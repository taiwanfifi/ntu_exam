## 教學特色

### 題型鑑別決策樹
遇到題目不知道用什麼方法？兩份「題型鑑別指南」提供：
- **機率**：離散 vs 連續分布判別、計算方法選擇（MGF/CDF/LOTUS）、分布關係圖
- **演算法**：DP vs Greedy vs D&C 判別、圖論演算法選擇、NPC 判定流程

### 完整推導與 Step-by-step 計算
每個定理、公式都包含：
- 為什麼要用這個方法（不是 A 不是 B 而是 C 的理由）
- 從零開始的推導過程
- 具體數字的計算範例

### 虛擬碼 + Trace Through
每個演算法都附：
- 完整虛擬碼
- 具體輸入的逐步執行過程（含表格）
- 時間/空間複雜度分析

### 零基礎友善設計
每章教學講義都包含：
- **基礎觀念區塊**：用生活化語言從零解釋核心概念
- **關鍵術語表**：中英對照 + 白話解釋 + 具體例子
- **豐富數值範例**：每個公式/定理後附完整的 step-by-step 計算
- **自我檢測題**：基礎→進階的漸進式練習（附參考答案）
- **直覺比喻**：Relaxation = 轉機比直飛便宜、MGF = 分布指紋、NP = 數獨驗證

## 涵蓋主題

### 機率與統計
基礎機率與計數 → 條件機率與貝氏定理 → 離散分布 → 連續分布 → 聯合分布與變數轉換 → 期望值、變異數與 MGF → 極限定理（LLN, CLT） → 統計推論（MLE, 假設檢定） → Poisson Process → Markov Chain → 資訊理論（Entropy, KL Divergence）

### 演算法
漸近分析與遞迴式 → 分治法 → 動態規劃 → 貪心演算法 → 圖論（BFS/DFS/SCC） → 最短路徑 → 最小生成樹 → 網路流 → NP 完全性 → 攤銷分析與進階資料結構

## 使用建議

1. **首次使用**：先閱讀 `台大考古題整理.md` 瞭解整體架構
2. **系統學習**：按 `tutorial/` 中的編號順序閱讀教學講義
3. **題型練習**：搭配題庫檔案，先自行嘗試再看詳解
4. **考前速查**：使用 `prob_00` 和 `algo_00` 的決策樹快速複習
5. **查漏補缺**：對照原始考卷確認是否遺漏重要題型
