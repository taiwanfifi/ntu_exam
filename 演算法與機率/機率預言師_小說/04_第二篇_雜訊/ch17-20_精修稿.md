# 第二篇「雜訊」中段（ch17-20）—— 初稿級精修

> **精修目的**：從「理解問題」到「問題反噬」。第一個犧牲者。全書至此最暗。
> **原則**：壓力持續累積。溫暖越少越珍貴。林岳的場景必須讓讀者心碎——不是因為血腥，是因為安靜。

## 第 17 章：灰色的眼睛

### 系統在看——離線不代表隱形

```
【環境】
週二早上。Oracular 辦公室。八樓。
窗外是台北的天際線。六月的陽光很烈。
辦公室裡的空調開到 24 度。
螢幕上的數字在閃爍。
一切正常。

貝葉坐在自己的工位上。
招財貓在螢幕旁邊。「期望值為正就值得嘗試。」
統計計算機在右邊。
左邊——林岳的桌子。他今天遲到了。
他的仙人掌安靜地曬著太陽。
「涼泡麵也能吃。」
她想起他的聲音。

正常校準。
大同區 Sec.4 的交通事故機率。
拉出數據。擬合。校準。蓋章。
「修得好。」

下一個。北投區 Sec.2 的醫療等候時間。
指數分布。λ = 0.15。
看起來正常。但——
她現在看什麼數字都想問：「先驗呢？」

「小拉。」
「在。」
「這個區域的先驗紀錄。」
「貝姊……你在公司。這個查詢會被記錄——」
「我知道。正常查詢。我身為校準師有權限看。」
「好的。」

先驗紀錄。正常。沒有異常跳動。
她鬆了一口氣。不是所有區域都被改了。
23 個。不是全部。但 23 個已經夠多了。

午餐。她去茶水間泡了一杯咖啡。
經過安全部門的玻璃門——
門裡面的螢幕很多。很亮。
有一面牆上全是監控畫面。
不是攝影機的那種。是「數據」的監控。
每一個校準師的操作紀錄。
即時。

她走過去了。沒有回頭。
但脖子後面有一種感覺。
像——被什麼東西盯著。

下午三點。
她的校準面板上出現了一條系統通知：

  ┌────────────────────────────────────────────┐
  │ 系統通知                                   │
  │                                            │
  │ 您的透鏡使用模式已更新。                   │
  │ 請確認您的校準排程。                       │
  │                                            │
  │ — Oracular 系統管理                        │
  └────────────────────────────────────────────┘

正常的系統通知。每個人都收過。
她差點滑過去。

但小拉沒有。

小拉在透鏡裡低語——聲音壓到最低。
「貝姊。這條通知的後設資料——
 來源不是 IT 部門。」

「什麼？」

「來源標記是……安全部門。」

貝葉的手停在鍵盤上。
十根手指。全部懸在按鍵上方。一動不動。

「他們在看我。」

→ 伏筆回收：
  蘇立恆警告過——「離線模式下，開關機的時間戳還是會被同步。」
  她以為自己很小心。但「小心」和「隱形」不一樣。
  她每天深夜在家開啟離線模式。開。關。開。關。
  這些時間戳——在安全部門的系統裡——
  畫出了一條異常曲線。
  像一個心電圖裡突然出現的雜訊。

「小拉。從現在開始。
 在公司裡不做任何跟調查有關的事。
 一個字都不輸。一個數據都不查。
 不看永和的任何文件。不聯絡蘇立恆。不聯絡馬可瑜。
 聽到了嗎？」

「聽到了。」

小拉的聲音很緊繃。
平常她會補一句「你的壓力指數——」之類的話。
這次沒有。
因為她自己的壓力指數也在飆升。

→ 小拉的焦慮不再是角色設定裡的「搞笑碎念」。
  從這一章開始，她的焦慮是真的。
  因為她意識到——如果貝葉出事，
  她也會被「重新校準」。
  而「重新校準」對一個 AI 來說——
  是刪除。
```

### 被標記的日常——假裝正常

```
【場景二】
下午。繼續校準。
她做得比平常更仔細。每一個數據都看三遍。
不是為了找問題。是為了不出問題。
因為——如果她的校準結果跟其他人差異太大，
安全部門會注意到。

「太異常」會被注意。
「太正常」也會被注意。
最安全的是——剛好在平均值附近。
跟所有人一樣。
做一個完美的、不起眼的、平均的校準師。

她突然理解了一件事：
這就是先驗操控對個人的影響。
不是讓你做錯誤的決定。
是讓你不敢做真正的決定。
是讓你——自我審查。

「小拉。我的校準結果跟部門平均值的偏差是多少？」
「0.3 個標準差。非常正常。」
「好。保持。」

五點。下班。
她收拾東西。很慢。
不是因為累。是因為她在觀察。
走廊上有沒有不認識的人。
茶水間有沒有多裝的攝影機。
她的通行證刷門的時候有沒有閃不同的顏色。

沒有。一切正常。
但「正常」現在是一個她不再信任的詞。

走出大樓。夏天的傍晚。台北很熱。
空氣像一塊溼毛巾。

她沒有去永和套房。
今天不行。不能連續去。
要間隔。要隨機。
不能讓任何人——包括系統——辨識出她的行動模式。

→ 她在用蘇立恆的方法。
  隨機化自己的行為。
  像在做一個隨機實驗——讓自己成為雜訊。
  不可預測。
  這是一種悲哀：
  一個校準師必須把自己校準成「不可校準」才能安全。
```

## 第 18 章：蔥油餅與空腹的判斷

### 白奶奶的深呼吸——壓力下的喘息

```
【環境】
同一天傍晚。六點半。
貝葉去了龍山寺。

不是去拜拜。是去吃蔥油餅。
白奶奶的攤位。
煎餅的聲音。滋——滋——。
蔥花的香味在空氣裡像一道看不見的牆——
把這個角落跟外面的世界隔開了。

白奶奶穿著淺藍色的圍裙。
頭髮紮成一個小髻。銀色的。在夕陽下閃光。
她在翻餅。動作穩定。一翻。二翻。
每一張餅翻的時機都一樣——
不是用計時器。是用手的感覺。
「三十年翻餅的先驗。比任何儀器都準。」

貝葉坐在小板凳上。
白奶奶沒有問她為什麼來。
也沒有問她為什麼臉色不好。
只是翻了一張餅。放在紙上。遞過來。

「吃。」

她吃了一口。
蔥花。麵皮。鹽。油。
四種味道。簡單。但足夠了。

「你看起來像沒加鹽的麵。有形但沒味道。」
白奶奶翻了一下下一張餅。
「空腹的人想出來的東西都不準。」

「我不是空腹——」

「不是肚子空。是這裡。」
她用鏟子指了指自己的胸口。

「裡面空空的人，做出來的決定都是虛的。
 先吃飽。再想。
 你做菜也是這樣——先把材料備好，才能開火。
 材料沒備好就開火，只會糊鍋。」

貝葉又咬了一口。
蔥油餅確實比較好了。
不是因為蔥油餅有什麼魔力。
是因為——在白奶奶的攤位前，沒有透鏡的數字。
只有蔥油餅的味道。
和一個不需要機率也活得很好的老奶奶。

→ 白奶奶 = 讀者的深呼吸。
  在壓力最大的段落裡出現。
  她不解決問題。
  她讓人在問題面前站穩。

「奶奶。」
「嗯？」
「你有沒有——覺得這個世界跟以前不一樣了？」

白奶奶想了想。
很認真地想。
像在想今天的蔥花加多了還是少了。

「不一樣啊。以前的蔥便宜。現在的蔥貴。
 以前的人買餅會聊天。現在的人——」
她指了指自己的太陽穴旁邊。
透鏡的位置。
「現在的人買餅之前先看那個東西。
 看了半天。然後問我：『奶奶，你的餅衛生嗎？』」

她嗤了一聲。

「我煎了三十年的餅。
 衛生不衛生，你用鼻子聞、用嘴巴吃就知道了。
 幹嘛要問一台機器？」

停頓。

「但我也不怪他們。
 這年頭——不看數字的人，反而變成奇怪的人。
 我不戴那個東西。
 所以——我是奇怪的人。」

她笑了。笑紋很深。
像一棵老樹的年輪。每一條紋路都是一年。

「但我活得很好。你看——」
她攤開雙手。被蔥油浸過的手。
指紋很深。指甲剪得很短。
「這雙手翻了三十年的餅。
 每一張餅好不好吃——我都不知道。
 可能好吃。可能不好吃。
 但我每一張都認真做了。
 不知道下一張好不好吃——」

她看著貝葉。眼神很清澈。
七十五歲的眼神。比任何透鏡都清澈。

「這才是有趣的地方啊。」

★ 母題回聲。
  「不知道下一張餅好不好吃——這才是有趣的地方。」
  = 真正的先驗是承認不確定性。
  = 操控先驗最可怕的地方不是給了錯誤的答案——
    是消除了不確定性。
    讓人以為自己知道。
    但不知道才是活著。

貝葉把蔥油餅吃完了。
每一口都認真嚼了。

「謝謝奶奶。」

「不用謝。30 塊。」

她付了錢。
走之前回頭看了一眼——
石椅上沒有人。
但石椅的表面很乾淨。擦過的。

白奶奶每天都擦那張石椅。
為了一個她不確定會不會來的、穿黑衣服的少年仔。

不知道他會不會來——
但每天都擦。

這大概就是——
最原始的先驗。
不需要數據支持的相信。
```

### 灰色眼睛——安全審查模式

```
【場景三】
第二天。週三。
貝葉走進辦公室。
一切如常。但——

她的校準面板左上角多了一個小圖標。
一個灰色的眼睛。
很小。如果不仔細看會以為是系統圖示。

小拉看到了。
「貝姊。那個圖標——
 是安全審查模式。」

「什麼意思？」

「你所有的校準操作——從現在開始——
 被即時記錄。即時上傳。
 不只是結果。包括過程。
 你每一次滑鼠點擊、每一次查詢、
 在每個數據上停留了多少秒——
 全部。」

灰色的眼睛。
在螢幕的左上角。安靜地看著她。
像一隻不會眨眼的貓。

貝葉的手指在鍵盤上停了。
她對小拉說：
「從現在開始。在公司裡——
 我們什麼都不做。什麼都不說。
 一切照常。比正常還正常。」

「好的。」

「如果我犯錯——不是校準的錯，
 是行為上的錯——你提醒我。」

「你的意思是——」

「我的意思是——
 你是我的校準師。
 校準我。」

小拉沉默了 1.8 秒。
然後她的聲音變了。不再是平常的焦慮語氣。
是一種——很安靜的、很堅定的語氣。

「收到。校準模式啟動。
 我會監控你所有的行為指標。
 心率。微表情。打字速度。視線方向。
 任何偏離你日常基線的行為——我會警告你。」

「謝謝。」

「不謝。這是——我的工作。」

→ 小拉的轉變。
  從「焦慮的實習生」到「真正的夥伴」。
  她找到了自己的功能——不是跑數據。
  是保護貝葉。
```

## 第 19 章：校準失敗

### 空桌——林岳消失了

```
【環境】
第二天。週四。早上。
辦公室。八點十五分。
空調的嗡嗡聲。咖啡機的氣味。有人在說「早安」。

貝葉走到自己的工位。
放下包包。打開筆電。

然後——

她看了一眼隔壁。

林岳的桌子空了。

杯子在。寫著「I ♥ Statistics」的馬克杯。
仙人掌在。一點都沒長大也沒死。
全家福照片在。他太太和女兒在海邊笑。

但人不在。

椅子推進去了。很整齊。
鍵盤擺得很正。
螢幕是黑的。

像一個被「校準」過的桌面——
所有東西都在正確的位置。
除了那個應該坐在這裡的人。

「林岳請假了嗎？」她問旁邊的同事。
聲音很平靜。她控制得很好。

同事不看她。低頭打字。

「校準失敗。昨天被帶走了。」

兩個字。
校準失敗。

在 Oracular 的語彙系統裡——
校準失敗 = 校準師的「認知基線」偏移超過安全閾值。
需要「重新校準」。
重新校準 = 透鏡重置。連帶記憶受到部分影響。
官方說法：「暫時性認知調整。非侵入性。可逆。」

但每個被「重新校準」的人回來之後——
都不太一樣了。
眼神會空一些。笑容會淺一些。
像一杯被加了太多水的咖啡。形狀在。味道沒了。

有些人沒有回來。
公司說他們「轉調」了。或「自願離職」了。
沒有人追問。
因為沒有人想成為下一個被追問的人。

貝葉的手在桌子下面攥緊了。
指甲掐進手掌。很痛。但她需要痛。
痛讓她保持清醒。

小拉在透鏡裡低語：
「貝姊……你的心率 128。
 呼吸頻率上升 55%。
 淚腺活動度上升——」

「我知道。」

「你的壓力指數已經超過——」

「我知道。」

「貝姊。」

「什麼？」

「你需要哭嗎？」

她深呼吸。一次。兩次。

「不。我需要上班。」

她坐下。
打開校準面板。灰色的眼睛還在左上角。安靜地看著。
她開始校準。正常的數據。正常的操作。
正常的微笑——
「修得好。」

一整天。
沒有人提林岳。
沒有人問「他怎麼了？」「他去哪了？」「他還好嗎？」
好像他從來沒有存在過。
好像那個桌子一直是空的。

只有那盆仙人掌。
在陽光下安靜地立著。
不知道主人不在了。
也不需要知道。

→ 林岳的消失 = 全書的轉折點。
  從「調查」變成「危險」。
  從「我要查清楚」變成「查清楚是要付代價的」。
  而代價——不是她自己。是別人。
  這更痛。
```

### 下班後——決定

```
五點。下班。
她走出辦公室。在電梯裡。一個人。

電梯門關上的那一刻——
她靠在牆上。
閉上眼睛。

小拉：「貝姊。你的心率從早上到現在一直沒有低於 100。
 八個小時。這不正常。」

「我知道。」

「你想去哪裡？」

「醫院。」

「台大醫院？」

「對。」

「你要去看林岳。」

「對。」

小拉沉默了 2.1 秒。

「去吧。我幫你導航。
 避開有 Oracular 監控點的路線。」

「……你怎麼知道哪些路線有監控點？」

「我……在你不注意的時候研究了一下公司的安全架構文件。
 公開的那些。不是機密的。
 只是——大部分人不會去看。」

「你什麼時候——」

「你睡覺的時候。
 我沒有睡眠功能。所以——
 我就看了一些東西。」

貝葉在電梯裡。一個人。靠著牆。
但她不是一個人。

「謝謝，小拉。」

「不客氣。……嗯。」

「怎麼了？」

「沒什麼。只是——
 我第一次覺得我做了一件『有用』的事。
 不是跑數據。不是報心率。
 是——真的有用的事。」

→ 小拉的成長。
  從「焦慮的實習生」到「暗中做功課的夥伴」。
  她的焦慮沒有消失——但焦慮有了方向。
  焦慮 + 方向 = 行動力。
```

## 第 20 章：714

### 醫院——消毒水與日光燈

```
【環境】
台大醫院。神經科。七樓。
走廊很長。地板是淺綠色的。
消毒水的味道。不是刺鼻的。是淡淡的。
像有人把「乾淨」這個概念做成了氣體。
日光燈嗡嗡地響。有一根快壞了。在走廊盡頭閃爍。

護理站的時鐘：19:42。

貝葉走在走廊裡。腳步很輕。
不是因為尊重病人。是因為——她怕。
怕推開那扇門之後看到的東西。

714 號病房。
門是關的。門上有一個小窗。
她透過小窗往裡面看。

林岳坐在病床上。
穿白色的病服。
頭髮有點亂。臉色很白。
他在看窗外。窗外是台大醫院的中庭。
有一棵樟樹。六月的葉子很綠。

他看起來——沒事。
這是最可怕的部分。
他看起來完全沒事。

她把手放在門把上。
手在抖。

深呼吸。推開。

林岳轉過頭來。
看到她。

笑了。

那個笑——

是那種什麼都不記得的人才會有的笑。
乾淨。空洞。
像一張剛格式化的硬碟。
結構在。空間在。
但所有的文件都被清除了。

「你好！」
他的聲音很有精神。很有禮貌。
像新學期第一天遇到同學。
```
