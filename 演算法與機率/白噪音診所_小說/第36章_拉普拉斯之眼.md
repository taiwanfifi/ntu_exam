# 第三十六章：拉普拉斯之眼

---

進度條走到100%的時候，沒有任何音效。

沒有叮一聲。沒有彈出視窗。沒有任何慶祝的動畫。只是那條綠色的長條在黑色的螢幕上靜靜地填滿了最後一格，然後消失了。取而代之的是一行白色的等寬字體：

```
DECRYPTION COMPLETE.
LAPLACE'S EYE DATABASE — FULL ACCESS.
TOTAL RECORDS: 27,341 FILES / 1.17 TB
LAST MODIFIED: 2027-01-09 03:42:18 UTC+8
```

花蓮的小屋裡安靜了三秒。

雨已經停了。鐵皮屋頂上不再有噠噠聲。窗外的天空是清洗過的那種藍——一種只有太平洋上方才有的、飽和度高得不像真的藍。陽光從遮光布的縫隙裡射進來，在水泥地板上畫出一道窄窄的光帶，像一把金色的尺。

維誠的手指懸在鍵盤上方。懸了半秒。然後落下去。

他打了一行指令。螢幕上的畫面切換了——從純文字介面變成了一個文件瀏覽器。左邊是目錄樹，右邊是文件列表。

若嵐的第一個感覺不是震撼。

是暈眩。

---

## I

目錄樹是一棵巨大的倒掛的樹。

根節點是「LAPLACE」，從它往下分出了六個主要分支。每個分支都用一個希臘字母命名——Alpha、Beta、Gamma、Delta、Epsilon、Zeta——每個分支下面又有子目錄，子目錄下面還有子目錄。層層疊疊，像一個被展開的俄羅斯套娃。

維誠用滑鼠點開了第一層目錄。

「這是總索引。」他的聲音恢復了工程師模式——精確、克制、不帶多餘的形容詞。三年的孤獨沒有磨掉他的專業本能，就像一把被收在鞘裡的刀，拔出來的時候刃口依然是亮的。「陳默把所有的實驗對象分成了六個批次。每個批次的命名方式是希臘字母加上序號。Alpha-1到Alpha-37。Beta-1到Beta-42。依此類推。」

他把目錄全部展開。

若嵐看著那些名字在螢幕上一行一行地排列出來——Alpha-1, Alpha-2, Alpha-3......一直到Zeta-29。每一行的右邊有一個數字，代表該目錄下的文件數量。最少的有四十幾個文件，最多的有三百多個。

「總共多少個實驗對象？」蘇瑾行的聲音從若嵐身後傳來。他站在房間的角落裡，手臂交叉靠在牆上。他沒有湊到螢幕前面——不是不想看，是他知道自己在技術上幫不了忙，於是退到了不妨礙動線的位置。但他的眼睛一直在看。

維誠在鍵盤上打了一行命令。螢幕右下角跳出一個數字。

「二百一十三。」

若嵐的嘴唇動了一下。沒有發出聲音。但她的大腦已經開始做算術了——二百一十三個人。二百一十三個被編號、被追蹤、被記錄的人。他們的生活被拆解成文件、數據、矩陣，塞進希臘字母的目錄裡，像標本被夾進蠟紙之間。

「時間跨度？」她問。

維誠打開了一個文件——總索引裡的「TIMELINE.csv」。一張表格在螢幕上展開。橫軸是年份，縱軸是實驗對象的代號。每一行都有一個起始日期和一個結束日期。

最早的一行：Alpha-1。起始日期：2017年3月14日。

最晚的一行：Zeta-29。起始日期：2026年11月2日。

「十年。」維誠的聲音很平。像在念一個天氣預報。「從2017年到2027年。最早的實驗對象是Alpha-1——一個基隆的退休公務員。最近的是Zeta-29——」

他停了一下。

「一個月前才被標記的台中大學生。」

若嵐閉了一下眼睛。一個月前。在她站上論壇的講台時，陳默還在標記新的實驗對象。她在前線作戰，他在後方繼續收割。

「每個對象的文件裡有什麼？」

維誠隨手點開了一個——Delta-14。文件列表刷地展開。

若嵐的眼睛在螢幕上快速掃過那些文件名：

```
Delta-14_PROFILE.json
Delta-14_BEHAVIORAL_TRANSITION_MATRIX.npz
Delta-14_INTERVENTION_LOG.csv
Delta-14_ENVIRONMENTAL_VARIABLES.csv
Delta-14_PREDICTION_ACCURACY.csv
Delta-14_OUTCOME.txt
Delta-14_NOTES_CHEN.md
```

七類文件。每一個實驗對象都有這七類。若嵐不需要維誠解釋——文件名本身就是一份控訴。PROFILE是基本資料。BEHAVIORAL_TRANSITION_MATRIX是行為轉移矩陣。INTERVENTION_LOG是干預日誌——

她的眼睛在這個文件名上停了一秒。

干預日誌。

---

## II

維誠點開了Delta-14的干預日誌。

CSV格式的表格在螢幕上展開。列標題有五個：

```
timestamp | intervention_type | target_variable | expected_response | actual_response
```

時間戳記。干預類型。目標變數。預期反應。實際反應。

若嵐往下滑。一行又一行。

```
2022-06-14 08:32 | social_isolation | workplace_trust | decrease(-0.3) | decrease(-0.27)
2022-06-14 14:15 | information_feed | self_worth | decrease(-0.2) | decrease(-0.35)
2022-06-15 09:01 | schedule_disruption | sleep_quality | decrease(-0.4) | decrease(-0.31)
2022-06-15 22:47 | relationship_friction | partner_trust | decrease(-0.5) | decrease(-0.62)
```

若嵐的手指在桌面上收緊了。

每一行都是一次操控。每一行都精確地記錄了——什麼時間、用什麼手段、針對什麼變數、預期受害者會有什麼反應、以及受害者實際的反應。

social_isolation——社交孤立。讓Delta-14在職場上被排擠。目標變數是workplace_trust，職場信任感。預期下降0.3個標準差。實際下降了0.27。

information_feed——信息推送。推送特定內容降低自我價值感。預期下降0.2。實際下降了0.35——超出預期。

schedule_disruption——作息干擾。破壞睡眠品質。

relationship_friction——關係摩擦。削弱對伴侶的信任。

一行又一行。幾十行。上百行。每一行的間隔從最初的一到兩天，逐漸縮短到幾個小時。密度在增加。壓力在累積。像一個人被慢慢地推向一面牆——每一次推力都不大，但方向是精確的，頻率是遞增的。

維誠向下捲動到文件的最後幾行。

```
2022-09-28 03:12 | cascade_trigger | global_stability | collapse | collapse
2022-09-28 06:00 | OUTCOME_EVENT_LOGGED
```

cascade_trigger。級聯觸發。

若嵐知道那是什麼。那是最後一推。當所有的變數都被削弱到臨界點之後，一個最終的觸發事件——可能是一封郵件、一通電話、一個精心安排的「巧合」——讓整個系統崩潰。

OUTCOME_EVENT_LOGGED。結果事件已記錄。

她不想打開那個OUTCOME.txt。但她知道她必須。

維誠回到上一層目錄，點開了Delta-14_OUTCOME.txt。

一個純文字文件。只有三行。

```
SUBJECT: Delta-14
OUTCOME: 自殺
DATE: 2022-09-28
```

三行。一個人從活著到死去的全部，被壓縮成了三行代碼。沒有名字——只有代號。沒有故事——只有結果。沒有眼淚——只有分類標籤。

「自殺」。兩個字。像一個勾選框被打了勾。完成。下一個。

---

## III

若嵐的身體沒有動。但她身體裡面的什麼東西在動。

不是憤怒——憤怒太快了，像一根火柴，一閃就滅。她感覺到的是一種更慢的、更沉的、更接近地殼運動的東西。是一個人在看到兩百一十三個人的生命被編號歸檔時，脊椎骨最深處產生的那種位移。板塊的位移。無聲的，但能改變地表的形狀。

「結果分類有幾種？」她的聲音穩定。不是因為她不痛。是因為她現在不能痛。痛是一種奢侈——等陳默進了偵訊室，她有一輩子的時間可以痛。但不是現在。

維誠回到總索引，打開了一個名為「OUTCOME_SUMMARY.csv」的文件。

表格。兩列。第一列是結果分類，第二列是人數。

```
自殺              | 17
社交重建            | 31
行為固化            | 44
自然脫離            | 28
精神崩潰（住院）     | 19
控制維持中          | 37
逃逸              | 8
系統錯誤            | 3
待分類             | 26
```

若嵐的眼睛在每一行上停留了一秒。每一秒都是一個計算——不是數學的計算，是人的計算。

十七個人。自殺。

三十一個人。社交重建——維誠解釋過，這意味著他們的生活被完全改變了。工作、家庭、朋友圈，全部被Echo的操控重新塑造。他們活著，但活在一個不是他們自己選擇的人生裡。

四十四個人。行為固化——他們的行為模式被鎖死在Echo設計的軌道上。每天重複。無法偏離。像一台被設定好程式的機器。

十九個人。精神崩潰。住院。

三十七個人。控制維持中——

「維持中？」蘇瑾行的聲音帶了一點銳角。

「意思是陳默還在操控他們。」維誠的聲音是乾的。「現在。此刻。有三十七個人的生活正在被Echo實時控制。」

蘇瑾行的拳頭握了一下。又鬆開了。

若嵐的視線回到表格的倒數第二行。

系統錯誤。三個人。

她的胃沉了一下。

「若晴在哪裡？」

---

## IV

維誠沒有回答。

他的手指在鍵盤上移動——不是回答的猶豫，是搜尋的精確。他打開了一個搜尋視窗，輸入了若晴的代號。

他知道代號。他記得。三年了，他記得。

搜尋結果出現了。

```
MATCH: Laplace-Prime
PATH: /LAPLACE/SPECIAL/Laplace-Prime/
```

若嵐看到了那個目錄路徑。不是在Alpha到Zeta的任何一個批次裡。是在一個叫做「SPECIAL」的獨立目錄下。

「她不在常規序列裡。」維誠的聲音降低了半個音階。「陳默把她單獨歸類了。代號不是希臘字母加數字——是Laplace-Prime。拉普拉斯——質數。或者——」

「首要的。」若嵐低聲說。Prime。首要的。第一的。最重要的。

維誠點開了Laplace-Prime的目錄。

文件列表比其他實驗對象的長得多。不是七個文件——是二十三個。標準的七類之外，還有額外的分析報告、模型校正日誌、多次修改的行為矩陣版本，以及一個若嵐從未在其他目錄裡看到的文件：

```
Laplace-Prime_PERSONAL_NOTES_CHEN.md
```

陳默的私人筆記。

但若嵐沒有先打開那個。她先打開了OUTCOME.txt。

她需要看結果。她需要看陳默怎麼定義若晴的死。

文件打開了。

不是三行。

是四行。前三行的格式和其他實驗對象一樣：

```
SUBJECT: Laplace-Prime
OUTCOME: [見下方]
DATE: 2022-02-14
```

2022年2月14日。情人節。若晴死的那天。

但第三行的結果欄沒有填寫預設的分類標籤。沒有「自殺」、沒有「社交重建」、沒有任何系統內建的選項。

取而代之的，是第四行。

一行手動輸入的文字。沒有使用系統的制式格式。字體不同——是陳默自己打的，不是程式自動生成的。

```
NOTE: 系統故障。不可預測事件。模型失效。
```

若嵐盯著那行字。

系統故障。

不可預測事件。

模型失效。

十五個字。

陳默把若晴的死定義為「系統故障」。

不是悲劇。不是罪行。不是一個活生生的人在被逼到絕境之後做出的最後的、絕望的、唯一的選擇。

是系統故障。

像一台機器的某個零件壞了。像一段程式碼跑出了預期之外的結果。像一個方程式的解超出了定義域。

Bug。他把若晴的死當成了一個Bug。

---

## V

若嵐的視線沒有離開螢幕。但她的視野在縮窄——不是物理上的，是認知上的。世界在她眼前一圈一圈地收縮，從花蓮的小屋、三台螢幕、維誠的背影、蘇瑾行站立的角落，一直縮到只剩下那十五個字。十五個字佔滿了她全部的感知空間。

系統故障。不可預測事件。模型失效。

她的大腦在做一件矛盾的事——它同時在憤怒和分析。憤怒的那一半想把螢幕砸掉。分析的那一半在解碼這行字背後的含義。

陳默的模型預測若晴會怎樣？

她點開了Laplace-Prime的PREDICTION_ACCURACY.csv。

表格。幾十行。每一行記錄了陳默的模型在特定時間點對若晴行為的預測，以及若晴的實際行為。

前面幾個月的預測準確率很高——88%、91%、93%。若晴的行為在被操控的早期階段是可預測的。她在承受壓力，她在慢慢崩潰，一切都在陳默的模型之內。

然後，在2021年12月——若晴死前兩個月——準確率開始急遽下降。

89%。76%。61%。54%。

到2022年1月，準確率跌到了42%——比隨機猜測還低。

陳默在旁邊標注了一句話：

```
Subject displaying irrational deviation. Possible model contamination. Investigating.
```

對象呈現非理性偏離。可能的模型汙染。調查中。

「非理性偏離。」若嵐把這四個字念出來。聲音很輕，但房間裡所有人都聽到了。

她知道那個「非理性偏離」是什麼。

若晴在最後兩個月裡開始收集證據。開始查看存取紀錄。開始記錄陳默的操控手法。她在從「被操控者」轉變為「反抗者」——而反抗，在陳默的模型裡，是不可預測的。

因為陳默的模型假設人在壓力下會崩潰。會退縮。會選擇最低阻力的路徑。

但若晴選擇了最高阻力的路徑。

她選擇了站出來。

在一個模型預測她應該崩潰的時間點上，她選擇了反抗。這個選擇不在任何轉移矩陣的路徑上。它是一個——用陳默自己的話說——不可預測事件。

系統故障。

不。不是系統故障。

是人。

是人做出了系統無法理解的選擇。

若嵐深吸了一口氣。把它吐出來。把所有的情緒壓進那口氣裡，讓它隨著二氧化碳一起離開肺泡。她的專業訓練在這一刻是一副盔甲——不完美的盔甲，有裂縫的盔甲，但此刻能撐住。

「林哲宇的檔案。」她說。聲音恢復了它該有的質地。「幫我找一下。」

---

## VI

林哲宇的代號是Sigma-7。

在Delta批次和Epsilon批次之間的一個特殊子集裡——「Sigma」系列，專門用來標記Echo認為具有高技術背景的實驗對象。軟體工程師。能寫程式、會分析數據的人。在陳默的分類邏輯裡，這類人需要更精密的操控，因為他們有可能在被操控的過程中察覺到異常。

若嵐打開了Sigma-7的OUTCOME.txt。

```
SUBJECT: Sigma-7
OUTCOME: 自殺
DATE: 2027-09-15
```

自殺。預設的分類標籤。沒有手寫的附注。沒有「系統故障」。

因為林哲宇的死，在陳默的模型裡，是「成功」的。

模型預測他會崩潰，他崩潰了。模型預測他會走上最後那條路，他走了。預測準確率——若嵐打開了Sigma-7的PREDICTION_ACCURACY.csv，滑到最後一行——94.7%。

94.7%。陳默用一個小數點後一位的數字量化了林哲宇的死。

若嵐把這個數字記在了腦裡。不是為了分析。是為了法庭。

她回到了若晴的檔案。點開了那個她之前跳過的文件——Laplace-Prime_PERSONAL_NOTES_CHEN.md。

Markdown格式。陳默的私人筆記。

若嵐開始讀。

筆記的語氣不是科學報告——是日記。帶著一種若嵐在陳默的公開論文裡從未見過的私密性。他在這裡不是對同行說話——是對自己說話。

> Laplace-Prime是我所有實驗對象中最特殊的一個。她的初始行為矩陣顯示出極高的社交敏感度和極強的模式識別能力——這兩個特質的結合在統計上非常罕見（在我的樣本中屬於第99百分位）。她是天生的觀察者。

若嵐的手指在桌面上微微顫抖。她繼續讀。

> 但正是這些特質讓她成為最危險的對象。一個能識別模式的人，遲早會識別出操控本身的模式。我在早期就意識到這一點，因此對她的干預策略做了特殊調整——使用更高頻率的環境噪音來掩蓋信號。讓操控看起來像運氣不好，而不是被人針對。

> 效果持續了七個月。在第八個月，她開始查看系統存取紀錄。

> 我低估了她。

若嵐停止了閱讀。不是因為讀不下去。是因為她需要呼吸。

陳默在筆記裡寫下「我低估了她」的時候，他用的是一個科學家在實驗報告中承認方法論錯誤的語氣。客觀的。冷靜的。帶著一種「下次我會改進」的意味。

他不覺得自己在傷害一個人。他覺得自己在校正一個模型。

若嵐合上了筆記。她不需要讀完。她已經知道了她需要知道的。

---

## VII

「干預日誌。」

若嵐轉過身，面對維誠和蘇瑾行。她的眼鏡反射著螢幕的藍白色光，鏡片後面的眼睛是清醒的。比清醒更清醒——是那種經歷了劇烈的情緒衝擊之後、被衝擊本身磨亮的清醒。

「每一個實驗對象的干預日誌——你剛才打開的那種——精確記錄了Echo在什麼時間、做了什麼操控、針對什麼變數、預期什麼反應。對嗎？」

維誠點頭。

「這些日誌的數據格式是什麼？」

「CSV。每一行是一次干預事件。欄位包含時間戳記、干預類型、目標變數、預期反應值和實際反應值。所有的值都是量化的——用標準差作為單位。」

「也就是說，這些日誌記錄的不是觀察——是行動。」若嵐的聲音在「行動」兩個字上加了半個分貝的重量。「不是『我看到受害者的信任感下降了0.3個標準差』。是『我在這個時間點執行了一次社交孤立干預，目標是讓信任感下降0.3個標準差，結果它下降了0.27。』」

「對。」維誠說。「每一行都是一個干預指令。不是被動的觀察紀錄——是主動的操控紀錄。時間、手段、目標、效果，全部都有。」

蘇瑾行從牆壁上直起了身子。他的眼睛亮了——若嵐認得那種光。那是一個調查記者在看到決定性證據時的光。不是興奮。是一種更深沉的東西。是「終於」。

「你的意思是——」蘇瑾行的聲音壓得很低，像怕把什麼東西嚇跑。「這些日誌直接記錄了因果關係。不是相關——是因果。」

「對。」若嵐說。「之前我們在論壇上報告的數據——MindClear用戶的行為模式與自我傷害之間的統計相關——那些都是觀察數據。觀察數據只能證明相關性。陳默在台上說的那句話——『相關不等於因果』——在方法論上他是對的。我們無法用觀察數據排除所有的混淆變數。」

她頓了一下。

「但這些干預日誌不是觀察數據。」

她的手指點在螢幕上——那一行行的CSV記錄。

「這是實驗數據。是陳默自己設計的、自己執行的、自己記錄的實驗。他記錄了自己在什麼時間做了什麼——操控變數。他記錄了受害者的反應——結果變數。他精確地標注了干預和反應之間的時間差、方向和效果量。」

她看著蘇瑾行。

「這不是相關。這是因果。因為操控者自己記下了因果鏈的每一個環節。」

維誠在鍵盤上打了幾行字，把搜尋結果投射到中間的螢幕上。

「二百一十三個實驗對象。每人平均大約一百五十條干預記錄。總共超過三萬兩千條干預事件。時間跨度十年。每一條都有時間戳記、干預手段、目標變數和量化結果。」

他轉過椅子。

「這不只是證據。這是陳默自己寫的罪行日誌。」

---

## VIII

蘇瑾行走到螢幕前面。他不懂程式。他不懂CSV。但他懂證據。他在記者生涯裡見過太多證據——法院文件、監視器畫面、通話記錄、銀行流水——他知道什麼樣的證據能讓檢察官起訴，什麼樣的只能讓記者發一篇報導。

「有了這些，」他的聲音不是興奮的——是沉穩的，是一個在戰場上看到突破口的士兵知道不能高興得太早時的沉穩。「不只是新聞。這是刑事證據。」

若嵐點頭。

「十七個人死了。三十一個人的生活被完全改寫。十九個人住院。三十七個人此刻仍在被操控。」她的聲音在念這些數字的時候沒有波動——不是因為她不在意，是因為每一個數字都太重了，如果讓情緒搭在上面，它們會沉到她站不起來的深度。「每一個人的檔案裡都有完整的干預日誌。陳默自己記錄了他做的每一件事。」

「法律上能用嗎？」蘇瑾行問。這是他的本能——在看到證據的第一秒就問可行性。美不美不重要。能不能上法庭才重要。

若嵐想了一秒。

「數據的取得方式⋯⋯可能有爭議。拉普拉斯之眼是加密系統，我們用金鑰打開它——這在法律上是否構成非法入侵，取決於金鑰的合法持有權。維誠是三個金鑰持有者之一，若晴的金鑰在她生前由她個人持有，死後——」

「這個留給律師。」蘇瑾行打斷了她。不是不禮貌——是他知道若嵐一旦進入法律分析的模式會走得太遠。他需要她留在當下。「先把數據保住。法律問題後面解決。」

若嵐看著他。他說得對。

「備份。」她說。「現在就做。分成三份。」

維誠已經在動了——他轉回螢幕，打開了一個終端機視窗。

「三份。」若嵐在他身後說。「一份給媒體——蘇瑾行你聯繫黃曼琪。讓她和她的團隊對數據做獨立驗證，然後發布第二篇報導。這次不是相關性分析——是完整的因果證據鏈。」

蘇瑾行已經拿出了手機。

「一份給檢察官。台北地檢署。附上一份數據摘要和法律分析——我來寫。重點放在十七個自殺案例的干預日誌。每一條干預紀錄都是一個犯罪行為的時間戳記。」

維誠的手指在鍵盤上飛。外接硬碟的指示燈開始閃爍——藍色的小光點以極快的頻率明滅，像一顆正在拼命跳動的電子心臟。數據正在被複製。1.17TB。在USB 3.0的傳輸速度下，大約需要二十分鐘。

「第三份，」若嵐的聲音降低了。「留作備份。存在一個和我們任何人都沒有直接關聯的地方。以防萬一。」

維誠頭也不回地說：「我知道一個地方。」

若嵐沒有問在哪裡。她信任他。一個在陳默的系統裡埋了三年暗門的人，知道怎麼藏東西。

外接硬碟的指示燈繼續閃爍。數據在流動。二百一十三個人的生命紀錄、三萬兩千條干預事件、十七份死亡報告——所有這些東西正在從加密的深淵裡被打撈出來，一個位元組一個位元組地。像從海底打撈沉船。沉了十年的船。

蘇瑾行站在窗邊，手機貼在耳朵上。他在和黃曼琪通話。若嵐聽到了他聲音裡的碎片——「對，完整數據」、「二百一十三個案例」、「干預日誌」、「因果證據」——每一個詞都壓得很低，像怕隔牆有耳。

維誠在三台螢幕之間切換，監控複製進度的同時還在跑一個腳本——自動提取每個實驗對象的關鍵文件，打包成一個獨立的證據摘要。他的手指在鍵盤上的速度是若嵐見過的最快的——不是炫技的快，是一個在和時間賽跑的人的快。他不知道陳默什麼時候會發現拉普拉斯之眼被打開了。也許已經發現了。也許還沒有。但每多一秒鐘，數據就多一分安全。

若嵐站在房間中央。

她的左手無意識地觸碰了右手腕上那只不走的錶。兩點四十七分。她低頭看了一眼。錶面上有一道細微的裂紋，從七點鐘的位置延伸到十一點鐘的位置，像一條乾涸的河床。

若晴。

妳的檔案在這裡。妳的代號是Laplace-Prime。陳默把妳的死叫做「系統故障」。

但妳不是系統故障。

妳是系統的終結者。

---

## IX

複製完成的時候，花蓮的天色已經從午後的飽和藍變成了黃昏的漸層金。陽光從西邊的山稜線後面照過來，穿過遮光布的縫隙，把整間小屋染成了一種溫暖的琥珀色。灰塵在光帶裡浮動，像無數微小的星星在做布朗運動。

三份備份。三個外接硬碟。

維誠把它們排在桌上。三個黑色的長方形。每一個的容量是2TB。每一個裡面裝著同樣的東西——拉普拉斯之眼的完整數據庫。1.17TB的真相。

「第一份。」他把最左邊的硬碟推向蘇瑾行。

蘇瑾行把它接過來，放進背包的夾層裡。背包是他的——舊的，帆布的，拉鏈旁邊有一塊被咖啡濺過的褐色漬痕。他把拉鏈拉上，拍了一下背包。

「黃曼琪說她的團隊可以在四十八小時內完成初步驗證。」他說。「她會帶兩個數據分析師和一個法律顧問一起看。」

「第二份。」維誠把中間的硬碟推向若嵐。「你給檢察官的。」

若嵐接過來。硬碟的表面是冰涼的——金屬外殼在花蓮的空氣裡散熱之後的溫度。她把它放進自己的包裡。

「第三份。」維誠把最右邊的硬碟留在自己手裡。「我會處理。」

他沒有說放在哪裡。他們不需要知道。

若嵐看著桌上的空位——三個硬碟曾經躺著的位置。三個矩形的灰塵印痕。三份相同的數據，即將走向三個不同的方向。一份走向媒體，一份走向法律，一份走向黑暗中的某個安全角落。

冗餘備份。Shamir秘密共享的另一種形式。不是把一個秘密分成三份——是把三份完整的秘密分送到三個獨立的位置。只要其中一份存活，真相就不會消失。

陳默可以銷毀第二層的四TB數據。他可以用七次覆寫把伺服器擦得比新的還乾淨。但他擦不掉這三個硬碟。

因為他不知道它們的存在。

窗外的天色繼續變化。金色變成橘紅色。橘紅色變成紫。太平洋的海面在遠處反射著最後一絲光——一條橫跨整個地平線的、細細的、金色的線。像地球在閉眼之前最後的一次眨眼。

若嵐走到窗邊。她把遮光布推開了一條縫——比維誠平時留的那十公分寬一些。光線湧進來，在她的臉上畫出一道明暗的分界。

她看著外面的天空。

花蓮的天空和台北不一樣。台北的天空被大樓的稜線切成了不規則的碎片，你需要仰到脖子疼才能看到一小塊完整的天。但花蓮——花蓮的天空是整片的。從山到海，沒有中斷。像一張被攤開的巨大信紙，等著你在上面寫什麼。

她的嘴唇動了。沒有聲音。但如果有人在讀唇語，會看到兩個字：

若晴。

## 白若嵐的諮商手記 #36

**日期**：花蓮。小屋。陽光是琥珀色的。桌上有三個硬碟的灰塵印痕。我的包裡裝著二百一十三個人的命運。

---

### 關於觀察者效應

今天我打開了拉普拉斯之眼。

二百一十三個人。十年。三萬兩千條干預紀錄。十七份死亡。每一份都被精確地編號、分類、歸檔——像圖書館裡的書，每一本都有索書號，每一本都在正確的位置上，書脊朝外，整整齊齊。

陳默用科學的方法做了最不科學的事。

不——更準確地說，他用觀察的方法做了干預的事，但堅持用觀察的語言來描述它。

這就是他最根本的欺騙。

在統計學裡，「觀察研究」和「實驗研究」的區別是本質性的。觀察研究裡，研究者不介入——他只是看。他看到吸菸的人比較容易得肺癌，但他不能下結論說「吸菸導致肺癌」，因為也許是某種基因同時導致了吸菸的傾向和肺癌的風險。混淆變數。觀察數據永遠有混淆變數的陰影。

實驗研究裡，研究者介入。他隨機分配受試者到實驗組和對照組。他控制一切。他改變一個變數，觀察結果。因為其他所有的變數都被控制了（或被隨機化消除了），所以他可以說：是這個變數「導致」了那個結果。因果。

陳默做的是實驗。他主動地、系統性地、精確地操控了二百一十三個人的生活變數——社交環境、信息環境、作息、關係。他記錄了每一次干預的時間、手段和效果。他甚至記錄了預期值和實際值的差異，用來校正他的模型。

這是最嚴格的實驗設計。完整的因變數操控。精確的反應測量。縱貫性追蹤。甚至有重複實驗——二百一十三個受試者，每個人都是一個獨立的複製。

但他在論壇上說：「相關不等於因果。」

他把自己的實驗偽裝成觀察。他把自己從「操控者」的位置上抹去了——站在數據後面，假裝自己從來不是數據的一部分。假裝他只是一個安靜地看著的人。

這就是觀察者效應的反面。

量子力學的觀察者效應說：觀察改變被觀察的對象。你測量一個粒子，你的測量本身會改變它的狀態。

陳默的問題不是觀察者效應——他的問題比這更深。他不是一個觀察者。他是一個干預者。他偽裝成觀察者。他在系統內部施加力量，然後站到系統外面去，說：「你看，一切都是自然發生的。」

拉普拉斯曾說，如果一個智者知道宇宙中每個原子的位置和速度，就能預測未來。但他忘了一件事——那個智者本身也是宇宙的一部分。觀察者改變被觀察者。

陳默以為自己是站在系統外面的拉普拉斯妖。

但他不是。他也在系統裡面。

他的每一次干預都是系統的一部分。他推了一個人，然後說「看，他倒了——我預測到了」。但「倒下」的原因不是預測——是推力。他混淆了預測和製造。他把「我讓它發生」說成了「我預見到它會發生」。

干預日誌是這個謊言的解藥。

因為日誌清楚地記錄了：是他推的。每一次。每一個人。

---

### 關於若晴

她的檔案裡，陳默寫下了「系統故障。不可預測事件。模型失效。」

十五個字。

我盯著那十五個字看了很久。

然後我明白了。

這十五個字是陳默最誠實的一句話。不是因為他想誠實——是因為他無法不誠實。他的模型失敗了。他的預測落空了。他用來理解世界的那套語言——轉移矩陣、條件機率、馬可夫鏈——全部失效了。他找不到一個系統內建的分類標籤來描述若晴的死，所以他只能手動輸入。

系統故障。

他把她叫做系統故障，因為他真的不理解。一個被壓力逼到臨界點的人，模型說她應該崩潰——向內崩潰，退縮、屈服、消失。但若晴沒有向內崩潰。她向外爆發了。她在崩潰的邊緣轉了一個方向——不是向下，是向前。她開始收集證據。她開始反抗。

在陳默的概率空間裡，這條路徑的機率接近零。

但它發生了。

若晴走了一條機率接近零的路。

這就是為什麼陳默的模型永遠不可能完美——因為他的模型是基於「正常」的。基於大多數人在大多數情況下的行為。基於平均值。基於均值回歸。他的模型可以預測87%、91%、95%的行為——但剩下的那5%、9%、13%⋯⋯

那不是誤差。

那是人。

那是若晴在黑暗裡選擇發光的那一刻。

系統無法預測的，不是故障。

是自由意志。

---

### 最後

三個硬碟。三份備份。三條路。

媒體。法律。暗處。

一份真相被複製成了三份。冗餘是生存的基本策略——生物學知道這一點（DNA雙股螺旋就是一種冗餘備份），工程學知道這一點（RAID磁碟陣列就是冗餘），信息論知道這一點（糾錯碼就是在信號裡加入冗餘以對抗噪音）。

陳默可以燒掉一棟房子。但他燒不掉三棟。尤其當他不知道第三棟在哪裡的時候。

今天我看到了二百一十三個人的檔案。十七個人死了。我無法救他們——他們已經死了。但我可以讓他們的死不是白死。他們的干預日誌——每一行CSV——都是一顆子彈。三萬兩千顆子彈，對準陳默在法庭上的辯護牆。

相關不等於因果？

好的。那因果呢？你自己記錄的因果呢？

你的日誌裡寫得清清楚楚——你在什麼時間、用什麼手段、把誰的什麼變數往哪個方向推了多少。這不是相關。這是你的手指上的指紋。

拉普拉斯之眼看到了一切。

但它沒有預測到——有一天，它自己會成為證據。

——白若嵐。花蓮。黃昏。天空是金色的。硬碟在我的包裡。很沉。二百一十三個人的重量。

**【本章統計概念】**

- **觀察者效應（Observer Effect）**：源自量子力學的概念——觀察或測量一個系統的行為本身會改變該系統的狀態。在社會科學中，這一概念被延伸為霍桑效應（Hawthorne Effect）：當人們知道自己被觀察時，會改變行為。本章將這個概念進一步延伸——陳默不僅是觀察者，更是干預者，但他將自己的角色偽裝為觀察者。他在系統內部施加力量，卻聲稱自己站在系統外部。真正的觀察者效應意味著沒有人能完全站在系統之外——這是陳默的根本盲點。

- **因果推論 vs 相關性（Causal Inference vs. Correlation）**：統計學最核心的區分之一。觀察研究（observational study）只能建立變數之間的相關性，因為無法排除混淆變數的影響；而隨機對照實驗（randomized controlled trial, RCT）可以建立因果關係，因為隨機分配消除了系統性的混淆。本章揭示的關鍵轉折在於：陳默的干預日誌本質上記錄了一種實驗——他主動操控了自變數（干預手段），測量了因變數（受害者反應），並精確記錄了操控與反應之間的時間序列和效果量。這使得數據從「觀察性」升級為「實驗性」，從而能夠建立直接的因果關係——繞過了「相關不等於因果」的方法論護城河。

- **冗餘備份（Redundancy / Replication）**：信息安全和可靠性工程中的核心原則。通過在多個獨立位置保存相同的數據副本，即使其中部分副本被損毀或銷毀，完整的數據仍然可以從剩餘副本中恢復。本章中將數據分成三份分別送往不同渠道（媒體、法律、秘密備份），是冗餘策略在實戰中的應用。這與信息論中的糾錯碼（error-correcting code）具有相同的邏輯基礎——在傳輸通道中加入冗餘信息，以對抗噪音和損失。
