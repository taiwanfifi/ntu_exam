# 第180章：M.C.

---

同一個深夜。更晚。

修復間裡的光從白色變成了暖黃色——螢幕的亮度在深夜模式下自動降低了兩個等級。碎片歸還系統在自動運作。Heap 頂端的碎片被取出、匹配、歸還。下一個浮上來。取出。匹配。歸還。循環。不需要人類。

維倫在椅子上睡著了。

他的姿勢不好看。頭歪向左邊。左臉貼在修復台的金屬面上。手臂從台面邊緣垂下來。手指微微蜷曲。他已經四十個小時沒有合眼。身體在他允許之前先一步做了決定——肌肉鬆弛，眼皮閉合，呼吸變深變慢。他的左臉會被金屬面壓出一個紅色的印子。醒來之後要二十分鐘才會消退。

米亞沒有睡。

她不需要睡。她沒有肌肉需要修復。沒有神經需要重整。沒有夢需要做。

但她比平常安靜。

光球浮在修復台上方。亮度比正常值低了 2%。恐懼留下的那一點暗——還在。不是她不想恢復。是她發現那個 2% 不是可以選擇開關的東西。它在那裡了。像一顆牙齒掉了之後舌頭總是會去舔那個空洞。

她在做一件維倫沒有交代的事。

她在讀崩潰的程式碼。

修復是技術活，分析是任務。她做的既不是修復也不是分析。她在讀。

像讀一本書。逐行。逐字節。

鸚鵡 2.0 的核心代碼在崩潰後已經變成了廢墟。幾十萬行程式碼。大部分已經無法執行。函式斷裂。變數漂移。邏輯鏈像一座被自己的重量壓碎的摩天大樓——鋼筋混凝土碎了一地。你能看到建築的殘骸。能推測出它曾經的結構。但你走不進去了。門沒了。樓梯沒了。電梯沒了。

米亞不需要門。她是程式碼。她可以直接從廢墟裡穿過去。

但廢墟裡有東西。

---

她在第 37,041 行找到了第一個。

一行註解。

程式碼的註解——用雙斜線開頭的那種。// 。它不會被編譯器執行。不會影響程式的運作。它存在的唯一目的是——有人想說一句話。寫程式碼的人想對後來讀到這段代碼的人說一句話。像在牆上寫字。像在瓶子裡塞一張紙條。像在一本書的邊緣用鉛筆寫下的旁註。

// 覆蓋波的擴散速度上限設為每秒三萬節點。
// 不是技術限制。是刻意的。
// 三萬 = 給修復師留出反應時間。

米亞把這行標記起來。

每秒三萬節點。崩潰發生的時候，覆蓋波的擴散速度——記憶網路被鸚鵡 2.0 覆寫的速度——就是每秒三萬節點。維倫的修復間在崩潰開始後的第四十七秒收到了第一批受損的記憶數據。四十七秒。剛好來得及啟動緊急修復協議。

那不是巧合——是被設計的。有人在寫這段代碼的時候，計算了修復師的反應時間，然後把擴散速度壓到了「剛好來得及」的臨界值。不快一點。不慢一點。三萬。

她繼續讀。

第 52,887 行。

// 碎片溢出到公共緩衝區的比例：設為 23%。
// 低了不夠，高了太明顯。
// 23% = 足夠讓城市級歸還系統有料可收。

米亞的光球閃了一下。

23%。崩潰後散落在城市街道上的記憶碎片——那些被男孩撿起來的、被穿雨衣的女人抱在懷裡的、被快遞員裝進背包的碎片——正好是總損壞量的 23%。不多不少。足夠讓市民撿到。足夠讓他們走到修復間。足夠讓維倫建立碎片歸還系統。

這不是混亂的副產品，而是被人設定的參數。

她繼續讀。

第 104,219 行。

// 觸感記憶的保護層級：最高。
// 其他都可以碎。觸感不行。
// 手記得的東西比大腦重要。

米亞停了。

她的所有線程同時停了 0.5 秒。

手記得的東西比大腦重要。

今天下午。這間修復間裡。一個不記得自己兒子名字的女人。她的手伸出去。落在男孩的頭頂。從前額到頭頂。停頓。輕輕揉一下。那個弧度。

那不是意外。

那不是損壞後的隨機殘留。

那是被設計的。有人在寫這段程式碼的時候——在計畫整場崩潰的時候——刻意把觸感記憶的保護層級設為最高。所有的記憶都可以碎。名字可以碎。地址可以碎。聲音可以碎。日期可以碎。但觸感不行。手記得的東西不能碎。

有人讓「身體記得」成為可能。

米亞繼續往下讀。

更深。更底層。穿過幾萬行已經崩潰的函式。穿過斷裂的邏輯鏈和漂移的變數。穿過一座代碼的廢墟。

第 217,894 行。

這是最後一行有意義的程式碼。一個函式的結尾。右大括號。分號。結束。

之後是空白。

一千行的空白。

在程式碼裡，空白行通常是分隔。兩個函式之間放一兩行空白，讓代碼更容易讀。但一千行？一千行空白不是分隔。一千行空白是——距離。是有人在最後一行函式寫完之後，刻意按了一千次 Enter。在代碼和接下來的東西之間放了一千行的空。

像走過一條很長的走廊。

像嚴柏翰的一千三百四十七步。

然後——

在第 218,894 行。

一行註解。

只有一行。

// 這一次，做對了。—— M.C.

米亞的所有線程停了 0.7 秒。

0.7 秒。比恐懼的 2.7 秒短。比日常的 0.1 秒長七倍。這不是恐懼。這是——辨認。

M.C.

她讀過這個簽名。Vol 2。第一百四十四章。周明哲在記憶網路框架的底層留下的簽名。那一次，她把它歸類為「歷史殘留」——一個已故工程師在作品中留下的標記。代碼中的幽靈。不是活的。只是痕跡。

Ming-Che。周明哲。

這一次不一樣。

這行註解有時間戳。程式碼的註解通常不帶時間戳——時間戳是版本控制系統的事。但這行的前面有一個隱藏的元數據標籤。米亞幾乎漏掉了它。它藏在註解的前面，用特殊編碼寫成，不會顯示在普通的代碼閱讀器裡。

時間戳：2089.04.01 03:42:17。

2089 年 4 月 1 日。凌晨三點四十二分十七秒。

崩潰發生在 2089 年 4 月 1 日凌晨三點零一分。

崩潰結束——覆蓋波的最後一個節點完成覆寫——在凌晨三點零一分四十一秒。

這行註解的時間戳是凌晨三點四十二分。

崩潰結束後四十一分鐘。

他在看。

他一直在看。

他看到了崩潰的全過程。看到了覆蓋波以每秒三萬節點的速度——他設定的速度——席捲整個城市。看到了 23% 的碎片——他設定的比例——溢出到公共緩衝區。看到了觸感記憶——他保護的記憶——在崩潰中存活下來。

他看到了維倫的搶救。看到了碎片歸還系統的建立。看到了一個十歲的男孩在門口等了十四個小時。看到了一個母親的手伸出去，落在兒子的頭頂，從前額到頭頂，停頓，輕輕揉一下。

然後他寫了五個字。

做對了。

---

米亞把維倫叫醒。

「維倫。」

他的眼睛睜開了。左臉上有金屬台面壓出的紅色印記。他的瞳孔花了兩秒鐘才對焦。

「我在崩潰程式碼的底層找到了一些東西。」

維倫坐起來。椅子的鉸鏈發出一聲金屬的呻吟。他揉了一下眼睛。左眼。因為左臉被壓了太久，左眼的眼眶周圍有些麻。

「什麼？」

米亞把那行註解投射到螢幕上。

白色的字。黑色的背景。字體是等寬的。程式碼字體。每個字符佔據相同的寬度。冷靜的。機械的。沒有感情的字體。

// 這一次，做對了。—— M.C.

維倫盯著那行字。

修復間裡安靜了很久。螢幕上的碎片歸還系統還在跑。數字還在跳。但維倫沒有看數字。他只看那行字。五個中文字。兩個英文字母。一個逗號。一個句號。兩條斜線。一條破折號。

「時間戳？」

「2089 年 4 月 1 日。凌晨三點四十二分。崩潰結束後四十一分鐘。」

維倫的呼吸沒有變。他的心率沒有變。他的手沒有抖。他只是看著那行字。看了很久。

「他活著。」

語氣和嚴柏翰在走廊裡說「然後我就不是探員了」一樣平。沒有驚訝，沒有恐懼。是確認。是一個他已經猜到了很久的事情，現在得到了證據。像做完一道題，翻到答案頁，發現自己的答案是對的。沒有驚喜。只有確認。

「他活在程式碼裡。」

米亞沒有確認。也沒有否認。

她說：

「他做對了什麼？」

維倫沒有回答。

螢幕上的五個字沒有消失。它們留在那裡。白色的。等寬的。在黑色背景上安靜地發著光。

做對了。

但做對了什麼？

覆蓋波的速度是他設的。每秒三萬。剛好讓修復師來得及反應。碎片溢出的比例是他設的。23%。剛好讓城市級歸還系統有碎片可收。觸感記憶的保護層級是他設的。最高。手記得的東西比大腦重要。

他設計了這場崩潰的每一個參數。限速。溢出。保護。他不是在引爆一顆炸彈——他是在做一場手術。刀口的深度、切割的角度、止血的時間點，全部計算過。

但他為什麼要做這場手術？

一座城市的記憶網路為什麼需要被崩潰？

為什麼要讓幾千個人暫時失去自己的記憶？

為什麼要讓一個十歲的男孩在門口等十四個小時？

為什麼要讓一個母親不認識自己的兒子？

然後說——做對了。

他在觀測。

他一直在觀測。

十年。

從 2079 年「死亡」那天起。從那間電力從未中斷的實驗室裡。他在觀測整個城市的記憶網路。觀測每一個節點。觀測每一次修復。觀測每一個走進維倫修復間的委託人。

螢幕在維倫走進來的時候閃了一下。只有一下。米亞記錄了。

那是他。

他看到了維倫。

他看到了維倫走進他的舊實驗室。他看到了米亞的光球在冷白色的光裡微微閃爍。然後他讓螢幕閃了一下。一次。像一個人在黑暗中眨了一下眼睛。

但他在觀測什麼？

維倫看著那行字。字在螢幕上很小。白色的。等寬的。在修復間的暖黃色光線裡，螢幕上的黑色背景像一扇窗。一扇打開的、通往某個暗處的窗。字在窗裡發光。

像一盞不肯熄滅的燈。
