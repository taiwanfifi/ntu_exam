## 第 33 章：凌晨兩點三十七分

### 觸發——蜜罐尖叫了

```
【環境】
五月十一日。凌晨兩點三十七分。
貝葉家。
她沒有睡。她已經五天沒有好好睡了。
每天晚上躺在床上。閉著眼睛。
但大腦不關機。像一台永遠在跑模擬的電腦。

陽台上。薄荷長得很好。五月的薄荷像小型森林。
客廳的燈關著。只有手機的待機光——綠色的。很微弱。

兩點三十七分。

透鏡邊緣閃了一下。紅色。
小拉的聲音在黑暗裡響起——

「貝姊。」

聲音不大。但是——
小拉的聲音沒有抖。
平常她的聲音會有一點碎念的質感。
現在——清澈。精確。像一台剛校準完的儀器。

「蜜罐觸發了。」

貝葉從床上坐起來。
心率從 62 跳到 108。
小拉記錄了。但沒有說出來。
不是時候。

「數據回傳。」小拉的聲音在黑暗裡像手術燈。

  ┌─ 蜜罐觸發報告 ─────────────────────────────┐
  │                                               │
  │ 區域：新北市三重區 Sec.3                     │
  │ 觸發時間：02:17:43                           │
  │ 系統自動校準完成時間：01:52:11               │
  │ 間隔：25 分 32 秒                            │
  │                                               │
  │ 修改前先驗：Gamma(α=3.2, β=1.1)            │
  │   E[λ] = 2.91                               │
  │                                               │
  │ 修改後先驗：Gamma(α=1.8, β=1.4)            │
  │   E[λ] = 1.29                               │
  │                                               │
  │ 偏移：|2.91 - 1.29| = 1.62                  │
  │ 偏移/σ₀ = 1.62/0.31 = 5.23σ                │
  │                                               │
  │ Chebyshev 判定：                              │
  │ P(正常波動 ≥ 5.23σ) ≤ 1/5.23² ≈ 3.66%     │
  │ ⚠ 超過 5σ 門檻。觸發確認。                 │
  │                                               │
  │ 狀態：■ 紅色警報                            │
  └───────────────────────────────────────────────┘

「超過 5σ。」貝葉的聲音在黑暗裡像一根火柴被劃著。

她拿起手機。撥了蘇立恆。
響了一聲。接了。
一秒都沒等。像他一直拿著手機。

「觸發了。」
「我知道。我看到了。」
「你也沒睡？」
「我每三個月確認後門。但今天——」
「你猜到了。」
「操控者有排程。三個月一次。上次二月。這次五月。
 ……十號或十一號。我猜十一號。因為十一號是週三。
 上兩次也是週三。他有固定的排程。」

他的聲音平淡。像在報告天氣。
但——他接得太快了。
一秒。
如果他真的什麼都沒感覺——
他會等三秒再接。讓自己看起來不那麼急。
一秒——意味著他也在等。
他也在怕。

「叫上馬可瑜和老高。
 我們需要在天亮之前把數據鎖定。
 一旦操控者發現觸發了——他可能會清除痕跡。」

「好。」

她掛了電話。穿上外套。
走出家門。

凌晨兩點四十五分的台北。
路上沒有人。只有路燈和遠處便利商店的光。
空氣裡有一點夜露的味道。涼的。

她走向捷運站——不。凌晨沒有捷運。
她叫了計程車。

計程車司機看了她一眼。
「小姐，這麼晚要去哪？」
「台大。」
「讀書讀到這麼晚啊？」
「……是。讀一種很難的書。」

她看著窗外。
台北的夜景在車窗上流過。
每一盞路燈像一個數據點。
連起來——是一條城市的脈搏線。

她的手裡攥著手機。
手機裡有一條數據。
5.23σ。
一個數學上幾乎不可能是巧合的偏離。

抓到了。
```

## 第 34 章：動差生成函數

### MGF——數學的鐵證

```
【環境】
凌晨四點。台大 301 研究室。
四個人到齊。theta 也到了——馬可瑜用背包把他背來的。
theta 在沙發上打了一個大哈欠。然後趴下。
仿佛在說：「你們自己忙，我先睡。」

白板被擦乾淨了。
蘇立恆已經把蜜罐的數據整理好了。
兩組完整的數據——
修改前和修改後的先驗。
以及它們各自跟觀測數據結合後的後驗。

「我們有了超過 5σ 的偏移。」貝葉說。
「Chebyshev 告訴我們——正常波動造成這個偏移的機率不超過 4%。
 這夠了嗎？」

高思遠搖頭。

「不夠。4% 太大了。
 法庭需要更強的證據。
 而且——Chebyshev 只是上界。
 它告訴你機率不超過 4%。
 但真實的機率可能更小——也可能接近 4%。
 我們需要一個精確的結論。不是上界。」

「什麼結論？」

「不是『很可能不同』。
 是『絕對不同』。
 數學上的絕對。」

他走到白板前。拿起粉筆。
他的粉筆——從口袋裡拿出來的。
跟了他三十年。
不是同一支。但同一個品牌。法國的。

「動差生成函數。」

  ┌─ 動差生成函數（MGF）─────────────────────────┐
  │                                                 │
  │ 定義：                                         │
  │ M_X(t) = E[e^{tX}]                            │
  │                                                 │
  │ 為什麼叫「動差生成函數」？                     │
  │ 因為對 t 求 n 次導數，代入 t=0——              │
  │ 就能得到 X 的第 n 階動差：                     │
  │ M_X^(n)(0) = E[X^n]                           │
  │                                                 │
  │ 第一階動差 = E[X] = 平均值                     │
  │ 第二階動差 = E[X²] → 可算出變異數             │
  │ 第三階動差 → 偏態（分布是否歪斜）             │
  │ 第四階動差 → 峰度（分布的尾巴有多厚）         │
  │                                                 │
  │ 所有的「動差」都藏在 M_X(t) 裡面。             │
  │ 它是分布的「DNA」。                            │
  │                                                 │
  │ ★ 唯一性定理：                                │
  │ 如果 M_X(t) = M_Y(t) 在 t=0 附近成立——       │
  │ 那 X 和 Y 有相同的分布。                       │
  │                                                 │
  │ ★ 逆否命題：                                  │
  │ 如果 M_X(t) ≠ M_Y(t)——                        │
  │ 那 X 和 Y 不是同一個分布。                     │
  │ 這不是「很可能不同」。                         │
  │ 是「絕對不同」。數學的絕對。                   │
  └─────────────────────────────────────────────────┘

高思遠放下粉筆。
「MGF 是分布的指紋。
 兩個人的指紋不同——他們就是不同的人。
 不需要看長相。不需要看身高。不需要 p-value。
 指紋不同——就是不同。」

「所以——」貝葉走到白板前。
「如果修改前的後驗 A 和修改後的後驗 B——
 它們的 MGF 不同——
 那它們就是不同的分布。
 這意味著先驗被修改了。不是波動。不是巧合。
 是——人為改變。」

「對。而且——
 Gamma 分布的 MGF 有封閉形式。很好算。」

蘇立恆接上。他在筆電上已經準備好了。

「後驗 A——用原始先驗 Gamma(3.2, 1.1) + 觀測數據：」

  M_A(t) = (1.1/(1.1 - t))^{3.2 + n}

「後驗 B——用被修改的先驗 Gamma(1.8, 1.4) + 同樣的觀測數據：」

  M_B(t) = (1.4/(1.4 - t))^{1.8 + n}

「β 不同——1.1 vs 1.4。
 α 不同——3.2+n vs 1.8+n。」

他在筆電上打了幾行代碼。跑了。

  ┌─ MGF 比對結果 ──────────────────────────────┐
  │                                               │
  │ M_A(t) vs M_B(t)                             │
  │                                               │
  │ 在 t = 0.1: M_A = 1.387  vs  M_B = 1.219   │
  │ 在 t = 0.2: M_A = 1.924  vs  M_B = 1.497   │
  │ 在 t = 0.5: M_A = 5.017  vs  M_B = 2.413   │
  │                                               │
  │ 結論：M_A(t) ≠ M_B(t) 在所有 t > 0          │
  │                                               │
  │ 唯一性定理判定：                              │
  │ ■ 後驗 A 和後驗 B 是不同的分布。            │
  │ ■ 先驗被修改。                               │
  │ ■ 這是數學的證明。不是統計的推論。           │
  └───────────────────────────────────────────────┘

「不同的分布。」貝葉說。

「不同的分布。」蘇立恆說。

「不同的分布。」馬可瑜說。

theta 在沙發上翻了一個身。

「這是數學的鐵證。」高思遠說。
他的聲音比平常更慢。每個字都像粉筆在黑板上按下去的重量。
「不是假設。不是猜測。不是 p-value。
 是——證明。」

他把粉筆放在白板的邊緣。

「好了。我們有了槍。
 現在——需要一個開槍的地方。」

→ MGF = 全書的「破案神器」。
  它不是最難的概念。但它是最關鍵的。
  因為它做了一件統計檢定做不到的事——
  給出了「絕對」的結論。
  不是 p < 0.05。不是「拒絕虛無假設」。
  是——「這兩個東西不一樣。句號。」
  讀者在這個高潮裡學會了 MGF。
  不是因為考試要考。
  是因為——它拯救了所有人。
```

## 第 35 章：備註

### 蘇立恆的三年前——「當你找到一個值得的人」

```
【環境】
天亮了。清晨六點。
研究室的窗戶透進來第一道光。
四個人都還在。
咖啡喝了四輪。紙杯堆了一小塔。

蘇立恆在做最後的驗證。
把 MGF 的計算從頭跑一遍。
確認沒有錯誤。

他的螢幕上。代碼。
先驗更新模組的核心——他三年前寫的。
一行一行。像自己三年前的日記。

然後他停了。

螢幕上有一行文字。
不是代碼。是備註。
用綠色的字——註解的顏色。

  // 當你找到一個值得的人，把這個給她看。

他盯著那行字。
手指懸在鍵盤上。一動不動。

貝葉走過來。
「怎麼了？」

她看了一眼螢幕。

// 當你找到一個值得的人，把這個給她看。

「這是……你寫的？」

蘇立恆的表情沒有變化。
但他的呼吸——慢了。
像一個人在刻意控制什麼。

「三年前。離開的前一天。凌晨三點。
 我把所有該留的東西留好了。
 後門。密碼。驗證機制。
 然後——大概太累了。
 寫了這行。」

他看著螢幕。

「我不記得用了這些字。
 我只記得——我留了備註。
 但不記得——寫了什麼。」

「當你找到一個值得的人。」
貝葉唸出來。聲音很輕。

她沒有問「值得的人是什麼意思」。
她沒有問「你是在寫給誰」。
因為——有些問題。
如果你需要問，答案就不夠大。
如果你不需要問——答案就在你不問的那個沉默裡。

蘇立恆把螢幕上的備註截了圖。
然後——他把那行備註刪掉了。

「為什麼刪掉？」

「因為它完成了它的任務。」

停頓。

他扣上筆電。

「備註不應該比代碼更重要。」

他站起來。走到窗邊。
晨光照在他的臉上。
三年來第一次——有陽光照到他的臉上而他沒有避開。

高思遠在沙發上看著他。
然後低頭看了看手裡的粉筆。
很輕地搖了搖頭。
是搖給自己看的。
意思是：「年輕人啊。」

馬可瑜在旁邊。她的眼眶有點紅。
但她忍住了。
她轉過頭去假裝看 theta。
theta 在沙發上。四腳朝天。完全沒有在意周圍發生了什麼。

小拉記錄了蘇立恆的心率。
從 58 到 76。
維持了三分鐘。
然後慢慢回到 62。

她沒有把這筆數據存進報告。
她把它存進了「私人」資料夾。
然後鎖起來。

但這次——她在旁邊加了一行註解：

  「有些數據不是用來分析的。
   是用來——記住的。」

→ 蘇立恆的情感線在這一刻完成了第一次完整的弧線。
  從「不會感覺的人」到「感覺了但不知道怎麼表達的人」。
  他的表達方式——是刪除。
  刪掉備註 = 承認備註很重要。
  因為不重要的東西不需要特意刪掉。
```

## 第 36 章：你的先驗假設是什麼？

### 研討會——第一槍

```
【環境】
十月。
三個月過去了。
從五月到十月——他們做了什麼？
蒐集更多數據。驗證 MGF。準備學術論文。

蘇立恆找到了更多區域的偏移證據。
高思遠聯繫了學術研討會的主辦方——
以「特邀評論人」身份出席。

台大。國際會議廳。
「2045 台灣統計學會年度研討會」。

大部分是統計學家、數據科學家、精算師、研究生。

講台上有投影幕、麥克風、一張長桌。
長桌上有水、名牌。

穿深藍色西裝。銀灰色的頭髮梳得很整齊。

穿灰色外套。坐得很直。

馬可瑜在第八排。theta 不能帶進來。
她把 theta 託給一個研究生照顧。
「如果他叫——就給他吃零食。如果他不叫——也給他吃零食。」
```

### 上台——粉筆與投影

```
「下一位發表者。
 陳貝葉。Oracular 校準工程師。
 題目：『匿名數據異常分析——基於動差生成函數的分布辨識』。」

貝葉站起來。走上講台。
她穿了一件白色的襯衫。簡單。乾淨。
沒有化妝。
高思遠說：「你的臉就是你的投影片。誠實就夠了。」

她站在麥克風前面。
看了一眼台下。
三百個人的眼睛。
（好。兩百七十個。）

她的手裡——
不是雷射筆。
是粉筆。

投影幕旁邊。
她放了一塊小黑板。帶來的。
是高思遠研究室裡的那塊。她借了。
高思遠說：「拿去。用手寫出來的公式才有重量。」

有人在台下竊竊私語——
「她帶了黑板？」
「現在誰還用粉筆啊？」

她深呼吸。一次。

小拉在透鏡裡低語：「心率 98。偏高但可控。你可以的。」

「謝謝大家。
 我今天要展示的是——
 如何用動差生成函數的唯一性定理，
 判斷兩組時間序列數據是否來自同一個分布。」

她切到第一張投影片。
數據。圖表。兩組後驗分布的直方圖。
沒有提 Oracular。只說「某大型資訊系統的校準數據」。
但——在場的人都知道。
台灣只有一個「大型資訊系統」在做機率校準。

她展示了蜜罐的方法。
沒有說「蜜罐」。說「受控實驗」。
展示了 Chebyshev 的觸發。5.23σ。
展示了兩組 MGF——

然後她放下遙控器。
拿起粉筆。
走到黑板前。

粉筆在黑板上的聲音。
沙沙的。
在空調的嗡嗡聲裡——
像一個人在說悄悄話。

她寫了：

  M_A(t) = (1.1/(1.1-t))^{3.2+n}
  M_B(t) = (1.4/(1.4-t))^{1.8+n}

  M_A(t) ≠ M_B(t)  ∀ t > 0

  ∴ A 和 B 是不同的分布。
  ∴ 先驗被修改了。

她放下粉筆。
轉向台下。

「這兩組數據——不可能來自同一個先驗設定。
 唯一的解釋是——先驗在觀測期間被修改了。
 而且——修改的幅度超過 Chebyshev 5σ 上界。」

會場安靜了。
連空調的聲音都像在屏息。

她切到最後一張投影片。
上面沒有圖表。沒有公式。
只有一行字。白底黑字。

  「每個人都有權利看到真實的機率。
   即使真實讓人害怕。」

全場沉默。
三秒。
很長的三秒。
像三個世紀。
```

### 柏朗的掌聲

```
然後——

柏朗站起來。
帶頭鼓掌。

掌聲很響。
不是禮貌性的。是用力的。
他的手掌每一次相擊都帶著重量。
像他在拍碎什麼。
也許是在拍碎自己的某個部分。

所有人都跟著鼓掌了。
因為柏朗鼓掌了。
他是 Oracular 的創始人。
他是全場最有分量的人。
他鼓掌——你就鼓掌。

掌聲停了。

柏朗微笑。
溫和的。長輩式的。
像一個教授對學生的報告表示讚賞。

「非常精彩的分析。陳小姐。」

他的聲音很好聽。沉穩。像大提琴。
低音區的。那種你會不自覺相信的聲音。

「但——」

停頓。
全場屏息。

「我想請教一個問題。」

「請。」貝葉的聲音穩定。
小拉在背後：「心率 88。很好。保持。」

柏朗走到過道的麥克風前。
他走路的方式跟費雪不同——
費雪的步伐是精確的。像機器。
柏朗的步伐是從容的。像散步。
一個已經看過所有風景的人的步伐。

「您的分析基於一個假設——
 您所謂的『受控實驗』的基線先驗是正確的。
 也就是說——您認為 Gamma(3.2, 1.1) 是『真實的』先驗。
 而 Gamma(1.8, 1.4) 是『被修改的』先驗。」

「是。」

「但——」

他微笑了。那個微笑比費雪的可怕一百倍。
因為費雪的微笑是計算過的。
柏朗的微笑——是真的。
他真的覺得自己有道理。

「您怎麼知道您的先驗是對的？」

全場安靜。

「換句話說——」
他看著貝葉。不是俯視。是平視。
像看一個值得認真對待的對手。

「您質疑了系統的先驗。
 但您用什麼來質疑？
 您自己的先驗。
 那——」

他的聲音放慢了。每個字都像被稱過重量。

「您的先驗假設是什麼？」

★★★ 全書核心問題。

一切回到起點。
P(θ|data) ∝ P(θ) × P(data|θ)。
後驗 ∝ 先驗 × 似然。
如果你質疑別人的先驗——你自己的先驗是什麼？
誰來校準校準師？
誰來質疑質疑者？

這是一個無限迴圈。
一個蛇咬自己尾巴的問題。
一個數學無法回答的數學問題。

貝葉沒有立刻回答。

她看了一眼台下。
高思遠在第五排。他的表情平靜。
但她知道——他在為她擔心。
他的手——放在膝蓋上——握得很緊。
指節發白。

走廊外面。
蘇立恆靠在牆上。
他聽到了柏朗的問題。
他的拳頭也握緊了。
不是因為憤怒。
是因為——這個問題他也問過自己。
三年來。每一天。
他沒有找到答案。
所以他希望——她能。

「柏朗先生。」

貝葉的聲音穩定。
不是那種刻意壓住的穩定。
是——真的穩定。
像一個已經想通了的人。

「您的問題非常好。」

她放下粉筆。
沒有看投影片。沒有看筆記。
看著柏朗。

「我的先驗假設是什麼？
 這是一個好問題。
 因為——您說得對。
 任何統計推論都建立在先驗上。
 我質疑了系統的先驗。
 但我的質疑本身——也有先驗。」

停頓。

「我的先驗是——
 數據應該被看到。
 真相應該被知道。
 每個人都有權利用自己的方式更新自己的信念。
 不是被別人替換。
 不是在他們不知道的情況下——
 被人把起點改了。」

她的聲音在會場裡回響。空調的嗡嗡聲消失了。

「這是一個我無法用數學證明的先驗。
 我沒有辦法寫出 P(真相重要) = 多少。
 我沒有辦法用 MGF 驗證它。
 我沒有辦法做假設檢定。」

停頓。

「但我選擇相信它。
 因為——如果這個先驗是錯的——
 如果真相不重要——
 那我們為什麼還在做統計？
 如果數據不值得被看到——
 那我們為什麼還在收集數據？
 如果人不應該知道真相——
 那——科學存在的意義是什麼？」

她看著柏朗。

「您的先驗是——穩定比真相重要。
 我尊重。
 但我的先驗不同。」

柏朗看著她。
他的微笑沒有變。
但他的眼神——變了。
不是憤怒。
是悲傷。

一種很深的、很老的悲傷。
像一口三十年的老井。
你看不到底。但你知道水在裡面。

他坐下了。
筆在指間停止了轉動。

全場再次沉默。
但這次的沉默——跟之前不同。
之前的沉默是「不知道該怎麼反應」。
這次的沉默是——每個人都在想。
在問自己。

我的先驗假設是什麼？
```

### 第三篇收束

```
─── 第三篇「信號」完 ───

研討會結束。
人群散了。
台下的椅子一排排空了。
燈還亮著。空調還開著。

貝葉站在講台上。一個人。
手裡拿著粉筆。短了一截。
她看了一眼黑板。
上面的公式還在——

  M_A(t) ≠ M_B(t)

白色的粉筆字在黑色的板上。
像一道裂縫。
看起來只是一行公式。
但它劈開了——一些東西。

她走下講台。走出會場。
走廊上。陽光很亮。十月的台北偶爾有這種好天氣。

蘇立恆靠在牆上。
他的眼睛睜開了。
他聽到了一切。

兩個人對視了。
他沒有說「說得好」。
沒有說「你很勇敢」。
沒有說任何一句——他擅長說的那種精確的話。

他做了一件事。
他伸出手。
把她手裡的粉筆接過去。
幫她放進外套口袋。

他的手指碰到了她的手背。
很短。不到一秒。
像一個數據點——存在過就消失了。

但存在過的東西——不會消失。
就像數據被記錄了，就永遠是真的。
不管你怎麼刪除。
它存在過。

小拉記了一筆。
然後鎖進「私人」資料夾。

馬可瑜從會場裡走出來。
「柏朗鼓掌的時候他的手在抖。」

「我沒注意到。」

「我注意到了。
 他不是在鼓勵你。
 他是在——忍耐。」

高思遠最後走出來。
站在走廊上。看了一眼天空。

十月的天空。很藍。
跟三十年前一樣藍。
他年輕的時候——跟柏朗——跟貝葉的父親——
就是在這條走廊上。
討論統計。討論世界。討論——
「如果我們有能力改變所有人的信念，
 我們應不應該？」

那時候的答案很簡單。
三個年輕人。三杯咖啡。
「不應該。」

但三十年後——
其中一個人改變了所有人的信念。
另一個人——已經不在了。
而他——坐在台下。
看著第三代人重新問同一個問題。

他的口袋裡多了一支粉筆。
貝葉用完了還給他的。
短了一截。

他握著那支粉筆。
走進走廊的陽光裡。

眼角。
一滴。
只有一滴。

落在地板上。
乾了。
像一個先驗——
存在過。影響了一切。
然後消失了。

但後驗還在。
後驗永遠在。

因為——
後驗 ∝ 先驗 × 似然。
先驗消失了。
但似然在。
所以後驗——
比先驗更真。

─── 翻頁。第四篇「檢定」開始。 ───
```
