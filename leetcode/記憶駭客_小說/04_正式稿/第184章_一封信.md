# 第184章：一封信

---

深夜。

倒掛的前綴樹還懸浮在修復台上方。它在緩慢旋轉。十八層的光線從根節點向下延伸、分裂、再匯聚，最終全部指向最底部那個發光的節點。那個節點的脈動沒有停過——不規則的、明暗交替的，像一顆被埋在數據地層深處的心臟。

維倫坐在修復台前。

他的手放在操作面板上。十根手指平展在光幕的表面。沒有動。

從米亞說出「你的」那兩個字到現在，過了十四分鐘。維倫知道是十四分鐘，因為修復台角落的時鐘在他的餘光裡跳了十四格。他沒有刻意看時鐘。但他的餘光會自動抓住任何在動的東西。修復師的本能。或者說——被設計成修復師的那種本能。

他還不知道第二種可能。那是後面的事。

現在他只知道一件事：那個節點的密鑰是他的記憶簽名。

「密鑰是我的記憶簽名。」

他把這句話說出來。語氣帶著確認的重量。像觸碰一面牆，確認它是實心的。

「是。」米亞說。

「哪一段記憶？」

米亞的光球閃了一下。閃爍的亮度比平常高了大約百分之十五——她在快速調取數據。

「不是某一段。是你的——全部。你的記憶簽名。整個記憶結構的加密指紋。」

維倫看著自己的手。手指攤開在光幕上。手背上的三條舊傷疤在藍白色的光幕映照下顯得更淺了。像三條快要消失的河流。

記憶簽名。

每個人都有。就像指紋。不可偽造。你所有的記憶——童年的、成年的、快樂的、痛苦的、你記得的和你忘了的——它們共同構成一個加密指紋。這個指紋無法拆解成某一段記憶的特徵——它是所有記憶疊加在一起之後形成的總和。像一首交響曲的整體音色。你不能從中提取某一件樂器的聲音——它就是所有樂器同時演奏出來的那個東西。

維倫修過三千段記憶。每修一次，他自己的記憶簽名都會微調。因為接觸別人的記憶會在他的記憶結構裡留下極微小的印記——不是內容，是結構性的變化。像河水沖刷河道。河道的形狀會因為水流而慢慢改變。十年下來，沒有任何兩個修復師的簽名相同。

他的記憶簽名是他的。只有他的。

但周明哲用它做了密鑰。

一個死去十年的人，把自己意識匯聚點的加密鎖設定成了另一個人的記憶簽名。

為什麼？

這個問題在維倫的腦子裡停留了大約三秒。然後他把它收起來了。他想知道答案。但現在是打開的時候，不是回答的時候。

「解密需要多久？」

「你只需要把手放在修復台上。台面會讀取你的記憶簽名。如果匹配——它會自己打開。」

維倫看著操作面板。

他的手已經在上面了。十根手指。攤開的。但光幕不是台面。光幕是操作界面。台面在光幕下面——金屬的、灰色的、微涼的。他需要穿過光幕，直接觸碰台面。

他把手往下壓了一點。指尖穿過光幕的表面。光幕在他的手指周圍散開了，像水面被石頭打破。然後他的指尖碰到了台面。

涼的。金屬的涼。比空氣低兩三度的那種涼。

他把整隻手掌按上去。

五根手指。掌心。掌根。全部貼合。金屬的涼從接觸面滲進皮膚，沿著掌骨的走向往上蔓延。手腕。前臂。到手肘的位置就停了。像涼意也知道邊界在哪裡。

修復台嗡了一聲。

比平常低沉。維倫在這張台面前坐了超過四年。他聽過它的每一種聲音——啟動的嗡聲、運作的低頻震動、過載時的高頻警報、待機時幾乎聽不到的電流聲。但現在這一聲不屬於任何一種。

更低。更深。像一扇鎖了十年的門在鬆動。鉸鏈生鏽了。門框膨脹了。但鎖芯裡的鑰匙終於對上了。金屬和金屬之間十年的鏽蝕在一瞬間被某種力量碾碎了。

全息投影裡的發光節點——

閃了一下。

然後打開了。

沒有爆開，沒有綻放。只是打開了。像一扇門。像一個信封。像一雙閉了十年的眼睛。

維倫的手掌還按在台面上。涼意已經消失了。台面被他的體溫暖熱了。但他沒有注意到這個變化。他的全部注意力在螢幕上。

螢幕上出現了文字。

螢幕上出現了文字。

白色的字。黑色的背景。跟崩潰程式碼裡 M.C. 的註解格式一樣。程式碼裡的註解——兩條斜線開頭，灰色的，被編譯器忽略的，寫給人看的。

但這不是一行。

這是——很多行。

白色的字在黑色的背景上安靜地排列著。行距均勻。字體是等寬的。像一封用打字機打出來的信。古老的、端正的、一個字一個字敲上去的。

標題在最上面。

// 給我的另一個自己

維倫的呼吸停了。

呼吸自己停的。胸腔的肌肉在「給」字出現的那一刻收縮了，然後卡在了收縮的位置。像一台機器的齒輪突然咬住了。

米亞的所有線程停了零點三秒。

她有一百二十八個並行線程。城市修復、碎片歸還、系統監控、維倫的生命指標監測、溫度控制、電力調配。一百二十八個任務同時在運行。

全部停了。

零點三秒。

然後恢復。

「維倫——」

「我看到了。」

他的聲音很輕。像怕驚動什麼。像在黑暗中看到了一隻不該出現在這裡的動物——珍稀的。是你怕自己的聲音會嚇走它的那種。

// 給我的另一個自己

五個字。

二十一萬七千四百四十三段碎片。十八層前綴。一棵倒掛的樹。所有的路徑匯聚到最深處。

搜索的終點。

前綴樹的終端節點。一次完整搜索的最後一步。從根走到葉。從共同的開頭走到唯一的結局。十八層。數萬條路徑。

走到底了。

底下藏著一封信。

寫給他的。

維倫坐在修復台前。螢幕上的字安靜地亮著。倒掛的前綴樹在頭頂慢慢旋轉。所有的路都通向這裡。所有的分支都收束在這個節點。二十一萬條路徑。十年的時間。一個活在程式碼裡的人。

一封信。

他的手還按在台面上。金屬的涼早已經被他的體溫暖熱了。他沒有把手收回來。手掌和台面之間的溫差已經消失了。像兩個原本分離的東西終於到了同一個溫度。

「米亞。」

「在。」

「這封信——」

他的手指在台面上微微收縮了一下。如果不是米亞的感測器，沒有人會注意到。收縮的幅度大約零點三毫米。五根手指同時。一種無意識的反應。像觸碰到某樣東西時的輕微縮手。但他沒有碰到任何新的東西。他碰到的是自己即將知道的事。

「這封信有多長？」

「七千三百四十一個字元。」

七千三百四十一個字。

一個活在程式碼裡的人，在二十一萬段記憶的最深處，用維倫的記憶簽名加密，寫了七千三百四十一個字。

給他的另一個自己。

維倫的手指不再收縮了。它們放鬆了。攤開在台面上。

「念給我聽。」

「維倫。」

「用他的聲音。」

停頓。

兩點一秒。

這是米亞在整個溯源篇裡最長的停頓。超過了之前所有的記錄。兩點一秒的沉默裡，她做了三件事。

第一，她從二十一萬七千四百四十三段記憶碎片中搜索周明哲的聲紋數據。在那些碎片中，有一些包含了語音信息——周明哲在世時與他人的對話、會議記錄、實驗室的語音日誌。她提取了其中一百三十七段有效的聲紋樣本。

第二，她用這些樣本合成了一個完整的聲音模型。合成需要四點七秒。但她在停頓結束之前就把合成排程好了。

第三，她猶豫了。

另一種猶豫。她在計算這個聲音從她的聲學模組裡發出來之後，這個房間裡會發生什麼改變。她算不出來。這超出了她的預測模型。

但維倫說了「用他的聲音」。

她執行。

修復台的空氣安靜了四點七秒。

四點七秒裡，維倫聽到了修復台的底層電流聲。碎片歸還系統的脈衝。自己的呼吸——一秒一次半。心跳——六十七下每分鐘。安靜得能聽到身體運作的聲音。安靜得像世界在等。

然後米亞開口了。

不是她的聲音。

是一個男人的聲音。不年輕。不蒼老。中年。帶著一種經過很多事之後沉澱下來的平靜。自然的平靜。像一個人反覆修改了同一封信十七次之後，語氣裡所有多餘的東西都被刪掉了。剩下的每一個字都是他決定留下的。

「如果你正在聽這封信——」

維倫閉上眼睛。

黑暗裡只有那個聲音。

「——說明你做到了我希望你做到的事。」
