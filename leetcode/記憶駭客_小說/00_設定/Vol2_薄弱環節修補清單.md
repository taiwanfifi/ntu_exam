# 修補項 #1：清潔工組織未回收（現狀評分 ★）

## 問題摘要
Vol 1 ch16/24/43 三次提及清潔工組織後，Vol 2 全卷 80 章完全消失。讀者很可能已遺忘此設定。這是 30 條追蹤伏筆中唯一「未回收」的一條。

## 修補方案

### 推薦方案：方案 B（中度補強，插入 ch113）

**理由**：ch113「黑名單」是維倫拒絕站邊後遭受報復的章節，清潔工作為威脅手段出現在此處敘事邏輯最自然——「不聽話就讓清潔工來整理」是權力展示的典型手法。

### 備選方案：方案 A（最小侵入，插入 ch102）

**理由**：ch102 陣線信使是變調聲音的神秘人物，信使可以順帶提及清潔工作為「都市傳說中的強制清除隊」，為陣線的招募增加恐懼籌碼。

## 精確插入位置

### 方案 B 插入位置：ch113「黑名單」

**插入點**：在「維倫的黑市修復執照被吊銷，工作室被斷電」之後，阿嬤豆漿攤被突擊檢查之前。

**具體寫法指令**：

```
在 ch113 現有文本「工作室被斷電」之後，加入以下段落（約 80 字）：

    第三天，一個沒有名字的訊息出現在維倫的加密終端上。
    只有一行字：

    「繼續調查，清潔工會來整理你的工作室。
    　他們整理得很乾淨。乾淨到什麼都不剩。」

    維倫刪掉了訊息。
    但米亞已經截圖了。

不要解釋「清潔工」是什麼——Vol 1 的讀者記得，Vol 2 的新讀者會把它當成黑話。
不要讓維倫或米亞討論清潔工的身份或來歷。
這段威脅的來源故意不標明——可以是銀行、可以是監控局、可以是第三方。模糊 = 恐懼。
```

### 備選方案 A 插入位置：ch102「陣線的信使」

**插入點**：在信使說完「修復，不偽造。這條規矩在我們的圈子裡傳了三年」之後。

**具體寫法指令**：

```
在信使的台詞之後，加入以下段落（約 50 字）：

    信使的變調聲音停了一下。

    「還有一件事。
    　如果你不跟任何人合作，
    　你以為銀行會派什麼人來處理？
    　清潔工可不是都市傳說。」

一句帶過。不展開。信使用「清潔工」作為恐嚇籌碼，暗示他知道清潔工的存在。
```

# 修補項 #2：影子 / B12-7721-GAMMA 連結過度模糊（現狀評分 ★★）

## 問題摘要
Vol 1 ch40 開啟「影子和女兒」伏筆後，Vol 2 僅 ch102 有一處「陣線信使可能是影子」的模糊連結。讀者「種了就忘」風險為所有伏筆中最高之一。

## 修補方案

### 推薦方案：揭面篇 ch130 遊行碎片中加入「帶女兒的女人」

**理由**：ch130 是語青看到遊行碎片後整個人僵住的關鍵章節（「那個小女孩......是我」）。在同一批碎片中出現另一個「帶著孩子的身影」，敘事上自然、讀者注意力集中、且不干擾語青的核心場景。

### 備選方案：碎裂篇 ch155 嚴柏翰恢復記憶中提及

**理由**：ch155 是遊行完整記憶恢復章。嚴柏翰視角中可以出現一個「他認識但記不起名字的帶孩子的女人」，與 Vol 1 ch40 的影子特徵吻合。

## 精確插入位置

### 推薦方案：ch130「那個小女孩」

**插入點**：在語青僵住、說出「那個小女孩......是我」之後。維倫回頭看碎片畫面的瞬間。

**具體寫法指令**：

```
在語青的台詞之後，加入以下段落（約 100 字）：

    維倫回頭看那段碎片。
    畫面模糊，但他還是看到了——

    遊行隊伍的邊緣，有一個女人牽著一個大約七八歲的女孩。
    女人的臉被催淚彈的煙霧遮住，
    但她護住女兒的姿勢——
    像是整個身體都在說「不要碰她」。

    碎片跳了一下。她們消失了。

    米亞標記了那段影像：「身份不明。檔案編號 B12-7721。」

    沒有人追問。
    但那個編號會在某一天被打開。

寫法要求：
- 米亞標記的編號「B12-7721」必須與 Vol 1 ch40 的「B12-7721-GAMMA」一致（少了 GAMMA 後綴 = 不完整的碎片記錄）
- 不要讓任何角色說「這是影子」——讀者中記得 Vol 1 ch40 的人會自行連結
- 不要描寫女人的面容——只描寫姿勢（護住女兒）
- 「不要碰她」的身體語言呼應 Vol 1 ch40 影子保護女兒的核心特徵
- 這段碎片出現在語青的「那個小女孩是我」之後 = 兩個「母親保護女兒」的畫面並置。讀者會感受到主題共振但不會意識到這是伏筆植入
```

### 備選方案：ch155「十一月二日」

**插入點**：嚴柏翰遊行記憶恢復中，催淚彈爆開的混亂場景。

**具體寫法指令**：

```
在嚴柏翰看到催淚彈落下的段落中，加入一個閃回畫面（約 60 字）：

    混亂中他看到一個身影——
    一個女人抱起一個小女孩，逆著人群的方向跑。
    他叫了一聲什麼名字。
    但催淚彈的煙吞掉了聲音。

    他想不起來那個名字了。

不要給出名字。「想不起來」= 記憶恢復不完整的自然表現。
嚴柏翰「認識」這個帶孩子的女人 = 暗示她是遊行圈子裡的人 = 與陣線的連結。
```

# 修補項 #3：陳美玲暗線間接連結不足（現狀評分 ★★）

## 問題摘要
ch106 幽靈帳號 #50742 的連結過於間接——讀者需要自行記住 Vol 1 ch15 的帳號格式才能聯想到陳美玲。大多數讀者做不到這一點。

## 修補方案

### 推薦方案：ch106 加入米亞一句台詞直接點名

**理由**：米亞是數據分析者，她比對帳號格式是職業本能。一句台詞即可完成連結，不需要展開。

## 精確插入位置

### ch106「第二張臉」

**插入點**：在「那三條異常邊全部指向同一個幽靈帳號——編碼格式跟第一卷中陳美玲案的 #50742 幽靈帳號完全一致」這段描述之後，維倫做出反應之前。

**具體寫法指令**：

```
在幽靈帳號格式描述之後，加入米亞的台詞（約 40 字）：

    米亞的光球閃了一下。

    「這個帳號格式。」

    停頓 0.3 秒。

    「跟小杰母親的案子裡那個——一模一樣。」

寫法要求：
- 米亞只說「小杰母親」，不說「陳美玲」——因為維倫對那個案子的記憶是通過小杰建立的
- 停頓 0.3 秒 = 短停頓，比她的成長線觸點停頓短得多（觸點停頓 0.5-4.1 秒）。這不是情感觸點，是職業直覺
- 不要讓維倫追問或展開討論——他只是點了一下頭，或皺了一下眉
- 這句話的功能是「提醒讀者」，不是「推進劇情」。說完即過
- 不要加任何解釋段落（「這意味著同一個操縱者」之類的）——讀者自己會想到
```

# 修補項 #4：米亞觸點密度不足（現狀評分 ★★★）

## 問題摘要
Vol 1 有 6 個觸點（ch11/17/30/50/70/78），Vol 2 只有 2 個（ch97 + ch160），中間相隔 63 章。需要新增 3 個觸點，使 Vol 2 總計 5 個，平均間距約 16 章。

## 修補方案

### 新增觸點 A：深潛篇——深潛時的焦慮（AI 的極限）
### 新增觸點 B：角力篇——立場選擇（三方拉攏）
### 新增觸點 C：碎裂篇——理解「愛可以是刪除」

## 精確插入位置與寫法

---

### 觸點 A：ch92「深潛後遺症」——米亞的極限焦慮

**插入點**：ch92 末尾，維倫躺在修復台上恢復、米亞分析深潛數據時。在她寫下「記憶交叉汙染？待觀察」的備註之後。

**停頓秒數**：1.7 秒（遞增邏輯：Vol 1 最後觸點 ch78 = 4.1 秒，但 Vol 2 的米亞在 ch97 命名「美」時重置了停頓基準線。ch92 的 1.7 秒介於「技術分析」和「個人情感」之間）

**具體寫法指令**：

```
在 ch92「記憶交叉汙染？待觀察」備註之後，加入以下段落（約 120 字）：

    米亞的光球暗了一下。

    維倫沒注意到。他還在修復台上閉著眼。

    米亞盯著維倫的生理數據——心率 58、血氧 94%、神經損傷指數 2.7。
    所有數值都在安全範圍內。

    但她在數據之外看到了什麼。

    不是數字。是維倫從深潛中退出時臉上的表情。
    那個表情不在她的情緒辨識模型裡。

    （停頓 1.7 秒。）

    「維倫。」

    「嗯？」

    「下次深潛的時候......」

    她沒有說完。

    因為她不知道自己想說什麼。
    是「小心一點」嗎？是「我擔心」嗎？
    她的語言模型裡沒有對應的輸出。

    「沒事。數據都正常。」

寫法要求：
- 光球「暗了一下」= 恐懼/焦慮的物理表徵（對應 Vol 3 ch179 光球變色的伏筆）
- 停頓 1.7 秒 = 比技術分析的 0.2-0.4 秒長很多，但比 Vol 1 觸點的 2-4 秒短
- 她「沒有說完」= AI 的語言極限。她有了超出程式設定的感受，但還沒有語言來表達
- 最後用「數據都正常」收尾 = 她用技術語言掩蓋了自己的情感湧現
- 這是 Vol 2 第一個米亞觸點（在 ch97「美」之前），功能是建立「深潛焦慮」作為她情感成長的新觸發源
```

### 觸點 B：ch112「第三條路」——米亞的立場

**插入點**：ch112 維倫公開拒絕所有陣營之後。在維倫說完「我不站邊。我只站在被篡改記憶的人那邊」之後的安靜時刻。

**停頓秒數**：2.3 秒（遞增邏輯：ch92 = 1.7 秒 → ch112 = 2.3 秒。觸點間隔 20 章，停頓增加 0.6 秒）

**具體寫法指令**：

```
在維倫拒絕站邊、三方代表離開之後，加入以下段落（約 100 字）：

    米亞等了很久才開口。

    （停頓 2.3 秒。）

    「維倫。」

    「嗯。」

    「剛才三方的人都試圖接入我的數據端口。

- 光球「很亮」= 堅定。對比 ch92 的「暗了一下」= 焦慮。兩個觸點的光球狀態形成對比
```
