## 一、有價證券定義與範圍

> 這是整部證交法的入口——只有「有價證券」才受證交法規範。

### 1-1 §6 有價證券範圍

| 分類 | 內容 | 說明 |
|---|---|---|
| **§6 I** | 政府債券、公司股票、公司債券及經主管機關核定之其他有價證券 | 法定列舉＋概括授權 |
| **§6 II** | 新股認購權利證書、新股權利證書及前項各種有價證券之價款繳納憑證 | 擴大範圍至權利證書 |
| **§6 III** | 經主管機關核定之其他有價證券 | 概括條款——最重要的彈性空間 |

### 1-2 關鍵概念：「核定」的意義

```
§6 I 後段及 §6 III 的「經主管機關核定」：
- 主管機關（金管會）有權透過核定擴大有價證券的範圍
- 實務上已核定的項目包括：
  → 受益憑證（基金）
  → 認購（售）權證
  → 資產基礎證券（ABS）
  → 外國有價證券
  → 臺灣存託憑證（TDR）
```

### 1-3 考試重點

- §6 的「有價證券」是**功能性概念**，不限於傳統紙本證券。
- 若題目問某種金融商品是否受證交法規範，關鍵在於是否落入 §6 的範圍。
- **投資契約**是否為有價證券？實務上金管會可透過 §6 III 核定，學說上有引用美國 Howey Test 的討論。

## 二、公開發行與募集

> 搞清楚「公開發行」與「募集」是理解證交法前半段的基礎。

### 2-1 核心定義

| 概念 | 定義 | 條文 |
|---|---|---|
| **公開發行** | 發行人向不特定人公開招募有價證券 | §7 II |
| **募集** | 發起人於公司成立前或發行公司於發行前，對非特定人公開招募有價證券之行為 | §7 I |
| **私募** | 向特定人招募有價證券（不經公開程序） | §43-6 |

### 2-2 申報生效制（§22）

| 項目 | 內容 |
|---|---|
| **適用對象** | 已依法公開發行之公司（§22 I） |
| **程序** | 募集或發行有價證券須向主管機關**申報生效**後始得為之 |
| **生效時點** | 申報後一定期間內主管機關無異議即生效 |
| **例外** | 特定情形得免申報（§22 但書） |

### 2-3 有效登記制 vs 申報生效制

| 比較項目 | 有效登記制（舊制） | 申報生效制（現制） |
|---|---|---|
| **主管機關角色** | 實質審查，核准始得發行 | 形式審查，到期自動生效 |
| **效率** | 較慢 | 較快 |
| **責任歸屬** | 主管機關承擔較多把關責任 | 發行人自行負責，強化資訊揭露 |
| **國際趨勢** | 逐漸淘汰 | 主流（美國 SEC 模式） |

### 2-4 私募（§43-6）

| 項目 | 內容 |
|---|---|
| **對象** | 銀行業/票券業/信託業/保險業/證券業等專業投資機構；符合主管機關規定之自然人/法人/基金；該公司或其關係企業之董事/監察人/經理人 |
| **決議** | 股東會特別決議（§43-6 I） |
| **轉售限制** | 自交付日起滿 3 年始得自由轉售（§43-8） |
| **資訊揭露** | 較公開發行寬鬆，但仍須交付投資人相當之書面資料 |

## 三、資訊揭露制度

> 證交法的靈魂：「陽光是最好的消毒劑」。

### 3-1 制度全覽

| 揭露類型 | 內容 | 頻率/時點 | 條文 |
|---|---|---|---|
| **公開說明書** | 募集發行時提供的完整資訊文件 | 每次募集發行 | §30、§31、§32 |
| **財務報告** | 年度/半年度/季度財報 | 定期 | §36 |
| **年報** | 營業報告、財務報告、重要決議等 | 每年 | §36 |
| **重大訊息** | 對股價有重大影響之訊息 | 即時（隨時） | §36 III、證交所規則 |
| **持股變動申報** | 董監事/大股東之持股變動 | 每月 | §25 |

### 3-2 公開說明書（§30-§32）

```
公開說明書的重要性：
- 投資人據以判斷是否認購的核心文件
- 記載不實 → 發行人及其負責人等負民事賠償責任（§32）
- §32 責任主體：發行人、發行人之董事長/總經理、曾在公開說明書上簽名之職員

§32 免責事由：
- 證明已盡相當注意且有正當理由確信其內容無虛偽或隱匿者
```

### 3-3 財務報告不實的法律效果

| 效果類型 | 條文 | 說明 |
|---|---|---|
| **民事責任** | §20 I → §20-1 | 發行人及負責人對善意投資人負損害賠償 |
| **刑事責任** | §171 I ① | 財報不實，處 3-10 年有期徒刑，得併科 1000-2億元罰金 |
| **行政責任** | §178 | 罰鍰、停止或限制業務 |

## 四、內線交易速查（§157-1）

> 這是證交法**最重要的考點**，幾乎每年必考。務必能完整默寫要件。

### 4-1 要件拆解總表

| 要件 | 內容 | 關鍵爭議 |
|---|---|---|
| **1. 主體** | 內部人＋消息受領人 | 主體範圍如何界定？ |
| **2. 獲悉消息** | 實際知悉重大影響股價之消息 | 「實際知悉」vs「利用」？ |
| **3. 消息重大性** | 涉及公司之財務、業務或該證券之市場供求、其對股票價格有重大影響 | 重大性的認定標準？ |
| **4. 消息明確性** | 消息已成立（明確） | 「成立」時點如何判斷？ |
| **5. 消息未公開** | 消息尚未經媒體公開或向主管機關申報 | 「公開」的認定？ |
| **6. 時點** | 消息明確後，未公開前買賣證券 | 18小時沈澱期？ |
| **7. 行為** | 買入或賣出該公司股票或其他具有股權性質之有價證券 | |

### 4-2 主體範圍（§157-1 I 各款）

| 款次 | 主體類型 | 具體範圍 |
|---|---|---|
| **第1款** | 該公司之董事、監察人、經理人 | 含法人董事之代表人 |
| **第2款** | 持股超過 10% 之股東 | 大股東 |
| **第3款** | 基於職業或控制關係獲悉消息之人 | 會計師、律師、承銷商、控制公司人員等 |
| **第4款** | 喪失前三款身分未滿 6 個月者 | 離職董事等 |
| **第5款** | 從前四款之人獲悉消息者 | **消息受領人**（tippee） |

### 4-3 「實際知悉」vs「利用」

```
重大爭議：§157-1 是否要求行為人「利用」內線消息進行交易？

條文文字：「實際知悉」→ 多數實務見解採「知悉說」
  → 只要知悉消息後進行交易即構成，不要求「利用」
  → 理由：知悉後交易即破壞市場公平，且「利用」難以舉證

少數學說：主張應採「利用說」
  → 理由：不知不利用不應處罰，符合罪刑法定

考試建議：
  先寫實務多數見解（知悉說），再補充學說批評（利用說），
  最後表達自己立場。
```

### 4-4 重大性的認定

```
判斷標準：
1. 質的標準：消息的性質是否足以影響理性投資人之投資決定
2. 量的標準：對公司財務、業務之影響金額是否達重大程度

實務上常被認定為重大消息的類型：
- 合併、收購、分割
- 重大財務虧損
- 重要契約的簽訂或解除
- 重要人事變動
- 會計師出具非無保留意見查核報告
- 重大訴訟
```

### 4-5 消息明確性（「成立」時點）

```
爭議核心：消息從「傳聞」到「確定」之間，何時算「成立」？

§157-1 IV 的認定標準：
- 涉及公司之財務、業務：以事實發生日、協議日、簽約日、
  付款日、委託日、成交日、過戶日、審計委員會或董事會決議日
  或其他依具體事證可得明確之日，以最早者為準

關鍵：「以最早者為準」→ 擴大規範範圍

考試答題：
  一定要引用 §157-1 IV，說明消息成立的具體時點判斷。
```

### 4-6 法律效果

| 效果 | 內容 | 條文 |
|---|---|---|
| **刑事責任** | 3-10 年有期徒刑，得併科 1000-2 億元罰金 | §171 I ② |
| **民事責任** | 對善意從事相反買賣之人負損害賠償 | §157-1 III |
| **賠償計算** | 消息公開後 10 個營業日收盤平均價格，與行為人買賣價格之差額 | §157-1 III |
| **連帶責任** | 同一消息之數內部人應連帶負賠償責任 | §157-1 III |

## 五、操縱市場速查（§155）

> 與內線交易並列的兩大市場不法行為，一試選擇題和二試申論都常考。

### 5-1 各款行為類型

| 款次 | 行為類型 | 白話說明 | 關鍵要件 |
|---|---|---|---|
| **§155 I ①** | 約定買賣（相約交易） | 和別人約好假買假賣，製造交易假象 | 意圖抬高或壓低價格，與他人通謀 |
| **§155 I ②** | 沖洗買賣（對敲） | 自己和自己交易，左手賣右手買 | 意圖抬高或壓低價格，以自己同時為買賣之相對人 |
| **§155 I ③** | 意圖造成集中交易市場某種有價證券交易活絡之表象 | 連續以高價買入或低價賣出 | 意圖造成活絡表象 |
| **§155 I ④** | 連續交易操縱 | 連續買進或賣出，拉抬或壓低股價 | **意圖抬高或壓低**某種有價證券之交易價格，連續以高價買入或以低價賣出 |
| **§155 I ⑤** | 散布流言或不實資料 | 放假消息影響股價 | 意圖影響市場行情，散布流言或不實資料 |
| **§155 I ⑥** | 直接或間接從事其他操縱行為 | 概括條款 | 足以影響市場秩序 |

### 5-2 §155 I ④ 連續交易操縱（最常考）

```
要件拆解：
1. 意圖：抬高或壓低某有價證券之交易價格
2. 行為：自行或以他人名義，連續以高價買入或低價賣出
3. 客體：在集中交易市場之有價證券
4. 因果關係：使該有價證券之價格發生偏離

實務上認定「連續」：
- 不需要天天交易，但須有一定密度
- 法院綜合交易頻率、數量、價格偏離幅度判斷

「高價」的認定：
- 不限於當日最高價
- 若以接近或等於委託當時揭示賣價買入，可認為高價買入
```

### 5-3 法律效果

| 效果 | 內容 | 條文 |
|---|---|---|
| **刑事責任** | 3-10 年有期徒刑，得併科 1000-2 億元罰金 | §171 I ① |
| **民事責任** | 對善意買賣之人負損害賠償 | §155 III |
| **賠償範圍** | 善意取得人之買入或賣出價格，與操縱行為結束後 10 日收盤平均價之差額 | 實務見解 |

## 六、短線交易歸入權（§157）

> 不需要證明有內線消息，只要「六個月內反向交易有獲利」就要歸入。

### 6-1 要件總表

| 要件 | 內容 | 說明 |
|---|---|---|
| **主體** | 公司董事、監察人、經理人、持股超過 10% 之股東 | 與 §157-1 的內部人範圍類似但不完全相同 |
| **客體** | 該公司之股票或其他具有股權性質之有價證券 | 含轉換公司債、認購權證等 |
| **行為** | 取得後 6 個月內賣出，或賣出後 6 個月內再買進 | 不問先買後賣或先賣後買 |
| **結果** | 因此獲得利益 | 歸入權標的 |
| **主觀** | **不問主觀意圖** | 這是最重要的特色——客觀認定 |

### 6-2 歸入計算方法

```
最高價減最低價法（實務通說）：
- 將六個月內的所有買入與賣出配對
- 以最高賣價減最低買價、次高賣價減次低買價...
- 所有正差額加總即為應歸入利益
- 負差額不予扣抵

例：
  1月買入 100股 @50元
  3月買入 100股 @60元
  4月賣出 100股 @80元
  5月賣出 100股 @55元

  配對：80（最高賣價）- 50（最低買價）= 30 × 100 = 3,000元
       55（次高賣價）- 60（次低買價）= -5 → 不計

  應歸入利益 = 3,000元
```

### 6-3 歸入權行使

| 項目 | 內容 |
|---|---|
| **請求權人** | 公司（由董事會代表） |
| **股東代位** | 公司未於 30 日內行使 → 股東得以 30 日之限期請求董事會行使；届期不行使者，請求之股東得**為公司行使歸入權** |
| **歸入標的** | 獲得之利益歸於公司 |
| **除斥期間** | 自獲得利益之日起 2 年 |

### 6-4 與內線交易的關鍵區別

| 比較 | 短線交易歸入權（§157） | 內線交易（§157-1） |
|---|---|---|
| **是否需要消息** | 不需要 | 必須有重大消息 |
| **主觀要件** | 不問主觀 | 須實際知悉 |
| **時間限制** | 6 個月內反向交易 | 消息公開前 |
| **法律效果** | 民事歸入（利益歸公司） | 刑事＋民事（賠償投資人） |
| **規範目的** | 事前嚇阻（推定利用內線） | 事後追訴（保護市場公平） |

## 七、公開收購（§43-1 以下）

### 7-1 強制公開收購 vs 任意公開收購

| 比較項目 | 強制公開收購 | 任意公開收購 |
|---|---|---|
| **法源** | §43-1 III | §43-1 I |
| **觸發條件** | 任何人單獨或共同取得公開發行公司已發行股份總數**超過一定比例**（依子法規定） | 收購人自行決定 |
| **是否有選擇** | 依法強制，須以公開收購方式為之 | 自願發動 |
| **目的** | 保護市場上其他股東——大量收購應讓所有股東都有出售機會 | 收購人策略性取得經營權 |
| **申報** | 向主管機關申報（§43-1 I） | 向主管機關申報 |

### 7-2 公開收購程序

| 步驟 | 內容 | 條文 |
|---|---|---|
| 1. 申報 | 向主管機關提出公開收購申報書 | §43-1 I |
| 2. 公告 | 公告收購條件（價格、期間、數量等） | §43-2 |
| 3. 收購期間 | 不得少於 20 日，不得超過 50 日 | §43-2 |
| 4. 條件限制 | 收購期間內不得變更收購條件（原則） | §43-2 |
| 5. 撤回限制 | 公告後除主管機關核准外不得撤回 | §43-2 |
| 6. 應賣人權利 | 期間屆滿前得撤回 | §43-2 |
| 7. 按比例收購 | 應賣數量超過預定收購數量 → 按比例收購 | §43-2 |

### 7-3 重要規定

| 規定 | 內容 |
|---|---|
| **競爭公開收購** | 他人得於收購期間內提出競爭公開收購 |
| **防禦措施限制** | 被收購公司之董事會不得有不合理之防禦措施 |
| **資訊揭露** | 被收購公司董事會應就收購條件發表意見，供股東參考 |

## 八、證券詐欺（§20）

> §20 是證交法的「一般反詐欺條款」，類似美國 Rule 10b-5。

### 8-1 §20 I 反詐欺條款

| 要件 | 內容 |
|---|---|
| **主體** | 任何人（不限內部人） |
| **行為** | 有價證券之募集、發行、私募或買賣，不得有虛偽、詐欺或其他足致他人誤信之行為 |
| **客觀** | 虛偽（misrepresentation）、詐欺（fraud）、足致他人誤信 |
| **主觀** | 故意（通說；有爭議是否包含重大過失） |
| **因果關係** | 行為與他人投資決定之間具因果關係 |

### 8-2 §20 II — 發行人財報不實責任

| 項目 | 內容 |
|---|---|
| **主體** | 發行人申報或公告之財務報告及財務業務文件 |
| **行為** | 有虛偽或隱匿之情事 |
| **效果** | 發行人及其負責人對善意信賴之人負賠償責任 |

### 8-3 §20 III — 損害賠償

| 項目 | 內容 |
|---|---|
| **請求權人** | 善意之有價證券取得人或出賣人 |
| **賠償義務人** | 違反 §20 I 之人 |
| **舉證責任** | 被害人就「行為」舉證；行為人就「免責事由」舉證 |

### 8-4 §20 vs §20-1 的關係

| 項目 | §20 | §20-1 |
|---|---|---|
| **性質** | 一般反詐欺 | 針對財報不實的特別規定 |
| **適用範圍** | 所有證券詐欺行為 | 主要財務報告或財務業務文件不實 |
| **責任主體** | 任何行為人 | 發行人、負責人、會計師等專業人士 |
| **免責** | 行為人證明無故意過失 | §20-1 II-V 各款有不同免責要件 |
| **關係** | 普通法 | 特別法（優先適用） |

## 九、董監事與大股東義務

### 9-1 持股申報義務

| 義務 | 主體 | 內容 | 條文 |
|---|---|---|---|
| **就任申報** | 董事、監察人、經理人及持股超過 10% 之股東 | 就任或持股超過 10% 之日起 2 日內申報 | §25 I |
| **持股變動申報** | 同上 | 每月持股變動情形於次月 5 日前申報 | §25 II |
| **全體董監持股成數** | 全體董事、監察人 | 應維持最低持股成數 | §26 |

### 9-2 轉讓限制

| 限制 | 內容 | 條文 |
|---|---|---|
| **事前申報** | 董監事及大股東欲轉讓持股，應於轉讓前向主管機關申報 | §22-2 |
| **禁止轉讓期間** | 股東會前停止過戶期間內不得轉讓 | §22-2 |
| **數量限制** | 每一交易日轉讓不得超過一定比例 | 子法規定 |

### 9-3 全體董監持股成數（§26）

```
重要規定：
- 全體董事、監察人合計最低持股成數由主管機關規定
- 不足者 → 主管機關得限期補足
- 届期未補足 → 全體董監事當然解任（這是嚴厲效果！）

考試陷阱：
- 是「全體」合計，不是個別董監事
- 效果是「當然解任」，非經主管機關或法院裁定
```

## 十、民事賠償責任體系

> 證交法的民事責任條文散落各處，考試常要求「比較各條的差異」。

### 10-1 核心條文比較表

| 條文 | 適用情形 | 責任主體 | 請求權人 | 賠償範圍 | 歸責原則 |
|---|---|---|---|---|---|
| **§20 I** | 一般證券詐欺 | 任何行為人 | 善意取得人/出賣人 | 因詐欺所受損害 | 故意（通說） |
| **§20-1** | 財報/財務文件不實 | 發行人/負責人/會計師 | 善意取得人/出賣人 | 差額損害 | 發行人無過失；負責人推定過失；會計師推定過失 |
| **§32** | 公開說明書不實 | 發行人/負責人/專業人士/證券商 | 善意取得人 | 取得價格與真實價值之差額 | 推定過失（舉證責任倒置） |
| **§157-1 III** | 內線交易 | 內部人/消息受領人 | 善意從事相反買賣之人 | 消息公開後10日均價與交易價之差額 | 知悉即負責 |

### 10-2 §20-1 深度解析

| 責任主體 | 歸責原則 | 免責事由 |
|---|---|---|
| **發行人** | 無過失責任 | 不得免責 |
| **發行人之負責人** | 推定過失 | 證明已盡相當注意且有正當理由可合理確信無虛偽隱匿 |
| **曾簽證之會計師** | 推定過失 | 證明已經合理調查且有正當理由確信無虛偽隱匿 |
| **承銷商** | 推定過失 | 同上 |

### 10-3 §32 公開說明書責任

```
責任主體（§32各款）：
1. 發行人及其負責人
2. 發行人之職員，曾在公開說明書上簽章者
3. 該有價證券之證券承銷商
4. 會計師、律師、工程師或其他專門職業或技術人員，
   曾在公開說明書上簽章或出具意見書者

免責事由：
  證明已盡相當之注意，並有正當理由確信其主要內容無虛偽隱匿

特色：
  - 舉證責任倒置（由被告證明已盡注意）
  - 保護善意取得人（不問是否在初級市場取得）
```

### 10-4 賠償範圍速記

| 條文 | 計算方式 | 白話 |
|---|---|---|
| §20 I | 所受損害（因果關係） | 因為被騙多付或少收的錢 |
| §20-1 | 差額說 | 不實揭露前後的價格差 |
| §32 | 取得價格與真實價值之差額 | 公開說明書騙你多付的錢 |
| §157-1 III | 消息公開後 10 日均價與交易價差額 | 內線交易讓你損失的價差 |

## 十二、易混淆概念對照表

### 對照 1：內線交易（§157-1）vs 操縱市場（§155）

| 項目 | 內線交易（§157-1） | 操縱市場（§155） |
|---|---|---|
| **核心行為** | 利用未公開之重大消息買賣 | 人為操控市場價格或交易量 |
| **有無消息** | 有重大未公開消息 | 不需要特定消息 |
| **主體限制** | 限內部人及消息受領人 | 任何人 |
| **侵害法益** | 資訊平等（fairness） | 價格形成機制（integrity） |
| **典型案例** | 知悉公司將被併購，先行買入 | 連續大量高價買入拉抬股價 |
| **刑事責任** | §171 I ②，3-10年 | §171 I ①，3-10年 |

### 對照 2：強制公開收購 vs 任意公開收購

| 項目 | 強制公開收購 | 任意公開收購 |
|---|---|---|
| **法源** | §43-1 III | §43-1 I |
| **發動原因** | 取得股份超過法定比例，依法必須 | 收購人自行決定 |
| **目的** | 保護小股東（被動） | 取得經營權（主動） |
| **收購價格** | 不得低於一定條件 | 收購人自訂（但不得低於法定下限） |
| **能否附條件** | 原則不得附條件 | 得附最低收購數量條件 |

### 對照 3：短線交易歸入權（§157）vs 內線交易（§157-1）

| 項目 | 短線交易歸入（§157） | 內線交易（§157-1） |
|---|---|---|
| **是否需消息** | 不需要 | 必須有重大未公開消息 |
| **主觀要件** | 完全不問（客觀歸責） | 須實際知悉 |
| **時間要件** | 6 個月內反向交易 | 消息公開前 |
| **法律效果** | 利益歸入公司（民事） | 刑事＋民事賠償投資人 |
| **規範邏輯** | 事前嚇阻 | 事後追訴 |
| **行使權人** | 公司（或股東代位） | 善意相反交易之投資人 |

### 對照 4：§20 vs §20-1 vs §32

| 項目 | §20 | §20-1 | §32 |
|---|---|---|---|
| **適用情形** | 一般證券詐欺 | 財報/文件不實 | 公開說明書不實 |
| **性質** | 一般法 | 特別法 | 特別法 |
| **歸責** | 故意（通說） | 發行人無過失；其餘推定過失 | 推定過失 |
| **時機** | 隨時 | 定期/不定期報告 | 募集發行時 |

### 對照 5：「沖洗買賣」vs「相對委託」

| 項目 | 沖洗買賣（§155 I ②） | 相對委託（§155 I ①） |
|---|---|---|
| **行為人** | 一人（自己跟自己交易） | 二人以上（通謀） |
| **行為態樣** | 自己同時為買賣之相對人 | 與他人約定，於同一時間以相同價格買賣 |
| **交易本質** | 虛假交易（無真實移轉） | 虛假交易（預謀配合） |
| **目的** | 製造交易量假象 | 製造交易量假象 |

### 對照 6：「實際知悉」vs「獲悉」vs「利用」

| 用語 | 出現條文 | 意義 |
|---|---|---|
| **實際知悉** | §157-1 I | 主觀上確實知道該消息（不限書面告知） |
| **獲悉** | §157-1 I ⑤ | 消息受領人從內部人處得知消息 |
| **利用** | 學說主張 | 不僅知悉，還據此進行交易決策 |

### 二試答題公式

```
證交法申論題的標準答題步驟：

1. 爭點提出：本題涉及「○○制度」之適用問題
2. 法律依據：依證券交易法第○條規定...
3. 要件分析：本條之要件為①...②...③...④...
4. 涵攝（套用事實）：本題中，甲之行為...符合/不符合上開要件
5. 法律效果：依法應...（區分刑事/民事/行政責任）
6. 附帶爭點（若有）：此外，尚涉及...

內線交易題答題模板：
Step 1：甲是否為 §157-1 I 之規範主體？→ 檢驗各款
Step 2：系爭消息是否具「重大性」？→ 質量標準
Step 3：消息是否已「明確（成立）」？→ §157-1 IV 判斷
Step 4：甲是否於消息公開前「實際知悉」？→ 知悉說/利用說
Step 5：甲是否於消息未公開前進行交易？→ 時點認定
Step 6：法律效果 → 刑事（§171）＋民事（§157-1 III）

時間分配建議（以 25 分鐘一題為準）：
- 讀題＋構思：5 分鐘
- 書寫：18 分鐘
- 檢查：2 分鐘
```
