# Vol 3 崩潰篇（ch161-180）情感密度圖

> **覆蓋範圍**：Vol 3 第一篇「末日搶救」
> **核心命題**：世界崩了。六十秒。你只能救四成。剩下六成——你能做的只有站在空白的城市裡，看著忘了結帳的人茫然地走。
> **情感主軸**：從末日紅色警報 → 六十秒搶救 → 四成的絕望 → 空白的城市 → 孕婦的胎動 → 橘子貓 → 嚴柏翰交出金鑰 → 地底第五層 → 茉莉花再現 → 碎片拾荒者的等候 → 手記得的弧度 → 米亞的恐懼 → M.C. 的確認
> **設計挑戰**：崩潰篇是 Vol 3 的第一篇，也是全書第一次「城市級災難」。它從 Vol 2 ch160 的 9 分「碎裂中的希望」直接跳到 10 分「希望被末日碾壓」——零緩衝。20 章必須完成三件事：(1) 引爆鸚鵡 2.0 並展現後果；(2) 從恐懼過渡到悲傷再到微弱的希望；(3) 為溯源篇（ch181-200）鋪設追蹤線索。最大的風險是「災難疲勞」——連續 20 章的高壓可能讓讀者麻木。解決方案是「三波結構」+ 精確配置呼吸章。
