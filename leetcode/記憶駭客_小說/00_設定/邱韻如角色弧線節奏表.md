# 邱韻如 角色弧線節奏表

> **核心主題**：愛是刪除自己也要保護對方——但當「保護」本身是被植入的信念呢？
> **弧線全長**：ch35（名字首次出現）→ ch157（按下刪除鍵）→ ch217（信念崩塌）→ ch293（花園中微笑）
> **設計原則**：她的主體性是一條拋物線——從零（傀儡）升到最高點（覺醒），再自願歸零（選擇遺忘）。自願的零 ≠ 被迫的零。

## 壹、四重身份結構

```
【第一層：原始身份（2050-2079）】
  台大神經科學研究者 → 記憶解放陣線創始人之一（織網者）
  與嚴柏翰相戀（實驗室同事 → 戰友 → 情人）
  語青的母親（語青 ~6 歲時的茉莉花茶、紙鶴、撫臉）
  鸚鵡計畫的共同設計者（初衷：保護記憶的備份系統）

【第二層：周明哲的傀儡（2079 後）】
  記憶銀行「首席技術顧問」= 合法身份
  織網者 = 記憶解放陣線地下領袖
  雙重身份看似矛盾，實則兩邊都在替周明哲辦事
  核心信念「記憶改寫 = 消除人類痛苦」= 被植入的

【第三層：覺醒者（Vol 3 ch217 後）】
  發現自己的信念是被設計的
  「什麼才是真正屬於我的？」——全書最痛的自問
  開始用自己的意志行動（而非執行被植入的使命）

【第四層：選擇遺忘者（Vol 4 ch293）】
  主動改寫自己的記憶
  不是被迫。不是逃避。是「我選擇讓一切重來」。
  花園裡澆花的女人。不記得任何事。但手記得種茉莉。
```

### ch59 咖啡店——身體記憶的極致

```
她坐在靠窗的位置。
短髮，深色系套裝，金框眼鏡。
像一個普通的中年女性技術顧問。

嚴柏翰在三桌之外。
他沒有走過去。
她不認識他。

但她起身走向出口時，
經過他的桌邊——
她的手在空中停了一下。

不是猶豫。
不是認出。
是她的手記得一個人的輪廓，
但她的腦不知道那是誰。

然後她的手收回去了。
她走了。
```

**寫作指令**：
1. 不要讓她認出嚴柏翰——身體記得 ≠ 腦記得
2. 「手停在空中」不超過一行描寫。越短越有力
3. 嚴柏翰不追。他只是看著她離開。像看一場自己無法參與的夢

### ch70 十分鐘——母女重聚

```
邱韻如的眼神在第三秒改變。
空白 → 困惑 → 恐懼 → 認出。

她的手摸上語青的臉。
（手的軌跡跟二十年前一模一樣。）

「妳長大了。」

十分鐘後，
保護機制恢復。
她的眼神退回空白。
手從語青的臉上滑落。

語青沒有哭。
她已經練習過無數次不哭了。
```

**寫作指令**：
1. 不描寫邱韻如「想起」的過程——直接跳到「認出」的結果
2. 「手的軌跡」這個細節只寫一次。讀者記住就夠了
3. 語青「不哭」比哭更痛。因為她準備好了。她知道只有十分鐘

### ★★★ ch156-157 按下刪除鍵——全書最安靜的暴力

```
記憶的最後一幀：

邱韻如坐在嚴柏翰的床邊。
他已經被注射了鎮靜劑。意識模糊。

她沒有哭。
（這是她的 DNA——面對最大的痛苦時，她不哭。）

她的手指滑過他左眼的傷疤——
那道疤是遊行中被鎮暴盾牌劃的。
是她沒能保護他的證明。

她輕輕吻了他的額頭。

然後轉身面對操作台。
螢幕上跳動著一行字：
「確認覆寫？」

她按下了刪除鍵。
手指沒有顫抖。
```

**寫作指令**：
1. **不要寫她的內心獨白**——讓動作說話。吻額頭 + 按刪除 = 全部
2. **不要讓她猶豫**——她已經猶豫了很久了。這個場景是結果，不是過程
3. **「手指沒有顫抖」是最殘忍的一句**——因為她做過最痛的決定，但她的身體執行得很精準
4. 嚴柏翰意識模糊 = 他在半夢半醒中看到了這一切。這比完全不知道更痛
5. 這個場景不超過 200 字。少一個字都比多一個字好

### ch203 茉莉花——「我不記得為什麼種它」

```
她辦公桌上有一盆茉莉花。
半死不活。葉子邊緣發黃。

她不記得什麼時候種的。
不記得為什麼種。
但她每天早上第一件事是澆水。

這不是記憶。
這是肌肉。
是手記得一個味道、一個人、一段時光——
腦子全部忘了，手沒忘。
```

**寫法規則**：像阿嬤的那碗留著的豆漿。同一個母題的變奏。不需要解釋——讀者已經在 ch3 見過了。

### ch217 信念崩塌——全書最安靜的核爆之一

```
維倫把證據攤開：
周明哲的原始程式碼中，
有一段針對邱韻如的「信念注入」模組。

「記憶改寫可以消除人類痛苦」——
這句話不是她自己想的。
是被寫進去的。

她用這個信念活了十年。
用它改寫了嚴柏翰的記憶。
用它成為織網者。
用它推動了整個鸚鵡計畫。

現在她知道了：
她不是在拯救世界。
她是在執行一段程式碼。

（沉默。很長的沉默。）

「什麼才是真正屬於我的？」
```

**寫作指令**：
1. 這是她弧線的最低點。比被刪除記憶更痛——因為那次至少她以為是自己的選擇
2. 不要讓別人安慰她。這個問題只有她自己能回答
3. 答案在 Vol 4 ch293：**「我選擇遺忘」——這一次，是真正的選擇**

## 陸、第四卷弧線（ch241-293）：選擇與解放

### ch270 真正的茶葉——最小的選擇 = 最大的自由

```
她以前喝的都是合成茉莉花茶。
濃郁，完美，穩定。
是記憶銀行標配的。

今天她用了真正的茶葉。
味道比較淡。
有一點點苦。
不穩定——每一杯濃度都不同。

「味道比較淡。」
她看著杯中淡黃色的茶湯。
「但比較真。」
```

**為什麼這個場景重要**：
- 合成茶 = 被設計的人生
- 真茶 = 不完美但屬於自己的選擇
- 這是她回答 ch217「什麼才是真正屬於我的」的方式——不是宣言。是一杯茶

### ch293 花園——弧線終點

```
嚴柏翰站在社區花園的圍欄外面。

花園裡，一個短髮女人正在澆花。
茉莉花。開得很好。

她不認識他。
她不認識任何人。
她選擇了改寫自己的全部記憶。

不是逃避。
不是崩潰。
是她在所有選擇中——
選了唯一一個真正屬於自己的。

「跟十年前一模一樣。」
嚴柏翰低聲說。

她的微笑。
沒有記憶的負擔。
純粹得像花園裡的陽光。

他轉身走了。
沒有進去。
```

**寫作指令**：
1. 不要讓嚴柏翰進花園。進去 = 替她做決定 = 違反選項 C
2. 「跟十年前一模一樣」= 跨全書呼應。她的微笑從未改變——改變的是記住微笑的人
3. 她選擇遺忘 ≠ 悲劇。這是**自由**。全書唯一一個真正做到「選項 C」的人

## 拾、寫作指引

### 語言 DNA
- **不哭。手穩。話少。**——她的情感全部在動作裡
- 她說話像寫程式碼：精準、簡短、不重複
- 唯一的「失控」是 ch202 的眼淚——那一次要寫好
- 內心獨白越少越好。讓讀者猜她在想什麼

### 與嚴柏翰的寫法規則
1. ❌ Vol 1-2 不要讓他們正面對話——她不認識他
2. ❌ 不要讓嚴柏翰說「我原諒她」——理解 ≠ 原諒
3. ❌ ch293 不要讓嚴柏翰進花園——那等於替她做選擇
4. ✅ 每次寫他們的場景，至少一個「身體記得」的細節
5. ✅ 茉莉花在兩人同框時永遠存在（味道、實物、記憶）

### 與語青的寫法規則
