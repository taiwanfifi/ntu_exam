# 第二卷新角色與勢力設定表

> **用途**：Vol 2 新登場角色的完整設定 + 三方勢力動態 + 既有角色 Vol 2 弧線轉變
> **與 Vol 1 的銜接**：Vol 1 只有「維倫的世界」——工作室、地下街、委託人。Vol 2 世界炸開：三方政治、城市危機、歷史真相。新角色是這個膨脹世界的載體。

### 1. 何志鴻——議員的深淵（ch86-92）

| 欄位 | 設定 |
|------|------|
| 身份 | 新台北市議員，連任三屆，表面形象清廉 |
| 年齡 | 52 歲 |
| 記憶損壞 | 深層記憶中藏有記憶賄賂紀錄 + 外來碎片（蜂巢意識伏筆 #1） |
| 真相 | 不是記憶賄賂——是記憶「清洗」：有人替他刪除了投票前的猶豫和良心不安 |
| 哲學命題 | 記憶裡的選擇被刪掉後，那個選擇還算「你的」嗎？ |
| 出場章節 | ch86-92（委託案 #11） |
| 演算法載體 | DFS/BFS/Tree——他的大腦是 Vol 2 第一座記憶迷宮 |

**角色功能**：
- 「深潛」教學的第一個試驗場——讀者透過他的大腦學會 DFS/BFS
- 「清洗」比「賄賂」更可怕的反轉 = Vol 2 第一次主題升級：系統性的惡 > 個體的惡
- ch92 的「外來碎片」= 蜂巢意識伏筆 #1 的物理容器

**寫作要點**：
- 不要讓他成為「壞人」。他是受害者——被刪掉良心的人更可憐
- 深潛進他大腦時，迷宮的「恐怖」來自防禦體（第四層的巨大眼球），不是他本人
- 出場 7 章後完全退場。不要讓他回來
