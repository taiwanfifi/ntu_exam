> 核心策略：記類型、背要件、跑流程、辨概念

## 一、行政行為類型速查表

### 口訣：「處契命規指計」（處分、契約、命令、規則、指導、計畫）

### 1-1 行政處分（最重要！考試佔比最高）

| 項目 | 內容 |
|------|------|
| **定義** | 行政機關就公法上具體事件所為之決定或其他公權力措施，而對外直接發生法律效果之單方行政行為（行政程序法 §92） |
| **白話** | 政府對你做了一個有法律效力的決定（核發執照、開罰單、拆除命令等） |
| **五要件** | ① 行政機關所為 ② 公法上行為 ③ 具體事件（非抽象通案） ④ 對外（非機關內部） ⑤ 直接發生法律效果 |
| **法律效果** | 存續力（確定力）、構成要件效力、執行力 |
| **救濟方式** | 訴願 → 行政訴訟（撤銷訴訟為主） |
| **考試重點** | 判斷是否為行政處分是行政法最基本的功夫，不是處分就不能提撤銷訴訟 |

### 1-2 行政契約

| 項目 | 內容 |
|------|------|
| **定義** | 行政機關與人民間，就公法上法律關係之設定、變更或消滅，互相意思表示一致所締結之契約（§135） |
| **白話** | 政府跟你「談好」一個公法上的合約（如 BOT、公地放領契約） |
| **類型** | ① 隸屬契約（機關 vs 人民，地位不對等）② 對等契約（機關 vs 機關） |
| **合法要件** | ① 不得違反法律強制規定 ② 須有法規依據或屬於職權 ③ 不得代替行政處分（除非法規許可或有正當理由）④ 人民之給付與機關之對待給付須合比例 |
| **法律效果** | 契約拘束力、可依 §147 因情事變更調整或終止 |
| **救濟方式** | 直接提行政訴訟（一般給付訴訟），不經訴願 |

### 1-3 法規命令

| 項目 | 內容 |
|------|------|
| **定義** | 行政機關基於法律授權，對多數不特定人民就一般事項所作抽象之對外發生法律效果之規定（§150） |
| **白話** | 政府根據法律授權訂的「實施細則」、「辦法」等，對外有約束力 |
| **要件** | ① 有法律授權 ② 授權內容、目的、範圍須明確 ③ 須經發布程序 |
| **法律效果** | 對外拘束人民，法院審查時得附帶審查其合法性 |
| **救濟方式** | 不能直接提訴願或訴訟（因為不是處分）；但可在個案中附帶挑戰（爭執依該命令作成之處分時，一併主張命令違法） |

### 1-4 行政規則

| 項目 | 內容 |
|------|------|
| **定義** | 上級機關對下級機關，或長官對屬官，依其權限或職權為規範機關內部秩序及運作之規定（§159） |
| **白話** | 政府內部的「作業手冊」，原則上不對外生效 |
| **類型** | ① 組織性規則 ② 作業性規則 ③ 裁量基準（最重要！間接對外產生效力） ④ 解釋性規則 |
| **法律效果** | 原則上僅拘束內部；但裁量基準透過行政自我拘束原則，間接對外發生效力 |
| **救濟方式** | 不能直接爭訟；但可在個案中主張行政機關違反裁量基準→平等原則 |
| **考試重點** | 法規命令 vs 行政規則的區分是必考題 |

### 1-5 行政指導

| 項目 | 內容 |
|------|------|
| **定義** | 行政機關在其職權範圍內，為實現一定之行政目的，以輔導、協助、勸告等不具法律上強制力之方法，促請特定人為一定作為或不作為（§165） |
| **白話** | 政府「建議」你做或不做某事，但沒有強制力 |
| **要件** | ① 在職權範圍內 ② 不具法律上強制力 ③ 相對人明確拒絕後不得繼續 |
| **法律效果** | 無法律上強制力，相對人不服從也不會被處罰 |
| **救濟方式** | 原則上無須救濟（因無法律效果）；但若行政指導實質上已構成強制，可認定為事實上的行政處分 |

### 1-6 行政計畫

| 項目 | 內容 |
|------|------|
| **定義** | 行政機關為將來一定期限內達成特定之目的或實現一定之構想，事前就達成該目的或實現該構想有關之方法、步驟或措施所為之設計與規劃（§163） |
| **白話** | 政府的長期施政藍圖（如都市計畫、國土計畫） |
| **法律效果** | 視計畫性質而定：確定計畫（如都市計畫個別變更）有行政處分性質；綱要計畫則無 |
| **救濟方式** | 若有處分性質 → 訴願＋行政訴訟；若無 → 原則上不可爭訟 |

## 二、行政處分核心整理

### 2-1 合法要件

| 類別 | 合法要件 | 白話說明 |
|------|---------|---------|
| **主體** | 有管轄權之行政機關 | 對的機關做的決定 |
| **程序** | 踐行正當程序（陳述意見、聽證等） | 做決定之前有讓你講話 |
| **方式** | 書面或其他法定方式 | 用對的形式通知你 |
| **內容** | 明確、可能、合法、合目的 | 內容要說清楚，不能違法，也不能叫你做不到的事 |
| **法律依據** | 有法律保留之依據 | 要有法律讓政府做這件事 |

### 2-2 瑕疵類型：無效 vs 得撤銷

| 比較項目 | 無效（§111） | 得撤銷（§117） |
|---------|-------------|---------------|
| **白話** | 處分錯得太離譜，根本不算數 | 處分有錯，但先算數，除非被撤銷 |
| **效力** | 自始無效，從來沒生效過 | 在被撤銷前仍有效 |
| **法定事由** | ① 不能由書面明確知悉 ② 應以書面而未給書面 ③ 內容對任何人皆無法實現 ④ 所命為犯罪行為 ⑤ 內容違背善良風俗 ⑥ 未經授權而附款逾越法定限制 ⑦ 其他有重大明顯瑕疵 | 凡不屬於無效事由的其他違法（要件不符、程序瑕疵等） |
| **主張方式** | 任何人隨時可主張（不受期限限制） | 須在法定期限內提訴願→行政訴訟 |
| **判斷口訣** | 「重大＋明顯」＝無效 | 「一般違法」＝得撤銷 |
| **考試技巧** | 選擇題看到「無效」，先想有沒有符合 §111 的七款事由。不確定就選「得撤銷」，因為無效是例外 |

### 2-3 違法處分的撤銷 vs 合法處分的廢止

| 比較項目 | 撤銷（§117） | 廢止（§122-§125） |
|---------|-------------|------------------|
| 前提 | 處分「違法」 | 處分「合法」 |
| 效力 | 原則溯及失效（回到沒做過的狀態） | 原則向將來失效 |
| 信賴保護 | 須衡量公益 vs 信賴利益 | 須衡量；若因廢止致人民損害，應予補償 |
| 行使期限 | 知有撤銷原因起 2 年內（§121） | 廢止原因發生後 2 年內（§124） |

### 2-4 附款類型（§93）

| 附款類型 | 白話說明 | 舉例 |
|---------|---------|------|
| **期限** | 處分效力有開始或結束的時間點 | 營業許可至 2027 年 12 月 31 日止 |
| **條件** | 處分效力繫於未來不確定事實的成就 | 「若通過環評，核發建築許可」 |
| **負擔** | 要求相對人做某事（額外義務） | 核發建照，但須設置無障礙設施 |
| **負擔之保留** | 保留事後附加負擔的權利 | 「主管機關得視情況追加條件」 |
| **廢止權之保留** | 保留事後廢止處分的權利 | 「主管機關必要時得廢止本許可」 |

**附款合法要件（考試常考）：**
- 法規允許或處分為裁量處分
- 附款不得違反處分目的
- 附款與處分目的須有合理關聯（禁止不當聯結原則，§94）
- 附款須合比例原則

**授益處分 vs 負擔處分（貫穿行政法的核心區分）：**

| 比較項目 | 授益處分 | 負擔處分 |
|---------|---------|---------|
| 白話 | 給你好處的處分（如核發許可、補助金） | 給你義務或不利益的處分（如罰鍰、拆除命令） |
| 撤銷限制 | 撤銷受信賴保護限制（§117、§119） | 撤銷較無限制（人民不會信賴違法的負擔） |
| 廢止限制 | 廢止受嚴格限制（§123 列舉事由） | 原則上可廢止（對人民有利） |
| 陳述意見 | 限制或剝奪授益前，應給予陳述意見機會（§102） | 作成負擔處分前，應給予陳述意見機會 |
| 考試重點 | 先判斷是授益還是負擔，決定適用哪套撤銷/廢止規定 |

## 三、行政程序法重點

### 3-1 正當程序：陳述意見 vs 聽證

| 比較項目 | 陳述意見（§102-§109） | 聽證（§107、§155） |
|---------|---------------------|-------------------|
| **白話** | 簡易版的聽你說話程序 | 正式版的聽你說話程序（像開庭一樣） |
| **形式** | 書面或口頭均可 | 須公開舉行、作成紀錄 |
| **適用時機** | 作成限制或剝奪人民權利的處分前（原則） | 法規明定應舉行聽證時 |
| **程序要求** | 通知相對人、給予陳述機會 | 公告、通知、言詞辯論、製作紀錄、斟酌全部結果 |
| **得免除情形** | §103 八款事由（如大量作成同種類處分、情況急迫等） | 無免除規定（法定應聽證就要聽證） |

### 3-2 信賴保護原則

```
信賴保護三要件：
□ 1. 信賴基礎：有一個「國家行為」讓你產生信賴（如行政處分、法規）
□ 2. 信賴表現：你基於信賴做了具體的安排或處置
□ 3. 信賴值得保護：你的信賴沒有不值得保護的情形
     ├ 不值得保護的情形（§119）：
     │  ① 以詐欺、脅迫、賄賂取得
     │  ② 對重要事項提供不正確資料或不完全陳述
     │  └ ③ 明知或因重大過失不知行政處分違法
```

**信賴保護的法律效果：**
- 存續保護（維持原處分）→ 優先適用
- 財產保護（撤銷但補償信賴損失）→ 當公益明顯大於私益時

### 3-3 比例原則在行政法的適用（§7）

| 子原則 | 內容 | 白話 |
|--------|------|------|
| 適當性 | 手段有助於目的之達成 | 你這樣做有用嗎？ |
| 必要性 | 有多種手段可達成目的時，應選擇對人民侵害最小者 | 有沒有更溫和的方式？ |
| 狹義比例 | 手段造成的損害不得與目的利益顯失均衡 | 值得嗎？ |

### 3-4 行政裁量與不確定法律概念

| 比較項目 | 行政裁量 | 不確定法律概念 |
|---------|---------|--------------|
| **白話** | 法律給行政機關「選擇空間」（可以罰也可以不罰、可罰多也可罰少） | 法律用模糊的字眼（如「公共利益」、「情節重大」），行政機關要去解釋 |
| **所在位置** | 法律效果面（罰多少、怎麼罰） | 構成要件面（符不符合條件） |
| **法院審查** | 審查有無裁量瑕疵（逾越、濫用、怠惰），不能代替行政機關做決定 | 原則上可完全審查；例外有判斷餘地（如考試評分、專業判斷） |
| **瑕疵類型** | ① 裁量逾越 ② 裁量濫用 ③ 裁量怠惰（收縮至零） | 涵攝錯誤（把法律概念套到事實時搞錯了） |

**裁量瑕疵三類型詳解（必考）：**

| 瑕疵類型 | 白話 | 舉例 |
|---------|------|------|
| **裁量逾越** | 超出法律授權的範圍（法律說罰 3-30 萬，你罰 50 萬） | 法定罰鍰上限 30 萬，機關裁處 50 萬 |
| **裁量濫用** | 形式上在範圍內，但考慮了不該考慮的因素 | 因為討厭某人就罰最重；考慮了與目的無關的因素 |
| **裁量怠惰** | 有裁量權卻不行使，一律照公式處理 | 法律規定「得」處罰，機關一律都罰，完全不個案考量 |

**裁量收縮至零（考試愛考）：**

| 概念 | 說明 |
|------|------|
| 意思 | 雖然法律給行政機關裁量空間，但在特定情況下，只剩一個合法選擇 |
| 白話 | 法律說你「可以」做，但情況特殊到你「必須」做 |
| 典型情形 | 生命身體有急迫危險時，警察「得」採取措施 → 收縮為「應」採取措施 |
| 法院審查 | 若裁量收縮至零而機關不作為，構成違法，人民可提課予義務訴訟 |

### 3-5 依法行政原則總整理

```
依法行政原則
├─ 法律優位原則（消極面）
│   └→ 行政行為不得牴觸法律
│       白話：政府做事不能違反法律
│
└─ 法律保留原則（積極面）
    └→ 行政行為須有法律依據
        白話：政府做事要有法律授權
        ├─ 侵害保留（最低限度）：侵害人民權利的行為才需法律依據
        ├─ 全面保留（最高標準）：所有行政行為都需法律依據
        └─ 重要事項保留（通說）：重要事項需法律依據（釋字 443）
```

## 四、行政救濟流程圖

### 4-1 完整救濟流程

```
人民權利受侵害
    │
    ├─ 是行政處分嗎？
    │   ├─ 是 ─→ 訴願（30日內）─→ 行政訴訟
    │   │                              ├─ 撤銷訴訟
    │   │                              ├─ 課予義務訴訟
    │   │                              └─ 確認訴訟
    │   │
    │   └─ 否 ─→ 依行為類型選擇救濟
    │       ├─ 行政契約爭議 → 一般給付訴訟（不經訴願）
    │       ├─ 事實行為損害 → 一般給付訴訟 或 國家賠償
    │       └─ 法規命令違法 → 於個案中附帶爭執
    │
    └─ 公務員身分爭議？
        ├─ 改變身分之處分（免職等）→ 復審 → 行政訴訟
        └─ 管理措施（記過等）→ 申訴、再申訴
```

### 4-2 訴願制度

| 項目 | 內容 |
|------|------|
| **功能** | 行政體系內部的自我審查（上級機關審查下級機關的處分） |
| **管轄** | 原則上向原處分機關的上級機關提起 |
| **期限** | 處分達到次日起 30 日內 |
| **審查範圍** | 合法性＋合目的性（比法院審查範圍廣！法院只審合法性） |
| **決定** | 撤銷原處分、駁回訴願、不受理 |
| **注意** | 訴願前置主義：提行政訴訟前原則上須先經訴願（但有例外） |

### 4-3 行政訴訟類型

| 訴訟類型 | 適用情形 | 訴訟要件 | 白話 |
|---------|---------|---------|------|
| **撤銷訴訟**（§4） | 人民不服行政處分 | ① 有行政處分 ② 經訴願 ③ 處分違法 ④ 損害權利 | 政府對我做了一個錯誤的決定，我要把它撤掉 |
| **課予義務訴訟**（§5） | 人民申請被拒絕或怠為處分 | ① 有申請 ② 被駁回或未處理 ③ 經訴願 ④ 有公法上請求權 | 我申請了，政府不理我或拒絕我，我要法院命令政府准我 |
| **確認訴訟**（§6） | 確認法律關係存否、處分無效或違法 | ① 確認利益 ② 不能用撤銷或課予義務訴訟者（補充性） | 我要法院宣告某個法律關係存不存在 |
| **一般給付訴訟**（§8） | 非行政處分的公法上給付請求 | ① 公法上之給付請求 ② 非處分事項 | 政府欠我錢或應賠償我，但不是處分問題 |

### 4-4 各訴訟類型的關鍵區分

```
判斷流程（必背）：

Q1: 人民要「撤掉」一個處分？ → 撤銷訴訟
Q2: 人民要「政府做」一個處分？ → 課予義務訴訟
Q3: 人民要確認法律關係？ → 確認訴訟（補充性）
Q4: 人民要公法上的給付（非處分）？ → 一般給付訴訟

口訣：「撤掉它、給我做、說清楚、給我錢」
```

### 4-5 暫時權利保護

| 類型 | 適用情形 | 條文 | 要件 |
|------|---------|------|------|
| 停止執行 | 撤銷訴訟中，防止處分繼續執行造成難以回復損害 | 行訴 §116 | ① 本案訴訟已提起或將提起 ② 處分之執行將造成難以回復之損害 ③ 有急迫情事 ④ 停止不違反公益 |
| 假處分 | 課予義務或一般給付訴訟中，暫時確保權利 | 行訴 §298 | ① 爭執之公法上法律關係 ② 有防止發生重大損害或避免急迫危險之必要 |

**重要觀念：行政處分的「執行不停止原則」**

| 項目 | 說明 |
|------|------|
| 原則 | 提起訴願或行政訴訟，不停止原處分的執行（行訴 §116 I） |
| 白話 | 你告歸你告，政府的決定照樣執行。除非你聲請停止執行且法院准許。 |
| 例外 | 原處分機關、受理訴願機關或行政法院依職權或聲請停止執行 |
| 考試重點 | 這是原則，停止執行是例外。選擇題常考「提起訴願是否停止執行？」答案是「不停止」。 |

### 4-6 行政訴訟重要期限整理

| 事項 | 期限 | 起算點 |
|------|------|--------|
| 訴願 | 30 日 | 行政處分達到或公告期滿次日 |
| 撤銷訴訟 | 2 個月 | 訴願決定書送達次日 |
| 課予義務訴訟 | 2 個月 | 訴願決定書送達次日（怠為處分：訴願決定書送達次日） |
| 確認訴訟 | 無不變期間 | 但須有確認利益 |
| 一般給付訴訟 | 無不變期間 | 但公法上請求權有消滅時效（5 年，§131 行程） |
| 國賠請求 | 2 年（知有損害時起）/ 5 年（損害發生時起） | 取較短者 |

## 五、國家賠償法速查

### 5-1 兩大請求權基礎

| 比較項目 | §2 公務員責任 | §3 公共設施責任 |
|---------|-------------|---------------|
| **白話** | 公務員執行職務搞砸了 | 政府的公共設施害人了 |
| **要件** | ① 公務員 ② 執行職務行使公權力 ③ 故意或過失 ④ 不法侵害 ⑤ 人民自由或權利 ⑥ 因果關係 | ① 公有公共設施 ② 設置或管理有欠缺 ③ 致人民生命、身體、人身自由或財產受損害 ④ 因果關係 |
| **歸責原則** | 過失責任（須證明公務員故意或過失） | 無過失責任（不需要證明政府有過失！只要設施有問題就賠） |
| **賠償義務機關** | 公務員所屬機關 | 設施之設置或管理機關 |
| **求償權** | 公務員有故意或重大過失時，賠償機關對其有求償權 | 原因可歸責他人時，對該他人有求償權 |

### 5-2 §2 第一項 vs 第二項

| 比較項目 | §2 I（積極作為） | §2 II（消極不作為） |
|---------|-----------------|-------------------|
| 白話 | 公務員「做了」不該做的事 | 公務員「沒做」該做的事 |
| 要件差異 | 行為不法即可 | 須有作為義務而怠於執行 |
| 舉例 | 警察執法過當打傷人 | 消防隊接獲通報不出勤，導致火災擴大 |

### 5-3 國賠程序

```
第一步：書面向賠償義務機關請求（協議先行主義）
    │
    ├─ 協議成立 → 取得賠償
    │
    └─ 協議不成立（拒絕賠償或逾 30 日未開始協議、逾 60 日協議未成立）
        │
        └─ 向普通法院（民事庭）提國賠訴訟
           （注意：不是行政法院！）
```

**考試重點提醒：國家賠償訴訟由「普通法院民事庭」管轄，不是行政法院。這是常考陷阱。**

### 5-4 國賠 vs 損失補償

| 比較項目 | 國家賠償 | 損失補償 |
|---------|---------|---------|
| 前提 | 國家行為「違法」 | 國家行為「合法」 |
| 白話 | 政府做錯事，要賠你 | 政府做對的事，但犧牲了你，要補償你 |
| 舉例 | 公務員違法拆除你的房子 | 政府合法徵收你的土地 |
| 法律依據 | 國家賠償法 | 各個別法（如土地徵收條例）、特別犧牲理論 |

## 六、行政罰法重點

### 6-1 基本原則

| 原則 | 內容 | 條文 |
|------|------|------|
| **處罰法定** | 違反行政法義務之處罰，以行為時之法律或自治條例有明文規定者為限 | §4 |
| **責任條件** | 故意或過失，始受處罰。不知法規不得免責，但得減輕 | §7、§8 |
| **責任能力** | 未滿 14 歲不罰；14 歲以上未滿 18 歲得減輕 | §9 |

### 6-2 故意過失

| 比較項目 | 故意 | 過失 |
|---------|------|------|
| 白話 | 明知故犯 | 應注意而未注意 |
| 行政罰的要求 | 須有故意或過失才能處罰（不是嚴格責任） | 推定過失：行為人違反行政法義務，推定有過失（舉證責任轉換） |
| 考試陷阱 | 行政罰法施行前，實務曾認為行政罰不以故意過失為要件。施行後已改採「故意過失為必要」 |

### 6-3 一行為不二罰（§24-§26）

| 情形 | 處理方式 | 白話 |
|------|---------|------|
| 一行為違反數個行政法義務 | 從一重處罰（§24 I） | 做一件壞事同時違反好幾條法，只罰最重的那條 |
| 一行為同時觸犯刑法和行政法 | 依刑法處罰；但其他種類行政罰（如停業、吊照）仍得裁處（§26） | 如果行為同時是犯罪，原則上用刑罰；但非罰鍰的行政罰還是可以另外罰 |
| 數行為違反同一行政法義務 | 分別處罰 | 做了好幾件壞事，每件都罰 |

**§26 的重點（刑罰 vs 行政罰）：**

```
一行為同時觸犯刑法與行政法：
├─ 罰鍰部分 → 依刑事法律處罰之（不再罰鍰，避免雙重處罰）
│   └─ 但若經不起訴、無罪、免刑、緩起訴等 → 仍得裁處罰鍰
└─ 非罰鍰之行政罰（吊照、停業等）→ 仍得裁處（不受影響）
```

### 6-4 裁處權時效（§27）

| 項目 | 內容 |
|------|------|
| 一般時效 | 3 年（自違反行政法上義務之行為終了時起算） |
| 特殊時效 | 法律另有規定者從其規定 |
| 效果 | 裁處權因時效消滅，行政機關不得再處罰 |

### 6-5 行政罰的種類（§1-§2）

| 種類 | 內容 | 舉例 |
|------|------|------|
| **罰鍰** | 最常見的行政罰，命繳一定金額 | 交通罰鍰、環保罰鍰 |
| **沒入** | 剝奪財物所有權 | 沒收違禁品 |
| **其他裁罰性不利處分** | 限制或禁止行為、剝奪或消滅資格或權利、警告性處分 | 吊銷執照、停業處分、公布姓名 |

### 6-6 行政罰 vs 刑罰 vs 秩序罰

| 比較項目 | 行政罰 | 刑罰 |
|---------|--------|------|
| 法律依據 | 行政罰法 | 刑法 |
| 處罰機關 | 行政機關 | 法院 |
| 程序 | 行政程序 | 刑事訴訟程序 |
| 不利益類型 | 罰鍰、沒入、吊照等 | 死刑、自由刑、罰金等 |
| 競合處理 | 一行為同時該當→刑罰優先（§26）；但非罰鍰的行政罰仍可並罰 |

## 七、10 大必考考點排行

| 排行 | 考點 | 重要度 | 一句話結論 |
|------|------|--------|-----------|
| 1 | 行政處分的認定 | ★★★★★ | 五要件缺一不可：行政機關＋公法上＋具體事件＋對外＋直接發生法律效果。是不是處分，決定了後面怎麼救濟。 |
| 2 | 撤銷訴訟 vs 課予義務訴訟 | ★★★★★ | 撤銷＝把不要的處分消掉；課予義務＝要法院命行政機關做一個你要的處分。 |
| 3 | 行政處分瑕疵（無效 vs 得撤銷） | ★★★★★ | 重大明顯瑕疵→無效；一般違法→得撤銷（原則）。 |
| 4 | 信賴保護原則 | ★★★★☆ | 三要件：信賴基礎＋信賴表現＋值得保護。§119 列出三種不值得保護情形。 |
| 5 | 裁量 vs 不確定法律概念 | ★★★★☆ | 裁量在法律效果面（法院審瑕疵），不確定概念在構成要件面（法院原則可全審）。 |
| 6 | 法規命令 vs 行政規則 | ★★★★☆ | 法規命令＝有法律授權＋對外生效；行政規則＝職權命令＋原則內部。裁量基準是行政規則但間接對外。 |
| 7 | 國家賠償（§2 vs §3） | ★★★★☆ | §2 是公務員故意過失責任，§3 是公共設施無過失責任。國賠訴訟走民事法院。 |
| 8 | 一行為不二罰 | ★★★☆☆ | 一行為違反數法→從重；刑罰優先於罰鍰，但非罰鍰之行政罰不受影響。 |
| 9 | 行政契約的合法要件 | ★★★☆☆ | 不得代替處分（除非法規許可）、不違反強制規定、人民給付須合比例。 |
| 10 | 訴願前置 vs 直接提訴訟 | ★★★☆☆ | 撤銷訴訟/課予義務訴訟須先訴願；一般給付訴訟/確認訴訟不經訴願。 |

## 八、易混淆概念對照表

### 8-1 行政處分 vs 事實行為

| 比較項目 | 行政處分 | 事實行為 |
|---------|---------|---------|
| **白話** | 政府做了一個有法律效力的「決定」 | 政府做了一個沒有法律效力的「動作」 |
| **法律效果** | 直接發生法律上的效果（設定、變更、消滅權利義務） | 不直接發生法律效果（但可能有事實上的影響） |
| **舉例** | 核發建照、開罰單、拒絕申請 | 道路施工、環境清潔、行政指導、提供資訊 |
| **救濟** | 訴願 → 撤銷訴訟 | 一般給付訴訟或國家賠償（不能提撤銷訴訟！） |
| **判斷關鍵** | 有沒有「對外直接發生法律效果」 | 只有事實上的作用，沒有法律上的效果 |

### 8-2 羈束處分 vs 裁量處分

| 比較項目 | 羈束處分 | 裁量處分 |
|---------|---------|---------|
| **白話** | 法律規定「要件符合就必須這樣做」，行政機關沒選擇 | 法律規定「要件符合，行政機關可以選擇做或不做、怎麼做」 |
| **行政機關角色** | 只是執行法律（像機器一樣） | 有判斷空間（像裁判一樣） |
| **法律用語辨識** | 「應」、「須」 | 「得」、「可以」 |
| **法院審查** | 完全審查（對錯分明） | 僅審查有無裁量瑕疵（逾越、濫用、怠惰） |
| **舉例** | 符合條件「應」核發執照 | 「得」處以罰鍰 3-30 萬元 |
| **考試技巧** | 看法條用「應」還是「得」，90% 可以判斷 |

### 8-3 法規命令 vs 行政規則

| 比較項目 | 法規命令（§150） | 行政規則（§159） |
|---------|----------------|----------------|
| **白話** | 政府依法律授權訂的「對外規定」 | 政府內部的「作業手冊」 |
| **法律授權** | 需要（一定要有母法授權） | 不需要（基於職權即可訂定） |
| **對外效力** | 有（直接對人民產生拘束力） | 原則上無（但裁量基準間接有） |
| **名稱** | 通常叫「辦法」、「規則」、「細則」、「標準」 | 通常叫「要點」、「注意事項」、「作業規定」 |
| **發布** | 須經法定發布程序（刊登政府公報） | 下達即生效（不需刊登公報） |
| **法院審查** | 可附帶審查合法性（是否逾越授權） | 原則不審查；但可審查裁量基準有無違反平等原則 |

### 8-4 訴願 vs 行政訴訟

| 比較項目 | 訴願 | 行政訴訟 |
|---------|------|---------|
| **性質** | 行政救濟（行政體系內部自我審查） | 司法救濟（獨立的法院審查） |
| **審理機關** | 上級行政機關（訴願管轄機關） | 行政法院 |
| **審查範圍** | 合法性＋合目的性（比法院廣） | 合法性（不審查目的性） |
| **期限** | 處分達到次日起 30 日 | 訴願決定達到次日起 2 個月（行訴 §106） |
| **順序** | 先訴願 | 原則上須先經訴願（訴願前置） |

### 8-5 更多易混淆概念速查

| 概念 A | 概念 B | 關鍵區分 |
|--------|--------|---------|
| 撤銷（§117） | 廢止（§122） | 撤銷＝原處分「違法」→溯及失效；廢止＝原處分「合法」→向將來失效 |
| 訴願不受理 | 訴願駁回 | 不受理＝程序不合法（遲到、不是處分等）；駁回＝實體審查後認為處分沒錯 |
| 形式存續力 | 實質存續力 | 形式＝不得再爭訟（期限過了）；實質＝處分內容拘束當事人及機關 |
| 公權力行為 | 私經濟行為 | 公權力＝單方高權行為（適用公法）；私經濟＝政府以私人地位行為（適用私法） |
| 行政執行 | 行政罰 | 行政執行＝強制實現義務（你不做我幫你做、強制你做）；行政罰＝違反義務的制裁（罰你） |
| 代履行 | 直接強制 | 代履行＝可由他人代為完成的義務（如拆違建）；直接強制＝直接對人或物實施（如斷水斷電） |

### 8-6 行政處分常見類型判斷表（選擇題必備）

| 行為 | 是不是行政處分？ | 理由 |
|------|---------------|------|
| 核發建築執照 | 是（授益處分） | 對特定人核准申請，直接發生法律效果 |
| 交通違規罰單 | 是（負擔處分） | 對特定人課予義務（罰鍰） |
| 拒絕營業許可申請 | 是（駁回處分） | 拒絕也是處分（消極處分） |
| 都市計畫通盤檢討 | 否（法規性質） | 對不特定多數人的一般性規劃，非具體事件 |
| 交通號誌設置 | 否（一般處分） | 但如果是「個別路段的禁止通行」可能是一般處分 |
| 公務員年終考績 | 視情形 | 考績結果影響身分（如丙等免職）＝處分；一般考績通知＝管理措施 |
| 教育部函釋 | 否（行政規則） | 解釋性行政規則，不對外直接發生法律效果 |
| 主管機關口頭勸導 | 否（行政指導） | 不具法律上強制力 |
| 觀念通知（催繳通知單） | 否（事實行為） | 只是通知你已有的義務，不「設定」新的法律效果 |
| 行政機關核定徵收 | 是（行政處分） | 對特定土地所有人的具體決定 |

### 8-7 行政執行法重點速查

| 項目 | 內容 |
|------|------|
| **公法上金錢給付** | 逾期未繳→移送行政執行署強制執行（限制出境、拍賣財產等） |
| **行為或不行為義務** | ① 代履行（可替代行為，如拆違建）② 怠金（不可替代行為，如命停止污染）③ 直接強制（斷水斷電、強制驅離） |
| **即時強制** | 無須行政處分為前提，直接對人或物實施（§36-§40）。例如：警察對醉漢的管束（限 24 小時）。 |
| **比例原則適用** | 行政執行仍須遵守比例原則，不得逾越必要程度 |

### 答題架構萬用模板（申論題）

```
一、行為定性
  → 本題中行政機關的行為屬於（行政處分 / 行政契約 / 事實行為...）
  → 理由：符合（§92 / §135 / 其他）之定義

二、合法性審查
  (一) 形式合法
       ① 管轄權：有無管轄權？
       ② 程序：有無踐行正當程序（陳述意見/聽證）？
       ③ 方式：書面？送達？
  (二) 實質合法
       ① 法律依據：法律保留、授權明確？
       ② 比例原則：手段適當、必要、合比例？
       ③ 裁量有無瑕疵？

三、救濟途徑
  → 應提起（訴願→撤銷訴訟 / 課予義務訴訟 / 一般給付訴訟 / 國賠）
  → 期限：（30日 / 2個月 / 其他）

四、結論
```

### 選擇題秒殺技巧

| 題型 | 秒殺口訣 |
|------|---------|
| 問「是否為行政處分」 | 跑五要件：行政機關＋公法＋具體＋對外＋法律效果，缺一就不是 |
| 問「應提何種訴訟」 | 先看有無行政處分→有＝撤銷/課予義務；無＝一般給付/確認 |
| 問「是否合法」 | 先查程序（有無陳述意見機會），再查實體（比例原則、裁量） |
| 問「國賠要件」 | §2 看故意過失，§3 看設置管理欠缺（無過失），國賠走民事法院 |
| 問「訴願 vs 行政訴訟」 | 訴願審合法＋合目的；行政訴訟只審合法性 |

### 考前口訣總整理

| 口訣 | 對應內容 |
|------|---------|
| 「處契命規指計」 | 六大行政行為類型 |
| 「機公具外效」 | 行政處分五要件（行政機關、公法上、具體事件、對外、法律效果） |
| 「重大＋明顯＝無效」 | 行政處分無效的判斷基準 |
| 「撤掉它、給我做、說清楚、給我錢」 | 四大訴訟類型（撤銷、課予義務、確認、一般給付） |
| 「訴願 30、行訴 2 月」 | 救濟期限 |
| 「國賠走民事」 | 國家賠償訴訟由普通法院管轄 |
| 「§2 要過失、§3 不用」 | 國賠兩種類型的歸責差異 |
| 「基表現值」 | 信賴保護三要件（信賴基礎、信賴表現、值得保護） |
| 「逾濫怠」 | 裁量三種瑕疵（逾越、濫用、怠惰） |
| 「期條負負廢」 | 五種附款（期限、條件、負擔、負擔之保留、廢止權之保留） |

### 常見陷阱題整理

| 陷阱敘述 | 正確觀念 |
|---------|---------|
| 「提訴願就停止執行」 | 錯。原則是執行不停止。除非聲請並獲准停止執行。 |
| 「行政規則不需要任何依據」 | 對一半。行政規則不需「法律授權」，但仍基於「職權」訂定，且裁量基準仍受平等原則拘束。 |
| 「國家賠償由行政法院管轄」 | 錯。國家賠償訴訟由普通法院民事庭管轄。 |
| 「行政處分一定要書面」 | 錯。行政處分可以口頭、書面或其他方式作成（§95）。但相對人可要求補發書面（§95 II）。 |
| 「所有行政訴訟都要先經訴願」 | 錯。撤銷訴訟和課予義務訴訟須經訴願，但一般給付訴訟和確認訴訟不經訴願。 |
| 「行政契約和民事契約一樣」 | 錯。行政契約適用行政程序法（公法），不適用民法（私法），救濟走行政法院。 |
| 「裁量處分法院不能管」 | 錯。法院可以審查裁量瑕疵（逾越、濫用、怠惰），只是不能代替機關做裁量決定。 |
| 「行政指導沒有任何救濟」 | 原則上是，但如果行政指導實質上構成強制（有事實上的不利益），可能被認定為行政處分。 |

### 行政法上重要原則總整理

| 原則 | 法條 | 白話 | 考試怎麼用 |
|------|------|------|-----------|
| 依法行政 | 行程 §4 | 政府做事要依法 | 所有行政法題目的大前提 |
| 明確性原則 | 行程 §5 | 處分內容要說清楚 | 審查處分合法性時必提 |
| 平等原則 | 行程 §6 | 同類案件同樣處理 | 挑戰裁量基準時常用 |
| 比例原則 | 行程 §7 | 手段不能太超過 | 幾乎每題都可以提 |
| 誠信原則 | 行程 §8 | 政府要守信用 | 信賴保護的上位原則 |
| 有利不利一律注意 | 行程 §9 | 不能只找你的碴 | 程序合法性審查時提 |
| 禁止不當聯結 | 行程 §94（附款） | 附款要跟處分有關 | 審查附款合法性 |

---

> 最後叮嚀：行政法考試的核心是兩件事 —— **「行為定性」** 和 **「救濟選擇」**。
> 先判斷行政機關做的事是什麼類型（處分？契約？事實行為？），
> 再決定人民該走什麼救濟途徑（訴願？哪種訴訟？國賠？）。
> 只要這兩步做對，答題架構就穩了。加油，你可以的。
