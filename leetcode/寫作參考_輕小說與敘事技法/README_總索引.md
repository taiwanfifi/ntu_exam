## 最關鍵的 8 個寫作模式

| 模式 | 代表作 | 核心技法 |
|------|--------|----------|
| 對話教學法 | 數學少女 | 角色對話中自然嵌入概念推導 |
| 期限驚悚法 | Phoenix Project | 用 deadline 壓力讓技術學習充滿緊迫感 |
| 日常之謎法 | 冰菓 | 演算法不需要大場面，生活問題就夠了 |
| 關卡遞增法 | 死亡萬花筒 | 每個章節弧線教一個新概念，難度遞增 |
| 程式碼嵌入法 | Cryptonomicon | 直接在敘事中嵌入可運行的虛擬碼/程式碼 |
| 超短章節法 | Digital Fortress | 2-4 頁的極短章節，讓讀者停不下來 |
| 硬核引擎法 | 三體 | 科學嚴謹性本身就是劇情引擎 |
| 多格式混搭法 | Listen for the Lie | 散文+程式碼區塊+debug 日誌+問題描述交替 |
