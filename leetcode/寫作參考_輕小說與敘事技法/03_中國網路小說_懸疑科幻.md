# 中國網路小說 — 懸疑 / 科幻排行榜作品

> 起點中文網、晉江文學城等平台的熱門懸疑/科幻作品。
> 這些作品深諳中文讀者的閱讀習慣，結構和節奏值得細究。

## 1. 《詭秘之主》(Lord of the Mysteries) ⭐⭐ 必讀

| 項目 | 內容 |
|------|------|
| **作者** | 愛潛水的烏賊 |
| **類型** | 克蘇魯 / 蒸汽朋克 / 推理 / 黑暗奇幻 |
| **規模** | ~1,400 章，400 萬+ 字 |
| **平台** | 起點中文網（2018.04-2020.05）；英文版在 WebNovel |
| **續作** | 《宿命之環》(2023，已完結) |
| **維基** | https://en.wikipedia.org/wiki/Lord_of_Mysteries |
| **WebNovel** | https://www.webnovel.com/book/lord-of-mysteries_11022733006234505 |

### 寫作技法分析

**伏筆回收——業界天花板**
- 被廣泛讚譽為「環環相扣」的伏筆大師
- 幾百章前埋下的細節，會在意想不到的時刻完美回收
- 讀者回看時會驚呼「原來那個時候就暗示了」

**系統化知識層級 = 世界觀**
- 「序列途徑」系統本質上是一個結構化知識體系（就像演算法的分類體系）
- 每個序列有層級，越高越強但代價越大（就像演算法優化的 trade-off）
- 讀者在追故事的同時，不知不覺學會了這個體系

**慢熱型世界觀建構**
- 前期大量鋪墊，後期爆發式收穫
- 耐心的讀者會得到極大的滿足感

**入選國家圖書館「中國首批 100 部網路小說典藏」**

### 可借鑑的技法
> **「序列途徑」≈ 演算法分類體系**：
> - 「占卜師途徑」= Searching 系列（Binary Search → DFS/BFS → A*）
> - 「戰士途徑」= Sorting 系列（Bubble → Quick → Merge → Heap）
> - 每往上一層，能力更強但也更危險（更難理解/實現）
>
> **伏筆技法**：在第 3 章提到的 hash collision，在第 15 章的案件裡成為破案關鍵。

## 2. 《壞小孩》/ 推理之王系列（紫金陳）

| 項目 | 內容 |
|------|------|
| **作者** | 紫金陳 |
| **類型** | 社會派懸疑 / 犯罪小說 |
| **規模** | 三部曲，每部 ~30 章：《無證之罪》《壞小孩》《長夜難明》 |
| **平台** | 天涯論壇首發（2012）；後出版實體書 |
| **改編** | 《隱秘的角落》入選 Variety 2020 年最佳國際電視劇 |
| **維基** | https://zh.wikipedia.org/zh-hant/紫金陳 |
| **獎項** | 《壞小孩》入圍 2023 英國犯罪作家協會「匕首獎」 |

### 寫作技法分析

**多線平行敘事**
- 《低智商犯罪》用了「七線同步敘事」技法
- 多條線索同時展開，在關鍵節點交匯——產生驚人的化學反應

**嚴密邏輯 + 意外翻轉**
- 每一步都有邏輯基礎，但結論出人意料
- 不是「為了翻轉而翻轉」，而是「你漏看了一個前提」

**社會批判深度**
- 犯罪只是表面，底下是教育、階級、制度的批判

### 可借鑑的技法
> **「多線平行」結構**非常適合教複雜演算法：
> - 線 A：主角用暴力解卡關
> - 線 B：學姊在背景默默做優化
> - 線 C：入侵者的行動在同步推進
> - 三條線交匯的瞬間 = 演算法真正的 "aha moment"

## 3. 《死亡萬花筒》⭐

| 項目 | 內容 |
|------|------|
| **作者** | 西子緒 |
| **類型** | 超自然推理 / 無限流 |
| **規模** | 139 章（2018 完結） |
| **平台** | 晉江文學城 VIP |
| **晉江** | https://www.jjwxc.net/onebook.php?novelid=3435320 |
| **豆瓣** | https://m.douban.com/book/subject/37049635/ |

### 寫作技法分析

**「關卡制」遞增難度結構**
- 12 扇鐵門 = 12 個自成一體的場景
- 每個場景是一個獨立的解謎挑戰，難度逐漸遞增
- **這個結構直接對應「LeetCode 難度遞增的題目」**

**密室逃脫式推理**
- 每個場景都有嚴格的規則和限制
- 必須在規則內找到「生存演算法」

**恐怖氛圍營造**
- 壓力和恐懼成為學習的催化劑（就像 deadline）

### 可借鑑的技法
> **最適合直接借鑑的結構**：
> - 第 1 扇門（Easy）：Linear Search / Two Sum
> - 第 2 扇門（Easy+）：Binary Search
> - 第 3 扇門（Medium）：Hash Map + DFS
> - ...
> - 第 12 扇門（Hard）：Dynamic Programming + Graph Theory
>
> 每個「場景」自成一體但又與大故事線相連。

| 項目 | 內容 |
|------|------|
| **作者** | 我會修空調 |
| **類型** | 恐怖推理 / 解謎 |
| **規模** | 1,000+ 章 |
| **平台** | 起點中文網 |
| **PTT** | https://www.ptt.cc/bbs/CFantasy/M.1542073436.A.108.html |

### 寫作技法分析
- **「每日任務」遞增難度**：本質上是遊戲化的課程結構
- **主角用邏輯推理（= debug）解決每個場景**
- **「探索禁地」= 探索新演算法領域**

### 可借鑑的技法
> 「遊戲化」的概念：每解決一個演算法，解鎖新的能力/工具。

## 5. 《我的治癒系遊戲》

| 項目 | 內容 |
|------|------|
| **作者** | 我會修空調 |
| **類型** | 現實-遊戲嵌套 / 懸疑 |
| **規模** | 1,000+ 章（完結） |
| **平台** | 起點中文網 |

### 寫作技法分析
- **現實-遊戲雙層結構**：在遊戲中發現線索，在現實中解決案件
- **每日強制登入機制**驅動快節奏
- **恐怖場景的描寫水準極高**

### 可借鑑的技法
> **「雙層結構」**：
> - 現實層 = 偵探案件
> - 「遊戲層」= LeetCode 題目 / 演算法挑戰
> - 兩層互相映照、互相提供線索

## 6. 《網內人》(Inside the Net) ⭐

| 項目 | 內容 |
|------|------|
| **作者** | 陳浩基（**資工系畢業**，香港中文大學計算機科學學位） |
| **類型** | 科技驚悚 / 推理 |
| **規模** | 20+ 章 |
| **平台** | 皇冠出版社（台灣） |
| **獎項** | 島田莊司推理小說獎、書展大獎、誠品書店推薦獎 |
| **Readmoo** | https://readmoo.com/book/210087022000101 |
| **博客來** | https://www.books.com.tw/products/0010759449 |

### 寫作技法分析
- **資工背景作者**：技術元素不是裝飾，而是故事引擎
- **網路霸凌、VC 投資、數位鑑識**全部有專業深度
- **駭客技能驅動推理**：程式設計和駭客技術是破案工具

### 可借鑑的技法
> **這是最接近「有資工背景的作者寫推理小說」的中文範本**。
> 證明：技術細節可以成為劇情的核心驅動力，而不只是點綴。
