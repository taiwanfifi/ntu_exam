# 第一篇「校準」序篇（ch1-4）—— 初稿級精修

> **精修目的**：將大綱級敘事提升為初稿級密度——加入具體感官、對白節奏、內心獨白、鏡頭感。
> **原則**：遵循《角色語言風格指南》十條戒律，機率潛移默化，貝葉的力量在「安靜的堅定」。

---

## 第 1 章：透鏡日常

### 開場鏡頭（600 字以內）

```
【環境】
早上六點五十二分。中山區。
一棟三十年老公寓的四樓。窗外是巷子裡永遠停不滿的機車。

鬧鐘沒有響。
因為鬧鐘是小拉。

「貝姊。貝姊。貝——」
「不要叫我貝姊。」
陳貝葉把臉從枕頭裡拔出來。頭髮亂成一個 Gamma 分布——
右邊厚、左邊薄，尾巴拖到肩膀。

「好的貝姊。今日天氣預報：降雨機率 62%。分布 Beta(6.2, 3.8)。
 鑒於你昨天、前天、大前天都沒有帶傘——」
「Beta(6.2, 3.8) 的期望值是 0.62，標準差大概……」
「0.098。意味著實際降雨機率可能從 42% 到 82%——」
「所以不一定會下。」
「也不一定不下。」
「那帶不帶傘。」
「根據你過去三十天的行為數據——」
「好了好了。」

貝葉拿起傘。出門。
小拉在透鏡裡閃了一下：「！今天帶了！我要修改先驗——
你主動帶傘的條件機率從 0.12 上調到 0.15。」

「才上升 0.03 是不是看不起我。」

「是科學。」小拉說。聲音裡有委屈。

出門前回頭看了一眼陽台。
薄荷還活著。勉強。
葉片邊緣有點焦，但中間冒了一個新芽。
她拿了杯子澆了一口水。

「你對植物的關注度比對自己的身體高 47%。」小拉碎念。
「因為植物不會碎念。」

她關上門。
巷子裡有人在騎摩托車暖車。排氣管的聲音在六點五十五分的台北特別清楚。
像一個不太準時的 Poisson 過程。

不對——摩托車不是 Poisson。
Poisson 是「獨立」事件。但這台摩托車每天都在。
「固定的雜訊不是隨機。」
她搖搖頭。職業病。
```

### 通勤場景——透鏡的日常

```
【節奏】輕快。用小拉的碎念推動。
讓讀者像在看日常漫畫——舒服、有趣、愛上這個世界。

捷運中山站。早上七點半。人群像一個高峰期的排隊模型。

透鏡自動開機。
貝葉不喜歡通勤時開透鏡——太多數字會讓早晨變得很擠。
但公司規定校準工程師必須全時段開啟。

她看著月台。

  ┌─ 機率透鏡 v4.2 ───────────────────────┐
  │ 捷運板南線                              │
  │ 下一班：2 min 14 sec                   │
  │ 到達分布：Normal(μ=134s, σ=12s)        │
  │ 本班車廂有座位 P(seat) = 0.23          │
  │ 建議：等下一班 ██░░░░░░░░ 23%          │
  └─────────────────────────────────────────┘

「小拉，23% 有座位。」
「建議等下一班。下一班有座位的機率是 61%。」
「但我會遲到。」
「遲到的機率是……12%。嗯，其實不太高——」
「但我不想遲到。」
「那就擠上去。」

她擠上去了。沒有座位。
站在車廂裡，透鏡的數字浮在每個乘客的方向——
「此人手機被偷的機率：0.002」
「此人下一站下車的機率：0.71」
「此人今天心情好的機率：0.33」

這些數字有用嗎？
也許有用。也許沒有。
但所有人都已經習慣了。
像習慣呼吸一樣習慣了被數字包圍。

貝葉閉上眼。透鏡的數字在眼皮後面變成微弱的餘像。
她想起高思遠教授的話：
「機率不是命運。機率只是地圖。走哪條路，是你自己的事。」

列車到站了。她睜開眼。沒有座位。但也沒有遲到。
0.23 的機率不代表不可能。
也不代表值得。
只是——一個數字。

【寫作提示】
這一段的目的不是教讀者「什麼是常態分布」。
是讓讀者感覺到：生活在一個被數字填滿的世界，是什麼感覺。
一點點方便。一點點煩躁。一點點——說不清楚的不安。
```

### Oracular 辦公室——日常與伏筆

```
【環境描寫】
南港軟體園區。三棟弧形玻璃建築。
大廳地板是白色的，乾淨到能看到倒影。
連接三棟樓的空中走廊上有一面巨大的螢幕，
24 小時滾動顯示全台灣的即時機率熱力圖。
現在：北部降雨機率偏高（一片淺藍），南部晴天（暖黃色）。

「歡迎回來，陳貝葉。今日校準指數：88.7。」門禁系統的語音。
貝葉走進校準組的辦公室。
十二張桌子，排成三排。日光燈很亮。空調永遠是 23 度。

她的桌子在靠窗的位置。
桌上三樣東西：半杯昨天的珍奶（已經分層了，珍珠沉底）、
統計計算機（她不信任透鏡的內建計算——「自己算一遍比較安心」）、
一隻橘色招財貓。
招財貓是高思遠教授給她的。上面寫著一行字：
「期望值為正就值得嘗試。」
字跡很舊了。但她從不擦。

旁邊的桌子是林岳的。
林岳 31 歲。瘦。戴眼鏡。永遠在跑數據。
桌上有一棵假的仙人掌和一張全家福照片——妻子和兩歲的女兒。
他是校準組除了貝葉之外，唯一會「手動驗算」的人。

「早。」林岳推了推眼鏡。
「早。」貝葉坐下。打開電腦。
「你昨天加班到幾點？」
「十一點。妳呢？」
「十二點。」
「……妳贏了。」
「這不是比賽。」
「聽起來就是。」

林岳笑了。轉回去看他的數據。
然後突然——

「欸，貝葉。你覺得 Poisson(λ=2.1) 的殘差分布，偏態係數
 0.03 正常嗎？」

貝葉停了一下。
「0.03？λ=2.1 的 Poisson 偏態應該在 0.4 到 0.7 之間。
 0.03 太低了。」
「嗯。我也覺得。可能是我哪裡跑錯了。」
「你在看哪個區域？」
「中正區。但沒什麼——可能是我的模型設定有問題。」

他轉回去了。沒有再提。
貝葉也轉回去了。
但那個數字——0.03——留在她腦子裡。

像一顆種子。
還沒有發芽。但已經落土了。

【寫作提示】
林岳的這段對話在第 19 章回看時要讓讀者心碎——
「他比我更早發現。但他沒有來得及說完。」
所以這裡要寫得輕描淡寫。像日常。
越日常，之後的痛越痛。
```

### 午休＆下午校準——機率公理的自然呈現

```
【午休——小拉的營養分析】
便當。排骨飯。公司地下一樓的自助餐。
小拉自動啟動營養分析。

「這個排骨飯的熱量分布——」
「小拉。」
「什麼？」
「我只是想吃飯。」
「……好。但我想讓你知道，在你吃完之後有 71% 的機率會想睡覺。」
「所以？」
「所以你應該買杯咖啡。」
「我不想花錢。」
「公司的免費咖啡還有。」
「那個不叫咖啡。那叫深色的水。」
「……以化學成分來看，你說得對。」

她吃飯的時候，同事們在討論新的校準任務。
有人說信義區的數據又要重跑。有人在抱怨系統更新太頻繁。
周副理走過來：「大家不要忘了下午三點的週會。」
然後壓低聲音對貝葉說：
「下午有一個到府校準的案子。老先生。你去一趟。」
「什麼情況？」
「透鏡數值全面偏高。可能是機械故障，也可能是……」他停了一下。
「認知基線偏移。」
他的語氣微妙地軟了一下。
「帶點心。他可能需要有人陪他說說話。」

周副理走了。

小拉：「周副理剛才的語調偏差了 1.3 個標準差——
       比他正常的語氣更溫柔。」
貝葉：「他只是……善良。」
小拉：「善良在我的模型裡不是一個參數。但我正在考慮加入。」

【下午——校準工作的日常展示】
三點。週會結束。貝葉收拾東西去做到府校準。

出發前，她打開今天的校準面板。
三個例行區域。全部正常。

在跑每一個區域的時候，她的腦子裡自動跑著校準的邏輯——

（← 以下用「內心獨白」方式呈現機率公理，不是教科書）

  校準的第一步永遠是確認樣本空間。
  「所有可能發生的事」。
  聽起來簡單。但這是一切的起點。
  如果你連「什麼事有可能發生」都搞錯了——
  後面算的所有機率都是假的。

  第二步：確認三條公理。
  所有事件的機率 ≥ 0。不會有負的機率。
  Ω 的機率 = 1。所有可能加起來是全部。
  互斥事件可以相加。A 和 B 不能同時發生，那 P(A∪B) = P(A) + P(B)。

  這三條規則——就是整個機率世界的地基。
  透鏡上的每一個數字，都站在這三條規則上面。
  如果地基出了問題——

她停了一下。
為什麼會想到「地基出問題」？
沒有原因。只是……一個念頭。

「修得好。」她蓋了章。三個區域全部正常。
收拾東西。出發去老先生家。

【寫作提示】
「如果地基出了問題」= 全書的伏筆。
這裡要寫得像一個普通的念頭——來了又走了。
讀者不會注意到。但回頭看的時候會起雞皮疙瘩。
```

### 章末——孤獨的量化

```
【收尾鏡頭】
晚上九點。回到家。
老先生的校準明天才去。今天在公司做完了例行工作。

開門。四樓。走廊的燈壞了一盞，閃爍。
她進門。脫鞋。把傘放在門邊（今天真的下了）。

打開冰箱。裡面有：半盒牛奶（小拉說明天過期）、
兩顆蘋果（小拉說後天過期）、一盒微波義大利麵（小拉說上週過期）。

「微波麵過期了。」
「知道了。」
「你不打算丟嗎？」
「明天。」
「你三天前也這樣說。」
「……嗯。」

她坐在客廳的地板上。背靠著沙發。
打開手機。沒有未讀訊息。
打開透鏡的社交面板。
友好度排行：馬可瑜 92%、高思遠 89%、林岳 71%。
其他人都在 50% 以下。

透鏡彈出一條訊息：
「今晚感到孤獨的機率：78%。」

她盯著那行字。
78%。

一個很精確的數字。
但精確不代表有用。
因為——就算她知道自己有 78% 的機率感到孤獨，
她也不知道該怎麼辦。

「機率告訴你可能會發生什麼。不告訴你該怎麼辦。」
高思遠教授的話。
十年前說的。十年後還是對的。

她把透鏡切到夜間模式。數字消失了。
房間裡只剩下冰箱的嗡嗡聲和隔壁小孩練鋼琴的聲音。
鋼琴彈得不好。但很努力。
C-D-E 停頓 C-D-E 停頓 C-D-E-F——
每次到 F 都會卡住。
但每次都會重來。

貝葉閉上眼。
不知道什麼時候睡著的。
地板很硬。但也沒有那麼硬。

陽台上的薄荷在風裡搖了一下。
沒有人替它算過存活機率。
但它明天還是會長。
```

## 第 2 章：校準悲傷的人

### 到府校準——老先生的家

```
【開場】
下午兩點。大安區。一棟老公寓。

貝葉站在四樓門口。門鈴是那種很舊的——按下去會「叮咚」兩聲。
不是電子音。是物理的震動。

門開了。
鄧伯伯。七十五歲。退休的高中數學老師。
個子不高。頭髮全白了。穿一件格子襯衫，扣到最上面一顆。

「你好。Oracular 的校準工程師——」
「啊，你就是那個小陳啊。進來進來。鞋子脫這裡。」
他指了指門口的鞋架。鞋架上有兩雙拖鞋。
一雙是鄧伯伯的。藍色。鬆了。
另一雙是粉紅色的。小一號。整齊地放著。
上面有一層薄薄的灰。

貝葉看了一眼。沒有問。

「坐坐坐。我泡茶。」鄧伯伯走進廚房。動作很慢但很穩。
客廳不大。沙發是舊的，但套了乾淨的布。
茶几上有一疊報紙（實體報紙——現在很少人看了）、
一副老花眼鏡、一個馬克杯——上面印著「世界最棒的老師」。

電視櫃上有一張照片。
鄧伯伯年輕的時候。旁邊是一個笑得很燦爛的女人。
她穿著旗袍。頭髮盤起來。手裡捧著一束花。

照片旁邊有一個空的花瓶。
以前插過花。現在沒有了。

小拉在透鏡裡小聲說：「環境分析完成。
此住宅近期有人員減少的跡象——鞋架多餘拖鞋、未使用的花瓶——」
「小拉。」
「什麼？」
「安靜。」
「……好的。」
```

### 校準過程——悲傷改變了先驗

```
【技術場景——融入情感】
鄧伯伯坐下來。貝葉打開校準設備。
一台手持式校準儀——像一個小平板。
放在桌上，連接鄧伯伯的透鏡。

  ┌─ 校準面板 ─────────────────────────────────┐
  │ 用戶：鄧XX（75 歲，男性）                   │
  │ 校準異常：全域偏高 +15%                      │
  │                                               │
  │ 降雨機率：顯示 87%（實際 62%）               │
  │ 路口事故：顯示 23%（實際 8%）                │
  │ 食物中毒：顯示 31%（實際 11%）               │
  │ 詐騙電話：顯示 44%（實際 15%）               │
  │                                               │
  │ 所有數值均偏高。非機械故障。                 │
  │ 判定：認知基線偏移（Cognitive Baseline Shift） │
  └───────────────────────────────────────────────┘

「鄧伯伯，您的透鏡數值全面偏高了大概 15%。
 不是機器壞了。是……」她猶豫了一下。
「是我看什麼都覺得比較危險了，對不對？」鄧伯伯替她說完。

她點頭。
「透鏡的算法會根據使用者的認知狀態自動調整。
 您最近的情緒基線……可能有比較大的變化。」

鄧伯伯看了一眼照片。
「上個月。美玲走了。」
停頓。
「四十七年了。」

他沒有哭。他的眼睛很平靜。
但那種平靜不是沒有感覺——是感覺太深了，深到表面看不到。
像一個水很深的湖。風吹不動。

小拉在貝葉腦中低語（只有她聽到）：
「使用者情緒數據顯示深度悲傷。持續時間：34 天。
 這個……這是什麼感覺？我的模型裡沒有——」
「小拉。等一下再說。」

貝葉放下校準儀。
她沒有馬上開始調整。
她倒了一杯茶。跟鄧伯伯坐在一起。

「鄧伯伯……美玲阿姨是怎樣的人？」

她不需要問這個。校準不需要這個資訊。
但她問了。

鄧伯伯笑了。
是那種「想到某個人就會自動笑」的笑。
「她啊……她煮的紅豆湯全台北最好喝。
 我退休之後她說你不要在家裡晃來晃去的，
 去學一個什麼才藝。我就去學書法了。
 結果她每次看我寫都說：『你寫的字跟雞爪一樣。』
 然後自己拿去裱框掛在客廳。」

他指了指牆上。
果然有一幅書法。「靜」。
字確實不太好看。但裱得很用心。

「她走的那天……我在廚房。
 聽到客廳很安靜。平常她都會看電視，很吵的。
 我走出去。她坐在沙發上。像睡著了。」

他的聲音很穩。但手指在微微發抖。
貝葉看到了。

「透鏡那天開始就不對了。出門看什麼都覺得危險。
 過馬路的時候，數字跳很高。我不敢過。
 後來連下樓買菜都……」

他沒有說完。不需要說完。

貝葉開始校準。
但不是冷冰冰地調數字。
她一邊調，一邊跟鄧伯伯說話。

「鄧伯伯，您知道嗎——透鏡的計算有一個東西叫做『先驗』。
 就是……在你看到任何數據之前，你已經相信的東西。
 您太太走了之後，您的先驗變了。
 您開始相信世界比較危險。
 所以透鏡算出來的所有數字，都跟著偏高了。」

他想了想。
「所以……不是世界變危險了。是我的心變了。」
「是的。」
「那……她在的時候，我的心讓世界變安全了嗎？」

貝葉愣了一下。
她沒有想過這個。

「也許是。」她說。聲音比預期的小。

鄧伯伯笑了。「那她比你的透鏡厲害。」
```

### 校準完成——雨停的機率 ★★


## 第 3 章：白奶奶的先驗

### 龍山寺旁——蔥油餅攤

```
【開場——氣味先行】
週六下午。萬華區。

還沒看到攤位，就先聞到了。
蔥油餅。
不是那種精品餐廳把蔥油餅做成「法式蔥油酥」的味道。
是四十年鐵板上累積的、洗不掉的、混合著蔥花焦香和老油的味道。

滋——
```

### 關於透鏡的對話——全書核心哲學的種子

```
「奶奶，你真的不想裝透鏡嗎？」
貝葉邊吃邊問。她每次來都問。答案每次都一樣。

白奶奶翻餅。鐵板滋滋響。
「不要。」
「免費的欸。政府有補助。」
「免費的也不要。」
「為什麼？」

白奶奶把餅翻了個面。動作流暢。不看鐵板。
手知道在哪裡翻。手知道什麼時候翻。

「我做蔥油餅四十年。每一張都不一樣。
 麵團的溼度——看天氣。蔥的量——看心情。
 火候——看鐵板的脾氣。今天它比較燙。」

她抬頭看貝葉。

「你那個透鏡仔，可以告訴我下一張餅好不好吃嗎？」

貝葉想了想。
「理論上……可以建一個模型。根據麵團含水量、
 蔥花分布密度、鐵板溫度——」
「好了好了。」白奶奶揮揮手。像在趕蒼蠅。
「你聽我說。」

她把做好的餅放到貝葉面前。

「做餅這件事啊——不知道下一張會不會好吃，
 這才是有趣的地方。
 如果每張都一樣好吃，那我做了四十年是在做什麼？
 就是因為每一張都不一樣，我才每天都想做。」

她用鏟子拍了拍鐵板。鏘。

「你們啊，什麼都要算。算來算去的。
 算到最後——連吃一張餅都要先看數字。
 那吃餅還有什麼意思？」

貝葉咬了一口餅。沒有看數字。
嗯。好吃。

「而且——」白奶奶壓低聲音。
「我的膝蓋比你的電腦準。它說明天會下雨。

「三次是多少百分比？」貝葉忍不住問。
白奶奶看了她一眼。
「你看，又在算了。」

白奶奶不是在「否定」機率。
她是在說：有些東西不需要被量化也有價值。

在第 58 章貝葉跟柏朗辯論時，
白奶奶的這些話會從貝葉的記憶裡浮上來。
「不知道，才有趣。」
= 「不確定性不是敵人。」
```

## 第 4 章：十一秒

### 高思遠的週末聚餐

```
【開場——固定的風景】
週日。台大附近。一間叫「老張」的小餐館。
在羅斯福路三段的巷子裡。從台大正門走路七分鐘。
老闆姓張。但大家都叫他「老張」。不是因為他老。是因為他從不笑。
但他做的滷肉飯是附近最好的。
高思遠教授說：「不笑的廚師通常做菜比較好。
因為他把所有的善意都放進食物裡了。」

每個月第一個週日。固定。二十年了。
從貝葉十五歲開始。那時候她還在讀高中。
高思遠每次都點一樣的：滷肉飯、荷包蛋、一碗豆腐湯。
「你爸以前也點這個。」他每次都說。
「我知道。老高你每次都說。」她每次都回。
這個對話本身已經變成一種儀式了。

高思遠今天穿那件淺藍色的襯衫。扣到倒數第二顆。
褐色的老舊領帶。口袋裡有粉筆（雖然他已經退休兩年了）。
他的透鏡——第一代原型機——偶爾會在左眼邊緣閃一下。
那個bug。從 2037 年到現在。他從不修。
「這是歷史的痕跡。」

「吃飯吧。」他說。
然後咳嗽了一聲。
比上次多了一聲。

【吃飯時的對話——日常裡的暗流】
「最近的校準工作怎麼樣？有沒有什麼……不尋常的？」

他每次都問。
但今天的語氣——不一樣。
多了一個停頓。在「什麼」和「不尋常」之間。
貝葉的透鏡分析不出這個停頓的含義。
但她的直覺可以。

「還好。前幾天校準了一個老先生的透鏡。偏高 15%。認知基線偏移。」
「因為什麼？」
「太太去世了。」

高思遠的筷子停了一下。
不是停在空中——是停在碗的邊緣。很穩。但停了。

「……悲傷確實會改變先驗。」他說。

停頓。

「月華走的那年。我的透鏡也偏了。」

月華。李月華。高思遠的太太。
五年前因為透鏡焦慮症引發的恐慌症發作，
在過馬路時未注意來車。

「但我沒有去校準。」
他放下筷子。看著碗裡的滷肉飯。
「我想留著那個偏移。
 因為那是我唯一能感覺到她還在的方式。」

★★ 催淚點。
不校準悲傷 = 不忘記。
偏移 = 思念的量化。
「先驗」在這裡不是數學術語。是一個老人對亡妻的眷戀。

貝葉的喉嚨緊了一下。
她吃了一口飯。嚼了很久。

「老高。」
「嗯。」
「你現在的偏移……還在嗎？」
他笑了。皺紋在眼角展開。像一條河流經了很多地方。

「在。但小了很多。
 時間會讓先驗慢慢更新。
 新的數據會慢慢稀釋舊的信念。
 這就是……貝氏更新吧。」

他的眼鏡反射了窗外的光。

「但有些先驗——不管多少新數據，都不會完全消失。
 它只會越來越小。趨近零。但永遠不到零。
 那就是……你永遠不會完全忘記一個人的數學證明。」

小拉在透鏡裡閃了一下。
她想說什麼。但這一次，她選擇不說。
有些東西不需要被量化。
```

### 回憶——十一秒

```
【場景轉換——用光線切換時間】
走出餐館。高思遠走在前面。步伐比去年慢了一點。
陽光從樟樹的葉子之間漏下來。

光斑在路面上搖晃。
像壞掉的時間戳。

貝葉走在他身後。
看著他的背影。
突然想起了另一個背影。

——十六年前。她十二歲。

  放學回家。客廳裡坐滿了人。
  母親在沙發上。臉上有淚痕，但不是在哭。是哭完了。
  那種比哭更安靜的狀態。

  高思遠站在角落。臉色蒼白。
  他看到貝葉進門的時候，嘴唇動了一下。
  但沒有發出聲音。

  「爸爸呢？」

  沒有人回答。
  那就是答案。

  爸爸出門買便當。
  在巷口——離家 200 公尺的巷口——
  被一輛闖紅燈的卡車撞了。
  當場。

  後來她才知道：
  爸爸本來不需要自己去買。
  是外送系統剛好當機了。
  「那就我自己去吧。」他說。
  穿了鞋。出門。
  門在他身後關上的聲音——她記了十六年。

——三年後。她十五歲。已經學會了基本的統計。

  高思遠沒有教她統計。是她自己學的。
  動機不是好奇。是——
  「我要算出來。爸爸的死，有沒有可能避免。」

  她蒐集了所有數據：
  ・出門時間：18:32:14
  ・卡車到達巷口時間：18:32:41（肇事紀錄）
  ・紅綠燈週期：60 秒（綠燈 25 秒 / 紅燈 35 秒）
  ・巷口車流量：Poisson(λ=0.8/min)
  ・爸爸步行速度：4.2 km/h（從家門口到巷口 200m ≈ 171 秒）

  她算了又算。用了兩個月。三個筆記本。
  最後得出一個結論。

  「如果爸爸提早 11 秒出門——
   他到達巷口的時間會落在紅燈期間。
   他會停下來等。
   卡車通過。
   他安全。
   存活機率從 2% 上升到 97%。」

  11 秒。

  多穿一隻鞋的時間。
  多看一眼手機的時間。
  多說一句「等一下我忘了帶錢包」的時間。

  任何一個微小的改變——
  都能製造出 11 秒。
  但那天。
  所有微小的事件。
  以最糟糕的方式排列了。

「小拉。」貝葉突然說。
「什麼？」
「如果……如果那時候有透鏡。
 爸爸出門前看到『這個時間出門在巷口遇到事故的條件機率偏高——
 建議延遲 11 秒』……」
「他會等嗎？」小拉問。
「……不知道。但至少他有機會選擇。」

這是她加入 Oracular 的原因。
不是薪水。不是名聲。
是因為她相信——
如果每個人都能看到機率，很多悲劇可以避免。

這個信念很美。
也很危險。
因為如果有一天她發現——
機率本身是假的呢？

那她的信念就不只是錯了。
是被背叛了。
```

### 章末——質疑先驗

```
【高思遠的最後一句話】
走回台大正門。椰林大道。
椰子樹在風裡搖。像倒過來的鐘擺。

高思遠站在正門口。

「貝葉。」
「嗯。」

「質疑先驗。永遠質疑先驗。」

停頓。他看著她。
眼神很溫柔。但底下有一層什麼——像湖底的暗流。

「包括……你自己的。」

他轉身走了。走進椰林大道。
背影在斑駁的樹影裡漸漸縮小。
比去年的背影更瘦了一點。

貝葉站在原地。

質疑自己的先驗？
她的先驗是什麼？

「透鏡是好的。Oracular 是好的。」

 根據我的情緒模型……」

害怕什麼？

但她把那句話記住了。
像一顆種子。
掉進了已經有另一顆種子的土裡。

透鏡自動顯示：「此花店一年內倒閉的機率：34%。」

然後想起鄧伯伯家裡那個空的花瓶。

她走進去。買了一束百合。
不是為了自己。
明天上班的時候，順路送過去。

「因為花瓶空了。」
「這不是一個邏輯充分的理由。」
「我知道。」
```
