# 第三十九章：蒙地卡羅

---

台北地檢署在博愛路上。

一棟灰色的建築物，外觀沒有任何多餘的裝飾——直線、直角、水泥、玻璃。它的設計語言和它的功能完全一致：冷硬、精確、不容情面。大門口的台階在十一月的晨光裡泛著一層淺灰色的光澤，被無數雙皮鞋磨出了微微的凹弧。

沈維誠在早上八點四十五分走上了那些台階。

他穿了一件若嵐從來沒見他穿過的衣服——白色的襯衫，塞進深灰色的長褲裡。不是新的。襯衫的領口有一道極細的摺痕，顯示它在衣櫃裡被折了很久。褲腳的熨痕也不太銳利——不是沒有燙過，是燙的人手藝普通。大概是曉夢。曉夢昨天晚上到安全屋來送這套衣服的時候，若嵐看到了她手指上的微小燙傷——食指和中指之間，一個新的、半圓形的、皮膚輕微泛紅的痕跡。熨斗留下的。

維誠沒有帶律師。

蘇瑾行提議過——他說他認識一個做刑事辯護的律師，可以陪維誠去。但維誠拒絕了。他說：「我不是去當被告的。我是去提供證據的。吹哨者不需要帶律師。吹哨者需要帶的是——數據。」

他帶了數據。

一個加密的外接硬碟。黑色的。三吋的。大約一百二十克。裡面裝著拉普拉斯之眼的完整備份——1.2TB的數據，包含兩百多個受害者的干預日誌、行為操控的完整紀錄、陳默親手標註的「成功案例報告」、以及若嵐整理的統計分析初稿。

一百二十克。

比一包泡麵輕。但比任何武器都重。

---

## I

方正怡檢察官的辦公室在三樓。

走廊很長。日光燈管在天花板上發出恆定的、不帶感情的白光，把走廊的每一個角落都照得無處藏匿。地板是淺灰色的磨石子，每一步腳步都會發出清脆的、規律的咚聲，在走廊的牆壁之間產生短暫的回音。

蘇瑾行走在維誠前面半步。他昨天晚上通過三層關係——前同事的大學同學的法律事務所合夥人——聯繫上了方正怡的助理。助理在電話裡只說了一句話：「方檢願意見。明天早上九點。帶齊所有資料。」

九點。維誠提前了十五分鐘到。

辦公室的門是關的。門上有一塊名牌：「方正怡 檢察官」。字體是標準的楷書，金色的字刻在深色的木板上。名牌旁邊貼了一張小小的便條——不是公文，是手寫的，字跡很小：「開會中。請稍候。」

蘇瑾行和維誠在走廊的長椅上坐下來。

長椅是木頭的。表面的漆磨掉了一半，露出底下淺色的木紋。椅背的角度不太舒服——大約九十五度，不夠讓人放鬆地靠著，但又不至於直挺挺的。一個讓等待者保持清醒但不至於焦慮的角度。也許是設計者有意為之。也許不是。

九點零三分。門開了。

一個男人從辦公室裡走出來。四十多歲，穿西裝，表情陰沉。他經過蘇瑾行和維誠身邊的時候，目光沒有停留——他在想自己的事。他的皮鞋在磨石子地板上發出急促的咚咚聲，在走廊盡頭的轉角消失了。

然後方正怡出現在門口。

---

## II

若嵐看過很多種面孔。

作為臨床心理師，她的職業就是閱讀面孔——微表情、肌肉群的協同模式、情緒在五官上的分布方式。她建立了一個龐大的面孔數據庫，分類方式不是按照美醜，而是按照「這張臉的主人如何處理壓力」。有些臉是迴避型的：嘴角在壓力下會微微收緊，像拉上拉鍊。有些臉是對抗型的：下巴會往前推，頸部肌肉會繃緊。有些臉是解離型的：所有的表情在壓力下歸零，變得像一面空白的牆。

但方正怡的臉不屬於任何一個類別。

後來蘇瑾行向她描述了方正怡，若嵐才在腦海中補全了這張臉。短髮。非常短。耳朵上方一公分的長度，乾淨利落，不需要任何整理就能維持的那種。沒有染色——是自然的黑，帶著四十三歲的人會有的、零星的銀灰色髮絲。臉型偏方。不是那種溫潤的鵝蛋臉——是稜角分明的，顴骨略高，下巴線條清楚得像用尺畫的。眼睛不大，但非常亮。不是那種溫暖的亮——是X光機的亮。你覺得她在看你的時候，她看的不是你的臉，而是你臉底下的東西。

她穿深色的套裝。沒有首飾。手腕上一只黑色的電子錶——不是時尚配件，是計時工具。

「進來。」

兩個字。沒有寒暄。沒有「你好」或「請坐」。就是「進來」。語調像一顆子彈：短、快、直線。

蘇瑾行後來告訴若嵐：方正怡說話一直都是這種風格。在法庭上，她的交叉詢問方式被律師圈稱為「方氏連射」——一個問題接一個問題，每一個問題都短於十五個字，每兩個問題之間的間隔不超過三秒。她不給被詢問者喘息的空間。她像一台壓力測試機，用最少的力道、最高的頻率，找到材料斷裂的那一個點。

維誠走進了辦公室。蘇瑾行跟在後面。門在他們身後關上了。

---

## III

辦公室比若嵐想像的小。

後來蘇瑾行描述了細節：一張L型的辦公桌，上面疊了至少三十公分高的卷宗。一台電腦螢幕。一盞枱燈。牆上掛著一幅字——不是書法，是用打字機打的一行字，裝在簡單的黑色相框裡：

「沒有證據的正義不是正義。」

蘇瑾行告訴過若嵐這句話。現在他親眼看到了。字體是Courier New，十四號。紙張是普通的A4影印紙，白色的，邊角微微泛黃。這幅「字」可能已經掛了很多年。

方正怡坐在辦公桌後面。她沒有坐辦公椅——她坐的是一張和走廊上一樣的木頭椅子。沒有扶手。沒有靠墊。維誠和蘇瑾行坐在對面的訪客椅上。

「說。」

一個字。

維誠深吸了一口氣。然後他開始說。

從2021年說起。他加入Resonance Analytics的經過。他參與的系統設計。他第一次發現陳默的真正目的——不是數據分析，是行為操控。他看到的受害者案例。若晴的角色。Subject Gamma的死亡。他做出的決定——在代碼裡埋後門、準備證據、計畫脫離。他的消失。三年的躲藏。花蓮的老房子。

方正怡在整個過程中沒有打斷他一次。

這讓蘇瑾行覺得不尋常——根據他的經驗，方正怡的風格是頻繁打斷、頻繁追問。她不打斷，意味著她在聽。真正的在聽。不是禮貌性的，而是結構性的——她在用維誠的陳述在腦海裡建構一個框架，每一個細節都是框架上的一根鋼筋。她不打斷，因為她不想在框架完成之前移動任何一根鋼筋。

維誠說了大約四十分鐘。

然後方正怡開口了。

不是提問。是一句話。

「如果你說的是真的，這會是台灣史上最大的科技犯罪案。」

她的聲音在「台灣史上最大」這幾個字上沒有任何抑揚頓挫。像在念一份氣象報告。明天台北會下雨。這會是台灣史上最大的科技犯罪案。同一種語調。不是因為她不在乎——是因為她在乎到了一個程度，讓情緒表達變得多餘。

「但如果你在騙我——」

她的眼睛盯著維誠。那個眼神——蘇瑾行後來對若嵐描述的時候用了一個很精確的比喻——像一台工業級的光譜分析儀。它不是在看你。它是在分解你。把你臉上的每一個像素拆開來，逐一分析其中的頻率分布，判斷哪些是真實的光，哪些是反射、折射、散射。

「——你會坐牢。」

維誠回視她。

他的眼睛沒有閃躲。在三年的躲藏中，他學會了很多種逃避——逃避追蹤、逃避數位足跡、逃避人群。但此刻，面對這雙X光般的眼睛，他沒有逃避。他的虹膜在方正怡的視線壓力下微微收縮了——那是交感神經的自動反應，無法偽裝——但他的眼球沒有移動。穩的。像一顆被定位系統鎖定的衛星。

「我等這一天等了三年。」維誠說。

方正怡看了他三秒。

三秒。在她的評估體系裡，三秒足夠完成一次可信度初判。她不看面部表情——面部表情可以被訓練。她看的是生理指標的一致性：瞳孔大小是否與語調的壓力水平匹配、頸部肌肉的緊張度是否與手部姿勢協調、呼吸頻率是否在講述高壓事件時出現了符合預期的波動。

「硬碟。」她伸出手。

維誠把那個黑色的外接硬碟放在她的手心裡。一百二十克。

方正怡接過硬碟。她沒有立刻打開——她把硬碟放在桌面上，按了桌上電話的一個按鍵。

「小林，帶數位取證設備上來。」

她看向蘇瑾行。

「你是記者。」不是問句。

「獨立調查記者。」蘇瑾行說。

「嗯。」方正怡的「嗯」不帶任何評價——既不是肯定也不是質疑，只是一個「我已經把這個信息歸檔了」的確認音。「記者的部分我們之後談。現在我要先驗證數據。如果數據是真的——」

她的手指輕輕叩了一下硬碟的外殼。

「——我會親自辦。」

---

## IV

若嵐不在檢察署。

她在安全屋裡。坐在維誠的電腦前面。三台螢幕亮著。

她在做最後的統計分析。

不是為了自己——是為了方正怡。維誠帶去的硬碟裡有原始數據，但原始數據本身不會說話。1.2TB的資料對一個不具備統計學背景的人來說，就是一堆數字的汪洋。你可以在裡面游泳游到溺死，也不會自動看見模式。

你需要有人幫你看見模式。

若嵐要寫一份統計報告。一份連不懂統計的法官都能看懂的報告。一份能回答一個關鍵問題的報告：

**「如果這兩百多個受害者的崩潰都是自然發生的——如果沒有人在操控——我們觀察到如此一致的模式的機率是多少？」**

這是虛無假設。統計推論的起點。

H₀：這些受害者的行為崩潰是獨立的、隨機的、源於他們各自的人生壓力和心理脆弱性。Echo Protocol不存在。環境操控不存在。一切都是巧合。

H₁：這些行為崩潰不是獨立的。存在一個共同的外部操控因素。Echo Protocol是真的。

若嵐的工作就是計算：在H₀為真的條件下，觀察到現有數據的機率是多少。如果那個機率足夠小——小到任何理性的人都無法接受「這只是巧合」的解釋——那麼H₀就可以被拒絕。

問題是：怎麼算？

干預日誌裡的數據結構極其複雜。每一個受害者的記錄不是一個單一的數字——是一條時間序列。幾個月甚至幾年的行為追蹤。環境變數的變化。情緒狀態的估計值。社交互動的頻率。睡眠模式。消費行為。GPS軌跡。每一個變數都在時間軸上波動，像一條獨立的河流。

兩百多條河流。

要證明這兩百多條河流不是各自獨立地流向崩潰——而是被某個人同時改變了水流方向——你不能只看單一的河流。你要看所有的河流。你要看它們之間的關聯性。你要看模式。

但「模式」是一個模糊的概念。法庭不接受模糊。法庭接受數字。

所以若嵐選擇了蒙地卡羅模擬。

---

## V

蒙地卡羅。

這個名字來自摩納哥的蒙地卡羅賭場。1940年代，曼哈頓計畫的物理學家斯坦尼斯瓦夫·烏拉姆（Stanislaw Ulam）在病床上玩單人紙牌遊戲時，想計算一手牌成功的機率。他發現用解析的方法太複雜了——排列組合的數量太大。但如果他玩上一百次、一千次——用大量的隨機試驗來估計成功率——答案就會自然浮現。

他和約翰·馮·紐曼（John von Neumann）把這個想法應用到了核武器的中子擴散計算中。他們需要計算一個中子在複雜的幾何結構中穿透多少層物質後會引發連鎖反應——解析解不可行，但如果你模擬一百萬個中子的隨機路徑，統計有多少個穿透了，你就得到了一個足夠精確的估計。

他們把這個方法命名為「蒙地卡羅」。因為烏拉姆的叔叔常去蒙地卡羅賭場。因為賭場是隨機性的聖殿——骰子、輪盤、紙牌——每一次投擲都是一次隨機試驗。

若嵐要做的事情和烏拉姆在本質上一樣：她有一個太複雜而無法用解析方法直接計算的問題，所以她要用大量的隨機模擬來逼近答案。

她在Python的編輯器裡開始寫代碼。

邏輯如下：

第一步：從干預日誌中提取每一個受害者的關鍵指標——「崩潰前的行為模式一致性指數」。這是她自己設計的一個指標，測量的是：在崩潰發生前的三十天內，受害者的行為序列與Echo的干預指令序列之間的互相關係數（cross-correlation）。如果Echo沒有在操控——如果行為崩潰是自然發生的——那麼受害者的行為和Echo的指令之間應該沒有顯著的相關性。互相關係數應該在零附近。

但實際觀測到的數據顯示：兩百多個受害者的互相關係數的平均值是0.847。

0.847。幾乎完美的正相關。受害者的行為波動和Echo的指令波動之間，有84.7%的一致性。

這個數字本身已經很驚人了。但若嵐不滿足於「驚人」。她要量化「驚人的程度」。她要回答：**在隨機條件下，觀察到這麼高的平均互相關係數的機率是多少？**

第二步：蒙地卡羅模擬。

她設定虛無假設的世界：在這個世界裡，每一個受害者的行為序列都是獨立的隨機過程——和Echo的指令無關。她用自助法（bootstrap）從原始數據中隨機重新排列每一個受害者的時間序列，打斷它和Echo指令之間的時間對應關係。然後重新計算互相關係數的平均值。

這樣做一次，就得到了一個「在虛無假設下可能觀察到的平均互相關係數」。

做一百萬次，就得到了虛無假設下的完整分布。

然後，把實際觀測到的0.847放到那個分布裡，看看它落在什麼位置。

若嵐的手指在鍵盤上快速地打字。代碼一行一行地在螢幕上生長。

```python
import numpy as np
from scipy import signal

n_simulations = 1_000_000
observed_mean_xcorr = 0.847
simulated_means = []

for i in range(n_simulations):
    # 隨機打亂每個受害者的時間序列
    shuffled_xcorrs = []
    for victim in victims:
        shuffled_behavior = np.random.permutation(victim.behavior_sequence)
        xcorr = signal.correlate(shuffled_behavior, echo_commands, mode='full')
        max_xcorr = np.max(xcorr) / (len(victim.behavior_sequence) *
                    np.std(shuffled_behavior) * np.std(echo_commands))
        shuffled_xcorrs.append(max_xcorr)
    simulated_means.append(np.mean(shuffled_xcorrs))

p_value = np.mean(np.array(simulated_means) >= observed_mean_xcorr)
```

她按下了執行。

螢幕上跳出了一個進度條。

```
Monte Carlo Simulation: [░░░░░░░░░░░░░░░░░░░░] 0.0% — Est. time: 4h 12m
```

四個小時。

若嵐靠回椅背。三台螢幕的光打在她的臉上。進度條在最左邊的螢幕上以每秒大約0.007%的速度前進——肉眼幾乎看不到移動。像地質學的時間尺度。像大陸漂移。

她站起來。走到窗邊。

安全屋的窗戶很小，被百葉窗遮了一大半。從縫隙裡看出去，是中和的老社區——頂樓加蓋的鐵皮屋頂、晾衣架上的衣服在風裡搖擺、對面公寓的鐵窗上種了一排萬年青。天空是十一月的灰白色。不是陰天——是那種台北冬天特有的、介於晴和陰之間的、沒有表態的天空。

維誠現在在檢察署裡。正在被方正怡用X光般的眼睛審視。

若嵐能做的，都做了。數據備份了三份。統計分析在跑。維誠去自首了。蘇瑾行在聯繫媒體的備案管道。阿鬼在做最後的數位安全加固。

現在，只有等待。

---

## VI

蘇瑾行在十點半回到了安全屋。

他沒有和維誠一起出來——方正怡在蘇瑾行離開之前對他說了一句話：「這是偵查程序。你是記者，不是當事人。你可以走了。」語氣不帶任何商量的餘地。蘇瑾行識相地站起來，在維誠肩膀上拍了一下，走出了辦公室。

他回到安全屋的時候，若嵐正坐在螢幕前面。進度條走到了3.7%。

「她怎麼說？」若嵐問。

蘇瑾行把連帽外套拉下來，露出被冷風吹紅的耳廓。

「她在聽。真正的在聽。不是敷衍——是那種⋯⋯」他想了一下。「妳知道那種感覺嗎？有些人聽你說話的時候，你能看到他們的注意力在某些詞上面停頓。像一台雷達，掃到一個目標的時候會鎖定幾秒。方正怡就是那樣。維誠每說到一個技術細節——伺服器架構、加密協定、數據銷毀的方式——她的眼睛就會微微收窄。不是懷疑。是對焦。」

「她相信了？」

「還不確定。但她收了硬碟。而且她叫了數位取證的人上來。」蘇瑾行坐在若嵐旁邊的椅子上。椅子的輪子在地板上滑了幾公分，他用腳止住了。「我的判斷——」

他停了一下。記者的謹慎。不過度推斷。

「——她至少認為這件事值得花時間查。」

若嵐點了一下頭。

她回頭看螢幕。進度條。4.1%。每一個百分比背後是大約一萬次模擬。每一次模擬裡，一百萬個隨機排列的時間序列被生成、計算互相關、取平均值、存儲。一萬個微型的平行宇宙——在每一個宇宙裡，Echo不存在，受害者的行為是獨立的隨機事件——被創造、測量、然後丟棄。

到目前為止，四萬一千個平行宇宙。

沒有一個宇宙裡，平均互相關係數達到0.847。

最高的一次是0.23。

若嵐盯著那個數字。0.23。在最最最幸運的隨機條件下——四萬一千次嘗試中最極端的一次——平均互相關也只有0.23。而她觀測到的是0.847。

差距不是數量級的。差距是類別的。0.23和0.847之間的距離不是從台北到高雄——是從台北到火星。它們不在同一個分布裡。它們屬於不同的宇宙。

---

## VII

時間在安全屋裡以一種奇怪的方式流動。

不是快，不是慢。是稠的。像蜂蜜。每一分鐘都有質量。每一分鐘都被焦慮和希望的混合物灌滿了，沉甸甸地掛在時鐘的秒針上，讓秒針走得吃力。

蘇瑾行在若嵐旁邊坐了下來。

他們沒有說話。

若嵐的眼睛在三台螢幕之間移動——左邊是模擬的進度條（7.3%）、中間是已經完成的模擬結果的直方圖（一個以0.02為中心的、窄窄的鐘形曲線，0.847標示在曲線的極右側，遠遠超出了分布的邊界，像一顆流星飛出了銀河系的邊緣）、右邊是維誠離開前登入的伺服器監控界面（Echo的數據銷毀程序還在跑，紅色的DELETE指令一行一行地滾動，像一條流血的河）。

蘇瑾行的手機震了一下。他看了一眼。是阿鬼的加密訊息。

「媒體備份已上傳。三份。分散式儲存。就算陳默炸了整個AWS，數據也還在。」

若嵐聽到了但沒有反應。她的注意力全部集中在進度條上。8.2%。

然後她的手機也震了。

不是阿鬼。

是維誠。

訊息只有三個字：「還在談。」

若嵐看了一眼時間。十一點十五分。維誠進去已經兩個半小時了。

兩個半小時。方正怡用兩個半小時在聽一個吹哨者的陳述，而不是在十分鐘之內把他打發走。這本身就是一個信號。一個好的信號。

但若嵐不敢把它放進她的貝氏模型裡。不敢更新先驗。不敢讓希望的機率上升太快——因為她知道，在量子力學裡，觀測會改變被觀測的對象；但在心理學裡，期望也會改變期望者。如果她讓自己相信「方正怡會接案」，那麼當結果不是這樣的時候，落差造成的傷害會比「一開始就沒有期望」更大。

她選擇不更新。

她選擇等。

進度條。9.1%。

---

## VIII

中午。蘇瑾行出去買了兩個便當。

排骨飯。從安全屋樓下的自助餐店。便當盒是那種白色保麗龍的，打開蓋子的時候會冒出一股混合了高麗菜和滷排骨的熱氣。若嵐吃了三口就放下了。不是不餓——是胃在焦慮的狀態下拒絕工作。胃壁的平滑肌在交感神經的控制下收縮成一個僵硬的拳頭，不給食物通過的空間。

蘇瑾行也沒吃多少。他坐在椅子上，筷子夾著一塊排骨，停在半空中，眼睛看著螢幕上的進度條。18%。

「蒙地卡羅。」他忽然說。

若嵐看著他。

「我在想這個方法的隱喻。」蘇瑾行把排骨放回便當盒裡。「妳跑一百萬次模擬——相當於在黑暗裡投擲一百萬次骰子。每一次投擲都是一個假想的宇宙：如果世界是公平的，如果沒有人在操控，骰子會怎麼落？」

「對。」若嵐說。

「然後妳把真實的世界和那一百萬個假想的宇宙做比較。如果真實的世界和所有假想的宇宙都不一樣——如果它是一百萬分之一的異類——那就意味著⋯⋯」

「意味著真實的世界不是公平的。」若嵐替他說完了。「骰子被動過手腳。」

蘇瑾行點頭。

他沉默了幾秒。然後：

「一百萬次。妳問了一百萬次『如果世界是公平的，會發生這樣的事嗎？』」

「是。」

「答案是什麼？」

若嵐看了一眼螢幕。到目前為止已經完成了十八萬次模擬。十八萬個假想的宇宙。

「答案是：不會。」她說。「十八萬次裡，沒有一次的平均互相關係數超過0.35。而我們觀測到的是0.847。」

蘇瑾行消化了這個數字。

「所以在十八萬個公平的宇宙裡——」

「——沒有一個宇宙能產生我們觀察到的模式。」

窗外有一陣風吹過。百葉窗的葉片在風裡微微搖晃，光線在地板上投下的條紋跟著波動了一下。像水面。像不確定性在房間裡短暫地湧動了一下，然後又平靜了。

進度條。19%。

---

## IX

下午一點四十分。維誠發來了第二條訊息。

「出來了。」

若嵐的心跳在看到那三個字的時候加速了——不是劇烈的，是從每分鐘七十二下跳到了八十四下的那種。一個可以被血壓計記錄到但不至於觸發焦慮的增幅。

然後第二條訊息來了：「回去再說。」

若嵐把手機放在桌上。面朝下。

蘇瑾行看了她一眼。「要去接他嗎？」

「不。他自己會回來。」

若嵐的聲音是平的。但她的手在發抖。

不是大幅度的顫抖——是那種只有在把手指平伸出來、靜止不動的時候才看得到的微小震顫。指尖以每秒大約六到八次的頻率在空中畫出幾乎不可見的弧線。生理性震顫。每個人都有。但在焦慮的狀態下，震顫的幅度會增加到肉眼可見的程度——腎上腺素提高了肌肉纖維的自發放電頻率。

蘇瑾行看到了。

他沒有說話。他把椅子往若嵐的方向推了十五公分——不多，但足以讓他的右手自然地落在兩張椅子之間的扶手上。離若嵐的左手大約五公分。

三十秒之後——也可能是一分鐘，若嵐沒有計時——她的左手移動了。五公分。

蘇瑾行的手接住了它。

不是握——是包。他的手指沒有收緊。只是攤開了掌心，讓若嵐的手落進去。像接住一片從樹上掉下來的葉子。不是用力抓住——是讓它停在那裡。讓它不再飄。

若嵐的手指不再抖了。

不是因為蘇瑾行的手有什麼神奇的穩定效果。是因為——當你的手被另一隻手接住的時候，你的指尖的震顫會被另一個人的體溫傳導吸收。就像在兩個彈簧之間加了一個阻尼器。物理學。熱力學。兩個系統之間的能量耦合。

或者不是物理學。

或者就是一個人握住了另一個人的手。

進度條。23%。

---

## X

維誠在下午三點十七分回到了安全屋。

他推門進來的時候，若嵐首先看到的是他的臉。

面色疲憊。那種被連續高壓對話抽乾了的疲憊——不是肌肉的疲勞，是神經的。他的眼窩下面有淡灰色的陰影，瞳孔稍微散大了一點，嘴唇微微發白。這些是交感神經在長時間激活後、副交感神經接管時出現的典型體徵：瞳孔開始放鬆，但還沒完全恢復；嘴唇的血管在應激狀態下收縮過度，現在才開始慢慢回流。

但他的表情是平靜的。

不是空洞的平靜。是完成了某件事之後的平靜。像一個長跑選手跑過終點線之後的那幾步——腿還在動，但力量已經不是來自意志，是慣性。慣性的平靜。

他站在安全屋的門口。看著若嵐。看著蘇瑾行。

然後他說了八個字。

「她相信了。調查開始了。」

若嵐的呼吸在「調查開始了」四個字上停了一拍。

不是那種戲劇化的屏息——是橫膈膜在接收到一個超出預期的正面信號時的短暫失調。像一台精密的機器收到了一個它一直在等但已經開始懷疑永遠不會來的指令。處理器在那一毫秒裡做了一次完整的上下文切換：從「等待模式」切換到了「執行模式」。然後呼吸恢復了。

「細節。」若嵐說。一個字。

維誠走到桌邊。他看了一眼進度條——31%——然後坐下來。

「方正怡花了四個小時問我。」他說。聲音是沙的。四個小時的連續對話把他的聲帶磨出了粗糙的紋理。「不是隨便問——是按照一個嚴格的架構在問。她先問了組織結構。然後問技術架構。然後問受害者的選擇標準。然後問操控的具體方法。然後問我個人的參與程度。每一個環節她都追到底。」

他停了一下。手伸向桌上的水杯，發現杯子是空的，放下了。若嵐遞了一瓶礦泉水過去。他擰開瓶蓋，喝了三口。喉結上下滑動了三次。

「她花了最長時間在數據上。」他繼續說。「她讓數位取證的助理當場打開了硬碟——在她的電腦上，用隔離環境——然後她讓我指給她看。每一個文件夾的結構。每一份干預日誌的格式。她特別問了日誌裡的時間戳記是不是原始的——有沒有被篡改的痕跡。我給她看了SHA-256的校驗碼——每一份文件在創建時就計算了雜湊值，存在獨立的日誌裡。如果文件被修改過，雜湊值會不匹配。」

「匹配了？」若嵐問。

「全部匹配。兩百多份干預日誌。一份都沒有被動過。」

蘇瑾行的身體前傾了。「她怎麼說？」

維誠的嘴角出現了一個極淡的弧度——那個弧度裡有疲憊、有如釋重負、有三年的等待終於得到回應的微小的感動。

「她說了兩件事。第一：她會以科技犯罪的框架啟動正式的偵查程序。第二：她需要一份統計分析報告——用法庭能理解的語言寫的。她說她會找司法統計的專家來審閱，但她需要我們先提供一份初稿。」

他看向若嵐。

「她說：『如果妳們的統計學家能告訴我，這些模式在隨機條件下出現的機率有多低——低到讓我無法忽視——我就有足夠的理由申請搜索票。』」

若嵐轉頭看向螢幕。

進度條。33%。

一百萬次模擬。目前完成了三十三萬次。三十三萬個假想的宇宙。

在所有這些宇宙裡，平均互相關係數的最大值是0.31。

若嵐要告訴方正怡的是：在一百萬個公平的宇宙裡，沒有一個能解釋我們在真實世界中看到的東西。

---

## XI

模擬在晚上十一點四十三分跑完了。

安全屋裡只剩若嵐一個人。蘇瑾行在傍晚出去處理媒體管道的事——他聯繫了一家他信任的網路媒體，準備在適當的時機發布第二波報導。維誠回到了曉夢的花店——不是因為他不想留下來，是因為若嵐看出了他的疲憊已經超過了神經系統的安全閾值，逼他去休息。「你今天做了最難的部分。剩下的交給我。」她這樣說。

維誠走的時候在門口回頭看了一眼。他的眼睛裡有一種若嵐很熟悉的東西——信任。不是對若嵐個人的信任——是對整個團隊的信任。是一個孤獨了三年的人，終於發現自己不再需要一個人扛所有東西之後的那種信任。

門關了。

若嵐一個人面對三台螢幕。

十一點四十三分。進度條：

```
Monte Carlo Simulation: [████████████████████] 100.0% — COMPLETE
```

她點開了結果。

螢幕上彈出了一個直方圖。橫軸是「平均互相關係數」，縱軸是「頻率」。一百萬個模擬結果在直方圖上堆疊成了一個漂亮的鐘形曲線——以0.018為中心，標準差0.047。分布的右尾在0.35左右就已經完全歸零了。

在直方圖的最右邊，遠遠地——遠到若嵐不得不把橫軸的範圍擴大兩倍才能把它顯示在畫面裡——是一條紅色的垂直虛線。

0.847。

實際觀測值。

紅色的虛線和鐘形曲線之間的距離是荒誕的。像一個人站在撒哈拉沙漠的中央，指著地平線以外的某一點說「那裡有一棵樹」。沒有人能看到那棵樹。因為那棵樹不在這個沙漠裡。那棵樹在另一個星球上。

P值：

```
p-value = 0.000000
(0 out of 1,000,000 simulations produced a mean
cross-correlation ≥ 0.847)
```

零。

一百萬次模擬。沒有一次——零次——產生了大於或等於0.847的平均互相關係數。

P < 0.000001。

若嵐盯著那個數字。她的大腦在自動翻譯它的含義：如果Echo不存在——如果沒有人在操控——如果這兩百多個受害者的行為崩潰真的是各自獨立、各自隨機的——那麼我們觀察到如此一致的模式的機率，小於百萬分之一。

小於百萬分之一。

在統計學的語言裡，這意味著：虛無假設被拒絕。以任何合理的顯著水準。不是0.05。不是0.01。不是0.001。是0.000001。

一百萬次模擬，問了一百萬次「如果世界是公平的，會發生這樣的事嗎？」

一百萬次的回答都是：不會。

---

## XII

若嵐開始寫報告。

她打開了一個新的文件。文件名：「統計鑑定報告_Echo Protocol行為操控之機率分析.pdf」。

她寫了一整夜。

不是那種靈感湧現的寫作——是那種每一句話都被反覆斟酌、每一個數字都被交叉驗證、每一個邏輯步驟都被檢查三遍的寫作。因為這份報告不是寫給學術期刊的——是寫給法庭的。它的讀者不是統計學教授——是法官、檢察官、律師、和十二個可能完全沒有統計學背景的陪審員（如果最終進入審判的話）。

她必須讓一個不懂P值的人理解P值意味著什麼。

她必須讓一個從來沒有聽過蒙地卡羅模擬的人理解為什麼一百萬次模擬的結果是可信的。

她必須用最簡單的語言解釋最複雜的統計推論，同時不犧牲任何精確性。

報告的核心段落，她改了七遍：

---

*「本報告使用蒙地卡羅模擬方法，檢驗以下虛無假設：『被害人之行為崩潰係各自獨立之隨機事件，與Echo Protocol之環境操控無關。』*

*模擬過程如下：將每位被害人之行為時間序列進行隨機重新排列，消除其與Echo指令序列之時間對應關係，然後重新計算行為—指令互相關係數。此過程重複一百萬（1,000,000）次。*

*結果：在一百萬次模擬中，沒有任何一次產生之平均互相關係數達到或超過實際觀測值（0.847）。模擬產生之最大平均互相關係數為0.31。*

*以白話表述：想像我們問了一百萬次這個問題——『如果沒有人在操控，這些人的行為崩潰有沒有可能碰巧形成如此一致的模式？』一百萬次的回答都是：不可能。*

*P值 < 0.000001。*

*統計結論：拒絕虛無假設。被害人之行為崩潰模式與Echo Protocol之操控指令之間存在高度一致性，此一致性在隨機條件下出現之機率低於百萬分之一。此結果強烈支持以下結論：被害人之行為崩潰並非獨立之隨機事件，而是受到系統性之外部干預。」*

---

她寫完最後一個句號的時候，窗外的天空已經開始發白了。

凌晨五點十二分。

若嵐把報告存檔。列印了一份——安全屋裡有一台維誠帶來的小型印表機，用的是沒有聯網功能的USB直連。紙張從出紙口一頁一頁地吐出來，帶著碳粉的溫熱和新鮮印刷品特有的微苦氣味。

她把報告整理好。釘書機。封面。一個牛皮紙的資料夾。

然後她靠在椅背上，閉上了眼睛。

在黑暗裡，她看到了一百萬個宇宙。

一百萬個假想的、公平的、沒有Echo的宇宙。在每一個宇宙裡，那兩百多個人——林哲宇、若晴、所有被陳默的模型碾過的人——他們的生活是隨機的、獨立的、不受控制的。他們有的會遇到壞事。有的會崩潰。有的可能——在最不幸的隨機組合下——會選擇結束。但他們的崩潰不會形成模式。不會有0.847的互相關。不會有時間和手段和結果的精確對應。

在那一百萬個宇宙裡，悲劇是隨機的。隨機的悲劇是殘酷的，但它是公平的——沒有人要為骰子的結果負責。

但我們不在那一百萬個宇宙裡。

我們在這一個宇宙裡。

在這個宇宙裡，悲劇不是隨機的。骰子被動過手腳。有人在控制。有人要負責。

若嵐睜開眼睛。

天亮了。安全屋的窗戶上有一層薄薄的霧氣——夜裡的溫差讓玻璃表面凝結了一層水珠。光線透過那些水珠進來，變成了一種散射的、柔軟的、帶著微微暖意的白。

桌上的牛皮紙資料夾。裡面是一份可以改變一切的報告。

今天，她要把這份報告送到方正怡手裡。

## 白若嵐的諮商手記 #39

**日期**：安全屋。凌晨。報告寫完了。印表機的碳粉味道還在空氣裡。窗外的天空正在從黑變灰變白——一個連續的、不可逆的轉換。就像我們正在做的事。

### 關於蒙地卡羅

蒙地卡羅模擬。

名字來自賭場。方法來自原子彈。用途是：在你無法直接計算答案的時候，用大量的隨機試驗去逼近答案。

這個方法的哲學前提很簡單：如果你不知道真相是什麼，就去想像一萬種、十萬種、一百萬種「不是真相」的可能。把所有這些可能性排列出來，形成一個分布。然後看看你手中的現實——你觀察到的數據、你親眼見證的事件——落在這個分布的什麼位置。

如果它落在正中間——恭喜，你手中的現實完全可以被「巧合」解釋。不需要陰謀論。不需要幕後黑手。骰子就是這樣落的。

如果它落在邊緣——嗯，有點不尋常，但還在合理的範圍內。值得注意，但不必驚慌。

如果它落在分布的外面——如果它不在一百萬種可能性的任何一種裡——

那你手中的現實就不是巧合。

那就有人動了手腳。

我跑了一百萬次模擬。

一百萬次。

我的電腦花了將近十二個小時。在這十二個小時裡，它創造了一百萬個虛擬的宇宙——在每一個宇宙裡，Echo不存在，陳默是一個普通的退休教授，那兩百多個人的生活是獨立的、隨機的、不受任何外力控制的。我的電腦在每一個宇宙裡測量了「行為崩潰模式的一致性」，然後記錄下來，然後銷毀那個宇宙，創造下一個。

一百萬個宇宙。生。測量。死。生。測量。死。

像一個不知疲倦的、沒有感情的上帝，在用排列組合窮盡所有的可能性。

結果：

一百萬個宇宙裡，沒有一個——零個——能產生我們在真實世界中觀察到的模式。

P < 0.000001。

---

這個結果的意義是什麼？

不是「陳默一定有罪」。統計學不做有罪判決。統計學做的是：排除巧合。

P < 0.000001的意思是：如果你堅持相信「這只是巧合」，那你需要接受一個事實——這個巧合在一百萬次嘗試中連一次都沒有出現過。你不是在相信一個不太可能的事件。你是在相信一個在已知的隨機條件下機率為零的事件。

零。

不是很小。是零。

這不是統計顯著。這是統計學在對你大喊：「這裡有東西不對。」

### 關於等待

今天有一個場景我會記住很久。

在檢察署外面等維誠。四個小時。蘇瑾行陪著我。

我們沒有說話。只是坐在走廊的長椅上。長椅的木頭表面很涼——十一月的室內溫度加上磨石子地板的散熱效應，讓椅面的溫度大約在十八度左右。我的手在發抖。不是冷。是焦慮。是腎上腺素讓肌纖維的自發放電頻率超過了正常值。

蘇瑾行握住了我的手。

他沒有說「會沒事的」。沒有說「別擔心」。沒有說任何試圖降低我焦慮值的話——因為他知道，對一個數據導向的人來說，沒有數據支持的安慰詞是空洞的。你不能用「會沒事的」來降低一個後驗機率。你不能用語言來改變貝氏更新的結果。

他只是握住了我的手。

這在統計學上沒有意義。一個人握住另一個人的手，不會改變檢察官的決定。不會改變P值。不會改變證據的強度或法律的程序。

但在不是統計學的那個世界裡——在人活著的那個世界裡——

它有意義。

因為等待是一種特殊的時間狀態。在等待中，你的大腦會自動跑一種人類版本的蒙地卡羅模擬——你會不由自主地想像所有可能的結果。方正怡相信了。方正怡不相信。維誠被逮捕了。維誠安全出來了。一萬種可能性在你的腦海裡同時展開，像薛丁格的貓——在你打開盒子之前，所有的結果都同時存在。

在這種疊加態裡，一個人的手握住你的手，不會讓好的結果更可能發生。但它會讓你在等待的過程中不至於被那些壞的可能性壓垮。

物理學家可能會把這叫做阻尼。

我把它叫做陪伴。

### 最後

蒙地卡羅模擬的意義不在於每一次的結果。

每一次模擬都是隨機的。每一個假想的宇宙都是獨一無二的、不可預測的、一次性的。你無法從任何一次模擬中得出結論。

但一百萬次之後，模式浮現了。

這就像人生。

你不能預測任何一天會發生什麼。你不知道明天的太陽從哪邊升起——好吧，你知道，太陽從東邊升起。但你不知道明天的心情從哪邊升起。你不知道明天會遇到誰、失去誰、被什麼意想不到的事情擊中。每一天都是一次隨機試驗。

但如果你活得夠久——如果你經歷得夠多——如果你願意回頭看——

模式就會浮現。

你會發現某些人總是在你最需要的時候出現。你會發現某些選擇——在做的時候看起來完全隨機的選擇——在回顧的時候形成了一條線。你會發現，生活不是完全隨機的。它有一個底層的結構。不是因為有誰在操控——而是因為你自己的選擇、你自己的價值觀、你自己反覆做出的微小的決定——它們在大數法則之下，會收斂成一個模式。

你的模式。

而有些模式，不應該存在。

兩百多個人的行為崩潰形成了一個不應該存在的模式。不是因為他們的選擇——而是因為有人覆蓋了他們的選擇。有人把他們的隨機變成了確定。有人把他們的人生變成了一組被控制的實驗變數。

蒙地卡羅告訴我：在公平的世界裡，這個模式不會出現。

所以世界不公平。

所以有人要為此負責。

明天我把報告送出去。

一百萬次模擬的結論，裝在一個牛皮紙的資料夾裡。一百二十克的數據——和維誠帶去檢察署的硬碟一樣重。一百二十克。

兩個一百二十克。

一個裝著兩百多個人的命運。一個裝著一百萬個假想宇宙的回答。

兩個一百二十克加在一起，應該足以讓天平傾斜了。

——白若嵐。天亮了。碳粉的味道。進度條100%。

**【本章統計概念】**

- **蒙地卡羅模擬（Monte Carlo Simulation）**：一種利用大量隨機抽樣來估計複雜系統行為的計算方法。名稱來自摩納哥的蒙地卡羅賭場，由數學家斯坦尼斯瓦夫·烏拉姆（Stanislaw Ulam）和約翰·馮·紐曼（John von Neumann）在1940年代曼哈頓計畫期間發展。核心思想：當一個問題的解析解過於複雜或不可能求得時，可以通過大量隨機試驗的統計結果來逼近答案。例如：估計圓周率π可以通過在正方形中隨機投點、計算落在內切圓中的比例來實現。模擬次數越多，估計值越精確（根據大數法則）。蒙地卡羅方法廣泛應用於金融風險評估、物理模擬、工程可靠度分析、貝氏統計推論等領域。本章中若嵐使用蒙地卡羅模擬來估計虛無假設下觀測到特定模式的機率——即P值的數值估計。

- **互相關（Cross-Correlation）**：衡量兩個時間序列之間相似程度的統計指標。給定兩個時間序列x(t)和y(t)，互相關函數R_xy(τ)測量的是x(t)和y(t+τ)之間的相關性，其中τ是時間延遲。當τ=0時，互相關等價於皮爾森相關係數。互相關的值介於-1和1之間：1表示完美正相關，-1表示完美負相關，0表示無相關。本章中若嵐用互相關來測量受害者的行為時間序列和Echo操控指令時間序列之間的一致性。

- **自助法（Bootstrap）**：由布拉德利·艾弗隆（Bradley Efron）於1979年提出的統計重抽樣方法。核心思想：從已有的樣本數據中進行有放回的重複抽樣，生成大量「模擬樣本」，以此估計統計量的分布。自助法不需要對數據的分布做任何假設（非參數方法），因此特別適用於分布未知或複雜的情況。本章中若嵐使用排列檢定（permutation test）的變體——隨機打亂時間序列的順序來模擬虛無假設——這在精神上與自助法類似，都是通過重抽樣來建構統計量的分布。

- **P值（P-value）的蒙地卡羅估計**：傳統的P值計算依賴已知的統計分布（如t分布、卡方分布）。但當檢定統計量的分布過於複雜而無法用解析方法求得時，可以使用蒙地卡羅方法來估計P值：在虛無假設下生成大量模擬數據，計算每次模擬的檢定統計量，然後計算「模擬統計量大於等於實際觀測統計量的比例」。這個比例就是P值的蒙地卡羅估計。模擬次數N越大，估計越精確。當N = 1,000,000時，P值估計的精確度約為±0.001（即如果真實P值為0.05，估計值有95%的信心落在0.049到0.051之間）。本章中P = 0/1,000,000 = 0，表示在一百萬次模擬中沒有任何一次超過觀測值，意味著真實的P值極小（< 10⁻⁶）。

- **SHA-256校驗碼**：安全雜湊演算法（Secure Hash Algorithm）的一種，可以將任意長度的數據映射為固定的256位元（32位元組）雜湊值。其特性包括：（1）確定性——相同的輸入永遠產生相同的輸出；（2）雪崩效應——輸入的微小變化會導致輸出完全不同；（3）不可逆性——從雜湊值無法反推原始輸入；（4）碰撞抗性——找到兩個不同輸入產生相同輸出在計算上不可行。本章中用於驗證干預日誌文件的完整性——如果文件未被篡改，其SHA-256雜湊值應與創建時記錄的值完全一致。
