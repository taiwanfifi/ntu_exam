### 題目 1：電阻串並聯與等效電阻

**來源**：經典題型
**難度**：★☆☆☆☆
**主題**：基本電路定律 — 電阻串並聯

**題目**：

如下圖所示電路，由端子 a-b 看入，求等效電阻 R_eq。

```
    a ──┬── R1=6Ω ──┬── R3=4Ω ──┐
        │            │            │
       R2=3Ω       R4=12Ω       R5=6Ω
        │            │            │
    b ──┴────────────┴────────────┘
```

**詳解**：

**解題思路**：觀察電路拓撲，R4 與 R3+R5 串聯後並聯，再與 R1 串聯，最後與 R2 並聯。由內而外逐步化簡。

**步驟**：

1. R3 與 R5 串聯：
   R_35 = R3 + R5 = 4 + 6 = 10 Ω

2. R_35 與 R4 並聯：
   R_345 = R4 ∥ R_35 = (R4 × R_35) / (R4 + R_35) = (12 × 10) / (12 + 10) = 120 / 22 ≈ 5.45 Ω

3. R1 與 R_345 串聯：
   R_1345 = R1 + R_345 = 6 + 120/22 = 132/22 + 120/22 = 252/22 = 126/11 Ω

4. R2 與 R_1345 並聯：
   R_eq = R2 ∥ R_1345 = (3 × 126/11) / (3 + 126/11) = (378/11) / (33/11 + 126/11) = (378/11) / (159/11) = 378/159 = 126/53 Ω

**答案**：

> **R_eq = 126/53 ≈ 2.377 Ω**

**延伸思考**：若某個電阻改為開路或短路，等效電阻如何變化？可練習橋式電路（Wheatstone Bridge）的等效電阻計算。

### 題目 2：KVL 與 KCL 綜合應用

**來源**：台大電機 108-1 期中考風格
**難度**：★★☆☆☆
**主題**：基本電路定律 — KVL、KCL

**題目**：

在以下電路中，求電流 I 及電壓 V_x。

```
        I →
   ┌────────── 4Ω ──────────┐
   │                          │
 (+)                         (+)
 12V                        V_x
 (-)                         (-)
   │         6Ω              │
   ├────────────────┬────────┤
   │                │        │
   │              3Ω         │
   │                │        │
   └────────────────┴────────┘
```

12V 電壓源正端在上，左側迴路上方為 4Ω 電阻，下方 6Ω 與 3Ω 並聯，V_x 跨在並聯組合上。

**詳解**：

**解題思路**：先求 6Ω 與 3Ω 的並聯等效，再用 KVL 求迴路電流 I，最後求 V_x。

**步驟**：

1. 求 6Ω 與 3Ω 並聯等效電阻：
   R_parallel = (6 × 3) / (6 + 3) = 18 / 9 = 2 Ω

2. 迴路總電阻：
   R_total = 4 + 2 = 6 Ω

3. 由 KVL 繞迴路（順時針）：
   -12 + 4I + V_x = 0

   其中 V_x = I × R_parallel = 2I

   代入：-12 + 4I + 2I = 0
   6I = 12
   I = 2 A

4. 求 V_x：
   V_x = 2I = 2 × 2 = 4 V

5. 驗證（KVL）：-12 + 4(2) + 4 = -12 + 8 + 4 = 0 ✓

**答案**：

> **I = 2 A，V_x = 4 V**

**延伸思考**：利用 KCL 驗證流過 6Ω 與 3Ω 的個別電流。I_6 = 4/6 = 2/3 A，I_3 = 4/3 A，I_6 + I_3 = 2/3 + 4/3 = 2 A = I ✓。

### 題目 3：功率計算與能量守恆

**來源**：經典題型
**難度**：★★☆☆☆
**主題**：基本電路定律 — 功率計算

**題目**：

下列電路中有五個元件，已知各元件的電壓電流如下表所示（電壓與電流皆遵循被動符號慣例 Passive Sign Convention）。判斷每個元件是吸收功率還是提供功率，並驗證功率守恆。

| 元件 | 電壓 (V) | 電流 (A) |
|------|----------|----------|
| A    | +10      | -2       |
| B    | +6       | +3       |
| C    | -4       | -1       |
| D    | +8       | +2       |
| E    | -12      | +1       |

**詳解**：

**解題思路**：在被動符號慣例下，P = V × I。若 P > 0，元件吸收功率；若 P < 0，元件提供（釋放）功率。所有元件功率代數和應為零（功率守恆，Tellegen's Theorem）。

**步驟**：

1. 元件 A：P_A = (+10)(−2) = −20 W → 提供 20 W
2. 元件 B：P_B = (+6)(+3) = +18 W → 吸收 18 W
3. 元件 C：P_C = (−4)(−1) = +4 W → 吸收 4 W
4. 元件 D：P_D = (+8)(+2) = +16 W → 吸收 16 W
5. 元件 E：P_E = (−12)(+1) = −12 W → 提供 12 W

6. 驗證功率守恆：
   ΣP = −20 + 18 + 4 + 16 + (−12) = 6 ≠ 0

   注意：此處功率不守恆，代表題目數據有誤或電路未封閉。若將元件 D 的電流改為 +1 A：
   P_D = 8 × 1 = 8 W
   ΣP = −20 + 18 + 4 + 8 − 12 = −2 ≠ 0

   重新設定正確數據：將元件 B 改為電流 +2 A：
   P_B = 6 × 2 = 12 W
   ΣP = −20 + 12 + 4 + 16 − 12 = 0 ✓

   功率守恆成立：提供功率 = 20 + 12 = 32 W，吸收功率 = 12 + 4 + 16 = 32 W。

**答案**：

> **提供功率的元件：A（20W）、E（12W）；吸收功率的元件：B（12W）、C（4W）、D（16W）。全電路功率守恆，ΣP = 0。**

**延伸思考**：Tellegen's Theorem 指出滿足 KVL 與 KCL 的任意電路中，所有元件功率代數和必為零。此定理與元件本身的特性無關，只要求電路拓撲正確。

### 題目 4：相依電源與 KCL/KVL

**來源**：台大電機 109-1 期中考風格
**難度**：★★★☆☆
**主題**：基本電路定律 — 相依電源

**題目**：

求下列含有電流控制電壓源（CCVS）的電路中，電流 I_o 的值。

```
       2Ω       I_o →       4Ω
  ┌────┤────────────────────┤────┐
  │                              │
(+)                            (+)
 20V                          5·I_x
(-)                            (-)
  │         I_x →               │
  ├────────── 8Ω ───────────────┤
  │                              │
  └──────────────────────────────┘
```

左側為 20V 獨立電壓源，右側為相依電壓源 5·I_x（CCVS），其中 I_x 為流過 8Ω 的電流。上方支路有 2Ω 和 4Ω 串聯，流過電流 I_o。

**詳解**：

**解題思路**：此電路有兩個迴路，含有一個相依電源。使用 KVL 列兩個迴路方程式求解。

**步驟**：

1. 定義迴路電流：上方迴路電流為 I_o（順時針），下方迴路電流為 I_x（順時針）。

2. 由 KCL 可知：上方支路電流為 I_o（從左到右），下方支路電流為 I_x（從左到右）。

3. 對上方迴路（順時針方向）列 KVL：
   -20 + 2·I_o + 4·I_o + 5·I_x = 0
   -20 + 6·I_o + 5·I_x = 0  ... (1)

4. 對下方迴路（順時針方向）列 KVL：
   -20 + 8·I_x = 0  ... 但注意需考慮相依電源的極性。

   重新分析：左側 20V 電壓源為兩迴路共用。

   下方迴路 KVL（順時針）：
   -20 + 8·I_x = 0
   I_x = 20/8 = 2.5 A

5. 代入方程式 (1)：
   -20 + 6·I_o + 5(2.5) = 0
   -20 + 6·I_o + 12.5 = 0
   6·I_o = 7.5
   I_o = 1.25 A

6. 驗證：
   上方迴路 KVL：-20 + 2(1.25) + 4(1.25) + 5(2.5) = -20 + 2.5 + 5 + 12.5 = 0 ✓

**答案**：

> **I_o = 1.25 A**

**延伸思考**：若相依電源改為 VCCS（電壓控制電流源），分析方法會有何不同？含相依電源的電路不能直接使用疊加定理消去相依電源。

### 題目 5：Wheatstone Bridge 平衡條件

**來源**：經典題型
**難度**：★★★☆☆
**主題**：基本電路定律 — 橋式電路

**題目**：

如圖所示為一惠斯登電橋（Wheatstone Bridge），求使電橋平衡（即 R5 上無電流流過）時 R_x 的值，並在平衡條件下求電源提供的總電流。

```
            a
           ╱ ╲
         R1    R2
        =2kΩ  =4kΩ
         ╱     ╲
    b ──R5=1kΩ── c
         ╲     ╱
        R3    R_x
       =3kΩ   =?
         ╲   ╱
           d
           │
          12V
           │
          GND ── a (回到 a 點)
```

12V 電壓源接在 a-d 之間（a 為正端）。

**詳解**：

**解題思路**：Wheatstone Bridge 平衡條件為對角乘積相等：R1 × R_x = R2 × R3。平衡時 R5 上電流為零，可視為開路。

**步驟**：

1. 平衡條件：
   R1 × R_x = R2 × R3
   2k × R_x = 4k × 3k
   R_x = (4k × 3k) / 2k = 12k²/2k = 6 kΩ

2. 平衡時 R5 無電流，等效為開路。電路簡化為兩個獨立支路並聯：
   - 左支路：R1 + R3 = 2k + 3k = 5 kΩ
   - 右支路：R2 + R_x = 4k + 6k = 10 kΩ

3. 等效電阻：
   R_eq = (5k × 10k) / (5k + 10k) = 50k²/15k = 10/3 kΩ ≈ 3.333 kΩ

4. 電源提供的總電流：
   I_total = V / R_eq = 12 / (10000/3) = 12 × 3/10000 = 36/10000 = 3.6 mA

5. 驗證平衡：
   V_b = 12 × R3/(R1+R3) = 12 × 3/5 = 7.2 V
   V_c = 12 × R_x/(R2+R_x) = 12 × 6/10 = 7.2 V
   V_b = V_c → R5 上電壓差為零 → 無電流 ✓

**答案**：

> **R_x = 6 kΩ，平衡時電源總電流 I_total = 3.6 mA**

**延伸思考**：Wheatstone Bridge 在實際中用於精密電阻量測。若電橋略微不平衡，可用 Thévenin 等效分析 R5 上的微小電流。

### 題目 6：節點電壓法（Node Voltage Method）

**來源**：台大電機 107-1 期中考風格
**難度**：★★☆☆☆
**主題**：電路分析技巧 — 節點電壓法

**題目**：

使用節點電壓法求下列電路中 V1 和 V2 的值。

```
              V1              V2
    ┌─────────●─────────────────●─────────┐
    │         │                 │         │
   5A↑      2Ω               4Ω        10Ω
   (源)       │                 │         │
    │         │      8Ω        │         │
    └─────────┤───────────────┤─────────┘
              │               │
             GND             GND
```

5A 電流源從接地端流入 V1 節點（方向朝上）。V1 與 V2 之間有 8Ω 電阻連接。V1 節點接 2Ω 到地，V2 節點接 4Ω 到地和 10Ω 到地。

**詳解**：

**解題思路**：選定參考節點（GND），對非參考節點 V1、V2 列 KCL 方程式。節點電壓法適用於節點數少的電路。

**步驟**：

1. 節點 V1 的 KCL（流出 = 流入）：
   V1/2 + (V1 - V2)/8 = 5

   將等式乘以 8 消去分母：
   4·V1 + V1 - V2 = 40
   5·V1 - V2 = 40  ... (1)

2. 節點 V2 的 KCL（流出電流之和 = 0，因無電流源）：
   V2/4 + V2/10 + (V2 - V1)/8 = 0

   找最小公倍數 40，乘以 40：
   10·V2 + 4·V2 + 5·(V2 - V1) = 0
   10·V2 + 4·V2 + 5·V2 - 5·V1 = 0
   -5·V1 + 19·V2 = 0  ... (2)

3. 由方程式 (1)：V2 = 5·V1 - 40

   代入 (2)：
   -5·V1 + 19·(5·V1 - 40) = 0
   -5·V1 + 95·V1 - 760 = 0
   90·V1 = 760
   V1 = 760/90 = 76/9 ≈ 8.444 V

4. V2 = 5(76/9) - 40 = 380/9 - 360/9 = 20/9 ≈ 2.222 V

5. 驗證（節點 V1 的 KCL）：
   V1/2 + (V1-V2)/8 = (76/9)/2 + (76/9 - 20/9)/8 = 38/9 + 56/(9×8) = 38/9 + 7/9 = 45/9 = 5 ✓

**答案**：

> **V1 = 76/9 ≈ 8.44 V，V2 = 20/9 ≈ 2.22 V**

**延伸思考**：若將 5A 電流源改為電壓源，需改用其他方法或做源轉換。節點電壓法在電流源多的電路中特別方便。

### 題目 7：網目電流法（Mesh Current Method）

**來源**：經典題型
**難度**：★★☆☆☆
**主題**：電路分析技巧 — 網目電流法

**題目**：

使用網目電流法求下列電路中的網目電流 I1、I2，以及 R3 上的電壓降。

```
     ┌──── R1=5Ω ────┬──── R2=10Ω ────┐
     │                │                 │
   (+)               R3=20Ω            (+)
   50V                │               30V
   (-)                │                (-)
     │                │                 │
     └────────────────┴────────────────┘
```

左迴路電流 I1（順時針），右迴路電流 I2（順時針）。

**詳解**：

**解題思路**：電路有兩個網目。對每個網目列 KVL 方程式，注意共用支路 R3 上的電流為 I1 - I2（因兩個網目電流方向在此支路相反）。

**步驟**：

1. 網目 1（順時針方向）的 KVL：
   -50 + R1·I1 + R3·(I1 - I2) = 0
   -50 + 5·I1 + 20·(I1 - I2) = 0
   -50 + 5·I1 + 20·I1 - 20·I2 = 0
   25·I1 - 20·I2 = 50  ... (1)

2. 網目 2（順時針方向）的 KVL：
   R3·(I2 - I1) + R2·I2 + 30 = 0
   20·(I2 - I1) + 10·I2 + 30 = 0
   -20·I1 + 20·I2 + 10·I2 + 30 = 0
   -20·I1 + 30·I2 = -30  ... (2)

3. 由 (2)：I1 = (30·I2 + 30) / 20 = (3·I2 + 3) / 2

   代入 (1)：
   25·(3·I2 + 3)/2 - 20·I2 = 50
   (75·I2 + 75)/2 - 20·I2 = 50
   75·I2 + 75 - 40·I2 = 100
   35·I2 = 25
   I2 = 25/35 = 5/7 ≈ 0.714 A

4. I1 = (3×5/7 + 3)/2 = (15/7 + 21/7)/2 = (36/7)/2 = 18/7 ≈ 2.571 A

5. R3 上的電壓降：
   V_R3 = R3 × (I1 - I2) = 20 × (18/7 - 5/7) = 20 × 13/7 = 260/7 ≈ 37.14 V

6. 驗證（網目 1 KVL）：
   -50 + 5(18/7) + 260/7 = -350/7 + 90/7 + 260/7 = 0 ✓

**答案**：

> **I1 = 18/7 ≈ 2.571 A，I2 = 5/7 ≈ 0.714 A，V_R3 = 260/7 ≈ 37.14 V**

**延伸思考**：網目電流法特別適用於平面電路（Planar Circuit），且電壓源多的電路比節點電壓法更方便。非平面電路無法使用網目電流法。

### 題目 8：超節點（Supernode）

**來源**：台大電機 110-1 期中考風格
**難度**：★★★☆☆
**主題**：電路分析技巧 — 超節點

**題目**：

使用節點電壓法（含超節點技巧）求下列電路中各節點電壓。

```
           V1         6V         V2
    ┌───────●───── (+)──(-)──────●───────┐
    │       │                     │       │
   2A↑     5Ω                   2Ω      10Ω
    │       │                     │       │
    └───────┴─────────────────────┴───────┘
                     GND
```

2A 電流源從 GND 流入 V1 節點。V1 與 V2 之間有 6V 電壓源（V1 端為正）。V1 接 5Ω 到地，V2 接 2Ω 和 10Ω 到地（2Ω 和 10Ω 並聯）。

**詳解**：

**解題思路**：V1 與 V2 之間有電壓源直接相連，無法直接寫出通過此電壓源的電流，因此將 V1 與 V2 合併為一個「超節點」。超節點的 KCL 方程式加上電壓源的約束方程式即可求解。

**步驟**：

1. 電壓源約束方程式：
   V1 - V2 = 6  ... (1)

2. 超節點（包含 V1 和 V2）的 KCL：
   流入超節點的電流 = 流出超節點的電流
   2 = V1/5 + V2/2 + V2/10

   將右側通分（公倍數 10）：
   2 = 2·V1/10 + 5·V2/10 + V2/10
   2 = (2·V1 + 6·V2) / 10
   20 = 2·V1 + 6·V2
   V1 + 3·V2 = 10  ... (2)

3. 聯立 (1) 和 (2)：
   由 (1)：V1 = V2 + 6
   代入 (2)：(V2 + 6) + 3·V2 = 10
   4·V2 = 4
   V2 = 1 V

4. V1 = 1 + 6 = 7 V

5. 驗證（超節點 KCL）：
   V1/5 + V2/2 + V2/10 = 7/5 + 1/2 + 1/10 = 14/10 + 5/10 + 1/10 = 20/10 = 2 ✓

**答案**：

> **V1 = 7 V，V2 = 1 V**

**延伸思考**：當電壓源一端接地時不需要超節點——該節點電壓直接等於電壓源的值。超節點技巧也適用於含相依電壓源的情況。

### 題目 9：超網目（Supermesh）

**來源**：台大電機 108-1 期中考風格
**難度**：★★★☆☆
**主題**：電路分析技巧 — 超網目

**題目**：

使用網目電流法（含超網目技巧）求下列電路中的 I1、I2、I3。

```
     ┌──── 2Ω ────┬──── 4Ω ────┬──── 1Ω ────┐
     │             │             │             │
   (+)            │            6A↑           (+)
   12V          3Ω            (源)           8V
   (-)            │             │            (-)
     │             │             │             │
     └─────────────┴─────────────┴─────────────┘
```

三個網目：I1（左，順時針）、I2（中，順時針）、I3（右，順時針）。6A 電流源位於網目 2 和網目 3 的共用支路上。

**詳解**：

**解題思路**：6A 電流源位於網目 2 和網目 3 之間的共用支路。無法直接列出此電流源的電壓降，因此將網目 2 和網目 3 合併為超網目。列出超網目 KVL、網目 1 的 KVL，以及電流源的約束方程式。

**步驟**：

### 題目 13：Norton 等效電路與源轉換

```

**解題思路**：逐步將電壓源+串聯電阻 ↔ 電流源+並聯電阻做源轉換，化簡電路至最終只剩一個源與 R_L。

1. **第一次源轉換**：12V 電壓源串聯 4Ω → 電流源並聯 4Ω

8. 求 I_L：

### 題目 14：最大功率轉移定理

**來源**：台大電機 110-2 期中考風格
**難度**：★★★☆☆
**主題**：電路定理 — 最大功率轉移

**題目**：

求負載電阻 R_L 為何值時可獲得最大功率，並求此最大功率。

```
       10Ω        5Ω
  ┌────┤────┬──────┤──── a
  │         │
(+)       20Ω          R_L
 40V        │
(-)         │
  │         │
  └─────────┴──────────── b
```

**詳解**：

**解題思路**：最大功率轉移定理（Maximum Power Transfer Theorem）指出，當 R_L = R_Th 時，負載可獲得最大功率。先求 Thévenin 等效，再計算最大功率。

**步驟**：

**求 V_Th：**

1. a-b 開路，5Ω 無電流。
   電流流過 10Ω 和 20Ω 串聯迴路：
   I = 40/(10 + 20) = 40/30 = 4/3 A

2. 20Ω 上的電壓：V_20 = 20 × 4/3 = 80/3 V

3. V_Th = V_20 − 0 = 80/3 V（5Ω 無電流，無壓降）
   V_Th = 80/3 ≈ 26.67 V

**求 R_Th：**

4. 關閉 40V（短路），從 a-b 看入：
   R_Th = 5 + (10 ∥ 20) = 5 + (10×20)/(10+20) = 5 + 200/30 = 5 + 20/3 = 35/3 Ω

**最大功率轉移：**

5. R_L = R_Th = 35/3 ≈ 11.67 Ω

6. 最大功率：
   P_max = V_Th² / (4·R_Th) = (80/3)² / (4 × 35/3)
   = (6400/9) / (140/3) = (6400/9) × (3/140) = 6400/(9×140/3) = 6400×3/(9×140)
   = 19200/1260 = 640/42 = 320/21 ≈ 15.24 W

**答案**：

> **R_L = 35/3 ≈ 11.67 Ω，P_max = 320/21 ≈ 15.24 W**

**延伸思考**：最大功率轉移時的效率僅為 50%（一半功率消耗在 R_Th 上）。在電力工程中，追求的是高效率而非最大功率轉移；在通訊工程中，信號功率微弱，追求最大功率轉移更為重要。

### 題目 15：含相依源的 Thévenin 等效

**解題思路**：利用 Op-Amp 理想假設：(1) 虛短路（Virtual Short）：V⁺ = V⁻；(2) 虛斷路（Virtual Open）：流入 Op-Amp 輸入端的電流為零。

**步驟**：

1. 由虛短路：V⁻ = V⁺ = 0 V（因為非反相端接地）

2. 由虛斷路：流入 (−) 端的電流為零。因此通過 R1 的電流全部流過 Rf。

3. R1 上的電流：
   I₁ = (v_s − V⁻) / R1 = (0.5 − 0) / 2k = 0.25 mA

4. Rf 上的電流（方向從 V⁻ 到 v_o）：
   I_f = I₁ = 0.25 mA

5. v_o = V⁻ − I_f × Rf = 0 − 0.25m × 10k = −2.5 V

6. 增益：A_v = v_o / v_s = −2.5 / 0.5 = −5

   通用公式驗證：A_v = −Rf / R1 = −10k / 2k = −5 ✓

**答案**：

> **v_o = −2.5 V，電壓增益 A_v = −Rf/R1 = −5**

**延伸思考**：反相放大器的輸入阻抗等於 R1（因為虛短路使 V⁻ = 0V，等效於 R1 一端接地），這是其缺點。若需要高輸入阻抗，應改用非反相放大器。

### 題目 17：非反相放大器（Non-Inverting Amplifier）

**來源**：經典題型
**難度**：★★☆☆☆
**主題**：運算放大器 — 非反相放大器增益

**題目**：

如圖所示為一非反相放大器電路，R1 = 4 kΩ，Rf = 20 kΩ，輸入電壓 v_s = 1 V。假設 Op-Amp 為理想，求輸出電壓 v_o。

```
                 Rf = 20kΩ
           ┌────────┤────────┐
           │                  │
           │    ┌──┐          │
      ┌────┤(-)    ├──┬── v_o
      │    │ Op-Amp │  │
v_s ──┤(+)    ├  │
           └──┘      │
     R1=4kΩ          GND
      │
     GND
```

**詳解**：

**解題思路**：非反相放大器的 (+) 端直接接輸入信號，利用虛短路和虛斷路推導增益。

**步驟**：

1. 由虛短路：V⁻ = V⁺ = v_s = 1 V

2. 由虛斷路：流入 (−) 端電流為零。R1 和 Rf 形成分壓器，將 v_o 分壓後等於 V⁻。

3. 分壓關係：
   V⁻ = v_o × R1 / (R1 + Rf)
   1 = v_o × 4k / (4k + 20k)
   1 = v_o × 4/24
   v_o = 24/4 = 6 V

4. 增益：A_v = v_o / v_s = 6 / 1 = 6

   通用公式驗證：A_v = 1 + Rf/R1 = 1 + 20k/4k = 1 + 5 = 6 ✓

**答案**：

> **v_o = 6 V，電壓增益 A_v = 1 + Rf/R1 = 6**

**延伸思考**：非反相放大器的最小增益為 1（當 Rf = 0 或 R1 → ∞ 時），稱為電壓隨耦器（Voltage Follower）。其輸入阻抗趨近無限大，是其比反相放大器的優勢。

### 題目 18：加法器（Summing Amplifier）

**來源**：台大電機 108-1 期中考風格
**難度**：★★★☆☆
**主題**：運算放大器 — 反相加法器

**題目**：

如圖所示反相加法器（Summing Amplifier），R1 = 10 kΩ，R2 = 5 kΩ，R3 = 20 kΩ，Rf = 40 kΩ。輸入電壓 v₁ = 1 V，v₂ = −2 V，v₃ = 0.5 V。求輸出電壓 v_o。

```
              Rf = 40kΩ
    ┌────────────┤────────────┐
    │                          │
v₁ ──R1=10kΩ──┐               │
               │   ┌──┐       │
v₂ ──R2=5kΩ───┤(-)    ├──┬── v_o
               │ Op-Amp │
v₃ ──R3=20kΩ──┘   │       │
              ┌───┤(+)    ├
              │   └──┘
             GND
```

**詳解**：

**解題思路**：反相加法器的輸出是各輸入的加權反相疊加。利用虛短路得 V⁻ = 0，再由虛斷路列 KCL 方程式。

**步驟**：

1. 虛短路：V⁻ = V⁺ = 0 V

2. 虛斷路：在 (−) 節點列 KCL（流入 = 流出）：
   I₁ + I₂ + I₃ = I_f

   (v₁ − 0)/R1 + (v₂ − 0)/R2 + (v₃ − 0)/R3 = (0 − v_o)/Rf

3. 代入數值：
   1/10k + (−2)/5k + 0.5/20k = −v_o/40k

   0.1m + (−0.4m) + 0.025m = −v_o/40k

   −0.275m = −v_o/40k

   v_o = 0.275m × 40k = 11 V

4. 通用公式驗證：
   v_o = −Rf × (v₁/R1 + v₂/R2 + v₃/R3)
   = −40k × (1/10k + (−2)/5k + 0.5/20k)
   = −40k × (0.1m − 0.4m + 0.025m)
   = −40k × (−0.275m)
   = 11 V ✓

**答案**：

> **v_o = 11 V**

**延伸思考**：若所有輸入電阻相等（R1 = R2 = R3 = R），則 v_o = −(Rf/R)(v₁ + v₂ + v₃)，成為等權加法器。加法器在類比計算機和 DAC（數位類比轉換器）中有廣泛應用。

### 題目 19：差動放大器（Difference Amplifier）

**來源**：台大電機 110-1 期中考風格
**難度**：★★★☆☆
**主題**：運算放大器 — 差動放大器

**題目**：

如圖所示差動放大器，R1 = R3 = 10 kΩ，R2 = Rf = 50 kΩ。輸入 v₁ = 3 V 接在 (−) 端側 R1，v₂ = 1 V 接在 (+) 端側 R3。求輸出電壓 v_o。

```
              Rf = 50kΩ
    ┌────────────┤────────────┐
    │                          │
v₁ ──R1=10kΩ──┤(-)    ┌──┐   │
               │ Op-Amp ├──┬── v_o
v₂ ──R3=10kΩ──┤(+)    └──┘
               │
             R2=50kΩ
               │
              GND
```

**詳解**：

**解題思路**：差動放大器的輸出與兩輸入的差成正比。先用分壓求 V⁺，再由虛短路條件列方程式。

**步驟**：

1. 求 V⁺（非反相輸入端電壓）：
   V⁺ = v₂ × R2 / (R3 + R2) = 1 × 50k / (10k + 50k) = 50/60 = 5/6 V

2. 虛短路：V⁻ = V⁺ = 5/6 V

3. 在 (−) 端列 KCL：
   (v₁ − V⁻)/R1 = (V⁻ − v_o)/Rf

   (3 − 5/6)/10k = (5/6 − v_o)/50k

   (13/6)/10k = (5/6 − v_o)/50k

   交叉相乘：
   50k × (13/6) = 10k × (5/6 − v_o)

   5 × 13/6 = 5/6 − v_o

   65/6 = 5/6 − v_o

   v_o = 5/6 − 65/6 = −60/6 = −10 V

4. 通用公式驗證（當 R2/R3 = Rf/R1 時）：
   A_d = Rf/R1 = 50k/10k = 5
   v_o = A_d × (v₂ − v₁) = 5 × (1 − 3) = 5 × (−2) = −10 V ✓

**答案**：

> **v_o = −10 V，差動增益 A_d = Rf/R1 = 5**

**延伸思考**：差動放大器要有良好的共模拒斥比（CMRR），需要 Rf/R1 = R2/R3 精確匹配。若不匹配，共模信號會洩漏到輸出。儀表放大器（Instrumentation Amplifier）用三個 Op-Amp 來解決匹配問題。

### 題目 20：積分器與微分器（Integrator and Differentiator）

**來源**：經典題型
**難度**：★★★☆☆
**主題**：運算放大器 — 積分器

**題目**：

如圖所示為一理想反相積分器（Integrator），R = 10 kΩ，C = 0.1 μF。輸入為一方波信號：v_s(t) = 2 V（常數，於 t = 0 時施加）。假設電容初始電壓為零，求 t = 1 ms 時的輸出電壓 v_o(t)。

```
            C = 0.1μF
    ┌──────── ┤├ ────────┐
    │                      │
    │     ┌──┐             │
v_s ── R=10kΩ ──┤(-)    ├──┬── v_o
                │ Op-Amp │
           ┌───┤(+)    ├
           │   └──┘
          GND
```

**詳解**：

**解題思路**：反相積分器用電容取代回授電阻 Rf。由虛短路和虛斷路推導輸出為輸入的積分。

**步驟**：

1. 虛短路：V⁻ = V⁺ = 0 V

2. 虛斷路：通過 R 的電流 = 通過 C 的電流
   i_R = (v_s − 0) / R = v_s / R

3. 電容電流-電壓關係：
   i_C = C × d(0 − v_o)/dt = −C × dv_o/dt

4. i_R = i_C：
   v_s / R = −C × dv_o/dt
   dv_o/dt = −v_s / (RC)

5. 積分得：
   v_o(t) = v_o(0) − (1/RC) ∫₀ᵗ v_s(τ) dτ

6. 代入數值：
   RC = 10k × 0.1μ = 10⁴ × 10⁻⁷ = 10⁻³ = 1 ms
   1/RC = 1000 s⁻¹

   v_o(t) = 0 − 1000 × ∫₀ᵗ 2 dτ = −2000t

7. 在 t = 1 ms = 10⁻³ s：
   v_o(1 ms) = −2000 × 10⁻³ = −2 V

**答案**：

> **v_o(1 ms) = −2 V，通用公式：v_o(t) = −(1/RC) ∫₀ᵗ v_s(τ) dτ + v_o(0)**

**延伸思考**：實際積分器會加上一個大電阻並聯電容以避免直流漂移（DC drift）。反相微分器則將 R 與 C 的位置對調：v_o(t) = −RC × dv_s/dt，但對高頻雜訊敏感，實務中較少單獨使用。

### 題目 21：RC 自然響應（Natural Response）

**來源**：經典題型
**難度**：★★☆☆☆
**主題**：一階電路 — RC 自然響應

**題目**：

下圖 RC 電路中，開關 SW 在 t = 0 時從位置 a 切換到位置 b。已知 R = 5 kΩ，C = 2 μF，電容在 t = 0⁻ 時已充電至 V₀ = 20 V。求 t > 0 時電容電壓 v_C(t) 及電阻電流 i_R(t)。求 v_C(t) 降至初始值的 37% 所需時間。

```
     a      SW(t=0)    b
  ┌──●────/────●──────────┐
  │                        │
(+)                       R = 5kΩ
 V₀=20V                   │
(初始充電)               C = 2μF
  │                        │
  └────────────────────────┘
```

t < 0 時 SW 在 a（電容已充電），t = 0 時切到 b（電容通過 R 放電）。

**詳解**：

**解題思路**：RC 自然響應：電容透過電阻放電，無外部電源。通解為指數衰減函數，時間常數 τ = RC。

**步驟**：

1. 時間常數：
   τ = RC = 5k × 2μ = 5 × 10³ × 2 × 10⁻⁶ = 10 × 10⁻³ = 10 ms

2. RC 自然響應公式：
   v_C(t) = V₀ × e^(−t/τ) = 20 × e^(−t/0.01)    （t 的單位為秒）
   v_C(t) = 20e^(−100t) V，t ≥ 0

3. 電阻電流（由電容放電提供，注意方向）：
   i_R(t) = v_C(t) / R = 20e^(−100t) / 5000 = 4 × 10⁻³ × e^(−100t)
   i_R(t) = 4e^(−100t) mA，t ≥ 0

4. v_C 降至 37% 的時間：
   37% ≈ 1/e ≈ e⁻¹
   v_C(t₁) = 20 × e⁻¹ = 20 × 0.368 = 7.36 V
   此時 t₁ = τ = 10 ms

5. 驗證：v_C(10 ms) = 20 × e^(−100×0.01) = 20 × e⁻¹ = 7.36 V ✓

**答案**：

> **v_C(t) = 20e^(−100t) V，i_R(t) = 4e^(−100t) mA，降至 37% 所需時間 = τ = 10 ms**

**延伸思考**：經過 5τ 後，響應降至初始值的 e⁻⁵ ≈ 0.67%，工程上常視為「完全放電」。時間常數 τ 的物理意義是指數函數在 t = 0 處切線與時間軸交點的時間。

### 題目 22：RL 自然響應（Natural Response）

**來源**：台大電機 107-2 期中考風格
**難度**：★★☆☆☆
**主題**：一階電路 — RL 自然響應

**題目**：

下圖 RL 電路中，t < 0 時開關 SW 閉合，電感電流已達穩態。t = 0 時 SW 打開，電感通過 R₂ 放電。已知 V_s = 24 V，R₁ = 8 Ω，R₂ = 12 Ω，L = 60 mH。求 t > 0 時電感電流 i_L(t) 及 R₂ 上的電壓 v_R₂(t)。

```
       SW(t=0)
  ┌──/──── R₁=8Ω ────┐
  │                    │
(+)                    ├──── R₂=12Ω ────┐
 V_s=24V               │                 │
(-)                    L=60mH            │
  │                    │                 │
  └────────────────────┴─────────────────┘
```

t < 0：SW 閉合，V_s 通過 R₁ 供電，L 穩態。t = 0：SW 打開，L 通過 R₂ 放電。

**詳解**：

**解題思路**：先求 t = 0⁻ 的電感電流（初始條件），再用 RL 自然響應公式求 t > 0 的行為。

**步驟**：

1. 初始條件（t = 0⁻，穩態）：
   電感在直流穩態下等效為短路。
   i_L(0⁻) = V_s / R₁ = 24 / 8 = 3 A
   （穩態時 R₂ 被短路，因為 L 的兩端電壓為零，R₂ 與 L 並聯，L 等效短路。）

   等一下，需要確認電路拓撲。R₂ 與 L 並聯。穩態時 L 短路，所以所有電流走 L 支路。
   i_L(0⁻) = V_s / R₁ = 24 / 8 = 3 A ✓

2. 電感電流連續性：i_L(0⁺) = i_L(0⁻) = 3 A

3. t > 0 時 SW 打開，V_s 和 R₁ 斷開。電感與 R₂ 構成迴路。
   時間常數：τ = L / R₂ = 60m / 12 = 5 ms

4. RL 自然響應：
   i_L(t) = I₀ × e^(−t/τ) = 3 × e^(−t/0.005) = 3e^(−200t) A，t ≥ 0

5. R₂ 上的電壓（注意極性，電流從上往下流過 R₂）：
   v_R₂(t) = −i_L(t) × R₂ = −3 × 12 × e^(−200t) = −36e^(−200t) V
   （負號因為電感釋放能量時，R₂ 上的電壓極性與原供電方向相反。）

6. 驗證初始值：
   v_R₂(0⁺) = −36 V
   KVL 在 t = 0⁺：v_L(0⁺) + v_R₂(0⁺) = 0
   v_L(0⁺) = L × di_L/dt|₀₊ = 60m × (−200)(3) = −36 V
   v_L(0⁺) + v_R₂(0⁺) = −36 + (−36)... 不對。

   重新檢查：迴路中 L 和 R₂ 構成封閉迴路。KVL：v_L + v_R₂ = 0。
   v_R₂ = i_L × R₂ = 3e^(−200t) × 12 = 36e^(−200t) V
   v_L = −v_R₂ = −36e^(−200t) V
   v_L = L × di_L/dt = 60m × 3 × (−200) × e^(−200t) = −36e^(−200t) V ✓

**答案**：

> **i_L(t) = 3e^(−200t) A，v_R₂(t) = 36e^(−200t) V，τ = 5 ms**

**延伸思考**：電感的「電流連續性」是一階 RL 電路分析的關鍵。電感不允許電流瞬間改變（否則需要無限大電壓）。開關打開瞬間，電感可能在電阻上產生很大的電壓，這在實際電路中可能造成電弧放電。

### 題目 23：RC 步階響應（Step Response）

**來源**：台大電機 109-1 期中考風格
**難度**：★★★☆☆
**主題**：一階電路 — RC 步階響應

**題目**：

如下圖所示，開關 SW 在 t = 0 時閉合。已知 R = 20 kΩ，C = 5 μF，V_s = 10 V，電容初始電壓 v_C(0⁻) = 2 V。求 t > 0 時 v_C(t)。

```
       SW(t=0)    R = 20kΩ
  ┌──/──────────── ┤ ──────┐
  │                         │
(+)                       C = 5μF
V_s=10V                     │ v_C(0⁻)=2V
(-)                         │
  │                         │
  └─────────────────────────┘
```

**詳解**：

**解題思路**：一階電路步階響應通用公式：x(t) = x(∞) + [x(0⁺) − x(∞)] × e^(−t/τ)。需要三個關鍵量：初始值 x(0⁺)、最終值 x(∞)、時間常數 τ。

**步驟**：

1. 初始值：
   v_C(0⁺) = v_C(0⁻) = 2 V（電容電壓連續性）

2. 最終值（t → ∞，穩態）：
   電容充電完成後等效開路，無電流流過 R。
   v_C(∞) = V_s = 10 V

3. 時間常數：
   τ = RC = 20k × 5μ = 20 × 10³ × 5 × 10⁻⁶ = 0.1 s = 100 ms

4. 代入通用公式：
   v_C(t) = v_C(∞) + [v_C(0⁺) − v_C(∞)] × e^(−t/τ)
   v_C(t) = 10 + (2 − 10) × e^(−t/0.1)
   v_C(t) = 10 − 8e^(−10t) V，t ≥ 0

5. 驗證：
   - t = 0：v_C(0) = 10 − 8e⁰ = 10 − 8 = 2 V ✓
   - t → ∞：v_C(∞) = 10 − 8 × 0 = 10 V ✓
   - t = τ = 0.1 s：v_C = 10 − 8e⁻¹ = 10 − 2.943 = 7.057 V（已充至 63.2% 的變化量）

**答案**：

> **v_C(t) = 10 − 8e^(−10t) V，t ≥ 0**

**延伸思考**：通用公式 x(t) = x(∞) + [x(0) − x(∞)]e^(−t/τ) 適用於任何一階電路的步階響應，其中 x 可以是電容電壓或電感電流。解題關鍵三步驟：(1) 找初始值 (2) 找最終值 (3) 找時間常數。

### 題目 24：RL 步階響應搭配多電阻

**來源**：台大電機 110-1 期中考風格
**難度**：★★★☆☆
**主題**：一階電路 — RL 步階響應

**題目**：

如下圖，t < 0 時開關 SW 打開，電路已達穩態，電感電流為零。t = 0 時 SW 閉合。已知 V_s = 30 V，R₁ = 6 Ω，R₂ = 12 Ω，R₃ = 4 Ω，L = 0.2 H。求 t > 0 時的電感電流 i_L(t)。

```
       SW(t=0)    R₁=6Ω
  ┌──/──────────── ┤ ──────┬─────────┐
  │                         │         │
(+)                       R₂=12Ω    R₃=4Ω
V_s=30V                    │      ┌──┤──┐
(-)                        │      L=0.2H
  │                         │      │
  └─────────────────────────┴──────┘
```

R₂ 與（R₃ 串聯 L）並聯，整組再與 R₁ 串聯，接到 V_s。

**詳解**：

**解題思路**：先求初始值、最終值和戴維寧等效電阻（從電感端看入），再套用通用公式。

**步驟**：

1. 初始條件（t = 0⁺）：
   i_L(0⁺) = i_L(0⁻) = 0 A（電感電流連續，且 t < 0 時 SW 打開，無電流）

2. 最終值（t → ∞，穩態）：
   電感在直流穩態等效短路。R₃ 與 L 串聯，L 短路後只剩 R₃ = 4 Ω。
   R₂ ∥ R₃ = (12 × 4)/(12 + 4) = 48/16 = 3 Ω
   總電流：I_total = V_s / (R₁ + R₂∥R₃) = 30 / (6 + 3) = 30/9 = 10/3 A
   電流分配（流過 R₃-L 支路）：
   i_L(∞) = I_total × R₂/(R₂ + R₃) = (10/3) × 12/(12 + 4) = (10/3) × (3/4) = 10/4 = 2.5 A

3. 時間常數：
   從電感端看入的戴維寧等效電阻（V_s 短路）：
   R_Th = R₃ + R₁∥R₂ = 4 + (6 × 12)/(6 + 12) = 4 + 4 = 8 Ω
   τ = L / R_Th = 0.2 / 8 = 0.025 s = 25 ms

4. 代入通用公式：
   i_L(t) = i_L(∞) + [i_L(0⁺) − i_L(∞)] × e^(−t/τ)
   i_L(t) = 2.5 + (0 − 2.5) × e^(−t/0.025)
   i_L(t) = 2.5(1 − e^(−40t)) A，t ≥ 0

5. 驗證：
   - t = 0：i_L(0) = 2.5(1 − 1) = 0 A ✓
   - t → ∞：i_L(∞) = 2.5(1 − 0) = 2.5 A ✓

**答案**：

> **i_L(t) = 2.5(1 − e^(−40t)) A，t ≥ 0，τ = 25 ms**

**延伸思考**：求時間常數時，要從儲能元件（電感或電容）的端點看入，求戴維寧等效電阻。τ = L/R_Th（RL 電路）或 τ = R_Th × C（RC 電路）。不要將不相關的電阻納入計算。

### 題目 25：一階電路的切換問題

**來源**：台大電機 108-2 期中考風格
**難度**：★★★★☆
**主題**：一階電路 — 分段求解

**題目**：

下圖電路中有兩個開關。SW1 在 t = 0 時閉合，SW2 在 t = 50 ms 時閉合。已知 R₁ = 10 kΩ，R₂ = 10 kΩ，C = 5 μF，V_s = 20 V，v_C(0⁻) = 0 V。求 v_C(t) 在 0 ≤ t < 50 ms 及 t ≥ 50 ms 兩個區間的表達式。

```
       SW1(t=0)    R₁=10kΩ           SW2(t=50ms)
  ┌──/──────────── ┤ ────────┬──────/──── R₂=10kΩ ──┐
  │                           │                        │
(+)                         C=5μF                     │
V_s=20V                      │                        │
(-)                          │                        │
  │                           │                        │
  └───────────────────────────┴────────────────────────┘
```

**詳解**：

**解題思路**：此題需分段求解。第一段（0 ≤ t < 50 ms）只有 SW1 閉合，電容通過 R₁ 充電。第二段（t ≥ 50 ms）SW2 也閉合，等效電阻改變，電容繼續充電。

**步驟**：

**第一段：0 ≤ t < 50 ms（僅 SW1 閉合）**

1. 初始值：v_C(0⁺) = 0 V
2. 最終值（若此狀態持續到 ∞）：v_C₁(∞) = V_s = 20 V
3. 時間常數：τ₁ = R₁ × C = 10k × 5μ = 50 ms
4. v_C(t) = 20(1 − e^(−t/0.05)) V，0 ≤ t < 50 ms

**求 t = 50 ms 時的電容電壓（作為第二段的初始條件）：**

5. v_C(50 ms) = 20(1 − e^(−50m/50m)) = 20(1 − e⁻¹) = 20 × 0.6321 = 12.642 V

**第二段：t ≥ 50 ms（SW1 和 SW2 都閉合）**

6. 新初始值：v_C(50ms⁺) = 12.642 V

7. 新最終值：SW2 閉合後 R₂ 與 C 並聯，但最終電容仍充到 V_s。
   v_C₂(∞) = V_s = 20 V

8. 新等效電阻（從電容端看入，V_s 短路）：
   R_eq = R₁ ∥ R₂ = (10k × 10k)/(10k + 10k) = 5 kΩ

9. 新時間常數：τ₂ = R_eq × C = 5k × 5μ = 25 ms

10. 令 t' = t − 50 ms（時間平移）：
    v_C(t) = 20 + (12.642 − 20) × e^(−t'/0.025)
    v_C(t) = 20 − 7.358 × e^(−(t−0.05)/0.025) V，t ≥ 50 ms

11. 驗證連續性：
    t = 50 ms⁺：v_C = 20 − 7.358 × e⁰ = 20 − 7.358 = 12.642 V ✓

**答案**：

> **v_C(t) = 20(1 − e^(−20t)) V，0 ≤ t < 50 ms**
> **v_C(t) = 20 − 7.358 × e^(−40(t−0.05)) V，t ≥ 50 ms**

**延伸思考**：分段求解的關鍵是每段結束時的電容電壓（或電感電流）作為下一段的初始條件。實務中時序切換（Sequential Switching）常見於數位電路與電源管理。注意 τ₂ < τ₁，表示第二段充電更快（等效電阻減半）。

### 題目 26：RLC 串聯自然響應 — 過阻尼（Overdamped）

**來源**：經典題型
**難度**：★★★☆☆
**主題**：二階電路 — RLC 串聯自然響應

**題目**：

RLC 串聯電路中，R = 7 Ω，L = 1 H，C = 1/6 F。t = 0 時電容初始電壓 v_C(0) = 10 V，電感初始電流 i(0) = 0 A。求 t > 0 時的電流 i(t)。

```
    ┌── R=7Ω ── L=1H ──┐
    │                    │
    │                  C=1/6 F
    │                  v_C(0)=10V
    │                    │
    └────────────────────┘
```

**詳解**：

**解題思路**：列出二階 ODE，求特徵方程的根判斷阻尼類型。RLC 串聯的特徵方程為 s² + (R/L)s + 1/(LC) = 0。

**步驟**：

1. 求特徵方程係數：
   α = R/(2L) = 7/(2×1) = 3.5 Neper/s
   ω₀ = 1/√(LC) = 1/√(1 × 1/6) = √6 ≈ 2.449 rad/s

2. 比較 α 與 ω₀：
   α = 3.5 > ω₀ = 2.449 → **過阻尼（Overdamped）**

3. 特徵方程：s² + 7s + 6 = 0
   s₁ = −α + √(α² − ω₀²) = −3.5 + √(12.25 − 6) = −3.5 + 2.5 = −1
   s₂ = −α − √(α² − ω₀²) = −3.5 − 2.5 = −6

   驗證：(s + 1)(s + 6) = s² + 7s + 6 ✓

4. 通解：
   i(t) = A₁e^(s₁t) + A₂e^(s₂t) = A₁e^(−t) + A₂e^(−6t)

5. 由初始條件求 A₁、A₂：
   條件 1：i(0) = 0
   A₁ + A₂ = 0 → A₁ = −A₂ ... (1)

   條件 2：di/dt|_{t=0}。由 KVL：L·di/dt + R·i + v_C = 0
   在 t = 0⁺：1 × di/dt(0⁺) + 7 × 0 + 10 = 0
   di/dt(0⁺) = −10 A/s

   di/dt = −A₁e^(−t) − 6A₂e^(−6t)
   di/dt(0) = −A₁ − 6A₂ = −10 ... (2)

6. 由 (1) A₁ = −A₂ 代入 (2)：
   −(−A₂) − 6A₂ = −10
   A₂ − 6A₂ = −10
   −5A₂ = −10
   A₂ = 2，A₁ = −2

7. 最終結果：
   i(t) = −2e^(−t) + 2e^(−6t) = 2(e^(−6t) − e^(−t)) A，t ≥ 0

8. 驗證：
   i(0) = 2(1 − 1) = 0 ✓
   di/dt(0) = 2(−6 − (−1)) = 2(−5) = −10 A/s ✓

**答案**：

> **i(t) = 2(e^(−6t) − e^(−t)) A，t ≥ 0（過阻尼響應）**

**延伸思考**：過阻尼響應沒有振盪，由兩個不同速率的指數衰減項組成。α² > ω₀² 對應兩個相異負實數根。

### 題目 27：RLC 串聯自然響應 — 臨界阻尼（Critically Damped）

**來源**：經典題型
**難度**：★★★☆☆
**主題**：二階電路 — 臨界阻尼

**題目**：

RLC 串聯電路中，R = 4 Ω，L = 1 H，C = 1/4 F。初始條件：v_C(0) = 0 V，i(0) = 4 A（電感已有電流）。求 t > 0 時的電流 i(t)。

**詳解**：

**解題思路**：判斷阻尼類型，臨界阻尼時特徵方程有重根。

**步驟**：

1. 求特徵方程係數：
   α = R/(2L) = 4/(2×1) = 2 Neper/s
   ω₀ = 1/√(LC) = 1/√(1 × 1/4) = 1/(1/2) = 2 rad/s

2. 比較 α 與 ω₀：
   α = 2 = ω₀ = 2 → **臨界阻尼（Critically Damped）**

3. 特徵方程：s² + 4s + 4 = 0 → (s + 2)² = 0
   重根 s₁ = s₂ = −2

4. 通解（臨界阻尼形式）：
   i(t) = (A₁ + A₂t)e^(−2t)

5. 由初始條件求 A₁、A₂：
   條件 1：i(0) = 4
   A₁ = 4 ... (1)

   條件 2：di/dt|_{t=0}。
   KVL 在 t = 0⁺：L·di/dt(0⁺) + R·i(0) + v_C(0) = 0
   1 × di/dt(0⁺) + 4 × 4 + 0 = 0
   di/dt(0⁺) = −16 A/s

   di/dt = A₂e^(−2t) + (A₁ + A₂t)(−2)e^(−2t) = [A₂ − 2(A₁ + A₂t)]e^(−2t)
   di/dt(0) = A₂ − 2A₁ = −16
   A₂ = −16 + 2(4) = −8 ... (2)

6. 最終結果：
   i(t) = (4 − 8t)e^(−2t) A，t ≥ 0

7. 驗證：
   i(0) = (4 − 0) × 1 = 4 A ✓
   di/dt(0) = (−8) − 2(4) = −16 A/s ✓，
   但用推導：di/dt(0) = A₂ − 2A₁ = −8 − 8 = −16 ✓

**答案**：

> **i(t) = (4 − 8t)e^(−2t) A，t ≥ 0（臨界阻尼響應）**

**延伸思考**：臨界阻尼是響應回到零最快的非振盪情形，在阻尼器設計（如汽車避震器、精密儀表）中被廣泛追求。略微降低 R 就會進入欠阻尼（振盪），略微增加 R 則為過阻尼（較慢衰減）。

### 題目 28：RLC 串聯自然響應 — 欠阻尼（Underdamped）

**來源**：台大電機 109-2 期中考風格
**難度**：★★★★☆
**主題**：二階電路 — 欠阻尼

**題目**：

RLC 串聯電路中，R = 2 Ω，L = 1 H，C = 1/5 F。初始條件：v_C(0) = 10 V，i(0) = 0 A。求 t > 0 時的電容電壓 v_C(t)。

**詳解**：

**解題思路**：判斷為欠阻尼後，通解含衰減正弦項。先求 i(t)，再由 i = C·dv_C/dt 積分或直接對 v_C 列二階 ODE。

**步驟**：

1. 特徵方程係數：
   α = R/(2L) = 2/(2×1) = 1 Neper/s
   ω₀ = 1/√(LC) = 1/√(1 × 1/5) = √5 ≈ 2.236 rad/s

2. α = 1 < ω₀ = 2.236 → **欠阻尼（Underdamped）**

3. 阻尼振盪頻率：
   ω_d = √(ω₀² − α²) = √(5 − 1) = 2 rad/s

4. 特徵根：s₁,₂ = −α ± jω_d = −1 ± j2

5. 對 v_C 的通解（v_C 的特徵方程與 i 相同）：
   v_C(t) = e^(−αt)[B₁cos(ω_d·t) + B₂sin(ω_d·t)]
   v_C(t) = e^(−t)[B₁cos(2t) + B₂sin(2t)]

6. 初始條件：
   條件 1：v_C(0) = 10
   B₁ = 10 ... (1)

   條件 2：dv_C/dt|_{t=0} = i(0)/C = 0/(1/5) = 0

   dv_C/dt = e^(−t)[−2B₁sin(2t) + 2B₂cos(2t)] + (−1)e^(−t)[B₁cos(2t) + B₂sin(2t)]
   dv_C/dt(0) = 2B₂ − B₁ = 0
   B₂ = B₁/2 = 5 ... (2)

7. 最終結果：
   v_C(t) = e^(−t)[10cos(2t) + 5sin(2t)] V，t ≥ 0

   也可寫為極座標形式：
   振幅 = √(10² + 5²) = √125 = 5√5 ≈ 11.18 V
   相位 = arctan(5/10) = arctan(0.5) ≈ 26.57°
   v_C(t) = 5√5 × e^(−t) × cos(2t − 26.57°) V

8. 驗證：
   v_C(0) = 10 × 1 + 5 × 0 = 10 V ✓
   dv_C/dt(0) = 2(5) − 10 = 0 ✓

**答案**：

> **v_C(t) = e^(−t)[10cos(2t) + 5sin(2t)] V，t ≥ 0（欠阻尼響應，ω_d = 2 rad/s）**

**延伸思考**：欠阻尼響應會產生振盪（Oscillation），振幅以 e^(−αt) 的速率衰減。振盪頻率為 ω_d（非 ω₀），且 ω_d < ω₀。當 R → 0（α → 0）時趨近無阻尼振盪（LC 諧振）。

### 題目 29：RLC 並聯自然響應

**來源**：台大電機 110-2 期中考風格
**難度**：★★★★☆
**主題**：二階電路 — RLC 並聯

**題目**：

RLC 並聯電路中，R = 5 Ω，L = 1 H，C = 1/20 F。初始條件：v(0) = 0 V（電容電壓），i_L(0) = 2 A（電感電流）。求 t > 0 時端電壓 v(t)。

```
    ┌──────┬──────┬──────┐
    │      │      │      │
    │     R=5Ω  L=1H   C=1/20F
    │      │    i_L(0)   │
    │      │    =2A     v(0)=0V
    │      │      │      │
    └──────┴──────┴──────┘
```

**詳解**：

**解題思路**：RLC 並聯電路的特徵方程為 s² + s/(RC) + 1/(LC) = 0。注意並聯的 α 公式與串聯不同。

**步驟**：

1. 特徵方程係數（並聯 RLC）：
   α = 1/(2RC) = 1/(2 × 5 × 1/20) = 1/(1/2) = 2 Neper/s
   ω₀ = 1/√(LC) = 1/√(1 × 1/20) = √20 = 2√5 ≈ 4.472 rad/s

2. α = 2 < ω₀ = 4.472 → **欠阻尼**

3. 阻尼振盪頻率：
   ω_d = √(ω₀² − α²) = √(20 − 4) = 4 rad/s

4. 通解：
   v(t) = e^(−2t)[A₁cos(4t) + A₂sin(4t)]

5. 初始條件：
   條件 1：v(0) = 0
   A₁ = 0 ... (1)

   條件 2：dv/dt|_{t=0}。由 KCL 在頂部節點：
   C·dv/dt + v/R + i_L = 0
   dv/dt(0) = −v(0)/(RC) − i_L(0)/C = −0/(5×1/20) − 2/(1/20) = 0 − 40 = −40 V/s

   dv/dt = e^(−2t)[−4A₁sin(4t) + 4A₂cos(4t)] + (−2)e^(−2t)[A₁cos(4t) + A₂sin(4t)]
   dv/dt(0) = 4A₂ − 2A₁ = 4A₂ − 0 = −40
   A₂ = −10 ... (2)

6. 最終結果：
   v(t) = −10e^(−2t)sin(4t) V，t ≥ 0

7. 驗證：
   v(0) = −10 × 1 × 0 = 0 V ✓
   dv/dt(0) = 4(−10) − 2(0) = −40 V/s ✓

**答案**：

> **v(t) = −10e^(−2t)sin(4t) V，t ≥ 0（欠阻尼，ω_d = 4 rad/s）**

**延伸思考**：RLC 並聯與串聯的 α 公式不同：串聯 α = R/(2L)，並聯 α = 1/(2RC)。串聯中增大 R 使阻尼增大，並聯中增大 R 使阻尼減小——物理意義上，並聯電阻越大，能量散逸越少，振盪越持久。

### 題目 30：RLC 串聯步階響應

**來源**：台大電機 108-2 期中考風格
**難度**：★★★★☆
**主題**：二階電路 — RLC 步階響應

**題目**：

RLC 串聯電路受到直流步階電壓 V_s = 24 V 驅動。R = 10 Ω，L = 1 H，C = 1/25 F。初始條件：v_C(0) = 0 V，i(0) = 0 A。求 t > 0 時的電容電壓 v_C(t)。

```
       SW(t=0)   R=10Ω     L=1H
  ┌──/────────── ┤ ────── ┤ ──────┐
  │                                │
(+)                              C=1/25F
V_s=24V                            │
(-)                                │
  │                                │
  └────────────────────────────────┘
```

**詳解**：

**解題思路**：二階步階響應 = 穩態響應 + 暫態響應。先求特徵方程判斷阻尼類型，暫態部分與自然響應形式相同，穩態部分為最終值。

**步驟**：

1. 特徵方程係數：
   α = R/(2L) = 10/(2×1) = 5 Neper/s
   ω₀ = 1/√(LC) = 1/√(1 × 1/25) = 5 rad/s

2. α = 5 = ω₀ = 5 → **臨界阻尼**

3. 特徵根：s₁ = s₂ = −5（重根）

4. 穩態值：v_C(∞) = V_s = 24 V（電容充電至電源電壓）

5. 完全解（步階響應 = 穩態 + 暫態）：
   v_C(t) = 24 + (D₁ + D₂t)e^(−5t)

6. 初始條件求 D₁、D₂：
   條件 1：v_C(0) = 0
   24 + D₁ = 0 → D₁ = −24 ... (1)

   條件 2：dv_C/dt|_{t=0} = i(0)/C = 0/(1/25) = 0

   dv_C/dt = D₂e^(−5t) + (D₁ + D₂t)(−5)e^(−5t)
   dv_C/dt(0) = D₂ − 5D₁ = 0
   D₂ = 5D₁ = 5(−24) = −120 ... (2)

7. 最終結果：
   v_C(t) = 24 + (−24 − 120t)e^(−5t)
   v_C(t) = 24 − (24 + 120t)e^(−5t) V，t ≥ 0

8. 驗證：
   v_C(0) = 24 − 24 = 0 V ✓
   v_C(∞) = 24 − 0 = 24 V ✓
   dv_C/dt(0) = −120 − 5(−24) = −120 + 120 = 0 ✓

**答案**：

> **v_C(t) = 24 − (24 + 120t)e^(−5t) V，t ≥ 0（臨界阻尼步階響應）**

**延伸思考**：二階步階響應中，欠阻尼會產生超越量（Overshoot），即 v_C 暫時超過最終值。臨界阻尼和過阻尼則不會超越。步階響應的通解形式為 x(t) = x_f + x_n(t)，其中 x_f 為最終值，x_n(t) 為對應的自然響應形式。

### 題目 31：相量轉換與阻抗計算

**來源**：經典題型
**難度**：★★☆☆☆
**主題**：交流穩態 — 相量與阻抗

**題目**：

電路操作頻率 ω = 1000 rad/s。R = 4 Ω，L = 3 mH，C = 250 μF。求：
(a) 電阻、電感、電容各自的阻抗（Impedance）
(b) 三者串聯的總阻抗 Z_total
(c) 若施加電壓 v(t) = 10cos(1000t + 30°) V，求穩態電流 i(t)

**詳解**：

**解題思路**：將時域元件轉換為頻域（相量域）阻抗，用歐姆定律的相量版本求解。

**步驟**：

(a) 各元件阻抗：
1. Z_R = R = 4 Ω
2. Z_L = jωL = j × 1000 × 3 × 10⁻³ = j3 Ω
3. Z_C = 1/(jωC) = 1/(j × 1000 × 250 × 10⁻⁶) = 1/(j0.25) = −j4 Ω

(b) 串聯總阻抗：
4. Z_total = Z_R + Z_L + Z_C = 4 + j3 − j4 = 4 − j1 Ω

5. 極座標形式：
   |Z_total| = √(4² + 1²) = √17 ≈ 4.123 Ω
   ∠Z_total = arctan(−1/4) = −14.04°
   Z_total = 4.123∠−14.04° Ω

(c) 穩態電流：
6. 電壓相量：**V** = 10∠30° V（取振幅相量）

7. 電流相量：**I** = **V** / Z_total = 10∠30° / 4.123∠−14.04°
   = (10/4.123)∠(30° − (−14.04°))
   = 2.425∠44.04° A

8. 轉回時域：
   i(t) = 2.425cos(1000t + 44.04°) A

**答案**：

> **(a) Z_R = 4 Ω，Z_L = j3 Ω，Z_C = −j4 Ω**
> **(b) Z_total = 4 − j1 = 4.123∠−14.04° Ω**
> **(c) i(t) = 2.425cos(1000t + 44.04°) A**

**延伸思考**：Z_total 的虛部為負（容性），表示電流超前電壓。若 ω 增大使 Z_L 增大且 Z_C 減小，電路可能變為感性。當 ωL = 1/(ωC) 時達到串聯諧振，Z_total = R（純電阻）。

### 題目 32：相量域 KVL/KCL 電路分析

**來源**：台大電機 108-2 期末考風格
**難度**：★★★☆☆
**主題**：交流穩態 — 相量域分析

**題目**：

在下圖相量域電路中，求電流相量 **I**。ω = 2 rad/s。

```
         4Ω        j6Ω
    ┌─────┤────────── ┤ ──────┐
    │                          │
  (+)                       −j2Ω
 20∠0° V                      │
  (-)                         │
    │         8Ω              │
    └──────────┤──────────────┘
```

上方支路：4Ω 串聯 j6Ω（電感），下方支路：8Ω。右側：−j2Ω（電容）並聯在兩支路的右端到底部。**I** 為流過 −j2Ω 的電流。

**詳解**：

**解題思路**：先用阻抗合併化簡電路，或用節點電壓法/網目電流法在相量域中求解。

**步驟**：

1. 定義節點：左側為參考（V = 0），右側節點電壓為 **V₁**。

2. 重新解讀：電壓源 20∠0° 在左側。上方支路 (4 + j6) Ω 從左到右。下方支路 8 Ω 從左到右。−j2 Ω 從右節點到底部（即從右節點到左側參考節點）。

3. 所以 −j2 Ω 與 8 Ω 並聯（都連接右節點到左參考節點）。

4. 8 ∥ (−j2) = (8 × (−j2))/(8 + (−j2)) = (−j16)/(8 − j2)

   有理化：(−j16)(8 + j2) / ((8)² + (2)²) = (−j128 − j²32) / 68 = (32 − j128) / 68 = (8 − j32)/17

   8∥(−j2) = (8 − j32)/17 ≈ 0.471 − j1.882 Ω

5. 總阻抗：Z_total = (4 + j6) + (8 − j32)/17 = (68 + j102 + 8 − j32)/17 = (76 + j70)/17

6. 總電流（從電源流出）：
   **I_total** = 20∠0° / Z_total = 20 / ((76 + j70)/17) = 340 / (76 + j70)

   有理化：340(76 − j70) / (76² + 70²) = 340(76 − j70) / (5776 + 4900) = 340(76 − j70) / 10676

   **I_total** = (25840 − j23800) / 10676 = 2.420 − j2.229 A

7. 右節點電壓：
   **V₁** = **I_total** × (8∥(−j2)) = (2.420 − j2.229) × (8 − j32)/17

   先算分子：(2.420 − j2.229)(8 − j32)
   = 19.36 − j77.44 − j17.832 + j²71.328
   = 19.36 − 71.328 − j(77.44 + 17.832)
   = −51.968 − j95.272

   **V₁** = (−51.968 − j95.272)/17 = −3.057 − j5.604 V

8. 流過 −j2 Ω 的電流：
   **I** = **V₁** / (−j2) = (−3.057 − j5.604) / (−j2)

   = (−3.057 − j5.604) × (j2) / ((−j2)(j2)) = (−3.057 − j5.604)(j2) / 4

   分子：(−3.057)(j2) + (−j5.604)(j2) = −j6.114 − j²11.208 = 11.208 − j6.114

   **I** = (11.208 − j6.114) / 4 = 2.802 − j1.529 A

   極座標：|**I**| = √(2.802² + 1.529²) = √(7.851 + 2.338) = √10.189 = 3.192 A
   ∠**I** = arctan(−1.529/2.802) = −28.63°

   **I** = 3.192∠−28.63° A

**答案**：

> ****I** = 3.192∠−28.63° A，即 i(t) = 3.192cos(2t − 28.63°) A**

**延伸思考**：相量域分析將微分方程轉化為代數方程，KVL/KCL/戴維寧等直流分析技巧都可直接套用，只是計算涉及複數運算。

### 題目 33：交流電路的戴維寧等效

**來源**：台大電機 109-2 期末考風格
**難度**：★★★★☆
**主題**：交流穩態 — 相量域戴維寧等效

**題目**：

求下圖交流電路端子 a-b 的戴維寧等效（**V_Th** 和 Z_Th），ω = 4 rad/s。

```
         2Ω       j4Ω
    ┌─────┤──────── ┤ ────── a
    │
  (+)
 12∠0° V          −j1Ω
  (-)               │
    │               │
    └───────────────┴──────── b
```

2Ω 和 j4Ω 串聯從左上到 a。−j1Ω 從中間節點（2Ω 和 j4Ω 之間）到 b。

**詳解**：

**解題思路**：與直流戴維寧方法相同——先求開路電壓 **V_Th**，再關閉獨立源求 Z_Th。

**步驟**：

**求 V_Th（開路電壓 V_ab）：**

1. a-b 開路時，j4Ω 無電流（a 端開路）。
2. 電流迴路：12∠0° → 2Ω → −j1Ω → 回到電源。
   **I** = 12∠0° / (2 + (−j1)) = 12 / (2 − j1)
   有理化：= 12(2 + j1) / (4 + 1) = 12(2 + j1)/5 = (24 + j12)/5 = 4.8 + j2.4 A

3. 中間節點電壓（相對於 b）：
   **V_mid** = **I** × (−j1) = (4.8 + j2.4)(−j) = −j4.8 − j²2.4 = 2.4 − j4.8 V

4. **V_Th** = **V_mid** − j4Ω × 0 = **V_mid** = 2.4 − j4.8 V
   極座標：= √(2.4² + 4.8²)∠arctan(−4.8/2.4) = √(5.76 + 23.04)∠(−63.43°) = √28.8∠−63.43°
   = 5.367∠−63.43° V

**求 Z_Th：**

5. 關閉 12∠0°（短路），從 a-b 看入：
   2Ω 與 −j1Ω 並聯，再與 j4Ω 串聯。

   2 ∥ (−j1) = 2(−j1)/(2 + (−j1)) = (−j2)/(2 − j) = (−j2)(2 + j)/5 = (−j4 − j²2)/5 = (2 − j4)/5
   = 0.4 − j0.8 Ω

   Z_Th = j4 + 0.4 − j0.8 = 0.4 + j3.2 Ω
   極座標：= √(0.16 + 10.24)∠arctan(3.2/0.4) = √10.4∠82.87° = 3.225∠82.87° Ω

**答案**：

> ****V_Th** = 5.367∠−63.43° V，Z_Th = 0.4 + j3.2 = 3.225∠82.87° Ω**

**延伸思考**：在交流電路中，最大平均功率轉移的條件是 Z_L = Z_Th*（共軛匹配），而非簡單的 Z_L = Z_Th。

### 題目 34：平均功率與無效功率計算

**來源**：經典題型
**難度**：★★★☆☆
**主題**：交流穩態 — 交流功率

**題目**：

已知某負載的端電壓 v(t) = 100cos(377t + 20°) V，電流 i(t) = 8cos(377t − 40°) A。求：
(a) 平均功率 P（Average Power / Real Power）
(b) 無效功率 Q（Reactive Power）
(c) 視在功率 |S|（Apparent Power）
(d) 功率因數 pf（Power Factor）

**詳解**：

**解題思路**：利用交流功率公式。相量域中 S = ½**V****I*** = P + jQ。

**步驟**：

(a) 平均功率：

1. 電壓振幅 V_m = 100 V，電流振幅 I_m = 8 A
2. 電壓與電流的相位差：θ_v − θ_i = 20° − (−40°) = 60°
3. P = ½ × V_m × I_m × cos(θ_v − θ_i) = ½ × 100 × 8 × cos(60°)
   = 400 × 0.5 = 200 W

(b) 無效功率：

4. Q = ½ × V_m × I_m × sin(θ_v − θ_i) = ½ × 100 × 8 × sin(60°)
   = 400 × 0.866 = 346.4 VAR

   Q > 0 → 感性負載（電流落後電壓）

(c) 視在功率：

5. |S| = ½ × V_m × I_m = ½ × 100 × 8 = 400 VA

   驗證：|S| = √(P² + Q²) = √(200² + 346.4²) = √(40000 + 120000) = √160000 = 400 VA ✓

(d) 功率因數：

6. pf = cos(60°) = 0.5（落後，Lagging）

   也可以用 pf = P/|S| = 200/400 = 0.5 ✓

**答案**：

> **P = 200 W，Q = 346.4 VAR（感性），|S| = 400 VA，pf = 0.5 lagging**

**延伸思考**：若用有效值（RMS）表示：V_rms = 100/√2 ≈ 70.71 V，I_rms = 8/√2 ≈ 5.657 A。此時 P = V_rms × I_rms × cos(60°) = 70.71 × 5.657 × 0.5 = 200 W，結果一致。工程實務中常用 RMS 值。

### 題目 35：複數功率與功率三角形

**來源**：台大電機 108-1 期末考風格
**難度**：★★★☆☆
**主題**：交流穩態 — 複數功率

**題目**：

一個阻抗為 Z = 3 + j4 Ω 的負載，接上有效值為 V_rms = 20 V 的電壓源。求負載吸收的複數功率 **S**、平均功率 P、無效功率 Q 及功率因數。

**詳解**：

**解題思路**：先求電流有效值，再用 **S** = V_rms × I_rms* 或 **S** = |I_rms|² × Z 計算。

**步驟**：

1. 阻抗大小：|Z| = √(3² + 4²) = 5 Ω

2. 電流有效值：I_rms = V_rms / |Z| = 20 / 5 = 4 A

3. 複數功率（用 **S** = |I_rms|² × Z）：
   **S** = 4² × (3 + j4) = 16 × (3 + j4) = 48 + j64 VA

4. 平均功率：P = Re{**S**} = 48 W

   驗證：P = I_rms² × R = 16 × 3 = 48 W ✓

5. 無效功率：Q = Im{**S**} = 64 VAR

   驗證：Q = I_rms² × X = 16 × 4 = 64 VAR ✓

6. 視在功率：|**S**| = √(48² + 64²) = √(2304 + 4096) = √6400 = 80 VA

   驗證：|**S**| = V_rms × I_rms = 20 × 4 = 80 VA ✓

7. 功率因數：pf = P / |**S**| = 48/80 = 0.6 lagging（因 X > 0，感性）

   也可以：pf = cos(arctan(4/3)) = cos(53.13°) = 0.6 ✓

**答案**：

> ****S** = 48 + j64 VA，P = 48 W，Q = 64 VAR，|S| = 80 VA，pf = 0.6 lagging**

**延伸思考**：功率三角形中 P（水平）、Q（垂直）、|S|（斜邊）形成直角三角形。感性負載 Q > 0（向上），容性負載 Q < 0（向下）。

### 題目 36：功率因數校正（Power Factor Correction）

**來源**：台大電機 109-1 期末考風格
**難度**：★★★★☆
**主題**：交流穩態 — 功率因數校正

**題目**：

某工廠負載消耗 P = 100 kW，功率因數為 0.6 lagging，電源為 V_rms = 480 V（有效值），頻率 f = 60 Hz。求：
(a) 負載電流有效值
(b) 需要並聯多大的電容使功率因數校正至 0.95 lagging
(c) 校正後的電源電流有效值

**詳解**：

**解題思路**：功率因數校正（PFC）透過並聯電容提供部分無效功率，減少電源需提供的無效功率，從而降低電源電流。

**步驟**：

(a) 校正前的負載電流：

1. 視在功率：|S₁| = P / pf₁ = 100k / 0.6 = 166.67 kVA
2. I₁_rms = |S₁| / V_rms = 166670 / 480 = 347.2 A

(b) 需要的電容值：

3. 校正前的 Q₁：
   θ₁ = arccos(0.6) = 53.13°
   Q₁ = P × tan(θ₁) = 100k × tan(53.13°) = 100k × 4/3 = 133.33 kVAR

4. 校正後的 Q₂：
   θ₂ = arccos(0.95) = 18.19°
   Q₂ = P × tan(θ₂) = 100k × tan(18.19°) = 100k × 0.3287 = 32.87 kVAR

5. 電容需提供的無效功率：
   Q_C = Q₁ − Q₂ = 133.33k − 32.87k = 100.46 kVAR

6. Q_C = V_rms² × ωC
   C = Q_C / (V_rms² × ω) = 100460 / (480² × 2π × 60)
   = 100460 / (230400 × 376.99)
   = 100460 / 86,862,336
   = 1.157 × 10⁻³ F ≈ 1.157 mF

(c) 校正後的電源電流：

7. |S₂| = P / pf₂ = 100k / 0.95 = 105.26 kVA
8. I₂_rms = |S₂| / V_rms = 105260 / 480 = 219.3 A

9. 電流減少比例：(347.2 − 219.3) / 347.2 = 36.8% 的電流降低

**答案**：

> **(a) I₁ = 347.2 A**
> **(b) C ≈ 1.157 mF**
> **(c) I₂ = 219.3 A（電流降低 36.8%）**

**延伸思考**：功率因數校正不改變負載消耗的實功率 P，但大幅降低電源電流和線路損耗（I²R）。電力公司對低功率因數用戶加收罰款。不校正到 pf = 1 是因為所需電容太大（成本高），且若過度校正變為 leading 反而不利。

### 題目 37：最大平均功率轉移（交流電路）

**來源**：經典題型
**難度**：★★★★☆
**主題**：交流穩態 — 最大功率轉移

**題目**：

已知交流電路的戴維寧等效為 **V_Th** = 10∠0° V（振幅相量），Z_Th = 3 + j4 Ω。
(a) 若負載阻抗 Z_L 可以任意選擇，求使負載獲得最大平均功率的 Z_L 值及最大功率。
(b) 若負載限定為純電阻 R_L，求最佳 R_L 值及對應功率。

**詳解**：

**解題思路**：交流最大功率轉移定理：Z_L = Z_Th*（共軛匹配）時功率最大。純電阻限定時 R_L = |Z_Th|。

**步驟**：

(a) 共軛匹配（Z_L 可任意選）：

1. 最佳負載：Z_L = Z_Th* = 3 − j4 Ω

2. 總阻抗：Z_total = Z_Th + Z_L = (3 + j4) + (3 − j4) = 6 Ω（純電阻）

3. 電流振幅：I_m = V_Th_m / |Z_total| = 10 / 6 A

4. 最大平均功率：
   P_max = ½ × I_m² × R_L = ½ × (10/6)² × 3 = ½ × 100/36 × 3 = 150/36 = 25/6 ≈ 4.167 W

   通用公式：P_max = |V_Th|² / (8 × R_Th) = 100 / (8 × 3) = 100/24 = 25/6 ≈ 4.167 W ✓

(b) 純電阻限定（Z_L = R_L）：

5. 最佳 R_L = |Z_Th| = √(3² + 4²) = 5 Ω

6. 總阻抗：Z_total = (3 + j4) + 5 = 8 + j4 Ω
   |Z_total| = √(64 + 16) = √80 = 4√5 Ω

7. 電流振幅：I_m = 10 / 4√5 A

8. 平均功率：
   P = ½ × I_m² × R_L = ½ × (100/80) × 5 = 250/80 × ½ = 500/160 = 25/8 = 3.125 W

**答案**：

> **(a) Z_L = 3 − j4 Ω（共軛匹配），P_max = 25/6 ≈ 4.167 W**
> **(b) R_L = 5 Ω（= |Z_Th|），P = 25/8 = 3.125 W**

**延伸思考**：共軛匹配的功率 (25/6 W) 大於純電阻匹配的功率 (25/8 W)，因為共軛匹配消除了虛部使總阻抗最小化。通訊系統中常用共軛匹配（如天線匹配網路），而電力系統中通常不追求最大功率轉移。

### 題目 38：三相電路功率計算

**來源**：台大電機 110-2 期末考風格
**難度**：★★★★☆
**主題**：交流穩態 — 平衡三相電路

**題目**：

一個平衡三相 Y-Y 系統，線電壓（Line Voltage）有效值為 V_L = 208 V，頻率 60 Hz，每相負載阻抗 Z_Y = 6 + j8 Ω。求：
(a) 相電壓（Phase Voltage）有效值
(b) 線電流（Line Current）有效值
(c) 總三相功率（P_total、Q_total、S_total）

**詳解**：

**解題思路**：平衡三相 Y 接：V_phase = V_line/√3，線電流 = 相電流。總功率 = 3 × 單相功率。

**步驟**：

(a) 相電壓：

1. V_p = V_L / √3 = 208 / √3 = 120.1 V

(b) 線電流：

2. |Z_Y| = √(6² + 8²) = √100 = 10 Ω
3. I_L = I_p = V_p / |Z_Y| = 120.1 / 10 = 12.01 A

(c) 三相功率：

4. 功率因數角：θ = arctan(8/6) = arctan(4/3) = 53.13°
   pf = cos(53.13°) = 0.6 lagging

5. 單相功率：
   P₁ = V_p × I_p × cos(θ) = 120.1 × 12.01 × 0.6 = 865.2 W
   Q₁ = V_p × I_p × sin(θ) = 120.1 × 12.01 × 0.8 = 1153.6 VAR

6. 三相總功率：
   P_total = 3 × P₁ = 3 × 865.2 = 2595.6 W ≈ 2.596 kW
   Q_total = 3 × Q₁ = 3 × 1153.6 = 3460.8 VAR ≈ 3.461 kVAR
   |S_total| = √(P² + Q²) = √(2595.6² + 3460.8²) = √(6737152 + 11977136) = √18714288 ≈ 4327.2 VA

   用公式驗證：P_total = √3 × V_L × I_L × cos(θ) = √3 × 208 × 12.01 × 0.6 = 2595.5 W ✓

7. 若改為 Y-Δ 負載轉換：Z_Δ = 3 × Z_Y = 3(6 + j8) = 18 + j24 Ω。
   Δ 接時相電流 = V_L/|Z_Δ| = 208/30 = 6.93 A，線電流 = √3 × 6.93 = 12.01 A，功率相同。

**答案**：

> **(a) V_p = 120.1 V**
> **(b) I_L = 12.01 A**
> **(c) P_total = 2.596 kW，Q_total = 3.461 kVAR，|S_total| = 4.327 kVA，pf = 0.6 lagging**

**延伸思考**：三相功率公式 P = √3 × V_L × I_L × cos(θ) 適用於任何平衡三相電路（無論 Y 或 Δ 接法）。兩瓦特計法（Two-Wattmeter Method）可以用兩個單相功率計量測三相功率，且可由兩讀數之和求 P，之差求 Q。

### 題目 39：一階低通濾波器轉移函數

**來源**：經典題型
**難度**：★★★☆☆
**主題**：頻率響應 — 低通濾波器

**題目**：

一 RC 低通濾波器，R = 1 kΩ，C = 1 μF。
(a) 求轉移函數 H(jω) = V_out/V_in
(b) 求截止頻率（−3 dB 頻率）f_c
(c) 在 ω = 10ω_c 時，增益為多少 dB？

```
      R = 1kΩ
V_in ── ┤ ──┬── V_out
             │
           C = 1μF
             │
            GND
```

**詳解**：

**解題思路**：RC 低通濾波器用分壓公式求轉移函數，再分析頻率響應特性。

**步驟**：

(a) 轉移函數：

1. H(jω) = Z_C / (Z_R + Z_C) = (1/jωC) / (R + 1/jωC) = 1 / (1 + jωRC)

2. 定義 ω_c = 1/(RC) = 1/(10³ × 10⁻⁶) = 1000 rad/s

3. H(jω) = 1 / (1 + jω/ω_c)

   這是一階低通濾波器的標準形式。

(b) 截止頻率：

4. ω_c = 1000 rad/s
   f_c = ω_c / (2π) = 1000 / (2π) = 159.2 Hz

5. 在 ω = ω_c 時：|H(jω_c)| = 1/√(1 + 1) = 1/√2 ≈ 0.707
   20log₁₀(1/√2) = −3.01 dB ✓

(c) 在 ω = 10ω_c 時的增益：

6. |H(j·10ω_c)| = 1/√(1 + 10²) = 1/√101 ≈ 0.0995

7. 增益 = 20log₁₀(0.0995) = 20 × (−1.002) = −20.04 dB ≈ −20 dB

   這符合一階系統在截止頻率以上每十倍頻（decade）衰減 −20 dB 的規律。

**答案**：

> **(a) H(jω) = 1/(1 + jω/1000)**
> **(b) f_c = 159.2 Hz（ω_c = 1000 rad/s）**
> **(c) 在 ω = 10ω_c 時增益 ≈ −20 dB**

**延伸思考**：一階低通濾波器的波德圖特徵：低頻增益 0 dB，截止頻率後斜率 −20 dB/decade（−6 dB/octave）。相位從 0° 逐漸變化到 −90°，在 ω_c 處為 −45°。

### 題目 40：一階高通濾波器與波德圖

**來源**：經典題型
**難度**：★★★☆☆
**主題**：頻率響應 — 高通濾波器與波德圖

**題目**：

一 RC 高通濾波器，R = 10 kΩ，C = 0.01 μF。
(a) 求轉移函數 H(s) 和 H(jω)
(b) 求截止頻率
(c) 繪製波德圖（增益圖和相位圖）的漸近線

**詳解**：

**解題思路**：高通濾波器將輸出取在電阻上。用 s 域轉移函數分析。

**步驟**：

(a) 轉移函數：

1. H(s) = R / (R + 1/(sC)) = sRC / (1 + sRC)

2. 令 ω_c = 1/(RC) = 1/(10⁴ × 10⁻⁸) = 10⁴ rad/s

3. H(s) = s/ω_c / (1 + s/ω_c) = (s/10⁴) / (1 + s/10⁴)

4. H(jω) = jω/ω_c / (1 + jω/ω_c)

(b) 截止頻率：

5. ω_c = 10⁴ rad/s = 10000 rad/s
   f_c = ω_c/(2π) = 10000/(2π) = 1591.5 Hz

(c) 波德圖漸近線：

6. **增益圖**：
   - ω ≪ ω_c（低頻）：|H| ≈ ω/ω_c → 20log₁₀(ω/ω_c)，斜率 +20 dB/decade
   - ω ≫ ω_c（高頻）：|H| ≈ 1 → 0 dB（水平）
   - ω = ω_c：|H| = 1/√2 → −3 dB

7. **相位圖**：
   - ω ≪ ω_c/10：φ ≈ +90°
   - ω = ω_c：φ = +45°
   - ω ≫ 10ω_c：φ ≈ 0°
   - 相位從 +90° 漸減到 0°

8. 波德圖漸近線繪製（增益）：
   ```
   |H|_dB
   0 --|-------------- (高頻漸近線)
      |         /
   -20|        /
      |       /  (+20 dB/dec)
   -40|      /
      |     /
      +---+---+---+---→ log(ω)
       10² 10³ 10⁴ 10⁵
                ω_c
   ```

**答案**：

> **(a) H(s) = (s/10⁴)/(1 + s/10⁴)，H(jω) = (jω/10⁴)/(1 + jω/10⁴)**
> **(b) f_c = 1591.5 Hz**
> **(c) 低頻斜率 +20 dB/dec，高頻 0 dB，ω_c 處 −3 dB；相位從 90° 到 0°**

**延伸思考**：高通濾波器可用於交流耦合（AC Coupling），阻擋直流分量而通過交流信號。實際應用中常見於音訊處理、感測器信號調理等。

### 題目 41：二階帶通濾波器

**來源**：台大電機 109-2 期末考風格
**難度**：★★★★☆
**主題**：頻率響應 — RLC 帶通濾波器

**題目**：

RLC 串聯帶通濾波器，輸出取在電阻上。R = 10 Ω，L = 10 mH，C = 10 μF。求：
(a) 中心頻率（共振頻率）f₀
(b) 品質因數 Q
(c) 頻寬 BW
(d) 在共振頻率時的增益

```
      L=10mH     C=10μF
V_in ── ┤ ──── ┤├ ───┬── V_out
                       │
                     R = 10Ω
                       │
                      GND
```

**詳解**：

**解題思路**：RLC 串聯帶通濾波器的轉移函數為 H(s) = (sR/L)/(s² + sR/L + 1/(LC))。共振時電感和電容的阻抗互消。

**步驟**：

(a) 共振頻率：

1. ω₀ = 1/√(LC) = 1/√(10 × 10⁻³ × 10 × 10⁻⁶) = 1/√(10⁻⁷) = 1/(√10 × 10⁻⁴)
   = 10⁴/√10 = 10⁴ × 10⁻⁰·⁵ = 10³·⁵ ≈ 3162.3 rad/s

2. f₀ = ω₀/(2π) = 3162.3/(2π) ≈ 503.3 Hz

(b) 品質因數：

3. Q = ω₀L/R = 3162.3 × 0.01 / 10 = 31.623/10 = 3.162

   也可以用 Q = 1/(R) × √(L/C) = (1/10)√(0.01/10⁻⁵) = (1/10)√1000 = 31.623/10 = 3.162 ✓

(c) 頻寬：

4. BW = ω₀/Q = 3162.3/3.162 = 1000 rad/s

   也可以用 BW = R/L = 10/0.01 = 1000 rad/s ✓

   f_BW = BW/(2π) = 1000/(2π) ≈ 159.2 Hz

5. 半功率頻率：
   ω₁ = −R/(2L) + √((R/(2L))² + ω₀²) = −500 + √(250000 + 10⁷) = −500 + √10250000
   ≈ −500 + 3201.6 = 2701.6 rad/s

   ω₂ = 500 + 3201.6 = 3701.6 rad/s

   ω₂ − ω₁ = 1000 rad/s = BW ✓

(d) 共振時增益：

6. 在 ω = ω₀ 時，Z_L + Z_C = jω₀L + 1/(jω₀C) = jω₀L − j/(ω₀C)
   = j(ω₀L − 1/(ω₀C)) = 0（因為 ω₀²LC = 1）

7. H(jω₀) = R / (R + 0) = 1 → |H| = 1 → 0 dB

**答案**：

> **(a) f₀ ≈ 503.3 Hz（ω₀ ≈ 3162 rad/s）**
> **(b) Q ≈ 3.162**
> **(c) BW = 1000 rad/s ≈ 159.2 Hz**
> **(d) 共振時增益 = 0 dB（|H| = 1）**

**延伸思考**：Q 值越高，帶通濾波器越窄越銳利（選擇性高）。在無線電接收機中，高 Q 帶通濾波器用於頻道選擇。Q 也等於中心頻率除以頻寬：Q = f₀/BW。

### 題目 42：帶拒濾波器（Notch Filter）

**來源**：經典題型
**難度**：★★★★☆
**主題**：頻率響應 — 帶拒濾波器

**題目**：

RLC 串聯帶拒濾波器，輸出取在 L-C 串聯組合上。R = 50 Ω，L = 5 mH，C = 20 μF。求：
(a) 拒帶中心頻率 f₀
(b) 轉移函數 H(s)
(c) 在 ω = ω₀ 時增益為何
(d) 在 ω → 0 和 ω → ∞ 時增益為何

```
      R=50Ω
V_in ── ┤ ──┬── V_out
             │
           L=5mH
             │
           C=20μF
             │
            GND
```

V_out 取在 L 和 C 串聯上。

**詳解**：

**解題思路**：帶拒（Band-Reject / Notch）濾波器在共振頻率處衰減最大，因為 L-C 串聯在共振時阻抗為零。

**步驟**：

(a) 拒帶中心頻率：

1. ω₀ = 1/√(LC) = 1/√(5×10⁻³ × 20×10⁻⁶) = 1/√(10⁻⁷) = 10³·⁵ ≈ 3162.3 rad/s
   f₀ = ω₀/(2π) ≈ 503.3 Hz

(b) 轉移函數：

2. H(s) = (sL + 1/(sC)) / (R + sL + 1/(sC)) = (s²LC + 1) / (s²LC + sRC + 1)

3. 標準形式（除以 LC）：
   H(s) = (s² + 1/(LC)) / (s² + (R/L)s + 1/(LC))
   = (s² + ω₀²) / (s² + (R/L)s + ω₀²)

4. 代入數值：R/L = 50/0.005 = 10000，ω₀² = 10⁷
   H(s) = (s² + 10⁷) / (s² + 10000s + 10⁷)

(c) 在 ω = ω₀ 時：

5. H(jω₀) = (−ω₀² + ω₀²) / (−ω₀² + jω₀R/L + ω₀²) = 0 / (jω₀R/L) = 0

   |H(jω₀)| = 0 → −∞ dB

   在拒帶中心頻率處，信號完全被抑制（理想情況下）。

(d) 極端頻率：

6. ω → 0：H(j·0) = (0 + ω₀²)/(0 + 0 + ω₀²) = 1 → 0 dB（低頻通過）
7. ω → ∞：H(jω) ≈ ω²/ω² = 1 → 0 dB（高頻通過）

**答案**：

> **(a) f₀ ≈ 503.3 Hz**
> **(b) H(s) = (s² + 10⁷)/(s² + 10000s + 10⁷)**
> **(c) |H(jω₀)| = 0（−∞ dB），完全衰減**
> **(d) |H(0)| = |H(∞)| = 1（0 dB），完全通過**

**延伸思考**：帶拒濾波器常用於消除特定頻率的干擾（如 60 Hz 電源雜訊）。其轉移函數的分子有虛數軸上的零點（s = ±jω₀），分母有左半平面的極點。帶拒 = 低通 + 高通的合成效果。

### 題目 43：波德圖繪製 — 二階系統

**來源**：台大電機 110-1 期末考風格
**難度**：★★★★☆
**主題**：頻率響應 — 波德圖

**題目**：

給定轉移函數 H(s) = 100(s + 10) / (s(s + 100))。
(a) 將 H(s) 化為標準波德形式
(b) 畫出增益波德圖的漸近線（標明各段斜率和轉折頻率）
(c) 求 ω = 1、10、100、1000 rad/s 時的精確增益（dB）

**詳解**：

**解題思路**：將轉移函數分解為基本因子的乘積，各因子的波德圖漸近線疊加即得。

**步驟**：

(a) 標準波德形式（常數項提出使各括號內常數為 1）：

1. H(s) = 100(s + 10) / (s(s + 100))
   = 100 × 10(1 + s/10) / (s × 100(1 + s/100))
   = (100 × 10 / 100) × (1 + s/10) / (s × (1 + s/100))
   = 10 × (1 + s/10) / (s(1 + s/100))

2. H(jω) = 10 × (1 + jω/10) / (jω × (1 + jω/100))

(b) 波德圖漸近線分解：

3. 組成因子：
   - 常數 K = 10 → 20log₁₀(10) = 20 dB
   - 分母 1/jω：斜率 −20 dB/dec，通過 ω = 1 時 0 dB
   - 分子 (1 + jω/10)：ω < 10 為 0 dB，ω > 10 為 +20 dB/dec
   - 分母 1/(1 + jω/100)：ω < 100 為 0 dB，ω > 100 為 −20 dB/dec

4. 漸近線疊加：
   - ω < 10：斜率 = −20 dB/dec（來自 1/jω）
   - 10 < ω < 100：斜率 = −20 + 20 = 0 dB/dec（水平）
   - ω > 100：斜率 = 0 − 20 = −20 dB/dec

5. 基準點：在 ω = 10（第一轉折頻率）：
   由 1/jω 因子：20log₁₀(1/10) = −20 dB
   加上常數 20 dB → 0 dB
   這是水平段的增益水平。

   波德圖增益漸近線：
   ```
   |H|_dB
   40 |╲
   20 | ╲  (-20 dB/dec)
    0 |  ╲___________ (0 dB/dec)
   -20|              ╲
   -40|               ╲ (-20 dB/dec)
      +---+---+---+---+→ log(ω)
       1  10  100  1k
   ```

(c) 精確增益值：

6. ω = 1：|H(j1)| = 10 × |1+j0.1| / (1 × |1+j0.01|)
   = 10 × √1.01 / √1.0001 = 10 × 1.005/1.00005 ≈ 10.05
   增益 = 20.05 dB

7. ω = 10：|H(j10)| = 10 × |1+j1| / (10 × |1+j0.1|)
   = 10 × √2 / (10 × √1.01) = √2/√1.01 = 1.4142/1.005 ≈ 1.407
   增益 = 2.97 dB（漸近線為 0 dB，差 3 dB）

8. ω = 100：|H(j100)| = 10 × |1+j10| / (100 × |1+j1|)
   = 10 × √101 / (100 × √2) = 10 × 10.05 / 141.4 = 100.5/141.4 ≈ 0.7107
   增益 = −2.97 dB

9. ω = 1000：|H(j1000)| = 10 × |1+j100| / (1000 × |1+j10|)
   = 10 × 100.005 / (1000 × 10.05) ≈ 1000/10050 ≈ 0.0995
   增益 = −20.04 dB

**答案**：

> **(a) H(jω) = 10(1 + jω/10) / (jω(1 + jω/100))**
> **(b) 轉折頻率 ω = 10 和 100；斜率：−20→0→−20 dB/dec**
> **(c) ω = 1: 20 dB，ω = 10: 3 dB，ω = 100: −3 dB，ω = 1000: −20 dB**

**延伸思考**：漸近線與精確值的最大誤差發生在轉折頻率處，一階因子的誤差為 3 dB。二階因子在轉折頻率處的誤差取決於阻尼比 ζ（或 Q 值），可能遠大於 3 dB。

### 題目 44：波德圖反推轉移函數

**來源**：經典題型
**難度**：★★★★☆
**主題**：頻率響應 — 由波德圖求轉移函數

**題目**：

已知某系統的波德增益圖漸近線如下。求轉移函數 H(s)。

```
|H|_dB
40 |                 _______________
   |                /  (0 dB/dec)
20 |   (+20 dB/dec)/
   |              /
 0 |_____________/
   |  (0 dB/dec)
-20|
   +---+---+---+---+---→ log(ω)
    1  10  100  1k  10k
```

低頻（ω < 100）增益 0 dB，ω = 100 開始以 +20 dB/dec 上升，ω = 1000 處增益達到 20 dB 後恢復水平（0 dB/dec）。

**詳解**：

**解題思路**：分析漸近線的斜率變化，找出轉折頻率和對應的因子類型。

**步驟**：

1. 斜率變化分析：
   - ω < 100：斜率 0 → 無積分/微分因子在低頻
   - ω = 100：斜率從 0 變為 +20 → 分子有零點 (1 + jω/100)
   - ω = 1000：斜率從 +20 變為 0 → 分母有極點 (1 + jω/1000)

2. 低頻增益 = 0 dB → K = 1

3. 組合：
   H(jω) = (1 + jω/100) / (1 + jω/1000)

4. H(s) = (1 + s/100) / (1 + s/1000)

5. 化簡為有理函數形式：
   H(s) = (s + 100)/100 / ((s + 1000)/1000) = 1000(s + 100) / (100(s + 1000)) = 10(s + 100) / (s + 1000)

6. 驗證：
   - ω = 1：|H| ≈ 10 × 100/1000 = 1 → 0 dB ✓
   - ω = 10000：|H| ≈ 10 × 10000/10000 = 10 → 20 dB ✓
   - ω = 100：|H| = 10|100+j100|/|100+j1000| ≈ 10 × 141.4/1005 ≈ 1.407 → 3 dB ✓（漸近線為 0 dB）

**答案**：

> **H(s) = 10(s + 100)/(s + 1000)**

**延伸思考**：此轉移函數稱為「Lead Compensator（超前補償器）」，在控制系統中用於改善相位裕度。零點在極點的左邊（ω_z < ω_p），提供相位超前。

### 題目 45：RLC 串聯諧振電路

**來源**：台大電機 108-1 期末考風格
**難度**：★★★★☆
**主題**：頻率響應 — 串聯諧振

**題目**：

RLC 串聯諧振電路，在共振頻率 f₀ = 1 MHz 操作，品質因數 Q = 50，共振時電路阻抗 Z = 5 Ω。求：
(a) R、L、C 的值
(b) 半功率頻率 f₁ 和 f₂
(c) 共振時若施加 V_s = 1 V，電容上的電壓振幅為何

**詳解**：

**解題思路**：利用串聯諧振的基本關係：Z(ω₀) = R，Q = ω₀L/R = 1/(ω₀CR)。

**步驟**：

(a) R、L、C 的值：

1. R = Z(ω₀) = 5 Ω（共振時阻抗為純電阻）

2. ω₀ = 2πf₀ = 2π × 10⁶ rad/s

3. Q = ω₀L/R → L = QR/ω₀ = 50 × 5 / (2π × 10⁶) = 250/(2π × 10⁶)
   = 39.79 × 10⁻⁶ H ≈ 39.79 μH

4. Q = 1/(ω₀CR) → C = 1/(Qω₀R) = 1/(50 × 2π × 10⁶ × 5)
   = 1/(500π × 10⁶) = 636.6 × 10⁻¹² F ≈ 636.6 pF

   驗證：ω₀ = 1/√(LC) = 1/√(39.79×10⁻⁶ × 636.6×10⁻¹²) = 1/√(25.33×10⁻¹⁵)
   = 1/(5.033×10⁻⁷·⁵) ≈ 1/(159.2×10⁻⁹) ≈ 6.283×10⁶ = 2π×10⁶ ✓

(b) 半功率頻率：

5. BW = f₀/Q = 10⁶/50 = 20 kHz

6. 近似（高 Q 時）：f₁ ≈ f₀ − BW/2 = 1000000 − 10000 = 990 kHz
   f₂ ≈ f₀ + BW/2 = 1000000 + 10000 = 1010 kHz

7. 精確公式：
   ω₁ = −α + √(α² + ω₀²)，α = R/(2L) = 5/(2×39.79μ) = 62832 rad/s
   ω₂ = α + √(α² + ω₀²)

   由於 Q = 50 ≫ 1，近似值已非常準確。

(c) 共振時電容電壓：

8. 共振時電流：I = V_s/R = 1/5 = 0.2 A

9. 電容上電壓振幅：
   |V_C| = I × |Z_C(ω₀)| = I / (ω₀C) = 0.2 / (2π×10⁶ × 636.6×10⁻¹²)
   = 0.2 / (4×10⁻³) = 50 V

   或直接用 Q 值：|V_C| = Q × V_s = 50 × 1 = 50 V ✓

   電容（和電感）上的電壓是電源電壓的 Q 倍！

**答案**：

> **(a) R = 5 Ω，L ≈ 39.79 μH，C ≈ 636.6 pF**
> **(b) f₁ ≈ 990 kHz，f₂ ≈ 1010 kHz，BW = 20 kHz**
> **(c) |V_C| = Q × V_s = 50 V**

**延伸思考**：串聯諧振的「電壓放大」效應（V_C = QV_s）在高 Q 電路中非常顯著，可能導致元件電壓過高而損壞。此效應在電力系統中需要避免，但在無線電調諧電路中被利用來選擇信號。

### 題目 46：RLC 並聯諧振

**來源**：台大電機 109-1 期末考風格
**難度**：★★★★☆
**主題**：頻率響應 — 並聯諧振

**題目**：

RLC 並聯電路，R = 100 kΩ，L = 10 mH，C = 10 nF。一電流源 I_s = 1 mA 驅動此電路。求：
(a) 共振頻率 f₀
(b) 品質因數 Q
(c) 頻寬 BW
(d) 共振時端電壓 V

```
    ┌──────┬──────┬──────┐
    │      │      │      │
  I_s↑    R=100kΩ L=10mH C=10nF
  1mA      │      │      │
    │      │      │      │
    └──────┴──────┴──────┘
```

**詳解**：

**解題思路**：並聯 RLC 諧振：共振時阻抗最大（= R），Q = R/(ω₀L) 或 Q = ω₀RC（與串聯相反）。

**步驟**：

(a) 共振頻率：

1. ω₀ = 1/√(LC) = 1/√(10×10⁻³ × 10×10⁻⁹) = 1/√(10⁻¹⁰) = 10⁵ rad/s

2. f₀ = ω₀/(2π) = 10⁵/(2π) ≈ 15.915 kHz

(b) 品質因數：

3. Q = R/(ω₀L) = 100×10³/(10⁵ × 10×10⁻³) = 10⁵/10³ = 100

   或 Q = ω₀RC = 10⁵ × 100×10³ × 10×10⁻⁹ = 10⁵ × 10⁻³ = 100 ✓

(c) 頻寬：

4. BW = ω₀/Q = 10⁵/100 = 1000 rad/s

   f_BW = BW/(2π) ≈ 159.2 Hz

   或 BW = 1/(RC) = 1/(100k × 10n) = 1/(10⁻³) = 1000 rad/s ✓

(d) 共振時端電壓：

5. Z(ω₀) = R = 100 kΩ（並聯諧振時 L 和 C 的導納互消，只剩 R）

6. V = I_s × Z(ω₀) = 1×10⁻³ × 100×10³ = 100 V

7. 電流放大效應：
   I_L = V/(ω₀L) = 100/(10⁵ × 0.01) = 100/1000 = 0.1 A = 100 mA
   I_C = V × ω₀C = 100 × 10⁵ × 10⁻⁸ = 100 × 10⁻³ = 0.1 A = 100 mA
   I_L/I_s = 100m/1m = 100 = Q ✓

**答案**：

> **(a) f₀ ≈ 15.915 kHz**
> **(b) Q = 100**
> **(c) BW = 1000 rad/s ≈ 159.2 Hz**
> **(d) V = 100 V，電感/電容電流 = Q × I_s = 100 mA**

**延伸思考**：並聯諧振又稱「反諧振（Anti-Resonance）」，因為其阻抗在共振時最大（與串聯諧振相反）。並聯諧振中流過 L 和 C 的電流為電源電流的 Q 倍（電流放大效應），這與串聯諧振的電壓放大效應對偶。

### 題目 47：s 域電路模型 — 含初始條件的電容

**來源**：經典題型
**難度**：★★★☆☆
**主題**：拉普拉斯轉換 — s 域電路分析

**題目**：

下圖 RC 電路中，R = 5 Ω，C = 0.1 F，電容初始電壓 v_C(0⁻) = 10 V。t = 0 時接上 V_s = 25 V 直流電壓源。用拉普拉斯轉換法求 t > 0 時的電容電壓 v_C(t) 和電流 i(t)。

```
       SW(t=0)   R = 5Ω
  ┌──/────────── ┤ ──────┐
  │                       │
(+)                     C = 0.1F
V_s=25V                   │ v_C(0)=10V
(-)                       │
  │                       │
  └───────────────────────┘
```

**詳解**：

**解題思路**：將電路轉換到 s 域：電壓源 → V_s/s，電阻 → R，電容 → 1/(sC) 串聯初始條件電壓源 v_C(0)/s。用 s 域的 KVL 求解後做反轉換。

**步驟**：

1. s 域等效電路（電容用阻抗模型，含初始條件電壓源）：

   KVL：V_s/s = R × I(s) + (1/(sC)) × I(s) + v_C(0)/s

   25/s = 5·I(s) + (1/(0.1s))·I(s) + 10/s

   25/s − 10/s = I(s)·(5 + 10/s)

   15/s = I(s)·(5s + 10)/s

   I(s) = 15/(5s + 10) = 15/(5(s + 2)) = 3/(s + 2)

2. 反拉普拉斯轉換求 i(t)：
   i(t) = L⁻¹{3/(s + 2)} = 3e^(−2t) A，t ≥ 0

3. 求 V_C(s)：
   V_C(s) = v_C(0)/s + I(s)/(sC) = 10/s + 3/((s+2)·0.1s) = 10/s + 30/(s(s+2))

4. 部分分式展開 30/(s(s+2))：
   30/(s(s+2)) = A/s + B/(s+2)
   A = 30/(0+2) = 15
   B = 30/(−2+0) = −15（代入 s = −2）：B = 30/(−2) = −15

   30/(s(s+2)) = 15/s − 15/(s+2)

5. V_C(s) = 10/s + 15/s − 15/(s+2) = 25/s − 15/(s+2)

6. 反轉換：
   v_C(t) = 25 − 15e^(−2t) V，t ≥ 0

7. 驗證：
   - v_C(0) = 25 − 15 = 10 V ✓
   - v_C(∞) = 25 V ✓
   - τ = RC = 5 × 0.1 = 0.5 s → 指數中 1/τ = 2 ✓
   - 與時域通用公式比較：v_C(t) = 25 + (10 − 25)e^(−2t) = 25 − 15e^(−2t) ✓

**答案**：

> **v_C(t) = 25 − 15e^(−2t) V，i(t) = 3e^(−2t) A，t ≥ 0**

**延伸思考**：s 域分析法將微分方程轉化為代數方程，初始條件自然地融入 s 域模型中。電容的 s 域模型有兩種等效形式：(1) 阻抗 1/(sC) 串聯電壓源 v_C(0)/s；(2) 導納 sC 並聯電流源 Cv_C(0)。

V_s=20V                   │ i_L(0)=3A
(-)                       │

```

**詳解**：

**解題思路**：電感的 s 域模型：阻抗 sL 串聯電壓源 Li_L(0)（極性：初始電流方向同端為正）。

**步驟**：

   - i_L(0) = 5 − 2 = 3 A ✓
   - i_L(∞) = 5 − 0 = 5 A = V_s/R = 20/4 ✓
   - τ = L/R = 2/4 = 0.5 s → 指數中 1/τ = 2 ✓

**答案**：

> **i_L(t) = 5 − 2e^(−2t) A，t ≥ 0**

**延伸思考**：電感的 s 域模型也有兩種等效形式：(1) 阻抗 sL 串聯電壓源 Li_L(0)；(2) 導納 1/(sL) 並聯電流源 i_L(0)/s。選擇哪種取決於用節點電壓法（用導納模型）還是網目電流法（用阻抗模型）。

### 題目 49：部分分式展開 — 相異實數根

**來源**：經典題型
**難度**：★★★☆☆
**主題**：拉普拉斯轉換 — 部分分式展開

**題目**：

求下列 s 域函數的反拉普拉斯轉換。
F(s) = (3s² + 17s + 30) / (s(s + 2)(s + 5))

**詳解**：

**解題思路**：分母有三個相異實數根，用部分分式展開為三個一階項的和。

**步驟**：

1. 部分分式展開：
   F(s) = A/s + B/(s + 2) + C/(s + 5)

2. 求各係數（Cover-up Method / Heaviside Method）：

   A = s·F(s)|_{s=0} = (0 + 0 + 30)/((0 + 2)(0 + 5)) = 30/10 = 3

   B = (s+2)·F(s)|_{s=−2} = (3(4) + 17(−2) + 30)/((−2)(−2 + 5))
   = (12 − 34 + 30)/(−2 × 3) = 8/(−6) = −4/3

   C = (s+5)·F(s)|_{s=−5} = (3(25) + 17(−5) + 30)/((−5)(−5 + 2))
   = (75 − 85 + 30)/(−5 × (−3)) = 20/15 = 4/3

3. F(s) = 3/s − (4/3)/(s + 2) + (4/3)/(s + 5)

4. 驗證：設 s = 1：
   F(1) = (3 + 17 + 30)/(1 × 3 × 6) = 50/18 = 25/9
   展開式：3/1 − (4/3)/3 + (4/3)/6 = 3 − 4/9 + 4/18 = 3 − 4/9 + 2/9 = 3 − 2/9 = 25/9 ✓

5. 反拉普拉斯轉換：
   f(t) = 3 − (4/3)e^(−2t) + (4/3)e^(−5t)，t ≥ 0

   也可寫為：f(t) = 3 + (4/3)(e^(−5t) − e^(−2t))，t ≥ 0

**答案**：

> **f(t) = 3 − (4/3)e^(−2t) + (4/3)e^(−5t)，t ≥ 0**

**延伸思考**：Cover-up Method 只適用於分母中不重複的一階因子。若有重根或複數根，需要用其他方法（如比較係數法或微分法）。

### 題目 50：部分分式展開 — 重根

**來源**：台大電機 109-2 期末考風格
**難度**：★★★★☆
**主題**：拉普拉斯轉換 — 重根展開

**題目**：

求反拉普拉斯轉換：F(s) = (2s + 3) / ((s + 1)²(s + 3))

**詳解**：

**解題思路**：分母含重根 (s+1)²，部分分式展開需包含 (s+1) 和 (s+1)² 兩項。

**步驟**：

1. 部分分式展開：
   F(s) = A/(s + 1) + B/(s + 1)² + C/(s + 3)

2. 求 C（Cover-up，用 s = −3）：
   C = (s+3)·F(s)|_{s=−3} = (2(−3) + 3)/((−3+1)²) = (−3)/(4) = −3/4

3. 求 B（Cover-up，用 s = −1 消去重根）：
   B = (s+1)²·F(s)|_{s=−1} = (2(−1) + 3)/((−1+3)) = 1/2

4. 求 A（對 (s+1)²·F(s) 微分後代入 s = −1）：
   (s+1)²·F(s) = (2s + 3)/(s + 3)

   d/ds[(2s + 3)/(s + 3)] = [2(s + 3) − (2s + 3)] / (s + 3)²
   = (2s + 6 − 2s − 3) / (s + 3)² = 3/(s + 3)²

   代入 s = −1：A = 3/(−1 + 3)² = 3/4

5. F(s) = (3/4)/(s + 1) + (1/2)/(s + 1)² + (−3/4)/(s + 3)

6. 驗證（代入 s = 0）：
   F(0) = (0 + 3)/(1 × 3) = 3/3 = 1
   展開式：3/4/1 + 1/2/1 + (−3/4)/3 = 3/4 + 1/2 − 1/4 = 3/4 + 2/4 − 1/4 = 4/4 = 1 ✓

7. 反轉換（L⁻¹{1/(s+a)²} = te^(−at)）：
   f(t) = (3/4)e^(−t) + (1/2)te^(−t) − (3/4)e^(−3t)，t ≥ 0

   f(t) = (3/4)(e^(−t) − e^(−3t)) + (1/2)te^(−t)，t ≥ 0

**答案**：

> **f(t) = (3/4)e^(−t) + (1/2)te^(−t) − (3/4)e^(−3t)，t ≥ 0**

**延伸思考**：重根在時域中對應 te^(−at) 項（t 的多項式乘以指數衰減），這與二階電路臨界阻尼響應一致。n 重根會產生 t⁰, t¹, ..., t^(n−1) 乘以 e^(−at) 的各項。

### 題目 51：部分分式展開 — 共軛複數根

**來源**：經典題型
**難度**：★★★★☆
**主題**：拉普拉斯轉換 — 複數根展開

**題目**：

求反拉普拉斯轉換：F(s) = 10(s + 3) / ((s + 1)(s² + 4s + 13))

**詳解**：

**解題思路**：s² + 4s + 13 = (s+2)² + 9 有共軛複數根 s = −2 ± j3。展開時保持二次項以避免複數運算。

**步驟**：

1. 部分分式展開：
   F(s) = A/(s + 1) + (Bs + C)/(s² + 4s + 13)

2. 求 A（Cover-up，s = −1）：
   A = 10(−1 + 3)/((−1)² + 4(−1) + 13) = 10(2)/(1 − 4 + 13) = 20/10 = 2

3. 求 B 和 C：
   F(s) = 2/(s + 1) + (Bs + C)/(s² + 4s + 13)

   通分：10(s + 3) = 2(s² + 4s + 13) + (Bs + C)(s + 1)

   展開右側：
   = 2s² + 8s + 26 + Bs² + Bs + Cs + C
   = (2 + B)s² + (8 + B + C)s + (26 + C)

   比較係數：
   - s²：0 = 2 + B → B = −2
   - s¹：10 = 8 + B + C = 8 − 2 + C → C = 4
   - s⁰：30 = 26 + C = 26 + 4 = 30 ✓

4. F(s) = 2/(s + 1) + (−2s + 4)/(s² + 4s + 13)

5. 處理第二項，配合標準形式 (s+2)² + 9：
   (−2s + 4)/((s+2)² + 9) = −2(s + 2)/((s+2)² + 9) + (4 + 4)/((s+2)² + 9)
   = −2(s + 2)/((s+2)² + 9) + 8/((s+2)² + 9)
   = −2(s + 2)/((s+2)² + 9) + (8/3) × 3/((s+2)² + 9)

6. 反轉換：
   L⁻¹{(s+a)/((s+a)² + b²)} = e^(−at)cos(bt)
   L⁻¹{b/((s+a)² + b²)} = e^(−at)sin(bt)

   f(t) = 2e^(−t) − 2e^(−2t)cos(3t) + (8/3)e^(−2t)sin(3t)，t ≥ 0

7. 驗證（初值定理）：
   f(0⁺) = lim_{s→∞} s·F(s) = lim_{s→∞} 10s(s+3)/(s(s+1)(s²+4s+13))
   = lim_{s→∞} 10(s+3)/((s+1)(s²+4s+13)) = 0 ✓

   f(0) = 2 − 2 + 0 = 0 ✓

**答案**：

> **f(t) = 2e^(−t) − 2e^(−2t)cos(3t) + (8/3)e^(−2t)sin(3t)，t ≥ 0**

**延伸思考**：共軛複數根在時域中產生衰減正弦/餘弦項（即振盪衰減），對應欠阻尼響應。保持二次因子不分解可以避免複數運算，直接得到 cos/sin 形式的結果。

### 題目 52：s 域電路分析 — 二階電路完整求解

**來源**：台大電機 110-1 期末考風格
**難度**：★★★★★
**主題**：拉普拉斯轉換 — RLC 電路

**題目**：

RLC 串聯電路，R = 6 Ω，L = 1 H，C = 1/8 F。初始條件：v_C(0) = 4 V，i(0) = 2 A。t = 0 時接上 V_s = 12 V 直流電壓源。用 s 域分析法求 v_C(t)。

**詳解**：

**解題思路**：在 s 域中列 KVL，包含所有初始條件，求 V_C(s) 後做部分分式展開和反轉換。

**步驟**：

1. s 域 KVL：
   V_s/s = R·I(s) + [sL·I(s) − L·i(0)] + [I(s)/(sC) + v_C(0)/s]

   12/s = 6·I(s) + s·I(s) − 2 + 8·I(s)/s + 4/s

2. 整理：
   12/s − 4/s + 2 = I(s)·(s + 6 + 8/s)
   8/s + 2 = I(s)·(s² + 6s + 8)/s
   (8 + 2s)/s = I(s)·(s² + 6s + 8)/s
   I(s) = (2s + 8)/(s² + 6s + 8) = 2(s + 4)/((s + 2)(s + 4)) = 2/(s + 2)

3. 求 V_C(s)：
   V_C(s) = v_C(0)/s + I(s)/(sC) = 4/s + 2·8/(s·(s + 2) × s)...

   不對，讓我重新用 V_C(s) = I(s)/(sC) + v_C(0)/s：
   V_C(s) = I(s) × (1/(sC)) + v_C(0)/s = (2/(s+2)) × (8/s) + 4/s
   = 16/(s(s+2)) + 4/s

4. 部分分式展開 16/(s(s+2))：
   16/(s(s+2)) = A/s + B/(s+2)
   A = 16/2 = 8
   B = 16/(−2) = −8

   16/(s(s+2)) = 8/s − 8/(s+2)

5. V_C(s) = 8/s − 8/(s+2) + 4/s = 12/s − 8/(s+2)

6. 反轉換：
   v_C(t) = 12 − 8e^(−2t) V，t ≥ 0

7. 驗證：
   - v_C(0) = 12 − 8 = 4 V ✓
   - v_C(∞) = 12 V = V_s ✓
   - 特徵方程：s² + 6s + 8 = (s+2)(s+4) = 0 → s = −2, −4
   - 注意 I(s) = 2/(s+2) 中 (s+4) 被消掉了，這是因為初始條件恰好使 s = −4 的模態消失。
   - i(t) = 2e^(−2t)，i(0) = 2 A ✓

**答案**：

> **v_C(t) = 12 − 8e^(−2t) V，t ≥ 0**

**延伸思考**：此題的特徵方程有兩個根 s = −2 和 s = −4（過阻尼），但因為初始條件的特殊值，s = −4 的模態被消去，最終響應看起來像一階系統。這種「零極點消去」在系統理論中是重要的概念。

### 題目 53：轉移函數與衝激響應

**來源**：經典題型
**難度**：★★★★☆
**主題**：拉普拉斯轉換 — 轉移函數

**題目**：

某線性非時變（LTI）電路的轉移函數為 H(s) = (s + 3) / (s² + 5s + 6)。
(a) 求衝激響應 h(t)（Impulse Response）
(b) 若輸入為 v_in(t) = 10e^(−t)u(t)，用 s 域求輸出 v_out(t)
(c) 判斷系統的穩定性

**詳解**：

**解題思路**：衝激響應 = 轉移函數的反拉普拉斯轉換。輸出 = H(s) × V_in(s) 的反轉換。

**步驟**：

(a) 衝激響應：

1. H(s) = (s + 3)/((s + 2)(s + 3)) = 1/(s + 2)

   （分子分母有共同因子 (s+3) 被消去）

2. h(t) = L⁻¹{1/(s + 2)} = e^(−2t)u(t)

(b) 輸出：

3. V_in(s) = L{10e^(−t)u(t)} = 10/(s + 1)

4. V_out(s) = H(s) × V_in(s) = (1/(s + 2)) × (10/(s + 1)) = 10/((s + 1)(s + 2))

5. 部分分式展開：
   10/((s+1)(s+2)) = A/(s+1) + B/(s+2)
   A = 10/(−1+2) = 10
   B = 10/(−2+1) = −10

6. v_out(t) = 10e^(−t) − 10e^(−2t)，t ≥ 0
   v_out(t) = 10(e^(−t) − e^(−2t))u(t)

7. 驗證（初值定理）：
   lim_{s→∞} s·V_out(s) = lim_{s→∞} 10s/((s+1)(s+2)) = 0
   v_out(0) = 10(1 − 1) = 0 ✓

(c) 穩定性：

8. 消去前的極點：s = −2 和 s = −3（都在左半平面 LHP）
9. 消去後的極點：s = −2（在 LHP）

10. 所有極點都在 s 平面的左半平面 → **系統 BIBO 穩定（Bounded-Input Bounded-Output Stable）**

    但注意：被消去的極點 s = −3 代表一個不可觀測（Unobservable）或不可控制（Uncontrollable）的模態，在控制理論中需要額外討論內部穩定性。

**答案**：

> **(a) h(t) = e^(−2t)u(t)**
> **(b) v_out(t) = 10(e^(−t) − e^(−2t))u(t)**
> **(c) 系統 BIBO 穩定（所有極點在 LHP）**

**延伸思考**：轉移函數的極點決定系統的自然頻率和穩定性。零極點消去在數學上簡化了轉移函數，但在物理上被消去的模態仍然存在。若被消去的極點在 RHP（右半平面），系統表面穩定但內部不穩定。

### 題目 54：初值定理與終值定理

**來源**：台大電機 110-2 期末考風格
**難度**：★★★☆☆
**主題**：拉普拉斯轉換 — 初值/終值定理

**題目**：

已知 F(s) = (5s² + 23s + 20) / (s(s + 2)(s + 4))。
(a) 用終值定理（Final Value Theorem）求 f(∞)
(b) 用初值定理（Initial Value Theorem）求 f(0⁺)
(c) 求完整的 f(t) 並驗證 (a)(b) 的結果

**詳解**：

**解題思路**：初值定理：f(0⁺) = lim_{s→∞} sF(s)。終值定理：f(∞) = lim_{s→0} sF(s)（前提是 sF(s) 的所有極點在 LHP）。

**步驟**：

(a) 終值定理：

1. 先確認適用條件：sF(s) = (5s² + 23s + 20)/((s+2)(s+4)) 的極點為 s = −2, −4，都在 LHP → 可用。

2. f(∞) = lim_{s→0} sF(s) = lim_{s→0} (5s² + 23s + 20)/((s + 2)(s + 4))
   = 20/(2 × 4) = 20/8 = 5/2 = 2.5

(b) 初值定理：

3. f(0⁺) = lim_{s→∞} sF(s) = lim_{s→∞} s × (5s² + 23s + 20)/(s(s+2)(s+4))
   = lim_{s→∞} (5s² + 23s + 20)/((s+2)(s+4))
   = lim_{s→∞} (5s²)/(s²) = 5

(c) 完整求解 f(t)：

4. 部分分式展開：
   F(s) = A/s + B/(s+2) + C/(s+4)

   A = sF(s)|_{s=0} = 20/(2×4) = 5/2

   B = (s+2)F(s)|_{s=−2} = (5(4) + 23(−2) + 20)/((−2)(−2+4)) = (20 − 46 + 20)/(−4)
   = (−6)/(−4) = 3/2

   C = (s+4)F(s)|_{s=−4} = (5(16) + 23(−4) + 20)/((−4)(−4+2)) = (80 − 92 + 20)/(8)
   = 8/8 = 1

5. F(s) = (5/2)/s + (3/2)/(s+2) + 1/(s+4)

6. f(t) = 5/2 + (3/2)e^(−2t) + e^(−4t)，t ≥ 0

7. 驗證：
   - f(0⁺) = 5/2 + 3/2 + 1 = 5 ✓（與初值定理一致）
   - f(∞) = 5/2 + 0 + 0 = 5/2 ✓（與終值定理一致）

**答案**：

> **(a) f(∞) = 5/2 = 2.5**
> **(b) f(0⁺) = 5**
> **(c) f(t) = 5/2 + (3/2)e^(−2t) + e^(−4t)，t ≥ 0**

**延伸思考**：終值定理的適用前提是 f(∞) 必須存在（sF(s) 的極點都在 LHP 或至多在原點有一階極點）。若 F(s) 有右半平面極點或虛軸上的極點（如 sin(ωt) 的情形），終值定理不適用。初值定理則無此限制，但要求 F(s) 是嚴格真分式（proper rational function）。

### 題目 55：Z 參數（開路阻抗參數）

**來源**：經典題型
**難度**：★★★☆☆
**主題**：雙埠網路 — Z 參數

**題目**：

求下列 T 型（T-Network）雙埠網路的 Z 參數矩陣。

```
   I₁ →          I₂ →
  ●──── Z_a=6Ω ──┬── Z_b=4Ω ────●
  │               │               │
Port 1          Z_c=12Ω        Port 2
  │               │               │
  ●───────────────┴───────────────●
```

**詳解**：

**解題思路**：Z 參數定義：V₁ = z₁₁I₁ + z₁₂I₂，V₂ = z₂₁I₁ + z₂₂I₂。求法：z₁₁ = V₁/I₁|_{I₂=0}（Port 2 開路），z₂₁ = V₂/I₁|_{I₂=0}，以此類推。

**步驟**：

**I₂ = 0（Port 2 開路）：**

1. z₁₁ = V₁/I₁|_{I₂=0}：
   電流 I₁ 流過 Z_a 和 Z_c 串聯（I₂ = 0 代表 Port 2 開路，Z_b 無電流）。
   V₁ = I₁ × (Z_a + Z_c) = I₁ × (6 + 12) = 18·I₁
   z₁₁ = 18 Ω

2. z₂₁ = V₂/I₁|_{I₂=0}：
   V₂ = I₁ × Z_c = 12·I₁（Z_b 上無電流，無壓降）
   z₂₁ = 12 Ω

**I₁ = 0（Port 1 開路）：**

3. z₂₂ = V₂/I₂|_{I₁=0}：
   電流 I₂ 流過 Z_b 和 Z_c 串聯。
   V₂ = I₂ × (Z_b + Z_c) = I₂ × (4 + 12) = 16·I₂
   z₂₂ = 16 Ω

4. z₁₂ = V₁/I₂|_{I₁=0}：
   V₁ = I₂ × Z_c = 12·I₂
   z₁₂ = 12 Ω

5. Z 參數矩陣：
   ```
   [z] = | z₁₁  z₁₂ | = | 18  12 | Ω
         | z₂₁  z₂₂ |   | 12  16 |
   ```

6. 驗證互易性（Reciprocity）：z₁₂ = z₂₁ = 12 Ω ✓
   （無相依源的被動網路必滿足互易性）

**答案**：

> **[z] = | 18  12 |  Ω，互易網路（z₁₂ = z₂₁）**
> **&emsp;&emsp;&emsp;| 12  16 |**

**延伸思考**：T 型網路的 Z 參數有簡潔的公式：z₁₁ = Z_a + Z_c，z₂₂ = Z_b + Z_c，z₁₂ = z₂₁ = Z_c。反之，由 Z 參數可反推 T 型等效：Z_a = z₁₁ − z₁₂，Z_b = z₂₂ − z₂₁，Z_c = z₁₂。

### 題目 56：Y 參數（短路導納參數）

**來源**：台大電機 109-1 期末考風格
**難度**：★★★☆☆
**主題**：雙埠網路 — Y 參數

**題目**：

求下列 π 型（Pi-Network）雙埠網路的 Y 參數矩陣。

```
   I₁ →                    I₂ →
  ●─────────── Y_b=0.5S ───────────●
  │                                 │
Port 1    Y_a=0.2S          Y_c=0.3S    Port 2
  │           │                 │       │
  ●───────────┴─────────────────┴───────●
```

Y_a 從 Port 1 到地，Y_b 從 Port 1 到 Port 2，Y_c 從 Port 2 到地。

**詳解**：

**解題思路**：Y 參數定義：I₁ = y₁₁V₁ + y₁₂V₂，I₂ = y₂₁V₁ + y₂₂V₂。求法：y₁₁ = I₁/V₁|_{V₂=0}（Port 2 短路），以此類推。

**步驟**：

**V₂ = 0（Port 2 短路）：**

1. y₁₁ = I₁/V₁|_{V₂=0}：
   Port 2 短路使 Y_c 被短路。V₁ 驅動 Y_a（到地）和 Y_b（到 Port 2 = 地）並聯。
   I₁ = V₁ × (Y_a + Y_b) = V₁ × (0.2 + 0.5) = 0.7·V₁
   y₁₁ = 0.7 S

2. y₂₁ = I₂/V₁|_{V₂=0}：
   I₂ 從短路端流出。Y_b 上電流 = V₁ × Y_b = 0.5·V₁（從 Port 1 到 Port 2），此電流流入 Port 2。
   但 I₂ 的正方向是流入 Port 2，而通過 Y_b 的電流從 Port 1 流向 Port 2 的短路端，在 Port 2 端流出。
   I₂ = −V₁ × Y_b = −0.5·V₁
   y₂₁ = −0.5 S

**V₁ = 0（Port 1 短路）：**

3. y₂₂ = I₂/V₂|_{V₁=0}：
   V₂ 驅動 Y_c 和 Y_b 並聯（Y_a 被短路）。
   I₂ = V₂ × (Y_c + Y_b) = V₂ × (0.3 + 0.5) = 0.8·V₂
   y₂₂ = 0.8 S

4. y₁₂ = I₁/V₂|_{V₁=0}：
   I₁ = −V₂ × Y_b = −0.5·V₂
   y₁₂ = −0.5 S

5. Y 參數矩陣：
   ```
   [y] = | y₁₁  y₁₂ | = |  0.7  -0.5 | S
         | y₂₁  y₂₂ |   | -0.5   0.8 |
   ```

6. 驗證：y₁₂ = y₂₁ = −0.5 S ✓（互易網路）

**答案**：

> **[y] = |  0.7  −0.5 |  S**
> **&emsp;&emsp;&emsp;| −0.5   0.8 |**

**延伸思考**：π 型網路的 Y 參數公式：y₁₁ = Y_a + Y_b，y₂₂ = Y_c + Y_b，y₁₂ = y₂₁ = −Y_b。注意 y₁₂ 為負值。Z 參數和 Y 參數互為矩陣逆：[y] = [z]⁻¹。

### 題目 57：h 參數（混合參數）

**來源**：台大電機 108-2 期末考風格
**難度**：★★★★☆
**主題**：雙埠網路 — h 參數

**題目**：

某雙埠網路的 Z 參數矩陣為：
```
[z] = | 20  5 | Ω
      |  5 10 |
```
(a) 求 h 參數矩陣
(b) 說明各 h 參數的物理意義和單位

**詳解**：

**解題思路**：h 參數定義：V₁ = h₁₁I₁ + h₁₂V₂，I₂ = h₂₁I₁ + h₂₂V₂。由 Z 參數轉換。

**步驟**：

(a) 由 Z 參數轉換為 h 參數：

1. 轉換公式：
   h₁₁ = Δz / z₂₂
   h₁₂ = z₁₂ / z₂₂
   h₂₁ = −z₂₁ / z₂₂
   h₂₂ = 1 / z₂₂

   其中 Δz = z₁₁z₂₂ − z₁₂z₂₁

2. Δz = 20 × 10 − 5 × 5 = 200 − 25 = 175

3. h₁₁ = 175/10 = 17.5 Ω
   h₁₂ = 5/10 = 0.5（無因次）
   h₂₁ = −5/10 = −0.5（無因次）
   h₂₂ = 1/10 = 0.1 S

4. h 參數矩陣：
   ```
   [h] = | 17.5   0.5  |
         | -0.5   0.1  |
   ```

5. 驗證：對於互易網路，h₁₂ = −h₂₁ → 0.5 = −(−0.5) = 0.5 ✓

(b) 各參數的物理意義和單位：

6. h₁₁ = V₁/I₁|_{V₂=0}：Port 2 短路時的輸入阻抗。單位：Ω
   h₁₂ = V₁/V₂|_{I₁=0}：Port 1 開路時的反向電壓增益。無因次
   h₂₁ = I₂/I₁|_{V₂=0}：Port 2 短路時的正向電流增益。無因次
   h₂₂ = I₂/V₂|_{I₁=0}：Port 1 開路時的輸出導納。單位：S

**答案**：

> **(a) [h] = | 17.5  0.5 |**
> **&emsp;&emsp;&emsp;&emsp;| −0.5  0.1 |**
> **(b) h₁₁：輸入阻抗 (Ω)，h₁₂：反向電壓增益（無因次），h₂₁：正向電流增益（無因次），h₂₂：輸出導納 (S)**

**延伸思考**：h 參數常用於電晶體（BJT）的小信號模型分析。h₂₁ 即為 β（電流增益 h_fe）。混合參數的「混合」指的是方程式中混合了電壓和電流作為獨立變數（V₁ 由 I₁ 和 V₂ 決定，I₂ 由 I₁ 和 V₂ 決定），因此各參數的單位不同。

### 題目 58：ABCD 參數（傳輸參數）

**來源**：經典題型
**難度**：★★★★☆
**主題**：雙埠網路 — ABCD 參數

**題目**：

求下列 T 型網路的 ABCD 參數（傳輸參數）矩陣。Z₁ = 3 Ω，Z₂ = 5 Ω，Z₃ = 10 Ω。

```
   I₁ →           I₂ →
  ●── Z₁=3Ω ──┬── Z₂=5Ω ──●
  │            │            │
Port 1       Z₃=10Ω     Port 2
  │            │            │
  ●────────────┴────────────●
```

**詳解**：

**解題思路**：ABCD 參數定義：V₁ = A·V₂ − B·I₂，I₁ = C·V₂ − D·I₂。注意 I₂ 的方向是「流出」Port 2（與 Z/Y 參數不同）。

**步驟**：

**I₂ = 0（Port 2 開路）：**

1. A = V₁/V₂|_{I₂=0}：
   I₂ = 0 時，I₁ 流過 Z₁ 和 Z₃ 串聯。
   V₂ = I₁ × Z₃ = 10·I₁
   V₁ = I₁ × (Z₁ + Z₃) = 13·I₁
   A = V₁/V₂ = 13/10 = 1.3

2. C = I₁/V₂|_{I₂=0}：
   C = I₁/(10·I₁) = 1/10 = 0.1 S

**V₂ = 0（Port 2 短路）：**

3. B = −V₁/I₂|_{V₂=0}（注意負號因為 I₂ 流出）：
   V₂ = 0 時，Z₂ 的右端短路到地，Z₃ 也接地。
   Z₃ 與 Z₂ 並聯：Z₃∥Z₂ = (10 × 5)/(10 + 5) = 50/15 = 10/3 Ω

   V₁ = I₁ × (Z₁ + Z₃∥Z₂) = I₁ × (3 + 10/3) = I₁ × 19/3

   I₂ 流出 Port 2（從右端進入短路）：
   I₂ = −I₁ × Z₃/(Z₂ + Z₃) = −I₁ × 10/15 = −(2/3)I₁
   （負號因為電流分配後是從中間節點流向 Port 2 再流出）

   B = −V₁/I₂ = −(I₁ × 19/3)/(−(2/3)I₁) = (19/3)/(2/3) = 19/2 = 9.5 Ω

4. D = −I₁/I₂|_{V₂=0}：
   D = −I₁/(−(2/3)I₁) = 3/2 = 1.5

5. ABCD 矩陣：
   ```
   [T] = | A  B | = | 1.3  9.5 |
         | C  D |   | 0.1  1.5 |
   ```

6. 驗證 AD − BC = 1（被動互易網路的必要條件）：
   AD − BC = 1.3 × 1.5 − 9.5 × 0.1 = 1.95 − 0.95 = 1.0 ✓

7. T 型網路 ABCD 的通用公式驗證：
   A = 1 + Z₁/Z₃ = 1 + 3/10 = 1.3 ✓
   B = Z₁ + Z₂ + Z₁Z₂/Z₃ = 3 + 5 + 15/10 = 9.5 ✓
   C = 1/Z₃ = 0.1 ✓
   D = 1 + Z₂/Z₃ = 1 + 5/10 = 1.5 ✓

**答案**：

> **[T] = | 1.3  9.5 |，AD − BC = 1（互易網路）**
> **&emsp;&emsp;&emsp;| 0.1  1.5 |**

**延伸思考**：ABCD 參數的最大優勢是串級（Cascade）連接：兩個雙埠網路串級後的 ABCD 矩陣等於兩者 ABCD 矩陣的乘積。這在傳輸線、多級放大器、微波電路分析中極為方便。

### 題目 59：雙埠網路串級連接

**來源**：台大電機 110-1 期末考風格
**難度**：★★★★★
**主題**：雙埠網路 — ABCD 串級連接

**題目**：

兩個相同的 T 型網路串級（Cascade）連接，每個網路的 ABCD 矩陣為：
```
[T] = | 2   3 |
      | 1   2 |
```
(a) 求串級後的總 ABCD 矩陣
(b) 求串級後的 Z 參數矩陣
(c) 若 Port 2 接負載 Z_L = 5 Ω，Port 1 接電壓源 V_s = 10 V（內阻為零），求負載電流 I₂

**詳解**：

**解題思路**：串級連接的總 ABCD = 前級 × 後級。再由 ABCD 轉換為 Z 參數。

**步驟**：

(a) 串級 ABCD 矩陣：

1. [T_total] = [T₁] × [T₂]

   ```
   | 2  3 | × | 2  3 | = | 2×2+3×1  2×3+3×2 | = | 7   12 |
   | 1  2 |   | 1  2 |   | 1×2+2×1  1×3+2×2 |   | 4    7 |
   ```

2. 驗證 AD − BC = 7 × 7 − 12 × 4 = 49 − 48 = 1 ✓

(b) ABCD 轉換為 Z 參數：

3. 轉換公式：
   z₁₁ = A/C = 7/4 = 1.75 Ω
   z₁₂ = (AD − BC)/C = 1/4 = 0.25 Ω
   z₂₁ = 1/C = 1/4 = 0.25 Ω
   z₂₂ = D/C = 7/4 = 1.75 Ω

4. Z 參數矩陣：
   ```
   [z] = | 1.75  0.25 | Ω
         | 0.25  1.75 |
   ```

5. z₁₂ = z₂₁ = 0.25 ✓（互易網路）
   z₁₁ = z₂₂ ✓（對稱網路，因為兩級相同）

(c) 求負載電流：

6. ABCD 方程式：V₁ = A·V₂ − B·I₂，I₁ = C·V₂ − D·I₂

   V₂ = Z_L × I₂ = 5I₂（注意 ABCD 中 I₂ 流出）

   但這裡 I₂ 流出 Port 2 進入負載，所以 V₂ = Z_L × I₂ 但 I₂ 的正方向是流出。
   V₂ = 5 × I₂... 不對，如果 I₂ 流出 Port 2 且流入負載，則 V₂ = Z_L × I₂。

   V₁ = A·V₂ − B·I₂ = 7(5I₂) − 12·I₂ = 35I₂ − 12I₂ = 23I₂

   但 V_s = V₁ = 10 V（內阻為零）

   V₁ = A·V₂ + B·I₂（使用正號版本：V₁ = AV₂ + BI₂，其中 I₂ 定義為流出）

   讓我確認 ABCD 的標準定義。

   標準定義：V₁ = A·V₂ − B·(−I₂) = A·V₂ + B·I₂'
   其中有些教科書將 I₂ 定義為流入 Port 2。

   使用 Nilsson/Riedel 的慣例：
   V₁ = A·V₂ − B·I₂（I₂ 流出 Port 2）

   Load: V₂ = −Z_L·I₂（因為 I₂ 流出 Port 2 即流入負載，V₂ 跨在負載上）
   等等，V₂ 正端在上，I₂ 流出代表從上端流出。如果負載在 Port 2 外，電流 I₂ 從 Port 2 上端流出進入負載上端再從下端流回，那 V₂ = Z_L × I₂（I₂ 流入負載的方向與 V₂ 的被動符號一致...）

   讓我直接用替代法。

   V₂ = Z_L × I₂（ABCD 中 I₂ 流出 Port 2，等於流入負載）
   V₁ = AV₂ + BI₂ = A·Z_L·I₂ + B·I₂ = (AZ_L + B)I₂

   注意：在 ABCD 矩陣中，正式定義為：
   V₁ = AV₂ + BI₂ 且 I₁ = CV₂ + DI₂
   其中 I₂ 是流出 Port 2 的電流。

   但如果教科書版本是 V₁ = AV₂ - BI₂（I₂ 流入 Port 2），則 I₂ 定義不同。

   根據 AD − BC = 1（已驗證），使用 V₁ = AV₂ + BI₂ 的慣例（I₂ 流出）：

   V₁ = (AZ_L + B)I₂ = (7 × 5 + 12)I₂ = 47I₂

   10 = 47I₂
   I₂ = 10/47 ≈ 0.2128 A

7. 驗證：V₂ = 5 × 10/47 = 50/47 ≈ 1.064 V

**答案**：

> **(a) [T_total] = | 7  12 |，AD − BC = 1**
> **&emsp;&emsp;&emsp;&emsp;&emsp;&emsp;| 4   7 |**
> **(b) [z] = | 1.75  0.25 |  Ω**
> **&emsp;&emsp;&emsp;&emsp;| 0.25  1.75 |**
> **(c) I₂ = 10/47 ≈ 0.213 A**

**延伸思考**：ABCD 矩陣相乘即可處理串級連接，這大幅簡化了多級網路的分析。在微波工程中，傳輸線、匹配網路等都用 ABCD（或等效的散射參數 S-parameter）來分析。注意串級的順序很重要——矩陣乘法不可交換。

### 題目 60：雙埠參數互轉綜合題

**來源**：台大電機 109-2 期末考風格
**難度**：★★★★★
**主題**：雙埠網路 — 參數互轉

**題目**：

某雙埠網路的 Y 參數矩陣為：
```
[y] = |  0.4  -0.1 | S
      | -0.1   0.2 |
```
(a) 求 Z 參數矩陣（= [y]⁻¹）
(b) 求 h 參數矩陣
(c) 求 ABCD 參數矩陣
(d) 畫出此網路的 π 型等效電路

**詳解**：

**解題思路**：利用雙埠參數轉換公式表（或矩陣運算）在各組參數間轉換。

**步驟**：

(a) Z = Y⁻¹：

1. Δy = y₁₁y₂₂ − y₁₂y₂₁ = 0.4 × 0.2 − (−0.1)(−0.1) = 0.08 − 0.01 = 0.07

2. [z] = (1/Δy) × | y₂₂  -y₁₂ | = (1/0.07) × | 0.2   0.1 |
                    | -y₂₁  y₁₁ |               | 0.1   0.4 |

3. z₁₁ = 0.2/0.07 = 20/7 ≈ 2.857 Ω
   z₁₂ = 0.1/0.07 = 10/7 ≈ 1.429 Ω
   z₂₁ = 0.1/0.07 = 10/7 ≈ 1.429 Ω
   z₂₂ = 0.4/0.07 = 40/7 ≈ 5.714 Ω

4. [z] = | 20/7  10/7 | Ω
         | 10/7  40/7 |

5. 驗證互易性：z₁₂ = z₂₁ = 10/7 ✓

(b) Y 轉 h 參數：

6. 轉換公式（Y → h）：
   h₁₁ = 1/y₁₁ = 1/0.4 = 2.5 Ω
   h₁₂ = −y₁₂/y₁₁ = −(−0.1)/0.4 = 0.25
   h₂₁ = y₂₁/y₁₁ = (−0.1)/0.4 = −0.25
   h₂₂ = Δy/y₁₁ = 0.07/0.4 = 0.175 S

7. [h] = | 2.5    0.25  |
         | -0.25  0.175 |

8. 驗證互易性：h₁₂ = −h₂₁ → 0.25 = −(−0.25) = 0.25 ✓

(c) Y 轉 ABCD 參數：

9. 轉換公式（Y → ABCD）：
   A = −y₂₂/y₂₁ = −0.2/(−0.1) = 2
   B = −1/y₂₁ = −1/(−0.1) = 10 Ω
   C = −Δy/y₂₁ = −0.07/(−0.1) = 0.7 S
   D = −y₁₁/y₂₁ = −0.4/(−0.1) = 4

10. [T] = | 2    10 |
          | 0.7   4 |

11. 驗證：AD − BC = 2 × 4 − 10 × 0.7 = 8 − 7 = 1 ✓

(d) π 型等效電路：

12. 由 Y 參數反推 π 型網路：
    Y_b = −y₁₂ = −(−0.1) = 0.1 S（即 R_b = 10 Ω）
    Y_a = y₁₁ − Y_b = 0.4 − 0.1 = 0.3 S（即 R_a = 10/3 Ω）
    Y_c = y₂₂ − Y_b = 0.2 − 0.1 = 0.1 S（即 R_c = 10 Ω）

13. π 型等效電路：
    ```
       I₁ →              R_b=10Ω              I₂ →
      ●─────────────────── ┤ ───────────────────●
      │                                         │
    Port 1    R_a=10/3Ω              R_c=10Ω      Port 2
      │           │                     │         │
      ●───────────┴─────────────────────┴─────────●
    ```

**答案**：

> **(a) [z] = | 20/7  10/7 |  Ω**
> **&emsp;&emsp;&emsp;&emsp;| 10/7  40/7 |**
>
> **(b) [h] = |  2.5   0.25  |**
> **&emsp;&emsp;&emsp;&emsp;| −0.25  0.175 |**
>
> **(c) [T] = | 2    10  |**
> **&emsp;&emsp;&emsp;&emsp;| 0.7   4  |**
>
> **(d) π 型：R_a = 10/3 Ω，R_b = 10 Ω，R_c = 10 Ω**

**延伸思考**：六組雙埠參數（Z、Y、h、g、ABCD、ABCD⁻¹）之間可以互轉，但前提是轉換公式中的分母不為零（即某些參數可能不存在，例如理想變壓器沒有 Z 和 Y 參數）。工程上選擇哪組參數取決於量測方便性和電路分析需求：低頻電晶體用 h 參數，微波電路用 S 參數（散射參數），電力傳輸線用 ABCD 參數。
