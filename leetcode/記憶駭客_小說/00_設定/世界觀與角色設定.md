### 下城區的一天（程維倫的日常）

```
05:30  地下街的日光燈永遠亮著，分不清白天黑夜。
       程維倫被米亞的鬧鐘叫醒。鬧鐘是她故意設成最吵的音效。
       「早安，距離你上次吃正餐已經 19 小時了。」

06:00  走到地下街 B 區的「阿嬤豆漿」買早餐。
       老闆娘是個記憶負債者，只記得怎麼做豆漿和油條。
       其他的——兒子的名字、自己的年齡——全賣了。
       但她炸的油條是整條地下街最好吃的。
       程維倫每次都多付一點錢。

07:00  打開修復台。檢查昨晚的委託進度。
       桌上三個記憶晶片在排隊：一段損壞的婚禮記憶、
       一個老兵的戰場記憶碎片、一段不知道誰寄來的匿名記憶。

12:00  午餐通常是便利商店的微波便當。
       地下街的「記憶の窩」便利商店什麼都賣：
       泡麵、電池、衛生紙、3GB 的廉價快樂記憶體驗包。
       收銀台旁邊的小招牌寫著：
       「本店記憶體驗包皆為合法授權。品質不保證。不接受退貨。」

15:00  黑市客戶上門。通常是些不能走正規管道的案子：
       被前任刪除記憶的人想找回來、
       想偷看死去親人最後一段記憶的人、
       或者——更危險的——想「植入」從未發生過的記憶的人。
       程維倫有底線：他修復，不偽造。

22:00  地下街的霓虹燈開始變成暖色調（模擬日落，雖然看不到太陽）。
       程維倫買一杯珍珠奶茶，半糖少冰。
       坐在 Y 區出口的鐵樓梯上，看「上面」的人搭手扶梯下來。
       有時候他會想：上面的世界是什麼樣的？
       然後他會想：算了。上面的記憶也不一定是真的。

02:00  還在修記憶。米亞已經把她的光球調成暗橘色（她說這是「打瞌睡模式」）。
       「你知道人類需要睡眠這種東西吧？」
       「再五分鐘。」
       「你三個小時前也這麼說。」
```

## 1.3 文化與氛圍：台灣味賽博朋克

```
這不是東京。不是洛杉磯。不是香港。

這是台北。2089年的台北。

  ▸ 夜市還在。但攤位除了賣雞排和臭豆腐，
    也賣「5分鐘快樂記憶體驗」和「前任的道歉記憶（非官方）」。
    最受歡迎的攤位叫「記憶滷味」——
    老闆把各種廉價記憶裝在滷味的塑膠袋裡，
    你挑幾個、他幫你「加熱」（載入），像挑滷味一樣。
    一份 50 新台幣。品質參差不齊。偶爾會挑到驚喜。

  ▸ 機車還在。但全部電動化、自動駕駛。
    下城區的人還是喜歡手動騎——
    「自動駕駛的機車就像被修改過的記憶，
      方向盤不在你手裡，你只是個乘客。」
    ——某個下城區機車行老闆

  ▸ 便利商店還在。而且更多了。
    7-ELEVEN 改名叫 MEMORY-ELEVEN（記憶拾壹）。
    全家變成「記憶全家」。萊爾富變成「記憶萊富」。
    除了賣食物和日用品，也賣記憶相關的小商品：
    記憶晶片清潔棒、記憶防篡改貼紙（其實沒什麼用）、
    記憶芳香劑（「讓你的記憶聞起來像薰衣草」）。

  ▸ 廟宇還在。但供的神不太一樣了。
    龍山寺還是龍山寺。但除了拜月老，
    現在也有人拜「記憶大帝」——一個民間自發的信仰。
    據說誠心祈禱，就能讓遺失的記憶自動回來。
    科學家說這是迷信。但每到記憶大崩潰週年紀念日，
    龍山寺的人潮比跨年還多。

  ▸ 台語還在。
    下城區的老人大多說台語。
    「記憶」在台語裡叫「記持」（kì-tî）。
    「你的記持賣了了啊？」
    是下城區最殘忍的一句問候。

  ▸ 選舉還在。
    2089 年的市長選舉，最大的政見是：
    「記憶備份全民免費化」vs「記憶自由市場化」。
    沒有人討論其他議題。因為在這個時代，
    記憶就是一切——身份、財產、關係、歷史。
    連選舉的口號都是：「你記得我們的承諾嗎？」
```

```
║  │                                                                │   ║
║  │  節點 = 記憶片段                                              │   ║
║  │  邊   = 關聯（因果、時間、情感、人物）                       │   ║

║  │  篡改一段記憶 → 關聯的記憶也會出現「邊的張力」               │   ║

║  └──────────────────────────────────────────────────────────────┘   ║
║                                                                      ║
║  ┌──────────────────────────────────────────────────────────────┐   ║

║  │                                                                │   ║

║  │                                                                │   ║
```

### 背景故事

```

  他的整個人生是周明哲的設計。

  周明哲在他空白的大腦裡植入了精心設計的記憶：

  自學程式（引導他走向技術）、台大電機（賦予他能力）、
  對體制的不信任（讓他離開主流、進入地下世界）。

  他是周明哲的「觀測器」：
  如果程維倫能用演算法還原被重寫的記憶 → 重寫不夠完美，需要改進。
  如果程維倫無法還原 → 計畫成功。
  無論哪種結果，對周明哲來說都是「有用的數據」。

  但周明哲沒有預料到一件事：
  程維倫在修復記憶的過程中，產生了「湧現記憶」。
  對委託人的同理心、對真相的執著、對林語青的感情——
  這些不在設計圖裡。這些是真實的。
  他是唯一能「超越設計」的存在。
```

### 修復風格（他的技術流派）

```
  程維倫的修復風格在下城區被稱為「裸手流」。

  大多數修復師依賴自動診斷軟體——讓程式跑完分析，再按建議修。
  程維倫不用。
  他直接看原始數據流——就像有些老中醫不看儀器，只把脈。

  他的方法：
  1. 先「感覺」。把自己的神經接上記憶數據流，用身體感受異常。
     （這很危險——如果記憶有陷阱，修復師的大腦也會受傷。）
  2. 找到異常點後，手動選擇演算法。他的選擇永遠是最精準的。
     不是最快的，但是副作用最小的。
  3. 修復時極度溫柔。他的操作速度比自動軟體慢三倍。
     但修復品質是業界（包括黑市）公認的頂級。
  4. 修完後會做一件事：在記憶的末尾加上一個微小的「修復印記」。
     不是簽名——是一段只有他能辨識的數據模式。
     像書法家的印章。這是他的驕傲。

  台大電機的背景在這裡發揮作用：
  他理解硬體層面的記憶晶片架構（電機系的底層功力），
  也理解軟體層面的數據結構和演算法（自學的程式設計天賦）。
  這讓他能從「矽到靈魂」全方位修復——
```

### 背景故事（含時間悖論解釋）

```
【林語青的身世：三層真相】

  第一層（她告訴程維倫的）：
    她是一個普通的記憶大崩潰倖存者。
    擁有「未被篡改」的原始記憶——記得十年前的真相。
    她帶著一段「不可能存在的記憶」來找程維倫修復。

  第二層（第四層揭露）：
    她是邱韻如（織網者）的女兒。
    她偷了母親的核心數據才逃出來的。
    她帶來的「未來記憶」裡藏著鸚鵡計畫的全部細節。

  第三層（第五層揭露——時間悖論）：
    她不是「來自現在」。
    周明哲的「記憶即現實」實驗——
    當足夠多的人的記憶被統一改寫時，
    客觀現實也會被「拖曳」。
    這不是魔法，是「集體意識塑造感知現實」的極端版本。

    在「原本的時間線」裡：
    程維倫修復了所有人的真實記憶。
    所有人都記起了遊行、屠殺、篡改。
    但真相太殘酷了。社會崩潰。新台北陷入內戰。
    林語青在那條時間線裡看著一切毀滅。

    然後周明哲啟動了「時間線重寫」——
    不是時間旅行。是「把所有人的記憶改寫成另一個版本」。
    當所有人都「記得」一個不同的過去，
    對這些人來說，那個不同的過去就是現實。

    但林語青的記憶沒有被完全覆蓋。
    因為她有兩個核心身份（母親的基因 + 父親的技術殘留）。
    她的記憶樹有兩個根——兩條時間線的她疊加在一起。

    所以：
    「未來記憶」= 她在「原本時間線」中 2091 年的記憶。
    在那條時間線裡，現在是 2091 年。
    但那條時間線被重寫了。新的時間線的「現在」是 2089 年。
    她帶著 2091 年的記憶，活在 2089 年。
    她是一個時間悖論的產物。

    「你會想起來的。你每次都會。」
    這句話的完整含義：
    → 在「原本的時間線」裡，程維倫也曾經歷過真相揭露。
    → 他「每次」都會走到同樣的終點——發現真相。
    → 但「每次」的結果都是毀滅。
    → 林語青帶著記憶回來，是想讓這一次的結局不同。
    → 「你每次都會」= 她看過很多次他覺醒的過程。
       每一次，她都在旁邊。每一次，他都不記得她。
       但每一次，他都會重新認出她。
       這是她堅持下去的唯一原因。
```

```
  她可能不存在於現實中。

  她會分裂成兩個不完整的意識。
  兩個都不是完整的林語青。
  兩個都認識程維倫。但認識的方式不同。

  她一直沒有告訴程維倫這件事。
  因為她不知道，如果告訴他，
  他會想要救「哪一個」她。
```

### 辦案風格：像 debug

```

  5. 審訊的時候不威脅也不利誘。他只問問題。
     問到對方自己發現自己在說謊。

  ├─「記憶會騙人。但資料不會。」
```

### 完整故事：從父愛到毀滅

```

  他不存在於任何一台伺服器上。
  他存在於所有伺服器上。

```
