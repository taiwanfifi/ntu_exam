# 民事訴訟法考前衝刺筆記 — 考前一週速讀版

> 本筆記專為非法律背景轉職考生設計，聚焦高頻考點與易混淆概念。
> 條號以現行民事訴訟法為準，標示★者為近年必考熱點。

## 壹、訴訟要件速查

### 一、什麼是訴訟要件？

訴訟要件是法院進入「本案審理」（實體判決）之前，必須先審查的程序門檻。
欠缺訴訟要件 → 法院以「裁定駁回」或「判決駁回」方式處理，不會審理有無理由。

### 二、訴訟要件一覽表

| 要件類別 | 核心內容 | 欠缺效果 | 常考條文 |
|---------|---------|---------|---------|
| **當事人能力** | 有無成為訴訟當事人之一般資格（類似民法權利能力） | 裁定駁回 | §40 |
| **訴訟能力** | 有無獨立為訴訟行為之能力（類似民法行為能力） | 需法定代理人代為訴訟 | §45 |
| **當事人適格** | 就特定訴訟有無實施訴訟之權能（正當當事人） | 判決駁回（以無理由駁回） | 無明文，學理要件 |
| **訴之利益** | 有無利用訴訟制度之必要與實益（權利保護必要） | 判決駁回 | 無明文，學理要件 |
| **管轄權** | 該法院對該事件有無審判權限 | 裁定移送 | §1~§31-3 |
| **起訴程式合法** | 訴狀是否具備法定格式、繳納裁判費 | 裁定命補正/駁回 | §244、§249 |
| **無重複起訴** | 同一事件不得重複起訴 | 裁定駁回 | §253 |

### 三、當事人能力 ★

| 類型 | 說明 |
|-----|-----|
| 自然人 | 有權利能力者均有當事人能力（出生～死亡） |
| 法人 | 依法設立登記之法人 |
| 非法人團體 | 設有代表人或管理人者（§40 III），例如：公寓大廈管理委員會 |
| 胎兒 | 涉及其利益之保護時，視為既已出生（民法§7） |

**考試重點**：非法人團體有當事人能力，但無權利能力，勝訴確定後強制執行以團體財產為限。

### 四、當事人適格 ★★

定義：在具體訴訟中，原告是否為「適合提起該訴訟之人」，被告是否為「適合被訴之人」。

| 類型 | 說明 | 舉例 |
|-----|-----|-----|
| 固有適格 | 主張自己為權利義務主體之人 | 債權人訴請債務人清償 |
| 第三人訴訟擔當 | 以自己名義就他人之法律關係實施訴訟 | 破產管理人、遺囑執行人 |

**口訣**：當事人能力＝「抽象資格」；當事人適格＝「具體資格」。

### 五、管轄重點整理

| 管轄類型 | 內容 | 條文 |
|---------|-----|------|
| 以原就被 | 原則上由被告住所地法院管轄 | §1 |
| 不動產專屬管轄 | 不動產物權訴訟由不動產所在地法院管轄 | §10 |
| 合意管轄 | 當事人以書面合意定第一審管轄法院 | §24 |
| 應訴管轄 | 被告不抗辯而為本案言詞辯論 | §25 |
| 指定管轄 | 管轄權有爭議或不能辨別時，由共同直接上級法院指定 | §23 |

## 貳、訴之三要素 ★★

### 一、三要素概覽

| 要素 | 內容 | 功能 |
|-----|-----|-----|
| **當事人** | 原告與被告 | 確定主觀範圍 |
| **訴訟標的** | 原告在訴訟中所主張的法律上地位或權利 | 確定審判對象 |
| **聲明（訴之聲明）** | 原告請求法院為特定內容裁判之聲明 | 確定裁判範圍 |

### 二、舊訴訟標的理論 vs 新訴訟標的理論 ★★★

| 比較項目 | 舊訴訟標的理論 | 新訴訟標的理論（一分肢） | 新訴訟標的理論（二分肢） |
|---------|--------------|---------------------|---------------------|
| 識別標準 | 以實體法上之請求權為準 | 以訴之聲明為準 | 以訴之聲明＋事實關係為準 |
| 請求權競合處理 | 數個請求權＝數個訴訟標的 | 同一聲明＝一個訴訟標的 | 同一聲明＋同一事實＝一個訴訟標的 |
| 舉例（車禍求償） | 侵權＋債務不履行＝二訴訟標的 | 同一給付聲明＝一訴訟標的 | 同一事故同一賠償＝一訴訟標的 |
| 實務立場 | **我國實務目前採此說** | — | 學說傾向此說 |
| 優點 | 概念明確、易操作 | 避免重複起訴問題 | 兼顧明確性與訴訟經濟 |
| 缺點 | 請求權競合時，漏未主張即遮斷 | 概念過於抽象 | 「事實關係」範圍不易界定 |

**考試提醒**：實務採舊說，但作答時應兩說並陳，並說明對既判力範圍、訴之變更追加、重複起訴之影響。

### 三、訴訟標的理論對各制度之影響 ★★

| 影響制度 | 舊說結論 | 新說結論 |
|---------|---------|---------|
| 重複起訴（§253） | 僅同一請求權始為同一事件 | 同一聲明（＋同一事實）即為同一事件，範圍較廣 |
| 訴之變更追加（§255） | 更換請求權基礎＝訴之變更 | 聲明不變則非訴之變更 |
| 既判力客觀範圍（§400） | 僅及於該請求權 | 及於同一聲明所涵蓋之全部請求權 |
| 訴之合併（競合合併） | 數請求權合併主張有實益 | 本質為一個訴訟標的，無合併問題 |

### 四、重複起訴禁止 §253 ★★

| 項目 | 內容 |
|-----|------|
| 條文 | 當事人不得就已起訴之事件，於訴訟繫屬中更行起訴 |
| 判斷標準 | 同一事件＝當事人同一＋訴訟標的同一＋聲明同一（或相反） |
| 違反效果 | 後訴以裁定駁回 |
| 與既判力之關係 | 重複起訴禁止＝「訴訟繫屬中」之規範；既判力＝「判決確定後」之規範 |
| 實務爭議 | 前後訴訟標的不同但有牽連者，是否構成重複起訴？通說認為不構成 |

## 參、訴的類型比較 ★★

### 一、三種訴訟類型

| 比較項目 | 給付之訴 | 確認之訴 | 形成之訴 |
|---------|---------|---------|---------|
| 目的 | 請求被告為一定給付 | 確認法律關係存在或不存在 | 請求法院以判決形成新法律關係 |
| 聲明範例 | 「被告應給付原告100萬元」 | 「確認兩造間僱傭關係存在」 | 「准原告與被告離婚」 |
| 訴之利益 | 原則上有給付請求即有 | 須有「確認利益」（§247） | 以法律有特別規定者為限 |
| 判決效力 | 確認力＋執行力 | 僅有確認力 | 確認力＋形成力 |
| 執行力 | 有 | 無 | 無（但法律關係已變動） |
| 形成力 | 無 | 無 | 有（對世效力） |
| 典型考法 | 所有權移轉登記、損害賠償 | 確認僱傭關係、確認本票債權不存在 | 離婚、撤銷股東會決議 |

### 二、確認之訴的確認利益 ★

四要件（口訣：「法不過去即受」）：
1. **法律關係**：須以法律關係為標的（非單純事實）
2. **不限於現在**：過去法律關係有確認利益時亦可
3. **即受確認判決之法律上利益**：原告法律上地位有不安危險，且能以確認判決除去
4. **無更適當之訴訟類型**：能提給付之訴者，原則上不得提確認之訴（補充性）

### 三、將來給付之訴 §246

| 項目 | 內容 |
|-----|------|
| 定義 | 請求被告於將來為一定給付之訴 |
| 要件 | 有預為請求之必要（例如：清償期尚未屆至但有不履行之虞） |
| 執行力 | 判決確定後，俟條件成就或期限屆至始得執行 |
| 典型案例 | 承租人訴請出租人於租約屆滿時返還押租金 |

### 四、形成之訴特殊問題

| 項目 | 說明 |
|-----|------|
| 形成權行使 | 形成之訴以法律明定須以訴訟方式行使形成權者為限（訴訟法上之形成之訴） |
| 形成力之時點 | 判決確定時發生形成效力 |
| 對世效力 | 形成判決對第三人亦生效力（不限於當事人間） |
| 除斥期間 | 許多形成之訴有除斥期間限制（如：撤銷股東會決議須於決議後30日內提起，公司法§189） |

## 肆、共同訴訟速查 ★★

### 一、類型比較表

| 比較項目 | 普通共同訴訟 §53 | 固有必要共同訴訟 | 類似必要共同訴訟 |
|---------|----------------|----------------|----------------|
| 定義 | 數人因同一或同種類事實上法律上原因而為共同訴訟 | 依法律規定必須一同起訴/被訴，否則當事人不適格 | 數人雖非必須共同起訴，但若共同起訴則判決須合一確定 |
| 當事人適格 | 各自獨立判斷 | 全體一同起訴/被訴始適格 | 各自均有適格，但共同起訴時須合一確定 |
| 訴訟行為 | 各人獨立（§55） | 合一確定：有利行為及於全體、不利行為全體同意始生效（§56） | 同固有必要，準用§56 |
| 裁判 | 可分別裁判 | 須合一確定 | 須合一確定 |
| 典型案例 | 數被害人分別向加害人求償 | 公同共有人全體起訴請求返還共有物、撤銷詐害行為之訴（債務人＋受益人） | 數債權人中之一人提起撤銷股東會決議之訴、必要共同訴訟之一造提上訴 |

### 二、§56 合一確定原則 ★

| 規則 | 內容 |
|-----|-----|
| 有利行為效力擴張 | 共同訴訟人中一人之行為有利於共同訴訟人者，其效力及於全體 |
| 不利行為效力不及 | 不利益行為對全體不生效力 |
| 典型適用 | 一人上訴→效力及於全體；一人撤回上訴→不生效力 |

### 三、共同訴訟判斷流程

```
第一步：是否為「必須」一同起訴/被訴？
  → 是 → 固有必要共同訴訟（全體共同始適格）
  → 否 → 進入第二步

第二步：共同起訴時判決是否須合一確定？
  → 是 → 類似必要共同訴訟
  → 否 → 普通共同訴訟
```

### 四、常考固有必要共同訴訟類型

| 類型 | 全體為原告或被告 | 說明 |
|-----|----------------|------|
| 公同共有物之處分/返還 | 全體公同共有人為原告 | 處分公同共有物須共有人全體同意 |
| 撤銷詐害行為之訴（民§244） | 債務人＋受益人為共同被告 | 須同時以二者為被告 |
| 婚姻無效之訴 | 夫妻二人分為兩造 | 身分關係訴訟之特殊要求 |
| 確認親子關係訴訟 | 父母子女間 | 身分關係訴訟之特殊要求 |

## 伍、訴訟參加速查 ★★

### 一、三種參加類型比較表

| 比較項目 | 獨立參加（主參加）§54 | 從參加（輔助參加）§58 | 被告參加（訴訟告知）§65 |
|---------|-------------------|-------------------|---------------------|
| 參加人地位 | 獨立之當事人 | 輔助一造之從屬地位 | 被告知人決定是否參加 |
| 參加要件 | 對訴訟標的全部或一部有權利主張 | 對訴訟結果有法律上利害關係 | 當事人訴訟告知第三人 |
| 參加目的 | 保護自己權利 | 輔助被參加人勝訴，避免日後對己不利 | 使被告知人有參加機會，日後受參加效力拘束 |
| 訴訟行為效力 | 獨立進行訴訟行為 | 不得與被參加人之行為牴觸（§61） | 視其參加與否而定 |
| 參加效力 | 判決對其有既判力 | 產生「參加效力」（§63），拘束從參加人與被參加人間 | 告知後不參加仍受§63效力（§67） |
| 典型案例 | 甲乙爭房屋所有權，丙主張自己才是所有人 | 承租人輔助出租人（所有權人）對抗第三人之所有物返還訴訟 | 被告敗訴可能對第三人求償，告知該第三人參加 |

### 二、參加效力（§63）★

- 內容：從參加人於其後與被參加人間之訴訟，不得主張被參加人之訴訟行為不當或裁判不當。
- 例外：§63但書排除情形（從參加人受限於訴訟進行程度無法提出攻防方法等）。
- 白話：從參加人日後不能翻臉說「當初你打官司打得不好」。

### 三、從參加（輔助參加）要件深入 ★

| 項目 | 說明 |
|-----|------|
| 法律上利害關係 | 須為「法律上」利害關係，非僅事實上、經濟上利害關係 |
| 判斷標準 | 本訴結果是否在法律上影響從參加人之權利義務 |
| 合法審查 | 他造得異議（§60），法院以裁定判斷參加是否合法 |
| 輔助行為限制 | 不得為捨棄、認諾、撤回、和解等處分行為 |

### 四、訴訟告知的實務重要性 ★

| 場景 | 說明 |
|-----|------|
| 買賣瑕疵連環訴訟 | 買受人A告出賣人B→B告知上游供應商C參加→C若不參加，日後B向C求償時，C受參加效力拘束 |
| 保證人訴訟 | 債權人告主債務人→主債務人告知保證人參加 |
| 告知之效力 | 經合法告知而不參加者，視為已參加（§67），受§63參加效力拘束 |

## 陸、證據法則速查 ★★★

### 一、舉證責任分配 §277 ★★★

**基本原則**：當事人主張有利於己之事實者，就其事實有舉證之責任。

| 概念 | 內容 |
|-----|-----|
| 客觀舉證責任 | 事實真偽不明時，由負舉證責任之一方承擔不利益（敗訴風險） |
| 主觀舉證責任 | 當事人為避免敗訴而提出證據之行為責任 |
| 分配原則 | 主張權利存在者→就權利發生要件負舉證責任；主張權利消滅/排除者→就消滅/排除要件負舉證責任 |
| §277但書 | 依情形顯失公平者，法院得減輕或免除舉證責任 |
| 但書適用情形 | 醫療訴訟、公害訴訟、證據偏在一方、武器不對等 |

### 二、自認 §279 ★★

| 項目 | 內容 |
|-----|-----|
| 定義 | 當事人對他造主張之事實，於言詞辯論時明示承認 |
| 效果 | 他造無庸舉證（§279 I）；法院應以之為裁判基礎（拘束法院） |
| 擬制自認 | 不爭執且經審判長告知仍不爭執（§280） |
| 撤銷自認 | 須證明與事實不符且係出於錯誤（§279 III） |
| 不適用 | 法律上主張（非事實）、涉及公益事項、訴訟外陳述 |

### 三、文書提出命令 §342 以下 ★

| 步驟 | 內容 |
|-----|-----|
| 聲請 | 舉證人聲請法院命他造或第三人提出文書 |
| 提出義務 | §344 列舉他造負有提出義務之情形（引用文書、利益文書、商業帳簿、法律上有提出義務之文書、就訴訟關係所作文書） |
| 違反效果（他造）| 法院得認舉證人關於該文書之主張為真實（§345） |
| 違反效果（第三人）| 科處罰鍰（§349） |

### 四、證據契約與自由心證 ★

| 項目 | 說明 |
|-----|-----|
| 自由心證 | 法院依調查證據之結果，斟酌全辯論意旨，依自由心證判斷事實之真偽（§222 I） |
| 心證門檻 | 通說採「高度蓋然性」（非科學確定性） |
| 證據契約 | 當事人約定特定證據方法或證明度之契約（實務態度保守） |

### 五、表見證明與事實推定 ★

| 概念 | 說明 |
|-----|------|
| 表見證明（初步推定） | 依經驗法則，由已知事實推認待證事實之存在；他造得提出反證推翻 |
| 法律上推定 | 法律明文規定由特定事實推定另一事實（如：民法§184 II推定過失） |
| 事實上推定 | 法院依經驗法則由間接事實推認待證事實 |
| 與舉證責任之關係 | 推定不移轉舉證責任，但使他造負提出反證之負擔（行為責任轉換） |

### 六、鑑定與勘驗

| 項目 | 鑑定 §324 以下 | 勘驗 §364 以下 |
|-----|---------------|---------------|
| 定義 | 法院指定具特別學識經驗之人就特定事項報告意見 | 法院以五官直接觀察物體或場所之狀態 |
| 性質 | 證據方法之一 | 證據方法之一 |
| 鑑定人義務 | 具結（§334）、到場陳述 | — |
| 當事人指定 | 法院得依當事人合意指定鑑定人（§326） | — |

## 柒、既判力三維度 ★★★

### 一、整體架構

既判力＝確定終局判決對後訴之拘束力，分三個維度分析。

### 二、客觀範圍 §400 ★★★

| 項目 | 內容 |
|-----|-----|
| 原則 | 既判力僅及於「訴訟標的」經裁判者（§400 I） |
| 判決理由 | 原則上不生既判力（判決理由中之判斷僅為攻防方法） |
| 抵銷抗辯例外 | 以主張抵銷之額度為限，有既判力（§400 II） |
| 一部請求 | 明示一部請求→其餘部分不受既判力遮斷；默示一部請求→有爭議（實務傾向遮斷） |

### 三、主觀範圍 §401 ★★

| 對象 | 說明 | 條文 |
|-----|------|------|
| 當事人 | 既判力及於雙方當事人 | §401 I |
| 繼受人 | 確定判決對當事人之繼受人亦有效力 | §401 I |
| 訴訟繫屬後為當事人或其繼受人占有請求標的物者 | 擴張至占有人 | §401 I |
| 為他人為原告或被告者（訴訟擔當） | 該他人亦受既判力拘束 | §401 II |

### 四、時間範圍（基準時）★★★

| 項目 | 內容 |
|-----|-----|
| 基準時 | 事實審言詞辯論終結時 |
| 遮斷效 | 基準時前存在之攻防方法，不論是否已提出，均受既判力遮斷，不得於後訴再行主張 |
| 基準時後新事由 | 不受既判力遮斷，可另訴主張 |
| 考試重點 | 判斷某事由發生在基準時「前」或「後」，決定是否受遮斷 |

### 五、既判力相關概念比較

| 概念 | 性質 | 範圍 | 條文基礎 |
|-----|------|------|---------|
| 既判力 | 法律明定 | 訴訟標的 | §400、§401 |
| 爭點效 | 學說/判例 | 重要爭點經當事人充分攻防且法院為判斷 | 無明文（最高法院承認但限縮適用） |
| 反射效 | 學說 | 對第三人之事實上不利影響 | 無明文 |

### 六、一部請求與既判力 ★★

| 情形 | 既判力範圍 | 說明 |
|-----|-----------|------|
| 明示一部請求 | 僅及於該一部請求 | 原告明確表示「先就100萬元中之50萬元起訴」，其餘50萬元不受遮斷 |
| 默示一部請求 | 爭議：是否遮斷其餘部分 | 實務傾向認為：未明示者視為就全部請求起訴，敗訴時全部受遮斷 |
| 擴張聲明（§255 I③） | 非訴之變更 | 一部請求後擴張聲明至全部，屬聲明之擴張，不構成訴之變更追加 |

### 七、爭點效四要件 ★★

| 要件 | 說明 |
|-----|------|
| (1) 重要爭點 | 該爭點為前訴判決之重要基礎（非枝微末節） |
| (2) 當事人充分攻防 | 雙方已就該爭點為充分之舉證與辯論 |
| (3) 法院為實質判斷 | 法院確實就該爭點為實質之認定（非迴避或附帶提及） |
| (4) 非顯失公平 | 適用爭點效不致對當事人產生顯失公平之結果 |

**實務見解**：最高法院 96 年台上字第 307 號判決承認爭點效理論，但要求嚴格審查四要件。

## 捌、保全程序速查 ★★

### 一、三種保全處分比較表

| 比較項目 | 假扣押 §522 | 假處分 §532 | 定暫時狀態處分 §538 |
|---------|-----------|-----------|-------------------|
| 保全對象 | 金錢請求或得易為金錢請求之請求 | 金錢以外之請求 | 爭執之法律關係 |
| 目的 | 防止債務人脫產，保全將來強制執行 | 防止現狀變更致日後不能執行 | 防止重大損害或急迫危險，定暫時狀態 |
| 釋明要件 | 請求原因＋假扣押原因（§523） | 請求原因＋假處分原因 | 爭執法律關係＋防止重大損害或急迫危險之必要（§538） |
| 擔保 | 法院得命供擔保後為之；債權人未充分釋明時以擔保補足（§526） | 同假扣押 | 法院得命供擔保 |
| 管轄 | 本案管轄法院或假扣押標的所在地法院（§524） | 本案管轄法院或爭執標的所在地法院 | 本案管轄法院 |
| 執行期間 | 裁定送達後30日內執行（§529） | 同假扣押 | 裁定確定或送達後執行 |
| 典型案例 | 甲怕乙脫產，聲請假扣押乙之銀行存款 | 甲怕乙拆屋，聲請假處分禁止拆除 | 僱傭關係爭議，聲請定暫時狀態命雇主繼續給付薪資 |

### 二、假扣押三要件 ★

1. **請求原因**：對債務人有金錢債權（或得易為金錢之請求）
2. **假扣押原因**：日後有不能強制執行或甚難執行之虞（§523）
3. **釋明**：須釋明請求及原因；釋明不足得以擔保補之（§526）

### 三、撤銷保全裁定

| 事由 | 條文 |
|-----|------|
| 債權人未於期限內起訴 | §529 |
| 假扣押/假處分原因消滅 | §530 |
| 債務人供擔保聲請撤銷 | §530 |
| 本案敗訴確定 | §530 |

### 四、保全程序之程序要點

| 項目 | 內容 |
|-----|------|
| 釋明 vs 證明 | 保全程序僅需「釋明」（使法院產生薄弱心證即可），非「證明」（高度蓋然性） |
| 擔保之性質 | 為日後假扣押/假處分錯誤時賠償相對人損害之擔保 |
| 假扣押損害賠償 | 假扣押裁定因自始不當而撤銷，或債權人未於期限內起訴致撤銷者，債務人得請求損害賠償（§531） |
| 抗告 | 對保全裁定得抗告（§528） |
| 緊急性 | 保全程序強調迅速，法院得不經言詞辯論即為裁定 |

### 五、定暫時狀態處分之特殊審查 ★

| 審查要素 | 說明 |
|---------|------|
| 本案勝訴可能性 | 聲請人於本案訴訟中勝訴之蓋然性 |
| 急迫危險 | 不定暫時狀態是否造成無法回復之重大損害 |
| 利益衡量 | 准許處分與否對雙方利益之影響比較 |
| 公共利益 | 是否涉及公共利益之維護 |
| 勞動事件法 §49 | 勞動事件中，勞工得以釋明有相當理由聲請定暫時狀態處分，法院於審酌時應特別考量勞工之生計 |

## 拾壹、易混淆概念對照表

### 一、訴之變更 vs 訴之追加 ★

| 比較項目 | 訴之變更 | 訴之追加 |
|---------|---------|---------|
| 定義 | 以新訴訟標的取代原訴訟標的 | 在原訴訟標的外，增加新訴訟標的 |
| 效果 | 原訴消滅，僅審新訴 | 原訴與新訴併存，一同審理 |
| 共同條文 | §255 均規範 | §255 均規範 |
| §255 I本文 | 被告同意者，得為之 | 同左 |
| §255 I但書例外 | 不變更訴訟標的但補充或更正事實上法律上陳述（第2款）、請求基礎事實同一（第2款）、擴張或減縮應受判決事項之聲明（第3款）、不甚礙被告防禦及訴訟終結（第7款） | 同左 |
| 考試注意 | 擴張減縮聲明（第3款）嚴格而言非訴之變更追加，而是聲明之擴張減縮 | 主觀追加（追加新當事人）有更嚴格限制 |

### 二、既判力 vs 爭點效 ★★

| 比較項目 | 既判力 | 爭點效 |
|---------|-------|-------|
| 法律依據 | §400、§401明文規定 | 無明文，源自判例法理 |
| 拘束對象 | 訴訟標的（判決主文） | 判決理由中之重要爭點 |
| 發生要件 | 確定終局判決即生 | (1)重要爭點 (2)當事人充分舉證攻防 (3)法院為實質判斷 (4)非顯失公平 |
| 範圍 | 限於訴訟標的 | 及於重要爭點之判斷 |
| 實務態度 | 完全承認 | 最高法院有條件承認（限縮適用） |
| 考法 | 直接引用條文 | 須論述四要件，並討論採否之影響 |

### 三、假扣押 vs 假處分 vs 定暫時狀態處分

| 比較項目 | 假扣押 | 假處分 | 定暫時狀態處分 |
|---------|-------|-------|--------------|
| 請求類型 | 金錢請求 | 非金錢請求 | 法律關係有爭執 |
| 保全方法 | 查封債務人財產 | 禁止現狀變更 | 暫定法律關係 |
| 急迫程度 | 中（防脫產） | 高（現狀將變更） | 最高（重大損害或急迫危險） |
| 審理密度 | 低（釋明即可） | 中 | 高（須審酌保全必要性） |

### 四、訴訟上和解 vs 訴訟外和解 vs 調解

| 比較項目 | 訴訟上和解 §377 | 訴訟外和解 | 調解 §403以下 |
|---------|---------------|-----------|-------------|
| 時點 | 訴訟繫屬中 | 任何時點 | 起訴前（強制調解）或訴訟中 |
| 效力 | 與確定判決有同一效力（§380） | 僅有契約效力 | 與確定判決有同一效力（§416） |
| 執行力 | 有 | 無 | 有 |
| 救濟 | 得以再審之訴或宣告和解無效之訴撤銷 | 依民法撤銷或無效 | 宣告調解無效之訴 |

### 五、訴之合併類型

| 類型 | 說明 | 舉例 |
|-----|------|------|
| 單純合併 | 數請求間無依存關係，分別獨立請求 | 請求返還借款100萬＋損害賠償50萬 |
| 預備合併 | 先位聲明有理由時，不審備位聲明 | 先位：請求給付買賣價金；備位：請求返還不當得利 |
| 選擇合併 | 數請求中擇一為有理由之判決 | 依侵權或債務不履行擇一請求賠償（舊說下有實益） |
| 重疊合併（競合合併） | 以同一給付為目的，主張數個訴訟標的 | 依買賣契約＋不當得利請求返還同一筆款項 |

### 六、當事人恆定 vs 訴訟承擔

| 比較項目 | 當事人恆定 §254 | 訴訟承擔 §254 II |
|---------|----------------|-----------------|

| 效力 | 判決效力及於繼受人（§401） | 繼受人取代原當事人地位 |
| 目的 | 防止當事人藉由移轉權利拖延訴訟 | 使真正權利人直接參與訴訟 |

### 七、訴之撤回 vs 捨棄認諾

| 比較項目 | 訴之撤回 §262 | 捨棄 §384 | 認諾 §384 |
|---------|-------------|----------|----------|
| 定義 | 原告撤回起訴，訴訟繫屬消滅 | 原告於言詞辯論時捨棄其訴訟上之請求 | 被告於言詞辯論時認諾原告之訴訟上請求 |
| 效果 | 視為未起訴（§263 I） | 法院應為原告敗訴之判決 | 法院應為被告敗訴之判決 |
| 時點限制 | 被告已為本案言詞辯論者，須得被告同意（§262 I但書） | 言詞辯論中 | 言詞辯論中 |
| 再訴限制 | 撤回後不得再提同一之訴（§263 II） | 有既判力 | 有既判力 |
| 性質 | 訴訟行為（程序法） | 訴訟行為 | 訴訟行為 |

### 八、闡明義務 §199 ★

| 項目 | 說明 |
|-----|------|
| 內容 | 審判長應向當事人發問或告知，令其就事實上及法律上之陳述為必要之補充或敘明 |
| 適時提出 | §196：攻擊防禦方法應依訴訟進行程度適時提出 |
| 法律觀點闡明 | 法院認定之法律觀點與當事人不同時，應曉諭當事人（防止突襲裁判） |
