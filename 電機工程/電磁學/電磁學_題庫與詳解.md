### 題目 1：柱座標系中的梯度計算
**來源**：台大電機 經典題型
**難度**：★★
**主題**：向量分析 — 梯度(Gradient)

**題目**：

已知純量場 V(ρ, φ, z) = ρ²cos(2φ) + z²ρ，求梯度 ∇V 在點 P(2, π/4, 3) 處的值。

**詳解**：

**解題思路**：在柱座標(Cylindrical Coordinates)中套用梯度公式，分別對 ρ、φ、z 偏微分。

**步驟**：

柱座標梯度公式：

$$\nabla V = \frac{\partial V}{\partial \rho}\hat{\rho} + \frac{1}{\rho}\frac{\partial V}{\partial \phi}\hat{\phi} + \frac{\partial V}{\partial z}\hat{z}$$

分別計算各分量：

(1) ρ 分量：

$$\frac{\partial V}{\partial \rho} = 2\rho\cos(2\phi) + z^2$$

(2) φ 分量：

$$\frac{1}{\rho}\frac{\partial V}{\partial \phi} = \frac{1}{\rho}\cdot(-2\rho^2\sin(2\phi)) = -2\rho\sin(2\phi)$$

(3) z 分量：

$$\frac{\partial V}{\partial z} = 2z\rho$$

因此：

$$\nabla V = (2\rho\cos 2\phi + z^2)\hat{\rho} + (-2\rho\sin 2\phi)\hat{\phi} + 2z\rho\,\hat{z}$$

代入 P(ρ=2, φ=π/4, z=3)：

- cos(2·π/4) = cos(π/2) = 0
- sin(2·π/4) = sin(π/2) = 1

ρ 分量：2(2)(0) + 9 = 9

φ 分量：-2(2)(1) = -4

z 分量：2(3)(2) = 12

**答案**：

> ∇V|_P = 9ρ̂ − 4φ̂ + 12ẑ

其大小為 |∇V| = √(81 + 16 + 144) = √241 ≈ 15.52

**延伸思考**：∇V 的方向即為 V 增加最快的方向。若要求 V 在某方向 â 的方向導數(Directional Derivative)，只需計算 ∇V · â。

### 題目 2：球座標系中的散度
**來源**：台大電機 經典題型
**難度**：★★
**主題**：向量分析 — 散度(Divergence)

**題目**：

已知向量場 **D** = (3r²sinθ)r̂ + (r·cosθ·cosφ)θ̂ （球座標），求 ∇·**D**，並在點 Q(2, π/6, π/3) 處求值。

**詳解**：

**解題思路**：套用球座標散度公式，注意各項的 scale factor。

**步驟**：

球座標(Spherical Coordinates)散度公式：

$$\nabla\cdot\mathbf{D} = \frac{1}{r^2}\frac{\partial}{\partial r}(r^2 D_r) + \frac{1}{r\sin\theta}\frac{\partial}{\partial\theta}(\sin\theta\, D_\theta) + \frac{1}{r\sin\theta}\frac{\partial D_\phi}{\partial\phi}$$

其中 D_r = 3r²sinθ，D_θ = r·cosθ·cosφ，D_φ = 0。

(1) 第一項：

$$\frac{1}{r^2}\frac{\partial}{\partial r}(r^2 \cdot 3r^2\sin\theta) = \frac{1}{r^2}\frac{\partial}{\partial r}(3r^4\sin\theta) = \frac{1}{r^2}\cdot 12r^3\sin\theta = 12r\sin\theta$$

(2) 第二項：

$$\frac{1}{r\sin\theta}\frac{\partial}{\partial\theta}(\sin\theta \cdot r\cos\theta\cos\phi)$$

$$= \frac{\cos\phi}{r\sin\theta}\cdot r\frac{\partial}{\partial\theta}(\sin\theta\cos\theta)$$

$$= \frac{\cos\phi}{\sin\theta}\cdot\frac{\partial}{\partial\theta}\left(\frac{\sin 2\theta}{2}\right)$$

$$= \frac{\cos\phi}{\sin\theta}\cdot\cos 2\theta$$

(3) 第三項：D_φ = 0，故此項為 0。

因此：

$$\nabla\cdot\mathbf{D} = 12r\sin\theta + \frac{\cos 2\theta\cdot\cos\phi}{\sin\theta}$$

代入 Q(r=2, θ=π/6, φ=π/3)：

- sinθ = sin(π/6) = 1/2
- cos2θ = cos(π/3) = 1/2
- cosφ = cos(π/3) = 1/2

第一項：12(2)(1/2) = 12

第二項：(1/2)(1/2)/(1/2) = 1/2

**答案**：

> ∇·**D**|_Q = 12 + 0.5 = 12.5

**延伸思考**：由散度定理(Divergence Theorem)，∮**D**·d**S** = ∫(∇·**D**)dv。若 **D** 為電通密度(Electric Flux Density)，則 ∇·**D** = ρ_v（體電荷密度）。

### 題目 3：旋度與斯托克斯定理驗證
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：向量分析 — 旋度(Curl)與斯托克斯定理(Stokes' Theorem)

**題目**：

已知向量場 **F** = xy²x̂ + yz²ŷ + zx²ẑ（直角座標）。
(a) 計算 ∇ × **F**。
(b) 取 z = 0 平面上以原點為圓心、半徑 a 的圓為路徑 C，驗證斯托克斯定理 ∮**F**·d**l** = ∫∫(∇ × **F**)·d**S**。

**詳解**：

**解題思路**：(a) 直接展開行列式。(b) 分別計算線積分和面積分做比較。

**步驟**：

**(a) 計算旋度**

$$\nabla\times\mathbf{F} = \begin{vmatrix} \hat{x} & \hat{y} & \hat{z} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ xy^2 & yz^2 & zx^2 \end{vmatrix}$$

x 分量：∂(zx²)/∂y − ∂(yz²)/∂z = 0 − 2yz = −2yz

y 分量：∂(xy²)/∂z − ∂(zx²)/∂x = 0 − 2xz = −2xz

z 分量：∂(yz²)/∂x − ∂(xy²)/∂y = 0 − 2xy = −2xy

$$\nabla\times\mathbf{F} = -2yz\,\hat{x} - 2xz\,\hat{y} - 2xy\,\hat{z}$$

**(b) 驗證斯托克斯定理**

在 z = 0 平面上，d**S** = dS ẑ（法向量朝 +z）。

面積分（右側）：

$$\iint(\nabla\times\mathbf{F})\cdot d\mathbf{S} = \iint (-2xy)\,dS$$

z = 0 平面上旋度的 z 分量為 -2xy。轉柱座標 x = ρcosφ, y = ρsinφ：

$$= \int_0^{2\pi}\int_0^a (-2\rho\cos\phi\cdot\rho\sin\phi)\,\rho\,d\rho\,d\phi$$

$$= -2\int_0^{2\pi}\sin\phi\cos\phi\,d\phi \int_0^a \rho^3\,d\rho$$

$$= -2\int_0^{2\pi}\frac{\sin 2\phi}{2}\,d\phi \cdot \frac{a^4}{4}$$

$$= -\frac{a^4}{4}\left[-\frac{\cos 2\phi}{2}\right]_0^{2\pi} = -\frac{a^4}{4}\cdot\left(-\frac{1}{2}+\frac{1}{2}\right) = 0$$

線積分（左側）：

在 z = 0 平面上的圓：x = acosφ, y = asinφ, z = 0。

d**l** = (−asinφ x̂ + acosφ ŷ) dφ

**F**|_{z=0} = xy² x̂ + 0 ŷ + 0 ẑ = a³cos(φ)sin²(φ) x̂

$$\oint\mathbf{F}\cdot d\mathbf{l} = \int_0^{2\pi} a^3\cos\phi\sin^2\phi\cdot(-a\sin\phi)\,d\phi$$

$$= -a^4\int_0^{2\pi}\cos\phi\sin^3\phi\,d\phi$$

令 u = sinφ, du = cosφ dφ。在 φ: 0→2π 中 sinφ 回到原值，故積分為 0。

**答案**：

> 線積分 = 面積分 = 0，斯托克斯定理得證。

**延伸思考**：若 ∇ × **F** = 0 則 **F** 為保守場(Conservative Field)，線積分與路徑無關。本題 **F** 並非保守場，但在此特定路徑與曲面上積分恰為 0。

### 題目 4：庫侖定律 — 有限長線電荷的電場
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 庫侖定律(Coulomb's Law)

**題目**：

一均勻線電荷密度 ρ_L (C/m) 分布在 z 軸上 −L ≤ z ≤ L 的區段。求在 ρ 軸上距原點 ρ₀ 處的電場 **E**。當 L → ∞ 時化簡結果。

**詳解**：

**解題思路**：用微分庫侖定律逐段積分，利用對稱性消去 z 分量。

**步驟**：

取觀測點 P 在 ρ 軸上 (ρ₀, 0, 0)，源點 Q 在 z 軸上 (0, 0, z')。

位置向量：**R** = ρ₀ρ̂ − z'ẑ

|**R**| = √(ρ₀² + z'²)

微分電場：

$$d\mathbf{E} = \frac{\rho_L\,dz'}{4\pi\varepsilon_0}\cdot\frac{\rho_0\hat{\rho} - z'\hat{z}}{(\rho_0^2 + z'^2)^{3/2}}$$

由對稱性，z' 從 −L 到 +L 積分時 ẑ 分量互相抵消（奇函數積分為零）。

只剩 ρ̂ 分量：

$$E_\rho = \frac{\rho_L\rho_0}{4\pi\varepsilon_0}\int_{-L}^{L}\frac{dz'}{(\rho_0^2 + z'^2)^{3/2}}$$

令 z' = ρ₀ tanα, dz' = ρ₀ sec²α dα：

$$(\rho_0^2 + z'^2)^{3/2} = \rho_0^3\sec^3\alpha$$

$$E_\rho = \frac{\rho_L\rho_0}{4\pi\varepsilon_0}\int_{-\alpha_0}^{\alpha_0}\frac{\rho_0\sec^2\alpha\,d\alpha}{\rho_0^3\sec^3\alpha} = \frac{\rho_L}{4\pi\varepsilon_0\rho_0}\int_{-\alpha_0}^{\alpha_0}\cos\alpha\,d\alpha$$

$$= \frac{\rho_L}{4\pi\varepsilon_0\rho_0}\cdot 2\sin\alpha_0$$

其中 tanα₀ = L/ρ₀，故 sinα₀ = L/√(L² + ρ₀²)。

$$E_\rho = \frac{\rho_L}{4\pi\varepsilon_0\rho_0}\cdot\frac{2L}{\sqrt{L^2+\rho_0^2}}$$

當 L → ∞ 時，2L/√(L²+ρ₀²) → 2：

**答案**：

> 有限長線電荷：**E** = [ρ_L · L / (2πε₀ρ₀√(L²+ρ₀²))] ρ̂
>
> 無限長線電荷(L→∞)：**E** = ρ_L / (2πε₀ρ₀) ρ̂

**延伸思考**：無限長線電荷的結果也可用高斯定律直接得到（取同軸柱面高斯面）。有限長度時沒有足夠對稱性，必須用庫侖定律積分。

### 題目 5：面電荷的電場 — 有限圓盤
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 面電荷分布

**題目**：

一均勻面電荷密度 ρ_s (C/m²) 分布在 z = 0 平面上半徑為 a 的圓盤。求 z 軸上任意點 (0, 0, h)（h > 0）的電場。當 a → ∞ 時化簡結果。

**詳解**：

**解題思路**：利用環狀微元素積分，由對稱性知 ρ̂ 方向分量抵消，只剩 ẑ 分量。

**步驟**：

取圓盤上的微元素面積 dS' = ρ' dρ' dφ'，位於 (ρ', φ', 0)。

觀測點 P(0, 0, h)，源點到觀測點向量：

**R** = −ρ'ρ̂' + hẑ，|**R**| = √(ρ'² + h²)

由圓對稱性，ρ̂ 分量在 φ' 積分後為零。

z 分量微分電場：

$$dE_z = \frac{\rho_s\,\rho'\,d\rho'\,d\phi'}{4\pi\varepsilon_0}\cdot\frac{h}{(\rho'^2+h^2)^{3/2}}$$

對 φ' 積分 0 到 2π 得 2π：

$$E_z = \frac{\rho_s h}{2\varepsilon_0}\int_0^a\frac{\rho'\,d\rho'}{(\rho'^2+h^2)^{3/2}}$$

令 u = ρ'² + h²，du = 2ρ' dρ'：

$$E_z = \frac{\rho_s h}{2\varepsilon_0}\cdot\frac{1}{2}\int_{h^2}^{a^2+h^2} u^{-3/2}\,du$$

$$= \frac{\rho_s h}{4\varepsilon_0}\left[-2u^{-1/2}\right]_{h^2}^{a^2+h^2}$$

$$= \frac{\rho_s h}{4\varepsilon_0}\cdot 2\left(\frac{1}{h} - \frac{1}{\sqrt{a^2+h^2}}\right)$$

$$= \frac{\rho_s}{2\varepsilon_0}\left(1 - \frac{h}{\sqrt{a^2+h^2}}\right)$$

當 a → ∞：h/√(a²+h²) → 0

**答案**：

> 有限圓盤：**E** = (ρ_s / 2ε₀)[1 − h/√(a²+h²)] ẑ
>
> 無限大平面(a→∞)：**E** = (ρ_s / 2ε₀) ẑ

**延伸思考**：無限大平面的結果也可由高斯定律得到（取扁圓柱高斯面，又稱 pillbox）。注意無限大平面電場為均勻場，與距離無關。

### 題目 6：高斯定律 — 球對稱電荷分布
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 高斯定律(Gauss's Law)

**題目**：

一半徑為 a 的球體，其體電荷密度為 ρ_v = ρ₀(1 − r/a)（0 ≤ r ≤ a），r > a 無電荷。利用高斯定律求 r < a 及 r > a 區域的電場 **E**(r)。

**詳解**：

**解題思路**：球對稱問題，取球面為高斯面(Gaussian Surface)。**E** = E_r(r) r̂。

**步驟**：

高斯定律：∮**D**·d**S** = Q_enc

D_r · 4πr² = Q_enc（球面上 **D** 均勻分布）

**Case 1：r < a**

$$Q_{enc} = \int_0^r\int_0^\pi\int_0^{2\pi}\rho_0\left(1-\frac{r'}{a}\right)r'^2\sin\theta\,dr'\,d\theta\,d\phi$$

$$= 4\pi\rho_0\int_0^r\left(r'^2 - \frac{r'^3}{a}\right)dr'$$

$$= 4\pi\rho_0\left[\frac{r'^3}{3} - \frac{r'^4}{4a}\right]_0^r$$

$$= 4\pi\rho_0\left(\frac{r^3}{3} - \frac{r^4}{4a}\right)$$

由 D_r · 4πr² = Q_enc：

$$D_r = \frac{\rho_0}{r^2}\left(\frac{r^3}{3} - \frac{r^4}{4a}\right) = \rho_0\left(\frac{r}{3} - \frac{r^2}{4a}\right)$$

$$E_r = \frac{D_r}{\varepsilon_0} = \frac{\rho_0}{\varepsilon_0}\left(\frac{r}{3} - \frac{r^2}{4a}\right) \quad (r < a)$$

**Case 2：r > a**

$$Q_{enc} = Q_{total} = 4\pi\rho_0\left(\frac{a^3}{3} - \frac{a^4}{4a}\right) = 4\pi\rho_0\left(\frac{a^3}{3} - \frac{a^3}{4}\right) = 4\pi\rho_0\cdot\frac{a^3}{12}$$

$$= \frac{\pi\rho_0 a^3}{3}$$

$$D_r = \frac{Q_{total}}{4\pi r^2} = \frac{\rho_0 a^3}{12r^2}$$

$$E_r = \frac{\rho_0 a^3}{12\varepsilon_0 r^2} \quad (r > a)$$

**驗證**：在 r = a 處兩式應連續：

- 內部：ρ₀(a/3 − a²/4a)/ε₀ = ρ₀(a/3 − a/4)/ε₀ = ρ₀a/12ε₀
- 外部：ρ₀a³/(12ε₀a²) = ρ₀a/12ε₀ ✓

**答案**：

> r < a：**E** = (ρ₀/ε₀)(r/3 − r²/4a) r̂
>
> r > a：**E** = ρ₀a³/(12ε₀r²) r̂

**延伸思考**：在 r > a 區域電場與 1/r² 成正比，等效於總電荷集中在原點的點電荷。可進一步求 E_r 的最大值位置，令 dE_r/dr = 0 得 r = 2a/3。

### 題目 7：高斯定律 — 同軸柱對稱
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 高斯定律應用

**題目**：

一無限長同軸結構：內柱半徑 a 帶面電荷密度 ρ_sa，外柱半徑 b 帶面電荷密度 ρ_sb。若系統總電荷為零（即 ρ_sa · 2πa + ρ_sb · 2πb = 0），求各區域（ρ < a, a < ρ < b, ρ > b）的電場。

**詳解**：

**解題思路**：柱對稱問題，取同軸圓柱面為高斯面，高度 L。

**步驟**：

由電荷中性條件：ρ_sb = −ρ_sa · a/b

取高度 L 的同軸圓柱面為高斯面。端面上 **E** ⊥ d**S**，無貢獻。側面上 **E** = E_ρ ρ̂。

$$\oint\mathbf{D}\cdot d\mathbf{S} = D_\rho\cdot 2\pi\rho L = Q_{enc}$$

**Case 1：ρ < a**

Q_enc = 0（無包圍電荷）

$$E_\rho = 0 \quad (\rho < a)$$

**Case 2：a < ρ < b**

Q_enc = ρ_sa · 2πa · L

$$D_\rho\cdot 2\pi\rho L = \rho_{sa}\cdot 2\pi a L$$

$$E_\rho = \frac{a\rho_{sa}}{\varepsilon_0\rho} \quad (a < \rho < b)$$

**Case 3：ρ > b**

Q_enc = ρ_sa · 2πaL + ρ_sb · 2πbL = 0（電荷中性）

$$E_\rho = 0 \quad (\rho > b)$$

**答案**：

> ρ < a：**E** = 0
>
> a < ρ < b：**E** = (aρ_sa)/(ε₀ρ) ρ̂
>
> ρ > b：**E** = 0

**延伸思考**：此即同軸電纜(Coaxial Cable)的靜電場分布。電場僅存在於 a < ρ < b 區域，這是同軸電纜具有良好電磁屏蔽特性的基礎。此結構的電容可進一步求得 C = 2πε₀L/ln(b/a)。

### 題目 8：電位計算 — 路徑積分
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 電位(Electric Potential)

**題目**：

已知一球座標中的電場 **E** = (A/r³)r̂ + (Bsinθ/r)θ̂（A, B 為常數）。
(a) 判斷此電場是否為保守場(Conservative Field)。
(b) 若 B = 0，求電位 V(r)，取 V(∞) = 0。

**詳解**：

**解題思路**：保守場等價於 ∇ × **E** = 0。若為保守場才可定義唯一電位。

**步驟**：

**(a) 判斷保守性**

球座標旋度 φ̂ 分量（選取可能不為零的項）：

$$(\nabla\times\mathbf{E})_\phi = \frac{1}{r}\left[\frac{\partial(rE_\theta)}{\partial r} - \frac{\partial E_r}{\partial\theta}\right]$$

$$= \frac{1}{r}\left[\frac{\partial}{\partial r}(r\cdot\frac{B\sin\theta}{r}) - \frac{\partial}{\partial\theta}\left(\frac{A}{r^3}\right)\right]$$

$$= \frac{1}{r}\left[\frac{\partial}{\partial r}(B\sin\theta) - 0\right] = 0$$

檢查 r̂ 分量：

$$(\nabla\times\mathbf{E})_r = \frac{1}{r\sin\theta}\left[\frac{\partial(\sin\theta\cdot E_\theta)}{\partial\theta} - \frac{\partial E_\phi}{\partial\phi}\right]$$

$$= \frac{1}{r\sin\theta}\cdot\frac{\partial}{\partial\theta}\left(\frac{B\sin^2\theta}{r}\right) = \frac{1}{r\sin\theta}\cdot\frac{2B\sin\theta\cos\theta}{r} = \frac{2B\cos\theta}{r^2}$$

此值不為零（除非 B = 0），故一般情況下 **E** 不是保守場。

**(b) B = 0 時**

此時 **E** = (A/r³)r̂，∇ × **E** = 0，為保守場。

$$V(r) = -\int_\infty^r \mathbf{E}\cdot d\mathbf{l} = -\int_\infty^r \frac{A}{r'^3}\,dr' = -A\left[-\frac{1}{2r'^2}\right]_\infty^r = -A\left(-\frac{1}{2r^2}+0\right)$$

$$V(r) = \frac{A}{2r^2}$$

**答案**：

> (a) B ≠ 0 時不是保守場（∇ × **E** ≠ 0）
>
> (b) B = 0 時，V(r) = A/(2r²)

**延伸思考**：只有保守場（無旋場）才能定義純量電位。靜電場必為保守場（∇ × **E** = 0），但時變場中 ∇ × **E** = −∂**B**/∂t ≠ 0，此時需引入向量位 **A** 使得 **E** = −∇V − ∂**A**/∂t。

### 題目 9：電場疊加 — 對稱電荷分布
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 電場疊加原理

**題目**：

四個點電荷 +Q 分別置於正方形的四個頂角，正方形邊長為 a，位於 z = 0 平面，中心在原點。求正方形中心正上方 z = h 處的電場。

**詳解**：

**解題思路**：利用對稱性，四個電荷對中心上方點的水平分量互相抵消，只剩 ẑ 分量。

**步驟**：

正方形頂角到中心的距離 d = a√2/2 = a/√2。

每個點電荷到 P(0,0,h) 的距離 R = √(d² + h²) = √(a²/2 + h²)。

每個電荷產生的電場大小：

$$|\mathbf{E}_i| = \frac{Q}{4\pi\varepsilon_0 R^2} = \frac{Q}{4\pi\varepsilon_0(a^2/2+h^2)}$$

由對稱性，四個電荷的 x̂ 和 ŷ 分量兩兩抵消。

每個電荷貢獻的 ẑ 分量：

$$E_{iz} = |\mathbf{E}_i|\cdot\frac{h}{R} = \frac{Qh}{4\pi\varepsilon_0(a^2/2+h^2)^{3/2}}$$

四個電荷的總電場：

$$E_z = 4\cdot\frac{Qh}{4\pi\varepsilon_0(a^2/2+h^2)^{3/2}}$$

$$= \frac{Qh}{\pi\varepsilon_0(a^2/2+h^2)^{3/2}}$$

**答案**：

> **E** = Qh / [πε₀(a²/2 + h²)^(3/2)] ẑ

**延伸思考**：當 h >> a 時，(a²/2 + h²)^(3/2) ≈ h³，故 **E** ≈ Q/(πε₀h²) ẑ = 4Q/(4πε₀h²) ẑ，等效於總電荷 4Q 的點電荷場，符合物理直覺。當 h = 0 時 **E** = 0，中心處電場為零也符合對稱性。

### 題目 10：電位與電場的關係 — 等位面
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電場 — 電位與電場

**題目**：

兩個等量異號點電荷 +Q 和 −Q 分別位於 (0, 0, d/2) 和 (0, 0, −d/2)。
(a) 寫出空間中任意點 (x, y, z) 的電位 V。
(b) 證明 z = 0 平面為等位面，V = 0。
(c) 求 z = 0 平面上 (ρ, 0, 0) 處的電場（用柱座標表示）。

**詳解**：

**解題思路**：由疊加原理得電位，再取梯度得電場。

**步驟**：

**(a) 電位**

$$V(x,y,z) = \frac{Q}{4\pi\varepsilon_0}\left[\frac{1}{\sqrt{x^2+y^2+(z-d/2)^2}} - \frac{1}{\sqrt{x^2+y^2+(z+d/2)^2}}\right]$$

定義 R₊ = √(x²+y²+(z−d/2)²)，R₋ = √(x²+y²+(z+d/2)²)：

$$V = \frac{Q}{4\pi\varepsilon_0}\left(\frac{1}{R_+} - \frac{1}{R_-}\right)$$

**(b) z = 0 平面**

z = 0 時：R₊ = √(x²+y²+d²/4) = R₋

故 V = Q/(4πε₀)(1/R₊ − 1/R₊) = 0。z = 0 平面確為 V = 0 等位面。

**(c) z = 0 平面上的電場**

由對稱性，在 z = 0 平面上 E_ρ = 0（兩電荷的 ρ 分量大小相等方向相反，互相抵消）。

只剩 E_z 分量。由 E_z = −∂V/∂z：

$$\frac{\partial V}{\partial z}\bigg|_{z=0} = \frac{Q}{4\pi\varepsilon_0}\left[\frac{-(z-d/2)}{R_+^3} - \frac{-(z+d/2)}{R_-^3}\right]_{z=0}$$

z = 0 時 R₊ = R₋ = √(ρ² + d²/4)：

$$= \frac{Q}{4\pi\varepsilon_0}\cdot\frac{1}{(\rho^2+d^2/4)^{3/2}}\left[\frac{d}{2} - \left(-\frac{d}{2}\right)\right]$$

等等，仔細計算：

$$\frac{\partial}{\partial z}\left(\frac{1}{R_+}\right) = \frac{-(z-d/2)}{R_+^3}$$

z = 0 時：= −(−d/2)/(ρ²+d²/4)^(3/2) = (d/2)/(ρ²+d²/4)^(3/2)

$$\frac{\partial}{\partial z}\left(\frac{1}{R_-}\right) = \frac{-(z+d/2)}{R_-^3}$$

z = 0 時：= −(d/2)/(ρ²+d²/4)^(3/2)

$$\frac{\partial V}{\partial z}\bigg|_{z=0} = \frac{Q}{4\pi\varepsilon_0}\left[\frac{d/2}{(\rho^2+d^2/4)^{3/2}} + \frac{d/2}{(\rho^2+d^2/4)^{3/2}}\right]$$

$$= \frac{Qd}{4\pi\varepsilon_0(\rho^2+d^2/4)^{3/2}}$$

$$E_z = -\frac{\partial V}{\partial z} = -\frac{Qd}{4\pi\varepsilon_0(\rho^2+d^2/4)^{3/2}}$$

**答案**：

> (a) V = Q/(4πε₀)(1/R₊ − 1/R₋)
>
> (b) z = 0 時 R₊ = R₋，故 V = 0
>
> (c) **E**|_{z=0} = −Qd / [4πε₀(ρ² + d²/4)^(3/2)] ẑ

電場在 z = 0 平面上指向 −z 方向（從正電荷指向負電荷），且垂直於等位面。

**延伸思考**：此結構即為電偶極(Electric Dipole)。z = 0 平面可等效為一接地(Grounded)導體平面，這是鏡像法(Image Method)的基礎。若只有 +Q 在 z = d/2 且 z = 0 為接地導體，可用 −Q 在 z = −d/2 的鏡像電荷取代導體。

### 題目 11：導體-真空邊界條件
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：邊界條件(Boundary Conditions) — 導體表面

**題目**：

一半徑為 a 的導體球帶電荷 Q，置於自由空間中。
(a) 求球外電場 **E** 與電位 V(r)。
(b) 驗證導體表面的邊界條件：E_n = ρ_s/ε₀，E_t = 0。
(c) 求導體表面的面電荷密度 ρ_s 分布。

**詳解**：

**解題思路**：利用高斯定律求場，再驗證邊界條件。導體為等位體(Equipotential Body)，內部電場為零。

**步驟**：

**(a) 球外電場與電位**

由球對稱性，取半徑 r 的球面為高斯面：

$$D_r\cdot 4\pi r^2 = Q$$

$$\mathbf{E} = \frac{Q}{4\pi\varepsilon_0 r^2}\hat{r} \quad (r \geq a)$$

電位（取 V(∞) = 0）：

$$V(r) = -\int_\infty^r \frac{Q}{4\pi\varepsilon_0 r'^2}\,dr' = \frac{Q}{4\pi\varepsilon_0 r} \quad (r \geq a)$$

導體表面電位：V(a) = Q/(4πε₀a)

導體內部為等位：V(r) = Q/(4πε₀a)，r ≤ a

**(b) 邊界條件驗證**

導體表面 r = a 處：

法向電場：E_n = E_r|_{r=a} = Q/(4πε₀a²)

切向電場：E_t = E_θ = E_φ = 0 ✓（只有 r̂ 分量）

由 D_n = ε₀E_n = Q/(4πa²) = ρ_s

所以 E_n = ρ_s/ε₀ ✓

**(c) 面電荷密度**

$$\rho_s = D_n|_{r=a} = \varepsilon_0 E_r|_{r=a} = \frac{Q}{4\pi a^2}$$

面電荷均勻分布，總電荷驗證：

$$Q_{total} = \rho_s\cdot 4\pi a^2 = \frac{Q}{4\pi a^2}\cdot 4\pi a^2 = Q \quad \checkmark$$

**答案**：

> (a) **E** = Q/(4πε₀r²) r̂，V(r) = Q/(4πε₀r)
>
> (b) E_n = ρ_s/ε₀ 且 E_t = 0 均滿足
>
> (c) ρ_s = Q/(4πa²)，均勻分布

**延伸思考**：若導體球不在自由空間而是在外加均勻電場 **E₀** 中，面電荷密度將不均勻，需用拉普拉斯方程(Laplace's Equation)求解。

### 題目 12：介電質-介電質邊界條件
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：邊界條件 — 介電質界面

**題目**：

z > 0 區域為介電質 ε₁ = 3ε₀，z < 0 區域為介電質 ε₂ = 5ε₀。界面無自由電荷。若 z > 0 區域的電場為 **E₁** = 2x̂ + 3ŷ − 4ẑ (V/m)，求 z < 0 區域的電場 **E₂** 及 **D₂**。

**詳解**：

**解題思路**：應用邊界條件 — 切向 E 連續，法向 D 連續（界面無自由電荷）。

**步驟**：

界面法向量 n̂ = ẑ（由區域 2 指向區域 1）。

分解 **E₁**：

- 切向分量：E₁t = 2x̂ + 3ŷ（平行於 z = 0 平面）
- 法向分量：E₁n = −4ẑ（指向 −z 方向，即指向區域 2）

**邊界條件 1：切向 E 連續**

$$E_{2t} = E_{1t} = 2\hat{x} + 3\hat{y}$$

**邊界條件 2：法向 D 連續（無自由面電荷）**

$$D_{1n} = D_{2n}$$

$$\varepsilon_1 E_{1n} = \varepsilon_2 E_{2n}$$

$$3\varepsilon_0\cdot(-4) = 5\varepsilon_0\cdot E_{2n}$$

$$E_{2n} = -\frac{12}{5} = -2.4$$

因此：

$$\mathbf{E_2} = 2\hat{x} + 3\hat{y} - 2.4\hat{z} \text{ (V/m)}$$

$$\mathbf{D_2} = \varepsilon_2\mathbf{E_2} = 5\varepsilon_0(2\hat{x} + 3\hat{y} - 2.4\hat{z}) = \varepsilon_0(10\hat{x} + 15\hat{y} - 12\hat{z}) \text{ (C/m²)}$$

驗證 D_n 連續：

D₁n = 3ε₀(−4) = −12ε₀

D₂n = 5ε₀(−2.4) = −12ε₀ ✓

**計算折射角：**

$$\tan\theta_1 = \frac{|E_{1t}|}{|E_{1n}|} = \frac{\sqrt{4+9}}{4} = \frac{\sqrt{13}}{4}$$

$$\tan\theta_2 = \frac{|E_{2t}|}{|E_{2n}|} = \frac{\sqrt{13}}{2.4}$$

$$\frac{\tan\theta_1}{\tan\theta_2} = \frac{2.4}{4} = \frac{3}{5} = \frac{\varepsilon_1}{\varepsilon_2} \quad \checkmark$$

此即介電質界面折射定律：tanθ₁/tanθ₂ = ε₁/ε₂。

**答案**：

> **E₂** = 2x̂ + 3ŷ − 2.4ẑ (V/m)
>
> **D₂** = ε₀(10x̂ + 15ŷ − 12ẑ) (C/m²)

**延伸思考**：若界面有自由面電荷密度 ρ_sf，則法向 D 的邊界條件變為 D₁n − D₂n = ρ_sf。切向 E 的連續性不受影響。

### 題目 13：平行板電容計算（含介電質）
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電容(Capacitance) — 平行板

**題目**：

平行板電容器，板面積 S，兩板間距 d。下半部（厚度 d/2）填充介電常數 ε₁ 的介電質，上半部（厚度 d/2）填充介電常數 ε₂ 的介電質。求等效電容。

**詳解**：

**解題思路**：兩層介電質串聯，等效為兩個電容串聯。

**步驟**：

設上板電壓 V₀，下板接地。設 **D** 在兩區域中沿 ẑ 方向均勻，大小為 D。

由無自由電荷的法向邊界條件，D 在兩層中相同：

D₁ = D₂ = D

各層電場：

$$E_1 = \frac{D}{\varepsilon_1}, \quad E_2 = \frac{D}{\varepsilon_2}$$

總電壓：

$$V_0 = E_1\cdot\frac{d}{2} + E_2\cdot\frac{d}{2} = \frac{Dd}{2}\left(\frac{1}{\varepsilon_1}+\frac{1}{\varepsilon_2}\right)$$

面電荷密度 ρ_s = D，總電荷 Q = DS。

$$C = \frac{Q}{V_0} = \frac{DS}{\frac{Dd}{2}\left(\frac{1}{\varepsilon_1}+\frac{1}{\varepsilon_2}\right)} = \frac{2S}{d\left(\frac{1}{\varepsilon_1}+\frac{1}{\varepsilon_2}\right)} = \frac{2S\varepsilon_1\varepsilon_2}{d(\varepsilon_1+\varepsilon_2)}$$

等效驗證 — 串聯電容：

$$C_1 = \frac{\varepsilon_1 S}{d/2} = \frac{2\varepsilon_1 S}{d}, \quad C_2 = \frac{2\varepsilon_2 S}{d}$$

$$\frac{1}{C} = \frac{1}{C_1}+\frac{1}{C_2} = \frac{d}{2S}\left(\frac{1}{\varepsilon_1}+\frac{1}{\varepsilon_2}\right)$$

$$C = \frac{2S\varepsilon_1\varepsilon_2}{d(\varepsilon_1+\varepsilon_2)} \quad \checkmark$$

**答案**：

> C = 2Sε₁ε₂ / [d(ε₁ + ε₂)]

**延伸思考**：若兩層介電質不是上下排列（串聯）而是左右排列（並聯），則 C = (ε₁S₁ + ε₂S₂)/d，其中 S₁ + S₂ = S。串聯時 D 相同，並聯時 E 相同。

### 題目 14：同軸電容計算
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電容 — 同軸結構

**題目**：

同軸電纜(Coaxial Cable)內導體半徑 a，外導體內半徑 b，長度 L（L >> b）。填充介電常數 ε 的介電質。求單位長度電容 C'。若 a = 1 mm, b = 3.5 mm, ε_r = 2.3（聚乙烯），求 C' 的數值。

**詳解**：

**解題思路**：利用高斯定律求電場，再積分求電壓，最後 C = Q/V。

**步驟**：

設內導體帶線電荷密度 ρ_L (C/m)。

取高度 L、半徑 ρ 的同軸圓柱面為高斯面：

$$D_\rho\cdot 2\pi\rho L = \rho_L L$$

$$D_\rho = \frac{\rho_L}{2\pi\rho}$$

$$E_\rho = \frac{D_\rho}{\varepsilon} = \frac{\rho_L}{2\pi\varepsilon\rho}$$

兩導體間電壓：

$$V = -\int_b^a E_\rho\,d\rho = \int_a^b \frac{\rho_L}{2\pi\varepsilon\rho}\,d\rho = \frac{\rho_L}{2\pi\varepsilon}\ln\frac{b}{a}$$

單位長度電容：

$$C' = \frac{\rho_L}{V} = \frac{2\pi\varepsilon}{\ln(b/a)}$$

數值計算：

$$C' = \frac{2\pi\times 2.3\times 8.854\times 10^{-12}}{\ln(3.5/1)} = \frac{2\pi\times 20.36\times 10^{-12}}{1.253}$$

$$= \frac{127.9\times 10^{-12}}{1.253} = 102.1 \text{ pF/m}$$

**答案**：

> C' = 2πε / ln(b/a)
>
> 數值：C' ≈ 102 pF/m

**延伸思考**：同軸電纜的特性阻抗 Z₀ = (1/2π)√(μ/ε)·ln(b/a)。對於此例（聚乙烯填充），Z₀ ≈ 50 Ω，這正是常用的 RG-58 同軸電纜的標準阻抗。

### 題目 15：球形電容
**來源**：台大電機 經典題型
**難度**：★★
**主題**：電容 — 球形結構

**題目**：

同心球形電容器，內球半徑 a，外球內半徑 b，之間填充介電常數 ε 的介電質。
(a) 求電容 C。
(b) 令 b = a + t（t << a），證明結果趨近平行板電容公式。
(c) 令 b → ∞，求孤立球的電容。

**詳解**：

**解題思路**：高斯定律求場，積分求電壓。

**步驟**：

**(a) 球形電容**

設內球帶電荷 Q，取半徑 r（a < r < b）的球面為高斯面：

$$D_r\cdot 4\pi r^2 = Q$$

$$E_r = \frac{Q}{4\pi\varepsilon r^2}$$

兩球間電壓：

$$V = -\int_b^a E_r\,dr = \int_a^b\frac{Q}{4\pi\varepsilon r^2}\,dr = \frac{Q}{4\pi\varepsilon}\left[-\frac{1}{r}\right]_a^b = \frac{Q}{4\pi\varepsilon}\left(\frac{1}{a}-\frac{1}{b}\right)$$

$$C = \frac{Q}{V} = \frac{4\pi\varepsilon}{\frac{1}{a}-\frac{1}{b}} = \frac{4\pi\varepsilon ab}{b-a}$$

**(b) b = a + t, t << a**

$$C = \frac{4\pi\varepsilon a(a+t)}{t} \approx \frac{4\pi\varepsilon a^2}{t} = \frac{\varepsilon\cdot 4\pi a^2}{t} = \frac{\varepsilon S}{t}$$

其中 S = 4πa² 為球面面積，t 為間距。此即平行板電容公式 C = εS/d。

**(c) b → ∞**

$$C = \frac{4\pi\varepsilon ab}{b-a} = \frac{4\pi\varepsilon a}{1-a/b} \xrightarrow{b\to\infty} 4\pi\varepsilon a$$

**答案**：

> (a) C = 4πεab/(b−a)
>
> (b) 當 t << a 時，C ≈ ε·4πa²/t（平行板公式）
>
> (c) 孤立球電容 C = 4πεa

**延伸思考**：地球的電容 C = 4πε₀R_E ≈ 4π(8.854×10⁻¹²)(6.371×10⁶) ≈ 709 μF。雖然數值不小，但要使地球電位升高 1V 就需要 709 μC 的電荷。

### 題目 16：鏡像法 — 點電荷與接地平面
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：鏡像法(Image Method)

**題目**：

一點電荷 +Q 位於 (0, 0, d)，z = 0 平面為接地導體(Grounded Conductor)。
(a) 用鏡像法求 z > 0 區域的電位 V(x, y, z)。
(b) 求導體表面（z = 0）的感應面電荷密度 ρ_s(ρ)。
(c) 驗證導體上感應電荷總量為 −Q。
(d) 求電荷 Q 受到的作用力（鏡像力）。

**詳解**：

**解題思路**：用位於 (0, 0, −d) 的鏡像電荷 −Q 取代導體平面，確保 z = 0 上 V = 0。

**步驟**：

**(a) 電位**

$$V(x,y,z) = \frac{Q}{4\pi\varepsilon_0}\left[\frac{1}{\sqrt{x^2+y^2+(z-d)^2}} - \frac{1}{\sqrt{x^2+y^2+(z+d)^2}}\right]$$

$$= \frac{Q}{4\pi\varepsilon_0}\left(\frac{1}{R_1} - \frac{1}{R_2}\right), \quad z \geq 0$$

其中 R₁ = √(ρ²+(z−d)²)，R₂ = √(ρ²+(z+d)²)，ρ² = x²+y²。

在 z = 0 上：R₁ = R₂ = √(ρ²+d²)，故 V = 0 ✓

**(b) 感應面電荷密度**

$$\rho_s = -\varepsilon_0\frac{\partial V}{\partial z}\bigg|_{z=0^+} = \varepsilon_0 E_z|_{z=0^-}$$

但更直接用 D_n = ε₀E_n = ρ_s，且 n̂ = −ẑ（導體法向指向上方即 +ẑ，但 ρ_s = D·n̂_out），需要仔細處理方向。

導體表面法向量朝外（+ẑ 方向），ρ_s = D_z|_{z=0⁺}：

$$E_z = -\frac{\partial V}{\partial z}$$

$$\frac{\partial V}{\partial z} = \frac{Q}{4\pi\varepsilon_0}\left[\frac{-(z-d)}{R_1^3} - \frac{-(z+d)}{R_2^3}\right]$$

z = 0 時 R₁ = R₂ = (ρ²+d²)^(1/2)：

$$\frac{\partial V}{\partial z}\bigg|_{z=0} = \frac{Q}{4\pi\varepsilon_0}\cdot\frac{1}{(\rho^2+d^2)^{3/2}}\left[d + (-d)\right]$$

等一下，讓我重新計算：

$$\frac{\partial V}{\partial z}\bigg|_{z=0} = \frac{Q}{4\pi\varepsilon_0}\left[\frac{d}{(\rho^2+d^2)^{3/2}} - \frac{d}{(\rho^2+d^2)^{3/2}}\right]$$

這不對，讓我更仔細地展開：

$$\frac{\partial}{\partial z}\left(\frac{1}{R_1}\right) = \frac{-(z-d)}{R_1^3}$$

$$\frac{\partial}{\partial z}\left(\frac{1}{R_2}\right) = \frac{-(z+d)}{R_2^3}$$

$$\frac{\partial V}{\partial z} = \frac{Q}{4\pi\varepsilon_0}\left[\frac{-(z-d)}{R_1^3} + \frac{(z+d)}{R_2^3}\right]$$

z = 0 時：

$$\frac{\partial V}{\partial z}\bigg|_{z=0} = \frac{Q}{4\pi\varepsilon_0}\left[\frac{d}{(\rho^2+d^2)^{3/2}} + \frac{d}{(\rho^2+d^2)^{3/2}}\right] = \frac{Qd}{2\pi\varepsilon_0(\rho^2+d^2)^{3/2}}$$

$$E_z|_{z=0^+} = -\frac{\partial V}{\partial z}\bigg|_{z=0} = -\frac{Qd}{2\pi\varepsilon_0(\rho^2+d^2)^{3/2}}$$

$$\rho_s = \varepsilon_0 E_z|_{z=0^+} = -\frac{Qd}{2\pi(\rho^2+d^2)^{3/2}}$$

**(c) 總感應電荷**

$$Q_{induced} = \int_0^\infty\int_0^{2\pi}\rho_s\,\rho\,d\rho\,d\phi = 2\pi\int_0^\infty\left(-\frac{Qd}{2\pi(\rho^2+d^2)^{3/2}}\right)\rho\,d\rho$$

$$= -Qd\int_0^\infty\frac{\rho\,d\rho}{(\rho^2+d^2)^{3/2}}$$

令 u = ρ² + d²，du = 2ρ dρ：

$$= -Qd\cdot\frac{1}{2}\int_{d^2}^{\infty} u^{-3/2}\,du = -Qd\cdot\frac{1}{2}\left[-2u^{-1/2}\right]_{d^2}^{\infty}$$

$$= -Qd\cdot\frac{1}{2}\cdot\left(0+\frac{2}{d}\right) = -Q \quad \checkmark$$

**(d) 鏡像力**

電荷 +Q 受鏡像電荷 −Q（距離 2d）的庫侖力：

$$\mathbf{F} = \frac{Q\cdot(-Q)}{4\pi\varepsilon_0(2d)^2}\hat{z} = -\frac{Q^2}{16\pi\varepsilon_0 d^2}\hat{z}$$

力指向 −z（吸引力），電荷被導體吸引。

**答案**：

> (a) V = Q/(4πε₀)(1/R₁ − 1/R₂)，z ≥ 0
>
> (b) ρ_s = −Qd/[2π(ρ²+d²)^(3/2)]
>
> (c) Q_induced = −Q
>
> (d) **F** = −Q²/(16πε₀d²) ẑ（吸引力）

**延伸思考**：鏡像法可推廣到介電質界面（鏡像電荷大小需乘以修正係數）、球形導體（鏡像電荷位置和大小需由球面邊界條件決定）等更複雜的問題。

### 題目 17：鏡像法 — 點電荷與接地導體球
**來源**：台大電機 經典題型
**難度**：★★★★★
**主題**：鏡像法 — 球形導體

**題目**：

一接地導體球半徑 a，球心在原點。一點電荷 +Q 位於 z 軸上 (0, 0, d)，d > a。
(a) 求鏡像電荷的大小和位置。
(b) 求球外電位。
(c) 求球面上感應電荷密度。

**詳解**：

**解題思路**：導體球的鏡像電荷位於球心到原電荷連線上，位於球內反演點(Inverse Point)。

**步驟**：

**(a) 鏡像電荷**

對於球心在原點、半徑 a 的接地導體球，原電荷 +Q 在 z = d 處的鏡像電荷為：

- 大小：Q' = −Qa/d
- 位置：z' = a²/d（球內反演點）

驗證：在球面上任一點 P(a, θ, φ)：

從 +Q 到 P 的距離：R₁ = √(a² + d² − 2ad cosθ)

從 Q' 到 P 的距離：R₂ = √(a² + a⁴/d² − 2a·a²cosθ/d) = (a/d)√(d² + a² − 2ad cosθ)

$$V|_{r=a} = \frac{1}{4\pi\varepsilon_0}\left[\frac{Q}{R_1} + \frac{Q'}{R_2}\right]$$

$$= \frac{1}{4\pi\varepsilon_0}\left[\frac{Q}{\sqrt{a^2+d^2-2ad\cos\theta}} + \frac{-Qa/d}{(a/d)\sqrt{a^2+d^2-2ad\cos\theta}}\right]$$

$$= \frac{1}{4\pi\varepsilon_0}\cdot\frac{1}{\sqrt{a^2+d^2-2ad\cos\theta}}\left[Q - Q\right] = 0 \quad \checkmark$$

**(b) 球外電位**

$$V(r,\theta) = \frac{1}{4\pi\varepsilon_0}\left[\frac{Q}{\sqrt{r^2+d^2-2rd\cos\theta}} - \frac{Qa/d}{\sqrt{r^2+a^4/d^2-2ra^2\cos\theta/d}}\right]$$

有效範圍：r ≥ a。

**(c) 球面感應電荷密度**

$$\rho_s = -\varepsilon_0\frac{\partial V}{\partial r}\bigg|_{r=a}$$

由電荷 Q 在 (0,0,d) 和電荷 Q' = −Qa/d 在 (0,0,a²/d) 對球面的法向電場：

$$E_r|_{r=a} = -\frac{Q}{4\pi\varepsilon_0}\cdot\frac{a-d\cos\theta}{(a^2+d^2-2ad\cos\theta)^{3/2}}$$
$$+ \frac{Qa/d}{4\pi\varepsilon_0}\cdot\frac{a-a^2\cos\theta/d}{(a^2+a^4/d^2-2a^3\cos\theta/d)^{3/2}}$$

經簡化（利用 R₂ = (a/d)R₁ 的關係）：

$$\rho_s = -\frac{Q}{4\pi a}\cdot\frac{d^2-a^2}{(a^2+d^2-2ad\cos\theta)^{3/2}}$$

在 θ = 0（最近點）處密度最大，θ = π（最遠點）處密度最小。

**答案**：

> (a) 鏡像電荷 Q' = −Qa/d，位於 z = a²/d
>
> (b) V(r,θ) = (1/4πε₀)[Q/R₁ − (Qa/d)/R₂]
>
> (c) ρ_s = −Q(d²−a²)/[4πa(a²+d²−2ad cosθ)^(3/2)]

**延伸思考**：若導體球不接地而是孤立的（帶總電荷為 0），需要額外加一個 +Qa/d 的鏡像電荷於球心，以保持球面等位且總感應電荷為 0。

### 題目 18：電偶極場
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電偶極(Electric Dipole)

**題目**：

一電偶極矩 **p** = p ẑ（p = Qd）位於原點。
(a) 寫出遠場近似下的電位 V(r, θ) 和電場 **E**(r, θ)。
(b) 畫出電場線和等位線的定性圖樣。
(c) 求 θ = 0, π/2, π 處的電場方向和大小。

**詳解**：

**解題思路**：電偶極遠場為靜電場中重要的多極展開(Multipole Expansion)首項。

**步驟**：

**(a) 電位與電場**

電偶極電位（r >> d 遠場近似）：

$$V(r,\theta) = \frac{p\cos\theta}{4\pi\varepsilon_0 r^2}$$

電場由 **E** = −∇V 在球座標中求得：

$$E_r = -\frac{\partial V}{\partial r} = \frac{2p\cos\theta}{4\pi\varepsilon_0 r^3}$$

$$E_\theta = -\frac{1}{r}\frac{\partial V}{\partial\theta} = \frac{p\sin\theta}{4\pi\varepsilon_0 r^3}$$

$$E_\phi = 0 \quad \text{（軸對稱）}$$

$$\mathbf{E} = \frac{p}{4\pi\varepsilon_0 r^3}(2\cos\theta\,\hat{r} + \sin\theta\,\hat{\theta})$$

**(b) 電場線與等位線**

- 電場線從 +Q 出發到 −Q 結束，在遠場呈「8」字形（沿 z 軸拉長）
- 等位線 V = const 即 cosθ/r² = const，在 θ = π/2 平面上 V = 0
- 電場線與等位線處處正交

**(c) 特殊方向的電場**

θ = 0（+z 軸方向）：

$$\mathbf{E} = \frac{2p}{4\pi\varepsilon_0 r^3}\hat{r} = \frac{p}{2\pi\varepsilon_0 r^3}\hat{z}$$

θ = π/2（赤道面）：

$$\mathbf{E} = \frac{p}{4\pi\varepsilon_0 r^3}\hat{\theta} = -\frac{p}{4\pi\varepsilon_0 r^3}\hat{z}$$

θ = π（−z 軸方向）：

$$\mathbf{E} = \frac{-2p}{4\pi\varepsilon_0 r^3}\hat{r} = \frac{p}{2\pi\varepsilon_0 r^3}\hat{z}$$

注意：沿軸線（θ = 0 和 π）的電場大小是赤道面的 2 倍。

**答案**：

> V = p cosθ/(4πε₀r²)
>
> **E** = p/(4πε₀r³)(2cosθ r̂ + sinθ θ̂)
>
> 軸線上 |E| = p/(2πε₀r³)，赤道面上 |E| = p/(4πε₀r³)

**延伸思考**：在外加均勻電場 **E₀** 中，電偶極受力矩 **τ** = **p** × **E₀**，傾向於讓 **p** 轉向與 **E₀** 同向。電偶極在非均勻場中還會受淨力 **F** = ∇(**p** · **E**)。

### 題目 19：靜電能計算
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜電能(Electrostatic Energy)

**題目**：

同軸電纜內導體半徑 a，外導體內半徑 b，長度 L，填充介電常數 ε 的介電質，兩導體間電壓為 V₀。用兩種方法計算儲存的靜電能：
(a) W = ½CV₀²
(b) W = ½∫∫∫**D**·**E** dv

**詳解**：

**解題思路**：兩種方法應得到相同結果，互相驗證。

**步驟**：

**(a) 用電容公式**

由題目 14 的結果，同軸電容：

$$C = \frac{2\pi\varepsilon L}{\ln(b/a)}$$

$$W = \frac{1}{2}CV_0^2 = \frac{\pi\varepsilon L V_0^2}{\ln(b/a)}$$

**(b) 用場能量密度積分**

電場（a < ρ < b）：

由 V₀ = (ρ_L/2πε)ln(b/a)，得 ρ_L = 2πεV₀/ln(b/a)。

$$E_\rho = \frac{\rho_L}{2\pi\varepsilon\rho} = \frac{V_0}{\rho\ln(b/a)}$$

能量密度：

$$w_e = \frac{1}{2}\varepsilon E^2 = \frac{\varepsilon V_0^2}{2\rho^2[\ln(b/a)]^2}$$

體積積分（柱座標）：

$$W = \int_0^L\int_0^{2\pi}\int_a^b \frac{\varepsilon V_0^2}{2\rho^2[\ln(b/a)]^2}\,\rho\,d\rho\,d\phi\,dz$$

$$= \frac{\varepsilon V_0^2}{2[\ln(b/a)]^2}\cdot 2\pi L\int_a^b\frac{d\rho}{\rho}$$

$$= \frac{\varepsilon V_0^2}{2[\ln(b/a)]^2}\cdot 2\pi L\cdot\ln\frac{b}{a}$$

$$= \frac{\pi\varepsilon L V_0^2}{\ln(b/a)} \quad \checkmark$$

兩種方法結果完全一致。

**答案**：

> W = πεLV₀²/ln(b/a)

**延伸思考**：能量密度 w_e = εE²/2 在 ρ = a 處最大，隨 1/ρ² 衰減。電場集中在內導體附近，這也是同軸電纜內導體附近容易發生電擊穿(Breakdown)的原因。

### 題目 20：拉普拉斯方程 — 球座標
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：拉普拉斯方程(Laplace's Equation)

**題目**：

一介電球（半徑 a，介電常數 ε）置於均勻外加電場 **E₀** = E₀ẑ 中。利用拉普拉斯方程求球內外電位，並求球內電場。

**詳解**：

**解題思路**：具有 φ 軸對稱性，用球座標拉普拉斯方程的勒讓德多項式(Legendre Polynomial)解。

**步驟**：

邊界條件：
1. r → ∞ 時，V → −E₀r cosθ（均勻場）
2. r = 0 時，V 有限
3. r = a 時，V 連續（電位連續）
4. r = a 時，ε₁(∂V₁/∂r) = ε₂(∂V₂/∂r)（法向 D 連續，球面無自由電荷）

其中 ε₁ = ε（球內），ε₂ = ε₀（球外）。

由對稱性，電位只含 cosθ = P₁(cosθ) 項：

球外（r ≥ a）：

$$V_{out} = -E_0 r\cos\theta + \frac{A\cos\theta}{r^2}$$

（第一項為外加場，第二項為感應偶極場）

球內（r ≤ a）：

$$V_{in} = Br\cos\theta$$

（必須在原點有限，只取 r 的正冪次）

**邊界條件 3**（r = a，V 連續）：

$$-E_0 a\cos\theta + \frac{A\cos\theta}{a^2} = Ba\cos\theta$$

$$-E_0 a + \frac{A}{a^2} = Ba \quad \cdots (1)$$

**邊界條件 4**（r = a，法向 D 連續）：

$$\varepsilon\frac{\partial V_{in}}{\partial r}\bigg|_{r=a} = \varepsilon_0\frac{\partial V_{out}}{\partial r}\bigg|_{r=a}$$

$$\varepsilon\cdot B\cos\theta = \varepsilon_0\left(-E_0 - \frac{2A}{a^3}\right)\cos\theta$$

$$\varepsilon B = \varepsilon_0\left(-E_0 - \frac{2A}{a^3}\right) \quad \cdots (2)$$

由 (1)：A = a²(Ba + E₀a) = a³(B + E₀)

代入 (2)：

$$\varepsilon B = \varepsilon_0\left(-E_0 - 2(B+E_0)\right) = \varepsilon_0(-3E_0 - 2B)$$

$$\varepsilon B + 2\varepsilon_0 B = -3\varepsilon_0 E_0$$

$$B(\varepsilon + 2\varepsilon_0) = -3\varepsilon_0 E_0$$

$$B = \frac{-3\varepsilon_0 E_0}{\varepsilon + 2\varepsilon_0}$$

$$A = a^3(B + E_0) = a^3\cdot\frac{-3\varepsilon_0 E_0 + (\varepsilon+2\varepsilon_0)E_0}{\varepsilon+2\varepsilon_0} = a^3\cdot\frac{(\varepsilon-\varepsilon_0)E_0}{\varepsilon+2\varepsilon_0}$$

球內電位：

$$V_{in} = \frac{-3\varepsilon_0 E_0}{\varepsilon+2\varepsilon_0}\cdot r\cos\theta = \frac{-3\varepsilon_0}{\varepsilon+2\varepsilon_0}E_0 z$$

球內電場（均勻！）：

$$\mathbf{E}_{in} = -\nabla V_{in} = \frac{3\varepsilon_0}{\varepsilon+2\varepsilon_0}E_0\hat{z}$$

定義相對介電常數 ε_r = ε/ε₀：

$$\mathbf{E}_{in} = \frac{3}{\varepsilon_r+2}E_0\hat{z}$$

由於 ε_r > 1，故 E_in < E₀，球內電場小於外加場（介電質的屏蔽效應）。

**答案**：

> 球內電場為均勻場：**E**_in = 3ε₀/(ε+2ε₀) · E₀ ẑ = 3/(ε_r+2) · E₀ ẑ
>
> 球外電位 = 外加場 + 等效偶極場，偶極矩 p = 4πε₀a³(ε_r−1)/(ε_r+2) · E₀

**延伸思考**：若 ε_r → ∞（導體球），E_in → 0，球內無電場。球外偶極場係數 A → a³E₀，這與導體球在均勻場中的結果一致。此問題也是介質球散射(Scattering)的基礎。

### 題目 21：Biot-Savart 定律 — 有限長直線電流
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — Biot-Savart 定律

**題目**：

一載有電流 I 的直導線沿 z 軸，從 z = −L 到 z = +L。用 Biot-Savart 定律求距離導線 ρ₀ 處（ρ 軸上）的磁場 **B**。並驗證 L → ∞ 時與安培定律結果一致。

**詳解**：

**解題思路**：類似有限長線電荷的電場積分，利用對稱性。

**步驟**：

Biot-Savart 定律：

$$d\mathbf{B} = \frac{\mu_0 I}{4\pi}\frac{d\mathbf{l}'\times\hat{R}}{R^2}$$

源點在 z 軸上 (0, 0, z')，觀測點在 (ρ₀, 0, 0)。

d**l**' = dz' ẑ

**R** = ρ₀ρ̂ − z'ẑ，R = √(ρ₀² + z'²)

R̂ = (ρ₀ρ̂ − z'ẑ)/R

$$d\mathbf{l}'\times\hat{R} = dz'\,\hat{z}\times\frac{\rho_0\hat{\rho}-z'\hat{z}}{R} = \frac{\rho_0\,dz'}{R}\,(\hat{z}\times\hat{\rho}) = \frac{\rho_0\,dz'}{R}\hat{\phi}$$

（因為 ẑ × ρ̂ = φ̂，ẑ × ẑ = 0）

$$d\mathbf{B} = \frac{\mu_0 I}{4\pi}\cdot\frac{\rho_0\,dz'}{R^3}\hat{\phi} = \frac{\mu_0 I\rho_0}{4\pi}\cdot\frac{dz'}{(\rho_0^2+z'^2)^{3/2}}\hat{\phi}$$

積分：

$$\mathbf{B} = \frac{\mu_0 I\rho_0}{4\pi}\hat{\phi}\int_{-L}^{L}\frac{dz'}{(\rho_0^2+z'^2)^{3/2}}$$

此積分與題目 4 相同。令 z' = ρ₀tanα：

$$\int_{-L}^{L}\frac{dz'}{(\rho_0^2+z'^2)^{3/2}} = \frac{1}{\rho_0^2}\cdot\frac{2L}{\sqrt{L^2+\rho_0^2}}$$

$$\mathbf{B} = \frac{\mu_0 I}{4\pi\rho_0}\cdot\frac{2L}{\sqrt{L^2+\rho_0^2}}\hat{\phi} = \frac{\mu_0 IL}{2\pi\rho_0\sqrt{L^2+\rho_0^2}}\hat{\phi}$$

L → ∞ 時：L/√(L²+ρ₀²) → 1

$$\mathbf{B} = \frac{\mu_0 I}{2\pi\rho_0}\hat{\phi}$$

此即安培定律(Ampere's Law)對無限長直線電流的結果。

**答案**：

> 有限長：**B** = μ₀IL / [2πρ₀√(L²+ρ₀²)] φ̂
>
> 無限長(L→∞)：**B** = μ₀I/(2πρ₀) φ̂

**延伸思考**：對有限長導線，也可用角度表示 **B** = μ₀I(sinα₁+sinα₂)/(4πρ₀) φ̂，其中 α₁、α₂ 為導線兩端對觀測點的仰角。此公式在計算多段折線電流的磁場時很方便。

### 題目 22：Biot-Savart 定律 — 圓環電流
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 圓環電流的軸線磁場

**題目**：

半徑為 a 的圓形載流線圈，電流 I，位於 z = 0 平面，中心在原點。求 z 軸上任意點 (0, 0, h) 的磁場 **B**。

**詳解**：

**解題思路**：利用 Biot-Savart 定律，由對稱性知 z 軸上只有 ẑ 分量。

**步驟**：

源點在圓環上：(a, φ', 0)（柱座標）

d**l**' = a dφ' φ̂' = a dφ'(−sinφ' x̂ + cosφ' ŷ)

觀測點：P(0, 0, h)

**R** = −acosφ' x̂ − asinφ' ŷ + hẑ

R = √(a² + h²)

$$d\mathbf{l}'\times\mathbf{R} = a\,d\phi'(-\sin\phi'\hat{x}+\cos\phi'\hat{y})\times(-a\cos\phi'\hat{x}-a\sin\phi'\hat{y}+h\hat{z})$$

展開叉積：

$$= a\,d\phi'\left[(-\sin\phi')(h)(\hat{x}\times\hat{z}) + (-\sin\phi')(-a\sin\phi')(\hat{x}\times\hat{y})\right.$$
$$\left.+ (\cos\phi')(-a\cos\phi')(\hat{y}\times\hat{x}) + (\cos\phi')(h)(\hat{y}\times\hat{z})\right]$$

利用 x̂×ẑ = −ŷ, x̂×ŷ = ẑ, ŷ×x̂ = −ẑ, ŷ×ẑ = x̂：

$$= a\,d\phi'\left[-h\sin\phi'(-\hat{y}) + a\sin^2\phi'\hat{z} - a\cos^2\phi'\hat{z} + h\cos\phi'\hat{x}\right]$$

等等，讓我重新整理。更簡潔的做法：

$$= a\,d\phi'\left[h\cos\phi'\hat{x} + h\sin\phi'\hat{y} + a(\sin^2\phi'+\cos^2\phi')\hat{z}\right]$$

等一下，仔細用行列式計算：

$$d\mathbf{l}'\times\mathbf{R} = a\,d\phi'\begin{vmatrix}\hat{x}&\hat{y}&\hat{z}\\-\sin\phi'&\cos\phi'&0\\-a\cos\phi'&-a\sin\phi'&h\end{vmatrix}$$

x̂ 分量：cosφ'·h − 0·(−asinφ') = h cosφ'

ŷ 分量：0·(−acosφ') − (−sinφ')·h = h sinφ'

ẑ 分量：(−sinφ')(−asinφ') − cosφ'(−acosφ') = a sin²φ' + a cos²φ' = a

所以：

$$d\mathbf{l}'\times\mathbf{R} = a\,d\phi'(h\cos\phi'\hat{x} + h\sin\phi'\hat{y} + a\hat{z})$$

在 φ' 從 0 到 2π 積分時，cosφ' 和 sinφ' 的積分為零。只剩 ẑ 分量：

$$\mathbf{B} = \frac{\mu_0 I}{4\pi}\oint\frac{d\mathbf{l}'\times\hat{R}}{R^2} = \frac{\mu_0 I}{4\pi R^3}\int_0^{2\pi}a\cdot a\,d\phi'\hat{z}$$

$$= \frac{\mu_0 I a^2}{4\pi(a^2+h^2)^{3/2}}\cdot 2\pi\hat{z}$$

$$= \frac{\mu_0 I a^2}{2(a^2+h^2)^{3/2}}\hat{z}$$

**答案**：

> **B** = μ₀Ia²/[2(a²+h²)^(3/2)] ẑ
>
> 在圓心處(h=0)：**B** = μ₀I/(2a) ẑ

**延伸思考**：圓環電流在遠處(h >> a)的磁場可近似為磁偶極(Magnetic Dipole)場，磁偶極矩 m = Iπa² = IS（S 為環面積）。遠場 B ≈ μ₀m/(2πh³)，與電偶極遠場的形式完全類比。

### 題目 23：安培環路定律 — 螺線管與環形線圈
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 安培環路定律(Ampere's Circuital Law)

**題目**：

(a) 無限長密繞螺線管(Solenoid)，每單位長度匝數 n，電流 I。用安培定律求內外磁場。
(b) 環形線圈(Toroid)，平均半徑 R，匝數 N，電流 I。求內部磁場。

**詳解**：

**解題思路**：利用高度對稱性選取安培環路(Amperian Loop)。

**步驟**：

**(a) 無限長螺線管**

取矩形安培環路 ABCD：AB 在螺線管內部（長 l，平行於軸），CD 在外部。

由對稱性：
- 螺線管外 **B** = 0（可由安培定律配合對稱性論證：取橫截面上的環路可知 B 與距軸距離無關，取包含無窮遠的環路可知外部 B = 0）
- 螺線管內 **B** 均勻，沿軸向

安培定律：∮**B**·d**l** = μ₀I_enc

$$B\cdot l + 0 + 0 + 0 = \mu_0\cdot nI\cdot l$$

$$B = \mu_0 nI$$

**B** = μ₀nI ẑ（ẑ 為螺線管軸向，方向由右手定則決定）

**(b) 環形線圈**

取以環形中心為圓心、半徑 ρ 的圓形安培環路。

由對稱性，**B** 沿 φ̂ 方向，大小只與 ρ 有關。

若 ρ 在環形內部（R−a < ρ < R+a，a 為環截面半徑）：

$$B_\phi\cdot 2\pi\rho = \mu_0 NI$$

$$B_\phi = \frac{\mu_0 NI}{2\pi\rho}$$

若 ρ 不在環形內部（ρ < R−a 或 ρ > R+a）：

$$B_\phi\cdot 2\pi\rho = 0 \implies B_\phi = 0$$

在平均半徑 ρ = R 處：

$$B = \frac{\mu_0 NI}{2\pi R}$$

**答案**：

> (a) 螺線管內：**B** = μ₀nI ẑ；螺線管外：**B** = 0
>
> (b) 環形線圈內：**B** = μ₀NI/(2πρ) φ̂；外部：**B** = 0

**延伸思考**：螺線管是均勻磁場的理想來源。環形線圈的磁場完全被限制在環內，無漏磁，這是理想變壓器(Transformer)的基礎。若 a << R（薄環），可近似 B ≈ μ₀NI/(2πR) 為均勻值。

### 題目 24：磁向量位 — 無限長直線電流
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 磁向量位(Magnetic Vector Potential)

**題目**：

一無限長直線電流 I 沿 z 軸流動。
(a) 求磁向量位 **A**。
(b) 驗證 **B** = ∇ × **A** 給出正確的磁場。
(c) 驗證 ∇·**A** = 0（庫侖規範, Coulomb Gauge）。

**詳解**：

**解題思路**：由對稱性 **A** 只有 ẑ 分量且只與 ρ 有關。由 **B** = ∇ × **A** 反推 **A**。

**步驟**：

**(a) 磁向量位**

已知 **B** = μ₀I/(2πρ) φ̂。

由柱座標旋度的 φ̂ 分量：

$$B_\phi = -\frac{\partial A_z}{\partial\rho}$$

（若 **A** = A_z(ρ)ẑ，其旋度 ∇ × **A** 的 φ̂ 分量為 −∂A_z/∂ρ）

$$-\frac{\partial A_z}{\partial\rho} = \frac{\mu_0 I}{2\pi\rho}$$

$$A_z = -\frac{\mu_0 I}{2\pi}\ln\rho + C$$

取 ρ = ρ₀ 處 A_z = 0 的參考點（因無限長結構 A 對數發散，不能取 A(∞) = 0）：

$$A_z = -\frac{\mu_0 I}{2\pi}\ln\frac{\rho}{\rho_0}$$

$$\mathbf{A} = -\frac{\mu_0 I}{2\pi}\ln\frac{\rho}{\rho_0}\,\hat{z}$$

**(b) 驗證 B = ∇ × A**

柱座標中 **A** = A_z(ρ)ẑ 的旋度：

$$\nabla\times\mathbf{A} = -\frac{\partial A_z}{\partial\rho}\hat{\phi} = -\left(-\frac{\mu_0 I}{2\pi\rho}\right)\hat{\phi} = \frac{\mu_0 I}{2\pi\rho}\hat{\phi} \quad \checkmark$$

**(c) 驗證庫侖規範**

$$\nabla\cdot\mathbf{A} = \frac{\partial A_z}{\partial z} = 0 \quad \checkmark$$

（A_z 不依賴 z）

**答案**：

> **A** = −μ₀I/(2π) ln(ρ/ρ₀) ẑ
>
> ∇ × **A** = μ₀I/(2πρ) φ̂ = **B** ✓
>
> ∇ · **A** = 0 ✓

**延伸思考**：磁向量位 **A** 在量子力學中有直接物理意義（Aharonov-Bohm 效應）。在電磁波問題中，**A** 與純量位 V 構成四維向量位(Four-potential)，是電動力學的基礎。

### 題目 25：載流導線在磁場中的受力
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 安培力(Ampere Force)

**題目**：

兩條平行無限長直線電流，相距 d，分別載有電流 I₁ 和 I₂。
(a) 求兩導線間的作用力（每單位長度）。
(b) 判斷同向電流和反向電流時的力是吸引還是排斥。
(c) 若 I₁ = I₂ = 1 A，d = 1 m，求力的數值（這就是安培的定義）。

**詳解**：

**解題思路**：先求一電流產生的磁場，再計算另一電流在此磁場中受到的安培力。

**步驟**：

**(a) 每單位長度的力**

設 I₁ 沿 z 軸，I₂ 在 y = d 處平行於 z 軸。

I₁ 在 y = d 處產生的磁場：

$$\mathbf{B_1} = \frac{\mu_0 I_1}{2\pi d}\hat{\phi_1}$$

在 I₂ 所在位置，φ̂₁ = −x̂（因為 I₂ 在 I₁ 的 +y 方向，此處 φ̂ = −x̂）。

**等一下，讓我更仔細處理座標。**

設 I₁ 在 x = 0 處沿 +z 方向，I₂ 在 x = d 處沿 +z 方向。

I₁ 產生的磁場在 x = d 處（ρ = d，φ = 0 的柱座標）：

$$\mathbf{B_1}(x=d) = \frac{\mu_0 I_1}{2\pi d}\hat{\phi} = \frac{\mu_0 I_1}{2\pi d}\hat{y}$$

（在 x 軸正方向處 φ̂ = ŷ）

I₂ 的電流元素：d**l₂** = dz ẑ

單位長度受力：

$$\frac{d\mathbf{F}}{dz} = I_2(\hat{z}\times\mathbf{B_1}) = I_2\left(\hat{z}\times\frac{\mu_0 I_1}{2\pi d}\hat{y}\right) = \frac{\mu_0 I_1 I_2}{2\pi d}(\hat{z}\times\hat{y})$$

$$= \frac{\mu_0 I_1 I_2}{2\pi d}(-\hat{x})$$

力指向 −x 方向，即朝向 I₁，為吸引力。

**(b) 方向判別**

- 同向電流（I₁I₂ > 0）：力指向對方，**吸引**
- 反向電流（I₁I₂ < 0）：力背離對方，**排斥**

**(c) 安培定義**

$$\frac{F}{L} = \frac{\mu_0 I_1 I_2}{2\pi d} = \frac{4\pi\times 10^{-7}\times 1\times 1}{2\pi\times 1} = 2\times 10^{-7} \text{ N/m}$$

**答案**：

> (a) F/L = μ₀I₁I₂/(2πd)
>
> (b) 同向吸引，反向排斥
>
> (c) F/L = 2×10⁻⁷ N/m（此為舊制安培的定義基礎）

**延伸思考**：2019 年國際單位制改革後，安培的定義改為基於基本電荷 e = 1.602176634×10⁻¹⁹ C，但上述力的計算結果不變。兩平行載流導線的作用力原理也是電流秤(Current Balance)的工作基礎。

### 題目 26：自感計算 — 環形線圈
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 自感(Self-Inductance)

**題目**：

環形線圈(Toroid)：平均半徑 R，截面為矩形（內半徑 a，外半徑 b，高度 h），匝數 N，繞在磁導率 μ 的鐵心上。求自感 L。

**詳解**：

**解題思路**：先求磁場，再算磁通量，最後用 L = NΦ/I。

**步驟**：

由安培環路定律，環形線圈內部（a < ρ < b）：

$$B_\phi = \frac{\mu NI}{2\pi\rho}$$

通過截面的磁通量：

$$\Phi = \int_a^b\int_0^h B_\phi\,d\rho\,dz = \int_a^b\int_0^h\frac{\mu NI}{2\pi\rho}\,d\rho\,dz$$

$$= \frac{\mu NIh}{2\pi}\int_a^b\frac{d\rho}{\rho} = \frac{\mu NIh}{2\pi}\ln\frac{b}{a}$$

磁通量鏈(Flux Linkage)：

$$\Lambda = N\Phi = \frac{\mu N^2 Ih}{2\pi}\ln\frac{b}{a}$$

自感：

$$L = \frac{\Lambda}{I} = \frac{\mu N^2 h}{2\pi}\ln\frac{b}{a}$$

若截面為圓形（半徑 r₀，r₀ << R），可近似為：

$$L \approx \frac{\mu N^2 \cdot \pi r_0^2}{2\pi R} = \frac{\mu N^2 r_0^2}{2R}$$

或用矩形截面近似（a ≈ R，b−a ≈ 2r₀ << R）：

$$\ln\frac{b}{a} \approx \frac{b-a}{R}$$

$$L \approx \frac{\mu N^2 h(b-a)}{2\pi R} = \frac{\mu N^2 S}{2\pi R}$$

其中 S = h(b−a) 為矩形截面積。

**答案**：

> L = μN²h/(2π) · ln(b/a)
>
> 薄環近似(b−a << R)：L ≈ μN²S/(2πR)

**延伸思考**：環形線圈的自感與 N² 成正比，增加匝數可有效提高電感。自感也可用磁能公式 L = 2W_m/I² 來計算，其中 W_m = ∫(B²/2μ)dv。

### 題目 27：互感計算 — 同軸螺線管
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 互感(Mutual Inductance)

**題目**：

兩個同軸密繞螺線管，內管半徑 a₁、匝數密度 n₁，外管半徑 a₂（a₂ > a₁）、匝數密度 n₂，長度均為 l。求互感 M，並驗證 M² ≤ L₁L₂（耦合係數 k ≤ 1）。

**詳解**：

**解題思路**：設一管通電流求磁場，算通過另一管的磁通量鏈。

**步驟**：

設內管通電流 I₁，產生的磁場（近似無限長螺線管）：

管內（ρ < a₁）：B = μ₀n₁I₁

管外（ρ > a₁）：B ≈ 0

通過外管單匝的磁通量：

$$\Phi_{21} = B\cdot\pi a_1^2 = \mu_0 n_1 I_1\pi a_1^2$$

（只有 ρ < a₁ 的區域有磁場穿過外管截面）

外管總磁通量鏈：

$$\Lambda_{21} = n_2 l\cdot\Phi_{21} = \mu_0 n_1 n_2 l\pi a_1^2 I_1$$

互感：

$$M = \frac{\Lambda_{21}}{I_1} = \mu_0 n_1 n_2\pi a_1^2 l$$

**驗證耦合係數**

各自自感：

$$L_1 = \mu_0 n_1^2\pi a_1^2 l$$

$$L_2 = \mu_0 n_2^2\pi a_2^2 l$$

$$L_1 L_2 = \mu_0^2 n_1^2 n_2^2\pi^2 a_1^2 a_2^2 l^2$$

$$M^2 = \mu_0^2 n_1^2 n_2^2\pi^2 a_1^4 l^2$$

$$k = \frac{M}{\sqrt{L_1 L_2}} = \frac{a_1}{a_2} \leq 1 \quad \checkmark$$

k = 1 當且僅當 a₁ = a₂（兩管完全重合）。

**答案**：

> M = μ₀n₁n₂πa₁²l
>
> 耦合係數 k = a₁/a₂ ≤ 1

**延伸思考**：耦合係數 k 反映了磁通的耦合程度。k = 1 為完全耦合（理想變壓器），k < 1 表示有漏磁通。在實際變壓器設計中，使用鐵心可大幅提高 k。

### 題目 28：磁場的邊界條件
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 邊界條件

**題目**：

z > 0 為磁導率 μ₁ = 5μ₀ 的鐵磁材料，z < 0 為空氣 μ₂ = μ₀。界面無自由面電流。若空氣中的磁場為 **B₂** = 2x̂ + 5ẑ (mT)，求鐵磁材料中的 **B₁** 和 **H₁**。

**詳解**：

**解題思路**：套用磁場邊界條件 — 法向 B 連續，切向 H 連續（無面電流時）。

**步驟**：

法向量 n̂ = ẑ（由區域 2 指向區域 1）。

**分解 B₂**：

- 法向分量：B₂n = 5ẑ (mT)
- 切向分量：B₂t = 2x̂ (mT)

**邊界條件 1：法向 B 連續**

$$B_{1n} = B_{2n} = 5\hat{z} \text{ (mT)}$$

**邊界條件 2：切向 H 連續（無面電流）**

$$H_{1t} = H_{2t}$$

$$\frac{B_{1t}}{\mu_1} = \frac{B_{2t}}{\mu_2}$$

$$B_{1t} = \frac{\mu_1}{\mu_2}B_{2t} = \frac{5\mu_0}{\mu_0}\cdot 2\hat{x} = 10\hat{x} \text{ (mT)}$$

因此：

$$\mathbf{B_1} = 10\hat{x} + 5\hat{z} \text{ (mT)}$$

$$\mathbf{H_1} = \frac{\mathbf{B_1}}{\mu_1} = \frac{10\hat{x}+5\hat{z}}{5\mu_0} = \frac{1}{\mu_0}(2\hat{x}+1\hat{z}) \text{ (A/m 量級)}$$

數值：H₁ = (1/5μ₀)(10x̂ + 5ẑ) × 10⁻³

= (1/(5×4π×10⁻⁷))(10x̂ + 5ẑ) × 10⁻³

= (10⁻³/(2π×10⁻⁶))(2x̂ + ẑ)

= (1000/(2π))(2x̂ + ẑ) ≈ (318.3)(2x̂ + ẑ) A/m

= 636.6x̂ + 318.3ẑ A/m

驗證 H₂：

$$\mathbf{H_2} = \frac{\mathbf{B_2}}{\mu_2} = \frac{2\hat{x}+5\hat{z}}{\mu_0}\times 10^{-3}$$

H₂t = 2×10⁻³/μ₀ x̂ = 1591.5 x̂ A/m

H₁t = 10×10⁻³/(5μ₀) x̂ = 2×10⁻³/μ₀ x̂ = 1591.5 x̂ A/m ✓

**答案**：

> **B₁** = 10x̂ + 5ẑ (mT)
>
> **H₁** = (2x̂ + ẑ)/(5μ₀) × 10⁻³ ≈ 636.6x̂ + 318.3ẑ (A/m)

**折射角的關係**：

tanθ₁/tanθ₂ = μ₁/μ₂ = 5。鐵磁材料中磁力線更趨近平行界面。

**延伸思考**：高磁導率材料會「吸引」磁力線（磁力線傾向於走磁導率高的路徑），這是鐵心能有效集中磁通的物理原因。在 μ₁/μ₂ → ∞ 的極限下，鐵磁材料中的 **B** 幾乎平行於界面。

### 題目 29：磁能與電感
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：靜磁場 — 磁能(Magnetic Energy)

**題目**：

同軸電纜內導體半徑 a（導體磁導率 μ₀），外導體內半徑 b，電流 I 在內導體均勻分布。
(a) 求 ρ < a 和 a < ρ < b 區域的磁場。
(b) 求單位長度的內部電感 L_int'（ρ < a）和外部電感 L_ext'（a < ρ < b）。

**詳解**：

**解題思路**：分區域用安培定律求 B，再用 W_m = ∫(B²/2μ₀)dv 求能量，由 L = 2W_m/I² 得電感。

**步驟**：

**(a) 磁場**

ρ < a：取半徑 ρ 的圓形安培環路

$$B_\phi\cdot 2\pi\rho = \mu_0\cdot I\cdot\frac{\pi\rho^2}{\pi a^2} = \frac{\mu_0 I\rho^2}{a^2}$$

$$B_\phi = \frac{\mu_0 I\rho}{2\pi a^2} \quad (\rho < a)$$

a < ρ < b：

$$B_\phi\cdot 2\pi\rho = \mu_0 I$$

$$B_\phi = \frac{\mu_0 I}{2\pi\rho} \quad (a < \rho < b)$$

**(b) 電感計算**

內部磁能（單位長度）：

$$\frac{W_{int}}{l} = \frac{1}{2\mu_0}\int_0^a B_\phi^2\cdot 2\pi\rho\,d\rho = \frac{1}{2\mu_0}\int_0^a\left(\frac{\mu_0 I\rho}{2\pi a^2}\right)^2 2\pi\rho\,d\rho$$

$$= \frac{\mu_0 I^2}{4\pi a^4}\int_0^a\rho^3\,d\rho = \frac{\mu_0 I^2}{4\pi a^4}\cdot\frac{a^4}{4} = \frac{\mu_0 I^2}{16\pi}$$

$$L_{int}' = \frac{2W_{int}/l}{I^2} = \frac{\mu_0}{8\pi}$$

這就是著名的結果：內部電感為 μ₀/(8π) H/m，與導體半徑無關！

外部磁能（單位長度）：

$$\frac{W_{ext}}{l} = \frac{1}{2\mu_0}\int_a^b\left(\frac{\mu_0 I}{2\pi\rho}\right)^2 2\pi\rho\,d\rho = \frac{\mu_0 I^2}{4\pi}\int_a^b\frac{d\rho}{\rho} = \frac{\mu_0 I^2}{4\pi}\ln\frac{b}{a}$$

$$L_{ext}' = \frac{2W_{ext}/l}{I^2} = \frac{\mu_0}{2\pi}\ln\frac{b}{a}$$

**答案**：

> (a) ρ < a：B_φ = μ₀Iρ/(2πa²)；a < ρ < b：B_φ = μ₀I/(2πρ)
>
> (b) L_int' = μ₀/(8π) ≈ 50 nH/m（與 a 無關！）
>
> L_ext' = μ₀/(2π) · ln(b/a)
>
> 總電感：L' = μ₀/(8π) + μ₀/(2π) · ln(b/a)

**延伸思考**：在高頻時，趨膚效應(Skin Effect)使電流集中在導體表面，內部電感趨近於零，只剩外部電感。同軸電纜的特性阻抗 Z₀ = √(L'/C') = (1/2π)√(μ₀/ε)·ln(b/a)。

### 題目 30：磁路分析
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：靜磁場 — 磁路(Magnetic Circuit)

**題目**：

一環形鐵心（磁導率 μ = 1000μ₀），平均磁路長度 l_c = 0.5 m，截面積 S = 4 cm²，開有氣隙(Air Gap) l_g = 2 mm。繞 N = 200 匝線圈，通電流 I。
(a) 畫出等效磁路。
(b) 求使氣隙中 B = 0.5 T 所需的電流 I。
(c) 求漏磁(Fringing)忽略下的電感。

**詳解**：

**解題思路**：磁路與電路類比 — 磁動勢(MMF)對應 EMF，磁通(Flux)對應電流，磁阻(Reluctance)對應電阻。

**步驟**：

**(a) 等效磁路**

NI（磁動勢）驅動磁通 Φ 通過串聯的鐵心磁阻 R_c 和氣隙磁阻 R_g：

$$NI = \Phi(R_c + R_g)$$

**(b) 計算電流**

磁阻：

$$R_c = \frac{l_c}{\mu S} = \frac{0.5}{1000\times 4\pi\times 10^{-7}\times 4\times 10^{-4}} = \frac{0.5}{5.027\times 10^{-7}} = 9.947\times 10^5 \text{ A·t/Wb}$$

$$R_g = \frac{l_g}{\mu_0 S} = \frac{2\times 10^{-3}}{4\pi\times 10^{-7}\times 4\times 10^{-4}} = \frac{2\times 10^{-3}}{5.027\times 10^{-10}} = 3.979\times 10^6 \text{ A·t/Wb}$$

注意：氣隙磁阻遠大於鐵心磁阻（即使氣隙很短），因為 μ_r = 1000。

R_g/R_c = (l_g/μ₀S)/(l_c/μS) = μl_g/(μ₀l_c) = 1000 × 0.002/0.5 = 4

所需磁通：

$$\Phi = BS = 0.5\times 4\times 10^{-4} = 2\times 10^{-4} \text{ Wb}$$

磁動勢：

$$NI = \Phi(R_c+R_g) = 2\times 10^{-4}\times(9.947\times 10^5+3.979\times 10^6)$$

$$= 2\times 10^{-4}\times 4.974\times 10^6 = 994.7 \text{ A·t}$$

$$I = \frac{994.7}{200} = 4.97 \text{ A}$$

驗證：鐵心中 H_c = B/(μ) = 0.5/(1000×4π×10⁻⁷) = 397.9 A/m

氣隙中 H_g = B/μ₀ = 0.5/(4π×10⁻⁷) = 397887 A/m

NI = H_c·l_c + H_g·l_g = 397.9×0.5 + 397887×0.002 = 198.9 + 795.8 = 994.7 A·t ✓

**(c) 電感**

$$L = \frac{N\Phi}{I} = \frac{N^2}{R_c+R_g} = \frac{200^2}{4.974\times 10^6} = \frac{40000}{4.974\times 10^6} = 8.04\times 10^{-3} \text{ H} \approx 8 \text{ mH}$$

**答案**：

> (b) I ≈ 4.97 A
>
> (c) L ≈ 8.04 mH

### 題目 31：法拉第定律 — 移動導線
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電磁感應 — 法拉第定律(Faraday's Law)

**題目**：

一矩形導線迴路（寬 w，長 l）以速度 **v** = v₀x̂ 沿 x 方向移動，通過一非均勻磁場 **B** = B₀(x/a)ẑ（a 為常數）。迴路在 xy 平面上，其左邊在 x = x₀ 處。
(a) 求穿過迴路的磁通量 Φ(x₀)。
(b) 用法拉第定律 EMF = −dΦ/dt 求感應電動勢。
(c) 用動生電動勢(Motional EMF) ∮(**v**×**B**)·d**l** 驗證結果。

**詳解**：

**解題思路**：三種方法求感應 EMF，應得一致結果。

**步驟**：

迴路的左邊在 x = x₀，右邊在 x = x₀ + l。寬度 w 沿 y 方向。

**(a) 磁通量**

$$\Phi = \int_0^w\int_{x_0}^{x_0+l}\frac{B_0 x}{a}\,dx\,dy = w\cdot\frac{B_0}{a}\left[\frac{x^2}{2}\right]_{x_0}^{x_0+l}$$

$$= \frac{B_0 w}{2a}\left[(x_0+l)^2-x_0^2\right] = \frac{B_0 w}{2a}(2x_0 l+l^2)$$

$$= \frac{B_0 wl}{2a}(2x_0+l)$$

**(b) 法拉第定律**

$$\text{EMF} = -\frac{d\Phi}{dt} = -\frac{B_0 wl}{2a}\cdot 2\frac{dx_0}{dt} = -\frac{B_0 wl v_0}{a}$$

（因為 dx₀/dt = v₀）

**(c) 動生電動勢驗證**

**v** × **B** = v₀x̂ × B₀(x/a)ẑ = v₀B₀(x/a)(x̂ × ẑ) = v₀B₀(x/a)ŷ 的方向...

等一下，x̂ × ẑ = −ŷ。

所以 **v** × **B** = −v₀B₀x/(a) ŷ

動生 EMF 只在與 **v** × **B** 平行的導線段上產生，即 y 方向的兩邊：

左邊 (x = x₀)：d**l** = dy ŷ

$$\int_{left}(\mathbf{v}\times\mathbf{B})\cdot d\mathbf{l} = \int_0^w\left(-\frac{v_0 B_0 x_0}{a}\right)dy = -\frac{v_0 B_0 x_0 w}{a}$$

右邊 (x = x₀+l)：d**l** = −dy ŷ（迴路方向回來）

$$\int_{right}(\mathbf{v}\times\mathbf{B})\cdot d\mathbf{l} = \int_0^w\left(-\frac{v_0 B_0(x_0+l)}{a}\right)(-dy) = \frac{v_0 B_0(x_0+l)w}{a}$$

上下兩邊（x 方向）與 **v**×**B**（y 方向）垂直，無貢獻。

總動生 EMF：

$$\text{EMF} = -\frac{v_0 B_0 x_0 w}{a} + \frac{v_0 B_0(x_0+l)w}{a} = \frac{v_0 B_0 lw}{a}$$

注意正負號的約定：上面法拉第定律得到 −v₀B₀lw/a，動生 EMF 得到 +v₀B₀lw/a。差異在於 EMF 的正方向定義，兩者在物理上一致（電流方向相同）。

**答案**：

> |EMF| = v₀B₀lw/a

**延伸思考**：當磁場隨時間也在變化時，感應 EMF 有兩個來源：(1) 磁場本身的時變（感生電動勢 Transformer EMF），(2) 迴路的運動（動生電動勢 Motional EMF）。法拉第定律 EMF = −dΦ/dt 自動包含了兩者。

### 題目 32：法拉第定律 — 感應電場
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電磁感應 — 感應電場(Induced Electric Field)

**題目**：

一長螺線管（半徑 a，匝數密度 n），其中磁場以 B(t) = B₀sinωt 隨時間變化。求：
(a) 螺線管內部 (ρ < a) 的感應電場 **E**。
(b) 螺線管外部 (ρ > a) 的感應電場 **E**。
(c) 在 ρ = a/2 處放一半徑為 a/2 的圓形導線環（電阻 R），求環上的感應電流。

**詳解**：

**解題思路**：用法拉第定律的積分形式 ∮**E**·d**l** = −dΦ/dt，取圓形路徑利用軸對稱性。

**步驟**：

**(a) ρ < a**

取半徑 ρ（ρ < a）的圓形安培迴路。由對稱性，**E** = E_φ(ρ)φ̂。

$$\oint\mathbf{E}\cdot d\mathbf{l} = E_\phi\cdot 2\pi\rho$$

$$-\frac{d\Phi}{dt} = -\frac{d}{dt}(B\cdot\pi\rho^2) = -\pi\rho^2 B_0\omega\cos\omega t$$

$$E_\phi\cdot 2\pi\rho = -\pi\rho^2 B_0\omega\cos\omega t$$

$$E_\phi = -\frac{\rho B_0\omega\cos\omega t}{2}$$

$$\mathbf{E} = -\frac{\rho B_0\omega\cos\omega t}{2}\hat{\phi} \quad (\rho < a)$$

**(b) ρ > a**

取半徑 ρ（ρ > a）的圓形路徑。磁通只在 ρ < a 內：

$$E_\phi\cdot 2\pi\rho = -\frac{d}{dt}(B\cdot\pi a^2) = -\pi a^2 B_0\omega\cos\omega t$$

$$E_\phi = -\frac{a^2 B_0\omega\cos\omega t}{2\rho}$$

$$\mathbf{E} = -\frac{a^2 B_0\omega\cos\omega t}{2\rho}\hat{\phi} \quad (\rho > a)$$

**(c) 感應電流**

導線環在 ρ = a/2 處（完全在螺線管內部），半徑 r = a/2：

$$\text{EMF} = \oint\mathbf{E}\cdot d\mathbf{l}$$

但注意環不在原點中心，它偏移了。讓我重新思考。

題目說「在 ρ = a/2 處放一半徑為 a/2 的圓形導線環」，也就是環的中心在 ρ = a/2，環的半徑也是 a/2。此環恰好經過原點和 ρ = a 處。

此時不能直接用 E_φ × 2πr，因為 E 場沿此環不是常數。應直接用磁通量法：

穿過此環的磁通量 = B(t) × 環面積

由於環完全在 ρ < a 範圍內（環心在 ρ = a/2，半徑 a/2，所以最遠點恰在 ρ = a），而且 B 在 ρ < a 內均勻：

$$\Phi = B(t)\cdot\pi\left(\frac{a}{2}\right)^2 = \frac{\pi a^2}{4}B_0\sin\omega t$$

$$\text{EMF} = -\frac{d\Phi}{dt} = -\frac{\pi a^2}{4}B_0\omega\cos\omega t$$

$$I = \frac{\text{EMF}}{R} = -\frac{\pi a^2 B_0\omega\cos\omega t}{4R}$$

**答案**：

> (a) ρ < a：**E** = −(ρB₀ωcosωt/2)φ̂
>
> (b) ρ > a：**E** = −(a²B₀ωcosωt/2ρ)φ̂
>
> (c) I = −πa²B₀ωcosωt/(4R)

**延伸思考**：感應電場 **E** 的旋度不為零（∇ × **E** = −∂**B**/∂t），因此它不是保守場，不能用純量電位描述。在 ρ < a 處 |E| ∝ ρ（線性增長），在 ρ > a 處 |E| ∝ 1/ρ（反比衰減），在 ρ = a 處連續。

### 題目 33：楞次定律與渦電流
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電磁感應 — 楞次定律(Lenz's Law)

**題目**：

一金屬環（電阻 R，自感 L）從高處自由落下，穿過一水平放置的磁鐵上方區域（磁場向上且不均勻）。
(a) 當環接近磁鐵（下落中）時，感應電流方向為何？
(b) 列出環中電流的微分方程。
(c) 說明環的運動如何受影響。

**詳解**：

**解題思路**：楞次定律 — 感應電流的方向總是抗拒引起感應的磁通變化。

**步驟**：

**(a) 電流方向**

環接近磁鐵時，穿過環的向上磁通在增加（磁鐵附近磁場更強）。

由楞次定律，感應電流的方向應使其產生的磁場抵消磁通的增加，即感應電流產生向下的磁場。

由右手定則，從上方看，感應電流為順時針方向。

**(b) 微分方程**

設磁通量 Φ(z) 為環位置 z 的函數（z 向下為正）。

法拉第定律：

$$\text{EMF} = -\frac{d\Phi}{dt} = -\frac{d\Phi}{dz}\cdot\frac{dz}{dt} = -\frac{d\Phi}{dz}\cdot v$$

迴路方程（含自感）：

$$L\frac{di}{dt} + Ri = -\frac{d\Phi}{dz}\cdot v$$

配合牛頓第二定律（環質量 m）：

$$m\frac{dv}{dt} = mg - F_{drag}$$

其中制動力（磁力對環向上的分力）：

$$F_{drag} = i\cdot\frac{d\Phi}{dz}$$

（負號已含在方向中）

**(c) 運動的影響**

1. 接近磁鐵時：磁通增加 → 感應電流 → 電流與磁場交互產生向上的力 → 減速（楞次定律的力學效應）
2. 離開磁鐵時：磁通減少 → 感應電流反向 → 仍然產生向上的力 → 同樣減速
3. 環的動能部分轉化為焦耳熱 P = i²R

結果：金屬環的下落速度比無磁場時慢，部分重力位能轉化為熱能。若 R → 0（超導環），制動力最大，環可能懸浮（磁浮效應）。

**答案**：

> (a) 從上方看為順時針（產生向下磁場以抵抗向上磁通增加）
>
> (b) L di/dt + Ri = −(dΦ/dz)v，配合 m dv/dt = mg − i(dΦ/dz)
>
> (c) 環受到電磁制動力，下落減速，能量以焦耳熱形式耗散

**延伸思考**：這就是電磁煞車(Electromagnetic Braking)的原理，應用於高鐵煞車系統、自由落體減速裝置等。超導體的完美抗磁性（邁斯納效應, Meissner Effect）也是楞次定律的極端情形。

### 題目 34：位移電流
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：馬克士威方程 — 位移電流(Displacement Current)

**題目**：

一平行板電容器（板面積 S，間距 d，介電常數 ε），正在被充電，充電電流 I(t) = I₀cosωt。
(a) 求電容器內的位移電流密度 **J_d**。
(b) 驗證位移電流 I_d = I（電流連續性）。
(c) 求電容器內的磁場 **H**(ρ)（ρ 為到中心軸的距離，ρ < R，R 為板半徑）。

**詳解**：

**解題思路**：位移電流密度 **J_d** = ∂**D**/∂t，由廣義安培定律 ∇ × **H** = **J** + ∂**D**/∂t 統一處理。

**步驟**：

**(a) 位移電流密度**

電容器中電場（均勻）：

$$E = \frac{V}{d} = \frac{Q}{Cd} \cdot \frac{1}{C\cdot d/Q}$$

更直接地，由 I = dQ/dt 且 Q = CV，D = εE = Q/S：

$$\frac{\partial D}{\partial t} = \frac{1}{S}\frac{dQ}{dt} = \frac{I(t)}{S}$$

$$\mathbf{J_d} = \frac{\partial\mathbf{D}}{\partial t} = \frac{I_0\cos\omega t}{S}\hat{z}$$

**(b) 位移電流**

$$I_d = \int_S\mathbf{J_d}\cdot d\mathbf{S} = \frac{I_0\cos\omega t}{S}\cdot S = I_0\cos\omega t = I(t) \quad \checkmark$$

位移電流等於傳導電流(Conduction Current)，保證了電流的連續性。

**(c) 電容器內磁場**

取半徑 ρ（ρ < R）的圓形安培環路，面積內無傳導電流，只有位移電流：

$$\oint\mathbf{H}\cdot d\mathbf{l} = H_\phi\cdot 2\pi\rho = I_{d,enc} = J_d\cdot\pi\rho^2 = \frac{I_0\cos\omega t}{S}\cdot\pi\rho^2$$

$$H_\phi = \frac{I_0\cos\omega t\cdot\rho}{2S} = \frac{I_0\rho\cos\omega t}{2\pi R^2}$$

（因 S = πR²）

**答案**：

> (a) **J_d** = I₀cosωt/S ẑ
>
> (b) I_d = I₀cosωt = I(t)，電流連續
>
> (c) **H** = I₀ρcosωt/(2πR²) φ̂，(ρ < R)

**延伸思考**：馬克士威加入位移電流項是電磁學史上最偉大的理論預測之一。正是位移電流使得馬克士威方程具有波動解，預言了電磁波的存在。在高頻電路中，電容器「通交流」的物理機制正是位移電流。

### 題目 35：馬克士威方程 — 積分形式與微分形式
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：馬克士威方程(Maxwell's Equations)

**題目**：

寫出馬克士威四方程的積分形式和微分形式，並說明每個方程的物理意義。從微分形式推導連續性方程(Continuity Equation)。

**詳解**：

**解題思路**：系統性整理四方程，並利用向量恆等式推導連續性方程。

**步驟**：

**馬克士威四方程**

| 名稱 | 微分形式 | 積分形式 | 物理意義 |
|------|---------|---------|---------|
| 高斯定律(電) | ∇·**D** = ρ_v | ∮**D**·d**S** = Q_enc | 電荷是電力線的源 |
| 高斯定律(磁) | ∇·**B** = 0 | ∮**B**·d**S** = 0 | 不存在磁單極(Magnetic Monopole) |
| 法拉第定律 | ∇×**E** = −∂**B**/∂t | ∮**E**·d**l** = −dΦ_B/dt | 時變磁場產生電場 |
| 廣義安培定律 | ∇×**H** = **J** + ∂**D**/∂t | ∮**H**·d**l** = I_enc + dΦ_D/dt | 電流和時變電場產生磁場 |

**推導連續性方程**

對廣義安培定律取散度：

$$\nabla\cdot(\nabla\times\mathbf{H}) = \nabla\cdot\mathbf{J} + \nabla\cdot\frac{\partial\mathbf{D}}{\partial t}$$

左側：∇·(∇×**H**) ≡ 0（任何向量旋度的散度為零）

右側：

$$0 = \nabla\cdot\mathbf{J} + \frac{\partial}{\partial t}(\nabla\cdot\mathbf{D}) = \nabla\cdot\mathbf{J} + \frac{\partial\rho_v}{\partial t}$$

得到連續性方程：

$$\boxed{\nabla\cdot\mathbf{J} + \frac{\partial\rho_v}{\partial t} = 0}$$

積分形式（配合散度定理）：

$$\oint_S\mathbf{J}\cdot d\mathbf{S} = -\frac{dQ_{enc}}{dt}$$

意義：流出封閉面的淨電流等於內部電荷的減少率，即電荷守恆(Conservation of Charge)。

**答案**：

> 馬克士威四方程如上表。
>
> 連續性方程：∇·**J** + ∂ρ_v/∂t = 0（電荷守恆）

**延伸思考**：若沒有位移電流項 ∂**D**/∂t，安培定律 ∇×**H** = **J** 取散度後得 ∇·**J** = 0，這只在穩態(Steady State)成立。馬克士威加入位移電流正是為了使方程與連續性方程（電荷守恆）自洽。

### 題目 36：波動方程推導
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：馬克士威方程 — 波動方程

**題目**：

從自由空間中的馬克士威方程（ρ_v = 0, **J** = 0）推導 **E** 和 **H** 的波動方程(Wave Equation)，並求電磁波在自由空間中的傳播速度。

**詳解**：

**解題思路**：對法拉第定律取旋度，利用向量恆等式和高斯定律消去 **B**。

**步驟**：

自由空間馬克士威方程：

(i) ∇·**E** = 0

(ii) ∇·**B** = 0

(iii) ∇×**E** = −∂**B**/∂t

(iv) ∇×**B** = μ₀ε₀ ∂**E**/∂t

對 (iii) 取旋度：

$$\nabla\times(\nabla\times\mathbf{E}) = -\frac{\partial}{\partial t}(\nabla\times\mathbf{B})$$

左側用向量恆等式：

$$\nabla\times(\nabla\times\mathbf{E}) = \nabla(\nabla\cdot\mathbf{E}) - \nabla^2\mathbf{E} = -\nabla^2\mathbf{E}$$

（因為 ∇·**E** = 0）

右側代入 (iv)：

$$-\frac{\partial}{\partial t}(\mu_0\varepsilon_0\frac{\partial\mathbf{E}}{\partial t}) = -\mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2}$$

因此：

$$\nabla^2\mathbf{E} = \mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2}$$

同理對 (iv) 取旋度可得：

$$\nabla^2\mathbf{B} = \mu_0\varepsilon_0\frac{\partial^2\mathbf{B}}{\partial t^2}$$

這是三維波動方程，一般形式 ∇²f = (1/v²)∂²f/∂t²，波速為：

$$v = \frac{1}{\sqrt{\mu_0\varepsilon_0}}$$

代入數值：

$$v = \frac{1}{\sqrt{4\pi\times 10^{-7}\times 8.854\times 10^{-12}}} = \frac{1}{\sqrt{1.113\times 10^{-17}}}$$

$$= \frac{1}{3.336\times 10^{-9}} = 2.998\times 10^8 \text{ m/s} = c$$

這就是光速！馬克士威由此預言光是電磁波。

**答案**：

> ∇²**E** = μ₀ε₀ ∂²**E**/∂t²
>
> ∇²**B** = μ₀ε₀ ∂²**B**/∂t²
>
> 傳播速度 v = 1/√(μ₀ε₀) = c ≈ 3×10⁸ m/s

**延伸思考**：在介質中波速 v = 1/√(με) = c/n，其中 n = √(μ_rε_r) 為折射率(Refractive Index)。在良導體中波動方程需加入歐姆損耗項(Ohmic Loss)σ∂**E**/∂t，導致電磁波衰減。

### 題目 37：時諧場(Time-Harmonic Field)與相量表示
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：馬克士威方程 — 相量(Phasor)

**題目**：

一均勻平面波在自由空間中沿 +z 方向傳播，電場為 **E** = E₀cos(ωt − βz)x̂。
(a) 寫出相量形式 **Ẽ**。
(b) 由馬克士威方程求磁場 **H**。
(c) 求波數 β、波長 λ、相速度 v_p。

**詳解**：

**解題思路**：時諧場用相量表示可將微分方程轉為代數方程，∂/∂t → jω。

**步驟**：

**(a) 相量形式**

時域：**E** = E₀cos(ωt − βz)x̂ = Re{E₀e^{j(ωt−βz)}x̂}

相量（去掉 e^{jωt}）：

$$\tilde{\mathbf{E}} = E_0 e^{-j\beta z}\hat{x}$$

**(b) 由法拉第定律求 H**

相量形式的法拉第定律：∇ × **Ẽ** = −jω**B̃** = −jωμ₀**H̃**

$$\nabla\times\tilde{\mathbf{E}} = \begin{vmatrix}\hat{x}&\hat{y}&\hat{z}\\\frac{\partial}{\partial x}&\frac{\partial}{\partial y}&\frac{\partial}{\partial z}\\E_0 e^{-j\beta z}&0&0\end{vmatrix}$$

x̂ 分量：0 − 0 = 0

ŷ 分量：0 − (−jβ)E₀e^{-jβz} = jβE₀e^{-jβz}

ẑ 分量：0 − 0 = 0

$$\nabla\times\tilde{\mathbf{E}} = j\beta E_0 e^{-j\beta z}\hat{y}$$

$$\tilde{\mathbf{H}} = \frac{\nabla\times\tilde{\mathbf{E}}}{-j\omega\mu_0} = \frac{j\beta E_0 e^{-j\beta z}}{-j\omega\mu_0}\hat{y} = \frac{-\beta E_0 e^{-j\beta z}}{\omega\mu_0}\hat{y}$$

注意到 β/ωμ₀ = 1/η₀，其中 η₀ = √(μ₀/ε₀) ≈ 377 Ω 為自由空間波阻抗(Wave Impedance)：

$$\tilde{\mathbf{H}} = -\frac{E_0}{\eta_0}e^{-j\beta z}\hat{y}$$

等一下，方向需要仔細確認。**E** 沿 x̂，波沿 +z，則 **H** 應沿 +ŷ：

$$\tilde{\mathbf{H}} = \frac{E_0}{\eta_0}e^{-j\beta z}\hat{y}$$

讓我重新計算：β = ω√(μ₀ε₀) = ω/c > 0

$$\frac{\beta}{\omega\mu_0} = \frac{\omega\sqrt{\mu_0\varepsilon_0}}{\omega\mu_0} = \sqrt{\frac{\varepsilon_0}{\mu_0}} = \frac{1}{\eta_0}$$

所以 H̃ = −(1/η₀)E₀e^{-jβz}(−ŷ)...

讓我直接用結果：對沿 +z 傳播的均勻平面波，**H** = (1/η₀)(ẑ × **E**)：

$$\tilde{\mathbf{H}} = \frac{1}{\eta_0}(\hat{z}\times E_0 e^{-j\beta z}\hat{x}) = \frac{E_0}{\eta_0}e^{-j\beta z}\hat{y}$$

時域：

$$\mathbf{H} = \frac{E_0}{\eta_0}\cos(\omega t-\beta z)\hat{y}$$

**(c) 波參數**

$$\beta = \omega\sqrt{\mu_0\varepsilon_0} = \frac{\omega}{c} = \frac{2\pi f}{c} = \frac{2\pi}{\lambda}$$

$$\lambda = \frac{2\pi}{\beta} = \frac{c}{f}$$

$$v_p = \frac{\omega}{\beta} = \frac{1}{\sqrt{\mu_0\varepsilon_0}} = c$$

**答案**：

> (a) **Ẽ** = E₀e^{-jβz} x̂
>
> (b) **H** = (E₀/η₀)cos(ωt−βz) ŷ，η₀ = √(μ₀/ε₀) ≈ 377 Ω
>
> (c) β = ω/c, λ = c/f, v_p = c

**延伸思考**：**E** 和 **H** 同相位、互相垂直、且都垂直於傳播方向，這是 TEM（橫電磁波, Transverse Electromagnetic Wave）的特徵。能量傳播方向由波印亭向量 **S** = **E** × **H** 確定。

### 題目 38：時變場中的邊界條件
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：馬克士威方程 — 時變邊界條件

**題目**：

一均勻平面波從自由空間垂直入射到一完美導體(Perfect Electric Conductor, PEC)表面（z = 0 平面）。
(a) 列出 PEC 表面的邊界條件。
(b) 若入射波 **E_i** = E₀e^{-jβz} x̂，求反射波 **E_r** 和總電場 **E**。
(c) 求合成波的磁場 **H** 和表面電流密度 **J_s**。

**詳解**：

**解題思路**：PEC 表面切向電場為零，以此決定反射波。

**步驟**：

**(a) PEC 邊界條件**

1. n̂ × **E** = 0（切向 E 為零）
2. n̂ × **H** = **J_s**（切向 H 等於表面電流密度）
3. n̂ · **B** = 0（法向 B 為零）
4. n̂ · **D** = ρ_s（法向 D 等於面電荷密度）

其中 n̂ = −ẑ（指向自由空間）。

**(b) 反射波與總電場**

反射波沿 −z 方向傳播：**Ẽ_r** = E_r₀ e^{+jβz} x̂

在 z = 0 處，切向 E 為零：

$$E_0 + E_{r0} = 0 \implies E_{r0} = -E_0$$

$$\tilde{\mathbf{E}}_r = -E_0 e^{+j\beta z}\hat{x}$$

總電場相量：

$$\tilde{\mathbf{E}} = E_0 e^{-j\beta z}\hat{x} - E_0 e^{+j\beta z}\hat{x} = E_0(e^{-j\beta z}-e^{+j\beta z})\hat{x} = -2jE_0\sin(\beta z)\hat{x}$$

時域：

$$\mathbf{E} = 2E_0\sin(\beta z)\sin(\omega t)\hat{x}$$

這是駐波(Standing Wave)！

**(c) 磁場與表面電流**

入射波磁場：**H̃_i** = (E₀/η₀)e^{-jβz} ŷ

反射波磁場（注意方向）：**H̃_r** = (E₀/η₀)e^{+jβz} ŷ

（反射波沿 −z 傳播，**H** = (1/η₀)(−ẑ) × (−E₀x̂)e^{+jβz} = (E₀/η₀)e^{+jβz}ŷ）

總磁場：

$$\tilde{\mathbf{H}} = \frac{E_0}{\eta_0}(e^{-j\beta z}+e^{+j\beta z})\hat{y} = \frac{2E_0}{\eta_0}\cos(\beta z)\hat{y}$$

時域：

$$\mathbf{H} = \frac{2E_0}{\eta_0}\cos(\beta z)\cos(\omega t)\hat{y}$$

表面電流密度（z = 0 處）：

$$\mathbf{J_s} = \hat{n}\times\mathbf{H}|_{z=0} = (-\hat{z})\times\frac{2E_0}{\eta_0}\hat{y} = \frac{2E_0}{\eta_0}(-\hat{z}\times\hat{y}) = \frac{2E_0}{\eta_0}\hat{x}$$

相量形式：**J̃_s** = (2E₀/η₀) x̂

**答案**：

> (b) **E** = 2E₀sin(βz)sin(ωt) x̂（駐波）
>
> (c) **H** = (2E₀/η₀)cos(βz)cos(ωt) ŷ
>
> **J_s** = (2E₀/η₀)cos(ωt) x̂

**延伸思考**：E 和 H 在時間上相差 90°（一個是 sinωt，另一個是 cosωt），時間平均波印亭向量為零，能量不被傳輸，只在空間中來回振盪。E 的節點(Node)在 z = nλ/2，H 的節點在 z = (2n+1)λ/4。

### 題目 39：波印亭定理
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：馬克士威方程 — 波印亭定理(Poynting's Theorem)

**題目**：

(a) 從馬克士威方程推導波印亭定理。
(b) 一同軸電纜，內導體半徑 a、外導體內半徑 b，通直流電流 I，兩導體間電壓 V。用波印亭向量驗證傳輸功率 P = VI。

**詳解**：

**解題思路**：(a) 用 **E**·(iv) − **H**·(iii) 的向量技巧。(b) 求同軸結構中的 **S** = **E** × **H** 並積分。

**步驟**：

**(a) 推導波印亭定理**

由法拉第定律 (iii) 兩邊點乘 **H**：

$$\mathbf{H}\cdot(\nabla\times\mathbf{E}) = -\mathbf{H}\cdot\frac{\partial\mathbf{B}}{\partial t} = -\mu\mathbf{H}\cdot\frac{\partial\mathbf{H}}{\partial t} = -\frac{\partial}{\partial t}\left(\frac{\mu H^2}{2}\right)$$

由廣義安培定律 (iv) 兩邊點乘 **E**：

$$\mathbf{E}\cdot(\nabla\times\mathbf{H}) = \mathbf{E}\cdot\mathbf{J} + \mathbf{E}\cdot\frac{\partial\mathbf{D}}{\partial t} = \sigma E^2 + \frac{\partial}{\partial t}\left(\frac{\varepsilon E^2}{2}\right)$$

利用向量恆等式 ∇·(**E**×**H**) = **H**·(∇×**E**) − **E**·(∇×**H**)：

$$\nabla\cdot(\mathbf{E}\times\mathbf{H}) = -\frac{\partial}{\partial t}\left(\frac{\mu H^2}{2}\right) - \sigma E^2 - \frac{\partial}{\partial t}\left(\frac{\varepsilon E^2}{2}\right)$$

定義波印亭向量 **S** = **E** × **H**，能量密度 w = εE²/2 + μH²/2：

$$\nabla\cdot\mathbf{S} = -\frac{\partial w}{\partial t} - \sigma E^2$$

積分形式（波印亭定理）：

$$-\oint_S\mathbf{S}\cdot d\mathbf{S} = \frac{\partial}{\partial t}\int_V w\,dv + \int_V\sigma E^2\,dv$$

物理意義：流入封閉面的功率 = 內部電磁能增加率 + 歐姆損耗功率。

**(b) 同軸電纜功率傳輸**

電場（a < ρ < b）：

$$\mathbf{E} = \frac{V}{\rho\ln(b/a)}\hat{\rho}$$

磁場：

$$\mathbf{H} = \frac{I}{2\pi\rho}\hat{\phi}$$

波印亭向量：

$$\mathbf{S} = \mathbf{E}\times\mathbf{H} = \frac{V}{\rho\ln(b/a)}\cdot\frac{I}{2\pi\rho}(\hat{\rho}\times\hat{\phi}) = \frac{VI}{2\pi\rho^2\ln(b/a)}\hat{z}$$

沿 +z 方向傳輸！

總功率（對截面積分）：

$$P = \int_a^b\int_0^{2\pi}\frac{VI}{2\pi\rho^2\ln(b/a)}\rho\,d\rho\,d\phi$$

$$= \frac{VI}{\ln(b/a)}\int_a^b\frac{d\rho}{\rho} = \frac{VI}{\ln(b/a)}\cdot\ln\frac{b}{a} = VI \quad \checkmark$$

**答案**：

> (a) ∇·**S** + ∂w/∂t + σE² = 0（波印亭定理）
>
> (b) P = ∫**S**·d**S** = VI（功率由電磁場攜帶，在導體之間的空間傳輸）

**延伸思考**：令人驚訝的是，同軸電纜中的功率並非在導體中傳輸，而是由導體間空間中的電磁場攜帶。導體只是引導電磁場的邊界。這個觀念對理解傳輸線理論(Transmission Line Theory)非常重要。

### 題目 40：電磁波在良導體中的衰減
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：馬克士威方程 — 良導體(Good Conductor)

**題目**：

一頻率 f = 1 MHz 的電磁波入射到銅（σ = 5.8×10⁷ S/m, μ = μ₀, ε = ε₀）表面。
(a) 驗證銅在此頻率下為良導體（σ >> ωε）。
(b) 求衰減常數 α、相位常數 β、趨膚深度 δ。
(c) 波在銅中傳播 1 mm 後，振幅衰減到原來的幾分之幾？

**詳解**：

**解題思路**：良導體近似下 α ≈ β ≈ √(πfμσ)，趨膚深度 δ = 1/α。

**步驟**：

**(a) 良導體判別**

$$\frac{\sigma}{\omega\varepsilon} = \frac{5.8\times 10^7}{2\pi\times 10^6\times 8.854\times 10^{-12}} = \frac{5.8\times 10^7}{5.563\times 10^{-5}} = 1.043\times 10^{12} >> 1$$

σ/ωε 比 1 大了 12 個數量級，銅在 1 MHz 下絕對是良導體。

**(b) 衰減常數、相位常數、趨膚深度**

良導體近似：

$$\alpha = \beta = \sqrt{\pi f\mu_0\sigma}$$

$$= \sqrt{\pi\times 10^6\times 4\pi\times 10^{-7}\times 5.8\times 10^7}$$

$$= \sqrt{\pi\times 10^6\times 72.88}$$

$$= \sqrt{72.88\pi\times 10^6}$$

$$= \sqrt{2.289\times 10^8}$$

$$= 1.513\times 10^4 \text{ Np/m (或 rad/m)}$$

趨膚深度：

$$\delta = \frac{1}{\alpha} = \frac{1}{1.513\times 10^4} = 6.61\times 10^{-5} \text{ m} = 66.1 \text{ μm}$$

或用公式直接計算：

$$\delta = \frac{1}{\sqrt{\pi f\mu_0\sigma}} = \sqrt{\frac{1}{\pi\times 10^6\times 4\pi\times 10^{-7}\times 5.8\times 10^7}} = 66.1 \text{ μm}$$

**(c) 1 mm 深度的衰減**

$$\frac{E(z=1\text{mm})}{E(z=0)} = e^{-\alpha z} = e^{-1.513\times 10^4\times 10^{-3}} = e^{-15.13}$$

$$= 2.68\times 10^{-7}$$

衰減至原來的約 2.7×10⁻⁷，即幾乎完全衰減。

以 dB 表示：20 log₁₀(2.68×10⁻⁷) = −131.4 dB

**答案**：

> (a) σ/ωε ≈ 10¹² >> 1，確為良導體
>
> (b) α = β ≈ 1.513×10⁴ m⁻¹，δ ≈ 66.1 μm
>
> (c) 1 mm 深度衰減至原來的 2.68×10⁻⁷（約 −131 dB）

**延伸思考**：趨膚深度 δ ∝ 1/√f，頻率越高穿透越淺。在 60 Hz 工頻下銅的 δ ≈ 8.5 mm，在 10 GHz 微波下 δ ≈ 0.66 μm。這就是為什麼高頻電路中導體只需鍍薄層銅即可，也是電磁屏蔽(Electromagnetic Shielding)的基礎。

### 題目 41：均勻平面波 — E, H, S 完整計算
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電磁波 — 均勻平面波(Uniform Plane Wave)

**題目**：

一均勻平面波在介質（ε_r = 4, μ_r = 1, σ = 0）中沿 +z 方向傳播，頻率 f = 100 MHz，電場振幅 E₀ = 10 V/m。
(a) 求波數 β、波長 λ、相速度 v_p。
(b) 求波阻抗 η 和磁場振幅 H₀。
(c) 求時間平均波印亭向量(Time-Average Poynting Vector) **S**_av。
(d) 寫出 **E**(z,t) 和 **H**(z,t) 的完整表達式。

**詳解**：

**解題思路**：套用無損介質中平面波的基本公式。

**步驟**：

**(a) 波參數**

$$v_p = \frac{c}{\sqrt{\varepsilon_r\mu_r}} = \frac{3\times 10^8}{\sqrt{4}} = 1.5\times 10^8 \text{ m/s}$$

$$\lambda = \frac{v_p}{f} = \frac{1.5\times 10^8}{10^8} = 1.5 \text{ m}$$

$$\beta = \frac{2\pi}{\lambda} = \frac{2\pi}{1.5} = 4.189 \text{ rad/m}$$

或 β = ω√(με) = ω√(μ₀ε₀·μ_rε_r) = (ω/c)√(ε_r) = (2π×10⁸/3×10⁸)×2 = 4.189 rad/m ✓

**(b) 波阻抗**

$$\eta = \sqrt{\frac{\mu}{\varepsilon}} = \sqrt{\frac{\mu_0}{\varepsilon_0\varepsilon_r}} = \frac{\eta_0}{\sqrt{\varepsilon_r}} = \frac{377}{2} = 188.5 \text{ Ω}$$

$$H_0 = \frac{E_0}{\eta} = \frac{10}{188.5} = 53.1 \text{ mA/m}$$

**(c) 時間平均波印亭向量**

$$\mathbf{S}_{av} = \frac{1}{2}\text{Re}(\tilde{\mathbf{E}}\times\tilde{\mathbf{H}}^*) = \frac{E_0^2}{2\eta}\hat{z} = \frac{100}{2\times 188.5}\hat{z} = 0.265\hat{z} \text{ W/m}^2$$

**(d) 完整表達式**

設 **E** 沿 x̂ 極化：

$$\mathbf{E}(z,t) = 10\cos(2\pi\times 10^8 t - 4.189z)\hat{x} \text{ (V/m)}$$

$$\mathbf{H}(z,t) = 53.1\times 10^{-3}\cos(2\pi\times 10^8 t - 4.189z)\hat{y} \text{ (A/m)}$$

**答案**：

> (a) β = 4.189 rad/m, λ = 1.5 m, v_p = 1.5×10⁸ m/s
>
> (b) η = 188.5 Ω, H₀ = 53.1 mA/m
>
> (c) S_av = 0.265 W/m² (沿 +z)
>
> (d) 如上式

**延伸思考**：在 ε_r = 4 的介質中，波速只有真空中的一半，波長也減半。這就是為什麼微波電路中使用高介電常數基板可以縮小電路尺寸（但代價是頻寬變窄）。

### 題目 42：極化判別
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：電磁波 — 極化(Polarization)

**題目**：

判別以下各電場的極化類型（線極化 Linear、圓極化 Circular、橢圓極化 Elliptical），並說明旋轉方向（左旋 LHC 或右旋 RHC）：
(a) **E** = (3x̂ + 4ŷ)cos(ωt − βz)
(b) **E** = E₀[cos(ωt − βz)x̂ + cos(ωt − βz + 90°)ŷ]
(c) **E** = E₀[cos(ωt − βz)x̂ + 2sin(ωt − βz)ŷ]
(d) **E** = E₀[cos(ωt − βz)x̂ − sin(ωt − βz)ŷ]

**詳解**：

**解題思路**：判別方法 — 寫出相量的 x 和 y 分量，分析振幅比和相位差。

**步驟**：

判別規則：設 Ẽ_x = A e^{jφ_x}，Ẽ_y = B e^{jφ_y}，相位差 δ = φ_y − φ_x

- δ = 0 或 π：線極化
- δ = ±π/2 且 A = B：圓極化
- 其他：橢圓極化
- 旋轉方向：面對波來的方向（看向 −z），若電場向量逆時針旋轉為左旋(LHC)，順時針為右旋(RHC)。等價地，sinδ > 0 為左旋，sinδ < 0 為右旋（IEEE 定義）。

**(a) E = (3x̂ + 4ŷ)cos(ωt − βz)**

兩分量同相位（δ = 0），振幅不同。

→ **線極化**，極化方向與 x 軸夾角 θ = arctan(4/3) ≈ 53.1°

**(b) E = E₀[cos(ωt−βz)x̂ + cos(ωt−βz+90°)ŷ]**

相量：Ẽ_x = E₀, Ẽ_y = E₀ e^{j90°} = jE₀

振幅相同，相位差 δ = 90° = π/2。

→ **圓極化**

旋轉方向：sinδ = sin(π/2) = 1 > 0

→ **左旋圓極化(LHCP)**

驗證：在 z = 0 處，E_x = E₀cosωt, E_y = E₀cos(ωt+π/2) = −E₀sinωt

t = 0：E = E₀x̂

t = π/2ω：E = −E₀ŷ

面對波來的方向（看向 −z 方向），x̂ 轉到 −ŷ 是順時針...

讓我用 IEEE 標準更仔細地判斷。觀察者在 +z 方向遠處看向 −z（波朝觀察者前進）：

t = 0：(E₀, 0)
t = T/4：(0, −E₀)

從 (E₀, 0) 到 (0, −E₀) 是順時針 → **右旋圓極化(RHCP)**

（注意：不同教科書對左旋/右旋的定義可能不同。此處採用 IEEE 標準：面對波傳播方向，順時針為右旋。）

**(c) E = E₀[cos(ωt−βz)x̂ + 2sin(ωt−βz)ŷ]**

相量：Ẽ_x = E₀, Ẽ_y = 2E₀ e^{-jπ/2}（因 sin = cos(·−π/2)所以相位為 −90°）

等一下：2sin(ωt−βz) = 2cos(ωt−βz−π/2)

所以 Ẽ_y = 2E₀ e^{-jπ/2} = −2jE₀

振幅不同（E₀ vs 2E₀），相位差 δ = −π/2。

→ **橢圓極化**

sinδ = sin(−π/2) = −1 < 0

→ **右旋橢圓極化(RHEP)**（IEEE 標準下）

但需要再確認，若用物理學的定義（根據角動量），結論可能相反。此處統一用 IEEE/工程定義。

**(d) E = E₀[cos(ωt−βz)x̂ − sin(ωt−βz)ŷ]**

相量：Ẽ_x = E₀

−sin(ωt−βz) = cos(ωt−βz+π/2)，所以 Ẽ_y = E₀ e^{jπ/2} = jE₀

振幅相同，相位差 δ = π/2。

→ **圓極化**

在 z = 0 處：t = 0：(E₀, 0)；t = T/4：(0, E₀)

面對波前進方向，(E₀, 0) → (0, E₀) 是逆時針 → **左旋圓極化(LHCP)**

**答案**：

> (a) 線極化，與 x 軸夾 53.1°
>
> (b) 圓極化（右旋 RHCP，IEEE 定義）
>
> (c) 橢圓極化（右旋 RHEP）
>
> (d) 圓極化（左旋 LHCP）

**延伸思考**：任何極化可分解為兩個正交圓極化的疊加（LHCP + RHCP）。圓極化在衛星通訊中很重要，因為接收天線無需對準極化方向。GPS 信號使用右旋圓極化(RHCP)。

### 題目 43：損耗介質中的電磁波傳播
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：電磁波 — 損耗介質(Lossy Dielectric)

**題目**：

一電磁波在損耗介質（ε_r = 4, μ_r = 1, σ = 0.01 S/m）中傳播，頻率 f = 1 GHz。
(a) 計算損耗正切 tanδ = σ/(ωε)。
(b) 計算傳播常數 γ = α + jβ。
(c) 求趨膚深度 δ 和波長 λ。
(d) 判斷此介質是低損耗介質、高損耗介質還是良導體。

**詳解**：

**解題思路**：用完整的複數傳播常數公式計算，再根據 tanδ 的大小判別損耗類型。

**步驟**：

**(a) 損耗正切**

$$\tan\delta = \frac{\sigma}{\omega\varepsilon} = \frac{\sigma}{2\pi f\varepsilon_0\varepsilon_r}$$

$$= \frac{0.01}{2\pi\times 10^9\times 8.854\times 10^{-12}\times 4} = \frac{0.01}{2.224\times 10^{-1}} = 0.04496$$

tanδ ≈ 0.045，遠小於 1 → **低損耗介質(Low-Loss Dielectric)**。

**(b) 傳播常數**

一般公式：

$$\gamma = j\omega\sqrt{\mu\varepsilon}\sqrt{1-j\frac{\sigma}{\omega\varepsilon}} = j\omega\sqrt{\mu\varepsilon}\sqrt{1-j\tan\delta}$$

低損耗近似（tanδ << 1）：

$$\alpha \approx \frac{\sigma}{2}\sqrt{\frac{\mu}{\varepsilon}} = \frac{\sigma\eta}{2}$$

$$\beta \approx \omega\sqrt{\mu\varepsilon}\left(1+\frac{1}{8}\tan^2\delta\right) \approx \omega\sqrt{\mu\varepsilon}$$

計算：

$$\eta = \sqrt{\frac{\mu_0}{\varepsilon_0\varepsilon_r}} = \frac{377}{2} = 188.5 \text{ Ω}$$

$$\alpha = \frac{0.01\times 188.5}{2} = 0.9425 \text{ Np/m}$$

$$\beta = \frac{2\pi\times 10^9}{3\times 10^8}\times\sqrt{4} = \frac{2\pi\times 10^9\times 2}{3\times 10^8} = 41.89 \text{ rad/m}$$

精確計算驗證：

$$\gamma = j\omega\sqrt{\mu_0\varepsilon_0\varepsilon_r}\sqrt{1-j\cdot 0.04496}$$

$$|1-j\cdot 0.04496| = \sqrt{1+0.002021} = 1.001$$

角度 = −arctan(0.04496)/2 ≈ −0.02248 rad

$$\gamma \approx j\times 41.89\times 1.001\times e^{-j0.02248}$$

$$= j\times 41.89\times(0.9997-j\times 0.02248)$$

$$= j\times 41.89\times 0.9997 + 41.89\times 0.02248$$

$$= 0.9415 + j\times 41.88$$

α ≈ 0.94 Np/m, β ≈ 41.9 rad/m ✓（與近似結果一致）

**(c) 趨膚深度和波長**

$$\delta = \frac{1}{\alpha} = \frac{1}{0.9425} = 1.061 \text{ m}$$

$$\lambda = \frac{2\pi}{\beta} = \frac{2\pi}{41.89} = 0.15 \text{ m} = 15 \text{ cm}$$

**(d) 判別**

tanδ = 0.045 << 1 → **低損耗介質**

- 低損耗介質：tanδ << 1（通常 < 0.1）
- 高損耗/良導體：tanδ >> 1（通常 > 10）
- 中等損耗：tanδ ≈ 1

**答案**：

> (a) tanδ ≈ 0.045
>
> (b) α ≈ 0.94 Np/m, β ≈ 41.9 rad/m
>
> (c) δ ≈ 1.06 m, λ = 15 cm
>
> (d) 低損耗介質

**延伸思考**：在微波工程中，常用介質的 tanδ 值：鐵氟龍(Teflon, PTFE) tanδ ≈ 0.0004，FR-4 基板 tanδ ≈ 0.02，水 tanδ ≈ 0.05 (1 GHz)。微波爐利用水的介電損耗來加熱食物。

### 題目 44：垂直入射 — 反射與透射
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：電磁波 — 垂直入射(Normal Incidence)

**題目**：

一均勻平面波從自由空間（η₁ = η₀ = 377 Ω）垂直入射到半無限大介質（ε_r = 9, μ_r = 1, σ = 0，η₂ = η₀/3）。
(a) 求反射係數 Γ 和透射係數 τ。
(b) 求 VSWR（電壓駐波比, Voltage Standing Wave Ratio）。
(c) 求反射功率和透射功率佔入射功率的比例。
(d) 寫出入射區域和透射區域的電場表達式。

**詳解**：

**解題思路**：利用垂直入射公式，反射係數由兩介質波阻抗決定。

**步驟**：

**(a) 反射係數與透射係數**

$$\Gamma = \frac{\eta_2-\eta_1}{\eta_2+\eta_1} = \frac{\eta_0/3-\eta_0}{\eta_0/3+\eta_0} = \frac{1/3-1}{1/3+1} = \frac{-2/3}{4/3} = -\frac{1}{2}$$

$$\tau = 1+\Gamma = 1-\frac{1}{2} = \frac{1}{2}$$

或 τ = 2η₂/(η₁+η₂) = 2(η₀/3)/(η₀+η₀/3) = (2/3)/(4/3) = 1/2 ✓

Γ = −0.5（負號表示反射波電場與入射波反相）。

**(b) VSWR**

$$\text{VSWR} = \frac{1+|\Gamma|}{1-|\Gamma|} = \frac{1+0.5}{1-0.5} = \frac{1.5}{0.5} = 3$$

**(c) 功率比例**

反射功率比（反射率, Reflectance）：

$$R = |\Gamma|^2 = 0.25 = 25\%$$

透射功率比（透射率, Transmittance）：

$$T = 1-R = 0.75 = 75\%$$

驗證：T 也可用 τ 計算

$$T = \frac{\eta_1}{\eta_2}|\tau|^2 = \frac{\eta_0}{\eta_0/3}\times\frac{1}{4} = 3\times\frac{1}{4} = \frac{3}{4} = 75\% \quad \checkmark$$

注意：T ≠ |τ|²，因為兩介質的波阻抗不同，需要乘以 η₁/η₂ 的修正因子。

**(d) 電場表達式**

設入射波 **E_i** = E₀ e^{-jβ₁z} x̂

β₁ = ω/c（自由空間），β₂ = 3ω/c（介質中）

入射區域（z < 0）：

$$\tilde{\mathbf{E}} = E_0(e^{-j\beta_1 z} - \frac{1}{2}e^{+j\beta_1 z})\hat{x}$$

透射區域（z > 0）：

$$\tilde{\mathbf{E}} = \frac{E_0}{2}e^{-j\beta_2 z}\hat{x}$$

**答案**：

> (a) Γ = −1/2, τ = 1/2
>
> (b) VSWR = 3
>
> (c) 反射 25%, 透射 75%
>
> (d) 入射區：E₀(e^{-jβ₁z} − ½e^{jβ₁z})x̂；透射區：(E₀/2)e^{-jβ₂z}x̂

**延伸思考**：若要消除反射（Γ = 0），需要 η₁ = η₂，即兩介質阻抗匹配(Impedance Matching)。可在界面插入四分之一波長匹配層(Quarter-Wave Matching Layer)，這與傳輸線中的阻抗匹配概念完全對應。

### 題目 45：斜入射 — Snell 定律與全反射
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：電磁波 — 斜入射(Oblique Incidence)

**題目**：

一均勻平面波從介質 1（n₁ = 1.5，玻璃）斜入射到介質 2（n₂ = 1.0，空氣），入射角 θ_i。
(a) 由 Snell 定律求折射角 θ_t。
(b) 求全反射(Total Internal Reflection)的臨界角 θ_c。
(c) 若 θ_i = 60°，求反射和折射的情形。

**詳解**：

**解題思路**：Snell 定律 n₁sinθ_i = n₂sinθ_t，當 n₁ > n₂ 時存在全反射臨界角。

**步驟**：

**(a) Snell 定律**

$$n_1\sin\theta_i = n_2\sin\theta_t$$

$$1.5\sin\theta_i = 1.0\sin\theta_t$$

$$\sin\theta_t = 1.5\sin\theta_i$$

**(b) 全反射臨界角**

全反射發生在 sinθ_t = 1（折射角 θ_t = 90°）時：

$$\sin\theta_c = \frac{n_2}{n_1} = \frac{1.0}{1.5} = \frac{2}{3}$$

$$\theta_c = \arcsin\frac{2}{3} = 41.81°$$

當 θ_i > 41.81° 時發生全反射。

**(c) θ_i = 60° 的情形**

$$\sin\theta_t = 1.5\sin 60° = 1.5\times\frac{\sqrt{3}}{2} = 1.299 > 1$$

sinθ_t > 1，不存在實數折射角 → **全反射**！

此時 |Γ| = 1（全部能量被反射），但反射波有相位偏移(Phase Shift)。

折射角為複數：cosθ_t = √(1−sin²θ_t) = √(1−1.687) = j√0.687 = j0.829

透射波：e^{-jβ₂(x sinθ_t + z cosθ_t)} = e^{-jβ₂·1.299x} · e^{+β₂·0.829z}（z < 0 方向）

z 方向為指數衰減（衰逝波, Evanescent Wave），能量不傳入介質 2。

$$\delta_{evanescent} = \frac{1}{\beta_2\cdot 0.829} = \frac{\lambda_2}{2\pi\times 0.829}$$

衰逝波的穿透深度約為波長的量級。

**答案**：

> (a) sinθ_t = 1.5 sinθ_i
>
> (b) θ_c = arcsin(2/3) ≈ 41.81°
>
> (c) θ_i = 60° > θ_c，發生全反射。透射波為衰逝波(Evanescent Wave)。

**延伸思考**：全反射是光纖(Optical Fiber)導波的基本原理。光纖纖芯的折射率 n₁ 略大於包層 n₂，使光線在纖芯內以大於臨界角的角度全反射傳播。全反射受挫(Frustrated Total Reflection)現象（兩稜鏡靠近時衰逝波耦合）等效於量子力學中的穿隧效應(Tunneling Effect)。

### 題目 46：Brewster 角與 TE/TM 極化
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：電磁波 — Brewster 角(Brewster's Angle)

**題目**：

一電磁波從空氣(n₁ = 1)斜入射到玻璃(n₂ = 1.5)。
(a) 推導 TM 波（平行極化, P-polarization）的反射係數公式(Fresnel 方程)。
(b) 求 Brewster 角 θ_B（TM 波反射係數為零的入射角）。
(c) 在 Brewster 角入射時，TE 波（垂直極化, S-polarization）的反射係數為多少？

**詳解**：

**解題思路**：套用 Fresnel 方程，令 Γ_TM = 0 求 Brewster 角。

**步驟**：

**(a) TM 波反射係數 (Fresnel 方程)**

TM 波（**E** 在入射面內）：

$$\Gamma_{TM} = \frac{\eta_2\cos\theta_t - \eta_1\cos\theta_i}{\eta_2\cos\theta_t + \eta_1\cos\theta_i}$$

對非磁性介質（μ₁ = μ₂ = μ₀）：

η₁ = η₀/n₁，η₂ = η₀/n₂

$$\Gamma_{TM} = \frac{n_1\cos\theta_t - n_2\cos\theta_i}{n_1\cos\theta_t + n_2\cos\theta_i}$$

（分子分母同乘 n₁n₂/η₀）

或等價地用折射率表示：

$$\Gamma_{TM} = \frac{n_2\cos\theta_i - n_1\cos\theta_t}{n_2\cos\theta_i + n_1\cos\theta_t}$$

（注意：不同教科書可能因 E 的正方向定義不同而有正負號差異）

**(b) Brewster 角**

令 Γ_TM = 0：

$$n_2\cos\theta_i = n_1\cos\theta_t$$

配合 Snell 定律 n₁sinθ_i = n₂sinθ_t：

$$n_2\cos\theta_B = n_1\cos\theta_t \quad \cdots (1)$$

$$n_1\sin\theta_B = n_2\sin\theta_t \quad \cdots (2)$$

由 (1)：cosθ_t = (n₂/n₁)cosθ_B

由 (2)：sinθ_t = (n₁/n₂)sinθ_B

利用 sin²θ_t + cos²θ_t = 1：

$$\frac{n_1^2}{n_2^2}\sin^2\theta_B + \frac{n_2^2}{n_1^2}\cos^2\theta_B = 1$$

$$n_1^4\sin^2\theta_B + n_2^4\cos^2\theta_B = n_1^2 n_2^2$$

$$n_1^4\sin^2\theta_B + n_2^4(1-\sin^2\theta_B) = n_1^2 n_2^2$$

$$(n_1^4-n_2^4)\sin^2\theta_B = n_1^2 n_2^2-n_2^4 = n_2^2(n_1^2-n_2^2)$$

$$\sin^2\theta_B = \frac{n_2^2(n_1^2-n_2^2)}{(n_1^2-n_2^2)(n_1^2+n_2^2)} = \frac{n_2^2}{n_1^2+n_2^2}$$

$$\tan\theta_B = \frac{\sin\theta_B}{\cos\theta_B} = \frac{n_2}{n_1}$$

$$\theta_B = \arctan\frac{n_2}{n_1} = \arctan\frac{1.5}{1} = 56.31°$$

**(c) Brewster 角下 TE 波的反射**

TE 波（**E** 垂直於入射面）的反射係數：

$$\Gamma_{TE} = \frac{\eta_2\cos\theta_i - \eta_1\cos\theta_t}{\eta_2\cos\theta_i + \eta_1\cos\theta_t} = \frac{n_1\cos\theta_i - n_2\cos\theta_t}{n_1\cos\theta_i + n_2\cos\theta_t}$$

在 θ_i = θ_B = 56.31° 時：

由 Snell 定律：sinθ_t = (1/1.5)sin56.31° = 0.5547，θ_t = 33.69°

$$\Gamma_{TE} = \frac{1\times\cos 56.31° - 1.5\times\cos 33.69°}{1\times\cos 56.31° + 1.5\times\cos 33.69°}$$

$$= \frac{0.5547 - 1.5\times 0.8321}{0.5547 + 1.5\times 0.8321} = \frac{0.5547-1.2481}{0.5547+1.2481} = \frac{-0.6934}{1.8029} = -0.3846$$

|Γ_TE|² = 0.148，即 TE 波仍有約 14.8% 的反射功率。

**答案**：

> (a) Γ_TM = (n₂cosθ_i − n₁cosθ_t)/(n₂cosθ_i + n₁cosθ_t)
>
> (b) θ_B = arctan(n₂/n₁) = arctan(1.5) = 56.31°
>
> (c) |Γ_TE| = 0.385，TE 波反射率約 14.8%

**延伸思考**：在 Brewster 角下，TM 波完全透射而 TE 波部分反射，故反射光為 TE 極化（偏振）。這就是偏光太陽眼鏡的工作原理：濾除水面或路面的眩光（主要是 TE 極化的反射光）。雷射腔(Laser Cavity)的布儒斯特窗(Brewster Window)也利用此原理來實現無損耗的特定極化輸出。

### 題目 47：波印亭向量與輻射壓力
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電磁波 — 波印亭向量(Poynting Vector)與功率

**題目**：

一雷射束(Laser Beam)功率 P = 1 W，光束截面積 A = 1 mm²。
(a) 求光束的平均功率密度(Intensity) I。
(b) 求光束中電場和磁場的振幅 E₀、H₀。
(c) 求光束打在完全反射鏡上的輻射壓力(Radiation Pressure)。

**詳解**：

**解題思路**：從功率密度反推場強，利用動量守恆求輻射壓力。

**步驟**：

**(a) 功率密度**

$$I = \frac{P}{A} = \frac{1}{10^{-6}} = 10^6 \text{ W/m}^2 = 1 \text{ MW/m}^2$$

**(b) 場振幅**

$$I = S_{av} = \frac{E_0^2}{2\eta_0}$$

$$E_0 = \sqrt{2\eta_0 I} = \sqrt{2\times 377\times 10^6} = \sqrt{7.54\times 10^8} = 2.746\times 10^4 \text{ V/m} \approx 27.5 \text{ kV/m}$$

$$H_0 = \frac{E_0}{\eta_0} = \frac{27460}{377} = 72.8 \text{ A/m}$$

**(c) 輻射壓力**

電磁波的動量密度(Momentum Density)：

$$g = \frac{S}{c^2} = \frac{I}{c^2}$$

完全反射時，動量改變為原來的 2 倍（反彈）：

$$P_{rad} = \frac{2I}{c} = \frac{2\times 10^6}{3\times 10^8} = 6.67\times 10^{-3} \text{ Pa} \approx 6.67 \text{ mPa}$$

完全吸收時為 I/c = 3.33 mPa。

**答案**：

> (a) I = 1 MW/m²
>
> (b) E₀ ≈ 27.5 kV/m, H₀ ≈ 72.8 A/m
>
> (c) P_rad = 2I/c ≈ 6.67 mPa（完全反射）

**延伸思考**：輻射壓力雖小，但在太空中可用於太陽帆(Solar Sail)推進。在地球軌道處太陽光的功率密度約 1370 W/m²，輻射壓力約 9.1 μPa。強聚焦雷射的輻射壓力可用於光鑷(Optical Tweezers)操控微小粒子，此技術的發明者 Arthur Ashkin 於 2018 年獲得諾貝爾物理獎。

### 題目 48：多層介質的垂直入射
**來源**：台大電機 經典題型
**難度**：★★★★★
**主題**：電磁波 — 四分之一波長抗反射膜

**題目**：

為消除光從空氣(n₁ = 1)入射到玻璃(n₃ = 1.5)的反射，在中間加一層厚度 d、折射率 n₂ 的介電膜。
(a) 用傳輸線類比法，求此結構的等效輸入阻抗。
(b) 求使反射為零的條件（n₂ 和 d 的值）。
(c) 若入射光波長 λ₀ = 550 nm（綠光），求膜的實際厚度。

**詳解**：

**解題思路**：多層介質的垂直入射等價於串聯傳輸線問題，可用傳輸線阻抗轉換公式。

**步驟**：

**(a) 等效輸入阻抗**

將介質膜視為特性阻抗 η₂、長度 d 的傳輸線段，負載為 η₃（玻璃）：

$$Z_{in} = \eta_2\frac{\eta_3 + j\eta_2\tan(\beta_2 d)}{\eta_2 + j\eta_3\tan(\beta_2 d)}$$

其中 β₂ = 2πn₂/λ₀。

**(b) 反射為零的條件**

要使反射為零：Z_in = η₁（輸入阻抗等於入射介質的波阻抗）。

取 d = λ₂/4（四分之一波長），即 β₂d = π/2，tanβ₂d → ∞：

$$Z_{in} = \eta_2\cdot\frac{j\eta_2}{j\eta_3} = \frac{\eta_2^2}{\eta_3}$$

令 Z_in = η₁：

$$\frac{\eta_2^2}{\eta_3} = \eta_1$$

$$\eta_2 = \sqrt{\eta_1\eta_3}$$

由 η = η₀/n：

$$\frac{\eta_0}{n_2} = \sqrt{\frac{\eta_0}{n_1}\cdot\frac{\eta_0}{n_3}}$$

$$n_2 = \sqrt{n_1 n_3} = \sqrt{1\times 1.5} = 1.225$$

（氟化鎂 MgF₂ 的折射率 n ≈ 1.38 常用於光學鍍膜，但不完全匹配 1.225）

**(c) 膜厚度**

$$d = \frac{\lambda_2}{4} = \frac{\lambda_0}{4n_2} = \frac{550}{4\times 1.225} = 112.2 \text{ nm}$$

**驗證**

若 n₂ = √1.5 = 1.225：

Γ = (Z_in − η₁)/(Z_in + η₁) = (η₁ − η₁)/(η₁ + η₁) = 0 ✓

完全消除反射。

**答案**：

> (a) Z_in = η₂(η₃ + jη₂tanβ₂d)/(η₂ + jη₃tanβ₂d)
>
> (b) n₂ = √(n₁n₃) = √1.5 ≈ 1.225, d = λ₀/(4n₂)
>
> (c) d ≈ 112.2 nm

**延伸思考**：這就是抗反射鍍膜(Anti-Reflection Coating, AR Coating)的原理，廣泛應用於相機鏡頭、眼鏡片、太陽能電池等。單層鍍膜只能在一個波長下完全消除反射，多層鍍膜可在更寬的波段內降低反射。此概念與傳輸線中的四分之一波長匹配(Quarter-Wave Transformer)完全類比。

### 題目 49：損耗介質中的波阻抗與複數傳播常數
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：電磁波 — 損耗介質中的複數波阻抗

**題目**：

海水的電磁參數為 ε_r = 81, μ_r = 1, σ = 4 S/m。一潛艇通訊系統使用 f = 100 Hz 的 ELF（極低頻, Extremely Low Frequency）信號。
(a) 計算 σ/(ωε) 並判斷海水在此頻率下的行為。
(b) 求衰減常數 α、趨膚深度 δ。
(c) 在海面下 10 m 處，信號衰減多少 dB？
(d) 解釋為何潛艇通訊使用極低頻。

**詳解**：

**解題思路**：在超低頻下海水為良導體，用良導體近似公式。

**步驟**：

**(a) 判別**

$$\frac{\sigma}{\omega\varepsilon} = \frac{4}{2\pi\times 100\times 81\times 8.854\times 10^{-12}} = \frac{4}{4.506\times 10^{-7}} = 8.88\times 10^6$$

σ/(ωε) >> 1 → 海水在 100 Hz 下為**良導體**。

**(b) 衰減常數與趨膚深度**

良導體近似：

$$\alpha = \sqrt{\pi f\mu_0\sigma} = \sqrt{\pi\times 100\times 4\pi\times 10^{-7}\times 4}$$

$$= \sqrt{100\times 16\pi^2\times 10^{-7}} = \sqrt{1.579\times 10^{-3}}$$

$$= 0.03974 \text{ Np/m}$$

$$\delta = \frac{1}{\alpha} = 25.16 \text{ m}$$

**(c) 10 m 深度的衰減**

$$\frac{E(10)}{E(0)} = e^{-\alpha\times 10} = e^{-0.3974} = 0.672$$

dB 衰減：20 log₁₀(0.672) = −3.45 dB

10 m 深處信號衰減約 3.45 dB（約 67.2% 的電場振幅保留）。

**(d) 為何使用極低頻**

趨膚深度 δ = 1/√(πfμ₀σ) ∝ 1/√f

- f = 100 Hz：δ ≈ 25 m
- f = 1 kHz：δ ≈ 8 m
- f = 1 MHz：δ ≈ 0.25 m
- f = 1 GHz：δ ≈ 0.008 m

頻率越低穿透越深。100 Hz 的信號在 25 m 深處才衰減到 e⁻¹ ≈ 37%，因此潛艇在淺潛深度（< 20 m）即可接收。若使用較高頻率，信號在幾公分深就被完全衰減。

代價：極低頻的頻寬極窄，數據傳輸速率極低（約每分鐘幾個字元），只能傳送簡短的指令代碼。

**答案**：

> (a) σ/(ωε) ≈ 8.88×10⁶ >> 1，良導體
>
> (b) α ≈ 0.0397 Np/m, δ ≈ 25.2 m
>
> (c) 10 m 深度衰減約 3.45 dB
>
> (d) 低頻穿透深，潛艇可在水下接收；代價是頻寬極窄

**延伸思考**：美國海軍曾運營的 ELF 通訊系統使用 76 Hz 頻率，天線長達數十公里。Sanguine/Seafarer 計畫的天線建在威斯康辛州和密西根州。俄羅斯的 ZEVS 系統使用 82 Hz。這些系統現已退役，被更先進的通訊方式取代。

### 題目 50：電磁波的能量與動量
**來源**：台大電機 經典題型
**難度**：★★★
**主題**：電磁波 — 能量密度與動量

**題目**：

在自由空間中，一均勻平面波的電場瞬時值為 **E**(z,t) = E₀cos(ωt−βz) x̂。
(a) 計算瞬時波印亭向量 **S**(z,t)。
(b) 求時間平均波印亭向量 **S**_av。
(c) 證明電場能量密度和磁場能量密度的時間平均值相等。
(d) 求總電磁能量密度的時間平均值 ⟨w⟩ 與 S_av 的關係。

**詳解**：

**解題思路**：直接計算 **S** = **E** × **H**，再取時間平均。

**步驟**：

**(a) 瞬時波印亭向量**

**H** = (E₀/η₀)cos(ωt−βz) ŷ

$$\mathbf{S} = \mathbf{E}\times\mathbf{H} = E_0\cos(\omega t-\beta z)\hat{x}\times\frac{E_0}{\eta_0}\cos(\omega t-\beta z)\hat{y}$$

$$= \frac{E_0^2}{\eta_0}\cos^2(\omega t-\beta z)\hat{z}$$

**(b) 時間平均**

$$\mathbf{S}_{av} = \langle\mathbf{S}\rangle = \frac{E_0^2}{\eta_0}\langle\cos^2(\omega t-\beta z)\rangle\hat{z} = \frac{E_0^2}{2\eta_0}\hat{z}$$

（因 ⟨cos²⟩ = 1/2）

**(c) 電場能量密度 = 磁場能量密度**

電場能量密度：

$$w_e = \frac{1}{2}\varepsilon_0 E^2 = \frac{1}{2}\varepsilon_0 E_0^2\cos^2(\omega t-\beta z)$$

$$\langle w_e\rangle = \frac{1}{4}\varepsilon_0 E_0^2$$

磁場能量密度：

$$w_m = \frac{1}{2}\mu_0 H^2 = \frac{1}{2}\mu_0\frac{E_0^2}{\eta_0^2}\cos^2(\omega t-\beta z)$$

$$= \frac{1}{2}\frac{\mu_0 E_0^2}{\mu_0/\varepsilon_0}\cos^2(\omega t-\beta z) = \frac{1}{2}\varepsilon_0 E_0^2\cos^2(\omega t-\beta z)$$

$$\langle w_m\rangle = \frac{1}{4}\varepsilon_0 E_0^2$$

因此 ⟨w_e⟩ = ⟨w_m⟩ ✓ 電磁波中電場和磁場各攜帶一半能量。

**(d) 能量密度與功率密度的關係**

$$\langle w\rangle = \langle w_e\rangle+\langle w_m\rangle = \frac{1}{2}\varepsilon_0 E_0^2$$

$$S_{av} = \frac{E_0^2}{2\eta_0} = \frac{E_0^2}{2}\sqrt{\frac{\varepsilon_0}{\mu_0}} = \frac{1}{2}\varepsilon_0 E_0^2\cdot\frac{1}{\sqrt{\mu_0\varepsilon_0}} = \langle w\rangle\cdot c$$

$$\boxed{S_{av} = \langle w\rangle\cdot c}$$

功率密度 = 能量密度 × 傳播速度，符合直覺。

**答案**：

> (a) **S** = (E₀²/η₀)cos²(ωt−βz) ẑ
>
> (b) **S**_av = E₀²/(2η₀) ẑ
>
> (c) ⟨w_e⟩ = ⟨w_m⟩ = ε₀E₀²/4
>
> (d) S_av = ⟨w⟩ · c

**延伸思考**：S_av = ⟨w⟩c 的關係也可推廣到介質中（v_p 取代 c）。在色散介質中，能量傳播速度為群速度 v_g = dω/dβ，而非相速度 v_p = ω/β。

### 題目 51：傳輸線基本方程 — 特性阻抗與傳播常數
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★
**主題**：傳輸線 — 特性阻抗(Characteristic Impedance)

**題目**：

一無損(Lossless)同軸傳輸線，分布參數為：L' = 250 nH/m（單位長度電感），C' = 100 pF/m（單位長度電容）。
(a) 從傳輸線方程(Telegrapher's Equations)推導特性阻抗 Z₀ 和相速度 v_p 的公式。
(b) 計算此傳輸線的 Z₀ 和 v_p。
(c) 若傳輸線長度 l = 3 m，工作頻率 f = 100 MHz，求電長度(Electrical Length) βl。

**詳解**：

**解題思路**：從分布參數模型推導傳輸線方程，求解得到特性阻抗和傳播速度。

**步驟**：

**(a) 推導**

傳輸線的電報方程(Telegrapher's Equations)：

$$-\frac{\partial v(z,t)}{\partial z} = L'\frac{\partial i(z,t)}{\partial t} + R'i(z,t)$$

$$-\frac{\partial i(z,t)}{\partial z} = C'\frac{\partial v(z,t)}{\partial t} + G'v(z,t)$$

無損情況 R' = 0, G' = 0，取時諧場（e^{jωt}）：

$$-\frac{d\tilde{V}}{dz} = j\omega L'\tilde{I}$$

$$-\frac{d\tilde{I}}{dz} = j\omega C'\tilde{V}$$

消去 Ĩ 得波動方程：

$$\frac{d^2\tilde{V}}{dz^2} = -\omega^2 L'C'\tilde{V} = -\beta^2\tilde{V}$$

其中 β = ω√(L'C')。

通解：Ṽ(z) = V₊e^{-jβz} + V₋e^{+jβz}

電流通解：Ĩ(z) = (V₊e^{-jβz} − V₋e^{+jβz})/Z₀

其中特性阻抗：

$$Z_0 = \sqrt{\frac{L'}{C'}}$$

相速度：

$$v_p = \frac{\omega}{\beta} = \frac{1}{\sqrt{L'C'}}$$

對有損傳輸線，一般公式為：

$$Z_0 = \sqrt{\frac{R'+j\omega L'}{G'+j\omega C'}}, \quad \gamma = \sqrt{(R'+j\omega L')(G'+j\omega C')}$$

**(b) 數值計算**

$$Z_0 = \sqrt{\frac{250\times 10^{-9}}{100\times 10^{-12}}} = \sqrt{\frac{250}{100}\times 10^3} = \sqrt{2500} = 50 \text{ Ω}$$

$$v_p = \frac{1}{\sqrt{250\times 10^{-9}\times 100\times 10^{-12}}} = \frac{1}{\sqrt{2.5\times 10^{-17}}} = \frac{1}{5\times 10^{-9}} = 2\times 10^8 \text{ m/s}$$

v_p = 2×10⁸ m/s = 2c/3，即光速的 2/3（典型的同軸電纜值）。

**(c) 電長度**

$$\beta = \frac{\omega}{v_p} = \frac{2\pi\times 10^8}{2\times 10^8} = \pi \text{ rad/m}$$

$$\beta l = \pi\times 3 = 3\pi = 9.42 \text{ rad} = 540°$$

電長度 540° = 3 × 180°，相當於 3 個半波長。

**答案**：

> (a) Z₀ = √(L'/C'), v_p = 1/√(L'C')
>
> (b) Z₀ = 50 Ω, v_p = 2×10⁸ m/s (= 2c/3)
>
> (c) βl = 3π rad = 540°

**延伸思考**：50 Ω 是射頻(RF)系統最常用的特性阻抗，因為它大致是最小衰減（約 77 Ω）和最大功率容量（約 30 Ω）的折中值。75 Ω 常用於電視電纜系統。特性阻抗是傳輸線的固有屬性，與長度和頻率無關。

### 題目 52：反射係數與輸入阻抗
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★★
**主題**：傳輸線 — 反射係數(Reflection Coefficient)與輸入阻抗(Input Impedance)

**題目**：

一特性阻抗 Z₀ = 50 Ω 的無損傳輸線，長度 l = λ/8，終端接負載 Z_L = (100 + j75) Ω。
(a) 求負載反射係數 Γ_L（大小和相位）。
(b) 求輸入端反射係數 Γ_in。
(c) 求輸入阻抗 Z_in。
(d) 求 VSWR。

**詳解**：

**解題思路**：系統性地使用反射係數和輸入阻抗公式。這是傳輸線問題的核心計算流程。

**步驟**：

**(a) 負載反射係數**

$$\Gamma_L = \frac{Z_L-Z_0}{Z_L+Z_0} = \frac{(100+j75)-50}{(100+j75)+50} = \frac{50+j75}{150+j75}$$

分子：50 + j75，|分子| = √(2500+5625) = √8125 = 90.14，∠分子 = arctan(75/50) = 56.31°

分母：150 + j75，|分母| = √(22500+5625) = √28125 = 167.7，∠分母 = arctan(75/150) = 26.57°

$$|\Gamma_L| = \frac{90.14}{167.7} = 0.5375$$

$$\angle\Gamma_L = 56.31° - 26.57° = 29.74°$$

$$\Gamma_L = 0.5375\angle 29.74° = 0.5375 e^{j0.519}$$

或直接做複數除法：

$$\Gamma_L = \frac{(50+j75)(150-j75)}{(150+j75)(150-j75)} = \frac{7500-j3750+j11250+5625}{22500+5625}$$

$$= \frac{13125+j7500}{28125} = 0.4667+j0.2667$$

|Γ_L| = √(0.4667²+0.2667²) = √(0.2178+0.0711) = √0.2889 = 0.5375 ✓

∠Γ_L = arctan(0.2667/0.4667) = 29.74° ✓

**(b) 輸入端反射係數**

$$\Gamma_{in} = \Gamma_L e^{-j2\beta l}$$

l = λ/8，βl = 2π/λ × λ/8 = π/4

$$\Gamma_{in} = 0.5375 e^{j29.74°}\cdot e^{-j2\times 45°} = 0.5375 e^{j(29.74°-90°)} = 0.5375 e^{-j60.26°}$$

$$= 0.5375\angle -60.26°$$

= 0.5375(cos60.26° − j sin60.26°) = 0.5375(0.4969 − j0.8679)

= 0.2671 − j0.4665

**(c) 輸入阻抗**

方法一：由 Γ_in 求 Z_in：

$$Z_{in} = Z_0\frac{1+\Gamma_{in}}{1-\Gamma_{in}} = 50\times\frac{1+(0.2671-j0.4665)}{1-(0.2671-j0.4665)}$$

$$= 50\times\frac{1.2671-j0.4665}{0.7329+j0.4665}$$

分子：|1.2671−j0.4665| = √(1.6056+0.2176) = √1.8232 = 1.3503

分母：|0.7329+j0.4665| = √(0.5371+0.2176) = √0.7548 = 0.8688

分子/分母 = 1.3503/0.8688 = 1.5543

分子角度 = arctan(−0.4665/1.2671) = −20.22°

分母角度 = arctan(0.4665/0.7329) = 32.49°

Z_in = 50 × 1.5543 ∠(−20.22°−32.49°) = 77.72 ∠−52.71°

= 77.72(cos52.71° − j sin52.71°) = 77.72(0.6040 − j0.7970)

= (46.94 − j61.94) Ω

方法二：直接用傳輸線阻抗轉換公式：

$$Z_{in} = Z_0\frac{Z_L+jZ_0\tan\beta l}{Z_0+jZ_L\tan\beta l}$$

βl = π/4, tanβl = tan45° = 1

$$Z_{in} = 50\times\frac{(100+j75)+j\times 50\times 1}{50+j(100+j75)\times 1}$$

$$= 50\times\frac{100+j75+j50}{50+j100+j^2 75} = 50\times\frac{100+j125}{50-75+j100}$$

$$= 50\times\frac{100+j125}{-25+j100}$$

做複數除法：

$$\frac{(100+j125)(-25-j100)}{(-25+j100)(-25-j100)} = \frac{-2500-j10000-j3125-j^2 12500}{625+10000}$$

$$= \frac{-2500+12500-j13125}{10625} = \frac{10000-j13125}{10625}$$

$$= 0.9412-j1.2353$$

$$Z_{in} = 50\times(0.9412-j1.2353) = (47.06-j61.76) \text{ Ω}$$

兩方法結果一致（微小差異來自四捨五入）。✓

**(d) VSWR**

$$\text{VSWR} = \frac{1+|\Gamma_L|}{1-|\Gamma_L|} = \frac{1+0.5375}{1-0.5375} = \frac{1.5375}{0.4625} = 3.32$$

**答案**：

> (a) Γ_L = 0.5375∠29.74°
>
> (b) Γ_in = 0.5375∠−60.26°
>
> (c) Z_in ≈ (47 − j62) Ω
>
> (d) VSWR = 3.32

**延伸思考**：注意 |Γ_in| = |Γ_L| = 0.5375（無損傳輸線上反射係數的大小不變，只是相位隨位置旋轉）。VSWR 也沿線不變。輸入阻抗中的虛部為負值（容性），表明此段傳輸線在此頻率下等效為一個電容性負載。

### 題目 53：特殊長度傳輸線
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★
**主題**：傳輸線 — 特殊長度的性質

**題目**：

分析以下特殊長度無損傳輸線（特性阻抗 Z₀）的輸入阻抗：
(a) l = λ/4（四分之一波長）：Z_in = ?
(b) l = λ/2（半波長）：Z_in = ?
(c) l = λ/4 且短路終端（Z_L = 0）：Z_in = ?
(d) l = λ/4 且開路終端（Z_L = ∞）：Z_in = ?
(e) 短路截線（Z_L = 0），長度 l < λ/4：Z_in 的性質。

**詳解**：

**解題思路**：將特殊值代入傳輸線阻抗轉換公式，利用 tan 函數的特殊值。

**步驟**：

傳輸線阻抗轉換公式：

$$Z_{in} = Z_0\frac{Z_L+jZ_0\tan\beta l}{Z_0+jZ_L\tan\beta l}$$

**(a) l = λ/4，βl = π/2，tan(π/2) → ∞**

$$Z_{in} = Z_0\frac{Z_L+jZ_0\cdot\infty}{Z_0+jZ_L\cdot\infty} = Z_0\cdot\frac{jZ_0}{jZ_L} = \frac{Z_0^2}{Z_L}$$

**四分之一波長轉換器(Quarter-Wave Transformer)：Z_in = Z₀²/Z_L**

這是阻抗反轉器！高阻抗變低阻抗，低阻抗變高阻抗。

**(b) l = λ/2，βl = π，tan(π) = 0**

$$Z_{in} = Z_0\frac{Z_L+0}{Z_0+0} = Z_L$$

**半波長傳輸線：Z_in = Z_L**

半波長傳輸線完全透明，輸入阻抗等於負載阻抗。

**(c) l = λ/4，Z_L = 0（短路）**

$$Z_{in} = \frac{Z_0^2}{0} = \infty \quad \text{（開路！）}$$

四分之一波長短路線等效為開路。

**(d) l = λ/4，Z_L = ∞（開路）**

$$Z_{in} = \frac{Z_0^2}{\infty} = 0 \quad \text{（短路！）}$$

四分之一波長開路線等效為短路。

**(e) 短路截線(Short-Circuited Stub)，Z_L = 0**

$$Z_{in} = Z_0\frac{0+jZ_0\tan\beta l}{Z_0+0} = jZ_0\tan\beta l$$

- 0 < l < λ/4：tanβl > 0 → Z_in = jX_L（純感性，等效電感）
- λ/4 < l < λ/2：tanβl < 0 → Z_in = −jX_C（純容性，等效電容）
- l = λ/4：Z_in → ∞（等效開路，如 (c) 所述）

同理，開路截線(Open-Circuited Stub)，Z_L = ∞：

$$Z_{in} = -jZ_0\cot\beta l = \frac{Z_0}{j\tan\beta l}$$

- 0 < l < λ/4：Z_in = −jX_C（純容性）
- λ/4 < l < λ/2：Z_in = jX_L（純感性）

**答案**：

> (a) λ/4 轉換器：Z_in = Z₀²/Z_L（阻抗反轉）
>
> (b) λ/2 線：Z_in = Z_L（透明）
>
> (c) λ/4 短路 = 開路；(d) λ/4 開路 = 短路
>
> (e) 短路截線：Z_in = jZ₀tan(βl)（純電抗，可調長度模擬電感或電容）

**延伸思考**：四分之一波長轉換器是微波電路中最基本的阻抗匹配元件。短路/開路截線是微波濾波器和匹配網路的基本構建塊。在印刷電路板(PCB)上，開路截線比短路截線更容易實作（不需要通孔接地）。

### 題目 54：四分之一波長阻抗匹配器設計
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★★
**主題**：傳輸線 — 四分之一波長轉換器(Quarter-Wave Transformer)

**題目**：

一 50 Ω 傳輸線需要匹配 200 Ω 的純電阻負載（f = 2 GHz）。
(a) 設計四分之一波長轉換器：求所需特性阻抗 Z_T。
(b) 若傳輸線介質 ε_r = 2.2（微帶線基板），求轉換器的物理長度。
(c) 計算此匹配器在設計頻率下的 VSWR 和反射損耗(Return Loss)。
(d) 討論頻寬限制。

**詳解**：

**解題思路**：四分之一波長轉換器是最經典的阻抗匹配方法，設計直觀但有頻寬限制。

**步驟**：

**(a) 轉換器特性阻抗**

要求：Z_in = Z₀ = 50 Ω，負載 Z_L = 200 Ω

$$Z_{in} = \frac{Z_T^2}{Z_L}$$

$$50 = \frac{Z_T^2}{200}$$

$$Z_T = \sqrt{50\times 200} = \sqrt{10000} = 100 \text{ Ω}$$

**(b) 物理長度**

在介質中：

$$v_p = \frac{c}{\sqrt{\varepsilon_r}} = \frac{3\times 10^8}{\sqrt{2.2}} = 2.024\times 10^8 \text{ m/s}$$

$$\lambda = \frac{v_p}{f} = \frac{2.024\times 10^8}{2\times 10^9} = 0.1012 \text{ m} = 101.2 \text{ mm}$$

$$l = \frac{\lambda}{4} = 25.3 \text{ mm}$$

**(c) VSWR 和反射損耗**

在設計頻率（f₀ = 2 GHz）下，βl = π/2 恰好是四分之一波長：

Z_in = Z_T²/Z_L = 100²/200 = 50 Ω = Z₀

Γ = (Z_in − Z₀)/(Z_in + Z₀) = 0

**VSWR = 1（完美匹配！）**

**反射損耗(Return Loss) = −20 log|Γ| = ∞ dB（完美匹配）**

**(d) 頻寬分析**

偏離設計頻率時，βl ≠ π/2，匹配不再完美。

設 f = f₀ + Δf，則 βl = (π/2)(f/f₀) = (π/2)(1 + Δf/f₀)

$$Z_{in}(f) = Z_T\frac{Z_L+jZ_T\tan\beta l}{Z_T+jZ_L\tan\beta l}$$

在 f ≠ f₀ 時，tanβl ≠ ∞，Z_in ≠ 50 Ω，產生反射。

反射係數：

$$\Gamma = \frac{Z_{in}-Z_0}{Z_{in}+Z_0}$$

經推導，當 |Γ| < Γ_max 時的頻寬（VSWR < S_max 的範圍）：

$$\frac{\Delta f}{f_0} \approx \frac{2}{\pi}\arccos\left(\frac{2\Gamma_{max}\sqrt{Z_0 Z_L}}{|Z_L-Z_0|\sqrt{1-\Gamma_{max}^2}}\right)$$

例如要求 VSWR < 1.5（|Γ| < 0.2）：

$$\frac{\Delta f}{f_0} \approx \frac{2}{\pi}\arccos\left(\frac{2\times 0.2\times\sqrt{10000}}{150\times\sqrt{0.96}}\right) = \frac{2}{\pi}\arccos\left(\frac{40}{147.0}\right)$$

$$= \frac{2}{\pi}\arccos(0.272) = \frac{2}{\pi}\times 74.2° = \frac{2\times 74.2}{180}\times 100\% \approx 82.4\%$$

分數頻寬約 82%，對應頻率範圍約 1.18 GHz 到 2.82 GHz。

（注意：Z_L/Z₀ = 4 的阻抗比不大，所以頻寬相當寬。阻抗比越大頻寬越窄。）

**答案**：

> (a) Z_T = √(Z₀·Z_L) = √(50×200) = 100 Ω
>
> (b) l = λ/(4√ε_r) = 25.3 mm
>
> (c) 設計頻率下 VSWR = 1，Return Loss = ∞ dB
>
> (d) VSWR < 1.5 的頻寬約 82%（寬頻匹配需多段轉換器）

**延伸思考**：若需更寬的頻寬，可用多段四分之一波長轉換器（多段匹配, Multi-Section Matching）。常用的設計方法有 Binomial（最大平坦, Maximally Flat）和 Chebyshev（等波紋, Equal Ripple）轉換器，這是微波工程中的重要設計課題。

### 題目 55：Smith 圖基礎 — 歸一化阻抗與反射係數
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★★
**主題**：傳輸線 — Smith 圖(Smith Chart)基礎

**題目**：

Smith 圖是射頻/微波工程中最重要的圖形工具。回答以下問題：
(a) 歸一化阻抗(Normalized Impedance) z_L = Z_L/Z₀ = 2 + j1 在 Smith 圖上的位置（用 Γ 表示）。
(b) 將此點轉換為歸一化導納 y_L。
(c) 在 Smith 圖上標出以下特殊點：匹配點、短路點、開路點。
(d) 若沿傳輸線向訊號源方向移動 0.15λ，新的歸一化阻抗為何？

**詳解**：

**解題思路**：Smith 圖上的每個點對應一個反射係數（或歸一化阻抗），向訊號源移動等於順時針旋轉。

**步驟**：

**(a) 歸一化阻抗轉反射係數**

z_L = 2 + j1

$$\Gamma_L = \frac{z_L-1}{z_L+1} = \frac{(2+j1)-1}{(2+j1)+1} = \frac{1+j1}{3+j1}$$

$$= \frac{(1+j)(3-j)}{(3+j)(3-j)} = \frac{3-j+j3-j^2}{9+1} = \frac{4+j2}{10} = 0.4+j0.2$$

$$|\Gamma_L| = \sqrt{0.16+0.04} = \sqrt{0.20} = 0.4472$$

$$\angle\Gamma_L = \arctan\frac{0.2}{0.4} = 26.57°$$

在 Smith 圖上，此點位於中心偏右上方，距中心 0.4472（在半徑 1 的圓內），角度 26.57°。

Smith 圖上的讀法：
- 找 r = 2 的等電阻圓（通過 Smith 圖右側 z = 2 + j0 點的圓）
- 找 x = 1 的等電抗圓弧（上半部的圓弧）
- 兩者交點即為 z = 2 + j1

**(b) 歸一化導納**

方法一（計算）：

$$y_L = \frac{1}{z_L} = \frac{1}{2+j1} = \frac{2-j1}{(2+j1)(2-j1)} = \frac{2-j1}{5} = 0.4-j0.2$$

方法二（Smith 圖法）：

在 Smith 圖上，導納是阻抗的對徑點（旋轉 180°），因為 y = 1/z 等價於 Γ_y = −Γ_z。

$$\Gamma_y = -\Gamma_z = -(0.4+j0.2) = -0.4-j0.2$$

$$y_L = \frac{1+\Gamma_y}{1-\Gamma_y} = \frac{1+(-0.4-j0.2)}{1-(-0.4-j0.2)} = \frac{0.6-j0.2}{1.4+j0.2}$$

$$= \frac{(0.6-j0.2)(1.4-j0.2)}{(1.4)^2+(0.2)^2} = \frac{0.84-j0.12-j0.28+j^2 0.04}{2.0}$$

$$= \frac{0.80-j0.40}{2.0} = 0.40-j0.20 \quad \checkmark$$

**(c) 特殊點**

| 點 | 歸一化阻抗 z | Γ | Smith 圖位置 |
|---|---|---|---|
| 匹配(Matched) | z = 1 + j0 | Γ = 0 | 圓心 |
| 短路(Short Circuit) | z = 0 | Γ = −1 (最左) | 最左端 |
| 開路(Open Circuit) | z = ∞ | Γ = +1 (最右) | 最右端 |
| 純電抗 jX | z = jX | \|Γ\| = 1 | 圓周上 |

**(d) 向訊號源移動 0.15λ**

在 Smith 圖上，向訊號源(Toward Generator)移動 d 距離，等於反射係數相位減少 2βd：

$$\Gamma_{new} = \Gamma_L e^{-j2\beta d} = 0.4472 e^{j26.57°}\cdot e^{-j2\times\frac{2\pi}{\lambda}\times 0.15\lambda}$$

$$= 0.4472 e^{j26.57°}\cdot e^{-j\times 0.6\pi} = 0.4472 e^{j26.57°}\cdot e^{-j108°}$$

$$= 0.4472 e^{j(26.57°-108°)} = 0.4472 e^{-j81.43°}$$

= 0.4472(cos81.43° − j sin81.43°) = 0.4472(0.1489 − j0.9889)

= 0.0666 − j0.4422

$$z_{new} = \frac{1+\Gamma_{new}}{1-\Gamma_{new}} = \frac{1.0666-j0.4422}{0.9334+j0.4422}$$

$$= \frac{(1.0666-j0.4422)(0.9334-j0.4422)}{0.9334^2+0.4422^2}$$

分子 = 0.9956 − 0.4717j − 0.4127j + 0.1955j²

= (0.9956 − 0.1955) − j(0.4717+0.4127)

= 0.8001 − j0.8844

分母 = 0.8712 + 0.1955 = 1.0668

$$z_{new} = \frac{0.8001-j0.8844}{1.0668} = 0.750-j0.829$$

或等價地用 Smith 圖讀取（順時針轉 0.15λ = 54°）。

在 Smith 圖的外圈刻度上，原始位置約在 0.214λ 處（向訊號源方向）。移動 0.15λ 後到達 0.214 + 0.15 = 0.364λ 處，讀取對應的歸一化阻抗。

**答案**：

> (a) Γ_L = 0.4472∠26.57°
>
> (b) y_L = 0.4 − j0.2
>
> (c) 匹配=圓心，短路=最左，開路=最右
>
> (d) z_new ≈ 0.75 − j0.83（向訊號源移動 0.15λ）

**延伸思考**：Smith 圖上順時針旋轉（向訊號源方向）一整圈是 λ/2（不是 λ）。這是因為反射係數的相位因子是 e^{-j2βl}，每 λ/2 距離相位變化 2π。Smith 圖的外圈刻度直接以 λ 為單位標示，非常方便。

### 題目 56：VSWR 與功率計算
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★
**主題**：傳輸線 — VSWR 與功率

**題目**：

一 50 Ω 傳輸線上量測到 VSWR = 3，入射功率 P_inc = 100 W。
(a) 求 |Γ|。
(b) 求反射功率 P_ref 和傳遞到負載的功率 P_load。
(c) 求傳輸線上的最大電壓 V_max 和最小電壓 V_min。
(d) 求最大電流 I_max 和最小電流 I_min。
(e) 求 Return Loss (dB) 和 Mismatch Loss (dB)。

**詳解**：

**解題思路**：從 VSWR 推算各物理量，這是業界常用的分析流程。

**步驟**：

**(a) |Γ| 由 VSWR 求得**

$$|\Gamma| = \frac{\text{VSWR}-1}{\text{VSWR}+1} = \frac{3-1}{3+1} = \frac{2}{4} = 0.5$$

**(b) 功率計算**

反射功率：

$$P_{ref} = |\Gamma|^2 P_{inc} = 0.25\times 100 = 25 \text{ W}$$

負載功率：

$$P_{load} = P_{inc}-P_{ref} = (1-|\Gamma|^2)P_{inc} = 0.75\times 100 = 75 \text{ W}$$

**(c) 最大和最小電壓**

入射波電壓振幅：P_inc = V₊²/(2Z₀)

$$V_+ = \sqrt{2Z_0 P_{inc}} = \sqrt{2\times 50\times 100} = \sqrt{10000} = 100 \text{ V}$$

$$V_{max} = V_+(1+|\Gamma|) = 100\times 1.5 = 150 \text{ V}$$

$$V_{min} = V_+(1-|\Gamma|) = 100\times 0.5 = 50 \text{ V}$$

驗證：VSWR = V_max/V_min = 150/50 = 3 ✓

**(d) 最大和最小電流**

$$I_{max} = \frac{V_+}{Z_0}(1+|\Gamma|) = \frac{100}{50}\times 1.5 = 3 \text{ A}$$

$$I_{min} = \frac{V_+}{Z_0}(1-|\Gamma|) = \frac{100}{50}\times 0.5 = 1 \text{ A}$$

注意：V_max 和 I_min 出現在同一位置，V_min 和 I_max 出現在同一位置。

V_max 處的阻抗：Z_max = V_max/I_min = 150/1 = 150 Ω = VSWR × Z₀ ✓

V_min 處的阻抗：Z_min = V_min/I_max = 50/3 = 16.67 Ω = Z₀/VSWR ✓

**(e) Return Loss 和 Mismatch Loss**

Return Loss (RL)：

$$RL = -20\log_{10}|\Gamma| = -20\log_{10}(0.5) = -20\times(-0.301) = 6.02 \text{ dB}$$

Mismatch Loss (ML)：

$$ML = -10\log_{10}(1-|\Gamma|^2) = -10\log_{10}(0.75) = -10\times(-0.125) = 1.25 \text{ dB}$$

意義：入射功率中有 1.25 dB 的功率因為反射而未能傳到負載。

**答案**：

> (a) |Γ| = 0.5
>
> (b) P_ref = 25 W, P_load = 75 W
>
> (c) V_max = 150 V, V_min = 50 V
>
> (d) I_max = 3 A, I_min = 1 A
>
> (e) Return Loss = 6.02 dB, Mismatch Loss = 1.25 dB

**延伸思考**：業界常用的 Return Loss 標準 — 一般要求 RL > 10 dB（|Γ| < 0.316, VSWR < 1.92），較嚴格的要求 RL > 15 dB（|Γ| < 0.178, VSWR < 1.43），高精度系統要求 RL > 20 dB（|Γ| < 0.1, VSWR < 1.22）。

### 題目 57：單截線匹配設計
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★★★
**主題**：傳輸線 — 單截線匹配(Single-Stub Matching)

**題目**：

一 50 Ω 傳輸線終端接負載 Z_L = (25 − j25) Ω。使用並聯短路截線(Shunt Short-Circuited Stub)進行阻抗匹配。
(a) 求歸一化負載阻抗和導納。
(b) 求截線的接入位置 d（距負載的距離）。
(c) 求截線的長度 l_s。
(d) 驗證匹配效果。

**詳解**：

**解題思路**：單截線匹配的核心思想 — 先找到傳輸線上導納實部 g = 1 的位置（截線接入點），再用截線提供虛部相消的電納(Susceptance)。由於是並聯結構，用導納(Admittance)分析最方便。

**步驟**：

**(a) 歸一化**

$$z_L = \frac{Z_L}{Z_0} = \frac{25-j25}{50} = 0.5-j0.5$$

$$y_L = \frac{1}{z_L} = \frac{1}{0.5-j0.5} = \frac{0.5+j0.5}{0.25+0.25} = \frac{0.5+j0.5}{0.5} = 1+j1$$

在 Smith 圖上，y_L = 1 + j1 位於 g = 1 圓上，且 b = +1（感性電納）。

**(b) 截線接入位置**

由於 y_L 已在 g = 1 圓上（幸運的情況！），d 可以是 0。

但讓我們用一般方法處理。沿傳輸線從負載向訊號源移動，需要找到 y(d) 的實部 = 1 的位置。

y_L 的反射係數：

$$\Gamma_L^y = \frac{1-y_L}{1+y_L} = \frac{1-(1+j1)}{1+(1+j1)} = \frac{-j1}{2+j1} = \frac{-j(2-j)}{(2+j)(2-j)} = \frac{-j2+j^2}{5} = \frac{-1-j2}{5}$$

$$= -0.2-j0.4$$

|Γ| = √(0.04+0.16) = √0.20 = 0.4472

∠Γ = arctan(0.4/0.2) + 180° = 243.4°（第三象限）

在 y 的 Smith 圖上（導納 Smith 圖），y_L 已在 g = 1 圓上。

但因為題目要求完整設計，讓我找到 g = 1 圓上的兩個交點。

向訊號源方向移動距離 d，導納變為：

$$y(d) = \frac{1+\Gamma_L e^{-j2\beta d}}{1-\Gamma_L e^{-j2\beta d}}$$

（這裡用阻抗 Smith 圖的反射係數轉導納）

讓我改用更系統的方法。

直接用公式：

$$y_{in}(d) = \frac{y_L+j\tan\beta d}{1+jy_L\tan\beta d}$$

設 t = tanβd，y_L = 1 + j1：

$$y_{in} = \frac{(1+j)+jt}{1+j(1+j)t} = \frac{1+j(1+t)}{(1+t)+jt}$$

（注意分母 = 1+j(1+j)t = 1 + jt + j²t = 1−t + jt = (1−t)+jt，等一下）

重算分母：1 + j(1+j)t = 1 + jt + j²t = 1 + jt − t = (1−t) + jt

分子：(1+j) + jt = 1 + j(1+t)

$$y_{in} = \frac{1+j(1+t)}{(1-t)+jt}$$

有理化（乘以共軛）：

$$= \frac{[1+j(1+t)][(1-t)-jt]}{[(1-t)+jt][(1-t)-jt]}$$

分母 = (1−t)² + t² = 1 − 2t + 2t²

分子實部 = 1·(1−t) + (1+t)·t = 1−t + t + t² = 1 + t²

分子虛部 = (1+t)(1−t) − 1·t = 1 − t² − t

$$y_{in} = \frac{(1+t^2)+j(1-t-t^2)}{1-2t+2t^2}$$

令實部 g = 1：

$$\frac{1+t^2}{1-2t+2t^2} = 1$$

$$1+t^2 = 1-2t+2t^2$$

$$0 = -2t+t^2$$

$$t(t-2) = 0$$

解：t = 0 或 t = 2

**解 1：t = tanβd = 0 → d = 0**

此時 y_in = y_L = 1 + j1，b = +1

需要截線提供 b = −1 來消除。

**解 2：t = tanβd = 2 → βd = arctan(2) = 63.43° → d = 0.1762λ**

此時：

$$b = \frac{1-t-t^2}{1-2t+2t^2} = \frac{1-2-4}{1-4+8} = \frac{-5}{5} = -1$$

y_in = 1 − j1

需要截線提供 b = +1 來消除。

**(c) 截線長度**

短路截線的電納：b_stub = −cotβl_s（短路截線導納為 −jcotβl_s，等一下）

短路截線的輸入導納：

$$y_{stub} = \frac{1}{jZ_0\tan\beta l_s/Z_0} = \frac{1}{j\tan\beta l_s} = -j\cot\beta l_s$$

所以 b_stub = −cotβl_s。

**解 1（d = 0）**：需要 b_stub = −1

−cotβl_s = −1 → cotβl_s = 1 → βl_s = π/4 → l_s = λ/8

**解 2（d = 0.1762λ）**：需要 b_stub = +1

−cotβl_s = +1 → cotβl_s = −1 → βl_s = 3π/4 → l_s = 3λ/8

**(d) 驗證（以解 1 為例）**

截線接在 d = 0（直接在負載處並聯）：

負載導納：y_L = 1 + j1

截線導納：y_stub = −j·cot(π/4) = −j·1 = −j1，即 b_stub = −1

總並聯導納：

y_total = y_L + y_stub = (1+j1) + (−j1) = 1 + j0 = 1（匹配！）

Z_in = Z₀/y_total = 50/1 = 50 Ω ✓

Γ = 0，VSWR = 1，完美匹配。

**答案**：

> (a) z_L = 0.5−j0.5, y_L = 1+j1
>
> (b) 解 1：d = 0；解 2：d = 0.1762λ
>
> (c) 解 1：l_s = λ/8；解 2：l_s = 3λ/8
>
> (d) 驗證：y_total = 1，完美匹配

**延伸思考**：單截線匹配可以匹配任何負載阻抗。兩個解（d₁ 和 d₂）中，截線較短的解通常較佳（頻寬較寬且損耗較小）。在微帶線(Microstrip)設計中，通常選擇開路截線（較容易實作），短路截線需要通孔(Via Hole)接地。雙截線匹配(Double-Stub Matching)使用兩個固定間距的截線，提供更多設計自由度。

### 題目 58：Smith 圖完整阻抗匹配設計
**來源**：台大電機 / 業界面試高頻題
**難度**：★★★★★
**主題**：傳輸線 — Smith 圖阻抗匹配

**題目**：

使用 Smith 圖設計一個匹配網路：Z₀ = 50 Ω 傳輸線，負載 Z_L = (100 + j80) Ω。用並聯短路截線匹配。詳細描述 Smith 圖上的操作步驟。

**詳解**：

**解題思路**：在 Smith 圖上一步一步操作，這是業界面試中最常見的考題之一。

**步驟**：

**Step 1：歸一化**

$$z_L = \frac{Z_L}{Z_0} = \frac{100+j80}{50} = 2+j1.6$$

**Step 2：在 Smith 圖上標出 z_L 點**

找 r = 2 等電阻圓和 x = 1.6 等電抗弧的交點。標記為點 A。

計算 Γ_L 以確認位置：

$$\Gamma_L = \frac{z_L-1}{z_L+1} = \frac{1+j1.6}{3+j1.6} = \frac{(1+j1.6)(3-j1.6)}{9+2.56}$$

$$= \frac{3-j1.6+j4.8+2.56}{11.56} = \frac{5.56+j3.2}{11.56} = 0.481+j0.277$$

|Γ_L| = √(0.231+0.0767) = √0.308 = 0.555

∠Γ_L = arctan(0.277/0.481) = 29.9°

**Step 3：轉換到導納**

在 Smith 圖上，旋轉 180° 到達對徑點（點 B）：

$$y_L = \frac{1}{z_L} = \frac{1}{2+j1.6} = \frac{2-j1.6}{4+2.56} = \frac{2-j1.6}{6.56} = 0.305-j0.244$$

在導納 Smith 圖（或同一 Smith 圖旋轉 180°後讀取）上標出 y_L = 0.305 − j0.244。

**Step 4：沿等 |Γ| 圓順時針旋轉到 g = 1 圓**

從 y_L 點（點 B）出發，沿 |Γ| = 0.555 的圓順時針旋轉（向訊號源方向），找到與 g = 1 圓的交點。

g = 1 圓在導納 Smith 圖上通過中心點（y = 1），是一個穿過圖中心的圓。

需要解方程求精確交點。使用公式：

$$y(d) = \frac{y_L+j\tan\beta d}{1+jy_L\tan\beta d}$$

設 y_L = 0.305 − j0.244 = g_L + jb_L，令 t = tanβd：

$$g(d) = \text{Re}\{y(d)\} = 1$$

這需要數值求解。使用 Smith 圖的刻度讀取：

y_L 在 Smith 圖外圈的位置（WTG，Wavelengths Toward Generator 刻度）：

先求 ∠Γ_y = ∠Γ_L − 180° = 29.9° − 180° = −150.1°

WTG 位置 = 0.5 + (−150.1°)/(720°) = 0.5 − 0.2085 = 0.2915λ

（因為 Smith 圖外圈從 Γ = 1 即最右端的 0 開始，順時針一圈是 0.5λ）

沿 |Γ| = 0.555 的圓順時針旋轉，找 g = 1 圓的交點。

經 Smith 圖讀取（或數值計算），第一個交點約在 WTG = 0.335λ 處：

移動距離 d₁ = 0.335 − 0.2915 = 0.0435λ

此處的導納約為 y₁ = 1 + j1.35

第二個交點約在 WTG = 0.415λ 處：

移動距離 d₂ = 0.415 − 0.2915 = 0.1235λ

此處的導納約為 y₂ = 1 − j1.35

**Step 5：用截線消除電納**

選擇第二個解 d₂ = 0.1235λ（y₂ = 1 − j1.35），需要截線提供 b = +1.35：

短路截線：b_stub = −cotβl_s = +1.35

cotβl_s = −1.35

tanβl_s = −1/1.35 = −0.741

βl_s = π − arctan(0.741) = 180° − 36.5° = 143.5°

l_s = 143.5°/360° × λ = 0.399λ

或選擇第一個解 d₁ = 0.0435λ（y₁ = 1 + j1.35），需要截線提供 b = −1.35：

短路截線：b_stub = −cotβl_s = −1.35

cotβl_s = 1.35

tanβl_s = 0.741

βl_s = 36.5° → l_s = 0.1014λ

**第一個解更佳**（截線較短），取 d₁ = 0.0435λ，l_s = 0.1014λ。

**Step 6：驗證**

在距負載 d₁ = 0.0435λ 處的導納：y₁ = 1 + j1.35

截線導納：y_stub = −j·cot(36.5°) = −j·1.35

總導納：y_total = (1+j1.35) + (−j1.35) = 1

Z_in = Z₀ = 50 Ω ✓

**答案**：

> Smith 圖操作流程：
> 1. 標 z_L = 2+j1.6（點 A）
> 2. 旋轉 180° 得 y_L = 0.305−j0.244（點 B）
> 3. 沿等|Γ|圓順時針旋轉到 g=1 圓交點
> 4. 讀取電納值，設計截線長度消除電納
>
> 最終設計：d = 0.044λ, l_s = 0.101λ

**延伸思考**：Smith 圖操作的核心邏輯是：(1) 沿傳輸線移動 = 沿等 |Γ| 圓旋轉（順時針向源，逆時針向負載），(2) 並聯元件 = 在導納圖上沿等 g 圓移動。熟練使用 Smith 圖後，可以在幾分鐘內完成複雜的匹配設計，這也是為什麼它至今仍是射頻工程師的必備技能。

### 題目 59：傳輸線暫態分析 — 脈衝反射
**來源**：台大電機 經典題型
**難度**：★★★★
**主題**：傳輸線 — 暫態分析(Transient Analysis)

**題目**：

一無損傳輸線（Z₀ = 50 Ω，單程傳播延遲 T = l/v_p = 1 ns），源端接 V_s = 10 V 電壓源（內阻 R_s = 150 Ω），終端接負載 R_L = 150 Ω。在 t = 0 時電壓源突然開啟。
(a) 求初始入射波電壓 V₁⁺。
(b) 求源端反射係數 Γ_s 和負載端反射係數 Γ_L。
(c) 畫出彈跳圖(Bounce Diagram)前 4 次反射。
(d) 求 t → ∞ 時負載上的穩態電壓。

**詳解**：

**解題思路**：傳輸線暫態分析用彈跳圖(Lattice Diagram / Bounce Diagram)追蹤信號在源端和負載端之間的多次反射。

**步驟**：

**(a) 初始入射波**

t = 0⁺ 時，傳輸線呈現特性阻抗 Z₀（尚未有反射波返回）：

$$V_1^+ = V_s\cdot\frac{Z_0}{R_s+Z_0} = 10\times\frac{50}{150+50} = 10\times\frac{50}{200} = 2.5 \text{ V}$$

**(b) 反射係數**

源端反射係數（從傳輸線看向源端）：

$$\Gamma_s = \frac{R_s-Z_0}{R_s+Z_0} = \frac{150-50}{150+50} = \frac{100}{200} = 0.5$$

負載端反射係數：

$$\Gamma_L = \frac{R_L-Z_0}{R_L+Z_0} = \frac{150-50}{150+50} = \frac{100}{200} = 0.5$$

**(c) 彈跳圖**

```
        源端(z=0)                               負載端(z=l)
t=0     |------ V₁⁺ = 2.5 V -------->           |
        |                                t=T     |
t=2T    |<------ V₁⁻ = Γ_L·V₁⁺ -----           |
        |        = 0.5×2.5 = 1.25 V             |
t=2T    |------ V₂⁺ = Γ_s·V₁⁻ ------>           |
        |        = 0.5×1.25 = 0.625 V           |
        |                                t=3T    |
t=4T    |<------ V₂⁻ = Γ_L·V₂⁺ -----           |
        |        = 0.5×0.625 = 0.3125 V         |
t=4T    |------ V₃⁺ = Γ_s·V₂⁻ ------>           |
        |        = 0.5×0.3125 = 0.15625 V       |
        ...
```

**負載端電壓隨時間的變化：**

| 時間 | 事件 | 負載電壓增量 | 負載累計電壓 |
|------|------|-------------|-------------|
| t = T | V₁⁺ 到達，反射 V₁⁻ | V₁⁺(1+Γ_L) = 2.5×1.5 = 3.75 V | 3.75 V |
| t = 3T | V₂⁺ 到達 | V₂⁺(1+Γ_L) = 0.625×1.5 = 0.9375 V | 4.6875 V |
| t = 5T | V₃⁺ 到達 | V₃⁺(1+Γ_L) = 0.15625×1.5 = 0.234 V | 4.922 V |
| t = 7T | V₄⁺ 到達 | 0.0586 V | 4.980 V |

**(d) 穩態電壓**

t → ∞ 時，所有反射消失，傳輸線等效為直流穩態（電感短路、電容開路）：

$$V_L = V_s\cdot\frac{R_L}{R_s+R_L} = 10\times\frac{150}{150+150} = 5 \text{ V}$$

用級數驗證：

$$V_L = V_1^+(1+\Gamma_L)\sum_{n=0}^{\infty}(\Gamma_s\Gamma_L)^n = 2.5\times 1.5\times\frac{1}{1-0.5\times 0.5}$$

$$= 3.75\times\frac{1}{0.75} = 5 \text{ V} \quad \checkmark$$

**答案**：

> (a) V₁⁺ = 2.5 V
>
> (b) Γ_s = Γ_L = 0.5
>
> (c) 彈跳圖如上，負載電壓逐級接近 5V
>
> (d) 穩態 V_L = 5 V

**延伸思考**：此問題在數位電路的信號完整性(Signal Integrity)分析中極為重要。當 PCB 走線未做阻抗匹配時，高速數位信號會在驅動端和接收端之間反覆彈跳，造成振鈴(Ringing)、過衝(Overshoot)等問題。常見的端接(Termination)策略包括串聯端接(Source Termination, R_s = Z₀)和並聯端接(Load Termination, R_L = Z₀)。

### 題目 60：綜合設計 — 微帶線阻抗匹配系統
**來源**：台大電機 / 業界面試綜合題
**難度**：★★★★★
**主題**：傳輸線 — 綜合設計

**題目**：

一微帶線(Microstrip Line)射頻放大器系統：
- 源阻抗 Z_s = 50 Ω
- 傳輸線特性阻抗 Z₀ = 50 Ω，介質 ε_eff = 2.5
- 工作頻率 f = 2.4 GHz（Wi-Fi 頻段）
- 負載（天線）阻抗 Z_L = (30 − j40) Ω

要求設計匹配網路使 VSWR < 1.5。

(a) 計算未匹配時的 VSWR 和反射損耗。
(b) 方案一：設計四分之一波長轉換器（若負載為實數）。先用截線將負載虛部消除。
(c) 方案二：設計單截線匹配（並聯短路截線）。
(d) 計算各方案中微帶線段的物理長度。

**詳解**：

**解題思路**：綜合運用前面所學，完成實際的射頻匹配設計。

**步驟**：

**(a) 未匹配時的性能**

$$\Gamma_L = \frac{Z_L-Z_0}{Z_L+Z_0} = \frac{(30-j40)-50}{(30-j40)+50} = \frac{-20-j40}{80-j40}$$

$$= \frac{(-20-j40)(80+j40)}{(80-j40)(80+j40)} = \frac{-1600-j800-j3200-j^2 1600}{6400+1600}$$

$$= \frac{-1600+1600-j4000}{8000} = \frac{0-j4000}{8000} = -j0.5$$

$$|\Gamma_L| = 0.5, \quad \angle\Gamma_L = -90°$$

$$\text{VSWR} = \frac{1+0.5}{1-0.5} = 3$$

$$\text{Return Loss} = -20\log(0.5) = 6.02 \text{ dB}$$

反射功率 = |Γ|² = 25%，即 75% 的功率傳到負載。

$$\text{Mismatch Loss} = -10\log(1-0.25) = 1.25 \text{ dB}$$

VSWR = 3 遠超過 1.5 的要求，必須做匹配。

**(b) 方案一：截線 + 四分之一波長轉換器**

Step 1：用截線消除負載的虛部

在負載 Z_L = 30−j40 旁並聯一個截線，使虛部為零。

負載導納：

$$Y_L = \frac{1}{Z_L} = \frac{1}{30-j40} = \frac{30+j40}{900+1600} = \frac{30+j40}{2500} = (12+j16)\times 10^{-3} \text{ S}$$

需要截線提供 B_stub = −16×10⁻³ S = −j0.016 S

歸一化：y_L = Z₀·Y_L = 50(0.012+j0.016) = 0.6+j0.8

需要截線提供 b_stub = −0.8

短路截線：b_stub = −cotβl_s = −0.8

cotβl_s = 0.8 → tanβl_s = 1.25 → βl_s = arctan(1.25) = 51.34°

l_s1 = 51.34°/360° × λ = 0.1426λ

並聯後的導納：y_total = 0.6 + j0.8 + (−j0.8) = 0.6

Z_total = Z₀/0.6 = 50/0.6 = 83.33 Ω（純電阻）

Step 2：設計四分之一波長轉換器

$$Z_T = \sqrt{Z_0\times Z_{total}} = \sqrt{50\times 83.33} = \sqrt{4166.7} = 64.55 \text{ Ω}$$

轉換器長度 l_T = λ/4

**(c) 方案二：單截線匹配**

歸一化負載導納：y_L = 0.6 + j0.8

在 Smith 圖上找 g = 1 圓的交點。

$$y(d) = \frac{y_L+jt}{1+jy_Lt} \quad \text{其中 } t = \tan\beta d$$

y_L = 0.6 + j0.8

$$y(d) = \frac{(0.6+j0.8)+jt}{1+j(0.6+j0.8)t} = \frac{0.6+j(0.8+t)}{(1-0.8t)+j0.6t}$$

有理化：

分母共軛 = (1−0.8t) − j0.6t

分母模方 = (1−0.8t)² + (0.6t)² = 1 − 1.6t + 0.64t² + 0.36t² = 1 − 1.6t + t²

分子 × 分母共軛的實部：

= 0.6(1−0.8t) + (0.8+t)(0.6t)

= 0.6 − 0.48t + 0.48t + 0.6t²

= 0.6 + 0.6t²

令 g = 1：

$$\frac{0.6+0.6t^2}{1-1.6t+t^2} = 1$$

$$0.6+0.6t^2 = 1-1.6t+t^2$$

$$0.4t^2-1.6t+0.4 = 0$$

$$t^2-4t+1 = 0$$

$$t = \frac{4\pm\sqrt{16-4}}{2} = \frac{4\pm\sqrt{12}}{2} = 2\pm\sqrt{3}$$

解 1：t₁ = 2+√3 = 3.732 → βd₁ = arctan(3.732) = 75° → d₁ = 0.2083λ

解 2：t₂ = 2−√3 = 0.2679 → βd₂ = arctan(0.2679) = 15° → d₂ = 0.04167λ

計算各解對應的電納：

分子虛部 × 分母共軛：

= (0.8+t)(1−0.8t) − 0.6·(0.6t)

= 0.8 − 0.64t + t − 0.8t² − 0.36t

= 0.8 − 0.8t² + t(1 − 0.64 − 0.36)

= 0.8 − 0.8t²

等一下，讓我重算虛部：

分子 × 共軛 的虛部 = (0.8+t)(1−0.8t) − 0.6(0.6t)

= 0.8 − 0.64t + t − 0.8t² − 0.36t

= 0.8 + t(1−0.64−0.36) − 0.8t²

= 0.8 − 0.8t²

b = (0.8 − 0.8t²)/(1 − 1.6t + t²)

解 1（t₁ = 3.732）：

b₁ = (0.8 − 0.8×13.93)/(1 − 5.971 + 13.93) = (0.8 − 11.14)/(8.96) = −10.34/8.96 = −1.155

解 2（t₂ = 0.2679）：

b₂ = (0.8 − 0.8×0.0718)/(1 − 0.4287 + 0.0718) = (0.8 − 0.0574)/(0.6431) = 0.7426/0.6431 = 1.155

**解 2 較佳**（截線更短）：

d₂ = 0.04167λ，需要截線提供 b_stub = −1.155

短路截線：−cotβl_s = −1.155

cotβl_s = 1.155 → tanβl_s = 0.8660 → βl_s = 40.89°

l_s = 0.1136λ

**(d) 物理長度**

介質中波長：

$$\lambda = \frac{c}{f\sqrt{\varepsilon_{eff}}} = \frac{3\times 10^8}{2.4\times 10^9\times\sqrt{2.5}} = \frac{3\times 10^8}{3.795\times 10^9} = 0.07906 \text{ m} = 79.06 \text{ mm}$$

**方案一**（截線 + λ/4 轉換器）：

- 截線長度：l_s1 = 0.1426 × 79.06 = 11.27 mm
- 轉換器長度：l_T = λ/4 = 79.06/4 = 19.77 mm
- 轉換器阻抗：Z_T = 64.55 Ω

**方案二**（單截線匹配）：

- 截線到負載距離：d = 0.04167 × 79.06 = 3.29 mm
- 截線長度：l_s = 0.1136 × 79.06 = 8.98 mm

**方案比較**：

| | 方案一 | 方案二 |
|---|---|---|
| 總長度 | ~31 mm | ~12 mm |
| 複雜度 | 需兩種阻抗線 | 只需一種阻抗線 |
| 頻寬 | 受 λ/4 限制 | 受截線限制 |
| 實用性 | 較佳（系統化） | 較佳（緊湊） |

**答案**：

> (a) 未匹配：VSWR = 3, Return Loss = 6 dB, Mismatch Loss = 1.25 dB
>
> (b) 方案一：Z_T = 64.55 Ω，截線 11.27 mm + 轉換器 19.77 mm
>
> (c) 方案二：截線距負載 3.29 mm，截線長度 8.98 mm
>
> (d) 物理長度如上（λ = 79.06 mm）

**延伸思考**：在實際 Wi-Fi 天線設計中，常用的匹配技術包括微帶線截線匹配、集總元件(Lumped Element)匹配（在低頻可行）、以及天線本身的阻抗調整（如改變饋入點位置）。現代射頻設計常用 CAD 軟體（如 ADS, HFSS, CST）進行電磁場模擬和最佳化，但 Smith 圖仍是理解匹配原理的最佳工具，也是面試中必考的知識。傳輸線理論不僅用於射頻/微波，在高速數位電路（PCB 走線、DDR 記憶體介面、USB/PCIe 等）中同樣至關重要。
