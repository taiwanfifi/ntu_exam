# 第266章：遮罩

---

白板上多了一排方格。

八個。整整齊齊地排成一行。每個方格大約兩公分見方。藍色的線條。每個方格裡面是一個數字——0或1。

1　0　1　1　0　0　1　0

維倫指著它們。

「這是一個人的記憶遮罩。」

他在每個方格上面標了編號。1到8。紅色的。跟方格裡的藍色數字形成對比。

「一號記憶。童年。可見。」他指著第一格。1。

「二號記憶。初戀。隱藏。」第二格。0。

「三號記憶。母親的聲音。可見。」第三格。1。

他沒有繼續列舉剩下的五個。不需要。三個例子就夠了。讀者不需要知道四號到八號分別是什麼。他們只需要理解一件事：1等於可見，0等於隱藏。你自己選。

---

米亞的光球飄到白板旁邊。

比昨天更暗了一點。暗到——如果你不看她，你會以為角落裡只有一個影子。安全屋的牆壁是灰白色的，她的光球現在的亮度跟牆壁的反光差不多。你必須刻意去看她，才能看到她在那裡。

「位元遮罩。Bit mask。每一個bit對應一段記憶。1等於可見。0等於隱藏。」

她的語速比昨天又慢了一點。不明顯。像錶走慢了半格。如果你不把今天的語速和昨天的做對比，你不會注意到差別。但如果你注意到了——你會開始在意。開始計算。

「你的記憶。你自己決定——哪些給別人看。哪些只有你知道。」

---

維倫在白板上畫了第二排方格。

0　0　0　0　0　0　0　0

「這是一個全部隱藏的遮罩。什麼都看不到。」

第三排。

1　1　1　1　1　1　1　1

「這是全部公開的。所有人都能看見你的一切。」

他回到第一排。

1　0　1　1　0　0　1　0

「大部分人的選擇——會在這兩者之間。有些事願意讓人看。有些事——留給自己。」

---

嚴柏翰靠在門框上。手插在口袋裡。

他看著那排0和1。

他在想。他不常在團隊面前想事情——通常他的想法都在行動裡，快到連他自己都不確定中間有沒有經過「想」這個步驟。但現在他在想。眼睛盯著白板上那排方格。表情沒有變化。

如果十年前有這個系統，他會把哪些記憶設成0。

答案是：跟她有關的。全部。

不是因為羞恥，不是因為那些記憶裡有什麼見不得人的東西——是因為那些是他的。只有他的。

一個人折紙鶴的手指。一杯擺在桌上的溫茶。一個背影走進走廊的最後一幀。這些記憶不屬於這個世界的任何公共空間。它們屬於他口袋裡的那個位置——手伸進去就能碰到。只有他的手伸進去才能碰到。

0。全部0。

不是隱藏——是保護。

---

語青走到白板前面。

她的影子比昨天更淡了。日光燈從天花板照下來，正常情況下一個人站在燈下會投出一個清晰的、邊緣銳利的影子。語青的影子不銳利。邊緣在發散。像墨水滴進水裡——中心還有一點濃度，但四周已經模糊了，正在往地板的灰白色裡消融。

她看著那排方格。

「後門呢？」

她的聲音很輕。

維倫看著她。

「什麼後門？」

「如果有人想——繞過別人的遮罩，看見別人設成0的記憶——」

維倫搖頭。

「沒有後門。」

---

他在白板上寫了一段話。紅色的。字很小。

簡單到沒有後門——
因為後門需要至少一個多餘的bit。
而我們不給任何人多餘的bit。

他退後一步。看著那段話。像看一件剛完成的作品——不是滿意，是確認。確認每一個字都在它該在的位置。

米亞的光球亮了一下。然後暗回去。像是在點頭。

「數學上——如果每個人的遮罩只有本人能修改——系統中不存在任何第三方能改變你的bit。」

「不是『不應該』。是『不可能』。物理級別的不可能。」

她的聲音在「不可能」三個字上放慢了，因為這三個字的重量比其他字更大。「不應該」是法律——法律可以被修改、被繞過、被違反。「不可能」是物理。物理是宇宙的底線。

---

維倫在白板上畫了一個小鎖。鎖的圖案。一個U形的鎖環，一個方形的鎖體。

「遮罩加密鑰。你的記憶用你的密鑰加密。你的記憶用你的遮罩控制可見性。兩層。都只有你自己能動。」

他退後一步。看著白板的全貌。昨天的紅色。今天的藍色。0和1。方格。鑰匙。鎖。

「周明哲的系統——他有萬能鑰匙。也有萬能遮罩。他能看見所有人的0。他能改任何人的1。」

他看著白板上的小鎖。

「我們的系統——沒有萬能。因為簡單。簡單到沒有多餘的位子留給後門。」

---

邱韻如的聲音從角落傳來。她沒有離開筆電。十根手指還在鍵盤上。螢幕上是伺服器的配置介面。

「所有偉大的安全系統——都輸給了人性的弱點。你怎麼確保不會有人找到繞過遮罩的方法？」

她的語氣是職業的。精確的。二十年管理記憶銀行後台的人問出來的問題。不是質疑。是審核。

維倫沒有回答。

米亞替他回答了。零點七秒之後。

「因為繞過遮罩需要——修改你不持有的bit。在這個系統裡——你只持有你自己的bit。別人的bit對你來說——不存在。」

她停了一下。光球微微轉了幾度。

「對你來說，那些bit根本不存在。你不能駭入一個不存在的東西。」

邱韻如安靜了幾秒。螢幕上的數字在跳。伺服器的負載率。她沒有看。她在消化米亞的話。

然後她推了一下眼鏡。食指和拇指。右上角。兩毫米。

「好。」

一個字。

但從一個二十年來管理萬能遮罩的人嘴裡說出來——這個「好」比任何技術認證都重。她見過萬能遮罩的力量。她也見過萬能遮罩的危險。她比任何人都清楚，一把能打開所有門的鑰匙最終會被用來做什麼。

好。

不需要萬能鑰匙了。

---

維倫在白板上畫了兩個方框。一個裡面寫著「A的記憶」。一個寫著「B的記憶」。中間畫了一條線連接它們。

「密鑰解決了加密。遮罩解決了權限。」

他在兩個方框之間畫了一個問號。

「但如果兩個人想——把記憶藏在彼此裡面呢？」

米亞的光球微微轉了一下。暗了一個色階。然後亮回來。

「XOR。」她說。

停了零點八秒。

「異或。」
