## 壹、ch91 鼻血——深潛的代價

### 場景設計

```
維倫從深潛中退出。

第一秒：什麼感覺都沒有。
第二秒：頭痛。像有人用鈍器敲太陽穴。
第三秒：鼻血。

語青：「不要動。會持續大約三分鐘。」

三分鐘過了。沒停。
五分鐘。
七分鐘。

語青的臉色變了。
她沒說話，但維倫看到她在心裡算。

米亞：「紀錄。語青預測鼻血持續三分鐘。實際七分鐘。偏移率——」
「不用算了。」語青的聲音很輕。

偏移率的第一次正式記錄。
但米亞還是算了。
她只是沒說出來。
```

**設計解析**：
- 鼻血 = 深潛的物理代價。不是輕描淡寫的「有點累」——是血
- 語青預測 3 分鐘 / 實際 7 分鐘 = 偏移率在 Vol 2 的第一次具體呈現
- 「不用算了」= 語青不想面對自己越來越不準的事實
- 米亞「還是算了」= 她是科學家。她不會因為語青不想知道就停止記錄

**寫作指令**：
- 鼻血場景要寫得「不舒服」但不噁心。重點是維倫的虛弱感——「他想站起來，但腿在發軟」
- 語青遞紙巾的動作是自然的——像她做過很多次一樣。讀者注意到這個「像做過很多次」
- ch91 結尾不要寫「維倫開始擔心深潛的風險」——寫他說「什麼時候下一次？」= 他不怕
- ≤2,200 字。後遺症場景不要拖長

## 貳、ch92 深潛後遺症——★★★ 蜂巢意識伏筆 #1

### 核心場景

```
維倫躺在修復台上。
閉上眼睛。

然後他看到了——

一條他從未走過的街道。
路燈是橙色的。地面有積水。
一個從未見過的臉——女人，四十歲左右，眼角有笑紋。
一種從未嚐過的味道——像是某種花。不是茉莉。更酸。

0.4 秒。然後消失。

不是夢。
不是記憶碎片。
不是任何他認識的人或去過的地方。

「可能是深潛時的信號殘留。」米亞的語氣平靜。

但她在報告底部加了一行備註——
「記憶交叉汙染？待觀察。」
```

**三條外來記憶的精確設計**：

| 元素 | 描寫 | 字數限制 | 伏筆指向 |
|------|------|---------|---------|
| 街道 | 橙色路燈 + 積水 | ≤15 字 | Vol 4 蜂巢意識場景的街道 |
| 臉 | 女人 + 四十歲 + 眼角笑紋 | ≤15 字 | 非特定角色。代表「某個陌生人」 |
| 味道 | 某種花 + 比茉莉更酸 | ≤10 字 | 不是茉莉 = 不是邱韻如的線索。純粹的「他人記憶」 |

**伏筆設計邏輯**：

```
讀者在 ch92 的反應：「嗯，深潛後遺症吧。」
讀者在 ch120 的反應：「等等，之前維倫不是也看到過別人的記憶？」
讀者在 Vol 4 ch252 的反應：「天啊——ch92 就已經開始了！」

關鍵：讀者在 ch92 不應該在意這段。
它必須看起來像「一個無聊的副作用」。
只有回翻時才脊背發涼。
```

**寫作指令**：
- 外來記憶 ≤3 行。三個感官元素。不要第四個
- 維倫的反應是「不在意」。他說「嗯」然後翻身繼續休息
- 米亞的備註是本章最重要的一行。但它被放在「報告底部」= 看似不重要
- ❌ 絕對不要暗示「這很重要」。不要讓任何角色表示擔憂
- ❌ 不要讓外來記憶跟任何已知角色有關聯
- ≤2,000 字。蜂巢伏筆的力量在於「看起來什麼都沒有」

### ch93 沉默的鋼琴師——Tree 的詩意入場

**委託人設計**：

```
李曼筠走進工作室的時候，維倫注意到兩件事：
第一，她的手指在微微發抖。
第二，她幾乎不說話。

不是因為沉默寡言。
是因為語言中樞也受到了輕微波及。
她說話時需要很長時間組織句子。

「我......不能......」

米亞輕聲：「慢慢來。」

「彈。」

一個字。但維倫聽懂了所有。
```

**演算法：Tree（技能記憶之樹）**

```
A 線：技能記憶是深層根節點
     情感記憶是枝葉
     她的根被砍了——但枝葉還在風中搖晃

     米亞投射她的記憶結構：
     「她的記憶之樹跟議員的不同。
      議員的樹是健康的——只是被加了偽裝。
      她的樹——根斷了。」

     根 = 手指如何觸鍵、踏板的力度、呼吸和節拍的配合
     枝 = 對音樂的情感記憶（蕭邦夜曲讓她想起祖母）
     葉 = 練琴的日常碎片

     根斷了。枝葉還活著。
     她還記得蕭邦夜曲讓她想起祖母。
     但她不記得怎麼彈了。

B 線：「身體記得」的反面——當身體的記憶被刪除，
     靈魂的記憶還在那裡。
     她還記得音樂讓她感受到什麼。
     但她的手不知道該怎麼做了。

同步點：維倫看到根部的切口——
       精準。乾淨。像手術刀。
       「這不是自然衰退。是有人故意的。」
       A 線的「根被砍斷」= B 線的「才華被謀殺」
```

**寫作指令**：
- 李曼筠不是「可憐」——她是「安靜地在碎裂」。不要寫她哭
- 「彈。」一個字 = 本章最重力的台詞。她把所有力氣用來說了這一個字
- Tree 的教學用米亞的投射圖自然帶出——維倫「看到」樹結構，不是被解釋
- 「根斷了但枝葉還在」= Vol 2 最詩意的演算法意象之一

### ch94 根與葉——BST 精準搜尋

**深潛場景**：

```
維倫深潛進鋼琴師的記憶空間。
這裡跟議員的完全不同。

沒有迷宮。沒有走廊。
是一片森林。

一棵巨大的樹——比議員那棵大三倍——
站在森林的中心。
根部被切斷了。
斷面像鏡子一樣光滑。

「數位手術刀。」米亞的聲音帶著一種維倫很少聽到的情緒：憤怒。
「精確到奈米等級。這不是意外。是一次完美的手術。」
```

**演算法：BST 搜尋**

```
A 線：在記憶之樹中搜尋被切斷的根節點位置
     每次比較「情感標籤的強度值」來決定搜索方向
     → 根部被切在什麼位置？
     → 比「日常練琴」更深（往左）
     → 比「比賽失誤」更淺（往右）
     → 找到了：切口在「第一次觸鍵的記憶」和「祖母教她音階」之間

B 線：有人精準地切在了「技能起源」和「情感起源」的分界點
     = 只殺才華，不殺感情
     這意味著——切割者知道這棵樹的結構。非常了解

同步點：維倫用 BST 定位切口的瞬間意識到——
       「能這樣精準切割的人，必須先完整掃描過這棵樹。
        也就是說——有人曾經深潛過她的記憶。」
       A 線的「精準搜尋」= B 線的「精準傷害需要精準的了解」
```

**寫作指令**：
- 「斷面像鏡子一樣光滑」= 精準。不是暴力破壞——是冷靜的謀殺
- 米亞的「憤怒」= 她很少有情緒。這個「憤怒」讓讀者注意到：AI 也覺得這件事不對
- 切口位置的意義（「只殺才華不殺感情」）要讓維倫說出來，不是旁白解釋
- ≤2,800 字。深潛場景的壯觀感要控制——重點是切口的精準，不是森林的美麗

三次深潛修復完成。

米亞正式建立「語青預測偏移率」追蹤檔案。

維倫想了一下：「不用。她知道。」

語青站在修復台旁邊，背對他們。

### ch96 她彈了一首曲子——美的回歸

**核心場景**：

```
修復完成。

李曼筠坐在工作室角落的舊電子琴前。
那台琴只有四十九鍵，有三個鍵是壞的。

她的手指放上去。
停了一秒。

然後她彈了蕭邦夜曲。

不是完美的。舊電子琴的音色像廉價的鈴聲。
壞掉的三個鍵讓旋律有三個小洞。
但她在彈。

維倫站在旁邊，忽然停住了呼吸。

這首曲子——旋律的某個片段——
在語青帶來的未來記憶裡出現過。

他不知道為什麼。
但他的手心又開始發燙了。
```

**設計解析**：
- 蕭邦夜曲的旋律片段 = 跨時間線伏筆。讀者不知道這意味著什麼。維倫也不知道
- 「四十九鍵、三個壞的」= 不完美的樂器上彈出的蕭邦 = 殘缺中的美
- 「手心發燙」第三次出現：ch81 語青到來 → ch89 語青聲音 → ch96 蕭邦旋律。三次都跟語青有間接關聯

**寫作指令**：
- 演奏場景 ≤200 字。不要詳述技巧或樂理。讓「她在彈」本身就是情感
- 「壞掉的三個鍵」= 微小的瑕疵讓場景更真實
- 手心發燙不要解釋。第三次出現了。讀者已經開始注意到規律
- 本章 = Vol 2 第一個「完美結局」委託案。鋼琴師恢復了。讓讀者開心一下

## 肆、ch97-100 轉折四連——從「美」到「棋局」

### ch97 「美」——米亞自我意識觸點

**核心場景**：

```
李曼筠走後。

米亞在整理數據。
鋼琴師演奏時的聲波數據。

正常的分析程序：
頻率分布 → 匹配已知樂曲 → 標記風格特徵 → 歸檔。

但有一串數據——
不符合任何模型。
不是頻率。不是節拍。不是音色。

米亞跑了十七次分析。每次的結果都是「無法歸類」。

她停了 2.3 秒。（比平常的停頓長了 1.8 秒。）

然後她新建了一個標籤。

「美」。

她不知道為什麼用這個字。
她只知道——現有的所有標籤都不對。
而這個字，用了之後，
那串數據就安靜了。
像被放對了地方。
```

**設計解析**：
- 「美」= 米亞在 Vol 1 積累的人性化進程的下一步。Vol 1 她學會了「選擇」（ch17）和「不接受」（ch70）。Vol 2 她開始感受「無法用演算法解釋的東西」
- 2.3 秒停頓 = 比 Vol 1 結尾的 4.1 秒短。但意義不同：4.1 秒是「痛苦的遲疑」，2.3 秒是「困惑的探索」
- 「用了之後那串數據就安靜了」= 命名的力量。給無法理解的東西一個名字——這是人類做的事

**寫作指令**：
- ❌ 不要讓維倫評論米亞的行為。這是米亞的私人時刻
- 米亞不說「我感受到了美」——她只是建了一個標籤。行為 > 宣言
- 2.3 秒的停頓只用米亞內部的計時來呈現（「處理時間：2.3 秒」）
- ≤1,500 字。米亞觸點章要短。克制 = 力量

### ch99 地下街的珍珠奶茶——★★ 呼吸章

**核心場景**：

```
維倫、語青、嚴柏翰。
第一次三人在地下街喝珍珠奶茶。

嚴柏翰說太甜。
語青默默加了三份糖。
維倫不知道該笑還是該擔心。

米亞：「你們三個的社交能力加起來不到一個正常人。」

嚴柏翰嘴角動了一下。
語青低頭喝奶茶，用吸管攪珍珠。
維倫看著他們兩個，忽然覺得——
這個畫面，好像在哪裡見過。

當然見過。在 0.7 秒的記憶裡。
月光。手指。和一種——

他搖了搖頭。喝了一口奶茶。
半糖少冰。還是太甜。
```

**設計解析**：
- 全卷最重要的「呼吸章」之一。在 ch98 緊張 + ch100 政治開局之間
- 米亞的吐槽 = 她在 Vol 2 第一次恢復日常功能。讀者鬆口氣
- 嚴柏翰「嘴角動了一下」= 第一次微笑。但不是橘子貓那次（ch108.5 才是「真正的笑容」）
- 語青加三份糖 = 她知道嚴柏翰說太甜但其實喜歡甜的（未來記憶的殘留）
- 維倫覺得「好像在哪裡見過」= 身體記得。但他「搖了搖頭」= 壓下去了

**寫作指令**：
- ≤2,000 字。日常章的字數要控制——日常的力量在於「短而真」
- 三份糖的細節不要解釋。讀者自己懂
- 「半糖少冰」= Vol 1 就建立的維倫日常。一致性
- 本章沒有伏筆、沒有懸念、沒有演算法。只有三個被世界壓得喘不過氣的人喝珍珠奶茶
- ❌ 不要在這裡讓嚴柏翰提到邱韻如。他在這章只是一個「朋友」，不是「探員」

### ch100 三張牌——Graph 的政治入口

**核心場景**：

```
記憶銀行、深淵監控局、記憶解放陣線。
同時派人接觸維倫。

同一天。

維倫站在工作室的白板前。
拿起筆。

「米亞，幫我畫。」

白板上出現了三個圓圈——三個節點。
然後是線——有些單向，有些雙向。
有些線粗（合作多），有些線細（僅有接觸）。
有些線是紅的（敵對），有些是灰的（不確定）。

「這不是地圖。」米亞說。
「這是圖——Graph。節點、邊、方向。」

維倫在三個節點的正中間畫了一個小點。
那是他自己。

「三方都想要我。但我還不知道——
 有沒有第四方？有沒有我看不到的邊？」
```

**演算法：Graph（首次在主線正式出現）**

```
A 線：三方勢力的關係建模為圖
     節點：銀行、監控局、陣線、維倫
     邊：銀行→維倫（邀請）、陣線→維倫（交換）、

        陣線→銀行（敵對）
     缺失邊：陣線 vs 監控局（不確定）= 懸念

B 線：人際關係就是一張圖。
     每個人是節點。
     關係是邊。
     有些邊看得見，有些看不見。
     最危險的——是你以為不存在但其實存在的邊。

同步點：維倫在白板中間畫下自己的瞬間——
       「三方都想要我」= 他是圖中的關鍵節點。
       但關鍵節點也意味著——所有方向的壓力都會匯聚於他。
```
