# 第八章：頻率響應與拉普拉斯轉換

> **適用對象**：零基礎大一大二學生
> **先修知識**：第一至七章（交流穩態、阻抗、相量）
> **學習時間**：約 5-6 小時

## 🔰 **本章基礎觀念（零基礎必讀）**

上一章我們學了**固定頻率**的交流電路分析。但在真實世界，訊號通常包含**很多不同的頻率**。

例如：音樂訊號中同時有低音（低頻）和高音（高頻）。你的音響有低音和高音旋鈕——這些旋鈕就是在調節電路對**不同頻率的放大倍率**。

### 本章兩大主題

1. **頻率響應**：同一個電路，對不同頻率的訊號會有不同的增益和相位偏移
2. **拉普拉斯轉換**：一個更強大的數學工具，把微分方程變成代數方程

### 比喻

把電路想成一個餐廳：
- **轉移函數**就是「菜單」——告訴你輸入什麼（原料）會得到什麼（菜色）
- **波德圖**就是「評分表」——在不同價位（頻率）下，食物品質（增益）如何
- **拉普拉斯轉換**就是「翻譯機」——把困難的食譜翻譯成簡單的步驟

## 一、轉移函數（Transfer Function）

### 1.1 定義

$$
\mathbf{H}(\omega) = \frac{\mathbf{V}_{out}}{\mathbf{V}_{in}} = \frac{\text{輸出相量}}{\text{輸入相量}}
$$

H(ω) 是一個**複數函數**，包含：
- **大小** |H(ω)|：增益（輸出振幅 / 輸入振幅）
- **角度** ∠H(ω)：相位偏移

### 1.2 用 s 域表示

用 s = jω 代替：

$$
H(s) = \frac{V_{out}(s)}{V_{in}(s)}
$$

> s 域的好處：可以同時處理穩態和暫態。

### 1.3 數值例題 ①

> **題目**：RC 低通濾波器，R = 1kΩ，C = 1μF。求轉移函數 H(jω)。

**解答**：

```
  Vin ──[R]──●── Vout
              |
             [C]
              |
              ⏚
```

用分壓器：

$$
H(j\omega) = \frac{Z_C}{Z_R + Z_C} = \frac{\frac{1}{j\omega C}}{R + \frac{1}{j\omega C}} = \frac{1}{1 + j\omega RC}
$$

代入數值：RC = 1000 × 10⁻⁶ = 10⁻³ s

$$
H(j\omega) = \frac{1}{1 + j\omega \times 10^{-3}}
$$

截止頻率：ω_c = 1/RC = 1000 rad/s（f_c = 159.2 Hz）

答：**H(jω) = 1/(1 + jω/1000)**。

## 二、波德圖（Bode Plot）

### 2.1 什麼是波德圖？

波德圖是用來視覺化頻率響應的標準工具，由兩張圖組成：

1. **增益圖**：20log₁₀|H(jω)| (dB) vs log₁₀(ω)
2. **相位圖**：∠H(jω) (°) vs log₁₀(ω)

### 2.2 分貝（dB）

$$
|H|_{dB} = 20 \log_{10} |H|
$$

| 增益倍數 | dB 值 |
|----------|-------|
| 1 | 0 dB |
| 2 | 6 dB |
| 10 | 20 dB |
| 100 | 40 dB |
| 0.1 | -20 dB |
| 1/√2 ≈ 0.707 | -3 dB |

> -3 dB 就是「半功率點」（功率減半），也是**截止頻率**的定義！

### 2.3 常見的波德圖基本型

#### 型一：常數 K

- 增益：20log|K| dB（水平線）
- 相位：0°（K > 0）或 -180°（K < 0）

#### 型二：零點 (1 + jω/ω_z)

- ω << ω_z：增益 ≈ 0 dB，相位 ≈ 0°
- ω = ω_z：增益 = +3 dB，相位 = +45°
- ω >> ω_z：斜率 +20 dB/decade，相位 → +90°

#### 型三：極點 1/(1 + jω/ω_p)

- ω << ω_p：增益 ≈ 0 dB，相位 ≈ 0°
- ω = ω_p：增益 = -3 dB，相位 = -45°
- ω >> ω_p：斜率 -20 dB/decade，相位 → -90°

#### 型四：原點零點 jω/ω₀

- 斜率 +20 dB/decade（穿過 ω₀ 時為 0 dB）
- 相位固定 +90°

#### 型五：原點極點 1/(jω/ω₀)

- 斜率 -20 dB/decade
- 相位固定 -90°

### 2.4 如何手繪近似波德圖

**步驟**：

| 步驟 | 操作 |
|------|------|
| ① | 把 H(jω) 分解成基本型的乘積 |
| ② | 找出所有零點和極點的頻率 |
| ③ | 從低頻開始，畫出初始增益（水平線） |
| ④ | 每遇到一個極點頻率，斜率 **-20 dB/dec** |
| ⑤ | 每遇到一個零點頻率，斜率 **+20 dB/dec** |
| ⑥ | 相位圖：在轉折頻率附近的一個十倍頻範圍內，相位變化 ±90° |

### 2.5 數值例題 ②

> **題目**：H(jω) = 100/(1 + jω/10)(1 + jω/1000)。手繪波德圖的關鍵特徵。

**解答**：

常數項：20log(100) = 40 dB

極點 1：ω_p1 = 10 rad/s
極點 2：ω_p2 = 1000 rad/s

**增益圖**：
- ω < 10：水平，40 dB
- 10 < ω < 1000：斜率 -20 dB/dec（一個極點作用）
  - 在 ω = 1000 時：40 - 20×log(1000/10) = 40 - 40 = 0 dB
- ω > 1000：斜率 -40 dB/dec（兩個極點同時作用）

**相位圖**：
- ω << 10：0°
- ω = 10：-45°
- ω = 100（兩個極點之間的幾何平均）：約 -90°
- ω = 1000：-135°
- ω >> 1000：-180°

答：低頻增益 40dB，在 10 rad/s 開始下降，1000 rad/s 後加速下降。

## 三、濾波器分類

### 3.1 四種基本類型

| 濾波器 | 通過的頻率 | 阻擋的頻率 | 轉移函數特徵 |
|--------|-----------|-----------|------------|
| 低通（Low-pass） | 低頻 | 高頻 | H(0) = K，H(∞) = 0 |
| 高通（High-pass） | 高頻 | 低頻 | H(0) = 0，H(∞) = K |
| 帶通（Band-pass） | 中間頻帶 | 低頻和高頻 | 在 ω₀ 附近有峰值 |
| 帶拒（Band-stop / Notch） | 低頻和高頻 | 中間頻帶 | 在 ω₀ 附近有凹谷 |

### 3.2 電路實現

| 濾波器 | 簡單電路 | 轉移函數 |
|--------|---------|---------|
| RC 低通 | R-C 串聯，取 C 電壓 | 1/(1 + jωRC) |
| RC 高通 | R-C 串聯，取 R 電壓 | jωRC/(1 + jωRC) |
| RLC 帶通 | RLC 串聯，取 R 電壓 | — |
| RLC 帶拒 | RLC 串聯，取 LC 電壓 | — |

## 四、共振、品質因數 Q 與頻寬 BW

### 4.1 共振頻率

RLC 串聯電路的共振頻率：

$$
\omega_0 = \frac{1}{\sqrt{LC}}
$$

在共振時：
- 阻抗最小（Z = R，純電阻）
- 電流最大
- Z_L 和 Z_C 相消

### 4.2 品質因數 Q

$$
Q = \frac{\omega_0 L}{R} = \frac{1}{\omega_0 RC} = \frac{1}{R}\sqrt{\frac{L}{C}}
$$

Q 代表**選頻的尖銳程度**：
- Q 高 → 共振峰很尖 → 選頻能力強
- Q 低 → 共振峰很平 → 選頻能力差

### 4.3 頻寬 BW

$$
BW = \frac{\omega_0}{Q} = \frac{R}{L}
$$

頻寬 = 上截止頻率 - 下截止頻率 = ω₂ - ω₁

### 4.4 數值例題 ③

> **題目**：RLC 串聯電路 R = 10Ω，L = 0.1H，C = 10μF。求共振頻率、Q 值和頻寬。

**解答**：

$$
\omega_0 = \frac{1}{\sqrt{LC}} = \frac{1}{\sqrt{0.1 \times 10 \times 10^{-6}}} = \frac{1}{\sqrt{10^{-6}}} = \frac{1}{10^{-3}} = 1000 \text{ rad/s}
$$

$$
f_0 = \frac{\omega_0}{2\pi} = \frac{1000}{6.283} = 159.2 \text{ Hz}
$$

$$
Q = \frac{\omega_0 L}{R} = \frac{1000 \times 0.1}{10} = 10
$$

$$
BW = \frac{\omega_0}{Q} = \frac{1000}{10} = 100 \text{ rad/s}
$$

$$
f_{BW} = \frac{100}{2\pi} = 15.92 \text{ Hz}
$$

答：**ω₀ = 1000 rad/s，Q = 10，BW = 100 rad/s**。Q = 10 表示共振峰相當尖銳。

## 五、拉普拉斯轉換（Laplace Transform）

### 5.1 為什麼需要拉普拉斯轉換？

相量法只能處理**穩態的單一頻率**正弦訊號。但拉普拉斯轉換可以處理：
- 暫態響應
- 任意波形的輸入（步階、脈衝、斜坡...）
- 初始條件自然融入

### 5.2 定義

$$
F(s) = \mathcal{L}\{f(t)\} = \int_0^{\infty} f(t) e^{-st} \, dt
$$

其中 s = σ + jω 是複數頻率。

### 5.3 常見轉換對

| f(t)（t ≥ 0） | F(s) |
|---------------|------|
| δ(t)（脈衝） | 1 |
| u(t)（步階） | 1/s |
| t（斜坡） | 1/s² |
| t^n | n!/s^(n+1) |
| e^{-at} | 1/(s+a) |
| te^{-at} | 1/(s+a)² |
| sin(ωt) | ω/(s²+ω²) |
| cos(ωt) | s/(s²+ω²) |
| e^{-at}sin(ωt) | ω/((s+a)²+ω²) |
| e^{-at}cos(ωt) | (s+a)/((s+a)²+ω²) |

### 5.4 重要性質

| 性質 | 時域 | s 域 |
|------|------|------|
| 線性 | af₁ + bf₂ | aF₁ + bF₂ |
| 微分 | df/dt | sF(s) - f(0⁻) |
| 二次微分 | d²f/dt² | s²F(s) - sf(0⁻) - f'(0⁻) |
| 積分 | ∫f dt | F(s)/s |
| 時移 | f(t-a)u(t-a) | e^{-as}F(s) |
| 頻移 | e^{-at}f(t) | F(s+a) |

> 微分在 s 域變成乘以 s，而且**初始條件自動包含在內**！這就是拉普拉斯最強大的地方。

## 六、s 域電路分析

### 6.1 元件的 s 域模型

| 元件 | 時域 | s 域阻抗 | s 域模型（含初始條件） |
|------|------|---------|---------------------|
| 電阻 | v = Ri | Z(s) = R | 就是 R |
| 電感 | v = L(di/dt) | Z(s) = sL | sL 串聯電壓源 Li(0) |
| 電容 | i = C(dv/dt) | Z(s) = 1/(sC) | 1/(sC) 串聯電壓源 v(0)/s |

### 6.2 分析步驟

| 步驟 | 操作 |
|------|------|
| ① | 把所有元件轉換成 s 域模型（含初始條件） |
| ② | 把電源也轉換到 s 域 |
| ③ | 用一般電路分析方法（KVL、KCL、分壓器等）解出 V(s) 或 I(s) |
| ④ | 用**反拉普拉斯轉換**把結果轉回時域 |

### 6.3 數值例題 ④

> **題目**：RC 電路，R = 2Ω，C = 0.5F，v_C(0) = 5V，輸入 v_s(t) = 10u(t)（t=0 接上 10V 步階）。
> 用拉普拉斯轉換求 v_C(t)。

**解答**：

**s 域**：V_s(s) = 10/s

電容的 s 域阻抗：1/(sC) = 1/(0.5s) = 2/s

用分壓器（注意要加上初始條件）：

KVL 方程在 s 域：

$$
V_s(s) = RI(s) + \frac{I(s)}{sC} + \frac{v_C(0)}{s}
$$

等等，更直觀的方法：

$$
V_s(s) - \frac{v_C(0)}{s} = I(s)\left(R + \frac{1}{sC}\right)
$$

$$
\frac{10}{s} - \frac{5}{s} = I(s)\left(2 + \frac{2}{s}\right) = I(s) \cdot \frac{2s + 2}{s}
$$

$$
\frac{5}{s} = I(s) \cdot \frac{2(s + 1)}{s}
$$

$$
I(s) = \frac{5}{2(s + 1)}
$$

$$
V_C(s) = \frac{v_C(0)}{s} + \frac{I(s)}{sC} = \frac{5}{s} + \frac{I(s)}{0.5s} = \frac{5}{s} + \frac{5}{s(s+1)}
$$

部分分式展開 5/[s(s+1)]：

$$
\frac{5}{s(s+1)} = \frac{A}{s} + \frac{B}{s+1}
$$

5 = A(s+1) + Bs

s = 0：5 = A → A = 5
s = -1：5 = -B → B = -5

$$
V_C(s) = \frac{5}{s} + \frac{5}{s} - \frac{5}{s+1} = \frac{10}{s} - \frac{5}{s+1}
$$

反轉換：

$$
v_C(t) = 10 - 5e^{-t} \text{ V}, \quad t \geq 0
$$

**驗證**：v_C(0) = 10 - 5 = 5V ✓，v_C(∞) = 10V ✓

答：**v_C(t) = 10 - 5e^{-t} V**。

## 七、部分分式展開（Partial Fraction Expansion）

### 7.1 為什麼需要？

反拉普拉斯轉換時，我們需要把複雜的 F(s) 拆成簡單的形式，好查轉換表。

### 7.2 情況一：不同實數根

$$
F(s) = \frac{N(s)}{(s+a)(s+b)} = \frac{A}{s+a} + \frac{B}{s+b}
$$

求係數方法（掩蓋法, Cover-up method）：

$$
A = (s+a)F(s)\big|_{s=-a}, \quad B = (s+b)F(s)\big|_{s=-b}
$$

### 7.3 情況二：重複根

$$
F(s) = \frac{N(s)}{(s+a)^2} = \frac{A}{s+a} + \frac{B}{(s+a)^2}
$$

B 可以用掩蓋法，A 需要微分或其他方法。

### 7.4 情況三：共軛複數根

$$
F(s) = \frac{N(s)}{(s+\alpha)^2 + \omega^2}
$$

直接轉換為 e^{-αt}(A cos ωt + B sin ωt)

### 7.5 數值例題 ⑤

> **題目**：求 F(s) = (3s+5)/[(s+1)(s+3)] 的反拉普拉斯轉換。

**解答**：

$$
F(s) = \frac{A}{s+1} + \frac{B}{s+3}
$$

掩蓋法：

$$
A = \frac{3(-1)+5}{(-1)+3} = \frac{2}{2} = 1
$$

$$
B = \frac{3(-3)+5}{(-3)+1} = \frac{-4}{-2} = 2
$$

$$
F(s) = \frac{1}{s+1} + \frac{2}{s+3}
$$

反轉換：

$$
f(t) = e^{-t} + 2e^{-3t}, \quad t \geq 0
$$

答：**f(t) = e^{-t} + 2e^{-3t}**。

## 八、初值定理與終值定理

### 8.1 初值定理

$$
f(0^+) = \lim_{s \to \infty} sF(s)
$$

### 8.2 終值定理

$$
f(\infty) = \lim_{s \to 0} sF(s)
$$

（前提：sF(s) 的所有極點都在左半平面，即系統穩定）

### 8.3 用途

不需要做完整的反轉換，就能直接知道初始值和最終值！

### 8.4 數值例題 ⑥

> **題目**：F(s) = 10(s+2)/[s(s+1)(s+5)]。用初值和終值定理驗證。

**解答**：

**初值**：

$$
f(0^+) = \lim_{s \to \infty} s \cdot \frac{10(s+2)}{s(s+1)(s+5)} = \lim_{s \to \infty} \frac{10(s+2)}{(s+1)(s+5)}
$$

$$
= \lim_{s \to \infty} \frac{10s}{s^2} = 0
$$

**終值**：

$$
f(\infty) = \lim_{s \to 0} s \cdot \frac{10(s+2)}{s(s+1)(s+5)} = \frac{10 \times 2}{1 \times 5} = 4
$$

答：**f(0⁺) = 0，f(∞) = 4**。

## 九、雙埠網路簡介

### 9.1 什麼是雙埠網路？

雙埠（Two-port）網路有四個端子：兩個輸入端（port 1）和兩個輸出端（port 2）。

```
     I₁ →        I₂ →
  +──────┐    ┌──────+
  V₁  [  雙埠  ]  V₂
  -──────┘    └──────-
```

變數：V₁、I₁（輸入端）和 V₂、I₂（輸出端）

### 9.2 四種參數矩陣

| 參數 | 方程式 | 名稱 | 適用場合 |
|------|--------|------|---------|
| Z 參數 | V = ZI | 阻抗參數 | T 型電路 |
| Y 參數 | I = YV | 導納參數 | π 型電路 |
| h 參數 | 混合 | 混合參數 | 電晶體 |
| ABCD 參數 | [V₁,I₁] = f(V₂,I₂) | 傳輸參數 | 串接電路 |

### 9.3 Z 參數（阻抗參數）

$$
\begin{bmatrix} V_1 \\ V_2 \end{bmatrix} = \begin{bmatrix} z_{11} & z_{12} \\ z_{21} & z_{22} \end{bmatrix} \begin{bmatrix} I_1 \\ I_2 \end{bmatrix}
$$

- z₁₁：port 2 開路時，port 1 的驅動點阻抗
- z₂₁：port 2 開路時，V₂/I₁（轉移阻抗）

### 9.4 Y 參數（導納參數）

$$
\begin{bmatrix} I_1 \\ I_2 \end{bmatrix} = \begin{bmatrix} y_{11} & y_{12} \\ y_{21} & y_{22} \end{bmatrix} \begin{bmatrix} V_1 \\ V_2 \end{bmatrix}
$$

### 9.5 ABCD 參數（傳輸參數）

$$
\begin{bmatrix} V_1 \\ I_1 \end{bmatrix} = \begin{bmatrix} A & B \\ C & D \end{bmatrix} \begin{bmatrix} V_2 \\ -I_2 \end{bmatrix}
$$

最大優點：串接多個雙埠網路時，ABCD 矩陣直接相乘！

### 9.6 數值例題 ⑦

> **題目**：一個 T 型電路，Z₁ = 2Ω（port 1 到中間），Z₃ = 3Ω（中間到地），Z₂ = 4Ω（中間到 port 2）。求 Z 參數。

**解答**：

T 型電路的 Z 參數有簡單公式：

$$
z_{11} = Z_1 + Z_3 = 2 + 3 = 5 \text{ Ω}
$$
$$
z_{12} = z_{21} = Z_3 = 3 \text{ Ω}
$$
$$
z_{22} = Z_2 + Z_3 = 4 + 3 = 7 \text{ Ω}
$$

$$
[Z] = \begin{bmatrix} 5 & 3 \\ 3 & 7 \end{bmatrix} \text{ Ω}
$$

答：z₁₁ = 5Ω，z₁₂ = z₂₁ = 3Ω，z₂₂ = 7Ω。

## 十、題型鑑別指南

| 看到什麼 | 用什麼方法 |
|----------|-----------|
| 「求轉移函數」 | 用分壓器或節點法在 s 域（或 jω 域）列式 |
| 「畫波德圖」 | 找出零點和極點 → 手繪近似 |
| 「低通/高通/帶通」 | 判斷 H(0) 和 H(∞) 的值 |
| 「共振頻率」、「Q 值」 | ω₀ = 1/√(LC)，Q = ω₀L/R |
| 「用拉普拉斯求 v(t)」 | 轉 s 域 → 代數運算 → 部分分式 → 反轉換 |
| 「初值/終值」 | 直接用定理，不需要反轉換 |
| 「雙埠網路參數」 | 依電路拓撲選擇 Z/Y/h/ABCD |
| 「串接兩個電路」 | 用 ABCD 矩陣相乘 |

| 分貝 | Decibel (dB) | 增益的對數尺度 | 20log₁₀|H| |
| 截止頻率 | Cutoff Frequency | 增益下降到 -3dB 的頻率 | ω_c = 1/RC |
| 低通濾波器 | Low-pass Filter | 只讓低頻通過 | RC 電路取 C 電壓 |
| 高通濾波器 | High-pass Filter | 只讓高頻通過 | RC 電路取 R 電壓 |
| 帶通濾波器 | Band-pass Filter | 只讓某頻帶通過 | RLC 串聯取 R |
| 帶拒濾波器 | Band-stop / Notch Filter | 擋掉某頻帶 | 除雜訊 |
| 共振頻率 | Resonant Frequency (ω₀) | 阻抗最小/電流最大的頻率 | 1/√(LC) |
| 品質因數 | Quality Factor (Q) | 共振尖銳度 | Q 高=選頻好 |
| 頻寬 | Bandwidth (BW) | -3dB 頻率範圍 | ω₀/Q |
| 拉普拉斯轉換 | Laplace Transform | 時域→s 域的數學工具 | 微分→乘 s |
| 部分分式 | Partial Fractions | 把複雜分式拆成簡單項 | 為了反轉換 |
| 初值定理 | Initial Value Theorem | 不反轉換直接求 f(0⁺) | lim s→∞ sF(s) |
| 終值定理 | Final Value Theorem | 不反轉換直接求 f(∞) | lim s→0 sF(s) |
| 雙埠網路 | Two-port Network | 有輸入和輸出端的電路模組 | 放大器、濾波器 |
| Z 參數 | Z-parameters | 用開路測量的阻抗矩陣 | T 型電路 |
| Y 參數 | Y-parameters | 用短路測量的導納矩陣 | π 型電路 |
| ABCD 參數 | ABCD / Transmission Parameters | 串接時矩陣相乘 | 傳輸線分析 |

## ✅ 自我檢測

### 題目 1
RC 低通濾波器中 R = 10kΩ，C = 0.1μF。求截止頻率 f_c。

<details>
<summary>點擊查看解答</summary>

ω_c = 1/(RC) = 1/(10⁴ × 10⁻⁷) = 1/10⁻³ = 1000 rad/s

f_c = ω_c/(2π) = 1000/6.283 ≈ **159.2 Hz**

</details>

### 題目 2
什麼是 -3dB？為什麼它這麼重要？

<details>
<summary>點擊查看解答</summary>

-3dB 對應增益倍數 1/√2 ≈ 0.707。

重要性：
- 功率 P ∝ V²，當電壓降為 1/√2 倍時，功率降為 (1/√2)² = 1/2
- 所以 -3dB = **半功率點**
- 截止頻率就是定義在 -3dB 的點

</details>

### 題目 3
求 F(s) = 12/[s(s+2)(s+6)] 的反拉普拉斯轉換。

<details>
<summary>點擊查看解答</summary>

部分分式：

$$F(s) = \frac{A}{s} + \frac{B}{s+2} + \frac{C}{s+6}$$

A = 12/(2×6) = 1
B = 12/[(-2)(-2+6)] = 12/(-8) = -3/2
C = 12/[(-6)(-6+2)] = 12/(24) = 1/2

$$f(t) = 1 - 1.5e^{-2t} + 0.5e^{-6t}, \quad t \geq 0$$

驗算（終值定理）：f(∞) = 1 ✓（lim s→0 sF(s) = 12/(2×6) = 1）

</details>

### 題目 4
H(jω) = 1/(1 + jω/100)。在 ω = 100 時，增益是多少 dB？相位是多少？

<details>
<summary>點擊查看解答</summary>

ω = 100 時：H = 1/(1+j) = 1/(√2 ∠45°) = (1/√2)∠-45°

增益：20log(1/√2) = 20 × (-0.5 × log2) = 20 × (-0.1505) = **-3.01 dB ≈ -3 dB**

相位：**-45°**

這正是截止頻率的定義！

</details>

### 題目 5
用初值定理和終值定理求 F(s) = 5(s+3)/[s(s+1)(s+4)] 的 f(0⁺) 和 f(∞)。

<details>
<summary>點擊查看解答</summary>

**初值**：
$$f(0^+) = \lim_{s\to\infty} sF(s) = \lim_{s\to\infty} \frac{5s(s+3)}{s(s+1)(s+4)} = \lim_{s\to\infty} \frac{5(s+3)}{(s+1)(s+4)}$$
$$= \lim_{s\to\infty} \frac{5s}{s^2} = 0$$

**終值**：
$$f(\infty) = \lim_{s\to 0} sF(s) = \frac{5 \times 3}{1 \times 4} = \frac{15}{4} = 3.75$$

答：**f(0⁺) = 0，f(∞) = 3.75**

</details>
