## 伍、嚴柏翰的情感崩潰模型

```
每恢復一段記憶 = 一次微型五階段循環：

3. 無力（取代憤怒）：無法恨邱韻如——她是為了保護他
4. 沉默（取代抑鬱）：不是不說話，是找不到語言

最終的「接納」不是放下——是「帶著傷痛繼續活」。
```

## 陸、寫作指引

### 語言 DNA
- **軍事簡報式**：短句。不用形容詞。結論在前。
- **沉默 > 語言**：他最有力的場景都是無台詞的。
- **重複詞 = 崩潰信號**：正常時每句不同；崩潰時開始重複。
  - ch149：「三樣……三樣就好」——重複 = 他在壓制自己
- 超過三句話就檢查：他是不是說太多了？

### 與邱韻如的寫法規則
1. ❌ 不要讓他們在卷二正面見面——「帶我去見她」是 Vol 3 的事
2. ❌ 不要讓嚴柏翰說「我原諒她」——他理解但不原諒。這兩者不同。
3. ❌ 不要讓他在 ch149 選擇時猶豫太久——他是一個果斷的人。猶豫 = 出戲。
4. ✅ 每次提到邱韻如，他的身體都要有反應（手、心率、疤痕）
5. ✅ 茉莉花每卷出現不超過 3 次——稀缺 = 力量

### 與維倫的對手戲規則
- 兩個沉默的男人不要試圖讓他們傾訴
- 最好的友誼場景：並排站著，什麼都不說
- ch149「你是這十年裡唯一真的東西」——這句之後不要接維倫的反應。讓沉默留著。
