## 題目 S2.1（葉丙成 101-2 期末考）

**原始故事背景**：老葉建議魯君做家長調查，若七成以上家長覺得曾參是好人就聘用他。各家長獨立不受其他家長影響，軍士調查誤差百分之三以內。

**數學語言重述**：設母體比例 $p$ 為認為曾參是好人的家長比例。欲檢定 $p \geq 0.7$。要求在誤差 $E = 0.03$ 以內、信心水準至少 90% 的條件下，求所需最小樣本數 $n$。

### 詳解

這是**比例的信賴區間 / 樣本數決定**問題。

在最保守情況下（$p$ 未知時取 $p = 0.5$ 使變異數最大，或用 $p = 0.7$ 因為我們在 0.7 附近做檢定），信賴區間的半寬為：

$$E = z_{\alpha/2} \sqrt{\frac{p(1-p)}{n}}$$

90% 信心 $\Rightarrow z_{0.05} = 1.645$，取 $p = 0.7$（在 $H_0$ 邊界處）：

$$0.03 = 1.645 \sqrt{\frac{0.7 \times 0.3}{n}}$$

$$0.03 = 1.645 \sqrt{\frac{0.21}{n}}$$

$$\sqrt{\frac{0.21}{n}} = \frac{0.03}{1.645} = 0.01824$$

$$\frac{0.21}{n} = 0.000333$$

$$n = \frac{0.21}{0.000333} = 630.7$$

取上整數：

$$\boxed{n = 631}$$

若取最保守的 $p = 0.5$：$n = \frac{(1.645)^2 \times 0.25}{0.03^2} = \frac{0.6765}{0.0009} = 752$。

**解題思路**：此題的關鍵是利用信賴區間公式反求樣本數。$p = 0.7$ 是較合理的估計值（因為要判斷是否達到七成），若用最保守估計 $p=0.5$ 則需要更多樣本。在考試中，兩種假設皆可接受，建議說明所用的 $p$ 值。

## 題目 S2.2（葉丙成 101-2 期末考）

**原始故事背景**：魯君最多只能探詢千戶家長，詢問後恰恰六成人認為曾參是好人，誤差正負百分之三。問信心度為多少？

**數學語言重述**：$n = 1000$，$\hat{p} = 0.6$，誤差 $E = 0.03$。反求信心水準 $1 - \alpha$。

### 詳解

由信賴區間公式：

$$E = z_{\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}}$$

代入已知值：

$$0.03 = z_{\alpha/2} \sqrt{\frac{0.6 \times 0.4}{1000}} = z_{\alpha/2} \sqrt{\frac{0.24}{1000}} = z_{\alpha/2} \times 0.01549$$

$$z_{\alpha/2} = \frac{0.03}{0.01549} = 1.937$$

查標準常態分布表：

$$\Phi(1.937) \approx 0.9736$$

$$\alpha/2 = 1 - 0.9736 = 0.0264$$

$$1 - \alpha = 1 - 2 \times 0.0264 = 0.9472$$

$$\boxed{\text{信心度} \approx 94.7\%}$$

**解題思路**：這是上一題的反問。已知樣本數和誤差，反推 $z$ 值，再從標準常態分布表查出對應的信心水準。注意 $\hat{p} = 0.6$ 而非假設的 0.7，因為這是實際觀察到的比例。

## 題目 S2.3（葉丙成 101-2 期末考）

**原始故事背景**：東晉軍中共有 1000 名世家子弟，每人戰後存活機率為 0.5（獨立），每人收集讚數 $L_i \sim \text{Uniform}\{1, 2, \ldots, 200\}$（彼此獨立）。謝安戰後每讚發一兩金。求黃金總兩數之 MGF。

**數學語言重述**：有 $n = 1000$ 人，每人存活的機率為 $p = 0.5$（獨立）。第 $i$ 人若存活，貢獻金額 $L_i \sim \text{Uniform}\{1, 2, \ldots, 200\}$，若不存活則貢獻 0。$L_i$ 彼此獨立，與存活與否也獨立。求總金額 $S = \sum_{i=1}^{1000} L_i \cdot B_i$ 的 MGF，其中 $B_i \sim \text{Bernoulli}(0.5)$。

### 詳解

**第一步：求單人貢獻的 MGF**

設 $Z_i = L_i \cdot B_i$ 為第 $i$ 人的貢獻。

$$M_{Z_i}(t) = E[e^{tZ_i}] = P(B_i = 0) \cdot e^{t \cdot 0} + P(B_i = 1) \cdot E[e^{tL_i}]$$

$$= 0.5 + 0.5 \cdot M_{L_i}(t)$$

其中 $L_i \sim \text{Uniform}\{1, 2, \ldots, 200\}$ 的 MGF 為：

$$M_{L_i}(t) = \frac{1}{200} \sum_{k=1}^{200} e^{kt} = \frac{e^t}{200} \cdot \frac{1 - e^{200t}}{1 - e^t} = \frac{e^t(1 - e^{200t})}{200(1 - e^t)}$$

因此單人 MGF：

$$M_{Z_i}(t) = \frac{1}{2} + \frac{1}{2} \cdot \frac{e^t(1 - e^{200t})}{200(1 - e^t)}$$

**第二步：求總金額 MGF**

由於 $Z_1, Z_2, \ldots, Z_{1000}$ 彼此獨立且同分布：

$$\boxed{M_S(t) = \left[\frac{1}{2} + \frac{e^t(1 - e^{200t})}{400(1 - e^t)}\right]^{1000}}$$

**解題思路**：此題的核心是 **compound sum（複合和）** 的 MGF。每個人的貢獻是一個「帶 Bernoulli 開關」的隨機變數，先求單人 MGF（利用全機率公式分 $B_i=0$ 和 $B_i=1$ 兩種情況），再由獨立性取乘積。

## 題目 S2.4（葉丙成 101-2 期末考）

**原始故事背景**：承 S2.3，求犒賞黃金總兩數的期望值與標準差。

**數學語言重述**：承上題，$S = \sum_{i=1}^{1000} L_i \cdot B_i$，其中 $B_i \sim \text{Bernoulli}(0.5)$，$L_i \sim \text{Uniform}\{1, \ldots, 200\}$，全部獨立。求 $E[S]$ 與 $\text{SD}(S)$。

### 詳解

**求單人期望值：**

$$E[Z_i] = E[L_i \cdot B_i] = E[L_i] \cdot E[B_i] = \frac{1 + 200}{2} \times 0.5 = 100.5 \times 0.5 = 50.25$$

**求單人二階矩：**

$$E[Z_i^2] = E[L_i^2 \cdot B_i^2] = E[L_i^2] \cdot E[B_i^2]$$

注意 $B_i^2 = B_i$（因為 $B_i \in \{0,1\}$），所以 $E[B_i^2] = E[B_i] = 0.5$。

$$E[L_i^2] = \frac{1}{200}\sum_{k=1}^{200} k^2 = \frac{1}{200} \cdot \frac{200 \times 201 \times 401}{6} = \frac{201 \times 401}{6} = \frac{80601}{6} = 13433.5$$

$$E[Z_i^2] = 13433.5 \times 0.5 = 6716.75$$

**單人變異數：**

$$\text{Var}(Z_i) = E[Z_i^2] - (E[Z_i])^2 = 6716.75 - 50.25^2 = 6716.75 - 2525.0625 = 4191.6875$$

**總金額：**

$$E[S] = 1000 \times 50.25 = \boxed{50250}$$

$$\text{Var}(S) = 1000 \times 4191.6875 = 4191687.5$$

$$\text{SD}(S) = \sqrt{4191687.5} \approx \boxed{2047.4}$$

**解題思路**：由於所有 $Z_i$ 獨立同分布，總和的期望值和變異數直接用線性性質求得。關鍵是正確計算 $E[Z_i^2]$ 時，利用 $B_i^2 = B_i$ 這個 Bernoulli 變數的特性來簡化。

## 題目 S2.5（葉丙成 101-2 期末考）

**原始故事背景**：瓊博和老葉投注賠率 $W$ = 酒仙大賽贏家飲酒量 / 輸家飲酒量。問 $W$ 的機率分布。

**數學語言重述**：$X \sim \text{Exp}(1/2)$（$\lambda = 1/2$，均值 2），$Y \sim \text{Exp}(1)$（$\lambda = 1$，均值 1），$X, Y$ 獨立。定義

$$W = \frac{\max(X, Y)}{\min(X, Y)}$$

求 $W$ 的機率分布（PDF）。

### 詳解

**方法：條件分解**

注意 $W \geq 1$。我們分 $X > Y$ 和 $Y > X$ 兩種情況。

先求 $W$ 的 CDF，$P(W \leq w)$ for $w \geq 1$。

$$P(W \leq w) = P\left(\frac{\max(X,Y)}{\min(X,Y)} \leq w\right) = 1 - P(W > w)$$

$$P(W > w) = P\left(\frac{X}{Y} > w, X > Y\right) + P\left(\frac{Y}{X} > w, Y > X\right)$$

**Case 1：$X > Y$，此時 $W = X/Y$**

$$P\left(\frac{X}{Y} > w, X > Y\right) = P(X > wY) \quad (\text{若 } w \geq 1 \text{ 則 } X > wY \Rightarrow X > Y \text{ 自動成立})$$

$$= \int_0^\infty \int_{wy}^\infty \frac{1}{2}e^{-x/2} \cdot e^{-y} \, dx \, dy = \int_0^\infty e^{-y} \cdot e^{-wy/2} \, dy = \int_0^\infty e^{-y(1+w/2)} \, dy = \frac{1}{1 + w/2} = \frac{2}{2+w}$$

**Case 2：$Y > X$，此時 $W = Y/X$**

$$P\left(\frac{Y}{X} > w, Y > X\right) = P(Y > wX) \quad (\text{若 } w \geq 1 \text{ 則 } Y > wX \Rightarrow Y > X \text{ 自動成立})$$

$$= \int_0^\infty \int_{wx}^\infty \frac{1}{2}e^{-x/2} \cdot e^{-y} \, dy \, dx = \int_0^\infty \frac{1}{2}e^{-x/2} \cdot e^{-wx} \, dx = \frac{1}{2}\int_0^\infty e^{-x(1/2+w)} \, dx = \frac{1}{2} \cdot \frac{1}{1/2+w} = \frac{1}{1+2w}$$

因此：

$$P(W > w) = \frac{2}{2+w} + \frac{1}{1+2w}, \quad w \geq 1$$

$$F_W(w) = 1 - \frac{2}{2+w} - \frac{1}{1+2w}$$

取導數得 PDF：

$$f_W(w) = \frac{d}{dw}\left[1 - \frac{2}{2+w} - \frac{1}{1+2w}\right] = \frac{2}{(2+w)^2} + \frac{2}{(1+2w)^2}$$

$$\boxed{f_W(w) = \frac{2}{(2+w)^2} + \frac{2}{(1+2w)^2}, \quad w \geq 1}$$

**驗證**：$\int_1^\infty f_W(w)\,dw = \left[-\frac{2}{2+w} - \frac{1}{1+2w}\right]_1^\infty = 0 - \left(-\frac{2}{3} - \frac{1}{3}\right) = 1$ \checkmark

**解題思路**：$W = \max/\min$ 的分布需要分「誰大誰小」兩種情況。由於 $X, Y$ 有不同的指數分布參數，兩種情況不對稱，需分別計算。利用互補 CDF $P(W > w)$ 較方便積分，最後對 $w$ 微分得 PDF。

## 題目 S2.6（陳宏 103-2 期末考）

$X_1, X_2$ i.i.d.，密度 $f(x) = 3x^{-4}$, $x > 1$。令 $Y = X_2 / X_1$。對 $y \geq 1$，求 $P(Y \geq y)$。

### 詳解

**第一步：求 CDF**

$$F(x) = \int_1^x 3t^{-4}\,dt = \left[-t^{-3}\right]_1^x = 1 - x^{-3}, \quad x \geq 1$$

**第二步：計算 $P(Y \geq y)$ for $y \geq 1$**

$$P\left(\frac{X_2}{X_1} \geq y\right) = P(X_2 \geq yX_1)$$

由於 $X_1, X_2$ 獨立：

$$= \int_1^\infty P(X_2 \geq yx_1) \cdot f(x_1)\,dx_1 = \int_1^\infty [1 - F(yx_1)] \cdot 3x_1^{-4}\,dx_1$$

其中 $F(yx_1) = 1 - (yx_1)^{-3}$（當 $yx_1 \geq 1$，即 $x_1 \geq 1/y$；由於 $x_1 \geq 1$ 且 $y \geq 1$，此條件自動成立）。

$$= \int_1^\infty (yx_1)^{-3} \cdot 3x_1^{-4}\,dx_1 = \frac{3}{y^3}\int_1^\infty x_1^{-7}\,dx_1$$

$$= \frac{3}{y^3} \cdot \left[-\frac{x_1^{-6}}{6}\right]_1^\infty = \frac{3}{y^3} \cdot \frac{1}{6} = \frac{1}{2y^3}$$

$$\boxed{P(Y \geq y) = \frac{1}{2y^3}, \quad y \geq 1}$$

**驗證**：$P(Y \geq 1) = 1/2$，這與對稱性一致：$P(X_2 \geq X_1) = 1/2$（因 $X_1, X_2$ i.i.d.）。

**延伸**：$Y$ 的 PDF 為（$y \geq 1$）：

$$f_Y(y) = -\frac{d}{dy}\left(\frac{1}{2y^3}\right) = \frac{3}{2y^4}, \quad y \geq 1$$

由對稱性，完整的 PDF 為：$f_Y(y) = \frac{3}{2}\frac{1}{\max(y, 1/y)^4} \cdot \frac{1}{y^2}$... 不過本題只問 $y \geq 1$ 的部分，上述結果已足夠。

**解題思路**：由於兩個隨機變數都只取 $x > 1$ 的值，且 $y \geq 1$，積分下界自然是 1。利用互補 CDF $1 - F(yx_1) = (yx_1)^{-3}$ 可使積分大幅簡化。

## 題目 S2.7（陳宏 103-2 期末考）

$X, Y$ 為獨立隨機變數，MGF 分別為 $M_X(t) = (1 - 2t)^{-2}$，$M_Y(t) = (1 - 2t)^{-3}$。求 $\text{Var}(X + Y)$。

### 詳解

**第一步：辨認分布**

Gamma($\alpha, \lambda$) 的 MGF 為 $\left(\frac{\lambda}{\lambda - t}\right)^\alpha = (1 - t/\lambda)^{-\alpha}$。

比較 $M_X(t) = (1 - 2t)^{-2}$：令 $1/\lambda = 2$，即 $\lambda = 1/2$，$\alpha = 2$。

$$X \sim \text{Gamma}(\alpha=2, \lambda=1/2) \implies E[X] = \frac{\alpha}{\lambda} = 4, \quad \text{Var}(X) = \frac{\alpha}{\lambda^2} = \frac{2}{1/4} = 8$$

同理 $M_Y(t) = (1 - 2t)^{-3}$：$\alpha = 3, \lambda = 1/2$。

$$Y \sim \text{Gamma}(\alpha=3, \lambda=1/2) \implies E[Y] = 6, \quad \text{Var}(Y) = \frac{3}{1/4} = 12$$

**第二步：由獨立性**

$$\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y) = 8 + 12 = \boxed{20}$$

**替代方法（直接從 MGF 求矩）：**

$M_{X+Y}(t) = M_X(t) \cdot M_Y(t) = (1-2t)^{-5}$（因為獨立）。

$$M'(t) = 10(1-2t)^{-6}, \quad E[X+Y] = M'(0) = 10$$

$$M''(t) = 120(1-2t)^{-7}, \quad E[(X+Y)^2] = M''(0) = 120$$

$$\text{Var}(X+Y) = 120 - 10^2 = \boxed{20}$$

**解題思路**：本題有兩條路：(1) 辨認 Gamma 分布直接用公式；(2) 對 MGF 微分求矩。兩者殊途同歸。辨認分布的方法更快，但微分法更通用。注意 Gamma 分布的參數化方式（rate vs. scale）需小心比對。

## 題目 S2.8（陳宏 103-2 期末考）

在某個非常不穩定的產業中，公司倒閉率為每年 2.5 家，假設倒閉發生為 Poisson process。

(a) 求明年有 4 家或以上倒閉的機率。
(b) 求下兩次倒閉發生在彼此三個月之內的機率。

### 詳解

**(a)** 令 $N \sim \text{Poisson}(\lambda = 2.5)$（一年內倒閉數）。

$$P(N \geq 4) = 1 - P(N \leq 3) = 1 - \sum_{k=0}^{3} \frac{e^{-2.5} \cdot 2.5^k}{k!}$$

逐項計算：
- $P(N=0) = e^{-2.5} = 0.0821$
- $P(N=1) = 2.5 \cdot e^{-2.5} = 0.2052$
- $P(N=2) = \frac{2.5^2}{2} \cdot e^{-2.5} = \frac{6.25}{2} \cdot 0.0821 = 0.2565$
- $P(N=3) = \frac{2.5^3}{6} \cdot e^{-2.5} = \frac{15.625}{6} \cdot 0.0821 = 0.2138$

$$P(N \leq 3) = 0.0821 + 0.2052 + 0.2565 + 0.2138 = 0.7576$$

$$\boxed{P(N \geq 4) = 1 - 0.7576 = 0.2424}$$

**(b)** 在 Poisson process 中，相鄰事件間的時間間隔 $T_i \sim \text{Exp}(\lambda = 2.5 \text{/year})$。

「下兩次倒閉發生在彼此三個月之內」意即：設 $T_1$ 為從現在到下一次倒閉的時間，$T_2$ 為第一次到第二次倒閉之間的時間。要求 $T_2 \leq 0.25$ 年（三個月）。

注意：$T_1$（從任意時刻到下一事件的等待時間）在 Poisson process 中也服從 $\text{Exp}(2.5)$（memoryless property）。而「下兩次倒閉在彼此三個月之內」可理解為第一次和第二次倒閉的間隔 $T_2 \leq 0.25$。

$$P(T_2 \leq 0.25) = 1 - e^{-2.5 \times 0.25} = 1 - e^{-0.625}$$

$$= 1 - 0.5353 = \boxed{0.4647}$$

**解題思路**：(a) 是基本的 Poisson 機率計算，由互補事件 $P(N \geq 4) = 1 - P(N \leq 3)$ 入手。(b) 的關鍵是理解 Poisson process 中相鄰事件間的間隔為 i.i.d. 指數分布。「下兩次在三個月之內」指的是兩次倒閉之間的間隔（interarrival time），而不是從現在起三個月內發生兩次。

## 題目 S2.9（陳宏 103-2 期末考）

一個非負隨機變數 $X$，均值 100，變異數 100。

(a) Markov 不等式對 $P(X \geq 400)$ 給出什麼上界？
(b) Chebyshev 不等式對 $P(X \geq 400)$ 給出什麼上界？
(c) 令 $S_n$ 為 $n$ 個獨立同分布（與 $X$ 同分布）隨機變數之和。求一個數列 $x_n$，使得 $P(S_n / n > 100 + x_n) \to 1/4$ 當 $n \to \infty$。

### 詳解

**(a) Markov 不等式**

對非負隨機變數 $X$：

$$P(X \geq a) \leq \frac{E[X]}{a}$$

$$P(X \geq 400) \leq \frac{100}{400} = \boxed{\frac{1}{4} = 0.25}$$

**(b) Chebyshev 不等式**

$$P(|X - \mu| \geq k) \leq \frac{\sigma^2}{k^2}$$

$P(X \geq 400) \leq P(|X - 100| \geq 300)$（因為 $X \geq 400 \Rightarrow |X-100| \geq 300$）

$$P(X \geq 400) \leq P(|X - 100| \geq 300) \leq \frac{100}{300^2} = \frac{100}{90000} = \boxed{\frac{1}{900} \approx 0.00111}$$

Chebyshev 給出的上界（$1/900$）遠比 Markov（$1/4$）更緊。

**(c) 利用中央極限定理**

$\bar{X}_n = S_n/n$，$E[\bar{X}_n] = 100$，$\text{Var}(\bar{X}_n) = 100/n$，$\text{SD}(\bar{X}_n) = 10/\sqrt{n}$。

由 CLT：

$$\frac{\bar{X}_n - 100}{10/\sqrt{n}} \xrightarrow{d} Z \sim N(0,1)$$

要求：

$$P\left(\bar{X}_n > 100 + x_n\right) = P\left(\frac{\bar{X}_n - 100}{10/\sqrt{n}} > \frac{x_n}{10/\sqrt{n}}\right) \to \frac{1}{4}$$

$$P(Z > z) = 1/4 \implies z = \Phi^{-1}(3/4) \approx 0.6745$$

因此需要：

$$\frac{x_n}{10/\sqrt{n}} = 0.6745 \implies x_n = \frac{6.745}{\sqrt{n}}$$

$$\boxed{x_n = \frac{10 \cdot \Phi^{-1}(3/4)}{\sqrt{n}} \approx \frac{6.745}{\sqrt{n}}}$$

**解題思路**：(a)(b) 是直接套用不等式。(c) 要求的是 CLT 的應用——既然 $\bar{X}_n$ 漸近常態，我們設定偏移量 $x_n$ 使得標準化後的分位數恰好是 $\Phi^{-1}(3/4)$。$x_n$ 必須以 $1/\sqrt{n}$ 的速率趨於 0，才能讓尾部機率收斂到固定值。

## 題目 S2.10（劉長遠 103-2 期末考）

保險公司以保費 \$400 出售保額 \$10,000 的一年期汽車保險。每個投保人一年內發生事故的機率為 $1/10$。若公司賣出 100 張保單，求公司年末虧損的機率。

### 詳解

**建模**：設 $X_i$ 為第 $i$ 位投保人造成的理賠（$X_i = 10000$ 的機率 $1/10$，$X_i = 0$ 的機率 $9/10$）。

公司收入 = $100 \times 400 = 40000$。

公司理賠總額 = $S = \sum_{i=1}^{100} X_i$。

虧損 $\Leftrightarrow S > 40000$。

**參數計算**：

$$E[X_i] = 10000 \times 0.1 = 1000$$

$$E[X_i^2] = 10000^2 \times 0.1 = 10^7$$

$$\text{Var}(X_i) = 10^7 - 1000^2 = 10^7 - 10^6 = 9 \times 10^6$$

$$E[S] = 100 \times 1000 = 100000$$

$$\text{Var}(S) = 100 \times 9 \times 10^6 = 9 \times 10^8$$

$$\sigma_S = 30000$$

**等價表述**：虧損 $\Leftrightarrow S > 40000$。

但注意 $E[S] = 100000 > 40000$，我們需要 $P(S > 40000)$...

等一下，重新理解：公司收入 40000，理賠支出 $S$。

$$\text{利潤} = 40000 - S$$

$$P(\text{虧損}) = P(S > 40000)$$

等價於看多少人出事：設 $N = $ 出事人數 $\sim \text{Bin}(100, 0.1)$，$S = 10000N$。

$$P(S > 40000) = P(N > 4) = 1 - P(N \leq 4)$$

**CLT 近似**：$N \sim \text{Bin}(100, 0.1)$，$E[N] = 10$，$\text{Var}(N) = 9$，$\sigma_N = 3$。

$$P(N > 4) = P\left(\frac{N - 10}{3} > \frac{4 - 10}{3}\right) = P(Z > -2) = \Phi(2) \approx 0.9772$$

使用連續性修正：

$$P(N > 4) = P(N \geq 5) \approx P\left(Z > \frac{4.5 - 10}{3}\right) = P(Z > -1.833) = \Phi(1.833) \approx 0.9666$$

$$\boxed{P(\text{虧損}) \approx 1 - \Phi(-2) = \Phi(2) \approx 0.9772}$$

即公司虧損的機率非常高（約 97.7%）。這意味著 \$400 的保費對於 \$10,000 的保額和 10% 的事故率來說定價太低了（期望理賠 = \$1000 > 保費 \$400）。

**解題思路**：先建立模型，發現這本質上是 Binomial 問題。由於期望理賠遠超保費收入，公司虧損的機率自然非常高。用 CLT 將 Binomial 近似為 Normal 來計算。

## 題目 S2.11（劉長遠 103-2 期末考）

個別學生成績 $\sim N(\mu, 20^2)$，$\mu$ 為 70 或 80。虛無假設 $H_0: \mu = 80$，統計量為單一成績值，顯著水準 $\alpha = 0.01$。

(a) 應使用右尾、左尾、還是雙尾檢定？
(b) 求 $H_0$ 被錯誤接受的條件機率（Type II error）。
(c) 求 $H_0$ 被錯誤拒絕的條件機率（Type I error）。

### 詳解

**(a)** 對立假設 $H_1: \mu = 70 < 80$，值較小，所以應使用**左尾檢定（left-sided test）**。

理由：若 $\mu$ 實際上是 70，我們會觀察到較小的成績值。因此拒絕域應在分布的左尾。

**(b) Type II Error：$H_0$ 為假（真 $\mu = 70$）時，未能拒絕 $H_0$ 的機率。**

左尾檢定在 $\alpha = 0.01$ 下的臨界值：

$$x_c = 80 + z_{0.01} \times 20 = 80 + (-2.326) \times 20 = 80 - 46.52 = 33.48$$

（拒絕域：$X < 33.48$；接受域：$X \geq 33.48$）

當 $\mu = 70$ 時，$X \sim N(70, 400)$：

$$P(\text{Type II}) = P(X \geq 33.48 \mid \mu = 70) = P\left(Z \geq \frac{33.48 - 70}{20}\right) = P(Z \geq -1.826)$$

$$= \Phi(1.826) \approx \boxed{0.9661}$$

Type II Error 很高，因為 $\sigma = 20$ 很大，而 $\mu$ 的差距只有 10，以單一觀測值很難區分。

**(c) Type I Error：$H_0$ 為真（真 $\mu = 80$）時，錯誤拒絕 $H_0$ 的機率。**

這就是顯著水準本身：

$$\boxed{P(\text{Type I}) = \alpha = 0.01}$$

**解題思路**：(a) 看對立假設的方向——$H_1$ 的值比 $H_0$ 小，所以用左尾。(b) Type II error 需要在 $H_1$ 的分布下計算落入接受域的機率。(c) Type I error 就是 $\alpha$ 本身，這是設計檢定時已經固定的。本題的教訓是：樣本量太小（只有 1 個觀測值）時，Type II error 會非常高。

## 題目 S2.12（劉長遠 103-2 期末考）

母體密度為 $f(x) = \frac{1 + ax}{2}$，$-1 \leq x \leq 1$，其中 $-1 \leq a \leq 1$。估計量 $\hat{a} = 3\bar{X}$（$\bar{X}$ 為樣本平均）。問 $\hat{a}$ 是否為 $a$ 的不偏估計量？

### 詳解

**需驗證 $E[\hat{a}] = a$ 是否成立。**

$$E[\hat{a}] = 3E[\bar{X}] = 3E[X]$$

計算 $E[X]$：

$$E[X] = \int_{-1}^{1} x \cdot \frac{1 + ax}{2} \, dx = \frac{1}{2}\int_{-1}^{1} (x + ax^2) \, dx$$

$$= \frac{1}{2}\left[\int_{-1}^{1} x \, dx + a\int_{-1}^{1} x^2 \, dx\right]$$

$$\int_{-1}^{1} x \, dx = 0 \quad \text{（奇函數在對稱區間上積分為零）}$$

$$\int_{-1}^{1} x^2 \, dx = \frac{2}{3}$$

$$E[X] = \frac{1}{2}\left[0 + a \cdot \frac{2}{3}\right] = \frac{a}{3}$$

因此：

$$E[\hat{a}] = 3 \times \frac{a}{3} = a$$

$$\boxed{\hat{a} = 3\bar{X} \text{ 是 } a \text{ 的不偏估計量（unbiased estimator）。}}$$

**解題思路**：判斷不偏性只需驗證 $E[\hat{a}] = a$。先算出 $E[X]$，利用奇偶函數性質簡化積分。$\int_{-1}^{1} x\,dx = 0$（奇函數），只剩 $ax^2$ 項。最終 $E[X] = a/3$，所以 $3\bar{X}$ 的期望值恰好等於 $a$。
