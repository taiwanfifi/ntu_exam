# 備考指南：從零到上榜 — 非法律人的國考完全攻略

> 這份指南綜合了數十位 PTT 上榜者的真實經驗，專為非法律背景的轉職者重新編寫。
> 不管你現在是工程師、護理師、業務、設計師，還是任何其他職業——這份指南會告訴你，如何從完全不懂法律，走到通過律師或司法官考試。

---

## 第一章：在開始之前，你需要知道的事

### 1.1 這場考試到底在考什麼？

很多人以為法律考試就是「背法條」。錯。

法律考試考的是**你能不能像法律人一樣思考**。

想像一下這個場景：

> 小明在路上被一台闖紅燈的車撞了，送醫後發現骨折。他想要求賠償。

一般人的反應：「對方闖紅燈撞人，當然要賠啊！」

法律人的反應：
1. 小明對駕駛有什麼**請求權**？（侵權行為損害賠償？）
2. **構成要件**都滿足嗎？（加害行為、違法性、故意過失、因果關係、損害）
3. 對方有沒有可以**抗辯**的理由？（小明自己有沒有過失？）
4. 可以請求賠償**哪些項目**？（醫療費、工作損失、精神慰撫金？）
5. 除了告駕駛，還可以告**誰**？（車主？僱用人？）
6. 有沒有**時效**問題？（什麼時候之前必須提告？）

看到差別了嗎？法律人不是知道更多知識，而是**看事情的角度不同**。他們會系統性地拆解問題，找出每一個法律上的爭點，然後逐一分析。

**這就是我們要教會你的能力。**

### 1.2 考試的真實面貌

#### 一試（第一關）：選擇題海戰
- 600 分，全部選擇題
- 一天考完，4 個科目卷
- 錄取標準：全體到考者前 33%
- 性質：**純篩選**，分數不計入最終成績
- 策略：不需要滿分，只需要穩穩地在前三分之一

#### 二試（第二關）：申論題才是真正的決戰
- 律師 1000 分 / 司法官 900 分
- 兩天考完，全部申論題（寫作文式的法律分析）
- 考場會發法條，不用死背法條號碼
- 考的是：你能不能看到案例就拆解出法律問題，然後用法律的語言寫出分析

#### 三試（司法官限定）：口試
- 100 分，考儀態、溝通、人格、才識、應變
- 60 分及格

### 1.3 非法律人的優勢與劣勢

**你的劣勢：**
- 沒有法律基礎，起步比法律系晚
- 不熟悉法律的思考方式和寫作格式
- 可能不知道去哪裡找資源

**你的優勢（認真的）：**
- **動機更強烈**：轉職者通常比「被家長逼去念法律」的人更有決心
- **社會經驗更豐富**：很多法律案例你在職場上親身經歷過，理解起來比大學生更深刻
- **不同領域的專業**：如果你選智慧財產法（適合工程師）、勞動社會法（適合人資/社工）、財稅法（適合會計/金融），你的原本專業就是加分項
- **學習效率**：成年人有目標導向的學習能力，不會像大學生一樣隨便讀讀

PTT 上有大量非本科上榜的案例。他們不是天才，而是找對方法。

## 第二章：準備時程規劃

### 2.1 整體時程：需要多久？

根據 PTT 上榜者的經驗統計：

| 背景 | 建議準備時間 | 每日讀書時間 |
|------|-------------|-------------|
| 法律系畢業 | 6-12 個月 | 6-8 小時 |
| 非法律系，全職準備 | 12-18 個月 | 8-10 小時 |
| 非法律系，在職準備 | 18-24 個月 | 4-6 小時 |

重要提醒：這不是「讀完一遍就好」的考試。上榜者平均把核心科目讀 3-5 遍。

### 2.2 四階段準備法（適用全職考生，18 個月版本）

#### 第一階段：打地基（第 1-4 個月）

**目標：建立法律思維，理解基本框架**

這個階段最重要的事：**不要急著做考古題。** 先把基本概念搞懂。

順序建議：
1. **民法總則 + 債編總論**（4-6 週）— 這是所有法律的基礎中的基礎
2. **刑法總則**（3-4 週）— 建立犯罪分析的基本框架
3. **憲法**（2-3 週）— 建立公法體系的基礎
4. **行政法**（3-4 週）— 公法最常考的科目

怎麼讀：
- 用我們的教學講義，從第一頁開始讀
- 每讀完一個章節，用自己的話把概念解釋一遍（費曼學習法）
- 遇到不懂的，不要跳過，反覆讀到懂
- 這個階段**不需要**做題目，專注在「理解」

每日時間分配（8 小時版本）：
- 上午 3 小時：新進度（讀新章節）
- 下午 3 小時：新進度或整理筆記
- 晚上 2 小時：複習今天和前幾天的內容

#### 第二階段：蓋牆壁（第 5-10 個月）

**目標：完成所有科目的第一輪，開始做題目**

這個階段要把所有考試科目都讀過一遍，同時開始用考古題檢驗自己。

順序建議：
1. **民法（債編各論 + 物權 + 親屬繼承）**（4 週）
2. **刑法分則**（3 週）
3. **民事訴訟法**（4 週）— 很多人覺得最難，但極度重要
4. **刑事訴訟法**（3 週）
5. **公司法**（2 週）
6. **保險法**（1-2 週）
7. **證券交易法**（1-2 週）
8. **選試科目**（2-3 週）

同時開始：
- 每週做 2-3 題考古題申論（先從簡單的開始）
- 每週做一回一試的選擇題練習
- 建立自己的「爭點筆記」— 每個科目整理出最重要的 20-30 個爭點

每日時間分配：
- 上午 3 小時：新進度
- 下午 2 小時：做考古題（限時寫完）
- 下午 1 小時：對答案、檢討
- 晚上 2 小時：複習舊科目 + 整理筆記

#### 第三階段：裝修（第 11-14 個月）

**目標：第二輪複習 + 大量做題**

這個階段是「把讀過的東西變成你的」。

核心任務：
1. 所有科目讀第二遍（這次會快很多，約原來速度的 1/3）
2. **每天至少寫一題完整的申論題**
3. 開始做近 7 年的一試考古題（每科至少刷 3 遍）
4. 針對弱科加強

關鍵方法 — **題目會（讀書會）**：
- 找 2-4 個程度相近的考友
- 每週一次，每人限時寫 1-2 題
- 寫完互相批改、討論
- PTT 上幾乎所有高分上榜者都推薦這個方法

每日時間分配：
- 上午 2 小時：第二輪複習
- 上午 1 小時：寫申論題（限時）
- 下午 2 小時：檢討申論 + 整理爭點
- 下午 1 小時：刷一試選擇題
- 晚上 2 小時：弱科加強 + 複習爭點筆記

#### 第四階段：考前衝刺（第 15-18 個月）

**目標：快速複習 + 模擬考 + 心態調整**

一試前（最後 2 個月）：
- 一試選擇題瘋狂刷題（近 5 年至少刷 4 遍）
- 每天一回完整模擬（定時、定量）
- 錯題反覆看
- 小科（國際公法、國際私法、票據法、強制執行法、法律倫理）最後衝刺

一試後到二試（約 1-2 個月）：
- 回到申論題模式
- 每天寫 2-3 題完整申論
- 複習自己的爭點筆記（這時候應該已經非常精煉）
- 讀近一年的重要判決和大法官解釋

考前一週：
- 不要再讀新東西
- 只看自己的筆記
- 正常作息、正常飲食
- 信任自己的準備

### 2.3 在職考生的調整版本

如果你是在職準備，以下是調整建議：

每日時間分配（4-5 小時版本）：
- 通勤時間：聽法律課程音檔/複習筆記（0.5-1 小時）
- 早起或晚上：讀新進度（2-3 小時）
- 週末：集中做題 + 複習（每天 6-8 小時）

關鍵原則：
- **品質大於時間**：4 小時專注讀書 > 8 小時分心讀書
- **週末是決勝點**：在職考生的勝負在週末
- **善用零碎時間**：午休、通勤、等人時，都可以用來複習
- **考古題不能省**：就算時間不夠，考古題的練習量不能減

## 第三章：各科目的備考策略

### 3.1 民法 — 最重要的科目，沒有之一

**權重**：一試 100 分 + 二試 300 分 = 總共 400 分

**核心技能**：請求權基礎分析法（Anspruchsgrundlage）

這是民法最重要的方法論。考試題目給你一個案例，你要做的是：

1. **找出所有可能的請求權**（誰可以向誰主張什麼？）
2. **逐一檢驗每個請求權的要件**（構成要件都符合嗎？）
3. **考慮抗辯**（對方可以主張什麼來對抗？）
4. **得出結論**

檢驗順序口訣（PTT 鄉民共識）：
> 契約 → 類似契約（締約過失、無權代理人責任）→ 無因管理 → 物權請求權 → 不當得利 → 侵權行為

**常見陷阱**：
- 只寫一個請求權就結束 → 要窮盡所有可能的請求權
- 只寫結論不寫論證 → 每一步都要交代理由
- 忽略抗辯事由 → 對方的抗辯也要分析

**建議讀法**：
1. 先讀總則（搞懂法律行為、意思表示、代理、時效）
2. 再讀債總（搞懂債的發生原因和效力）
3. 然後債各（各種契約類型）
4. 最後物權、親屬、繼承
5. 全部讀完後，開始做請求權基礎分析的題目

### 3.2 刑法 — 三階層體系是你的武器

**權重**：一試 70 分 + 二試 200 分（含刑訴）

**核心技能**：三階層犯罪審查體系

刑法的分析永遠按照這個順序：

```
一、構成要件該當性
  （一）客觀構成要件：行為、結果、因果關係、客觀歸責
  （二）主觀構成要件：故意/過失
二、違法性
  有無阻卻違法事由？（正當防衛、緊急避難等）
三、罪責（有責性）
  （一）責任能力（年齡、精神狀態）
  （二）不法意識（有無禁止錯誤）
  （三）期待可能性
```

**新手最常犯的錯**：跳過步驟直接寫結論。

例如題目問「甲打乙，乙受傷，甲有罪嗎？」

❌ 新手寫法：「甲打乙致傷，成立傷害罪。」（這不是分析，這是結論）

✅ 正確寫法：逐一檢驗構成要件 → 違法性 → 罪責，每一步都寫清楚

**建議讀法**：
1. 先讀總則（犯罪論 → 未遂 → 共犯 → 競合）
2. 再讀分則（侵害個人法益最常考）
3. 總則是骨架，分則是血肉，兩者要結合練習

### 3.3 憲法 — 基本權分析是送分題

**權重**：一試 40 分 + 二試 200 分（含行政法）

**核心技能**：基本權限制的審查架構

```
一、基本權的保障範圍
  系爭措施涉及哪個基本權？（言論自由？工作權？財產權？）
二、基本權是否受到限制？
  國家行為是否構成對基本權的干預？
三、限制是否合憲？（比例原則審查）
  （一）目的正當性：限制的目的是否為重要公益？
  （二）適當性：手段能否達成目的？
  （三）必要性：有沒有侵害更小的替代手段？
  （四）狹義比例性：手段造成的損害與達成的利益是否相稱？
```

這個架構用在幾乎所有憲法題目上，掌握了就是送分。

**建議讀法**：
1. 先讀基本權總論（打底）
2. 再逐一讀各個基本權（每個都做架構練習）
3. 政府體制放在最後讀
4. 背重要大法官解釋的結論和理由

### 3.4 行政法 — 案例最貼近生活

**權重**：一試 70 分 + 二試 200 分（含憲法）

**核心技能**：行政處分的識別 + 行政救濟途徑的判斷

行政法的題目幾乎都是這個模式：

> 政府機關做了某件事（發處分、罰款、拒絕申請等），人民不服，問：
> 1. 這個行為是什麼性質？（行政處分？行政契約？事實行為？）
> 2. 合法嗎？
> 3. 人民可以怎麼救濟？（訴願？行政訴訟？哪種訴訟類型？）

**最重要的判斷**：是不是行政處分？
行政處分 = 行政機關就公法上具體事件所為之決定或其他公權力措施，直接對外發生法律效果。

**建議讀法**：
1. 行政法基本原則（依法行政、比例原則等）
2. 行政作用法（行政處分最重要）
3. 行政程序法
4. 行政救濟（訴願 + 行政訴訟 + 國家賠償）

### 3.5 訴訟法（民訴 + 刑訴）— 程序就是權利

**民訴核心**：處分權主義 + 辯論主義 + 既判力

**刑訴核心**：強制處分 + 證據法則（傳聞法則、自白法則、違法取證排除）

訴訟法的特色是「程序先於實體」。實體法告訴你誰對誰錯，程序法告訴你怎麼證明、怎麼打官司。

**新手盲點**：覺得程序法不重要。實際上，二試很多題目是「實體法 + 程序法」混合出題。例如：分析完侵權行為之後，接著問舉證責任怎麼分配——這就需要民訴的知識。

### 3.6 商事法（公司 + 保險 + 證交）— 投報率最高的組合

**權重**：一試 70 分 + 二試 100 分

二試的商事法是三科合考 100 分，每科大約出一題。相比民法和刑法動輒上百個爭點，商事法的常考重點相對集中，是「投報率最高」的科目。

**公司法必考重點**：
- 股東會決議瑕疵
- 董事的忠實義務與善管注意義務
- 少數股東權
- 揭穿公司面紗

**保險法必考重點**：
- 保險利益
- 告知義務
- 保險代位

**證交法必考重點**：
- 內線交易的構成要件
- 操縱市場行為
- 證券詐欺

### 3.7 選試科目 — 選對科目是策略

律師考試要選一個選試科目（100 分）。選擇策略：

| 科目 | 適合誰 | 特色 |
|------|--------|------|
| 智慧財產法 | 工程師、設計師、創作者 | 與科技產業最相關 |
| 勞動社會法 | 人資、社工、一般上班族 | 最貼近日常生活經驗 |
| 財稅法 | 會計、金融、財務 | 計算較多，邏輯強的人有利 |
| 海商法與海洋法 | 航運、貿易 | 考生最少，但資源也最少 |

PTT 多數上榜者建議：**選你原本專業最相關的**。例如，工程師選智慧財產法，因為你對專利、著作權已經有直覺理解。

### 3.8 一試專用小科 — 不要放棄也不要過度投入

國際公法（20）、國際私法（20）、票據法（20）、強制執行法（20）、法律倫理（30）、法學英文（30）

這六科只考一試選擇題，共 140 分。策略：
- **不要花太多時間**（每科 1-2 週就好）
- **但也不能完全放棄**（一試差 1 分沒過的大有人在）
- **考前最後兩週集中衝刺**效率最高
- 刷近 5 年考古題就能掌握出題模式

## 第四章：申論題的寫作方法

### 4.1 為什麼申論題是決勝關鍵？

一試只是門檻（選擇題，前 33% 就好）。
二試才是定生死——而二試全部是申論題。

申論題不是在考你「記不記得」，而是考你「會不會用」。

### 4.2 申論題的基本架構

不管哪一科，申論題的架構都類似：

```
一、爭點的提出
  本題涉及的法律問題是什麼？

二、法律規定的說明
  相關法條規定了什麼？學說/實務見解為何？

三、涵攝（把法律規定套用到具體案例）
  本案的事實是否符合法律要件？逐一對照分析。

四、結論
  綜上所述，本案的結論為何。
```

這個「爭點 → 規範 → 涵攝 → 結論」的架構，德文叫 Gutachtenstil（鑑定式寫法），是台灣法律考試最標準的寫作格式。

### 4.3 申論題的八個寫作技巧

#### 技巧一：先列大綱再寫

拿到題目後，先花 5-10 分鐘：
1. 讀題（至少讀兩遍）
2. 找出所有爭點（在題目上劃線、標註）
3. 列出答題大綱（用數字編號）
4. 然後才開始寫

#### 技巧二：開頭先破題

第一段就要讓閱卷者知道你看到了什麼問題：
> 「本題涉及之法律問題為：甲對乙之物之返還請求權是否成立，以及甲是否得依民法第 767 條向乙請求返還 A 車。」

不要寫廢話開頭，直接進入分析。

#### 技巧三：每個爭點都是一個小作文

每個爭點都要完整地走一遍「規範 → 涵攝 → 結論」：

> 按民法第 184 條第 1 項前段規定，因故意或過失不法侵害他人之權利者，負損害賠償責任。**（規範）**
>
> 本案甲酒後駕車闖紅燈撞傷乙，甲之駕車行為致乙身體受傷，具有加害行為；乙之身體權受侵害，具有權利侵害；甲闖紅燈違反道路交通管理處罰條例，具有違法性；甲酒後駕車，對於可能撞傷他人應有預見，具有過失；甲之闖紅燈行為與乙之受傷間具有相當因果關係。**（涵攝）**
>
> 從而，甲之行為符合民法第 184 條第 1 項前段之要件，乙得向甲請求損害賠償。**（結論）**

#### 技巧四：不要只寫一說

遇到有爭議的問題（學說或實務有不同見解），標準寫法是：

> **甲說（實務見解）**：認為……理由為……
> **乙說（學說見解）**：認為……理由為……
> **管見（本文見解）**：採甲說/乙說，蓋……

但要注意：不是每個問題都有爭議。如果是通說，直接寫就好，不需要硬湊不同見解。

#### 技巧五：條號要正確

寫法條要準確到「條項款」：
- ✅ 民法第 184 條第 1 項前段
- ❌ 民法第 184 條（太籠統）

如果真的記不清條號，可以寫概念：
- ✅ 「依民法關於侵權行為之規定」
- ❌ 空白不寫

#### 技巧六：字跡工整比內容多重要

PTT 上多位上榜者都強調：**字不用漂亮，但要整齊**。
- 用 0.5mm 以上的黑色筆
- 字體大一點（每格只寫一個字）
- 分段、編號清楚
- 閱卷者一天改幾百份，整齊的卷子天然加分

#### 技巧七：時間管理

二試每科通常 2-3 題，考 2-3 小時。
- 先花 10 分鐘通讀所有題目
- 按配分分配時間（100 分的題目花 60 分鐘，50 分的花 30 分鐘）
- 最後留 5-10 分鐘檢查
- **寧可每題都寫到，也不要一題寫超完美但其他題來不及**

#### 技巧八：答不出來也要寫

真的遇到不會的題目：
1. 寫出你能辨識的爭點（即使只是部分）
2. 寫出你知道的相關法條
3. 用一般法理推論（比例原則、誠信原則等萬用工具）
4. 空白 = 0 分，寫了至少有機會

## 第五章：一試選擇題的應考策略

### 5.1 選擇題的特色

- 全部單選，每題 2 分
- 不倒扣
- 考法條記憶、實務見解、學說知識
- 比二試更注重「廣度」而非「深度」

### 5.2 刷題策略

**第一輪：分科刷題**
- 每讀完一科，就做該科的近 5 年考古題
- 記錄錯題和猜對的題目
- 整理出「我不會的知識點」清單

**第二輪：混合模擬**
- 按照實際考試的科目組合和時間限制
- 每週做 1-2 回完整模擬
- 目標：每回得分逐步提升

**第三輪：錯題反覆**
- 只做之前做錯的題目
- 直到錯題率降到 10% 以下

### 5.3 各科選擇題技巧

**民法（100 分）**：
- 最多分，也最值得投入
- 注意「判例」和「最高法院判決」的結論
- 債編和物權的條文要熟

**刑法（70 分）**：
- 分則的具體罪名定義要精確
- 常考刑法修正後的新規定

**行政法（70 分）**：
- 行政程序法的條文要很熟
- 大量考「行政處分」的認定

**民訴（60 分）**：
- 管轄、訴訟要件是基本送分題
- 上訴審的規定常考

**刑訴（50 分）**：
- 傳聞法則的例外條文要背
- 強制處分的要件要清楚

**法律倫理（30 分）**：
- 律師倫理規範要讀過
- 不需要深入，但基本概念要有

## 第六章：讀書方法與工具

### 6.1 做筆記的方法

**推薦：爭點筆記法**

每個科目建一份「爭點清單」，格式如下：

```
【爭點名稱】無權代理 vs 表見代理的區分

【法條】民法 §169、§170

【實務見解】最高法院認為……

【學說見解】王澤鑑教授認為……

【結論/口訣】表見代理重點在「本人可歸責性」

【常考變形】配偶擅自出賣共有房屋——是無權處分還是無權代理？
```

到了考前衝刺階段，你只需要看這份爭點清單就好。

### 6.2 複習的節奏

遵循「間隔重複」原則：
- 當天讀完 → 當天睡前複習一次
- 隔天 → 快速瀏覽一次
- 一週後 → 做相關題目測驗
- 一個月後 → 再次複習

人的記憶曲線是會遺忘的，但每次複習都會讓記憶更牢固。

### 6.3 讀書會的組織方式

PTT 上幾乎所有高分上榜者都推薦讀書會。建議格式：

**人數**：3-5 人最理想

**頻率**：每週一次，固定時間

**模式**：
1. 每次選 2-3 題申論題
2. 每人限時 60 分鐘內寫完
3. 寫完後互相交換批改
4. 討論每題的最佳答法
5. 整理出該題的「模範架構」

**注意事項**：
- 找程度相近的人（差太多會沒效率）
- 要有紀律（不能常缺席）
- 以「寫題」為主，不是「聊天」

### 6.4 心態管理

**關於焦慮**：
- 準備國考的過程中，焦慮是正常的
- 不要和別人比進度
- 專注在「今天比昨天進步了多少」

**關於挫折**：
- 寫考古題寫到崩潰是正常的（每個上榜者都經歷過）
- 第一次寫申論題很可能只拿 20-30 分，不要氣餒
- 重點不是你現在的分數，而是你有沒有在進步

**關於多次應考**：
- PTT 上有人考了 8 次才上榜
- 每次應考都是寶貴的經驗
- 落榜不丟臉，放棄才可惜

## 第七章：從「非法律人」到「法律人」的意識轉變

### 7.1 法律人看世界的三個核心習慣

#### 習慣一：凡事問「法律依據是什麼？」

一般人說：「這不公平！」
法律人說：「依據哪條法律，你有什麼權利可以主張？」

法律人不會用道德或情感來判斷對錯（至少在法律分析時不會）。他們會問：法條怎麼規定的？判例怎麼說的？學理上有什麼支持？

**練習方式**：從今天開始，看到任何新聞事件，試著用法律的角度分析。

#### 習慣二：永遠考慮「另一邊怎麼說」

一般人聽到一方的說法就做判斷。
法律人永遠會想：「如果我是對方的律師，我會怎麼反駁？」

這就是法律上的「兩造」思維。原告有原告的理由，被告有被告的抗辯。法院要聽完雙方的說法才做判決。

**練習方式**：每次分析案例，先替原告找理由，然後替被告找理由，最後才做判斷。

#### 習慣三：區分「事實」和「法律評價」

一般人說：「他偷了我的東西！」
法律人說：「先確認事實——他拿走東西的具體經過是什麼？然後再評價——這在法律上構成竊盜還是侵占？還是根本不構成犯罪？」

事實是客觀發生的事情。法律評價是把事實放進法律的框架裡分析。兩者不能混為一談。

**練習方式**：讀案例時，先把事實整理清楚（什麼時間、什麼人、做了什麼事），然後才開始法律分析。

### 7.2 建立法律直覺的方法

法律直覺不是天生的，是練出來的。方法：

1. **大量讀判決**：去司法院法學資料檢索系統，找最高法院的判決來讀。一開始會很痛苦，但讀 100 篇之後，你會發現自己開始有「法感」了。

2. **每天分析一則新聞**：用法律的角度分析新聞事件。例如看到酒駕新聞，想：「這構成什麼罪？刑度是多少？有沒有加重事由？被害人可以民事求償嗎？」

3. **跟法律系的人聊天**：聽他們怎麼分析問題，觀察他們的思考方式。

4. **反覆做考古題**：做到看到案例就能自動分類爭點的程度。

### 7.3 常見的思維陷阱

#### 陷阱一：道德判斷 ≠ 法律判斷

「壞人就應該被判刑」→ 不一定。如果構成要件不該當、有阻卻違法事由、或證據不足，法律上就是無罪。

#### 陷阱二：「法律規定太死板」

初學者常覺得法律太僵化。但其實法律有很多彈性工具：不確定法律概念、比例原則、誠信原則、類推適用等。法律人的功力就在於如何運用這些工具。

#### 陷阱三：「我覺得應該這樣判」

法律分析不是個人感想。每一個結論都需要法律依據。「我覺得」在法律作文裡是零分關鍵字。要寫「依民法第 XX 條規定」「按最高法院 XX 年度台上字第 XX 號判決意旨」。

## 第八章：考試當天的實戰建議

### 8.1 考前準備

- 前一天：不要再讀新東西，早點睡
- 準備物品：身分證、准考證、深色原子筆（多帶幾支）、手錶（不能用手機看時間）
- 提前到考場：熟悉環境、找到教室和廁所

### 8.2 一試當天

- 每卷 80-100 分鐘，時間夠用但不能發呆
- 不確定的題目先跳過，最後回來做
- 不倒扣，所以每題都要填答案
- 答案卡填寫要小心，不要劃錯格

### 8.3 二試當天

- 拿到考卷先通讀所有題目（5 分鐘）
- 配分多的題目多花時間
- 大綱先列在試卷空白處
- 如果某一大題下有 (1)(2)(3) 小題，每小題都要回答，不要漏
- 字不用漂亮但要整齊
- 寫不完也要寫重點結論，不要留空白

### 8.4 最重要的一件事

**不要考完一科就對答案。**

考完就考完了，分數改不了。把精力放在下一科。等全部考完再對答案。
