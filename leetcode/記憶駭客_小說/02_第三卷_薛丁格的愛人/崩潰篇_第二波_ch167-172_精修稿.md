# 壹、ch167 空白的城市

---

## 核心場景

```
清晨六點。

維倫站在地下街的出口。陽光照進來，照在他臉上。
他瞇了一下眼睛。

街上有人。
很多人。

他們在走。
但不是走路。是——移動。
腳在動，但眼睛不在看。

一個穿著睡衣的中年男人停在十字路口。
他看了看左邊。看了看右邊。
又看了看左邊。

他不知道要去哪裡。

旁邊的紅綠燈還亮著——但沒有人看它。

便利商店的自動門開了又關。開了又關。
一個女人站在收銀台前面。手裡拿著一瓶水。
她看著螢幕上的虹膜掃描提示，臉上沒有表情。

「我不知道怎麼付。」

店員站在櫃檯後面。
也在看那個螢幕。

「我不知道怎麼收。」

他們對視了很久。
然後女人把水放下。走了。
店員沒有攔她。他也走了。

自動門開了又關。

                    ＊

有一個男孩。大約七八歲。
牽著一個男人的手。

男人應該是他的父親。

男孩在前面走。男人在後面跟。
男人的腳步不穩。左搖右晃。像喝醉了。
但他沒有喝醉。

他不記得怎麼走路。

男孩牽緊了他的手。
用力。

「爸，跟我走。」

男人沒有回答。
但他的腳繼續在動。
因為有人在拉。

                    ＊

維倫走過三個街區。

看見：
一個穿著醫生白袍的女人蹲在路邊哭。
一對老夫婦坐在公園長椅上，各自看著不同的方向。
一隻流浪狗用鼻子碰一個躺在騎樓下的人，碰了又碰，那人不動。

記憶之塔在遠處。全息投影還在播放。
藍色的字：「守護你的記憶。」

維倫把視線移開了。

                    ＊

捷運新莊線。廢棄隧道入口。

有人搬了幾張椅子在裡面。
有人在發水。
有人用噴漆在牆上寫了一行字：

「如果你不記得自己是誰，來這裡。」

一個短頭髮的女人站在那裡。三十出頭。
她在數人數。

維倫經過時，她看了他一眼。

「你是那個能把記憶還給我們的人？」

維倫的腳步停了。

「……是。」

她搖了搖頭。不是否認。是——

「有些人不想要回去。」

維倫張了張嘴。

她已經轉身去幫一個找不到出口的老人了。

                    ＊

維倫站在隧道入口。
看了很久。

三百萬人。

就這樣了嗎。

米亞的聲音從耳機裡傳來。
很輕。

「工作室有人在等你。」
```

# 貳、ch168 遺忘的胎動

---

## 核心場景

```
工作室。

修復台開著。燈光是冷白色的。

她坐在修復台的邊緣。
大概二十五六歲。長頭髮紮成馬尾。
肚子很大。七個月。

旁邊站著一個男人。大概同齡。
他的手一直放在她肩膀上。
她沒有靠過去。

維倫走進來的時候，她正低著頭。
在看自己的肚子。

她的表情不是好奇。
是恐懼。

「這是什麼？」

三個字。

旁邊的男人咬住了嘴唇。
他的手在她肩膀上抖了一下。

                    ＊

她叫林欣怡。她丈夫叫張瑞安。懷孕七個月又十二天。

這些是張瑞安說的。
林欣怡自己什麼都不記得。

不是「忘了一部分」。
是全部。

她不記得自己懷孕。
不記得什麼時候懷的。
不記得那個站在她旁邊、手在發抖的男人是誰。

她知道自己叫什麼名字——身份記憶還在。
她知道怎麼走路、怎麼說話——技能記憶還在。

但所有關於肚子裡這個生命的記憶——
產檢報告。超音波照片。第一次感覺到胎動時她握住丈夫的手。
取好的名字。買好的嬰兒衣服。
全部都被鸚鵡 2.0 的覆蓋波清除了。

她看著肚子。
不是看「我的孩子」。
是看「我身體裡有一個東西」。

她害怕。

                    ＊

張瑞安把維倫拉到走廊上。

「她不讓我碰肚子。」
他的聲音在抖。
「她說——她不知道那是什麼。」

維倫沒有說話。

「你能修好嗎？」

維倫看了他很久。

「我試。」

                    ＊

修復台啟動。

米亞投射掃描結果。

「她的記憶損傷模式很特殊。
 身份記憶保留——她知道自己是林欣怡。
 技能記憶保留——她能正常生活。
 但所有與嬰兒相關的記憶被覆蓋波精準清除。」

「精準？」

「不是隨機的。鸚鵡 2.0 的覆蓋波有一個特徵——
 它優先覆蓋情感連結最強的記憶。
 對她來說，七個月以來最強烈的情感連結全部指向胎兒。
 所以那些記憶被最先覆蓋。最徹底。」

維倫閉了一下眼睛。

「碎片還在嗎？」

「在。但散了。
 我在覆蓋波的殘餘緩衝區裡找到了十一段碎片。
 每一段都是她和嬰兒之間的記憶。」

米亞將十一段碎片投射出來。
大小不一。有的清晰，有的模糊。

「問題是——十一段碎片中只有三到四段能完整修復。
 時間有限。她的認知焦慮正在加速惡化。
 如果不盡快修復核心記憶，她可能產生永久性的情感斷裂。」

「意思是——」

「意思是你不能全部修。
 你得選。
 先修哪一段。」

維倫看著投射出的十一段碎片。

「哪一段是核心？」

「Greedy 策略。」
米亞的聲音變得低了一些。像在忍住什麼。
「不是修最完整的。是修——對她來說情感連結最強的。
 先找到錨點。
 她的記憶才能以這個錨點為基礎，重新建立與嬰兒的連結。」

「就像剛才的六十秒。」

「一樣。
 不是最好的。是——現在最重要的。」

                    ＊

維倫坐在修復台前。

十一段碎片。選一段先修。

他的手懸在操作杆上方。

六十秒搶救時，他靠的是速度。
現在不是速度。是——什麼是一個母親與孩子之間最核心的連結？

米亞列出了十一段碎片的描述：

  碎片 #1：產檢報告。懷孕第 8 週確認。
  碎片 #2：購買嬰兒床。
  碎片 #3：和丈夫討論名字。三個候選。
  碎片 #4：超音波照片。第一次看見輪廓。
  碎片 #5：——
  碎片 #6：孕吐。凌晨三點。丈夫遞水。
  碎片 #7：摸肚子。第一次感覺到胎動。
  碎片 #8：買了一件黃色小衣服。
  碎片 #9：在醫院走廊等候。丈夫握著她的手。
  碎片 #10：她對著肚子說話。說了什麼不清楚。
  碎片 #11：嬰兒踢了一下。她笑了。

碎片 #5 的描述欄是空的。
米亞標註：數據損壞嚴重，無法辨識內容。

維倫的目光在十一段之間掃過。

產檢報告？不。那是數據。
超音波？不。那是影像。
名字？不。那是語言。

他的手停在碎片 #7 上面。

摸肚子。第一次胎動。

「這個。」

米亞沒有問為什麼。

維倫開始修復。
```

很輕。

十段碎片從上到下排列，像一座倒過來的塔。

第二位：碎片 #11——嬰兒踢了一下，她笑了。

第四位：碎片 #6——凌晨三點孕吐，丈夫遞水。

最底下：碎片 #1——產檢報告。

產檢報告排在最後。

她在看自己的手。

抱在懷裡。

橘子沒有瞇眼睛。
牠看著老太太。
像看一個陌生人。

老太太的嘴唇抖了一下。

她已經哭完了。

「回家吧。」
她說。

她把橘子放在地上。

橘子站了一下。
看了看四周。
聞了聞空氣。

然後牠走了。

不是走向老太太。
不是走向門口。

牠走到工作室的角落。
靠近通風口的那個角落。

然後蹲下來了。

四隻腳併攏。尾巴繞過來。
眼睛半閉。

牠蹲在那裡。

就像牠在家裡蹲在陽台角落一樣。

牠不知道這個女人是誰。
但牠的身體記得——
蹲下來。找一個角落。蹲下來。

這是牠的習慣。
十三年的習慣。
晶片裡沒有了。
但四隻腳記得。

老太太看著蹲在角落的橘子。

她的手捂住了嘴。

然後她哭了。

維倫站在修復台旁邊。

他沒有轉開。
他看著老太太哭。
看著橘子蹲在角落。

然後他的眼睛也濕了。

他沒有擦。
就讓它在那裡。

工作室暗下來。
只剩通風口角落的應急燈。

打了一個很小的呼嚕。

語青站在門邊。
看著維倫的背影。

他的肩膀在動。很輕。

她什麼都沒說。
