# 《機率預言師》母題回聲與名場面追蹤

> **母題（Motif）**：重複出現的意象/台詞/物件，每次出現意義都在演化。
> 好的母題不是重複。是——變奏。
> 同一句話，在不同的語境裡，意思完全不同。

---

## 一、五大母題追蹤

### 母題 4：11 秒

```
追蹤：關鍵時刻的停頓長度。
11 秒 = 做重要決定所需要的時間。

┌──────┬──────────────────────────────────────────────────────┐
│ 章節 │ 場景 / 誰停了 11 秒                                 │
├──────┼──────────────────────────────────────────────────────┤

├──────┼──────────────────────────────────────────────────────┤
│ Ch11 │ 第一次關透鏡。用裸眼看世界。停了 11 秒。            │
│      │ 「世界——不一樣了。」                                 │
├──────┼──────────────────────────────────────────────────────┤
│ Ch25 │ 決定走法律途徑。團隊開會。沉默了 11 秒。            │
├──────┼──────────────────────────────────────────────────────┤
│ Ch37 │ 小拉暫停前的沉默。「如果我有意識——」                 │
├──────┼──────────────────────────────────────────────────────┤
│ Ch39 │ 法庭上。法官聽完統計證據。停了 11 秒。              │
├──────┼──────────────────────────────────────────────────────┤
│ Ch55 │ 天台。柏朗聽完「漁夫出海」之後的沉默。              │
│      │ 最長的一次——不確定是不是 11 秒。                     │

## 二、名場面總覽

> 評級：★ 有感 / ★★ 記得住 / ★★★ 回想時心跳加速 / ★★★★ 放下書之後還在想

### ★★★★ 級（全書最高潮）

```
1. Ch55「漁夫出海」
   貝葉對柏朗：「你願意活在一個漁夫永遠不出海的世界嗎？」
   → 全書的哲學支點。不確定性 = 希望。
   → 柏朗五十五年來第一次哭。左一滴。右一滴。常態分布。

2. Ch60「蔥花」
   P(蔥花香) = 1。
   → 全書的最後一句不是數學。是味道。
   → 有些先驗不需要更新。不需要檢定。它只是在那裡。
```

```
