### 一、情緒定位


### 二、核心場景

```
早晨。

維倫醒來的時候，修復台還在自動運作。

他不記得自己什麼時候睡著的。

米亞的光球在角落亮著。
比平常暗一點。
恐懼留下的那一點暗——四十個小時後還在。

「早安。」

「米亞。」

「你睡了三十七小時十二分鐘。
碎片歸還系統持續運作中。進度 34%。

六百萬個節點。
五千台核心伺服器。五萬個中繼站。

每一個。

不是集中在某一台伺服器。
是——每一台。」

維倫看著那張圖。
紅點像灰塵一樣遍布整個網路。
不是感染。不是入侵。
更像——

從網路內部。
即時的。
```

## 貳、ch182 共同的前綴——Trie 概念引入

### 一、情緒定位

```
情緒類型：好奇 + 發現
強度：6
曲線位置：追蹤展開
節奏功能：Trie 概念引入（AB 線同步第一擊）
前章銜接：ch181 米亞提出「找模式」的方向。
         本章開始用前綴匹配的方法分析周明哲標記過的記憶碎片。
一行法則：「所有以同一個編碼開頭的記憶——都被同一個人碰過。」
AB 線同步：A 線（在數據海洋中找一個人的痕跡）= B 線（Trie：共同前綴 = 共同源頭）
```

### 二、核心場景

```
下午。

米亞花了六個小時整理數據。

她從崩潰程式碼中提取了所有帶 M.C. 簽名的註解——
總共 1,247 行。

然後她做了一件維倫沒要求的事。

她把每一行註解關聯到的記憶碎片也提取出來。

「維倫。」

「嗯。」

「1,247 行註解。
關聯到 34,891 段記憶碎片。
範圍跨越整個記憶網路。
時間跨度——十年。」

維倫走到螢幕前。

34,891 段碎片像星星一樣漂浮在三維投影裡。
沒有明顯的規律。
沒有聚集。沒有排列。
隨機分布。

「看起來什麼都不是。」

「表面上是。」

米亞把投影旋轉了九十度。
然後把所有碎片的編碼顯示出來。

每段記憶碎片在記憶網路中都有一個唯一編碼。
像身分證字號。
長長的一串字母和數字。

維倫一開始看不出規律。

然後米亞把前八位元高亮。

「你看到了嗎？」

維倫看了五秒。

然後他看到了。

34,891 段碎片。
前八位元——全部一樣。

MC-2079。

「MC-2079。」維倫唸出來。

「M.C. 周明哲。2079——他『死亡』的那一年。」

「所有被他標記過的碎片——」

「都以同一組編碼開頭。
像字典裡所有以『周』字開頭的詞條。」

米亞的光球亮了。

「這叫共同前綴。
想像所有記憶是一本字典。
每個人的名字是一個詞條。
有些詞條以同一個字開頭——
你只要找到那個開頭，
就能找到所有跟那個人有關的東西。」

維倫靠近螢幕。

「不只前八位。」

他用手指劃過一段碎片的完整編碼。

MC-2079-A3-7F-01-...

「前十六位也一樣。」

「前二十四位。」

米亞開始比對。

「前三十二位——」

停頓 0.6 秒。

「98.7% 的碎片共享前三十二位元編碼。」

維倫坐下。

「一個人的痕跡，在三萬五千段記憶裡，
用了同一個前綴。」

「不是巧合。
是設計。
他在每一段他碰過的記憶上都蓋了同一個章。」

維倫安靜了一會兒。

「這個前綴——能帶我們找到更多嗎？」

「如果我用這三十二位元當關鍵字，
搜尋整個記憶網路——」

「會找到什麼？」

米亞的停頓比平常長了 0.3 秒。

「會找到所有他碰過的東西。
不只是帶註解的。
任何被他的微程序接觸過、修改過、甚至只是讀取過的記憶碎片——
都會帶著這個前綴的痕跡。」

維倫看著那三十二位元的編碼。

MC-2079-A3-7F-01-D9-E2-4B。

一個死去十年的人的指紋。
印在三萬五千段記憶上。
可能還有更多。

「做吧。」

米亞沒有回答。

她已經開始了。
```

### 三、逐行解讀

| 台詞/動作 | 設計意圖 |
|-----------|---------|
| 「1,247 行註解。34,891 段碎片」 | 用具體數字建立規模感。讀者理解追蹤的範圍 |
| 前八位元全部一樣 | Trie 概念的第一次觸發：共同前綴。用視覺化呈現 |
| 「像字典裡所有以『周』字開頭的詞條」 | 米亞的核心比喻。把 Trie 的前綴概念翻譯成讀者能理解的語言 |
| 「不只前八位」→ 十六位 → 二十四位 → 三十二位 | 逐步深入 = Trie 的路徑延伸。每一層更精確 |
| 98.7% 共享前三十二位元 | 不是 100%。1.3% 的異常為後續留下空間 |
| 「他在每一段碰過的記憶上都蓋了同一個章」 | 米亞的第二個比喻：蓋章 = 前綴標記 |
| 「她已經開始了」 | 米亞的主動性。不等指令就行動 = 觸點 #11（保護欲）的前奏 |

### 五、演算法 AB 線分析

```
A 線（劇情）：在 34,891 段記憶碎片中，發現所有碎片共享同一前綴編碼。
            = 在海量數據中找到「一個人碰過的所有東西」的方法。

B 線（Trie）：前綴匹配。
            共同前綴 = 共同源頭。
            前綴越長，匹配越精確。
            八位元 → 三十二位元 = 從「大致方向」到「精確定位」。

同步點：     前綴編碼 MC-2079 = 周明哲的指紋。
            「在字典裡找以同一個字開頭的詞條」
            = 「在記憶網路裡找被同一個人碰過的碎片」。
            技術名詞「前綴」被劇情化為「指紋」和「蓋章」。

品質評估：   ★★★★。概念引入自然。比喻清晰。
            但尚未到達 AB 線的「合一」——那要等 ch184。
```

## 參、ch183 字典樹——Trie 構建

### 一、情緒定位

```
情緒類型：敬畏 + 匯聚
強度：7
曲線位置：追蹤加速
節奏功能：Trie 構建教學（AB 線同步第二擊）
前章銜接：ch182 發現共同前綴。
         ch183 維倫開始用前綴搜尋整個記憶網路——結果遠超預期。
         他在修復台上構建一棵前綴樹。
一行法則：「它像一棵倒掛的樹。根在上面。所有的路徑都往下走——走到同一個地方。」
AB 線同步：A 線（從碎片拼出一個人的全貌）= B 線（Trie 構建：O(m) 精準匹配）
```

### 二、核心場景

```
夜晚。第三天。

前綴搜尋的結果回來了。

不是 34,891 段。

是 217,000 段。

米亞沉默了 1.1 秒。

「我擴大了搜索範圍。
不只是帶 M.C. 簽名的碎片——
任何在底層代碼中帶有 MC-2079 前綴痕跡的碎片，
全部提取出來。」

「多少？」

「217,443 段。」

維倫看著螢幕。
三維投影裡的星星從三萬五千顆變成了二十一萬顆。
密得像銀河。

「我需要看結構。」

「你要建——？」

「一棵樹。」

維倫打開修復台的全息操作面板。

他把 217,443 段碎片的編碼匯入系統。
然後開始建。

第一層：前八位元。MC-2079。
所有碎片匯聚到同一個根節點。

第二層：接下來的八位元。
根節點分出 47 條分支。
每條分支代表不同的類別——
A3 是核心伺服器區。
B7 是中繼站。
D1 是個人晶片。

第三層：再往下八位元。
47 條分支再分裂——成 2,300 多條。
每條分支更窄。更精確。

修復台上方，一棵樹正在生長。

但它是倒掛的。
根在上面。
枝幹向下延伸。

第四層。第五層。第六層。

像一棵倒過來的大樹。
根是天空。
枝椏向大地伸展。

米亞在旁邊看著。
她的光球從藍色慢慢變亮。

「你在建的東西——
計算機科學裡有個名字。」

「叫什麼？」

「Trie。前綴樹。也叫字典樹。

想像你把一本字典裡所有的字按筆畫排列——
第一筆相同的字歸到同一條路徑。
第二筆相同的再歸到一起。
一直到最後一筆。

每一條從根到葉的路徑——
就是一個完整的詞條。」

維倫沒有停手。

他繼續建。

第七層。第八層。

分支越來越窄。越來越細。
像大樹最末端的毛細根。

但奇怪的事情發生了。

從第九層開始——分支不再分裂。
相反。
```
