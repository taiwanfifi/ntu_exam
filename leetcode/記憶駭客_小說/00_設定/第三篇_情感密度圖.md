## 捌、第三篇的獨特挑戰

### 1. 深潛場景的感官設計

冰原（ch52-55）和記憶海洋（ch62）是全書最依賴「視覺想像」的場景。

**冰原規則**：
- 顏色只有白和黑。白 = 被刪除的空白。黑 = 記憶吞噬程式
- 聲音 = 冰裂的聲音 + 自己的呼吸。其他什麼都沒有
- 溫度 = 冷。維倫的手指在發麻

**記憶海洋規則**：
- 顏色 = 所有顏色同時存在又混成一片
- 聲音 = 三百萬人的情感同時湧入——像海嘯
- 感覺 = 「被一萬根燒紅的針同時刺入」

### 2. 周明哲的對話場景（ch63-65）

**最大風險**：變成「說教」。周明哲是全書最「哲學」的角色——如果不控制，他的對話會變成論文。

**規則**：
- 每段對話 ≤ 3 句。不允許長篇獨白
- 用物件代替論述：書架、茶杯、金色碎片
- 維倫不是「聽眾」——他反擊。「假的記憶裡長出來的真感情——不是假的」

### 3. ch59 咖啡廳——最難寫的「安靜」場景

三個人在咖啡廳。邱韻如不認識任何人。但她的手停了一秒。

**為什麼難寫**：因為什麼都沒有「發生」。但讀者要被心碎。

**寫法**：
- 嚴柏翰的視角。他坐在三張桌子外
- 不寫他的心理。只寫他的身體：手指發抖、心跳加速
- 邱韻如離開時手停一秒 = 50 字以內。不解釋
- 語青在車裡說「她是我媽媽。但她也不記得我了。」= 全章的重量
