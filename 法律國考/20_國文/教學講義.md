# 國文作文完全教學 — 從零開始拿下「最被低估的拉分科目」

> 律師／司法官國考專用
> 專為非法律背景的轉職者設計，無論你的寫作底子如何，都能學到實用的方法。

---

## 寫在最前面：這科到底在講什麼？

### 如果你對國考作文一無所知，先讀這段

**國文作文佔100分，是拉開差距的秘密武器。**

很多考生把所有時間花在法律科目上，完全忽略國文作文。結果法律科目大家都差不多，反而是作文拉開了差距。

想想看：法律科目動輒幾十個法條、上百個爭點要記，投入大量時間可能只能多拿幾分。但國文作文只要掌握方法、準備素材，從 50 分進步到 70 分是完全可能的——**這 20 分的差距，可能就是上榜和落榜的關鍵。**

### 國考作文的基本資訊

| 項目 | 內容 |
|------|------|
| **考試科目** | 國文（作文） |
| **配分** | 100 分（作文 60 分 + 公文 20 分 + 測驗 20 分） |
| **考試時間** | 2 小時 |
| **作文字數** | 建議 1,200-1,800 字 |
| **題目形式** | 通常一個題目或引文，要求寫一篇論說文 |

### 為什麼轉職考生更應該重視這科？

1. **公平**：不像法律科目有「科班 vs 非科班」的差距，作文人人平等
2. **可預測**：題型固定，準備方向明確
3. **投報率高**：花同樣的時間，作文進步的幅度比法律科目大
4. **有方法**：作文不是靠天分，是靠方法和素材

### 國考作文的分數分佈

根據經驗，國考作文的分數分佈大約如下：

| 分數區間 | 佔比 | 特徵 |
|---------|------|------|
| **80 分以上** | 極少（< 5%） | 文筆優美 + 論述深入 + 結構完美 |
| **65-79 分** | 約 15-20% | 論述清楚 + 結構合理 + 有素材 |
| **50-64 分** | 約 40-50% | 中規中矩、沒有明顯缺失 |
| **35-49 分** | 約 20-25% | 內容空洞或結構混亂 |
| **35 分以下** | 約 5-10% | 嚴重偏題或字數不足 |

**你的目標**：至少 65 分以上。只要掌握本講義的方法和素材，這是完全可以達到的。

## 第一部分：國考作文的特性 — 和你以前寫的作文不一樣

> 很多考生用大學聯考或學測的寫作方式來應對國考，結果分數很慘。因為國考作文有完全不同的評分標準和期待。

---

### 一、國考作文 vs 大學聯考作文

| 比較項目 | 大學聯考作文 | 國考作文 |
|---------|------------|---------|
| **讀者** | 國文老師 | 法律實務工作者 |
| **期待** | 文學性、創意、感性 | 邏輯性、論理能力、務實 |
| **文體** | 記敘文、抒情文、論說文均可 | 以論說文為主 |
| **內容** | 個人經驗、情感表達 | 社會議題、法律價值、公共政策 |
| **評分重點** | 文筆、修辭、想像力 | 論述邏輯、見解深度、結構完整 |
| **字數** | 600-800 字 | 1,200-1,800 字 |
| **素材** | 個人故事、文學典故 | 法律觀點、社會現象、名言佳句 |

### 二、國考作文的五個評分維度

根據考試院的評分參考標準，國考作文主要從以下五個維度評分：

| 維度 | 權重 | 說明 |
|------|------|------|
| **主旨掌握** | 20% | 是否切題、是否準確理解題意 |
| **論述能力** | 30% | 論點是否清晰、論據是否充分、邏輯是否嚴密 |
| **結構組織** | 20% | 文章架構是否合理、段落是否分明 |
| **文字表達** | 20% | 用字遣詞是否恰當、句子是否通順 |
| **見解深度** | 10% | 是否有獨到的見解、是否超越表面 |

### 三、國考作文的核心特質

**國考作文本質上是一篇「法律人的論說文」。**

它要求你展現的是：
1. **邏輯思維**：能夠有條理地分析問題
2. **價值判斷**：對社會議題有正確的價值取向
3. **論述能力**：能用論點、論據、論證來說服讀者
4. **文字素養**：用語精準、表達得體
5. **法律人視角**：從法治、人權、正義等角度思考問題

## 第二部分：審題技巧 — 讀懂題目是拿分的第一步

> 很多考生拿到題目就開始寫，結果寫了一半才發現方向不對。花三到五分鐘認真審題，是最值得的投資。

---

### 一、審題三步驟

#### 步驟一：找出核心概念

題目中一定有一個或幾個核心概念（關鍵詞）。找到它，就找到了寫作的方向。

**範例**：
- 題目：「論法律與道德的關係」
- 核心概念：**法律**、**道德**、**關係**
- 寫作方向：分析法律和道德如何互動、衝突、互補

**範例**：
- 題目：「論寬容的力量」
- 核心概念：**寬容**、**力量**
- 寫作方向：寬容能帶來什麼正面的影響和改變

#### 步驟二：確定題目的要求

題目要你做什麼？是「論」（分析論述）、「談」（自由發揮）、「試述」（有系統地說明）？

| 題目用語 | 要求 | 寫法 |
|---------|------|------|
| **論......** | 有系統地分析和論述 | 正反兩面分析，提出自己的立場 |
| **談......** | 較為自由地發表看法 | 可以加入個人經驗和感受 |
| **試述......** | 有條理地說明 | 按邏輯順序逐一說明 |
| **如何看待......** | 表達你的看法 | 先說明問題，再提出你的立場和理由 |
| **......之我見** | 個人觀點 | 明確表達立場，用論據支持 |

#### 步驟三：構思論點框架

在動筆之前，先花兩分鐘用關鍵詞列出你要寫的幾個論點。

**範例**：題目：「論法律人的社會責任」

```
論點框架：
1. 法律人的角色定位（為何有社會責任？）
2. 社會責任的具體內涵（哪些社會責任？）
   - 維護司法正義
   - 保障弱勢權益
   - 推動法治教育
3. 現實中的挑戰和困境
4. 如何實踐（從個人到制度）
```

---

### 二、常見審題陷阱

#### 陷阱一：偏題

**錯誤示範**：題目問「法律與道德的關係」，結果整篇都在講法律的定義，完全沒有提到道德。

**避免方法**：每寫完一段，回頭看一次題目，確認沒有偏離。

#### 陷阱二：只看到一半的題目

**錯誤示範**：題目是「論現代社會中法治的重要性與挑戰」，結果只寫了「重要性」，完全沒有寫「挑戰」。

**避免方法**：審題時把題目中的每一個關鍵詞都圈出來，確保每個都有回應到。

#### 陷阱三：題目的引文是提示不是限制

很多題目會附帶一段引文。這段引文是用來啟發你思考的，不是限制你只能寫引文的內容。

**正確做法**：
1. 閱讀引文，理解它的核心意思
2. 從引文出發，但延伸到更廣的討論
3. 可以在文章中引用這段話，但不要只是重複它

---

### 三、審題練習

**練習 1**：「論公平正義的追求」
- 核心概念：公平正義、追求
- 可以討論：什麼是公平正義？為什麼要追求？如何在法律和社會中實現？有什麼挑戰？

**練習 2**：「知識的力量與責任」
- 核心概念：知識、力量、責任
- 可以討論：知識帶來什麼力量？擁有知識的人有什麼責任？如何善用知識？

**練習 3**：「在變與不變之間」
- 核心概念：變、不變、之間
- 可以討論：什麼應該改變？什麼不應該改變？如何在兩者之間取得平衡？

## 第三部分：架構方法 — 文章的骨架

> 好的架構就像建築的鋼骨結構——有了它，文章再怎麼寫都不會倒。

---

### 一、基本架構：起承轉合

國考作文最穩妥的架構就是傳統的「起承轉合」四段式。

| 段落 | 功能 | 佔比 | 字數（以1,500字為例） |
|------|------|------|---------------------|
| **起**（開頭段） | 破題、引入、提出論點 | 15% | 約 200-250 字 |
| **承**（承接段） | 正面論述、展開論點 | 30% | 約 400-500 字 |
| **轉**（轉折段） | 反面思考、深入分析 | 30% | 約 400-500 字 |
| **合**（結尾段） | 總結、昇華、展望 | 25% | 約 300-350 字 |

#### 起（開頭段）

**功能**：抓住讀者注意力、點出文章主題、預告論述方向。

**常用開頭方式**：

| 方式 | 說明 | 範例 |
|------|------|------|
| **名言開頭** | 引用名人的話帶入主題 | 「孟子曰：『民為貴，社稷次之，君為輕。』這句話揭示了......」 |
| **故事開頭** | 用一個簡短的故事或案例引入 | 「二〇二三年某位律師放棄高薪，為冤案當事人無償辯護......」 |
| **問題開頭** | 提出一個引人思考的問題 | 「什麼是正義？這個問題困擾了人類數千年。」 |
| **對比開頭** | 用兩個對立的概念製造張力 | 「在追求效率的時代，我們是否遺忘了公平？」 |
| **定義開頭** | 先定義核心概念 | 「所謂法治，並非單純地以法律統治人民，而是......」 |
| **時事開頭** | 從近期的社會事件切入 | 「近年來頻繁發生的社會爭議，讓我們重新思考......」 |

**開頭的黃金公式**：

```
引子（名言/故事/問題）
  ↓
連結題目（這和今天要討論的主題有什麼關係）
  ↓
提出論點（我認為......）
```

#### 承（承接段）

**功能**：展開正面論述、提供論據和例子。

**承接段的結構**：

```
論點一
  ├── 論點陳述（一句話說明這個論點）
  ├── 論據（事例、數據、學說來支持）
  └── 小結（回扣主題）

論點二
  ├── 論點陳述
  ├── 論據
  └── 小結
```

**注意**：每個論點都要有「論據」支持。空泛的論述不會得高分。

#### 轉（轉折段）

**功能**：展現批判思維、提出反面觀點或深入分析困難和挑戰。

**轉折的方式**：

| 方式 | 說明 | 適用場景 |
|------|------|---------|
| **反面論述** | 提出相反的觀點，再加以回應 | 「然而，也有人認為......但我認為......」 |
| **困境分析** | 討論實踐中的困難和挑戰 | 「理想雖好，但現實中面臨......」 |
| **層次深入** | 從表面深入到本質 | 「表面上看是......但深層來看......」 |
| **不同角度** | 從另一個角度重新審視 | 「如果從弱勢者的角度來看......」 |

**為什麼需要「轉」？**

因為只有正面論述的文章太單薄，像是在「自說自話」。加入轉折段，展現你能看到問題的多面性，這是高分文章的關鍵。

**生活比喻**：就像律師在法庭上，不能只說自己的理由多好，還要預見對方可能的反駁，並提前回應。

#### 合（結尾段）

**功能**：總結全文、提升層次、給讀者留下深刻印象。

**結尾的方式**：

| 方式 | 說明 | 範例 |
|------|------|------|
| **總結回顧** | 簡潔回顧全文論點 | 「綜上所述，法治的實現需要......」 |
| **昇華提升** | 把討論提升到更高的層次 | 「這不僅是法律的問題，更是......」 |
| **展望未來** | 對未來提出期許或行動呼籲 | 「期許每一位法律人都能......」 |
| **呼應開頭** | 回應開頭的故事或名言 | 「回到開頭那位律師的故事......」 |
| **名言收尾** | 用一句有力的話結束 | 「正如......所言：『......』」 |

**結尾的黃金公式**：

```
總結全文（概括主要論點）
  ↓
提升層次（從個案到通則、從現在到未來）
  ↓
有力結語（名言或期許，餘韻不絕）
```

---

### 二、進階架構：論點佈局法

如果你希望文章更有層次感，可以用「論點佈局法」來規劃文章結構：

#### 總分總結構

```
第一段：開頭（提出中心論點）
第二段：論點一（例如：個人層面）
第三段：論點二（例如：社會層面）
第四段：論點三（例如：制度層面）
第五段：反思/困境
第六段：結論（總結 + 展望）
```

#### 遞進結構

```
第一段：開頭
第二段：是什麼（定義和現象）
第三段：為什麼（原因分析）
第四段：怎麼辦（解決方案）
第五段：深層思考（反思和提升）
第六段：結論
```

#### 對比結構

```
第一段：開頭
第二段：正面論述（好的一面）
第三段：反面論述（問題和挑戰）
第四段：綜合分析（如何平衡）
第五段：結論
```

---

### 三、段落內部的結構

每一個段落都應該有清楚的內部結構：

```
主題句（Topic Sentence）
  ↓
解釋或說明
  ↓
舉例或論據
  ↓
小結或過渡（連接到下一段）
```

**範例**：

> 法治精神的核心在於「法律之前，人人平等」。（主題句）
> 這意味著無論一個人的身分地位如何——無論是總統還是庶民、無論是富豪還是窮人——在法律面前都應該受到同等的對待。（解釋）
> 二〇一三年某前總統因貪污案被判刑入獄，這正是法治精神的具體實踐：沒有任何人可以凌駕於法律之上。（舉例）
> 由此可見，法治不僅是抽象的理念，更需要透過具體的司法實踐來落實。（小結，連接到下一段）

## 第四部分：法律人視角的論述方式

> 國考作文的閱卷老師多是法律人。如果你的文章能展現法律人的思維方式和價值關懷，會大大提升印象分數。

---

### 一、法律人的思維特質

| 特質 | 在作文中的展現 |
|------|--------------|
| **邏輯嚴密** | 論點之間有因果關係、不跳躍 |
| **重視程序** | 討論問題時注重「程序正義」 |
| **人權意識** | 從弱勢者和被害人的角度思考 |
| **價值衡量** | 討論不同價值之間的衝突和平衡 |
| **制度思維** | 從個案提升到制度層面的反思 |

### 二、法律人常用的論述框架

#### 框架一：權利衝突的衡量

很多國考作文的題目涉及不同價值或權利的衝突。法律人的分析方式是：

```
1. 指出衝突的雙方（例如：言論自由 vs 名譽權）
2. 分析各方的利益和立場
3. 提出衡量的標準（比例原則、公共利益等）
4. 做出你的判斷
```

#### 框架二：從個案到制度

法律人不會只看個案，而是會從個案提升到制度的反思：

```
1. 描述一個具體事件或現象
2. 分析背後的制度問題
3. 提出制度改善的建議
4. 反思制度改革的可能性和限制
```

#### 框架三：理想與現實的辯證

```
1. 描述理想的狀態（例如：完美的法治社會）
2. 指出現實的差距
3. 分析造成差距的原因
4. 提出縮小差距的途徑
5. 承認改革的困難，但仍保持希望
```

### 三、法律人視角的語言特色

| 一般說法 | 法律人的說法 |
|---------|------------|
| 「這不公平」 | 「此舉有違平等原則之精神」 |
| 「政府不能亂來」 | 「行政權之行使應受比例原則之拘束」 |
| 「保護弱勢」 | 「維護弱勢族群之基本人權」 |
| 「大家要守法」 | 「法治社會之建構，有賴全民法治意識之提升」 |
| 「法律不完美」 | 「現行法制仍有未臻完善之處，有待修法改善」 |

**注意**：適度使用法律人的語言可以加分，但不要通篇都是法律術語，那樣反而會讓文章變得生硬。最好的方式是「七分白話、三分法言法語」。

## 第五部分：名言佳句與素材庫

> 「巧婦難為無米之炊。」再好的架構方法，如果沒有素材填充，文章也會空洞無力。這一部分提供你豐富的素材庫，按主題分類，方便考前快速複習。

---

### 一、正義 (Justice)

#### 名言佳句

| 出處 | 原文 |
|------|------|
| **亞里斯多德** | 「正義乃百德之首。」 |
| **羅爾斯 (John Rawls)** | 「正義是社會制度的首要美德，正如真理是思想體系的首要美德。」 |
| **馬丁路德乘恩** | 「任何地方的不公義，都是對所有地方公義的威脅。」(Injustice anywhere is a threat to justice everywhere.) |
| **史記** | 「王侯將相，寧有種乎？」——陳勝起義，正是對不正義制度的反抗。 |
| **孟子** | 「無惻隱之心，非人也。」——正義感是人性的基本。 |

> 看優秀的範文，拆解它的結構和技巧，是最快速的學習方式。

### 範文一：「論法律人的使命」

#### 全文

> 韓非子曾言：「法不阿貴，繩不撓曲。」這句話道出了法律的本質——它不應因權勢而扭曲，不應因利益而偏斜。然而，法律不會自己說話，法律的精神需要透過法律人的雙手來實現。因此，法律人的使命，不僅在於精通法條、贏得訴訟，更在於以法律為器、以正義為志，守護社會的公平與秩序。
>
> 法律人的第一項使命，在於維護司法正義。法律是社會正義的最後一道防線，而法律人正是守護這道防線的衛士。無論是法官的公正裁判、檢察官的客觀追訴，還是律師的忠實辯護，每一個法律人都在各自的崗位上，為正義的實現貢獻力量。誠如美國大法官乘金斯伯格所言：「真正偉大的法律人，不是贏得最多案件的人，而是讓法律更接近正義的人。」
>
> 法律人的第二項使命，在於保障弱勢者的權益。法律之前人人平等，但現實中的資源分配並不平等。那些缺乏資源、缺乏知識的弱勢族群，往往無法有效地利用法律來保護自己。因此，法律人有責任透過法律扶助、公益訴訟等方式，讓法律不只是有錢人的工具，而是所有人的盾牌。當一位律師願意放下高額的律師費，為一位被冤枉的農工辯護，法律的溫度才真正被感受到。
>
> 然而，法律人的使命並非沒有挑戰。在現實的壓力下，法律人時常面臨利益與正義的兩難。商業利益的誘惑、政治力量的介入、社會輿論的壓力，都可能使法律人偏離初衷。更深層的挑戰在於，法律本身就不完美——法條可能過時、制度可能有漏洞、程序可能被濫用。面對這些不完美，法律人需要的不只是專業能力，更是道德勇氣和批判精神。
>
> 面對這些挑戰，法律人應該如何自處？我認為，法律人必須培養三種素養：一是持續學習的能力，讓自己的專業跟上時代的變化；二是同理心，能夠站在不同立場思考問題；三是道德勇氣，在面對不義時敢於發聲。范仲淹「先天下之憂而憂，後天下之樂而樂」的精神，正是法律人應有的胸懷。
>
> 法律不是冰冷的條文，而是一個社會對正義的承諾。法律人的使命，就是讓這份承諾不致落空。在這個瞬息萬變的時代，願每一位法律人都能謹記初衷，以專業維護正義，以溫度守護人權，讓法律成為照亮社會黑暗角落的光。

#### 結構拆解

| 段落 | 功能 | 技巧 |
|------|------|------|
| **第一段（起）** | 引用名言 → 連結主題 → 提出論點 | 韓非子名言 + 從法律延伸到法律人 |
| **第二段（承-1）** | 論點一：維護司法正義 | 具體說明 + 引用金斯伯格的話 |
| **第三段（承-2）** | 論點二：保障弱勢者 | 說明現實差距 + 具體案例（律師為農工辯護） |
| **第四段（轉）** | 提出挑戰和困境 | 列舉三種挑戰 + 深入到制度層面 |
| **第五段（承接轉折）** | 提出應對之道 | 三種素養 + 范仲淹名言 |
| **第六段（合）** | 總結 + 昇華 + 期許 | 從法條到承諾 + 光明的意象 |

#### 高分原因分析

1. **開頭有力**：用韓非子名言直接切入，簡潔有力
2. **論點明確**：兩個使命（正義、弱勢）條理清晰
3. **有反思**：轉折段不迴避困境，展現批判思考
4. **有解方**：不只提問題，還提出三種素養作為回應
5. **結尾昇華**：從「法條」提升到「承諾」和「光」的意象
6. **法律人視角**：全文貫穿法治精神和人權關懷

### 範文二的結構拆解（簡要）

題目：「論公平與效率」

```
第一段（起）：
  - 引用經濟學家的觀點：效率與公平往往矛盾
  - 提出問題：社會是否必須在兩者之間做出選擇？

  - 經濟發展需要效率

第五段（合）：
  - 總結：追求兼顧公平與效率的制度設計
  - 法律的角色：透過法律制度實現兩者的平衡
  - 期許：建立一個不犧牲任何一方的社會
```

## 第七部分：常見題型與因應策略

---

### 一、題型分類

| 題型 | 特徵 | 範例 |
|------|------|------|
| **價值探討型** | 探討某個抽象價值 | 「論正義」、「論自由的意義」 |
| **關係探討型** | 分析兩個概念的關係 | 「論法律與道德」、「論權利與義務」 |
| **議題分析型** | 分析特定社會議題 | 「論環境保護與經濟發展」 |
| **引文申論型** | 根據引文發揮 | 給一段古文或名言，要求寫感想 |
| **人生智慧型** | 討論人生道理 | 「論堅持」、「論寬恕」 |
| **法律相關型** | 與法律價值直接相關 | 「論法治社會的建構」 |

---

### 二、各題型的因應策略

#### 價值探討型

**策略**：

```
1. 先定義這個價值（是什麼）
2. 解釋為什麼重要（為什麼）
3. 如何在現實中實踐（怎麼做）
4. 實踐中的困難和反思
5. 總結
```

**範例**：「論正義」

- 第一段：定義正義（分配正義、矯正正義、程序正義）
- 第二段：正義為何重要（社會穩定的基礎）
- 第三段：正義的實踐（法律制度、社會福利）
- 第四段：正義的困境（多數決 vs 少數權利、國際不正義）
- 第五段：總結

#### 關係探討型

**策略**：

```
1. 分別定義兩個概念
2. 分析兩者的共通點
3. 分析兩者的差異和衝突
4. 探討如何調和或平衡
5. 總結
```

**範例**：「論法律與道德的關係」

- 第一段：法律和道德各是什麼
- 第二段：兩者的交集（法律反映道德底線）
- 第三段：兩者的衝突（法律允許但道德譴責的行為、道德要求但法律不管的事）
- 第四段：如何理解兩者的關係（法律是道德的最低標準？還是兩者各自獨立？）
- 第五段：總結

#### 引文申論型

**策略**：

```
1. 解讀引文的意思
2. 從引文延伸到更廣的主題
3. 提出自己的看法
4. 用例子和論據支持
5. 總結
```

**注意**：不要整篇都在解釋引文的意思，那會變成「翻譯」而不是「論述」。引文只是起點，你的論述才是重點。

當然，這些解方並非萬靈丹。[承認困難和局限]。
但我們仍應[保持希望和方向]。

【第五段：結尾】（約 200 字）
[總結 + 昇華]。
```

### 模板三：價值辯證型

```
【第一段：提出兩個價值的張力】（約 200 字）
[價值A]和[價值B]，自古以來就存在著深刻的張力。
[用一個例子說明這種張力]。

【第二段：論述價值A】（約 350 字）

[解釋 + 舉例]。

【第四段：綜合辯證】（約 350 字）

[提出你的平衡方案]。
[用例子說明可以兼顧的方式]。

【第五段：結尾】（約 250 字）

[有力的結尾]。
```

【第二段：第一層論述】（約 350 字）
[從引文延伸出的第一個論點]。
[解釋 + 舉例]。

【第三段：第二層論述】（約 350 字）
更進一步地看，[第二個論點/更深層的思考]。
[解釋 + 舉例]。

【第四段：反思與當代連結】（約 300 字）

[分析 + 提出你的看法]。

【第五段：結尾】（約 250 字）
[回到引文，用新的角度重新詮釋]。
[總結 + 展望]。
```

## 第九部分：寫作的細節技巧

> 大方向對了之後，細節決定了你是 60 分還是 75 分。

---

### 一、用字遣詞

#### 避免口語化

| 口語 | 書面用語 |
|------|---------|
| 很重要 | 至為關鍵 / 不可或缺 |
| 不好 | 有所不足 / 令人擔憂 |
| 所以 | 因此 / 是以 / 故而 |
| 但是 | 然而 / 惟 / 但 |
| 雖然......但是 | 固然......惟 / 誠然......然 |
| 我覺得 | 筆者以為 / 我認為 |
| 越來越多 | 日益增加 / 與日俱增 |
| 很多 | 為數不少 / 比比皆是 |
| 不一樣 | 截然不同 / 有所差異 |
| 要怎麼辦 | 應如何因應 |

#### 常用的書面連接詞

| 功能 | 連接詞 |
|------|--------|
| **承接** | 此外、再者、況且、不僅如此 |
| **轉折** | 然而、惟、但、不過、儘管如此 |
| **因果** | 因此、是以、故、由此可見 |
| **舉例** | 例如、譬如、以......為例 |
| **總結** | 綜上所述、總而言之、歸納而言 |
| **遞進** | 更進一步、不僅......更、除此之外 |
| **對比** | 相較之下、反觀、與之相反 |
| **讓步** | 固然、誠然、縱使、即便 |

---

### 二、段落轉換

段落之間的過渡要自然，不要突然跳到另一個話題。常用的過渡方式：

| 方式 | 範例 |
|------|------|
| **反面轉折** | 「然而，上述理想在現實中並非沒有挑戰。」 |
| **遞進深入** | 「在此基礎之上，我們還必須思考更深層的問題。」 |
| **對比引入** | 「相較於個人層面的實踐，制度層面的改革更為關鍵。」 |
| **時間推進** | 「回顧歷史的演變，我們可以看到......」 |
| **類比引入** | 「法治之於社會，猶如規則之於賽場。」 |

---

### 三、常見錯誤

| 錯誤類型 | 說明 | 修正方式 |
|---------|------|---------|
| **空泛論述** | 只有結論沒有論據 | 每個論點都要有具體例子或數據支撐 |
| **流水帳** | 東說一點西說一點，沒有邏輯 | 先列大綱，確定論點的順序 |
| **字數不足** | 低於 1,000 字 | 在每個論點上多展開、多舉例 |
| **開頭太長** | 花了 400 字還沒進入正題 | 開頭控制在 200 字以內 |
| **沒有結尾** | 寫到一半時間到了 | 預留最後 10 分鐘寫結尾 |
| **錯別字太多** | 扣分的致命傷 | 最後留 5 分鐘檢查 |
| **立場搖擺** | 前面說 A 好，後面又說 B 好 | 先確定立場再動筆 |
| **只有正面沒有反面** | 論述太單薄 | 一定要有「然而」的段落 |

---

### 四、時間分配

以 2 小時的考試時間為例：

| 時段 | 分配 | 任務 |
|------|------|------|
| **前 5 分鐘** | 審題 + 構思 | 找核心概念、列論點大綱 |
| **5-15 分鐘** | 寫開頭段 | 約 200-250 字 |
| **15-50 分鐘** | 寫正文（承 + 轉） | 約 800-1,000 字 |
| **50-65 分鐘** | 寫結尾段 | 約 250-350 字 |
| **65-70 分鐘** | 檢查 | 錯別字、語句通順 |
| **70-120 分鐘** | 公文 + 測驗 | 剩餘時間分配給公文和選擇題 |

**注意**：寫作文的時間最好控制在 60-70 分鐘內，不要佔用太多公文和測驗的時間。

### 一、公文的基本格式

```
                        函

受文者：○○○（機關名稱）

速別：普通件

附件：

  一、（說明背景、原因、法令依據）
  二、（進一步說明細節或情況）
  三、（如有必要，繼續說明）
辦法：
  一、（具體的做法或請求）
  二、（時程或配合事項）

正本：○○○
副本：○○○

                                     機關首長  ○○○
```

### 二、公文寫作的要點

| 要點 | 說明 |
|------|------|
| **格式正確** | 主旨、說明、辦法三段式 |
| **用語正式** | 使用公文慣用語 |
| **簡潔明確** | 一句話說清楚一件事 |
| **層次分明** | 用編號分點敘述 |
| **主旨精要** | 主旨要能獨立說明全文要義 |

| 類型 | 用語 |
|------|------|

| **期望** | 「請查照」、「請照辦」、「請核示」 |
| **上行文** | 「請鑒核」、「請核示」 |
| **平行文** | 「請查照」、「請惠予協助」 |

## 第十一部分：歷年常見考題實戰演練

> 光看方法不夠，實際演練才能內化。這部分提供歷年常見的國考作文題型，附上審題分析和寫作大綱。

---

### 演練一：「論法律與道德的關係」

#### 審題分析

- **核心概念**：法律、道德、關係
- **題型**：關係探討型
- **寫作方向**：分析法律與道德的共通點、差異、衝突與互補
- **法律人視角**：可以從法實證主義 vs 自然法學派的爭論切入

#### 參考大綱

```
第一段（起）：
  引言：「惡法亦法乎？」——這個古老的問題揭示了法律與道德之間
  永恆的張力。法律和道德都是規範人類行為的力量，但兩者的關係
  遠比想像中複雜。

第二段（承-1）：法律與道德的交集
  - 法律反映道德的最低標準（例如：不得殺人、不得偷竊）
  - 許多法律條文源自道德共識
  - 舉例：公序良俗條款（民法 §72）

第三段（承-2）：法律與道德的差異
  - 法律有國家強制力，道德靠社會輿論和個人良知
  - 法律有明確規範，道德較為模糊
  - 法律可以修改，道德演變較慢
  - 舉例：同性婚姻——法律已承認，但部分人的道德觀仍有不同

第四段（轉）：法律與道德的衝突
  - 有些事法律允許但道德譴責（例如：趁人之危的合法交易）
  - 有些事道德讚許但法律處罰（例如：安樂死在某些地區仍違法）
  - 法實證主義 vs 自然法：法律是否必須符合道德才有效？
  - 「惡法亦法」的困境

第五段（合）：
  總結：法律不等於道德，但兩者密不可分。法律人的使命在於
  讓法律盡可能接近道德的理想，同時維護法律的確定性和安定性。
```

---

### 演練二：「論寬恕的力量」

#### 審題分析

- **核心概念**：寬恕、力量
- **題型**：價值探討型
- **寫作方向**：寬恕能帶來什麼正面影響，以及法律中的寬恕精神
- **法律人視角**：可以連結到修復式正義、緩刑制度、更生保護

#### 參考大綱

```
第一段（起）：
  引言：曼德拉在獄中度過27年後說：「當我走出牢房、邁過通往
  自由的監獄大門時，我已經清楚，如果我不能把悲痛和怨恨留在
  身後，那麼我其實仍在獄中。」寬恕，不是軟弱的表現，而是
  一種需要巨大勇氣的力量。

第二段（承-1）：寬恕對個人的力量
  - 放下仇恨讓自己自由
  - 心理學研究：寬恕有助於心理健康
  - 舉例：曼德拉的故事

第三段（承-2）：寬恕對社會的力量
  - 南非的真相與和解委員會——用寬恕取代報復
  - 修復式正義——讓加害者與被害者對話
  - 台灣的轉型正義——如何面對歷史傷痕

第四段（轉）：寬恕的邊界
  - 寬恕不等於縱容——法律仍需懲罰犯罪
  - 受害者有權選擇不寬恕——強迫寬恕是二次傷害
  - 寬恕不能取代正義——正義是寬恕的前提
  - 舉例：對重大犯罪的寬恕引發的爭議

第五段（合）：
  總結：寬恕的力量在於打破仇恨的惡性循環，讓社會有可能
  走向和解和重建。但真正有意義的寬恕，必須建立在正義的
  基礎之上。法律人的智慧，在於在懲罰與寬恕之間找到平衡。
```

### 演練三：「在變與不變之間」

#### 審題分析

- **核心概念**：變、不變、平衡
- **題型**：辯證型
- **寫作方向**：什麼應該改變？什麼不應該改變？如何在兩者之間取得平衡？
- **法律人視角**：法律的安定性（不變）vs 法律的進步性（變）

#### 參考大綱

```
第一段（起）：
  引言：「苟日新，日日新，又日新」——古人早已理解變革的重要。
  但同時，「不以規矩，不能成方圓」——穩定的規則同樣不可或缺。
  在急遽變遷的現代社會，如何在變與不變之間取得智慧的平衡，
  是每一個時代的課題。

第二段（承-1）：「變」的必要性
  - 社會在進步，法律和制度必須與時俱進
  - 不變等於落後——過時的法律會阻礙社會發展
  - 舉例：同性婚姻的法律變革、數位時代的隱私權保護

第三段（承-2）：「不變」的價值
  - 核心價值不能隨意改變——人權、正義、誠信
  - 法律的安定性：頻繁修法會讓人民無所適從
  - 舉例：憲法的基本權利保障——歷經數十年不變，但始終重要

第四段（轉）：在變與不變之間的智慧
  - 變的是形式和手段，不變的是核心價值
  - 法律修正的藝術：在回應社會需求的同時維持法安定性
  - 舉例：大法官釋字——在不修改憲法文字的情況下，
    透過解釋來回應時代的需求

第五段（合）：
  總結：「變」與「不變」不是二擇一，而是辯證的統一。
  智慧在於知道什麼時候該堅持、什麼時候該改變。
  對法律人而言，這正是「法的生命在於經驗」的深刻體現。
```

### 演練四：「論公共利益與個人權利」

#### 審題分析

- **核心概念**：公共利益、個人權利、衝突與平衡
- **題型**：關係探討型
- **寫作方向**：公共利益和個人權利如何衝突？如何調和？
- **法律人視角**：比例原則、基本權利的限制

#### 參考大綱

```
第一段（起）：
  引言：新冠疫情期間，政府為了公共衛生實施了封城、強制隔離、
  疫苗通行證等措施。這些措施保護了公共健康，但也限制了個人
  的行動自由。公共利益與個人權利的衝突，是法治社會永恆的
  命題。

第二段（承-1）：個人權利的重要性
  - 個人權利是民主自由的基石
  - 沒有個人權利的保障，國家會淪為極權統治
  - 舉例：歷史上的極權體制如何以「公共利益」之名侵害人權

第三段（承-2）：公共利益的正當性
  - 人不是孤立存在的，社會需要公共秩序
  - 某些公共利益確實需要個人做出讓步
  - 舉例：都市計畫中的土地徵收、傳染病防治中的隔離措施

第四段（轉）：比例原則——法律人的衡量工具
  - 目的正當性：限制個人權利的目的是否正當？
  - 手段適當性：手段能否達成目的？
  - 必要性：有沒有侵害更小的替代手段？
  - 狹義比例性：所獲公益是否大於所失私益？
  - 舉例：大法官釋字的比例原則審查

第五段（合）：
  總結：在公共利益與個人權利之間，沒有一勞永逸的標準答案。
  每一次的衝突都需要具體的衡量和判斷。但有一個原則是不變的：
  對個人權利的限制，必須是為了正當的目的、以合比例的手段為之。
  這就是法治的精髓。
```

### 演練五：「知識分子的社會角色」

#### 審題分析

- **核心概念**：知識分子、社會角色
- **題型**：議題分析型
- **寫作方向**：知識分子（包括法律人）在社會中應扮演什麼角色
- **法律人視角**：法律人作為知識分子的特殊責任

#### 參考大綱

```
第一段（起）：
  引言：顧炎武「天下興亡，匹夫有責」，知識分子更是責無旁貸。
  在資訊爆炸的時代，知識分子的社會角色不減反增。

第二段（承-1）：知識分子作為社會的良心
  - 發出獨立的聲音，不隨波逐流
  - 為弱勢者代言
  - 舉例：知識分子在社會運動中的角色

第三段（承-2）：知識分子作為理性的力量
  - 以專業知識分析問題，提供理性的解方
  - 在情緒化的公共討論中保持冷靜
  - 舉例：法律人在公共議題中的理性分析

第四段（轉）：知識分子的困境
  - 象牙塔的誘惑——脫離現實
  - 政治力的壓迫——面臨噤聲的壓力
  - 商業化的侵蝕——知識被金錢收買
  - 反智主義的挑戰——專業知識被否定

第五段（合）：
  總結：知識分子的角色是「批判性的參與者」——既保持獨立的
  批判精神，又積極參與社會事務。范仲淹「先天下之憂而憂」
  的精神，正是知識分子應有的擔當。
```

## 第十二部分：進階寫作技巧

---

### 一、如何寫出有深度的文章

#### 技巧一：多層次分析

不要只在一個層面上討論問題。好的文章會在多個層次之間穿梭：

```
個人層次 → 社會層次 → 制度層次 → 文明層次
```

**範例**（論正義）：
- 個人層次：每個人內心的正義感
- 社會層次：社會如何定義和追求正義
- 制度層次：法律制度如何保障正義
- 文明層次：人類文明對正義的永恆追求

#### 技巧二：辯證思維

不要只看到事物的一面。高分文章會展現辯證的思維：

| 正題 | 反題 | 合題 |
|------|------|------|
| 法治很重要 | 但法律不完美 | 我們需要不斷改善法律來追求更好的法治 |
| 自由很珍貴 | 但自由不是無限的 | 真正的自由是在合理限制下的自由 |
| 效率很重要 | 但不能犧牲公平 | 追求兼顧效率與公平的制度設計 |

#### 技巧三：具體化抽象概念

抽象的論述容易空洞。把抽象概念具體化，文章會更有說服力：

| 抽象 | 具體化 |
|------|--------|
| 「正義很重要」 | 「當一個無辜的人被冤枉入獄，正義不只是一個抽象的概念，而是那個人失去的十年青春。」 |
| 「法治很重要」 | 「在一個沒有法治的社會，你今天努力工作買的房子，明天可能被權貴一句話就拿走。」 |
| 「人權很重要」 | 「對一個被非法拘禁的人而言，人權不是教科書上的名詞，而是能不能回家吃晚飯的差別。」 |

---

### 二、如何避免千篇一律

閱卷老師一天要看幾百篇作文，如果你的文章和別人一模一樣，很難拿高分。以下是讓你的文章脫穎而出的方法：

#### 方法一：獨特的開頭

不要用「自古以來」、「隨著時代的演變」這種萬用開頭。

| 千篇一律 | 獨特 |
|---------|------|
| 「自古以來，正義就是人類追求的目標。」 | 「西元前399年，蘇格拉底飲下毒酒。雅典的法庭判他有罪，但兩千四百年後，我們記住的不是那個判決，而是他對正義的堅持。」 |
| 「法律與道德一直是重要的議題。」 | 「一位母親偷了藥來救病危的孩子。你會判她有罪嗎？」 |

#### 方法二：個人經驗的適度運用

雖然國考作文以論說為主，但適度加入個人經驗可以增加真實感：

> 「在轉職準備法律國考的過程中，我深刻體會到法律不只是條文的堆砌，更是一種思考問題的方式。」

**注意**：個人經驗只是調味料，不是主菜。不要整篇都在講自己的故事。

#### 方法三：跨領域的素材

大部分考生只會引用法律和政治的素材。如果你能引用文學、哲學、科學、歷史的素材，會讓文章更有深度：

- 文學：卡夫卡《審判》——對官僚體系的批判
- 哲學：羅爾斯的「無知之幕」——公平的思想實驗
- 歷史：法國大革命——自由平等博愛的追求
- 科學：人工智慧的倫理挑戰——當機器開始做決定

---

### 三、修辭技巧的適度運用

適度的修辭可以讓文章更有表現力，但不要過度使用：

| 修辭 | 範例 | 效果 |
|------|------|------|
| **對比** | 「法律給了他自由，卻沒有給他尊嚴。」 | 突出兩者的差距 |
| **排比** | 「是正義的追求，是自由的守護，是弱勢者最後的依靠。」 | 增強氣勢 |
| **設問** | 「正義的代價，是不是任何一個社會都承擔得起？」 | 引發思考 |
| **引用** | 「如孟子所言：『雖千萬人，吾往矣。』」 | 借助權威 |
| **類比** | 「法治之於社會，猶如骨骼之於人體。」 | 化抽象為具體 |
| **回扣** | 開頭說的故事，在結尾再提一次 | 結構完整、首尾呼應 |

## 結語：國文作文的備考策略

### 四個關鍵行動

1. **每週練習一篇**：找歷年考題，限時 70 分鐘寫完。重點不是寫得好不好，而是「寫完」和「寫習慣」。

2. **建立你的素材庫**：把本講義的名言佳句整理成隨身小卡，利用零碎時間背誦。每個主題至少記住 2-3 個可用的素材。

3. **練習套用模板**：不同題目套用不同模板，反覆練習直到內化。到考場上，你的手指要能自動寫出起承轉合。

4. **請人閱讀你的作文**：如果可能的話，請朋友或老師看你的作文，給你回饋。看不到自己盲點的時候，別人的眼睛最有用。

### 考前一週的策略

| 天數 | 任務 |
|------|------|
| **7 天前** | 複習所有名言佳句和素材 |
| **6 天前** | 練習一篇完整的作文（限時） |
| **5 天前** | 複習四個寫作模板 |
| **4 天前** | 練習公文格式（背格式、練語氣） |
| **3 天前** | 整理常考主題的關鍵字清單 |
| **2 天前** | 再練一篇作文，特別注意開頭和結尾 |
| **考前 1 天** | 輕鬆複習，不要再寫新東西，保持手感 |

### 最後一句話

國文作文不是比誰的文筆最華麗，而是比誰的思考最清晰、論述最有力、態度最誠懇。只要你掌握了方法、準備了素材、反覆練習，這一科絕對是你拉開差距的利器。
