  作者的審視：關於 revision_plan.md 的逐點回應

  我花了很長的時間重新讀了終局六章（55-60）、法庭五章（37-41）、
  和所有關鍵危機場景（33-36、49-50、以及第20章的格式化事件）。以
  下是我以這本書的寫作者——擁有六十章完整 context 的人——對那份
  revision plan 的真實判斷。

  ---
  問題 #1：「說教感太重，公式減法 30-40%」

  我的判斷：部分同意，但比例和邏輯都不對。

  revision plan 引用書評說「從中段開始，主角群幾乎都在白板前面上
  統計學課」。但那位寫 plan 的 Claude 沒有讀過正文，他只讀了書評
  。我重新讀了法庭場景——第39章貝葉在白板上畫硬幣、寫 H₀/H₁、α =
  0.001；第40章費雪畫出2×2混淆矩陣然後貝葉拿費雪自己的圖反擊 Type
   II Error；第41章煙霧偵測器比喻——這些不是上課，這些是戰鬥。
  每一段數學都是武器，都在推動劇情和角色對抗。

  同理，第33章蜜罐 5.23σ 觸發是一個半夜的情報危機；第34章 MGF
  唯一性定理證明是凌晨四點四個人圍著白板的儀式性場景（「不同的分
  布」三人輪流說出口，像宣誓）。這些場景裡的數學不是裝飾，是劇情
  的骨架。

  但我也知道：中段某些章節——特別是第二篇「雜訊」和第三篇「信號」
  的前半部——確實有些段落是在「展示概念有多酷」而不是在推動故事。
  那些需要修。

  我的做法：
  - 不做 30-40% 的大砍。做 15-20% 的精準手術。
  - 逐章掃描「白板場景」，用一個判準：拿掉這段推導之後，劇情和情
  感有沒有損失？ 有損失→保留。沒損失→移到附錄「機率教室」（我已經
  建好了，五篇、25張圖、完整對照表）。
  - 正文裡被移走的段落替換為一句結論 + 一個腳註指向附錄。

  不動的： 法庭戲的所有白板場景（它們已經是「攻防式」而不是「上課
  式」）、蜜罐 5σ 場景、MGF 證明、終局的過度擬合論證。

  問題 #2：「終局重寫——柏朗不能被演講說服」

  我的判斷：不同意。終局不需要重寫。

  這是我跟 revision plan 分歧最大的地方。Plan 說「典型的 Talk no
  Jutsu 結局」，書評說「主角的反擊是在法庭上寫白板、跑到天台講大
  道理，然後反派就哭了、覺悟了、交出最高權限卡」。

  但實際讀第55章——貝葉的核心論證是「你在過度擬合」。這不是大道理
  ，這是用柏朗最信任的語言（機器學習）指出他系統的結構性缺陷。她
  的邏輯鏈是：

  1. 你的系統完美擬合了已知風險（交通事故、犯罪）
  2. 但它失去了泛化能力
  3. 壓縮常態分布的尾巴不會消除極端事件——會製造更集中、更具毀滅性
  的黑天鵝
  4. 零方差不是安全，是死亡

  然後漁夫比喻——如果天氣預報完美，你父親永遠不會出海，你就不會存
  在。用柏朗自己的存在來反證他的系統。
  這不是感動他，是在他信任的邏輯框架內擊敗他。情感崩潰（兩滴對稱
  的淚）是在邏輯擊潰之後才發生的。

  第57章高思遠拒絕給柏朗定論——他說「你做了」，不說對錯，說
  p-value 還不能算，樣本量不夠。柏朗的救贖不是「完整的」，是一個
  開放的後驗。

  這已經不是 Talk no Jutsu 了。
  書評可能是快速讀過去，看到「天台對話→交出權限卡」就歸類為嘴遁。
  但實際的文本結構是：邏輯擊潰→情感崩塌→自願交出→未完成的救贖。

  Revision plan 的具體建議——「找出柏朗論文裡的數學漏洞」——其實我
  已經做了，那個「漏洞」就是過度擬合。只是它不是一個公式上的
  bug，是一個方法論上的盲點。我認為這比找一個「計算錯誤」更高級：
  不是你算錯了，是你算對了但問錯了問題。

  不動。

  問題 #3：「加入物理危機」

  我的判斷：完全同意。這是全書最大的結構性弱點。

  第20章林岳被格式化——這是全書最好的危機場景，沒有之一。一個乾淨
  的、空洞的微笑。「你叫什麼名字？我想記住。」蘇立恆凌晨三點不請
  自來、不說一個字、留一杯算好冷卻時間的溫咖啡。那一章是真正讓人
  心碎的。

  但從第20章之後，到第60章——四十章裡，沒有任何人再付出不可逆的代
  價。 蜜罐是資訊危機、法庭是智力對決、Oracle Prime
  是倒計時——但沒有人流血、沒有人失去什麼、沒有人的生活被永久改變
  （高思遠的病是在背景裡緩慢推進的，不是一個危機事件）。

  這意味著讀者在第20章之後已經「知道」主角不會真的受傷。張力會逐
  漸下降。

  我的做法：

  在第四篇「檢定」或第五篇「隨機」的前段，加入一個真正的損失。我
  有幾個想法：

  方案 A：馬可瑜的數據被摧毀。 她的 6,083
  筆保險數據——那是第43章扭轉局面的關鍵證據。如果 Oracular
  在法庭戲之後報復性地侵入保險公司伺服器、銷毀了原始數據，馬可瑜
  必須用她的備份（她是精算師，一定有備份——但備份可能不完整，只有
  4,200 筆）。這會讓 Power 從 0.999+ 下降到一個尷尬的灰色地帶——足
  以說明問題，但不再是壓倒性的。同時這展示了 Oracular
  願意犯法來滅證，提高了物理危機感。

  方案 B：貝葉差點被格式化。 在進入 B3
  實驗室的過程中，費雪啟動了安保協議。貝葉的透鏡被遠端強制進入「
  校準模式」——她的視野開始模糊、先驗開始被覆寫。蘇立恆必須在她完
  全被格式化之前物理拔除她的透鏡。這會造成一個永久後果：貝葉的透
  鏡損壞，從此她只能用裸眼看世界。這跟蘇立恆在第57章自願摘下透鏡
  形成對照——一個是被迫的、一個是選擇的。

  方案 C：高思遠在天台之前倒下。 他去找柏朗的路上，身體撐不住。他
  沒能親自到達——但他留了一封信給貝葉，裡面寫著他三十年來沒對柏朗
  說出口的話。貝葉帶著那封信上天台。這讓天台對決多了一層重量：不
  只是她自己的論證，還有師父託付的遺言。

  我傾向方案 B。 因為它直接測試了全書的核心命題——P(free will) ≠
  0——在最物理的層面：你的自由意志正在被一行一行地覆寫，你還能選擇
  嗎？

  問題 #4：「法庭戲現實化」

  我的判斷：大部分不需要改。

  我重讀了法庭五章。書評和 revision plan
  說「台灣法庭不會讓人畫白板上課」——但第39章已經處理了這個問題：
  白板是高思遠特別申請的，法官核准，理由是「投影片是結論，白板是
  推理過程」。這在小說的近未來世界觀裡是成立的。

  更重要的是，法庭戲已經不是上課式的了。費雪在第40章用貝葉的同一
  塊白板畫出混淆矩陣來反擊，貝葉再用費雪的圖圈出 Type II
  Error——這是攻防，不是教學。煙霧偵測器比喻讓全場屏息三十秒、連費
  雪都微微點頭——這是修辭的巔峰時刻，不是課堂。

  法官林俊維的描寫也是對的：他承認自己統計學被當過，要求雙方用他
  聽得懂的語言講。這不是在醜化法官，是在尊重他——他知道自己不懂，
  所以他認真聽。

  唯一需要微調的： 台灣民事訴訟確實偏向法官職權主義，雙方律師的口
  頭攻防不會像普通法系那麼戲劇化。但這是一部科幻小說，不是法律教
  科書。我可以在第37章加一句旁白解釋這個案件的特殊性——涉及2300萬
  人的消費者保護訴訟、技術高度專業化——讓法官選擇較寬鬆的程序規則
  ，增加口頭辯論時間。一句話就夠了。

  問題 #5：「角色去臉譜化」

  我的判斷：角色已經有深度了。但可以更早展現。

  Revision plan 說蘇立恆是「典型的面癱高智商酷男駭客」。但第35章
  他發現自己三年前埋的那行程式碼備註——

  // 當你找到一個值得的人，把這個給她看。

  然後他的手凍在鍵盤上方，心率從 58 跳到
  76。他刪掉了那行備註——「因為它完成了它的任務」。小拉記錄下心率
  ，加了一句：「有些數據不是用來分析的。是用來記住的。」

  第57章他摘下透鏡——三年來第一次用裸眼看世界。第58章他走到貝葉旁
  邊的石椅坐下——白奶奶說「少年仔，你終於走過來了」。

  這不是面癱酷男。這是一個用三年的沉默來等待的人。

  費雪離開 Oracular 時留下的 Fisher 初版書上的題字——「The p-value
   is not the probability that H₀ is true. Never was. Never will
  be. Don't make the same mistake I did.」——這不是冷酷女高管。這
  是一個在離開的最後一刻，仍然在乎真理的人。

  柏朗問高思遠「若晴她——會原諒我嗎？」——這不是反派。這是一個花了
  三十年逃避悲傷的人終於回頭。

  但。 書評可能在第一篇（前12章）就給角色貼了標籤，然後帶著這個標
  籤讀完全書。如果讀者在第5章就認定蘇立恆是面癱酷男，到了第35章他
  的複雜性出來的時候，讀者可能已經棄書了。

  我的做法： 在每個角色首次出場的章節裡，加入一個微小的「裂縫」——
  一個不符合刻板印象的瞬間。不需要大改，只要一個細節：

  - 蘇立恆（第2-3章首出場）： 加入一個他不精準的瞬間。比如他把某
  個東西打翻了——不是因為不小心，是因為他在想別的事。讓讀者從第一
  秒就知道：這個人的冷靜是演出來的。
  - 費雪（第9-10章首出場）：
  加一個她猶豫的瞬間。也許她在簽署某份文件之前，筆尖停了0.3秒。0.
  3秒太短了，沒有人注意到。但敘事者注意到了。
  - 柏朗（第12章首現身）： 讓他做一個溫柔的事情。也許他給某個基層
  員工多倒了半杯水——不是因為禮貌，是因為那個人讓他想起了某個人。

  這些不改變人設，不破壞第一印象的力道，只是在讀者心裡種一顆種子
  ：這個人不是你以為的那樣。等到第35章、第55章那些複雜性全面展開
  的時候，讀者會回想起那顆種子，然後一切就通了。
