## ch201 你是誰

**情緒強度：8（薛丁格篇開場——緊繃的對峙起手式）**
**演算法：無**

---

### 核心場景

```
場景轉換。

離維倫的工作室四公里。
地下第五層。周明哲的舊實驗室。

三天前嚴柏翰走進這裡。

門是開的。
不是壞了——是從來沒有鎖。
像在等一個遲早會來的人。

走廊的牆壁是水泥原色。
每隔三公尺一盞應急燈。
白色。冷。把一切照得像手術室。
空氣裡有消毒水的殘味和一種更深的、
不知道從哪裡來的茉莉花的氣味。

淡得幾乎不存在。
但嚴柏翰的身體認出了它。
心跳從 72 跳到 110。
胸口悶了一下。

他走了六十步。
從門口到操作台。
他數的。

操作台前坐著一個人。

短髮。深色系套裝。
金框眼鏡推在額頭上。
螢幕的藍光打在她的臉上。
她在看數據——崩潰後第十四天的城市修復報告。

邱韻如。

她沒有抬頭。

嚴柏翰站在她背後三步遠的地方。
像十年來他站在每一個線人背後的距離。
職業習慣。安全距離。
也像一道牆。

他站了多久？
他不知道。
螢幕上的數據翻了兩頁。也許三分鐘。

然後她轉頭。

不是因為聽到腳步聲——
是因為心率監測儀響了。
她的生物指標在他進入走廊的第七秒時
出現了一次異常脈衝。

她不知道為什麼。
她只知道系統提示「生理異常」。
所以她轉頭。

然後她看到他。

一個男人。
黑風衣。左眼有一道垂直的疤。
站在三步之外。

她的眼神——
是空的。

不是敵意。不是冷漠。
是從資料庫裡查了一遍、什麼都沒找到的空。

「你是誰？」

三個字。
發音標準。語氣平穩。
像在問一個走錯辦公室的陌生人。

嚴柏翰沒有回答。

他的左手——
左手在口袋裡。
握著一隻紙鶴。
紙角已經被汗浸軟了。

他在口袋裡握了三天。
從 ch186 離開維倫工作室那天開始。

紙鶴的折法是歪的。
因為教他的人也是歪的。
而教他的人是她。

他沒有回答「你是誰」。

他說的是：

「第三天了。」

邱韻如的眉毛動了一下。
不是困惑。
是——她的臉在試圖做出一個
她的記憶裡不存在的表情。

「我不認識你。」

嚴柏翰的嘴角動了一下。
不是笑。不是苦。
是肌肉的痙攣。
十年的肌肉記憶在提醒他：
你認識她。

「我知道。」

他的手在口袋裡鬆開了紙鶴。
又握住。
又鬆開。

「但妳的心率認識我。」
```

## ch202 身體記得

**情緒強度：8（對峙第二天——身體背叛意識）**
**演算法：無**

---

### 核心場景

```
第四天的早晨。

嚴柏翰靠在走廊牆壁上。
他從第一天起就睡在走廊裡。
背靠水泥牆。軍用睡袋。
枕頭是他的黑風衣折成的。

他醒得很早。
探員的生理時鐘——凌晨五點。
改不掉。

走廊的應急燈在凌晨模式下只開一半。
光線像稀釋過的牛奶。
他的影子被拉得很長。

操作台那邊傳來鍵盤聲。
她也醒了。
或者她沒有睡。

他站起來。
走到操作台旁邊。
三步距離。

她在打字。
數據。報告。城市修復進度。
指尖在鍵盤上的節奏很穩。
像節拍器。

他站在她身後。
跟昨天一樣。
跟前天一樣。

他沒有說話。

她也沒有。

五分鐘。十分鐘。
鍵盤聲持續。
偶爾停一下——她在思考。
然後繼續。

嚴柏翰看著她的後腦勺。
短髮。髮際線後面有一顆小痣。
在她左耳後方一公分。

他看過無數次。
在另一個身份裡。
在被刪除的記憶裡。

現在他看到了。
用恢復後的眼睛。
痣還在那裡。

他開口了。

不是問句。不是命令。
他的聲音很低。
像在講一個與她無關的事實。

「左耳後面一公分有一顆痣。
  妳夏天會曬傷那裡。
  因為妳把頭髮紮起來的時候，
  那個位置剛好露出來。」

鍵盤聲停了。

邱韻如的手指懸在空中。

她的大腦在做什麼——
在搜索「左耳後面」「痣」「夏天」「曬傷」。
沒有結果。
資料庫裡沒有。

但她的手——
她的左手從鍵盤上抬起來。
慢慢地。
像被線牽著。

手指碰到了左耳後方。

碰到了那顆痣。

她的手知道那顆痣在哪裡。

然後——

她的眼淚掉下來了。

不是哭。
她的表情沒有變。
她的嘴唇沒有動。
她的呼吸沒有亂。

但眼淚——
在她的意識理解「為什麼」之前——
就掉下來了。

一滴。
從右眼。
沿著鼻翼滑下。
落在鍵盤上。

噠。

空調的嗡嗡聲。

她低頭看著鍵盤上的那滴水。
像看一件不屬於自己的東西。

她不知道為什麼哭。

但她的眼睛知道。


嚴柏翰站在原地。

他沒有靠近。

三步。
跟第一天一樣。
跟第二天一樣。
跟第三天一樣。

他站在那裡。
像一根柱子。
十年沒有倒的柱子。

不是因為堅強。
是因為他知道——
如果他靠近一步，
她的眼淚會停。
因為她的防禦機制會啟動。
「陌生人入侵安全距離」——
鸚鵡計畫寫在她意識裡的第一道防線。

但三步之外——
她的身體可以自由反應。
不觸發任何警報。

嚴柏翰站在三步之外。
用三步的距離，
給她的身體留一個哭泣的空間。

十年前他學會了靠近她的方式。
十年後他學會了不靠近她的方式。

都是愛。
```

### 逐行解讀

| 台詞/細節 | 設計意圖 |
|-----------|---------|
| 軍用睡袋、黑風衣當枕頭 | 探員本色。他不是來度假的。他睡在走廊裡 = 他不進她的空間，但不離開 |
| 凌晨五點生理時鐘 | 回收嚴柏翰的探員設定。十年的職業習慣。改不掉 = 身體記得 |
| 髮際線後面一顆小痣 | 全章核心道具。這顆痣是嚴柏翰恢復記憶後「看到」的細節——在另一個身份裡他看過無數次。現在他用它來觸發邱韻如的身體記憶 |
| 「妳夏天會曬傷那裡」 | 他的台詞不是「你認識我嗎」——是一個只有親密的人才知道的瑣碎細節。痣、夏天、紮頭髮。這些不是重要的記憶。正因為不重要，鸚鵡計畫才沒有清理它 |
| 她的手碰到左耳後方 | 身體記得的核心時刻。她的手知道痣在哪裡——但她的意識不知道為什麼知道。手的動作比意識快 |
| 眼淚在意識理解之前掉下來 | 章名「身體記得」的直接呈現。不是她「想哭」——是她的淚腺記得一個人。淚腺比大腦誠實 |
| 噠。 | 淚珠落在鍵盤上的聲音。在安靜的環境裡這個聲音被放大了。一個字。一滴淚。讀者也聽到了 |
| 三步距離不靠近 | 嚴柏翰弧線最深層的設計：他知道靠近會觸發防禦機制。所以他用「不靠近」來保護她的身體反應。不靠近 = 愛的最安靜的形式 |
| 「十年前學會靠近，十年後學會不靠近」 | 全章收尾。不是台詞——是敘述。嚴柏翰弧線的濃縮：他在兩個版本的自己裡用相反的方式愛同一個人 |

薛丁格的花苞。

沒有任何理性理由。

每天早上。
拿起水壺。
傾斜 15 度。
水流的粗細、持續的秒數、

幾十年的肌肉記憶。
不是「照顧一盆花」的記憶。

是她送的。
她每天早上幫他澆水。
因為他總忘。

她不記得了。
但她的手記得。

水壺的角度。
水流的粗細。
花盆邊緣的弧線。

一模一樣。


嚴柏翰走近了一步。

不是走向她。
是走向花。

他伸出手。
食指碰了碰那朵快開的花苞。

花瓣上有一滴水。

沒有聲音。

轉回去。
繼續打字。

但她的呼吸——
從每分鐘十六次變成了十八次。

嚴柏翰把手收回來。

他走回三步之外的位置。

站好。

他在口袋裡捏了捏那隻紙鶴。
紙已經軟了。

還在。

快枯了。但還在。
