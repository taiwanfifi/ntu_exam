# 電磁學教學講義 08：電磁波反射與透射

---

## 🔰 **本章基礎觀念（零基礎必讀）**

### 為什麼電磁波會反射？

想像你站在游泳池邊，把一條繩子綁在牆上，然後甩一個波過去。波碰到牆後會**彈回來**——這就是反射。

電磁波也一樣。當它從一種材料（如空氣）進入另一種材料（如玻璃），在界面上就會產生**反射**（部分能量彈回來）和**透射**（部分能量穿過去）。

反射的多寡取決於兩邊材料的「波阻抗」差異——差異越大，反射越多。這就像光線碰到鏡子（玻璃和鍍銀層的阻抗差很大 → 大部分反射）或碰到透明玻璃（阻抗差小 → 大部分透射）。

---

## 一、垂直入射（Normal Incidence）

### 1.1 基本設定

電磁波從介質 1（$\varepsilon_1, \mu_1, \eta_1$）垂直射入介質 2（$\varepsilon_2, \mu_2, \eta_2$），界面在 z = 0。

**入射波（Incident Wave）：**
$$\vec{E}_i = E_0 e^{-j\beta_1 z}\hat{x}$$

**反射波（Reflected Wave）：**
$$\vec{E}_r = \Gamma E_0 e^{+j\beta_1 z}\hat{x}$$

**透射波（Transmitted Wave）：**
$$\vec{E}_t = \tau E_0 e^{-j\beta_2 z}\hat{x}$$

### 1.2 反射係數與透射係數

由邊界條件（E 的切線分量連續、H 的切線分量連續）：

$$\boxed{\Gamma = \frac{\eta_2 - \eta_1}{\eta_2 + \eta_1}}$$

$$\boxed{\tau = \frac{2\eta_2}{\eta_2 + \eta_1}}$$

其中：
- $\Gamma$（Gamma）：**反射係數**（Reflection Coefficient），$-1 \leq \Gamma \leq 1$
- $\tau$（tau）：**透射係數**（Transmission Coefficient）

**重要關係：**
$$\tau = 1 + \Gamma$$

### 📝 例題 1：空氣-玻璃界面

**題目：** 電磁波從空氣（$\eta_1 = 377\;\Omega$）垂直入射到玻璃（$\varepsilon_r = 4$, $\mu_r = 1$，$\eta_2 = 377/\sqrt{4} = 188.5\;\Omega$）。求反射係數、透射係數和反射功率百分比。

**解：**

$$\Gamma = \frac{188.5 - 377}{188.5 + 377} = \frac{-188.5}{565.5} = -0.333$$

$$\tau = 1 + \Gamma = 1 - 0.333 = 0.667$$

反射功率百分比：$|\Gamma|^2 = 0.333^2 = 0.111 = 11.1\%$

透射功率百分比：$1 - |\Gamma|^2 = 88.9\%$

> 負的 $\Gamma$ 代表反射波的電場方向反轉（相位差 180°）。入射到較「密」的介質（$\eta_2 < \eta_1$）時 $\Gamma < 0$。

**驗證功率守恆：**

$$\frac{|\tau|^2 \eta_1}{eta_2} = \frac{0.667^2 \times 377}{188.5} = \frac{0.445 \times 377}{188.5} = \frac{167.6}{188.5} = 0.889$$

$|\Gamma|^2 + \frac{\eta_1}{\eta_2}|\tau|^2 = 0.111 + 0.889 = 1$ ✓

---

### 1.3 特殊情況

| 情況 | $\Gamma$ | 物理意義 |
|------|---------|---------|
| $\eta_2 = \eta_1$（匹配） | 0 | 完全透射，無反射 |
| $\eta_2 = 0$（完美導體） | -1 | 完全反射，E 反向 |
| $\eta_2 = \infty$ | +1 | 完全反射，E 同向 |
| $\eta_2 > \eta_1$ | > 0 | 部分反射，E 同向 |
| $\eta_2 < \eta_1$ | < 0 | 部分反射，E 反向 |

---

### 1.4 駐波（Standing Wave）

當有反射波時，入射波和反射波疊加產生**駐波**。

在介質 1 中，總電場：

$$E_{total} = E_0(e^{-j\beta_1 z} + \Gamma e^{+j\beta_1 z})$$

**電壓駐波比（VSWR, Voltage Standing Wave Ratio）：**

$$\boxed{\text{VSWR} = \frac{|E|_{max}}{|E|_{min}} = \frac{1 + |\Gamma|}{1 - |\Gamma|}}$$

- VSWR = 1：完美匹配（$\Gamma = 0$），無駐波
- VSWR = ∞：完全反射（$|\Gamma| = 1$），純駐波
- 一般情況：1 < VSWR < ∞

### 📝 例題 2：VSWR 計算

**題目：** 承例題 1，$|\Gamma| = 0.333$。求 VSWR。

**解：**

$$\text{VSWR} = \frac{1 + 0.333}{1 - 0.333} = \frac{1.333}{0.667} = 2.0$$

也就是說，駐波的最大值是最小值的 2 倍。

## 二、斜入射（Oblique Incidence）

### 2.1 基本設定

電磁波以角度 $\theta_i$ 入射到界面。

**入射面（Plane of Incidence）**：由入射波的傳播方向和界面法向量所組成的平面。

根據電場相對於入射面的方向，分成兩種極化：

| 極化 | 別名 | 電場方向 |
|------|------|---------|
| TE（Transverse Electric） | s 極化、垂直極化 | E 垂直於入射面 |
| TM（Transverse Magnetic） | p 極化、平行極化 | E 平行於入射面 |

### 2.2 Snell 定律

$$n_1\sin\theta_i = n_2\sin\theta_t$$

其中 $n = \sqrt{\mu_r\varepsilon_r}$ 是折射率。

也可以寫成：$\beta_1\sin\theta_i = \beta_2\sin\theta_t$

**反射定律：** $\theta_r = \theta_i$（反射角 = 入射角）

### 📝 例題 3：Snell 定律

**題目：** 光線從空氣（$n_1 = 1$）以 $\theta_i = 45°$ 入射到水中（$n_2 = 1.33$）。求折射角。

**解：**

$$\sin\theta_t = \frac{n_1}{n_2}\sin\theta_i = \frac{1}{1.33}\sin 45° = \frac{0.7071}{1.33} = 0.5317$$

$$\theta_t = \sin^{-1}(0.5317) = 32.1°$$

光線進入水中後「靠近法線」（$\theta_t < \theta_i$），因為水比空氣「密」（$n_2 > n_1$）。

---

### 2.3 Fresnel 方程

#### TE 波（s 極化）反射與透射係數

$$\Gamma_{TE} = \frac{\eta_2\cos\theta_i - \eta_1\cos\theta_t}{\eta_2\cos\theta_i + \eta_1\cos\theta_t}$$

$$\tau_{TE} = \frac{2\eta_2\cos\theta_i}{\eta_2\cos\theta_i + \eta_1\cos\theta_t}$$

#### TM 波（p 極化）反射與透射係數

$$\Gamma_{TM} = \frac{\eta_2\cos\theta_t - \eta_1\cos\theta_i}{\eta_2\cos\theta_t + \eta_1\cos\theta_i}$$

$$\tau_{TM} = \frac{2\eta_2\cos\theta_i}{\eta_2\cos\theta_t + \eta_1\cos\theta_i}$$

> 記憶技巧：TE 的公式中，$\cos\theta_i$ 配 $\eta_2$，$\cos\theta_t$ 配 $\eta_1$；TM 反過來。

---

## 三、特殊角度

### 3.1 布魯斯特角（Brewster Angle）

**定義：** TM 波（p 極化）反射係數為零的入射角。

$$\Gamma_{TM} = 0 \quad \Rightarrow \quad \eta_2\cos\theta_t = \eta_1\cos\theta_i$$

對於非磁性材料（$\mu_1 = \mu_2 = \mu_0$）：

$$\boxed{\theta_B = \tan^{-1}\left(\frac{n_2}{n_1}\right) = \tan^{-1}\sqrt{\frac{\varepsilon_2}{\varepsilon_1}}}$$

> 白話：在布魯斯特角入射時，p 極化完全不反射（100% 透射）。偏光太陽眼鏡就是利用這個原理——水面反射的光主要是 s 極化（因為 p 極化在接近布魯斯特角時反射很少），眼鏡阻擋 s 極化來減少眩光。

### 📝 例題 4：布魯斯特角

**題目：** 光線從空氣入射到玻璃（$n = 1.5$）。求布魯斯特角。在此角度，TE 波的反射係數是多少？

**解：**

$$\theta_B = \tan^{-1}(n_2/n_1) = \tan^{-1}(1.5/1) = \tan^{-1}(1.5) = 56.3°$$

在 $\theta_i = 56.3°$ 時：

$$\sin\theta_t = \frac{\sin 56.3°}{1.5} = \frac{0.832}{1.5} = 0.555 \quad \Rightarrow \quad \theta_t = 33.7°$$

注意 $\theta_i + \theta_t = 56.3° + 33.7° = 90°$（布魯斯特角的一個特性！）

TE 波的反射係數（在布魯斯特角）：

$$\Gamma_{TE} = \frac{\eta_2\cos\theta_i - \eta_1\cos\theta_t}{\eta_2\cos\theta_i + \eta_1\cos\theta_t}$$

$\eta_1 = 377\;\Omega$, $\eta_2 = 377/1.5 = 251.3\;\Omega$

$$= \frac{251.3\cos 56.3° - 377\cos 33.7°}{251.3\cos 56.3° + 377\cos 33.7°} = \frac{251.3 \times 0.555 - 377 \times 0.832}{251.3 \times 0.555 + 377 \times 0.832}$$

$$= \frac{139.5 - 313.7}{139.5 + 313.7} = \frac{-174.2}{453.2} = -0.384$$

所以在布魯斯特角，TM 波反射為零，但 TE 波仍有 38.4% 的反射（功率反射 = 14.7%）。

---

### 3.2 臨界角（Critical Angle）與全反射

當波從**較密的介質**（$n_1 > n_2$）入射到較疏的介質時，可能發生**全反射（Total Internal Reflection）**。

$$\boxed{\theta_c = \sin^{-1}\left(\frac{n_2}{n_1}\right)}$$

當 $\theta_i > \theta_c$ 時：
- $\sin\theta_t > 1$ → $\theta_t$ 不存在（折射角沒有實數解）
- 所有能量被反射回介質 1
- $|\Gamma| = 1$

> 白話：光從玻璃射向空氣，如果角度太大（超過臨界角），光就完全彈回玻璃裡出不去。光纖（光纜）就是靠全反射把光「困」在細玻璃棒裡傳輸的！

### 📝 例題 5：全反射

**題目：** 光纖核心 $n_1 = 1.5$，包覆層 $n_2 = 1.45$。求臨界角。

**解：**

$$\theta_c = \sin^{-1}\left(\frac{1.45}{1.5}\right) = \sin^{-1}(0.9667) = 75.2°$$

光線在核心中的入射角只要大於 75.2°（和界面法線的夾角），就會發生全反射而不會漏出包覆層。

數值孔徑（Numerical Aperture）：

$$\text{NA} = \sqrt{n_1^2 - n_2^2} = \sqrt{1.5^2 - 1.45^2} = \sqrt{2.25 - 2.1025} = \sqrt{0.1475} = 0.384$$

## 四、多層介面反射

### 4.1 等效阻抗法

對於多層介質結構，可以從最後一層往回推算，逐層計算「等效阻抗」。

在 z = -d 處（距離界面 d），介質 2 前面看到的等效阻抗：

$$Z_{in} = \eta_2 \frac{\eta_3 + \eta_2\tanh(\gamma_2 d)}{\eta_2 + \eta_3\tanh(\gamma_2 d)}$$

對無損介質（$\gamma_2 = j\beta_2$）：

$$Z_{in} = \eta_2 \frac{\eta_3 + j\eta_2\tan(\beta_2 d)}{\eta_2 + j\eta_3\tan(\beta_2 d)}$$

### 4.2 四分之一波長抗反射膜

如果中間層的厚度恰好是 $\lambda_2/4$（$\beta_2 d = \pi/2$）：

$$Z_{in} = \frac{\eta_2^2}{\eta_3}$$

要讓反射為零（$Z_{in} = \eta_1$），需要：

$$\eta_2 = \sqrt{\eta_1 \eta_3}$$

即 $n_2 = \sqrt{n_1 n_3}$

> 白話：在玻璃表面鍍一層「剛剛好」的薄膜（折射率是空氣和玻璃的幾何平均值，厚度是四分之一波長），就能消除反射！這就是相機鏡頭上那層「抗反射鍍膜」的原理。

### 📝 例題 6：抗反射膜

**題目：** 光線從空氣（$n_1 = 1$）入射到玻璃（$n_3 = 1.5$）。設計一層抗反射膜（$\lambda = 550$ nm，綠光）。

**解：**

所需折射率：$n_2 = \sqrt{n_1 \times n_3} = \sqrt{1 \times 1.5} = 1.225$

（實際常用 MgF₂，$n = 1.38$，不完美但效果還不錯）

薄膜厚度：

$$d = \frac{\lambda_2}{4} = \frac{\lambda_0/n_2}{4} = \frac{550/1.225}{4} = \frac{449}{4} = 112.2 \text{ nm}$$

## 五、題型鑑別

| 看到什麼 | 用什麼方法 |
|---------|----------|
| 「垂直入射到界面」 | $\Gamma = (\eta_2-\eta_1)/(\eta_2+\eta_1)$ |
| 「入射到完美導體」 | $\Gamma = -1$，完全反射 |
| 「求 VSWR」 | VSWR = $(1+|\Gamma|)/(1-|\Gamma|)$ |
| 「斜入射」 | 先分 TE/TM，再用 Fresnel 方程 |
| 「求折射角」 | Snell 定律 $n_1\sin\theta_i = n_2\sin\theta_t$ |
| 「p 極化無反射」 | 布魯斯特角 $\theta_B = \tan^{-1}(n_2/n_1)$ |
| 「從密 → 疏、角度很大」 | 全反射，臨界角 $\theta_c = \sin^{-1}(n_2/n_1)$ |
| 「多層結構」 | 等效阻抗法，從後往前推 |
| 「抗反射膜」 | $n_2 = \sqrt{n_1 n_3}$，$d = \lambda_2/4$ |

## ✅ 自我檢測

### Q1：垂直入射
電磁波從 $\varepsilon_r = 1$ 的介質入射到 $\varepsilon_r = 16$ 的介質（都是非磁性）。反射功率是多少百分比？

<details><summary>點擊展開答案</summary>

$$\eta_1 = \eta_0, \quad \eta_2 = \eta_0/\sqrt{16} = \eta_0/4$$

$$\Gamma = \frac{\eta_0/4 - \eta_0}{\eta_0/4 + \eta_0} = \frac{1/4 - 1}{1/4 + 1} = \frac{-3/4}{5/4} = -\frac{3}{5} = -0.6$$

反射功率：$|\Gamma|^2 = 0.36 = 36\%$

</details>

### Q2：VSWR
若 $\Gamma = 0.5$，VSWR 為何？

<details><summary>點擊展開答案</summary>

$$\text{VSWR} = \frac{1 + 0.5}{1 - 0.5} = \frac{1.5}{0.5} = 3$$

</details>

### Q3：布魯斯特角
光線從水（$n = 1.33$）入射到空氣（$n = 1$）。布魯斯特角為何？

<details><summary>點擊展開答案</summary>

$$\theta_B = \tan^{-1}\left(\frac{n_2}{n_1}\right) = \tan^{-1}\left(\frac{1}{1.33}\right) = \tan^{-1}(0.752) = 36.9°$$

</details>

### Q4：全反射
光線從鑽石（$n = 2.42$）到空氣。臨界角為何？這和鑽石很閃亮有什麼關係？

<details><summary>點擊展開答案</summary>

$$\theta_c = \sin^{-1}\left(\frac{1}{2.42}\right) = \sin^{-1}(0.413) = 24.4°$$

臨界角只有 24.4°！這意味著光線在鑽石內部只要碰到界面的角度稍微大一點（超過 24.4°），就會全反射而留在鑽石裡面。光在鑽石內部來回反射多次才最終從某些角度射出，所以鑽石看起來特別閃亮。

對比玻璃（$n = 1.5$）的臨界角是 41.8°，光更容易「漏」出去。

</details>
