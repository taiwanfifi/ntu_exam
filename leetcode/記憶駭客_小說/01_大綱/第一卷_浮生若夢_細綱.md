> **核心命題**：在記憶可以被偽造的世界，什麼才是真實的？
> **情感主軸**：從孤獨到信任，從信任到心碎
> **演算法覆蓋**：Array / Two Pointers, Sliding Window, HashMap / HashSet, Stack / Queue, Linked List, Binary Search

## 第 3 章：地下街的早餐

| 欄位 | 內容 |
|------|------|
| **章節標題** | 地下街的早餐 |
| **一句話概要** | 維倫在舊台北地下街吃早餐，遇見了新台北底層社會的眾生相。 |
| **劇情概要** | 維倫每天早上去地下街的老攤吃鹹豆漿配燒餅。攤主阿嬤是第四階的記憶負債者——為了還債被強制提取過三次記憶，現在只記得怎麼做鹹豆漿。「記住怎麼做這碗豆漿，比記住我是誰重要。」她笑著說。但維倫注意到一件事：阿嬤每天早上開攤前，都會先做一碗豆漿放在攤位角落。那碗不賣。等到收攤時倒掉。維倫問過一次：「那碗是留給誰的？」阿嬤想了半天：「我不記得了。但我每天都覺得該留一碗。」——她的記憶被提取了，但她的身體記得有一個人每天會來喝豆漿。那個人也許是她的兒子，也許是丈夫。她已經不知道了。但那碗永遠留著的豆漿，是整條地下街最安靜的悲劇。地下街是新台北的底層生態：有人兜售盜版技能記憶（「學鋼琴只要三百塊」），有人在角落交易禁忌的情感記憶（「想體驗初戀的感覺嗎？」），有巡邏的深淵監控局無人機在頭頂嗡嗡飛過。維倫在這裡是半個名人——地下街的人都知道他是最好的黑市修復師。今天有人等他：一個十歲的男孩，抱著一個記憶存儲盒，眼眶紅紅的。 |
| **情緒基調** | 煙火氣、溫暖中帶苦澀、市井人間 |
| **演算法** | 無 |
| **委託案** | 無（但為委託案 #2 埋下伏筆） |
| **主線推進** | 深化世界觀：記憶經濟的底層生態、階層壓迫的日常化、深淵監控局的存在感；維倫在社區中的角色（底層的守護者）；男孩小杰初次出現 |
| **章末鉤子** | 男孩把記憶存儲盒推到維倫面前。「叔叔，我媽媽走之前留了一段記憶給我。但是......我打開的時候，裡面全是雪花。」維倫打開盒子，掃了一眼數據。記憶確實嚴重損壞——但損壞的方式很奇怪。不像自然衰減，更像......被什麼東西覆蓋了。「你媽媽什麼時候離開的？」「三個月前。醫生說是記憶衰竭症。」米亞在維倫耳邊低語：「記憶衰竭症的損壞模式不是這樣的。這段記憶是被人動過手腳的。」 |

## 第 5 章：米亞的理論

| 欄位 | 內容 |
|------|------|
| **章節標題** | 米亞的理論 |
| **一句話概要** | 米亞提出「未來記憶」的三種可能假說，維倫決定先看一眼那段記憶。 |
| **劇情概要** | 維倫無法入睡。他坐在修復台前，和米亞討論林語青帶來的記憶可能是什麼。米亞提出三種假說：一、精密偽造（但時間戳無法偽造）；二、時間線殘留（理論上可能但從未被記錄過）；三、某種未知的記憶協議版本。維倫做了一個決定——他用最小權限模式打開那段未來記憶，只讀取最外層的感官數據。他看到的畫面是：一座燃燒的城市。新台北的天際線在火焰中扭曲。街上到處是失去記憶的空白人，像行屍走肉。然後畫面切到一個房間——他自己的工作室。他看到了自己。螢幕上的自己滿臉是血，對著鏡頭說了一句話。但音頻數據損壞了，他聽不見。只看到自己的嘴型。米亞做了唇語分析：「他說的是——『別相信她。』」 |
| **情緒基調** | 懸疑升級、不安、心跳加速 |
| **演算法** | 無 |
| **委託案** | 無 |
| **主線推進** | 未來記憶的第一次窺視——建立全卷的核心懸念：新台北會毀滅？維倫自己警告自己不要相信「她」（是語青嗎？）；三人組合的信任裂縫從第一卷開始就埋下 |
| **章末鉤子** | 「別相信她。」他——未來的他——到底在說誰？維倫關上了記憶檔案。手指在發抖。他做了第二個決定：明天，去找林語青。不是為了修復她的記憶。是為了問她一個問題。「你說裡面有我。那個『我』——是活著的，還是死了的？」第二天早上，他在地下街找到了她。她坐在阿嬤的攤位前，正在學怎麼喝鹹豆漿。看到他的時候，她笑了。那個笑容讓維倫突然覺得，不管未來記憶裡發生了什麼，此刻的她不像一個會被警告的人。 |

| **情緒基調** | 日常、輕鬆、淡淡的曖昧 |
| **演算法** | 無 |
| **委託案** | 無 |

| **章末鉤子** | 回工作室的路上，維倫注意到一件事：語青走路的時候，總是走在他的左邊。每一次。不管他怎麼換位置，她都會不自覺地繞到他的左邊。「你為什麼總走這邊？」語青愣了一下，像是被抓到了什麼：「因為......你以前——」她突然停住。「沒什麼。習慣了。」維倫沒有追問。但米亞在他的耳機裡說：「她剛才的心率飆升了 40%。她差一點說出什麼。」 |

## 第 7 章：記憶銀行的廣告
