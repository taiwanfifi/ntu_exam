## ch208 一鍋多煮的豆漿

**情緒強度：6（阿嬤弧線——另一種遊行）**

---

### 核心場景

```
ch207 Trie 搜尋帶回了更多碎片。
維倫在修復台上展開它們。

這一次不是阿嬤的主觀視角。
是她的攤位——被動收錄的環境記憶。
像一台忘了關的監視器。
只是那台監視器是一鍋冒著蒸氣的豆漿。


十一月一日。
遊行前一天。

清晨五點半。
天還是暗的。
阿嬤已經在磨豆子了。

記憶裡沒有畫面清晰度。
只有聲音。
磨石機的嗡嗡聲。
水管裡的嘶嘶聲。
鋁鍋碰到爐架時的一聲叮。

然後是味道。
黃豆磨開時那種——
帶著一點點甜的、生的、濃稠的氣味。
幾十年前叫豆漿。
現在也叫豆漿。
名字沒變過。


五點四十五。

第一鍋豆漿開始冒泡。
阿嬤把火調小。
泡沫從鍋邊緩緩往內收。

這個動作她做了一輩子。
火大了豆漿溢出來。
火小了煮不透。
剛剛好的火——
要靠手掌舉在鍋邊感覺溫度。

不是溫度計。
是手掌。

她的手知道。


六點整。
周明哲來了。

（ch198 的畫面。ch199 的「明天見」。
 讀者已經看過了。本章不重播。
 只用一句帶過。）

那個年輕人坐在塑膠凳上。
膝蓋到胸口。
阿嬤端了一碗鹹的。


六點十五。
第二批客人。

一個穿制服的年輕女人。
二十歲出頭。
臉上有明顯的黑眼圈。
她點了一碗甜豆漿。
雙手捧著碗，沒有喝。
只是把臉湊在碗邊。
蒸氣在她的睫毛上凝了水珠。

她在暖自己。

阿嬤看了她一眼。
什麼都沒說。
多給了她一根油條。

「阿嬤我沒點——」

「今天的油條炸太多了。吃吧。」

她其實炸得剛好。
每天都剛好。


六點半。
地下街開始喧鬧。

有人在傳單子。
紅色的紙。黑色的字。
阿嬤看不清上面寫什麼——
她的老花眼鏡放在攤位底下某個角落，
已經很久沒找到了。

有人走過她的攤位時加快了腳步。
有人停下來張望了一下又走掉。
有人拿著大聲公從通道那頭跑過去。

阿嬤不看他們。
她在看鍋裡的豆漿。

泡沫又起來了。
她調了火。


「阿嬤，明天是不是不開？」

一個常客。中年男人。工地的。
每天兩碗鹹豆漿配三根油條。

阿嬤轉頭。
「為什麼不開？」

「明天不是——」
他壓低了聲音。
「遊行嗎？」

阿嬤聽不太清楚。
她把耳朵湊過去。

「遊行？」

她的語氣像在問一個不認識的詞。

「遊行跟我賣豆漿有什麼關係？」

那個男人沒有回答。
他把碗裡的豆漿喝完了。
油條只吃了兩根。
第三根帶走了。

阿嬤收碗的時候嘀咕了一句。

「遊行遊行。
 每次一遊行油條就賣不完。
 上次也是。上上次也是。
 人一多就跑來跑去的。
 跑來跑去不吃東西。
 不吃東西我炸的油條怎麼辦。」

她把火關小了一點。

「一鍋多煮的豆漿最後都是我自己喝。」


——維倫退出播放。

螢幕暗了。

他坐在那裡。
嘴角動了一下。

不是笑。
是——理解。

遊行在教科書裡是歷史事件。
遊行在嚴柏翰的記憶裡是傷疤和鎮暴盾牌。
遊行在邱韻如的記憶裡是她做出選擇的前夜。
遊行在周明哲的記憶裡是他最後走向人群中心的那條路。

遊行在阿嬤的記憶裡——
是油條賣不完。
是一鍋多煮的豆漿。

她不在乎遊行。
她在乎明天還要不要開攤。

而答案永遠是：開。

不管外面發生什麼。
她開攤。磨豆子。炸油條。
等那個六點來的年輕人。

因為豆漿不會自己煮。
油條不會自己炸。
人不會自己吃飽。

這是她的遊行。
每天清晨五點半。
一個人的。安靜的。
跟歷史無關的遊行。
```

## ch209 那碗永遠留著的

```
記憶的更深處。

不是十一月一日了。
不是十一月二日了。
是——之後。

很久之後。


記憶裡的畫面模糊了。
不是因為損壞。
是因為每一天都太像了。

豆漿冒泡。調火。
塑膠凳。四把。

維倫看到的是一段——
沒有日期的記憶。

因為對阿嬤來說，
從那天起，
每一天都一樣。


鹹豆漿。醬油少一點。

放在那個位置。
那張塑膠凳前面。

六點。
六點零五。
六點一十。

涼了。
換一碗。

換一碗。

七點。

 在第三次提取時被帶走了。

米亞的語速慢了一點。
不是計算延遲。

一碗又一碗。
一天又一天。
三千六百五十碗。

她想起了那個零點五秒的背影。
灰色棉外套。口袋裡的筆。

那個年輕人走向人群中心。
然後消失了。

三千六百五十天后，
一個老婦人還在為他留一碗豆漿。

她不記得他了。

多一碗的量。


維倫關掉播放。

他沒有說話。

窗外下著雨。
十一月的雨。該下就下了。

米亞的光球轉了一圈。很慢。

 第一百天是本能。

 不需要理由的那種。」


語青轉過身。
走向門口。

維倫：「語青？」

她停了一下。
沒有回頭。

「我去買兩碗豆漿。」

她走出去了。

維倫看著她的背影。
門在她身後輕輕關上。

他低下頭看修復台的螢幕。
螢幕是黑的。

上面映出了他自己的臉。
和一雙——
不知道為什麼有點模糊的眼睛。
```
