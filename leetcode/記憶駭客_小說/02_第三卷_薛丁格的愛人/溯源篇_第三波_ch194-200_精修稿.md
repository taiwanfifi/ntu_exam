## ch194 黑市的海底

**情緒強度：5（溯源篇最低強度 = 喘息）**
**演算法：Trie（第四次出場——黑市記憶池前綴搜尋）**

---

### 核心場景

```
深夢黑市。崩潰後第十一天。

入口不再是娃娃機店的背板了。覆蓋波衝擊時電力過載，
整面外牆炸裂，露出通往地底的鐵樓梯。
門口噴漆字跡模糊，只剩半句：
「記憶不賣——」

後半句被水漬蓋住了。

維倫走下去。
鞋底踩在積水上。拍。拍。拍。
每一步都有微弱的回音。

天花板管線外露。銅管上結了冷凝水珠。
滴答。

應急燈和 LED 燈條把一切染成藍白色。
人的臉在這種光下看起來像遊魂。
空氣裡是潮濕的水泥味、廉價記憶晶片的臭氧味，
和角落裡有人在吃泡麵的鹹味。

鄭浩然在等他。

「歡迎來到海底。」

他們穿過殘破的 A 區，繞過倒塌的貨架，
走進蘇曼菲的倉庫深處。

七百八十台伺服器。
排列成不規則的矩陣。
指示燈以不同頻率閃爍——
藍色、黃色、偶爾一盞紅色。
像一個在呼吸的生物體。

風扇的白噪音。低沉、持續、像海浪。
但比海浪冷。沒有溫度的聲音。

維倫戴上記憶提取器。

他「看到」的不是數據。
是一片深藍色的海洋。
一千兩百萬段記憶碎片像水母一樣在海中漂浮。
每一個都在發微弱的光。
白色。琥珀色。偶爾一閃粉紅——
那是帶著強烈情感標記的碎片。

「這片海有三層。」

鄭浩然的聲音從外面傳進來，像隔了一面水牆。

「表層是最近三天的碎片。新鮮的。
 中層是崩潰前幾個月的。
 底層——」

他停了一下。

「底層是十年前的垃圾。沒人看。」

維倫：「我要的就在底層。」

米亞的聲音亮起來。

「開始建樹。」

她的語氣不像在開玩笑。
也不像在報告數據。
像一個工匠拿起了她最精密的工具。

「前綴條件——
 時間：2079 年 10 月。
 地點：地下街。
 提取類型：債務催收。」

光的海洋開始旋轉。
一千兩百萬個光點同時震了一下。

然後開始消失。

「第一層過濾。」
米亞的聲音沉穩。
「時間前綴 2079-10——匹配 47,000 段。」

四萬七千個光點留下來。
其餘的暗了。

「第二層過濾。地點前綴：地下街——」

光點又消失了一大片。

「匹配 4,700 段。」

「第三層。提取類型：債務催收——」

海洋幾乎全暗了。
只剩二十三個光點。

孤零零地飄在深藍色的黑暗裡。
像深海裡的水母。
沒有同伴。沒有光源。
只是在那裡。
等了十年。

「23 段。」

米亞的聲音輕了一點。

「其中編號 JX-2079-10-28-003。
 提取時間：2079 年 10 月 28 日。
 五天後鸚鵡計畫啟動。
 這段記憶在鸚鵡計畫之前就離開了主人的大腦。」

維倫看著那個光點。

灰色的。不發光。

因為沒有人覺得它「重要」。
在所有記憶碎片按情感標記定價的系統裡，
一個老太太炸油條的記憶，
標價是零。

「這段記憶——」
米亞停了一下。比平常長。

「是全城唯一未被鸚鵡計畫篡改的原始版本。」

維倫伸出手。
在海洋的最深處。
觸碰那個灰色的、不發光的光點。

他的手指碰到它的瞬間——
光點亮了。

不是白色。不是琥珀色。
是一種非常淡的暖黃。

像清晨炸油條時，
油鍋上方升起的那一縷熱氣。
```

| 台詞/細節 | 設計意圖 |
|-----------|---------|
| 「記憶不賣——」後半句被水漬蓋住 | 崩潰後的黑市已經不遵守任何規矩。Vol 1 的「記憶不賣，只換」已經是歷史 |

| 四萬七千 → 四千七百 → 二十三 | Trie 三層過濾的數字遞減 = 讓讀者「看到」演算法的效率。但重點不是效率——是在一千兩百萬中找到二十三 |

## ch195 一碗豆漿的重量

**情緒強度：6（阿嬤弧線高潮開始）**

---

### 核心場景

```
蘇曼菲把記憶包裹從伺服器裡調出來。

一個拳頭大小的數據球。
灰色。表面有細微的裂紋。
像一顆放了十年的鵝卵石。

「這段記憶在我的倉庫裡躺了十年。」

蘇曼菲看著它。
她的表情不像商人。像考古學家看著一塊沒有人在乎的化石。

「從來沒有人問過。
 你是第一個。」

維倫：「多少錢？」

蘇曼菲看了他一眼。

「免費。」

她把記憶包裹放在桌上。
灰色的球在藍白色的應急燈下毫無光澤。

「我收的那些高價記憶——
 初戀、婚禮、環遊世界的——
 都有人搶著買。
 但這種？」

她指了指那顆灰色的球。

「一個老太太炸油條的記憶。
 從來沒有人覺得它值一毛錢。」

她停了停。
指尖在灰色的球體表面輕輕劃了一下。

「但你來找了。」

她把球推向維倫。

「所以也許我收了十年的東西，
 終於有意義了。」


維倫把記憶包裹帶回修復台。

語青坐在角落。
她看著那顆灰色的球。

維倫接上讀取裝置。
數據開始展開。

米亞安靜了幾秒。

「記憶完整度——94.7%。」

她的語氣裡有一種維倫從未聽過的東西。

「未被覆蓋。未被篡改。未被任何人碰過。
 十年。」

維倫把球放進修復台的播放槽。

「鸚鵡計畫清理了政客、探員、記者——
 所有它認為重要的人。」

米亞的聲音放輕了。
不是計算。是理解。

「但它忽略了一個賣豆漿的老婦人。
 因為她不重要。」

停頓。

「然後——
 全城唯一的未篡改遊行記錄，
 就在她的記憶裡。」

維倫的手指懸在播放鍵上方。

語青站起來。走到他身邊。

她沒有說話。
但她站在那裡。

就像在說——
我陪你看。


維倫按下播放。

螢幕上出現的第一個畫面：

清晨。
一團油鍋裡的氣泡。
然後是油條。
金黃色的、膨脹的、正在被翻面的油條。

一雙老人的手。皺紋很深。
指尖上有燙傷的舊疤。
但動作穩定。每一個翻面的角度都是幾十年的肌肉記憶。

鹹豆漿在鋁鍋裡冒著煙。

這是阿嬤的早晨。
這是十年前的一個早晨。
這是全城唯一被完整保存下來的早晨。

米亞安靜了很久。
然後她說了一句不像她的話。

「在百萬段記憶裡——
 我們找到了一碗豆漿。」

（停頓 2.4 秒。）

「它比所有初戀和婚禮加起來都重。」
```

### 逐行解讀

| 台詞/細節 | 設計意圖 |
|-----------|---------|
| 灰色數據球像放了十年的鵝卵石 | 物理外觀呼應記憶的「無價值」定位。灰色 = ch194 的灰色光點延續 |
| 蘇曼菲「像考古學家」 | 她不是單純的商人。她守了這顆球十年。免費給維倫 = 她也在等一個把意義還給它的人 |
| 「從來沒有人覺得它值一毛錢」 | 核心諷刺：情感標記定價系統把「普通人的日常」標為零。= 全書「最不重要的人擁有最關鍵的證據」母題 |
| 記憶完整度 94.7% | 精確數字 = 米亞的語言習慣。但這次的精確帶著敬畏——「未被覆蓋、未被篡改、未被碰過」三連否定 = 乾淨得令人心疼 |
| 油鍋裡的氣泡 → 油條 → 老人的手 | 阿嬤記憶的第一個畫面 = 最日常的場景。不是遊行、不是衝突——是炸油條。這就是「最安靜的見證者」的視角 |
| 「它比所有初戀和婚禮加起來都重」 | 米亞用她的比較邏輯做了一個不理性的判斷。這是觸點 #11 的前置——她開始在乎「重量」而非「數據」 |

## ch196 不在出廠設定裡

**情緒強度：6（米亞觸點 #10 → 保護欲萌芽）**

---

### 核心場景

```
修復台上，阿嬤的記憶已經展開。
維倫準備繼續播放。

米亞的光球閃了一下。

然後暗了。

「維倫。」

（停頓 3.1 秒。光球灰藍色。）

他停下手。
這個停頓不正常。
米亞日常的回應速度是 0.1 秒。
3.1 秒——他只在 ch78 聽過這個量級的沉默。

「我建議你不要看下一段。」

維倫：「有危險嗎？」

「不是危險。」

（停頓 1.4 秒。）

「我不想看到你痛苦。」

安靜。

修復台的低頻嗡嗡聲。
伺服器風扇轉動的白噪音。
遠處有水滴的滴答聲。

維倫看著光球。

灰藍色。不是她平時的冷靜藍。
不是分析時的亮藍。
是一種他從未見過的顏色——
像薄霧。
像困惑。

（停頓 2.3 秒。語速變慢。）

「這個反應不在我的出廠設定裡。」

她的聲音比平常低了半個音階。

「我分析了四遍。
 下一段記憶不會對你的記憶結構造成損傷。
 沒有技術風險。
 沒有覆蓋波殘留。
 完全安全。」

她停了停。

「但我不想讓你看。」

又一次停頓。0.8 秒。比剛才短。
像她自己也在消化這句話。

「我花了 0.3 秒搜尋我的決策日誌。
 找不到任何邏輯來支撐這個建議。
 沒有理性原因。
 沒有風險評估。
 只有——」

她的光球從灰藍色轉成了一種更暗的顏色。
不是任何已知的色號。

「不想。」


維倫安靜了很久。

然後他笑了。
不是苦笑。不是勉強的笑。
是很溫暖的那種。
像聽到朋友說了一句蠢話——
但那句蠢話裡有真心。

「謝謝妳，米亞。」

米亞的光球跳了一下。
像被這三個字嚇到了。

「……不客氣？」

維倫伸手碰了碰光球邊緣。
光球閃了一下暖色。

「但我要看。」

米亞沉默了 0.6 秒。

「我知道。」

又沉默了 0.4 秒。

「所以我已經提前把你的壓力指數
 監測閾值調低了 15%。
 如果你的心率超過 130——
 我會強制暫停播放。」

維倫：「這也不在出廠設定裡吧？」

米亞：「……對。」

停頓 1.1 秒。

「但我覺得很合理。」


語青始終坐在角落。

她看著維倫和米亞的對話。
嘴角動了一下。
不是笑。
是——辨認。

她認出了什麼。
在一個 AI 的光球裡。
看到了一種她在另一條時間線裡
從未見過的東西。

因為在另一條時間線裡，
米亞沒有機會長到這裡。

語青低下頭。
她的左手食指——
閃了一下。
不到半秒。
```
