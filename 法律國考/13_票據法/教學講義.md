## 寫在最前面：這科到底在講什麼？

### 如果你從來沒學過法律，先讀這段

想像一個沒有銀行轉帳、沒有 LINE Pay、沒有信用卡的年代。

你是一個商人，要付一大筆貨款給供應商。帶著一箱現金上路？太危險了。於是人類發明了一種東西：**一張紙，上面寫著「我欠你錢」或「請某某銀行付錢給拿這張紙的人」。**

這張紙就是**票據**。

票據就是「紙上的錢」—— 它不是鈔票，但拿著它的人可以去要錢。而且最厲害的是，這張紙可以**轉讓**：A 欠 B 的錢，B 把這張紙交給 C，C 就可以直接去找 A 要錢。

### 三種票據，用生活場景理解

| 票據類型 | 白話說法 | 生活場景 |
|---------|---------|---------|
| **支票（Check）** | 「銀行，請從我的帳戶付錢給他」 | 你開一張支票給裝潢師傅，師傅拿去銀行領錢 |
| **本票（Promissory Note）** | 「我自己保證會付錢」 | 你寫一張本票給房東，承諾下個月 1 號付房租 |
| **匯票（Bill of Exchange）** | 「我叫別人替我付錢給你」 | 你叫你的客戶直接把貨款付給你的供應商 |

### 為什麼現在還要學這個？

你可能覺得：「現在大家都用轉帳了，誰還用票據？」

答案是：**票據法的法律原理在商業世界仍然非常重要。** 很多公司之間的大額交易還是會用票據（尤其是本票和支票），借貸時簽本票更是台灣非常常見的擔保方式。而且——

**國考超愛考！** 票據法在一試佔 20 分，題型固定，只要把核心觀念搞懂，這 20 分幾乎可以全拿。投資報酬率極高。

### 考試出題方向預告

根據歷年考古題分析，票據法最常考的主題：

1. **票據行為的獨立性**（偽造、變造的效力判斷）
2. **票據的無因性與抗辯限制**（善意取得人的保護）
3. **背書的連續性與類型**（空白背書、禁止轉讓背書、期後背書）
4. **本票裁定（§123）**（這是超級高頻考點！）
5. **追索權的行使要件**
6. **票據時效與利益償還請求權（§22 IV）**

## 第一部分：票據法通則 — 所有票據共通的基本原理

> 這個部分是整部票據法的「地基」。不管是匯票、本票還是支票，都適用這些基本原理。把這些搞懂，後面就是細節的差異而已。

---

### 一、票據的種類

票據法第 1 條規定，票據分為三種：

| 種類 | 法律定義 | 關係人 |
|------|---------|--------|
| **匯票（Bill of Exchange）** | 發票人簽發票據，委託付款人在到期日付款給受款人 | 三方關係：發票人、付款人、受款人 |
| **本票（Promissory Note）** | 發票人自己承諾在到期日付款給受款人 | 兩方關係：發票人、受款人 |
| **支票（Check）** | 發票人簽發票據，委託銀行付款給受款人 | 三方關係：發票人、付款人（銀行）、受款人 |

**白話理解：**

- 匯票：「我叫別人付錢給你」（三個人的事）
- 本票：「我自己保證付錢給你」（兩個人的事）
- 支票：「我叫銀行付錢給你」（其實是匯票的特殊版，付款人一定是銀行）

**生活場景：**

小明做生意，小華是他的供應商，小李欠小明一筆貨款。

- **匯票用法**：小明開一張匯票給小華，叫小李直接付錢給小華。（小明 = 發票人，小李 = 付款人，小華 = 受款人）
- **本票用法**：小明自己開一張本票給小華，承諾三個月後付清。（小明 = 發票人，小華 = 受款人）
- **支票用法**：小明開一張銀行支票給小華，小華拿去銀行領錢。（小明 = 發票人，銀行 = 付款人，小華 = 受款人）

### 二、票據行為的要式性（嚴格形式主義）

#### 白話解釋

票據是一種非常「講究格式」的法律文件。**少寫一個字、少蓋一個章，整張票可能就無效。**

為什麼要這麼嚴格？因為票據是會在不同人之間流通的「紙上的錢」。如果格式不統一、不嚴格，拿到票的人怎麼判斷這張票到底有沒有效？

#### 法律用語

票據法採**嚴格的要式主義（書面形式主義）**：票據行為必須依照法定方式為之，否則不生票據法上的效力。

#### 三種記載事項

| 類型 | 意思 | 效果 |
|------|------|------|
| **必要記載事項（絕對必要記載事項）** | 法律規定一定要寫的 | 缺少任何一項 → 票據無效 |
| **有害記載事項** | 法律不允許寫在票據上的 | 寫了 → 整張票據無效（或該記載無效） |
| **無害記載事項（任意記載事項）** | 可寫可不寫 | 寫了有效果，不寫也不影響票據效力 |

**生活場景：**

想像你在填寫一張支票：

- **必要記載事項**（不寫就作廢）：金額、日期、簽名……
- **有害記載事項**（不能寫）：「本支票以交貨為條件」—— 票據是無條件支付的，加了條件就毒化了整張票
- **無害記載事項**（可寫可不寫）：「禁止轉讓」—— 不寫的話可以自由轉讓，寫了就限制轉讓

**考試重點：** 要記住各種票據的「絕對必要記載事項」有哪些，這是選擇題的常客。

#### 各種票據的必要記載事項

**匯票（§24）：**
1. 表明為匯票的文字
2. 一定之金額
3. 無條件委託付款的文字
4. 付款人的姓名或商號
5. 受款人的姓名或商號
6. 發票日
7. 發票人簽名

**本票（§120）：**
1. 表明為本票的文字
2. 一定之金額
3. 無條件擔任付款的文字
4. 受款人的姓名或商號
5. 發票日
6. 發票人簽名

**支票（§125）：**
1. 表明為支票的文字
2. 一定之金額
3. 無條件付款的委託
4. 付款人（金融業者）的商號
5. 發票日
6. 發票人簽名

> **注意：** 到期日在匯票和本票是「相對必要記載事項」（沒寫就推定為見票即付），但支票一律見票即付，所以不需要寫到期日。

### 三、票據行為的獨立性

#### 白話解釋

一張票據上面可能有好多人簽名（發票人簽一次、背書人簽一次、保證人簽一次……）。**每個人的簽名都是獨立的法律行為。** 就算其中某個人的簽名有問題（比如是偽造的、或者簽名的人沒有行為能力），其他人的簽名照樣有效。

#### 法律用語

票據法第 8 條：「票據上之簽名，依票據上所載文義負責。」
票據法第 15 條（無行為能力或限制行為能力人的簽名不影響其他簽名的效力）。

#### 生活場景

小明開了一張本票給小華。小華在票上背書轉讓給小陳。小陳又背書轉讓給小林。

後來發現，小華的簽名其實是別人偽造的。

**問題：小明和小陳的票據責任會受影響嗎？**

**答案：不會。** 因為票據行為具有獨立性，小華的簽名被偽造，只影響「小華」那一筆簽名的效力。小明（發票人）和小陳（背書人）的票據責任照舊。小林仍然可以向小明和小陳追索。

#### 為什麼要有獨立性？

因為票據需要在人與人之間流通。如果一個簽名有問題就整張票報廢，誰敢收票據？獨立性保護的是**票據的流通安全**。

#### 考試重點整理

| 情境 | 結論 |
|------|------|
| A 的簽名被偽造 | A 不負票據責任，但其他真正簽名的人仍負責 |
| B 是未成年人（限制行為能力人） | B 的票據行為效力有疑問，但其他人的責任不受影響 |
| C 被詐欺而簽名 | C 可能主張意思表示瑕疵，但不影響其他人 |

### 四、票據的文義性

#### 白話解釋

票據上面寫什麼，法律效果就照著那個字面意思來。不能拿票據以外的資料來「補充」或「修改」票據上的文字。

**就像考試的答案卷——閱卷老師只看你寫在卷上的東西，不會去查你的草稿紙。**

#### 法律用語

票據法第 5 條第 1 項：「在票據上簽名者，依票上所載文義負責。」

這就是**文義性原則**：票據上的權利義務內容，完全以票據上記載的文字為準。

#### 生活場景

小明開了一張面額 100 萬元的本票給小華，雙方口頭約定「其實只要付 80 萬就好」。後來小華把這張本票背書轉讓給小陳。

小陳拿著票來找小明要 100 萬。小明說：「我跟小華說好只要付 80 萬啊！」

**結論：** 小明必須付 100 萬。因為票據上寫的是 100 萬，文義性原則之下，票上寫什麼就是什麼。至於小明跟小華之間的口頭約定，那是他們兩個之間的事，跟拿到票據的第三人無關。

#### 文義性 vs 契約自由

在一般契約法的世界裡，雙方的真意比字面重要（民法第 98 條：解釋意思表示應探求當事人真意）。但在票據法的世界裡，**字面意思就是一切**。

這是因為票據要流通，每一個收到票據的人只能看到票面上的文字，不可能去調查之前每一手之間到底講了什麼。

### 五、票據的無因性 — 票據行為與原因關係的分離

#### 白話解釋

這是票據法最核心、最常考的概念之一。

**無因性 = 票據上的權利跟「為什麼開這張票」無關。**

不管你是因為買賣、借貸、贈與還是賭債而開了一張票，只要票據本身合法有效，拿到票的人就有權利要錢。**票據不問原因。**

#### 法律用語

票據行為是一種**無因行為（抽象行為）**：票據債務的成立，不以原因關係的存在或有效為前提。

換言之，票據關係與原因關係是**分離的、獨立的**。

#### 生活場景

小明向小華買了一批貨，開了一張 50 萬的本票作為貨款。後來小華交的貨有瑕疵，小明想退貨。

**問題：小明可以因為「貨有瑕疵」而拒絕支付票據上的 50 萬嗎？**

**分析：**
- 票據關係（本票上的 50 萬）和原因關係（買賣契約）是分開的
- 原則上，貨有瑕疵是「原因關係」的問題，不能直接拿來對抗「票據關係」
- **但是！** 如果小華自己來要錢（直接當事人之間），小明可以主張「人的抗辯」（等一下會解釋）
- **重點是：** 如果小華把票背書轉讓給善意的小陳，小陳來要錢時，小明就不能拿「貨有瑕疵」來抗辯了

#### 無因性的例外 — 直接當事人之間

**非常重要！** 無因性保護的是「善意第三人」。在票據的**直接當事人之間**（也就是發票人和受款人之間），發票人還是可以用原因關係來抗辯的。

| 情境 | 能否用原因關係抗辯？ |
|------|-------------------|
| 發票人 vs 受款人（直接當事人） | 可以 |
| 發票人 vs 善意背書取得人（第三人） | 不可以（無因性保護） |
| 發票人 vs 惡意背書取得人 | 可以（因為不是善意） |

### 六、票據抗辯 — 人的抗辯 vs 物的抗辯

#### 白話解釋

「抗辯」就是「拒絕付錢的理由」。

有人拿票來跟你要錢，你不想付，你總得有個理由吧？票據法上的抗辯分兩大類：

1. **物的抗辯（絕對抗辯）**：跟「票據本身」有關的理由，不管誰來要錢你都可以拒絕
2. **人的抗辯（相對抗辯）**：跟「特定的人」有關的理由，只能對特定的人主張

#### 物的抗辯（絕對抗辯）

**白話：「這張票本身就有問題，誰來都不用付！」**

常見的物的抗辯：

| 抗辯事由 | 說明 |
|---------|------|
| 票據欠缺必要記載事項 | 票上少寫了法定必填項目，票據無效 |
| 票據偽造 | 簽名是假的，被偽造的人當然不用付 |
| 票據已經付款 | 已經付過錢了，不用再付一次 |
| 票據時效完成 | 超過法定時效，票據權利消滅 |
| 票據被除權判決宣告無效 | 法院已經判定這張票無效 |
| 票據文義記載本身無效 | 例如金額矛盾、有害記載事項 |

**特徵：** 不管票據落到誰手上，債務人都可以主張。這些理由是「對世」的。

#### 人的抗辯（相對抗辯）

**白話：「我跟『你』之間有糾紛，所以我不想付給『你』。但如果票到了別人手上，我就不能用這個理由了。」**

常見的人的抗辯：

| 抗辯事由 | 說明 |
|---------|------|
| 原因關係不存在或有瑕疵 | 買賣被撤銷、貨有瑕疵、借款已清償… |
| 雙方有特別約定 | 約定延期付款、約定減少金額… |
| 票據是被詐欺或脅迫而簽發 | 對騙你或逼你的那個人可以主張 |
| 受款人取得票據無對價 | 對方沒有付出對價就拿到票 |

**特徵：** 只能對特定的那個人主張，如果票據轉讓給了善意第三人，就不能再用。

#### 表格比較

| | 物的抗辯（絕對抗辯） | 人的抗辯（相對抗辯） |
|--|-------------------|-------------------|
| 對象 | 對任何人都可以主張 | 只能對特定相對人主張 |
| 理由 | 票據本身的瑕疵 | 當事人之間的關係瑕疵 |
| 善意第三人取得票據後 | 仍可主張 | 不可主張 |
| 常見例子 | 偽造、欠缺要件、時效 | 原因關係瑕疵、無對價 |

### 七、票據抗辯的限制 — 善意取得人的保護

#### 白話解釋

這是票據法最重要的政策決定之一：

**善意的第三人收到票據後，原本的債務人不能拿他跟前手之間的「私人恩怨」來拒絕付款。**

這就是「人的抗辯的限制」，也叫做「票據抗辯的切斷」。

#### 法律用語

票據法第 13 條：「票據債務人不得以自己與發票人或執票人之前手間所存抗辯之事由對抗執票人。但執票人取得票據出於惡意者，不在此限。」

這一條翻譯成白話就是：
- 債務人不能用他跟「前幾手」之間的糾紛來對付現在拿票的人
- **例外：** 除非現在拿票的人是「惡意」的（明知道有問題還故意收票）

#### 生活場景

1. A 向 B 買貨，開了一張 100 萬的本票給 B
2. B 交的貨有嚴重瑕疵，A 本來可以拒付
3. 但 B 趕在 A 退貨之前，把這張本票背書轉讓給善意的 C
4. C 是善意取得（不知道貨有瑕疵的事），而且有付對價
5. C 拿票來找 A 要錢

**結論：** A 必須付 100 萬給 C。A 不能用「B 的貨有瑕疵」這個理由來對抗善意的 C。A 只能付完錢之後，再回頭跟 B 打官司。

**如果 C 是惡意的呢？**（C 明明知道貨有瑕疵，故意去找 B 收這張票）

**結論：** A 可以拒絕付款給 C。因為 C 是惡意取得，§13 但書的例外適用。

#### 「善意」的判斷標準

| 要素 | 說明 |
|------|------|
| 不知情 | 不知道債務人與前手間有抗辯事由 |
| 無重大過失 | 不是因為自己太疏忽才不知道 |
| 判斷時點 | 取得票據的當時 |

### 八、票據的善意取得

#### 白話解釋

善意取得是票據法上另一個保護善意第三人的重要制度。

在一般民法的世界裡，**「沒有人可以給出比自己更多的權利」**——你不是物主，就不能把所有權轉讓給別人（「任何人不能讓與超過自己所有的權利」nemo dat quod non habet）。

但票據法為了保護交易安全，打了一個例外：**如果你善意取得票據，即使前手不是真正的權利人（例如票是被偷的），你也可以取得票據上的權利。**

#### 法律用語

票據法第 14 條：
- 第 1 項：以惡意或有重大過失取得票據者，不得享有票據上之權利。（反面解釋：善意且無重大過失取得者，可以享有票據上之權利）
- 第 2 項：無對價或以不相當之對價取得票據者，不得享有優於其前手之權利。

#### 善意取得的要件

| 要件 | 說明 |
|------|------|
| 1. 背書的連續 | 票據上的背書必須形式上連續（一手接一手） |
| 2. 善意 | 不知道讓與人無處分權 |
| 3. 無重大過失 | 不是因為自己太粗心才不知道 |
| 4. 有對價 | 有付出相當的代價取得票據 |

#### 生活場景

小明的一張面額 50 萬的本票被小偷偷走了。小偷偽造小明的背書，把票賣給不知情的小陳，小陳付了 48 萬買下這張票。

**問題：小陳能否取得票據上的權利？**

**分析：**
- 小偷不是票據的真正權利人
- 但小陳是善意（不知道票是偷來的）
- 小陳無重大過失
- 小陳有付出對價（48 萬）
- 票據上的背書形式上連續

**結論：** 小陳善意取得票據上的權利，可以向票據債務人請求付款。

#### §14 II 的「不得享有優於前手的權利」

如果你是**無對價**或**不相當對價**取得票據（例如受贈），你不能享有比你前手更好的權利。

**場景：** A 欠 B 100 萬，開了本票。B 因為原因關係的瑕疵，本來不能跟 A 要錢。B 把這張票「送」給 C（無對價）。

**結論：** C 不能享有優於 B 的權利。既然 B 不能跟 A 要錢，C 也不能。

### 九、票據的偽造與變造

#### 白話解釋

**偽造 = 假冒別人的名字在票據上簽名**

**變造 = 偷偷修改票據上已經記載的內容**（例如把 10 萬改成 100 萬）

兩者都是非法的行為，但法律效果不同。

#### 偽造票據

##### 法律用語

票據法第 15 條：「票據之偽造或票據上簽名之偽造，不影響於真正簽名之效力。」

##### 效力分析

| 對誰 | 效力 |
|------|------|
| 被偽造人（名字被冒用的人） | **不負票據責任** —— 你又沒簽名，當然不用負責 |
| 偽造人（冒用別人名字的人） | **不負票據責任**（因為票據上沒有他自己的名字），但負刑事偽造有價證券罪責 |
| 其他真正簽名的人 | **仍負票據責任** —— 票據行為的獨立性！ |

##### 生活場景

甲開了一張匯票。乙偽造丙的背書，將票轉讓給丁。丁又背書轉讓給戊。

| 人物 | 責任 |
|------|------|
| 甲（發票人） | 負票據責任 ✓ —— 甲是真正簽名的 |
| 乙（偽造者） | 不負票據責任 ✗ —— 票上沒有乙自己的名字 |
| 丙（被偽造人） | 不負票據責任 ✗ —— 丙根本沒簽名 |
| 丁（背書人） | 負票據責任 ✓ —— 丁是真正簽名的 |
| 戊（執票人） | 有票據權利，可以向甲和丁請求 |

##### 考試陷阱

很多人會搞混：偽造人到底負不負票據責任？

**答案是不負。** 因為票據責任是「依票據上的簽名」來判斷的，偽造人用的是別人的名字，票上沒有偽造人自己的名字，所以不負票據法上的責任。但偽造人可能負侵權行為的民事賠償責任和刑事責任。

#### 變造票據

##### 法律用語

票據法第 16 條：
- 第 1 項：「票據之變造，不影響於變造前簽名者之責任。」
- 第 2 項：「簽名在變造後者，依變造文義負責。」
- 第 3 項：「不能辨別前後時，推定簽名在變造前。」

##### 效力分析

| 簽名時間 | 責任 |
|---------|------|
| 變造**前**簽名的人 | 依**原來**的文義負責 |
| 變造**後**簽名的人 | 依**變造後**的文義負責 |
| 分不出先後 | **推定**在變造前 → 依原來的文義負責 |

##### 生活場景

甲開了一張 10 萬元的本票給乙。乙偷偷把金額改成 100 萬，然後背書轉讓給丙。丙又背書轉讓給丁。

| 人物 | 責任金額 |
|------|---------|
| 甲（發票人，變造前簽名） | 負 10 萬元的責任 |
| 乙（變造者+背書人，變造後簽名） | 負 100 萬元的責任 |
| 丙（背書人，變造後簽名） | 負 100 萬元的責任 |
| 丁（執票人） | 可以向甲要 10 萬，向乙、丙各要 100 萬 |

##### 變造 vs 偽造的比較

| | 偽造 | 變造 |
|--|------|------|
| 行為 | 假冒他人簽名 | 修改票據上的記載內容 |
| 被偽造/變造人 | 不負責任 | 依原文義負責（如果在變造前簽名） |
| 其他真正簽名人 | 不受影響 | 依簽名時的文義負責 |
| 行為人本人 | 不負票據責任（無自己的簽名） | 依變造後文義負責（如果在變造後簽名） |

---

### 十、空白授權票據

#### 白話解釋

有時候開票的人會**故意留下一些空白**，授權收票的人自己填上去。最常見的情況：開票的人先簽好名字和蓋好章，但金額和日期留空，讓對方視情況填寫。

#### 法律用語

票據法第 11 條：
- 第 1 項：「欠缺本法所規定票據上應記載事項之一者，其票據無效。但本法別有規定者，不在此限。」
- 第 2 項：「執票人善意取得已具備本法規定應記載事項之票據者，得依票據文義行使權利；票據債務人不得以票據原係欠缺應記載事項為由，對抗執票人。」
- 第 3 項：（空白授權之規定）

#### 重點整理

| 情境 | 效果 |
|------|------|
| 空白票據未經補充，直接提示 | 欠缺必要記載事項 → 無效 |
| 空白票據經授權人補充後提示 | 有效 |
| 被授權人超越授權範圍填寫 | 對善意取得人仍有效（善意取得人的保護） |
| 未經授權的人擅自填寫 | 對票據債務人不生效力，但善意取得人受保護 |

#### 生活場景

老闆甲先在本票上簽名蓋章，金額和日期留空，交給員工乙，授權乙在需要付貨款時填寫金額（限額 50 萬以內）。

乙在票上填了 80 萬（超越授權），然後把票交給善意的丙。

**問題：甲是否要對丙負 80 萬的票據責任？**

**分析：**
- 乙超越授權，但丙是善意取得
- 依 §11 的法理，善意取得人取得已具備法定要件的票據，得依票據文義行使權利
- 甲不得以「乙超越授權」來對抗善意的丙

**結論：** 甲對丙負 80 萬的責任。甲之後可以回頭向乙求償超出的 30 萬。

## 第二部分：匯票 — 票據法的「母法」

> 匯票是票據法中規定最詳細的票據類型。本票和支票的很多規則都是「準用」匯票的規定。所以學好匯票，本票和支票就容易了。

---

### 一、匯票概述

#### 白話解釋

匯票是三方關係的票據：
- **發票人**：開票的人，叫別人付錢
- **付款人**：被委託付錢的人
- **受款人**：收錢的人

**類比：** 你寫了一張紙條給你朋友 A 說：「A，請把 10 萬塊付給 B。」這就是匯票的基本概念。你 = 發票人，A = 付款人，B = 受款人。

#### 匯票的資金關係

| 關係 | 當事人 | 說明 |
|------|--------|------|
| 票據關係 | 所有在票據上簽名的人 | 票面上的權利義務 |
| 資金關係 | 發票人 ↔ 付款人 | 發票人為何有權叫付款人付錢（例如帳戶有錢） |
| 原因關係 | 發票人 ↔ 受款人 | 為何要付這筆錢（例如買賣貨款） |

---

### 二、發票（匯票的簽發）

#### 必要記載事項（§24）

再強調一次匯票的絕對必要記載事項：

1. **表明其為匯票之文字** — 票上要明確寫「匯票」兩個字
2. **一定之金額** — 金額必須確定，不能寫「大約 10 萬」
3. **無條件委託付款之委託** — 不能附加條件
4. **付款人之姓名或商號** — 誰要付錢
5. **受款人之姓名或商號** — 誰要收錢
6. **發票日** — 什麼時候開的票
7. **發票人之簽名** — 開票的人簽名

**缺任何一項 → 匯票無效！**

#### 有害記載事項

| 記載內容 | 為何有害 |
|---------|---------|
| 附條件的付款委託 | 「等貨到了才付」→ 違反「無條件」原則 |
| 分期付款 | 票據金額必須是一筆確定的金額 |
| 免除發票人擔保付款責任 | §29 但書例外：發票人可免除擔保承兌，但不得免除擔保付款 |

#### 相對必要記載事項（沒寫不會無效，有推定規則）

| 事項 | 未記載時的推定 |
|------|--------------|
| 到期日 | 視為見票即付（§24 II） |
| 付款地 | 以付款人之營業所或住所為付款地（§24 III） |
| 發票地 | 以發票人之營業所或住所為發票地（§24 IV） |

---

### 三、背書 — 票據轉讓的方法

#### 白話解釋

背書就是「在票據背面簽名，把票據轉讓給別人」。

你拿到一張支票，想把它「讓」給另一個人，不是直接交出去就好——你要在票的背面簽名，這就叫背書。背書讓票據可以像貨幣一樣在人與人之間流通。

#### 背書的種類

##### 1. 記名背書（完全背書）

**做法：** 在票據背面寫「付給 XXX」並簽名

**效果：** 票據轉讓給指定的那個人

**場景：** 小明在票據背面寫「付給小華」並簽名 → 票據轉讓給小華

##### 2. 空白背書

**做法：** 只在票據背面簽名，不指定被背書人

**效果：** 任何持有票據的人都可以行使票據權利。取得票據的人可以：
- 自己行使權利
- 再做空白背書轉讓
- 補寫自己或他人的名字變成記名背書

**法條：** §31 II —「空白背書之匯票，得依匯票之交付轉讓之。」

**場景：** 小明只在票據背面簽名（不寫「付給誰」）→ 誰拿到票誰就可以行使權利

##### 3. 禁止轉讓背書（限制背書）

**做法：** 背書時加註「禁止轉讓」

**法條：** §30 II —「記名匯票發票人有禁止轉讓之記載者，不得轉讓。」
§30 III —「背書人於票上記載禁止轉讓者，仍得依背書而轉讓之，但禁止轉讓之背書人，對於禁止後再由背書取得之人，不負責任。」

**非常重要的區分：**

| 誰記載禁止轉讓 | 效果 |
|--------------|------|
| **發票人**記載 | 該票據完全不得背書轉讓（只能用民法債權讓與方式轉讓） |
| **背書人**記載 | 票據仍可繼續背書轉讓，但該背書人對後手不負擔保責任 |

**場景：**

情境一：甲（發票人）開了一張匯票給乙，票上寫「禁止轉讓」。乙不能背書轉讓這張票。

情境二：甲開匯票給乙，沒有禁止轉讓。乙背書轉讓給丙時寫了「禁止轉讓」。丙還是可以再背書轉讓給丁，但乙只對丙負背書人的責任，**對丁不負責任**。

**考試高頻陷阱：** 發票人 vs 背書人禁止轉讓的效力差異，是選擇題的最愛！

##### 4. 期後背書

**白話解釋：** 在票據到期日之後（或拒絕付款之後）才做的背書。

**法條：** §41 —「到期日後之背書，僅有通常債權轉讓之效力。」

**效果：** 期後背書只有一般民法債權讓與的效力，不具有票據法上的特殊保護（例如：不適用善意取得、不切斷人的抗辯）。

**場景：** 甲開了一張到期日為 3/1 的匯票給乙。乙在 3/15 才背書轉讓給丙。

**結論：** 丙只取得一般債權讓與的效力。如果甲對乙有抗辯事由，甲也可以拿來對抗丙（因為期後背書不切斷抗辯）。

**為什麼期後背書效力較弱？** 因為票據已經到期了，流通的目的已經結束，再轉讓就像是普通債權的讓與，不需要給予票據法上的特殊保護。

##### 5. 回頭背書

**白話解釋：** 票據被背書轉讓了好幾手之後，又回到了之前某個票據債務人的手上。

**法條：** §38 —「匯票之被背書人為發票人、付款人或前手時，准許背書之。但其得依背書而取得之票據權利，仍以出讓時之權利為限。」（關於回頭背書的規範）

**場景：** 甲開票給乙 → 乙背書給丙 → 丙背書給丁 → 丁背書回給乙。

乙拿到票後，可以向甲（發票人）請求付款嗎？可以向丙、丁（後手背書人）追索嗎？

**分析：** 回頭背書的持票人（乙）原則上可以向其他票據債務人行使權利，但需注意不能對自己之前的被背書人（也就是自己的後手到回頭背書之前手這段）行使追索權，以免循環追索。

##### 6. 委任取款背書

**做法：** 背書時寫「委託取款」或類似文字

**效果：** 這不是真正的轉讓，只是授權被背書人代替自己去領錢。被背書人不取得票據權利，只有代理收款的權限。

### 四、背書的連續性

#### 白話解釋

票據要合法有效地流通，背書必須「一手接一手」，不能中間斷掉。

就像一個接力賽，第一棒交給第二棒，第二棒交給第三棒……如果中間少了一棒，鏈條就斷了。

#### 法律用語

票據法第 37 條：「執票人應以背書之連續，證明其權利。」

**背書連續 = 每一個前手的「被背書人」必須是下一個背書的「背書人」。**

#### 生活場景

正確的背書連續：
```
發票人：甲 → 受款人：乙
乙背書給丙（乙簽名，被背書人欄寫「丙」）
丙背書給丁（丙簽名，被背書人欄寫「丁」）
```
→ 背書連續 ✓：甲→乙→丙→丁

背書不連續的情況：
```
發票人：甲 → 受款人：乙
乙背書給丙
※「丁」背書給戊（但丙沒有背書給丁！）
```
→ 背書不連續 ✗：丙和丁之間斷了

**背書不連續的後果：** 執票人無法以背書連續來證明其權利，付款人可以拒絕付款。

#### 空白背書對連續性的影響

空白背書（只簽名不寫被背書人）不會造成背書不連續。因為空白背書的下一手不論是誰，都可以算是合法取得。

```
甲開票給乙
乙做空白背書（只簽名）
丙取得票據，再背書給丁
```
→ 背書連續 ✓

---

### 五、承兌

#### 白話解釋

承兌是**匯票專有**的制度（本票和支票沒有）。

還記得匯票的三方關係嗎？發票人叫付款人付錢給受款人。但問題是：**付款人沒有在票上簽名，他憑什麼要付錢？**

「承兌」就是付款人在匯票上簽名表示「好，我答應付」。付款人一旦承兌，就變成了匯票的**主債務人**。

#### 法律用語

票據法第 42 條：「付款人承兌後，應負付款之責。」

#### 承兌的效力

| 情境 | 效力 |
|------|------|
| 付款人承兌 | 付款人成為匯票的主債務人 |
| 付款人拒絕承兌 | 執票人可以行使期前追索權 |
| 付款人不為承兌 | 發票人仍負擔保付款的責任 |

#### 生活場景

甲開匯票給乙，委託丙付款。乙拿票去找丙，問丙「你願意付嗎？」

- 丙在票上簽名寫「承兌」→ 丙成為主債務人，到期時一定要付
- 丙拒絕承兌 → 乙可以不等到期，馬上回頭向甲追索

#### 承兌的方式

- 在匯票正面簽名即可（§43）
- 可以附條件承兌嗎？不行，附條件承兌視同拒絕承兌（§47），但「一部承兌」是例外允許的（只承兌部分金額）

#### 為什麼本票和支票不需要承兌？

- **本票**：發票人自己就是付款人，不需要「問別人答不答應」
- **支票**：付款人是銀行，支票一律見票即付，沒有承兌的必要（但有「保付」制度，稍後會講）

---

### 六、票據保證

#### 白話解釋

票據保證就是有人在票據上簽名說「如果主債務人不付錢，我來付」。

跟一般的保證（民法上的保證）不同，票據保證是**獨立的票據行為**，不像民法保證那樣有「先訴抗辯權」。

#### 法律用語

票據法第 58 條：「票據之保證，應由保證人在票據上記載保證意旨並簽名。」

票據法第 59 條：保證人與被保證人負同一責任。

票據法第 61 條：「保證人清償債務後，得行使執票人對承兌人、被保證人及其前手之追索權。」

#### 票據保證 vs 民法保證

| | 票據保證 | 民法保證 |
|--|---------|---------|
| 性質 | 獨立的票據行為 | 從屬於主債務的契約 |
| 先訴抗辯權 | **無** —— 不能要求「先去找主債務人」 | **有** —— 可以要求先找主債務人 |
| 被保證債務無效時 | 保證人**仍負責**（獨立性） | 保證人**免責**（從屬性） |
| 方式 | 在票據上記載並簽名 | 書面契約 |
| 範圍 | 與被保證人負同一責任 | 可以約定保證範圍 |

#### 保證的獨立性 — 超級重要！

票據法第 61 條第 2 項的法理：即使被保證人的票據債務因為行為能力的瑕疵而無效，保證人的責任仍然存在。

**場景：** 甲（未成年人）開了一張本票，乙在票上做保證。後來甲的法定代理人拒絕承認這個票據行為。

**結論：** 甲的票據債務因為行為能力瑕疵而有問題，但乙（保證人）的責任不受影響。保證的獨立性保護了執票人。

#### 隱存保證

**白話：** 有人實際上是要當保證人，但表面上用「共同發票」或「背書」的方式來做。

實務上很常見：甲向銀行借錢，銀行要求甲的朋友乙也在本票上簽名。乙實際上是保證人，但在票面上看起來像是共同發票人。

**效力：** 依票據的文義性，乙在票面上是發票人（或背書人），就要負發票人（或背書人）的責任。不能事後說「我其實只是保證人」。

---

### 七、付款

#### 白話解釋

付款就是到期的時候，付款人把錢付出去，票據上的權利義務就消滅了。

#### 付款提示

執票人要在法定期間內向付款人「提示」票據請求付款。

**匯票的到期日種類（§65）：**

| 種類 | 意思 | 到期日的計算 |
|------|------|------------|
| 定日付款 | 指定某一天付款 | 就是那一天 |
| 發票日後定期付款 | 發票日後若干天付款 | 從發票日起算 |
| 見票即付 | 一提示就要付 | 提示的那天 |
| 見票後定期付款 | 提示後若干天付款 | 從提示日起算 |

**付款提示期間（§69）：**
- 見票即付的匯票：自發票日起 **6 個月**內提示
- 其他匯票：到期日或其後 **2 日**內提示

#### 付款的效力

付款人付款後，票據上的所有債務消滅。全部的背書人、保證人都解放了。

---

### 八、追索權

#### 白話解釋

當票據到期了，付款人不付錢（或不承兌），執票人可以**回頭向前面的人要錢**，這就叫追索。

追索的對象包括：發票人、背書人、保證人——他們對執票人負「連帶責任」。

#### 追索權的種類

| 種類 | 時機 | 條件 |
|------|------|------|
| **到期追索** | 到期日後 | 付款被拒絕 |
| **期前追索** | 到期日前 | 承兌被拒絕、付款人破產、付款人逃匿… |

#### 到期追索的要件（§85）

1. 到期日已經到了
2. 向付款人提示付款被拒絕
3. 取得拒絕證書（或等同的證明）

#### 拒絕證書（§86-§88）

拒絕證書是由法院或公證人做成的證明，證明「我有去要錢，但對方不付」。

**實務上的重要例外：** 如果票據上記載「免除作成拒絕證書」（return without protest），執票人不需要做拒絕證書也可以行使追索權。

#### 期前追索的要件（§85 II）

不用等到期日，以下情況可以提前追索：
1. 付款人拒絕承兌（全部或一部）
2. 付款人受破產宣告
3. 付款人逃匿，無從提示承兌或付款

#### 追索的對象與金額

**對象（§96）：** 發票人、背書人及其保證人，對執票人負**連帶責任**。

**金額（§97）：** 執票人可以追索的金額包括：
1. 票據金額（未付的部分）
2. 自到期日起的利息（年利率 6%）
3. 作成拒絕證書及通知的費用
4. 其他必要費用

#### 再追索權

被追索而清償的人（例如背書人 C 被追索後付了錢），可以再向前手追索（向 B、A 追索），這叫**再追索權**。

**場景：**

甲（發票人）→ 乙（背書人）→ 丙（背書人）→ 丁（執票人）

付款人拒絕付款，丁行使追索權。

- 丁可以向甲、乙、丙任何一人追索（連帶責任）
- 如果丁選擇向丙追索，丙付錢後，可以再向甲和乙追索
- 如果丁直接向甲追索，甲付完就沒有再追索的對象了（甲是源頭）

#### 追索權的喪失

如果執票人沒有在法定期限內提示或沒有做拒絕證書，可能會喪失對部分追索對象的追索權。

**具體來說（§89、§95）：**

| 執票人的疏忽 | 喪失對誰的追索權 |
|------------|----------------|
| 未在期限內提示 | 對背書人及其保證人喪失追索權 |
| 未做拒絕證書 | 對背書人及其保證人喪失追索權 |
| 以上疏忽 | **不影響**對發票人、承兌人的權利 |

**白話理解：** 就算你太晚去要錢或忘了做拒絕證書，你還是可以找開票的人和已經答應付錢的人要。但中間經手的背書人就可以免責了。

---

## 第三部分：本票 — 「自己保證付錢」的票據

> 本票是台灣實務中最常見的票據。借錢要簽本票、租房子要簽本票、分期付款要簽本票——幾乎所有的債權擔保場景都會看到本票的身影。

---

### 一、本票概述

#### 白話解釋

本票就是「我自己保證付錢給你」的紙。只有兩方當事人：

- **發票人**：開票的人，也就是承諾付錢的人
- **受款人**：收錢的人

跟匯票不同的是，本票沒有「付款人」這個角色——因為發票人自己就是付款人。

#### 本票 vs 匯票的比較

| | 匯票 | 本票 |
|--|------|------|
| 當事人 | 三方（發票人、付款人、受款人） | 兩方（發票人、受款人） |
| 誰付款 | 付款人（第三人） | 發票人自己 |
| 有無承兌 | 有（付款人承兌） | 無（不需要，因為發票人自己就要付） |
| 發票人的地位 | 擔保承兌及付款的次要債務人 | **主債務人**（等同匯票承兌人） |
| 強制執行 | 要經過訴訟程序 | **本票裁定（§123）可以直接聲請強制執行！** |

#### 本票的必要記載事項（§120）

1. 表明其為本票之文字
2. 一定之金額
3. 無條件擔任付款之文字（注意：匯票是「委託」付款，本票是「擔任」付款）
4. 受款人之姓名或商號
5. 發票日
6. 發票人之簽名

> **注意：** 本票不需要記載「付款人」，因為發票人就是付款人。

---

### 二、本票裁定（§123）— 超級高頻考點！

#### 白話解釋

這是本票最強大、最獨特的功能，也是考試幾乎每年都考的考點。

一般來說，你要別人付錢，他不付，你得先打官司、拿到判決書，再聲請強制執行。這個過程可能要花一兩年。

但如果你手上有一張本票，你可以**跳過打官司的步驟**，直接向法院聲請「本票裁定」，法院准許後，你就可以直接去強制執行（查封、拍賣債務人的財產）。

#### 法律用語

票據法第 123 條：「執票人向本票發票人行使追索權時，得聲請法院裁定後強制執行。」

#### 本票裁定的要件

| 要件 | 說明 |
|------|------|
| 1. 票據是本票 | 匯票和支票不能用這招 |
| 2. 執票人向發票人行使追索權 | 只能對發票人用，不能對背書人用 |
| 3. 本票已到期 | 見票即付的本票自發票日起 6 個月內提示 |
| 4. 向法院聲請 | 向票據付款地的法院聲請 |

#### 法院審查的範圍

**非常重要！** 法院在本票裁定程序中只做**形式審查**，不做實質審查。

| 法院會審查的 | 法院不會審查的 |
|------------|--------------|
| 票據是否具備法定形式要件 | 原因關係是否存在 |
| 背書是否連續 | 票據是否被偽造 |
| 是否已到期 | 雙方之間有什麼約定 |
| 提示期間是否遵守 | 發票人有什麼抗辯事由 |

**白話：** 法院只看「這張本票格式上有沒有問題」，不管「你們之間到底有沒有真的欠錢」。

#### 發票人的救濟

如果發票人覺得「我不應該付這筆錢」（例如：原因關係不存在、票據是被偽造的……），他該怎麼辦？

1. **對本票裁定提起抗告**：但抗告理由僅限於形式要件的瑕疵，不能主張實體事由
2. **提起「確認票據債權不存在之訴」**：這才是主張實體抗辯的正確途徑
3. **聲請停止執行**：在確認之訴判決前，可以提供擔保金聲請法院停止強制執行

#### 生活場景

小明跟小華借了 100 萬，簽了一張本票。後來小明覺得小華收的利息太高（重利），想要主張原因關係有瑕疵。

但小華不管，直接拿本票去法院聲請本票裁定。法院看了本票，格式沒問題，就裁定准許強制執行。

**小明該怎麼辦？**

1. 先提起「確認本票債權不存在之訴」
2. 同時聲請法院「停止強制執行」（要提供擔保金）
3. 在訴訟中主張「原因關係有重利的問題」

**考試重點：** 本票裁定 = 形式審查，實體抗辯要另外提訴。這個觀念幾乎每年必考！

---

### 三、本票的到期日

#### 到期日的種類

本票準用匯票的規定，到期日有四種：

| 種類 | 說明 |
|------|------|
| 定日付款 | 指定某天付款 |
| 發票日後定期付款 | 發票後 X 天付款 |
| 見票即付 | 提示就付 |
| 見票後定期付款 | 提示後 X 天付款 |

**未記載到期日 → 視為見票即付。**

#### 見票即付本票的提示期限

票據法第 120 條準用 §66：見票即付的本票，自發票日起 **6 個月**內提示付款。

**超過這個期限還沒提示，會喪失對背書人的追索權。** 但對發票人（主債務人）的權利不受影響（只是要注意時效問題）。

---

### 四、本票的其他準用規定

本票大部分規定準用匯票，以下是需要注意的差異：

| 事項 | 匯票 | 本票 |
|------|------|------|
| 承兌 | 有 | 無（發票人 = 付款人） |
| 強制執行 | 無特別規定 | §123 本票裁定 |
| 發票人地位 | 次要債務人 | 主債務人（等同承兌人） |
| 追索 | 先向承兌人/付款人求償 | 直接向發票人求償 |
| 拒絕證書 | 需要 | 向發票人追索不需要（因為是主債務人） |

---

## 第四部分：支票 — 「叫銀行付錢」的票據

> 支票是日常生活中最常見的票據。雖然電子支付越來越普及，但支票在商業交易中仍然活躍，而且考試題目經常出現。

---

### 一、支票概述

#### 白話解釋

支票就是你寫一張紙條給銀行說：「請從我的帳戶付 X 元給拿這張紙的人。」

**支票其實就是一種特殊的匯票——付款人固定是金融業者（銀行）。**

#### 支票的特殊性

| 特點 | 說明 |
|------|------|
| 付款人 | 必須是**金融業者**（銀行、信用合作社等） |
| 到期日 | **一律見票即付**，不能約定其他到期日 |
| 承兌 | **無承兌制度**（付款人是銀行，不需要再問銀行答不答應） |
| 保證 | 無票據保證制度（但有「保付」制度） |
| 提示期限 | 比匯票短得多（國內 7 天） |

---

### 二、支票與匯票的差異

| | 匯票 | 支票 |
|--|------|------|
| 付款人 | 任何人 | 限金融業者 |
| 到期日 | 四種 | 一律見票即付 |
| 承兌 | 有 | 無 |
| 保證 | 有 | 無（但有保付） |
| 提示期限 | 到期日後 2 日內 | 發票日後 7 日（國內） |
| 遠期票據 | 可以開 | 不可以（但實務上有遠期支票） |
| 利息記載 | 見票即付可記載利息 | 不可記載利息 |

---

### 三、支票的必要記載事項（§125）

1. 表明其為支票之文字
2. 一定之金額
3. 付款人（金融業者）之商號
4. 無條件付款之委託
5. 發票日
6. 發票人之簽名

> **注意：** 支票不用記載受款人（可以開「無記名支票」）、不用記載到期日（一律見票即付）。

---

### 四、付款提示期限 — 支票的靈魂

#### 白話解釋

支票最重要的特徵之一就是：**你拿到支票後，必須在很短的時間內去銀行兌現。** 超過期限，銀行可能不付錢，你也可能喪失追索權。

#### 法定期限（§130）

| 情形 | 提示期限 |
|------|---------|
| 發票地與付款地在同一省（市）區 | 發票日後 **7 日**內 |
| 發票地與付款地不在同一省（市）區 | 發票日後 **15 日**內 |
| 發票地在國外，付款地在國內 | 發票日後 **2 個月**內 |

#### 逾期提示的效果

| 情境 | 效果 |
|------|------|
| 在提示期限內提示 | 銀行應付款；如果不付，可以行使追索權 |
| 超過提示期限才提示 | 銀行仍**得**付款（只要帳戶有錢且未撤銷付款委託），但喪失對前手的追索權 |
| 發票人撤銷付款委託 | 發票日後 6 個月內不得撤銷（§136 但書保護期） |

**重點：** 超過提示期限 ≠ 支票作廢。銀行可以選擇付或不付。但執票人喪失追索權。

#### 遠期支票的問題

台灣實務上很常見「遠期支票」—— 開票日期寫在未來（例如今天 1/1，但支票上的發票日寫 3/1）。

法律上：支票一律見票即付，但 §128 II 規定：「以發票日為提示期限之起算日。」所以就算你在 1/1 就去銀行提示，提示期限也是從 3/1 起算。

**但！** §128 I 規定：「支票限於見票即付，有相反之記載者，其記載無效。」所以即使發票日寫在未來，**執票人在發票日前提示，付款人仍應付款**。

這是一個看似矛盾但重要的規定：
- 遠期支票的「遠期」不影響見票即付的性質
- 但提示期限從發票日起算（保護發票人有時間準備資金）

---

### 五、空頭支票

#### 白話解釋

開了支票但帳戶裡沒那麼多錢，這就是空頭支票（俗稱「芭樂票」）。

#### 法律效果

| 層面 | 效果 |
|------|------|
| 票據法 | 支票仍然有效，執票人可以向發票人追索 |
| 行政管制 | 銀行會記錄退票紀錄；退票 3 張以上 → 列為拒絕往來戶（3 年不能開戶） |
| 刑事責任 | 以前有刑罰（票據法第 141 條），現已刪除；但仍可能構成詐欺罪 |

#### 發票人的擔保責任

票據法第 126 條：「發票人應照支票文義擔保支票之支付。」

**白話：** 就算銀行不付（因為帳戶沒錢），開票的人自己要負責付。

---

### 六、平行線支票（劃線支票）

#### 白話解釋

在支票正面劃兩條平行線，就是平行線支票。功能是**限制誰可以領這張支票的錢**，增加安全性。

#### 兩種平行線

| 種類 | 記載方式 | 效果 |
|------|---------|------|
| **普通平行線** | 只劃兩條線，或線內寫「銀行」 | 付款銀行只能對金融業者付款 |
| **特別平行線** | 線內寫特定銀行名稱 | 付款銀行只能對該特定銀行付款 |

#### 為什麼要劃平行線？

防止票據被偷或撿到的人直接去領錢。劃了平行線後，必須透過銀行帳戶才能兌現，這樣就有「跡」可查。

#### 法條

票據法第 139 條至第 141 條。

#### 考試重點

| 問題 | 答案 |
|------|------|
| 平行線可以取消嗎？ | 支票之平行線，**送款人得於平行線內記載照付現款或同義字樣**（§141），但依§139 III：「劃平行線之支票，得由發票人於平行線內記載照付現款或同義字樣，由發票人簽名或蓋章於其旁，支票上之平行線視為未記載。」只有發票人可以取消 |
| 普通平行線可以改成特別平行線嗎？ | 可以（§139 II） |
| 特別平行線可以改成普通平行線嗎？ | 不可以 |
| 違反平行線規定付款的後果 | 付款人應負賠償責任（§141） |

---

### 七、保付支票

#### 白話解釋

保付支票就是銀行在支票上蓋章說「保證付款」。有了銀行的保付，這張支票幾乎跟現金一樣安全。

#### 法律用語

票據法第 138 條：「付款人於提示期限內，經執票人之同意，得為保付。保付後，付款人應照支票文義負責，與匯票承兌人同。」

#### 保付的效力

| 效果 | 說明 |
|------|------|
| 銀行成為主債務人 | 等同匯票的承兌人 |
| 發票人和背書人免責 | §138 II：「付款人於支票上記載照付或保付或其他同義字樣並簽名後，其付款責任，與匯票承兌人同。發票人及背書人免除其責任。」 |
| 不得撤銷付款委託 | 保付後，發票人不能撤銷 |

#### 考試重點

**注意！** 保付後，發票人和背書人**免除責任**。這是很特殊的規定，因為一般情況下，付款人承諾付款並不會讓發票人和背書人免責（匯票承兌不會讓發票人免責）。

**場景：** 甲開支票給乙，銀行保付。乙背書給丙。

- 丙只能向銀行請求付款
- 丙不能向甲（發票人）或乙（背書人）追索
- 因為保付後，甲和乙都免責了

---

## 第五部分：票據時效 — 權利不行使就會消滅

> 時效制度的基本精神：法律不保護在權利上睡覺的人（Vigilantibus non dormientibus jura subveniunt）。你有權利要錢，但如果你太久不去要，法律就不保護你了。

---

### 一、各種票據的時效期間

#### 匯票時效（§22）

| 請求權 | 時效期間 | 起算日 |
|--------|---------|--------|
| 對承兌人（主債務人） | **3 年** | 到期日 |
| 對前手（背書人、發票人）的追索權 | **1 年** | 拒絕承兌或拒絕付款日；或免除作成拒絕證書者，自到期日起算 |
| 背書人對前手的再追索權 | **6 個月** | 清償之日或被訴之日 |

#### 本票時效（§22 準用）

| 請求權 | 時效期間 | 起算日 |
|--------|---------|--------|
| 對發票人（主債務人） | **3 年** | 到期日 |
| 對背書人的追索權 | **1 年** | 拒絕付款日 |
| 背書人對前手的再追索權 | **6 個月** | 清償之日或被訴之日 |

#### 支票時效（§22 III）

| 請求權 | 時效期間 | 起算日 |
|--------|---------|--------|
| 對發票人的追索權 | **1 年** | 發票日 |
| 對背書人的追索權 | **4 個月** | 提示日 |
| 背書人對前手的再追索權 | **2 個月** | 清償之日或被訴之日 |

#### 時效比較表（一目了然版）

| | 對主債務人 | 對背書人/發票人 | 再追索權 |
|--|---------|--------------|---------|
| 匯票 | 3 年 | 1 年 | 6 個月 |
| 本票 | 3 年 | 1 年 | 6 個月 |
| 支票 | 1 年（對發票人） | 4 個月 | 2 個月 |

**記憶口訣：** 匯票本票時效長（3-1-0.5），支票時效短（1-4月-2月）。支票因為見票即付、快速結算，所以時效特別短。

---

### 二、時效完成的效果

#### 白話解釋

時效完成不代表債務完全消滅，而是：
- 債務人取得「拒絕給付」的抗辯權
- 如果債務人自己願意付，付了也不能要回去（因為是「自然債務」）

#### 法律效果

| 效果 | 說明 |
|------|------|
| 票據上的權利消滅 | 不能再用票據法上的追索權或付款請求權 |
| 民法上的權利 | 如果有原因關係（例如借貸），仍可以根據民法請求 |
| 利益償還請求權 | §22 IV 的特殊救濟（下面詳述） |

---

### 三、利益償還請求權（§22 IV）— 重要考點

#### 白話解釋

這是票據法給「忘記要錢、過了時效的人」的最後一條救生繩。

即使票據時效已經完成，執票人仍然可以向**因時效完成而受利益的人**請求返還利益。

#### 法律用語

票據法第 22 條第 4 項：「匯票、本票之執票人，對前項之追索權時效消滅時，仍得對於發票人或承兌人，於其所受利益之限度，請求償還。」

#### 要件

| 要件 | 說明 |
|------|------|
| 1. 票據權利因時效而消滅 | 不是因為其他原因消滅的 |
| 2. 對象限於發票人或承兌人 | 不能對背書人主張 |
| 3. 在所受利益的限度內 | 不是全額，而是受利益的部分 |

#### 「利益」的範圍

這裡的「利益」是指什麼？

**通說：** 指票據債務人因票據時效完成而免除票據債務所受的利益。具體來說：

| 情境 | 有無「利益」 |
|------|------------|
| 發票人有原因關係的對價（例如收到貨物） | 有利益 → 可以請求 |
| 發票人無原因關係的對價（例如是贈與票據） | 無利益 → 不能請求 |
| 發票人已經將資金交給付款人（匯票的資金關係） | 發票人無利益（但可以向付款人請求） |

#### 生活場景

甲開本票給乙（因為買賣貨款 50 萬），到期日是 2020/1/1。乙太忙忘了去要錢，一直到 2023/3/1 才想起來。

- 票據時效（3 年）已於 2023/1/1 完成
- 乙不能再行使票據上的權利
- **但！** 甲收了 50 萬的貨，因時效完成而免除了付款義務，甲確實「受有利益」
- 乙可以依 §22 IV 向甲請求利益償還

#### 利益償還請求權的時效

這個請求權本身的時效是多久？法律沒有明文規定，通說認為適用民法的一般消滅時效（15 年，或依民法第 125 條以下的規定）。

#### 支票沒有利益償還請求權？

**非常重要的考試陷阱！**

§22 IV 的文字是「匯票、本票之執票人」—— **沒有提到支票**。

所以支票的執票人，在時效完成後，**不能**主張利益償還請求權。

但支票的執票人如果跟發票人之間有原因關係，還是可以依民法的規定（例如不當得利、買賣契約）來請求。

## 第六部分：重要爭議問題與進階概念

> 這個部分整理一些實務上常見的爭議和進階概念，幫助你應對比較難的題目。

---

### 一、票據法律關係的整體架構

#### 三層關係

要理解票據法，必須區分三層法律關係：

| 關係 | 說明 | 例子 |
|------|------|------|
| **票據關係** | 票據上的權利義務 | 執票人向背書人追索 |
| **票據預約** | 開票之前的約定 | 甲答應開票給乙 |
| **原因關係** | 為什麼要開票 | 買賣、借貸、贈與… |
| **資金關係** | 發票人跟付款人之間的關係（匯票） | 發票人在銀行有存款 |

#### 重要觀念：這三層關係是獨立的

- 原因關係無效 ≠ 票據關係無效（無因性）
- 資金關係不存在 ≠ 票據關係不存在
- 但在直接當事人之間，可以主張原因關係的抗辯

---

### 二、共同發票

#### 白話解釋

兩個人以上同時在票據上以「發票人」的身分簽名。

#### 法律效果

共同發票人對執票人負**連帶責任**。

**場景：** 甲和乙共同簽發一張 100 萬的本票給丙。丙可以向甲要 100 萬，也可以向乙要 100 萬，也可以分別向兩人各要一部分。

---

### 三、到期日前的善意取得與抗辯切斷

#### 整合分析

當一張票據經過多手轉讓，到了善意第三人手上時，會同時涉及：

1. **善意取得（§14）**：即使前手無處分權，善意取得人仍取得票據權利
2. **抗辯切斷（§13）**：債務人不能用與前手之間的抗辯事由對抗善意執票人

這兩個制度目的相同：保護票據的流通安全。

但要注意區分：
- §14 解決的是「**權利取得**」的問題（前手有沒有權利讓與？）
- §13 解決的是「**權利行使**」的問題（取得權利後能不能順利行使？）

---

### 四、利息的記載

#### 匯票

- 見票即付或見票後定期付款的匯票，可以記載利息（§28）
- 定日付款或發票日後定期付款的匯票，不得記載利息（因為已經可以把利息算進票面金額）

#### 本票

準用匯票的規定。

#### 支票

**不得記載利息。** 記載了利息，該記載視為無效（不影響支票其他部分的效力）。

---

### 五、付款地與管轄

票據上的付款地很重要，因為它決定：

1. **管轄法院**：票據訴訟由付款地法院管轄
2. **本票裁定的聲請法院**：向付款地法院聲請
3. **提示付款的地點**：在付款地提示

---

### 六、票據喪失的救濟

#### 白話解釋

你的支票弄丟了怎麼辦？

#### 救濟方式

| 方式 | 說明 |
|------|------|
| **止付通知** | 通知付款人暫停付款（§18） |
| **公示催告** | 向法院聲請公示催告，催告持有票據的人在期限內申報權利 |
| **除權判決** | 公示催告期間無人申報 → 法院做成除權判決 → 票據宣告無效 → 原持票人可以行使票據權利 |

**注意：** 止付通知後 5 日內必須向法院聲請公示催告，否則止付通知失效。

---

### 七、票據的塗銷

#### 白話解釋

塗銷就是把票據上的某些記載塗掉或劃掉。

#### 效力

| 情境 | 效力 |
|------|------|
| 故意塗銷背書 | 被塗銷的背書視為未曾存在 |
| 被塗銷的背書後手 | 也視為未曾存在（連鎖效應） |
| 非故意塗銷（如墨水暈開） | 需舉證非故意 |

**法條：** §17 —「票據上之簽名或記載被塗銷時，非由票據權利人故意為之者，不影響於票據上之效力。」

---

### 八、票據的抵銷

#### 白話解釋

A 拿票據來跟 B 要 100 萬，但 B 也對 A 有一筆 80 萬的債權。B 可以主張抵銷，只付 20 萬嗎？

#### 分析

這涉及票據抗辯的問題：

- 如果 A 和 B 是票據的直接當事人，B 可以主張抵銷（人的抗辯）
- 如果 A 是從第三人那裡善意取得票據的，B 不能主張對第三人的債權來抵銷

---

### 九、票據與電子支付的關係

雖然不是傳統考題範圍，但了解背景有助於理解：

隨著電子支付的普及，紙本票據的使用確實在減少。但：

1. 票據法的法律原理（文義性、無因性、獨立性）影響了電子支付法律的設計
2. 本票在民間借貸中仍然非常普遍
3. 支票在商業交易中仍有使用
4. 考試仍然會考，而且是固定的 20 分

## 第九部分：常見混淆概念對照表

### 偽造 vs 變造

| | 偽造 | 變造 |
|--|------|------|
| 行為 | 冒用他人簽名 | 變更票據上的記載 |
| 被害人責任 | 被偽造人不負責 | 變造前簽名者依原文義負責 |
| 行為人責任 | 不負票據責任（票上無自己的名字） | 依變造後文義負責（若在變造後簽名） |
| 其他簽名人 | 不受影響 | 依簽名時的文義負責 |

### 物的抗辯 vs 人的抗辯

| | 物的抗辯 | 人的抗辯 |
|--|---------|---------|
| 對象 | 對任何人 | 對特定人 |
| 是否被切斷 | 否 | 善意取得時被切斷 |
| 例子 | 偽造、缺要件、時效 | 原因關係瑕疵、無對價 |

### 發票人禁轉 vs 背書人禁轉

| | 發票人記載禁止轉讓 | 背書人記載禁止轉讓 |
|--|------------------|------------------|
| 效果 | 票據不得背書轉讓 | 票據仍可背書轉讓 |
| 原因 | 發票人是票據的創造者 | 背書人只是中間經手人 |
| 後手地位 | 不適用票據法的善意取得 | 善意取得仍適用 |

### 票據保證 vs 民法保證

| | 票據保證 | 民法保證 |
|--|---------|---------|
| 獨立性 | 有（被保證債務無效，保證仍有效） | 無（主債務無效，保證也無效） |
| 先訴抗辯權 | 無 | 有 |
| 方式 | 在票據上記載 | 書面契約 |

### 本票裁定 vs 一般訴訟

| | 本票裁定 | 一般訴訟 |
|--|---------|---------|
| 程序 | 非訟程序，快速 | 訴訟程序，耗時 |
| 審查 | 形式審查 | 實質審查 |
| 救濟 | 抗告 + 確認之訴 | 上訴 |
| 適用 | 僅限本票 vs 發票人 | 所有票據 |

### 各種背書的效力

| 背書類型 | 票據權利轉讓 | 擔保責任 | 抗辯切斷 | 善意取得 |
|---------|------------|---------|---------|---------|
| 記名背書 | ✓ | ✓ | ✓ | ✓ |
| 空白背書 | ✓ | ✓ | ✓ | ✓ |
| 禁止轉讓背書（背書人記載） | ✓（仍可轉讓） | 對後手免責 | ✓ | ✓ |

## 第十部分：學習建議

### 票據法的學習策略

1. **先搞懂「為什麼」**
   - 票據法的每一個規定都有背後的道理。搞懂「為什麼要這樣規定」，比死背法條有效一百倍。
   - 核心精神就一個：**保護票據的流通安全。**

2. **掌握三大原則**
   - 文義性：票面說了算
   - 無因性：不問原因
   - 獨立性：各人的簽名互不影響

3. **善用比較表**
   - 匯票 vs 本票 vs 支票
   - 偽造 vs 變造
   - 物的抗辯 vs 人的抗辯
   - 票據保證 vs 民法保證
   - 比較是這科最有效的學習方式

4. **必背數字**
   - 支票提示期限：7/15/2月
   - 匯票本票時效：3年/1年/6月
   - 支票時效：1年/4月/2月
   - 見票即付提示期限：6個月

5. **做題練習**
   - 票據法的選擇題有固定模式，多做題目很快就能掌握
   - 特別注意「組合題」：一題裡面可能同時考偽造+背書連續+善意取得

### 考試時間分配建議

票據法在一試佔 20 分（選擇題），大約 10 題左右。

- 建議作答時間：15-20 分鐘
- 如果準備充分，這些題目可以很快作答
- 不確定的題目先標記，最後再回來

### 最後叮嚀

票據法是一科「投資報酬率極高」的科目：

- 範圍明確：就一部法律，不像民法那麼龐大
- 題型固定：常考的就那幾個主題
- 觀念清晰：搞懂原理後，題目萬變不離其宗
- 20 分全拿不是夢！

加油！把這份教材讀熟，再搭配題庫練習，票據法就是你的囊中物。

## 附錄：票據法重要條文原文節錄

> 以下摘錄考試最常出現的條文原文，方便你對照理解。

### §5（文義性）
「在票據上簽名者，依票上所載文義負責。」

### §10（票據行為之代理）
「票據之簽名，得以蓋章代之。」

### §11（欠缺要件/空白票據）
「欠缺本法所規定票據上應記載事項之一者，其票據無效。但本法別有規定者，不在此限。」
「執票人善意取得已具備本法規定應記載事項之票據者，得依票據文義行使權利；票據債務人不得以票據原係欠缺應記載事項為由，對抗執票人。」

### §13（票據抗辯的限制）
「票據債務人不得以自己與發票人或執票人之前手間所存抗辯之事由對抗執票人。但執票人取得票據出於惡意者，不在此限。」

### §14（善意取得/惡意取得/無對價取得）
「以惡意或有重大過失取得票據者，不得享有票據上之權利。」
「無對價或以不相當之對價取得票據者，不得享有優於其前手之權利。」

### §15（偽造的效力）
「票據之偽造或票據上簽名之偽造，不影響於真正簽名之效力。」

### §16（變造的效力）
「票據經變造時，簽名在變造前者，依原有文義負責；簽名在變造後者，依變造文義負責。不能辨別前後時，推定簽名在變造前。」

### §22（票據時效與利益償還）
「票據上之權利，對匯票承兌人及本票發票人，自到期日起算，三年間不行使，因時效而消滅。對支票發票人自發票日起算，一年間不行使，因時效而消滅。」
「匯票、本票之執票人，對前項之追索權時效消滅時，仍得對於發票人或承兌人，於其所受利益之限度，請求償還。」

### §30（禁止轉讓）
「匯票依背書及交付而轉讓。記名匯票發票人有禁止轉讓之記載者，不得轉讓。」
「背書人於票上記載禁止轉讓者，仍得依背書而轉讓之，但禁止轉讓之背書人，對於禁止後再由背書取得之人，不負責任。」

### §41（期後背書）
「到期日後之背書，僅有通常債權轉讓之效力。」

### §123（本票裁定）
「執票人向本票發票人行使追索權時，得聲請法院裁定後強制執行。」

### §130（支票提示期限）
「支票之執票人，應於左列期限內，為付款之提示：一、發票地與付款地在同一省（市）區內者，發票日後七日內。二、發票地與付款地不在同一省（市）區內者，發票日後十五日內。三、發票地在國外，付款地在國內者，發票日後二個月內。」

### §138（保付支票）
「付款人於提示期限內，經執票人之同意，得為保付。保付後，付款人應負絕對付款之責。」
「發票人及背書人免除其責任。」
