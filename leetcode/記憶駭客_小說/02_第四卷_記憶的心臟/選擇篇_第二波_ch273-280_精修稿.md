## ch273 嚴柏翰與邱韻如的告別

**情緒強度：8（全波最安靜的一章。三句對白。其他全是動作和空間。A>B，零技術）**

---

### 核心場景

```
上線前夜。

安全屋的走廊。
很長。
兩盞燈。一盞在這頭，一盞在那頭。
中間有一段光照不到的地方。

嚴柏翰站在走廊這端。
黑風衣。左眼的疤。
他本來只是要去通訊室確認節點狀態。

邱韻如站在走廊那端。
她端著一杯茶。
茶杯上方有蒸氣。
很細的蒸氣。
在走廊的光裡看得到。

五公尺。

他們同時看見了彼此。


嚴柏翰的腳步停了。

不是猶豫。
是身體做的決定。
腳先停。然後大腦才追上來。


邱韻如沒有動。
她站在那盞燈下面。
光從頭頂落下來。
把她的影子拉得很長。

她的影子幾乎碰到他的鞋尖。

但她本人離他五公尺。


茶杯的蒸氣在空氣裡慢慢散開。
走廊裡有一種味道。
淡的。
不是合成茉莉花茶的味道。
是真正的茶葉。
——有一點點苦。有一點點澀。
每一口濃度都不同。


邱韻如看著他的臉。
準確地說——看著他左眼的疤。

她記得那道疤。
不是今天才記得。
是 ch217 之後就知道了。
那道疤的來源。
那道疤的真正含義。
她親手做的事。


嚴柏翰注意到了她的視線。

他沒有別開頭。
也沒有遮住。

他只是站在那裡。
讓她看。


然後他說了一句話。

「已經不疼了。」

聲音很平。
不是冷。不是原諒。不是釋懷。
是——陳述。
像在說今天的天氣。
像在說走廊的燈有一盞比較亮。

已經不疼了。

技術上，他在說疤。
實際上，他在說一切。


邱韻如低下頭。

她看著杯中的茶。
淡黃色的茶湯。
還在冒著微弱的蒸氣。
她今天沒有用合成茶包。
她用了真正的茶葉。

是她很久以前——
還不是織網者的時候——
會喝的那種茶。

「味道比較淡。」
她的聲音很穩。知識分子的精確。
每一個字都經過秤量。

停頓。

「但比較真。」


走廊的空氣安靜了。

蒸氣繼續上升。
光繼續照著。
影子繼續拉著。

五公尺距離。沒有縮短。也沒有拉大。


嚴柏翰的右手微微收緊。

他自己可能沒注意到。
但邱韻如注意到了。
她的眼睛在那隻手上停了不到一秒。

然後他的手鬆開了。

指節從緊繃到放鬆。
很慢。
像一個人在學放手。
不是放開什麼東西。
是放開——緊握本身。


他轉身了。

沒有回頭。

腳步聲在走廊裡漸漸遠去。
不快。不慢。
是一個做過決定的人走路的速度。


邱韻如站在原地。
她端著茶杯。
蒸氣越來越弱。
茶在變涼。

她低頭喝了一口。

淡的。
有一點苦。
不完美。

但是她泡的。
第一次，用她自己選的茶葉泡的。


走廊的那盞燈在他離開後
顯得格外安靜。

像一座城市在夜裡
關掉了最後一扇窗。

但燈還亮著。
一直亮著。
```

### 逐行解讀

| 台詞/細節 | 設計意圖 |
|-----------|---------|
| 五公尺距離 | 全章的物理核心。不靠近 = 不越界。五公尺 = 理解但不原諒的精確距離 |
| 蒸氣在光裡看得到 | 茶的蒸氣 = 唯一流動的東西。在兩個靜止的人之間，蒸氣替他們「說話」|
| 真正的茶葉 vs 合成茶 | 回收邱韻如「覺醒後的選擇」母題。合成 = 被設計的人生。真茶 = 不完美但屬於自己 |
| 「已經不疼了」 | 全章核心句。嚴柏翰最精簡的告別。三重含義：疤不疼了、恨不疼了、放下了 |
| 邱韻如看他的疤 | 回收 ch217 她知道真相後的反應。她記得那道疤。她知道是自己造成的 |
| 「味道比較淡。但比較真。」 | 邱韻如的回應。不是在說茶。是在說覺醒後的世界——不再濃烈，但是真的 |
| 右手收緊然後鬆開 | 嚴柏翰的身體記得。手曾經在提到她時收緊（ch123）。現在鬆開 = 放下 |
| 影子幾乎碰到鞋尖 | 身體保持距離，但影子試圖觸碰——暗示潛意識的靠近 |
| 轉身不回頭 | 嚴柏翰的 DNA 級動作標記。ch157「帶我去見她」是轉身走向。現在是轉身走開。同一個動作，完全不同的方向 |
| 城市關掉最後一扇窗 | Vol 4 文明類比基調。走廊 = 城市。燈 = 窗。他的離開 = 一個時代的結束 |

## ch274 最後的探員

**情緒強度：7（嚴柏翰交出身份 + 紙鶴告別式玩笑。A>B，零技術）**

---

### 核心場景

```
控制室。

螢幕上跳出一組數字。
六百萬個節點。
之前有三分之一是被封鎖的。
深淵監控局的安全協議。
在系統崩潰後自動鎖死。

嚴柏翰的權限金鑰
是打開這些節點的唯一方式。

螢幕上的進度條——
12%...
34%...
67%...
89%...
100%。

所有封鎖的網路節點全部開啟。

然後權限金鑰自動銷毀。
螢幕上閃了一下。
深淵監控局的標誌碎成像素。
然後消失了。

嚴柏翰看著空白的螢幕。


「我不是探員了。」

他的語氣很平。
像在說一件已經發生很久的事。
像在確認一個事實——
不是宣告，是驗屍報告。


維倫站在他旁邊。

「你是什麼？」

嚴柏翰想了一會兒。
不長。四五秒。
他看著空白的螢幕，
眼睛裡有一種很淡的——

不是悲傷。
是某種東西被放下之後的輕。

「一個做了選擇的人。」


語青坐在操作台邊。
她的手腕以下已經半透明了。
在控制室的白色燈光下，
可以隱約看到鍵盤的輪廓
穿過她的手指。

嚴柏翰看了她一眼。

他記得她。
不是現在的語青。
是記憶裡那個——
站在實驗室門口的小女孩。
裙子上有墨水漬。
手裡捏著一隻紙鶴。
歪歪的。

——那是他教她折的。
在那個週四的下午。
周明哲在隔壁開會。
語青一個人坐在走廊地板上無聊。
嚴柏翰蹲下來。
「要不要學折紙鶴？」


他看著眼前的語青。
二十年後。
半透明的。
不再是小女孩了。
但笑起來的弧度一模一樣。


「替我跟你爸說一聲——」

嚴柏翰的語氣很輕。
帶著一點冷幽默。
他講正經事的方式。

「他教妳折的紙鶴，
 折法一直都是歪的。」


語青笑了。

不是大笑。
是那種——嘴角往右邊多翹了一點的笑。
半透明的臉上，
笑紋像光做的。

「我知道。」

她低頭看了一眼自己的手。
透明的手指。
已經很久沒折過紙鶴了。

「我一直沒跟他說。」


嚴柏翰看著她笑。

他的嘴角也動了一下。
很小的動作。
像在肌肉記憶裡找到了一條
已經很久沒有走過的路。


然後他站起來。

黑風衣。舊疤。
背影比以前鬆了。
肩膀不再是
繃了十年的那種直。

他走到控制室的門口。

停了一秒。
沒有回頭。

「紙鶴歪的比較好。」

聲音很輕。
像是丟在身後的。
不確定是不是說給誰聽的。

「完美的東西沒有溫度。」

然後他走了。


語青坐在操作台邊。
她低頭看著自己透明的手指。
微微曲起。
像在折什麼。

她沒有折。
但手記得那個動作。

米亞的光球在角落裡
安靜地亮了一下。
沒有說話。
延遲了 0.6 秒之後——
光球的顏色稍微暖了一點。

那是米亞的哭法。
```
