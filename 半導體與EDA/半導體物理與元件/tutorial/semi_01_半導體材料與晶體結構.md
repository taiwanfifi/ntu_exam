# 第一章：半導體材料與晶體結構

> **學習目標**：理解半導體為何特別、晶體結構如何影響元件性能，以及常見半導體材料的差異。
> **預備知識**：高中化學（原子結構）、基礎物理（能量概念）
> **預估時間**：3～4 小時

---

## 🔰 本章基礎觀念（零基礎必讀）

### 為什麼半導體重要？

你的手機、筆電、電動車、AI 伺服器——幾乎所有現代電子產品的核心都是「半導體晶片」。全球最知名的科技聚落「矽谷（Silicon Valley）」之所以叫這個名字，正是因為矽（Silicon, Si）是最重要的半導體材料。

**半導體的神奇之處**：它的導電能力可以被精確控制。

- 銅線（導體）：電阻很低，永遠都導電
- 玻璃（絕緣體）：電阻極高，永遠不導電
- 矽（半導體）：可以透過摻雜、電壓、光照等方式，在「導電」與「不導電」之間切換

這就是為什麼我們能用半導體做出「開關」（電晶體），進而建構整個數位世界。

### 一分鐘抓住核心

```
導體 ──── 半導體 ──── 絕緣體
(銅)      (矽)       (玻璃)
│          │           │
能帶隙≈0   能帶隙≈1eV  能帶隙>5eV
│          │           │
永遠導電   可控導電     永遠不導電
```

## 一、導體、半導體與絕緣體

### 1.1 用能帶隙（Bandgap, Eg）分類

在固體中，電子的能量不是連續的，而是形成「能帶」（Energy Band）。兩個最重要的能帶是：

- **價帶（Valence Band, VB）**：電子平時待的地方（滿的）
- **導帶（Conduction Band, CB）**：電子到這裡才能自由移動導電（空的）

價帶頂端和導帶底端之間的能量差，就叫做**能帶隙（Bandgap, Eg）**。

| 分類 | 能帶隙 Eg | 典型材料 | 導電性 |
|------|----------|---------|--------|
| 導體 | ≈ 0 eV（能帶重疊） | 銅 Cu、鋁 Al、金 Au | 非常好 |
| 半導體 | 0.1～4 eV | Si (1.12eV)、GaAs (1.42eV) | 可控制 |
| 絕緣體 | > 4～5 eV | SiO₂ (9eV)、Si₃N₄ (5eV) | 極差 |

**白話理解**：能帶隙就像一道「牆」。牆很矮（導體），電子輕鬆跳過去就能導電；牆中等（半導體），給點能量就跳得過去；牆太高（絕緣體），電子怎麼跳都過不去。

### 1.2 溫度效應

半導體的導電性會隨溫度上升而增加（與金屬相反！）。因為溫度越高，越多電子有足夠的熱能跳過能帶隙。

```
金屬：溫度↑ → 電阻↑（晶格振動增加，散射更多）
半導體：溫度↑ → 電阻↓（更多電子跳到導帶）
```

## 二、晶體結構

### 2.1 為什麼晶體結構重要？

半導體晶片的性能取決於原子排列的「整齊程度」：

- **單晶（Single Crystal）**：原子排列完全規則 → 電子移動順暢 → 用於晶片
- **多晶（Polycrystalline）**：多個小區域各自規則 → 用於閘極、太陽能板
- **非晶（Amorphous）**：原子排列雜亂 → 用於顯示器薄膜電晶體

### 2.2 基本晶格結構

**布拉維晶格（Bravais Lattice）**：自然界的晶體可以歸納為 14 種基本排列方式。半導體最常見的是面心立方（FCC, Face-Centered Cubic）。

**晶格常數（Lattice Constant, a）**：晶胞（Unit Cell）的邊長。

| 材料 | 晶格常數 a (Å) |
|------|---------------|
| Si   | 5.431         |
| GaAs | 5.653         |
| GaN  | a=3.189（六方） |

### 2.3 鑽石結構（Diamond Structure）

矽和鍺的晶體結構就是鑽石結構，特徵如下：

- 由**兩個互相穿插的 FCC 晶格**組成
- 第二個 FCC 沿體對角線方向偏移 (a/4, a/4, a/4)
- 每個原子與 4 個最近鄰原子形成**共價鍵**（正四面體排列）
- 每個晶胞包含 **8 個原子**

```
鑽石結構 = FCC 晶格 + 基底（0,0,0）和（a/4, a/4, a/4）兩個原子

        ●───────────●
       /|          /|
      / |  ◆     / |      ● = FCC 晶格點
     ●───────────●  |      ◆ = 偏移的第二組原子
     |  |     ◆  |  |
     |  ●───────-|--●
     | /    ◆    | /
     |/          |/
     ●───────────●
```

**配位數**：每個原子有 4 個最近鄰（共價鍵的特徵）

### 2.4 閃鋅結構（Zincblende Structure）

GaAs、InP 等 III-V 族化合物半導體採用閃鋅結構：

- 結構與鑽石結構幾乎一樣
- 差別在於：兩個 FCC 晶格分別由**不同元素**佔據
  - 一個 FCC 全是 Ga，另一個全是 As
- 因此**沒有反轉對稱性**（與鑽石結構不同）
- 這個對稱性差異會影響壓電效應等物理性質

```
鑽石結構：Si-Si-Si-Si（同一種原子）
閃鋅結構：Ga-As-Ga-As（兩種原子交替）
```

### 2.5 纖鋅礦結構（Wurtzite Structure）

GaN 和一些寬能隙半導體採用六方晶系的纖鋅礦結構：

- 六方密堆積（HCP）+ 基底中兩種原子
- 具有自發極化（Spontaneous Polarization）特性
- 這在 GaN HEMT 高電子遷移率電晶體中非常重要

## 三、Miller 指標（Miller Indices）

### 3.1 什麼是 Miller 指標？

Miller 指標是描述晶體中**平面方向**和**直線方向**的標記法。在半導體製程中，不同晶面的性質不同（氧化速率、蝕刻速率、介面態密度等），所以必須掌握。

### 3.2 晶面 (hkl) 的決定步驟

1. 找出平面在 x, y, z 軸的截距（以晶格常數 a 為單位）
2. 取倒數
3. 化為最小整數比
4. 用小括號 (hkl) 表示

**範例**：一個平面在 x 軸截距 = 1a, y 軸截距 = 2a, z 軸截距 = ∞

```
截距：1, 2, ∞
倒數：1, 1/2, 0
最小整數：2, 1, 0
Miller 指標：(210)
```

### 3.3 常見晶面

| 晶面 | 截距 (x,y,z) | 特徵 |
|------|-------------|------|
| (100) | 1, ∞, ∞ | 垂直 x 軸的平面 |
| (110) | 1, 1, ∞ | 對角面 |
| (111) | 1, 1, 1 | 體對角面，原子排列最密 |

**半導體製程中的應用**：

- **(100) 面**：MOSFET 最常用，因為 Si/SiO₂ 介面態密度最低
- **(111) 面**：原子排列最密，化學蝕刻最慢
- **(110) 面**：在某些特殊應用中使用

### 3.4 晶向 [hkl]

- 用方括號 [hkl] 表示
- 等價晶向族用角括號 <hkl> 表示
- 等價晶面族用大括號 {hkl} 表示

| 符號 | 意義 | 範例 |
|------|------|------|
| (hkl) | 特定晶面 | (100) |
| {hkl} | 等價晶面族 | {100} = (100),(010),(001),(1̄00),(01̄0),(001̄) |
| [hkl] | 特定晶向 | [110] |
| <hkl> | 等價晶向族 | <110> |

### 3.5 負指標的表示

負數用上方加一橫表示，例如 (1̄00) 表示 x 軸負方向的截距。在文字中常寫成 (-100)。

## 四、常見半導體材料

### 4.1 元素半導體

| 材料 | 能帶隙 Eg (eV) | 帶隙類型 | 主要應用 |
|------|---------------|---------|---------|
| Si   | 1.12          | 間接    | 邏輯晶片、記憶體、太陽能電池 |
| Ge   | 0.66          | 間接    | 高速元件、光偵測器 |

### 4.2 化合物半導體

| 材料 | 能帶隙 Eg (eV) | 帶隙類型 | 主要應用 |
|------|---------------|---------|---------|
| GaAs | 1.42          | 直接    | LED、雷射、高頻RF |
| InP  | 1.35          | 直接    | 光通訊、高速元件 |
| GaN  | 3.4           | 直接    | 藍光LED、功率元件、5G |
| SiC  | 3.26 (4H-SiC) | 間接    | 電動車功率元件 |
| InGaAs | 可調         | 直接    | 光偵測器、高速電晶體 |

### 4.3 為什麼 Si 稱霸？

1. **地殼中含量豐富**（第二多的元素，佔約 28%）
2. **SiO₂ 天然絕緣層**：矽的氧化物品質極佳，是 MOSFET 成功的關鍵
3. **成熟的製程技術**：數十年的累積，良率極高
4. **機械性質好**：矽晶圓不易碎裂
5. **Eg = 1.12 eV 剛好適中**：足以在室溫下維持低漏電

## 五、直接帶隙 vs 間接帶隙

### 5.1 基本概念

在 E-k 圖（能量-波向量圖）中：

- **直接帶隙（Direct Bandgap）**：導帶最低點和價帶最高點在**同一個 k 值**
- **間接帶隙（Indirect Bandgap）**：導帶最低點和價帶最高點在**不同的 k 值**

```
直接帶隙 (GaAs)              間接帶隙 (Si)

  E ↑                          E ↑
    |    /\                      |         /\
    |   /  \  CB                 |   /\   /  \  CB
    |  /    \                    |  /  \ /    \
    | / ↕Eg  \                   | /    X      \
    |/________\___→ k            |/ ↕Eg  \______\___→ k
    |\   VB   /                  |\  VB   /
    | \      /                   | \     /
    |  \    /                    |  \   /
    |   \  /                     |   \ /

  電子跳躍：只需能量          電子跳躍：需要能量 + 動量（聲子）
  → 容易發光                → 不容易發光
```

### 5.2 對元件應用的影響

| 特性 | 直接帶隙 | 間接帶隙 |
|------|---------|---------|
| 代表材料 | GaAs, GaN, InP | Si, Ge, SiC |
| 光發射效率 | 高 | 極低 |
| 光吸收效率 | 高（吸收層薄） | 較低（需要厚一點） |
| 復合方式 | 輻射復合為主 | 非輻射復合為主 |
| 主要應用 | LED、雷射、光偵測 | 邏輯電路、功率元件 |

**白話解釋**：

- 直接帶隙：電子從導帶掉回價帶時，能量直接變成光子射出 → 適合做 LED
- 間接帶隙：電子掉回去時，需要聲子（晶格振動）協助 → 能量變成熱而不是光 → 適合做邏輯電路（不需要發光，反而怕發光浪費能量）

### 5.3 為什麼 GaAs 做 LED，Si 做邏輯？

**GaAs 做 LED / 雷射的原因**：
- 直接帶隙 → 輻射復合效率高
- Eg = 1.42 eV 對應紅外光波長 λ = hc/Eg ≈ 870 nm
- 可以與其他 III-V 材料（如 AlGaAs、InGaP）形成異質接面，調整波長

**Si 做邏輯電路的原因**：
- 天然的 SiO₂ 絕緣層 → MOSFET 的基礎
- 成本低、良率高
- 間接帶隙反而是優勢：不會因為輻射復合而浪費能量

## 六、晶格缺陷（補充）

完美晶體不存在於現實中，常見缺陷包括：

| 缺陷類型 | 說明 | 對元件的影響 |
|---------|------|------------|
| 點缺陷（空位、間隙原子） | 少了一個原子或多了一個原子 | 產生深能階，增加復合 |
| 線缺陷（位錯 Dislocation） | 一排原子偏移 | 漏電流增加 |
| 面缺陷（堆疊缺陷、晶界） | 整面原子排列錯誤 | 嚴重影響元件可靠度 |

在半導體製程中，減少晶格缺陷是提高良率的關鍵。

## 數值例題

### 【例題 1】能帶隙與光子波長

**題目**：GaAs 的 Eg = 1.42 eV，求其對應光子波長。

**解答**：

光子能量與波長的關係：E = hc/λ

其中：
- h = 6.626 × 10⁻³⁴ J·s（普朗克常數）
- c = 3 × 10⁸ m/s（光速）
- 1 eV = 1.602 × 10⁻¹⁹ J

快速公式：**λ (μm) = 1.24 / Eg (eV)**

```
λ = 1.24 / 1.42 = 0.873 μm = 873 nm
```

這是近紅外光，人眼看不到，但用於光纖通訊和遙控器。

---

### 【例題 2】Miller 指標計算

**題目**：一個晶面通過 (2a, 3a, 6a) 三個軸截距，求 Miller 指標。

**解答**：

```
步驟 1：截距 = 2, 3, 6
步驟 2：取倒數 = 1/2, 1/3, 1/6
步驟 3：乘以最小公倍數 6 → 3, 2, 1
步驟 4：Miller 指標 = (321)
```

---

### 【例題 3】晶胞中的原子數

**題目**：計算鑽石結構（Si）一個晶胞中有多少個原子。

**解答**：

```
FCC 晶格的原子數：
  - 8 個角上的原子：8 × (1/8) = 1
  - 6 個面上的原子：6 × (1/2) = 3
  - FCC 共 4 個原子

鑽石結構 = 2 個 FCC 晶格穿插：
  - 第一個 FCC：4 個原子
  - 第二個 FCC（偏移 a/4, a/4, a/4）：4 個原子
  - 共 8 個原子
```

**答案：一個鑽石結構晶胞含 8 個原子**

---

### 【例題 4】原子密度計算

**題目**：已知 Si 的晶格常數 a = 5.431 Å，求 Si 的原子密度。

**解答**：

```
每個晶胞有 8 個原子
晶胞體積 = a³ = (5.431 × 10⁻⁸ cm)³ = 1.602 × 10⁻²² cm³

原子密度 = 8 / a³ = 8 / (1.602 × 10⁻²²)
         = 4.995 × 10²² atoms/cm³
         ≈ 5.0 × 10²² atoms/cm³
```

**答案：Si 的原子密度約為 5.0 × 10²² cm⁻³**

（這個數字很重要，常用來計算摻雜比例。例如摻雜濃度 10¹⁸ cm⁻³ 代表每 50,000 個 Si 原子中才有一個雜質原子。）

## ✅ 自我檢測

### 基礎題

**Q1**：Si 的能帶隙是多少？它是直接帶隙還是間接帶隙？

<details>
<summary>點擊查看答案</summary>

Si 的 Eg = 1.12 eV，是**間接帶隙**。導帶最低點在 X 方向的 Δ 谷，而價帶最高點在 Γ 點，兩者的 k 值不同。

</details>

**Q2**：為什麼 MOSFET 通常使用 (100) 晶面的矽晶圓？

<details>
<summary>點擊查看答案</summary>

因為 (100) 面的 Si/SiO₂ 介面態密度（interface trap density）最低，約為 (111) 面的十分之一。較低的介面態意味著較少的電荷陷阱，能提供更穩定的閾值電壓和更好的元件特性。

</details>

**Q3**：鑽石結構和閃鋅結構的主要差異是什麼？

<details>
<summary>點擊查看答案</summary>

兩者的幾何結構相同（都是兩個 FCC 穿插），差異在於：
- 鑽石結構：兩個 FCC 都是同一種原子（如 Si-Si）
- 閃鋅結構：兩個 FCC 是不同原子（如 Ga-As）
- 閃鋅結構因此**沒有反轉對稱性**，會產生壓電效應等特性

</details>

### 進階題

**Q4**：GaN 的 Eg = 3.4 eV，計算其對應波長，並說明這是什麼顏色的光。

<details>
<summary>點擊查看答案</summary>

λ = 1.24 / 3.4 = 0.365 μm = 365 nm

這是**紫外光（UV）**區域。GaN 常用於藍光/紫外 LED（搭配 InGaN 量子井可調到藍光 450 nm），中村修二因發明藍光 LED 獲得 2014 年諾貝爾物理獎。

</details>

**Q5**：某晶面的截距為 (3a, 2a, 4a)，求 Miller 指標。

<details>
<summary>點擊查看答案</summary>

```
截距：3, 2, 4
倒數：1/3, 1/2, 1/4
乘以最小公倍數 12 → 4, 6, 3
Miller 指標：(463)
```

</details>

**Q6**：為什麼 SiC 和 GaN 近年在電動車和 5G 領域受到重視？

<details>
<summary>點擊查看答案</summary>

SiC (Eg = 3.26 eV) 和 GaN (Eg = 3.4 eV) 是**寬能隙（Wide Bandgap, WBG）**半導體：

1. **高擊穿電壓**：Eg 大 → 臨界電場高 → 可承受更高電壓 → 適合功率元件
2. **高溫操作**：Eg 大 → 本質載子濃度低 → 在高溫下仍能維持正常工作
3. **低導通電阻**：臨界電場高 → 可以做更薄的漂移層 → RDS(on) 更低
4. **高頻操作**：GaN 的高電子遷移率 → 適合 5G RF 功率放大器

電動車的逆變器從 Si IGBT 換成 SiC MOSFET，可以提高 5-10% 的能源效率。

## 本章重點整理

```
1. 半導體靠能帶隙 Eg 定義：0.1 ~ 4 eV

3. GaAs: Eg=1.42eV, 直接帶隙, 閃鋅結構
4. GaN: Eg=3.4eV, 直接帶隙, 纖鋅礦結構
5. 鑽石結構 = 2個FCC穿插, 8 atoms/cell
6. Miller指標：截距→倒數→整數化→(hkl)
7. (100)面：MOSFET首選（介面態最低）
8. 直接帶隙→發光元件；間接帶隙→邏輯元件
9. λ(μm) = 1.24 / Eg(eV)
```
