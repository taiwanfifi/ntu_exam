# 刑法完全教學 — 從零開始搞懂「社會畫的那條不能越過的線」

> 律師／司法官國考一試及二試專用
> 專為非法律背景的轉職者設計，不需要任何法律基礎就能讀懂。

---

## 寫在最前面：這科到底在講什麼？

### 如果你從來沒學過法律，先讀這段

Criminal law defines the boundaries of acceptable behavior in society. When someone crosses that line, the state punishes them.

**民法管人與人之間的公平，刑法管社會畫的那條不能越過的線。**

想像你住在一棟大樓裡：
- 你鄰居半夜放音樂太大聲 → 這是你們之間的糾紛，民法管（侵權、鄰損）
- 你鄰居拿刀追你 → 這是你越過了社會底線，刑法管（殺人未遂）
- 你鄰居偷你信箱裡的包裹 → 刑法管（竊盜罪）
- 你跟鄰居簽約他不付錢 → 民法管（債務不履行）

民法是「你跟他的事」，刑法是「你跟整個社會的事」。偷東西不只是對被害人不公平，而是動搖了社會上「別人的東西不能拿」這條基本規則。所以國家（檢察官）要代替社會出來追究你。

### 刑法的兩大特徵

**1. 罪刑法定原則（§1）**

> 行為之處罰，以行為時之法律有明文規定者為限。

白話：「法律沒說是犯罪的，就不是犯罪。」

這是刑法最根本的原則。不管你的行為多麼「不道德」，如果刑法沒有明文規定處罰，法官不能判你有罪。例如：通姦在 2020 年被大法官宣告違憲除罪化後，就算你覺得很不應該，法院也不能用刑法處罰了。

罪刑法定原則衍生四大禁止：
- **禁止類推適用**：不能「比照」其他罪名來定罪
- **禁止溯及既往**：不能用新制定的法律處罰過去的行為
- **禁止習慣法**：不能用民間習慣來定罪
- **禁止不明確的構成要件**：法律條文必須讓人看得懂什麼行為會被處罰

**2. 最後手段性（謙抑原則）**

刑罰是國家最嚴厲的制裁手段（剝奪自由甚至生命），所以只有在其他法律手段（民事賠償、行政罰鍰）都不足以保護法益時，才動用刑罰。

### 考試重點預告

刑法考試的核心：**犯罪三階理論** + **分則重要罪名的適用**。

二試最典型的題目是給你一個故事，然後問「甲的刑事責任如何？」你需要用三階理論把每個可能的罪名逐一檢驗。

### 本講義的學習路徑

```
第一輪（3-4天）：通讀全文，理解犯罪三階的架構
第二輪（3-5天）：把每個階層的爭點吃透，搭配例題
第三輪（持續）：用考古題練習「檢驗流程」，直到能自動化地跑完三階
```

## 第一部分：犯罪三階理論 — 刑法的骨架

> 這是刑法總則最重要的東西。如果你只能記住一件事，就記住這個架構。

### 整體架構

一個行為要成立犯罪，必須通過三道關卡：

```
              犯罪三階層體系

第一關：構成要件該當性（Tatbestand）
  → 行為符合法條描述嗎？

        ↓ 通過

第二關：違法性（Rechtswidrigkeit）
  → 有沒有正當理由？

        ↓ 通過（沒有正當理由）

第三關：罪責（有責性）（Schuld）
  → 能不能責怪這個人？

        ↓ 通過

  結論：成立犯罪
```

**生活比喻**：想像你是機場安檢。

- 第一關（構成要件）：「旅客行李裡有沒有違禁品？」→ 有刀子 → 通過第一關
- 第二關（違法性）：「有沒有正當理由帶刀？」→ 他是廚師要去比賽，有大會許可函 → 第二關就擋下來了，不算違規
- 第三關（罪責）：「就算沒有正當理由，這個人有沒有判斷能力？」→ 如果是一個 10 歲小孩不知道不能帶，不能用對待大人的方式苛責他

三道關卡是**層層遞進**的：前一關沒通過，就不用檢查下一關。

### 第一關：構成要件該當性

構成要件分成**客觀面**和**主觀面**兩大塊。

#### 一、客觀構成要件

##### 1. 行為

刑法只處罰「行為」，不處罰單純的思想或身分。

行為必須是出於意思支配的舉動。以下不算「行為」：
- **反射動作**：被蜜蜂叮到手自然揮出去打到旁邊的人
- **睡眠中的動作**：夢遊時推倒花瓶砸到人
- **絕對強制下的動作**：被人抓住手臂去打第三人（物理上完全無法抗拒）

##### 2. 結果

有些犯罪要求特定結果發生才成立，叫「結果犯」；有些只要做了特定行為就成立，叫「行為犯」。

| 類型 | 說明 | 例子 |
|------|------|------|
| 結果犯 | 要有特定結果 | 殺人罪（要有人死）、傷害罪（要有人受傷） |
| 行為犯 | 做了就成立 | 偽證罪（在法庭上說謊就成立，不管有沒有影響判決） |

##### 3. 因果關係

如果是結果犯，行為和結果之間必須有因果關係。

**條件理論（sine qua non）**：「如果沒有這個行為，結果還會不會發生？」如果不會發生，就有因果關係。

例子：
- 甲開槍打乙，乙死了 → 沒有甲開槍，乙不會死 → 有因果關係
- 甲打了乙一巴掌，乙去醫院途中被車撞死 → 沒有甲打乙，乙不會去醫院，不會被車撞 → 條件理論下有因果關係（但後面的「客觀歸責」會處理這個問題）

##### 4. 客觀歸責理論

條件理論的因果鏈太廣了（蝴蝶效應會導致什麼都有因果關係），所以需要用「客觀歸責」來限縮。

客觀歸責三要件：
1. **製造法所不容許的風險**：行為人的行為製造了法律不允許的危險
2. **風險實現**：這個危險確實導致了結果發生
3. **構成要件效力範圍**：結果在該法條要保護的範圍內

**重要案例分析**：

| 案例 | 因果關係 | 客觀歸責 | 說明 |
|------|---------|---------|------|
| 甲刺傷乙，乙送醫途中車禍死亡 | 有 | 否 | 車禍死亡不是「刺傷」的風險實現 |
| 甲勸乙搭飛機，飛機墜毀 | 有 | 否 | 勸人搭飛機不是「製造法所不容許的風險」 |
| 甲超速撞死闖紅燈的乙 | 有 | 要看 | 如果甲即使不超速也一定撞到 → 風險未實現 |
| 甲拿刀砍乙，乙是血友病患者因此失血過多死亡 | 有 | 有 | 砍人的風險確實實現（特殊體質不影響歸責） |

#### 二、主觀構成要件

客觀面確認後，還要檢查行為人主觀上有沒有故意或過失。

##### 1. 故意（§13）

> 行為人對於構成犯罪之事實，明知並有意使其發生者，為故意。
> 行為人對於構成犯罪之事實，預見其發生而其發生並不違背其本意者，以故意論。

兩種故意：

**直接故意（確定故意）**：「我就是要他死」→ 明知 + 有意
**間接故意（未必故意）**：「死了也無所謂」→ 預見 + 不違背本意

##### 2. 過失（§14）

> 行為人雖非故意，但按其情節應注意，並能注意，而不注意者，為過失。
> 行為人對於構成犯罪之事實，雖預見其能發生而確信其不發生者，以過失論。

兩種過失：

**無認識過失**：根本沒想到會出事 → 應注意、能注意、而不注意
**有認識過失**：有想到可能出事，但確信不會 → 預見可能發生、確信不發生

> 下面第二部分會詳細展開故意與過失的光譜。

### 第二關：違法性

通過第一關後，推定行為違法。但如果有「阻卻違法事由」，就不成立犯罪。

#### 一、正當防衛（§23）

> 對於現在不法之侵害，而出於防衛自己或他人權利之行為，不罰。但防衛行為過當者，得減輕或免除其刑。

四個要件：

| 要件 | 說明 | 白話 |
|------|------|------|
| 現在 | 侵害正在發生或迫在眉睫 | 不能「事後報復」（侵害已結束就不行） |
| 不法 | 對方的侵害是違法的 | 警察依法逮捕你，你不能主張正當防衛 |
| 侵害 | 有具體的攻擊行為 | 只是嘴巴罵不算（但恐嚇可能算） |
| 防衛意思 | 主觀上是為了防衛 | 不能藉機報仇 |

**防衛過當**：防衛手段明顯超過必要程度。例如對方用拳頭打你，你拿刀捅他三刀。防衛過當不是完全不罰，而是「得減輕或免除其刑」。

**考試常見爭點**：
- 「預防性防衛」：知道對方明天要來打你，今天先下手 → 不算正當防衛（「現在」不成立）
- 「互毆」：雙方都在攻擊，一般認為雙方都沒有防衛意思，不成立正當防衛
- 「挑唆防衛」：故意激怒對方讓他先動手，再主張正當防衛 → 通說認為不得主張

#### 二、緊急避難（§24）

> 因避免自己或他人生命、身體、自由、財產之緊急危難而出於不得已之行為，不罰。但避難行為過當者，得減輕或免除其刑。

與正當防衛的關鍵差異：

| | 正當防衛 | 緊急避難 |
|---|---------|---------|
| 對象 | 對「不法侵害者」反擊 | 犧牲「無辜第三人」的利益 |
| 手段限制 | 不要求補充性（可直接反擊） | 要求補充性（不得已才行） |
| 均衡要求 | 較寬鬆 | 避難造成的損害不能大於避免的損害 |

例子：你被瘋狗追，踹開別人的門躲進去 → 你損壞了別人的門（財產），但保全了自己的身體 → 緊急避難成立。但如果你為了保全自己的財產而傷害別人的身體 → 法益權衡不通過。

#### 三、依法令之行為（§21）

> 依法令之行為，不罰。

例如：警察依法對現行犯實施逮捕，拘束了犯人的自由 → 形式上符合「私行拘禁罪」的構成要件，但因為是依法令的行為，阻卻違法。

公務員依法執行職務、醫師依法為病患手術等，都屬此類。

#### 四、被害人承諾（同意）

雖然刑法沒有明文規定，但學理與實務承認「得被害人承諾」可以阻卻違法。

要件：
- 被害人有處分權限（個人法益才行，生命不可放棄）
- 承諾出於自由意思（不能被脅迫或欺騙）
- 承諾在行為前或行為時存在

例子：
- 你同意朋友在你手臂上刺青（傷害）→ 有效同意，阻卻違法
- 你叫朋友殺了你（殺人）→ 生命不可處分，同意無效（但可能成立加工自殺 §275）

#### 五、超法規阻卻違法事由

除了法條明列的，還有學理承認的阻卻違法事由：
- **義務衝突**：同時面臨兩個義務，只能選擇履行其中一個
- **推測承諾**：無法取得被害人同意，但依合理判斷被害人會同意（例如鄰居不在家，你幫他關掉漏水的水龍頭而進入他家）

### 第三關：罪責（有責性）

通過前兩關後，最後要問：「能不能『責怪』這個人？」

#### 一、責任能力（§18、§19）

**年齡標準（§18）**：

| 年齡 | 責任能力 | 效果 |
|------|---------|------|
| 未滿 14 歲 | 無責任能力 | 不罰 |
| 14-18 歲 | 限制責任能力 | 得減輕其刑 |
| 滿 18 歲 | 完全責任能力 | 正常處罰 |
| 滿 80 歲 | 限制責任能力 | 得減輕其刑 |

**精神狀態（§19）**：

| 狀態 | 責任能力 | 效果 |
|------|---------|------|
| 行為時因精神障礙或心智缺陷，致不能辨識其行為違法或欠缺依其辨識而行為之能力 | 無責任能力 | 不罰 |
| 行為時因上述原因，致其辨識或控制能力顯著減低 | 限制責任能力 | 得減輕其刑 |

**重要例外 — 原因自由行為（§19 III）**：如果行為人是「故意或過失使自己陷入無責任能力狀態」，不能免責。

經典例子：甲想殺乙但不敢，故意喝醉酒壯膽後持刀殺了乙。甲不能主張「我喝醉了是精神障礙」來脫罪，因為他是故意讓自己喝醉的。

#### 二、不法意識（§16）

> 除有正當理由而無法避免者外，不得因不知法律而免除刑事責任。但按其情節，得減輕其刑。

白話：「不知道犯法不是藉口。」但如果你真的有合理理由不知道，可能減輕其刑；極端情況下（正當理由且無法避免）可以免責。

例子：外國人來台灣，在其母國合法的行為（例如特定藥物使用）在台灣是犯罪。如果他完全沒有機會知道台灣的法律 → 可能主張§16免責；但如果他已經在台灣住了五年 → 很難主張不知法律。

#### 三、期待可能性

能不能期待行為人在當時的情況下做出合法行為？

如果客觀上一般人在同樣處境下都無法遵守法律，就欠缺期待可能性，阻卻罪責。

例子：卡車司機被歹徒拿槍抵住頭，被迫運送毒品 → 雖然他客觀上運了毒品（構成要件該當），也沒有阻卻違法事由，但不能期待他在槍口下拒絕 → 阻卻罪責。

### 犯罪三階快速檢驗口訣

```
構成要件：客觀（行為→結果→因果→歸責）+ 主觀（故意/過失）
         → 「長什麼樣？」「心裡想什麼？」

違法性：  有無阻卻違法事由（正當防衛/緊急避難/依法令/同意）
         → 「有沒有好理由？」

罪責：    責任能力 + 不法意識 + 期待可能性
         → 「能不能怪他？」
```

## 第二部分：故意與過失 — 那條微妙的光譜

### 四種主觀狀態的光譜

這是國考最愛考的區分之一，尤其是「間接故意 vs 有認識過失」的界線。

```
明確知道+積極追求          知道可能+無所謂          知道可能+確信不會          根本沒想到
     |                      |                      |                      |
  直接故意              間接故意              有認識過失              無認識過失
     |←———— 故意 ————→|←———— 過失 ————→|
     |←————————————— 處罰較重 ——————————————→ 處罰較輕 ——→|
```

### 逐一拆解

#### 1. 直接故意（確定故意）

「我就是要讓結果發生。」

- 認知面：明知構成犯罪事實
- 意欲面：積極追求結果發生

例子：甲恨乙入骨，買了一把刀，埋伏在乙下班的路上，等乙經過時捅他心臟。

#### 2. 間接故意（未必故意、不確定故意）

「結果發生也沒關係。」

- 認知面：預見結果可能發生
- 意欲面：結果發生不違背本意（容任）

例子：甲在鬧區朝仇人開槍，他知道可能打到旁邊的路人，但心想「打到就打到吧」。對路人 → 間接故意。

#### 3. 有認識過失

「我知道可能出事，但我覺得不會。」

- 認知面：預見結果可能發生
- 意欲面：確信結果不會發生

例子：甲是卡車司機，連續開車 18 小時很疲勞，知道自己可能打瞌睡出事，但心想「我經驗豐富，撐一下就到了」。結果打瞌睡撞死人。

#### 4. 無認識過失

「我根本沒想到會這樣。」

- 認知面：完全沒有預見
- 注意義務：依其情節「應注意、能注意、而不注意」

例子：甲開車變換車道沒看後照鏡，撞到後方機車。他完全沒想到會撞到人，但他有義務看後照鏡。

### 間接故意 vs 有認識過失：考試最愛考的界線

兩者都有「預見」，差別在「態度」：

| | 間接故意 | 有認識過失 |
|---|---------|-----------|
| 認知 | 預見可能發生 | 預見可能發生 |
| 意志 | 「發生就發生吧」（容任） | 「應該不會發生」（確信不發生） |
| 法律效果 | 依故意犯處罰 | 依過失犯處罰（刑度通常低很多） |

**判斷技巧**：問自己「如果行為人事先確定知道結果一定會發生，他還會不會做？」
- 「還是會做」→ 間接故意
- 「就不會做了」→ 有認識過失

**經典考題**：飆車族在市區飆到時速 150 公里，撞死行人。
- 如果他心態是「管他的，就算撞到人也要飆」→ 間接故意（殺人罪）
- 如果他心態是「我技術好，不可能撞到人」→ 有認識過失（過失致死罪）
- 實務上多數認定為間接故意，因為時速 150 撞到人幾乎必死，很難說「確信不會撞到」。

## 第三部分：不作為犯 — 什麼都沒做也會犯罪？

### 基本概念

刑法處罰的不只是「做了壞事」（作為犯），也包括「該做而沒做」（不作為犯）。

兩種不作為犯：

| 類型 | 說明 | 例子 |
|------|------|------|
| **純正不作為犯** | 法條明文規定「不作為」就是犯罪 | §149 公然聚眾不遵令解散罪；§294 遺棄罪 |
| **不純正不作為犯（§15）** | 原本要求「作為」的犯罪，用「不作為」方式實現 | 母親不餵食嬰兒導致嬰兒餓死 → 不作為殺人 |

### 不純正不作為犯的要件（§15）

> 對於犯罪結果之發生，法律上有防止之義務，能防止而不防止者，與因積極行為發生結果者同。
> 因自己行為致有發生犯罪結果之危險者，負防止其發生之義務。

要成立不純正不作為犯，需要：

1. **保證人地位**：行為人對結果的不發生有法律上的義務（不是道德義務）
2. **作為可能性**：有能力採取行動防止結果
3. **結果迴避可能性**：如果採取行動，結果很可能不會發生
4. **等價性**：不作為的惡性相當於積極作為

### 保證人地位的來源

這是不作為犯最核心的考點。「誰」有義務防止結果？

| 來源 | 說明 | 例子 |
|------|------|------|
| **法律明文規定** | 法律直接課予保護義務 | 父母對未成年子女（民法§1084）；公務員的職務義務 |
| **契約或自願承擔** | 基於契約接受保護他人的任務 | 保母對嬰兒、游泳池救生員對泳客、看護對病患 |
| **危險前行為（前行為義務）** | 自己先前的行為製造了危險 | 你把鄰居推入水中，你有義務把他救起來 |
| **密切生活關係** | 特定親密關係產生的保護義務 | 配偶之間、同居伴侶之間 |
| **危險源監督義務** | 對自己控制的危險源有管理義務 | 養了一隻兇猛的狗，你有義務不讓牠咬人 |

**經典考題**：

甲帶女友乙去海邊游泳。乙溺水呼救，甲明明會游泳也在旁邊，卻因為前一天跟乙大吵一架而故意不救。乙溺死。

分析：
- 甲有保證人地位嗎？→ 有。基於密切生活關係（男女朋友同遊），且甲帶乙去海邊是「危險前行為」（將乙帶入危險環境）
- 甲有能力救嗎？→ 有（會游泳且在旁邊）
- 甲不救是否等同殺人？→ 是的，甲成立不作為殺人罪

### 路人有沒有救助義務？

一般路人看到有人溺水，在刑法上**沒有**保證人地位。不救不構成殺人罪。但要注意：可能構成§294遺棄罪的問題（對於無自救力之人，依法令或契約應扶助而不扶助）。

## 第四部分：錯誤論 — 搞錯了怎麼辦？

### 錯誤論總覽

行為人對事實或法律產生誤認時，如何影響罪責？

```
錯誤論
├── 構成要件錯誤（事實錯誤）→ 影響「故意」
│   ├── 客體錯誤
│   ├── 打擊錯誤
│   └── 因果歷程錯誤
└── 禁止錯誤（法律錯誤）→ 影響「罪責」中的不法意識
```

### 一、構成要件錯誤

#### 1. 客體錯誤（對象錯誤）

行為人搞錯對象，但對象屬於同一類型。

例子：甲要殺乙，在暗巷中把丙誤認為乙，殺了丙。

通說結論：**不影響故意**。甲有殺「人」的故意，也確實殺了「人」，至於殺的是誰並不重要（法律保護的是「人的生命」，不是「特定某人的生命」）。甲成立殺人既遂。

但如果是「等價性不同」的客體錯誤：甲要殺乙，結果打到旁邊乙的狗 → 對乙成立殺人未遂，對狗不成立故意毀損（因為沒有毀損的故意）。

#### 2. 打擊錯誤（方法錯誤）

行為人認準了對象，但手段失誤打到別人。

例子：甲要射殺乙，瞄準乙開槍，但槍法太差打到旁邊的丙。

兩大學說：

| 學說 | 結論 | 理由 |
|------|------|------|
| **具體符合說（通說）** | 對乙：殺人未遂；對丙：過失致死 → 想像競合 | 甲的故意具體指向乙，對丙沒有故意 |
| **法定符合說** | 對丙：殺人既遂 | 甲有殺「人」的故意，丙也是「人」，故意存在 |

考試通常用**具體符合說**作答。

#### 3. 因果歷程錯誤

行為人對因果過程的想像與實際不同。

例子：甲把乙從橋上推下去，想讓乙溺死。結果乙頭撞到橋墩先死了。

結論：因果歷程的偏離如果在「一般生活經驗可預見的範圍內」，不影響故意。把人從橋上推下去，不管是溺死還是撞死，都在風險範圍內。甲成立殺人既遂。

**韋伯概括故意（延續的因果歷程錯誤）**：

甲掐乙脖子以為乙死了（其實只是昏迷），為了毀屍滅跡把乙丟入河中，乙實際上溺死在河裡。

分析：第一個行為（掐脖子）→ 殺人未遂；第二個行為（丟河裡）→ 過失致死。想像競合。
但如果認為兩個行為是「一個整體行為計畫」的一部分 → 因果歷程偏離在可預見範圍內 → 殺人既遂。通說多採後者。

### 二、禁止錯誤（§16）

行為人不知道自己的行為是違法的。

與構成要件錯誤的關鍵差異：
- 構成要件錯誤：「我不知道我在做什麼」→ 阻卻故意
- 禁止錯誤：「我知道我在做什麼，但我不知道這是犯法的」→ 不阻卻故意，影響罪責

| 禁止錯誤類型 | 效果 |
|-------------|------|
| 不可避免的禁止錯誤 | 阻卻罪責（不罰） |
| 可避免的禁止錯誤 | 得減輕其刑 |

判斷「可不可以避免」：行為人有沒有盡到查詢義務？一般人在同樣情況下會不會知道這是違法的？

## 第五部分：未遂犯 — 做了但沒成功

### 犯罪發展階段

```
陰謀 → 預備 → 著手 → 未遂 → 既遂
                 ↑
            處罰起點（原則上）
```

**原則**：刑法處罰的起點是「著手」。單純的陰謀和預備原則上不罰，除非法律有特別規定（例如§271 II 預備殺人罪）。

### 一、著手的認定

「著手」是區分預備與未遂的關鍵。何時算「著手」？

主要學說：

| 學說 | 標準 | 例子 |
|------|------|------|
| 形式客觀說 | 開始實行構成要件行為 | 殺人罪 → 開始砍的時候 |
| 實質客觀說 | 行為對法益產生直接危險 | 闖入被害人房間準備行兇時 |
| 主客觀混合說（通說） | 依行為人犯罪計畫，已達對法益直接危險 | 依甲的計畫，他埋伏在乙家門口舉起刀時 |

### 二、未遂的三種類型

#### 1. 普通未遂（障礙未遂）（§25）

> 已著手於犯罪行為之實行而不遂者，為未遂犯。未遂犯之處罰，以有特別規定者為限，並得按既遂犯之刑減輕之。

行為人已著手實行，但因為外部障礙沒有完成。

例子：甲拿刀砍乙，被路人制伏 → 殺人未遂。

#### 2. 不能未遂（§26）

> 行為不能發生犯罪之結果，又無危險者，不罰。

行為人著手實行了，但從客觀上看根本不可能產生結果。

例子：
- 甲用玩具槍射乙想殺他 → 玩具槍不可能射死人 → 不能未遂 → 不罰
- 甲要毒死乙，在乙的咖啡裡放砂糖（以為是毒藥）→ 不能未遂 → 不罰
- 甲朝乙開真槍但沒打中 → 這不是不能未遂（真槍有危險），而是普通未遂

關鍵判斷：行為有沒有「危險性」？有危險 → 普通未遂（可罰）；無危險 → 不能未遂（不罰）。

#### 3. 中止未遂（§27）

> 已著手於犯罪行為之實行，而因己意中止或防止其結果之發生者，減輕或免除其刑。

行為人「自己決定」放棄犯罪或積極防止結果發生。

與普通未遂的差異：

| | 普通未遂 | 中止未遂 |
|---|---------|---------|
| 停止原因 | 外部障礙（被抓、被阻止） | 出於己意（自己不想做了） |
| 法律效果 | 得減輕 | 必減輕或免除 |

**判斷公式（Frank公式）**：
- 「我想繼續但做不到」→ 普通未遂
- 「我做得到但不想繼續」→ 中止未遂

例子：
- 甲正在偷東西，聽到警笛聲嚇得逃跑 → 普通未遂（「做不到」）
- 甲正在偷東西，突然良心發現，把東西放回去離開 → 中止未遂（「不想做了」）
- 甲刺了乙一刀後後悔，立刻叫救護車救了乙 → 中止未遂（積極防止結果發生）

**中止未遂的優惠待遇**理由：法律鼓勵「回頭是岸」，給犯罪人一座「回頭的金橋」。

## 第六部分：正犯與共犯 — 一起犯罪怎麼算？

### 整體架構

```
犯罪參與
├── 正犯（自己犯罪的人，§28-§31）
│   ├── 直接正犯：自己動手
│   ├── 間接正犯：利用他人當工具
│   └── 共同正犯（§28）：一起合作犯罪
└── 共犯（參與他人犯罪的人）
    ├── 教唆犯（§29）：叫別人去犯罪
    └── 幫助犯（§30）：幫別人犯罪
```

### 一、直接正犯

自己親手實行犯罪行為。最單純的形態。

例子：甲自己拿刀殺了乙。甲是殺人罪的直接正犯。

### 二、間接正犯

利用他人作為「工具」來犯罪。被利用的人通常不知情或不具備責任能力。

常見類型：

| 類型 | 例子 |
|------|------|
| 利用無責任能力者 | 甲叫 10 歲小孩去偷東西 → 甲是竊盜罪間接正犯 |
| 利用不知情者 | 甲把毒藥交給護士，謊稱是藥品，護士不知情注射給病人 → 甲是殺人罪間接正犯 |
| 利用有阻卻違法事由者 | 甲騙乙說「丙要殺你」，讓乙對丙「正當防衛」→ 甲是間接正犯 |

### 三、共同正犯（§28）

> 二人以上共同實行犯罪之行為者，皆為正犯。

兩個要件：
1. **共同行為決意（犯意聯絡）**：大家事先說好要一起幹
2. **共同行為實行（行為分擔）**：每個人都有實際參與

**核心原則 — 「一部行為，全部責任」**：

只要有犯意聯絡和行為分擔，即使你只做了一部分，也要對全部結果負責。

例子：甲把風、乙撬鎖、丙搬東西 → 三人都成立竊盜罪的共同正犯，不是甲只負責把風的部分。

**考試爭點：共謀共同正犯**

甲是幕後老大，策劃了整個搶銀行的計畫，但他自己沒到現場。乙和丙按照甲的計畫去搶。甲算不算共同正犯？

通說＋實務見解：**算**。雖然甲沒有到場「實行」，但他的策劃行為對犯罪有重要支配力，屬於「共謀共同正犯」。

### 四、教唆犯（§29）

> 教唆他人使之實行犯罪行為者，為教唆犯。教唆犯之處罰，依其所教唆之罪處罰之。

要件：
1. 被教唆人本來沒有犯罪的意思（如果本來就想犯，你只是推了一把 → 可能是幫助犯）
2. 教唆行為：用勸說、利誘、激將等方式讓他人「產生犯意」
3. 被教唆人確實實行了犯罪（至少到著手階段）

**從屬性原則**：教唆犯的成立「從屬於」正犯。如果正犯沒有著手實行，教唆犯也不成立（但可能成立§29 III的未遂教唆）。

例子：甲跟乙說：「丙搶了你女朋友，你應該教訓他一頓。」乙因此去打了丙。甲是傷害罪的教唆犯。

### 五、幫助犯（§30）

> 幫助他人實行犯罪行為者，為幫助犯。幫助犯之處罰，得按正犯之刑減輕之。

要件：
1. 幫助行為：提供物質上或精神上的助力（借車、望風、提供工具、鼓勵打氣）
2. 幫助故意：知道對方要犯罪，且有意幫助
3. 正犯確實實行了犯罪

**教唆犯 vs 幫助犯**：

| | 教唆犯 | 幫助犯 |
|---|-------|-------|
| 對正犯的影響 | 「產生」犯意 | 「強化」既有犯意或提供便利 |
| 刑度 | 依正犯之刑（同等處罰） | 得按正犯之刑減輕 |

### 共犯的重要原則

**共犯從屬性說（限制從屬形式）**：共犯的處罰從屬於正犯行為的「構成要件該當＋違法性」。正犯至少要具備構成要件該當性和違法性，共犯才能成立。

## 第七部分：競合論 — 一個行為觸犯多條法律怎麼辦？

### 為什麼需要競合論？

現實生活中，一個事件可能同時觸犯好幾條法律。例如：甲開一槍同時打死了乙和丙 → 這算一個殺人罪還是兩個？怎麼量刑？

### 三種競合類型

#### 一、想像競合（§55）

> 一行為而觸犯數罪名者，從一重處斷。

**一個行為**，觸犯**多個罪名**，只按最重的那個罪處罰。

例子：
- 甲開一槍同時打死乙和打傷丙 → 一個開槍行為，觸犯殺人罪（對乙）和傷害罪（對丙）→ 從一重處斷 → 論殺人罪
- 甲闖進別人家偷東西 → 一個入侵行為同時構成侵入住宅罪和竊盜罪 → 從一重處斷

#### 二、實質競合（數罪併罰）（§50-§51）

**數個行為**，各自構成**不同犯罪**，分別論罪再合併處罰。

例子：甲星期一偷了A的錢包，星期三打了B一巴掌。兩個完全不同的行為，分別成立竊盜罪和傷害罪，數罪併罰。

數罪併罰的計算方式（§51）：在各刑中最長期以上、各刑合併之刑期以下，定其應執行刑。但不得超過法定上限。

#### 三、法條競合（法律競合）

一個行為看似符合數個法條，但法條之間有**邏輯關係**，只適用其中一個。

四種類型：

| 類型 | 關係 | 例子 |
|------|------|------|
| **特別關係** | 特別法優先於普通法 | 殺害直系血親尊親屬（§272）vs 普通殺人（§271）→ 用§272 |
| **補充關係** | 主要法排斥補充法 | 既遂犯排斥未遂犯 |
| **吸收關係** | 高度行為吸收低度行為 | 偽造文書後行使 → 行使偽造文書罪吸收偽造行為 |
| **擇一關係** | 互斥的法條 | 較少見 |

### 想像競合 vs 法條競合的差異

| | 想像競合 | 法條競合 |
|---|---------|---------|
| 本質 | 真的觸犯多個罪 | 看起來觸犯多個罪，其實只犯一罪 |
| 處理 | 從一重處斷（但輕罪仍存在） | 只適用一個法條，其他排除 |
| 前科紀錄 | 多個罪名都記錄 | 只有一個罪名 |

## 第八部分：刑法分則重要罪名 — 考試高頻精華

> 分則是把總則的理論套用到具體罪名上。以下是考試最常出現的罪名群組。

### 一、生命身體法益的罪

#### 殺人罪（§271）

> 殺人者，處死刑、無期徒刑或十年以上有期徒刑。

- 客觀：殺人行為 + 死亡結果 + 因果歸責
- 主觀：殺人故意（直接或間接）

**預備殺人（§271 II）**：預備犯罪也罰，刑法例外規定。

#### 傷害罪（§277）

> 傷害人之身體或健康者，處五年以下有期徒刑。

- 「傷害」包括身體完整性的破壞和生理機能的損害
- 傷害致死（§277 II）：傷害行為導致死亡，加重結果犯

#### 過失致死（§276）

> 因過失致人於死者，處五年以下有期徒刑。

- 行為人沒有殺人故意，但有過失（應注意能注意而不注意）
- 最常見：車禍致死、醫療過失致死

#### 三者的關鍵區分

| 罪名 | 主觀 | 結果 | 刑度（概略） |
|------|------|------|-------------|
| 殺人罪 §271 | 殺人故意 | 死亡 | 十年以上至死刑 |
| 傷害致死 §277 II | 傷害故意 + 對死亡結果有過失 | 死亡 | 較殺人輕 |
| 過失致死 §276 | 過失 | 死亡 | 五年以下 |
| 傷害罪 §277 I | 傷害故意 | 受傷 | 五年以下 |

考試最愛問的就是：「甲打了乙一拳，乙倒地撞到頭死了。甲成立什麼罪？」

分析思路：
- 甲有沒有殺人故意？→ 打一拳通常沒有 → 排除殺人罪
- 甲有沒有傷害故意？→ 有
- 死亡結果是否可預見？→ 打人倒地撞頭死亡，客觀上可預見 → 成立傷害致死

### 二、財產法益的罪

這是分則考試最大的罪名群。

#### 竊盜罪（§320）

> 意圖為自己或第三人不法之所有，而竊取他人之動產者。

- 核心行為：「竊取」→ 破壞他人持有，建立自己持有
- 對象：他人的動產
- 主觀：不法所有意圖（想據為己有）

#### 搶奪罪（§325）

> 意圖為自己或第三人不法之所有，而搶奪他人之動產者。

- 核心行為：「搶奪」→ 趁人不備，公然奪取
- 與竊盜的差異：竊盜是「偷偷拿」，搶奪是「乘其不及抵抗，公然奪取」
- 例子：飛車搶皮包

#### 強盜罪（§328）

> 意圖為自己或第三人不法之所有，以強暴、脅迫、藥劑、催眠術或他法，至使不能抗拒，而取他人之物或使其交付者。

- 核心行為：用強暴脅迫「至使不能抗拒」→ 取財
- 手段：必須達到「不能抗拒」的程度
- 與搶奪的差異：強盜有「強暴脅迫」，搶奪只是「趁不備」

#### 詐欺罪（§339）

> 意圖為自己或第三人不法之所有，以詐術使人將本人或第三人之物交付者。

構造五步驟（缺一不可）：
1. 施用詐術（積極欺騙或消極隱瞞）
2. 使被害人陷於錯誤
3. 被害人基於錯誤而處分財產
4. 行為人或第三人獲得財產利益
5. 財產損害

關鍵：被害人是「自願」交付的（雖然是被騙），這是詐欺和竊盜最大的差異。竊盜是違反被害人意思取得，詐欺是利用被害人的瑕疵意思取得。

#### 侵占罪（§335）

> 意圖為自己或第三人不法之所有，而侵占自己持有他人之物者。

- 核心特徵：東西本來就「合法地」在你手上，你卻據為己有
- 例子：朋友借你筆電，你不還了 → 侵占罪
- 與竊盜的差異：竊盜是「別人手上的東西我拿走」，侵占是「我手上別人的東西我不還」

#### 背信罪（§342）

> 為他人處理事務，意圖為自己或第三人不法之利益，或損害本人之利益，而為違背其任務之行為，致生損害於本人之財產或其他利益者。

- 主體：為他人處理事務的人
- 行為：違背任務
- 結果：造成本人的財產損害
- 例子：公司經理把公司的訂單轉給自己的私人公司

#### 財產犯罪的快速比較表

| 罪名 | 取得方式 | 核心特徵 |
|------|---------|---------|
| **竊盜** | 偷偷拿 | 破壞他人持有 |
| **搶奪** | 趁不備公然奪 | 乘人不及抵抗 |
| **強盜** | 暴力脅迫後拿 | 至使不能抗拒 |
| **詐欺** | 騙人家自己給 | 施詐術→陷於錯誤→處分 |
| **侵占** | 本來合法持有→不還 | 易持有為所有 |
| **背信** | 為人處理事務卻搞鬼 | 違背任務致損害 |

### 三、自由法益的罪

#### 強制罪（§304）

> 以強暴、脅迫使人行無義務之事或妨害人行使權利者。

- 白話：用暴力或威脅逼別人做不該做的事，或阻止別人做有權做的事
- 例子：擋住別人的車不讓他走、逼人下跪道歉

#### 恐嚇罪（§305）

> 以加害生命、身體、自由、名譽、財產之事恐嚇他人，致生危害於安全者。

- 要件：通知惡害 + 致生危害於安全（被害人真的感到害怕）
- 惡害內容必須是行為人能控制的（「你出門小心被車撞」不算，因為你控制不了）
- 例子：傳訊息說「再亂講話就砍你」

#### 私行拘禁（§302）

> 私行拘禁或以其他非法方法，剝奪人之行動自由者。

- 只要剝奪行動自由即成立，不問時間長短
- 例子：把人鎖在房間裡、不讓人離開車上

### 四、偽造文書罪群（§210-§220）

#### 基本結構

```
偽造文書罪群
├── 偽造私文書（§210）
├── 偽造公文書（§211）
├── 變造文書（§210、§211）
├── 行使偽造文書（§216）
└── 準文書（§220）：以文書論（如電磁紀錄）
```

核心概念：
- **偽造**：無製作權人，冒用他人名義製作文書（例如偽造別人的簽名）
- **變造**：有真正文書存在，擅自更改其內容（例如改成績單上的分數）
- **行使**：把偽造/變造的文書拿來用

**考試重點**：偽造後行使 → 行使偽造文書罪吸收偽造行為（吸收關係）→ 只論行使偽造文書罪。

#### 「足以生損害於公眾或他人」

偽造文書罪要求行為「足以生損害」，這是一個抽象危險，不需要真的已經造成損害，只要「有可能」造成損害就夠了。

### 五、公共危險罪（§173-§194）

保護的是「公共安全」，不是特定個人。

#### 重要罪名

| 條文 | 罪名 | 白話 |
|------|------|------|
| §173 | 放火燒燬現供人使用之住宅 | 燒有人住的房子 |
| §174 | 放火燒燬非現供人使用之住宅 | 燒沒人住的房子 |
| §175 | 放火燒燬前二條以外之物 | 燒其他東西（如草地） |
| §185-3 | 不能安全駕駛罪（酒駕） | 酒駕/毒駕 |
| §185-4 | 肇事逃逸罪 | 撞到人跑掉 |

#### 放火罪的特殊結構

放火罪是「抽象危險犯」，不需要真的造成人員傷亡就成立。燒有人住的房子本身就被認定有高度公共危險。

§173（燒有人住的房子）是重罪：處無期徒刑或七年以上有期徒刑。即使是自己的房子，只要現在有人住在裡面，也成立本罪。

#### 酒駕相關罪名（§185-3）

近年修法加重：
- 吐氣酒精濃度達 0.25mg/L 或血液酒精濃度達 0.05% → 成立本罪
- 因而致人於死：加重處罰
- 因而致重傷：加重處罰

## 第九部分：總整理 — 考試實戰框架

### 國考答題的標準檢驗流程

拿到題目後，按照以下流程逐一檢驗每個可能的罪名：

```
Step 1：找出所有行為人和行為
        → 甲做了什麼？乙做了什麼？

Step 2：對每個行為，判斷可能觸犯的罪名
        → 甲拿刀砍人 → 可能是殺人罪或傷害罪

Step 3：對每個罪名，跑犯罪三階

  (1) 構成要件該當性
      客觀面：行為→結果→因果→歸責
      主觀面：故意/過失/特殊主觀要件（如不法所有意圖）

  (2) 違法性
      有無阻卻違法事由？

  (3) 罪責
      責任能力？不法意識？期待可能性？

Step 4：處理參與問題
        → 正犯？共同正犯？教唆犯？幫助犯？

Step 5：處理競合問題
        → 想像競合？實質競合？法條競合？

Step 6：寫出結論
        → 「甲成立殺人罪（§271 I），乙成立殺人罪之教唆犯（§29、§271 I）」
```

### 答題格式範例

題目：甲教唆乙去殺丙。乙持刀前往，但刺了丙一刀後心生悔悟，立刻叫救護車送丙就醫。丙經搶救後保住一命。

```
一、乙刺丙一刀之行為，可能成立殺人未遂罪（§271 II）：
  (一) 構成要件該當性：
    1. 客觀面：乙持刀刺丙，已著手殺人行為之實行，
       丙未死亡，未達既遂。
    2. 主觀面：乙有殺人之直接故意。
    → 構成要件該當。

  (二) 違法性：無阻卻違法事由。

  (三) 罪責：乙具完全責任能力，有不法意識，具期待可能性。

  (四) 中止未遂之適用（§27）：
    乙因己意中止，且積極防止結果發生（叫救護車），
    丙經搶救存活 → 成立中止未遂，必減輕或免除其刑。

  結論：乙成立殺人罪之中止未遂（§271 II、§27），
       應減輕或免除其刑。

二、甲教唆乙殺丙，可能成立殺人未遂罪之教唆犯（§29、§271 II）：
  (一) 甲以言語使原無犯意之乙產生殺人犯意 → 教唆行為成立。
  (二) 乙已著手實行 → 正犯行為至少達未遂階段。
  (三) 甲具教唆故意。

  結論：甲成立殺人未遂罪之教唆犯（§29 I、§271 II）。

  注意：乙的中止未遂減免效果不及於甲（中止是一身專屬事由）。
```

### 各章節對應考點速查

| 章節 | 核心考點 | 出題頻率 |
|------|---------|---------|
| 犯罪三階 | 客觀歸責、因果關係 | 每年必考 |
| 故意過失 | 間接故意 vs 有認識過失 | 極高頻 |
| 不作為犯 | 保證人地位認定 | 高頻 |
| 錯誤論 | 打擊錯誤、客體錯誤 | 高頻 |
| 未遂犯 | 著手時點、中止未遂 | 高頻 |
| 正犯共犯 | 共同正犯、間接正犯 | 極高頻 |
| 競合論 | 想像競合 vs 實質競合 | 中高頻 |
| 殺人vs傷害 | 殺人故意 vs 傷害故意 | 極高頻 |
| 財產犯罪 | 竊盜/詐欺/侵占區分 | 極高頻 |
| 偽造文書 | 偽造vs變造、行使吸收 | 中高頻 |
| 公共危險 | 酒駕、放火 | 中頻 |
