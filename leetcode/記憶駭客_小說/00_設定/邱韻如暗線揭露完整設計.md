## 第一幕：維倫破解 W（揭露前的鋪墊）

### 破解過程分三步

**第一步：停滯（40 字）**

```
維倫在白板前。
W。
那個空圓已經盯了他三天。
```

寫法指令：三天的停滯 = 讓讀者跟維倫一起「卡住」。不要用倒敘或回憶填充——就是盯著白板。

**第二步：轉向語青（60 字）**

```
他把所有已知的線索重新歸併：
  ch131 Union-Find 結果 = 所有篡改指向同一根節點
  根節點簽名 = W
  W 不在任何已知代號庫中

「如果 W 不是代號——」

他轉頭看向語青。
語青的臉沒有表情。
但她的手指在微微發抖。
```

寫法指令：語青的手指抖 = 她知道維倫即將揭開她最大的秘密。抖不是恐懼，是預期中的解脫。讀者此時的注意力被語青的反應吸引——「她知道什麼？」

**第三步：語言跳躍——破解的確切瞬間（120 字）**

```
維倫重新看那個 W。

不是英文。
不是代號。

是名字。

他在腦中翻過所有跟這條暗線相關的人物：
  記憶銀行內部
  首席技術顧問
  ch123 的授權簽章

邱韻如。

韻。

ㄩㄣˋ。

注音的羅馬拼音——yùn。
首字母——

「W 不是 W。」
維倫的聲音很輕。
像是在確認一個他不想確認的答案。

「W 是 Y 的鏡像翻轉。
是『韻』的注音首字母。
是邱韻如。」
```

### 揭露的確切瞬間

讓讀者意識到真相的**那一個字**是：

> **韻。**

不是「邱韻如」三個字（太長，理性處理）。不是「W 不是 W」（太抽象）。是單獨成行的「韻。」——讀者的眼睛掃到這個字時，如果他們記得 Vol 1 ch35，腦中會閃電般連結：韻 → 邱韻如 → 嚴柏翰的前戀人 → 語青的母親。

**排版指令**：「邱韻如。」「韻。」「ㄩㄣˋ。」三個詞各自獨立成行。每行之間留一個空行。讓讀者的視線在三個詞之間跳躍，像維倫的腦在跳躍。

### 第一幕寫作指令

# 肆、ch133「她是我媽」的寫法銜接

## ch132 → ch133 之間的距離

- **文字距離**：0 字。ch133 第一段緊接 ch132 最後一段的沉默。
- **敘事時間距離**：數分鐘。沉默持續了「很久」——不精確量化。
- **物理距離**：讀者翻頁。這一個翻頁動作 = 唯一的緩衝。

**不要在 ch132 末尾和 ch133 開頭之間插入任何過渡。** 沉默→翻頁→語青站起來。零過渡 = 最大衝擊。

## 語青開口前的身體語言

```
沉默持續了很久。

語青站起來。
```

只有兩個動作：「沉默」和「站起來」。

- 「站起來」= 她選擇站著告白，不是坐著。**這是尊嚴。** 坐著說 = 求饒。站著說 = 承擔。
- 不寫她擦眼淚、不寫她深呼吸、不寫她看向誰。她站起來。就這樣。

## 告白段落

核心台詞三行獨立排列，每行比上行更重：

```
「邱韻如是我的母親。」
（停頓五秒。）
「從一開始。
從我第一天走進維倫的工作室。
我就知道。」
```

### 「從一開始，我就知道」的重量

- 「從一開始」= ch1。**她從第一頁就知道。** 讀者會翻回 ch1 重讀語青敲門的場景
- 「從第一天走進維倫的工作室」= 時間錨點。她不是需要幫助的人——她帶著一整盤棋走進來的

這不是可以被原諒的事。也不是可以被簡單定義為「惡」的事。它是全書最核心的道德灰區——愛可以是暴力嗎？保護可以是侵犯嗎？

## 三、邱韻如 x 周明哲：同事 → 被操控 → 殘像對峙
