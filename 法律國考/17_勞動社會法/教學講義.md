## 開場：為什麼你應該關心勞動法？

If you've ever had a job, you've been protected (or should have been) by labor law. Got fired unfairly? Labor law. Didn't get overtime pay? Labor law. Workplace injury? Labor law. This is the law that affects every working person in Taiwan — and as a career changer, it probably affects YOU right now.

你現在正在準備轉職。你可能已經離開了上一份工作，或者還在職邊準備考試。不管你的情況如何，以下這些問題你一定想過：

- 公司叫我簽的勞動契約，真的合法嗎？
- 我加班這麼多，為什麼薪水單上加班費這麼少？
- 公司說我是「責任制」，所以沒有加班費——這是真的嗎？
- 如果公司裁員把我裁了，我能拿到多少資遣費？
- 我在工作中受傷了，老闆說那不算職災，真的嗎？
- 離職後沒工作，政府有什麼補助可以領？

**這些全部都是勞動社會法在處理的問題。**

勞動社會法不像民法那樣抽象，也不像刑法那樣嚴肅。它是最「接地氣」的法律——每一條規定都直接影響你的工作權益、你的薪水、你的休假、你的退休金。

# 第一章：勞動基準法

## 本章概覽

勞動基準法（以下簡稱「勞基法」）是台灣最重要的勞動法律，沒有之一。它規定了勞動條件的「最低標準」——工資不能低於這個數字、工時不能超過這個上限、解僱不能違反這些規定。

**關鍵觀念：勞基法是「地板」，不是「天花板」。**

什麼意思？勞基法規定的是最低標準。雇主可以給勞工更好的條件（更高的工資、更多的休假），但不能給更差的。任何低於勞基法標準的勞動條件，即使勞工「同意」了，依法也是無效的（勞基法§1第2項）。

> **生活比喻：** 想像勞基法是一棟大樓的地基。你可以在上面蓋更高更好的樓層（更好的勞動條件），但你不能把地基挖低。地基挖低了，整棟大樓就會塌。

---

## 1.1 適用範圍：誰受勞基法保護？

### 1.1.1 適用行業

勞基法並不是所有行業都適用的。勞基法§3規定，本法適用於一切勞雇關係，但因經營型態、管理制度及工作特性等因素適用本法確有窒礙難行者，並經中央主管機關指定公告之行業或工作者，不適用之。

目前的實務是：**絕大多數行業都適用勞基法**。不適用的例外非常少，主要包括：
- 部分家事勞工（家庭幫傭、看護等，但已有專法草案討論中）
- 部分自營作業者

### 1.1.2 勞工的定義

**勞基法§2第1款：** 勞工，指受雇主僱用從事工作獲致工資者。

白話翻譯：如果你幫別人工作、聽別人指揮、領別人給你的薪水，你就是勞工。

**三個要素：**
1. **受僱用** — 你和雇主之間有從屬關係
2. **從事工作** — 你有實際提供勞務
3. **獲致工資** — 你因此得到報酬

#### 從屬性的判斷（超級重要！）

判斷一個人是不是「勞工」，最關鍵的標準是「從屬性」。這在考試和實務中都是高頻爭點。

從屬性有四個面向：

| 從屬性類型 | 內容 | 白話解釋 |
|------------|------|----------|
| 人格從屬性 | 勞工在雇主指揮監督下提供勞務 | 老闆叫你做什麼你就做什麼，不能自己決定 |
| 經濟從屬性 | 勞工為雇主的事業目的而工作，不是為自己 | 你是幫公司賺錢，不是自己當老闆 |
| 組織從屬性 | 勞工被納入雇主的組織體系中 | 你是公司的一員，不是獨立的外部合作者 |
| 技術從屬性 | 勞工的工作方式受雇主指示 | 老闆不只告訴你做什麼，還告訴你怎麼做 |

> **考試陷阱：** 公司把員工稱為「承攬人」、「合作夥伴」、「獨立接案者」，簽的合約名字叫「承攬契約」——但如果實質上有從屬性，法院仍然會認定為勞動契約。**名稱不重要，實質關係才重要。** 這就是所謂的「契約定性」問題。

**大法官釋字第740號：** 保險業務員與保險公司間的關係，應依實際勞務提供的狀態判斷，不以契約名稱為準。

**實務案例：外送員是勞工嗎？**

近年來最熱門的爭議之一。Uber Eats、foodpanda 的外送員到底是勞工還是承攬人？

- 如果外送員可以自由決定上線時間、可以拒絕接單、自備交通工具、不受公司出勤管理 → 傾向承攬
- 如果平台透過演算法實質控制派單、要求穿制服、限制價格、有懲罰機制 → 傾向勞動契約

這個問題目前在台灣仍有爭議，實務判決不一。但考試時的分析框架就是「從屬性四要素」。

### 1.1.3 雇主的定義

**勞基法§2第2款：** 雇主，指僱用勞工之事業主、事業經營之負責人或代表事業主處理有關勞工事務之人。

白話翻譯：雇主不只是「老闆」，連代替老闆處理勞工事務的人（例如人資主管、廠長）也算廣義的雇主。

**為什麼這個定義重要？** 因為勞基法課予「雇主」很多義務（付工資、給休假、不能亂開除人）。雇主的定義越廣，能被追究責任的人就越多，對勞工保護就越周全。

## 1.2 勞動契約

### 1.2.1 什麼是勞動契約？

**勞基法§2第6款：** 勞動契約，指約定勞雇關係而具有從屬性之契約。

白話翻譯：你和老闆之間「你幹活、我付錢」的約定，只要具有從屬性，就是勞動契約。

**勞動契約 vs 承攬契約 vs 委任契約：**

| 比較項目 | 勞動契約 | 承攬契約 | 委任契約 |
|----------|----------|----------|----------|
| 核心特徵 | 有從屬性 | 完成特定工作 | 處理一定事務 |
| 報酬依據 | 持續提供勞務 | 完成成果 | 處理事務本身 |
| 指揮監督 | 受雇主指揮 | 獨立完成 | 有一定獨立性 |
| 勞基法保護 | 有 | 無 | 無 |
| 典型例子 | 公司職員 | 水電師傅修繕 | 律師受任處理訴訟 |

> **考試重點：** 勞動契約與承攬、委任的區別，是每年必考的高頻考點。分析框架就是「從屬性」。

### 1.2.2 定期契約 vs 不定期契約

勞基法把勞動契約分成兩種：

**定期契約（勞基法§9）：**
- 只有「臨時性、短期性、季節性、特定性」的工作才能簽定期契約
- 臨時性工作：無法預期的非繼續性工作，期間在6個月以內
- 短期性工作：可預期在6個月內完成的非繼續性工作
- 季節性工作：受季節性原料、材料來源或市場銷售影響的非繼續性工作，期間在9個月以內
- 特定性工作：可在特定期間完成的非繼續性工作（超過1年需報主管機關核備）

**不定期契約：**
- 有繼續性工作的，就應該簽不定期契約
- 這是常態、是原則

**為什麼這個區別重要？**

定期契約到期就結束，雇主不需要付資遣費。不定期契約則需要有法定事由才能終止，而且通常要付資遣費。

所以很多不肖雇主會「故意」把應該是不定期的工作包裝成定期契約，好讓自己可以隨時不續約。**勞基法對此有防堵機制：**

> 勞基法§9第2項：定期契約屆滿後，如果勞工繼續工作而雇主不表示反對，或者雖訂新約但前後契約的工作期間超過90天、中斷期間不超過30天，視為不定期契約。

白話翻譯：如果你一直續簽短期契約做同樣的工作，法律會把你當成不定期契約的員工來保護。

## 1.3 工資

### 1.3.1 工資的定義與認定

**勞基法§2第3款：** 工資，指勞工因工作而獲得之報酬，包括工資、薪金及按計時、計日、計月、計件以現金或實物等方式給付之獎金、津貼及其他任何名義之經常性給與均屬之。

白話翻譯：只要是你「因為工作」而「經常性」拿到的錢，不管名目叫什麼，都是工資。

**判斷標準：兩個要素**
1. **勞務對價性** — 這筆錢是因為你提供勞務才拿到的
2. **經常性給與** — 這筆錢是固定、經常性會發的，不是偶爾的恩惠

**常見爭議：什麼算工資？什麼不算？**

| 項目 | 是否為工資 | 理由 |
|------|------------|------|
| 底薪 | 是 | 勞務對價，經常性給與 |
| 伙食津貼 | 通常是 | 經常性給與，且實務多認定為工資 |
| 交通津貼 | 通常是 | 如果是每月固定發放的，屬經常性給與 |
| 全勤獎金 | 是 | 與出勤有對價關係，經常性給與 |
| 業績獎金（固定發放） | 是 | 即使名為「獎金」，但如果是每月固定依業績計算發放 |
| 年終獎金 | 通常不是 | 恩惠性、非經常性（但如果勞動契約約定固定發放，則可能是） |
| 三節獎金 | 通常不是 | 恩惠性質 |
| 差旅費實報實銷 | 不是 | 不是勞務對價，是費用補償 |

> **考試重點：** 工資認定直接影響加班費計算、資遣費計算、退休金計算、勞保投保薪資。雇主有動機把工資「拆成」底薪加各種「非工資」名目，以降低加班費等成本。勞工如果打官司，第一步往往就是爭執「這些項目到底算不算工資」。

### 1.3.2 基本工資

**勞基法§21：** 工資由勞雇雙方議定之，但不得低於基本工資。

基本工資是由基本工資審議委員會審議後，由行政院核定。基本工資分為：
- **月薪基本工資**：適用於按月計酬的全時工作者
- **時薪基本工資**：適用於部分工時工作者

> **重要觀念：** 基本工資不只是「底薪」，而是勞工每月正常工作時間所得的「工資總額」不能低於基本工資。所以如果你的底薪只有18,000元，但加上每月固定的伙食津貼、交通津貼後超過基本工資，仍然合法。

### 1.3.3 加班費計算（超重要！）

這是考試和實務中最常出現的爭議之一。

**正常工時：** 勞基法§30規定，勞工正常工作時間，每日不得超過8小時，每週不得超過40小時。

**延長工時（加班）的工資計算（勞基法§24）：**

| 加班時段 | 加班費計算 | 白話解釋 |
|----------|------------|----------|
| 平日延長工時前2小時 | 按平日每小時工資額加給1/3以上 | 時薪 × 1.34 |
| 平日延長工時第3-4小時 | 按平日每小時工資額加給2/3以上 | 時薪 × 1.67 |
| 休息日前2小時 | 按平日每小時工資額加給1又1/3以上 | 時薪 × 2.34 |
| 休息日第3-8小時 | 按平日每小時工資額加給1又2/3以上 | 時薪 × 2.67 |
| 休息日第9-12小時 | 同上 | 時薪 × 2.67 |

> **計算範例：** 假設小明月薪30,000元，每月工作天數約30天，每天8小時。
> 平日每小時工資 = 30,000 ÷ 30 ÷ 8 = 125元
> （注意：實務上依勞動部函釋，月薪制勞工「平日每小時工資額」= 月工資 ÷ 30 ÷ 8）
>
> 如果小明在平日加班3小時：
> - 前2小時：125 × (1 + 1/3) × 2 = 333元
> - 第3小時：125 × (1 + 2/3) × 1 = 208元
> - 合計加班費：541元

### 1.3.4 工資給付原則

勞基法§22-§26規定了幾個重要原則：

1. **全額直接給付原則**（§22）：工資應全額直接給付勞工。除法令另有規定或勞雇雙方另有約定外，不得扣除任何款項。
2. **定期給付原則**（§23）：工資的給付，除當事人有特別約定或按月預付者外，每月至少定期發給二次。
3. **工資不得預扣原則**（§26）：雇主不得預扣勞工工資作為違約金或賠償費用。

> **實務案例：** 公司規定「離職需提前一個月告知，否則扣一個月薪水」——這就違反了工資不得預扣原則。即使勞工簽了同意書，這個約定也是無效的。

## 1.4 工時制度

### 1.4.1 正常工時

**勞基法§30：**
- 每日正常工作時間不得超過 **8小時**
- 每週正常工作時間不得超過 **40小時**

但可以經工會同意（無工會者經勞資會議同意），做以下彈性調整：
- **兩週彈性工時**（§30第2項）：兩週內的工作總時數不超過80小時，但每日不超過10小時
- **八週彈性工時**（§30第3項）：八週內的工作總時數不超過320小時
- **四週彈性工時**（§30之1）：四週內正常工時分配，每日不超過10小時

### 1.4.2 延長工時（加班）

**勞基法§32：**
- 雇主有使勞工在正常工作時間以外工作之必要者，經工會同意（無工會者經勞資會議同意），得延長工作時間
- 每日正常工時加延長工時合計不得超過 **12小時**
- 每月延長工時總時數不得超過 **46小時**（經勞資會議同意可延長至54小時，每3個月不超過138小時）

### 1.4.3 責任制的迷思

很多雇主說「你是責任制，沒有加班費」。這在法律上是什麼意思？

**勞基法§84之1** 規定，經中央主管機關核定公告的「特定工作者」，可以由勞雇雙方另行約定工時、例假、休假等，不受一般規定限制。但有嚴格條件：

1. 必須是經中央主管機關「核定公告」的工作者（例如：保全人員、航空機師、醫院住院醫師等）
2. 勞雇雙方必須「書面約定」
3. 約定內容必須報請當地主管機關「核備」
4. 約定的工時仍不得損害勞工的健康與福祉

> **考試重點：** 絕大多數的上班族都不是勞基法§84之1所核定的工作者。老闆說你是「責任制」不算數，要看你的職位是否真的在主管機關的公告名單上。即使是§84之1的工作者，也不是「無上限加班」——仍然有合理的工時限制。

### 1.4.4 一例一休制度（106年修法重點）

這是台灣勞動法近年來最重大的修法之一，也是考試高頻考點。

**基本架構：**

每七日中應有二日之休息，其中一日為「例假」，一日為「休息日」。

| | 例假 | 休息日 |
|--|------|--------|
| 性質 | 強制休息，不可加班（原則） | 可加班，但加班費加倍 |
| 加班條件 | 只有天災、事變或突發事件才可以 | 雇主有需要、勞工同意即可 |
| 加班費 | 加倍發給 | 依§24第2、3項加成計算 |
| 雇主違法代價 | 違反可處罰鍰 | 違反可處罰鍰 |

**例假 vs 休息日的關鍵區別：**

例假是「原則上絕對不能加班」的日子——除非遇到天災、事變或突發事件。這是為了保障勞工最低限度的休息權。

休息日是「可以加班但代價比較高」的日子——雇主必須付更高的加班費。

> **生活比喻：** 把一週想像成你的時間帳戶。週一到週五是「正常提領」（正常工作），休息日是「提領要付手續費」（可以加班但加班費更高），例假是「銀行關門不能提領」（原則上不能加班）。

### 1.4.5 休假制度

**國定假日（勞基法§37）：**
內政部所定應放假之紀念日、節日、勞動節及其他中央主管機關指定應放假日，均應休假。目前一年有12天國定假日。

**特別休假（勞基法§38）：**

| 年資 | 特休天數 |
|------|----------|
| 6個月以上未滿1年 | 3日 |
| 1年以上未滿2年 | 7日 |
| 2年以上未滿3年 | 10日 |
| 3年以上未滿5年 | 每年14日 |
| 5年以上未滿10年 | 每年15日 |
| 10年以上 | 每1年加給1日，最多30日 |

**重要原則：**
- 特休由勞工自行排定，雇主不能強制
- 年度終結或契約終止時，未休完的特休應折算工資
- 雇主應告知勞工特休天數

## 1.5 解僱制度（考試最高頻考點之一）

### 1.5.1 解僱的基本架構

勞基法把雇主終止勞動契約分成兩大類：

| | 經濟性解僱（§11） | 懲戒解僱（§12） |
|--|-------------------|-----------------|
| 性質 | 非勞工的錯，是經濟因素或不可歸責勞工的事由 | 勞工有重大過錯 |
| 預告 | 需要預告期間 | 不需要預告 |
| 資遣費 | 需要付資遣費 | 不需要付資遣費 |
| 事由 | 歇業、轉讓、虧損、業務緊縮、不可抗力、勞工不能勝任 | 詐欺、暴行、判刑、違約情節重大、洩密、無正當理由曠工等 |

### 1.5.2 經濟性解僱事由（勞基法§11）

雇主非有下列情事之一者，不得預告勞工終止勞動契約：

1. **歇業或轉讓時**
2. **虧損或業務緊縮時**
3. **不可抗力暫停工作在一個月以上時**
4. **業務性質變更，有減少勞工之必要，又無適當工作可供安置時**
5. **勞工對於所擔任之工作確不能勝任時**

逐一分析：

**「虧損或業務緊縮」：**
- 虧損：公司財務上確實有虧損，不能只是「預期可能虧損」
- 業務緊縮：業務量確實減少、部門裁撤等
- 實務上法院會審查雇主提出的財務資料，確認是否真的虧損

**「勞工不能勝任工作」（最常考！）：**
- 客觀上不能勝任：能力確實不足，無法完成工作
- 主觀上不能勝任：能做但不願做、態度消極、屢次違規
- 實務見解（最高法院96年度台上字第2630號判決）：包括客觀上的能力不足和主觀上「能為而不為」、「可以做而無意願做」

> **考試陷阱：** 「不能勝任」不只是「能力不足」，也包括「態度問題」。但雇主不能只因為一次表現不好就解僱——必須給勞工改善的機會。

### 1.5.3 懲戒解僱事由（勞基法§12）

勞工有下列情形之一者，雇主得不經預告終止契約：

1. 於訂立勞動契約時為虛偽意思表示，使雇主誤信而有受損害之虞者
2. 對於雇主、雇主家屬、雇主代理人或其他共同工作之勞工，實施暴行或有重大侮辱之行為者
3. 受有期徒刑以上刑之宣告確定，而未諭知緩刑或未准易科罰金者
4. 違反勞動契約或工作規則，**情節重大**者
5. 故意損耗機器、工具、原料、產品，或其他雇主所有物品，或故意洩漏雇主技術上、營業上之秘密，致雇主受有損害者
6. 無正當理由繼續曠工三日，或一個月內曠工達六日者

**「違反工作規則情節重大」的判斷（高頻考點）：**

什麼叫做「情節重大」？法院通常會考量：
- 違規的具體內容
- 違規造成的損害程度
- 勞工過去的工作表現
- 雇主是否已經給過警告或處分
- 違規行為是否有反覆性
- 考量比例原則——解僱是否為「最後手段」

> **除斥期間：** 勞基法§12第2項規定，雇主依§12終止契約者，應自知悉其情形之日起，30日內為之。超過30天就不能再以同一事由解僱勞工了。

### 1.5.4 解僱最後手段性原則（核心原則！）

**這是勞動法最重要的原則之一，幾乎每年必考。**

解僱最後手段性原則（ultima ratio原則）的意思是：解僱是終止勞動關係最嚴厲的手段，雇主在解僱之前，必須先嘗試其他較溫和的替代措施。只有在其他手段都不可行或都已嘗試過的情況下，才能走到解僱這一步。

**適用於§11第5款「不能勝任」的情形：**

雇主在解僱「不能勝任」的勞工之前，應該先：
1. 給予教育訓練或職務調整的機會
2. 告知勞工表現不佳之處，要求改善
3. 設定合理的改善期限
4. 考慮是否有其他適當的職位可以安置

如果雇主什麼都沒做就直接解僱，法院很可能認定解僱違法。

**適用於§12第4款「違規情節重大」的情形：**

雇主在以「違規情節重大」解僱勞工之前，應該考慮：
1. 是否可以用較輕的處分（警告、記過、減薪、調職）替代
2. 該違規行為是否真的嚴重到必須解僱
3. 解僱是否符合比例原則

> **經典判決：** 最高法院104年度台上字第129號判決指出，雇主解僱勞工，應符合最後手段性原則，即雇主於其使用各種手段後，仍無法改善之情況下，始得終止勞動契約。

**考試如何使用這個原則：**

幾乎所有的解僱題目，答題時都要討論「解僱最後手段性原則」。步驟如下：
1. 先判斷雇主主張的解僱事由是§11還是§12
2. 檢查該事由在事實上是否成立
3. 分析雇主是否已經窮盡其他手段（教育訓練、調職、警告等）
4. 結論：如果雇主沒有窮盡其他手段就直接解僱，解僱可能違法

### 1.5.5 勞工的終止契約權

不只雇主可以解僱勞工，勞工也有權利終止契約：

**勞基法§14：** 有下列情形之一者，勞工得不經預告終止契約：
1. 雇主於訂立勞動契約時為虛偽之意思表示
2. 雇主、其家屬或代理人對勞工實施暴行或重大侮辱
3. 契約所訂之工作，對勞工健康有危害之虞，經通知雇主改善而無效果者
4. 雇主、其代理人或其他勞工患有法定傳染病，有傳染之虞者
5. 雇主不依勞動契約給付工作報酬，或對於按件計酬之勞工不供給充分之工作者
6. 雇主違反勞動契約或勞工法令，致有損害勞工權益之虞者

**重點：** 勞工依§14終止契約時，可以請求資遣費（§14第4項準用§17），這跟自願離職不一樣！

### 1.5.6 預告期間

雇主依§11終止契約時，應提前預告：

| 年資 | 預告期間 |
|------|----------|
| 繼續工作3個月以上未滿1年 | 10日前預告 |
| 繼續工作1年以上未滿3年 | 20日前預告 |
| 繼續工作3年以上 | 30日前預告 |

雇主未依規定期間預告而終止契約者，應給付預告期間的工資。

勞工於接到預告後，為另謀工作得於工作時間請假外出，每星期不得超過二日之工作時間，請假期間工資照給（§16第2項）。這就是所謂的「謀職假」。

### 1.5.7 資遣費

**新制（勞退新制，94年7月1日以後適用）：**

每滿1年發給1/2個月之平均工資，未滿1年者按比例計算。最高以發給6個月平均工資為限。

計算公式：資遣費 = 平均工資 × 年資 × 0.5

**舊制（勞退舊制，94年7月1日以前適用）：**

每滿1年發給1個月之平均工資，剩餘月數或未滿1年者以比例計給，未滿1個月者以1個月計。

計算公式：資遣費 = 平均工資 × 年資

> **計算範例（新制）：** 小華工作了2年6個月，平均工資40,000元。
> 資遣費 = 40,000 × 2.5 × 0.5 = 50,000元

> **什麼是「平均工資」？** 勞基法§2第4款：指計算事由發生之當日前六個月內所得工資總額除以該期間之總日數所得之金額。白話說就是「過去六個月的工資平均」。

## 1.6 退休制度

### 1.6.1 舊制（勞基法§53-§58）

**自請退休（§53）：**
- 工作15年以上且年滿55歲
- 工作25年以上
- 工作10年以上且年滿60歲

**強制退休（§54）：**
- 年滿65歲
- 心神喪失或身體殘廢不堪勝任工作

**退休金計算（§55）：**
- 按工作年資，每滿1年給與2個基數，超過15年的部分每滿1年給與1個基數
- 最高以45個基數為限
- 因職業災害致殘而強制退休者，加給20%
- 基數 = 核准退休時1個月平均工資

**舊制的問題：**
- 退休金由雇主負擔，如果公司倒閉，勞工可能領不到
- 勞工必須在同一雇主處工作到符合條件才能退休，年資不可攜帶
- 實際上很多勞工在符合退休條件之前就被解僱了

### 1.6.2 新制（勞工退休金條例，94年7月1日施行）

為了解決舊制的問題，政府推出了勞退新制：

**核心架構：個人帳戶制**
- 雇主每月提繳不低於勞工月工資6%到勞工的個人退休金帳戶
- 勞工也可以自願提繳（最高6%，可節稅）
- 帳戶跟著勞工走，換工作也不影響
- 滿60歲即可請領

**新制 vs 舊制比較：**

| 比較項目 | 舊制 | 新制 |
|----------|------|------|
| 提撥方式 | 雇主提撥到退休基金 | 雇主提繳到個人帳戶 |
| 年資計算 | 限同一雇主 | 可攜式，跟著勞工走 |
| 請領條件 | 工作年資+年齡 | 年滿60歲 |
| 保障程度 | 公司倒閉可能領不到 | 個人帳戶，不受公司影響 |
| 適用時間 | 94.7.1前到職 | 94.7.1後到職（之前的可選擇） |

## 1.7 職業災害補償（勞基法§59）

### 1.7.1 什麼是職業災害？

職業災害是指勞工因執行職務而發生的傷害、疾病、失能或死亡。

**判斷標準：業務遂行性 + 業務起因性**

| 標準 | 意思 | 白話解釋 |
|------|------|----------|
| 業務遂行性 | 災害發生時，勞工是在雇主指揮監督下工作 | 你是在「上班時間做工作的事」的時候受傷的 |
| 業務起因性 | 災害與工作之間有因果關係 | 你受傷是「因為工作」造成的 |

**常見爭議案例：**

| 情境 | 是否為職災 | 分析 |
|------|------------|------|
| 工廠操作機器受傷 | 是 | 典型職災 |
| 上下班途中車禍 | 通常是 | 通勤災害，認定為職災（但有例外） |
| 出差途中受傷 | 是 | 業務遂行中 |
| 午休外出吃飯被車撞 | 爭議 | 看是否屬於合理的附隨行為 |
| 上班時間心臟病發 | 爭議 | 如果與過勞有關，可能認定為職災 |
| 在公司跟同事打架受傷 | 通常不是 | 非業務起因 |

### 1.7.2 雇主的補償責任（勞基法§59）

勞工因職業災害而致死亡、失能、傷害或疾病時，雇主應依下列規定予以補償：

| 補償項目 | 內容 |
|----------|------|
| **醫療費用補償** | 雇主應補償必需的醫療費用 |
| **工資補償** | 勞工在醫療中不能工作時，雇主應按其原領工資數額予以補償（最長2年） |
| **失能補償** | 經治療終止後，經指定之醫院診斷審定其身體遺存障害者，雇主應按其平均工資及其失能程度一次給予失能補償 |
| **死亡補償** | 勞工遭遇職業傷害或罹患職業病而死亡時，雇主應給予5個月平均工資之喪葬費及40個月平均工資之死亡補償 |

**職災補償的三大特色：**

1. **無過失責任：** 不管雇主有沒有過失，都要補償。即使職災是勞工自己不小心造成的，雇主仍然要補償。

2. **抵充原則（§59但書）：** 如果同一事故已經依勞工保險條例或其他法令取得相關補償或賠償，雇主可以抵充。

3. **不得以勞工有過失而免責：** 勞基法§61規定，勞工對職災補償的受領權，不因勞工離職而受影響，且不得讓與、抵銷、扣押或供擔保。

> **考試重點：** 職災補償（勞基法§59）是無過失的「補償」責任，和民法上的「損害賠償」不同。如果勞工要主張雇主有過失而請求完整的損害賠償，需要另外依民法§184以下主張侵權行為損害賠償。兩者可以並行，但已領取的補償金額可以抵充。

## 1.8 工作規則

### 1.8.1 工作規則的意義

**勞基法§70：** 僱用勞工人數在三十人以上者，應依其事業性質，就工時、休息、休假、工資、津貼及獎金、紀律、考勤、請假、獎懲及升遷、受僱、解僱、資遣、離職及退休、災害傷病補償及撫卹、福利措施等事項訂立工作規則，報請主管機關核備後並公開揭示之。

白話翻譯：員工超過30人的公司，要制定「員工手冊」（工作規則），內容要涵蓋各種勞動條件，而且要送政府備查、向員工公告。

### 1.8.2 工作規則的效力

工作規則的法律性質，學說上有爭議：

| 學說 | 內容 |
|------|------|
| 法規範說 | 工作規則本身就有法的效力，不需要個別勞工同意 |
| 契約說 | 工作規則須經勞工同意才對勞工生效（併入勞動契約） |
| 折衷說 | 合理且已公開揭示的工作規則，視為勞動契約的一部分 |

**實務見解（最高法院）：** 傾向認為工作規則如果經過主管機關核備並公開揭示，且內容不違反法律強制規定，就對勞工有拘束力，不以勞工個別同意為必要。但如果工作規則的變更不利於勞工，則需要有合理性。

### 1.8.3 工作規則不利益變更

雇主可以修改工作規則，但如果修改的內容對勞工不利（例如降低加班費、延長工時、增加處分種類），就涉及「不利益變更」的問題。

**判斷標準：** 法院通常會審查不利益變更是否具有「合理性」，考量因素包括：
- 變更的必要性（公司是否真有經營困難）
- 變更的內容是否適當
- 是否有與勞工或工會協商
- 是否有過渡措施或補償措施
- 同行業其他公司的情況

## 1.9 童工與女工保護

### 1.9.1 童工保護（勞基法§44-§48）

- **童工定義：** 15歲以上未滿16歲之受僱從事工作者
- 未滿15歲之人，原則上不得僱用（例外：國中畢業或經主管機關認定其工作性質及環境無礙身心健康者）
- 童工每日工作不得超過8小時，不得於夜間（晚上8時至翌晨6時）工作
- 不得從事危險性或有害性工作

### 1.9.2 女性勞工保護

**產假（§50）：**
- 分娩前後給予8星期產假
- 妊娠3個月以上流產者，給予4星期產假
- 工作滿6個月者，產假期間工資照給
- 未滿6個月者，減半發給

**生理假：** 依性別工作平等法§14，每月得請1日生理假，全年請假日數未逾3日者不併入病假計算，其餘日數併入病假。

**夜間工作限制（§49）：** 雇主不得使女工於午後10時至翌晨6時之時間內工作，但經勞資會議同意後，且提供安全衛生設施、交通工具或宿舍等，不在此限。

**哺乳時間（§52）：** 子女未滿2歲須女工親自哺（集）乳者，除規定的休息時間外，雇主應每日另給哺（集）乳時間60分鐘。

# 第二章：勞動事件法

## 本章概覽

勞動事件法於108年（2019年）12月6日公布，109年（2020年）1月1日施行。這是一部專門處理勞動訴訟的程序法，目的是讓勞動爭議能更快速、更公正、更便利地在法院獲得解決。

> **為什麼需要勞動事件法？** 在這部法律施行之前，勞工打官司面臨很多困難：訴訟費用高、程序冗長、舉證困難（很多證據都在雇主手上）、即使勝訴也可能被雇主拖延執行。勞動事件法就是為了解決這些問題而生的。

---

## 2.1 管轄

### 2.1.1 專業法庭

**勞動事件法§4：** 為處理勞動事件，各級法院應設立勞動專業法庭（或專股）。法官應遴選具有勞動法相關學識、經驗者任之。

### 2.1.2 管轄法院

**勞動事件法§6：** 勞動事件以勞工為原告的，除了被告住所地法院有管轄權外，勞工的勞務提供地法院也有管轄權。

白話翻譯：勞工可以選擇在自己工作的地方打官司，不用跑到公司總部所在地。這對勞工很友善。

**勞動事件法§7：** 如果勞動契約有約定管轄法院，但該約定對勞工不利的，勞工可以不受拘束，向其他有管轄權的法院起訴。

---

## 2.2 當事人

### 2.2.1 工會的當事人能力

**勞動事件法§40：** 工會得為會員的利益，以自己的名義提起訴訟（團體訴訟）。工會也可以選定會員為選定當事人。

### 2.2.2 勞工的訴訟救助

**勞動事件法§14：** 勞工符合社會救助法規定的低收入戶、中低收入戶，或其他顯無資力者，法院應以裁定准予訴訟救助，不受一般民事訴訟法規定的限制。

**勞動事件法§12：** 因確認僱傭關係或給付工資、退休金或資遣費的勞動事件，勞工免繳裁判費。

---

## 2.3 調解前置

### 2.3.1 強制調解

**勞動事件法§16：** 勞動事件，除法律別有規定外，應於起訴前先經法院行勞動調解程序。

白話翻譯：你不能直接告雇主，要先在法院進行調解。如果調解成功就結案，調解不成才進入正式訴訟。

**為什麼要調解前置？**
- 調解比訴訟快：法律規定調解應於三次期日內終結（§24）
- 調解比訴訟省錢
- 調解委員會的組成包括法官和勞資雙方代表的調解委員，更能兼顧雙方利益

### 2.3.2 調解委員會的組成

**勞動事件法§22：** 勞動調解，由勞動法庭之法官一人及勞動調解委員二人組成勞動調解委員會行之。調解委員由法院遴聘具有勞資事務、法律或其他專業知識、信望素孚之人任之。

### 2.3.3 調解方案

**勞動事件法§27：** 勞動調解委員會在調解不成立時，應依職權斟酌一切情形，求兩造利益之平衡，於不違反法令強制或禁止規定的範圍內，提出解決事件之適當方案。當事人於收到方案後10日內未提出異議者，視為調解成立。

白話翻譯：即使雙方談不攏，調解委員會也會提出一個「建議方案」。如果雙方在10天內沒有反對，就視為同意。

---

## 2.4 勞動訴訟的特色

### 2.4.1 舉證責任調整（超級重要！）

傳統民事訴訟的原則是「誰主張、誰舉證」。但在勞動事件中，很多證據（出勤紀錄、工資資料、人事記錄）都在雇主手上，勞工根本拿不到。因此勞動事件法做了重大調整：

**勞動事件法§35：** 勞工請求之事件，雇主就其依法令應備置之文書，有提出之義務。

白話翻譯：出勤紀錄、薪資明細等雇主依法應該備置的文件，法院可以命雇主提出。如果雇主不提出，法院可以認定勞工的主張為真。

**勞動事件法§36：** 文書、勘驗物或鑑定所需資料之持有人，無正當理由不從法院之命提出者，法院得以裁定處罰鍰，於必要時並得以強制處分。

**勞動事件法§37：** 勞工與雇主間關於工資之爭執，經證明勞工本於勞動關係自雇主所受領之給付，推定為勞工因工作而獲得之報酬。

白話翻譯：如果雇主付了一筆錢給勞工，而雙方對「這筆錢算不算工資」有爭議，法律推定這筆錢就是工資。雇主如果要主張「這不是工資」，雇主要負舉證責任。

**勞動事件法§38：** 出勤紀錄內記載之勞工出勤時間，推定勞工於該時間內經雇主同意而執行職務。

白話翻譯：打卡紀錄上寫你幾點到幾點在公司，就推定你那段時間都在工作（加班）。如果雇主說「你那段時間不是在加班，是在做自己的事」，雇主要負舉證責任。

> **考試重點：** 勞動事件法§37和§38是考試最愛考的條文。它們大幅翻轉了傳統的舉證責任分配，對勞工非常有利。

### 2.4.2 保全處分的特別規定

**定暫時狀態處分（勞動事件法§49）：**

勞工提起確認僱傭關係存在之訴，法院認勞工有勝訴之望，且雇主繼續僱用非顯有重大困難者，得依勞工之聲請，為繼續僱用及給付工資之定暫時狀態處分。

白話翻譯：如果你被開除了，你告雇主說「解僱違法」，法院可以在判決確定之前，先命令雇主繼續僱用你、繼續發薪水給你。這樣你在打官司期間就不會斷炊。

**這個規定的革命性意義：**

在勞動事件法施行之前，即使勞工認為解僱違法而提起訴訟，在判決確定之前，勞工是沒有收入的。一場官司可能打好幾年，勞工可能撐不住就放棄了。勞動事件法§49讓法院可以在訴訟進行中就先恢復勞工的工作和薪資，大大降低了勞工打官司的成本。

### 2.4.3 迅速審理原則

**勞動事件法§32：** 法院處理勞動事件，應以一次期日辯論終結為原則。第一審並應於六個月內審結。

# 第三章：集體勞動法

## 本章概覽

前面談的勞基法、勞動事件法，都是處理「個別勞工」和「雇主」之間的關係。但有些問題不是個別勞工能解決的——工資太低？一個人去跟老闆談，老闆理都不理你。工作環境惡劣？一個人反映，公司說你不滿意可以離開。

**這就是為什麼需要集體勞動法。**

集體勞動法的核心概念是：勞工個別力量太弱，只有團結起來才能和雇主平等對話。

集體勞動法有三根支柱，合稱「勞動三法」：

| 法律 | 功能 | 白話解釋 |
|------|------|----------|
| 工會法 | 團結權 | 勞工有權組織工會 |
| 團體協約法 | 協商權 | 工會有權和雇主進行集體談判 |
| 勞資爭議處理法 | 爭議權 | 當談判破裂時，勞工有權罷工 |

> **生活比喻：** 一根筷子容易折斷，一把筷子很難折斷。工會法讓勞工把筷子綁在一起（組織工會），團體協約法讓這把筷子有用（和雇主談判），勞資爭議處理法讓這把筷子有力量（罷工）。

---

## 3.1 工會法

### 3.1.1 工會的種類

**工會法§6：** 工會組織類型如下：

| 工會類型 | 定義 | 例子 |
|----------|------|------|
| 企業工會 | 結合同一事業單位的勞工組織之 | 台積電企業工會 |
| 產業工會 | 結合相關產業內之勞工組織之 | 台灣護理產業工會 |
| 職業工會 | 結合相關職業技能之勞工組織之 | 台北市水電工職業工會 |

**企業工會的特殊地位：**
- 同一事業單位只能組織一個企業工會（§9）
- 企業工會需要有同一事業單位30人以上的勞工連署發起
- 企業工會在團體協商上具有優先地位

### 3.1.2 工會的設立

**工會法§11：**
- 工會由勞工30人以上發起
- 組成籌備會後，召開成立大會
- 成立大會後30日內，向主管機關登記

### 3.1.3 會員資格

**工會法§12-§14：**
- 勞工有加入工會的自由
- 代表雇主行使管理權的主管人員，不得加入企業工會
- 企業工會如果達到該事業單位勞工二分之一以上，其他同一事業單位的勞工應加入該工會（但工會法§7條但書有排除）

### 3.1.4 不當勞動行為（超級重要！）

**工會法§35：** 雇主或代表雇主行使管理權之人，不得有下列行為：

1. 對於勞工組織工會、加入工會、參加工會活動或擔任工會職務，而拒絕僱用、解僱、降調、減薪或為其他不利之待遇
2. 對於勞工或求職者以不加入工會或擔任工會職務為僱用條件
3. 對於勞工提出團體協商之要求或參與團體協商相關事務，而拒絕僱用、解僱、降調、減薪或為其他不利之待遇
4. 對於勞工參與或支持爭議行為，而解僱、降調、減薪或為其他不利之待遇
5. 不當影響、妨礙或限制工會之成立、組織或活動

白話翻譯：雇主不可以因為你參加工會活動而找你麻煩，也不可以插手干預工會的事務。這就是「不當勞動行為」的禁止。

**不當勞動行為的救濟：裁決機制**

**勞資爭議處理法§39-§51：** 勞工或工會認為雇主有不當勞動行為時，可以向「不當勞動行為裁決委員會」申請裁決。

裁決委員會由具有勞動法學識經驗的委員組成，具有準司法機關的性質。裁決決定如果不服，可以向法院提起行政訴訟。

> **考試重點：** 不當勞動行為的認定，是集體勞動法中最常出考題的部分。常見的考試情境：
> - 員工參與工會活動後被調職 → 是否為不當勞動行為？
> - 公司提供額外福利給非工會會員 → 是否為不當勞動行為？
> - 公司在工會籌組期間解僱發起人 → 是否為不當勞動行為？

### 3.1.5 不當勞動行為的類型化分析

在考試答題中，不當勞動行為可以分為幾種類型：

**一、針對勞工個人的不利益待遇（工會法§35第1項第1、3、4款）：**

構成要件：
1. 雇主對勞工施以不利益待遇（解僱、降調、減薪等）
2. 勞工有參加工會活動、提出團體協商要求、參與爭議行為等正當行為
3. 兩者之間有因果關係

判斷重點：雇主的「真正動機」。雇主可能會以績效不佳、組織調整等理由來包裝，但裁決委員會會綜合各項事證判斷雇主的真正動機是否為打壓工會。

**二、支配介入（工會法§35第1項第5款）：**

雇主不當影響、妨礙或限制工會的成立、組織或活動。例如：
- 雇主在工會選舉時介入，支持特定候選人
- 雇主不提供工會辦公場所或會議室（依團體協約或法令應提供者）
- 雇主監視工會會議
- 雇主散布對工會不利的言論

**三、黃犬契約（工會法§35第1項第2款）：**

以不加入工會或擔任工會職務為僱用條件。這是最明顯的不當勞動行為，在台灣實務上較少見，但仍是考試會出的概念。

## 3.2 團體協約法

### 3.2.1 團體協約的意義

**團體協約法§2：** 本法所稱團體協約，指雇主或有法人資格之雇主團體，與依工會法成立之工會，以約定勞動關係及相關事項為目的所簽訂之書面契約。

白話翻譯：團體協約就是工會代表所有會員和雇主談判後簽訂的「集體勞動契約」。它的效力不只針對參與談判的人，而是及於全體會員。

### 3.2.2 誠信協商義務

**團體協約法§6第1項：** 勞資雙方應本誠實信用原則，進行團體協約之協商；對於他方所提團體協約之協商，無正當理由者，不得拒絕。

這是「誠信協商義務」，也叫做「誠實協商義務」。它不是要求雇主一定要同意工會的要求，而是要求雇主必須真誠地、有誠意地坐下來談判。

**什麼算違反誠信協商義務？**

**團體協約法§6第2項：** 下列情形，屬於違反誠信協商義務：
1. 對於他方提出合理適當之協商內容、時間、地點及進行方式，拒絕進行協商者
2. 未於60日內針對協商書面通知提出對應方案，並進行協商者
3. 拒絕提供進行協商所必要之資料者

> **重要：** 違反誠信協商義務也構成不當勞動行為（團體協約法§6第3項準用工會法§35第2項），可以申請裁決。

### 3.2.3 團體協約的效力

**規範效力（團體協約法§17）：**

團體協約得約定勞動條件的最低標準。個別勞動契約的約定如果低於團體協約所定標準，其低於部分無效，無效部分以團體協約所定標準代之。

白話翻譯：團體協約規定「加班費至少1.5倍」，如果個別員工的契約只約定1.34倍，那個約定無效，自動變成1.5倍。

**債法效力：**

團體協約也可以約定工會和雇主之間的權利義務（例如：雇主提供工會辦公室、工會不在特定期間罷工等），這些約定對雙方有拘束力。

**一般拘束力（團體協約法§18）：**

團體協約簽訂後，如果適用該團體協約的勞工已達到事業單位同類勞工人數二分之一以上時，其他同類未參加工會的勞工，也適用該團體協約。

白話翻譯：如果一個公司一半以上的勞工都受到團體協約保護，那麼即使沒加入工會的勞工也一起受到保護。這是為了避免雇主只對工會會員施加壓力。

---

## 3.3 勞資爭議處理法

### 3.3.1 勞資爭議的類型

**勞資爭議處理法§5：**

| 爭議類型 | 定義 | 例子 |
|----------|------|------|
| 權利事項之勞資爭議 | 勞資雙方基於法令、團體協約、勞動契約等的權利義務爭議 | 雇主積欠工資、違法解僱 |
| 調整事項之勞資爭議 | 勞資雙方關於勞動條件的維持或變更的爭議 | 要求調薪、要求增加休假 |

**為什麼區分很重要？**

- 權利事項爭議：可以調解，也可以直接訴訟
- 調整事項爭議：只能調解或仲裁，不能訴訟（因為法院不能代替勞資雙方決定勞動條件應該是什麼）
- 罷工只能針對調整事項，不能針對權利事項（§53第1項）

### 3.3.2 調解

**勞資爭議處理法§9-§24：**

調解是最常用的爭議處理方式。分為：
- **獨任調解人調解**：由一位調解人主持
- **調解委員會調解**：由委員會主持（主管機關代表1人、勞方代表1-3人、資方代表1-3人）

調解方案經雙方同意後，與法院確定判決具有同一效力。

### 3.3.3 仲裁

**勞資爭議處理法§25-§37：**

仲裁分為：
- **自願仲裁**：勞資雙方同意交付仲裁
- **強制仲裁**：主管機關依職權交付仲裁（限於影響大眾生命安全、國家安全或重大公共利益的事業）

仲裁判斷與法院確定判決具有同一效力。

### 3.3.4 爭議行為與罷工（高頻考點！）

**什麼是爭議行為？**

**勞資爭議處理法§5第4款：** 爭議行為，指勞資爭議當事人為達成其主張，所為之罷工或其他阻礙事業正常運作及與之對抗之行為。

**罷工的合法要件（§53、§54）：**

勞工要合法罷工，必須滿足以下條件：

1. **主體要件：** 必須由工會發動。個別勞工不能發動罷工。
2. **事由要件：** 只能針對「調整事項」的爭議，不能針對「權利事項」。
3. **程序要件：**
   - 必須先經過調解不成立（調解前置）
   - 經工會會員以直接、無記名投票且過半數同意（§54第1項）
4. **禁止事項：**
   - 教師不得罷工（§54第2項）
   - 國防部及其所屬機關（構）、學校的勞工不得罷工
   - 影響大眾生命安全、國家安全或重大公共利益的事業，主管機關可以交付強制仲裁以阻止罷工

**罷工期間的法律效果：**

- 罷工期間，勞工不提供勞務，雇主不付工資
- 雇主不得以罷工為理由解僱參與罷工的勞工
- 雇主不得僱用替代性勞工（§55第2項）：雇主不得以非正當理由僱用外部勞工替代罷工中的勞工，但可以調度內部非罷工的勞工

**罷工中的權利行使限制：**

**勞資爭議處理法§55第3項：** 工會及其會員所為之爭議行為，不得有下列情形：
1. 對於設備、原料為破壞或侵害行為
2. 妨礙他人正當權利之行使

> **考試重點：** 罷工的合法性分析是歷年考試的熱門題型。答題時要逐一檢視：主體（是否由工會發動？）→ 事由（是否為調整事項？）→ 程序（是否經過調解？是否經投票通過？）→ 方式（有無破壞行為？）

### 3.3.5 罷工與民刑事責任

**勞資爭議處理法§55第1項：** 爭議行為以罷工、怠工或其他影響事業正常運作的行為為限，不得侵害他人之生命、身體、自由或財產。

**民事免責（§55第2項的反面解釋）：** 合法罷工所造成的雇主營業損失，勞工不需要賠償。但如果罷工違法（例如暴力行為、破壞設備），勞工和工會可能要負民事賠償責任。

**刑事免責的限制：** 合法罷工本身不構成犯罪，但如果罷工過程中有暴力、脅迫、毀損等行為，仍可能構成刑事犯罪。

# 第四章：勞工保險條例

## 本章概覽

勞工保險（簡稱「勞保」）是台灣最重要的社會保險制度之一，保障勞工在遭遇生育、傷病、失能、老年、死亡等風險時，能夠獲得經濟上的保障。

> **白話解釋：** 勞保就像一個「勞工專屬的保險公司」——你每個月繳保費（勞工自己負擔20%，雇主負擔70%，政府負擔10%），當你發生特定的風險事件時，就可以領保險金。

---

## 4.1 加保對象

### 4.1.1 強制加保

**勞工保險條例§6：** 年滿15歲以上，65歲以下的下列勞工，應以其雇主或所屬團體或所屬機構為投保單位，全部參加勞工保險為被保險人：

1. 受僱於僱用五人以上公司、行號之員工
2. 受僱於依法不須登記且僱用五人以上之事業之員工
3. 公、民營工廠、礦場或鹽場、農場、牧場、林場、茶場之產業勞工及其事業主
4. 交通公用事業之員工
5. 公司行號或新聞、文化、公益、合作事業之員工
6. 依法不得參加公教人員保險之政府機關及公、私立學校之員工
7. 無一定雇主或自營作業而參加職業工會者
8. 無一定雇主或自營作業而參加漁會之甲類會員

### 4.1.2 自願加保

**勞工保險條例§6第2項、§8：** 不在強制加保範圍的勞工（例如受僱於未滿五人公司的員工），也可以自願參加勞保。

---

## 4.2 保險給付種類

### 4.2.1 生育給付

**勞工保險條例§31-§32：**
- 被保險人或其配偶分娩或早產者，可以請領生育給付
- 給付標準：按被保險人分娩或早產「當月」起，前6個月的平均月投保薪資，一次給與**60日**生育給付

### 4.2.2 傷病給付

**勞工保險條例§33-§37：**

| 類型 | 等待期 | 給付標準 | 給付期限 |
|------|--------|----------|----------|
| 普通傷病 | 住院第4日起 | 平均月投保薪資的50% | 最長6個月（加保滿1年者最長1年） |
| 職業傷病 | 不能工作第4日起 | 平均月投保薪資的70% | 最長2年 |

### 4.2.3 失能給付

**勞工保險條例§53-§57：**

依失能等級表，按失能程度給付：
- 完全失能：按月投保薪資一定比例按月發給（年金）
- 部分失能：一次給付或按月發給

### 4.2.4 老年給付（最重要！）

**勞工保險條例§58-§58之2：**

老年給付是勞保最核心的長期給付，分為：

**一、老年年金（按月領）：**
- 請領條件：年滿法定請領年齡（目前逐年提高，最終為65歲），保險年資合計滿15年
- 給付標準（擇優計算）：
  - 方式一：平均月投保薪資 × 年資 × 0.775% + 3,000元
  - 方式二：平均月投保薪資 × 年資 × 1.55%

**二、老年一次金（一次領）：**
- 請領條件：年滿法定請領年齡，保險年資未滿15年
- 給付標準：平均月投保薪資 × 年資（每滿1年按1個月計）

**三、減給老年年金 / 展延老年年金：**
- 提前5年請領：每提前1年減給4%（最多減20%）
- 延後請領：每延後1年增給4%（最多增20%）

### 4.2.5 死亡給付

**勞工保險條例§62-§65之1：**

| 給付項目 | 內容 |
|----------|------|
| 本人死亡喪葬津貼 | 按其死亡當月起前6個月平均月投保薪資，發給5個月 |
| 遺屬年金 | 按被保險人之月投保薪資 × 年資 × 1.55%，最低3,000元 |
| 遺屬津貼（一次金） | 依保險年資不同，發給10-30個月 |

---

## 4.3 職業災害保險

### 4.3.1 勞工職業災害保險及保護法

110年（2021年）制定公布，111年（2022年）5月1日施行的「勞工職業災害保險及保護法」（簡稱「災保法」），將職業災害保險從勞保中獨立出來，成為單獨的社會保險。

**主要特色：**

1. **擴大納保範圍：** 受僱於登記有案事業單位之勞工，不論事業單位僱用人數，全部強制加保
2. **提高給付水準：** 職災傷病給付前2個月按投保薪資100%發給（第3個月起70%）
3. **單獨立法：** 不再依附於勞保條例，有獨立的保險費率和給付體系
4. **加強保護：** 增設職業災害預防和職業重建專章

### 4.3.2 職災保險的給付

| 給付種類 | 內容要點 |
|----------|----------|
| 醫療給付 | 職災勞工就醫免繳健保部分負擔 |
| 傷病給付 | 不能工作第4日起，前2個月100%，第3個月起70%，最長2年 |
| 失能給付 | 依失能等級，年金或一次金 |
| 死亡給付 | 遺屬年金 + 喪葬津貼（10個月） |
| 失蹤給付 | 按月發給70%投保薪資 |

# 第五章：就業保險法

## 本章概覽

就業保險和勞保不一樣——勞保保障你在「工作期間」的風險（傷病、老年、死亡），就業保險保障你在「失業期間」的經濟安全。

> **白話解釋：** 就業保險就是「失業保險」。你在有工作的時候每個月繳一點保費，等到你失業了，政府會發給你一筆錢（失業給付），讓你在找到新工作之前不至於沒飯吃。

---

## 5.1 適用對象

**就業保險法§5：** 年滿15歲以上，65歲以下，受僱之本國籍勞工，應以其雇主或所屬機構為投保單位，參加本保險為被保險人。

**不適用的人：**
- 依法應參加公教人員保險或軍人保險者
- 已領取勞保老年給付或公教人員保險養老給付者
- 受僱於依法免辦登記且僱用員工未滿五人之事業

---

## 5.2 失業給付

### 5.2.1 請領條件

**就業保險法§11第1項第1款：**
1. 被保險人非自願離職（被裁員、被資遣）
2. 辦理退保當日前3年內，保險年資合計滿1年以上
3. 具有工作能力及繼續工作意願
4. 向公立就業服務機構辦理求職登記，自求職登記之日起14日內仍無法推介就業或安排職業訓練

### 5.2.2 給付標準

**就業保險法§16：**
- 按被保險人離職辦理退保當月起前6個月平均月投保薪資的**60%**發給
- 最長發給6個月
- 被保險人離職退保時年滿45歲或領有身心障礙證明者，最長發給9個月

### 5.2.3 「非自願離職」的認定

**就業保險法§11第3項：** 本法所稱非自願離職，指被保險人因投保單位關廠、遷廠、休業、解散、破產宣告離職；或因勞基法§11、§13但書、§14及§20規定各款情事之一離職。

白話翻譯：不是你自己想走的，是被逼走的或被開除的。如果你自己辭職，就不能領失業給付。

> **考試陷阱：** 勞工依勞基法§14終止契約（雇主有過錯），雖然形式上是「勞工主動終止」，但法律上視為「非自願離職」，可以領失業給付。

---

## 5.3 育嬰留職停薪津貼

### 5.3.1 請領條件

**就業保險法§11第1項第4款、§19之2：**
1. 被保險人保險年資合計滿1年以上
2. 子女滿3歲前
3. 依性別工作平等法規定辦理育嬰留職停薪

### 5.3.2 給付標準

- 按被保險人育嬰留職停薪當月起前6個月平均月投保薪資的**80%**發給（原為60%，後加計20%補助）
- 每一子女合計最長發給6個月
- 同時撫育子女2人以上者，以發給1人為限

> **近年修法重點：** 為鼓勵父母共同育兒，政府自110年7月起加發育嬰留職停薪薪資補助，使實質給付率從60%提升到80%。

---

## 5.4 其他就業保險給付

| 給付種類 | 內容 |
|----------|------|
| 提早就業獎助津貼 | 符合失業給付請領條件的人，如果在失業給付請領期限屆滿前再就業，可以領取尚未請領的失業給付金額的50%，一次發給 |
| 職業訓練生活津貼 | 被保險人非自願離職後，經公立就業服務機構安排參加職業訓練期間，按離職前6個月平均月投保薪資60%發給，最長6個月 |
| 補助全民健保保費 | 失業期間，被保險人及其眷屬的全民健保費由就業保險基金補助 |

# 第六章：社會法基礎

## 本章概覽

勞動社會法考試不只考「勞動法」，還會考「社會法」。社會法是什麼？簡單說，就是國家為了保障人民的基本生活需求而建立的各種制度——社會保險、社會救助、社會福利等。

> **為什麼要有社會法？** 因為人生充滿風險——你可能生病、失業、老了沒有收入、遭遇意外。如果完全靠個人和家庭自己承擔這些風險，很多人會活不下去。社會法就是國家透過集體的力量，幫助個人分擔這些風險。

---

## 6.1 社會安全的概念

### 6.1.1 社會安全制度的體系

台灣的社會安全制度可以分為三大支柱：

| 支柱 | 制度 | 特色 | 例子 |
|------|------|------|------|
| 第一支柱 | 社會保險 | 強制參加、繳保費、保險事故發生時給付 | 勞保、健保、國民年金 |
| 第二支柱 | 社會救助 | 針對貧困者、不需繳保費、由國家稅收支應 | 低收入戶補助、中低收入戶補助 |
| 第三支柱 | 社會福利/社會促進 | 針對特定群體提供服務或補助 | 老人福利、身心障礙福利、兒少福利 |

### 6.1.2 社會保險的基本原理

社會保險和商業保險不同：

| 比較項目 | 社會保險 | 商業保險 |
|----------|----------|----------|
| 參加方式 | 強制參加 | 自願參加 |
| 保費計算 | 依所得能力（量能原則） | 依風險程度 |
| 給付標準 | 法定、統一 | 依契約約定 |
| 目的 | 社會互助、風險分擔 | 個人風險管理、營利 |
| 財務原則 | 隨收隨付或部分準備 | 完全提存準備 |
| 盈虧 | 政府負最後擔保責任 | 保險公司自負盈虧 |

### 6.1.3 社會保險的基本原則

1. **強制加保原則：** 符合條件者必須參加，不能自行選擇退出
2. **量能負擔原則：** 保費依被保險人的經濟能力（薪資水準）計算，而非依風險高低
3. **社會互助原則：** 健康的人幫助生病的人，年輕人幫助老年人
4. **保險給付法定原則：** 給付的種類和標準由法律規定，不是由契約約定
5. **政府擔保原則：** 社會保險如果有財務缺口，由政府負擔最後責任

---

## 6.2 社會保險 vs 社會救助 vs 社會福利

### 6.2.1 三者的比較

| 比較項目 | 社會保險 | 社會救助 | 社會福利 |
|----------|----------|----------|----------|
| 對象 | 全體國民或特定身分者 | 經濟弱勢者 | 特定群體 |
| 財源 | 保險費 | 政府稅收 | 政府稅收+民間捐助 |
| 給付條件 | 發生保險事故 | 經資產調查認定為低收入 | 符合法定條件 |
| 給付方式 | 現金給付或醫療服務 | 現金補助或實物給付 | 服務提供或補助 |
| 權利性質 | 繳費即有給付請求權 | 需經審查認定 | 依法定條件 |
| 代表法律 | 勞保條例、健保法、國民年金法 | 社會救助法 | 老人福利法、身心障礙者權益保障法 |

### 6.2.2 社會救助法的重點

**社會救助法§4：** 本法所稱低收入戶，指經申請戶籍所在地直轄市、縣（市）主管機關審核認定，符合家庭總收入平均分配全家人口，每人每月在最低生活費以下，且家庭財產未超過中央、直轄市主管機關公告之當年度一定金額者。

**最低生活費：** 由中央、直轄市主管機關參照中央主計機關所公布當地區最近一年每人可支配所得中位數的60%定之。

**社會救助的類型：**
1. **生活扶助：** 現金給付，維持基本生活
2. **醫療補助：** 補助健保部分負擔及其他醫療費用
3. **急難救助：** 臨時性的緊急救助
4. **災害救助：** 天災人禍的救助

---

## 6.3 全民健康保險

### 6.3.1 基本架構

全民健康保險（全民健保）是台灣最重要的社會保險制度之一，保障全體國民就醫的權利。

**全民健康保險法§1：** 為增進全體國民健康，辦理全民健康保險，以提供醫療服務。

**核心原則：**
1. **全民納保：** 具有中華民國國籍者，應參加全民健保（§8）
2. **強制加保：** 不能選擇不參加
3. **量能負擔：** 保費依所得能力計算
4. **就醫平等：** 不分貧富，享有同等的醫療服務

### 6.3.2 保險對象分類

全民健保將被保險人分為六類：

| 類別 | 對象 | 投保單位 |
|------|------|----------|
| 第一類 | 公務人員、公私立學校教職員、受僱者等 | 服務機關或雇主 |
| 第二類 | 職業工會會員、外僱船員 | 職業工會或船長公會 |
| 第三類 | 農民、漁民、水利會會員 | 農會、漁會、水利會 |
| 第四類 | 義務役軍人、替代役役男、軍校學生等 | 國防部等 |
| 第五類 | 低收入戶 | 戶籍所在地主管機關 |
| 第六類 | 其他地區人口（不屬前五類者） | 戶籍所在地鄉鎮市區公所 |

### 6.3.3 保險給付

全民健保提供的是「醫療服務」，不是「現金給付」。也就是說，你生病了去看醫生，健保會幫你付大部分的醫療費用，但不會直接發錢給你。

**給付範圍包括：**
- 門診及住院診療服務
- 藥品
- 檢驗、檢查
- 手術
- 住院
- 居家照護

**不給付的項目：**
- 美容手術
- 指定醫師
- 掛號費
- 特殊病房差額
- 非具治療性的診療項目

### 6.3.4 部分負擔

為了避免醫療資源浪費，健保法規定被保險人就醫時須自行負擔部分費用：
- 門診部分負擔：依醫療院所層級不同，負擔不同比例
- 住院部分負擔：依住院天數不同，負擔5%-30%
- 藥品部分負擔：超過一定金額的藥費，需自行負擔

### 6.3.5 重要的憲法爭議

**大法官釋字第472號：** 全民健保具有強制性，但屬於國家為增進全體國民健康而建立的社會保險制度，符合憲法增修條文§10第5項國家應推行全民健保的意旨，尚屬合憲。但對於無力繳納保費者，國家應給予適當之救助。

**大法官釋字第676號：** 健保法規定被保險人欠繳保費即予以鎖卡（暫停就醫權益），如未設適當的緩衝機制，有違比例原則。

---

## 6.4 國民年金

### 6.4.1 制度目的

國民年金保險（97年10月1日開辦）是為了照顧25歲以上、未滿65歲，沒有參加其他社會保險（勞保、軍保、公保、農保）的國民。

> **白話解釋：** 有工作的人有勞保，軍人有軍保，公務員有公保，農民有農保。那沒有工作的人（家庭主婦、學生畢業後待業、自由工作者等）怎麼辦？國民年金就是為了填補這個缺口。

### 6.4.2 加保對象

**國民年金法§7：** 年滿25歲、未滿65歲，在國內設有戶籍，且未參加勞保、農保、公教保險、軍人保險的國民，應參加本保險為被保險人。

### 6.4.3 保險給付

| 給付種類 | 內容 |
|----------|------|
| 老年年金 | 年滿65歲即可請領，按月發給 |
| 身心障礙年金 | 被保險人在保險期間發生身心障礙 |
| 喪葬給付 | 被保險人死亡時發給 |
| 遺屬年金 | 被保險人死亡時，符合條件的遺屬按月領取 |
| 生育給付 | 被保險人分娩或早產，按月投保金額發給2個月 |

### 6.4.4 國民年金與勞保的銜接

許多人在工作期間參加勞保，離職後轉為國民年金被保險人，再就業後又轉回勞保。兩種保險的年資可以分別計算，在請領老年給付時各自依其規定請領。

**例子：** 小美工作了20年（勞保年資20年），之後在家帶小孩10年（國民年金年資10年）。她65歲時可以同時請領勞保老年年金和國民年金老年年金。

## 7.2 答題架構建議

### 解僱類題目的答題架構

```
一、問題提出
   → 本題涉及雇主解僱勞工是否合法的問題

二、解僱事由的認定
   → 雇主主張的事由是§11（經濟性解僱）還是§12（懲戒解僱）？
   → 該事由在事實上是否成立？

三、解僱程序的合法性
   → 是否遵守預告期間（§11的情形）？
   → 是否在30日除斥期間內行使（§12的情形）？

四、解僱最後手段性原則
   → 雇主是否已經嘗試其他替代手段？
   → 解僱是否符合比例原則？

五、違法解僱的法律效果
   → 解僱無效 → 僱傭關係繼續存在
   → 勞工可請求確認僱傭關係存在 + 給付工資
   → 勞動事件法§49定暫時狀態處分
```

### 罷工類題目的答題架構

```
一、問題提出
   → 本題涉及罷工是否合法的問題

二、主體要件
   → 是否由工會發動？
   → 工會是否合法成立？

三、事由要件
   → 是否為調整事項的爭議？
   → 調整事項 vs 權利事項的區辨

四、程序要件
   → 是否經過調解不成立？
   → 是否經工會會員直接無記名投票過半數同意？

五、方式要件
   → 有無暴力、破壞行為？
   → 有無侵害他人生命、身體、自由或財產？

六、主體限制
   → 是否為禁止罷工的對象（教師等）？

七、結論
   → 綜合判斷罷工的合法性
```

### 不當勞動行為類題目的答題架構

```
一、問題提出
   → 本題涉及是否構成不當勞動行為的問題

二、不當勞動行為的類型認定
   → 不利益待遇（§35第1項第1、3、4款）？
   → 支配介入（§35第1項第5款）？
   → 黃犬契約（§35第1項第2款）？

三、構成要件分析
   → 雇主的行為是什麼？
   → 勞工的正當活動是什麼？
   → 兩者之間有無因果關係？
   → 雇主的真正動機為何？

四、救濟途徑
   → 向不當勞動行為裁決委員會申請裁決
   → 裁決決定的效力
   → 不服裁決的救濟（行政訴訟）
```

## 7.4 容易混淆的概念對比

### 解僱 vs 資遣 vs 開除

| 概念 | 法律依據 | 性質 | 資遣費 | 預告 |
|------|----------|------|--------|------|
| 經濟性解僱（資遣） | §11 | 非勞工過錯 | 有 | 有 |
| 懲戒解僱（開除） | §12 | 勞工有重大過錯 | 無 | 無 |
| 勞工自請離職 | §15 | 勞工主動 | 無 | 有（§15準用§16） |
| 勞工被迫離職 | §14 | 雇主有過錯 | 有 | 無 |

### 職災「補償」vs 職災「賠償」

| 比較項目 | 職災補償（勞基法§59） | 職災賠償（民法§184以下） |
|----------|----------------------|------------------------|
| 歸責原則 | 無過失責任 | 過失責任 |
| 舉證責任 | 勞工只需證明職災事實 | 勞工需證明雇主有過失 |
| 賠償範圍 | 法定項目（醫療、工資、失能、死亡） | 完整損害賠償（含精神慰撫金） |
| 抵充關係 | 已領取的補償可抵充賠償金額 | — |

### 權利事項 vs 調整事項

| 比較項目 | 權利事項 | 調整事項 |
|----------|----------|----------|
| 定義 | 基於法令、契約已有的權利義務 | 尚未約定的勞動條件之變更 |
| 解決方式 | 調解 + 訴訟（或仲裁） | 調解 + 仲裁（不能訴訟） |
| 可否罷工 | 不可以 | 可以 |
| 例子 | 雇主欠薪、違法解僱 | 要求加薪、增加假日 |

## 7.6 學習心法：讓勞動法變成你的「常識」

勞動社會法和其他法律科目最大的不同在於——它就是你的日常生活。

每一個你經歷過的工作經驗，都是勞動法的素材：

- 面試時被問「你結婚了嗎？打算生小孩嗎？」→ 這可能涉及就業歧視（就業服務法§5）
- 公司規定「試用期3個月內離職不發薪水」→ 違反勞基法§22全額給付原則
- 老闆說「不加班的人考績打差」→ 可能涉及變相強迫加班
- 同事受傷後被要求用自己的假請假 → 可能違反職災補償規定

**當你開始用「勞動法的眼睛」看你的工作經驗時，你就已經在學習了。**

考試時，你需要做的就是：
1. 準確辨識法律爭點（這個問題涉及哪些法條？）
2. 正確適用法律要件（構成要件是什麼？事實是否符合？）
3. 運用重要原則（解僱最後手段性、比例原則、誠信協商）
4. 得出具有說服力的結論

祝你考試順利！
