## ch249 不想分開的人

**情緒強度：8（從夢中醒來的人，和不想醒來的人）**

---

### 核心場景

```
第一個人醒來的時候，維倫差點沒接住。

不是物理上的「接」。
是意識層面的——
一段記憶從融合體中被分離出來，
像從蜂蜜裡抽出一根筷子，
黏稠、緩慢、拉絲。

然後斷了。

那個人的意識碎片重新凝聚。
模糊的輪廓慢慢清晰。
一個男人。四十多歲。圓臉。
眼睛睜開的瞬間——

茫然。

不是恐懼。不是憤怒。
是一種比恐懼更深的東西。

空白。

他看著維倫。
看著自己的手。
看著周圍漂浮的記憶碎片海洋。

「我……」

聲音沙啞。像很久沒有單獨說過話。

「我叫什麼？」


維倫的胃收縮了一下。

「你叫什麼——我不知道。」
他說。聲音盡量穩。
「但你的記憶簽名裡有一個地址。
　新台北市大安區——」

「大安區。」

男人重複這三個字。
眼睛忽然亮了。
不是想起來的亮。
是身體反應——
某種比記憶更深的東西被觸發了。

「大安區。復興南路。」

他的手在空氣中比劃。
右轉。直走。第三個路口。

「有一棵——很大的榕樹。」

維倫沒有說話。

男人的手停在半空。
手指微微顫抖。

「我住在榕樹旁邊。」

他的聲音變了。
從沙啞變成——濕的。

「我住在榕樹旁邊。對不對？」

維倫查了記憶簽名。
地址：新台北市大安區復興南路二段。
門牌號碼被融合汙染了，讀不到。
但——有一棵榕樹。
城市地景資料庫裡的標記。

「對。」維倫說。

男人哭了。
不是嚎啕。
是眼淚直接流下來。
安靜地。
像水龍頭沒關緊。

他想起了一棵樹。
還沒想起自己的名字。
但他想起了一棵樹。


第二個人醒來的過程比較快。
一個年輕女孩。二十出頭。
她醒來的第一句話是：「現在幾點？」

維倫愣了一下。

「——我不確定。在集體潛意識裡沒有標準時間。」

「噢。」
她看了看四周。
「那我遲到了嗎？」

「遲到——什麼？」

「上班。我在便利商店上班。早班。」

她的語氣很平。
像在問一個完全正常的問題。
像她只是睡過頭了。

維倫看著她。
她的記憶簽名顯示她被融合了至少三年。

三年。

她以為自己只是睡過頭了。


第三個人不一樣。

維倫嘗試分離她的時候，
融合體的表面起了波紋。
不是被動的阻力——
是主動的抗拒。

像水母收縮觸手。
把那段記憶往內裡拉。

維倫加大力道。
記憶碎片開始震動。

然後——一個聲音。

不是從融合體的集體意識來的。
是從那段記憶本身。

「不要。」

女人的聲音。三十多歲。低沉。平靜。
平靜得不正常。

「不要把我拉出去。」

維倫停了。

「你知道你在融合體裡面？」

「我知道。」

「你知道你已經失去個體意識了？」

「我沒有失去。我選擇了。」

沉默。

融合體的波紋慢慢平息。
那段記憶在波紋中間安靜地浮著。
像一片葉子浮在湖面。

「我在外面的時候，」
她的聲音從那片葉子裡傳出來，
「每天早上醒來都要花十分鐘說服自己起床。
　不是因為累。
　是因為——沒有理由起來。」

維倫沒有回答。

「在這裡不一樣。
　在這裡我能感覺到所有人。
　他們的快樂。
　他們的期待。
　有人剛談戀愛。有人的孩子出生了。
　有人終於學會了游泳——四十七歲才學會。」

她的聲音裡有笑意。

「我在外面感覺不到這些。
　在外面我只有——我自己。
　我自己不夠。」

那個字。
「不夠。」

維倫的手停在融合體邊緣。
他知道這個字。
他在記憶修復師生涯裡聽過幾百次。

我的記憶不夠完整。
我的快樂不夠多。
我的人生不夠好。

不夠。

「為什麼要把我拉回去？」

她的聲音還是那麼平靜。

「你能保證外面比這裡好嗎？」

維倫張了張嘴。

關上了。

他不能。
他保證不了。

語青在他身後。
沒有說話。
但她的存在是一種提醒——

有些問題沒有答案。
```

本章無演算法主線。純情感章。

### 寫作指令

- **視角**：第三人稱，維倫主導。三次分離嘗試構成三段落，節奏遞進：快（成功）→ 快（成功但錯位）→ 慢（失敗/拒絕）。
- **語氣**：克制。維倫不做道德判斷。他是修理工，不是傳教士。
- **節奏**：前兩段分離快寫，各用 30-40 行。第三段拉長到 60 行以上。重心在女人的獨白。
- **禁忌**：不可讓維倫說教。不可出現「你應該珍惜自己」之類的話。他的沉默比任何話都有力。
- **五感**：記憶從融合體中抽離的黏稠觸感。男人眼淚安靜流下的視覺。女人聲音的平靜——平靜本身是一種感官衝擊。

## ch250 蜂巢中的身體記得

**情緒強度：9（身體拒絕被吞沒——「身體記得」母題的蜂巢極致考驗）**
**紙鶴第四次出場（ch8 → ch130 → ch153 → ch250）**

---

### 核心場景

```
那雙手不屬於任何人。

或者說——它曾經屬於某個人。
但那個人的意識已經融合了。
名字、記憶、喜好、恐懼——
全部溶進了集體的海洋。

只剩下手。

手指在動。
拇指壓住食指和中指。
食指翻折。
中指跟進。
無名指從下方托起。

一隻紙鶴的第一個摺痕。

但沒有紙。

手指在空氣中折著一隻不存在的紙鶴。
重複。
一遍。
又一遍。
又一遍。


維倫蹲下來。

他看著那雙手。
女性的手。指甲很短。
指腹有繭——不是打字的繭。
是長期折紙的繭。
拇指側邊有一道舊疤。
裁紙刀的。

「米亞。」

（0.4 秒。）

「這雙手的主人——能查到嗎？」

（3.2 秒。）

米亞的延遲說明了一切。
3.2 秒。她在融合體的數億條記憶中搜索指紋。

「部分匹配。
　指紋序列與新台北市戶政資料庫中的
　一名女性吻合度 74%。
　融合汙染導致生物特徵退化。
　姓名：無法確認。
　年齡：推估 28-35 歲。
　職業：無法確認。」

「74%。」

維倫看著那雙手。
手指沒有停。
折到第三步——展開翅膀——然後回到起點。
重複。

「還有什麼？」

（0.4 秒。）

「她的嘴唇。」

維倫這才注意到。

不只是手。
融合體的表面，在那雙手的上方大約三十公分處，
有一張嘴。
也是從融合體中微微突出的。
不完整——只有下唇和下巴。

嘴唇在動。

無聲地。
反覆地。

維倫湊近。

他讀唇。

兩個音節。
同一個形狀。
反覆。

一個名字。

他讀不出那個名字是什麼。
但他知道那是一個名字。
因為嘴唇的形狀在第一個音節時微微上揚——
叫一個人名字的時候，
嘴角會不自覺地——
帶一點笑。


語青走過來。

她站在維倫旁邊。
看著那雙手。那張嘴。

她的表情——

維倫從來沒見過她這個表情。

不是難過。不是同情。
是——認出。

她認出了什麼。

「語青？」

她沒有回答。

她慢慢蹲下來。
和維倫一樣的姿勢。
半透明的膝蓋貼著地面。

她伸出手。

她的手也是半透明的。
指尖微微發光。

她把手放在那雙折紙鶴的手旁邊。
沒有碰到。
隔了大概一公分。

然後她的手指開始動了。

跟著那雙手——
她也在折。

同樣的節奏。
同樣的步驟。
拇指壓住食指和中指。
食指翻折。
中指跟進。

兩雙手。
一雙實體的、被困在融合體中的。
一雙半透明的、自由的。

在空氣中折著同一隻不存在的紙鶴。


「身體記得。」語青說。

她的聲音很輕。
輕得像是在自言自語。

「意識可以被融合。
　記憶可以被覆寫。
　名字可以被刪除。
　但身體記得。」

她看著那雙手。

「手指記得紙鶴的形狀。
　嘴唇記得一個名字的發音。
　這些不是記憶。
　這些是——刻痕。」

她停頓了一下。

「蜂巢可以吞掉意識。
　可以融合思想。
　可以讓一個人忘記自己是誰。
　但它吞不掉肌肉纖維裡的重複。
　吞不掉神經末梢的習慣。
　吞不掉骨頭記得的彎曲角度。」

她的手停了。
那雙融合體中的手還在折。

「身體拒絕被融合。」

語青站起來。
她看著維倫。
眼睛是亮的——即使半透明。

「這就是答案。」

維倫看著她。

「什麼的答案？」

語青沒有回答這個問題。
她轉過身。
看著整片融合區。

數千人。
數千具身體。

維倫順著她的視線看過去。

然後他看見了。

不只是那一雙手。

到處都是。

角落裡，有人的腳趾在打拍子。
某種節奏。某首歌。意識不記得歌名了，
但腳趾記得節拍。

左邊，有人的肩膀在做一個動作——
反覆聳起、放下。聳起、放下。
像在背一個很重的背包。
背包早就不在了。
但肩膀還在背。

更遠處，有人的嘴在咀嚼。
沒有食物。
但下顎的節奏是真實的——
左邊三下、右邊兩下、左邊三下。
某種特定食物的咀嚼方式。
身體記得那個味道。

數千個身體。
每一個都在做著某個屬於「自己」的動作。
意識融合了。思想是集體的。

但身體是私人的。
身體拒絕成為「我們」。
身體堅持是「我」。


維倫站在融合區中間。

他忽然理解了一件事。

如果有一天——
如果有一天，所有人的意識真的融合成蜂巢。
如果個體思想真的消失。
如果「我」這個概念真的不存在了。

身體還會在。

手指還會折紙鶴。
嘴唇還會叫一個名字。
肩膀還會背一個不存在的背包。

身體是最後的防線。

身體是個體性的——遺書。

不。

不是遺書。

是抗議。

是身體寫在融合體表面的——
一封一封的抗議信。

「我在這裡。」
「我曾經在這裡。」
「我不同意。」
```

### AB線分析

本章無演算法主線。但「身體記得」母題在此完成蜂巢語境的極致測試。

| 層面 | 內容 |
|---|---|
| 紙鶴第四次出場 | ch8 初見 → ch130 作為記憶錨點 → ch153 語青的時間線碎片 → ch250 融合體中身體的抗議 |
| 身體 vs 意識 | 意識可融合，身體不可。肌肉記憶是比語意記憶更深的存在證明 |
| 語青的「認出」 | 她認出了什麼？她自己也是「身體記得」的受益者——半透明的存在，靠身體的殘留維繫 |
| 集體中的個體 | 數千個身體各自做著私人動作——融合體表面的「抗議信」 |

**母題進化筆記**：「身體記得」在 Vol 1-3 是溫暖的——邱韻如的手記得女兒的臉（ch227）。在 ch250，它變成了嚴峻的：如果意識已經不存在了，身體記得還有意義嗎？答案是有。因為身體記得不只是「記得」——它是「拒絕」。拒絕被同化。拒絕失去個體性。這為 ch297 語青寫入肌肉記憶提供了理論基礎：如果身體是最後的防線，那麼把記憶寫進身體，就是最不可摧毀的保存方式。

### 寫作指令

- **視角**：第三人稱。前半段維倫視角觀察那雙手和那張嘴。中段語青介入。後段維倫的視野擴大到整個融合區。
- **語氣**：安靜的震撼。不是大場面的震撼。是蹲在地上看著一雙手折紙鶴時的——胸口被輕輕按住的感覺。
- **節奏**：慢。整章的節奏都是慢的。紙鶴摺疊的動作要寫得像慢動作回放。語青跟著折的段落是全章的核心畫面——兩雙手，一實一虛，折同一隻不存在的紙鶴。

碎片的溫度和記憶的溫度一致。

語青坐在他旁邊。

她的存在方式很安靜。
半透明的身體微微發光。
在集體潛意識的黑暗裡，
她是一盞很小的燈。

他們沒有說話。

很久。

融合區在他們面前脈動。
裡面有數千人。
有些想出來。有些不想。
有些的身體在抗議。有些的身體已經安靜了。

**核心教學：Segment Tree + Lazy Propagation**

Segment Tree（線段樹）是一種二叉樹資料結構，用於高效處理區間查詢和區間修改：

1. **結構**：每個節點代表一段連續區間 [L, R]。根節點代表全部區間。每次二分，左子 [L, mid]、右子 [mid+1, R]。葉子是單一元素。

2. **區間查詢**：給定查詢區間 [qL, qR]，從根往下遞迴。若節點區間完全包含在查詢區間內，直接回傳。否則拆分到左右子樹。時間 O(log n)。

3. **區間修改**：要修改 [qL, qR] 內所有元素，不逐一修改葉子（O(n)太慢）。用 Lazy Propagation：在完全包含的中間節點上標記「待修改」，不往下傳。等到日後有查詢經過這個節點時，才把標記「推」下去一層。均攤時間 O(log n)。

4. **故事中的對應**：
   - 葉子 = 一段記憶
   - 中間節點 = 一群人的記憶集合
   - 區間查詢 = 「看」任何人的記憶
   - 區間修改 + Lazy Propagation = 在人們「回憶」的那一刻悄悄修改記憶
   - 4,700 萬 pending 標記 = 4,700 萬段待修改的記憶

5. **核心比喻**：「Segment Tree 不是在管理記憶——是在管理人。每一個區間的定義都是一個權力的邊界。」
