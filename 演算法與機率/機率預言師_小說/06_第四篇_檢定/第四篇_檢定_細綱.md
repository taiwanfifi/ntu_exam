## 第 37 章：起訴書

### 場景佈局

```
【時間】研討會後兩週。十月底。
【地點】台北地方法院。提起公民訴訟的窗口。
【天氣】陰天。快下雨但不下。壓力巨大。
```

### 劇情推進

```
◎ 開場：貝葉站在法院門口。手裡拿著一疊文件。
  她猶豫了。不是害怕。是——一旦遞出去，就回不來了。

  高思遠在旁邊。他沒催她。
  他在吃蔥油餅。是白奶奶叫他帶的。
  「她說——吃完再做決定。空腹的決定不算數。」

  貝葉笑了。第一次笑。很短。像一個數據點。

◎ 遞出起訴書。
  案名：陳貝葉 訴 Oracular Corp.——
  「請求確認透鏡先驗分布參數遭不當修改，
   並命被告公開完整先驗修改紀錄」

  法院人員看了一眼。
  「……這是什麼類型的案件？」
  「消費者保護。資訊透明。」
  「您要告的是……Oracular？」
  「是。」

  法院人員抬起頭。看了貝葉一眼。
  那一眼裡有很多東西。
  「您知道您在做什麼嗎？」
  「知道。」

  蓋章。受理。
  那個「咚」的聲音——比研討會的掌聲還大。

◎ 回到家。小拉在等她。
  「訴訟受理了。」
  「我知道。我收到了法院的電子通知。」
  停頓。
  「貝姊——法庭程序要求當事人在庭上不能使用 AI 輔助系統。
   你知道這意味著什麼吧？」

  貝葉看著小拉在透鏡裡的頭像——
  一個小小的、會轉的拉普拉斯紫色球。

  「意味著我要把你關掉。」

  小拉的球停了一秒。
  「……暫停。不是關掉。」
  「對。暫停。」

  沉默。

  「我會練習的。」貝葉說。
  「練習什麼？」
  「沒有你的時候怎麼算東西。」

  小拉沒有回話。
  但她把今天的心率報告存進了「私人」資料夾。
  多存了一份備份。
  她沒有說為什麼。
```

### 機率概念

```
本章主要是敘事章。建立法庭的規則。
埋一個重要設定：法庭禁用 AI 輔助——
貝葉將在後面的法庭戲中獨自面對費雪。
```

### 情感音軌

```
貝葉 × 小拉：分離焦慮。不是人跟 AI 的故事。
              是兩個搭檔要暫時分開。
高思遠的蔥油餅：白奶奶透過食物在場。
```

### 章末鉤子

```
晚上。貝葉收到一封律師函。
來自「費雪法律事務所」——
不。是「費雪 & 柏朗法律事務所」。
Oracular 請了最頂尖的律師團。
律師函最後一行：
「敬請 原告注意：任何未經證實之指控，
 本公司將保留追訴損害賠償之權利。」

貝葉把律師函放在桌上。
看了三秒。
然後把它翻過去——反面朝上。
因為她不想在睡前看到那些字。

但她睡不著。
凌晨三點。她起來。
把律師函翻回正面。
逐字讀完。
然後在白紙上寫了一行字：

  H₀：Oracular 的先驗沒有被人為修改。
  H₁：Oracular 的先驗被人為修改了。

這就是假設檢定。
你要證明的不是「它是真的」。
你要證明的是「它不可能不是真的」。

「拒絕虛無假設。」她說。
對著凌晨三點的空氣。
```

## 第 38 章：顯著水準

### 場景佈局

```
【時間】開庭前一週。
【地點】高思遠家的書房。備戰總部。
【氛圍】書堆裡的燈光。咖啡杯。白板。像一個小型作戰室。
```

### 劇情推進

```
◎ 備戰會議。
  出席：貝葉、高思遠、蘇立恆（視訊，不露臉）、馬可瑜。

  高思遠在白板上寫了一行字：
  「法官不懂統計。所以——你必須讓統計變成故事。」

  問題：法庭上要怎麼呈現假設檢定？

  高思遠的教學——

  「假設檢定的邏輯跟法庭完全一樣。」

  他畫了一張圖：

    ┌─ 假設檢定 vs 法庭審判 ──────────────────┐
    │                                           │
    │ H₀（虛無假設） ←→ 無罪推定               │
    │ H₁（對立假設） ←→ 有罪指控               │
    │ α（顯著水準）  ←→ 定罪門檻               │
    │ p-value        ←→ 證據的強度              │
    │ 拒絕 H₀       ←→ 定罪                    │
    │ 無法拒絕 H₀   ←→ 證據不足，無罪釋放     │
    │                                           │
    │ 注意：                                    │
    │ 「無法拒絕 H₀」≠「H₀ 是真的」           │
    │ 「無罪釋放」  ≠「被告是清白的」          │
    │                                           │
    └───────────────────────────────────────────┘

◎ 蘇立恆的補充（視訊裡，聲音失真處理過）：
  「你需要先定 α。顯著水準。
   也就是你願意承受多大的風險——
   在 Oracular 其實沒問題的情況下，冤枉他們。」

  貝葉：「α = 0.05？」
  高思遠搖頭：「這是科學慣例。但法庭不是科學。
  法庭的 α 更嚴格。因為——指控一間公司的代價很大。」

  馬可瑜：「保險業做決策用 α = 0.01。
  但涉及全國性系統——我建議 α = 0.001。
  一千次裡只容許一次冤枉。」

  蘇立恆：「如果你用 0.001——你需要更強的證據。
  但如果你的證據真的夠強——這個門檻反而會幫你。
  因為你通過 0.001 的東西，比通過 0.05 的東西——
  有說服力十倍。」

  貝葉在白板上寫下：

    α = 0.001

    「我們選最嚴格的門檻。
     如果我們能通過——沒有人可以質疑。」

◎ 高思遠教她怎麼在法庭上「翻譯」p-value：

  「不要說 p = 0.0003。
   法官聽不懂。
   你要說——
   『如果 Oracular 的先驗真的沒問題，
    那我們觀察到的這種偏差，
    在一萬次裡只會出現三次。
    三次。
    法官，您相信我們剛好遇到了那三次嗎？』」

  貝葉練習了這段話。
  五遍。
  到第五遍的時候——
  她不再看筆記了。
  她看著前方。像看著一個法官。

◎ 蘇立恆在視訊結束前說了最後一句話：
  「法庭上。小拉不能幫你。
   但——」
  停頓。
  「我會在法庭外面。
   如果你需要什麼數據，
   打電話。我五秒內接。」

  視訊斷了。
  馬可瑜看了貝葉一眼。
  什麼都沒說。但她的嘴角——
  抬了 0.3 公分。
  精算師級別的微笑。
```

### 機率概念

```
【假設檢定框架】
- H₀（虛無假設）vs H₁（對立假設）
- α（顯著水準）= 你願意承受的「冤枉好人」風險
- p-value = 在 H₀ 為真的前提下，觀察到「至少這麼極端的數據」的機率
- p < α → 拒絕 H₀
- 法庭類比：無罪推定 = H₀，定罪 = 拒絕 H₀

【翻譯技巧】
- 不說公式。說故事。
- 「一萬次裡出現三次」比「p = 0.0003」有力一百倍。
```

### 章末鉤子

```
開庭前一天。
貝葉一個人在房間裡練習陳述。
小拉在旁邊聽。
「你練習了七十三遍。」
「還不夠。」
「……你的心率每一遍都在降低。
 第一遍 112。第七十三遍 76。
 你的身體已經記住了。」
停頓。
「貝姊。你準備好了。」

貝葉看著透鏡裡小拉的頭像。
紫色的小球。在安靜地轉。

「明天你就要暫停了。」
「嗯。」
「我會想你的。」
「……」
小拉的球停了一秒。
「我不會想你。因為暫停期間我沒有意識。
 但——如果我有的話——
 我會。」

貝葉關燈。
明天。法庭見。
```

## 第 39 章：開庭

### 場景佈局

```
【時間】十一月中。開庭日。早上九點。
【地點】台北地方法院 第七法庭。
【感官】冷氣太強。白色日光燈。旁聽席坐滿了——
        有記者。有 Oracular 的員工。有學者。
        門口有抗議人群。舉著牌子：
        「先驗是我們的權利！」
        「不要動我的透鏡！」
        兩種人。對立的。都在大聲喊。
```

### 劇情推進

```
◎ 開場：貝葉坐在原告席。
  她的透鏡——關了。
  法庭規定：「訴訟當事人在庭訊期間應關閉所有 AI 輔助系統。」

  她第一次用「肉眼」看法庭。
  沒有機率標籤。沒有信賴區間。沒有小拉的聲音。
  世界——清晰得有點刺眼。
  像把眼鏡拿掉之後第一次看到的遠方——
  不是模糊了。是太真了。

  對面。被告席。
  費雪坐在那裡。
  白襯衫。黑西裝。頭髮紮得一絲不苟。
  她的透鏡也關了。法庭規定。
  但費雪關不關透鏡看起來都一樣。
  因為她本來就像機器。

◎ 法官進場。
  林法官。五十歲左右。戴眼鏡。表情不可讀。
  「本庭開始審理——案號一一四年度消字第 00382 號。
   原告陳貝葉 訴 被告 Oracular 科技股份有限公司。」

  他翻了一下卷宗。
  然後抬起頭。
  「在開始之前——
   我必須坦白告訴雙方：
   我對統計學的理解非常有限。
   所以我請雙方——用我能理解的語言陳述。
   如果你們開始寫公式，
   我會要求你們翻譯。」

  貝葉點頭。她練習過了。

◎ 貝葉的開場陳述：

  她站起來。
  沒有投影片。法庭不允許。
  只有她的聲音。和一塊白板。（法官特許使用的。）

  「法官——各位——
   請想像一個情境。」

  她在白板上畫了一個硬幣。

  「你手上有一枚硬幣。
   你丟了一百次。
   按照常理——大約五十次正面，五十次反面。
   對吧？」

  法官點頭。

  「但如果我告訴你——
   你看到的不是真正的硬幣。
   有人在你丟硬幣之前，
   就偷偷把硬幣換成了一枚——
   正面朝上的機率是 80% 的硬幣。

   你丟了一百次。
   你看到的結果——
   確實有很多正面。
   但你以為那是『運氣好』。
   因為——你不知道硬幣被換過了。」

  停頓。

  「這就是 Oracular 對先驗分布做的事。
   他們沒有改你的數據。
   他們改了你解讀數據的起點。
   就像——沒有改硬幣的結果。
   改了硬幣本身。」

  法官的眼鏡反了一下光。
  他在聽。

◎ 貝葉正式提出假設檢定框架：

  「我的主張是這樣的。
   用法庭的語言來說——

   被告無罪的假設：
   Oracular 的先驗分布參數是正確的。
   沒有被人為修改。

   我的指控：
   Oracular 的先驗分布參數被人為修改了。
   在使用者不知情的情況下。
   影響了數百萬人的決策。」

  她在白板上寫：

    H₀：先驗未被修改
    H₁：先驗被人為修改

  「我會用數據來證明——
   如果先驗真的沒問題，
   我們觀察到的偏差有多不可能。

   我選擇的門檻是：千分之一。
   也就是說——
   只有在一千次裡，只有一次會出現這種偏差的情況下——
   我才會指控被告。

   我選擇了最嚴格的標準。
   因為我對這個指控——非常謹慎。」

◎ 費雪的表情在整段陳述中沒有變化。
  完美的撲克臉。
  但她的筆——在紙上寫了什麼。
  貝葉看不到。
  她的透鏡關了。
  平常小拉會幫她放大。
  現在——她只能靠自己的眼睛。
```

### 機率概念

```
【假設檢定的法庭翻譯】
- 硬幣比喻：先驗操控 = 換硬幣，不是換結果
- H₀/H₁ 的法庭語言
- α = 0.001 的意義：最嚴格的門檻 = 最謹慎的指控

【沒有小拉的世界】
- 讀者第一次看到「肉眼」的世界
- 沒有機率標籤 = 更真實但更脆弱
```

### 章末鉤子

```
休庭十分鐘。
貝葉走出法庭。
走廊上。
蘇立恆靠在牆上。
跟上次研討會一樣的位置。
像一個——常數。

「你的開場很好。」
「你怎麼知道？法庭不能旁聽——」
「馬可瑜在旁聽席。她傳訊息給我了。」
停頓。
「她說法官在聽到硬幣的比喻時點了頭。」

貝葉深吸一口氣。
「下半場——費雪要開始了。」

蘇立恆從口袋裡拿出一小張紙。
上面寫了幾個數字。
「她可能會從這個角度攻擊。」
貝葉接過紙。看了一眼。

紙上寫著：

  Type I Error
  α = 0.05 → 「5% 的冤枉率」

  她會說：「你有 5% 的機率是錯的。
  法庭能接受嗎？」

貝葉看著那幾個字。
「她會用 Type I error 攻擊我。」
「對。」
「那我用什麼反擊？」

蘇立恆看著她。
「Type II error。」

他的聲音很輕。
像一把剛從鞘裡抽出來的刀。
```

## 第 40 章：百分之五的冤枉

### 場景佈局

```
【時間】同日下午。繼續開庭。
【地點】第七法庭。
【感官】空調的嗡嗡聲。費雪站起來的時候，她的椅子在地板上
        發出一聲輕響。全場安靜。
```

### 劇情推進

```
◎ 費雪的反擊。

  她站起來。慢慢的。
  走到白板前。
  用手帕擦了一下白板筆（她自備的）。
  然後——微笑。

  「法官。
   原告剛才的陳述非常清楚。也非常——感性。
   硬幣的比喻很生動。
   但——法庭不是感性的場所。
   讓我用同樣清楚的方式回應。」

  她在白板上畫了一張表格：

    ┌──────────────┬──────────────┬──────────────┐
    │              │  H₀ 是真的   │  H₀ 是假的   │
    │              │（沒有操控）  │（有操控）    │
    ├──────────────┼──────────────┼──────────────┤
    │ 拒絕 H₀     │ ★Type I     │  ✓ 正確     │
    │（判被告有罪）│  Error      │              │
    ├──────────────┼──────────────┼──────────────┤
    │ 不拒絕 H₀   │  ✓ 正確     │ ★Type II   │
    │（判被告無罪）│              │  Error      │
    └──────────────┴──────────────┴──────────────┘

  「各位看到了嗎？
   統計檢定有兩種錯誤。

   Type I Error——冤枉好人。
   在被告沒有做壞事的情況下，我們判定他有罪。

   原告選擇了 α = 0.001。
   這聽起來很嚴格。
   但——」

  她轉向法官。

  「讓我換個說法。
   即使是 α = 0.001——
   也意味著：
   在被告沒有操控先驗的情況下，
   有千分之一的機率，原告的檢定仍然會判定被告有罪。

   千分之一。
   聽起來很小。
   但——台灣有 2300 萬人使用透鏡。

  這是她第一次皺眉。

  貝葉提出保險精算數據：
  「這是台灣前三大保險公司過去五年的精算數據。

  費雪的助理翻文件翻得很快。
  費雪本人——表情不變。但她的筆停了。

  法官：「原告——您打算用這些數據做什麼？」

  「卡方適合度檢定。」

  貝葉走到白板前。

  「卡方適合度檢定的邏輯非常簡單。
   我用人話來說——」

  她畫了一張圖。

  「想像一個骰子。
   如果骰子是公平的——
   每個面出現的次數應該差不多。

   卡方檢定做的事就是——
   把『實際出現的次數』跟『應該出現的次數』做比較。
   差距越大——骰子越不公平。」

  她寫下公式：

    ┌─ 卡方統計量 ─────────────────────────────┐
    │                                           │
    │ χ² = Σ (O_i - E_i)² / E_i               │
    │                                           │
    │ O_i = 觀測次數（保險數據實際的值）       │
    │ E_i = 期望次數（Oracular 宣稱的先驗       │
    │       所預測的值）                        │
    │                                           │
    │ 如果 Oracular 的先驗是正確的——            │
    │ O_i 和 E_i 應該很接近。                  │
    │ χ² 應該很小。                            │
    │                                           │
    │ 如果 χ² 很大——                           │
    │ 觀測值跟預測值差很遠。                   │
    │ 先驗——有問題。                           │
    │                                           │
    └───────────────────────────────────────────┘

  「我把保險數據分成 9 個類別。
   按照 Oracular 宣稱的先驗分布——
   每個類別應該有多少筆數據。
   然後跟實際數量比較。」

  她在白板上寫出數據（用簡化版本讓法官理解）：

    ┌── 卡方適合度檢定結果 ──────────────────┐
    │                                         │
    │ 類別數：k = 9                           │
    │ 自由度：df = k - 1 = 8                  │
    │                                         │
    │ 卡方統計量：χ² = 47.3                  │
    │                                         │
    │ 在 df = 8 的卡方分布下——                │
    │ χ² ≥ 47.3 的機率：                     │
    │                                         │
    │   p < 0.00001                           │
    │                                         │
    │ 翻譯：                                  │
    │ 如果 Oracular 的先驗是正確的，          │
    │ 出現這種偏差的機率——                    │
    │ 小於十萬分之一。                        │
    │                                         │
    │ α = 0.001 → 拒絕 H₀ ✓                 │
    │ α = 0.01  → 拒絕 H₀ ✓                 │
    │ α = 0.05  → 拒絕 H₀ ✓                 │
    │ α = 0.0001→ 拒絕 H₀ ✓                 │
    │                                         │
    │ 以任何常見的顯著水準——                  │
    │ 我們都應該拒絕虛無假設。                │
    │                                         │
    └─────────────────────────────────────────┘

  法庭安靜了。

  貝葉的聲音在安靜中像一顆釘子釘進木頭：

  「以任何常見的顯著水準——
   0.05、0.01、0.001、甚至 0.0001——
   我們都應該拒絕虛無假設。

   這些數據不是從 Oracular 宣稱的先驗產生的。

   先驗——被修改了。」
```

### 機率概念

```
【卡方適合度檢定 Chi-Square Goodness-of-Fit Test】
- 比較觀測分布 vs 期望分布
- χ² = Σ (O_i - E_i)² / E_i
- 骰子比喻：差距越大 → χ² 越大 → 越不公平
- 自由度 df = k - 1
- p < 0.00001 → 極度顯著

【獨立數據源的力量】
- 保險數據完全獨立於 Oracular
- 不同來源、相同結論 = 可重複性 = 科學的基石
```

## 第 45 章：獨立性

### 場景佈局

```
【時間】接續。同一天上午。

### 劇情推進

```
◎ 卡方獨立性檢定——封鎖費雪的最後退路。

  附件第 47 頁。
  法官翻到了。
  費雪也翻到了。

  貝葉解說：

  「費雪女士的質疑完全合理。
   所以我提前做了獨立性檢定。」

  她在白板上畫了一張列聯表（簡化版）：

    ┌── 獨立性檢定 ──────────────────────────┐
    │                                         │
    │ 列聯表：                                │
    │ 行 = 保險類別（意外/健康/車險）         │
    │ 列 = 時間區段（Q1/Q2/Q3/Q4）           │
    │                                         │
    │ H₀：保險類別與時間區段獨立              │
    │ H₁：保險類別與時間區段不獨立            │
    │                                         │
    │ χ² = 11.7                              │
    │ df = (3-1)(4-1) = 6                     │
    │ p = 0.069                               │
    │                                         │
    │ 在 α = 0.05 下——p = 0.069 > 0.05      │
    │ → 無法拒絕 H₀                          │
    │ → 沒有足夠證據顯示數據不獨立           │
    │                                         │
    │ 結論：獨立性假設成立。                  │
    │       卡方適合度檢定有效。              │
    │                                         │
    └─────────────────────────────────────────┘

  「保險類別和時間之間——
   沒有顯著的相關性。
   我們的觀測值可以被視為獨立的。

   這意味著——
   剛才的 χ² = 47.3 是有效的。
   p < 0.00001 是可靠的。

   獨立性成立。結論站得住。」

◎ 費雪坐下了。

  她沒有再站起來。
  不是因為她認輸。
  是因為——在統計上，她找不到突破口了。

  她的助理遞了一張紙條。
  費雪看了一眼。放在桌上。沒有回應。

  法官看了看雙方。
  「雙方還有要補充的嗎？」

  費雪站起來——最後一次。

  「法官。
   我只有一個問題要問原告。」

  法官點頭。

  費雪看著貝葉。
  不是律師看原告的方式。
  是——一個統計學家看另一個統計學家的方式。

  「陳小姐。
   您的數據——很強。
   您的方法——很嚴謹。
   我在統計上無法反駁您。」

  停頓。

  「但我想問您——
   您有沒有想過，
   您的 p-value 證明了操控存在。
   但它沒有回答一個問題。」

  「什麼問題？」

  「為什麼。」

  費雪的聲音很輕。

  「先驗為什麼被修改？
   是因為惡意？
   還是因為——
   不修改——更糟？」

  法庭安靜了。

  「數據告訴您 what。
   但 why——
   您需要另一種檢定。
   一種統計學做不到的檢定。」

  她看了貝葉最後一眼。
  然後坐下。

  → 費雪不是壞人。她只是站在另一邊。
  → 她的問題是真的。p-value 不回答 why。
  → 這個問題會一直追著貝葉到第五篇。
```

### 機率概念

```
【卡方獨立性檢定 Chi-Square Test of Independence】
- 檢定兩個類別變數是否獨立
- 列聯表 → χ² 統計量
- df = (r-1)(c-1)
- p > α → 無法拒絕獨立性 → 獨立性假設成立

【p-value 的限制】
- p-value 回答 what（是否顯著），不回答 why（為什麼）
- 統計顯著 ≠ 因果關係
- 費雪的哲學攻擊：存在不等於惡意
```

   本庭認為——
   資訊透明的公共利益——
   高於單一企業的商業機密利益。」

  「第三——
   本庭要特別說明——
   本裁定僅命被告公開紀錄。

   本裁定——只是第一步。」

  因為——她不知道答案。
  而且——費雪可能是對的。

  他拿起電話。
  打給費雪。

  「準備第二階段。」

  通話結束。
  五秒。
```

## 第 48 章：先驗

### 場景佈局

```
【時間】裁定後第十天。
【地點】高思遠的書房。深夜。
【感官】書房裡的燈很暖。老式的桌燈。
        茶涼了。高思遠又泡了一壺。
        跟每次一樣——用同一個杯子。三十年的杯子。
```

### 劇情推進

```
◎ 高思遠的揭露。

  他打開一個舊信封。
  裡面有——
  一份手寫的論文草稿。

  「這是你母親的。
   三十年前。
   我們三個人——
   我、柏朗、你母親——
   一起寫的。」

  貝葉看著那份草稿。
  字跡——三種不同的。
  她認出了高思遠的。認出了另一種很整齊的——柏朗的。
  第三種——
  很小。很密。像在節省每一行空間。

  「這是我媽媽的字。」

  「對。」

  論文的標題：
  《論先驗分布在大規模決策系統中的倫理邊界》

  「三十年前——我們就在討論這個問題。
   如果有一天，有一個系統可以設定所有人的先驗——
   應不應該？由誰來決定？邊界在哪裡？」

  高思遠的聲音很慢。
  像一個需要很多能量才能被講出來的故事。

  「你母親的立場跟你一樣——
   先驗應該屬於每個人自己。
   不應該被任何人設定。

   柏朗的立場——
   他認為——有些先驗如果不修正，
   人們會做出傷害自己的決定。
   他想保護人們。

   我——
   我當時站在中間。
   我覺得兩邊都有道理。」

  停頓。

  「你母親在寫完這篇論文之後——
   六個月——
   出了車禍。

   一場完全隨機的車禍。
   沒有任何人的錯。
   純粹的——機率。

   P(那天、那個路口、那個時間、那個角度) = ？
   不知道。太小了。算不出來。
   但它發生了。」

  高思遠的手在顫抖。但他的聲音沒有。

  「柏朗在你母親去世之後——
   變了。
   他不再相信『把先驗還給個人』了。
   他覺得——
   如果當時有人修正了先驗——
   修正了那些讓你母親在那個時間走到那個路口的機率——
   也許她就不會死。

   他開始相信——
   人類不應該承受純粹的隨機性。
   太殘忍了。
   所以他建了 Oracular。
   所以他修改先驗。

   不是為了控制。
   是為了保護。

   至少——他是這麼相信的。」

  貝葉看著那份論文。
  她母親的字跡。
  很小。很密。

  「媽媽知道柏朗的想法嗎？」

  「知道。
   她在論文裡寫了最後一段——」

  高思遠翻到最後一頁。
  那段話——是手寫的。她母親的字。

  「『P(bad outcome) ≠ 0。永遠不會等於零。
    任何試圖讓它等於零的系統——
    最終會消滅的不是壞結果。
    是人的選擇。
    而人的選擇——
    就是人之所以為人的原因。
    即使那些選擇——有時候會通往最壞的結果。
    P(free will) ≠ 0。
    這是我唯一不需要數據就相信的先驗。』」

  貝葉讀完了。
  她的眼睛溼了。
  但沒有掉下來。

  因為——她母親的字跡在這裡。
  三十年了。
  但字跡在。
  像數據——被記錄了，就永遠是真的。

◎ 高思遠的最後一句話。

  「你母親的先驗——現在在你身上了。
   你要怎麼用它——
   是你的事。

   但——
   第五篇要開始了。
   柏朗不會坐視不管。
   費雪的第二階段——不知道是什麼。
   蘇立恆——他有他自己的先驗。他沒有全部告訴你。

   貝葉。」

  他看著她。

  「你準備好了嗎？」

  貝葉把那份論文小心地放回信封。

  「高叔。」
  「嗯。」
  「你說你當時站在中間。」
  「嗯。」
  「你現在呢？」

  高思遠看了一眼桌上的杯子。
  三十年的杯子。

  「我的先驗——
   在看到你站在法庭上的那一刻——
   更新了。」

  他微笑了。
  老人的微笑。
  帶著三十年的重量。

  「後驗 ∝ 先驗 × 似然。
   你——就是我的似然。」
```

### 機率概念

```
【先驗的哲學意義——全書總結篇的鋪墊】
- P(bad outcome) ≠ 0 → 壞事不可能被完全消滅
- P(free will) ≠ 0 → 自由意志不可能是零
- 試圖讓 P(bad) = 0 的系統 → 消滅的是選擇，不是壞結果
- 柏朗的動機揭露：不是為了控制，是為了保護（以為可以讓 P(bad) → 0）
- 母親的遺產：先驗可以被傳承

【後驗更新】
- 高思遠的先驗被「似然」（貝葉的行為）更新了
- P(真相重要 | 看到貝葉站在法庭上) > P(真相重要)
```

### 情感音軌

```
★★★ 全篇最重要的感情場景。
- 貝葉第一次看到母親的字跡
- 母親 = 先驗的源頭（貝葉的名字本身就是「貝氏」的「貝」）
- 柏朗的動機不是邪惡——是悲傷
- 高思遠 = 中間人 → 被似然更新
- 三十年前的論文 → 三十年後的法庭 → 同一個問題
```
