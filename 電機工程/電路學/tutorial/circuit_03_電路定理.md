# 第三章：電路定理

> **適用對象**：零基礎大一大二學生
> **先修知識**：第一、二章（KVL、KCL、節點法、網目法）
> **學習時間**：約 4-5 小時

---

## 🔰 **本章基礎觀念（零基礎必讀）**

上一章我們學了「暴力解法」——列出所有方程式然後解。這一章要學的是「聰明解法」——用定理把複雜電路簡化。

想像你在搬家，可以一箱一箱慢慢搬（暴力法），也可以叫搬家公司（定理），一次處理一大堆。

本章的五大定理就是你的「搬家工具箱」：

1. **疊加定理**：把多個電源的效果分開算，再加起來
2. **源轉換**：電壓源和電流源可以互相轉換
3. **戴維寧定理**：再複雜的電路都能簡化成「一個電壓源 + 一個電阻」
4. **諾頓定理**：再複雜的電路都能簡化成「一個電流源 ∥ 一個電阻」
5. **最大功率轉移**：什麼時候負載能吃到最多功率？

---

## 一、疊加定理（Superposition Theorem）

### 1.1 原理

在**線性電路**中，任何支路的電壓或電流等於**每個獨立源單獨作用時**的貢獻之和。

> 白話說：如果有三個電池，你可以一次只開一個、關掉其他兩個，分別算出答案，最後加起來。

### 1.2 如何「關掉」電源？

| 電源類型 | 關掉方式 | 原因 |
|---------|---------|------|
| 電壓源 | 用**短路**取代（導線直接連通） | 理想電壓源內阻 = 0 |
| 電流源 | 用**斷路**取代（把線剪斷） | 理想電流源內阻 = ∞ |

> 注意：**相依源不能關掉**！只有獨立源才能用疊加定理。

### 1.3 完整步驟

| 步驟 | 操作 |
|------|------|
| ① | 數一下電路中有幾個**獨立源** |
| ② | 每次只保留**一個**獨立源，把其他的關掉 |
| ③ | 求出每種情況下要求的電壓/電流 |
| ④ | 把所有結果**代數相加**（注意正負號！） |

### 1.4 數值例題 ①

> **題目**：電路中有 12V 電壓源和 3A 電流源。
> - 12V 電壓源串聯 4Ω 電阻（左邊）
> - 3A 電流源並聯 6Ω 電阻（右邊）
> - 兩部分通過中間 2Ω 電阻連接
> - 求 6Ω 電阻上的電壓 Vₒ。

**解答**：

**情況 1：只有 12V 電壓源（關掉 3A 電流源→斷路）**

電流源斷路後，6Ω 和 2Ω 串聯 = 8Ω，再和 4Ω 串聯 = 12Ω（因電流源斷路，6Ω 和 2Ω 串聯成一條路）

等等，讓我重新分析：電流源斷路後，電流只從 12V 經過 4Ω → 2Ω → 6Ω 流回。

$$
I_1 = \frac{12}{4 + 2 + 6} = \frac{12}{12} = 1 \text{ A}
$$
$$
V_{o1} = 1 \times 6 = 6 \text{ V}
$$

**情況 2：只有 3A 電流源（關掉 12V 電壓源→短路）**

電壓源短路後，4Ω 和 2Ω 串聯 = 6Ω，這 6Ω 再和右邊的 6Ω 並聯。

$$
R_{eq} = \frac{6 \times 6}{6 + 6} = 3 \text{ Ω}
$$

3A 電流源流過等效 3Ω：
$$
V_{o2} = 3 \times 3 = 9 \text{ V}
$$

**疊加**：
$$
V_o = V_{o1} + V_{o2} = 6 + 9 = 15 \text{ V}
$$

答：**Vₒ = 15V**。

## 二、源轉換（Source Transformation）

### 2.1 原理

一個**電壓源 Vs 串聯電阻 Rs** 可以等效為一個**電流源 Is 並聯電阻 Rp**，只要：

$$
I_s = \frac{V_s}{R_s}, \quad R_p = R_s
$$

反過來也行：

$$
V_s = I_s \cdot R_p, \quad R_s = R_p
$$

```
等效轉換：

  Vs ─[Rs]─         Is ↑  ─[Rp]─
  (串聯)        ↔       (並聯)
```

### 2.2 重要限制

- 只適用於**獨立源**
- 理想電壓源（Rs = 0）不能轉換
- 理想電流源（Rp = ∞）不能轉換

### 2.3 數值例題 ②

> **題目**：將 12V 電壓源串聯 4Ω 電阻轉換為電流源等效電路。

**解答**：

$$
I_s = \frac{V_s}{R_s} = \frac{12}{4} = 3 \text{ A}
$$

等效電路：**3A 電流源並聯 4Ω 電阻**

答：3A 電流源並聯 4Ω。

## 三、戴維寧定理（Thevenin's Theorem）

### 3.1 原理

任何由**獨立源、相依源和電阻**組成的線性兩端電路，對外部負載而言，都可以等效為一個**電壓源 V_th 串聯一個電阻 R_th**。

```
  ┌──────────────┐           ┌─────────────┐
  │              │           │             │
  │  複雜電路    ├─ a    ≡   │ Vth─[Rth]─ ├─ a
  │              │           │             │
  │              ├─ b        │             ├─ b
  └──────────────┘           └─────────────┘
```

### 3.2 完整步驟（SOP）

| 步驟 | 操作 | 說明 |
|------|------|------|
| ① | 把負載**拆掉**，標出 a、b 兩端 | 我們要對 a-b 求等效 |
| ② | 求**開路電壓 Voc** | 即 a-b 端斷路時的電壓差 Vₐ - V_b |
| ③ | 求**短路電流 Isc** | 即 a-b 端短路時流過的電流 |
| ④ | 求等效電阻 R_th = Voc / Isc | |
| ⑤ | 畫出等效電路：Vth = Voc，串聯 Rth | |

**替代求 Rth 的方法**（僅限無相依源的電路）：
- 關掉所有獨立源
- 直接從 a-b 端看進去計算等效電阻

### 3.3 數值例題 ③

> **題目**：求 a-b 兩端的戴維寧等效電路。
> - 32V 電壓源串聯 R₁ = 4Ω 後連到節點 a
> - 節點 a 和地之間有 R₂ = 12Ω
> - a 為正端，b 為地（GND）

**解答**：

**步驟 1**：拆掉負載，a-b 端開路。

**步驟 2**：求開路電壓 Voc

a-b 開路時，R₂ 上沒有電流流過（因為和開路端串聯），等等不對——R₂ 是從 a 到 GND（b），如果 a-b 開路，R₂ 仍然接在電路中。

讓我重新分析：32V 串聯 4Ω，然後接到 a 點。a 點和 b（GND）之間有 12Ω。開路是指在 a-b 端外部不接任何東西。

但 R₂ = 12Ω 已經從 a 到 b 了。所以 Voc 就是 R₂ 上的電壓（分壓器）：

$$
V_{oc} = 32 \times \frac{12}{4 + 12} = 32 \times \frac{12}{16} = 32 \times 0.75 = 24 \text{ V}
$$

**步驟 3**：求 Rth

關掉 32V 電壓源（短路），從 a-b 端看進去：

R₁ 和 R₂ 並聯：

$$
R_{th} = \frac{4 \times 12}{4 + 12} = \frac{48}{16} = 3 \text{ Ω}
$$

**戴維寧等效電路**：**24V 電壓源串聯 3Ω 電阻**。

答：**V_th = 24V，R_th = 3Ω**。

### 3.4 數值例題 ④

> **題目**：求 R_L 兩端的戴維寧等效電路。
> - 20V 電壓源、R₁ = 5Ω、R₂ = 10Ω、R₃ = 10Ω
> - 20V 串聯 R₁ 後到達節點 c
> - c 到 a 之間有 R₂ = 10Ω
> - c 到 b（GND）之間有 R₃ = 10Ω
> - R_L 接在 a-b 之間

**解答**：

**求 Voc**：

拆掉 R_L，a-b 開路。此時 R₂ 上沒有電流（因為 a 端開路，R₂ 的一端沒有通路）。

電流只流經 R₁ 和 R₃ 的迴路：

$$
I = \frac{20}{R_1 + R_3} = \frac{20}{5 + 10} = \frac{4}{3} \text{ A}
$$

節點 c 的電壓（對 GND）：

$$
V_c = 20 - I \times R_1 = 20 - \frac{4}{3} \times 5 = 20 - \frac{20}{3} = \frac{40}{3} \text{ V}
$$

因為 R₂ 上沒有電流，所以 Vₐ = V_c = 40/3 V

$$
V_{oc} = V_a - V_b = \frac{40}{3} - \frac{4}{3} \times 10
$$

等等，V_b = 0（GND），所以：

$$
V_{oc} = V_a = V_c = \frac{40}{3} \approx 13.33 \text{ V}
$$

不對，讓我重新想。R₂ 從 c 到 a，R₃ 從 c 到 GND(b)。a 是開路端。

V_a 的電壓 = V_c（因為 R₂ 上無電流，無壓降）

V_c = I × R₃ = (4/3) × 10 = 40/3 V

V_b = 0V

$$
V_{oc} = V_a - V_b = \frac{40}{3} - 0 = \frac{40}{3} \approx 13.33 \text{ V}
$$

**求 Rth**：

關掉 20V 電壓源（短路）。從 a-b 看進去：

R₂ 的一端接 a，另一端接 c。c 點同時連接 R₁ 和 R₃。R₁ 的另一端接地（因為電壓源短路），R₃ 的另一端也接地。

所以 R₁ 和 R₃ 並聯 = (5×10)/(5+10) = 50/15 = 10/3 Ω

再和 R₂ 串聯：Rth = 10 + 10/3 = 40/3 ≈ 13.33 Ω

**戴維寧等效**：**Vth = 40/3 V ≈ 13.33V，Rth = 40/3 Ω ≈ 13.33Ω**。

答：**V_th = 40/3 V，R_th = 40/3 Ω**。

## 四、諾頓定理（Norton's Theorem）

### 4.1 原理

任何線性兩端電路都可以等效為一個**電流源 I_N 並聯一個電阻 R_N**。

$$
I_N = I_{sc} = \frac{V_{th}}{R_{th}}, \quad R_N = R_{th}
$$

> 戴維寧和諾頓是**對偶關係**：
> - 戴維寧 = 電壓源串聯電阻
> - 諾頓 = 電流源並聯電阻
> - 兩者可以用源轉換互相轉換

### 4.2 數值例題 ⑤

> **題目**：承例題 ③，將戴維寧等效轉換為諾頓等效。

**解答**：

已知 V_th = 24V，R_th = 3Ω

$$
I_N = \frac{V_{th}}{R_{th}} = \frac{24}{3} = 8 \text{ A}
$$
$$
R_N = R_{th} = 3 \text{ Ω}
$$

**諾頓等效電路**：**8A 電流源並聯 3Ω 電阻**。

## 五、最大功率轉移定理（Maximum Power Transfer）

### 5.1 原理

當負載電阻等於戴維寧等效電阻時，負載能獲得**最大功率**：

$$
R_L = R_{th}
$$

此時最大功率為：

$$
P_{max} = \frac{V_{th}^2}{4R_{th}}
$$

### 5.2 推導

負載功率：

$$
P_L = I^2 R_L = \left(\frac{V_{th}}{R_{th} + R_L}\right)^2 R_L
$$

對 R_L 微分令其等於零：

$$
\frac{dP_L}{dR_L} = 0
$$

解出 R_L = R_th。

代入：

$$
P_{max} = \left(\frac{V_{th}}{R_{th} + R_{th}}\right)^2 R_{th} = \frac{V_{th}^2}{4R_{th}}
$$

### 5.3 數值例題 ⑥

> **題目**：承例題 ③ 的戴維寧等效（V_th = 24V，R_th = 3Ω），求能獲得最大功率的負載電阻值和最大功率。

**解答**：

$$
R_L = R_{th} = 3 \text{ Ω}
$$

$$
P_{max} = \frac{V_{th}^2}{4R_{th}} = \frac{24^2}{4 \times 3} = \frac{576}{12} = 48 \text{ W}
$$

答：**R_L = 3Ω，P_max = 48W**。

### 5.4 效率的權衡

當 R_L = R_th 時，效率只有 50%（一半功率被 R_th 消耗）。

| 目標 | 選擇 |
|------|------|
| 最大功率轉移 | R_L = R_th（效率 50%） |
| 最高效率 | R_L >> R_th（功率小但效率高） |

> 實際應用：通訊系統重視最大功率轉移（訊號微弱，每一點功率都很珍貴）；電力系統重視效率（不能浪費一半的電）。

## 六、題型鑑別指南

| 看到什麼 | 用什麼定理 |
|----------|-----------|
| 「多個獨立源，求某支路的電壓/電流」 | 疊加定理 |
| 「把電壓源轉換成電流源」或反過來 | 源轉換 |
| 「求等效電路」 | 戴維寧或諾頓 |
| 「求開路電壓和等效電阻」 | 戴維寧 |
| 「求短路電流和等效電阻」 | 諾頓 |
| 「最大功率」、「最佳負載」 | 最大功率轉移 |
| 題目給了相依源 | 不能用簡化求 Rth，要用 Voc/Isc |
| 需要分析同一電路的多個不同負載 | 先求戴維寧等效 |

## 七、關鍵術語表

| 中文 | 英文 | 白話解釋 | 例子 |
|------|------|---------|------|
| 疊加定理 | Superposition | 多個電源的效果可以分開算再加起來 | 3 個電池的電路分 3 次算 |
| 源轉換 | Source Transformation | 電壓源串 R ↔ 電流源並 R | 12V 串 4Ω = 3A 並 4Ω |
| 戴維寧定理 | Thevenin's Theorem | 複雜電路 = 一個電壓源串一個電阻 | 打包成最簡形式 |
| 諾頓定理 | Norton's Theorem | 複雜電路 = 一個電流源並一個電阻 | 戴維寧的對偶版本 |
| 開路電壓 | Open-circuit Voltage (Voc) | 拔掉負載後端子間的電壓 | 不接任何東西時量到的電壓 |
| 短路電流 | Short-circuit Current (Isc) | 端子直接連通時流過的電流 | 用導線把 a-b 短接 |
| 等效電阻 | Thevenin Resistance (Rth) | Voc/Isc 或關掉獨立源後看到的電阻 | 從端子往裡看的電阻 |
| 最大功率轉移 | Maximum Power Transfer | R_L = Rth 時負載獲得最大功率 | 通訊系統中的阻抗匹配 |
| 線性電路 | Linear Circuit | 滿足疊加和齊次性的電路 | 只含線性電阻和獨立源 |
| 獨立源 | Independent Source | 輸出不受電路其他部分影響 | 電池、直流電源供應器 |
| 相依源 | Dependent Source | 輸出由電路中其他電壓或電流控制 | 電晶體模型中常見 |
| 對偶 | Duality | 電壓↔電流、串聯↔並聯 的對應關係 | 戴維寧與諾頓互為對偶 |

## ✅ 自我檢測

### 題目 1
使用疊加定理時，「關掉」一個 5V 電壓源和「關掉」一個 2A 電流源，分別要怎麼做？

<details>
<summary>點擊查看解答</summary>

- 關掉 5V 電壓源：用**短路**取代（直接用導線連通）
- 關掉 2A 電流源：用**斷路**取代（把那條支路剪斷）

理由：理想電壓源內阻為零（短路），理想電流源內阻為無窮大（斷路）。

</details>

### 題目 2
一個戴維寧等效電路 Vth = 10V、Rth = 5Ω。求諾頓等效電路。

<details>
<summary>點擊查看解答</summary>

$$I_N = \frac{V_{th}}{R_{th}} = \frac{10}{5} = 2 \text{ A}$$

$$R_N = R_{th} = 5 \text{ Ω}$$

諾頓等效：**2A 電流源並聯 5Ω 電阻**。

</details>

### 題目 3
Vth = 20V、Rth = 10Ω 的電路，接上負載 R_L = 10Ω。
(a) 負載電流？(b) 負載功率？(c) 這是最大功率嗎？

<details>
<summary>點擊查看解答</summary>

(a) I = Vth/(Rth + R_L) = 20/(10+10) = 1A

(b) P_L = I² × R_L = 1² × 10 = 10 W

(c) 是的！因為 R_L = Rth = 10Ω，符合最大功率轉移條件。
驗算：P_max = Vth²/(4Rth) = 400/40 = 10W ✓

</details>

### 題目 4
為什麼疊加定理不能用來直接計算功率？

<details>
<summary>點擊查看解答</summary>

因為功率是**非線性**函數：P = I²R 或 P = V²/R

疊加定理只適用於線性關係（電壓、電流）。

例如：(I₁ + I₂)²R ≠ I₁²R + I₂²R

正確做法：先用疊加定理求出總電壓或總電流，再算功率。

</details>

### 題目 5
一個電路含有相依源。求 Rth 時，能否直接關掉所有源然後計算電阻？

<details>
<summary>點擊查看解答</summary>

**不能！** 相依源不能被關掉。

正確做法：
- 用 Rth = Voc / Isc
- 或者：關掉所有**獨立源**，在 a-b 端外加一個測試電壓 Vt（或測試電流 It），求 Rth = Vt/It

</details>
