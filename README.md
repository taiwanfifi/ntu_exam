## 電機工程 Phase 1（已完成 ✅）

### 電路學（60 題 + 8 篇教學）

| 類別 | 內容 | 說明 |
|------|------|------|
| 題庫與詳解 | 60 題 | 串並聯→節點/網目法→電路定理→Op-Amp→一階/二階→交流功率→頻率響應→拉普拉斯→雙埠網路 |
| 教學講義 | 8 篇 | circuit_01～circuit_08，從基本定律到拉普拉斯轉換 |

**教學講義目錄**：
1. 基本定律與電阻電路（KVL/KCL/Ohm's Law）
2. 電路分析方法（節點電壓法/網目電流法）
3. 電路定理（疊加/戴維寧/諾頓/最大功率轉移）
4. 運算放大器（反相/非反相/加法器/差動/積分微分）
5. 一階電路（RC/RL 自然響應與步階響應）
6. 二階電路（RLC 過阻尼/臨界/欠阻尼）
7. 交流穩態與功率（相量/阻抗/功率因數/三相）
8. 頻率響應與拉普拉斯（波德圖/濾波器/s域分析/雙埠網路）

## 教學設計原則

本系統的每個科目都遵循相同的教學設計：

1. **零基礎入門** — 每個概念用生活化語言從零解釋
2. **術語表** — 中英對照 + 白話解釋 + 具體例子
3. **完整推導** — 不跳步，從公式推導到數值計算
4. **題型鑑別** — 遇到題目知道該用什麼方法
5. **自我檢測** — 基礎→進階漸進式練習（含摺疊答案）
