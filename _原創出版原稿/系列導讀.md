# 系列導讀：三本小說，三門課

> *「如果你認真讀完這三本小說，你等於上完了統計學、機率論、演算法三門課。差別只是——你的教授是白若嵐、貝葉、和米亞。」*

---

## 這個系列在做什麼

一個台大電機系的研究生，在準備博士資格考的過程中，把三個最枯燥的學科——統計學、機率論、演算法——寫成了三部小說。

不是「小說裡穿插一點數學公式」。是把整個台大資格考的三科核心課程，用三個完整的故事從頭教到尾。

每本小說都有獨立的附錄教學系統：完整的數學定義、程式實作、視覺化圖表——但你不需要翻附錄也能讀懂故事。數學藏在劇情裡，你讀完一個案件、經歷一場危機，回頭才發現自己已經學會了貝氏定理。

### 《機率預言師》— 機率論（研究所等級）

> 60 章 ｜ 近未來賽博龐克 × 貝氏哲學 ｜ 門檻最高，哲學最深

貝氏校準師貝葉活在一個所有人的決策都被機率模型預測的世界。60 章，每一章都有對應的機率概念與精確的數學數字。

**正文涵蓋 30+ 個機率核心概念，按五篇分布：**

| 篇章 | 主題 | 涵蓋概念 |
|------|------|----------|
| **校準篇** Ch1-12 | 貝氏基礎 | Beta 分布、Poisson 分布、最大似然估計 (MLE)、似然函數、似然比檢定、貝氏定理完整公式 |
| **雜訊篇** Ch13-24 | 分布與更新 | Normal / Exponential / Gamma / Beta 四大家族、共軛先驗、核密度估計 (KDE)、聯合分布、邊際分布 |
| **信號篇** Ch25-36 | 期望值與不等式 | LOTUS 定理、變異數、大數法則 (LLN)、中央極限定理 (CLT)、Markov 不等式、Chebyshev 不等式、5σ 門檻、動差生成函數 (MGF) |
| **檢定篇** Ch37-48 | 假設檢定 | H₀ vs H₁、Type I / Type II Error、檢定力 (Power)、信賴區間、χ² 卡方檢定、p-value |
| **隨機篇** Ch49-60 | 馬可夫與漫步 | 馬可夫鏈、轉移矩陣、遍歷性定理、隨機漫步、Poisson 過程、指數分布 |

**這些不是背景裝飾。** 每個概念都綁定劇情——白奶奶四十年的蔥油餅攤是先驗信念的化身、「後驗是我們的」紙條是貝氏定理寫成的情書、柏朗用 overfitting 建造的預測帝國最終被自己的數學邏輯擊潰。

**附錄「機率教室」：** 5 篇系統性教學 + 25 張視覺化圖表，所有數值範例都取自小說場景（Beta(72,28)、λ=0.58、χ²=47.3、|λ₂|=0.847）。

## 閱讀順序建議

### 你是誰，決定你從哪本開始。

---

### 路線 A：準備考試的人（統計 / 機率 / 演算法）

**白噪音 → 機率預言師 → 記憶駭客**

從應用到理論，從直覺到抽象。

- **白噪音**是暖身。統計學是機率的應用層，概念最直覺——你看得到數據、畫得出圖。用懸疑劇的節奏把「為什麼要學統計」內化。讀完你會記得「常態分布是堡壘」、「相關不等於因果但有些真相是一把刀」，這些比課本定義更容易在考試時被你的大腦調出來。
- **機率預言師**是核心。機率是統計的理論基礎，也是演算法分析的前置知識。「先驗是偏見，似然是遭遇，後驗是成長」——如果讀完這本還不懂貝氏定理在幹嘛，大概沒有東西能教你了。
- **記憶駭客**是終局。演算法需要前兩本的數學成熟度，而且四卷 320 章的篇幅，需要前兩本養出的閱讀慣性。

**考試層面你會得到什麼：**
- 資格考最難的不是計算，是理解一個定理的動機。這三本做的事就是把動機寫成故事。
- 白噪音讓你記住統計工具的「使用情境」
- 機率預言師讓你理解推導背後的「哲學邏輯」
- 記憶駭客讓你感受演算法的「設計思維」——為什麼用 Segment Tree 而不是暴力解，不是因為比較快，是因為問題的結構長那個樣子

---

### 路線 B：資工系學生

**記憶駭客 → 機率預言師 → 白噪音**

從最熟悉的領域開始，重新理解你以為你已經懂的東西。

- **記憶駭客**不是在教你演算法——你已經會了。它是讓你經歷「原來我每天寫的 code 可以這樣被理解」的震撼。XOR 不只是位元運算，是兩個人「只有不同的部分才能還原彼此」的愛情。Segment Tree 不只是區間查詢，是極權體制的隱喻。
- **機率預言師**填補資工學生最常見的弱點——統計與機率的直覺。很多資工人可以推導 MLE 但不知道它在「說什麼」。這本用故事把直覺補上。
- **白噪音**最後讀。它最「非技術」，是社會派故事。讓你體驗「技術被用在人身上」的倫理衝擊——當你有能力用數據分析一個人的行為模式時，你的責任邊界在哪裡。

**資工系學生會得到什麼：**
- 記憶駭客：你寫的每一行 code 都有哲學意義。選擇 O(n log n) 而非 O(n²) 不只是效率，是「用有限資源面對無限問題」的人生態度
- 機率預言師：ML 課教你調參數，這本教你為什麼 prior matters——不只在模型裡，在人生的每一個決策裡
- 白噪音：數據會說謊，但也會替沉默的人說話

---

### 路線 C：一般讀者（非理工背景、純粹想看故事）

**白噪音 → 記憶駭客 → 機率預言師**

從最好看的開始，讓故事帶著你走進數學。

- **白噪音**最像一般人會讀的小說——懸疑、推理、有 CP 感。數學都藏在「諮商手記」裡，正文就是一個好看的犯罪故事。門檻最低，但讀完你會發現自己不知不覺懂了什麼是常態分布。
- **記憶駭客**的情感後座力最強。就算你不懂演算法，米亞的 0.4 秒延遲、阿嬤三十年每天多煮一碗不賣的鹹豆漿、語青消散時掌心 21.8 度的微溫——這些不需要技術背景就能被打動。
- **機率預言師**門檻最高，但如果你被前兩本圈粉了，你會有動力讀完。而且讀完這本，你會成為那種「在蔥油餅攤前突然想到貝氏定理」的人。

**一般讀者會得到什麼：**
- 理科不是你以為的那種冷冰冰的東西
- 數學是一種看世界的方式，不是折磨你的考試工具
- 白噪音教你：數據會說謊，但也會替沉默的人說話
- 記憶駭客教你：遺忘不是失去，記憶不是擁有
- 機率預言師教你：沒有人是客觀的，但承認自己的偏見就是成長的開始
