## 零基礎完全入門 — 從「跨國事務該用哪國法律」開始

---

# 第一部分：什麼是國際私法？

## 一、開場白 — 跨國生活中的法律選擇

當一個台灣人和一個日本公民結婚，要適用哪個國家的婚姻法？當一家台灣公司控告一家德國公司，要在哪個國家的法院打官司？國際私法就是回答這些問題的法律。

想像一下這個場景：

> 小明（台灣人）在法國留學時認識了小花（日本人）。他們在法國舉行婚禮，之後搬到德國定居。現在他們想離婚。請問：要在哪個國家的法院打官司？要適用哪個國家的離婚法？

這個看似簡單的問題，其實涉及國際私法最核心的兩大問題：
1. **國際裁判管轄** — 哪個國家的法院有權審理？
2. **準據法的決定** — 適用哪個國家的法律？

## 二、國際私法不是「國際的法」

### 1. 名稱的誤導

「國際私法」這個名字很容易讓人以為它是「國家之間的私法」。

**錯誤理解**：國際私法是國際法的一部分。

**正確理解**：國際私法是**各國國內法**的一部分。它是每個國家自己的法律，用來決定在涉外案件中應該適用哪個國家的法律。

### 2. 和國際公法的區別

| 比較 | 國際公法 | 國際私法 |
|-----|---------|---------|
| 處理的問題 | 國家之間的關係 | 跨國私人事務適用哪國法 |
| 法律性質 | 國際法 | 國內法 |
| 主要主體 | 國家 | 私人（自然人、法人） |
| 淵源 | 條約、國際習慣 | 各國立法 |
| 典型問題 | 領土爭端、人權保護 | 跨國婚姻、跨國買賣 |

### 3. 各國名稱

| 國家/地區 | 名稱 |
|----------|------|
| 台灣 | 國際私法 / 涉外民事法律適用法 |
| 中國大陸 | 涉外民事關係法律適用法 |
| 英美法系 | Conflict of Laws（法律衝突） |
| 歐陸法系 | Private International Law（國際私法） |

## 三、國際私法的三大任務

國際私法主要處理三個問題：

```
國際私法
├── 1. 國際裁判管轄 — 哪個國家的法院有權審理？
├── 2. 準據法的決定 — 適用哪個國家的實體法？
└── 3. 外國判決的承認與執行 — 外國法院的判決在本國有效嗎？
```

# 第二部分：涉外民事法律適用法（台灣的國際私法）

## 第一章：概述

### 一、法律依據

台灣國際私法的主要法律是**涉外民事法律適用法**（簡稱「涉民法」）。

現行的涉外民事法律適用法於2010年全文修正，2011年5月26日施行。

### 二、適用前提 — 涉外因素

涉外民事法律適用法只適用於有「涉外因素」的案件。

**什麼是涉外因素？**

就是案件和外國有連結。可能是：
- 當事人是外國人
- 行為發生在外國
- 標的物在外國
- 契約約定適用外國法
- 其他與外國有關的因素

**情境**：

> 兩個台灣人在台灣結婚，住在台灣。這個案件有涉外因素嗎？

**答案**：沒有。這是純粹的國內案件，直接適用台灣民法。

> 一個台灣人和一個美國人在台灣結婚，住在台灣。有涉外因素嗎？

**答案**：有。因為一方當事人是外國人。

### 三、涉民法的基本結構

```
涉外民事法律適用法
├── 第一章 總則（第1-8條）
│   ├── 法律之適用
│   ├── 反致
│   ├── 公序良俗
│   └── 外國法查明
├── 第二章 行為能力及監護（第9-14條）
├── 第三章 債（第15-24條）
│   ├── 法律行為方式
│   ├── 代理
│   ├── 契約
│   └── 侵權行為
├── 第四章 物權（第25-27條）
├── 第五章 婚姻（第28-36條）
│   ├── 婚約
│   ├── 結婚要件
│   ├── 婚姻效力
│   ├── 夫妻財產制
│   └── 離婚
├── 第六章 父母子女（第37-43條）
├── 第七章 監護及輔助（第44-47條）
├── 第八章 繼承（第48-49條）
└── 第九章 附則（第50-63條）
```

## 第二章：國際私法的基本概念

### 一、衝突規則（Choice-of-Law Rules）

#### 1. 什麼是衝突規則？

**白話解釋**：衝突規則就是「選法規則」— 它告訴你在特定的案件類型中，應該適用哪個國家的法律。

**比喻**：衝突規則就像GPS導航。你輸入目的地（案件類型），GPS告訴你走哪條路（適用哪國法）。

#### 2. 衝突規則的結構

每一條衝突規則都有三個部分：

```
法律關係（範疇） + 連結因素 → 準據法
```

**例子**：

> 涉民法第48條：「繼承，依被繼承人死亡時之本國法。」

- **法律關係**：繼承
- **連結因素**：被繼承人死亡時的國籍
- **準據法**：被繼承人的本國法

### 二、連結因素（Connecting Factor）

#### 1. 什麼是連結因素？

連結因素就是「把案件和某個國家法律連起來的橋樑」。

#### 2. 常見的連結因素

| 連結因素 | 英文 | 說明 | 常用領域 |
|---------|------|------|---------|
| 國籍 | Nationality | 當事人的國籍 | 身分關係（婚姻、繼承） |
| 住所 | Domicile | 當事人的住所地 | 身分關係（英美法系偏好） |
| 慣居地 | Habitual Residence | 通常居住地 | 現代國際私法趨勢 |
| 行為地 | Place of Acting | 行為發生的地方 | 法律行為方式、侵權 |
| 物之所在地 | Situs | 標的物所在地 | 物權 |
| 當事人意思 | Party Autonomy | 當事人自己選擇 | 契約 |
| 法院地 | Lex Fori | 審理案件的法院所在地 | 程序法 |
| 最密切關係地 | Closest Connection | 與案件關係最密切的地方 | 補充性連結因素 |

#### 3. 連結因素的具體運用

**情境**：

> 一個美國人在台灣開車撞傷一個法國人。

- 要確定美國人是否有賠償能力 → 看**國籍**（美國法）→ 行為能力
- 要確定是否構成侵權 → 看**行為地**（台灣法）→ 侵權行為
- 如果訴訟在台灣進行 → 程序法適用**法院地法**（台灣法）

### 三、準據法（Applicable Law / Governing Law）

#### 1. 什麼是準據法？

準據法就是「最終適用來解決案件的那個國家的法律」。

**白話解釋**：經過國際私法的「GPS導航」後，你最終到達的目的地就是準據法。

#### 2. 準據法 vs. 衝突規則

| | 衝突規則 | 準據法 |
|--|---------|-------|
| 功能 | 選法 | 被選用的法 |
| 性質 | 法院地的國內法 | 可能是本國法或外國法 |
| 例子 | 涉民法第48條 | 依第48條選出的某國繼承法 |

## 第三章：定性問題（Characterization / Classification）

### 一、什麼是定性？

**情境**：

> 一個英國人在台灣過世，留下一棟在台灣的房子。遺囑中把房子留給情人。

問題：遺囑把不動產留給情人，這是「繼承」問題還是「物權」問題？

- 如果定性為「繼承」問題 → 適用被繼承人本國法（英國法）
- 如果定性為「物權」問題 → 適用物之所在地法（台灣法）

不同的定性可能導致適用不同的法律，結果可能完全不同。

**白話解釋**：定性就是「給問題貼標籤」— 你把問題歸類到哪個法律類別，就適用哪個類別的衝突規則。

### 二、定性的困難

定性之所以困難，是因為不同國家的法律分類可能不同。

**經典案例 — 馬爾他夫妻案**：

一對在馬爾他結婚的夫妻，丈夫死後，妻子主張對丈夫在法國的不動產有權利。法國法院面臨的問題是：妻子的權利是「夫妻財產制」問題還是「繼承」問題？

- 馬爾他法認為這是夫妻財產制的問題
- 法國法認為這可能是繼承的問題
- 不同定性導致不同結果

### 三、定性的標準

#### 1. 依法院地法定性（法院地法說）

用法院地（審理案件的法院所在國）的法律來進行定性。

**優點**：法官最熟悉自己國家的法律分類。
**缺點**：可能不公平 — 用A國的法律概念去套外國的法律關係。

#### 2. 依準據法定性（準據法說）

用準據法所屬國家的法律來定性。

**問題**：這是一個邏輯上的悖論 — 定性的目的就是要找準據法，你不能用還沒找到的準據法來定性。

#### 3. 自主定性（比較法說）

不依任何特定國家的法律，而是從比較法的角度，用國際私法自主的概念來定性。

**台灣的做法**：實務上主要採用**法院地法說**，以台灣法的法律概念來定性。

## 第四章：反致（Renvoi）

### 一、什麼是反致？

**這是國際私法中最容易讓初學者困惑的概念。**

**情境**：

> 一個英國人（住所在法國）在台灣過世。台灣法院審理他的遺產繼承。

台灣的涉民法說：「繼承適用被繼承人的本國法。」→ 應適用**英國法**。

但是英國的國際私法說：「繼承適用被繼承人住所地法。」→ 應適用**法國法**。

法國的國際私法說：「繼承適用被繼承人本國法。」→ 應適用**英國法**。

你看到了嗎？法律一直在「互踢皮球」：
- 台灣說用英國法
- 英國說用法國法
- 法國說用英國法
- 英國又說用法國法
- ……（無限循環）

### 二、反致的類型

#### 1. 直接反致（Remission / 狹義反致）

A國的衝突規則指向B國法，B國的衝突規則指回A國法。

```
台灣衝突規則 → 英國法
英國衝突規則 → 台灣法 ← 直接反致
```

結果：台灣法院最終適用台灣法。

#### 2. 間接反致（Transmission / 轉致）

A國的衝突規則指向B國法，B國的衝突規則指向C國法。

```
台灣衝突規則 → 英國法
英國衝突規則 → 法國法 ← 間接反致（轉致）
```

結果：台灣法院最終適用法國法。

#### 3. 雙重反致

A國不只適用B國的衝突規則，還假設自己站在B國法官的立場，看B國法官會怎麼判。

### 三、台灣法的規定

**涉民法第6條**：

> 依本法適用當事人本國法時，如依其本國法就該法律關係須依其他法律而定者，應適用該其他法律；依該其他法律更應適用其他法律者亦同。但依該其他法律，應適用中華民國法律者，適用中華民國法律。

**白話翻譯**：

1. 台灣衝突規則指向A國法 →
2. 如果A國衝突規則指向B國法 → 適用B國法（轉致）
3. 如果B國衝突規則又指向C國法 → 適用C國法（繼續轉致）
4. 但如果中途有任何國家的衝突規則指回台灣法 → 就適用台灣法（反致回台灣就停止）

**注意**：反致只在適用「當事人本國法」時才發生。如果連結因素是行為地、物之所在地等，通常不適用反致。

### 四、反致的實際意義

**支持反致**：
- 增加法院適用自己法律的機會（法官比較熟悉）
- 可能促進判決結果的一致性

**反對反致**：
- 邏輯上有問題（互踢皮球）
- 增加法律適用的不確定性
- 使法律選擇更複雜

## 第五章：公序良俗條款（Public Policy Exception）

### 一、什麼是公序良俗條款？

**白話解釋**：即使衝突規則指向外國法，如果那個外國法的內容嚴重違反我們國家的基本價值觀，法院可以拒絕適用。

**情境**：

> 某中東國家的法律允許一夫多妻。一個台灣人和一個該國國民結婚。該國國民在該國已有一位配偶。如果依衝突規則應適用該國法律，台灣法院會承認這段重婚關係嗎？

**答案**：不會。因為一夫多妻違反台灣的公序良俗。

### 二、法律依據

**涉民法第8條**：

> 依本法適用外國法時，如其適用之結果有背於中華民國公共秩序或善良風俗者，不適用之。

### 三、公序良俗條款的適用要件

1. **不是審查外國法本身**，而是審查**適用外國法的結果**
2. 必須是**嚴重違反**台灣的基本價值觀（不是一般的不同意見）
3. 必須和台灣有**足夠的關聯**

### 四、不適用外國法後怎麼辦？

如果因為公序良俗條款而不適用外國法，通常**改適用法院地法**（台灣法）。

### 五、公序良俗的具體案例

| 情形 | 可能觸發公序良俗？ |
|-----|-----------------|
| 一夫多妻 | 是 |
| 童婚 | 是 |
| 宗教離婚（只有丈夫可以單方面離婚） | 是 |
| 外國法的損害賠償金額和台灣差很多 | 通常不是（只是金額差異不足以構成） |
| 外國法不承認女性繼承權 | 是 |
| 外國法的消滅時效比台灣短 | 通常不是 |

## 第六章：外國法的查明

### 一、外國法在訴訟中的地位

#### 1. 外國法是「法律」還是「事實」？

這是一個很重要的問題：

| 立場 | 說明 | 後果 |
|-----|------|-----|
| 外國法是「法律」 | 法官有義務自己去查 | 法官依職權調查 |
| 外國法是「事實」 | 當事人有義務舉證 | 當事人負舉證責任 |

**台灣的做法**：涉民法第2條規定，法院應依職權適用外國法，但可以請當事人協力提出外國法的內容。

### 二、查不到外國法怎麼辦？

如果法院怎麼查也查不到外國法的內容（例如那個國家太小、法律太冷門），怎麼辦？

**可能的做法**：
1. 適用與該外國法最相近的其他國家法律
2. 適用法院地法（台灣法）
3. 適用一般法律原則

實務上最常見的做法是改適用台灣法。

# 第三部分：各論 — 各類法律關係的準據法

## 第一章：行為能力的準據法

### 一、基本規則

**涉民法第9條**：

> 人之行為能力，依其本國法。

**白話解釋**：一個人有沒有行為能力（能不能自己簽契約），看他是哪國人，就適用哪國的法律。

### 二、為什麼用「本國法」？

因為行為能力涉及一個人的法律地位，屬於「身分」問題。身分問題通常適用當事人的本國法（屬人法）。

### 三、交易保護的例外

**涉民法第9條第2項**：

> 有行為能力人之行為能力，不因其本國法變更而受影響。

**涉民法第10條**：

> 關於由法律行為而生之債，依行為地法有行為能力者，視為有行為能力。

**白話解釋**：

> 一個17歲的日本人在台灣買手機。依日本法，17歲是未成年人，沒有完全行為能力。但依台灣法，18歲才是成年（2023年修法後）。

如果這個日本人在台灣做的法律行為，依台灣法（行為地法）有行為能力，那就視為有行為能力。這是為了保護交易安全。

**但**：這個例外只適用於**由法律行為而生之債**（例如買賣、租賃），不適用於身分行為（例如結婚）。

## 第二章：法律行為方式的準據法

### 一、基本規則

**涉民法第15條**：

> 法律行為之方式，依該行為所應適用之法律。但依行為地法所定之方式者，亦為有效。

**白話解釋**：
- **原則**：法律行為的形式（要不要書面、要不要公證），適用該行為的準據法。
- **例外**：如果你按照行為地法的形式要求做了，也算有效。

### 二、為什麼有這個例外？

**情境**：

> 一個台灣人和一個法國人在日本簽訂買賣契約。依準據法（假設是法國法）契約需要公證。但他們在日本，來不及找法國的公證人。

如果按照日本法的形式要求（假設日本法不需要公證），契約也是有效的。這是為了方便當事人 — 你不可能要求在日本的人按照法國的形式來簽約。

### 三、兩個選擇

法律行為的方式有兩個選擇，符合其中一個就有效（**擇一適用、從寬認定**）：
1. 準據法所定的方式
2. 行為地法所定的方式

## 第三章：契約的準據法

### 一、當事人意思自主原則（Party Autonomy）

#### 1. 基本規則

**涉民法第20條**：

> 法律行為發生債之關係者，其成立及效力，依當事人意思定其應適用之法律。

**白話解釋**：契約要適用哪國法，由當事人自己約定。這叫做「當事人意思自主原則」，是國際私法中最重要的原則之一。

#### 2. 為什麼允許當事人自己選法律？

**比喻**：你去餐廳吃飯，你可以自己選要吃什麼。契約是你自己的事情，你當然可以自己選要適用哪國法。

#### 3. 選法的方式

- **明示選擇**：在契約中明確寫出「本契約適用某國法律」
- **默示選擇**：雖然沒有明確寫出，但從契約的內容和相關情況可以推斷當事人的意思

#### 4. 選法的限制

當事人的選法自由不是絕對的：
- 不能選擇與契約完全無關的國家的法律
- 不能以選法的方式規避強行法規
- 消費者契約和勞動契約有特別保護（限制雇主/企業的選法自由）

### 二、最密切關係原則

#### 1. 當事人沒有選法怎麼辦？

**涉民法第20條第2項**：

> 當事人無明示之意思或其明示之意思依所定應適用之法律無效時，依關係最切之法律。

**白話解釋**：如果當事人沒有約定適用哪國法（或約定無效），就看這個契約和哪個國家的關係最密切，就適用那個國家的法律。

#### 2. 怎麼判斷「最密切關係」？

涉民法第20條第3項提供了一些推定：

| 契約類型 | 推定最密切關係地 |
|---------|-----------------|
| 一般契約 | 特徵履行義務人（characteristic performance）的住所地或營業地 |
| 不動產相關 | 不動產所在地 |
| 運送契約 | 運送人的營業地 |

#### 3. 什麼是「特徵履行」？

**情境**：

> 台灣公司向日本公司購買汽車零件。台灣公司付錢，日本公司交貨。

哪一方是「特徵履行」方？

付錢每個契約都一樣（不具特徵性），但交貨是買賣契約的核心特徵。所以**日本公司（交貨方）是特徵履行方**。推定最密切關係地是日本公司的營業地（日本）。

| 契約類型 | 特徵履行方 |
|---------|-----------|
| 買賣契約 | 出賣人（交貨方） |
| 租賃契約 | 出租人 |
| 委任契約 | 受任人 |
| 借貸契約 | 貸與人 |
| 保險契約 | 保險人 |

### 三、消費者契約的特別保護

#### 1. 為什麼要特別保護消費者？

因為消費者和企業的談判地位不對等。如果允許企業完全自由選法，企業可能會選擇對消費者最不利的法律。

#### 2. 涉民法第21條

> 關於由消費者保護法所生之債，當事人雖得依前條規定選擇應適用之法律，但消費者之本國法中之強制規定，仍應適用之。

**白話解釋**：企業可以在契約中約定適用某國法律，但消費者本國法中保護消費者的強行規定（例如解約權、保固義務），還是要適用。

## 第四章：侵權行為的準據法

### 一、基本規則

**涉民法第22條**：

> 關於由侵權行為而生之債，依侵權行為地法。但另有關係最切之法律者，依該法律。

**白話解釋**：
- **原則**：在哪裡出車禍，就適用那裡的法律。
- **例外**：如果有其他國家的法律和案件的關係更密切，就用那個法律。

### 二、侵權行為地的確定

**問題**：如果行為地和結果地不同，哪個算「侵權行為地」？

**情境**：

> A在台灣架設網站發布誹謗B的內容。B住在日本，在日本看到這些內容，名譽受損。

- 行為地 = 台灣（架設網站的地方）
- 結果地 = 日本（損害發生的地方）

**台灣的做法**：行為地和結果地都算侵權行為地。如果兩者不同，**適用對被害人最有利的法律**。

### 三、特殊侵權行為

#### 1. 商品責任

**涉民法第23條**：

> 關於由商品之通常使用或消費所生之損害，依本條之規定。

商品責任可能適用：
- 損害發生地法
- 商品取得地法
- 商品製造地法

但要考慮製造人能否合理預見商品在該地銷售。

#### 2. 不當得利

**涉民法第24條**：

> 關於由不當得利而生之債，依其利益之受領地法。但不當得利係因給付而發生者，依該給付所由發生之法律關係所應適用之法律。

**白話解釋**：
- 一般不當得利 → 受領利益的地方的法律
- 因為給付而產生的不當得利 → 原來那個給付的法律關係的準據法

## 第五章：物權的準據法

### 一、基本規則

**涉民法第25條**：

> 關於物權，依物之所在地法。

**白話解釋**：東西在哪裡，就適用那裡的法律。這叫做**物之所在地法原則 (Lex Rei Sitae)**。

### 二、為什麼用物之所在地法？

理由很簡單：
1. 物之所在地國對該物有實際管控力
2. 第三人交易安全（第三人通常會查當地法律）
3. 物權有排他性，需要統一適用一個法律

### 三、特殊問題

#### 1. 動產的運送中（Transit）

**情境**：

> 一批貨物從台灣運往德國，船還在海上時，買方在日本把貨物轉賣給第三人。這批貨物的所有權移轉適用哪國法？

這是物權準據法中的難題。貨物在運送中不斷移動，「物之所在地」一直在變。

**解決方法**：
- 適用**目的地法**
- 或適用**送出地法**
- 台灣涉民法第25條沒有明確規定，需要由法院根據具體情況判斷

#### 2. 無體財產權（智慧財產權）

智慧財產權（專利、商標、著作權）因為沒有實體形態，不能用「物之所在地」來決定。

一般適用**保護國法**（要求保護的國家的法律）。

#### 3. 不動產

不動產一定在某個國家的領土內，適用物之所在地法沒有爭議。而且物之所在地法的適用範圍特別廣，涵蓋不動產的物權、契約方式等。

## 第六章：婚姻的準據法

### 一、婚約（訂婚）

**涉民法第28條**：

> 婚約之成立，依各該當事人之本國法。

**白話解釋**：訂婚有沒有效，分別看雙方各自的本國法。

### 二、結婚要件

#### 1. 實質要件

**涉民法第29條**：

> 婚姻之成立，依各該當事人之本國法。但結婚之方式依當事人一方之本國法或依舉行地法者，亦為有效。

**白話解釋**：
- 結婚的實質要件（年齡、有無配偶、近親限制等）→ 分別看雙方各自的本國法
- 結婚的形式要件（要不要登記、要不要證人）→ 雙方之一的本國法或舉行地法

**情境**：

> 一個台灣人（25歲）和一個菲律賓人（16歲）想在日本結婚。

分析：
- 台灣人的實質要件看台灣法：25歲，OK
- 菲律賓人的實質要件看菲律賓法：要看菲律賓法是否允許16歲結婚
- 形式要件可以看日本法（舉行地法）

#### 2. 為什麼用「各該當事人之本國法」？

因為結婚的實質要件涉及每個人自己的身分。你多大可以結婚、能不能和近親結婚，這是你自己本國法的問題。

### 三、婚姻的效力

**涉民法第30條**：

> 婚姻之效力，依夫妻共同之本國法；無共同之本國法時，依共同之住所地法；無共同之住所地法時，依與夫妻婚姻關係最切地之法律。

**白話解釋**：婚姻的效力（例如同居義務、忠誠義務）的準據法，用**階梯式**的方法決定：

```
第一步：夫妻有共同的國籍嗎？ → 有 → 適用共同本國法
    ↓ 沒有
第二步：夫妻有共同的住所地嗎？ → 有 → 適用共同住所地法
    ↓ 沒有
第三步：適用和婚姻關係最密切的法律
```

### 四、夫妻財產制

**涉民法第31條**：

> 夫妻財產制，夫妻以書面合意適用其一方之本國法或住所地法者，依其合意所定之法律。

> 夫妻未為前項之合意者，適用第二十條第二項之規定。

**白話解釋**：
- 夫妻可以書面約定適用哪國法（但只能選一方的本國法或住所地法）
- 沒有約定的話，適用最密切關係法

### 五、離婚

**涉民法第32條**：

> 離婚及其效力，依協議時或起訴時夫妻共同之本國法；無共同之本國法時，依共同之住所地法；無共同之住所地法時，依與夫妻婚姻關係最切地之法律。

**白話解釋**：離婚的準據法和婚姻效力的準據法一樣，用階梯式的方法。

**重要注意**：時間點是「協議時」或「起訴時」，不是結婚時。

## 第七章：繼承的準據法

### 一、基本規則

**涉民法第48條**：

> 繼承，依被繼承人死亡時之本國法。但依中華民國法律中華民國國民應為繼承人者，得就其在中華民國之遺產繼承之。

**白話解釋**：
- 繼承適用被繼承人**死亡時的本國法**
- 但如果被繼承人在台灣有遺產，而且依台灣法某些台灣國民應該是繼承人，這些人可以繼承在台灣的遺產

#### 2. 統一主義 vs. 分割主義

| 主義 | 說明 | 台灣的立場 |
|-----|------|-----------|
| 統一主義 | 所有遺產（不管動產不動產）適用同一法律 | 台灣採此 |
| 分割主義 | 動產適用被繼承人住所地法，不動產適用物之所在地法 | 英美法系採此 |

台灣採**統一主義**— 不管遺產在哪裡、是什麼類型，全部適用被繼承人死亡時的本國法。

### 二、遺囑

**涉民法第49條**：

> 遺囑之成立及效力，依成立時遺囑人之本國法。

> 遺囑之撤回，依撤回時遺囑人之本國法。

**白話解釋**：
- 遺囑有沒有效 → 看立遺囑時遺囑人的本國法
- 遺囑可不可以撤回 → 看撤回時遺囑人的本國法

## 第八章：代理的準據法

### 一、基本規則

**涉民法第16條**：

> 代理權係以法律行為授與者，其代理權之範圍、代理人之義務及權限內之行為是否對本人發生效力等，依代理權授與地法。

**白話解釋**：代理的問題（代理人可以做什麼、做了有沒有效），看代理權在哪裡授與的。

# 第四部分：外國判決的承認與執行

## 第一章：為什麼需要承認外國判決？

### 一、問題意識

**情境**：

> 台灣公司A在美國法院告日本公司B，美國法院判A勝訴，B要賠償100萬美元。但B的財產都在日本。A怎麼拿到錢？

A必須拿著美國法院的判決，去日本法院申請「承認與執行」。如果日本法院承認了美國法院的判決，A就可以在日本對B的財產強制執行。

### 二、基本原則

外國判決**不會自動**在其他國家生效。需要經過承認程序。

---

## 第二章：台灣法上外國判決的承認

### 一、法律依據

**民事訴訟法第402條**：

> 外國法院之確定判決，有下列各款情形之一者，不認其效力：
> 一、依中華民國之法律，外國法院無管轄權者。
> 二、敗訴之被告未應訴者。但開始訴訟之通知或命令已於相當時期在該國合法送達，或依中華民國法律上之協助送達者，不在此限。
> 三、判決之內容或訴訟程序，有背中華民國之公共秩序或善良風俗者。
> 四、無相互之承認者。

### 二、逐一解析各要件

#### 1. 外國法院有管轄權

**白話解釋**：你不能跑到一個完全無關的國家去告人，然後拿那個判決來台灣執行。

**判斷標準**：不是看外國法院自己認為有沒有管轄權，而是看**依台灣法**，那個外國法院有沒有管轄權。

#### 2. 被告有應訴機會

**白話解釋**：被告必須知道有人告他，有機會為自己辯護。如果被告根本不知道有這場官司就被判輸了，台灣不承認這個判決。

**例外**：如果已經合法送達（包括司法互助送達），但被告自己不來，那就沒話說了。

#### 3. 不違反公序良俗

**白話解釋**：外國判決的內容或訴訟程序不能嚴重違反台灣的基本價值觀。

**例子**：如果外國法院用刑求逼供取得的證據來判決，這個判決在台灣不被承認。

#### 4. 互惠原則

**白話解釋**：你承認我的判決，我才承認你的。

**問題**：互惠原則在實務上有爭議。有些學者認為應該放寬或廢除這個要件。

### 三、承認的效果

外國判決一旦被台灣法院承認：
- 具有和台灣確定判決相同的效力
- 可以在台灣強制執行
- 但需要經過「許可執行」的程序（強制執行法第4-1條）

# 第五部分：國際裁判管轄

## 第一章：什麼是國際裁判管轄？

### 一、問題意識

**情境**：

> 一家台灣公司和一家法國公司在網路上簽了一份契約。契約中沒有約定管轄法院。後來發生糾紛。台灣公司可以在台灣法院告法國公司嗎？法國公司可以在法國法院告台灣公司嗎？

國際裁判管轄就是要回答：**哪個國家的法院有權審理這個案件？**

### 二、和國內管轄的區別

| | 國內管轄 | 國際裁判管轄 |
|--|---------|-------------|
| 問題 | 哪個法院（地方法院）有管轄權？ | 哪個國家的法院有管轄權？ |
| 規範 | 民事訴訟法明確規定 | 沒有統一的國際法規範 |
| 決定 | 依民訴法的管轄規定 | 各國自行決定 |

### 三、台灣法的規定

台灣沒有專門的「國際裁判管轄法」。實務上，是**類推適用**民事訴訟法的管轄規定，再參考國際民事訴訟法的理論。

## 第二章：管轄權的基礎

### 一、一般管轄

**原則**：以被告的住所地為管轄依據（「原告就被告」原則）。

**白話解釋**：你要告人，原則上要到被告住的地方去告。這是為了保護被告，不能讓你隨便找一個對被告不方便的地方打官司。

### 二、特別管轄

| 管轄原因 | 說明 | 適用情形 |
|---------|------|---------|
| 財產所在地 | 被告在該國有財產 | 被告在台灣有不動產 |
| 侵權行為地 | 侵權行為發生在該國 | 車禍發生在台灣 |
| 契約履行地 | 契約應在該國履行 | 貨物應在台灣交付 |
| 不動產所在地 | 不動產在該國 | 不動產糾紛 |

### 三、合意管轄

當事人可以在契約中約定由某國法院管轄。

**限制**：
- 合意管轄不能違反專屬管轄的規定
- 不能濫用合意管轄（例如約定在對消費者極不方便的國家）

### 四、專屬管轄

某些案件只能由特定國家的法院審理：
- 不動產物權糾紛 → 不動產所在地國
- 公司設立、解散等 → 公司所在地國

### 五、應訴管轄

如果被告沒有提出管轄權異議，直接就實體問題進行答辯，視為同意該法院的管轄。

# 第六部分：國際私法的特殊問題

## 第一章：法律規避（Evasion of Law / Fraude a la loi）

### 一、什麼是法律規避？

**情境**：

> 台灣法律規定20歲才能結婚（假設）。甲（台灣人，19歲）想和乙（也是台灣人）結婚。甲故意先歸化為A國國籍（A國法律規定16歲就可以結婚），然後以A國國民的身分結婚。

甲是故意改變連結因素（國籍），以規避台灣法的限制。這就是「法律規避」。

### 二、法律規避的構成要件

1. 當事人**故意**改變連結因素
2. 改變的目的是為了**規避**不利法律的適用
3. 規避的是**強行法規**（不是任意法規）

### 三、法律規避的效果

如果認定構成法律規避，可以不承認連結因素的改變，仍然適用原來應該適用的法律。

**但**：法律規避的認定在實務上很困難。你怎麼證明一個人換國籍是為了規避法律，而不是真的想移民？

---

## 第二章：先決問題（Preliminary Question / Incidental Question）

### 一、什麼是先決問題？

**情境**：

> 一個英國人在台灣死亡，留下遺產。他有一個「配偶」— 一個在泰國依泰國法和他結婚的泰國人。現在問題是：這個「配偶」有沒有繼承權？

主要問題是「繼承」→ 準據法是英國法（被繼承人本國法）

但在解決繼承問題之前，必須先解決一個先決問題：「這段婚姻有效嗎？」

先決問題應該適用哪國的衝突規則？
- 用台灣（法院地）的衝突規則？
- 用英國（主要問題準據法所屬國）的衝突規則？

### 二、處理方法

#### 1. 依法院地的衝突規則（獨立連結說）

先決問題和主要問題一樣，都依法院地（台灣）的衝突規則來決定準據法。

#### 2. 依主要問題準據法所屬國的衝突規則（非獨立連結說）

先決問題依主要問題準據法所屬國（英國）的衝突規則來決定。

**台灣多數見解**：採**獨立連結說**，先決問題依台灣的衝突規則處理。

## 第三章：適應問題（Adaptation）

### 一、什麼是適應問題？

當同一個案件的不同部分適用不同國家的法律時，可能產生矛盾或漏洞。適應就是調和這些矛盾的方法。

**情境**：

> 夫死亡，留下遺產。夫妻財產制依A國法（共同財產制），繼承依B國法。A國法規定妻子可以取得共同財產的一半，B國法基於共同財產制已經保障了妻子，所以繼承法中妻子的繼承份額較低。

但如果我們適用A國法算夫妻財產制，然後適用B國法算繼承，結果是妻子既沒有得到共同財產制的保護（因為B國的繼承法假設有共同財產制），也沒有得到適當的繼承份額。

### 二、解決方法

法官可以「適應」（調整）準據法的適用結果，以避免不公平的結果。這需要法官的創造性解釋。

## 第四章：國際私法與強行法規

### 一、什麼是強行法規的特別連結？

有些法規是如此重要，以至於不管衝突規則指向哪國法，這些法規都要強制適用。

**涉民法第7條**：

> 涉外民事，在中華民國法律所定應適用之範圍內，如依本法規定應適用外國法時，不得排除中華民國法律之強制規定。

### 二、兩種強行法規

#### 1. 國內強行法規

在純國內案件中必須遵守的法規。在涉外案件中，如果衝突規則指向外國法，可能可以排除。

#### 2. 國際強行法規（直接適用的法規 / Overriding Mandatory Rules）

不管衝突規則指向哪國法，這些法規都直接適用。

**例子**：
- 勞動基準法中保護勞工的規定
- 消費者保護法中保護消費者的規定
- 外匯管制法規
- 競爭法（公平交易法）

# 第七部分：國際私法的重要案例

## 一、台灣實務上的涉外案件

### 案例一：跨國婚姻效力

**事實**：台灣國民甲與越南國民乙在越南結婚。甲主張婚姻無效。

**分析**：
- 結婚實質要件 → 分別依各該當事人本國法
- 甲的部分看台灣法，乙的部分看越南法
- 結婚方式 → 依越南法（舉行地法）即可

### 案例二：涉外侵權

**事實**：台灣公司的產品出口到美國，在美國造成消費者受傷。

**分析**：
- 侵權行為地法 → 美國法（損害發生在美國）
- 但如果另有關係最切之法律 → 需要綜合考量
- 商品責任的特別規定（涉民法第23條）也要考慮

### 案例三：跨國繼承

**事實**：美國國民在台灣死亡，在台灣和美國都有遺產。

**分析**：
- 繼承準據法 → 美國法（被繼承人死亡時本國法）
- 但美國各州法律不同 → 需要再確定是哪一州的法律
- 如果台灣國民依台灣法應為繼承人 → 可以就在台灣的遺產繼承之

# 第八部分：考試策略與總複習

## 一、國際私法在國考中的特點

### 1. 配分

國際私法在律師/司法官考試第一試佔**20分**。

### 2. 出題方向

- 涉民法各條的適用是最常考的
- 衝突規則的三要素（法律關係、連結因素、準據法）是基礎
- 反致是必考考點
- 公序良俗條款常考
- 契約的準據法（當事人意思自主）是重點
- 婚姻、繼承的準據法常考
- 外國判決的承認（民訴法第402條）是必考

### 3. 準備策略

**(1) 涉民法條文要背熟**

涉民法全文只有63條，每一條都可能考。建議製作一張表格：

| 法律關係 | 準據法 | 條文 |
|---------|-------|------|
| 行為能力 | 本國法 | 第9條 |
| 法律行為方式 | 準據法或行為地法 | 第15條 |
| 契約 | 當事人意思 → 最密切關係 | 第20條 |
| 侵權行為 | 侵權行為地法 | 第22條 |
| 物權 | 物之所在地法 | 第25條 |
| 結婚 | 各該當事人本國法 | 第29條 |
| 婚姻效力 | 共同本國法 → 共同住所地法 → 最密切關係 | 第30條 |
| 離婚 | 共同本國法 → 共同住所地法 → 最密切關係 | 第32條 |
| 繼承 | 被繼承人死亡時本國法 | 第48條 |
| 遺囑 | 成立時遺囑人本國法 | 第49條 |

**(2) 掌握基本概念**

以下概念必須理解透徹：
- 衝突規則的結構
- 連結因素的意義
- 反致的類型
- 公序良俗條款
- 定性問題
- 當事人意思自主原則
- 特徵履行

**(3) 做考古題時注意解題架構**

涉外案件的標準解題步驟：

```
步驟一：確認有涉外因素
步驟二：定性（這是什麼法律關係？）
步驟三：找到對應的衝突規則（涉民法哪一條？）
步驟四：確定連結因素（國籍、住所、行為地……）
步驟五：確定準據法
步驟六：檢查是否有反致問題
步驟七：檢查公序良俗條款
步驟八：適用準據法解決案件
```

## 二、重要概念對比

### 1. 屬人法的選擇

| 連結因素 | 採用國家 | 優點 | 缺點 |
|---------|---------|------|------|
| 國籍 | 大陸法系（台灣、日本、德國） | 確定性高、不容易改變 | 雙重國籍問題 |
| 住所 | 英美法系（英國、美國） | 反映實際生活中心 | 住所的認定較困難 |
| 慣居地 | 海牙公約 | 折衷方案 | 定義不夠明確 |

### 2. 統一主義 vs. 分割主義

| | 統一主義 | 分割主義 |
|--|---------|---------|
| 適用法律 | 所有遺產適用同一法律 | 動產和不動產適用不同法律 |
| 採用國家 | 台灣、日本、德國 | 英國、美國、法國 |
| 優點 | 簡單明確 | 尊重不動產所在地利益 |
| 缺點 | 可能忽略不動產所在地利益 | 複雜、可能導致遺產處理困難 |

### 3. 各種「地法」

| 名稱 | 拉丁文 | 說明 |
|-----|--------|------|
| 法院地法 | Lex Fori | 審理案件的法院所在國法 |
| 物之所在地法 | Lex Rei Sitae | 標的物所在地法 |
| 行為地法 | Lex Loci Actus | 行為發生地法 |
| 契約締結地法 | Lex Loci Contractus | 契約簽訂地法 |
| 履行地法 | Lex Loci Solutionis | 債務履行地法 |
| 侵權行為地法 | Lex Loci Delicti | 侵權行為發生地法 |
| 本國法 | Lex Patriae | 當事人國籍國法 |

## 三、高頻考點速查

1. 衝突規則的三要素 — 法律關係、連結因素、準據法
2. 反致的類型 — 直接反致、間接反致
3. 涉民法第6條反致的適用條件 — 限於適用「本國法」時
4. 公序良俗條款 — 第8條，審查適用結果而非外國法本身
5. 契約準據法 — 當事人意思 → 最密切關係 → 特徵履行推定
6. 侵權行為準據法 — 侵權行為地法
7. 物權準據法 — 物之所在地法
8. 婚姻準據法 — 各該當事人本國法
9. 繼承準據法 — 被繼承人死亡時本國法
10. 外國判決承認四要件 — 管轄權、應訴機會、公序良俗、互惠

## 三、涉外案件解題模板

### 模板一：契約糾紛

```
一、涉外因素確認
   → 有涉外因素（說明原因）

二、國際裁判管轄
   → 台灣法院有無管轄權？

三、準據法的決定
   （一）定性：本件為契約糾紛
   （二）衝突規則：涉民法第20條
   （三）當事人有無合意選法？
        → 有 → 適用合意選定之法律
        → 無 → 適用最密切關係法
   （四）反致問題？（通常契約不適用反致）
   （五）公序良俗檢查

四、適用準據法解決案件
```

### 模板二：婚姻/繼承案件

```
一、涉外因素確認
   → 有涉外因素（說明原因）

二、準據法的決定
   （一）定性：本件為婚姻/繼承問題
   （二）衝突規則：涉民法第○○條
   （三）連結因素：當事人本國法/共同本國法
   （四）有無反致？（涉民法第6條：限於適用本國法時）
   （五）公序良俗檢查（涉民法第8條）

三、適用準據法解決案件
```

## 結語

國際私法看似抽象，但在全球化的今天，跨國婚姻、跨國商務、跨國繼承越來越常見。國際私法就是處理這些跨國事務的「交通規則」。

理解國際私法的關鍵在於：永遠記住它在解決什麼問題 — **當多個國家的法律都可能適用時，選擇最適當的那一個。**

掌握了這個核心思維，再搭配涉民法各條文的具體規定，你就能應付國考中的國際私法題目了。
