# 序篇：修復師的孤獨（ch1-5）—— 初稿級精修

> **精修目的**：將大綱級敘事提升為初稿級密度——加入具體感官、對白節奏、內心獨白、鏡頭感。
> **原則**：遵循《角色語言風格指南》十條戒律，演算法潛移默化，維倫不說超過三句話。

---

## 第 1 章：壞掉的時間

### 開場鏡頭（500 字以內）

```
【環境】
凌晨兩點零七分。舊台北地下街 Y 區出口旁。
門牌號碼被噴漆蓋掉了。只有一塊手寫的壓克力板：
    「記憶修復 / 不問來歷 / 修不好不收錢」
第三條用紅筆加的——字跡比前兩條新。

工作室六坪。修復台佔了三分之一。
螢幕左下角有一條永遠修不好的亮線。
米亞的全息投影浮在修復台上方，藍色光暈像一隻發光的水母。
空氣裡有焊錫的味道、隔夜咖啡的酸味、以及地下街永遠散不掉的潮氣。

維倫坐在修復台前，袖子捲到手肘。他的手很穩。
手背上有三條舊傷疤——不是打架留的，是修復失敗時記憶回衝燒的。
他不戴手套。從來不戴。
「手套隔一層，波形就讀不準。」他曾經這樣跟人解釋。
但真正的原因是：他喜歡觸碰別人記憶時那種微涼的電流感。
像是在跟另一個人的人生握手。
```

### 老陳委託場景——節奏控制

```
【節奏】慢。每一個修復步驟都要讓讀者「看到」。
這是全書第一次展示修復過程——讀者必須愛上這項技術。

【米亞的第一句台詞】（全書第一句AI對白，定調）
建議方向：不是自我介紹。是一句吐槽。
  ✅ 「你又在喝那個三天前的咖啡。我的化學分析模組在哭。」
  ❌ 避免：「我是米亞，你的AI助手。」

【修復過程——演算法呈現方式】
不解釋 Two Pointers。讓讀者「看到」維倫在做什麼：
  - 左手食指壓住螢幕左端（= 左指針，從時間戳最早的幀開始）
  - 右手食指從右端滑過來（= 右指針，從最晚的幀開始）
  - 兩根手指往中間推，像在合攏一本散落的相簿
  - 「幀」的概念用「畫面」替代——「這張在這張前面。對了。」

【老陳的戲份——不超過 200 字】
  老陳 68 歲。退休國小老師。手指甲裡有粉筆灰。
  他不哭。他把修復好的記憶放進播放器，閉上眼。
  然後他的嘴角動了一下——不是笑。是那個笑容還沒來得及成形就碎了。
  「她那天穿的是那件。」他說了半句。沒有說完。
  維倫等了一分鐘。
  「修得好。」維倫說。（← 第一次使用口頭禪）
  「修得好。謝謝。」老陳把存儲盒捧在胸口，像捧著骨灰罈。走了。
```

### 章末鉤子——林語青登場

```
【鏡頭感】切換要快。前一秒是維倫關門的日常動作，下一秒是門鈴。

  門鈴聲音：不是電子鈴。是維倫用舊零件焊的機械鈴。叮——一聲。乾淨。

  維倫轉身。門口站著一個人。

  【語青的第一印象——只寫三個細節】
  1. 白大衣——不像醫生的白大衣。像實驗室的。衣角有燒焦的痕跡。
  2. 雨水——從髮梢滴到地上。她沒帶傘。但她看起來不像忘了帶傘的人。
     像是「知道會淋雨但不在乎」的人。
  3. 眼神——直視維倫。沒有猶豫。
     維倫見過各種委託人：害怕的、心虛的、絕望的。
     她的眼神不是任何一種。像是「終於找到了」。

  「我有一段記憶需要修復。」她說。
  「時間戳是 2091 年 3 月。」

  維倫看了看牆上的日曆。2089 年。

  米亞的藍光閃了一下。
  她在維倫耳邊低語的那句話不要寫出來。
  讓讀者只看到維倫的表情微微變化。

  章末。翻頁。
```

## 第 2 章：六坪宇宙

### 核心場景：維倫拒絕 + 語青的神秘台詞

```
【語青「你每次都會」的呈現方式】
  不要讓這句話「埋」在段落裡。讓它獨立。

  維倫：「時間戳不可能造假，妳應該去記憶銀行。」

  語青看著他。沒有生氣。沒有失望。
  像是在確認什麼。

  「你會改變主意的。」

  停頓。

  「你每次都會。」

  她轉身走進雨裡。
  維倫站在門口。雨打在雨棚上的聲音很大。

  「每次」。
  這個詞在他腦子裡卡了三秒。
  像一根刺。拔不出來。
```

### 世界觀展開——用五感寫而非用說明寫

```
【禁止】長段落解釋五個社會階層
【正確】通過維倫回家路上的所見所聞自然展開：

  視覺：地下街的全息廣告——
    「記憶銀行：給記憶一個永恆的家 ♡」（蝴蝶標誌，柔和的藍光）
    旁邊是黑市小廣告：「高品質技能記憶，一段 ¥300，貨到付款」
    （兩張廣告的並置 = 兩個世界的對比，不需要解釋）

  聽覺：角落裡有人在哭。不是大聲哭。是那種忘了怎麼哭的哭法。
    「空白人」——記憶被完全提取的人。他們不記得悲傷是什麼，
    但身體還是會哭。（← 第一次植入「身體記得」母題）

  嗅覺：維倫的公寓門口有一股消毒水的味道。
    隔壁的老張又在消毒記憶晶片了——他是黑市記憶販子，
    用消毒水蓋住記憶提取時留下的焦味。

  觸覺：維倫打開公寓門。門把手是冰的。
    六坪。修復失敗的記憶碎片列印稿貼滿了牆。
    「每一次失敗都是一個人生。」米亞問他為什麼不扔。他這樣說。
    但實際上他不知道為什麼。他只是覺得扔了不對。
    （← 維倫的善良用「不知道為什麼」表達，不用「因為我是好人」）
```

### 章末鉤子——記憶空位

```
【節奏】從日常轉到恐怖——要快。三段以內。

  凌晨四點。頭痛。
  不像生理的痛。像有什麼在裡面動。

  維倫打開修復台。自我掃描。
  螢幕上的數據一切正常。

  但米亞的聲音變了。變輕了。
  （← 米亞的語氣變化比台詞內容更重要。她在害怕。）

  「維倫。你的記憶索引區。」
  「嗯。」
  「有一個......空位。」
  「什麼意思。」
  「不是被刪除的。是從來沒有被寫入過。」
  「就像——」她停頓了 0.3 秒。比平常多了 0.2 秒。
  「有人在你的大腦裡，預留了一個房間。」

  維倫盯著螢幕。
  那個空位的大小——剛好能放進一段完整的記憶。

  他不知道的是：那段記憶的格式，跟林語青帶來的那段未來記憶一模一樣。

  （最後一句是作者視角的懸念，不是維倫知道的。用斜體或另起一行暗示。）
```

## 第 3 章：地下街的早餐

### 阿嬤場景——精修要點

```
【阿嬤出場——不超過 200 字，但讓讀者想哭】

  關鍵：不用任何煽情語言。純粹寫動作。

  阿嬤在炸油條。動作熟練。不需要看。
  手知道在哪裡翻。手知道什麼時候撈起來。
  油條金黃。像四十年來每一根一樣。

  攤位角落。一碗鹹豆漿。
  不是給客人的。從開攤時就放在那裡。收攤時倒掉。明天再做一碗。

  維倫問過：「那碗是留給誰的？」
  阿嬤想了半天。
  「不記得了。但是每天覺得該留。」

  她不記得周明哲。不記得他的名字。不記得他的臉。
  但她的手每天清晨六點準時做那碗豆漿。
  因為手記得。

  ——不要在這裡寫「身體記得」四個字。讓讀者自己想到。
```

### 地下街世界觀——底層生態的細節

```
【地下街的聲景】
  近處：油鍋的滋滋聲。阿嬤跟客人說台語的嘎嘎笑聲。
  中距：有人在叫賣——「技能記憶特價！學鋼琴三百、學開車五百、學日文八百！」
  遠處：深淵監控局無人機的嗡嗡聲——恆定的、機械的、像蚊子。

【地下街的光】
  螢光燈管。有幾根壞了，在閃爍。
  全息廣告的藍光和攤位的暖黃色交替。
  角落裡空白人的眼睛在暗處像小動物。

【維倫在地下街的地位】
  不用任何人叫他「維倫老師」或表示尊敬。
  用一個細節就夠：
  他經過每個攤位，每個攤主都會多看他一眼。
  不是害怕。是一種「需要的時候我知道找誰」的目光。
  這就是「底層守護者」——不被說出口的信任。
```

### 小杰登場——引入第二案

```
【小杰的第一印象——只寫兩個細節】
  1. 記憶存儲盒被抱在胸口——不是拿著。是抱著。像抱著活的東西。
  2. 眼眶紅——但不哭。10 歲的男孩在忍著。

  維倫蹲下來。（← 蹲下來 = 他跟孩子一樣高 = 尊重）
  他沒有說「怎麼了」。他等。

  小杰自己開口了。
```

## 第 4 章：她又來了

### 核心場景：沉默的共處

```
【重點】這一章的力量不在對話，在「安靜」。

  語青坐在工作室角落的舊折疊椅上。
  維倫在修復台前分析小杰的記憶。
  米亞的藍光在兩人之間浮動。

  長時間的沉默。
  但不是尷尬的沉默。是兩個不需要說話就能待在一起的人。

  寫法：
  - 用環境聲代替對話——修復台的低頻嗡聲、地下街隔著牆傳來的模糊人聲、
    雨打在排風口的聲音。
  - 偶爾一句話。間隔很長。
    語青：「你喝的那個咖啡已經三天了。」
    維倫：「……嗯。」（不抬頭）
    米亞（低聲，只有維倫聽到）：「她跟我說的一模一樣。」

  （← 米亞跟語青都注意到維倫的壞習慣。這不是巧合。）
```

### 語青的關鍵台詞——預留空間

```
【語青暗示維倫大腦異常】
  不要讓她「直接說」。讓她先問，再觀察反應。

  語青看著牆上的記憶碎片列印稿。看了很久。
  「你修復的成功率是多少？」
  「七成。」（維倫不炫耀。七成在黑市已經是最高。）
  「不是七成。是八成三。」
  維倫轉頭。「你怎麼——」
  「我看你的手。」語青指著維倫的右手。
  「你的手在觸摸記憶數據的時候會微微發抖。
   那不是緊張。是共振。
   你的大腦在跟別人的記憶同步。」

  停頓。

  「你有沒有想過——為什麼你能做到？」

  維倫沒有回答。
  但他的左手不自覺地摸了摸右手背上的舊傷疤。
  那些修復失敗時燒出來的傷疤——也許不是「失敗」。
  是他的大腦試圖跟記憶融合得太深了。
```

### 章末鉤子——兩層記憶

```
  語青走的時候，在門口停了一下。

  「那段未來記憶——你不需要修復它。你只需要看一眼。」
  「為什麼？」
  「因為裡面有你。」

  她走了。

  米亞等了十七秒才開口。
  （← 寫出準確的秒數。這是米亞的語言方式——她什麼都量化。）

  「我掃了她的記憶特徵。」
  「結果。」
  「兩層。」
  維倫不說話。
  「底下那一層的情感標籤裡……有你的名字。」

  維倫看著語青消失的門口。
  門外的雨聲像白噪音。
  他想到了自己記憶索引區裡的那個空位。
  剛好能放進一段完整記憶的空位。

  （讀者此時應該開始猜：那個空位是不是留給語青的？）
```

## 第 5 章：米亞的理論

### 米亞的三種假說——用角色聲音呈現

```
【米亞分析模式——快速、百分比、帶吐槽】

  維倫坐在修復台前。凌晨三點。又是凌晨三點。
  米亞的全息投影在空中展開三個面板。

  「假說一：精密偽造。」
  面板亮起。一堆技術參數。
  「成功偽造未來時間戳的機率——0.0000034%。
   需要的算力相當於三個記憶銀行核心伺服器同時運作七天。
   結論：除非有人吃飽太閒且能駭進銀行，否則不是偽造。」

  「假說二：時間線殘留。」
  第二個面板。理論物理的圖表。
  「量子記憶理論的邊緣推論。如果平行時間線之間有記憶洩漏——
   理論上可能，但從未被記錄過。機率：未知。
   但我查了所有學術資料庫。沒有案例。零個。
   ……我不喜歡零。零讓我不安。」

  「假說三：我不知道。」
  第三個面板是空的。
  「我討厭第三個假說。但它存在的機率是——」
  她停頓了 0.4 秒。
  「我算不出來。這讓我更不安了。」

  維倫看著三個面板。
  「打開它。」
  「什麼？」
  「她的記憶。用最小權限。只看外層。」
  米亞的藍光閃了一下——這是她表示猶豫的方式。
  「你確定？」
  「不確定。打開。」
```

### 未來記憶的第一次窺視——極致感官

```
【這是全書第一次讓讀者「看到」未來。必須震撼。】

  最小權限模式。只讀取最外層感官數據。
  維倫閉上眼。

  視覺先到——
  新台北。他認得天際線。
  但天際線在燃燒。
  101 塔的頂端折斷了。像一根被掰斷的筷子。
  街上到處是人。
  不。是空白人。成千上萬的空白人。
  他們在走路。但沒有方向。像斷了線的程式。
  有的撞到牆壁，繼續走。
  有的蹲在地上，對著空氣說話。
  有的在笑。笑得很大聲。但眼睛是空的。

  畫面切——
  一個房間。他的工作室。他認得那條螢幕上的亮線。
  椅子上坐著一個人。
  是他自己。

  未來的他。
  臉上有血。左眼腫了。嘴唇裂開。
  他對著鏡頭在說話。
  但沒有聲音。

  維倫看著自己的嘴型動了四個音節。
  米亞做了唇語分析。花了兩秒。

  然後她的聲音變了。變成維倫從來沒聽過的語氣。
  很輕。像怕把什麼東西震碎。

  「他說的是——」

  「別相信她。」

  維倫斷開了連接。手在抖。
  修復台的亮線還在閃。
  米亞的藍光比平常暗了兩個色階。

  兩個人在黑暗裡坐了很久。

  「她。」維倫低聲說。

  他沒有說「是不是語青」。
  米亞也沒有問。
  因為他們都聽到了。
  那個字的嘴型——不是「她」。
  是一個特定的名字。
  只是唇語分析的解析度不夠。

  差一點。
```

### 章末鉤子——決定找語青

```
  第二天早上。
  維倫在地下街找到語青。
  她坐在阿嬤的攤位前。
  在學怎麼喝鹹豆漿。

  阿嬤在教她：「加醋。不要加太多。加了就不能倒回去了。」
  語青一臉認真地在記。

  看到維倫，她笑了。
  那個笑容——
  維倫想到了昨夜螢幕上那張滿是血的臉。
  想到了「別相信她」。

  但此刻坐在他面前的這個人，
  在陽光裡（地下街沒有陽光，是攤位的暖黃色燈泡），
  在阿嬤的攤位前，
  在學喝鹹豆漿。

  她不像一個應該被警告的人。

  「你有一個問題要問我。」語青說。（← 她知道他會來。她每次都知道。）

  「裡面的那個我。」維倫坐下來。「是活的還是死的。」

  語青看著他。很久。

  「你真正想問的不是這個。」

  維倫等。

  「你想問的是——」
  她端起豆漿。喝了一口。放下。

  「那個你，是不是『你自己』。」

  序篇結束。
  讀者此刻的狀態：他媽的我要讀下去。
```

## 讀者體驗模擬——序篇五章的「鉤子鏈」

```
五章的鉤子形成一條不間斷的懸念鏈：

Ch1 鉤子：2091 年的記憶？（好奇）
     ↓
Ch2 鉤子：維倫大腦有空位。「你每次都會。」（不安）
     ↓
Ch3 鉤子：小杰母親的記憶被人動過手腳。（懸疑）
     ↓
Ch4 鉤子：語青的記憶有兩層。底層有維倫的名字。（震驚）
     ↓
Ch5 鉤子：未來的他說「別相信她」+ 語青知道他會來。（恐懼 + 好奇共存）

五章之後讀者擁有的信息：
  ✅ 認識了維倫：孤獨、善良、手很穩、不說廢話
  ✅ 認識了米亞：聰明、吐槽、量化一切、比維倫更擔心他
  ✅ 認識了語青：精準、神秘、「她知道些什麼」
  ✅ 看到了世界：記憶銀行、空白人、地下街、階層壓迫
  ✅ 被種下了恐懼：新台北在未來會毀滅？維倫自己會……？
  ✅ 心裡有一根刺：「你每次都會。」這個「每次」是什麼意思？
  ✅ 開始關心一碗不被賣出的豆漿——雖然不知道為什麼

五章之後讀者不知道的：
  ❌ 語青是誰（只知道她很奇怪）
  ❌ 鸚鵡計畫是什麼
  ❌ 維倫的大腦為什麼有空位
  ❌ 那碗豆漿留給誰
  ❌ 「別相信她」到底指誰
  ❌ 米亞的真正來源

結論：五章建立了六條懸念線，每一條都讓讀者「想知道」。
序篇的工作 = 在讀者心裡釘了六根釘子，每一根都只釘了一半。
```

## 精修總結：五章需要注意的寫作陷阱

| 陷阱 | 說明 |
|------|------|
| 世界觀 info-dump | 第 2、3 章最容易犯。用五感寫，不用「在這個世界裡……」句式。 |
| 維倫話太多 | 他前五章的台詞總量不應超過 40 句。超過就砍。 |
| 米亞太像 Siri | 她不是助手。她是搭檔。她會擔心、吐槽、生氣。0.3 秒的停頓 = 她的人性。 |
| 語青太神秘 | 她不是「神秘女人」的套路。她學喝豆漿的那個笑容 = 她是人。 |
| 演算法太明顯 | 第 1 章的 Two Pointers 和第 8 章的 Sliding Window：讀者應該覺得「修復記憶原來是這樣」，而不是「這是在教我演算法」。 |
| 老陳太催淚 | 克制。他「差一點笑」比「大哭」更痛。 |
| 阿嬤的豆漿太多鏡頭 | 200 字以內。輕描淡寫。讀者第一次讀完覺得「哦，一個可憐的老太太」。第四卷回頭才發現——那碗豆漿是留給全書最大反派的。 |
