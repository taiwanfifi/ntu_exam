# 第二篇「雜訊」終章（ch21-24）—— 初稿級精修

> **精修目的**：從「犧牲」到「反擊」。數學成為解碼工具。費雪登場。第二篇收束。
> **原則**：節奏從壓抑轉向緊湊。聯合分布/邊際分布/條件分布必須是「解謎」而非「教學」。費雪的出場要讓讀者脊背發涼。

## 第 21 章：六個字元

### Beta(72, 28)——林岳的遺言是什麼意思？

```
【環境】
週五深夜。永和套房。
日光燈管今天特別穩——沒有閃。
像連燈管都知道今天的事情很嚴肅。

蘇立恆坐在小桌前。面前是筆電。
貝葉坐在他對面。面前——兩張紙條。
一張寫著「先驗端口。1,247 次。查 B2。」
一張寫著「Beta(72, 28)」。

白板已經被擦乾淨了。
蘇立恆用溼紙巾擦了三遍。
「這一題需要乾淨的空間。」

馬可瑜今天沒有來。
她在家裡用加密頻道連線。
theta 的呼聲偶爾從通話裡傳出來。

「Beta(72, 28)。」蘇立恆看著紙條。
他的眼神跟平常不一樣——
平常是冷的。精確的。
今天多了一個東西。
很微弱。但貝葉看到了。
是——尊重。
對林岳的。
對一個在被帶走之前還能把發現壓縮成六個字元的人。

「字面上：一個 Beta 分布。α=72，β=28。」

  ┌─ Beta(72, 28) 基本分析 ────────────────────┐
  │                                              │
  │ 期望值：E[X] = 72/(72+28) = 0.72           │
  │ 變異數：Var[X] = 72×28/(100²×101) ≈ 0.002 │
  │ α+β = 100 → 「信心總量」很大 → 分布很集中  │
  │                                              │
  │ 意義：有 72% 的信心度。                     │
  │ 關於什麼？                                  │
  └──────────────────────────────────────────────┘

「72% 的信心度。但——關於什麼？」貝葉問。

「這是關鍵。」蘇立恆站起來。走到白板前。

「林岳不會隨便寫一組參數。
 如果他只是想告訴你『有操控』——
 他不需要寫具體數字。
 但他寫了 72 和 28。
 不是 70 和 30。不是 75 和 25。
 是精確的 72 和 28。
 這意味著——這組參數本身就是資訊。」

通話裡馬可瑜的聲音：
「等一下。72 和 28 加起來是 100。
 100 很整。不太可能是偶然。」

蘇立恆點頭。
「α+β = 100。100 次觀測。
 72 次——成功。28 次——失敗。
 如果這是一個二元實驗——
 100 次裡成功了 72 次。」

「什麼實驗？」

「如果我是林岳——
 在發現先驗操控之後、在被帶走之前——
 我會做什麼？」

蘇立恆看著牆。23 個紅點。

「我會測試。
 從已知被操控的區域裡抽樣。
 對每個區域的先驗做一個二元檢驗：
 『這個先驗是否偏離了歷史基線？』
 是——標記為 1。否——標記為 0。
 100 個樣本。72 個是 1。」

「所以 Beta(72, 28) 不只是一個分布——
 它是一個實驗結果。」

「對。林岳做了 100 次檢驗。
 72 次確認先驗被修改。
 他用一組 Beta 參數把整個實驗的結果壓縮了。
 六個字元。」

→ 林岳的智慧不在於他發現了什麼。
  在於他把發現的東西——用最少的資訊傳遞出去。
  Beta(72, 28) = 100 次實驗。72 次異常。
  數學家的遺言——不是一段話。是一個分布。
```

### 但更深的問題——哪 100 個區域？

```
「但等一下。」貝葉說。
「我們知道的只有 23 個區域。
 林岳測了 100 個？」

蘇立恆點了一下白板。

「這就是他比我們走得更遠的地方。
 我查了三年。查到 23 個。
 他——在公司內部——可能查了更多。
 他有內部權限。他能看到更多區域的先驗紀錄。」

「100 個區域裡 72 個被操控。」
馬可瑜的聲音從通話裡傳出來。很小聲。
theta 停止打呼了。像也在聽。

「72%。」

「全台灣的行政分區大概 368 個。
 如果比例一致——
 0.72 × 368 = 264。
 264 個區域的先驗被修改了。」

沉默。

蘇立恆走到地圖前面。
23 個紅點在地圖上。
但如果是 264 個——

「整張地圖都會變成紅色。」他說。

這不是改幾個犯罪率的問題了。
這是——
整個國家的現實被重新編輯了。

每一個人透鏡上看到的數字。
每一個人做的每一個決定。
出門走哪條路。買不買保險。投不投那張票。
信不信那個人。

全部——基於一個被操控的先驗。

貝葉看著地圖。
地圖上的台灣很小。比她的手掌大不了多少。
但地圖上的每一個點——都是真實的城市。
有人在那裡上班、買菜、接小孩、看醫生。
有人在那裡——相信透鏡上的數字。

「我們需要更多證據。」她說。
「Beta(72, 28) 是結論。但不是證據。
 我們需要知道——具體是哪 72 個。
 被改了多少。什麼時候改的。
 才能讓任何人相信我們。」

「所以——」蘇立恆看著她。
「我們需要破解林岳的方法。
 用他的方式——重做一次。」

「怎麼做？」

「聯合分布。」
```

## 第 22 章：聯合與邊際

### 兩個變數的秘密——聯合分布

```
【環境】
週六。白天。永和套房。
難得有陽光從窗戶照進來。
灰塵在光柱裡飛舞。像一群微小的隨機變數。

馬可瑜帶了 theta 來。
theta 立刻趴到瑜伽墊上。圈起來。睡了。
「他把你的瑜伽墊當窩了。」馬可瑜對蘇立恆說。

蘇立恆看了 theta 一眼。
「……隨便。」

白板上。蘇立恆開始寫。

「林岳測了 100 個區域。
 他一定記錄了每個區域的兩個資訊：
 X——先驗修改的幅度（比如 0% 到 100%）。
 Y——修改的時間點（比如 2044 年幾月）。」

「為什麼是這兩個？」貝葉問。

「因為——單獨看修改幅度，你只知道『改了多少』。
 單獨看修改時間，你只知道『什麼時候改的』。
 但如果你同時看——」

他在白板上畫了一個坐標系。
X 軸是修改幅度。Y 軸是修改時間。

  ┌─ 聯合分布 f(x, y) ────────────────────────┐
  │                                              │
  │ 定義：                                      │
  │   兩個隨機變數 X 和 Y 的聯合分布            │
  │   描述「X 和 Y 同時取某個值」的機率         │
  │                                              │
  │ f(x, y) = P(X = x 且 Y = y)                │
  │                                              │
  │ 直覺：                                      │
  │   X 單獨看是一張照片。                      │
  │   Y 單獨看是另一張照片。                    │
  │   f(x, y) 是兩張照片疊在一起——              │
  │   你能看到它們之間的關係。                  │
  │                                              │
  │ 就像偵探不會只看一個嫌疑人的行蹤。          │
  │ 他會把所有嫌疑人的行蹤疊在一起——            │
  │ 看誰在同一個時間出現在同一個地點。          │
  └──────────────────────────────────────────────┘

「兩個變數同時看。
 你就能看到——修改幅度和修改時間之間有沒有關聯。
 如果有——
 那操控不是隨機的。是有計畫的。
 有特定的時間窗口。有特定的幅度控制。」

貝葉盯著白板。

「所以林岳做的事是——
 把 100 個區域的 (X, Y) 都記錄下來。
 然後建立了它們的聯合分布。」

「對。而 Beta(72, 28) 是——
 從這個聯合分布裡提取出來的某個邊際分布。」

「邊際分布？」
```

### 邊際分布——把不需要的維度積掉

```
蘇立恆在白板上畫了一個二維的散點圖。
然後在 X 軸的方向畫了一條直方圖。
在 Y 軸的方向也畫了一條。

  ┌─ 邊際分布 ─────────────────────────────────┐
  │                                              │
  │ 從聯合分布 f(x, y) 可以得到：              │
  │                                              │
  │ X 的邊際分布：fₓ(x) = ∫ f(x, y) dy        │
  │   → 把 Y 積掉。只看 X 的分布。             │
  │   → 「不管時間，只看幅度的整體分布」        │
  │                                              │
  │ Y 的邊際分布：f_Y(y) = ∫ f(x, y) dx        │
  │   → 把 X 積掉。只看 Y 的分布。             │
  │   → 「不管幅度，只看時間的整體分布」        │
  │                                              │
  │ 直覺：                                      │
  │   你拍了一張二維的照片。                    │
  │   邊際分布 = 把照片壓扁成一維。             │
  │   從上往下壓 → 得到 X 的分布。              │
  │   從左往右壓 → 得到 Y 的分布。              │
  │                                              │
  │ 壓扁的過程中，你丟失了 X 和 Y 之間的關係。 │
  │ 但你得到了每個變數「單獨」的全貌。          │
  └──────────────────────────────────────────────┘

「如果 Beta(72, 28) 是 X 的邊際分布——」

「那它描述的是：先驗修改幅度的整體分布。
 期望值 0.72。意味著——平均修改幅度是 72%。」

馬可瑜插嘴了。她一直在聽。theta 也醒了。
在瑜伽墊上翻了一個身。

「等一下。72% 是平均修改幅度？
 我們之前看到的最大幅度是 61%——」

「對。林岳看到了更多。
 他查的 100 個區域裡——
 有些修改幅度超過了 61%。
 Beta(72, 28) 的分布很集中——變異數只有 0.002——
 意味著大部分區域的修改幅度都在 70% 附近。」

「70% 的先驗偏移。」貝葉重複了一遍。
聲音像被什麼東西壓住了。

「幾乎是原始先驗的三分之二都被改了。
 居民看到的世界——
 跟真實世界只有 30% 的重疊。」

窗外有人在澆花。水聲嘩啦嘩啦。
很正常的聲音。
一個正常的週六下午。
一個 70% 被改寫的週六下午。
```

### 條件分布——如果我知道幅度，能推斷時間嗎？

```
蘇立恆繼續。

「但邊際分布不夠。
 我們需要的是——條件分布。」

  ┌─ 條件分布 ─────────────────────────────────┐
  │                                              │
  │ P(Y|X) = f(x, y) / fₓ(x)                  │
  │                                              │
  │ 已知 X 的值，Y 的條件分布。                 │
  │                                              │
  │ 直覺：                                      │
  │   你知道犯人偷了多少錢（X = 修改幅度）。    │
  │   那他是什麼時候偷的（Y = 修改時間）？      │
  │                                              │
  │ 條件分布 = 在已知一個線索的情況下，          │
  │           推斷另一個線索的分布。              │
  │                                              │
  │ 這就是為什麼聯合分布比邊際分布有用：        │
  │   邊際 = 你只知道一件事。                    │
  │   聯合 = 你知道兩件事的關係。                │
  │   條件 = 你用一件事去推斷另一件事。          │
  └──────────────────────────────────────────────┘

「P(Y|X) = f(x, y) / fₓ(x)。」

他寫在白板上。紅色。

「如果我們能重建聯合分布——
 然後用 Beta(72, 28) 作為 X 的邊際——
 就能反推 P(Y|X)。
 也就是——在已知修改幅度的情況下，
 推斷修改時間的分布。」

「然後我們就能知道——什麼時候被改的。」

「對。而且不只是單一時間點。
 是一個分布——
 有些時間窗口更可能。有些不太可能。」

貝葉看著白板。
聯合分布。邊際分布。條件分布。
三個概念。
在課本上是三頁紙。
在這裡——是解救 264 個區域的工具。

「所以我們的計畫是：」

「一。重建聯合分布 f(x, y)。
 用我們有的 23 個區域的數據——加上馬可瑜的保險數據——
 估計 (修改幅度, 修改時間) 的聯合分布。

 二。確認 Beta(72, 28) 是 X 的邊際分布。
 如果吻合——說明我們的模型跟林岳的發現一致。

 三。計算條件分布 P(Y|X)。
 找到修改發生的時間窗口。

 四。用時間窗口——交叉比對 B2 的門禁紀錄。
 看誰在那些時間進出了核心架構組。」

蘇立恆把白板筆放下。

「林岳給我們的不是一個答案。
 是一把鑰匙。
 Beta(72, 28) 是鑰匙的形狀。
 聯合分布是鎖。
 條件分布——是打開鎖之後看到的東西。」

→ 全書最重要的數學橋段之一。
  聯合分布 → 邊際分布 → 條件分布。
  三個概念串成一條邏輯鏈。
  讀者在「破案」的緊張感中學會了這三個概念。
  不是因為老師講了。是因為他們需要。
```

## 第 23 章：時間窗口

### 計算——凌晨的真相

```
【環境】
深夜。永和套房。
三台筆電。三個人。一隻狗。
（theta 趴在瑜伽墊上。偶爾抖一下腿。在做夢。
 馬可瑜說他在夢裡追松鼠。）

三天。
他們花了三天重建聯合分布。

方法：
  用 23 個區域的 (修改幅度 X, 修改時間 Y) 做核密度估計（KDE）。
  再加上馬可瑜從保險數據推算出的 17 個區域的估計值。
  總共 40 個數據點。

  不夠多。但——
  「不夠多不代表不能做。」蘇立恆說。
  「只是信賴度會低一些。但方向是對的。」

三天後。結果出來了。

  ┌─ 聯合分布估計結果 ──────────────────────────┐
  │                                               │
  │ f̂(x, y) 的主要特徵：                        │
  │                                               │
  │ ① X（修改幅度）和 Y（修改時間）正相關       │
  │    相關係數 ≈ 0.67                           │
  │    → 越晚被修改的區域，幅度越大              │
  │    → 操控在「加速」                          │
  │                                               │
  │ ② X 的邊際分布：                             │
  │    估計為 Beta(α̂=68.3, β̂=27.1)               │
  │    與 Beta(72, 28) 接近 ✓                    │
  │    （差異可能因為我們只有 40 個點）           │
  │                                               │
  │ ③ Y 的邊際分布：                             │
  │    呈現三個峰值（多峰分布）                  │
  │    峰值 1：2044年8月                         │
  │    峰值 2：2044年11月                        │
  │    峰值 3：2045年2月                         │
  │    間隔 ≈ 3個月                              │
  │                                               │
  │ ④ 條件分布 P(Y|X > 0.5)：                   │
  │    高幅度修改集中在三個時間窗口               │
  │    全部在凌晨 02:00 - 04:00                  │
  │    全部在「系統自動校準」排程之後             │
  └───────────────────────────────────────────────┘

「三個時間窗口。」貝葉看著結果。
「每隔三個月。凌晨兩點到四點。」

「系統自動校準在每天凌晨一點半執行。
 執行完大概一點五十。
 然後——有人在自動校準完成之後——
 手動進入系統——覆蓋了先驗。」

「自動校準是幌子。」馬可瑜說。
她的聲音不再有平常的笑意了。
「真正的修改在它之後。
 就像有人等清潔工打掃完——
 然後在乾淨的地板上重新畫了記號。」

「而且——」蘇立恆指著第三個發現。
「每隔三個月。像一個排程。
 2044 年 8 月。11 月。2045 年 2 月。
 下一次——」

「2045 年 5 月。」貝葉說。
「下個月。」

三個人看著彼此。

下一次操控——在一個月後。
他們有一個月的時間。
```

### 門禁紀錄——被清除的幽靈

```
「三個時間窗口。凌晨兩點到四點。
 有人在這些時間進出了 B2——核心架構組。」

蘇立恆打開另一個視窗。

「我有 B2 的門禁紀錄。
 不是從內部拿的——是從大樓管理公司的備份系統。
 他們每季備份一次門禁數據。
 備份的時候不會被 Oracular 的安全系統覆蓋。」

「三年前的你就拿到了？」

「兩年前。花了八個月找到管道。」

他打開門禁紀錄。

2044 年 8 月。凌晨 02:00 - 04:00。B2。

空白。

2044 年 11 月。凌晨 02:00 - 04:00。B2。

空白。

2045 年 2 月。凌晨 02:00 - 04:00。B2。

空白。

「紀錄被清除了。」蘇立恆的聲音比平常更低。
「不是沒有人進出。是——紀錄被人刪掉了。」

「能刪除門禁紀錄的人——」

「在 Oracular 裡只有兩個權限等級能做到。
 一。系統架構組組長。也就是我的前職位。
 我離開之後——位子空了一年。
 然後被一個人接了。」

「誰？」

「我不知道名字。只知道代號。
 內部文件裡用的是——『Fisher Protocol』。」

「Fisher。」貝葉說。

「二。安全部門主管。
 安全部門有權限清除任何門禁紀錄——
 以『安全理由』。不需要報備。」

「安全部門主管——」

「費雪。」

蘇立恆看著螢幕。

「費雪有動機、有權限、有能力。
 她管安全。她能清除紀錄。她能監控校準師。
```

## 第 24 章：精確得不尋常

### 週一早上——費雪

```
【環境】
週一。早上。Oracular 辦公室。八樓。

一切如常。
空調 24 度。咖啡機的氣味。鍵盤的點擊聲。
有人在討論午餐吃什麼。有人在打噴嚏。
貝葉的桌上：新的珍奶（今天記得帶了）、統計計算機、招財貓。

林岳的桌子還是空的。
全家福照片被他太太上週來收走了。
「我先拿回去。等他好了再帶來。」
她太太的眼睛是紅的。但她在笑。
那個笑——跟貝葉在病房裡擠出來的笑很像。

仙人掌還在。
沒有人記得澆水。
但仙人掌不在乎。它本來就不需要太多水。
像一個獨立的、不需要先驗的變數。

貝葉盯著那盆仙人掌。

小拉：「仙人掌可以一個月不澆水——」

「我知道。」

但她還是倒了一點水。
不是為了仙人掌。是為了林岳。
是為了——維持一個「他還會回來」的假設。
即使她知道先驗很低。

灰色的眼睛在螢幕左上角。
安靜地看著她。
她已經習慣了。
像習慣了一個每天都在的、不會眨眼的室友。

然後——

辦公室的門開了。

所有人的視線——不約而同地——轉向那個方向。
不是因為有人故意看。
是——一種本能的反應。
像動物聽到了掠食者的腳步。

一個人走進來。

短髮。齊耳。每一根頭髮都在它應該在的位置。
深灰色套裝。剪裁精準。沒有一條多餘的線。
平底鞋。黑色的。鞋底很薄。
走路的聲音幾乎沒有——但你知道她在走。
像一隻貓。不。比貓更安靜。
貓偶爾會踩到東西。她不會。

她掃了一眼辦公室。
不是看人。是掃描。
像一台校準儀在讀取環境數據。
每一張桌子。每一個人。每一個螢幕上的數字。
她全部看了。
但她看起來什麼都沒看。

★★ 費雪的出場。

  這個角色的設計原則：
  她不壞。她精確。
  她不邪惡。她有效率。
  她做的每一件事都有「合理的理由」。
  這比純粹的惡更可怕。
  因為你沒辦法簡單地恨她。

她走到貝葉的桌前。
站在她面前。
距離剛好——不近不遠。
近到能看清楚她的臉。
遠到不會讓她覺得被侵犯。
精確的距離。

「陳貝葉工程師？」

「是。」

聲音穩定。小拉在背後悄悄說：
「心率 78。很好。保持。」

「我是費雪。安全部門。」

她微笑了。
那個微笑有精確的弧度。
不多不少。
剛好讓你覺得她是友善的。
但不會讓你放鬆。
就像一個 p-value 剛好在 0.05 的邊緣——
你不知道該拒絕還是接受。

「最近的校準數據我看了。」

停頓。
她看了一眼貝葉桌上的統計計算機。

「非常精確。」

又一個停頓。更長。
她的眼睛是深棕色的。幾乎是黑的。
瞳孔的大小似乎不會隨光線改變——
永遠是同一個直徑。像被校準過。

「精確得有點……不尋常。」

每一個字。
每一個音節。
都像被稱過重量。
「精確」——是事實。
「不尋常」——是暗示。
中間那個「有點」——是留給你自己填空的空間。

她的眼神沒有離開貝葉。
像一台校準儀在讀取數據。
不是讀表面的。是讀深層的。
像在看——
不是你做了什麼。
是你知道什麼。

「繼續保持。」

她轉身。走了。
走路的聲音很輕。
但每一步都在地板上留下了壓力。
不是物理的壓力。是——
一種讓你胸口發緊的壓力。

辦公室的空氣過了五秒鐘才恢復正常。

同事們慢慢轉回去。
開始打字。開始說話。
好像什麼都沒發生。
但鍵盤的聲音比剛才輕了一些。
說話的音量比剛才低了一些。

小拉的聲音在透鏡裡：
「貝姊……她剛才看你的時候，
 你的皮膚電導率上升了 340%。
 這通常代表——」

「我知道。」

她知道。
費雪知道了。
不一定知道全部。
但知道「有人在查」。
而且知道那個人是她。

→ 費雪沒有說一個字的威脅。
  但她的每一句話都是威脅。
  「精確得不尋常」= 「我知道你在做什麼。」
  「繼續保持」= 「我在看。」
  這比直接的威脅更可怕。
  因為直接的威脅你可以反抗。
  暗示——你不知道該怎麼反應。

```

```
值得嗎？

溫的。不是熱的。
```
