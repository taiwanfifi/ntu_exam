# 阿嬤（豆漿攤老闆娘）跨卷隱藏弧線

> **核心設計理念**：全書最不重要的角色，擁有最重要的證據。
> **主題共振**：身體的記憶比數位記憶更持久，最卑微的人生裡藏著最關鍵的真相。

---

## 基本設定

- **身份**：地下街 B 區豆漿攤老闆娘
- **記憶狀態**：第四階記憶負債者，被強制提取過三次記憶
- **僅存記憶**：做鹹豆漿和油條的手藝
- **日常習慣**：每天留一碗不賣的豆漿，收攤時倒掉

---

## 那碗永遠留著的豆漿——真相

```
在記憶數位化改變世界之前——很久以前。

周明哲還是台大神經科學系的研究生。年輕、禮貌、每天早上六點準時出現。
他會到地下街買一碗鹹豆漿配油條。
那時候阿嬤還記得一切——兒子的名字、自己的年齡、
以及這個每天來的年輕人叫什麼。

「周老師，今天的油條特別酥。」
「謝謝阿嬤。」他笑著拿走。

後來——研究成功了。記憶數位化問世。世界改變了。
他變成了「改變人類文明的偉人」。
他來的次數越來越少。然後不來了。
然後是「官方死亡日期：2079.11.02」。

三十年了。

阿嬤的記憶被提取了三次。
周明哲的名字、臉、聲音——全被抽走。

但她的身體記得：
每天六點，有一個人會來喝豆漿。
她不知道那個人是誰了。
但她知道該留一碗。

那碗永遠留著的豆漿，
是留給全書最大的反派的。
```

> **時間線備註（Round 14 審計）**：
> 「三十年」作為全書阿嬤弧線的核心意象數字，指的是她留一碗豆漿不賣的習慣延續的大約時間跨度。
> 周明哲作為「年輕研究生」的時期更早（MDP 發明前），但精確年份取決於周明哲出生年的最終確認。
> 詳見迭代紀錄 Round 14「時間線矛盾清單」。

## 她被提取的記憶中藏著什麼

```
阿嬤的三次記憶提取時間：
  第一次：2075年（還清醫療債務）—— 提取了家族記憶
  第二次：2078年（兒子的教育費）—— 提取了個人經歷記憶
  第三次：2079年（債務催收）—— 提取了社交記憶

關鍵：第三次提取發生在 2079 年 10 月 28 日。
      記憶大崩潰發生在 2079 年 11 月 2 日。

她的記憶在鸚鵡計畫啟動之前五天就被提取了。
這意味著：
  ✅ 她的記憶沒有被鸚鵡計畫「覆蓋」——因為已經不在她腦中了
  ✅ 她的記憶進入了黑市流通系統
  ✅ 在黑市的記憶池中，她的記憶是「未被篡改的原始版本」

她的第三次提取包含：
  - 三十年來地下街的日常——包括周明哲年輕時的身影
  - 2079年10月的最後幾天——遊行前的社會氣氛
  - 遊行當天早上（她的攤位在地下街出口，能看到地面上的遊行隊伍）

  ★ 她是唯一一個從「普通人視角」記錄了遊行前夕的人。
  ★ 不是監控畫面。不是新聞報導。是一個賣豆漿的老婦人的日常。
  ★ 她的記憶裡有：
    - 遊行隊伍經過地下街出口時的嘈雜聲
    - 一個年輕女人抱著一個四五歲的小女孩路過她的攤位
      （那個女人是邱韻如，小女孩是語青）
    - 小女孩停下來看她炸油條，邱韻如拉著女孩的手說「走了走了」
    - 遠處的催淚彈聲、尖叫聲、然後突然的寂靜
    - 一個面容模糊的男人（周明哲）在混亂中逆行走向人群中心
```

## 主題共振

```
阿嬤的弧線與全書三大主題的呼應：

1. 身體記憶 vs 數位記憶
   阿嬤的炸油條手藝 = 身體記得
   嚴柏翰的「手記得」= 身體記得
   維倫結尾的「心記得」= 身體記得
   → 三重呼應。全書用三個不同的角色證明同一件事：
     最底層的記憶不在數位晶片裡。在骨頭裡。

2. 最卑微的人 = 最關鍵的證據
   鸚鵡計畫清理了政客、探員、記者——所有「重要的人」。
   但它忽略了一個賣豆漿的老婦人。
   因為她「不重要」。
   結果，全城唯一未被篡改的遊行記錄，就在她的記憶裡。
   → 這是一個關於「被忽視的人」的隱喻。
     系統總是忽略邊緣人。但邊緣人記得系統不想讓任何人記得的事。

3. 反派的人性
   周明哲重寫了全世界。控制了所有人的記憶。
   但他年輕時只是一個每天喝豆漿的研究生。
   阿嬤記憶中的那個「周老師」，跟後來的「零號病人」是同一個人。
   → 阿嬤的記憶是全書中唯一保存了「周明哲的人性」的地方。
     不是他的天才。不是他的瘋狂。只是他喜歡喝豆漿。
     這讓最大的反派在最後一卷有了被理解的可能。
```

## 寫作提醒

```
阿嬤永遠不需要成為重要角色。
她不需要戲份。不需要台詞量。
她只需要每次出場時，安靜地觸動一根
跟全書主題相連的神經。

寫阿嬤的準則：
  - 不要讓她「知道」任何事。她什麼都不知道。
  - 讓她的身體「記得」一切。她什麼都記得。
  - 她的每次出場不超過 200 字。
  - 但每次出場後，讀者應該想哭。
  - 她是一碗豆漿。豆漿不需要解釋它自己。
```
