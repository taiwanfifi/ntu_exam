## 第 1 章：透鏡日常

### 基本資訊
```
字數目標：4,000-5,000 字
情緒色調：淡藍灰。溫暖的日常。像一個普通的梅雨季早晨。
POV：第一人稱（貝葉）
機率概念：機率公理、樣本空間
本章目標：讓讀者愛上這個世界。愛上透鏡。然後才有資格在後面心碎。
```

### 場景分鏡

```
【開場】醒來。
  貝葉被小拉叫醒。
  小拉：「早安貝姊。今日存活機率 99.987%。天氣——」
  貝葉：「不要叫我貝姊。」
  小拉：「好的貝姊。今日降雨機率 62%。分布 Beta(6.2, 3.8)。
         建議帶傘。但你上次也沒帶。所以我覺得你這次也不會——」
  貝葉拿起傘出門。小拉：「！今天帶了！我要修改先驗！」

  → 小拉的碎念建立「AI助理」的人格。輕鬆幽默。
  → 順帶帶出「先驗」這個詞。第一次出現，像口語。
  → 讀者不需要理解「先驗」是什麼。但這顆種子種下了。

【場景一】通勤。捷運。
  透鏡自動啟動。
  貝葉看著捷運月台上的人群。
  每個人頭上浮著微小的透鏡提示——
  但只有配戴者自己看得到。

  ┌─ 機率透鏡 v4.2 ─────────────────────────┐
  │ 捷運板南線                                │
  │ 下一班車到達時間：2 min 14 sec           │
  │ 到達分布：Normal(μ=134s, σ=12s)          │
  │ 車廂有座位的機率：P(seat) = 0.23         │
  │ 建議：等下一班 ██░░░░░░░░ 23%            │
  └───────────────────────────────────────────┘

  貝葉擠上了這一班。她從不聽「建議」。
  「建議是根據平均值算的。但我不是平均值。」
  → 第一次展示透鏡的 UI。讓讀者習慣這個格式。
  → 貝葉的個性：不完全依賴機率。她有自己的判斷。

【場景二】Oracular 總部。上班。
  描寫公司的環境：乾淨、精確、每個角落都有數據。
  同事們打招呼的方式：
  「今天你的校準指數多少？」「87.3。你呢？」「89.1。」

  貝葉的桌上：半杯昨天的珍奶（已經分層了）、統計計算機、橘色招財貓。
  旁邊的同事林岳正在跑數據。
  林岳：「貝葉，你覺得 Poisson(λ=2.1) 的殘差長這樣正常嗎？」
  貝葉瞄了一眼：「正常。λ 在 2 附近的 Poisson 本來就偏態大。怎麼了？」
  林岳：「沒事。只是......覺得有點太整齊了。」
  貝葉：「那就多跑幾組看看。」
  林岳：「嗯。」
  → 林岳的伏筆。他也注意到了。但比貝葉早。
  → 這段對話在第 19 章回看時會讓讀者心痛。

【場景三】午休。便當。
  小拉自動分析便當的營養：
  「這個排骨便當會讓你今日攝取超標的機率：0.71。
   但你的快樂指數預估上升 0.34。
   ......所以嚴格來說，期望效用是正的。吃吧。」
  貝葉：「你現在是營養師嗎？」
  小拉：「我什麼都是。因為沒有人會另外請一個 AI 來當營養師。
         所以所有雜事都是我的。就像實習生一樣。嗚嗚嗚。」
  → 小拉的碎念提供輕鬆時刻。
  → 同時帶出「期望值」的概念，用口語方式。

【場景四】下午的校準工作。
  → 這是本章的核心場景。詳見下方「機率概念呈現」。

  貝葉接到日常校準任務。
  描寫「校準」的流程：
  ① 打開數據面板。
  ② 檢查指定區域的歷史觀測。
  ③ 確認先驗分布的參數。
  ④ 跟觀測數據比對。
  ⑤ 如果吻合 → 蓋章通過。如果不吻合 → 手動調整。

  今天的任務很普通。三個區域。全部正常。
  「修得好。」（自言自語。蓋章的聲音。）

【收尾】下班。珍奶。
  貝葉買珍奶。半糖少冰。
  走在回家的路上。
  透鏡投射了最後一條訊息：
  「今晚感到孤獨的機率：78%。」
  她把透鏡切到夜間模式。數字消失了。
  但孤獨沒有消失。

  → 最後用「機率量化孤獨」建立貝葉的內在矛盾：
    她活在一個什麼都能被量化的世界。
    但量化並不等於解決。
```

### 機率概念呈現

```
【機率公理 — 融入場景的方式】
  貝葉在做校準時，內心獨白（或對小拉的解釋）：

  「校準的第一步：確認樣本空間。
   也就是——所有可能發生的事。
   一個十字路口，30 秒內可能發生什麼？
   {0 輛車, 1 輛車, 2 輛車, 3 輛車, ......}
   這就是樣本空間 Ω。

   第二步：確認機率公理。
   所有事件的機率 ≥ 0。
   所有可能的事件加起來 = 1。
   互斥事件的機率可以直接相加。

   聽起來很簡單。但你知道嗎——
   透鏡看到的每一個數字，都建立在這三條公理上。
   如果公理出了問題......一切都會崩塌。」

  → 不是教科書式的解釋。是她工作時的自然思維。
  → 「如果公理出了問題」= 伏筆。全書的核心威脅。
```

### 關鍵台詞

```
① 小拉：「好的貝姊。」（第一次出現。全書反覆回響。）
② 貝葉：「建議是根據平均值算的。但我不是平均值。」
③ 貝葉：「修得好。」（自我確認。像程式裡的 return true。）
④ 透鏡：「今晚感到孤獨的機率：78%。」
```

### 章末鉤子

```
回到家。打開門。
貝葉看到陽台上的薄荷。快死了。
她給它澆了水。
「你的存活機率也不高啊。但你還是每天長一點。」
薄荷沒有回答。
她笑了一下。是今天第一次笑。

→ 溫暖的結尾。讓讀者喜歡貝葉。
→ 薄荷也是母題：一個「不看機率也在努力活著」的生命。
```

## 第 2 章：校準悲傷的人

### 基本資訊
```
字數目標：4,500-5,500 字
情緒色調：暖灰轉溫金。從日常進入「第一次被感動」的瞬間。
POV：第一人稱（貝葉）
機率概念：樣本空間（深化）、條件機率（引入）
本章目標：用一個老先生的故事，建立「校準」的溫暖意義。
         同時讓讀者第一次流淚。
```

### 場景分鏡

```
【開場】特殊校準任務。
  主管周副理走過來：「貝葉，有一個到府校準的案子。
  老先生 75 歲。透鏡數值全面偏高。去看看。」
  貝葉：「全面偏高？機械故障嗎？」
  周副理：「可能。也可能......是認知基線偏移。」
  周副理的語氣有點奇怪。不是公事公辦——是溫柔。
  「對了，帶點心。老先生可能......需要有人陪他說說話。」

【場景一】老先生的家。
  公寓。乾淨但空曠。
  客廳有兩張沙發。一張明顯有人長期坐的凹痕。另一張很新。
  老先生叫鄧伯伯。說話慢慢的。眼神溫和但有霧。

  貝葉開始校準。
  打開校準面板——老先生的透鏡數據全面偏高 15%。

  ┌─ 校準面板 ─────────────────────────────────┐
  │ 用戶：鄧XX（75 歲，男性）                  │
  │ 校準異常：全域偏高 +15%                     │
  │                                              │
  │ 降雨機率：87%（實際 62%）                   │
  │ 路口事故：23%（實際 8%）                    │
  │ 店家倒閉：45%（實際 19%）                   │
  │                                              │
  │ 所有數值均偏高。非機械故障。                │
  │ 判定：認知基線偏移（Cognitive Baseline Shift）│
  └──────────────────────────────────────────────┘

  「鄧伯伯，您的透鏡會讓所有事看起來比實際更危險。」
  鄧伯伯笑了一下：「是這樣啊。」
  「這個偏移大概是什麼時候開始的？」
  鄧伯伯沉默了。看了一眼旁邊那張沒人坐的沙發。
  「......上個月。我太太走了之後。」

  → 認知基線偏移的真正原因：悲傷。

【場景二】校準。
  技術層面：貝葉調整先驗的參數。
  但她發現——老先生的「主觀先驗」不是問題。
  問題是他的「認知基線」本身被悲傷改變了。
  透鏡的算法會根據使用者的「認知狀態」自動調整顯示方式。
  悲傷的人看什麼都更悲觀 → 透鏡的輸出跟著偏高。

  ※ 這裡帶出「條件機率」的概念：
  P(壞事|悲傷中) ≠ P(壞事)
  在「悲傷」這個條件下，主觀機率會偏移。
  透鏡忠實地反映了這個偏移——但這不代表世界真的變危險了。

  貝葉對小拉說：
  「小拉，這不是機械問題。是條件機率的問題。
   他的透鏡在算的是 P(事件|他的情緒狀態)，
   而不是 P(事件|客觀環境)。」
  小拉：「所以......他的悲傷改變了他看到的世界？」
  貝葉：「不是改變了世界。是改變了他的先驗。」

  → 用一個老人的悲傷，自然帶出「條件機率」和「先驗偏移」。
  → 讀者理解：先驗不只是數學——是信念。悲傷可以改變信念。

【場景三】校準完成。窗外。
  貝葉手動校準了老先生的認知基線。
  數值回到正常。
  老先生看了看窗外。天還是灰的。但透鏡上的數字不同了。

  「小姐......雨停的機率是多少？」
  貝葉看了一下：「目前是 58%。」
  老先生：「58%......蠻高的。」
  停頓。
  「原來雨停的機率是這麼高啊。」
  他的眼角有淚。
  不是因為數字。是因為——
  在太太走了之後，他第一次覺得「也許雨會停」。

  ★★ 全書第一個催淚點。

【收尾】回程。
  貝葉在公車上。戴著耳機但沒放音樂。
  小拉的數據顯示她的心率在老先生說那句話的時候降到了 55。
  小拉：「你剛才......怎麼了嗎？」
  貝葉：「沒事。」
  小拉：「你的淚腺在你說『沒事』的時候啟動了。
         以我的數據庫來看，這通常不叫『沒事』。」
  貝葉沒有回答。
  她在想：校準可以修正數字。但能修正悲傷嗎？

  她不知道。
  但她知道——那個老先生看到 58% 的時候，
  笑了。
  那就夠了。
```

### 機率概念呈現

```
【條件機率 — 融入場景的方式】
  P(壞事發生|悲傷) > P(壞事發生)
  → 悲傷不改變世界。但改變你看世界的方式。
  → 透鏡忠實反映了這個主觀偏移。
  → 校準的工作 = 分離「主觀先驗」和「客觀先驗」。

  這個概念不需要公式。
  讀者只需要理解：「一個悲傷的人，看到的機率跟快樂的人不一樣。」
  這就是條件機率的本質。
```

### 關鍵台詞

```
① 鄧伯伯：「原來雨停的機率是這麼高啊。」 ★★
② 貝葉：「不是改變了世界。是改變了他的先驗。」
③ 小拉：「你的淚腺在你說『沒事』的時候啟動了。」
④ 周副理：「帶點心。老先生可能需要有人陪他說說話。」
```

## 第 3 章：白奶奶的先驗

### 基本資訊
```
字數目標：3,500-4,000 字
情緒色調：暖黃。蔥油餅的焦香色。
POV：第一人稱（貝葉）
機率概念：主觀 vs 客觀機率（深化）
本章目標：建立白奶奶的角色。建立蔥油餅攤的「聖地」感。
         第一次暗示蘇立恆的存在。
```

### 場景分鏡

```
【場景一】週末。龍山寺旁。
  貝葉每個月會來萬華一次。
  買蔥油餅 → 去龍山寺拜拜（她自己也不確定為什麼要拜）→ 逛夜市。

  白奶奶的攤位在龍山寺左手邊的巷口。
  四十年了。鐵板生鏽了三層，但從不換。
  「換了就不是那個味道了。」

  白奶奶看到貝葉：
  「哎呀，陳家的那個丫頭。來啊來啊。今天的蔥特別香。」
  （每次都說。但貝葉每次聽到都覺得比較好了。）

【場景二】關於透鏡的對話。
  貝葉邊吃邊問：「奶奶，你真的不想裝透鏡嗎？」
  白奶奶：「不要。」
  「為什麼？」
  白奶奶翻餅。鐵板滋滋作響。
  「我做蔥油餅四十年。每一張都不一樣。
   麵團的溼度、蔥的量、火候、天氣。
   你那個透鏡可以告訴我下一張餅會不會好吃嗎？」
  貝葉想了想：「理論上......可以算出一個機率。」
  白奶奶笑了：「那你就不懂了。
   做餅這件事，不知道下一張會不會好吃——
   這才是有趣的地方。」

  → 白奶奶用「蔥油餅」說出了全書最核心的哲學：
    不確定性不是敵人。不確定性是人之所以為人的一部分。
  → 這段對話在第 58 章貝葉面對柏朗時會被呼應。

【場景三】穿黑衣的人。
  白奶奶壓低聲音：
  「對了。最近有一個穿黑衣服的少年仔每天來坐。
   不買東西。就坐在對面的石椅上。看著我做餅。」
  貝葉：「認識嗎？」
  白奶奶：「不認識。但他看人的眼神啊......
   跟你很像。都是那種『在找什麼東西』的眼神。」

  貝葉看了一眼石椅。空的。上面放了一張蔥油餅。
  「那是——」
  白奶奶：「多做的。他有時候會拿。有時候不拿。
   不拿我就丟掉。沒什麼。」

  → 蘇立恆的第一次「出場」——透過白奶奶的描述。
  → 石椅上的蔥油餅 = 母題。

【收尾】離開前。
  貝葉付錢。白奶奶不收。
  「你每次都多給。我又不是不會算。」
  貝葉：「奶奶，你不用透鏡也會算？」
  白奶奶拍了她一下：「呸！我算帳算了四十年。
   你那個透鏡算的叫機率，我算的叫人情。不一樣的！」

  貝葉走的時候回頭看了一眼。
  白奶奶在擦鐵板。嘴裡哼著歌。
  她不知道明天的天氣。不知道明天的客人會不會來。
  不知道下一張蔥油餅好不好吃。
  但她不怕。

  貝葉的透鏡提示：
  「目前使用者情緒狀態：平靜。」
  ......在離開白奶奶之後，她的機率分布總是比較穩定。
  也許白奶奶才是最好的校準師。
```

### 機率概念呈現

```
【主觀 vs 客觀機率 — 融入場景的方式】
  白奶奶的「直覺」vs 透鏡的「計算」。

  客觀機率：基於數據。可重複驗證。透鏡的計算方式。
  主觀機率：基於經驗和信念。白奶奶「膝蓋比電腦準」。

  哪個更準？
  答案是：看情境。
  有些事，數據量夠大，客觀機率更可靠。
  有些事——比如「下一張蔥油餅好不好吃」——
  四十年經驗的主觀先驗，可能比任何模型都準。
```

## 第 4 章：十一秒

### 基本資訊
```
字數目標：4,000-5,000 字
情緒色調：灰色轉冷藍。回憶的顏色。
POV：第一人稱（貝葉）
機率概念：條件機率（深化）、全機率公式（暗示）
本章目標：揭露貝葉的過去。父親的車禍。十一秒。
         解釋她為什麼加入 Oracular。
```

### 場景分鏡

```
【開場】高思遠的週末聚餐。
  台大附近的小餐館。滷肉飯加荷包蛋。
  高思遠的固定點餐。二十年沒變。
  「你爸以前也點這個。」
  貝葉：「我知道。老高你每次都說。」
  高思遠笑了。然後咳嗽了一聲。比上次多咳了一聲。

  → 高思遠健康的伏筆。每次出場多咳一聲。

【場景一】高思遠的問題。
  「最近的校準工作怎麼樣？有沒有什麼......不尋常的？」
  （他每次都問。但今天的語氣跟以前不一樣。多了一個停頓。）
```
