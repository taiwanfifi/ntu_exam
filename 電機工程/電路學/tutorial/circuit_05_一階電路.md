# 第五章：一階電路（First-Order Circuits）

> **適用對象**：零基礎大一大二學生
> **先修知識**：第一至三章（KVL、KCL、戴維寧等效）、基本微積分
> **學習時間**：約 4-5 小時

---

## 🔰 **本章基礎觀念（零基礎必讀）**

到目前為止，我們處理的都是「電阻電路」——電路中只有電阻，電壓和電流**不隨時間改變**。

但現實世界的電路通常包含**電容（Capacitor）** 和 **電感（Inductor）**——它們能夠**儲存能量**，讓電壓和電流隨時間變化。

### 水流比喻

- **電容**就像一個**水塔**：可以儲水（儲存能量），水塔裝滿後水慢慢流出（放電）
- **電感**就像一個**旋轉的水輪**：一旦開始轉就很難馬上停下來（電流不能瞬間改變）

### 電容和電感的基本關係

| 元件 | 符號 | 基本方程式 | 儲能公式 | 單位 |
|------|------|-----------|---------|------|
| 電容 | C | i = C(dv/dt) | W = ½Cv² | 法拉(F) |
| 電感 | L | v = L(di/dt) | W = ½Li² | 亨利(H) |

> 重要特性：
> - **電容的電壓不能瞬間改變**（v_C(0⁺) = v_C(0⁻)）
> - **電感的電流不能瞬間改變**（i_L(0⁺) = i_L(0⁻)）

## 一、RC 電路的自然響應（Natural Response）

### 1.1 什麼是自然響應？

自然響應 = **沒有外部電源驅動**，電路靠自身儲存的能量運作。

就像拔掉充電器後，手機電池的電量慢慢下降。

### 1.2 電路描述

一個充好電的電容 C（初始電壓 V₀）和一個電阻 R 構成迴路：

```
    ┌───[R]───┐
    |         |
   [C] V₀    |
    |         |
    └─────────┘
```

### 1.3 推導

對迴路列 KVL：

$$
v_C + v_R = 0
$$

$$
v_C + RC\frac{dv_C}{dt} = 0
$$

（因為 i = C(dv_C/dt)，而 v_R = iR = RC(dv_C/dt)）

整理：

$$
\frac{dv_C}{dt} + \frac{v_C}{RC} = 0
$$

這是一階線性常微分方程式，解為：

$$
\boxed{v_C(t) = V_0 \, e^{-t/\tau}, \quad \tau = RC}
$$

### 1.4 時間常數 τ = RC

| 時間 | v_C 剩餘 | 百分比 |
|------|---------|--------|
| t = 0 | V₀ | 100% |
| t = τ | V₀/e ≈ 0.368V₀ | 36.8% |
| t = 2τ | V₀/e² ≈ 0.135V₀ | 13.5% |
| t = 3τ | V₀/e³ ≈ 0.050V₀ | 5.0% |
| t = 5τ | V₀/e⁵ ≈ 0.007V₀ | 0.7% |

> 工程實務上：經過 **5τ** 後，認為電容已經「完全放電」（剩不到 1%）。

### 1.5 τ 的物理意義

- τ 越大 → 放電越慢（大水塔放水慢）
- R 大 → 電流小 → 放電慢
- C 大 → 儲能多 → 放電慢
- τ 的單位：秒（Ω × F = s）

### 1.6 電流的自然響應

$$
i(t) = C\frac{dv_C}{dt} = C \cdot \left(-\frac{V_0}{\tau}\right) e^{-t/\tau} = -\frac{V_0}{R} e^{-t/\tau}
$$

負號代表電流方向是放電方向（從 + 端流出）。

### 1.7 數值例題 ①

> **題目**：RC 電路中 R = 5kΩ，C = 2μF，電容初始電壓 V₀ = 10V。
> (a) 求時間常數 τ
> (b) 求 t = 5ms 時的電容電壓
> (c) 大約多久後電容「完全放電」？

**解答**：

(a)
$$
\tau = RC = 5 \times 10^3 \times 2 \times 10^{-6} = 10 \times 10^{-3} = 10 \text{ ms}
$$

(b)
$$
v_C(5\text{ms}) = 10 \cdot e^{-5/10} = 10 \cdot e^{-0.5} = 10 \times 0.6065 = 6.065 \text{ V}
$$

(c) 完全放電 ≈ 5τ = 5 × 10ms = **50 ms**

答：**τ = 10ms，v(5ms) ≈ 6.07V，約 50ms 後完全放電**。

## 二、RL 電路的自然響應

### 2.1 電路描述

一個已建立電流的電感 L（初始電流 I₀）和電阻 R 構成迴路：

```
    ┌───[R]───┐
    |         |
   [L] I₀    |
    |         |
    └─────────┘
```

### 2.2 推導結果

$$
\boxed{i_L(t) = I_0 \, e^{-t/\tau}, \quad \tau = \frac{L}{R}}
$$

### 2.3 對比 RC 和 RL

| 項目 | RC 電路 | RL 電路 |
|------|---------|---------|
| 衰減的量 | 電容電壓 v_C | 電感電流 i_L |
| 時間常數 | τ = RC | τ = L/R |
| τ 增大的方式 | R↑ 或 C↑ | L↑ 或 R↓ |
| 初始條件 | v_C(0) = V₀ | i_L(0) = I₀ |

> 注意：RL 的 τ = L/R，R 在分母！（R 越大衰減越快，和 RC 相反）

### 2.4 數值例題 ②

> **題目**：RL 電路中 R = 8Ω，L = 4H，初始電流 I₀ = 6A。
> (a) 求時間常數
> (b) 求 t = 0.25s 時的電流
> (c) 求 t = 0.25s 時電感上的電壓

**解答**：

(a)
$$
\tau = \frac{L}{R} = \frac{4}{8} = 0.5 \text{ s}
$$

(b)
$$
i_L(0.25) = 6 \cdot e^{-0.25/0.5} = 6 \cdot e^{-0.5} = 6 \times 0.6065 = 3.639 \text{ A}
$$

(c) 電感電壓：
$$
v_L = L\frac{di_L}{dt} = 4 \times 6 \times \left(-\frac{1}{0.5}\right) e^{-0.25/0.5} = -48 \times 0.6065 = -29.11 \text{ V}
$$

或用 v_L = -Ri_L = -8 × 3.639 = -29.11V（KVL 得 v_L + v_R = 0）

答：**τ = 0.5s，i(0.25s) ≈ 3.64A，v_L(0.25s) ≈ -29.11V**。

## 三、步階響應（Step Response）

### 3.1 什麼是步階響應？

步階響應 = 在 t = 0 時突然**接上直流電源**，電路從初始狀態過渡到最終穩態。

就像打開水龍頭的瞬間：水不會立刻達到最大流量，而是逐漸增加。

### 3.2 通用公式（超級重要！）

$$
\boxed{x(t) = x(\infty) + [x(0) - x(\infty)] \, e^{-t/\tau}}
$$

其中：
- x(t)：任意時刻的電壓或電流
- x(0)：初始值（t = 0 時）
- x(∞)：最終值（t → ∞ 時，即穩態值）
- τ：時間常數

> 這個公式可以解**所有**一階電路的問題！只需要找出三個東西：x(0)、x(∞)、τ。

### 3.3 如何找 x(0)、x(∞)、τ？

| 要找什麼 | 怎麼找 |
|----------|--------|
| x(0)：初始值 | 分析 t < 0 的穩態（電容 = 斷路，電感 = 短路） |
| x(∞)：最終值 | 分析 t → ∞ 的穩態（電容 = 斷路，電感 = 短路） |
| τ：時間常數 | 找「看到」電容（或電感）的等效電阻 Rth，τ = RthC（或 L/Rth） |

### 3.4 為什麼穩態時「電容 = 斷路，電感 = 短路」？

- 電容：穩態時 dv/dt = 0 → i = C(dv/dt) = 0 → 沒有電流 → 等效斷路
- 電感：穩態時 di/dt = 0 → v = L(di/dt) = 0 → 沒有壓降 → 等效短路

### 3.5 RC 步階響應的推導

電容初始電壓 V₀，在 t = 0 接上 Vs 電壓源和 R：

$$
v_C(\infty) = V_s \quad (\text{最終電容充電到 } V_s)
$$

$$
v_C(0) = V_0 \quad (\text{初始值})
$$

$$
v_C(t) = V_s + (V_0 - V_s) \, e^{-t/RC}
$$

如果 V₀ = 0（電容從零開始充電）：

$$
v_C(t) = V_s (1 - e^{-t/RC})
$$

### 3.6 數值例題 ③

> **題目**：RC 電路中 R = 10kΩ，C = 50μF，初始電容電壓為 0V。t = 0 時接上 20V 電壓源。
> (a) 求 τ
> (b) 寫出 v_C(t) 的表達式
> (c) 求 t = 0.5s 時的電容電壓
> (d) 多久後電容電壓達到 20V 的 90%？

**解答**：

(a)
$$
\tau = RC = 10 \times 10^3 \times 50 \times 10^{-6} = 0.5 \text{ s}
$$

(b)
$$
v_C(t) = 20(1 - e^{-t/0.5}) = 20(1 - e^{-2t}) \text{ V}
$$

(c)
$$
v_C(0.5) = 20(1 - e^{-1}) = 20(1 - 0.368) = 20 \times 0.632 = 12.64 \text{ V}
$$

(d) 要求 v_C = 0.9 × 20 = 18V：

$$
18 = 20(1 - e^{-2t})
$$
$$
0.9 = 1 - e^{-2t}
$$
$$
e^{-2t} = 0.1
$$
$$
-2t = \ln(0.1) = -2.303
$$
$$
t = 1.151 \text{ s} \approx 2.3\tau
$$

答：**τ = 0.5s，v_C(0.5s) ≈ 12.64V，約 1.15s 後達到 90%**。

### 3.7 數值例題 ④

> **題目**：RL 步階響應。R = 4Ω，L = 2H，初始電流 I₀ = 0A，t = 0 接上 12V。
> (a) 求最終電流
> (b) 寫出 i_L(t) 表達式
> (c) 求 t = 1s 時的電流

**解答**：

(a) 最終穩態：電感 = 短路，I(∞) = V/R = 12/4 = 3A

τ = L/R = 2/4 = 0.5s

(b)
$$
i_L(t) = 3 + (0 - 3)e^{-t/0.5} = 3(1 - e^{-2t}) \text{ A}
$$

(c)
$$
i_L(1) = 3(1 - e^{-2}) = 3(1 - 0.1353) = 3 \times 0.8647 = 2.594 \text{ A}
$$

答：**I(∞) = 3A，i_L(1s) ≈ 2.59A**。

### 3.8 數值例題 ⑤

> **題目**：一個 RC 電路，電容已經充電到 8V。t = 0 時電路改接到 2V 電壓源。R = 1kΩ，C = 10μF。
> 求 v_C(t)。

**解答**：

用通用公式：

- x(0) = v_C(0) = 8V
- x(∞) = 2V（最終充到電源電壓）
- τ = RC = 1000 × 10×10⁻⁶ = 0.01s = 10ms

$$
v_C(t) = 2 + (8 - 2)e^{-t/0.01} = 2 + 6e^{-100t} \text{ V}
$$

答：**v_C(t) = 2 + 6e^{-100t} V**。

電壓從 8V 指數衰減到 2V。

## 四、初始條件和最終條件的判斷技巧

### 4.1 t < 0 的穩態（找初始條件）

假設開關已經維持很久（穩態）：
- **電容 → 斷路**（開路）
- **電感 → 短路**（導線）

然後分析電路，求出 v_C(0⁻) 或 i_L(0⁻)。

### 4.2 t = 0 的連續性

- v_C(0⁺) = v_C(0⁻)（電容電壓連續）
- i_L(0⁺) = i_L(0⁻)（電感電流連續）

### 4.3 t → ∞ 的穩態（找最終條件）

再次假設到達穩態：
- **電容 → 斷路**
- **電感 → 短路**

分析新的電路（開關動作後的電路），求出 v_C(∞) 或 i_L(∞)。

### 4.4 決策流程圖

```
收到一階電路題目
    |
    v
找出儲能元件是 C 還是 L？
    |
    v
求 x(0)：分析 t < 0 穩態
    |         C→斷路，L→短路
    v
求 x(∞)：分析 t → ∞ 穩態
    |         C→斷路，L→短路
    v
求 τ：找等效電阻 Rth
    |    τ = RthC 或 L/Rth
    v
代入通用公式：
x(t) = x(∞) + [x(0) - x(∞)]e^{-t/τ}
    |
    v
完成！
```

## 五、題型鑑別指南

| 看到什麼 | 用什麼方法 |
|----------|-----------|
| 只有 RC 或 RL，無電源 | 自然響應（指數衰減到零） |
| RC 或 RL + 直流電源 | 步階響應（用通用公式） |
| 「求時間常數」 | τ = RC 或 τ = L/R |
| 「多久後達到某百分比」 | 用公式解 t |
| 開關在 t = 0 動作 | 分析 t < 0（初始）和 t → ∞（最終） |
| 電路比較複雜 | 先求等效戴維寧電路再分析 |
| 「初始條件」 | 電容電壓連續、電感電流連續 |

## 六、關鍵術語表

| 中文 | 英文 | 白話解釋 | 例子 |
|------|------|---------|------|
| 一階電路 | First-order Circuit | 含一個電容或一個電感的電路 | RC 或 RL 電路 |
| 電容 | Capacitor (C) | 儲存電能的元件（像水塔） | i = C(dv/dt)，單位法拉(F) |
| 電感 | Inductor (L) | 儲存磁能的元件（像飛輪） | v = L(di/dt)，單位亨利(H) |
| 自然響應 | Natural Response | 無外部電源，靠自身能量衰減 | 電容放電 |
| 步階響應 | Step Response | 突然接上電源後的過渡過程 | 電容充電 |
| 時間常數 | Time Constant (τ) | 衰減到 36.8% 所需時間 | RC 或 L/R |
| 穩態 | Steady State | 不再隨時間改變的最終狀態 | t → ∞ 時 |
| 暫態 | Transient | 從初始到穩態的過渡過程 | 指數衰減的部分 |
| 初始條件 | Initial Condition | t = 0 時的電壓/電流值 | v_C(0)、i_L(0) |
| 連續性 | Continuity | 電容電壓和電感電流不能跳變 | v_C(0⁺) = v_C(0⁻) |
| 等效電阻 | Thevenin Resistance | 從儲能元件端看到的電阻 | 用來算 τ |

## ✅ 自我檢測

### 題目 1
RC 電路中 R = 2kΩ，C = 100μF。求時間常數，並問大約多久後電容「完全放電」？

<details>
<summary>點擊查看解答</summary>

τ = RC = 2000 × 100 × 10⁻⁶ = 0.2 s = 200 ms

完全放電 ≈ 5τ = 5 × 0.2 = **1 秒**

</details>

### 題目 2
為什麼電容的電壓不能瞬間改變？

<details>
<summary>點擊查看解答</summary>

因為 i = C(dv/dt)。

如果電壓瞬間改變（dv/dt → ∞），則需要無窮大的電流，這在物理上不可能。

能量角度：電容儲能 W = ½Cv²，瞬間改變 v 意味著瞬間改變能量，需要無窮大功率。

</details>

### 題目 3
RL 電路中 L = 0.5H，R = 100Ω，I₀ = 2A。求 t = 2.5ms 時的電流。

<details>
<summary>點擊查看解答</summary>

τ = L/R = 0.5/100 = 5 ms

i(t) = 2 × e^(-t/0.005)

i(2.5ms) = 2 × e^(-0.0025/0.005) = 2 × e^(-0.5) = 2 × 0.6065 = **1.213 A**

</details>

### 題目 4
一個 RC 步階響應：V₀ = 0V，Vs = 15V，R = 50kΩ，C = 4μF。寫出 v_C(t) 並求 t = 0.3s 時的電壓。

<details>
<summary>點擊查看解答</summary>

τ = RC = 50000 × 4 × 10⁻⁶ = 0.2 s

v_C(t) = 15(1 - e^{-t/0.2}) = 15(1 - e^{-5t})

v_C(0.3) = 15(1 - e^{-1.5}) = 15(1 - 0.2231) = 15 × 0.7769 = **11.65 V**

</details>

### 題目 5
通用公式 x(t) = x(∞) + [x(0) - x(∞)]e^{-t/τ} 中的三個關鍵量分別怎麼找？

<details>
<summary>點擊查看解答</summary>

1. **x(0) 初始值**：分析 t < 0 的穩態（假設開關動作前已經很久），電容視為斷路，電感視為短路。利用連續性：v_C(0⁺) = v_C(0⁻)，i_L(0⁺) = i_L(0⁻)。

2. **x(∞) 最終值**：分析 t → ∞ 的新穩態（開關動作後的電路已經很久），電容再次視為斷路，電感再次視為短路。

3. **τ 時間常數**：找從儲能元件端子看進去的戴維寧等效電阻 Rth（關掉所有獨立源後），然後 τ = Rth × C 或 τ = L / Rth。

</details>
