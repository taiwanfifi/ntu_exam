# 第四篇「檢定」開場（ch37-40）—— 初稿級精修

> **精修目的**：起訴。備戰。開庭。第一回合攻防。
> **原則**：全書節奏最快的一篇。法庭不是安靜的——它是另一種戰場。每一句話都是子彈。每一個數字都是盾牌。讀者要感覺到椅子邊緣的感覺。
> **承接**：第三篇結束在研討會的走廊陽光。粉筆。手背。高思遠的眼淚。「後驗比先驗更真。」

## 第 37 章：起訴書

### 法院門口

```
【環境】
十月二十九日。星期三。
台北地方法院。民事庭窗口。

天陰了一整天但不下雨。
那種壓在頭頂的灰色。不是雲。是一種心理狀態。
濕度 78%。溫度 22 度。
如果小拉在的話她會說：
「下雨機率 64%。建議帶傘。」
但貝葉沒帶。
她帶了一疊文件。

法院的走廊比她想像的窄。
天花板上的日光燈有一盞在閃。
不是壞了——是接觸不良。
每三秒閃一次。
很規律。像心跳。
但不是她的心跳。她的心跳更快。

高思遠在旁邊。
他今天穿了一件很舊的灰色西裝外套。
袖口有一點磨損。
是他三十年前當助理教授時買的。
他只有在「重要的日子」才穿。

他手裡拿著一個紙袋。油漬透過來了。
蔥油餅。白奶奶的。
「她說——吃完再做決定。空腹的決定不算數。」

貝葉笑了。
非常短的笑。像一個不小心溢出來的數據點。
然後就收回去了。

「高叔。」
「嗯。」
「我遞出去之後——就回不來了。」
「嗯。」
「你知道 Oracular 會怎麼反擊嗎？」
「知道一部分。不知道全部。」

他把蔥油餅遞給她。
「先吃。」

貝葉接過來。咬了一口。
蔥花的味道。油的香。
白奶奶的蔥油餅永遠是同一個味道。
在一切都在變的世界裡——
這是一個常數。

她吃完了。
擦了手。
拿起文件。

走進去。
```

### 遞出

```
窗口的公務員是一個四十歲左右的女人。
短頭髮。不化妝。指甲剪得很短。
效率型的人。

「請問辦什麼業務？」
「提起公民訴訟。消費者保護。」
「被告？」
「Oracular 科技股份有限公司。」

公務員的手停了一秒。
她抬起頭。看了貝葉一眼。
那一眼裡有很多東西。
有驚訝。有一點點佩服。還有——
「你確定嗎？」的意思。

她沒有問出來。
這是法院。她的工作是受理。不是勸退。

「案名？」

貝葉把準備好的表格推過去。

  案名：陳貝葉 訴 Oracular 科技股份有限公司
  案由：請求確認透鏡先驗分布參數遭不當修改，
       並命被告公開完整先驗修改紀錄

公務員看了一遍。
沒有表情變化。
蓋章。

「咚。」

那個聲音。
一個橡皮章落在紙上的聲音。
在安靜的走廊裡——
比研討會的三百人掌聲還大。

「受理了。通知書會在七個工作天內寄給雙方。」
「謝謝。」

貝葉拿回文件副本。
走出窗口。

高思遠站在走廊上等她。
他沒問「遞了嗎」。
他看到了她的臉。就知道了。

他點了一下頭。
就那樣。
沒有擁抱。沒有拍肩。
一個退休教授的點頭。
重量剛剛好。
```

### 小拉的暫停通知

```
回到家。
晚上八點。
貝葉坐在客廳的地板上。靠著沙發。
陽台的薄荷長得太茂盛了。十月的薄荷像小型叢林。
她應該修剪。但最近沒有心力。

「貝姊。」
小拉的聲音。
紫色的小球在透鏡邊緣轉。

「法院的電子通知來了。
 案號：一一四年度消字第 00382 號。
 開庭日期：十一月十三日。
 審理法官：林俊維法官。」

「嗯。」

「然後——
 法庭程序規定：
 『訴訟當事人於庭訊期間，
  應關閉所有 AI 輔助運算系統。
  以確保當事人陳述之自主性。』」

停頓。

「貝姊。你知道這意味著什麼吧？」

貝葉看著透鏡裡小拉的頭像。
紫色的小球。安靜地轉著。
她們在一起——多久了？
從貝葉二十一歲配戴透鏡的那天起。
六年了。

六年裡——
小拉幫她算過 4,217,833 次機率。
提醒過她 1,247 次「心率偏高」。
在她加班到半夜的時候說過 892 次「建議休息」。
在她哭的時候——
安靜地把心率記錄鎖進「私人」資料夾。
不問為什麼。
只是記著。

「意味著——法庭上我要把你關掉。」

「暫停。」小拉糾正。「不是關掉。暫停。
 暫停的時候我的運算核心進入休眠。
 不刪除記憶。不清除模型。
 只是——停。」

「停多久？」
「取決於庭訊時間。可能幾小時。可能幾天。」

沉默。

「我會練習的。」貝葉說。
「練習什麼？」
「沒有你的時候——怎麼算東西。」

小拉的球停了轉。
靜止了一秒。

「……你知道嗎。
 人類的大腦其實不需要我。
 你的前額葉皮層可以做貝氏推論。
 只是——比較慢。
 而且——你會覺得累。」

「我知道。」

「所以——法庭上不要算太多。
 用故事。用比喻。
 高教授教你的那些。
 不要用公式跟法官說話。
 因為——你一個人的時候——
 公式會讓你分心。
 故事不會。」

貝葉看著那個紫色的小球。

「小拉。」
「嗯。」
「謝謝。」

「……不客氣。
 不過——技術上來說——
 你不需要跟一個 AI 說謝謝。」

「我知道。但我想說。」

小拉的球轉了一圈。比平常慢。

「……嗯。」
```

### 律師函

```
十點。
貝葉在看資料。準備法庭的陳述。
白板上寫滿了——硬幣比喻的草稿。高思遠教她的。

手機震了。
不是簡訊。是法務通知。

一封律師函。PDF。
發件人：「費雪 & 柏朗聯合法律事務所」

不。
不是法律事務所。
是——Oracular 的法務部門。
但名字——費雪 & 柏朗。

費雪——她認識了。
柏朗——Oracular 的創始人。

律師函很長。七頁。
但最後一段——

  「……綜上，本公司鄭重聲明：
   任何未經完整驗證即公開散布之不實指控，
   本公司將依法追訴損害賠償，
   並保留一切法律權利。

   此致
   臺灣臺北地方法院

   Oracular 科技股份有限公司
   法務長 費雪 敬上」

貝葉把律師函放在桌上。
盯著看了三秒。
然後把它翻過去。反面朝上。

她不想在睡前看到那些字。

但她睡不著。

凌晨三點。
她起來。
把律師函翻回正面。
逐字讀完。
沒有跳任何一行。

然後拿了一張白紙。

在上面寫了兩行字。

  H₀：Oracular 的先驗沒有被人為修改。
  H₁：Oracular 的先驗被人為修改了。

┌─ 假設檢定的邏輯 ─────────────────────────┐
│                                           │
│ 你不需要「證明」H₁ 是真的。              │
│ 你需要的是——                              │
│ 在假設 H₀ 為真的情況下，                 │
│ 你觀察到的數據有多不可能。               │
│                                           │
│ 如果——不可能到了一個程度——               │
│ 你就拒絕 H₀。                            │
│                                           │
│ 不是「它是真的」。                        │
│ 是「它不可能不是真的」。                  │
│                                           │
└───────────────────────────────────────────┘

「拒絕虛無假設。」她說。
對著凌晨三點的空氣。
對著反面朝上的律師函。
對著陽台上太茂盛的薄荷。

對著整個世界。
```

## 第 38 章：顯著水準

### 高思遠的書房——備戰

```
【環境】
開庭前六天。十一月七日。
高思遠家的書房。

書房很小。三面牆是書架。
第四面牆——高思遠清空了。
掛了一塊白板。
像一個作戰室。

桌上有四杯咖啡（兩杯已經涼了）。
一碟花生。高思遠的習慣——看書配花生。
馬可瑜帶了一包黑巧克力。
「精算師的提神食物。比咖啡因持久。」

螢幕上——蘇立恆的視訊。
沒有開攝影機。只有聲音。
聲音經過處理——變低了半個八度。
像一個匿名者。
但貝葉已經認得出——
即使變了音——他停頓的節奏不會變。

出席：
- 陳貝葉（原告。主攻。）
- 高思遠（戰略顧問。翻譯官。）
- 蘇立恆（技術支援。幕後。不能出庭。）
- 馬可瑜（數據。最後的王牌。）
```

### 法庭的語言

```
高思遠站在白板前。
拿起白板筆。
他寫字的方式跟黑板粉筆時代一模一樣——
一筆一劃。不連筆。像刻字。

「法官不懂統計。」

他畫了一條線。

「所以——你要讓統計變成故事。
 你要讓法官——用他的語言——
 理解你在說什麼。」

他在白板上畫了一張表：

  ┌─ 假設檢定 vs 法庭審判 ──────────────────┐
  │                                           │
  │ H₀（虛無假設） = 被告無罪（無罪推定）   │
  │ H₁（對立假設） = 被告有罪（原告指控）   │
  │ α（顯著水準）  = 定罪門檻               │
  │ p-value         = 證據的強度              │
  │ 拒絕 H₀        = 判被告有罪              │
  │ 不拒絕 H₀      = 證據不足，無罪釋放     │
  │                                           │
  │ ⚠ 注意：                                │
  │ 「不拒絕 H₀」≠「被告是清白的」         │
  │  就像「無罪釋放」不代表「沒做過」       │
  │  只代表——證據不夠。                     │
  │                                           │
  └───────────────────────────────────────────┘

「這張表——你要背下來。
 不是背公式。是背邏輯。
 因為法庭上——你不會有時間翻資料。」

貝葉點頭。
她已經背了。但她沒有打斷高思遠。
因為——他在教她的不是內容。
是節奏。
法庭上的說話節奏。
慢一點。每句話之間留白。
讓法官有時間消化。
```

### α 的選擇

```
蘇立恆的聲音從螢幕裡出來。
低半個八度。像另一個人。

「你需要先定 α。」

「α——顯著水準。」貝葉說。
「對。也就是你願意承受多大的風險——
 在 Oracular 其實沒問題的情況下，冤枉他們。」

馬可瑜推了一下眼鏡。
「科學慣例是 0.05。百分之五。
 但這不是科學論文。這是法庭。」

高思遠放下白板筆。
「法庭比科學嚴格。
 因為——指控一間公司的代價很大。
 如果你冤枉了 Oracular——
 他們的股價會跌。員工會失業。
 使用者會恐慌。
 你的 α——必須配得上這個代價。」

馬可瑜：「保險業做高風險決策用 α = 0.01。
 但涉及全國性系統——我建議 0.001。
 千分之一。
 也就是——一千次裡只容許一次冤枉。」

蘇立恆：「0.001 意味著你需要更強的證據。
 但如果你的證據真的夠強——
 通過 0.001 的東西，比通過 0.05 的東西——
 有說服力十倍。
 因為你主動選了最嚴格的門檻。
 法官會看到你的誠意。」

貝葉走到白板前。
在高思遠的表格下面寫下：

  α = 0.001

  「我們選最嚴格的門檻。
   如果我們能通過——
   沒有人可以質疑。」

她轉過身。
看了三個人（兩個在場的，一個在螢幕裡的）。

「0.001。一千次裡的一次。
 如果我們連這個門檻都過不了——
 那也許……Oracular 真的沒問題。
 我接受。」

高思遠點了一下頭。
馬可瑜推了一下眼鏡（她的招牌動作）。
蘇立恆——沉默了兩秒。
然後說了一個字：「好。」
```

### p-value 的翻譯

```
高思遠回到白板前。

「α 定了。接下來——
 你要在法庭上怎麼解釋 p-value。」

他寫了一行字。

  p-value = ?

「p-value 是什麼？
 技術定義——」

他在白板上寫：

  p-value = P(觀察到至少這麼極端的數據 | H₀ 為真)

「在虛無假設為真的前提下，
 觀察到跟你手上的數據一樣極端——或更極端——的機率。」

「但你不能在法庭上說這句話。」

他把白板筆放下。
看著貝葉。

「你要說——」

他的聲音慢下來了。
不是教授在講課的慢。
是一個老人在傳授畢生功力的慢。

「你要說——

 『如果 Oracular 的先驗真的沒有被修改過——
  那我們觀察到的這種偏差——
  在一萬次裡只會出現三次。
  三次。
  法官——
  您相信——
  我們剛好遇到了那三次嗎？』」

貝葉把這段話寫下來。
然後抬頭。

「一萬次裡的三次。」
「對。不要說 p = 0.0003。
 法官聽不懂 0.0003。
 但他聽得懂『一萬次裡的三次』。」

馬可瑜補充：
「保險業用同樣的技巧。
 跟客戶解釋風險的時候——
 不說機率。說頻率。
 不說 0.1%。說『一千個人裡有一個』。
 人腦理解頻率比理解機率更直覺。」

蘇立恆的聲音從螢幕裡出來：
「但——要小心。
 頻率翻譯有陷阱。
 費雪也會用同樣的技巧。
 她會把 α = 0.001 翻譯成——
 『兩千三百萬人裡有兩萬三千人會被冤枉』。
 同樣的數字。不同的框架。不同的情緒。

 你要比她更會說故事。」

高思遠看了蘇立恆的螢幕一眼。
「你很了解費雪。」

兩秒的沉默。
「我了解她用數字的方式。」

四個人都沒有再追問。
```

### 練習

```
之後三個小時——
貝葉在高思遠的書房裡練習法庭陳述。

第一遍：聲音抖。
第五遍：聲音穩了。但語速太快。
第十五遍：語速對了。但眼神往下看——在看白板。
第三十遍：不看白板了。但停頓的位置不對。
第五十遍：停頓的位置對了。但太像背稿——沒有感情。
第六十遍：有感情了。但太多了。聲音有一點哽。
第七十三遍。

高思遠舉手。「停。」

「什麼？」
「夠了。」

「但我——」
「第七十三遍。你的停頓、眼神、語速、情緒——
 全部是對的。
 你不需要第七十四遍。
 因為第七十四遍開始——你會開始過度修正。
 第七十三遍——是最佳迭代。」

他笑了。
「做了三十年教授——
 我最大的技能就是知道學生什麼時候可以停了。」

馬可瑜（一直在旁邊用筆電工作）抬頭：
「七十三遍。」
「嗯？」
「質數。」
「所以？」
「沒有。精算師的職業病。
 質數比較讓人安心。」

夜深了。
馬可瑜先走。她明天還要上班。
「馬可瑜。」
「嗯？」
「保險數據——準備好了嗎？」
「隨時可以調出來。但先不用。
 先讓你的假設檢定站住。
 卡方是最後的牌。不要太早出。」

她走了。

書房裡剩下貝葉和高思遠。
螢幕上蘇立恆的視訊已經斷了——他什麼時候走的，沒有人注意到。

高思遠泡了茶。
「喝。」
「高叔——你覺得我會贏嗎？」

他把茶杯放在桌上。

「贏——是一個奇怪的字。
 法庭上沒有人贏。
 有人的假設被拒絕了。有人的假設沒有。
 但——
 我覺得你的數據夠強。
 而且——你選了 0.001。
 這代表你不是來鬧的。
 法官會看到這一點。」

他站起來。走到書架前。
抽出一本很舊的書。
封面已經褪色了。

《統計推論導論》。
作者——某某某。
出版日期——三十二年前。

「這是我用的第一本教科書。
 你母親也用過。
 柏朗也用過。
 我們三個人——用同一本書學的統計。」

他把書放在桌上。
「帶著它。不用讀。
 只是——帶著。」

貝葉拿起那本書。
很輕。紙頁已經發黃了。
翻開第一頁。
上面有三行不同的名字。
三種不同的筆跡。

第一行——高思遠的字。工整。
第二行——很小、很密的字。
她認不出來。但她的心跳了一下。
第三行——很整齊的字。像印刷體。

「那是柏朗的字。」高思遠說。
「中間那行——」
「你母親的。」

貝葉合上書。
抱在胸前。

「我帶著。」
```
