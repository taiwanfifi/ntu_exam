# 寫在最前面：為什麼要學海商法與海洋法？

> **Taiwan is an island nation. Everything from international trade shipping to fishing rights involves maritime law. It's a specialized but fascinating field.**

打開世界地圖，台灣是一個四面環海的島嶼。我們進口的石油、天然氣、原物料，出口的半導體、電子產品，絕大多數靠「船運」。台灣的遠洋漁業在全球排名前列。台灣海峽是世界最繁忙的國際航道之一。

所以，海商法與海洋法對台灣來說，不是冷門科目——它是我們作為島國的「生存之法」。

**海商法**處理的是：船舶的買賣、租賃、碰撞，貨物的運送，海上事故的損害分擔，以及海上保險。簡單來說，就是**「海上的商業活動規則」**。

**海洋法**處理的是：哪片海域屬於誰？各國在海上有什麼權利和義務？漁業怎麼管理？海底資源誰可以開採？簡單來說，就是**「海洋的國際秩序」**。

# 第一編：海商法

# 第一章：海商法總論

## 第一節：海商法的基本概念

### 一、什麼是海商法？

**海商法**是規範海上商業活動的法律。它規定了與海上運輸、船舶經營有關的各種法律關係。

海商法的特色在於它的**國際性**和**歷史性**：
- **國際性**：海上貿易本質上是跨國的，因此海商法高度統一，許多國家的海商法都以國際公約為基礎。
- **歷史性**：海商法是最古老的商業法之一，可以追溯到中世紀的地中海貿易。

### 二、海商法的法源

**（一）國內法源**

1. **海商法**：民國18年制定，最近一次大幅修正是民國89年。共分8章，168條。
2. **民法**：海商法未規定的事項，適用民法的一般規定。
3. **相關法規**：船舶法、航業法、商港法等。

**（二）國際法源**

1. **海牙規則（Hague Rules, 1924）**：規範提單與貨物運送
2. **海牙威士比規則（Hague-Visby Rules, 1968）**：海牙規則的修正
3. **漢堡規則（Hamburg Rules, 1978）**：聯合國的貨物運送公約
4. **鹿特丹規則（Rotterdam Rules, 2009）**：最新的國際海上運送公約
5. **1976年海事索賠責任限制公約**：船舶所有人責任限制
6. **1989年國際救助公約**：海上救助

### 三、海商法的適用範圍

海商法第1條規定：「本法所稱船舶，謂在海上航行，或在與海相通之水面或水中航行之船舶。」

也就是說，海商法適用於：
- **海上航行的船舶**
- **在與海相通的水域航行的船舶**（例如河口、港灣）

但不適用於：
- 軍事船舶（軍艦）
- 漁船（漁船有專門的法規）
- 總噸位未滿20噸的船舶

## 第二節：船舶

### 一、船舶的法律地位

在法律上，船舶是一種非常特殊的「物」。它雖然是「動產」（可以移動），但法律上把它當作「準不動產」來處理，因為：

1. 船舶有**登記制度**（類似不動產）
2. 船舶有**國籍**（掛某國的旗幟）
3. 船舶可以設定**抵押權**（類似不動產抵押）

### 二、船舶所有權

**（一）取得方式**

船舶所有權的取得，跟一般動產類似：
- 建造（原始取得）
- 買賣（繼受取得）
- 繼承
- 捕獲（戰爭中捕獲敵國船舶）

**（二）移轉與登記**

海商法第8條規定：「船舶所有權之移轉，非經登記，不得對抗第三人。」

> **白話解釋**：你買了一艘船，但如果沒有去做所有權移轉登記，別人可以說「我不知道船已經賣給你了」。

### 三、船舶國籍

每艘商船都必須有「國籍」——懸掛某個國家的旗幟。船舶的國籍決定了：
1. 適用哪個國家的法律
2. 享受哪個國家的外交保護
3. 遵守哪個國家的安全標準

台灣的船舶法規定，只有中華民國國民或依中華民國法律設立的法人，才能取得中華民國國籍的船舶。

**「權宜旗」（Flag of Convenience）問題**：有些船東為了降低成本（較低的稅金、寬鬆的法規），把船籍登記在巴拿馬、賴比瑞亞等國家。這些國家的船舶登記條件非常寬鬆，被稱為「權宜旗國」。

### 四、船舶優先權

**船舶優先權（Maritime Lien）**是海商法中非常重要的制度。它是指特定的債權人，對船舶享有優先於其他債權人受償的權利。

海商法第24條列舉了船舶優先權擔保的債權：

| 順序 | 債權類型 |
|------|----------|
| 1 | 船長、海員及其他在船舶上服務之人員，本於僱傭契約所生之債權 |
| 2 | 因船舶碰撞或其他海上事故所生之損害賠償 |
| 3 | 救助報酬及共同海損分擔額 |
| 4 | 因契約或侵權行為而生之損害賠償 |

> **白話解釋**：如果一艘船出了事故，很多人都來要錢（船員要薪水、被撞的人要賠償、救助的人要報酬），按照上面的順序來分配。船員的薪水排最前面。

船舶優先權的特色：
1. **法定權利**：不需要登記，依法律規定直接產生
2. **優先於抵押權**：優先權的順位高於船舶抵押權
3. **追及效力**：即使船舶被賣掉，優先權仍然存在（但有一年的除斥期間）

### 五、船舶抵押權

船舶可以設定抵押權，作為借款的擔保。

海商法第33條規定：「船舶抵押權之設定，應以書面為之。非經登記，不得對抗第三人。」

船舶抵押權的順位**低於**船舶優先權。

## 第三節：船長與海員

### 一、船長的法律地位

船長在海商法上有非常特殊的地位。他不只是船上的「管理者」，更是船舶所有人的**法定代理人**。

海商法第38條規定：「船長在航海中，為船舶所有人之代理人。」

這意味著船長可以代表船東簽訂契約、處理緊急事務等。

### 二、船長的權限

**（一）航海上的權限**

1. 指揮船舶的航行
2. 維持船上的秩序和紀律
3. 在緊急情況下，有權採取一切必要措施

**（二）商業上的權限**

1. 在航程中必要時，得為船舶所有人簽訂契約
2. 在不得已的情況下，得處分船舶上的貨物
3. 為籌措航海繼續所需費用，得以船舶為擔保借款

> **白話解釋**：船長在大海中就像一個「國王」——因為離岸很遠，無法隨時聯絡船東，所以法律賦予船長很大的權限來處理各種狀況。

### 三、船長的義務

1. **開航前檢查義務**：確保船舶適航
2. **直接管理義務**：船舶進出港時，船長應親自在場
3. **文件保管義務**：保管航海日誌、船舶文書等
4. **緊急處置義務**：遇到海難時，應盡力搶救
5. **報告義務**：到達目的港後，應向當地航政機關報告

# 第二章：海上貨物運送

## 第一節：運送契約

### 一、什麼是海上貨物運送契約？

海上貨物運送契約，就是**託運人（貨主）委託運送人（船公司）以船舶將貨物從A港運到B港**的契約。

這是海商法中最重要的部分，也是考試的重點中的重點。

### 二、運送契約的類型

**（一）件貨運送（Liner Transport）**

船公司經營固定航線，按照固定班期出航。託運人把貨物交給船公司，船公司會開立**提單（Bill of Lading）**。

> **白話解釋**：就像搭公車——公車有固定路線和時刻表，你買票上車就好。

**（二）傭船契約（Charter Party）**

由傭船人（租船的人）整艘或部分租用船舶。又分為：

| 類型 | 說明 |
|------|------|
| **論時傭船（Time Charter）** | 按時間租用船舶，船員由船東提供 |
| **論程傭船（Voyage Charter）** | 按航程租用船舶 |
| **光船租賃（Bareboat Charter）** | 租用空船，租船人自己配備船員 |

> **白話解釋**：
> - 論時傭船 = 租一輛有司機的車，按天計費
> - 論程傭船 = 叫計程車，按路程計費
> - 光船租賃 = 租一輛車（沒司機），自己開

---

## 第二節：提單（Bill of Lading）

### 一、什麼是提單？

**提單**是海上貨物運送中最重要的文件。它具有三個功能：

1. **運送契約的證明**：證明運送人和託運人之間有運送契約
2. **收據功能**：證明運送人已經收到貨物
3. **物權證券**：持有提單就等於持有貨物的權利。要領貨，就要出示提單。

> **白話解釋**：提單就像是貨物的「身分證 + 存摺」。有了提單，你就可以去領貨，也可以把提單轉讓給別人，讓別人去領貨。

### 二、提單的種類

| 種類 | 說明 |
|------|------|
| **已裝船提單（Shipped B/L）** | 貨物已經裝上船 |
| **收貨待運提單（Received for Shipment B/L）** | 貨物已收到但尚未裝船 |
| **清潔提單（Clean B/L）** | 提單上沒有批註貨物瑕疵 |
| **不清潔提單（Foul B/L）** | 提單上有批註貨物瑕疵 |
| **記名提單（Straight B/L）** | 指定特定人領貨 |
| **指示提單（Order B/L）** | 可以背書轉讓 |
| **無記名提單（Bearer B/L）** | 持有人就可以領貨 |

### 三、提單的法律效力

**（一）文義性**

運送人與提單持有人之間的關係，以提單記載的內容為準。即使實際的運送契約內容與提單不同，也以提單記載為準（對善意第三人）。

海商法第60條第2項：「載貨證券之發給人，對於依載貨證券所記載應為之行為，均應負責。」

**（二）流通性**

指示提單和無記名提單具有流通性，可以轉讓。這使得貨物在海上運送途中就可以進行買賣——買方只需要取得提單，就等於取得了貨物的權利。

### 四、提單與信用狀

在國際貿易中，提單通常跟**信用狀（Letter of Credit, L/C）**搭配使用：

1. 買方請銀行開立信用狀
2. 賣方將貨物裝船後，取得提單
3. 賣方用提單向銀行請求付款
4. 銀行把提單轉給買方
5. 買方憑提單去領貨

這個機制確保了「一手交錢、一手交貨」的安全性。

## 第三節：運送人的責任

### 一、運送人的基本義務

**（一）適航性義務**

海商法第62條規定，運送人在發航前及發航時，應注意使船舶具有：

1. **船舶適航性**：船舶本身堪於航海
2. **貨物適運性**：船艙適宜且安全收受、運送及保存貨物
3. **配員與裝備**：配置適當的船員、設備及供應品

> **白話解釋**：開船前，船公司要確保船不會漏水、船艙溫度適合載貨、船員夠用。

**（二）注意義務**

運送人從收到貨物到交付貨物為止，應以善良管理人的注意義務，妥善裝載、搬移、堆存、保管、運送與卸載。

### 二、運送人的免責事由

海商法第69條列舉了運送人的免責事由。這是海商法考試中最重要的條文之一：

| 免責事由 | 說明 |
|----------|------|
| 1. 船長、海員在航行或管理船舶行為中之過失 | **航海過失免責**（最特殊的一項） |
| 2. 海上或航路上的危險或意外事故 | 天災海難 |
| 3. 非由於運送人本身之過失所生之火災 | 火災免責 |
| 4. 天災 | 颱風、地震等 |
| 5. 戰爭行為 | 戰爭、海盜 |
| 6. 政府或公權力的行為 | 政府禁運等 |
| 7. 罷工、暴動 | 勞工糾紛 |
| 8. 救助海上人命或財產 | 為救助而偏航 |
| 9. 託運人的行為或過失 | 貨主自己的錯 |
| 10. 貨物的固有瑕疵或本質 | 貨物本身易壞 |
| 11. 包裝不固 | 貨主包裝不好 |
| 12. 標誌不清 | 貨物標記不清楚 |
| 13. 其他非因運送人本人之故意或過失所致者 | 概括條款 |

**最重要的概念：航海過失免責（Nautical Fault Exemption）**

這是海商法最特殊的制度。如果船長或海員在「駕駛船舶」或「管理船舶」過程中有過失導致貨損，運送人可以免責。

> **白話解釋**：如果船長開船不小心撞到礁石，導致船上的貨物損壞，運送人不需要賠償。但如果是運送人沒有好好保管貨物（例如冷凍櫃壞了沒修），就不能免責。

**區分重點**：
- **航海過失**（免責）：與船舶的駕駛和管理有關的過失
- **商業過失**（不免責）：與貨物的裝卸、堆存、保管有關的過失

### 三、運送人的責任限制

即使運送人不能免責，其賠償金額也有上限。

海商法第70條規定：「運送人對於每一包件或每一單位之賠償，以特別提款權666.67單位計算之金額為限。」

> **白話解釋**：即使你的貨物價值一億元，運送人的賠償也有上限。這是為了保護海上運輸業者，因為海上風險太大了。

### 四、託運人的義務

託運人也有義務：
1. 告知貨物的性質（特別是危險物品）
2. 正確申報貨物的種類和數量
3. 支付運費
4. 適當包裝貨物

## 第四節：海上旅客運送

### 一、基本概念

海上旅客運送契約，是指運送人以船舶運送旅客的契約。

### 二、運送人的責任

**（一）對旅客的責任**

海商法第81條準用民法中關於旅客運送的規定。運送人對旅客的安全負有注意義務。

**（二）對行李的責任**

旅客的隨身行李，運送人的責任與貨物運送類似。

### 三、旅客的權利

1. 安全到達目的地的權利
2. 在運送遲延時，得請求賠償
3. 在船舶不適航時，得解除契約

# 第三章：船舶碰撞

## 第一節：什麼是船舶碰撞？

### 一、定義

**船舶碰撞**是指兩艘或多艘船舶在海上發生物理上的接觸，導致損害。

海商法也把「間接碰撞」包含在內——即使兩艘船沒有直接接觸，但因為一艘船的過失動作（例如製造大浪），導致另一艘船損壞，也算碰撞。

### 二、碰撞的類型與責任分配

| 類型 | 說明 | 責任 |
|------|------|------|
| **不可抗力碰撞** | 因天災等不可抗力導致 | 各自承擔損害 |
| **有過失碰撞** | 因一方或雙方過失導致 | 有過失的一方賠償 |
| **雙方過失碰撞** | 雙方都有過失 | 按過失比例分擔 |
| **過失不明碰撞** | 無法判斷誰有過失 | 各自承擔損害 |

### 三、雙方過失碰撞的特殊規定

海商法第97條規定：「碰撞係因於雙方船舶之過失所致者，雙方船舶所有人各依其過失程度之比例，負損害賠償責任。」

**對貨物損害的連帶責任**：

海商法第97條第3項：「碰撞之船舶，對於碰撞所生之損害，負連帶責任。但各船舶間之過失比例，不因此而受影響。」

> **白話解釋**：如果A船和B船碰撞，都有過失，貨主可以向任何一艘船索賠全部損害。但A船和B船之間，再按照過失比例去分擔。

### 四、碰撞的時效

碰撞所生的損害賠償請求權，時效為**2年**，自碰撞日起算。

# 第四章：海上救助

## 第一節：什麼是海上救助？

### 一、定義

**海上救助（Salvage）**是指在海上或與海相通的水域中，對遭遇危險的船舶或其他財產提供救助的行為。

### 二、救助報酬

成功的救助人有權獲得**救助報酬**。

海商法第103條：「救助者所得受之報酬，不得超過船舶及貨物被救助後之價值。」

**「無效果、無報酬」原則（No Cure, No Pay）**：

如果救助沒有成功（船舶和貨物都沒有救回來），救助人原則上不能要求報酬。

> **白話解釋**：如果你去救一艘快沉的船，成功救起來了，你可以拿報酬。但如果船還是沉了，你就拿不到報酬。

### 三、救助報酬的決定因素

海商法第103條第2項列舉了決定救助報酬金額的因素：

1. 被救助財產的價值
2. 救助者的技術與努力
3. 救助者所承擔的危險
4. 救助者投入的時間和費用
5. 救助者使用的船舶和設備
6. 環境保護的效果

### 四、特別補償

1989年國際救助公約引入了「特別補償」制度。即使救助沒有成功（船沉了），但如果救助行為有效防止或減輕了環境損害（例如防止油污擴散），救助人仍可獲得「特別補償」。

---

# 第五章：共同海損

## 第一節：什麼是共同海損？

### 一、定義

**共同海損（General Average）**是海商法中最具特色的制度之一。

海商法第110條：「稱共同海損者，謂在船舶航程期間，為求共同危險中全體財產之安全，所為有意及合理之處分，而直接造成之犧牲及費用。」

> **白話解釋**：船在海上遇到危險（例如快沉了），船長為了拯救全船，決定把一部分貨物丟進海裡（減輕重量讓船不會沉）。這些被丟進海裡的貨物的損失，不是只由那個貨主承擔，而是由全船的利害關係人（船東、全部貨主）按比例分擔。

### 二、共同海損的要件

1. **共同危險**：船舶和貨物必須面臨共同的危險
2. **有意的處分**：犧牲必須是「故意」為之，不是意外
3. **合理性**：處分必須是合理的，不能亂丟
4. **直接因果關係**：損失必須是處分直接造成的
5. **為了共同安全**：目的是拯救全體，不是只救特定人

### 三、共同海損的例子

| 行為 | 是否為共同海損 |
|------|---------------|
| 為減輕重量而拋棄貨物 | 是 |
| 為撲滅船上火災而注水，損壞貨物 | 是 |
| 為脫淺而使船舶機器過度運轉 | 是 |
| 因颱風導致甲板上的貨物被浪捲走 | 否（不是有意的處分） |
| 貨物本身因潮濕而腐壞 | 否（不是為了共同安全） |

### 四、共同海損的分擔

**（一）分擔義務人**

所有因共同海損行為而得以保全的利害關係人，都要按比例分擔損失。包括：
- 船舶所有人
- 貨物所有人（所有的貨主）
- 運費收入者

**（二）分擔比例**

按照各方「被保全的財產價值」的比例來分擔。

> **例子**：船舶價值3,000萬，甲的貨物價值1,000萬，乙的貨物價值1,000萬。為了避免船沉沒，船長丟棄了乙的全部貨物（損失1,000萬）。
>
> 分擔比例：
> - 船舶：3,000萬 / 5,000萬 = 60% → 分擔 600萬
> - 甲的貨物：1,000萬 / 5,000萬 = 20% → 分擔 200萬
> - 乙的貨物：1,000萬 / 5,000萬 = 20% → 分擔 200萬
>
> 乙損失1,000萬，但可以從船東和甲那裡拿回800萬（600+200），自己只承擔200萬。

### 五、共同海損的精算

共同海損的計算非常複雜，通常由專門的**海損精算人**（Average Adjuster）來處理。國際上通用的計算規則是**約克-安特衛普規則（York-Antwerp Rules）**。

---

# 第六章：船舶所有人責任限制

## 第一節：為什麼要限制責任？

### 一、制度背景

海上運輸的風險非常高——颱風、碰撞、海盜、貨損等。如果船舶所有人必須對所有的損害負完全賠償責任，可能一次事故就會讓船東破產。

為了維持海運業的持續發展，各國法律允許船舶所有人在一定條件下**限制其賠償責任**。

### 二、責任限制的金額

海商法第21條規定了責任限制的計算方式。按照船舶的噸位來計算責任限制的金額：

**（一）人身傷亡**
- 500噸以下：333,000特別提款權
- 超過500噸，每噸加500特別提款權

**（二）財產損害**
- 500噸以下：167,000特別提款權
- 超過500噸，每噸加167特別提款權

### 三、不得主張責任限制的情形

如果損害是因船舶所有人**故意或重大過失**所致，就不能主張責任限制。

> **白話解釋**：如果你故意把船開去撞人，就不能說「我的責任有上限」。

---

# 第七章：海上保險

## 第一節：什麼是海上保險？

### 一、定義

**海上保險**是指保險人對被保險人，在海上或與海相通水域中，因航海事業所生的危險所造成的損害，負賠償責任的保險契約。

### 二、海上保險的種類

| 種類 | 保險標的 |
|------|----------|
| **船舶保險（Hull Insurance）** | 保障船舶本身的損害 |
| **貨物保險（Cargo Insurance）** | 保障運送中貨物的損害 |
| **運費保險（Freight Insurance）** | 保障運費收入的損失 |
| **責任保險（P&I Insurance）** | 保障船東對第三人的賠償責任 |

### 三、保險利益

要投保海上保險，被保險人必須對保險標的有**保險利益**。例如：
- 船舶所有人對船舶有保險利益
- 貨物所有人對貨物有保險利益
- 運送人對運費有保險利益

### 四、承保範圍

海商法第129條規定了保險人承保的風險：

1. 海上固有的危險（颱風、觸礁、碰撞等）
2. 火災
3. 海盜
4. 拋棄貨物
5. 戰爭（需要特別約定）
6. 其他約定的危險

### 五、委付

**委付（Abandonment）**是海上保險中特有的制度。

當保險標的遭遇「推定全損」時（例如船舶雖然沒有完全沉沒，但修理費用超過船舶價值），被保險人可以把保險標的「丟給」保險人，然後要求保險人賠付全損。

海商法第143條規定了可以委付的情形：
1. 船舶行蹤不明已逾二個月
2. 船舶修繕費用超過保險金額四分之三
3. 船舶不能修繕
4. 貨物毀損已達保險金額四分之三以上

# 第二編：海洋法

# 第八章：海洋法概論

## 第一節：什麼是海洋法？

### 一、定義

**海洋法（Law of the Sea）**是規範海洋的使用、管轄和資源分配的國際法。

### 二、海洋法的核心法源：聯合國海洋法公約（UNCLOS）

**聯合國海洋法公約（United Nations Convention on the Law of the Sea, UNCLOS）**於1982年通過，1994年生效。被譽為**「海洋的憲法」**。

UNCLOS共320條、17編、9個附件，幾乎涵蓋了海洋法的所有議題。目前有168個締約國（截至2024年）。

### 三、海洋法的基本原則

1. **公海自由原則**：公海不屬於任何國家，所有國家都可以自由使用
2. **沿海國主權原則**：沿海國對其領海享有主權
3. **和平利用原則**：海洋應用於和平目的
4. **人類共同遺產原則**：深海海底資源是「人類共同遺產」

---

## 第二節：海域劃分

### 一、海域分區概覽

從海岸向外，海洋被劃分為不同的區域，各有不同的法律地位：

```
陸地 → 內水 → 領海(12浬) → 鄰接區(24浬) → 專屬經濟區(200浬) → 公海
                                                    ↓
                                              大陸棚(最遠350浬)
                                                    ↓
                                            深海海底（國際海底區域）
```

### 二、基線（Baseline）

**基線**是測量所有海域寬度的起點。有兩種劃法：

**（一）正常基線（Normal Baseline）**

沿海國海岸的低潮線。

**（二）直線基線（Straight Baseline）**

在海岸線非常曲折或有群島的地方，可以用直線連接海岸上的適當點作為基線。

> **白話解釋**：基線就像是海域的「起跑線」。從這條線開始往外量，量12浬是領海，量24浬是鄰接區，量200浬是專屬經濟區。

---

# 第九章：各海域的法律制度

## 第一節：內水（Internal Waters）

### 一、定義

**內水**是指基線向陸地一側的水域，包括港口、河口、海灣等。

### 二、法律地位

沿海國對內水享有**完全的主權**，如同陸地領土。外國船舶沒有進入內水的權利（除非獲得許可或遇到緊急情況）。

---

## 第二節：領海（Territorial Sea）

### 一、定義

**領海**是從基線向外延伸**12海浬**的海域。

### 二、法律地位

沿海國對領海享有**主權**。但這個主權受到「無害通過權」的限制。

### 三、無害通過權（Right of Innocent Passage）

**（一）什麼是無害通過？**

外國船舶有權通過他國領海，但必須是「無害」的——也就是不能損害沿海國的和平、良好秩序或安全。

UNCLOS第19條列舉了「有害」的行為，包括：
1. 使用武力或威脅
2. 軍事演習
3. 收集情報
4. 散播宣傳
5. 起降飛機
6. 嚴重污染
7. 捕魚
8. 研究或調查活動

**（二）通過的方式**

通過必須是「繼續不停且迅速」的。船舶不能在領海中停留、徘徊或從事其他活動。

**（三）潛水艇的通過**

潛水艇在領海中通過時，必須**浮出水面並懸掛國旗**。

### 四、沿海國的權利

沿海國可以：
1. 制定關於航行安全、環境保護的法規
2. 在必要時暫停無害通過（但不得歧視特定國家）
3. 對違反規定的外國船舶行使管轄權

---

## 第三節：鄰接區（Contiguous Zone）

### 一、定義

**鄰接區**是從基線向外延伸**24海浬**的海域（也就是領海之外再12海浬）。

### 二、法律地位

鄰接區**不屬於沿海國的主權範圍**，但沿海國可以在鄰接區內行使有限的管轄權：

1. 防止在其領土或領海內違反海關、財政、移民或衛生法律和規章
2. 處罰在其領土或領海內已經違反上述法律的行為

> **白話解釋**：鄰接區就像是領海的「緩衝地帶」。如果有走私船在往你的領海方向前進，你可以在鄰接區就攔下它，不用等它進入領海才行動。

---

## 第四節：專屬經濟區（Exclusive Economic Zone, EEZ）

### 一、定義

**專屬經濟區**是從基線向外延伸**200海浬**的海域。

### 二、法律地位

專屬經濟區是一種「混合區域」——既不是公海，也不是領海。沿海國在EEZ內有**特定的主權權利和管轄權**，但其他國家也保有**航行和飛越自由**。

### 三、沿海國的權利

**（一）主權權利（Sovereign Rights）**

1. **勘探和開發自然資源**：包括海床和底土的礦物資源，以及上覆水域的生物資源（漁業）
2. **其他經濟目的的活動**：例如利用海水發電

**（二）管轄權（Jurisdiction）**

1. **人工島嶼和設施的建造與使用**
2. **海洋科學研究**
3. **海洋環境的保護和保全**

### 四、其他國家的權利

在沿海國的EEZ內，其他國家享有：
1. **航行自由**
2. **飛越自由**
3. **鋪設海底電纜和管道的自由**

> **白話解釋**：EEZ就像是你家的「農場」——你有權種菜（開發資源），但鄰居可以從你農場中間的路走過去（航行自由）。

### 五、台灣的EEZ問題

台灣與周邊國家（中國大陸、日本、菲律賓）的EEZ存在重疊。EEZ的劃界是國際法上的重要議題，通常透過「等距中線原則」和「公平原則」來處理。

---

## 第五節：大陸棚（Continental Shelf）

### 一、定義

**大陸棚**是沿海國陸地領土向海洋自然延伸的海底區域。

### 二、範圍

- **最基本的範圍**：200海浬
- **可以延伸**：如果地質上大陸棚超過200海浬，可以延伸到最遠**350海浬**（但須向大陸棚界限委員會提出申請）

### 三、沿海國的權利

沿海國對大陸棚享有：
1. **勘探和開發自然資源的主權權利**：包括石油、天然氣、礦物等
2. 這些權利是**固有的**——不需要宣布或占領，天然就有

> **白話解釋**：大陸棚就是你家地下室——地下的石油、礦產都是你的。即使你還沒開採，也是你的。

### 四、上覆水域

大陸棚的法律地位不影響其上覆水域的法律地位。也就是說，即使你對大陸棚的海底有權利，海面上的水域仍然可能是公海或EEZ。

---

## 第六節：公海（High Seas）

### 一、定義

**公海**是不包括在任何國家的領海、EEZ、內水或群島水域內的海域。

### 二、公海自由原則

UNCLOS第87條規定，公海對所有國家開放。公海自由包括：

1. **航行自由**
2. **飛越自由**
3. **鋪設海底電纜和管道的自由**
4. **建造人工島嶼和設施的自由**
5. **捕魚自由**（受養護義務的限制）
6. **科學研究自由**

### 三、公海上的管轄權

在公海上，船舶原則上只受**船旗國**的管轄。但有例外：

**（一）普遍管轄權**

任何國家對以下行為都可以行使管轄權：
1. **海盜行為**：任何國家都可以逮捕海盜
2. **販賣奴隸**
3. **非法廣播**

**（二）臨檢權（Right of Visit）**

軍艦有權在公海上對有合理懷疑的外國船舶進行臨檢。但如果臨檢後發現船舶無辜，臨檢的國家要賠償損失。

**（三）緊追權（Right of Hot Pursuit）**

如果一艘外國船舶在你的領海或EEZ內違法，你的軍艦可以從領海一路追到公海。但追擊必須是「連續不斷」的——你不能追到一半停下來，隔天再追。

---

## 第七節：國際海峽

### 一、什麼是國際海峽？

**國際海峽**是連接兩個公海或EEZ部分的海峽，且用於國際航行。

例如：馬六甲海峽、荷莫茲海峽、直布羅陀海峽。

### 二、過境通行權（Right of Transit Passage）

在國際海峽中，所有國家的船舶和飛機享有**過境通行權**。這比領海的「無害通過權」更為寬廣：

| 特點 | 無害通過 | 過境通行 |
|------|----------|----------|
| 適用區域 | 領海 | 國際海峽 |
| 潛水艇 | 必須浮出水面 | 可以潛航 |
| 飛機 | 無飛越權 | 有飛越權 |
| 可否暫停 | 沿海國可暫停 | 不可暫停 |

> **白話解釋**：國際海峽的通行權比領海寬鬆得多，因為如果海峽被封鎖，會影響全球航運。

---

## 第八節：島嶼制度

### 一、島嶼的定義

UNCLOS第121條規定：「島嶼是四面環水並在高潮時高於水面的自然形成的陸地區域。」

### 二、島嶼的海域權利

原則上，島嶼與其他陸地領土一樣，可以擁有：
- 領海（12浬）
- 鄰接區（24浬）
- 專屬經濟區（200浬）
- 大陸棚

### 三、岩礁的限制（第121條第3項）

**「不能維持人類居住或其本身的經濟生活的岩礁，不應有專屬經濟區或大陸棚。」**

這就是著名的**「岩礁條款」**——不是所有突出水面的礁石都可以主張200浬的EEZ。

**南海仲裁案（2016年）**就涉及了這個問題。仲裁庭認為南沙群島中的多數地物（如太平島）不能維持「其本身的經濟生活」，因此只能主張領海，不能主張EEZ。（注：我國對此裁決不予承認。）

### 四、台灣相關的島嶼爭議

1. **釣魚台列嶼**：中華民國、中華人民共和國與日本均主張主權
2. **太平島**：我國實際管轄的南沙群島最大島嶼
3. **東沙群島**：我國實際管轄

---

## 第九節：深海海底制度（國際海底區域）

### 一、定義

**國際海底區域（The Area）**是指國家管轄範圍以外的海床和洋底及其底土。

### 二、人類共同遺產原則

UNCLOS第136條規定：「區域及其資源是人類的共同遺產。」

這意味著：
1. **不能占有**：任何國家不能對國際海底區域主張主權
2. **共同利益**：區域資源的開發利益應由全人類共享
3. **和平利用**：區域僅用於和平目的

### 三、國際海底管理局（ISA）

**國際海底管理局**是管理國際海底區域資源的國際組織。所有關於深海礦物開採的活動，都必須經過ISA的批准。

深海海底富含**多金屬結核**、**多金屬硫化物**和**富鈷結殼**等礦物資源，具有巨大的經濟潛力。

# 第十章：海洋環境保護

## 第一節：海洋污染

### 一、海洋污染的來源

1. **船舶污染**：油輪漏油、船舶排放廢水
2. **陸源污染**：工廠廢水排入海洋
3. **傾倒污染**：把廢物丟進海裡
4. **海底活動污染**：海底石油開採造成的污染
5. **大氣污染**：空氣中的污染物落入海洋

### 二、UNCLOS的環境保護規定

UNCLOS第十二部分專門規定了海洋環境的保護和保全：

1. **一般義務**：各國有義務保護和保全海洋環境
2. **國際合作**：各國應進行區域和全球合作
3. **技術援助**：發達國家應協助發展中國家
4. **損害賠償責任**：造成海洋污染的國家應承擔賠償責任

### 三、重要的環境保護公約

1. **MARPOL公約**（防止船舶造成污染國際公約）
2. **油污損害民事責任公約**
3. **防止傾倒廢物公約（倫敦公約）**

# 第十一章：海洋爭端解決

## 第一節：UNCLOS的爭端解決機制

### 一、機制概覽

UNCLOS建立了一套完整的爭端解決機制：

1. **自行選擇的程序**：談判、調停
2. **導致有拘束力裁判的強制程序**：
   - 國際海洋法法庭（ITLOS）
   - 國際法院（ICJ）
   - 仲裁法庭
   - 特別仲裁法庭

### 二、國際海洋法法庭（ITLOS）

設在德國漢堡，由21名法官組成。專門審理與海洋法有關的爭端。

### 三、海洋劃界爭端

海洋劃界（特別是EEZ和大陸棚的劃界）是最常見的海洋爭端之一。劃界的原則：

1. **等距中線原則**：從兩國海岸各取等距點連成中線
2. **公平原則**：考慮所有相關情勢，達成公平的結果
3. **特殊情況**：島嶼、海岸地形等可能影響劃界

# 考試準備策略

## 一、海商法的準備重點

1. **貨物運送**：提單、運送人責任（免責事由）、責任限制
2. **共同海損**：要件、分擔計算
3. **船舶碰撞**：責任分配
4. **海上保險**：保險利益、委付
5. **船舶優先權**：順位、特性

## 二、海洋法的準備重點

1. **各海域的法律地位**：領海、EEZ、大陸棚、公海的比較
2. **無害通過權 vs. 過境通行權**的比較
3. **島嶼制度**：特別是岩礁條款
4. **公海上的管轄權**：緊追權、臨檢權
5. **海洋劃界**的原則

## 三、答題技巧

海商法與海洋法的申論題通常會給你一個案例，要你分析其中的法律問題。答題時：

1. 先確認涉及的法律制度
2. 引用相關法條或公約規定
3. 分析具體案例事實
4. 提出結論

# 名詞解釋快速參考

| 術語 | 白話解釋 |
|------|----------|
| 提單 | 貨物的「提貨券」兼「所有權證書」 |
| 航海過失免責 | 船長開船出錯，運送人不用賠 |
| 共同海損 | 為全船安全犧牲部分貨物，大家一起分擔損失 |
| 船舶優先權 | 特定債權人可以優先從船舶拿錢 |
| 委付 | 把快報廢的船丟給保險公司，拿全額理賠 |
| 領海 | 從海岸往外12浬，沿海國的主權範圍 |
| EEZ | 從海岸往外200浬，沿海國的經濟權利範圍 |
| 大陸棚 | 海底資源歸沿海國所有 |
| 公海 | 大家的海，誰都可以用 |
| 無害通過 | 外國船可以路過你的領海，但不能搞事 |
| 過境通行 | 在國際海峽中的通行權，比無害通過更寬 |
| 緊追權 | 你在我的海域犯法，我可以追到公海 |
| 岩礁條款 | 住不了人的礁石不能主張EEZ |
| 人類共同遺產 | 深海海底資源屬於全人類 |
