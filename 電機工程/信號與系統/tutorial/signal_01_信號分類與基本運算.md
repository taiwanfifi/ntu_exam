# 信號與系統 第一章：信號分類與基本運算

---

## 🔰 本章基礎觀念（零基礎必讀）

### 什麼是「信號」？

你每天都在接觸信號，只是可能沒有意識到：

- **聲音**：空氣壓力隨時間的變化 → 這就是一個信號
- **影像**：每個像素的亮度隨位置變化 → 也是信號
- **體溫**：你的體溫隨時間高低起伏 → 還是信號
- **股價**：每一天的收盤價 → 離散的信號

> **信號（Signal）的本質：一個量隨另一個量變化的「對應關係」。**

用數學的話說，信號就是一個**函數**。例如溫度信號 T(t) 表示「在時間 t 時，溫度是多少」。

### 什麼是「系統」？

系統就是**對信號做處理的東西**：

- **音響的等化器**：輸入音樂信號 → 調整高低音 → 輸出修改後的音樂
- **Instagram 濾鏡**：輸入照片 → 套濾鏡處理 → 輸出修改後的照片
- **冷氣溫控**：輸入溫度信號 → 判斷要不要開壓縮機 → 輸出控制信號

> **系統（System）的本質：接收輸入信號，經過某種規則處理後，產生輸出信號。**

```
輸入 x(t) ──→ [ 系統 ] ──→ 輸出 y(t)
```

整門「信號與系統」課程就是在回答一個核心問題：

> **給定一個系統和一個輸入信號，輸出信號長什麼樣？**

## 一、連續時間 vs 離散時間信號

### 1.1 連續時間信號（CT Signal）

時間變數 t 可以取**任何實數值**，信號在每個時間點都有定義。

- 記號：**x(t)**，用小括號
- 例如：x(t) = sin(2πt)，在 t = 0.001, t = 1.414... 等任何時間都有值

### 1.2 離散時間信號（DT Signal）

時間變數 n 只能取**整數值**，信號只在 n = ..., -2, -1, 0, 1, 2, ... 有定義。

- 記號：**x[n]**，用方括號
- 例如：x[n] = (0.5)ⁿ u[n]，只在 n = 0, 1, 2, 3, ... 有值

> **重要提醒**：看到小括號 x(t) 就是連續時間，看到方括號 x[n] 就是離散時間。這個記號區分非常重要！

### 數值例題 1.1：辨識信號類型

> **問題**：以下哪些是連續時間信號？哪些是離散時間信號？
> (a) 麥克風收到的聲波
> (b) 數位照片的像素值
> (c) 心電圖（ECG）
> (d) 每月電費帳單金額

**解答**：
- (a) 連續時間 — 聲波壓力在每個時間點都有定義
- (b) 離散 — 像素是一格一格的（離散空間信號）
- (c) 連續時間 — 心臟電壓隨時間連續變化
- (d) 離散時間 — 只在每個月才有一筆資料

## 二、信號分類

### 2.1 週期信號 vs 非週期信號

**定義（連續）**：如果存在一個最小正數 T，使得

$$x(t + T) = x(t), \quad \forall t$$

則 x(t) 是週期信號，T 稱為**基本週期（Fundamental Period）**。

**定義（離散）**：如果存在一個最小正整數 N，使得

$$x[n + N] = x[n], \quad \forall n$$

則 x[n] 是週期信號，N 稱為基本週期。

> **注意**：連續正弦 sin(ω₀t) 一定是週期的；但離散正弦 sin(ω₀n) 不一定是週期的！
> 離散正弦要是週期的，必須 ω₀/(2π) 是有理數。

### 數值例題 2.1：判斷週期性

> **問題**：判斷 x(t) = cos(3t) + sin(5t) 是否為週期信號？若是，求基本週期。

**解答**：
- cos(3t) 的週期 T₁ = 2π/3
- sin(5t) 的週期 T₂ = 2π/5
- 兩個週期信號的和是否週期？看 T₁/T₂ 是否為有理數
- T₁/T₂ = (2π/3)/(2π/5) = 5/3 → 有理數 ✓
- 合成週期 T = LCM(5,3)/1 × (2π/15) ... 更精確地：
  - T₁ = 2π/3, T₂ = 2π/5
  - 我們找最小的 T 使得 T/T₁ 和 T/T₂ 都是正整數
  - T = 2π 時：T/T₁ = 3 ✓, T/T₂ = 5 ✓
- **答：是週期信號，基本週期 T = 2π**

### 2.2 能量信號 vs 功率信號

**能量**（連續時間）：

$$E = \int_{-\infty}^{\infty} |x(t)|^2 \, dt$$

**平均功率**（連續時間）：

$$P = \lim_{T \to \infty} \frac{1}{2T} \int_{-T}^{T} |x(t)|^2 \, dt$$

| 分類 | 條件 | 典型例子 |
|------|------|----------|
| 能量信號 | E < ∞, P = 0 | 短脈衝、衰減指數 e^(-at)u(t) |
| 功率信號 | E = ∞, 0 < P < ∞ | 正弦波 sin(t)、週期方波 |
| 都不是 | E = ∞, P = ∞ | 持續增長的信號如 e^(t) |

> **白話**：能量信號是「短暫出現又消失」的信號，總能量有限。功率信號是「永遠持續」的信號，平均功率有限。

### 數值例題 2.2：能量計算

> **問題**：計算 x(t) = 3e^(-2t)u(t) 的能量。

**解答**：

$$E = \int_{-\infty}^{\infty} |x(t)|^2 \, dt = \int_{0}^{\infty} 9e^{-4t} \, dt$$

$$= 9 \cdot \left[-\frac{1}{4}e^{-4t}\right]_{0}^{\infty} = 9 \cdot \frac{1}{4} = \frac{9}{4} = 2.25$$

因為 E = 2.25 < ∞，所以 x(t) 是**能量信號**。

### 2.3 偶信號 vs 奇信號

- **偶信號（Even）**：x(-t) = x(t)，對 t=0 左右對稱。例如 cos(t)。
- **奇信號（Odd）**：x(-t) = -x(t)，對 t=0 反對稱。例如 sin(t)。

> **重要性質**：任何信號都可以分解為偶部分和奇部分的和！

$$x_e(t) = \frac{x(t) + x(-t)}{2} \quad \text{（偶部分）}$$

$$x_o(t) = \frac{x(t) - x(-t)}{2} \quad \text{（奇部分）}$$

$$x(t) = x_e(t) + x_o(t)$$

### 數值例題 2.3：偶奇分解

> **問題**：將 x(t) = e^(2t) 分解為偶部分和奇部分。

**解答**：

$$x_e(t) = \frac{e^{2t} + e^{-2t}}{2} = \cosh(2t)$$

$$x_o(t) = \frac{e^{2t} - e^{-2t}}{2} = \sinh(2t)$$

驗證：cosh(2t) + sinh(2t) = e^(2t) ✓

## 三、信號的基本運算

### 3.1 時移（Time Shift）

$$x(t) \longrightarrow x(t - t_0)$$

- **t₀ > 0**：信號向**右**移（延遲）
- **t₀ < 0**：信號向**左**移（提前）

> **記憶口訣**：「減右加左」—— 減一個正數就往右移。

### 3.2 反褶（Time Reversal）

$$x(t) \longrightarrow x(-t)$$

把信號對 t=0 做鏡像翻轉，就像把影片**倒帶播放**。

### 3.3 尺度變換（Time Scaling）

$$x(t) \longrightarrow x(at)$$

- **|a| > 1**：信號在時間軸上被**壓縮**（快轉）
- **|a| < 1**：信號在時間軸上被**拉伸**（慢放）
- **a < 0**：還會做反褶

### 3.4 複合運算的順序（超重要！）

當同時有時移和尺度變換時：x(at - b)

> **先做時移，再做尺度變換**，或者更安全的方法：
>
> 把 x(at - b) 寫成 x(a(t - b/a))
> 1. 先把 x(t) 壓縮/拉伸 a 倍
> 2. 再右移 b/a

### 數值例題 3.1：複合運算

> **問題**：已知 x(t) 在 [0, 2] 之間為 t（斜坡），其餘為 0。畫出 x(2t - 3) 的圖形。

**解答**：

把 x(2t - 3) 寫成 x(2(t - 3/2))。

**步驟 1：尺度變換** x(t) → x(2t)
- 原來在 [0, 2] 的信號壓縮到 [0, 1]
- x(2t) 在 [0, 1] 之間為 2t

**步驟 2：時移** x(2t) → x(2(t - 3/2)) = x(2t - 3)
- 往右移 3/2 = 1.5
- 最終信號在 [1.5, 2.5] 之間非零
- x(2t-3) 在 t ∈ [1.5, 2.5] 時值為 2t - 3

**驗證**：
- t = 1.5 時：x(2×1.5 - 3) = x(0) = 0 ✓
- t = 2.5 時：x(2×2.5 - 3) = x(2) = 2 ✓

## 四、重要基本信號

### 4.1 單位脈衝信號 δ(t)（Dirac Delta / Unit Impulse）

δ(t) 不是一般的函數，而是一個「廣義函數」(Generalized Function)：

$$\delta(t) = 0, \quad t \neq 0$$

$$\int_{-\infty}^{\infty} \delta(t) \, dt = 1$$

> **白話理解**：想像一記極短極重的「雷擊」—— 作用時間趨近於零，但衝量（面積）恰好是 1。

**最重要的性質 —— 取樣性質（Sifting Property）**：

$$\int_{-\infty}^{\infty} x(t) \delta(t - t_0) \, dt = x(t_0)$$

> 用 δ(t - t₀) 去「掃過」x(t)，結果就是在 t₀ 那一點的值 x(t₀)。

**離散版本** δ[n]：

$$\delta[n] = \begin{cases} 1, & n = 0 \\ 0, & n \neq 0 \end{cases}$$

離散的 δ[n] 就是一個普通的序列，非常好理解。

### 數值例題 4.1：取樣性質

> **問題**：計算 ∫₋∞^∞ (t² + 3t) δ(t - 2) dt

**解答**：

利用取樣性質，把 t = 2 代入 (t² + 3t)：

$$= (2)^2 + 3(2) = 4 + 6 = 10$$

### 4.2 單位步階信號 u(t)（Unit Step）

$$u(t) = \begin{cases} 1, & t > 0 \\ 0, & t < 0 \end{cases}$$

> t = 0 的值通常不重要（有時定義為 1/2）。

**白話**：想像一個開關，t = 0 的瞬間從「關」變成「開」。

**與 δ(t) 的關係**：

$$\frac{du(t)}{dt} = \delta(t) \quad \text{（步階微分 = 脈衝）}$$

$$u(t) = \int_{-\infty}^{t} \delta(\tau) \, d\tau \quad \text{（脈衝積分 = 步階）}$$

**矩形脈衝可以用步階函數表示**：

寬度為 T、高度為 1 的矩形脈衝 = u(t) - u(t - T)

### 4.3 指數信號 e^(st)

這是信號與系統中**最核心**的信號，因為 s 可以是複數：

$$s = \sigma + j\omega$$

$$e^{st} = e^{(\sigma + j\omega)t} = e^{\sigma t} \cdot e^{j\omega t}$$

- **σ > 0**：振幅指數增長（不穩定）
- **σ = 0**：純振盪 e^(jωt)（永恆正弦波）
- **σ < 0**：振幅指數衰減（穩定）

> 到後面的拉普拉斯轉換，s 就是拉氏變數。這裡先建立直覺。

### 4.4 正弦信號

$$x(t) = A \cos(\omega_0 t + \phi)$$

- A：振幅（Amplitude）
- ω₀：角頻率（Angular Frequency），單位 rad/s
- φ：初始相位（Phase）
- 頻率 f₀ = ω₀/(2π)，單位 Hz
- 週期 T = 1/f₀ = 2π/ω₀

**尤拉公式（Euler's Formula）** —— 連結複指數與正弦：

$$e^{j\theta} = \cos\theta + j\sin\theta$$

$$\cos\theta = \frac{e^{j\theta} + e^{-j\theta}}{2}$$

$$\sin\theta = \frac{e^{j\theta} - e^{-j\theta}}{2j}$$

### 數值例題 4.2：信號參數辨識

> **問題**：x(t) = 5cos(100πt + π/4)，求振幅、角頻率、頻率、週期、初始相位。

**解答**：
- 振幅 A = 5
- 角頻率 ω₀ = 100π rad/s
- 頻率 f₀ = ω₀/(2π) = 100π/(2π) = **50 Hz**
- 週期 T = 1/f₀ = 1/50 = **0.02 s**
- 初始相位 φ = π/4 = **45°**

## 五、題型鑑別

| 看到什麼 | 用什麼方法 |
|----------|-----------|
| 「判斷是否為週期信號」 | 求各分量週期，看週期比是否為有理數 |
| 「求能量/功率」 | 用能量/功率公式積分 |
| 「偶奇分解」 | 用 xₑ = [x(t)+x(-t)]/2, x₀ = [x(t)-x(-t)]/2 |
| 「畫 x(at-b) 的圖形」 | 先壓縮再平移，或寫成 x(a(t-b/a)) |
| 「∫ f(t)δ(t-a) dt」 | 取樣性質：直接代 t=a 進 f(t) |
| 「用 u(t) 表示矩形脈衝」 | u(t-a) - u(t-b) 表示 [a,b] 的矩形 |

### 問題 1
x[n] = cos(πn/4) 是否為週期信號？若是，求基本週期 N。

<details>
<summary>點擊查看答案</summary>

ω₀ = π/4，ω₀/(2π) = 1/8，是有理數 → 是週期信號。

基本週期 N = 2π/ω₀ × (使 N 為最小正整數) = 8。

或直接：ω₀N = 2πm → (π/4)N = 2πm → N = 8m，最小正整數 N = 8。

</details>

### 問題 2
計算 ∫₋∞^∞ e^(-3t) δ(t - 1) dt。

<details>
<summary>點擊查看答案</summary>

取樣性質：把 t = 1 代入 e^(-3t)，得到 e^(-3) ≈ 0.0498。

</details>

### 問題 3
x(t) = tu(t)（斜坡信號），畫出 x(-t+2) 的圖形並寫出數學表示式。

<details>
<summary>點擊查看答案</summary>

x(-t+2) = x(-(t-2)) = (-t+2)u(-t+2) = (2-t)u(2-t)。

圖形：在 t < 2 時為直線 (2-t)，在 t ≥ 2 時為 0。
- 反褶 x(-t) 把原來 t>0 的斜坡變成 t<0 的下降線
- 再右移 2，非零區間變成 (-∞, 2]

在 t = 0 時：x(-0+2) = x(2) = 2
在 t = 2 時：x(-2+2) = x(0) = 0

</details>

### 問題 4
判斷 x(t) = e^(-|t|) 是能量信號還是功率信號？

<details>
<summary>點擊查看答案</summary>

$$E = \int_{-\infty}^{\infty} e^{-2|t|} dt = 2\int_{0}^{\infty} e^{-2t} dt = 2 \cdot \frac{1}{2} = 1$$

E = 1 < ∞，所以是**能量信號**。

</details>

### 問題 5
將 x(t) = t³ 分解為偶部分和奇部分。

<details>
<summary>點擊查看答案</summary>

$$x_e(t) = \frac{t^3 + (-t)^3}{2} = \frac{t^3 - t^3}{2} = 0$$

$$x_o(t) = \frac{t^3 - (-t)^3}{2} = \frac{t^3 + t^3}{2} = t^3$$

所以 t³ 是純奇信號（xₑ = 0，xₒ = t³）。

這很直覺：奇數次方函數本來就是奇函數！

</details>
