# Binary Search 四次使用對比——情感耦合遞進

> **設計核心**：Binary Search 在第一卷中被使用四次，每一次的「搜尋對象」都更深、更私密、更接近核心。從搜尋數據 → 搜尋愛 → 搜尋恐懼 → 搜尋記憶碎片。演算法不變，但情感重量每次翻倍。
> **教學目標**：讀者在 ch33 學會 Binary Search 的概念，在 ch35 看到它的變體，在 ch45 看到它的應用，在 ch53 看到它在生死場景中的使用。四次之後，Binary Search 不是「知識」——是「直覺」。

## 參、逐次詳細分析

### 第一次：ch33 記憶庫搜尋——「把人生對折再對折」

**A 線設計**：
```
嚴柏翰的記憶庫。三十四年的人生。

中間點：17 年前。大學時代。太早了。
右半邊：8 年前。監控局時期。太晚了。
縮小：10~11 年前——

「我把他整個人生對折再對折。
在無數個日夜的縫隙裡，
找到了那個大雪紛飛的瞬間。」
```

**B 線設計**：
- 嚴柏翰的脆弱揭露（脫下面具、不記得傷疤）
- 同步點：「大雪紛飛的瞬間」= 搜尋結果 = 情感結果

**字數**：BS 過程 ≤ 300 字。「對折再對折」是讀者需要記住的意象。

**教學功能**：這是讀者第一次見到 BS。必須讓他們理解「為什麼不用從頭找」——因為記憶按時間排序。排序 = BS 的前提。

### 第二次：ch35 身體記憶搜尋——「搜尋的不是數據——是愛」

**A 線設計**：
```
如果數位記憶被鎖死了，還有另一種記憶——身體記憶。

大範圍：視覺 vs 聽覺 vs 嗅覺 → 嗅覺最強
縮小：花香 vs 食物 vs 化學品 → 花香
再縮小：玫瑰 vs 茉莉 vs 桂花 → 茉莉

茉莉花的味道讓嚴柏翰心率飆升到 150。
```

**B 線設計**：
- 嚴柏翰的手反覆握緊、鬆開（握住某人的手，然後放開）
- 同步點：茉莉 = 邱韻如 = 「我的手記得」

**為什麼 ch35 是全卷最佳 AB 線融合？**

因為搜尋的過程本身就是情感的過程：
- 每次「縮小範圍」= 更接近嚴柏翰的核心
- 最終找到的不是「數據」——是他最深的愛
- BS 的邏輯（排除、逼近、定位）完美映射了「在碎裂的記憶中尋找愛」的過程

**三段式結構**（參照 ch31-40 精修稿）：
1. 方法說明（100字）：身體記憶不存在於晶片，無法篡改
2. BS 變體（200字）：感官縮小範圍的三步
3. 崩潰（≤150字）：茉莉 → 心率150 → 握拳鬆開 → 名字 → 哭了

### 第三次：ch45 三千份報告——「搜尋的結果是倒數計時」

**A 線設計**：
```
三千份報告按時間排序。

中間點：三個月前。異常 12 份
右半：一個半月前。43 份
再切：三週前。127 份

不是零星犯罪。
每次覆蓋範圍是前一次的三倍。
```

**B 線設計**：
- 不是個人情感——是「全城」的恐懼
- 同步點：「下一次高峰——全城」+ 347,000 的發現

**字數**：BS 過程 ≤ 150 字。這次 BS 是工具，不是主角。

**與前兩次的區別**：
- ch33/35：BS 用於「個人記憶」——搜尋一個人的秘密
- ch45：BS 用於「系統數據」——搜尋一個陰謀的規模
- 從個人 → 系統的跳躍呼應了全書的 L3 哲學遞進

### 第四次：ch53 冰原碎片——「在一個人的荒蕪中搜尋最後的希望」

**A 線設計**：
```
冰原下的記憶碎片按時間排列。

中間點：17 歲。上大學。太早了
跳到 24 歲：加入某組織。接近了
再跳到 27 歲：在寫安全系統程式碼。就是這裡

維倫在 26~28 歲的記憶層掃描。
找到了後門密碼的第一個碎片。
```

**B 線設計**：
- 冰原 = 十年被刪除的記憶 = 視覺化的荒蕪
- 觸碰碎片 → 黑色液體（記憶吞噬程式）→ 生死追逐
- 同步點：密碼「YUNR」= 邱韻如 = 「他用愛人的名字做了密碼」

**為什麼 ch53 的情感重量最大？**

```
刪除了他所有的記憶，
卻刪不掉密碼——
因為密碼不在記憶裡，在心裡。
```

- ch33 找到的是「被模糊的臉」——還不知道是誰
- ch53 找到的是「YUNRU-2079」——愛的名字成為最後的鑰匙
- 兩次 BS 搜尋嚴柏翰的記憶，隔了 20 章，情感深度天壤之別

### Binary Search 的全書象徵意義

```
Binary Search = 在無邊的混沌中，找到那一個瞬間。

它不是暴力搜索——它是智慧。
它不是隨機——它依賴「排序」（秩序的存在）。
它不是一步到位——它是「每次排除一半」（逐步接近真相）。

這是維倫的方法論：
在一個記憶被篡改的世界裡，
不要試圖知道所有的真相。
每次排除一半的謊言。
一步一步。
直到找到那個無法被偽造的瞬間。
```
