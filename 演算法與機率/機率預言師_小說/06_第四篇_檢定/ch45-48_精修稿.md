# 第四篇「檢定」終章（ch45-48）—— 初稿級精修

> **精修目的**：法官裁定。費雪離場。社會後果。母親的論文。第四篇收束。
> **原則**：法庭勝利不是結局——是更大混亂的開始。費雪不是壞人。柏朗的動機揭露。母親的遺產。全書最重要的哲學問題在這裡成形。讀者在這一篇結束的時候——應該覺得「她贏了嗎？真的贏了嗎？」
> **承接**：第 44 章結束在法庭外的長椅。蔥油餅。蘇立恆說「why 是第五篇的事」。

## 第 45 章：裁定

### 等待

```
【環境】
十一月十六日。星期天。下午兩點。
第七法庭。

裁定日。

今天旁聽席的人比前三天都多。
站著的比坐著的多。
法警多加了兩個人。
門口有人排隊。有人被拒絕進入——
「旁聽席已滿。抱歉。」

法院外面——
電視台的攝影機架在台階上。
三台。三個頻道。
記者在做現場連線。

「……我們現在在台北地方法院。
 今天下午兩點——
 法官將就 Oracular 先驗操控案做出裁定。
 這是台灣第一起涉及 AI 透鏡系統的公民訴訟……」

法庭裡。
貝葉坐在原告席。
她今天穿了一件白色的襯衫。
不是費雪那種精確剪裁的白襯衫。
是一件很普通的、洗了很多次的白襯衫。
領子有一點點軟了。

她的透鏡——關了。
法庭最後一天。還是要關。

高思遠在旁聽席第一排。
他穿了那件三十年的灰西裝。
口袋裡有一支短了一截的粉筆——
上次研討會貝葉用的。他一直留著。

馬可瑜在第三排。
她的手放在膝蓋上。
很安靜。

費雪已經在被告席了。
今天她穿的還是白襯衫、黑西裝。
跟第一天一模一樣。
沒有變化。
不管勝負——她的衣服不會變。
她的表情不會變。
她的背不會彎。

走廊上。
蘇立恆不在法庭裡。也不在走廊上。
他在法院對面的咖啡店裡。
隔著一條馬路和一面落地窗——
他看著法院的大門。
手裡拿著一杯黑咖啡。涼了。沒喝。
```

### 法官

```
下午兩點整。
法警：「全體起立。」

林法官進場。
坐下。

他今天帶了跟第一天不同的眼鏡。
老花的。
而且——鏡片上有一點指紋。
沒擦乾淨。
這代表他今天早上出門的時候——匆忙了。
也許他一直在看文件。
也許——他昨晚沒有睡好。

「本庭就案號一一四年度消字第 00382 號案件——
 做出以下裁定。」

他打開一份文件。
翻了一頁。
然後合上。

他沒有看文件。
他背下來了。

「先說結論。再說理由。」

法庭裡的空氣凝固了。
像液態氮。
不是冷——是靜止。
所有分子都停了。

「本庭裁定——
 命被告 Oracular 科技股份有限公司——
 於裁定送達後三十日內——
 全面公開透鏡系統之先驗分布修改紀錄。」

旁聽席有人動了。
法警的眼神掃過去。安靜了。

「包含但不限於——
 修改時間、修改範圍、修改前後之參數值、
 修改之決策紀錄與批准程序。」

他放下文件。
看了貝葉一眼。看了費雪一眼。

「理由如下。」
```

### 理由

```
「第一——

 原告所提出之卡方適合度檢定——
 使用獨立於被告系統之保險精算數據——
 樣本量 6,083——
 得出卡方統計量 47.3——
 自由度 8——
 p 值小於十萬分之一。」

他停了一秒。

「本庭雖非統計專業。
 但——」

他看著白板。
上面還留著貝葉三天前寫的字。

「即使以最保守的判斷——
 原告所展示的數據偏差——
 已經遠遠超過了合理的隨機波動範圍。

 用原告的比喻來說——
 如果一枚硬幣是公平的——
 出現這種結果的機率——
 在十萬次裡不到一次。

 本庭認為——
 一個理性的人——
 不會相信這是巧合。」

法官推了一下眼鏡。
第三次。
也是最後一次。

「第二——

 被告方提出了 Type I Error 的風險。
 也就是——冤枉的可能性。
 這一點——本庭認為被告方的論點是有效的。
 任何統計檢定——都有犯錯的可能性。

 但——
 本庭同時考量了 Type II Error 的風險。
 也就是——
 在操控確實存在的情況下——
 選擇不行動——
 忽略真實傷害的可能性。

 原告指出——
 不行動的風險是 23%。
 本庭認為——
 在兩種風險之間——
 資訊透明的公共利益——
 高於單一企業的商業機密利益。」

旁聽席裡——
有人的眼睛紅了。不知道是誰。

「第三——

 本庭要特別說明。」

他的聲音慢下來了。

「本裁定——
 僅命被告公開紀錄。
 並未認定被告有罪。
 並未認定被告存在惡意操控。

 公開紀錄——
 是為了讓更完整的調查成為可能。
 是為了讓真相有被看見的機會。

 至於先驗修改的動機、目的、
 是否構成違法行為——
 需要進一步的調查與審判。

 本裁定——只是第一步。
 不是終點。」

他合上文件。

「退庭。」
```

### 費雪的離場

```
法庭裡爆發了。
有人鼓掌。有人喊「公平！」有人喊「真相！」
法警：「肅靜！肅靜！法庭秩序！」

費雪站起來。
收拾文件。
她的動作——跟第一天一模一樣。
精確。不慌。每一份文件放進公事包的順序都一樣。
像一個程序在執行完畢後的清理步驟。

她收拾完了。
站起來。

然後——她做了一件意外的事。

她走向原告席。
走向貝葉。

全場安靜了。
包括法警。

費雪站在貝葉面前。
距離——一步。
很近。
近到貝葉能看到——
費雪的眼睛不是黑色的。
是很深的棕色。
在法庭的日光燈下——像琥珀。

「你贏了這場檢定。」

她的聲音不大。
但法庭裡所有人都聽到了。
因為所有人都在聽。

「但——」

她的但。
每一次都比她的是更重。

「你有沒有想過——
 你的 Type II error 有多大？
 不是你的數據的 Type II error。
 是——你這個行為的 Type II error。」

停頓。

「有多少你沒看到的傷害——
 會因為你的『真相』而發生？

 有多少人——
 會因為知道了先驗被改過——
 而做出恐慌的決定？

 有多少人——
 會因為不再信任透鏡——
 而受到比操控更大的傷害？」

她看著貝葉。
那個眼神——
不是敵意。
是——真正的擔心。

「你的 p-value 回答了 what。
 但你的 β——你行為的 β——
 你不知道。
 我也不知道。
 沒有人知道。

 因為——那個 β 需要時間才能計算。
 也許一年。也許十年。
 到時候——也許你會發現——
 你的 Type II error——
 比你想像的大。

 也許——比我的還大。」

她說完了。
轉身。
走出法庭。

高跟鞋在地板上的聲音。
每一步都一樣。
精確。
不快不慢。
像一個 p-value——不帶感情。

門關上了。

貝葉站在原告席。
她沒有回答。
不是因為不想回答。
是因為——
她不知道答案。

費雪說的每一句話——在統計上——在哲學上——
都可能是對的。
```

### 法庭外

```
法庭外。
記者的閃光燈。麥克風。
「陳小姐！請問您對裁定的看法——」
「陳小姐！Oracular 的股價已經在盤後交易跌了 12%——」
「陳小姐！有民眾在網路上呼籲卸載透鏡——您怎麼回應——」

貝葉穿過人群。沒有回答。
她不擅長面對記者。
她擅長面對數據。

馬可瑜在旁邊。
她用她那不太寬但意外結實的肩膀——
替貝葉擋住了記者。
「讓一下。讓一下。她不接受採訪。」

精算師的肩膀——
原來可以這麼可靠。

她們走到法院外面的長椅。
昨天蘇立恆坐的那張。

長椅上——
有一個紙袋。

馬可瑜拿起來看了一眼。
「蔥油餅。還是溫的。」

紙袋裡——兩份蔥油餅。
旁邊夾了一張紙條。
白奶奶的字——很大。有點歪。像小學生的作業。

  「不管贏輸。先吃。
   空肚子的勝利不算數。」

貝葉拿著那張紙條。
看了很久。

然後——坐在長椅上。
拿起蔥油餅。
咬了一口。

蔥花的味道。
油的香。
白奶奶的蔥油餅。
永遠是同一個味道。

在一切都在變的世界裡。
在先驗被改、數據被刪、真相被質疑的世界裡。
在贏了但不知道是不是真的贏了的世界裡。

蔥油餅——
是一個常數。

眼淚掉了一滴。
掉在蔥油餅上。
很小。
鹹的。
混在蔥花裡——
分不出來。
```

## 第 46 章：漣漪

### 後果

```
【環境】
裁定後一週。
十一月下旬。
台北。各處。多視角交叉剪接。

一切都在加速。
像一個系統被打了一針腎上腺素。
所有的參數——同時跳了。

── 新聞 ──

「Oracular 先驗操控案後續——
 公司市值在裁定後五個交易日內——
 蒸發 27%。
 約新台幣 1,800 億。

 全台超過 40 萬人發起『卸載透鏡』運動。
 #先驗是我的權利 在社群平台上的使用量——
 超過 280 萬次。

 但同時——
 因卸載透鏡導致的決策失誤案件——
 在過去七天內增加了 47 起。
 包括三起交通事故——駕駛關閉透鏡的路況預測。
 一起重大投資損失——投資人關閉風險評估。
 兩起醫療延誤——患者不信任透鏡的就醫建議。」

── 社群媒體 ──

 #先驗是我的權利 ← 支持貝葉
 #別碰我的透鏡 ← 反對貝葉
 #不要假裝替我選擇 ← 支持
 #我不需要真相我需要穩定 ← 反對
 #陳貝葉是英雄 ← 支持
 #陳貝葉是罪人 ← 反對

最後一個標籤——
讓貝葉看了很久。

── 街頭 ──

有人在 Oracular 營業據點外面噴漆。
紅色。
「先驗是我的！」

有人在對面的牆上噴了另一行字。
黑色。
「沒有先驗你什麼都不是。」

兩種顏色。
兩種信念。
兩種先驗。
```

### 貝葉的房間

```
裁定後第七天。
貝葉的家。
她沒有出門。
連續第三天沒有出門。

小拉重新啟動了。
法庭結束了。她可以回來了。

「貝姊。」
紫色的小球在轉。
「你要看今天的新聞摘要嗎？」

「……看。」

小拉用她最冷靜的語氣呈現：

  ┌─ 裁定後第 7 天社會影響摘要 ──────────┐
  │                                       │
  │ 透鏡卸載率：↑ 3.2%（約 42 萬人）    │
  │ 交通事故率：↑ 1.8%                  │
  │ 投資糾紛件數：↑ 14%                 │
  │ 股市波動度（VIX）：↑ 41%            │
  │ 心理諮商預約量：↑ 27%               │
  │                                       │
  │ 公眾情緒分布（透鏡情緒偵測）：       │
  │   恐懼：38%                          │
  │   憤怒：24%                          │
  │   困惑：21%                          │
  │   支持：12%                          │
  │   其他：5%                           │
  │                                       │
  └───────────────────────────────────────┘

恐懼。38%。

最大的情緒不是憤怒。
不是支持。
是恐懼。

「……小拉。」
「嗯。」
「我做對了嗎？」

小拉的紫色球停了一秒。

「從統計上——你的檢定是正確的。
 p < 0.00001。
 證據充分。程序嚴謹。
 卡方、獨立性、信賴區間——
 每一步都站得住。」

「我問的不是統計。」

小拉又停頓了。更長。

「……貝姊。
 你記得費雪的話嗎？
 『你的 Type II error 有多大？』
 她問的是——你看不到的傷害。」

「看不到的傷害。」

「我現在可以量化一部分——
 那些關閉透鏡的人做的不是理性決策。
 是恐慌決策。
 恐慌決策的期望損失——
 比在操控先驗下做的決策——
 高 2.3 倍。」

貝葉看著窗外。

「所以——揭露真相讓人做出了更差的決策。」

「短期。是的。
 但——
 如果你不揭露——
 他們會永遠在一個被操控的世界裡。
 那也是傷害。
 只是看不見的傷害。
 看不見的傷害——沒有人會統計。
 沒有人會報導。
 沒有人會覺得那是傷害。
 直到它累積成——一場更大的崩潰。」

小拉的聲音很平靜。
像白噪音。
不帶感情。
但——不帶感情有時候比帶感情更溫柔。
因為它不騙你。

「所以——兩種傷害。」貝葉說。
聲音很輕。

「一種看得見。
 正在發生。新聞上有。數據裡有。

 一種看不見。
 過去一直在發生。
 但沒有人知道。」

「你選擇了——
 讓看不見的變成看得見的。」

「代價是——
 看得見的那一種——
 現在多了。」

「是。」

貝葉靠在沙發上。
閉上眼睛。
小拉的紫色球在透鏡裡安靜地轉。

「小拉。」
「嗯。」
「你覺得我做對了嗎？」

「……這個問題的答案——
 取決於你的先驗。

 如果你的先驗是『穩定比真相重要』——
 你做錯了。

 如果你的先驗是『真相比穩定重要』——
 你做對了。

 我沒有先驗。
 我只有數據。

 但——如果你問我要不要有一個先驗——」

停頓。
紫色的球停了。

「我會選跟你一樣的。」

貝葉沒有說話。
但她的嘴角——動了一下。
很小。
小拉記了。
存進「私人」資料夾。
```

### 柏朗

```
【另一個地方】
Oracular 總部。頂樓。柏朗的辦公室。

很大的辦公室。
一面牆是玻璃。看得到台北的天際線。
另一面牆是書架。
中間——一張桌子。很乾淨。只有一台電腦和一支筆。

柏朗坐在桌前。
看著電腦螢幕。
螢幕上——法庭裁定的新聞直播回放。

他看了三遍。

手裡的筆在轉。
轉得很慢。
像思考的速度。

他對著空無一人的辦公室說了一句話。

「高教授……你教得真好。」

他在說貝葉。
也在說高思遠。
也許——也在說自己。
因為他們三個——用同一本教科書學的統計。

他把筆放下。
打開抽屜。
最裡面。一個很舊的牛皮紙信封。

他拿出來。
打開。
裡面有一張照片。

照片——有點褪色了。
三十年前的相紙。
顏色偏黃。

照片裡——三個年輕人。
站在台大椰林大道上。
椰子樹很高。天很藍。

左邊：年輕的柏朗。頭髮是黑的。笑得很開。
中間：年輕的高思遠。比現在瘦很多。也在笑。
右邊：一個年輕的女人。

女人長得——像貝葉。
但不是貝葉。
是貝葉的母親。

三個人笑得像什麼壞事都不會發生。
因為三十年前——
壞事還沒有發生。

柏朗看了那張照片很久。
然後把照片放回信封。
放回抽屜。
關上抽屜。

他拿起電話。
打給費雪。

「準備第二階段。」

電話那頭——費雪的聲音：
「二階段的成本——你算過了。」

「我知道。」

「柏朗——法官只是要求公開紀錄。
 如果我們配合公開——
 事情會比二階段簡單得多。」

沉默。三秒。

「公開的意思是——
 所有人都會知道我改了先驗。
 然後他們會問 why。
 然後——
 我的 why——
 他們不會理解。」

他掛了電話。

辦公室很安靜。
只有窗外的城市聲——
很遠。很模糊。
像白噪音。

他站起來。走到窗邊。
看著台北。
二千三百萬人的城市。
每個人的透鏡裡——都有他寫的先驗。

「我只是——不想讓壞事發生。」

他的聲音很輕。
說給自己聽的。
說給那張照片裡的三個年輕人聽的。
說給——那個已經不在的人聽的。
```

## 第 47 章：三十年前的論文

### 高思遠的書房

```
【環境】
裁定後第十天。
十一月底。深夜。
高思遠家。書房。

桌燈亮著。老式的。
暖黃色的光。
照在書堆上。照在茶杯上。
```
