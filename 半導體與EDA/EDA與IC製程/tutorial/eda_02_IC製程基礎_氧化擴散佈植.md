## 🔰 本章基礎觀念（零基礎必讀）

### IC 是怎麼「做」出來的？

IC 製造的核心就是在矽晶圓（Silicon Wafer）上，透過一連串的物理和化學製程，「蓋」出數十億個微小的電晶體。

把它想像成蓋一棟大樓：
- **矽晶圓** = 地基
- **氧化** = 刷防水漆（絕緣層）
- **擴散 / 離子佈植** = 灌入鋼筋（摻雜改變導電性）
- **微影** = 照設計圖切出形狀
- **蝕刻** = 挖掉不需要的部分
- **薄膜沉積** = 鋪上新的建材層

本章專注在前三項：**氧化、擴散、離子佈植**。

### 為什麼矽是半導體之王？

| 特性 | 說明 |
|------|------|
| 地殼含量豐富 | 地殼中第二多的元素（約 28%） |
| 天然氧化物好 | SiO₂ 是優秀的絕緣體，且與矽的界面品質極佳 |
| 可精確控制導電性 | 透過摻雜（Doping），可在絕緣體～導體之間精準調整 |
| 技術成熟 | 超過 60 年的製造經驗累積 |

## 一、晶圓製造：柴氏長晶法（Czochralski Method）

### 1.1 從沙子到晶圓

```
沙子 (SiO₂)
    │  冶金級矽精煉
    ▼
多晶矽 (Polycrystalline Si, 純度 99.9999999%, 9N)
    │  柴氏長晶法 (CZ method)
    ▼
單晶矽晶棒 (Ingot)
    │  切割 (Sawing)
    ▼
矽晶圓 (Wafer, 300mm / 12 吋)
    │  研磨、拋光 (Lapping, Polishing)
    ▼
拋光晶圓 (Polished Wafer) → 進入 IC 製造
```

### 1.2 柴氏長晶法步驟

1. 將高純度多晶矽放入石英坩堝中加熱到 **1414°C**（矽的熔點）以上
2. 將一根小的單晶「種晶（Seed Crystal）」浸入矽熔液表面
3. 緩慢旋轉並向上拉起種晶
4. 矽熔液在種晶表面結晶，沿著種晶的晶格方向生長
5. 控制拉速和溫度，逐漸長出圓柱形的單晶矽晶棒

### 1.3 晶圓規格

| 參數 | 典型值 |
|------|--------|
| 直徑 | 300mm（12 吋），下一代 450mm |
| 厚度 | ~775 μm |
| 晶向 | (100)（CMOS 最常用）或 (111) |
| 電阻率 | 1～50 Ω·cm（依應用調整） |
| 型態 | P-type（硼摻雜）最常見 |

## 二、氧化（Oxidation）

### 2.1 為什麼要氧化？

SiO₂（二氧化矽）在 IC 製程中扮演多種角色：

| 用途 | 說明 |
|------|------|
| 閘氧化層（Gate Oxide） | MOSFET 的核心絕緣層，控制開關 |
| 場氧化層（Field Oxide） | 隔離不同電晶體 |
| 遮蔽層（Masking） | 阻擋離子佈植或擴散 |
| 保護層（Passivation） | 保護晶片表面 |

### 2.2 氧化的化學反應

#### 乾式氧化（Dry Oxidation）

$$\text{Si} + \text{O}_2 \rightarrow \text{SiO}_2$$

- **特點**：生長慢，但薄膜品質好、緻密
- **用途**：閘氧化層（Gate Oxide）—— 需要極高品質
- **生長速率**：約 1～5 nm/min（1000°C）

#### 濕式氧化（Wet Oxidation）

$$\text{Si} + 2\text{H}_2\text{O} \rightarrow \text{SiO}_2 + 2\text{H}_2$$

- **特點**：生長快，但品質稍差、較不緻密
- **用途**：場氧化層 —— 需要較厚的氧化層（速度重要）
- **生長速率**：約 10～50 nm/min（1000°C）

### 2.3 氧化的物理機制

氧化物生長時，**氧化劑（O₂ 或 H₂O）必須穿過已經長出來的 SiO₂ 層才能到達 Si/SiO₂ 界面繼續反應**。

```
     氧化劑 (O₂ or H₂O)
          │
          │ ① 從氣體擴散到表面
          ▼
    ┌───────────────┐
    │   SiO₂ 層     │ ② 穿過 SiO₂ 擴散
    │  (越長越厚)    │
    ├───────────────┤
    │   Si 基板     │ ③ 在 Si/SiO₂ 界面反應
    └───────────────┘
```

關鍵觀察：
- 氧化層越厚 → 氧化劑要走越遠 → 生長越慢
- 這就是為什麼 **初期線性成長，後期拋物線成長**

### 2.4 Deal-Grove 模型 ★核心公式

Deal-Grove 模型描述氧化層厚度 $x_{ox}$ 與時間 $t$ 的關係：

$$\boxed{x_{ox}^2 + A \cdot x_{ox} = B \cdot (t + \tau)}$$

其中：
- $x_{ox}$：氧化層厚度（μm 或 nm）
- $t$：氧化時間
- $\tau$：初始氧化層的等效時間偏移
- $A$, $B$：與溫度和氧化劑類型有關的參數

#### 兩個極端情況

| 情況 | 條件 | 近似公式 | 物理意義 |
|------|------|---------|---------|
| **薄氧化層**（初期） | $x_{ox} \ll A/2$ | $x_{ox} \approx \frac{B}{A}(t + \tau)$ | 表面反應速率主導（線性） |
| **厚氧化層**（長時間） | $x_{ox} \gg A/2$ | $x_{ox} \approx \sqrt{B(t + \tau)}$ | 擴散速率主導（拋物線） |

- **$B/A$**：線性速率常數（Linear Rate Constant）
- **$B$**：拋物線速率常數（Parabolic Rate Constant）

#### 解 Deal-Grove 方程式

從 $x_{ox}^2 + A \cdot x_{ox} = B(t + \tau)$，解出 $x_{ox}$：

$$x_{ox} = \frac{A}{2}\left(\sqrt{1 + \frac{4B(t+\tau)}{A^2}} - 1\right)$$

### 2.5 氧化的重要觀念

#### 矽的消耗

每生長 $x_{ox}$ 厚的 SiO₂，會消耗約 **0.44·$x_{ox}$** 厚的矽。

```
          氧化前                    氧化後
    ─────────────────      ───── 原始表面 ──────
                           │  0.54·x_ox  │ SiO₂
                           │             │ (在原表面上方)
    ─── Si 表面 ────       ├─────────────┤
    │               │      │  0.46·x_ox  │ SiO₂
    │    Si         │      │             │ (在原表面下方)
    │               │      ├─────────────┤
                           │    Si       │
```

即：SiO₂ 總厚度 $x_{ox}$ 中，約 **46%** 長在原始矽表面下方，**54%** 在上方。

### 數值例題 1：Deal-Grove 基本計算

**題目**：在乾式氧化條件下（1100°C），已知 $A = 0.090$ μm, $B = 0.027$ μm²/hr。若初始無氧化層（$\tau = 0$），求：
(a) 氧化 1 小時後的氧化層厚度
(b) 長到 0.1 μm 需要多少時間？

**解答**：

**(a)** 使用 Deal-Grove 公式：

$$x_{ox} = \frac{A}{2}\left(\sqrt{1 + \frac{4Bt}{A^2}} - 1\right)$$

代入 $A = 0.090$, $B = 0.027$, $t = 1$：

$$x_{ox} = \frac{0.090}{2}\left(\sqrt{1 + \frac{4 \times 0.027 \times 1}{0.090^2}} - 1\right)$$

$$= 0.045\left(\sqrt{1 + \frac{0.108}{0.0081}} - 1\right) = 0.045\left(\sqrt{1 + 13.33} - 1\right)$$

$$= 0.045\left(\sqrt{14.33} - 1\right) = 0.045 \times (3.786 - 1) = 0.045 \times 2.786$$

$$\boxed{x_{ox} \approx 0.125 \text{ μm} = 125 \text{ nm}}$$

**(b)** 代入 $x_{ox} = 0.1$ μm 到 $x_{ox}^2 + A \cdot x_{ox} = Bt$：

$$0.1^2 + 0.090 \times 0.1 = 0.027 \times t$$

$$0.01 + 0.009 = 0.027t$$

$$t = \frac{0.019}{0.027} \approx \boxed{0.704 \text{ hr} \approx 42 \text{ min}}$$

### 數值例題 2：乾式 vs 濕式比較

**題目**：在 1000°C 下，乾式氧化參數 $B/A = 0.0117$ μm/hr, $B = 0.0049$ μm²/hr；濕式氧化 $B/A = 0.203$ μm/hr, $B = 0.287$ μm²/hr。若要生長 50 nm 的氧化層，兩種方法各需多少時間？

**解答**：

50 nm = 0.05 μm，屬於薄氧化層，用線性近似：$x_{ox} \approx (B/A) \cdot t$

**乾式**：$t = x_{ox}/(B/A) = 0.05/0.0117 \approx \boxed{4.27 \text{ hr}}$

**濕式**：$t = x_{ox}/(B/A) = 0.05/0.203 \approx \boxed{0.246 \text{ hr} \approx 14.8 \text{ min}}$

乾式需要約 4.3 小時，濕式只需約 15 分鐘。但閘氧化層仍選用乾式，因為品質要求極高。

## 三、擴散（Diffusion）

### 3.1 什麼是擴散？

擴散是將摻雜原子（如硼 B、磷 P、砷 As）從**高濃度區域向低濃度區域移動**的過程。高溫（900～1200°C）下，摻雜原子在矽晶格中「跳躍」前進。

### 3.2 Fick 第一定律

$$\boxed{J = -D \frac{\partial C}{\partial x}}$$

- $J$：擴散通量（atoms/cm²·s），單位時間通過單位面積的原子數
- $D$：擴散係數（cm²/s），取決於溫度和摻雜種類
- $\partial C/\partial x$：濃度梯度
- 負號表示：原子從高濃度往低濃度方向移動

### 3.3 Fick 第二定律

$$\boxed{\frac{\partial C}{\partial t} = D \frac{\partial^2 C}{\partial x^2}}$$

這是一個偏微分方程式（PDE），描述濃度如何隨時間和位置變化。根據不同的邊界條件，有兩個重要的解。

### 3.4 恆定表面濃度擴散（Constant Source Diffusion）

**邊界條件**：表面濃度 $C_s$ 恆定（源源不斷的摻雜源）

$$\boxed{C(x,t) = C_s \cdot \text{erfc}\left(\frac{x}{2\sqrt{Dt}}\right)}$$

其中 $\text{erfc}(z) = 1 - \text{erf}(z) = 1 - \frac{2}{\sqrt{\pi}} \int_0^z e^{-u^2} du$

**特性**：
- 表面濃度始終為 $C_s$（通常等於固溶度 Solid Solubility）
- 總劑量隨時間增加：$Q(t) = \frac{2 C_s \sqrt{Dt}}{\sqrt{\pi}}$
- 常用於**預沉積（Pre-deposition）**步驟

```
C (濃度)
▲
│ Cs ──┐
│      │╲
│      │ ╲  t₁
│      │  ╲
│      │   ╲╲  t₂ > t₁
│      │    ╲╲
│      │     ╲╲╲
└──────┴───────────────▶ x (深度)
```

### 3.5 固定總劑量擴散（Limited Source Diffusion）

**邊界條件**：總劑量 $Q$ 固定（摻雜原子總量不變）

$$\boxed{C(x,t) = \frac{Q}{\sqrt{\pi D t}} \exp\left(-\frac{x^2}{4Dt}\right)}$$

**特性**：
- 高斯分佈（Gaussian Distribution）
- 表面濃度隨時間**下降**：$C(0,t) = Q/\sqrt{\pi Dt}$
- 總劑量守恆：$\int_0^{\infty} C(x,t) dx = Q$
- 常用於**驅入（Drive-in）**步驟

```
C (濃度)
▲
│   t₁（窄而高）
│   ╱╲
│  ╱  ╲
│ ╱    ╲   t₂ > t₁（矮而寬）
│╱  ╱────╲╲
│──╱──────╲╲───
└──────────────────▶ x (深度)
```

### 3.6 接面深度（Junction Depth, $x_j$）

接面深度是摻雜濃度等於背景濃度 $C_B$ 的深度位置。

**對恆定表面濃度**：

$$C_B = C_s \cdot \text{erfc}\left(\frac{x_j}{2\sqrt{Dt}}\right) \implies x_j = 2\sqrt{Dt} \cdot \text{erfc}^{-1}\left(\frac{C_B}{C_s}\right)$$

**對固定總劑量**：

$$C_B = \frac{Q}{\sqrt{\pi Dt}} \exp\left(-\frac{x_j^2}{4Dt}\right) \implies x_j = 2\sqrt{Dt \cdot \ln\left(\frac{Q}{C_B\sqrt{\pi Dt}}\right)}$$

### 3.7 擴散係數的溫度依賴性

$$D = D_0 \exp\left(-\frac{E_a}{kT}\right)$$

- $D_0$：頻率因子（cm²/s）
- $E_a$：活化能（eV）
- $k$：波茲曼常數 = $8.617 \times 10^{-5}$ eV/K
- $T$：絕對溫度（K）

| 摻雜物 | $D_0$ (cm²/s) | $E_a$ (eV) | 在矽中的特性 |
|--------|---------------|-------------|-------------|
| 硼 (B) | 10.5 | 3.69 | P-type，擴散較快 |
| 磷 (P) | 10.5 | 3.69 | N-type，擴散中等 |
| 砷 (As) | 0.32 | 3.56 | N-type，擴散慢（原子大） |

### 數值例題 3：恆定表面濃度擴散

**題目**：磷在矽中的擴散，溫度 1100°C，$D = 3.0 \times 10^{-13}$ cm²/s，表面濃度 $C_s = 5 \times 10^{20}$ cm⁻³，背景濃度 $C_B = 1 \times 10^{15}$ cm⁻³。擴散 30 分鐘後，求：
(a) 接面深度 $x_j$
(b) 總劑量 $Q$

**解答**：

**(a)** 先求 $Dt$：

$$Dt = 3.0 \times 10^{-13} \times (30 \times 60) = 3.0 \times 10^{-13} \times 1800 = 5.4 \times 10^{-10} \text{ cm}^2$$

$$\sqrt{Dt} = 2.324 \times 10^{-5} \text{ cm} = 0.2324 \text{ μm}$$

求 erfc 的反函數引數：

$$\frac{C_B}{C_s} = \frac{10^{15}}{5 \times 10^{20}} = 2 \times 10^{-6}$$

查 erfc 反函數表：$\text{erfc}^{-1}(2 \times 10^{-6}) \approx 3.36$

$$x_j = 2\sqrt{Dt} \times 3.36 = 2 \times 0.2324 \times 3.36 \approx \boxed{1.56 \text{ μm}}$$

**(b)** 總劑量：

$$Q = \frac{2 C_s \sqrt{Dt}}{\sqrt{\pi}} = \frac{2 \times 5 \times 10^{20} \times 2.324 \times 10^{-5}}{\sqrt{\pi}}$$

$$= \frac{2.324 \times 10^{16}}{1.7725} \approx \boxed{1.31 \times 10^{16} \text{ cm}^{-2}}$$

---

### 數值例題 4：固定總劑量擴散（Drive-in）

**題目**：將例題 3 的結果做為 Drive-in 的初始條件。在 1200°C 驅入 2 小時，$D_{1200°C} = 2.0 \times 10^{-12}$ cm²/s，求驅入後的：
(a) 表面濃度
(b) 接面深度

**解答**：

**(a)** 先計算 Drive-in 的 $Dt$：

$$Dt = 2.0 \times 10^{-12} \times (2 \times 3600) = 2.0 \times 10^{-12} \times 7200 = 1.44 \times 10^{-8} \text{ cm}^2$$

$$\sqrt{Dt} = 1.2 \times 10^{-4} \text{ cm}$$

表面濃度（使用高斯解，$x = 0$）：

$$C(0) = \frac{Q}{\sqrt{\pi Dt}} = \frac{1.31 \times 10^{16}}{\sqrt{\pi \times 1.44 \times 10^{-8}}}$$

$$= \frac{1.31 \times 10^{16}}{2.127 \times 10^{-4}} \approx \boxed{6.16 \times 10^{19} \text{ cm}^{-3}}$$

**(b)** 接面深度（$C_B = 10^{15}$ cm⁻³）：

$$x_j = 2\sqrt{Dt \cdot \ln\left(\frac{C(0)}{C_B}\right)} = 2\sqrt{1.44 \times 10^{-8} \times \ln\left(\frac{6.16 \times 10^{19}}{10^{15}}\right)}$$

$$= 2\sqrt{1.44 \times 10^{-8} \times \ln(6.16 \times 10^{4})} = 2\sqrt{1.44 \times 10^{-8} \times 11.03}$$

$$= 2\sqrt{1.588 \times 10^{-7}} = 2 \times 3.985 \times 10^{-4} \approx \boxed{7.97 \times 10^{-4} \text{ cm} \approx 7.97 \text{ μm}}$$

## 四、離子佈植（Ion Implantation）

### 4.1 為什麼用離子佈植取代擴散？

| 比較 | 擴散 | 離子佈植 |
|------|------|---------|
| 劑量控制 | 較差（靠時間/溫度） | 精確（直接量測離子電流） |
| 濃度分佈 | erfc 或高斯 | 可設計任意分佈 |
| 溫度 | 高溫 900～1200°C | 室溫（佈植本身） |
| 光阻遮蔽 | 需要 SiO₂ 遮蔽 | 光阻即可遮蔽 |
| 橫向擴散 | 嚴重 | 輕微 |
| 現代製程 | 幾乎不用 | ★主流方法 |

### 4.2 離子佈植原理

```
    離子源              加速管             晶圓
  ┌────────┐        ┌─────────┐       ┌─────┐
  │ BF₃,   │──▶ 離子 │  電場加速 │──▶ 高速 │     │
  │ PH₃,   │   提取  │ 10～500  │   離子  │ Si  │
  │ AsH₃   │        │  keV    │  打入→  │     │
  └────────┘        └─────────┘       └─────┘
       ↑                 ↑                ↑
    質量分析器          精確控制          掃描
   (選擇離子)          能量             均勻佈植
```

### 4.3 佈植分佈

離子打入矽後，形成近似**高斯分佈**：

$$\boxed{C(x) = \frac{\phi}{\sqrt{2\pi} \Delta R_p} \exp\left(-\frac{(x - R_p)^2}{2(\Delta R_p)^2}\right)}$$

其中：
- $\phi$：劑量（dose, atoms/cm²），即單位面積打入的總離子數
- $R_p$：投射範圍（Projected Range），峰值位置深度
- $\Delta R_p$：散佈（Straggle），分佈寬度（標準差）

```
C (濃度)
▲
│           峰值 = φ/(√(2π)·ΔRp)
│            ╱╲
│           ╱  ╲
│          ╱    ╲
│         ╱      ╲
│        ╱   ΔRp  ╲
│       ╱    ←→    ╲
│──────╱──────┼──────╲──────
└──────────── Rp ──────────▶ x (深度)
              ↑
          峰值深度
```

### 4.4 峰值濃度

$$\boxed{C_{peak} = C(R_p) = \frac{\phi}{\sqrt{2\pi} \cdot \Delta R_p}}$$

### 4.5 $R_p$ 和 $\Delta R_p$ 的影響因素

| 因素 | $R_p$ 的變化 | $\Delta R_p$ 的變化 |
|------|-------------|-------------------|
| 加速能量 ↑ | 增加（打更深） | 增加（散佈更廣） |
| 離子質量 ↑ | 減少（重離子停得快） | 減少（散佈較窄） |
| 基板原子質量 ↑ | 減少 | 減少 |

典型值（矽基板）：

| 離子 | 能量 | $R_p$ (nm) | $\Delta R_p$ (nm) |
|------|------|-----------|-------------------|
| B⁺ | 50 keV | 160 | 53 |
| P⁺ | 100 keV | 120 | 45 |
| As⁺ | 100 keV | 60 | 20 |

### 4.6 通道效應（Channeling）

當離子沿著晶格的對稱軸方向打入時，可能「穿過」原子間的通道而深入很遠，導致佈植分佈出現長尾巴。

**對策**：
- 晶圓傾斜 **7°** 角
- 表面先長一層非晶質 SiO₂
- 先做一次低劑量預非晶化佈植

### 4.7 退火（Annealing）

離子佈植會破壞矽晶格結構（產生缺陷）。需要高溫退火來：
1. **修復晶格損傷**
2. **活化摻雜原子**（讓它們進入晶格替位位置才能提供載子）

| 退火方式 | 溫度 | 時間 | 特點 |
|---------|------|------|------|
| 爐管退火 | 800～1000°C | 30～60 min | 傳統方式，擴散較多 |
| 快速熱退火（RTA） | 1000～1100°C | 1～60 sec | ★主流，減少擴散 |
| 毫秒退火（MSA） | ~1300°C | ~ms | 先進製程，極低擴散 |
| 雷射退火（LSA） | 表面熔融 | ~μs | 最低擴散 |

### 數值例題 5：離子佈植劑量計算

**題目**：磷離子佈植能量 80 keV（$R_p = 100$ nm, $\Delta R_p = 40$ nm），劑量 $\phi = 5 \times 10^{13}$ cm⁻²。求：
(a) 峰值濃度
(b) 表面濃度（$x = 0$）
(c) 深度 200 nm 處的濃度

**解答**：

**(a)** 峰值濃度：

$$C_{peak} = \frac{\phi}{\sqrt{2\pi} \cdot \Delta R_p} = \frac{5 \times 10^{13}}{\sqrt{2\pi} \times 40 \times 10^{-7}}$$

$$= \frac{5 \times 10^{13}}{2.507 \times 4 \times 10^{-6}} = \frac{5 \times 10^{13}}{1.003 \times 10^{-5}} \approx \boxed{4.99 \times 10^{18} \text{ cm}^{-3}}$$

**(b)** 表面濃度（$x = 0$）：

$$C(0) = C_{peak} \cdot \exp\left(-\frac{R_p^2}{2\Delta R_p^2}\right) = 4.99 \times 10^{18} \times \exp\left(-\frac{100^2}{2 \times 40^2}\right)$$

$$= 4.99 \times 10^{18} \times \exp(-3.125) = 4.99 \times 10^{18} \times 0.0439$$

$$\approx \boxed{2.19 \times 10^{17} \text{ cm}^{-3}}$$

**(c)** 深度 200 nm 處：

$$C(200) = C_{peak} \cdot \exp\left(-\frac{(200 - 100)^2}{2 \times 40^2}\right) = C_{peak} \cdot \exp(-3.125)$$

$$= 4.99 \times 10^{18} \times 0.0439 \approx \boxed{2.19 \times 10^{17} \text{ cm}^{-3}}$$

注意：$x = 0$ 和 $x = 200$ nm 的濃度相同！因為它們到峰值 ($R_p = 100$ nm) 的距離都是 100 nm，高斯分佈是對稱的。

### 數值例題 6：佈植後退火的擴散效應

**題目**：承例題 5，佈植後做 RTA 退火，1050°C 持續 10 秒，磷在此溫度的 $D = 5 \times 10^{-14}$ cm²/s。求退火後的等效散佈。

**解答**：

佈植的原始散佈 $\Delta R_p = 40$ nm = $4 \times 10^{-6}$ cm

退火導致的額外擴散：$\sqrt{2Dt} = \sqrt{2 \times 5 \times 10^{-14} \times 10} = \sqrt{10^{-12}} = 10^{-6}$ cm = 10 nm

等效總散佈（兩個高斯卷積）：

$$\sigma_{total} = \sqrt{(\Delta R_p)^2 + 2Dt} = \sqrt{40^2 + 10^2} = \sqrt{1600 + 100} = \sqrt{1700} \approx \boxed{41.2 \text{ nm}}$$

RTA 只讓散佈從 40 nm 增加到 41.2 nm（增加 ~3%），這就是 RTA 的優勢 —— 退火時間短，擴散極少。

### 數值例題 7：兩步驟製程（佈植 + 擴散）

**題目**：先做硼離子佈植 $\phi = 2 \times 10^{14}$ cm⁻²，$R_p = 50$ nm。然後在 1100°C 做 Drive-in 1 小時，$D = 4 \times 10^{-13}$ cm²/s，背景濃度 $C_B = 10^{15}$ cm⁻³。假設 Drive-in 後分佈近似高斯（忽略初始 $\Delta R_p$），求接面深度。

**解答**：

$$Dt = 4 \times 10^{-13} \times 3600 = 1.44 \times 10^{-9} \text{ cm}^2$$

由於 $2Dt = 2.88 \times 10^{-9}$ cm² 遠大於 $(\Delta R_p)^2 = (5 \times 10^{-6})^2 = 2.5 \times 10^{-11}$ cm²，初始散佈可以忽略。

表面濃度：

$$C(0) = \frac{\phi}{\sqrt{\pi Dt}} = \frac{2 \times 10^{14}}{\sqrt{\pi \times 1.44 \times 10^{-9}}} = \frac{2 \times 10^{14}}{6.726 \times 10^{-5}} = 2.97 \times 10^{18} \text{ cm}^{-3}$$

接面深度：

$$x_j = 2\sqrt{Dt \cdot \ln\left(\frac{C(0)}{C_B}\right)} = 2\sqrt{1.44 \times 10^{-9} \times \ln\left(\frac{2.97 \times 10^{18}}{10^{15}}\right)}$$

$$= 2\sqrt{1.44 \times 10^{-9} \times \ln(2970)} = 2\sqrt{1.44 \times 10^{-9} \times 8.0}$$

$$= 2\sqrt{1.152 \times 10^{-8}} = 2 \times 1.073 \times 10^{-4} = \boxed{2.15 \times 10^{-4} \text{ cm} \approx 2.15 \text{ μm}}$$

## ✅ 自我檢測

### 基礎題

<details>
<summary>Q1：乾式氧化和濕式氧化的主要差異是什麼？各適用於什麼場合？</summary>

**答案**：
- **乾式（O₂）**：生長慢，但薄膜緻密、品質好 → 用於**閘氧化層**
- **濕式（H₂O）**：生長快，品質稍差 → 用於**場氧化層**等需要厚氧化層的場合

化學反應：
- 乾式：Si + O₂ → SiO₂
- 濕式：Si + 2H₂O → SiO₂ + 2H₂
</details>

<details>
<summary>Q2：Deal-Grove 模型中，B/A 和 B 分別代表什麼物理意義？</summary>

**答案**：
- **B/A（線性速率常數）**：在氧化層很薄時主導，此時**表面反應速率**是瓶頸（氧化劑很容易就到達 Si/SiO₂ 界面）
- **B（拋物線速率常數）**：在氧化層很厚時主導，此時**擴散速率**是瓶頸（氧化劑穿過厚 SiO₂ 很慢）
</details>

<details>
<summary>Q3：Fick 第一和第二定律各描述什麼？</summary>

**答案**：
- **Fick 第一定律** $J = -D(\partial C/\partial x)$：描述擴散通量（穩態，濃度不隨時間變）
- **Fick 第二定律** $\partial C/\partial t = D(\partial^2 C/\partial x^2)$：描述濃度隨時間如何變化（非穩態）
</details>

<details>
<summary>Q4：離子佈植相比擴散的三個主要優勢？</summary>

**答案**：
1. **劑量控制精確**：直接量測離子電流，精度可達 ±1%
2. **室溫製程**：不需高溫，減少已有結構的擴散
3. **橫向擴散極少**：離子幾乎垂直打入，不像高溫擴散會橫向擴展
4. （加分）可穿過薄膜佈植，光阻即可遮蔽
</details>

### 計算題

<details>
<summary>Q5：若 B/A = 0.01 μm/hr, B = 0.005 μm²/hr，求 2 小時乾式氧化的氧化層厚度（初始無氧化層）。</summary>

**答案**：

先算 $A = B/(B/A) = 0.005/0.01 = 0.5$ μm

$$x_{ox} = \frac{0.5}{2}\left(\sqrt{1 + \frac{4 \times 0.005 \times 2}{0.5^2}} - 1\right)$$

$$= 0.25\left(\sqrt{1 + \frac{0.04}{0.25}} - 1\right) = 0.25(\sqrt{1.16} - 1)$$

$$= 0.25 \times (1.077 - 1) = 0.25 \times 0.077 = 0.019 \text{ μm} = 19 \text{ nm}$$

驗證：$x_{ox}$ = 19 nm 遠小於 $A/2$ = 250 nm，所以用線性近似也可以：

$x_{ox} \approx (B/A) \times t = 0.01 \times 2 = 0.02$ μm = 20 nm（近似值，與精確值接近）
</details>

<details>
<summary>Q6：砷離子佈植 100 keV，Rp = 60 nm, ΔRp = 20 nm, φ = 1×10¹⁵ cm⁻²。求峰值濃度和表面濃度。</summary>

**答案**：

峰值濃度：
$$C_{peak} = \frac{10^{15}}{\sqrt{2\pi} \times 20 \times 10^{-7}} = \frac{10^{15}}{5.013 \times 10^{-6}} = 1.995 \times 10^{20} \text{ cm}^{-3} \approx 2.0 \times 10^{20} \text{ cm}^{-3}$$

表面濃度（$x = 0$）：
$$C(0) = C_{peak} \times \exp\left(-\frac{60^2}{2 \times 20^2}\right) = 2.0 \times 10^{20} \times e^{-4.5}$$
$$= 2.0 \times 10^{20} \times 0.0111 = 2.22 \times 10^{18} \text{ cm}^{-3}$$
</details>

<details>
<summary>Q7：為什麼先進製程（< 10 nm）不能只靠擴散來做摻雜？</summary>

**答案**：
1. **擴散需要高溫（900～1200°C）**：會導致先前已做好的淺接面擴散變深，破壞結構
2. **無法精確控制劑量**：擴散劑量靠時間和溫度間接控制，精度不夠
3. **橫向擴散嚴重**：先進製程的特徵尺寸只有幾奈米，高溫擴散的橫向展開不可接受
4. **無法形成突變接面（Abrupt Junction）**：現代元件需要極陡峭的摻雜分佈，擴散做不到
5. 因此現代製程 100% 使用離子佈植 + 毫秒/雷射退火
</details>
