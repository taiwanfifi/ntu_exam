### Poisson

```
他們在市場裡穿行。
腳下是溼的——魚攤的水流到地上。
貝葉一邊走一邊跟小拉討論。

「小拉——在過去十二個小時的 Random Walk 中——
 我們遇到了幾次意外？」

小拉算了一下：

  ┌─ 意外事件統計 ──────────────────────┐
  │                                       │
  │ 過去 12 小時：                       │
  │ 意外事件次數：7                      │
  │                                       │
  │  正向：3                             │
  │   - 便利商店充電（手機快沒電）       │
  │   - 路人提供捷徑資訊                 │
  │   - 林安（B3 消防梯情報）★★★       │
  │                                       │
  │  負向：4                             │
  │   - 巷口死路（折返 15 分鐘）        │
  │   - 攝影機偵測                       │
  │   - 路口施工封閉                     │
  │   - 便利商店店員多看了兩眼           │
  │                                       │
  │  平均：每 1.71 小時一次              │
  │  λ ≈ 0.58 事件/小時                 │
  │                                       │
  └───────────────────────────────────────┘

「λ 等於 0.58。
 意外事件在時間軸上的分布——
 如果事件之間是獨立的——
 且平均速率是常數——
 就是 Poisson Process。

 P(N(t) = k) = (λt)^k × e^{-λt} / k!

 在接下來的一個小時裡——
 遇到零次意外的機率：
 P(0) = e^{-0.58} ≈ 0.56

 至少一次意外的機率：
 1 - 0.56 = 0.44。

 接近一半。」

「下一個意外——什麼時候來？」

「Poisson Process 的下一個事件等待時間——
 服從指數分布。Exp(λ)。
 期望值 = 1/λ = 1/0.58 ≈ 1.72 小時。

 但——指數分布有一個特殊性質。」

  ┌─ 指數分布的無記憶性 ─────────────────┐
  │                                       │
  │ Memoryless Property：                 │
  │                                       │
  │ P(T > s+t | T > s) = P(T > t)       │
  │                                       │
  │ 意思：                               │
  │ 不管你已經等了多久——                 │
  │ 下一次的等待時間分布——               │
  │ 跟從頭等是一樣的。                   │
  │                                       │
  │ 你等了 10 分鐘沒遇到意外——          │
  │ 未來遇到意外的期望等待時間           │
  │ 還是 1.72 小時。                     │
  │ 不是 1.72 - (10/60) = 1.55 小時。  │
  │                                       │
  │ 過去不影響未來。                      │
  │ 這是「無記憶」的意思。               │
  │                                       │
  └───────────────────────────────────────┘

「無記憶。」

「是。
 過去不影響未來。
 跟馬可夫性質對應——
 現在的狀態決定一切。
 不是歷史。

 這——其實就是隨機的定義。
 如果你的未來取決於過去——
 那你是可預測的。
 如果你的未來——只取決於現在——
 你就是自由的。」

貝葉看著市場裡的人。
每一個人——都是一個隨機變數。
他們不知道下一秒會發生什麼。
但他們還是在走。在買菜。在殺價。在活著。

「隨機——不只是不可預測。」

小拉等著。

「隨機——也是一種自由。」
```
