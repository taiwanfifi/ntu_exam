> **學習目標**：理解載子的產生與復合機制，掌握連續性方程和少數載子擴散方程，為 PN 接面分析打下基礎。
> **預備知識**：第二章（載子濃度）、第三章（漂移與擴散電流）
> **預估時間**：3～4 小時

## 🔰 本章基礎觀念（零基礎必讀）

### 為什麼要學產生與復合？

前兩章我們學了：
- 半導體裡有多少載子（第二章）
- 載子怎麼移動（第三章）

但還有一個關鍵問題：**載子會被「創造」和「消滅」**。

- **產生（Generation）**：電子從價帶跳到導帶，同時產生一個電子和一個電洞
- **復合（Recombination）**：電子從導帶掉回價帶，消滅一個電子和一個電洞

這兩個過程決定了：
- LED 發光效率（復合是否輻射）
- 太陽能電池效率（光產生的載子能否被收集）
- 二極體的反向漏電流（產生電流）
- 電晶體的漏電和增益

### 一分鐘抓住核心

```
產生(G)：能量注入 → 產生電子-電洞對
  - 光照、熱能、碰撞游離

復合(R)：電子和電洞碰面 → 消失
  - 直接復合：Band-to-Band（LED發光靠這個）
  - SRH復合：透過缺陷（Si中最常見）
  - Auger復合：高摻雜時重要

平衡：G = R → np = ni²
非平衡：G ≠ R → np ≠ ni² → 連續性方程
```

## 一、載子的產生與復合

### 1.1 熱平衡下的動態平衡

在熱平衡中，產生和復合持續進行，但速率相等：

$$G_0 = R_0 \quad \Rightarrow \quad n_0 p_0 = n_i^2$$

其中 n₀, p₀ 是平衡載子濃度。

### 1.2 非平衡狀態

當外界注入能量（光照、正偏壓等），額外的載子被產生：

$$n = n_0 + \Delta n \quad \text{（Δn = 超額電子）}$$
$$p = p_0 + \Delta p \quad \text{（Δp = 超額電洞）}$$

由於電子和電洞總是成對產生：**Δn = Δp**（電荷中性近似）

此時 np > ni²，淨復合率 > 0，系統會透過復合來恢復平衡。

## 二、復合機制

### 2.1 直接復合（Band-to-Band Recombination）

電子直接從導帶掉到價帶，與電洞結合。

$$R_{direct} = B(np - n_i^2)$$

其中 B 是復合係數（Recombination Coefficient）。

| 材料 | B (cm³/s) | 說明 |
|------|----------|------|
| GaAs | ~10⁻¹⁰ | 直接帶隙，B 大 → 發光效率高 |
| Si   | ~10⁻¹⁴ | 間接帶隙，B 極小 → 幾乎不發光 |

**關鍵**：
- 直接帶隙材料中：能量以**光子**形式釋放 → **輻射復合** → LED、雷射
- 間接帶隙材料中：需要聲子協助，機率極低 → 大部分能量變成**熱**

### 2.2 SRH 復合（Shockley-Read-Hall Recombination）

**最重要的復合機制（在 Si 中）**

透過帶隙中的**缺陷能階（Trap Level, Et）**作為中繼站：

```
      CB ────────── Ec
          │
          ↓ 電子被缺陷捕獲
     ─── Et ─── 缺陷能階（深能階）
          ↓ 電洞被缺陷捕獲（或說電子掉下去）
      VB ────────── Ev
```

SRH 復合率：

$$R_{SRH} = \frac{np - n_i^2}{\tau_p(n + n_1) + \tau_n(p + p_1)}$$

其中：
- τn, τp = 電子和電洞的捕獲時間常數
- n₁ = ni·exp((Et-Ei)/kT)
- p₁ = ni·exp((Ei-Et)/kT)

**重要結論**：

1. 當缺陷能階在帶隙中央（Et ≈ Ei）時，SRH 復合率**最大**
   - 這就是為什麼「深能階雜質」（如 Au, Fe 在 Si 中）會大幅降低載子壽命
2. SRH 復合是非輻射的 → 能量變成熱（聲子）

### 2.3 簡化：少數載子生命期

在低注入條件下（Δn << 多數載子濃度），SRH 復合可以簡化：

**N 型半導體**（少數載子是電洞）：

$$R = \frac{\Delta p}{\tau_p}$$

**P 型半導體**（少數載子是電子）：

$$R = \frac{\Delta n}{\tau_n}$$

其中 τn, τp 是**少數載子生命期（Minority Carrier Lifetime）**：

| 材料 | 典型 τ 值 | 說明 |
|------|----------|------|
| Si（高品質） | 1 ms ~ 10 ms | 太陽能電池用 |
| Si（IC 製程） | 0.1 ~ 10 μs | 有更多缺陷 |
| GaAs | 1 ~ 10 ns | 直接復合主導，τ 較短 |

### 2.4 Auger 復合

在高載子濃度（高摻雜或強注入）時，Auger 復合變得重要：

```
電子-電子-電洞 (eeh) Auger 過程：
      CB ──── ← 電子獲得能量（被「踢」到更高能量）
          ↘
      CB ──── ← 電子從這裡復合
          ↓
      VB ──── ← 電洞消失
```

$$R_{Auger} = C_n n^2 p + C_p n p^2$$

- Cn, Cp：Auger 復合係數（~10⁻³¹ cm⁶/s for Si）
- 在 N 型重摻雜中：R_Auger ≈ Cn·Nd²·Δp → **∝ Nd²**
- 這限制了太陽能電池重摻雜區的效率

### 2.5 表面復合

半導體表面有大量懸掛鍵（Dangling Bonds），形成缺陷態：

$$R_{surface} = S \cdot \Delta n_s$$

其中 S 是**表面復合速度（Surface Recombination Velocity）**，單位：cm/s。

| 界面 | S (cm/s) | 說明 |
|------|----------|------|
| 裸露 Si 表面 | 10⁴ ~ 10⁶ | 很多缺陷 |
| Si/SiO₂（良好鈍化） | 1 ~ 100 | 製程優化後 |
| Si/Al₂O₃ | < 10 | 新型鈍化技術 |

**鈍化（Passivation）**：用氧化物或其他材料覆蓋表面，填滿懸掛鍵，降低表面復合。這在太陽能電池和 MOSFET 中都極其重要。

## 四、連續性方程（Continuity Equation）

### 4.1 基本概念

連續性方程就是載子的「質量守恆」：

> **載子濃度的變化率 = 流入的 - 流出的 + 產生的 - 復合的**

### 4.2 電子的連續性方程

$$\frac{\partial n}{\partial t} = \frac{1}{q}\frac{\partial J_n}{\partial x} + G_n - R_n$$

**各項的意義**：
- ∂n/∂t：電子濃度隨時間的變化
- (1/q)(∂Jn/∂x)：電子電流的空間變化帶來的淨流入
- Gn：電子的產生率
- Rn：電子的復合率

### 4.3 電洞的連續性方程

$$\frac{\partial p}{\partial t} = -\frac{1}{q}\frac{\partial J_p}{\partial x} + G_p - R_p$$

（注意電洞的電流項有負號，因為電洞帶正電。）

### 4.4 白話理解連續性方程

```
想像一個水槽：

水位變化 = 水龍頭進水 - 排水口出水 + 下雨 - 蒸發
  ∂n/∂t  =  電流淨流入    + 產生G - 復合R
```

## 五、少數載子擴散方程

### 5.1 假設條件

在許多實際問題中，可以做以下簡化：
1. 只關注**少數載子**（它決定了元件特性）
2. **低注入**（Δn << 多數載子濃度）→ 電場可忽略
3. 多數載子的電場和濃度變化由空間電荷中性維持

### 5.2 少數載子擴散方程

以 P 型半導體中的少數電子為例：

$$\frac{\partial(\Delta n)}{\partial t} = D_n \frac{\partial^2(\Delta n)}{\partial x^2} - \frac{\Delta n}{\tau_n} + G_n$$

穩態（∂/∂t = 0）且無光照（Gn = 0）：

$$D_n \frac{d^2(\Delta n)}{dx^2} = \frac{\Delta n}{\tau_n}$$

### 5.3 擴散長度

上述方程的解是指數衰減形式：

$$\Delta n(x) = \Delta n(0) \cdot \exp\left(-\frac{x}{L_n}\right)$$

其中**擴散長度**：

$$L_n = \sqrt{D_n \tau_n}$$

**物理意義**：少數載子在復合之前，平均能擴散的距離。

| 材料 | Dn (cm²/s) | τn | Ln |
|------|-----------|-----|-----|
| Si (IC) | 35 | 1 μs | √(35×10⁻⁶) = 59 μm |
| Si (太陽能) | 35 | 1 ms | √(35×10⁻³) = 1.87 mm |
| GaAs | 200 | 10 ns | √(200×10⁻⁸) = 14 μm |

**擴散長度的重要性**：
- PN 接面：少數載子必須在 Ln 內到達接面才能被收集
- 太陽能電池：Ln 必須大於吸收深度才有高效率
- BJT：基極寬度必須 << Ln 才有高電流增益

## 六、準費米能階（Quasi-Fermi Level）

### 6.1 為什麼需要準費米能階？

在熱平衡下，一個費米能階 Ef 就夠了：

$$n = n_i \exp\left(\frac{E_f - E_i}{kT}\right), \quad p = n_i \exp\left(\frac{E_i - E_f}{kT}\right)$$

但在非平衡下（如照光、外加偏壓），np ≠ ni²。我們定義兩個**準費米能階**：

$$n = n_i \exp\left(\frac{E_{Fn} - E_i}{kT}\right) \quad \text{（電子準費米能階 EFn）}$$

$$p = n_i \exp\left(\frac{E_i - E_{Fp}}{kT}\right) \quad \text{（電洞準費米能階 EFp）}$$

### 6.2 準費米能階的性質

- 平衡時：EFn = EFp = Ef（回到單一費米能階）
- 非平衡時：EFn ≠ EFp
- 電子注入（np > ni²）：EFn 上移，EFp 下移，EFn - EFp > 0

### 6.3 電流與準費米能階的關係

電流可以用準費米能階的梯度來表達：

$$J_n = n\mu_n \frac{dE_{Fn}}{dx}$$

$$J_p = p\mu_p \frac{dE_{Fp}}{dx}$$

**意義**：電流流動的方向就是準費米能階下降的方向。如果準費米能階是平的（梯度為零），則該載子沒有淨電流。

### 6.4 在 PN 接面中的應用預告

在正偏 PN 接面中：
- 接面兩側的 EFn - EFp = qV（外加電壓）
- 在空乏區外，少數載子的準費米能階幾乎是平的
- 這是 PN 接面 I-V 特性推導的基礎（下一章會詳細討論）

## 數值例題

### 【例題 1】超額載子與復合

**題目**：P 型 Si，Na = 10¹⁶ cm⁻³，τn = 5 μs。照光使 Δn = Δp = 10¹⁴ cm⁻³。求穩態下的復合率。

**解答**：

```
確認低注入：Δn = 10¹⁴ << Na = 10¹⁶ ✓

復合率（低注入近似）：
R = Δn/τn = 10¹⁴ / (5×10⁻⁶)
  = 2×10¹⁹ cm⁻³·s⁻¹

穩態時：R = G（產生率等於復合率）
所以光照的產生率 G = 2×10¹⁹ cm⁻³·s⁻¹
```

---

### 【例題 2】擴散長度計算

**題目**：N 型 Si，Nd = 10¹⁷ cm⁻³，μp = 400 cm²/V·s，τp = 1 μs。求少數載子（電洞）的擴散長度。

**解答**：

```
步驟 1：由 Einstein 關係求 Dp
Dp = μp × VT = 400 × 0.026 = 10.4 cm²/s

步驟 2：計算擴散長度
Lp = √(Dp × τp) = √(10.4 × 10⁻⁶)
   = √(1.04 × 10⁻⁵)
   = 3.22 × 10⁻³ cm
   = 32.2 μm

答案：Lp ≈ 32 μm
```

**意義**：在 PN 接面中，注入到 N 區的少數電洞平均能走 32 μm 就會被復合。

---

### 【例題 3】少數載子擴散方程的求解

**題目**：P 型 Si，在 x = 0 處注入超額電子 Δn(0) = 10¹⁵ cm⁻³。已知 Ln = 50 μm。求 x = 100 μm 處的 Δn。假設穩態、無光照。

**解答**：

```
穩態、無光照、半無窮邊界條件下：

Δn(x) = Δn(0) × exp(-x/Ln)

Δn(100μm) = 10¹⁵ × exp(-100/50)
           = 10¹⁵ × exp(-2)
           = 10¹⁵ × 0.1353
           = 1.353 × 10¹⁴ cm⁻³
```

每經過一個擴散長度 Ln，超額載子衰減到 1/e ≈ 37%。

---

### 【例題 4】Auger 復合的影響

**題目**：重摻雜 N 型 Si，Nd = 10¹⁹ cm⁻³，Auger 係數 Cn = 2.8×10⁻³¹ cm⁶/s。求 Auger 限制的少數載子生命期。

**解答**：

```
在 N 型重摻雜中，eeh Auger 過程主導：
R_Auger ≈ Cn × n² × Δp = Cn × Nd² × Δp

等效 Auger 生命期：
τ_Auger = 1/(Cn × Nd²)
        = 1/(2.8×10⁻³¹ × (10¹⁹)²)
        = 1/(2.8×10⁻³¹ × 10³⁸)
        = 1/(2.8×10⁷)
        = 3.57×10⁻⁸ s
        ≈ 36 ns
```

**重要意義**：在 Nd = 10¹⁹ 的重摻雜區，Auger 復合把少數載子壽命限制在 ~36 ns，遠短於一般 SRH 的 μs 等級。這就是為什麼太陽能電池的射極（重摻雜）復合損失嚴重。

---

### 【例題 5】準費米能階分裂

**題目**：P 型 Si，Na = 10¹⁶ cm⁻³。照光使穩態下 Δn = 10¹⁴ cm⁻³。求 EFn - EFp。

**解答**：

```
n = n₀ + Δn ≈ ni²/Na + Δn = (1.5×10¹⁰)²/10¹⁶ + 10¹⁴
  = 2.25×10⁴ + 10¹⁴ ≈ 10¹⁴ cm⁻³

p = p₀ + Δp ≈ Na + Δp = 10¹⁶ + 10¹⁴ ≈ 10¹⁶ cm⁻³

np = 10¹⁴ × 10¹⁶ = 10³⁰ cm⁻⁶ >> ni² = 2.25×10²⁰

EFn - EFp = kT × ln(np/ni²)
           = 0.026 × ln(10³⁰/2.25×10²⁰)
           = 0.026 × ln(4.44×10⁹)
           = 0.026 × 22.21
           = 0.577 eV
```

**意義**：EFn - EFp ≈ 0.58 eV。在太陽能電池中，這個分裂的大小決定了開路電壓的上限（Voc ≤ (EFn-EFp)/q）。

## ✅ 自我檢測

### 基礎題

**Q1**：在 Si 中，哪種復合機制最常見？為什麼？

<details>
<summary>點擊查看答案</summary>

**SRH 復合（Shockley-Read-Hall）**最常見。

原因：Si 是間接帶隙材料，直接復合（band-to-band）需要聲子協助，機率極低。而 Si 晶體中不可避免地有些缺陷（深能階），SRH 復合透過這些缺陷進行，效率高得多。

只有在非常高的摻雜濃度（> 10¹⁸~10¹⁹ cm⁻³）時，Auger 復合才會超過 SRH。

</details>

**Q2**：少數載子生命期 τ 和擴散長度 L 的關係是什麼？

<details>
<summary>點擊查看答案</summary>

$$L = \sqrt{D \cdot \tau}$$

其中 D 是少數載子的擴散係數。

例如 Si 中少數電子：Dn = 35 cm²/s，τn = 1 μs → Ln = √(35×10⁻⁶) ≈ 59 μm。

物理意義：τ 是載子的「壽命」，D 是擴散的「快慢」，L 是載子在壽命內能走的「距離」。

</details>

**Q3**：什麼條件下 Auger 復合比 SRH 復合重要？

<details>
<summary>點擊查看答案</summary>

**高載子濃度**時 Auger 復合重要：

- 重摻雜（Nd 或 Na > 10¹⁸~10¹⁹ cm⁻³）
- 強注入（Δn 很大）

因為 Auger 復合率 ∝ n²p（或 np²），而 SRH ∝ Δn，所以高濃度時 Auger 增長更快。

典型場景：太陽能電池的重摻雜射極、功率元件的大電流區域。

</details>

### 進階題

**Q4**：為什麼缺陷能階在帶隙中央時，SRH 復合率最大？

<details>
<summary>點擊查看答案</summary>

SRH 復合需要兩步：
1. 電子從導帶被缺陷捕獲
2. 電洞從價帶被缺陷捕獲（等效於電子從缺陷掉到價帶）

如果 Et 靠近 Ec：步驟 1 容易（電子多），但步驟 2 困難（離 Ev 遠，電洞少）
如果 Et 靠近 Ev：步驟 2 容易，但步驟 1 困難

只有當 Et ≈ Ei（帶隙中央）時，兩步的速率相當，總復合率最大。

這就像接力賽：兩棒速度都快才能最快完成。

</details>

**Q5**：LED 中，為什麼需要直接帶隙材料？SRH 復合對 LED 有什麼影響？

<details>
<summary>點擊查看答案</summary>

LED 要求：注入的電子-電洞對在復合時釋放光子（輻射復合）。

- **直接帶隙**：輻射復合（band-to-band）佔主導 → 發光效率高
- **間接帶隙**（如 Si）：輻射復合極弱，SRH 復合（非輻射）主導 → 幾乎不發光

SRH 復合對 LED 的影響：即使在直接帶隙材料中，如果有太多缺陷（高 SRH 復合率），電子-電洞對會透過缺陷非輻射復合而不是發光 → 降低內部量子效率。

這就是為什麼 LED 的晶體品質要求極高。

</details>

**Q6**：在穩態、無光照下，P 型 Si 中少數電子滿足什麼方程？邊界條件通常是什麼？

<details>
<summary>點擊查看答案</summary>

方程：

$$D_n \frac{d^2(\Delta n)}{dx^2} = \frac{\Delta n}{\tau_n}$$

即：

$$\frac{d^2(\Delta n)}{dx^2} = \frac{\Delta n}{L_n^2}$$

通解：Δn(x) = A·exp(-x/Ln) + B·exp(+x/Ln)

常見邊界條件：
1. x = 0：Δn(0) = 已知（由偏壓決定）
2. x → ∞：Δn → 0（載子不能無限增長）→ B = 0

所以：Δn(x) = Δn(0)·exp(-x/Ln)

</details>

## 本章重點整理

```
1. 平衡：G₀ = R₀, np = ni²
2. 非平衡：G ≠ R, Δn = Δp（低注入）
3. 復合機制排名（Si中）：SRH > Auger（高摻雜）>> 直接復合
4. 低注入復合：R = Δn/τ
5. 直接復合：R = B(np - ni²)，直接帶隙材料重要
6. SRH：Et ≈ Ei 時復合最快
7. Auger：R ∝ n²p + np²，高濃度主導
8. 表面復合：R = S·Δn，鈍化很重要
9. 連續性方程：∂n/∂t = (1/q)∂Jn/∂x + G - R
10. 擴散長度：L = √(Dτ)，載子復合前的平均擴散距離
11. 準費米能階：非平衡下 EFn ≠ EFp
```
