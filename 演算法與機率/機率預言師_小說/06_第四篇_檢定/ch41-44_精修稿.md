# 第四篇「檢定」中段（ch41-44）—— 初稿級精修

> **精修目的**：Type II Error 反擊。數據消耗戰。馬可瑜的保險數據。卡方適合度檢定。
> **原則**：法庭攻防進入白熱化。貝葉和費雪是兩個鏡像——都用正確的統計。但選擇了不同的故事。讀者要感覺到「兩邊都有道理」但「必須選邊」的撕裂感。馬可瑜在第 43 章成為關鍵——不是主角的主角。
> **承接**：第 40 章結束在貝葉的手指指向 Type II Error 的位置。全場等她的下一句話。

## 第 41 章：百分之二十三的沉默

### 反擊

```
【環境】
接續第 40 章。同一個下午。第七法庭。
貝葉站在白板前。
費雪坐在被告席。
法官的筆懸在筆記本上方。
全場等著。

貝葉的手指從 Type II Error 的位置移開。
她拿起白板筆。

在費雪的表格旁邊——留了一塊空白的地方——
她寫下了一個新的符號。

  β

「β。」
她的聲音穩定。第七十三遍練出來的穩定。

「β 是 Type II Error 的機率。
 也就是——
 在操控確實存在的情況下——
 我們的檢定——偵測不到。
 我們放了他。
 我們跟兩千三百萬人說——一切正常。
 但一切不正常。」

她在白板上寫：

  β = P(不拒絕 H₀ | H₁ 為真)
    = P(放過操控 | 操控確實存在)

「費雪女士剛才問——
 法庭能接受千分之一的冤枉率嗎？

 我也要問——
 法庭能接受多大的放過率？」

她轉向白板。開始算。
手在穩。
沒有小拉。沒有計算機。
只有她的腦袋和七十三遍的練習。

「要計算 β——
 我需要三個東西。

 第一：α。我選了 0.001。
 第二：樣本量。我有 n = 347 個區域日。
 第三：效果量。根據我們觀察到的偏移——
 d = 0.42。」

她在白板上寫下計算過程——
不是完整的公式。是簡化的。法庭版的。

  ┌─ 檢定力計算 ─────────────────────────────┐
  │                                           │
  │ α = 0.001                                │
  │ n = 347                                   │
  │ d = 0.42                                  │
  │                                           │
  │ 檢定力 Power = 1 - β                     │
  │                                           │
  │ Power ≈ 0.77                              │
  │                                           │
  │ ∴ β = 1 - 0.77 = 0.23                   │
  │                                           │
  │ 翻譯：                                    │
  │ 如果操控確實存在——                        │
  │ 我們有 23% 的機率偵測不到。              │
  │                                           │
  └───────────────────────────────────────────┘

「β = 0.23。」

她放下白板筆。
轉向法官。

「讓我翻譯。

 費雪女士擔心的——冤枉好人——千分之一。
 我擔心的——放過壞人——23%。

 百分之二十三。」

停頓。
她讓這個數字在空氣裡停了三秒。
三秒很長。在法庭裡像三分鐘。

「法官——
 讓我用一個每個人都能理解的例子。」

她在白板上畫了一個方塊。

「想像您家的煙霧偵測器。
 如果有人告訴你——
 你的煙霧偵測器——
 有 23% 的機率會漏報火災。

 也就是——
 每四次火災裡——
 有一次——偵測器不會響。

 你會安心地睡覺嗎？」

法庭裡——
有人吸了一口氣。
不知道是誰。
但很清楚。

「這就是我們面對的選擇。

 不是在『有風險』和『沒有風險』之間選。
 是在兩種風險之間選。

 冤枉好人——千分之一。
 放過壞人——23%。

 各位。

 您選哪一個？」

法庭安靜了。
比費雪陳述後的安靜更深。
更重。
像水面下的安靜——
水面上看起來平靜。
但水面下——每個人的腦袋都在翻滾。

### 法庭——費雪的消耗戰

```
回到法庭。

費雪繼續。
她的攻擊——不是一記重拳。
是連續的刺拳。每一拳都不重。
但每一拳都在消耗貝葉的血量。

「原告。您的數據清單中——
 第 14 到第 31 項已經不存在。
 18 筆。
 您原本的 n = 347。
 現在——n = 329。」

「329。」貝葉說。

「329。少了 18 筆。5.2%。
 您知道——樣本量的減少——
 對信賴區間的影響嗎？」

貝葉走到白板前。
心裡在算。沒有計算機。沒有小拉。
她深吸一口氣。
開始寫。

  信賴區間寬度 ∝ 1/√n

  n = 347 → 1/√347 ≈ 0.0537
  n = 329 → 1/√329 ≈ 0.0551

  寬度增加：(0.0551 - 0.0537)/0.0537 ≈ 2.6%

「影響不大。」貝葉說。
「信賴區間只寬了 2.6%。
 結論不受影響。」

費雪微笑了。
那個微笑——意味著她還有下一手。

「2.6%。好的。不大。

 但——
 如果我告訴你——
 我即將對你的其他數據來源提出質疑呢？

 清單中第 1 到第 13 項——
 是原告在 Oracular 系統中使用『非授權查詢工具』取得的。
 我會質疑這些數據的合法性。

 第 32 到第 47 項——
 是透過匿名來源取得的二手數據。
 匿名來源的可靠性——在法庭上從來就是問題。」

她看著法官。

「法官——
 如果原告最終可驗證的數據只剩 200 筆呢？
 150 筆？100 筆？」

她轉向貝葉。

「您的信賴區間會有多寬？
 您的檢定力會剩多少？
 您的結論——還站得住腳嗎？」

┌─ 信賴區間與樣本量 ──────────────────────┐
│                                           │
│ 信賴區間寬度 ∝ 1/√n                     │
│                                           │
│ n = 347 → 寬度 ∝ 0.0537                 │
│ n = 200 → 寬度 ∝ 0.0707  （寬了 32%）   │
│ n = 100 → 寬度 ∝ 0.1000  （寬了 86%）   │
│ n =  50 → 寬度 ∝ 0.1414  （寬了 163%）  │
│                                           │
│ n 越小 → 區間越寬 → 結論越不確定        │
│                                           │
│ 費雪的策略：                              │
│ 不攻擊貝葉的數學。攻擊貝葉的數據。      │
│ 刪除數據 = 擴大不確定性 = 削弱結論。     │
│                                           │
└───────────────────────────────────────────┘

這是一種消耗戰。
費雪不需要贏。
她只需要讓貝葉的 n 越來越小。
小到——結論站不住腳。

貝葉站在白板前。
她的手——
有一點點在抖。
不多。只有一點點。
但她感覺到了。

因為——費雪說得沒有錯。
如果 n 掉到 100 以下——
她的 power 會從 0.77 跌到 0.40 以下。
β 會超過 60%。
她的整個論證——會像沙堡一樣塌掉。

法官看了看時鐘。四點二十分。
「今天的審理到此結束。
 明天上午九點繼續。
 休庭。」
