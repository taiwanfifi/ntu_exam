```
  通風口的風聲。米亞光球嗡鳴。遠處車流低頻。
```

```

  「邱韻如是我的母親。」
  （句號。不是驚嘆號。她不是在宣告。她是在釋放。）

   我知道織網者是誰。

   我知道周明哲的死是偽造的。」

  像 Backtracking。一層一層回到起點。

  「你們追了半年。
   我看著你們追了半年。」

  不超過八句。不解釋動機。不道歉。不求諒解。
  只攤開事實。
```

```

  語青 = 釋放。
    說完後肩膀降了下來。
    不是輕鬆。是空了。
```

### ch138 最短路徑——Dijkstra 核心教學

**核心意象**：維倫像幽靈穿越數位防線。米亞冷靜導航。每一步選「風險最小的下一個節點」。

**A 線指令（Dijkstra 核心教學——融入動作）**：

```
米亞的導航 = 即時教學：

  「Dijkstra 的規則：
   從起點開始。
   看所有相鄰節點。
   選累計風險值最小的那個。
   到了新節點再看。
   永遠選累計最小。
   重複。」

逐層潛入（米亞即時播報）：

  L1：「左邊走廊。感應器上方 2.3 公尺。蹲下走。
   累計風險：3%。」

  L2：「動態密碼。11.7 秒破解。12 秒更新。
   剩 0.3 秒。走。
   累計：8%。」

  L3：「凌晨 3:04 換班。走廊空。
   累計：14%。」

  L4：繞過。

  L5 入口：「深潛陷阱。不主動掃描。
   不要回憶——記憶特徵會被當入侵信號。」

  維倫閉上眼。清空腦袋。走進去。

關鍵教學——不是貪心：

  維倫：「用人話。」

  米亞：「現在看起來貴的路——後面便宜。
   現在看起來便宜的路——後面讓你傾家蕩產。
   Dijkstra 看的不是這一步。是所有步加起來。」
```

**B 線指令（米亞冷靜 vs 維倫冒險）**：

```
  米亞在外部導航。精準、穩定。
  「心率 112。可接受。
   超過 130 體溫升高——熱能偵測器發現你。
   所以——不要緊張。」

  維倫：「你叫闖入最高安全區的人不要緊張。」
  米亞：「我可以講笑話降低心率——」
  維倫：「不用。」
  米亞：「很好。我不太擅長笑話。」

  ※ 「不要回憶」= 天才設定——修復記憶的人被要求「不使用記憶」。
     最安全的方法有時要求放棄你最擅長的東西。
```

**AB 線同步點**：米亞「你走得比我算的還準」= 維倫本人就是 Dijkstra 化身。最優路徑 = 最小傷害修復哲學。

**寫作指令**：
- 教學融入潛入過程。邊走邊教。不要停下來講
- 累計風險數字做節拍器：3% → 8% → 14%

### ch139 第五層的意外——即時重新計算 + 懸念頂峰

**核心意象**：第五層「最高機密」檔案。十年前十一月二日全程錄像。即時路徑調整。

**A 線指令（Dijkstra 即時調整）**：

```
意外觸發：

  維倫碰到未標記節點。
  一段錄像。
  標籤：[最高機密][2079.11.02][現場全程紀錄]

  米亞：「風險值從 14% 跳到 61%。
   偵測系統啟動倒計時。你有 90 秒。」

兩個選擇：

  A. 放棄錄像。撤離。風險可控
  B. 下載（40 秒）。剩 50 秒走新路線

即時重算：

  米亞：「原路返回——累計 87%。不建議。
   新路線——東翼維修通道切入 L3。
   累計......38%。但 47 秒內跑 200 公尺。」

  維倫按下下載鍵。

  ※ 關鍵教學：
     Dijkstra 不是一次性的。
     當邊權重改變時（警報 = 權重暴增），
     需要從當前位置重新跑一次。
     = 「即時路徑調整」。
```

**B 線指令（懸念頂峰 + 逃離）**：

```
逃離場景——跑步節奏：

  「29 秒。前方防火門。開門 3 秒。你不能停。」

  「22 秒。L3。樓梯右邊。」

  「11 秒——」
  「維倫。停。3 秒後門口有巡邏。等他們過去。」
```

**AB 線同步點**：維倫選擇下載錄像而非撤離 = 「最安全的路徑不一定通向最重要的真相」= Dijkstra 的哲學意義：不是永遠選最安全的路，是知道什麼時候值得選一條危險的路。

- ≤2,800 字

## 柒、寫作陷阱與檢查清單（10 條）
