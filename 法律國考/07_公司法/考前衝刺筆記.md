## 一、公司類型比較表

> 國考一試選擇題愛考「哪種公司有哪種特徵」，這張表務必記熟。

| 比較項目 | 無限公司 | 有限公司 | 兩合公司 | 股份有限公司 |
|---|---|---|---|---|
| **條文依據** | §40 以下 | §98 以下 | §114 以下 | §128 以下 |
| **股東人數** | 2 人以上 | 1 人以上 | 2 人以上（至少各1） | 2 人以上（政府/法人 1 人即可） |
| **責任型態** | 全體股東負無限連帶責任 | 各股東就出資額負責 | 無限＋有限責任股東並存 | 各股東就所認股份負責 |
| **出資種類** | 現金/其他財產/勞務/信用 | 現金/其他財產（不含勞務信用） | 無限責任股東同無限公司；有限責任股東同有限公司 | 現金/其他財產/技術（§156 V） |
| **是否有股份** | 無（出資額） | 無（出資額） | 無 | 有，劃分為股份 |
| **轉讓出資/股份** | 須全體同意（§55） | 須全體同意（§111 I）但可章程另訂 | 無限責任股東須全體同意；有限責任股東依§115準用 | 原則自由轉讓（§163） |
| **機關設計** | 全體股東均執行業務（§45）或選任 | 至少1董事（§108） | 無限責任股東執行業務 | 股東會＋董事會＋監察人/審計委員會 |
| **實務重要度** | 極少見 | 中小企業常見 | 幾乎消失 | **考試絕對核心** |

### 記憶口訣

- **無限公司**：「有難大家扛」——全體連帶無限責任。
- **有限公司**：「各管各的額度」——就出資額負責。
- **兩合公司**：「混搭組合」——同時有無限＋有限責任股東。
- **股份有限公司**：「買股票負責到買的那些」——就認購股份金額負責。

### 高頻考點提醒

1. **有限公司出資轉讓**：原則須全體股東同意（§111 I），但 111 年修法後**得以章程訂定較低之同意門檻**。
2. **一人公司**：有限公司可以 1 人設立（§98 I）；股份有限公司原則需 2 人以上，但政府或法人股東 1 人即可（§128-1）。
3. **兩合公司**的有限責任股東不得執行業務（§122），若執行則視為無限責任股東。

## 二、股份有限公司機關速查

> 股份有限公司的三大機關（股東會、董事會、監察人）是考試出題最密集的區域。

### 2-1 三大機關職權分配表

| 機關 | 角色定位 | 核心職權 | 關鍵條文 |
|---|---|---|---|
| **股東會** | 最高意思機關（萬年考點） | 選任/解任董監事、承認財報、盈餘分配、章程變更、合併分割解散、重大處分 | §170-§191 |
| **董事會** | 業務執行機關 | 業務執行決定、選任董事長、召集股東會、編造財報、發行新股 | §192-§218-2 |
| **監察人** | 監督機關 | 監督業務執行、查核財報、代表公司對董事訴訟、少數股東請求時為公司提起訴訟 | §218-§227 |

### 2-2 董事會 vs 股東會的職權界線

這是國考二試常考的爭議問題。重點：

| 權限歸屬爭議 | 結論 | 說明 |
|---|---|---|
| 具體業務執行 | 董事會 | 經營判斷屬董事會權限（經營判斷法則） |
| 重大資產處分 | 股東會特別決議 | §185 I：讓與全部或主要部分營業或財產 |
| 董事報酬 | 章程或股東會決議 | §196 I |
| 董事競業許可 | 股東會（§209） | 不是董事會自己同意 |
| 盈餘分配 | 股東會承認 | §184 I、§228 |

### 2-3 監察人 vs 審計委員會

| 比較項目 | 監察人 | 審計委員會 |
|---|---|---|
| **法源** | §216-§227 | §14-4（證交法） |
| **組成** | 自然人，至少 1 人（公開發行 2 人） | 全部由獨立董事組成（至少 3 人） |
| **獨立行使** | 各得單獨行使（§221） | 委員會合議制 |
| **設置義務** | 非公開發行公司 | 上市櫃公司應設置 |
| **與董事會關係** | 外部監督 | 內部監督（獨董兼具董事身分） |

### 考試陷阱

- 監察人**各得單獨行使**職權（§221），不需要多數決。
- 審計委員會是**合議制**，決議門檻不同。
- 設有審計委員會者，**不再設監察人**。

## 三、股東會決議速查

### 3-1 普通決議 vs 特別決議

| 決議類型 | 法定出席門檻 | 法定表決門檻 | 條文 |
|---|---|---|---|
| **普通決議** | 已發行股份總數過半數出席 | 出席股權過半數同意 | §174 |
| **特別決議** | 已發行股份總數 2/3 以上出席 | 出席股權過半數同意 | §185 |
| **假決議**（普通決議出席不足時） | 1/3 以上出席 | 出席股權過半數同意 | §175 |

> 注意：§175 假決議須於 1 個月內再開確認會議，出席不足法定數額時，以假決議視為正式決議。

### 3-2 哪些事項需特別決議？（§185）

這是高頻考點，務必背熟：

| 事項 | 條文 |
|---|---|
| 1. 章程變更 | §277 |
| 2. 締結、變更或終止關於出租全部營業之契約 | §185 I ① |
| 3. 讓與全部或主要部分之營業或財產 | §185 I ② |
| 4. 受讓他人全部營業或財產 | §185 I ③ |
| 5. 解散 | §316 I |
| 6. 合併 | §316 I |
| 7. 分割 | §316-2 準用 |
| 8. 減資 | §168 I（§277 準用） |
| 9. 發行特別股（變更章程） | §157（§277 準用） |
| 10. 私募有價證券 | §248 I |

### 3-3 股東會決議瑕疵三層次

| 瑕疵類型 | 法律效果 | 救濟方式 | 除斥期間 | 條文 |
|---|---|---|---|---|
| **程序瑕疵**（召集程序/決議方法違法） | 得撤銷 | 訴請法院撤銷 | 決議日起 30 日內 | §189 |
| **內容違法** | 當然無效 | 確認無效之訴 | 無期間限制 | §191 |
| **決議不成立**（判例法理） | 不成立 | 確認不成立之訴 | 無期間限制 | 判例法理 |

### 經典考試辨析

- 未通知某股東 → **程序瑕疵** → 得撤銷（§189）
- 決議內容違反強行規定（如違法分配盈餘）→ **內容違法** → 無效（§191）
- 根本沒開會就做成決議 → **決議不成立**

## 四、董事義務與責任全覽

> 這是國考二試的「年年必考」區域，配分極高。

### 4-1 義務體系總表

| 義務類型 | 條文 | 核心要件 | 法律效果 |
|---|---|---|---|
| **忠實義務** | §23 I | 董事應忠實執行業務，不得以權謀私 | 對公司負損害賠償（§23 I） |
| **善良管理人注意義務** | §23 I | 如同處理自己事務以上的注意程度 | 對公司負損害賠償 |
| **競業禁止** | §209 | 為自己或他人為屬於公司營業範圍內之行為 | 須經股東會許可；違反者公司得行使歸入權（§209 V） |
| **自我交易限制** | §223 | 董事為自己或他人與公司為買賣等法律行為 | 由監察人代表公司（§223） |
| **利益迴避** | §206 III→§178 | 董事對會議事項有自身利害關係 | 不得加入表決，不算入已出席表決權數 |

### 4-2 競業禁止（§209）深度解析

```
要件拆解：
1. 主體：董事（含法人董事之代表人）
2. 行為：為自己或他人為屬於公司營業範圍內之行為
3. 未經許可：未經股東會普通決議許可
   → 注意：許可機關是「股東會」，不是董事會！

法律效果：
- 公司得主張歸入權（§209 V）
- 歸入：將董事因競業行為所得之利益，歸入公司
- 除斥期間：知悉後 1 年內，或行為後滿 1 年
```

### 4-3 自我交易（§223）深度解析

```
要件拆解：
1. 董事為自己或他人與公司進行交易
2. 代表公司的不能是該董事
3. 應由監察人代表公司

注意事項：
- §223 僅處理「代表權」問題，不處理交易實質公平性
- 若違反 §223，依通說該交易為無權代理
- 設有審計委員會者，由審計委員會指定之委員代表
```

### 4-4 §23 II — 對第三人責任

| 要件 | 說明 |
|---|---|
| 主體 | 公司負責人（董事、經理人等§8所列） |
| 行為 | 執行業務 |
| 主觀 | 違反法令 |
| 結果 | 致他人受損害 |
| 效果 | **負責人與公司連帶賠償** |

> 「執行業務」的解釋：實務採廣義說，包括與業務有關之一切行為。

### 4-5 經營判斷法則（Business Judgment Rule）

```
核心概念：
- 董事在經營判斷時若已善盡注意義務，即使結果不利，法院原則上不介入審查。
- 非法定明文，但實務與學說廣泛援引。

適用要件（學說整理）：
1. 經營判斷事項（非個人利益事項）
2. 善意（good faith）
3. 無利益衝突
4. 合理資訊蒐集
5. 合理相信符合公司最佳利益

考試提醒：
- 題目若問「董事做了某個商業決策導致公司虧損，是否應負賠償責任？」
  → 答題框架：先論忠實義務與善管注意 → 再論經營判斷法則的適用
```

## 五、少數股東權速查表

> 一試選擇題的送分題區域，但門檻數字一定要記對。

### 5-1 按持股門檻分類

| 持股門檻 | 權利內容 | 條文 |
|---|---|---|
| **不需門檻（單一股東即可）** | 股東會出席/表決權、提案權（§172-1，持股1年以上）、股東會撤銷決議之訴（§189） | §174、§189 |
| **繼續1年以上持有3%** | 請求監察人對董事提起訴訟 | §214 I |
| **繼續1年以上持有3%** | 監察人不提起 → 股東得自行為公司提起代位訴訟 | §214 II |
| **繼續6個月以上持有1%** | 書面請求監察人為公司對董事提起訴訟（公開發行公司） | §214 之適用 |
| **繼續1年以上持有3%** | 法院許可檢查公司業務帳目及財產狀況 | §245 I |
| **繼續3個月以上持有10%** | 請求董事會召集股東臨時會 | §173 I |
| **繼續3個月以上持有10%** | 董事會不召集 → 報主管機關許可自行召集 | §173 II |
| **繼續1年以上持有3%** | 請求法院選派檢查人 | §245 I |

### 5-2 重點比較

| 權利 | 持股比例 | 持有期間 | 對象 |
|---|---|---|---|
| 代位訴訟 | 3% | 繼續 1 年 | 先請求監察人 → 不為時自行提起 |
| 聲請檢查人 | 3% | 繼續 1 年 | 向法院聲請 |
| 臨時會召集請求 | 10% | 繼續 3 個月 | 先請求董事會 → 不為時報主管機關 |

### 考試陷阱

- **持股比例**與**持有期間**都要同時滿足，缺一不可。
- 代位訴訟（§214）必須**先請求監察人**，監察人 30 日內不提起才能自行提起。
- §173 臨時會召集，股東**不能直接自行召集**，必須先請求董事會，董事會不為時再報主管機關許可。

## 六、股份相關制度

### 6-1 股份轉讓自由原則（§163）

```
原則：股份之轉讓，不得以章程禁止或限制之。
例外（法定限制）：
1. 發起人股份：§163 但書，公司設立登記後 1 年內不得轉讓
   → 但公開發行公司改為「設立登記後不得轉讓」已廢除此限制
2. 員工庫藏股：§167-3 轉讓對象限制
3. 閉鎖性公司：§356-5 得以章程限制
```

### 6-2 庫藏股（§167-1）

| 項目 | 內容 |
|---|---|
| **法源** | §167-1 |
| **事由**（限3款） | ① 轉讓股份予員工 ② 配合附認股權之公司債等 ③ 維護公司信用及股東權益 |
| **決議機關** | 董事會（2/3 以上董事出席、過半數同意） |
| **買回上限** | 已發行股份總數 10% |
| **期限** | 3 年內轉讓予員工，届期未轉讓視為公司未發行股份，辦理變更登記 |
| **限制** | 庫藏股**無表決權**、**不得質押**、不得享有股東權利 |

### 6-3 特別股（§157）

| 項目 | 內容 |
|---|---|
| **法源** | §157 |
| **設置方式** | 章程訂明 |
| **特別股類型** | 優先分配股息紅利、優先分配剩餘財產、複數表決權、對特定事項否決權、保障當選一定名額董監事、限制被選舉權 |
| **表決權限制** | 得限制或無表決權（§157 I ③④） |
| **修改特別股權利** | 須特別股股東會同意（§159） |

### 6-4 記憶重點

- 庫藏股是**公司買回自己的股票**，不是「庫存的股票」。
- 庫藏股三大限制：**無表決權、不得質押、限期處分**。
- 特別股的權利內容**完全看章程怎麼訂**，但不得違反法律強行規定。
- §163 股份自由轉讓是**原則**，考試重點在**例外**。

## 七、公司變動速查

### 7-1 增資

| 類型 | 程序 | 關鍵規定 |
|---|---|---|
| **發行新股增資** | 董事會決議（§266 I） | 原股東有新股認購優先權（§267），但公開發行公司得保留 10%-15% 對外公開發行 |
| **盈餘轉增資** | 股東會特別決議（§240 I→§241準用§277） | 法定盈餘公積達資本總額 50% 始得發行新股 |
| **公積轉增資** | 股東會特別決議 | 資本公積得依 §241 撥充資本 |

### 7-2 減資

| 類型 | 程序 | 關鍵規定 |
|---|---|---|
| **實質減資**（退還股款） | 股東會特別決議（§168→§277） | 須通知債權人（§73 準用）、債權人得異議 |
| **形式減資**（彌補虧損） | 股東會特別決議 | 不退還股款，僅帳面調整 |

### 7-3 合併

| 項目 | 內容 |
|---|---|
| **決議** | 股東會特別決議（§316 I） |
| **合併契約** | 應載明合併條件、存續或新設公司章程等（§317-1） |
| **股東保護** | 反對股東得請求公司按當時公平價格收買其股份（§317） |
| **債權人保護** | 通知及公告債權人（§73 準用）；債權人得異議 |
| **效力** | 消滅公司權利義務由存續或新設公司概括承受 |

### 7-4 分割

| 項目 | 內容 |
|---|---|
| **法源** | §316-2 |
| **決議** | 準用合併之規定 → 股東會特別決議 |
| **反對股東** | 得請求收買（準用§317） |
| **效力** | 受讓公司承受分割部分之權利義務 |

### 7-5 解散與清算

| 階段 | 程序 | 關鍵規定 |
|---|---|---|
| **解散事由** | 章程所定事由、股東會特別決議、合併/分割/破產、命令解散、裁定解散 | §315（任意解散）、§11（命令解散） |
| **清算人** | 原則由董事充任（§322 I），章程另訂或股東會另選除外 | §322 |
| **清算程序** | 了結現務→收取債權→清償債務→分配剩餘財產 | §326 |
| **債權人保護** | 清算人應於就任後通知債權人申報債權（§327） | 申報期間不得少於 2 個月 |
| **剩餘財產分配** | 債務清償後始得分配（§330） | 違反者清算人對債權人連帶賠償 |

## 八、揭穿公司面紗（§154 II）

> 這是近年最熱門的考點之一，國考二試幾乎隔年就考。

### 8-1 制度基礎

公司為法人，具有獨立人格，股東僅就出資額（或認股金額）負責。但若股東**濫用公司獨立人格**，法律允許「揭穿面紗」，直接追究股東個人責任。

### 8-2 §154 II 要件

```
法條：股東濫用公司之法人地位，致公司負擔特定債務且清償顯有困難，
      其情節重大而有必要者，該股東應負清償之責。

要件拆解：
1. 股東（包括控制股東）
2. 濫用公司法人地位
   → 例如：公司資本顯著不足、公司與股東財產混同、
          公司為股東個人的工具（alter ego）
3. 致公司負擔特定債務
4. 清償顯有困難
5. 情節重大而有必要
```

### 8-3 實務常見適用情形

| 類型 | 具體情形 | 說明 |
|---|---|---|
| **資本顯著不足** | 設立資本與營業規模顯不相當 | 如：資本額 10 萬卻經營數億元業務 |
| **財產混同** | 股東個人財產與公司財產不分 | 股東把公司帳戶當私人帳戶用 |
| **人格混同** | 數公司間無實質區分 | A、B 公司同一辦公室、同一員工、同一負責人 |
| **脫產工具** | 設立公司專門逃避個人債務 | 將個人財產移轉給自己設立的公司 |

### 8-4 答題框架

```
考試遇到揭穿面紗的題目，建議依以下步驟論述：

Step 1：點出公司法人格獨立原則（§154 I）
Step 2：說明揭穿公司面紗的法理基礎（§154 II）
Step 3：逐一檢驗要件
  → 是否有「濫用」行為？
  → 公司是否「清償顯有困難」？
  → 是否「情節重大而有必要」？
Step 4：結論——是否應揭穿，股東應否負清償責任
```

## 九、關係企業速查

### 9-1 核心定義

| 概念 | 定義 | 條文 |
|---|---|---|
| **關係企業** | 控制公司與從屬公司之總稱，或相互投資公司 | §369-1 |
| **控制公司** | 直接或間接控制他公司之人事、財務或業務經營者 | §369-2 |
| **從屬公司** | 受他公司直接或間接控制者 | §369-2 |
| **相互投資公司** | 相互持有對方已發行有表決權股份總數或資本總額三分之一以上者 | §369-9 |

### 9-2 控制從屬推定（§369-3）

| 推定情形 | 說明 |
|---|---|
| 持有他公司已發行有表決權股份總數或資本總額**超過半數** | 最常見的推定 |
| 直接或間接控制他公司之**人事、財務或業務經營** | 實質控制 |

### 9-3 從屬公司的保護機制

| 保護機制 | 內容 | 條文 |
|---|---|---|
| **關係報告書** | 控制公司應於每會計年度終了編製關係企業合併營業報告書 | §369-12 |
| **補償責任** | 控制公司使從屬公司為不合營業常規之行為，應於會計年度終了時為適當補償 | §369-4 I |
| **連帶賠償** | 未為補償致從屬公司受損害者，控制公司負責人應與控制公司連帶賠償 | §369-4 II |
| **債權人保護** | 控制公司對從屬公司之債權，於從屬公司清算或破產時，次於其他債權人受清償 | §369-7 |

### 9-4 相互投資（§369-9 以下）

| 要點 | 內容 |
|---|---|
| **定義** | 相互持股達 1/3 以上 |
| **通知義務** | 知悉後應即通知他方（§369-10） |
| **表決權限制** | 相互投資公司行使表決權，各不得超過被投資公司已發行有表決權股份總數或資本總額之 1/3（§369-10） |

## 十一、易混淆概念對照表

### 對照 1：§189 撤銷 vs §191 無效 vs 決議不成立

| 項目 | 得撤銷（§189） | 無效（§191） | 不成立 |
|---|---|---|---|
| **瑕疵原因** | 召集程序或決議方法違法 | 決議內容違反法令或章程 | 根本無合法的決議行為 |
| **效力** | 經法院判決撤銷前有效 | 自始無效 | 自始不成立 |
| **救濟** | 形成之訴 | 確認之訴 | 確認之訴 |
| **當事人** | 股東（出席並當場表示異議、未出席均可） | 利害關係人 | 利害關係人 |
| **期間** | 30 日除斥期間 | 無限制 | 無限制 |
| **典型案例** | 漏未通知股東、未達法定數額仍強行表決 | 決議違反法律強行規定 | 根本沒開會、偽造紀錄 |

### 對照 2：競業禁止（§209）vs 自我交易（§223）

| 項目 | 競業禁止（§209） | 自我交易（§223） |
|---|---|---|
| **行為態樣** | 董事為自己或他人從事與公司營業範圍相同的行為 | 董事為自己或他人與公司交易 |
| **規範目的** | 防止董事利用職務之便搶公司的生意 | 確保交易的公平性 |
| **許可/處理** | 須經股東會許可 | 由監察人代表公司 |
| **違反效果** | 歸入權（§209 V） | 無權代理（通說） |

### 對照 3：監察人代表訴訟 vs 少數股東代位訴訟

| 項目 | 監察人代表訴訟 | 少數股東代位訴訟（§214） |
|---|---|---|
| **原告** | 監察人代表公司 | 少數股東自行為公司提起 |
| **前提** | 公司（監察人）主動提起 | 股東先請求監察人提起，監察人 30 日未提起 |
| **持股門檻** | 不需要 | 繼續 1 年持有 3% |
| **勝訴利益** | 歸公司 | 歸公司 |

### 對照 4：有限公司 vs 閉鎖性股份有限公司

| 項目 | 有限公司 | 閉鎖性股份有限公司 |
|---|---|---|
| **法源** | §98 以下 | §356-1 以下 |
| **股東人數** | 1 人以上 | 50 人以下 |
| **出資/股份** | 出資額 | 股份 |
| **轉讓限制** | 須全體同意（可章程另訂） | 得以章程限制（§356-5） |
| **特別股** | 無 | 彈性極大（§356-7），得有複數表決權等 |

### 對照 5：發起設立 vs 募集設立

| 項目 | 發起設立 | 募集設立 |
|---|---|---|
| **股份認購** | 全部由發起人認足 | 發起人認一部分，其餘向外招募 |
| **門檻** | 發起人須認足第一次應發行股份 | 發起人須認足第一次應發行股份總數 1/4 以上 |
| **程序** | 較簡便 | 須經主管機關許可、開創立會 |
| **實務現況** | 絕大多數公司採此方式 | 極為罕見 |

### 對照 6：經理人（§29）vs 董事（§192）

| 項目 | 經理人 | 董事 |
|---|---|---|
| **選任** | 董事會決議委任（§29 I） | 股東會選任（§192 I） |
| **身分** | 受僱人（委任關係） | 受任人（委任關係） |
| **職權** | 執行業務（在授權範圍內） | 決定業務執行方針（合議） |
| **競業禁止** | §32（須董事會許可） | §209（須股東會許可） |

### 對照 7：普通決議 vs 特別決議 vs 假決議

| 項目 | 普通決議（§174） | 特別決議（§185） | 假決議（§175） |
|---|---|---|---|
| **出席門檻** | 過半數 | 2/3 以上 | 1/3 以上 |
| **表決門檻** | 出席過半數 | 出席過半數 | 出席過半數 |
| **適用事項** | 一般事項 | 重大事項（章程變更、合併等） | 普通決議出席不足時的替代 |
| **後續程序** | 直接生效 | 直接生效 | 須 1 個月內再開確認會議 |

### 對照 8：董事會決議 vs 股東會決議的效力瑕疵

| 項目 | 董事會決議瑕疵 | 股東會決議瑕疵 |
|---|---|---|
| **明文規定** | 無明文（類推適用） | §189、§191 明文規定 |
| **撤銷事由** | 類推 §189（有爭議） | §189 召集程序/決議方法違法 |
| **無效事由** | 類推 §191 | §191 內容違法 |
| **不成立** | 判例法理 | 判例法理 |
| **實務見解** | 最高法院多認董事會決議瑕疵**不適用** §189 的 30 日除斥期間 | 嚴格適用 30 日除斥期間 |

## 附錄：高頻情境題型速查

> 以下整理國考常見的典型題目情境，幫助你快速連結知識點。

### 情境 1：「甲未被通知而未出席股東會，該次決議有何效力？」

```
答題方向：
→ 召集程序違法 → §189 得撤銷
→ 甲得於決議日起 30 日內訴請法院撤銷
→ 若甲未於 30 日內起訴 → 決議確定有效（除斥期間經過）
→ 若該決議即使甲出席也不影響結果 → 法院得依裁量駁回（§189-1）
```

### 情境 2：「董事甲私下經營與公司同類型之業務」

```
答題方向：
→ §209 競業禁止 → 甲有無經股東會許可？
→ 未經許可 → 公司得行使歸入權（§209 V）
→ 歸入權除斥期間：知悉後 1 年，行為後滿 1 年
→ 同時可能構成忠實義務（§23 I）之違反 → 損害賠償
```

### 情境 3：「控制公司指示從屬公司為不合營業常規之交易」

```
答題方向：
→ §369-4 I：控制公司應為適當補償
→ 未補償致從屬公司受損害 → §369-4 II 連帶賠償
→ 若從屬公司破產/清算 → §369-7 控制公司債權劣後受償
→ 可能同時涉及揭穿公司面紗（§154 II）之適用
```

### 情境 4：「股東持有 3% 股份，欲對董事提起訴訟」

```
答題方向：
→ §214 少數股東代位訴訟
→ 須繼續 1 年持有 3%（兩要件均須滿足）
→ 先以書面請求監察人為公司提起訴訟
→ 監察人 30 日內不提起 → 股東得自行為公司提起
→ 勝訴利益歸於公司
```

### 情境 5：「公司面臨虧損，股東個人財產能否被追索？」

```
答題方向：
→ 原則：§154 I 股東有限責任，不得追索
→ 例外：§154 II 揭穿公司面紗
  → 檢驗是否有濫用法人地位之情形
  → 資本顯著不足？財產混同？人格混同？
  → 清償顯有困難＋情節重大有必要
→ §23 II：若公司負責人執行業務違反法令致第三人受損害
  → 負責人與公司連帶賠償（走不同路徑）
```
