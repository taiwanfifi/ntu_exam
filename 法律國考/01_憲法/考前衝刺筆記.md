# 憲法考前衝刺筆記 — 考前一週速讀版

> 適用對象：非法律背景轉職考生｜最後一週快速複習用
> 核心策略：背釋字、記架構、練比較、抓考點

## 一、必背釋字清單（25 號精選）

以下是考試出現頻率最高的大法官解釋，每號請至少能說出「爭點＋結論」。

### 1-1 平等權相關

| 釋字 | 爭點 | 結論 | 考試怎麼用 |
|------|------|------|------------|
| **釋字 748** | 同性婚姻是否受憲法保障？ | 民法未保障同性婚姻違憲，應於 2 年內修法。婚姻自由＋平等權。 | 平等權＋婚姻自由的經典題，必考。答題時強調「實質平等」概念。注意本號同時涉及婚姻自由（§22）與平等權（§7）。 |
| **釋字 807** | 勞基法禁止女性夜間工作是否違憲？ | 違憲。以性別作差別待遇，未通過中度審查（重要關聯性）。 | 平等權審查密度的最佳範例，說明中度審查如何操作。注意「性別」分類採中度審查，須「實質關聯」。 |
| **釋字 365** | 父母對子女權利行使意見不一致時以父行之？ | 違憲。以性別為分類標準，無正當理由。 | 早期平等權釋字，可和 807 對比說明審查密度演變。選擇題常考。 |
| **釋字 682** | 中醫特考限制應考資格（須修習一定學分）？ | 合憲。為確保中醫師專業能力，限制應考資格合比例。 | 平等權與應考試權的交叉考題，說明合理差別待遇的正當理由。 |

### 1-2 言論自由相關

| 釋字 | 爭點 | 結論 | 考試怎麼用 |
|------|------|------|------------|
| **釋字 509** | 刑法誹謗罪是否侵害言論自由？ | 合憲，但被告只要能證明有相當理由確信為真，即不罰。 | 言論自由 vs 名譽權的衡量基準，常出選擇題。「真實惡意原則」的台灣版。 |
| **釋字 644** | 人團法禁止主張共產主義之結社？ | 違憲。事前審查言論內容，違反比例原則。 | 「事前限制」vs「事後處罰」的區分，答題必提。 |
| **釋字 414** | 藥物廣告事前審查是否違憲？ | 合憲。商業言論保障程度較低，事前審查合比例。 | 商業言論 vs 政治言論的審查密度不同，重要區分。 |
| **釋字 445** | 集會遊行法「許可制」是否違憲？ | 部分違憲。許可制本身合憲，但「主管機關」認定等規定違憲。 | 集會自由的經典釋字，注意「內容中立」原則。 |

### 1-3 人身自由相關

| 釋字 | 爭點 | 結論 | 考試怎麼用 |
|------|------|------|------------|
| **釋字 384** | 檢肅流氓條例的秘密證人制度？ | 違憲。人身自由限制須符合正當法律程序。 | 正當法律程序在台灣的里程碑釋字，必背。 |
| **釋字 392** | 檢察官有無羈押權？ | 羈押決定權應歸法官，檢察官羈押權違憲。 | 法官保留原則的經典，考「憲法 §8 的 24 小時」常出。 |
| **釋字 708** | 入出國及移民法對外國人收容？ | 暫時收容逾 15 日應經法院裁定，否則違憲。 | 人身自由保障擴及非國民，重要觀念。 |
| **釋字 737** | 偵查中羈押審查程序未賦予辯護人閱卷權？ | 違憲。應保障被告及辯護人在羈押審查程序中的閱卷權。 | 正當法律程序的延伸，配合 384、392 一起記。 |

### 1-4 財產權 / 工作權 / 其他基本權

| 釋字 | 爭點 | 結論 | 考試怎麼用 |
|------|------|------|------------|
| **釋字 400** | 既成道路是否構成徵收？ | 既成道路成立公用地役關係，但國家應補償。特別犧牲理論。 | 財產權考題必出，記住「特別犧牲＝應補償」。 |
| **釋字 747** | 既成道路地主可否請求徵收？ | 地主有權請求國家徵收，不能無限期不補償。 | 補充 400 號，強化「補償請求權」觀念。 |
| **釋字 649** | 限制非視障者從事按摩業？ | 違憲。工作權限制未通過中度審查。 | 工作權＋平等權交錯考題的好素材。 |
| **釋字 584** | 計程車司機曾犯特定罪不得執業？ | 合憲。職業自由限制通過比例原則審查。 | 職業自由三階理論的適用，和 649 對比。 |
| **釋字 603** | 全民按捺指紋制度？ | 違憲。資訊隱私權受憲法保障（隱私權的資訊自決面向）。 | 隱私權＝資訊自主權，近年數位時代常考。 |

### 1-5 權力分立相關

| 釋字 | 爭點 | 結論 | 考試怎麼用 |
|------|------|------|------------|
| **釋字 520** | 行政院片面停建核四？ | 行政院停止執行法定預算義務，應向立法院報告。 | 行政立法關係＋覆議權的考題。 |
| **釋字 585** | 立法院調查權的界限？ | 立法院有調查權，但不得侵害其他機關核心權限，亦受比例原則拘束。 | 國會調查權的經典，注意界限＝不得侵害司法個案。 |
| **釋字 613** | NCC 組織條例（委員由立院推薦）？ | 違憲。侵害行政院人事決定權，破壞權力分立。 | 獨立機關的合憲界限，近年組織法常考。 |
| **釋字 499** | 國大延任修憲案？ | 違憲。修憲有界限，不得牴觸自由民主憲政秩序。 | 修憲界限理論的唯一釋字，必背。 |
| **釋字 632** | 總統遲未提名監察委員？ | 總統有義務適時提名，消極不提名構成違憲。 | 憲法忠誠義務，冷門但曾出過。 |

## 二、基本權審查架構速查

### 口訣：「保→限→限限」（保護範圍→限制→限制的限制）

```
第一步：保護範圍
  └→ 系爭行為是否落入某基本權的保護範圍？
       ├ 人的範圍：本國人？外國人？法人？
       └ 事的範圍：該行為是否屬於該基本權保障的內容？

第二步：限制（國家行為是否構成限制？）
  └→ 干預、侵害、不利益處分 → 構成「限制」
       ├ 直接限制（法律明文禁止）
       └ 間接限制（事實上產生寒蟬效應等）

第三步：限制的限制（該限制是否合憲？）
  ├ 形式合憲性
  │   ├ 法律保留原則（是否有法律依據？）
  │   ├ 授權明確性（若是授權立法，授權條款是否明確？）
  │   └ 法律明確性（法律本身是否可理解、可預見、可審查？）
  └ 實質合憲性
      ├ 比例原則（目的正當＋手段適當＋必要性＋狹義比例）
      └ 其他原則（平等原則、信賴保護等視情況適用）
```

### 審查架構對照表

| 審查步驟 | 要問的問題 | 常用條文 | 答題時寫什麼 |
|----------|-----------|---------|-------------|
| 保護範圍 | 這件事受哪個基本權保護？ | 憲法 §7-§22（概括基本權 §22） | 「系爭行為落入 §__ 之保護範圍」 |
| 限制 | 國家做了什麼干預行為？ | 看具體事實 | 「系爭法規/行為對上開基本權構成限制」 |
| 形式合憲 — 法律保留 | 有法律授權嗎？ | 憲法 §23、中央法規標準法 §5 | 「系爭規定有/無法律依據，符合/違反法律保留」 |
| 形式合憲 — 授權明確性 | 授權條款內容、目的、範圍夠明確嗎？ | 釋字 524、614 | 「授權條款已/未具體明確規定目的、內容及範圍」 |
| 形式合憲 — 法律明確性 | 人民看得懂法律在說什麼嗎？ | 釋字 432、521 | 「法律用語可/不可理解、預見、司法審查」 |
| 實質合憲 — 比例原則 | 手段與目的間是否合比例？ | 憲法 §23（四要件） | 逐一分析四個子原則 |
| 實質合憲 — 平等原則 | 差別待遇有無正當理由？ | 憲法 §7 | 「以 __ 為分類標準，採 __ 審查密度」 |

### 比例原則四階段詳解

| 子原則 | 問什麼 | 白話 | 答題常用句型 |
|--------|--------|------|-------------|
| **目的正當性** | 立法目的是否合憲？ | 政府為什麼要管這件事？理由正當嗎？ | 「系爭規定之立法目的在於保護 ___，屬合憲目的」 |
| **適當性** | 手段是否有助於目的之達成？ | 這樣做有用嗎？ | 「系爭手段確能達成上開目的，具適當性」 |
| **必要性** | 有無侵害更小的替代手段？ | 有沒有更溫和的方式？ | 「尚有 ___ 等侵害較小之替代手段，系爭規定不具必要性」 |
| **狹義比例性** | 手段造成的不利益與目的利益是否均衡？ | 值得嗎？ | 「手段所造成之損害與目的所追求之利益顯失均衡」 |

### 審查密度三級制

| 密度 | 適用情形 | 審查標準 | 白話 |
|------|---------|---------|------|
| **嚴格審查** | 涉及種族、出生地等可疑分類；核心基本權限制 | 目的須為「極重要的政府利益」，手段須為「最小侵害」 | 政府要拿出超強理由，幾乎都會被判違憲 |
| **中度審查** | 涉及性別分類；重要基本權限制 | 目的須「重要」，手段與目的須有「實質關聯」 | 政府理由要夠好，手段要合理 |
| **寬鬆審查** | 一般社經立法；非可疑分類 | 目的「合法」即可，手段與目的有「合理關聯」 | 政府只要說得通就好，通常合憲 |

## 三、各基本權比較表

### 3-1 言論自由（憲法 §11）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 發表言論、不表意自由、接收資訊的自由。包含政治言論、商業言論、學術言論、藝術表現。 |
| **審查密度** | 政治言論 → 嚴格審查；商業言論 → 中度審查（釋字 414）；猥褻言論 → 較低保障。 |
| **重要釋字** | 509（誹謗）、414（藥物廣告）、644（人團法）、445（集遊法）、617（猥褻出版品） |
| **答題關鍵** | 區分言論類型→對應不同審查密度；事前限制原則上違憲（除非商業言論）。 |

**言論自由子類型速查：**

| 言論類型 | 保障程度 | 代表釋字 | 特殊規則 |
|---------|---------|---------|---------|
| 政治性言論 | 最高保障 | 644 | 事前限制幾乎不允許；對民主運作有核心價值 |
| 學術自由 | 高度保障 | 380、563 | 含研究自由、教學自由；大學自治為制度性保障 |
| 商業性言論 | 中度保障 | 414、577 | 容許較多限制（如藥品廣告事前審查）；虛偽不實之商業言論不受保護 |
| 猥褻性言論 | 低度保障 | 617 | 硬蕊（hard-core）不受保護；軟蕊（soft-core）附條件保護 |
| 仇恨性言論 | 爭議中 | — | 台灣尚無明確釋字，但涉及言論自由 vs 平等尊嚴的衡量 |

### 3-2 宗教自由（憲法 §13）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 內在信仰自由（絕對保障）＋外在宗教行為自由（相對保障，可限制）。 |
| **審查密度** | 內在信仰：不得限制（絕對保障）。外在行為：比例原則審查。 |
| **重要釋字** | 490（兵役 vs 宗教信仰）、573（寺廟監督條例） |
| **答題關鍵** | 信仰本身不可限制；行為面才能審查。良心自由（§13）與言論自由（§11）的區分。 |

### 3-3 人身自由（憲法 §8）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 身體不受非法逮捕、拘禁、審問、處罰。核心＝法官保留＋正當法律程序。 |
| **審查密度** | 嚴格審查。憲法 §8 是制度性保障，有具體程序要求（24 小時移審）。 |
| **重要釋字** | 384（正當法律程序）、392（法官保留）、708（外國人收容）、737（閱卷權） |
| **答題關鍵** | 憲法 §8 是唯一有具體程序規定的基本權條文，注意「24 小時」、「法官保留」。 |

### 3-4 財產權（憲法 §15）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 既有財產的存續保障＋財產的使用收益處分。包含公法上的財產請求權。 |
| **審查密度** | 中度審查為主。社會義務（內在限制）vs 特別犧牲（應補償）。 |
| **重要釋字** | 400（既成道路）、747（請求徵收）、440（地價稅）、516（公營事業移轉民營） |
| **答題關鍵** | 區分「社會義務」（不補償）vs「特別犧牲」（應補償）是得分關鍵。 |

### 3-5 平等權（憲法 §7）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 等者等之、不等者不等之。禁止恣意的差別待遇。不是齊頭式平等。 |
| **審查密度** | 依分類標準而異：種族/性別 → 嚴格/中度；其他 → 寬鬆（合理審查）。 |
| **重要釋字** | 748（同婚）、807（女性夜間工作）、365（父權優先）、649（按摩業） |
| **答題關鍵** | 先找「分類標準」→再決定審查密度 → 最後看有無正當理由。 |

### 3-6 訴訟權（憲法 §16）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 接近法院的權利＋有效權利保護＋公正程序保障。包含聽審權。 |
| **審查密度** | 中度審查。立法者有一定的制度形成自由，但不得侵害核心。 |
| **重要釋字** | 418（訴訟救濟不得恣意限制）、574（再審限制）、752（上訴限制） |
| **答題關鍵** | 核心＝不得完全剝奪救濟途徑；程序上的限制可以，但不能過度。 |

### 3-7 工作權 / 職業自由（憲法 §15）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 選擇職業之自由＋執行職業之自由。包含營業自由。 |
| **審查密度** | 依職業自由三階理論，逐階提高密度。 |
| **重要釋字** | 584（計程車駕駛）、649（按摩業）、659（私立學校教師）、702（教師行為不檢限制） |
| **答題關鍵** | 套用三階理論是得分關鍵。 |

**職業自由三階理論詳解（源自德國藥房判決）：**

| 階層 | 限制類型 | 審查密度 | 白話 | 舉例 |
|------|---------|---------|------|------|
| 第一階 | 執行職業自由之限制 | 寬鬆 | 你可以當醫生，但政府規定營業時間 | 營業時間限制、廣告限制 |
| 第二階 | 選擇職業自由之主觀限制 | 中度 | 你想當醫生，必須先考取執照（看你個人條件） | 證照考試、學歷要求 |
| 第三階 | 選擇職業自由之客觀限制 | 嚴格 | 政府說這個行業只開放 100 個名額（跟你個人無關） | 計程車總量管制、藥局距離限制 |

### 3-8 學術自由 / 大學自治（憲法 §11 延伸）

| 項目 | 內容 |
|------|------|
| **保護範圍** | 研究自由＋教學自由＋學習自由。大學自治為制度性保障。 |
| **審查密度** | 涉及學術核心事項（如課程、教師聘任）→ 嚴格；行政管理事項 → 中度。 |
| **重要釋字** | 380（大學自治與課程自主）、450（大學內部組織自主）、563（退學處分與學術自由）、684（大學生訴訟權）、797（學生受教育權保障） |
| **答題關鍵** | 大學自治 ≠ 完全不受法律拘束。仍須遵守法律保留與比例原則。學生的訴訟權也受保障（釋字 684 打破特別權力關係）。 |

## 四、權力分立速查

### 4-1 總統

| 項目 | 內容 |
|------|------|
| 產生方式 | 直選，任期 4 年，連選得連任一次（§47） |
| 核心權限 | 統帥權（§36）、公布法律（§37）、締結條約（§38）、宣布戒嚴（§39）、赦免權（§40）、任命權（§41）、緊急命令（增修 §2 III）、被動解散立法院（增修 §2 V）、提名權（司法院/考試院/監察院正副首長及大法官等） |
| 重要爭議 | ① 總統 vs 行政院長的關係（雙首長制的擺盪）；② 國安大政方針決定權（增修 §2 IV）的範圍；③ 緊急命令須經立法院追認（10 日內） |
| 考試重點 | 釋字 627（總統刑事豁免權）、釋字 419（副總統兼任行政院長） |
| 免副署事項 | 解散立法院、任命行政院長、提出緊急命令（以上依增修條文，免經行政院長副署） |

**釋字 627 總統豁免權要點：**
- 總統在任期間享有刑事豁免權（不受刑事訴追）
- 但不及於民事與彈劾
- 豁免權不等於免責，卸任後仍可追訴
- 總統的國家機密特權須受司法審查

## 五、憲法訴訟法重點（111 年新制）

### 重要觀念：特別權力關係的突破

過去認為公務員、軍人、學生等「特別權力關係」下的人，不能對國家的管理措施提起訴訟。大法官逐步打破這個觀念：

| 釋字 | 打破了什麼 | 白話 |
|------|-----------|------|
| **釋字 187** | 公務員退休金可以爭訟 | 公務員的財產權也受保護 |
| **釋字 298** | 改變公務員身分的處分可爭訟 | 免職、降級這種大事可以告 |
| **釋字 430** | 軍人權益受侵害可爭訟 | 軍人也是人 |
| **釋字 684** | 大學生對大學處分可爭訟 | 學生不再「忍氣吞聲」，退學以外的處分也能告 |
| **釋字 785** | 公務員對影響權益重大的管理措施可爭訟 | 不再區分「改變身分」或「非改變身分」，統一看對權益影響是否重大 |

> 考試口訣：特別權力關係已經「打得差不多了」，現在只要影響權益重大就可以爭訟。

---

### 5-1 新舊制比較

| 項目 | 舊制（大審法） | 新制（憲法訴訟法） |
|------|--------------|------------------|
| 性質 | 解釋（抽象、客觀） | 訴訟（裁判形式、主觀權利保護） |
| 程序 | 聲請解釋 | 聲請裁判（聲請書狀→言詞辯論→判決） |
| 效力 | 解釋文＋理由書 | 判決主文＋理由 |
| 最大變革 | 無 | 新增「裁判憲法審查」（審查法院裁判本身有無違憲） |

### 5-2 聲請類型與要件

| 聲請類型 | 聲請人 | 要件 | 條文 |
|---------|--------|------|------|
| **法規範憲法審查**（抽象審查） | 國家最高機關（1/4 立委、行政院、司法院等） | 行使職權適用法律有違憲疑義 | §47-§48 |
| **法規範憲法審查**（具體審查） | 法官（聲請停止審判＋移送） | 審理案件所應適用之法律確信違憲 | §55 |
| **裁判憲法審查**（新制亮點） | 人民 | ① 用盡審級救濟；② 確定終局裁判所適用之法規範或裁判本身牴觸憲法；③ 裁判送達後 6 個月內 | §59 |
| **機關爭議** | 國家最高機關 | 行使職權與他機關發生憲法上權限爭議 | §65 |
| **政黨違憲解散** | 由主管機關聲請 | 政黨之目的或行為危害自由民主憲政秩序 | §69 |
| **地方自治保障** | 地方自治團體 | 立法院所議決之法律有牴觸憲法致侵害其自治權 | §82 |
| **統一解釋** | 人民或法官 | 不同審判權終審法院裁判見解歧異 | §84 |

### 5-3 裁判憲法審查（最重要的新制）

白話說明：過去人民只能挑戰「法律」違憲，現在可以直接挑戰「法院判決」違憲。

```
要件檢查清單：
□ 1. 窮盡一般救濟途徑（用盡上訴、抗告等）
□ 2. 對象是「確定終局裁判」
□ 3. 主張裁判所適用之法規範違憲，或裁判本身違憲
□ 4. 有具體指明違反憲法哪一規定
□ 5. 在裁判送達後 6 個月內提出
□ 6. 非屬不受理事由（§15 各款）
```

### 5-4 判決效力

| 效力類型 | 說明 |
|---------|------|
| 對世效力 | 判決對所有人都有效（不只是聲請人） |
| 溯及效力 | 原則上自判決生效日起；但大法官可定過渡期間 |
| 定期失效 | 宣告違憲但給立法者修法時間（通常 1-2 年） |
| 個案效力 | 裁判憲法審查：廢棄原確定終局裁判，發回管轄法院 |

### 5-5 不受理事由（§15，選擇題常考）

| 不受理事由 | 白話 |
|-----------|------|
| 聲請不合程式且未補正 | 書狀格式不對又不改 |
| 聲請人無聲請權 | 你沒資格聲請 |
| 聲請逾期 | 過了 6 個月期限 |
| 聲請不備理由或未具體指明違憲 | 你沒說清楚哪裡違憲 |
| 對憲法法庭之裁判聲請再審以外之救濟 | 憲法法庭的判決不能再上訴 |
| 聲請顯無理由 | 一看就知道不會贏 |

### 5-6 暫時處分（§43）

| 項目 | 內容 |
|------|------|
| 性質 | 類似假處分，在本案判決前暫時凍結系爭法規的效力 |
| 要件 | 為避免憲法所保障之權利或公益遭受難以回復之重大損害，且有急迫必要 |
| 期限 | 暫時處分之裁定，其效力不得逾 6 個月 |
| 考試重點 | 暫時處分 ≠ 本案判決，只是暫時措施 |

## 六、10 大必考考點排行

| 排行 | 考點 | 重要度 | 一句話結論 |
|------|------|--------|-----------|
| 1 | 比例原則的操作 | ★★★★★ | 四階段（目的正當→適當→必要→狹義比例），是所有基本權限制的核心審查工具。 |
| 2 | 平等權審查密度 | ★★★★★ | 先找分類標準，再依標準決定嚴格/中度/寬鬆審查，最後看有無正當理由。 |
| 3 | 裁判憲法審查 | ★★★★★ | 111 年新制，人民可直接主張法院「判決」違憲，6 個月內聲請。 |
| 4 | 言論自由的類型與審查 | ★★★★☆ | 政治言論最高保護、商業言論中度保護；事前限制原則上違憲。 |
| 5 | 正當法律程序 | ★★★★☆ | 法官保留、聽審權、24 小時移審（§8）。不只刑事，行政領域也適用。 |
| 6 | 法律保留 vs 國會保留 | ★★★★☆ | 法律保留＝行政行為需有法律依據；國會保留＝重大事項必須由國會親自立法，不得授權。 |

| 8 | 職業自由三階理論 | ★★★☆☆ | 執行（寬鬆）→ 主觀許可（中度）→ 客觀許可（嚴格）。源自德國藥房判決。 |
| 9 | 憲法 §22 概括基本權 | ★★★☆☆ | 不在 §7-§21 列舉中的自由權利，只要不妨害社會秩序公共利益，均受保障。如隱私權、人格權、資訊自主權。 |
| 10 | 修憲界限 | ★★★☆☆ | 釋字 499：修憲不得牴觸自由民主憲政秩序（國民主權、權力分立、基本權保障）。 |

## 七、易混淆概念對照表

### 7-1 比例原則 vs 平等原則

| 比較項目 | 比例原則 | 平等原則 |
|---------|---------|---------|
| 功能 | 審查「手段與目的的關係」 | 審查「差別待遇有無正當理由」 |
| 適用時機 | 國家限制人民基本權時 | 國家對不同群體作不同處理時 |
| 操作方式 | 四階段：目的正當→適當性→必要性→狹義比例 | 找分類標準→決定審查密度→審查正當理由 |
| 條文 | 憲法 §23 | 憲法 §7 |
| 關鍵口訣 | 「目適必狹」 | 「等者等之，不等者不等之」 |
| 常見搭配 | 言論自由、人身自由等自由權 | 性別、族群、身分的差別待遇 |
| 一起出現？ | 可以。同一案件可能同時審查比例原則和平等原則（如釋字 649、807） |

### 7-2 法律保留 vs 國會保留

| 比較項目 | 法律保留 | 國會保留 |
|---------|---------|---------|
| 白話意思 | 行政機關做事要有法律依據 | 重要的事必須國會自己決定，不能丟給行政機關 |
| 層級 | 有法律或法律授權的命令即可 | 必須是「法律」本身規定，不能授權 |
| 理論基礎 | 依法行政原則 | 重要性理論（Wesentlichkeitstheorie） |
| 白話舉例 | 政府要收你的稅→要有稅法依據 | 要不要課某種新稅→立法院必須自己立法決定，不能說「授權財政部決定」 |
| 判斷基準 | 有無法律依據即可 | 涉及「基本權的重大限制」就需要國會保留 |
| 重要釋字 | 釋字 443（層級化保留體系） | 釋字 443 也是這號（同時建立兩個概念） |

**釋字 443 層級化法律保留（必背口訣「憲法→絕對→相對→非」）：**

| 層級 | 內容 | 白話說明 |
|------|------|---------|
| 憲法保留 | 憲法直接規定，連立法者都不能改 | 例如：人身自由 24 小時移審（§8） |
| 絕對法律保留（國會保留） | 必須由法律直接規定 | 例如：刑罰、稅捐 |
| 相對法律保留 | 法律或法律授權的命令均可 | 例如：一般行政管制 |
| 非法律保留 | 不需要法律依據 | 例如：給付行政（但仍受預算拘束） |

### 7-3 抽象審查 vs 具體審查

| 比較項目 | 抽象審查 | 具體審查 |
|---------|---------|---------|
| 白話意思 | 不需要有具體案件，直接挑戰法律是否違憲 | 在審理具體案件時，法官發現要用的法律可能違憲 |
| 聲請人 | 國家最高機關（如 1/4 立委） | 各級法院法官 |
| 有無具體案件 | 不需要 | 需要（正在審理的案件） |
| 程序 | 機關認為法律違憲→直接聲請 | 法官確信違憲→裁定停止審判→移送憲法法庭 |
| 聲請目的 | 維護客觀的憲法秩序 | 保護具體案件當事人的基本權 |
| 條文 | 憲法訴訟法 §47-§48 | 憲法訴訟法 §55 |

### 7-4 更多易混淆概念速查

| 概念 A | 概念 B | 關鍵區分 |
|--------|--------|---------|
| 制度性保障 | 基本權保障 | 制度性保障保護的是「制度」（如婚姻、大學自治），不是個人主觀權利，但可轉化為個人基本權 |
| 緊急命令 | 戒嚴 | 緊急命令由總統發布（增修 §2 III），須經行政院會議決議＋10 日內送立法院追認；戒嚴由總統宣布（§39），經立法院通過或追認，可限制人民基本權 |
| 覆議 | 否決 | 覆議是行政院對立法院決議不服提出（增修 §3 II），立法院 1/2 維持就確定；美國否決是總統對國會法案否決，2/3 推翻。台灣門檻較低 |
| 法律明確性 | 授權明確性 | 法律明確性：法律本身用語是否可理解、預見、審查；授權明確性：法律授權行政機關訂命令時，目的、內容、範圍是否具體明確 |
| 彈劾 | 罷免 | 彈劾是由監察院（對公務員）或立法院（對總統）提出，追究法律責任；罷免是由人民直接行使，是政治責任，不問法律理由 |
| 解散國會 | 不信任案 | 不信任案由立法院對行政院長提出；若通過，總統可被動解散立法院（不是主動），解散後 60 日內重新選舉 |
| 主觀權利 | 客觀法秩序 | 主觀權利＝個人可以主張的權利（基本權的防禦面向）；客觀法秩序＝國家有義務建構的制度（基本權的客觀面向，如保護義務、組織與程序保障） |

### 7-5 基本權的功能面向

| 功能面向 | 白話 | 舉例 |
|---------|------|------|
| **防禦權功能** | 國家不要來干涉我 | 言論自由：國家不得禁止人民發言 |
| **給付請求權功能** | 國家要提供我某些東西 | 受教育權：國家應提供基本教育 |
| **保護義務功能** | 國家要保護我不被第三人侵害 | 生命權：國家有義務立法保護人民免受他人侵害 |
| **組織與程序保障** | 國家要建立制度讓我的權利能實現 | 訴訟權：國家要設法院、建立審級制度 |
| **制度性保障** | 國家要維護某些重要制度 | 婚姻制度、大學自治、地方自治 |

> 白話：基本權不只是「國家不要管我」，還包括「國家要幫我」、「國家要保護我」、「國家要建好制度」。

## 八、考前最後提醒

### 答題架構萬用模板（申論題）

```
一、本題涉及（某基本權 / 某權力分立問題）

二、基本權審查（若為基本權題）
  (一) 保護範圍
       → 系爭自由屬於憲法 §_ 保障之 ___ 權
  (二) 限制
       → 系爭法律/行政行為對上述基本權構成限制
  (三) 限制之合憲性審查
       1. 形式合憲：法律保留（有法律依據 / 授權明確？）
       2. 實質合憲：比例原則
          (1) 目的正當性：_____
          (2) 適當性：_____
          (3) 必要性：_____
          (4) 狹義比例性：_____

三、結論
  → 系爭規定（合憲 / 違憲）
```

### 選擇題秒殺技巧

| 題型 | 秒殺口訣 |
|------|---------|
| 問「何者違憲」 | 找「絕對限制」、「完全剝奪」、「概括授權」的選項 |
| 問「何者合憲」 | 找「有法律依據＋通過比例原則」的選項 |
| 問釋字結論 | 記住 80% 的釋字結論是「違憲」，選「合憲」要特別小心確認 |
| 問憲法本文 vs 增修 | 增修優先適用；總統直選、行政院長免同意、國大廢除都是增修 |
| 問機關權限 | 注意「最高」用語：行政院＝最高行政、立法院＝最高立法、但總統≠最高行政 |
| 問人民聲請釋憲 | 裁判憲法審查（新制）：用盡救濟＋6 個月＋具體指明違憲 |
| 問基本權主體 | 自然人享有所有基本權；法人只享有性質上可享有者（如財產權、訴訟權，但不含人身自由） |
| 問數字門檻 | 覆議 1/2、彈劾總統 2/3、修憲 3/4 出席 3/4 通過、不信任 1/2 |

### 常見陷阱題整理

| 陷阱敘述 | 正確觀念 |
|---------|---------|
| 「總統是最高行政首長」 | 錯。行政院是最高行政機關（§53）。總統不等於行政首長。 |
| 「大法官可以主動釋憲」 | 錯。須有人聲請，大法官不能主動啟動違憲審查。 |
| 「行政院長由立法院同意」 | 錯。增修後已改為總統直接任命，免經立法院同意。 |
| 「總統可以主動解散國會」 | 錯。只有在立法院通過不信任案後，總統才能被動解散。 |
| 「監察委員由人民選舉」 | 錯。監察委員由總統提名、立法院同意。 |
| 「法律保留等於國會保留」 | 錯。國會保留是法律保留的最高等級，要求國會親自立法，不得授權。 |
| 「外國人不享基本權」 | 錯。外國人也享有基本權保障（如人身自由，釋字 708），但部分權利可合理限制。 |
| 「比例原則只有三個子原則」 | 看怎麼算。傳統三個（適當＋必要＋狹義比例），但考試常加「目的正當性」成四階段。 |

| 「保限限限」 | 基本權審查三步驟（保護範圍→限制→限制的限制） |

### 地方自治重要考點

| 項目 | 內容 |
|------|------|
| 憲法依據 | 憲法 §10 章（§108-§128）、增修 §9 |
| 自治事項 | 地方自治團體就自治事項有立法權與執行權 |
| 自治條例 | 地方立法機關制定，可設罰則（罰鍰、停止營業等），但不得與中央法律牴觸 |
| 重要釋字 | 釋字 498（地方首長列席義務）、釋字 550（健保費用分擔）、釋字 553（里長延選爭議） |
| 監督方式 | 合法性監督為原則，合目的性監督為例外（僅限委辦事項） |
| 考試重點 | 自治事項 vs 委辦事項的區分；中央與地方權限爭議由司法院解決 |

### 基本國策考點整理（增修 §10）

基本國策是選擇題的常客，以下整理最容易出題的規定：

| 條次 | 內容 | 考試怎麼考 |
|------|------|-----------|
| 增修 §10 I | 國家應獎勵科學技術發展及投資 | 基本國策 = 方針條款，不直接賦予人民請求權 |
| 增修 §10 II | 經濟及科學技術發展應與環境生態保護兼籌並顧 | 環境權的憲法依據之一 |
| 增修 §10 V | 國家應消除性別歧視，促進兩性地位實質平等 | 平等權的方針規定，和 §7 搭配 |
| 增修 §10 VI | 國家應保障身心障礙者之保險、就醫、教育等 | 身心障礙者權益保障 |
| 增修 §10 VII | 國家應重視社會救助、福利服務、社會保險、醫療保健等 | 社會國原則的體現 |
| 增修 §10 XI | 國家肯定多元文化，積極維護原住民族語言文化 | 原住民族權益 |
| 增修 §10 XII | 國家應依民族意願保障原住民族地位及政治參與 | 原住民保障名額的憲法依據 |

**基本國策的法律效力（必考觀念）：**

| 效力類型 | 說明 | 白話 |
|---------|------|------|
| 方針條款 | 僅指示國家施政方向，不直接賦予人民請求權 | 國家「應該」做，但人民不能直接拿來告 |
| 憲法委託 | 課予立法者制定法律的義務 | 國家「必須」制定相關法律來落實 |
| 制度性保障 | 保障特定制度的核心不被廢除 | 如：公務員制度、大學自治 |

### 憲法前 20 條速查（選擇題基本功）

| 條文 | 內容 | 一句記憶 |
|------|------|---------|
| §1 | 國體 | 中華民國基於三民主義，為民有民治民享之民主共和國 |
| §2 | 主權 | 主權屬於國民全體 |
| §3 | 國民 | 具中華民國國籍者 |

> §23 的四個限制目的（口訣「妨危序益」）：
> 防止妨礙他人自由、避免緊急危難、維持社會秩序、增進公共利益。
> 答題時可以將具體案例對應到其中一個目的。
