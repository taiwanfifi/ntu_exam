## 壹、ch81-84 重建期——四章的「慢呼吸」

### 設計原則

```
Vol 1 ch1-4 = 讀者第一次認識這個世界
Vol 2 ch81-84 = 讀者重新認識同一個世界——但一切都不同了

差異：
- ch1 維倫是孤獨的。ch81 維倫是失去了什麼但不知道失去了什麼
- ch1 語青是陌生人。ch81 語青假裝是陌生人
- ch1 米亞在吐槽。ch81 米亞「什麼都沒說，但光球特別亮」
- ch1 讀者好奇。ch81 讀者心碎（因為讀者記得，但維倫不記得）
```

**節奏指令**：ch81-84 是 Vol 2 的「序篇」。不要引入大懸念。不要加速。讓讀者在熟悉的場景中感受「不同」。每章 ≤3,000 字。

### ch81 白衣與雨——與 ch1 的三重鏡像

**開場第一段寫法**：

```
不要：「第二卷開始了。時間是……」
要：從感官開始。

雨的聲音。
白大衣的布料質感——像醫院走廊那種冷白。
門關上的聲音。

維倫面前坐著一個陌生女人。
她說她的名字叫林語青。
```

**與 ch1 的精確對應表**：

| 元素 | ch1 | ch81 | 讀者的感受 |
|------|-----|------|-----------|
| 語青的到來 | 第一次來 | 第三次「第一次來」| 心碎——她要再來一次 |
| 維倫的反應 | 拒絕（「我不接非法委託」）| 說「請進」| 他變了。即使不記得為什麼 |
| 手的狀態 | 正常 | 手心痕跡在發燙 | ★ 身體記得 |
| 米亞 | 吐槽（「又一個想修記憶的」）| 什麼都沒說。光球特別亮 | 米亞記得一切 |
| 嚴柏翰 | 不存在 | 不在場（ch82-84 都不在）| 讀者焦慮——他去哪了？ |

**ch81 新增元素（區別於 ch1）**：
- 手心痕跡發燙 = Vol 2「身體記得」的升級版（Vol 1 是阿嬤的身體記得，Vol 2 是維倫自己的）
- 0.7 秒記憶盒仍在口袋裡 = 物理證據。維倫不知道它是什麼
- 雨 = Vol 2 的核心意象。ch81 開場下雨，ch160 結尾月光。從「洗去」到「照見」

**寫作陷阱**：
- ❌ 不要讓維倫表現出「似曾相識」的感覺——太早。他完全不認識語青
- ❌ 不要讓語青表現出悲傷——她的面具是「專業委託人」
- ✅ 唯一的裂縫：語青說出維倫的名字時，停頓了 0.3 秒。讀者注意到。維倫沒有
- ✅ 米亞的光球特別亮 = 她唯一的「表達」。不要多解釋

### ch82 0.7 秒——維倫的理性分析

**核心場景**：維倫發現口袋裡的記憶盒

```
一個標準的記憶晶片盒。
打開。裡面只有 0.7 秒的記憶。

月光。
一隻手。手指在月光下微微發白。
有人在握他的手。

0.7 秒。然後黑。
```

**維倫的反應 = 純分析（他不知道這段記憶的意義）**：
- 「誰的手？」→ 米亞分析指紋形狀：女性。年齡 20-30
- 「為什麼只有 0.7 秒？」→ 米亞推測：記憶被截斷，不是損壞
- 「為什麼在我的口袋裡？」→ 沒有答案

**米亞的發現（推動劇情）**：
```
記憶中嵌入了一段加密座標。
米亞解碼後指向——一個地下實驗室。
不在維倫的記憶地圖上。
他從來不知道這個地方存在。
```

**寫作指令**：
- 維倫分析 0.7 秒記憶的場景用「技術語言」——他在用職業技能處理一段「不明來源的數據」
- 讀者知道那是語青的手。維倫不知道。這個落差 = 心疼的來源
- 加密座標 = ch85 地下實驗室的鋪墊。不要在 ch82 揭示是什麼地方
- 米亞在本章可以恢復一點吐槽：「你口袋裡有別人的記憶。要我建議你檢查一下你最近的社交圈嗎？」

### ch83 第二碗豆漿——★ 身體記得的跨卷回收

**核心台詞**：

```
阿嬤擦了擦手上的油，看著維倫看了很久。
然後她問了一句話：

「你身邊是不是少了一個人？」

維倫愣住。「什麼？」

「以前你來的時候，旁邊總有一個穿白衣服的女孩子。
 她不怎麼說話。但她每次都會幫你把油條的油擦乾淨。」

阿嬤低頭繼續炸油條：
「我記不清她的臉了。但我的手記得——
 每次她來，我都會多做一碗豆漿。」
```

**設計解析**：
- 阿嬤的記憶（負債者，只記得怎麼做豆漿）= Vol 1 設定的回收
- 「我的手記得」= 「身體記得」母題從 Vol 1 延伸到 Vol 2 的第一次呼應
- 「多做一碗豆漿」= 身體的慣性。數位記憶被買賣和刪除，但身體的肌肉記憶留下了
- 語青「幫維倫擦油條的油」= 一個讀者在 Vol 1 可能沒注意到的日常細節——現在它變成了最心碎的證據

**寫法指令**：
- 阿嬤的語氣是平靜的。她不覺得這很嚴重。她只是在陳述一個事實
- 維倫的反應不要寫「震驚」——寫「安靜」。他聽到了，但不知道該怎麼辦
- 本章結尾：維倫走回工作室。手心的痕跡又開始發燙了。他握了握拳。不知道為什麼
- ≤2,500 字。這是一個「安靜章」。安靜 = 力量

### ch84 錨——語青的交易

**場景設計**：

```
語青第二次來到工作室。
這次她不是「委託人」。

「我有一個提議。」
「我幫你找回你失去的記憶碎片。
 你幫我深潛進一個危險人物的大腦。」

維倫：「為什麼是我？新台北有其他修復師。」

語青：「因為你是唯一一個——
 可以在深潛空間裡不迷失的人。
 周明哲設計你的時候就是這樣設計的。」

這是維倫第二次（在 Vol 2）聽到「周明哲」這個名字。
第一次是米亞在 ch82 分析加密座標時提到的。
但語青說出來的方式不一樣——像是她認識他。
```

**「錨」的雙重含義**：
1. 技術層面：深潛時需要「錨」——一個外部信號源，防止深潛者迷失在他人記憶中
2. 情感層面：語青就是維倫的錨。即使他不記得她

**寫作指令**：
- 語青提出交易時的語氣是「專業的」——不帶感情。這是她的面具
- 但有一個破綻：她說「危險人物」時，手指微微收緊了一下。讀者能猜到「危險人物」可能跟她有關
- 維倫接受交易的理由不是信任語青——是好奇。0.7 秒的記憶讓他想知道自己失去了什麼
- ch84 結尾 = ch85 的起手：「那個地下實驗室在哪裡？」「你口袋裡那段記憶的座標。」

## 貳、ch85 深潛協議——技術世界觀建立

### 場景：地下實驗室

```
深入地底層第三層。
一扇被遺忘的門。上面有鸚鵡標誌——但已經被劃掉了。

裡面：
周明哲留下的原型深潛機。
金屬椅。腦波感應頭盔。三十二根連接線。
牆上的白板還留著算式——墨水已經乾了十年。

米亞掃描後：
「這台機器的核心程式碼跟我的基底架構有 34% 的重疊。」
「什麼意思？」
「意思是，建造我的人——也建造了這台機器。」
```

**深潛規則建立（Vol 2 核心技術設定）**：

| 規則 | 內容 | 情感意義 |
|------|------|---------|
| 1. 記憶之樹 | 深潛空間的基本架構像一棵倒長的巨樹。根在最深處，葉在表層 | 人的記憶有「根」——砍掉根，葉子會枯 |
| 2. 每次深潛有時限 | 視腦波承受力而定。維倫首次上限約 30 分鐘 | 時間壓力 = 所有深潛場景的張力來源 |
| 3. 神經損傷率 | 米亞估計 40%。語青說「可以降低」但沒說怎麼 | 後果是真的。不是遊戲 |
| 4. 錨定信號 | 深潛者需要外部聲音錨定，防止意識迷失 | 語青的聲音 = 維倫的錨 = 情感隱喻 |
| 5. 防禦體 | 他人大腦的自我保護機制，會攻擊入侵者 | 記憶迷宮的「怪物」= 恐怖感來源 |

**演算法：Tree（首次出現在 Vol 2）**

```
A 線：深潛空間像一棵倒長的巨樹
     ├── 根（最深處）= 核心記憶（身份、最重要的人）
     ├── 主幹 = 長期記憶
     ├── 枝 = 中期記憶
     └── 葉 = 短期記憶、感官碎片

B 線：人的記憶有結構。砍掉表層，根還在。但砍掉根——

同步點：米亞在白板上畫出「記憶之樹」的結構圖時，
       維倫忽然理解了為什麼有些人失去了表層記憶卻保留了身體記得——
       因為身體記憶藏在根部。
```

**米亞比喻**：「想像一棵倒著長的樹。你看到的記憶是葉子。但真正定義你是誰的——在根部。大多數記憶篡改者只修剪葉子。但有人——學會了砍根。」

**寫作指令**：
- 地下實驗室的氛圍 = 被遺忘的聖殿。不是恐怖——是莊嚴+荒涼
- 「34% 重疊」= 周明哲造了米亞也造了這台機器。一行台詞。不要展開
- Tree 的教學融在場景描述中——維倫「看到」樹形結構，不是被「講解」
- 語青說「深潛有風險」時，她的聲音是平靜的。但讀者知道——她見過維倫在深潛中出事

### ch86 議員的大腦——DFS 第一次出現

**入口設計**：

```
深潛啟動。
維倫的意識沉入。

第一個畫面：一扇門。
門在變形——像融化的金屬，不斷改變形狀。

「進去之後往最深處走。」語青的聲音從外部傳來，帶著電流雜音。
「怎麼知道哪邊是最深處？」
「一直往下。遇到岔路選更暗的那條。」

這就是 DFS 的邏輯：深度優先。
先往最深的地方走。
走不通了，回到上一個岔路。
再走。
```

**演算法：DFS（深度優先搜尋）**

```
A 線：在記憶迷宮中不斷向更深層推進
     第一層：表面記憶（辦公室、演講、握手）→ 通過
     第二層：私人記憶（家庭、習慣）→ 通過
     第三層：隱藏記憶（不想被看見的）→ 開始有阻力
     第四層：防禦體出現 → 被擋

B 線：越深的記憶越不想被看見 = 人心的防禦

同步點：維倫穿過第三層時感到強烈的排斥感——
       不是技術上的阻力，是「這個人不想讓你看到」。
       A 線的「往下走」= B 線的「侵入隱私」。
       DFS 的「深度優先」= 直面最不舒服的真相。
```

**感官設計**：
- 視覺：記憶空間不是現實——牆壁有紋路（像腦溝）。光源不固定。影子方向會變
- 聽覺：議員的聲音在空間裡迴盪——是他的記憶碎片在播放。斷斷續續。像壞掉的收音機
- 觸覺：越深的地方越冷。「像走進冷凍庫，但冷的是你的意識，不是身體」
- 情緒滲透：深潛者會感受到被深潛對象的情緒。維倫在第三層開始感到一種「不屬於自己的羞恥感」

**寫作指令**：
- 第一次深潛要寫得「慢」——維倫是新手。他在適應環境
- DFS 的「選更暗的那條」不是語青的專業術語——是她用日常語言解釋演算法
- 不要在 ch86 就揭露真相。ch86 只走到第四層被擋。到 ch90 才找到真相

### ch87 走廊盡頭的眼睛——恐怖的頂峰

**防禦體設計**：

```
走廊盡頭。
黑暗中有什麼在看他。

維倫舉起手中的記憶探針照明——

一隻眼球。
直徑至少三米。
佔滿了整條走廊。
瞳孔在收縮。
它認出了入侵者。

「回來！」語青的聲音帶著罕見的急迫。
```

**防禦體 = 恐懼的凝聚**：
- 議員最恐懼的記憶碎片凝聚成的保護機制
- 它不會「攻擊」——它會「讓你看到你最害怕的東西」
- 維倫如果直視它太久，會被灌入議員的恐懼記憶，導致意識混亂

**演算法：DFS 回溯**

```
A 線：碰到死路（防禦體堵住走廊）→ 返回上一個分叉點
     維倫退回第三層的岔路 → 選另一條路 → 繞過防禦體

B 線：不是所有真相都能正面衝撞。有時候，回頭找另一條路 = 更聰明的勇氣

同步點：維倫回溯時說「繞路不是放棄」→ 米亞記錄：「回溯節點。」
```

**寫作指令**：
- 防禦體的恐怖感來自「它在看你」的壓迫感，不是暴力。不要寫成怪物戰鬥
- 維倫的回溯是被動的（語青叫他回來）→ ch88 他主動理解回溯邏輯 = 成長
- 本章 ≤2,500 字。恐怖場景不能太長，否則變成恐怖小說不是記憶修復小說
- 結尾：維倫退出防禦體視線後，發現自己在發抖。不是因為冷。是因為他剛才看到了防禦體的瞬間——那隻眼球裡映出了另一張臉。不是議員的。他不認識

### ch88 分岔——BST 的詩意

**核心場景**：米亞從外部將迷宮視覺化

```
米亞在深潛空間中投射了一個半透明的結構圖。
維倫第一次從「上帝視角」看到了這座記憶迷宮的全貌。

不是迷宮。
是一棵樹。

每一個記憶選擇都是一次分岔——
左邊是「做了的事」。
右邊是「沒做的事」。
越痛苦的記憶排得越深。
越快樂的排得越淺。

「它在按情感強度排列。」維倫說。
米亞：「準確地說，是二元搜尋樹。每個節點的左子樹是『比它更痛的記憶』。」
```

**演算法：BST（二元搜尋樹）**

```
A 線：記憶按情感強度排列成 BST
     搜尋方向：找到「投票前的猶豫」= 痛苦指數中等
     → 先到根節點（最痛的記憶 = 喪親）
     → 太深了。往右（較淺的方向）
     → 中間節點 = 失戀 → 還是太深
     → 再往右 → 找到了：投票前的猶豫

B 線：人的記憶不是隨機擺放的。它們自動按照「多痛」來排列
     最深處放著你最不願意面對的事
     最淺處放著你最容易忘記的小確幸

同步點：維倫找到目標記憶的瞬間意識到——
       他自己的記憶之樹，最深處會放著什麼？
       一段 0.7 秒的記憶。月光。一隻手。
       那是他「最痛但不知道為什麼痛」的記憶。
```

**米亞比喻**：「你的記憶不是抽屜——是一棵樹。每次你經歷一件事，它就長出一個新枝。痛苦的往左長。快樂的往右長。如果你想找一段記憶，不需要翻遍所有抽屜——只要問自己：它比我最痛的記憶更痛，還是沒那麼痛？然後往那個方向走。」

**寫作指令**：
- 這是 Vol 2 第一個「美」的教學場景——記憶之樹的全貌應該是壯觀的、詩意的
- 維倫看到議員記憶之樹全貌時的反應不是「分析」——是「敬畏」
- BST 的教學用「搜尋目標記憶」的過程自然帶出，不是米亞的課堂講解
- 「他自己的記憶之樹最深處會放什麼？」= 維倫開始把技術跟自己連結。一句話。不要展開

### ch89 語青的聲音——錨定與信任

**核心場景**：維倫在第六層迷失

```
第六層。
所有走廊看起來都一樣。
灰色的牆。相同的轉角。相同的光。

維倫停下來。
他已經走了十分鐘（深潛空間裡的時間感）。
但他不確定自己有沒有在前進。

然後——

「維倫。」

語青的聲音。從外部傳來。帶著電流雜音和微微的失真。
但他能辨認。不是因為頻率或音色。
是因為——他的手心在發燙。

「你在第六層的 B-7 節點。往左走。前面有一個下行通道。」

「妳怎麼知道？」

「因為我在看米亞的即時投影。你離真相只剩兩層了。」

維倫往左走了。
不是因為相信她的判斷。
是因為她的聲音——不知道為什麼——讓他覺得安全。
```

**設計解析**：
- 語青的聲音 = 深潛的錨定信號（技術層面）+ 維倫重新信任的起點（情感層面）
- 「手心在發燙」= ch81 的回收。身體記得她的聲音
- 「不是因為相信她的判斷。是因為她的聲音讓他覺得安全」= Vol 2 信任的起點
- 本章無演算法。純情感章

**寫作指令**：
- ch89 是五章深潛中唯一的「溫暖」。前面是恐怖（ch86-87），後面是反轉（ch90）。這裡需要讓讀者喘氣
- 語青的聲音描寫要具體：「帶著電流雜音和微微的失真」= 她不是在他身邊，是在外面
- 「手心在發燙」不要解釋。讓它存在。讀者自己懂
- ≤2,000 字。短。安靜的力量

### ch90 賄賂的味道——BFS 橫向掃描 + 反轉

**真相揭露**：

```
議員記憶的最深處。

維倫本來以為會找到賄賂紀錄。
但他找到的——

不是記憶賄賂。
是記憶「清洗」。

有人替議員刪除了投票前的猶豫和良心不安。
那些「我這樣做真的對嗎」的瞬間。
那些凌晨三點睡不著、盯著天花板的夜晚。

全部被清洗了。
只留下「果斷」「堅定」「正確」。

維倫看著清洗後的記憶：
一個永遠不會猶豫的人。
一個永遠不會失眠的人。
一個完美的政客。

「這比賄賂更可怕。」維倫說。
「賄賂是改變你的選擇。清洗是——刪掉你選擇的能力。」
```

**演算法：BFS（廣度優先搜尋）**

```
A 線：在最深層的記憶平面上橫向掃描所有關聯碎片
     DFS 讓維倫到達了最深處
     BFS 讓他在最深處「橫向展開」——
     一圈一圈向外擴散，收集所有與「投票」相關的記憶碎片
     拼湊出完整的交易場景

B 線：真相不在一個點上——它散布在同一層的很多碎片裡
     你需要「廣度」來看到全貌

同步點：維倫用 BFS 拼湊出完整場景的瞬間——
       反轉不是「找到證據」而是「發現真相跟想像不同」
       A 線的「橫向掃描」= B 線的「全貌比單點更重要」
```

**米亞比喻**：「DFS 是挖井——越挖越深。BFS 是撒網——在同一個深度鋪開。你已經挖到底了。現在需要撒網。」

**寫作指令**：
- 反轉（不是賄賂是清洗）要在 200 字內完成。快。讀者的衝擊來自「突然」
- 「清洗」的描寫用「缺失」而非「存在」——維倫看到的不是「被添加的記憶」而是「被刪掉的猶豫」
- 「刪掉你選擇的能力」= Vol 2 的哲學升級：Vol 1 是「改記憶」，Vol 2 是「刪判斷力」
- BFS 教學融在「收集碎片」的動作中。維倫「一圈一圈向外」= 讀者自然理解廣度搜索
- ch90 結尾：維倫從深潛退出。不要寫「他學到了很多」。寫「他出來後看了語青一眼，然後說：『有人在系統性地清洗這座城市的良心。』」
