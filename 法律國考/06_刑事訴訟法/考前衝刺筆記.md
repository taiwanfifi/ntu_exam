## 壹、強制處分速查 ★★★

### 一、強制處分基本原則

| 原則 | 內容 |
|-----|------|
| 令狀原則 | 強制處分原則上須經法官核發令狀（法官保留） |
| 比例原則 | 手段與目的間須具備合理關聯，不得過度侵害人權 |
| 法律保留 | 須有法律明文依據始得為之 |

### 二、五大強制處分比較表

| 比較項目 | 逮捕 | 搜索 | 扣押 | 羈押 | 通訊監察 |
|---------|------|------|------|------|---------|
| 侵害法益 | 人身自由 | 居住/隱私自由 | 財產權 | 人身自由（最嚴重） | 秘密通訊自由 |
| 令狀要求 | 拘票（§77）/通緝書/無令狀逮捕 | 搜索票（§128）/無令狀搜索 | 扣押裁定/附隨搜索之扣押 | 羈押票（§102） | 通訊監察書（通保法§5） |
| 核發機關 | 法官（拘票）/檢察官（拘提）/司法警察（逕行逮捕） | 法官 | 法官/檢察官 | 法官 | 法官 |
| 期間限制 | 24小時內移送（§93） | 依票載期間 | 無固定期限 | 2月，延長限1次（§108） | 30日，得延長（通保法§12） |

### 三、逮捕類型 ★

| 類型 | 要件 | 條文 |
|-----|------|------|
| 通緝逮捕 | 經通緝之被告，利害關係人得逕行逮捕 | §87 |
| 現行犯逮捕 | 犯罪在實施中或實施後即時發覺者；不問何人得逕行逮捕 | §88 |
| 準現行犯逮捕 | §88 II 各款情形（被追呼為犯罪人、持有兇器贓物、身體衣服有犯罪痕跡、逃跑） | §88 II |
| 緊急拘提 | 檢察官、司法警察（官）因情況急迫，有相當理由認為有逃亡或湮滅證據之虞 | §88-1 |

### 四、搜索類型 ★★

| 類型 | 要件 | 條文 |
|-----|------|------|
| 有令狀搜索 | 法官核發搜索票 | §128 |
| 同意搜索 | 經受搜索人自願同意，無須搜索票 | §131-1 |
| 附帶搜索 | 逮捕被告或執行拘提時，得搜索其身體、隨身物件及所在處所 | §130 |
| 緊急搜索 | 有明顯事實足信有人犯罪嫌疑且情況急迫 | §131 I |
| 逕行搜索 | §131 II 各款（現行犯或準現行犯之住居所等） | §131 II |

### 五、羈押要件 ★★★

**三要件全備始得羈押**：

| 要件 | 內容 |
|-----|------|
| (1) 犯罪嫌疑重大 | 有相當理由認被告涉嫌犯罪 |
| (2) 法定原因（§101 I 各款） | 第1款：逃亡或有事實足認有逃亡之虞 |
| | 第2款：有事實足認有湮滅、偽造、變造證據或勾串共犯或證人之虞 |
| | 第3款：所犯為死刑、無期徒刑或最輕本刑五年以上之罪，有相當理由認有逃亡、湮滅證據之虞 |
| (3) 非予羈押顯難進行追訴、審判或執行 | 羈押必要性（比例原則之體現） |

**預防性羈押 §101-1**：特定犯罪（如放火、強制性交、殺人等），有事實足認有反覆實施同一犯罪之虞。

### 六、羈押替代處分 §116-2

| 替代手段 | 說明 |
|---------|------|
| 具保 | 繳納保證金（§110） |
| 責付 | 責付適當之人（§115） |
| 限制住居 | 限制被告住居所（§116） |
| 限制出境出海 | §93-2 以下 |

### 七、羈押審查程序 ★

| 項目 | 內容 |
|-----|------|
| 聲請權人 | 檢察官（偵查中）向法院聲請 |
| 法官審查 | 法官須訊問被告後始得裁定羈押（§101 I） |
| 被告防禦 | 被告得選任辯護人到場（§31-1偵查中指定辯護） |
| 羈押期間 | 偵查中不得逾2月，延長以1次為限（§108 I）；審判中不得逾3月，每延長一次不得逾2月 |
| 撤銷羈押 | 羈押原因消滅或無羈押必要時，法院應依職權或聲請撤銷之 |
| 提審 | 被告或其辯護人得聲請法院撤銷或變更羈押處分 |

### 八、通訊監察重點（通訊保障及監察法）★

| 項目 | 內容 |
|-----|------|
| 令狀核發 | 法官核發通訊監察書（通保法§5） |
| 重罪要件 | 限於通保法§5所列之特定犯罪（最輕本刑3年以上等） |
| 最後手段 | 須有相當理由可信有通保法§5 I各款情形，非有通訊監察不能或難以蒐集證據者 |
| 期間 | 每次不得逾30日，得延長，每次延長不得逾30日（通保法§12） |
| 另案監聽 | 監聽中發現他案犯罪者，須符合通保法§18-1之要件始得使用 |
| 違法監聽 | 違法取得之通訊內容不得作為證據（通保法§18-1 III），屬「絕對排除」 |

## 貳、證據法則速查 ★★★

### 一、自白法則 §156 ★★★

| 項目 | 內容 |
|-----|------|
| 任意性要求 | 被告之自白，非出於強暴、脅迫、利誘、詐欺、疲勞訊問或其他不正方法者，始得為證據（§156 I） |
| 補強法則 | 被告之自白不得作為有罪判決之唯一證據，仍應調查其他必要之證據，以察其是否與事實相符（§156 II） |
| 共犯自白 | 共同被告之自白，不得作為其他共同被告犯罪之證據（§156 II 後段之意旨）；須以證人身分具結交互詰問（大法官釋字582） |
| 毒樹果實 | 非任意性自白所衍生之證據（毒樹果實），實務採「個案權衡」 |

### 二、違法證據排除 §158-4 ★★★

| 項目 | 內容 |
|-----|------|
| 條文 | 除法律另有規定外，實施刑事訴訟程序之公務員因違背法定程序取得之證據，其有無證據能力之認定，應審酌人權保障及公共利益之均衡維護 |
| 審酌因素（實務見解） | (1)違背法定程序之程度 (2)違背法定程序時主觀意圖 (3)侵害犯罪嫌疑人權益之種類及輕重 (4)犯罪所生危險或實害 (5)禁止使用證據對預防將來違法取證之效果 (6)偵審人員如依法定程序有無發現該證據之必然性 (7)證據取得之違法對被告訴訟上防禦不利益之程度 |
| 性質 | 相對排除法則（權衡理論），非絕對排除 |

### 三、證據能力與證明力

| 概念 | 說明 |
|-----|------|
| 證據能力 | 得為證據之適格（法律判斷） |
| 證明力 | 證據足以證明事實之程度（法官自由心證） |
| 嚴格證明 | 須有證據能力＋經合法調查程序（用於犯罪事實之認定） |
| 自由證明 | 不限證據種類與調查方式（用於程序事項之釋明） |

### 四、補強法則深入 ★★

| 項目 | 內容 |
|-----|------|
| 適用對象 | 被告之自白（含審判中及審判外之自白） |
| 補強證據範圍 | 須為自白以外之獨立證據，且足以擔保自白之真實性 |
| 補強程度 | 不須達到獨立證明犯罪之程度，但須與自白互相印證 |
| 共犯自白之補強 | 共同被告之自白不得互為補強（釋字582號解釋意旨） |
| 實務見解 | 最高法院認為補強證據須與犯罪事實具有「關聯性」，且係合法取得之證據 |

### 五、私人違法取證 ★

| 項目 | 說明 |
|-----|------|
| 問題 | §158-4 係針對「公務員」違法取證，私人取證是否適用？ |
| 實務見解 | 私人取證原則上有證據能力，例外以嚴重侵害被告權利者始排除 |
| 學說 | 有主張類推適用§158-4權衡者；亦有主張私人取證不受拘束者 |
| 典型案例 | 配偶私自錄音通話、員工竊取公司文件作為證據 |

## 參、傳聞法則與例外整理 ★★★

### 一、傳聞法則基本架構

| 項目 | 內容 |
|-----|------|
| 傳聞定義 | 被告以外之人於審判外之言詞或書面陳述（§159 I） |
| 原則 | 無證據能力（§159 I） |
| 例外 | §159-1 ~ §159-5 各有例外規定 |
| 立法理由 | 保障被告之對質詰問權，確保陳述之可信性 |

### 二、傳聞例外整理表 ★★★

| 條文 | 陳述場景 | 要件 | 備註 |
|-----|---------|------|------|
| **§159-1 I** | 於法官面前之陳述 | 原則有證據能力；除顯有不可信之情況外 | 含另案法官前陳述 |
| **§159-1 II** | 於檢察官面前之陳述 | 除顯有不可信之情況外，有證據能力；但被告得聲明異議，經傳喚不到或有正當理由未到始得使用 | 實務最常考 ★★★ |
| **§159-2** | 於檢察事務官、司法警察（官）前之陳述 | 與審判中不符部分，經證明先前陳述具有較可信之特別情況且為證明犯罪事實之存否所必要 | 要件最嚴格 |
| **§159-3** | 陳述人不能到庭之情形 | 因死亡、身心障礙、滯留國外、所在不明或到庭有重大困難，經證明其先前陳述具有可信之特別情況且為證明犯罪事實所必要 | 不能到庭＋可信＋必要 |
| **§159-4** | 特信性文書 | 第1款：公務員職務上製作之紀錄或證明文書 | 公文書 |
| | | 第2款：業務上或通常業務過程所須製作之紀錄、文書、證明 | 業務文書（如醫療紀錄） |
| | | 第3款：其他可信之特別情況下所製作之文書 | 概括條款 |
| **§159-5 I** | 同意作為證據 | 經當事人於審判程序同意作為證據，法院審酌適當者 | 明示同意 |
| **§159-5 II** | 未於言詞辯論聲明異議 | 當事人於法院調查證據時知其為傳聞而未聲明異議，視為同意 | 擬制同意（實務最常用）★★ |

### 三、傳聞法則答題口訣

> 「法（§159-1）→ 檢（§159-1 II）→ 警（§159-2）→ 不能到（§159-3）→ 文書（§159-4）→ 同意（§159-5）」

### 四、傳聞法則答題流程 ★★

```
步驟一：確認陳述人是否為「被告以外之人」
  → 被告本人之陳述非傳聞，不適用§159

步驟二：確認陳述是否於「審判外」作成
  → 審判中之當庭陳述非傳聞

步驟三：確認屬於何種傳聞類型
  → 在法官前（§159-1 I）
  → 在檢察官前（§159-1 II）
  → 在司法警察前（§159-2）
  → 陳述人不能到庭（§159-3）
  → 特信性文書（§159-4）
  → 當事人同意（§159-5）

步驟四：逐一檢驗該款例外之要件

步驟五：結論
```

### 五、§159-1 II 檢察官前陳述之實務操作 ★★★

| 項目 | 說明 |
|-----|------|
| 原則 | 有證據能力（推定可信） |
| 例外排除 | 「顯有不可信之情況」始否定證據能力 |
| 被告對質詰問權 | 被告得於審判中聲明異議，要求傳喚證人到庭行交互詰問 |
| 傳喚不到 | 經合法傳喚證人不到或有正當理由不能到庭時，始得逕使用偵查中陳述 |
| 實務爭議 | 偵查中未給予被告在場機會之陳述，是否仍有證據能力？實務認為尚須審酌被告對質詰問權之保障 |

## 肆、偵查程序重點 ★★

### 一、偵查主體

| 主體 | 角色 |
|-----|------|
| 檢察官 | 偵查主體，指揮司法警察偵查犯罪（§228、§229） |
| 司法警察（官） | 偵查輔助機關，受檢察官指揮（§229~§231） |

### 二、強制偵查 vs 任意偵查 ★

| 比較項目 | 任意偵查 | 強制偵查 |
|---------|---------|---------|
| 定義 | 不涉及強制力之偵查方法 | 以強制力干預人民基本權利之偵查方法 |
| 法律保留 | 不以法律明文為必要 | 須有法律明文依據（強制處分法定原則） |
| 令狀原則 | 不適用 | 原則上須令狀（法官保留原則） |
| 舉例 | 詢問、跟監（有爭議）、調閱監視器 | 搜索、扣押、羈押、逮捕、通訊監察 |

### 三、偵查終結 ★

| 終結方式 | 要件 | 條文 |
|---------|------|------|
| **提起公訴** | 足認被告有犯罪嫌疑 | §251 |
| **不起訴處分** | 犯罪嫌疑不足、行為不罰、欠缺訴訟條件等 | §252 |
| **緩起訴處分** | 被告所犯為死刑、無期徒刑或最輕本刑三年以上有期徒刑以外之罪，審酌公共利益之維護、被害人損害賠償等，認以緩起訴為適當 | §253-1 |

### 四、不起訴處分確定之效力

| 項目 | 內容 |
|-----|------|
| 效力 | 非既判力，但有「實質確定力」（拘束檢察官） |
| 再行起訴 | 發現新事實或新證據，得再行起訴（§260） |
| 救濟 | 告訴人得聲請再議（§256）→ 高檢署命令續行偵查或駁回 → 得聲請交付審判/准許提起自訴（§258-1） |

### 五、偵查不公開原則 §245

| 項目 | 內容 |
|-----|------|
| 原則 | 偵查不公開（§245 I） |
| 目的 | 保護被告名譽、確保偵查順利、防止串供滅證 |
| 例外 | 依法令或為維護公共利益或保護合法權益有必要者 |
| 違反效果 | 偵查人員違反偵查不公開者，得依法追究責任，但不影響證據能力 |

### 六、緩起訴處分詳解 ★

| 項目 | 內容 |
|-----|------|
| 適用罪名 | 非死刑、無期徒刑、最輕本刑3年以上有期徒刑之罪 |
| 期間 | 1年以上3年以下之緩起訴期間（§253-1 I） |
| 附條件 | §253-2：得命被告向被害人道歉、賠償、向公庫支付一定金額、義務勞務、完成戒癮治療等 |
| 撤銷事由 | §253-3：緩起訴期間內故意更犯有期徒刑以上之罪、未遵守條件情節重大等 |
| 撤銷效果 | 檢察官得繼續偵查或起訴 |
| 期滿效果 | 緩起訴期滿未經撤銷者，非經發現新事實新證據不得對同一案件再行起訴（§260） |

## 伍、起訴審查與公訴 ★★

### 一、起訴法定主義 vs 起訴便宜主義

| 比較項目 | 起訴法定主義 | 起訴便宜主義 |
|---------|------------|------------|
| 意義 | 有犯罪嫌疑者，檢察官即應起訴 | 檢察官得裁量是否起訴 |
| 我國制度 | 原則（§251 I） | 例外：緩起訴（§253-1）、微罪不舉（§253） |
| 目的 | 貫徹國家追訴義務 | 訴訟經濟、給予自新機會 |

### 二、起訴之效力 ★

| 效力 | 內容 | 條文 |
|-----|------|------|
| 訴訟繫屬 | 案件繫屬於法院，法院取得審判權 | — |
| 起訴不可分（公訴不可分） | 起訴之效力及於「犯罪事實之全部」 | §267 |
| 審判不可分 | 法院應就起訴效力所及之全部犯罪事實審判 | — |

### 三、公訴之同一性 ★★★

| 概念 | 內容 | 條文 |
|-----|------|------|
| 案件同一性 | 被告同一＋犯罪事實同一 | — |
| 犯罪事實同一 | 社會事實同一（基本社會事實關係相同） | 學理 |
| 變更起訴法條 | 法院認應變更檢察官所引應適用法條者，應踐行§95告知義務後變更之 | §300 |
| 典型考法 | 檢察官以竊盜起訴→法院認定為搶奪→得依§300變更起訴法條（犯罪事實同一） | §300 |

### 四、§300 變更起訴法條 ★★

| 項目 | 內容 |
|-----|------|
| 要件 | (1)犯罪事實同一性之範圍內 (2)法院認應適用之法條與檢察官起訴法條不同 |
| 程序保障 | 法院應踐行§95 告知變更後法條，並給予被告辯解機會（§96） |
| 不得變更 | 超出犯罪事實同一性範圍者，法院不得審判（訴外裁判禁止） |

### 五、起訴書應記載事項 §264

| 項目 | 內容 |
|-----|------|
| 被告姓名等 | 被告之姓名、性別、年齡、住所等基本資料 |
| 犯罪事實 | 犯罪事實及證據並所犯法條 |
| 起訴範圍 | 起訴之效力及於起訴書所記載犯罪事實之全部（§267） |
| 卷證移送 | 檢察官應將卷宗及證物一併送交法院（起訴卷證併送制） |

### 六、單一性與同一性 ★★

| 概念 | 說明 |
|-----|------|
| 案件單一性 | 一個犯罪事實構成一個案件（實體法上一罪＝訴訟法上一案件） |
| 案件同一性 | 前後兩訴是否為同一案件（被告同一＋犯罪事實同一） |
| 實質上一罪 | 繼續犯、接續犯、集合犯、吸收犯等→一個案件 |
| 裁判上一罪 | 想像競合、牽連犯（舊法）→一個案件 |
| 單一性之功能 | 決定起訴不可分（§267）之範圍 |
| 同一性之功能 | 決定變更起訴法條（§300）之範圍 |

## 拾、易混淆概念對照表

### 一、逮捕 vs 拘提 ★

| 比較項目 | 逮捕 | 拘提 |
|---------|------|------|
| 定義 | 直接以強制力拘束人身自由之強制處分 | 以拘票強制被告到場之強制處分 |
| 令狀 | 不需令狀（現行犯、通緝犯）或需令狀 | 需拘票（§77） |
| 執行主體 | 任何人（現行犯）、司法警察、檢察官 | 司法警察（依法官或檢察官簽發之拘票） |
| 事由 | 現行犯逮捕（§88）、通緝逮捕（§87）、緊急拘提（§88-1） | 經合法傳喚不到（§75）、或有§76各款事由 |
| 時間限制 | 24小時內移送（§93） | 24小時內移送 |
| 考試口訣 | 逮捕偏「緊急」，不待傳喚 | 拘提偏「程序」，先傳後拘為原則 |

### 二、搜索 vs 扣押 ★

| 比較項目 | 搜索 | 扣押 |
|---------|------|------|
| 定義 | 對人之身體、物件、住宅或其他處所之搜查 | 取得物之占有而留置之強制處分 |
| 目的 | 發現被告或犯罪證據 | 保全證據或可得沒收之物 |
| 令狀 | 搜索票（§128） | 扣押裁定或附隨於搜索 |
| 關係 | 搜索後發現應扣押之物，得逕行扣押 | 扣押常為搜索之結果 |
| 客體 | 處所、身體、物件 | 可為證據或得沒收之物（§133） |

### 三、自訴 vs 公訴 ★★

| 比較項目 | 公訴 | 自訴 |
|---------|------|------|
| 追訴主體 | 檢察官 | 犯罪被害人（§319） |
| 起訴方式 | 提起公訴（§264） | 向管轄法院提起自訴（§319） |
| 強制辯護 | 依§31規定 | 自訴須委任律師為代理人（§319 II），否則不合法 |
| 偵查前置 | 經偵查後起訴 | 不經偵查，直接起訴；但已經檢察官開始偵查者不得自訴（§323 I） |
| 撤回 | 檢察官原則上不得撤回起訴 | 自訴人於一審辯論終結前得撤回（§325） |
| 不起訴效力 | 有（§260） | 不適用 |
| 上訴 | 檢察官得上訴 | 自訴人得上訴 |

### 四、緩起訴 vs 緩刑 ★★

| 比較項目 | 緩起訴 §253-1 | 緩刑 刑法§74 |
|---------|-------------|-------------|
| 決定機關 | 檢察官（偵查階段） | 法院（審判階段） |
| 時點 | 偵查終結時 | 判決宣告時 |
| 性質 | 不起訴之替代，附條件暫緩起訴 | 有罪判決之附條件暫緩執行 |
| 期間 | 1年以上3年以下 | 2年以上5年以下 |
| 條件 | 得命向被害人道歉、賠償、支付公益金、義務勞務等（§253-2） | 得命向被害人賠償、義務勞務、完成治療等（刑法§74 II） |
| 撤銷 | §253-3 各款事由（如於期間內故意犯罪） | 刑法§75、§75-1 |
| 撤銷效果 | 檢察官得繼續偵查或起訴 | 撤銷緩刑，執行原宣告之刑 |
| 前科紀錄 | 無 | 有（有罪判決），但期滿刑之宣告失效 |

|---------|------|------|------|

### 六、證據禁止類型整理

| 類型 | 內容 | 條文 |

## 拾壹、答題架構模板

### 刑訴申論題萬用架構

```
一、爭點提出
   「本題涉及......之問題」

二、法律規定
   「按刑事訴訟法第○條規定......」

三、要件涵攝
   「查本題事實......（逐一涵攝各要件）」

四、學說與實務見解（如有爭議）
   「就此，學說上有以下不同見解：
    甲說（實務）：......
    乙說（學說）：......」

五、本題結論
   「綜上所述，本題......有/無證據能力。」
```

### 證據能力題答題 SOP ★

```
步驟一：確定證據種類（自白？證人陳述？物證？文書？）
步驟二：檢驗取得程序有無違法（令狀原則？告知義務？）
步驟三：若為傳聞→檢驗有無例外
步驟四：若有違法→適用§158-4 權衡
步驟五：結論（有/無證據能力）
```

### 關鍵答題提醒

| 題型 | 注意事項 |
|-----|---------|
| 強制處分題 | 先辨別處分類型→檢驗令狀/法律依據→檢驗比例原則 |
| 證據能力題 | 依SOP逐步檢驗，不可跳步 |
| 傳聞法則題 | 先確定陳述場景（法官前/檢察官前/警詢），再對應條文 |
| 上訴救濟題 | 先判斷得否上訴（§376限制），再論不利益變更禁止 |
