## 1.2 機率透鏡的日常

### 配戴者的一天（以一般市民為例）

```
07:00  鬧鐘響。透鏡自動開機。第一個彈出的訊息：
       ┌─ 機率透鏡 v4.2 ─────────────────────┐
       │ 今日天氣預測                          │
       │ 降雨機率：P(rain) = 0.62              │
       │ 分布：Beta(6.2, 3.8)                  │
       │ 建議：帶傘 ██████░░░░ 62%             │
       └────────────────────────────────────────┘
       你看一眼窗外灰色的天空，嘆口氣，拿起傘。

08:30  搭捷運。透鏡自動計算車廂擁擠度。
       「下一班車有座位的機率：0.23。建議等第二班。」
       你選擇擠上這一班。有時候，人類就是喜歡跟機率作對。

12:00  午餐。便當店的菜色旁邊浮現透鏡資訊：
       「此雞腿便當讓你今日攝取超標的機率：0.71」
       「此沙拉便當讓你下午三點餓的機率：0.89」
       你選了雞腿。有些事情機率管不了。

15:30  開會。老闆報告新專案。透鏡背景跑出：
       「此專案如期完成的機率：0.12」
       你跟同事對看一眼。大家的透鏡都算出了同一個數字。
       沒人敢說出來。

19:00  約會。對方是交友軟體配對的。透鏡資訊：
       「此對象與你長期穩定交往的機率：P(stable|data) = 0.34」
       「相性分布：Normal(μ=0.6, σ=0.15)」
       你不知道該不該看。
       你看了。
       對方也看了。
       兩個人都假裝沒看。
       然後尷尬地笑了。

23:00  睡前。透鏡自動切換夜間模式，螢幕變暗。
       最後一條訊息：
       「明日生存機率：0.99987。晚安。」
       你在想，那 0.00013 是什麼。
       然後你決定不想了。
```

```
║                                                                      ║

║    即使數據（data）是真的，結論也會被扭曲。                          ║
║    這就是整本書的核心陰謀。                                          ║
║                                                                      ║
```

## 2.3 校準流程

```

  沉默。
  「但你會去嗎？」蘇立恆問。
  「會。」
  「為什麼？」
  「因為有些事的價值不能用機率衡量。」
  蘇立恆看著她。第一次——真正地——笑了。
  「這是我聽過最不像統計人的話。」
  「這是我說過最像人的話。」

  → 他們的吻（如果有的話）不會有任何機率數字。
  → 因為那是兩個人同時選擇關閉透鏡的瞬間。
  → 小拉在被關閉前最後碎念了一句：
  「心率數據異常......但我覺得這次是好的那種異常......」
```

## 5.2 暗線：高思遠的遺憾

```
高思遠的妻子李月華，五年前因為透鏡焦慮症引發的恐慌症發作，
在過馬路時未注意來車而過世。

諷刺的是——她死前看著透鏡上的數字：
「被車撞的機率：0.0004」
但她正在恐慌發作。她的注意力全在另一個數字上：
「心臟病發作的機率：23.7%」

一個幾乎不可能的事件殺死了她。
而機率——她一直在看的、讓她焦慮的機率——沒有救到她。

高思遠從此不再完全信任透鏡。
但他也不否定透鏡。
「工具沒有善惡。但使用工具的方式有。」

在第三篇，高思遠把妻子的透鏡交給貝葉。
裡面有一段她去世前最後看到的畫面——
那些數字。那些讓她焦慮到看不見真實危險的數字。
「貝葉，這就是為什麼校準很重要。
 不是校準數字——是校準人和機率之間的關係。」
```

# 第六章：核心衝突——全書的哲學主題

---

```
全書探討的核心問題：

  ┌─────────────────────────────────────────────────┐
  │                                                   │
  │  如果你能讓所有人看到「正確」的機率，            │
  │  你有權利「修正」他們看到的機率嗎？              │
  │                                                   │
  │  如果修正後的社會更安全、更幸福、更有效率，      │
  │  那「真實的機率」還重要嗎？                      │
  │                                                   │
  │  如果不確定性本身是人類自由意志的根基，          │
  │  那消除不確定性——是解放，還是囚禁？              │
  │                                                   │
  └─────────────────────────────────────────────────┘

五篇對應的哲學層次：

  第一篇「校準」：真相是什麼？（認識論）
    → 你看到的數字是真的嗎？「真」的定義是什麼？

  第二篇「雜訊」：你能信任誰？（信任論）
    → 當數據可以被操控，你還能信任數據嗎？信任人？

  第三篇「信號」：集體 vs 個人（社會哲學）
    → 一個人的真相，值不值得為了大多數人的安全而犧牲？

  第四篇「檢定」：正義 vs 秩序（法哲學）
    → 法庭上的「統計顯著」等於「道德正確」嗎？

  第五篇「隨機」：自由意志（存在主義）
    → 最終的選擇：完美的機率世界 vs 不確定但自由的世界？

貝葉在最後選擇了——不摧毀透鏡，也不維持操控。
她選擇把「先驗的控制權」還給每個人。
讓每個人可以選擇自己的先驗——
即使那意味著有人會選錯。
「不確定性不是敵人。不確定性是人之所以為人的證明。」
```

# 附錄 B：寫作鐵律

```
1. 故事第一，教學第二。
   如果一個場景「好讀」但沒有教到機率——可以。
   如果一個場景教了機率但「不好讀」——重寫。

2. 機率不是裝飾。每個機率概念都必須跟角色的命運綁定。
   「Poisson 分布」不是知識點。是貝葉發現真相的武器。
   「p-value」不是考試題。是法庭上保護正義的盾牌。

3. 讀者不需要學會公式。但他們會記得——
   「那個女生用 Poisson 分布找出了犯罪數據的假」
   「那場法庭戰用 p-value 扳倒了壞人」
   這就夠了。

4. 透鏡的 ASCII 框是視覺記憶點。善用它。
   但不要每頁都出現。出現在關鍵時刻，才有衝擊力。

5. 角色說「統計術語」時，必須自然。
   不是「讓我來解釋什麼是先驗分布」。
   而是「你的先驗偏了」——像罵人一樣自然。

6. 白奶奶的場景 = 讀者的深呼吸。
   在緊張的情節後放一段白奶奶的蔥油餅。
   讓讀者吸一口氣。然後再把他們拉回深淵。

7. 小拉的碎念 = 調味料。
   不要太多。但沒有的話會覺得少了什麼。

8. 感情線隱藏在數據裡。
   蘇立恆不說「我擔心你」。
   他說「你的操作頻率異常。建議休息。」
   貝葉不說「我想見你」。
   她說「你有空的話......那個數據要不要再看一次？」
   讓讀者自己讀出弦外之音。這比明說更動人。
```
