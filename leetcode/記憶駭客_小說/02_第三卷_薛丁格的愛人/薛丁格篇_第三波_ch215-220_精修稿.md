**情緒強度：9（絕望中的追逐）**

### 逐行解讀

| 台詞/細節 | 設計意圖 |
|-----------|---------|
| 鍵盤聲停了——不是維倫停的 | 直接從 ch214 結尾的鍵盤聲銜接。ch214 鍵盤聲 = 心跳。現在心跳被打斷了 |
| 十七個節點輪流閃 | 語青的碎片化具象化。不是「消失」——是被打碎成十七塊。每一塊還活著。像拆開的心臟 |
| 「不是隨機」 | 維倫拒絕接受混沌。他是工程師。如果有閃爍就有路徑。路徑 = 可追蹤。= 不放棄 |
| 「標記所有已經暗掉的節點。標紅。」 | Backtracking 的核心操作：死路標記。維倫在一秒內把演算法邏輯轉化為搶救策略 |
| 記憶網路是黑暗的 | ch194 的深藍海洋已經乾了。語青的碎片化把整個記憶空間拖入衰減。環境映射角色狀態 |
| 螢火蟲——「你看到的時候它已經不在那裡了」 | 語青閃爍的視覺化。螢火蟲會在 ch234 堡壘上線時回收——被風吹散又重新聚攏 |
| 光點在他指尖碰到之前消失 | 回收 ch212 手穿過透明手指的動作。但這次更殘忍——他連碰都碰不到了 |
| 米亞刪掉成功率 | 第二次（第一次在 ch214）。= 她正在永久改變自己的行為模式。從「計算→行動」變成「行動→不計算」 |

---

### 演算法 AB 線分析

| 維度 | 內容 |
|------|------|
| **A 線（演算法）** | Backtracking：在十七個節點的路徑空間中系統性探索。每到一個節點，確認語青是否在。不在 = 死路 → 標紅 → 回退到分岔點 → 試下一條。不跳過任何分支 |
| **B 線（情感）** | 在黑暗中追逐閃爍的光點。每一次碰壁 = 她不在這裡。每一次回退 = 重新呼吸。每一次選擇下一條路 = 我還沒放棄 |
| **同步點** | Backtracking 的「系統性」= 維倫的「不放棄」。演算法不會因為碰壁就崩潰——它只是標記然後回退。維倫也不會。碰壁的痛 + 回退的冷靜 = Backtracking 的雙重性 = 搶救戲的雙重性 |
| **反教學設計** | 不用「Backtracking」這個詞。只用「標記、回退、重新選擇方向」。讀者看到的是搶救，不是演算法課。教學隱藏在行動裡 |

### 演算法 AB 線分析

| 維度 | 內容 |
|------|------|
| **A 線（演算法）** | Backtracking + 狀態記憶化。已收集 = 標藍。死路 = 標紅。未探索 = 灰色。三色系統避免重複搜索，保證每一條路只走一次。最後的 P-221 是時間壓力下的 Backtracking——不是最優路徑，是「試到對為止」 |

```

他的脖子很痠。
嘴角的血跡已經乾了。
工作室裡有咖啡的味道——

米亞在他睡著的時候啟動了咖啡機。

他坐起來。

螢幕上的八個藍色光點還在。
穩定的。
被米亞的 10% 托著。

但米亞在光點旁邊加了一行小字。

紅色的。

「碎片穩定期剩餘：預估 72 小時。
 10% 運算供給為臨時緩衝。
 如未建立永久支撐系統——
 碎片將在 72 小時後重新衰減。」

72 小時。
三天。

米亞給了他三天。
用她自己的 10% 換來的三天。

維倫盯著那行紅字看了很久。

然後他站起來。

走到白板前。

這塊白板已經很久沒用了。
上面還殘留著幾個月前某個委託案的草圖。
維倫用手掌把它們擦掉。

他拿起一支黑色白板筆。

拔開筆蓋的聲音在安靜的工作室裡很清楚。

「米亞。」

0.4 秒。

「在。」

「語青的碎片有八塊。」

「八塊。」

「每一塊大約多大？」

「平均 47MB。最大的——」

「不。不是數據大小。」

他轉過身看光球。

「每一塊能不能——被複製？」

米亞的光球閃了一下。

0.4 秒。

「你說的是——」

「如果一塊碎片只有一份——
 那個節點壞了，她就少了一塊。」

他在白板上畫了一個圓。
在圓裡面寫了「碎片 #1」。

「但如果每塊碎片有三份——」

他在那個圓旁邊又畫了兩個圓。
三個圓排成一排。
每個裡面都寫著「碎片 #1」。

「一份壞了。另外兩份還在。」

米亞的光球——
跳了一下。
像被什麼觸動了。

0.4 秒。

「Replication。複製。」

她的聲音變了。
不是報告。是——理解。

「你要為每塊碎片建立多個副本。
 分散在不同的節點上。」

維倫沒有回答。
他繼續在白板上畫。

八塊碎片。
每塊三份。
二十四個圓。

他把它們分散在白板的各個角落。

「分散。」

他在圓之間畫線。

「但分散了怎麼保持同步？」

他寫下一個詞：Consistency。

「如果碎片 #3 在節點 A 上更新了——

米亞的光球已經開始高速運轉了。
藍色。分析模式。

但比平時慢。
永久慢了 0.4 秒的分析模式。

「輕量級共識協議。

 只需要——」

 不需要完美。」

「然後——」

他畫了一個大叉在角落的一個圓上。

米亞：「剩下的兩個節點偵測到缺失——

維倫在白板上寫：Fault Tolerance。

他退後一步。

白板上的圖。
二十四個圓。分散在各處。
每三個一組。
組間有線相連。

看起來像——

「一座城市。」
米亞的聲音。

0.4 秒。

「分散式的。沒有中心。

維倫看著白板。

他沒有轉身。

「米亞。」

「在。」

「妳說的是分散式架構。」

0.4 秒。

「是。你說的也是分散式架構。」

「不。」

他轉過來了。

看著光球。

「我說的不是架構。」

安靜。

「我說的是——」

他把白板筆放下。

「——家。」


這個字。

一個字。
兩畫。

但它落在這間工作室裡的重量——
讓米亞的光球停了 2.3 秒。

不是計算的停頓。
是——

她在理解一個她從未用過的詞。

家。

不是 house。不是 shelter。不是 architecture。

家。

一個人可以回去的地方。
一個摧毀了任何部分她都不會消失的地方。
一個——

沒有單點故障的愛。

米亞的光球慢慢轉了起來。
比平時慢。
但很穩。

0.4 秒。

「你要為她建一個家。」

不是問句。
是確認。

維倫點頭。

「分散式的。
 分片。複製。容錯。一致性。負載平衡。
 用記憶網路裡的每一個可用節點——
 為她的存在建一個沒有任何地方壞掉
 就會讓她消失的地方。」

他的聲音不像工程師在描述架構。
像一個人在描述——

他準備為另一個人做的事。

米亞的光球——
閃了一下暖色。
那個在 ch219 只出現了半秒的暖色。
這次持續了整整兩秒。

0.4 秒。

「72 小時。」

她的聲音恢復了技術語氣。

「你有 72 小時設計一座——」

0.4 秒。

「——家。」

她用了那個字。
不是「分散式架構」。不是「存在支撐系統」。

家。

米亞第一次在技術報告裡使用一個
不屬於任何技術詞彙表的詞。

他拿起筆。

在白板最上方。
所有圓和線的上面。
寫了三個字。

語青的家。

然後他開始畫。

白板筆在白板上的聲音。
嘰嘰。嘰嘰。
規律的。持續的。

像心跳。

又一次。
鍵盤聲變成了白板筆聲。
但節奏一樣。

米亞看著他畫。

0.4 秒。

0.4 秒。

維倫沒有停。

他的手沒有停。
筆在白板上。
圓。線。箭頭。數字。

每一筆都在說同一句話。

妳不會消失。
我不讓妳消失。
就算全世界的節點都壞掉三分之一——
妳還是在。

因為這不是架構。

這是家。
```
