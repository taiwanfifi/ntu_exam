# 第一卷→第二卷銜接設計

> **核心挑戰**：讀者合上 Vol 1 時的狀態是「心碎但有希望」。翻開 Vol 2 時不能讓這個情緒斷裂。ch80→ch81 的過渡必須像呼吸一樣自然——吐氣（ch80 的安靜）然後吸氣（ch81 的重新開始）。

### ch81「白衣與雨」的開場設計

**開場第一段**：
- 不要寫「第二卷」的任何提示。讓讀者以為還在 ch80 的延續
- 第一句話從感官開始：雨的聲音、白大衣的布料質感、門關上的聲音
- 維倫的視角：他面前坐著一個陌生女人。她說她的名字叫林語青

**與 ch1 的三重鏡像**：
| ch1 | ch80 | ch81 |
|-----|------|------|
| 語青第一次來 | 語青再次來 | 語青第三次「第一次來」|
| 維倫拒絕 | 維倫說「請進」| 維倫接受委託 |
| 他好奇 | 他的手在發抖 | 手心的痕跡在發燙 |
| 米亞吐槽 | 米亞沉默 | 米亞「什麼都沒說，但光球特別亮」|

**ch81 的新增元素**（區別於 ch1）：
- 手心痕跡發燙 = Vol 2 的「身體記得」升級版
- 0.7 秒記憶盒仍在口袋裡 = 物理證據
- 嚴柏翰不在場 → ch82 或 ch83 才出現 = 讀者的焦慮

## 參、情緒銜接設計

### 讀者的情緒過渡

```
Vol 1 ch80：心碎 + 希望 + 好奇
     ↓（翻頁）
Vol 2 ch81：熟悉（同一個場景！）+ 疼痛（他不記得她）+ 期待（重新開始）
     ↓（3-4 章後）
Vol 2 ch84：接受（新的冒險開始）+ 信任（語青的交易 = 新的合作起點）
```

**關鍵**：ch81-84 不能引入新的大懸念。讓讀者在「熟悉的環境」中適應「不同的關係」。前 4 章的節奏應該像 Vol 1 序篇——慢、安靜、用日常建立新的常態。

## 肆、世界觀銜接

### Vol 1 建立的規則（Vol 2 必須遵守）

1. 記憶修復只能外部操作或深潛。不能「遠程修復」
2. 空白人是不可逆的（記憶完全喪失 = 永久狀態）
3. 記憶銀行的蝴蝶標誌 = 舊標誌鸚鵡的替代品
4. 深淵監控局 = 政府的記憶監控機構
5. 五個社會階層系統
6. 生物共振 = 母女/血緣限定的記憶連接方式
7. 記憶疫苗 = 用自己的記憶為基底穩定他人意識

### Vol 2 新增的規則

1. **深潛有後遺症**（ch92）：鼻血、外來記憶殘留、神經損傷風險
2. **記憶可以跨個體滲透**（ch92）：蜂巢意識的前兆
3. **記憶銀行有七層安全防護**（ch137）
4. **記憶容量有上限**（ch147）：背包問題的物理基礎
5. **記憶覆寫可以被逐層剝離**（ch150）：但需要 Backtracking 的系統性方法
6. **三方勢力**：記憶銀行 vs 深淵監控局 vs 記憶解放陣線
