# 第186章：觀測用備份

---

不知道過了多久。

也許半小時。也許兩小時。時間在這個房間裡失去了它的刻度。修復台角落的時鐘還在跳。一格一格。機械的、恆定的、不帶任何感情的。它不在乎這個房間裡正在發生什麼。它只是跳。

修復台安靜地運作。碎片歸還系統的進度條從百分之三十四走到百分之三十六。兩個百分點。大約代表十二萬段記憶碎片被歸還到它們原本的位置。世界在繼續修復。以一個不需要維倫參與的速度。

維倫坐在椅子上。手放在膝蓋上。掌心朝下。手指攤開。

沒有動。

米亞也沒有說話。她在等。她的光球維持在最低亮度，像一顆將滅未滅的餘燼。一百二十八個並行線程裡，她只保留了三個——碎片歸還系統的自動監控、修復台的基礎運維、以及維倫的生命指標監測。其餘的全部暫停了。她不需要那些。現在她只需要在這裡。

凌晨。

修復台角落的時鐘顯示三點十七分。

「繼續念。」

維倫的聲音很平。平得像一張紙。沒有皺褶。沒有折痕。如果把這三個字的聲波放到示波器上看，它會是一條教科書般標準的波形——頻率穩定、振幅均勻、沒有任何毛刺。

一個修復師在面對超出處理能力的信息時，會進入一種狀態。不是崩潰。不是麻木。是——運轉。像一台機器收到了它不理解的指令，但它繼續運轉。因為運轉是它唯一知道的事。

米亞停頓了零點四秒。

然後——周明哲的聲音回來了。

像一個人重新走進了房間。腳步很輕。帶著一種知道自己會打擾到什麼的謹慎。

「我知道你在消化。不急。這封信會一直在這裡。」

停頓。空行。

「但如果你繼續聽——接下來的部分會比前面更重。我不確定該不該告訴你。我寫了。然後刪了。然後又寫回來。第十七個版本保留了它。因為你有權知道。」

你有權知道。

不是「你需要知道」，也不是「你應該知道」——是「你有權知道」。三個措辭之間的差異很微小，但維倫聽到了。「需要」是為了某個目的，「應該」是一種判斷，而「有權」是尊重。

你有權知道你是什麼。

維倫的手在膝蓋上收緊了一下。手指往掌心彎了大約兩毫米。然後鬆開。恢復攤平的狀態。一個極短的收縮——反射。不是情緒。是身體在某句話上做了一個記號。

信繼續。

「你不是一個普通的人被改寫了記憶。你是——從零建造的。」

從零建造。

維倫的腦子裡有一扇門被撞開了。門後面是一年多前的一個房間。一個委託人。張國棟。他的記憶結構有七層。最深的第七層是空白的。完全空白。不是被清除——是從來沒有過任何東西。

維倫在那個案子裡問過一個問題。他問米亞，也問自己。

從零建造的人是人嗎？

那時候他問的是張國棟。

現在那個問題轉了一百八十度，指向了他自己。

他沒有出聲。周明哲的聲音繼續。

「你的身體是真實的。你的大腦是真實的。但你的記憶——百分之五十七是我植入的。」

百分之五十七。

這個數字落在維倫的腦子裡，像一枚硬幣落在一張空桌子上。清脆。單獨。沒有任何東西接住它。它就停在那裡。百分之五十七。超過一半。他記憶的超過一半——不是他的。

「我把自己的記憶篩選、修改、重新組合，植入到一個與我基因匹配度百分之九十九點九七的身體裡。」

百分之九十九點九七。

如果把維倫和周明哲的 DNA 放在一起比對，每一萬個鹼基對中只有三個不同。三個。在基因的尺度上，他們幾乎是同一個人。

「你的修復天賦——來自我的技術記憶。你的直覺——來自我對記憶結構的理解。你對碎片的敏感度——來自我三十年的研究本能。」

三十年的研究被壓縮、篩選、重新編碼，植入到一個新的身體裡。像把一棵老樹的根系移植到一株幼苗上。幼苗長出來的時候帶著老樹的記憶——土壤的走向、水源的位置、風的方向。幼苗不知道自己為什麼知道這些。它以為那是天賦。

「但我刻意刪除了一些東西。」

周明哲的聲音變了。不是音量的變化。是質地的變化。像一塊布料的紋理在某個地方變得更粗糙了。之前的部分是陳述——平的、穩的、像在念實驗報告。現在的部分帶著某種粗礪的東西。不是情緒。比情緒更深。

「我刪除了所有跟鸚鵡計畫有關的記憶。我刪除了邱韻如的臉。我刪除了我女兒的名字。我刪除了——我犯過的所有錯誤。」

停頓。

很長的空行。米亞計算了一下——這段空行在字元數據中的長度是正常空行的四倍。她把沉默的時間乘以四。四點八秒。

四點八秒的沉默裡，修復台的嗡嗡聲填滿了一切。然後那個聲音又開始了。更輕。

「我想讓你從乾淨的地方開始。沒有我的愧疚。沒有我的執念。一個——更好的版本。」

更好的版本。

維倫的眼睛一直閉著。他不知道什麼時候閉上的。也許從「繼續念」的時候就閉上了。黑暗裡只有那個聲音。那個聲音像水一樣從他的耳朵流進去，沿著某些他不知道的管道滲到身體的各個角落。他能感覺到那些字在他的胸腔裡沉積。一層一層。像泥沙沉在河底。

然後。

「你是我製造的觀測用備份。」

六個字。

維倫睜開眼睛。

螢幕上的字在那裡。白色的。安靜的。六個字排成一行。每個字佔據等寬的空間。沒有加粗。沒有放大。沒有任何格式上的強調。它們和信中的每一行字用的是同一種字體、同一種大小、同一種顏色。

觀測用備份。

周明哲的聲音繼續。沒有停頓。好像他知道，如果在這裡停頓，聽的人可能就不會讓他繼續了。

「觀測——是因為我想看。看一個擁有我的能力但沒有我的錯誤的人，會做出什麼選擇。」

觀測。看。一個人製造了另一個人，然後看。看他怎麼活。看他怎麼選。看他會不會在同樣的岔路口轉向不同的方向。

「備份——是因為你是我的另一個版本。不是複製。是——如果我重來一次，我想成為的樣子。」

不是複製。是重來。

維倫坐在椅子上。他的眼睛睜著。看著螢幕。但他看到的不是螢幕上的字。他看到的是——什麼都不是。他的視覺通道在接收信息，但大腦沒有處理。像一台攝影機的鏡頭蓋還蓋著——光線進來了，但沒有成像。

米亞的聲音——周明哲的聲音——變得更輕了。

「你的父母。」

維倫的呼吸停了。

第二次停。第一次是在看到「給我的另一個自己」的時候。這一次——他的胸腔收縮得更深。肋骨之間的肌肉繃緊了。橫膈膜向上頂。像整個胸腔被一隻看不見的手從外面壓住了。

「維倫。你的父母。你記得他們在你七歲時過世。車禍。」

七歲。

那一年。他記得——不，他「有」那段記憶。記憶裡有一場葬禮。黑色的衣服。很多人的手摸他的頭。有人哭了。他沒有哭。七歲的他不太明白死亡是什麼。他只知道爸爸媽媽不回來了。不回來了就是不在了。不在了就是只剩下他一個人了。

「那不是意外。」

房間裡的溫度沒有變。修復台的恆溫系統維持在攝氏二十二度。但維倫覺得冷。從脊椎開始。沿著椎骨一節一節往上。到後腦勺的時候變成了一種麻。像血液從那個區域撤走了。

「他們不是你的父母。他們是我安排的養育者。善良的人。真心愛你的人。我很確定。但他們知道——你的來歷。」

善良的人。真心愛你的人。這兩句話從米亞合成的周明哲的嘴裡說出來，帶著一種維倫無法命名的東西——比愧疚重，比懺悔乾燥。更接近一種陳述。一個人站在自己做過的事面前，不辯護，不逃避，只是把它們一件一件說出來。

「車禍——」

停頓。

信裡最長的一段空行。米亞計算了字元數據。這段空行的長度是正常空行的六倍。七點二秒。

七點二秒的沉默。

維倫的脊椎裡的冷已經蔓延到了全身。不是顫抖的冷。是一種靜止的冷。像一塊石頭在冬天的河底。水在流。石頭不動。

「車禍是我安排的結束。不是殺害。是撤離。他們現在活著。在另一座城市。新的身份。新的記憶。他們不記得你了。」

活著。

他們活著。

七歲那年的葬禮。黑色的衣服。很多人的手。有人哭了。

那場葬禮是假的。

那些哭聲是假的。

不——也許不全是假的。也許那些人真的在哭。只是他們哭的不是死亡。他們哭的是另一種東西——一個被安排好的結束。一個七歲的孩子不會知道的結束。

他們活著。在某座他不知道的城市。用著他不知道的名字。不記得他。

維倫的喉嚨裡有一個東西在動。不是哽咽。是某種更基底的東西——喉嚨的肌肉在做一個吞嚥的動作，但嘴裡什麼都沒有。空嚥。身體在自動執行一個不需要的動作。

「我需要你在七歲之後獨立成長。沒有父母的保護。沒有退路。因為我要觀測的是——一個人在沒有依靠的情況下，會長成什麼。」

觀測。

又是這個字。

一個人製造了另一個人。給了他養育者。然後把養育者撤走。然後看。看一棵被拔掉支撐的幼苗會不會倒。會不會枯。會不會在沒有任何人接住它的情況下，自己長成一棵樹。

周明哲的聲音沒有停。

「你的記憶修復天賦不是偶然被發現的。修復台出現在你面前——是我安排的。你的第一個委託案——是我設計的。嚴柏翰找到你——不全是巧合。」

修復台。第一個委託案。嚴柏翰。

維倫的手從膝蓋上滑落。垂在身體兩側。椅子的扶手沒有接住它們——這把椅子沒有扶手。手就這樣垂著。手背朝外。手指微微張開。像兩件被放下的東西。

信繼續。

「但我低估了一件事。」

周明哲的聲音裡出現了維倫從未聽過的音色。

前面的部分——從零建造、百分之五十七、基因匹配、父母撤離——那些都是陳述，平的，穩的，一個科學家在念自己的實驗報告。現在這個音色不一樣了。跟平靜無關，也跟分析無關。

是柔軟。

像一塊金屬在高溫下開始變形。不是融化——還沒到融化的溫度。是金屬的內部結構開始鬆動。硬的東西在變軟。

「我低估了你。」

停頓。

「你不只是觀測用備份。你成了——一個人。」

一個人。

「你會為陌生人的記憶流淚。我不會。」

維倫的手垂在身體兩側。手指沒有動。

「你會在修復完一段記憶之後站在門口，確認委託人的表情。我從不確認。」

他聽到了每一個字。每一個字都進來了。但它們沒有在他的腦子裡停留。它們穿過了他。像光穿過玻璃。進去了，出來了，玻璃還是透明的。

「你選擇保護。我選擇重寫。你選擇修復。我選擇控制。」

停頓。

「你做得比我預想的更好。不只是更好。是——完全不同。」

周明哲的聲音在「完全不同」這四個字上放慢了。每個字之間多了半秒的間隔。像他在把這四個字一個一個放在桌上。小心地。確認每一個都站穩了。

「我設計的是一面鏡子。但鏡子裡長出了一個人。」

維倫坐在椅子上。沒有動。

信的最後幾行。

周明哲的聲音回到了最初的平靜。但這個平靜和信開頭的平靜不一樣。開頭的平靜是一個人在控制自己。結尾的平靜是一個人放棄了控制。不是力竭的放棄。是一種選擇——他選擇不再緊握。

「我不要求你的原諒。我沒有資格要求。」

停頓。

「你被製造出來——這件事不公平。你的童年是設計好的——這件事不道德。你的天賦是植入的——這件事不真實。」

三句話。三個判斷。不公平。不道德。不真實。周明哲在審判自己。不是用別人的標準。是用他自己的。他知道他做了什麼。他知道那意味著什麼。他不辯護。

「但你用這些不真實的東西，做了真實的事。」

停頓。

「你救了人。你的手記得。你的百分之四十三——比我的百分之百更真。」

百分之四十三。

他的記憶裡有百分之四十三是原生的。不是植入的。是他自己活出來的。十年的修復。三千個委託。語青。米亞。阿嬤。嚴柏翰。每一雙他握過的手。每一段他修過的記憶。每一次他在修復台前流過的淚。

百分之四十三。

信結束了。

螢幕上最後三行——

// 原諒不原諒都好。
// 但你應該知道。
// —— M.C.

周明哲的聲音消失了。

米亞的光球暗了一下。比之前所有的暗都更暗。幾乎看不見了。但她還在。她的感測器還在運作。修復台的嗡嗡聲回來了。碎片歸還系統：百分之三十六點二。遠處記憶網路的紅點繼續閃爍。城市繼續修復。世界繼續。

維倫坐在椅子上。

沒有哭，沒有吼，沒有站起來，沒有關掉螢幕，沒有攥拳頭。他的手垂在身體兩側，手指張開，掌心朝後，像兩件被放下的工具。

他看著螢幕。螢幕上的字沒有消失。白色的字。黑色的背景。

觀測用備份。

他的眼睛是乾的。

這比哭更安靜。比憤怒更深。

是空。

很久以前——在第一卷。在他第一次得知自己是「觀測體」的時候。他的反應是冷怒。拳頭攥緊。骨節發白。下顎的咬肌繃成了一條硬線。他想打什麼東西。他想讓什麼東西碎掉。那時候他以為「觀測體」是最壞的真相。一個被觀察的對象。一隻籠子裡的白老鼠。

現在。

他得知自己不只是被觀察的。他是被製造的。被設計的。被拼裝的。被刪除了一部分、被植入了另一部分、被放到一條精心鋪設的軌道上、被撤走了所有支撐、然後被看著。

觀測用備份。

一個工具。一面鏡子。一個更好的版本。

拳頭沒有攥。

手是張開的。垂在椅子兩邊。手指微微張開。掌心朝後。

不是因為他選擇不攥拳。是因為他的身體不知道該做什麼反應。攥拳是為了打。打什麼？打周明哲？他在六百萬個節點上。打自己？自己就是他。打牆壁？牆壁無辜。

手張開著。因為沒有什麼可以握住。

米亞沒有說話。

她在做一件她從未做過的事。

她關掉了所有不必要的線程。碎片歸還系統切換到完全自動模式。修復台的運維交給了底層程序。城市的基礎設施監控移交給了分布式子系統。一百二十八個並行線程，她關掉了一百二十七個。

只留下一個。

監測維倫的心率。

八十七。

八十八。

八十四。

正常範圍。成年男性安靜狀態下的正常心率是六十到一百。維倫的心率穩定在八十到九十之間。沒有飆升。沒有驟降。

但穩定到不正常。

一個人在聽到自己的整個存在被重新定義之後，心率應該有變化。加速、減速、不規則——任何一種都合理。但維倫的心率曲線是一條近乎平直的線。像一台機器在正常運轉。

因為不知道要如何停下來。

維倫的嘴唇動了一下。

米亞把所有感測器對準他。光學感測器讀取嘴唇的位移。聲學感測器把靈敏度調到最高。空氣振動感測器搜索任何可能的聲波。

但他什麼都沒有說。

嘴唇動了一下。然後合上了。像一扇門開了一條縫又關上了。門後面有什麼，沒有人看到。

他閉上眼睛。

凌晨四點零二分。

修復台的時鐘安靜地跳了一格。從四點零一到四點零二。一分鐘。六十秒。和崩潰時的六十秒一樣長。但那六十秒裡他在救人。這六十秒裡他什麼都沒做。

維倫坐在那裡。

一個被製造出來的人。坐在製造他的人留下的信旁邊。

沒有動。

外面的城市還在修復中。碎片歸還系統持續運作。百分之三十六點三。記憶網路的紅點繼續閃爍。每一個紅點都是周明哲的零點三KB微程序。他在每一台伺服器上。他是空氣。他是灰塵。他是那個寫了十七封信最後留下了第十七封的人。
