## 參、雙峰結構分析

### 第一峰：ch35 茉莉花崩潰

```
    ch31 ─── ch33 ─── ch35 ★
      ↗         ↗        ↘
   對峙 →    揭露 →    崩潰 → ch37 重建 → ch38 ♥ 呼吸
```

- **上坡段**（ch31-34）：嚴柏翰從冷硬對峙 → 合作 → 脫下面具 → 被防禦擋住
- **頂峰**（ch35）：茉莉花＋「我的手記得」＋嚴柏翰哭泣
- **下坡段**（ch36-37）：照片牆確認 → 身份崩塌重建
- **呼吸段**（ch38）：蛋炒飯。用溫暖消化衝擊

**寫法關鍵**：ch35 前必須有 3-4 章的張力蓄積。不能跳過 ch33-34 直接到茉莉花——讀者需要時間與嚴柏翰建立連結，否則他的哭泣只是「角色在哭」，不是「讀者跟著碎」。

### 第二峰：ch46 三十四萬七千人

```
    ch39 ─── ch43 ─── ch46 ★
      ↘         ↗        ↘
   黑暗 →    危機 →    攤牌 → ch47 ♥ 月光 → ch48 崩塌
```

- **下沉段**（ch39-42）：深夢市場 → 影子 → 驚悚 → 道德錨定
- **危機段**（ch43-45）：清潔工入侵 → 影子失蹤 → 指數增長
- **頂峰**（ch46）：347,000 + 語青半揭露 + 維倫握手 + 十九天倒數
- **呼吸段**（ch47）：屋頂月光。用浪漫消化末日感
- **崩塌段**（ch48）：「所有案子都是測試」—— 信任裂縫

**寫法關鍵**：ch46 的力量來自數字（347,000）和語青的「假揭露」。語青說的每一句都是真的——但她隱藏了 10%。讀者在此處會鬆一口氣，為 ch75 的真正炸彈蓄力。

### 呼吸段設計規則

1. **呼吸段不是「沒事發生」**——是「用安靜的方式推進關係和主題」
2. **呼吸段必須有一句話讓讀者記住**：
   - ch38：「在所有的時間線裡——你從來沒有教過我做飯。」
   - ch47：「不管接下來發生什麼——記住今晚。」
3. **呼吸段之後必須急轉**：ch38→ch39（深夢黑市）；ch47→ch48（信任崩塌）

### 品質檢查結論

**3/3 全部達標**。三個日常章節形成了完美的遞進序列：

1. **ch38** 的力量在於「第一次」——第一次一起吃飯、第一次嚴柏翰放下防備、第一次語青做「在所有時間線都沒做過」的事。「第一次」= 這條時間線的獨特價值。

2. **ch47** 的力量在於「克制」——描輪廓不碰觸、問了不回答、「今晚是真的」暗示明天可能不是。美在「差一點」。

3. **ch57.5** 的力量在於「意外」——阿嬤不知道怎麼找到他們、鋼琴走音但旋律清晰、米亞不知道「放鬆」是什麼但她存下了這段。最安靜的場景有最大的後勁。

**唯一建議**：ch57.5 的阿嬤出場需要一個合理的動線說明（她怎麼知道廢棄實驗室的位置？）。可以用一句話帶過：「她說她每天都會走到這條路，不知道為什麼。身體記得路。」——這樣既解決了邏輯問題，又強化了「身體記得」的母題。
