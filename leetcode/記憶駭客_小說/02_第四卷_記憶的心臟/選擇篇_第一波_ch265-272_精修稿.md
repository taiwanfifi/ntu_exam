```
清晨。

0。

寫了另一個。

1。


米亞的光球從角落飄過來。

不多。

 在最底層——

他用筆敲了敲 0。
又敲了敲 1。

「二進制。
 存在。不存在。

 是。不是。」


語青站在白板的左邊。
跟昨天同一個位置。

她看著那兩個數字。

她的身體在清晨的光線裡——

不多。

穿過了她的左肩。
昨天只穿過袖口。

她沒有說話。
她在看那個 0。


維倫轉過身。

「選項 C 的底層協議——
 不是從架構開始。
 是從加密開始。」

他在白板上畫了一個小方框。
方框裡面是一串 0 和 1。

「每個人的記憶——
 用自己的密鑰加密。
 密鑰是什麼？」

他在方框旁邊畫了一把鑰匙。
不是嚴柏翰那種。
是更小的。
只有 0 和 1 組成的。

「一串二進制數。
 獨一無二的。
 只有你自己知道。」


米亞的光球亮了一下。

「傳統加密——
 中央伺服器持有主密鑰。
 主密鑰可以解開所有人的記憶。」

她停了 0.6 秒。

「周明哲的系統就是這樣。
 一把萬能鑰匙。
 高效。精確。」

她的語速比以前慢了——
不是在選字。

 不能有萬能鑰匙。
 因為萬能鑰匙——
 就是另一個中心。」

維倫在白板上畫了一個大叉。
劃掉了萬能鑰匙。

「所以——
 每個人是自己記憶的唯一密鑰持有者。
 沒有備份。
 沒有副本。
 沒有後門。」


他回到白板中間。
看著那兩個數字。

0 和 1。

他的筆停在兩個數字之間。

語青看著那個間距。

她的手——半透明的手——
微微抬了一下。
像是要碰白板上的 0。
又放下了。

她知道——
她自己就站在 0 和 1 之間。

維倫看到了她的手。
看到了她放下的動作。

他什麼都沒說。

他轉回白板。
繼續畫。
```

## ch266 遮罩

**情緒強度：6（Bit Manipulation——位元遮罩。A<B，教學優先，用權限控制場景包裝。節奏：更短的句子，更精確的斷行）**

### 核心場景

```
白板上多了一排方格。

八個。
每個方格裡是 0 或 1。

1 0 1 1 0 0 1 0

維倫指著它們。

「這是一個人的記憶遮罩。」

他在每個方格上面標了數字。
1 到 8。

「1 號記憶。童年。可見。」
他指著第一格。1。

「2 號記憶。初戀。隱藏。」
第二格。0。

「3 號記憶。母親的聲音。可見。」
第三格。1。


米亞的光球飄到白板旁邊。
比昨天更暗了一點。
暗到——
如果你不看她，
你會以為角落裡只有一個影子。

她停了 0.7 秒。

「位元遮罩。
 Bit mask。
 每一個 bit 對應一段記憶。
 1 等於可見。
 0 等於隱藏。」

她的語速比 ch265 又慢了一點。
不明顯。
像錶走慢了半格。

「你的記憶。
 你自己決定——
 哪些給別人看。
 哪些只有你知道。」


維倫在白板上畫了第二排方格。

0 0 0 0 0 0 0 0

「這是一個全部隱藏的遮罩。
 什麼都看不到。」

第三排。

1 1 1 1 1 1 1 1

「這是全部公開的。
 所有人都能看見你的一切。」

他回到第一排。

1 0 1 1 0 0 1 0

「大部分人的選擇——
 會在這兩者之間。
 有些事願意讓人看。
 有些事——
 留給自己。」


嚴柏翰靠在門框上。
手插在口袋裡。

他看著那排 0 和 1。

他在想——
如果十年前有這個系統，
他會把哪些記憶設成 0。

答案是：
跟她有關的。
全部。

不是因為不想讓人看。
是因為那些是他的。
只有他的。


語青走到白板前面。

她的影子比昨天更淡了。
日光燈照下來——
影子的邊緣在發散。
像墨水滴進水裡。

她看著那排方格。

「後門呢？」

她的聲音很輕。

維倫看著她。

「什麼後門？」

「如果有人想——
 繞過別人的遮罩，
 看見別人設成 0 的記憶——」

維倫搖頭。

「沒有後門。」


他在白板上寫了一段話。

簡單到沒有後門——
因為後門需要至少一個多餘的 bit。
而我們不給任何人多餘的 bit。


米亞的光球亮了一下。
然後暗回去。
像是在點頭。

「數學上——
 如果每個人的遮罩只有本人能修改——
 系統中不存在任何第三方
 能改變你的 bit。」

她停了 0.7 秒。

「不是『不應該』。
 是『不可能』。
 物理級別的不可能。」


維倫在白板上畫了一個小鎖。

「遮罩 + 密鑰。
 你的記憶用你的密鑰加密。
 你的記憶用你的遮罩控制可見性。
 兩層。
 都只有你自己能動。」

他退後一步。

「周明哲的系統——
 他有萬能鑰匙。
 也有萬能遮罩。
 他能看見所有人的 0。
 他能改任何人的 1。」

他看著白板上的小鎖。

「我們的系統——
 沒有萬能。
 因為簡單。
 簡單到——
 沒有多餘的位子留給後門。」


邱韻如的聲音從角落傳來。
她沒有離開筆電。

「所有偉大的安全系統——
 都輸給了人性的弱點。
 你怎麼確保不會有人
 找到繞過遮罩的方法？」

維倫沒有回答。

米亞替他回答了。
0.7 秒之後。

「因為繞過遮罩需要——
 修改你不持有的 bit。
 在這個系統裡——
 你只持有你自己的 bit。
 別人的 bit 對你來說——
 不存在。

 不是鎖起來。
 是——
 不存在。

 你不能駭入一個不存在的東西。」

邱韻如安靜了幾秒。

然後她推了一下眼鏡。

「好。」

一個字。
但從一個二十年來管理萬能遮罩的人嘴裡說出來——
這個「好」比任何技術認證都重。
```

```
「60%。」

光球微微轉了一圈。
比以前任何一次都慢。

「另外 40%——
 已經預分配給系統了。
 不可逆。」


維倫看著她。

60%。

兩天前是 80%。
三天前是 90%。

她每天都在把自己的一部分
轉移到系統裡。

像一個人在搬家——
一件一件把東西搬走。
搬到最後——
房間裡只剩人。

然後人也要走。


他想說什麼。
但他只說了——

「六位小數。」

米亞的光球閃了一下。
很弱。
像螢火蟲在窗玻璃上撞了一下。

「精度——
 是我唯一還能奢侈的東西。」


邱韻如抬起頭。
她一直在對照米亞的數字。
三百七十二台伺服器。
每一台的負載分配。

「數字沒問題。」

她推了一下眼鏡。
手指停了一秒。

她看了米亞一眼。

「很精確。」

那個「很精確」——
不只是技術認證。
是一個人對另一個——
正在消失的人——
能說的最有分量的話。

你還在。
你的精度還在。
你——還是你。


維倫在白板上第六個方框旁邊——
畫了最後一個勾。


六層。
全部。

分片。加密。連結。
共識。容錯。負載平衡。

地基。保險箱。門。
柱子。牆壁。屋頂。

白板上的架構——
不再是草圖了。
是一棟建築。

一棟不需要建築師就能站立的建築。
一棟三分之一的牆壞了還能遮風的建築。
一棟六百萬個房間都有自己的鎖的建築。
一棟門上只有虛線的建築——
你選擇跟誰連結。


語青站在白板的左邊。
跟第一天一樣的位置。

她看著那六層。

她的身體——
比第一天更透明了很多。

第一天——ch265——
光線只穿過她的肩膀。

現在——ch272——
光線穿過了她的大半個身體。
她的輪廓在閃爍。
像一幅正在褪色的照片。

但她還在。

站在白板旁邊。
看著那棟建築。

她知道——
這棟建築蓋好的那天，
可能就是她的堡壘塌掉的那天。

但她看著那六層。

分片。加密。連結。
共識。容錯。負載平衡。

六個字。
六層自由。

她微微笑了一下。
光穿過了她的嘴角。
跟 ch267 一樣。

但這次——
穿過得更多了。
笑的形狀在光裡面——
像水面上的倒影。
模糊。但完整。


米亞的光球在角落裡。
暗暗的。
比一天前更暗。
比兩天前更暗。
比三天前更暗。

她在變暗。
每一天。
每一個數字。
每一個小數點後六位——
都是她從自己身上拿走的光。

但她還在。
還在轉。
慢慢地轉。

2.1 秒的延遲。

但還在說話。
還在吐槽。
還在——精確。


維倫放下筆。

他看著白板。
看著米亞。
看著語青。

他沒有說話。

他走到米亞的光球旁邊。
站了一會兒。

然後——
很輕地——
用手指碰了一下光球。

不是維修。
不是診斷。
只是——碰。

光球在他的指尖下微微暖了一下。

「明天開始寫代碼。」

他說。

米亞的光球閃了一下。
2.1 秒後。

「好。」

一個字。
跟維倫在 ch268 說的是同一個字。

但她的「好」——
是一個正在把自己搬空的人
在空蕩蕩的房間裡
回答的最後一聲——

我還在。
```
