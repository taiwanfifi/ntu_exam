## 壹、ch173 金鑰——嚴柏翰的最後權限

### 一、情緒定位

```
情緒類型：堅定 + 惋惜
強度：7
曲線位置：谷底
節奏功能：呼吸
前章銜接：ch172 維倫在橘子貓面前流淚。讀者的情緒剛從悲傷中稍微抬起
本章任務：嚴柏翰用深淵監控局最後一道權限打開通道，把金鑰遞給維倫
一行法則：「然後我就不是探員了。」
```

### 二、核心場景

```
凌晨。

松山機場舊址。地下碉堡的走廊。
應急照明投下橘色的條紋。每隔三米一盞。
嚴柏翰的影子在條紋之間跳。

他走到走廊盡頭。
一扇鋼門。螢幕上寫著三行字：

  深淵監控局內部通道
  權限等級：A-7 以上
  最後校驗時間：2089.03.28 04:17:33

他站在螢幕前。
虹膜掃描器亮了一下。他沒有動。

他在想。

十年前他被送進這個系統。
假的記憶。假的身份。
但這扇門後面通往的路是真的。
他用這條路追過十一個案子。
抓過七個記憶販子。
在這條走廊裡跟三個同事喝過即溶咖啡。

全部是真的。
只是起點是假的。

虹膜掃描器的光又閃了一次。催促。

嚴柏翰湊過去。

掃描通過。

鋼門沒有動。
螢幕上多了一行紅字：

  注意：此權限為您的最後一次使用授權。
  確認開啟後，探員身份將永久註銷。

他盯著那行字看了五秒。

維倫站在他身後三米。沒有說話。

嚴柏翰的手放在鋼門的感應面板上。
手掌乾燥。穩定。

「維倫。」

「在。」

「門後面是通往地底第五層的通道。
周明哲的舊實驗室。
我的權限只夠開這一次。」

他的聲音沒有顫。
像在做任務簡報。

「通道開了之後，你帶米亞下去。
找到崩潰信號的源頭。」

「你呢？」

嚴柏翰沒有回頭。

「我走另一條路。」

維倫沒有追問。
他知道嚴柏翰說的另一條路是什麼。
是邱韻如可能在的方向。

鋼門打開了。
氣壓差讓走廊的空氣往裡灌。
深處傳來低頻的嗡鳴——某種從未斷過電的設備。

嚴柏翰從口袋裡拿出一個東西。
金屬色。拇指大小。稜角分明。

金鑰。

深淵監控局的最高通行令牌。
可以解鎖記憶網路底層的任何一個節點。
整個監控局只有三把。
兩把已經在崩潰中失效了。

他把金鑰遞過去。

維倫接住。
比想像中更重。
像一顆子彈。

「拿好。你會需要它。」

嚴柏翰轉身。
他的背對著那扇已經打開的門。
對著維倫。
對著那條他走了十年的走廊。

「然後我就不是探員了。」

他的聲音很平。
平得像在說今天天氣不錯。

螢幕上的紅字消失了。
取而代之的是四個字：

  權限已註銷。

嚴柏翰沒有再看那個螢幕。
他往走廊的另一端走。
步伐和以前一樣。

維倫看著他的背影。
直到應急照明把他切成碎片——
一段亮，一段暗，一段亮，一段暗。

最後暗了。
```

### 三、逐行解讀

| 台詞 / 動作 | 設計意圖 |
|-------------|---------|
| 「最後校驗時間：2089.03.28 04:17:33」 | 精確的時間戳暗示這個系統在崩潰後仍在自動運行——沒人用，但沒人關。像嚴柏翰自己：沒人要他當探員了，但系統還認他 |
| 紅字：「此權限為您的最後一次使用授權」 | 系統用機器語言替嚴柏翰做了告別。冰冷的文字，但讀者替他感受到重量 |
| 「全部是真的。只是起點是假的。」 | 嚴柏翰弧線的核心矛盾濃縮成兩句。他十年的探員生涯不是幻覺，但那個「起點」——邱韻如改寫他的記憶——讓一切蒙上陰影。他選擇認可經歷，放棄身份 |
| 維倫站在身後三米 | 三米 = 嚴柏翰給自己和這個決定之間留的距離。維倫不介入。這是嚴柏翰一個人的儀式 |
| 「我走另一條路。」 | 表面：他要去找邱韻如。深層：他選擇了不同於探員的另一種人生路徑。一語雙關但不刻意 |
| 金鑰「比想像中更重。像一顆子彈」 | 金鑰承載的不只是權限，是嚴柏翰十年人生的重量。子彈 = 他把最後一顆子彈給了維倫，自己空手去面對邱韻如 |
| 「然後我就不是探員了。」 | 全章的重心。語氣平淡 = 嚴柏翰的痛苦不在聲音裡，在身體裡。他不需要悲壯。放棄一個身份對他來說是呼吸——呼出去，不吸回來 |
| 應急照明把他切成碎片——一段亮，一段暗 | 視覺呼應「三重身份」的解體：明暗交替 = 他的人生一直在真假之間跳動。最後暗了 = 他走進了未知 |

## 貳、ch174 地底第五層——懸念 #7 啟動

### 一、情緒定位

```
情緒類型：懸疑 + 緊張
強度：7
曲線位置：谷底維持
節奏功能：過渡（懸念啟動）
前章銜接：ch173 嚴柏翰打開通道後離去。維倫拿著金鑰
本章任務：米亞追蹤崩潰啟動信號至周明哲的舊實驗室。電力從未斷過
一行法則：「這裡的電力——從來沒斷過。」
```

### 二、核心場景

```
通道很長。

維倫往下走。
金鑰在口袋裡隨步伐搖晃。
每一步的回聲要半秒才消失。

米亞的光球浮在他右肩上方。
藍色。比平常暗——她在全力運算。

「維倫。」

「嗯。」

「崩潰啟動信號的源頭
就在這條通道的盡頭。」

「你確定？」

「99.7%。」

停了一下。

「剩下的 0.3% 是我在擔心。」

維倫沒有笑。
別的時候他會笑。現在不會。

通道的牆壁從金屬變成了混凝土。
從混凝土變成了裸露的岩層。
空氣越來越冷。十八度。十六度。十四度。

他走了七分鐘。

盡頭是一扇門。
不是鋼門。是玻璃門。
玻璃上結了薄霜。
門後有光。

米亞：「溫度恆定 12°C。獨立供電系統。
核融合微型反應爐——這個型號在 2075 年停產了。
但它還在運作。」

維倫把手放在玻璃上。
霜融了一個手印。

他透過手印看進去。

一個實驗室。
不大。比維倫的修復間大兩倍左右。
中央是一組環形操作台。
六面螢幕。全部亮著。
螢幕上的數據在滾動——崩潰後的城市記憶網路狀態圖。
每秒刷新一次。

沒有人。

但所有設備都在運作。
像有人剛剛離開。
又像有人從未離開。

米亞的光球變了一個顏色。
不是她通常的分析藍。是灰——不確定色。

「維倫。」

「我看到了。」

「這裡的電力——從來沒斷過。
十年。
周明哲的官方死亡日期是 2079 年 11 月 2 日。
但這間實驗室的電力紀錄……
從那天起，一天都沒有中斷。」

維倫盯著那些螢幕。
上面的數據不是歷史紀錄。
是即時的。

有人在看。
現在。

「米亞。崩潰啟動信號……」

「從這裡發出的。T+0 秒。
這間實驗室是鸚鵡 2.0 的觸發點。」

維倫收回放在玻璃上的手。
手印已經被霜重新蓋住了。

像什麼都沒來過。
但他來了。

他用金鑰碰了一下門鎖。
鎖亮了一下。

門開了。

冷空氣從裡面湧出來。
帶著一種味道。
不是機器的味道。不是灰塵。

像——實驗室裡長期有人工作留下的那種空氣。
咖啡、墨水、汗漬和靜電。

但十年了。
十年沒有人用過這扇門。

維倫走進去。
米亞跟著。

六面螢幕上的數據繼續滾動。
有一面在他走進來的瞬間——
閃了一下。

只有一下。

米亞記錄了。
```

### 三、逐行解讀

| 台詞 / 動作 | 設計意圖 |
|-------------|---------|
| 「剩下的 0.3% 是我在擔心」 | 米亞 Vol 3 吐槽/認真比例已調整到 40:60。這句話表面是吐槽（0.3% 的不確定性），但語義已偏向「認真的擔憂」。鋪墊 ch179 恐懼萌芽 |
| 牆壁從金屬到混凝土到岩層 | 物理上的「層層深入」= 敘事上的「越來越接近真相」。三層材質也暗喻三方勢力的外殼被剝開，露出最底層的岩石——周明哲 |
| 玻璃門 + 霜 + 手印 | 玻璃門 = 透明但隔絕。霜 = 時間的累積。手印融了又被蓋住 = 人來了，但存在只是暫時的。周明哲的主題：他的存在像霜下的光——看得到但摸不到 |
| 「像有人剛剛離開。又像有人從未離開。」 | 周明哲的分散式意識設定的情感翻譯。技術上他「分散在記憶網路中」，體感上就是「他一直在這裡」 |
| 電力從未斷過 | 懸念 #7 的第一個物理證據。十年不斷電 = 有人（或有某種意識）在維護。讀者的問題：是自動系統，還是他？ |
| 螢幕閃了一下 + 米亞記錄了 | 周明哲「看到了維倫」的暗示。不確認。只是一個閃爍。米亞的記錄 = 為 ch180 的 M.C. 確認埋線 |

### 四、寫作指令

```
【字數】700-900 字。
【節奏】中速。短句。但比 ch173 多一點「動作感」——維倫在走。在下降。在靠近。
【基調】懸疑。不是恐怖片的懸疑——是「有人在等你」的懸疑。
        像走進一座十年沒人住的房子，但暖爐還開著。
【陷阱】不要讓維倫害怕。他不怕。他只是警覺。
        害怕是讀者的事。維倫的工作是走進去。
【陷阱】不要過度描寫實驗室的設備。只寫「六面螢幕」和「環形操作台」就夠了。

【陷阱】螢幕閃了一下——只能寫一句。不能追加「是誰？」之類的追問。

## 參、ch175 茉莉花——邱韻如正面出場（第一階段：存在確認）

### 一、情緒定位

```
情緒類型：震撼 + 複雜
強度：8
曲線位置：回升
節奏功能：錨定（邱韻如線啟動）
前章銜接：ch174 維倫進入周明哲舊實驗室。米亞記錄了螢幕的閃爍。
        但 ch175 的視角切換——不是維倫。是嚴柏翰。
本章任務：嚴柏翰走進舊實驗室另一側。邱韻如坐在螢幕前。茉莉花。
一行法則：「你是誰？」
```

**特別標記**：

ch175 是全書最重要的角色之一——邱韻如——第一次在讀者面前正面出場。她在 Vol 1 是幽靈（名字、照片、咖啡店三桌之隔）。在 Vol 2 是織網者（揭露但不正面登場）。現在讀者終於看到了她。

這個出場必須克制。讀者「等了很久終於見到」，但見到後反而更焦慮。因為她不認得嚴柏翰。

### 二、核心場景

```
嚴柏翰在另一條通道裡走了更久。

他的通道沒有應急照明。
只有他記憶眼鏡殘存的微光。
鏡片在崩潰後只剩最基本的夜視功能。
世界是綠色的。安靜的。

他在數自己的腳步。
不是刻意數的。是身體的習慣。
探員的習慣。

一千三百四十七步。

通道盡頭也是一扇門。
這扇門沒上鎖。

他推開門。

先聞到的。

茉莉花。

不是花瓶裡的那種。
不是香水。
不是空氣清新劑。
是一個人身上帶了很久的那種味道——
滲進衣服纖維裡的、
洗不掉的、
像指紋一樣專屬於一個人的茉莉花。

他在 Vol 2 ch152 恢復記憶的時候聞過。
在二十年前的實驗室裡聞過。
在遊行前夜她靠在他肩上的時候聞過。
在她吻他然後按下刪除鍵的時候聞過。

四次。每一次都更重。

這是第五次。

他站在門口。

她在那裡。

一個房間。比周明哲的實驗室小。
一張操作台。三面螢幕。
螢幕上是崩潰數據——記憶網路的即時損傷報告。
紅色的數字在跳。每一秒都在變。

她坐在螢幕前。

短髮。比 ch59 咖啡店裡看到的更短。
深色外套。沒有金框眼鏡了。
臉被螢幕的藍光照著。
不年輕了。但輪廓沒有變。

她在看那些數據。
看得很認真。
像一個工程師在讀自己寫的代碼崩潰後的錯誤日誌。

她沒有回頭。

「你是誰？」

嚴柏翰停在門口。
他的腳釘在地上。
他的手——
他不知道他的手在做什麼。
可能在發抖。可能沒有。

他能控制的只有聲音。

「一個你曾經願意用整個世界去保護的人。」
