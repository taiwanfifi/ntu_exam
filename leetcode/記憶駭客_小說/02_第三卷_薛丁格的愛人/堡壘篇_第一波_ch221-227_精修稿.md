### AB線分析

| A線（技術） | B線（情感） |
|---|---|
| Sharding：將語青的意識分為 800 個分片 | 把一個人拆開來，每一塊都好好保存 |
| Replication：每片 3 副本（情緒片 5 副本） | 情緒記得最多，所以備份最多份 |
| 一致性哈希分配 | 每段記憶都有自己該去的位置 |
| 分片鍵 = 感官類型 | 用五感分類靈魂 |
| 總節點 ~3,000 | 三千個地方同時記住一個人 |

**技術真實性筆記**：分片（Sharding）與複製（Replication）是分散式系統的兩大基石。分片解決容量問題——單一伺服器存不下全部資料，所以拆開來分散存放。複製解決可靠性問題——單一副本會因硬體故障丟失，所以每塊資料存多份。維倫用感官類型當分片鍵是故事虛構，但「按業務邏輯選擇分片鍵」是真實的 System Design 最佳實踐。情緒記憶副本數提高到五份——就像真實系統中對核心資料提高冗餘度。

### 寫作指令

- **視角**：第三人稱，維倫主導。嚴柏翰從門口走到白板前的過程 = 從旁觀者到參與者的物理軌跡。
- **語氣**：低調的熱血。不是喊口號，是工程師面對不可能的規格書時，一行一行拆解的專注。
- **節奏**：前半段快——數字、筆蓋、白板，節奏密。後半段慢——「你值得被記住」那段要拉長呼吸。
- **禁忌**：不可出現「讓我們一起努力」之類的熱血台詞。維倫的熱血是安靜的——他畫圖，他算數字，他多備份情緒記憶。
- **五感**：油性筆的化學味。白板表面的觸感。筆蓋塑膠咬在齒間的硬度。

### 寫作指令

- **視角**：嚴柏翰主導，但透過維倫的眼睛觀察。嚴柏翰不會自我剖白——他的情感藏在動作裡（識別證「還溫的」、手指按在卡面上）。
- **語氣**：公務體系的乾燥語言。「已登記銷毀」「稽核日誌」「監察組」——用體制的語言描述反體制的行為。反差感是這章的核心。
- **節奏**：中段嚴柏翰的獨白是全章最長的連續語句。前後用短句夾住。獨白本身也不煽情——他在陳述事實，不是演講。
- **禁忌**：嚴柏翰不可流淚。不可聲音顫抖。他是專業的。他的叛變也是專業的。
- **五感**：識別證的金屬溫度。讀卡機的嗶聲。螢幕藍光打在臉上。鍵盤敲擊的觸感——和維倫的敲法不同，嚴柏翰打字是用兩根食指。

### AB線分析

| A線（技術） | B線（情感） |
|---|---|
| 強一致性：所有副本都要確認才通過 | 要求所有人完全理解彼此——不可能，太慢 |
| Quorum：多數同意即可 | 不需要完美理解，只需要夠多的信任 |
| 最終一致性：短暫不同步，但最終會補上 | 人和人之間的理解也是——不會同時到位，但最終會 |
| 副本 C 遲到了 312 毫秒但還是到了 | 有的人慢一點，但他會來 |

**技術真實性筆記**：Quorum 共識是分散式系統的核心概念。在 N 個副本中，只要 W（寫入確認數）+ R（讀取確認數）> N，就能保證讀寫一致性。本章用 N=3, W=2 的設定，是真實系統中最常見的配置（如 Apache Cassandra 的 QUORUM 一致性級別）。強一致性 vs. 最終一致性的取捨也是 System Design 面試的經典考題。

### 寫作指令

- **視角**：上帝視角。不進入任何人的深層思緒。只描述外部——聲音、動作、距離。
- **語氣**：極簡。句子短。段落短。大量留白。這一章的情感全部靠「不說」來傳達。
- **節奏**：非常慢。像深夜三點的呼吸。讀者應該能感覺到時間在這個場景裡變稠了。
- **禁忌**：絕對不可以出現「他們之間有一種無聲的默契」「她知道他在為她努力」之類的旁白。減法寫作——讀者自己感受。
- **核心手法**：只用環境音。鍵盤聲的節奏 = 維倫的情感節奏。咖啡機的循環 = 時間的標記。語青數七分鐘 = 她的存在方式。
- **五感**：鍵盤的觸覺和聲音（最主要）。咖啡的氣味（研磨時最濃）。螢幕藍光的色溫。折疊椅金屬腿在地板上的涼意。壞掉的日光燈偶爾抽搐一下的視覺閃爍。

```
「亮。」

邱韻如的眼睛是亮的。

不是淚光。
是——比淚光更深的什麼。

她的嘴張開了。又合上。
又張開。

她想說什麼。
她想說——

但她不記得。

她不記得這個女孩的任何事情。
不記得抱過她。不記得餵過她。
不記得每天晚上用手指摸她的臉。
不記得她第一次叫媽的聲音。

都不記得。

但她的手記得。

她的手自己動的——
沒有經過大腦——
掌心服貼在語青的臉頰上。

穿過了一半。
只碰到一半。

但那一半——

夠了。

邱韻如的聲音從她喉嚨深處出來。
很小很小。
像二十三年沒用過的一條通道突然打開了——
裡面的空氣是舊的。

「在。」

她說。

「我在。」


語青的臉上——
半透明的臉上——
出現了一種維倫從來沒見過的東西。

不是笑。
不是哭。

是——

安全。

一種非常古老的安全感。
來自身體最深處的。
比記憶早。比語言早。

一千多個夜晚積累下來的——
媽媽的手在臉上的路線。

語青的眼睛閉了一半。
像嬰兒被摸著臉——
快要睡著的那個瞬間。

她靠向邱韻如的手。

穿過了一半。
但靠了。


維倫沒有動。

他坐在他的位置上。
手放在鍵盤上。
螢幕上的代碼還亮著。
游標還在閃。

但他沒有看螢幕。

他在看她們。

母親和女兒。
一個不記得。一個也不記得。
但她們的身體——
在二十三年後——
找到了對方。

維倫的眼睛濕了。

他沒有擦。
他怕擦的動作會發出聲音。
會打破什麼。

他就讓眼淚留在那裡。

鏡片起霧。
螢幕上的代碼變成一團光。

沒關係。

他知道下一行要寫什麼。


米亞終於說話了。

（0.4 秒。）

但她只說了六個字。

聲音比平常輕了一個級距。
像把音量旋鈕往下轉了五度。

「記憶完整性——」

（又一個 0.4 秒。）

「——不只在大腦。」

然後她安靜了。

工作室裡的聲音回到了三種：

語青的呼吸——重新開始了。
邱韻如的呼吸——和語青的頻率很接近。
還有——

很遠很遠的地方——

鍵盤聲。

維倫把眼鏡擦了。
轉回螢幕。
游標閃著。

他的手指落下去。

噠——噠噠——噠噠噠——

繼續寫。

堡壘。

她們的家。
```

### 寫作指令

- **視角**：三段式。第一段——邱韻如的手（外部觀察，不進入她的內心）。第二段——語青的感受（「身體記得」是從她的角度確認的）。第三段——維倫的旁觀。三段像三個同心圓，母女在最中心。
- **語氣**：前半段壓到最低——像 ch226 的延續。「媽」那個字是全章唯一的爆破點。之後的文字立刻收回安靜。不過度渲染。讓那一個字的重量自己擴散。
- **節奏**：邱韻如手指碰到語青臉的那段——極慢。一個動作拆成六七個句子。路線描述（太陽穴→顴骨→臉頰→下巴→右邊臉頰）要逐步寫出。「媽」之後——時間靜止。然後慢慢恢復。用鍵盤聲的重新出現標記時間重啟。
- **禁忌**：
  - 邱韻如不可以在「媽」之後說出「我記得了」。她不記得。重點是她的手記得。
  - 不可出現任何角色說「真感人」或類似的旁白。
  - 維倫不可以走過去擁抱她們。他留在原地。距離是這個場景的一部分。
  - 不可解釋「身體記得」的科學原理。這一章用感受，不用解釋。
- **五感**：邱韻如手指的溫度降低（碰到半透明語青時的涼意）。咖啡冷卻的蒸氣越來越細。語青臉頰被碰觸的路線——這是本章最重要的觸覺描寫，必須精確。
- **核心意象**：邱韻如的手指在語青臉上的路線 = 一千多個夜晚的肌肉記憶。這不是大腦的記憶——是手的記憶。手比人活得更久。手記得的東西，連記憶管理局都刪不掉。
