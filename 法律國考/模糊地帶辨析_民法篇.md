# 民法易混淆概念深度辨析

> 法律中最難的不是個別概念，而是「類似但不同」的概念之間的界線。
> 本篇用工程師的精確思維，把每一組容易混淆的概念用「判斷條件」明確切割。
> 適用對象：有理工/程式背景的非法律轉職考生。

---

# Part I：債法混淆組

## 混淆組 1：給付不能 vs 給付遲延 vs 不完全給付

### 一眼判斷法（30秒速查）

| 判斷條件 | → 給付不能 | → 給付遲延 | → 不完全給付 |
|----------|-----------|-----------|-------------|
| 給付還做得到嗎？ | 做不到了 | 做得到但超時 | 做了但有瑕疵 |
| 債務人有沒有做？ | 想做也不能做 | 能做但還沒做 | 做了但做壞/做錯 |
| 時間點 | 給付期屆至時已不可能 | 給付期屆至後仍可補做 | 給付已經提出 |

### 為什麼會搞混？

因為同一個裝潢工程，改一個事實條件就跳到不同類型。三者都是「債務不履行」的下位概念，差別僅在「不履行的樣態」。

### 關鍵差異深度解析

```
if (給付客觀上或主觀上已不可能) {
    return "給付不能 §226";
} else if (給付可能 && 已屆清償期 && 債務人未為給付) {
    return "給付遲延 §229, §231";
} else if (給付已提出 && (有瑕疵 || 加害給付)) {
    return "不完全給付 §227";
}
```

- **給付不能**（§226）：又分「自始不能」（契約無效§246）與「嗣後不能」。重點在可歸責性——可歸責於債務人→損害賠償；不可歸責→債務消滅（§225 I）。
- **給付遲延**（§229-§231）：須經催告（確定期限者不用）。遲延後責任加重：即使不可抗力也要負責（§231 I）。
- **不完全給付**（§227）：分「瑕疵給付」（可補正→準用遲延；不可補正→準用不能）與「加害給付」（造成履行利益以外的損害，如漏水毀損傢俱）。

### 案例對比

**大情境**：甲請裝潢師乙裝修房屋，約定12月31日完工。

| 微調條件 | 類型 | 法律效果 |
|----------|------|----------|
| 12月20日房屋遭雷擊全毀（天災） | 不可歸責的給付不能 | 乙免給付義務（§225 I），甲免對待給付（§266 I） |
| 12月20日乙施工不慎引發火災燒毀房屋 | 可歸責的給付不能 | 甲得請求損害賠償（§226），並得解除契約（§256） |
| 1月15日乙仍未完工，房屋完好 | 給付遲延 | 甲催告後定期催告仍不履行→得解除契約（§254） |
| 12月30日完工，但浴室防水沒做好、漏水 | 不完全給付（瑕疵給付） | 可補正→催告補正→準用遲延規定 |
| 12月30日完工但漏水，水滲到樓下毀損鄰居傢俱 | 不完全給付（加害給付） | 甲得請求「履行利益以外」的損害賠償（§227 II） |

### 邊界案例

乙遲延中（1月10日還沒完工），結果1月15日發生地震把房子震塌——這算遲延還是不能？答案：**遲延中發生不可抗力，債務人仍須負責**（§231 I「遲延後之給付，於債權人無利益者，債權人得拒絕其給付，請求賠償」）。這就是遲延的「責任加重」效果。

### 實務見解

最高法院77年台上字第1989號判例：不完全給付之債務人所負責任，與給付不能或給付遲延相同，僅發生原因不同。法院強調須先判斷「不完全」是否可補正，再決定準用哪套規則。

### 考試出題陷阱

- 陷阱一：題目設定「嗣後不能」但不告訴你可不可歸責——你必須自己判斷，因為法律效果完全不同。
- 陷阱二：「加害給付」容易和侵權行為搞混——加害給付是契約責任，可同時成立侵權（請求權競合）。
- 陷阱三：不完全給付「可補正」時準用遲延，不是準用不能。很多人搞反。

## 混淆組 2：債務不履行 vs 侵權行為（§184）

### 一眼判斷法（30秒速查）

| 判斷條件 | → 債務不履行 | → 侵權行為 |
|----------|-------------|-----------|
| 當事人之間有契約嗎？ | 有（以契約為基礎） | 不需要 |
| 舉證責任誰扛？ | 債務人證明不可歸責（§230反面） | 原則上被害人舉證（§184） |
| 消滅時效多長？ | 一般15年（§125） | 知悉起2年/行為起10年（§197） |
| 精神慰撫金？ | 原則不行（除非特別約定） | 可以（§195） |

### 為什麼會搞混？

因為同一個事件可以同時構成兩者。醫生開刀失誤：對病人而言既是醫療契約的債務不履行，也是侵害身體的侵權行為。這就是「請求權競合」。

### 關鍵差異深度解析

| 比較項目 | 債務不履行 | 侵權行為 |
|----------|-----------|----------|
| 成立前提 | 須有有效契約 | 不須契約關係 |
| 歸責事由 | 推定債務人有過失（舉證責任倒置） | 原則由被害人證明加害人故意或過失 |
| 賠償範圍 | 履行利益（契約預期利益） | 固有利益 + 所受損害 + 所失利益 |
| 時效 | 15年（一般）或依各契約類型 | 2年/10年 |
| 慰撫金 | 原則否定 | §195列舉情形可請求 |

### 案例對比

**大情境**：病人甲到乙醫院，由丙醫師執刀開膝蓋手術，結果丙疏失切到神經導致左腳癱瘓。

**走債務不履行**：甲與乙醫院有醫療契約→乙的履行輔助人丙有過失→乙應負債務不履行損害賠償（§224、§227）。舉證上甲只要證明「有契約＋有損害＋因果關係」，乙要反證不可歸責。時效15年。

**走侵權行為**：丙醫師過失侵害甲的身體權（§184 I 前段），乙醫院為僱用人連帶負責（§188）。甲須證明丙有過失。但甲可額外請求精神慰撫金（§195）。時效2年。

**微調**：如果甲是路邊被車撞送急診，還沒簽同意書丙就先開刀→可能無契約關係→只能走侵權。

### 邊界案例

實務上採「請求權競合說」（非「請求權自由競合」也非「法條競合」），允許被害人就同一事實分別依契約或侵權主張，但不能雙重受償。考試時兩條路都要寫。

### 考試出題陷阱

- 陷阱一：時效差異是最常考的。侵權2年很短，考生忘記時效已過就只能走契約。
- 陷阱二：慰撫金只有侵權能請求，走契約拿不到——所以實務上通常兩條路併主張。
- 陷阱三：§224履行輔助人 vs §188僱用人責任，要件不同。§224不以選任監督過失為要件，§188有免責事由。

## 混淆組 3：無權代理 vs 表見代理

### 一眼判斷法（30秒速查）

| 判斷條件 | → 無權代理（§170） | → 表見代理（§169） |
|----------|-------------------|-------------------|
| 本人有無授權？ | 完全沒授權 | 沒授權，但有「授權外觀」 |
| 本人有可歸責性嗎？ | 不一定 | 有——本人自己造成或容忍外觀 |
| 法律效果？ | 效力未定→本人可拒絕承認 | 本人須負授權人責任 |
| 保護誰？ | 保護本人（讓本人選擇） | 保護善意第三人 |

### 為什麼會搞混？

兩者的共通點是「代理人實際上都沒有代理權」。差別在於本人有沒有製造讓第三人合理信賴的外觀。

### 關鍵差異深度解析

```
if (代理人無代理權) {
    if (本人有表示授權外觀 || 本人知情而不反對) {
        if (第三人善意 && 無過失) {
            return "表見代理 §169 → 本人負授權人責任";
        }
    }
    return "無權代理 §170 → 效力未定 → 待本人承認";
}
```

### 案例對比

**大情境**：甲公司的前業務員乙（已離職），拿著甲公司的名片和空白契約書去跟丙談生意，以甲公司名義簽約。

| 微調條件 | 結論 |
|----------|------|
| 甲已公告乙離職，收回所有文件，但乙私藏一張舊名片 | 無權代理——甲無可歸責外觀，丙應注意查證 |
| 甲未公告乙離職，乙仍持有公司大小章與空白契約 | 表見代理——甲容忍外觀存在，丙善意信賴受保護 |

### 邊界案例

乙是甲的現任員工但只有「採購權」，卻跑去跟丙簽「銷售契約」——這是「逾越代理權限」。依§107準用§169：如果甲知道乙常常這樣做卻不制止，可能構成表見代理。

### 實務見解

最高法院70年台上字第657號：表見代理之成立，須本人有表見之事實，足使第三人信該他人有代理權之正當理由，且第三人係善意無過失。

### 考試出題陷阱

- 陷阱一：表見代理的舉證責任在第三人——第三人要證明自己善意且無過失。
- 陷阱二：無權代理未經承認→代理人自己要負§110的履行或損害賠償責任。
- 陷阱三：本人的「沉默」不等於承認（§170 II），但可能構成表見代理的外觀。

## 混淆組 4：不當得利 vs 侵權行為

### 一眼判斷法（30秒速查）

| 判斷條件 | → 不當得利（§179） | → 侵權行為（§184） |
|----------|-------------------|-------------------|
| 核心關注點 | 受益人「得到什麼」 | 加害人「造成什麼損害」 |
| 需要故意/過失嗎？ | 不需要（客觀有無法律上原因） | 需要（故意或過失） |
| 返還/賠償範圍 | 所受利益（善意者現存利益） | 全部損害（回復原狀） |
| 功能 | 調整財產秩序 | 填補損害 |

### 為什麼會搞混？

偷東西同時構成兩者：偷了→無法律上原因受利益（不當得利）＋故意侵害所有權（侵權）。但法律效果不同。

### 關鍵差異深度解析

不當得利的要點：（1）一方受利益（2）他方受損害（3）無法律上原因（4）兩者間有因果關係。不問主觀善惡意，只問客觀有無原因。

侵權行為的要點：（1）加害行為（2）侵害權利或利益（3）有因果關係（4）有故意或過失（5）無阻卻違法事由。

### 案例對比

**情境一**：甲誤以為乙的腳踏車是自己的，騎走使用三天後歸還。
- 不當得利：甲無法律上原因使用乙的車→應返還使用利益（相當於租金）。
- 侵權行為：甲無過失（誤認），§184 I 前段要求故意或過失→不成立。
- 結論：只能走不當得利。

**情境二**：甲明知是乙的腳踏車，故意偷走並摔壞。
- 不當得利：甲受有使用利益→應返還。但車子壞了不是「利益」而是「損害」。
- 侵權行為：甲故意毀損乙的車→應賠償修復費用。
- 結論：兩者競合，但賠償範圍不同——不當得利只管「得到的」，侵權管「壞掉的」。

### 邊界案例

甲把乙的木材拿去蓋房子——木材已經「添附」（§811-§816）變成房屋的一部分。乙喪失木材所有權，但可以依不當得利請求甲返還木材的價額。此時侵權的損害賠償和不當得利的返還範圍可能一致，但請求權基礎不同。

### 考試出題陷阱

- 陷阱一：不當得利的善意受領人只返還「現存利益」（§182 I），如果花光了就不用還。侵權沒有這個減免。
- 陷阱二：惡意受領人要返還全部利益加利息，且須賠償損害（§182 II）——這時候和侵權的效果很接近。
- 陷阱三：非給付型不當得利（侵害型）vs 給付型不當得利——判斷「無法律上原因」的標準不同。

## 混淆組 5：買賣瑕疵擔保 vs 不完全給付 vs 詐欺

### 一眼判斷法（30秒速查）

| 判斷條件 | → 瑕疵擔保（§354） | → 不完全給付（§227） | → 詐欺（§92） |
|----------|-------------------|-------------------|---------------|
| 賣方知道瑕疵嗎？ | 不問（無過失責任） | 須可歸責於債務人 | 賣方故意隱瞞 |
| 時效 | 通知後6個月（§365） | 15年 | 撤銷權1年（§93） |
| 法律效果 | 解約或減價（§359） | 損害賠償 | 撤銷意思表示 |
| 需要通知嗎？ | 要（§356怠於通知視為承認） | 不用 | 不用 |

### 為什麼會搞混？

同一個情境（買到泡水車）可以同時走三條路。三個制度目的不同但結果可能重疊。

### 關鍵差異深度解析

**大情境**：甲向乙中古車商買車，交車後發現是重大事故車。

**路線一——瑕疵擔保（§354-§366）**：
- 不問乙知不知情，只要車有「滅失或減少通常效用或契約預定效用」的瑕疵
- 甲須在發現後「通知」乙（§356），然後6個月內行使解除或減價（§365）
- 優點：不用證明乙有過失。缺點：時效超短，且只有解約/減價，不能直接請求損害賠償

**路線二——不完全給付（§227）**：
- 須證明乙有可歸責事由（知情或應注意而未注意）
- 時效15年
- 可請求損害賠償，包括「加害給付」造成的其他損害
- 優點：時效長、可請求賠償。缺點：要多證明可歸責

**路線三——詐欺撤銷（§92）**：
- 乙故意隱瞞事故紀錄→詐欺→甲得撤銷買賣契約
- 撤銷後契約自始無效→回復原狀（§114、§179）
- 優點：契約整個打掉。缺點：要證明「故意」

### 案例對比

| 微調條件 | 最佳策略 |
|----------|----------|
| 乙不知道是事故車（前手隱瞞） | 瑕疵擔保（不問過失）＋不完全給付可能不成立 |
| 乙知道是事故車但沒說 | 三條路都走。瑕疵擔保＋不完全給付＋如果構成詐欺就撤銷 |
| 甲發現瑕疵後拖了8個月才行動 | 瑕疵擔保6個月時效已過→只能走不完全給付（15年） |

### 邊界案例

甲買車後事故車的引擎爆炸，炸傷甲和甲的家人。引擎爆炸的損害超出「車子本身瑕疵」的範圍→這是「加害給付」（§227 II），可以請求車價以外的人身和財物損害賠償。瑕疵擔保救不了這塊。

### 考試出題陷阱

- 陷阱一：瑕疵擔保的6個月時效是「除斥期間」（不是消滅時效），不能中斷。
- 陷阱二：§356「怠於通知視為承認」——買方發現瑕疵後必須「即時」通知，否則喪失權利。
- 陷阱三：三條路可以併行主張，但不能雙重受償。答題時三個都要分析。

## 混淆組 6：承攬 vs 委任 vs 僱傭

### 一眼判斷法（30秒速查）

| 判斷條件 | → 承攬（§490） | → 委任（§528） | → 僱傭（§482） |
|----------|---------------|---------------|---------------|
| 契約標的 | 工作之完成 | 事務之處理 | 勞務之給付 |
| 有沒有「成果」？ | 有（蓋好房子） | 不一定（盡力即可） | 無（付出時間勞力） |
| 報酬怎麼算？ | 工作完成後給付 | 可有可無（§547） | 按時間給付 |
| 誰指揮？ | 承攬人自主決定方法 | 受任人有裁量權 | 僱用人指揮監督 |

### 為什麼會搞混？

三個都是「花錢請人做事」，差別在「做的事情性質」和「誰說了算」。

### 關鍵差異深度解析

```
if (重點在「完成特定成果」&& 方法由受託人自主決定) {
    return "承攬";
} else if (重點在「處理事務」&& 不保證結果 && 有一定獨立裁量) {
    return "委任";
} else if (重點在「提供勞力/時間」&& 受僱用人指揮監督) {
    return "僱傭";
}
```

### 案例對比

| 情境 | 類型 | 理由 |
|------|------|------|
| 請營造廠蓋一棟房子 | 承攬 | 重點在「房子蓋好」這個成果 |
| 請律師打一場官司 | 委任 | 律師盡力處理訴訟事務，不保證贏 |
| 請保全站崗值班 | 僱傭 | 按時間提供勞務，受保全公司指揮 |
| 請設計師設計LOGO | 承攬 | 重點在「LOGO完成」的成果 |
| 請顧問提供企業診斷 | 委任 | 處理分析事務，有專業裁量 |
| 請工讀生在超商收銀 | 僱傭 | 受店長指揮，按時計薪 |

### 邊界案例

**醫師開刀是承攬還是委任？** 通說認為是**委任**（或類似委任的混合契約）。理由：醫療行為不保證治癒的結果，醫師是「盡善良管理人注意處理醫療事務」。如果認定為承攬，開刀沒治好就要負「工作未完成」的責任，這不合理。

**外送員是僱傭還是承攬？** 近年爭議。如果平台指定路線、時間、制服→偏僱傭；如果外送員自由接單、自備車輛→偏承攬。實務上個案判斷。

### 實務見解

最高法院99年台上字第1017號：承攬與委任之區別，在於承攬之目的在於一定工作之完成，委任則為事務之處理；其受有報酬者，前者於工作完成後始得請求，後者則得隨時請求。

### 考試出題陷阱

- 陷阱一：承攬人的瑕疵擔保（§492-§501）只有承攬才有，委任和僱傭沒有。
- 陷阱二：承攬人的損害賠償時效僅1年（§514），比一般的15年短非常多。
- 陷阱三：定作人可隨時終止承攬契約但須賠償（§511），委任人也可隨時終止（§549）但不用賠——搞反就扣分。

## 混淆組 7：解除契約 vs 終止契約

### 一眼判斷法（30秒速查）

| 判斷條件 | → 解除（§254-§260） | → 終止（各論規定） |
|----------|---------------------|-------------------|
| 效力方向 | 溯及既往（回到沒訂約時） | 向將來消滅 |
| 已給付部分 | 互負回復原狀義務（§259） | 不影響已履行部分 |
| 適用契約類型 | 一次性給付契約（買賣） | 繼續性契約（租賃、僱傭） |

### 為什麼會搞混？

日常用語混用。一般人說「我要解約」，法律上可能是解除也可能是終止，效果天差地別。

### 關鍵差異深度解析

**解除**：按下 Ctrl+Z 復原到契約成立前。買了瑕疵品→解除→雙方返還（買方退貨、賣方退錢）。

**終止**：按下 Stop 鍵從現在起停止。房客住了半年後終止→前半年的租金不用退，從現在起不用再付租金。

### 案例對比

| 情境 | 正確用法 |
|------|----------|
| 甲買乙的手機，發現是瑕疵品 | 解除買賣契約→退貨退款 |
| 甲租乙的房子住了三年，乙要收回 | 終止租賃契約→已付租金不退，搬家即可 |
| 甲跟乙簽一年顧問委任約，做了半年不想做 | 終止委任契約（§549）→已付報酬不退 |

### 邊界案例

繼續性契約能不能「解除」？通說認為繼續性契約原則上只能「終止」，因為已經履行的部分無法完全回復。但如果契約剛成立還沒開始履行，也有例外允許解除的餘地。

### 考試出題陷阱

- 陷阱一：題目用「解約」二字時，要自己判斷是解除還是終止。看契約類型和已履行狀態。
- 陷阱二：解除契約後的回復原狀請求權時效為15年，不要和原契約的時效搞混。

## 混淆組 8：債權讓與 vs 債務承擔 vs 契約承擔

### 一眼判斷法（30秒速查）

| 判斷條件 | → 債權讓與（§294） | → 債務承擔（§300） | → 契約承擔 |
|----------|-------------------|-------------------|-----------|
| 什麼東西在移動？ | 債權人換人 | 債務人換人 | 整個契約地位換人 |
| 需要誰同意？ | 不需債務人同意 | 需債權人同意 | 需他方同意 |
| 從屬權利？ | 擔保隨同移轉 | 抗辯隨同移轉 | 全部隨同移轉 |

### 為什麼會搞混？

三者都是「換人」，但換的位置不同。

### 案例對比

**大情境**：甲欠乙100萬元。

- **債權讓與**：乙把「向甲收100萬的權利」讓給丙。甲現在要還錢給丙。不需甲同意（§297通知即可）。
- **債務承擔**：丙願意替甲還這100萬。乙必須同意（§301），因為換了債務人，償債能力可能不同。
- **契約承擔**：甲乙間的契約中，甲的整個地位（權利＋義務全部）由丙接手。如頂讓店面時，承租人地位由新老闆全部接手。需出租人同意。

### 邊界案例

債權讓與中，如果債權有「禁止讓與」的特約（§294 I 但書），讓與無效。但§294 II規定此特約不得對抗善意第三人。實務上常考。

另一個邊界：「未來債權」能不能讓與？通說肯定——只要債權可以特定，即使尚未發生也可以讓與。例如將「未來一年對A公司的貨款債權」預先讓與給銀行作為融資擔保。

### 實務見解

最高法院42年台上字第626號判例：債權之讓與，非經通知債務人，對於債務人不生效力。此所謂通知，不以讓與人自為通知為必要，受讓人亦得為之。重點在於「通知」只是對抗要件，不是生效要件。

### 考試出題陷阱

- 陷阱一：債權讓與不需要債務人同意，但要「通知」債務人（§297），未通知不得對抗債務人。注意：未通知時，讓與在讓與人和受讓人之間仍然有效，只是不能對抗債務人。
- 陷阱二：債務承擔後，原債務人的保證人和物上擔保人原則上免責（§304），除非他們同意。
- 陷阱三：契約承擔目前民法無明文規定，是基於契約自由原則由實務承認的制度。答題時要先說明法律基礎。

## 混淆組 9：同時履行抗辯 vs 不安抗辯

### 一眼判斷法（30秒速查）

| 判斷條件 | → 同時履行抗辯（§264） | → 不安抗辯（§265） |
|----------|----------------------|-------------------|
| 誰先給付？ | 雙方同時給付 | 有先給付義務的一方 |
| 行使原因 | 對方未提出給付 | 對方「財產顯著減少」有難以對待給付之虞 |
| 功能 | 「你不給，我也不給」 | 「你快破產了，我先不給」 |

### 為什麼會搞混？

兩者都是「我先不給」的抗辯權，但適用時機和要件完全不同。

### 案例對比

**同時履行**：甲賣車給乙，交車日到了。甲說：「你先付錢」，乙說：「你先交車」→雙方都可以主張同時履行抗辯。

**不安抗辯**：甲先給付（先交車再收款）。但交車前甲發現乙的公司快倒了、帳戶被凍結→甲可以主張不安抗辯，在乙提出擔保或對待給付前拒絕先交車。

### 邊界案例

不安抗辯的「財產顯著減少」標準：不是稍微虧損就能主張，必須達到「有難為對待給付之虞」的程度。如果乙只是這個月營收下降但整體財務穩健，甲不能主張不安抗辯。

### 關鍵差異深度解析

同時履行抗辯的要件：(1)雙務契約 (2)雙方互負對待給付 (3)對方未提出給付 (4)無先為給付義務。

不安抗辯的要件：(1)雙務契約 (2)有先給付義務 (3)他方財產於訂約後「顯形減少」(4)有難為對待給付之虞。

```
if (雙方同時給付義務 && 對方未給付) {
    return "同時履行抗辯 §264";
} else if (己方有先給付義務 && 對方財產顯著減少) {
    return "不安抗辯 §265";
}
```

### 實務見解

最高法院50年台上字第1550號判例指出：同時履行抗辯權之行使，須以他方之給付與自己之給付立於互為對待給付之關係為要件。如果雙方的給付不具對價關係（例如附隨義務），不能主張同時履行抗辯。

### 考試出題陷阱

- 陷阱一：同時履行抗辯是「延緩抗辯」，不是「永久抗辯」——不消滅對方請求權，只是暫時拒絕。
- 陷阱二：法院不能依職權主張同時履行抗辯，必須被告自己提出。
- 陷阱三：判決主文會寫「被告應於原告給付XX之同時，給付原告XX」——這是同時履行判決的特殊格式，也是常考點。

## 混淆組 10：連帶債務 vs 不真正連帶債務

### 一眼判斷法（30秒速查）

| 判斷條件 | → 連帶債務（§272-§282） | → 不真正連帶債務 |
|----------|------------------------|-----------------|
| 債務發生原因 | 同一原因（法律規定或契約約定） | 不同原因，但給付內容相同 |
| 內部分擔 | 有（§280-§281求償權） | 原則無，但有終局責任者 |
| 絕對效力事項 | §274-§278規定（清償、免除等有絕對效力） | 僅清償有絕對效力 |
| 法律明文 | §272明文規定 | 無明文，由學說判例承認 |

### 為什麼會搞混？

外觀上都是「多人對同一債權人負同一給付義務」，債權人都可以向任何一人請求全部。

### 關鍵差異深度解析

**連帶債務**：甲乙丙三人共同向銀行借100萬→銀行可以向任何一人要100萬→被要到的人可以向其他人求償各自分擔額。

**不真正連帶債務**：甲被乙打傷（侵權），甲的保險公司丙理賠了→乙（侵權責任）和丙（保險契約）對甲負的是不真正連帶。原因不同，但甲只能拿到一份賠償。乙是終局責任人，丙賠了可以向乙追償。

### 案例對比

| 情境 | 類型 | 理由 |
|------|------|------|
| 甲乙共同打傷丙 | 連帶債務（§185） | 共同侵權，同一原因 |
| 乙員工丙執行職務撞傷甲 | 不真正連帶 | 丙是自己的侵權責任（§184），乙是僱用人責任（§188），原因不同 |

### 關鍵差異深度解析

連帶債務有「絕對效力」事項——某些事由對一債務人發生效力時，對全體連帶債務人均生效力：
- 清償（§274）：一人清償全部→全體免責
- 免除（§276）：債權人免除一人的債務→就該人的分擔部分，其他人也免責
- 混同（§275）：一債務人與債權人混同→他人就其分擔額免責
- 時效完成（§276）：對一人時效完成→就該人分擔部分，他人免責

不真正連帶債務沒有上述絕對效力規定。只有「清償」具有事實上的絕對效力（因為債權已滿足）。

### 實務見解

最高法院85年台上字第975號判決：不真正連帶債務之各債務人，係基於不同之法律原因，對同一債權人各負給付全部之義務，且各債務之發生原因不同，僅因偶然競合而致對同一債權人各負全部給付之義務。

### 考試出題陷阱

- 陷阱一：連帶債務中，一人被免除→是否影響其他人？§276有絕對效力（就該人應分擔部分免除）。不真正連帶中，免除一人原則上不影響他人。
- 陷阱二：不真正連帶是實務與學說發展出來的概念，民法沒有明文，但國考常考。
- 陷阱三：連帶債務人之間的「內部分擔額」預設平均分擔（§280），但可另外約定。不真正連帶的「終局責任」由法律關係本質決定（如：侵權行為人是終局責任人，僱用人只是中間責任人）。

## 混淆組 11：物權行為 vs 債權行為（物權行為獨立性與無因性）

### 一眼判斷法（30秒速查）

| 判斷條件 | → 債權行為 | → 物權行為 |
|----------|-----------|-----------|
| 產生什麼效果？ | 債權債務關係（請求權） | 物權直接變動（得喪變更） |
| 舉例 | 簽買賣契約 | 交付動產/登記不動產 |
| 需要什麼要件？ | 意思表示合致 | 意思表示合致 + 交付或登記 |

### 為什麼會搞混？

日常生活中「買東西付錢拿走」是一氣呵成的動作。但法律上拆成兩層：(1)買賣契約（債權行為）→(2)交付（物權行為）。

### 關鍵差異深度解析

一筆買賣中，至少有三個法律行為：
1. **買賣契約**（債權行為）：產生賣方交貨義務＋買方付款義務
2. **標的物交付/移轉登記**（物權行為）：所有權從賣方移轉到買方
3. **價金交付**（物權行為）：金錢所有權從買方移轉到賣方

**獨立性**：物權行為獨立於債權行為。即使買賣契約不存在，物權行為可以單獨成立。

**無因性**：物權行為的效力不受債權行為（原因行為）影響。買賣契約無效→物權移轉仍然有效→但可以依不當得利請求返還。

### 案例對比

**情境**：甲賣A畫給乙，乙付錢，甲交畫。事後發現甲是被詐欺才簽的。

| 分析步驟 | 結果 |
|----------|------|
| 甲撤銷買賣契約（§92） | 買賣契約自始無效 |
| A畫的物權移轉有效嗎？ | 依無因性→物權行為不受影響→乙仍為所有人 |
| 甲怎麼拿回畫？ | 依不當得利（§179）→乙無法律上原因取得A畫→應返還 |

**微調**：如果甲是被「脅迫」→物權行為的意思表示本身也有瑕疵→物權行為也可撤銷→甲仍為所有人，直接依§767請求返還。

### 邊界案例

我國通說及實務採「物權行為獨立性及無因性」（但學說上有爭議）。王澤鑑教授等學者主張採「有因性」或「相對無因性」。考試上以通說為主，但最好加一句「學說上有不同見解」。

### 實務見解

最高法院83年台上字第1179號判決肯認物權行為無因性：「物權行為有其獨立之意思表示，不因原因行為即債權契約之不成立、無效或被撤銷而受影響。」但須注意部分學者批評此見解過度保護交易安全而犧牲原所有人利益。

### 考試出題陷阱

- 陷阱一：物權行為無因性導致一個重要結果——契約無效時，所有權已移轉的要用不當得利討回，不能直接用物權請求權。答題時一定要寫清楚請求權基礎是§179而非§767。
- 陷阱二：但若物權行為本身有瑕疵（如意思表示錯誤、脅迫），物權行為也可能無效或被撤銷——這時候原所有人可以直接用§767物權請求權。
- 陷阱三：動產和不動產的物權行為形式不同——動產是「交付」（§761），不動產是「登記」（§758）。不要混用。

## 混淆組 12：善意取得 vs 時效取得

### 一眼判斷法（30秒速查）

| 判斷條件 | → 善意取得（§801+§948） | → 時效取得（§768-§772） |
|----------|------------------------|----------------------|
| 取得方式 | 從無權處分人處受讓 | 長期占有他人之物 |
| 時間要求 | 即時（交易當下） | 動產5年/不動產（登記）10-20年 |
| 主觀要件 | 善意（不知無權處分）且無重大過失 | 以所有之意思（自主占有） |
| 客體 | 動產為主（不動產有§759-1） | 動產和未登記不動產 |

### 為什麼會搞混？

兩者都是「非所有人取得所有權」的制度。但一個是「即時取得」，一個是「經過長時間取得」。

### 關鍵差異深度解析

**善意取得**：保護交易安全。你去跳蚤市場買了一幅畫，賣家其實是小偷→如果你善意不知情且無重大過失→你取得所有權。但注意：占有脫離物（被偷、遺失的東西）的善意取得有限制（§949-§951）——被盜物2年內原所有人可以回復。

**時效取得**：獎勵長期利用。你占用鄰居一塊空地種菜已經20年→可以主張時效取得所有權（但須向地政機關請求登記）。

### 案例對比

| 情境 | 適用制度 |
|------|----------|
| 甲在二手店買到乙被偷的手錶，不知情 | 善意取得（但乙2年內可請求回復） |
| 甲在乙的荒地上蓋屋居住20年 | 時效取得（§769和平公然繼續占有20年） |
| 甲向乙的受託人丙購車，丙無權處分 | 善意取得 |

### 邊界案例

已登記的不動產能否善意取得？依§759-1：「因信賴不動產登記之善意第三人，已依法律行為為物權變動之登記者，其變動之效力，不因原登記物權之不實而受影響。」所以是可以的，但要件更嚴格。

### 實務見解

最高法院64年台上字第2219號判例：善意受讓動產之占有者，縱讓與人無讓與之權利，受讓人仍取得其所有權，但受讓人明知或因重大過失而不知讓與人無讓與之權利者，不在此限。此判例明確「善意無重大過失」的舉證分配。

### 考試出題陷阱

- 陷阱一：善意取得只適用於「交易行為」取得，繼承或贈與等無償取得有爭議。通說認為善意取得以「有償取得」為原則。
- 陷阱二：時效取得要「和平、公然、繼續」占有——偷偷摸摸占用不算。如果中途有中斷占有，年數要重新計算。
- 陷阱三：時效取得後不是自動取得所有權，須請求登記（不動產）或主張（動產）。§769和§770的差異：前者要求20年（不問善惡意），後者要求10年但須善意且無過失。

## 混淆組 13：抵押權 vs 質權 vs 留置權

### 一眼判斷法（30秒速查）

| 判斷條件 | → 抵押權（§860） | → 質權（§884） | → 留置權（§928） |
|----------|-----------------|---------------|-----------------|
| 占有物品嗎？ | 不用（設定登記即可） | 要（交付占有） | 要（合法占有中） |
| 設定方式 | 契約 + 登記 | 契約 + 交付 | 法律規定（法定擔保物權） |
| 客體 | 不動產為主 | 動產、權利 | 動產 |
| 當事人意願 | 雙方約定 | 雙方約定 | 不需約定，法律直接賦予 |

### 為什麼會搞混？

三者都是「擔保物權」——確保債權能受清償。差別在設定方式和占有關係。

### 案例對比

| 情境 | 類型 |
|------|------|
| 甲拿房子向銀行貸款，銀行設定抵押 | 抵押權（不動產不用交給銀行，登記即可） |
| 甲拿金錶去當鋪借錢 | 質權（金錶交給當鋪占有） |
| 甲送車去修理廠修車，修好後甲不付修理費 | 留置權（修理廠合法占有車輛，甲不付錢就不還車） |

### 邊界案例

抵押權能不能設定在動產上？民法原則只有不動產。但「動產擔保交易法」有「動產抵押」制度——用登記代替交付，車貸就是典型例子。嚴格說這不是民法上的抵押權，但功能類似。

### 關鍵差異深度解析

三種擔保物權的「實行」（債務人不還錢時怎麼辦）也不同：
- **抵押權**：聲請法院拍賣抵押物（§873），就賣得價金優先受償
- **質權**：可以聲請法院拍賣（§893），也可以約定「流質契約」（§893-1，但動產質權禁止流質）
- **留置權**：先定相當期限催告清償（§936），逾期仍不清償→得聲請法院拍賣留置物

### 實務見解

抵押權的「最高限額抵押權」（§881-1以下）是實務上最常見的類型。與普通抵押權的差異：最高限額抵押權擔保的是「不特定債權」，在最高限額範圍內，將來發生的債權都受擔保。一般抵押權擔保的是「特定債權」。

### 考試出題陷阱

- 陷阱一：留置權是「法定」擔保物權，不是約定的——當事人不能用契約創設留置權，也不能用契約排除（§928是強制規定）。但§929規定，如果留置權之行使違反公共秩序或善良風俗，或與債權人應負之義務相抵觸，不得行使。
- 陷阱二：質權人不能使用質物（§888），違反者須負損害賠償責任。抵押物則由抵押人繼續使用。
- 陷阱三：抵押權有「次序」問題（第一順位、第二順位），質權和留置權沒有。
- 陷阱四：「權利質權」（§900以下）可以拿債權、股權等「權利」來設質，這和動產質權不同——不需要「交付有體物」。

## 混淆組 14：所有權 vs 占有

### 一眼判斷法（30秒速查）

| 判斷條件 | → 所有權（§765） | → 占有（§940） |
|----------|-----------------|---------------|
| 法律性質 | 物權（權利） | 事實狀態（不是權利） |
| 保護方式 | 物權請求權（§767） | 占有保護（§960-§962） |
| 需要什麼？ | 法律上正當原因 | 事實上管領力 |

### 為什麼會搞混？

日常語言中「我擁有這個東西」既可以指所有也可以指占有。法律上嚴格區分。

### 關鍵差異深度解析

**所有權**是「法律上的歸屬」。房子登記在你名下→你是所有權人，即使你人在國外從來沒住過。

**占有**是「事實上的支配」。房客住在房子裡→房客是占有人，但不是所有權人。

### 案例對比

| 情境 | 所有權人 | 占有人 |
|------|----------|--------|
| 甲把房子租給乙 | 甲 | 乙 |
| 小偷丙偷走甲的車 | 甲 | 丙（無權占有） |
| 甲把書借給乙，乙轉借給丙 | 甲 | 丙（間接占有鏈：甲→乙→丙） |

### 邊界案例

占有人可以主張「占有保護」（§960自力救濟、§962占有之訴），即使是小偷也有占有保護——因為占有保護的目的是維護社會和平秩序，禁止私力救濟。但所有權人可以透過§767物權請求權（返還、妨害排除、妨害防止）取回。

### 考試出題陷阱

- 陷阱一：占有不是權利，但法律保護占有的「事實狀態」——因為禁止任何人用私力改變現狀。
- 陷阱二：§943占有推定：占有人推定為合法占有、推定為以所有之意思占有。這個推定在善意取得和時效取得中很重要。
- 陷阱三：間接占有（§941）的概念——出租人是間接占有人，承租人是直接占有人。兩人都有占有保護。

## 混淆組 15：意思表示瑕疵大亂鬥——錯誤 vs 詐欺 vs 脅迫 vs 虛偽意思表示

### 一眼判斷法（30秒速查）

| 判斷條件 | 錯誤（§88） | 詐欺（§92） | 脅迫（§92） | 單獨虛偽（§86） | 通謀虛偽（§87） |
|----------|------------|------------|------------|----------------|----------------|
| 表意人知道嗎？ | 不知道自己搞錯 | 被騙才做的 | 被逼才做的 | 知道（故意假裝） | 雙方都知道是假的 |
| 法律效果 | 得撤銷 | 得撤銷 | 得撤銷 | 原則有效 | 無效 |
| 對第三人效力 | 撤銷不得對抗善意第三人 | 第三人詐欺→須相對人明知 | 撤銷得對抗任何人 | — | 不得對抗善意第三人 |
| 除斥期間 | 1年 | 1年（脅迫終止後1年） | 1年 | — | — |

### 為什麼會搞混？

五種瑕疵的「表面行為」都是「做了一個不想做的法律行為」，但原因不同→效果不同。

### 關鍵差異深度解析

```
if (表意人自己搞錯 && 不知道搞錯) {
    return "錯誤 §88 → 得撤銷（但要賠償信賴利益）";
} else if (第三方施詐導致錯誤意思) {
    return "詐欺 §92 → 得撤銷（1年除斥期間）";
} else if (被脅迫而為意思表示) {
    return "脅迫 §92 → 得撤銷（保護最強，得對抗善意第三人）";
} else if (表意人自知與真意不符 && 單方) {
    if (相對人明知) {
        return "§86但書 → 無效";
    }
    return "§86 → 有效（自己活該）";
} else if (雙方通謀虛偽) {
    return "§87 → 無效（但不得對抗善意第三人）";
}
```

### 案例對比

| 情境 | 類型 |
|------|------|
| 甲要標「100萬」按成「10萬」送出投標 | 錯誤（§88）——表示行為錯誤 |
| 乙跟甲說這是真跡（其實是仿品），甲花100萬買下 | 詐欺（§92）——乙故意使甲陷於錯誤 |
| 丙威脅甲說不簽約就傷害甲的家人 | 脅迫（§92）——意思表示自由被侵害 |
| 甲嘴上說要送車給乙（開玩笑），乙信以為真 | 單獨虛偽（§86）——甲心裡知道是假的，但乙不知→有效 |
| 甲乙為避稅假裝買賣房屋 | 通謀虛偽（§87）——雙方都知道是假的→無效 |

### 邊界案例

**脅迫 vs 詐欺**最大差異在第三人保護：詐欺的撤銷不得對抗善意第三人（§92 I 但書類推），但**脅迫的撤銷得對抗任何人**，包括善意第三人。理由：脅迫對意思自由的侵害最嚴重，法律給最強保護。

### 考試出題陷阱

- 陷阱一：§88錯誤撤銷有「信賴利益賠償」義務（§91），其他瑕疵類型沒有。
- 陷阱二：「動機錯誤」原則上不能撤銷，除非該動機已被表示為意思表示的內容（§88 II 性質錯誤）。
- 陷阱三：通謀虛偽的「隱藏行為」（§87 II）→如果隱藏行為本身合法有效，依其性質定之。例如名為買賣實為贈與→買賣無效，但贈與可能有效。

## 混淆組 16：無效 vs 得撤銷 vs 效力未定

### 一眼判斷法（30秒速查）

| 判斷條件 | → 無效 | → 得撤銷 | → 效力未定 |
|----------|--------|---------|-----------|
| 預設狀態 | 自始無效（不用做任何事） | 有效（直到被撤銷） | 懸而未決（等待確定） |
| 誰來決定？ | 不用決定，當然無效 | 撤銷權人行使撤銷權 | 特定人的承認或拒絕 |
| 有時間限制嗎？ | 無（永遠無效） | 有（除斥期間，通常1年） | 催告後確答期限 |
| 撤銷/確定後效果 | — | 溯及自始無效（§114） | 承認→自始有效/拒絕→確定無效 |

### 為什麼會搞混？

三者最後都「可能」變成無效，但過程完全不同。

### 關鍵差異深度解析

把它想成程式中的三種狀態：

```
// 無效：compile error——程式根本跑不起來
// 得撤銷：程式跑了，但有bug，使用者可以選擇rollback
// 效力未定：程式需要管理員批准才能執行
```

**無效的典型情形**：違反強制規定（§71）、違反公序良俗（§72）、通謀虛偽（§87）、自始客觀不能（§246）

**得撤銷的典型情形**：錯誤（§88）、詐欺脅迫（§92）、限制行為能力人未得允許的「單方行為」（§78）...等等，注意：限制行為能力人的契約行為是「效力未定」不是「得撤銷」

**效力未定的典型情形**：限制行為能力人未經法定代理人允許的契約行為（§79）、無權代理（§170）、無權處分（§118）

### 案例對比

| 情境 | 類型 | 理由 |
|------|------|------|
| 甲（16歲）未經父母同意買機車 | 效力未定 | §79——等父母承認或拒絕 |
| 甲乙假裝買賣房屋避稅 | 無效 | §87——通謀虛偽 |
| 甲被乙騙而簽保證契約 | 得撤銷 | §92——在甲撤銷之前，契約有效 |
| 甲賣乙的車給丙（甲無處分權） | 效力未定 | §118——等乙承認 |

### 邊界案例

限制行為能力人（滿7歲未滿18歲）的法律行為：
- **純獲法律上利益**→有效（§77但書）
- **日常生活必需**→有效（§77但書）
- **其他契約行為**→效力未定（§79，等法定代理人承認）
- **單方行為**→無效（§78，不是效力未定！）

§78和§79的差別是超級高頻考點。

### 考試出題陷阱

- 陷阱一：「得撤銷」在撤銷前是有效的。如果超過除斥期間沒撤銷→確定有效，不能再翻。
- 陷阱二：效力未定中，相對人可以催告（§80），一個月內不確答→視為拒絕。
- 陷阱三：無效不能補正、不能承認。但「得撤銷」可以因為不行使撤銷權而變成確定有效。

## 混淆組 17：代理 vs 代表 vs 使者

### 一眼判斷法（30秒速查）

| 判斷條件 | → 代理（§103） | → 代表 | → 使者 |
|----------|---------------|--------|--------|
| 誰的意思表示？ | 代理人自己形成意思 | 代表人即為法人本身 | 傳達本人已形成的意思 |
| 行為能力要求 | 代理人須有行為能力嗎？不須完全（§104） | 代表人通常須完全行為能力 | 不須行為能力（傳話就好） |
| 意思表示瑕疵看誰？ | 看代理人（§105） | 看代表人 | 看本人 |

### 為什麼會搞混？

三者外觀相似：都是「A替B做法律行為」。但內部法律結構完全不同。

### 關鍵差異深度解析

**代理**：代理人以「本人名義」獨立形成意思表示。甲授權乙代理購買房屋→乙自己決定買哪間、跟誰談→效果歸屬甲。

**代表**：代表人與法人是「同一人格」。董事長代表公司簽約→不是「替公司做」，而是「就是公司在做」。法人沒有手腳，只能透過代表人行為。

**使者**：傳聲筒。甲寫好契約書，叫乙拿去交給丙→乙是使者，只是物理上傳遞甲已經做好的意思表示。

### 案例對比

| 情境 | 類型 | 意思表示瑕疵看誰？ |
|------|------|-------------------|
| 甲授權乙去買一幅畫，乙看錯標價 | 代理→看乙（§105） | 乙的錯誤可撤銷 |
| 公司董事長代表公司簽約 | 代表→看董事長 | 董事長的意思即公司的意思 |
| 甲寫好出價信，叫乙送去 | 使者→看甲 | 甲的意思表示有無瑕疵看甲 |
| 甲叫乙送信，乙傳錯金額 | 使者傳達錯誤 | 類推§89（表示錯誤），甲得撤銷 |

### 邊界案例

代理與使者的灰色地帶：甲說「幫我去買一瓶水」——乙是代理人（自己決定買哪個品牌）還是使者（甲已決定好一切）？如果甲給了非常具體的指示（哪個品牌、哪家店、多少錢），乙比較像使者；如果甲只給概括授權，乙比較像代理人。

### 考試出題陷阱

- 陷阱一：代理人不需要有完全行為能力（§104），但使者連行為能力都不需要（因為不是自己做意思表示）。
- 陷阱二：代理的法律效果直接歸屬本人（§103），但代表的效果是法人自己的——這在判斷法人侵權責任（§28）時很重要。

## 混淆組 18：條件 vs 期限

### 一眼判斷法（30秒速查）

| 判斷條件 | → 條件（§99-§101） | → 期限（§102） |
|----------|-------------------|---------------|
| 事件會不會發生？ | 不確定（可能發生也可能不發生） | 確定會到來 |
| 舉例 | 「如果你考上律師，我送你車」 | 「明年1月1日，我送你車」 |
| 效力影響 | 條件成就→效力發生/消滅 | 期限屆至→效力發生/消滅 |

### 為什麼會搞混？

兩者都是「附加在法律行為上的約款」，讓效力不立即發生或不永久存續。差別在「確不確定」。

### 關鍵差異深度解析

**條件**分為：
- 停止條件（§99 I）：條件成就時法律行為「生效」。如：考上律師就送車。
- 解除條件（§99 II）：條件成就時法律行為「失效」。如：你結婚前可以住這間房，結婚就搬走。

**期限**分為：
- 始期（§102 I）：期限到了開始生效。如：明年起租給你。
- 終期（§102 II）：期限到了停止生效。如：租約到年底。

### 案例對比

| 約款內容 | 條件 or 期限？ | 理由 |
|----------|---------------|------|
| 「你考上律師我就送你車」 | 停止條件 | 不確定會不會考上 |
| 「明年1月1日送你車」 | 始期 | 確定會到 |
| 「你死的時候這房子給你兒子」 | 期限（不確定期限） | 人一定會死，只是不知道何時→確定會發生 |
| 「如果明天下雨就不用來上班」 | 停止條件 | 不確定會不會下雨 |

### 邊界案例

「死亡」是條件還是期限？人一定會死→確定會發生→是**期限**（不確定期限），不是條件。這是經典考題。

### 考試出題陷阱

- 陷阱一：「不確定期限」（確定會發生但不知何時）是期限不是條件。
- 陷阱二：不法條件（§101 II 以不正當行為促成條件成就）→視為條件不成就。以不正當行為阻止條件成就→視為條件已成就（§101 I）。

## 混淆組 19：婚生子女 vs 非婚生子女 vs 養子女

### 一眼判斷法（30秒速查）

| 判斷條件 | → 婚生子女（§1061） | → 非婚生子女（§1065） | → 養子女（§1072） |
|----------|-------------------|---------------------|-----------------|
| 父母有婚姻關係？ | 有 | 無 | 不論（透過收養創設） |
| 親子關係來源 | 血緣 + 婚姻推定 | 血緣（認領/準正後同婚生） | 法律行為（收養） |
| 成立方式 | 受胎期間在婚姻中（§1062推定） | 生父認領或法院判決 | 收養契約 + 法院認可 |

### 為什麼會搞混？

三者在「權利義務」上逐步趨同（非婚生子女經認領後與婚生同），但成立要件和身分確認程序完全不同。

### 關鍵差異深度解析

**婚生推定**（§1063）：妻之受胎，係在婚姻關係存續中者，推定其所生子女為婚生子女。丈夫若否認，須提起「否認之訴」（§1063 II，知悉後2年內）。

**非婚生子女**：生父可以「認領」（§1065），認領後視為婚生子女。未認領者，生母或子女可提起「認領之訴」（§1067）。經生父母結婚者，視為婚生（準正，§1064）。

**養子女**：透過收養契約（§1072以下）＋法院認可（§1079）成立。收養效力：養子女與養父母的關係與婚生子女同（§1077）。但養子女與本生父母的權利義務於收養關係存續中停止（§1077 II）。

### 案例對比

| 情境 | 身分 | 繼承權 |
|------|------|--------|
| 甲乙已婚，乙生下丙 | 丙為婚生子女 | 當然繼承甲乙遺產 |
| 甲乙未婚，乙生下丙，甲認領丙 | 丙為非婚生子女經認領→視為婚生 | 同婚生子女 |
| 甲乙已婚，收養丁 | 丁為養子女 | 繼承甲乙遺產，與本生父母的繼承關係停止 |

### 邊界案例

「代孕」所生子女的親子關係：目前我國無代孕專法，依現行法以「分娩者為母」為原則。即使卵子來自委託者，生出孩子的代理孕母是法律上的母親。委託方需透過收養程序取得親子關係。

### 實務見解

司法院大法官釋字第587號解釋：子女獲知其血統來源之權利，應受憲法保障。因此，即使超過否認之訴的除斥期間，在特殊情形下仍有司法救濟途徑。此見解對婚生推定制度形成重要補充。

### 考試出題陷阱

- 陷阱一：婚生否認之訴的「2年除斥期間」從知悉起算，不是從出生起算。但有上限規定：子女出生之日起10年。
- 陷阱二：養子女與本生父母的權利義務是「停止」不是「消滅」——終止收養後會恢復。
- 陷阱三：非婚生子女的「認領」是單方行為（生父的意思表示即可），不需要子女或生母同意。但生父認領後不得撤銷（§1070）。
- 陷阱四：收養需要法院認可（§1079），這是民國101年修法後的重大改變。未經法院認可的收養無效。

## 混淆組 20：應繼分 vs 特留分

### 一眼判斷法（30秒速查）

| 判斷條件 | → 應繼分（§1141-§1144） | → 特留分（§1223） |
|----------|------------------------|------------------|
| 是什麼？ | 法定繼承比例（沒有遺囑時按此分配） | 最低保障比例（有遺囑也不能低於此） |
| 可以被遺囑排除嗎？ | 可以（遺囑自由） | 不可以（強制保障） |
| 特留分比例 | — | 通常是應繼分的1/2（直系血親卑親屬、父母、配偶）或1/3（兄弟姐妹、祖父母） |
| 救濟方式 | — | 特留分被侵害→扣減權（§1225） |

### 為什麼會搞混？

兩者都和「繼承比例」有關，名稱相似，但功能完全不同。

### 關鍵差異深度解析

**應繼分**是「沒有遺囑時的預設分配方案」。例如配偶＋兩個子女：配偶和子女均分→各得1/3。

**特留分**是「即使有遺囑也不能低於的底線」。被繼承人不能把遺產100%給陌生人而讓自己的孩子一毛都沒有。特留分制度保障法定繼承人的最低權利。

### 案例對比

**情境**：甲（被繼承人）有配偶乙、兒子丙、女兒丁。遺產1200萬。

**沒有遺囑**：依應繼分→乙、丙、丁各得400萬（§1144：配偶與直系血親卑親屬均分）。

**甲立遺囑「全部給慈善機構」**：
- 乙的特留分 = 應繼分(1/3) x 1/2 = 1/6 = 200萬
- 丙的特留分 = 同上 = 200萬
- 丁的特留分 = 同上 = 200萬
- 三人合計特留分 = 600萬
- 遺囑侵害特留分→乙丙丁可行使扣減權（§1225），從遺贈中扣回各自的200萬
- 慈善機構最多拿600萬

**微調**：如果甲的遺囑是「800萬給慈善機構，400萬給乙」——丙丁的特留分各200萬被侵害，乙的400萬超過特留分所以沒被侵害。丙丁可以扣減800萬中的400萬。

### 邊界案例

特留分的扣減權（§1225）是「形成權」，一經行使即生效力。但扣減的對象有優先順序：先扣遺贈，再扣死因贈與，最後扣生前特種贈與。實務上繼承人常忽略行使扣減權的時間限制（知悉後2年），錯過就無法主張。

另一個邊界：配偶的應繼分因與不同順位繼承人共同繼承而不同。
- 與第一順位（直系血親卑親屬）共同繼承→均分（§1144第1款）
- 與第二順位（父母）或第三順位（兄弟姐妹）共同繼承→配偶得1/2（§1144第2款）
- 與第四順位（祖父母）共同繼承→配偶得2/3（§1144第3款）
- 無其他繼承人→配偶得全部

### 實務見解

最高法院58年台上字第1279號判例：繼承人之特留分，如因被繼承人所為之遺贈，致受侵害者，得按其不足之數由遺贈財產扣減之。此為「比例扣減」而非「全部無效」——遺贈不會因為侵害特留分而全部無效，只是扣減到特留分獲得滿足為止。

### 考試出題陷阱

- 陷阱一：特留分是應繼分的「一定比例」，不是遺產的比例。先算應繼分再乘比例。
- 陷阱二：§1223各款的特留分比例不同——直系血親卑親屬、配偶、父母是1/2；兄弟姐妹、祖父母是1/3。
- 陷阱三：扣減權是對「遺贈」或「應繼分指定」行使，不能對「生前贈與」行使（除非§1173歸扣的特種贈與）。

# 附錄：20組概念速查決策樹

以下是全部20組混淆概念的快速判斷流程，考前30分鐘速看。

```
【債法】

Q1: 債務人沒有好好履行→是哪種？
    ├── 根本做不到了 → 給付不能（§226）
    ├── 做得到但拖延 → 給付遲延（§229）
    └── 做了但有問題 → 不完全給付（§227）

Q2: 同一事件要走契約還是侵權？
    ├── 有契約關係 → 可走債務不履行（舉證較輕）
    ├── 無契約關係 → 只能走侵權（§184）
    └── 都有 → 請求權競合，兩條都寫

Q3: 代理人沒有代理權→結果？
    ├── 本人有可歸責外觀 + 第三人善意 → 表見代理（§169）
    └── 本人無可歸責外觀 → 無權代理（§170）→ 效力未定

Q4: 有人拿了不該拿的東西→走哪條路？
    ├── 看「他得到什麼」→ 不當得利（§179）
    └── 看「你損失什麼」→ 侵權行為（§184）

Q5: 買到瑕疵品→三條路？
    ├── 不問過失、6個月時效 → 瑕疵擔保（§354）
    ├── 要證明可歸責、15年時效 → 不完全給付（§227）
    └── 要證明故意欺騙 → 詐欺撤銷（§92）

Q6: 花錢請人做事→什麼契約？
    ├── 要完成成果 → 承攬（§490）
    ├── 要處理事務 → 委任（§528）
    └── 要提供勞力 → 僱傭（§482）

Q7: 契約要結束→解除 or 終止？
    ├── 一次性契約，要全部還原 → 解除
    └── 繼續性契約，往後不做了 → 終止

Q8: 契約中有人要換→換什麼？
    ├── 債權人換 → 債權讓與（§294）
    ├── 債務人換 → 債務承擔（§300）
    └── 整個地位換 → 契約承擔

Q9: 我想先不給→什麼抗辯？
    ├── 對方也沒給 → 同時履行抗辯（§264）
    └── 對方快破產了 → 不安抗辯（§265）

Q10: 多人負同一債務→什麼關係？
    ├── 同一原因 → 連帶債務（§272）
    └── 不同原因、同一給付 → 不真正連帶債務

【物權】

Q11: 簽約 vs 交付/登記→分別是什麼？
    ├── 簽約 → 債權行為
    └── 交付/登記 → 物權行為

Q12: 非所有人取得所有權→怎麼取得？
    ├── 從無權處分人處即時取得 → 善意取得（§801+§948）
    └── 長期占有後取得 → 時效取得（§768）

Q13: 擔保物權→哪一種？
    ├── 不用占有、登記設定 → 抵押權（§860）
    ├── 要交付占有、約定設定 → 質權（§884）
    └── 合法占有中、法律直接賦予 → 留置權（§928）

Q14: 「我拿著東西」=「東西是我的」？
    ├── 法律上歸屬 → 所有權
    └── 事實上支配 → 占有（不等於所有權）

【總則】

Q15: 意思表示有問題→什麼瑕疵？
    ├── 自己搞錯（不知情）→ 錯誤（§88）
    ├── 被別人騙 → 詐欺（§92）
    ├── 被別人逼 → 脅迫（§92）
    ├── 自己知道是假的（對方不知）→ 單獨虛偽（§86）
    └── 雙方都知道是假的 → 通謀虛偽（§87）

Q16: 法律行為的效力狀態？
    ├── 從頭到尾都沒效 → 無效
    ├── 先有效，可以選擇打掉 → 得撤銷
    └── 暫時不確定，等某人決定 → 效力未定

Q17: 替別人做法律行為→什麼角色？
    ├── 自己形成意思表示 → 代理（§103）
    ├── 作為法人的機關 → 代表
    └── 只是傳話 → 使者

Q18: 法律行為附加的約款→什麼？
    ├── 不確定會不會發生 → 條件（§99）
    └── 確定會到來 → 期限（§102）

【身分法】

Q19: 親子關係→什麼類型？
    ├── 父母有婚姻 + 受胎期間在婚姻中 → 婚生子女
    ├── 父母無婚姻 → 非婚生子女（認領後視為婚生）
    └── 透過法院認可的收養 → 養子女

Q20: 繼承比例→應繼分 or 特留分？
    ├── 沒有遺囑時的法定分配比例 → 應繼分
    └── 有遺囑也不能低於的最低保障 → 特留分
```

# 結語：混淆概念的學習策略

**第一步：先建立「預設判斷」。** 拿到一個事實，30秒內用「一眼判斷法」做初步分類。

**第二步：檢查「邊界條件」。** 初步分類後，問自己：「有沒有可能是另一個？」把微調條件一個個檢查。

**第三步：寫答案時「先排除再肯定」。** 考試作答時，先說明「本題非X，因為缺少某要件」，再論證「應為Y，因為符合全部要件」。這種寫法最能展現你清楚區分的能力，也是閱卷老師最想看到的。

**第四步：多組混淆交叉練習。** 現實考題不會乖乖只考一組。可能同一題裡，你要先判斷「是承攬還是委任」，再判斷「是不完全給付還是給付遲延」，最後判斷「走契約還是走侵權」。交叉練習才能真正內化。

> 法律概念的邊界，就是考試分數的來源。把每一條邊界都畫清楚，你就贏了。
