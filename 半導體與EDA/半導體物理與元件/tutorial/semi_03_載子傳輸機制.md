# 第三章：載子傳輸機制

> **學習目標**：掌握半導體中電流產生的兩大機制——漂移與擴散，以及 Einstein 關係和 Hall 效應。
> **預備知識**：第二章（載子濃度計算）、歐姆定律
> **預估時間**：3～4 小時

---

## 🔰 本章基礎觀念（零基礎必讀）

### 為什麼要學傳輸機制？

上一章我們知道了半導體裡有多少電子和電洞。但光知道「有多少」還不夠——元件的電流（I-V 特性）取決於載子**怎麼移動**。

半導體中有兩種驅動載子移動的力：

1. **電場力** → 產生**漂移電流（Drift Current）**
2. **濃度梯度** → 產生**擴散電流（Diffusion Current）**

### 一分鐘抓住核心

```
漂移（Drift）：電場推著載子走
  → J_drift = σE = q(nμn + pμp)E
  → 像風吹著船走

擴散（Diffusion）：載子從多的地方往少的地方跑
  → Jn = qDn(dn/dx)
  → 像茶包在水中自動擴散開

兩者的橋樑：Einstein 關係
  → Dn/μn = kT/q = VT ≈ 26 mV (300K)
```

## 一、漂移電流（Drift Current）

### 1.1 基本物理圖像

當半導體施加外部電場 E（V/cm）時：

- 電子（帶負電）被電場**反方向**加速
- 電洞（帶正電）被電場**同方向**加速
- 但載子會不斷與晶格和雜質碰撞，達到一個穩定的**漂移速度**

### 1.2 漂移速度與遷移率

**漂移速度（Drift Velocity）**：

$$v_d = \mu E$$

其中 μ 是**遷移率（Mobility）**，單位：cm²/(V·s)。

遷移率越大，載子在相同電場下移動越快。

| 材料 | μn (cm²/V·s) | μp (cm²/V·s) | 說明 |
|------|-------------|-------------|------|
| Si   | 1350        | 480         | 室溫，低摻雜 |
| GaAs | 8500        | 400         | 電子遷移率是 Si 的 6 倍 |
| GaN  | 1000        | 30          | 寬能隙材料 |
| Ge   | 3900        | 1900        | 遷移率最高的常見半導體 |

### 1.3 漂移電流密度

**電子漂移電流密度**：

$$J_{n,drift} = q \cdot n \cdot \mu_n \cdot E$$

**電洞漂移電流密度**：

$$J_{p,drift} = q \cdot p \cdot \mu_p \cdot E$$

**總漂移電流密度**：

$$J_{drift} = q(n\mu_n + p\mu_p)E = \sigma E$$

其中 **σ = q(nμn + pμp)** 是電導率（Conductivity），單位：(Ω·cm)⁻¹。

**注意**：電子帶負電但移動方向和電場相反，兩個負號抵銷，所以電子和電洞的漂移電流方向相同（都是電場方向）。

### 1.4 電阻率

電阻率 ρ = 1/σ：

$$\rho = \frac{1}{q(n\mu_n + p\mu_p)}$$

對 N 型半導體（n >> p）：

$$\rho \approx \frac{1}{qN_d\mu_n}$$

對 P 型半導體（p >> n）：

$$\rho \approx \frac{1}{qN_a\mu_p}$$

## 二、遷移率的物理與限制

### 2.1 散射機制

載子在晶體中不是自由飛行的，會被各種「障礙物」散射（碰撞）：

| 散射機制 | 溫度依賴性 | 說明 |
|---------|----------|------|
| 晶格散射（Lattice Scattering） | μL ∝ T⁻³/² | 溫度越高，晶格振動越大，散射越多 |
| 雜質散射（Impurity Scattering） | μI ∝ T³/²/Ni | 溫度越高，載子速度越快，反而不易被雜質偏折 |

- **晶格散射（聲子散射）**：原子在平衡位置附近振動（聲子），溫度越高振動越劇烈
- **雜質散射**：摻雜的施體或受體離子帶電，會偏折路過的載子

### 2.2 Matthiessen 規則

多種散射機制同時存在時，遷移率的合成：

$$\frac{1}{\mu} = \frac{1}{\mu_L} + \frac{1}{\mu_I}$$

**白話理解**：就像一條路上有兩種障礙物，總減速效果是各自效果的疊加。遷移率由「最差的那個」主導。

```
遷移率 μ
(log) ↑
      |         ╱ μI ∝ T^(3/2)/Ni
      |        ╱
      |      ╱
      | ○○○○─── μtotal（取低的那個）
      |○        ╲
      |          ╲ μL ∝ T^(-3/2)
      |           ╲
      └─────────────→ T (溫度)
      低溫：        高溫：
      雜質散射主導   晶格散射主導
```

### 2.3 遷移率與摻雜濃度的關係

```
遷移率 μ
(cm²/V·s)
1400 ┤ ────────
     │          ╲  μn (電子)
1000 ┤           ╲
     │            ╲
 600 ┤──────       ╲
     │      ╲       ╲
 400 ┤ μp    ╲       ╲
     │ (電洞) ╲       ╲
 200 ┤          ╲       ╲
     │           ╲───────╲───
 100 ┤
     └──┬────┬────┬────┬────→ 摻雜濃度 (cm⁻³)
       10¹⁴ 10¹⁵ 10¹⁶ 10¹⁷ 10¹⁸
```

- 低摻雜（< 10¹⁵）：μ 幾乎不變（晶格散射主導）
- 高摻雜（> 10¹⁷）：μ 明顯下降（雜質散射主導）

### 2.4 高電場效應：速度飽和

在低電場下，v = μE（線性關係）。但電場增大到某個程度時：

$$v_{sat} \approx 10^7 \text{ cm/s}$$

載子速度不再增加，達到**飽和速度**。這在短通道 MOSFET 中非常重要。

```
漂移速度 v
      ↑
vsat  │─ ─ ─ ─ ─ ─ ─ ○○○○○○○
      │            ○
      │         ○
      │      ○  ← 斜率 = μ
      │   ○
      │ ○
      └──────────────────→ 電場 E
          低電場：v=μE    高電場：v→vsat
```

## 三、擴散電流（Diffusion Current）

### 3.1 基本物理圖像

即使沒有電場，如果載子濃度分佈不均勻，載子會自發地從**高濃度區**移動到**低濃度區**——這就是擴散。

**日常類比**：在房間一角點一根香，煙味會慢慢擴散到整個房間，不需要風扇（電場）。

### 3.2 擴散電流密度

**電子擴散電流密度**：

$$J_{n,diff} = qD_n\frac{dn}{dx}$$

**電洞擴散電流密度**：

$$J_{p,diff} = -qD_p\frac{dp}{dx}$$

其中 Dn, Dp 是**擴散係數（Diffusion Coefficient）**，單位：cm²/s。

**注意正負號**：
- 電子帶負電，往低濃度方向擴散，電流方向相反 → 正號
- 電洞帶正電，往低濃度方向擴散，電流方向相同 → 負號（因為 dp/dx < 0 而電流向正方向）

### 3.3 Einstein 關係

擴散係數和遷移率之間存在深刻的聯繫：

$$\frac{D_n}{\mu_n} = \frac{D_p}{\mu_p} = \frac{kT}{q} = V_T$$

其中 **VT = kT/q ≈ 26 mV**（室溫 300K）稱為**熱電壓（Thermal Voltage）**。

這是一個非常優美的物理關係：漂移（有序運動）和擴散（無序運動）本質上都來自載子的熱運動。

**Si 在 300K 的數值**：

| 參數 | 電子 | 電洞 |
|------|------|------|
| μ (cm²/V·s) | 1350 | 480 |
| D (cm²/s) | 1350 × 0.026 = 35.1 | 480 × 0.026 = 12.5 |

## 四、總電流

### 4.1 總電流密度

半導體中的總電流是漂移和擴散的疊加：

$$J_n = qn\mu_nE + qD_n\frac{dn}{dx} \quad \text{（電子總電流）}$$

$$J_p = qp\mu_pE - qD_p\frac{dp}{dx} \quad \text{（電洞總電流）}$$

$$J_{total} = J_n + J_p$$

### 4.2 什麼時候哪個重要？

| 情境 | 主要電流機制 | 例子 |
|------|------------|------|
| 均勻摻雜 + 外加電場 | 漂移 | 電阻器 |
| 不均勻摻雜，無外加電場 | 擴散 | PN 接面的少數載子注入 |
| 一般情況 | 兩者都有 | MOSFET 通道 |

### 4.3 平衡時的漂移-擴散平衡

在熱平衡下，沒有淨電流流動。這意味著：

$$J_n = qn\mu_nE + qD_n\frac{dn}{dx} = 0$$

漂移電流和擴散電流完全抵銷。這個條件正是 Einstein 關係的來源之一，也是 PN 接面內建電場的形成原因。

## 五、Hall 效應

### 5.1 基本原理

Hall 效應是測量半導體載子類型和濃度的標準方法。

當電流 I 通過導體，且有垂直的磁場 B 施加時，載子會受到 Lorentz 力而偏向一側，在橫向建立一個電壓差——這就是 **Hall 電壓 VH**。

```
         ← w →
    ┌────────────┐
    │   ↑B(向上) │ t
    │   ⊙⊙⊙⊙⊙⊙ │ (厚度)
    │            │
 I→ │  ─→─→─→─  │ →I
    │   載子流動  │
    │            │
    │ ⊕         ⊖│ ← Hall 電壓 VH
    └────────────┘
         ← w →

N型：電子偏向一側，VH 正
P型：電洞偏向另一側，VH 負
→ 由 VH 的正負可判斷載子類型
```

### 5.2 Hall 係數

**Hall 係數（Hall Coefficient）**：

$$R_H = \frac{V_H \cdot t}{I \cdot B}$$

- N 型半導體：RH = **-1/(qn)** （負值）
- P 型半導體：RH = **+1/(qp)** （正值）

### 5.3 從 Hall 測量得到的資訊

1. **載子類型**：由 VH（或 RH）的正負號判斷
2. **載子濃度**：n = 1/(q|RH|)（或 p = 1/(q|RH|)）
3. **Hall 遷移率**：μH = |RH| · σ = |RH| / ρ

### 5.4 Hall 效應的實際應用

- 半導體材料的例行測試
- 磁場感測器（Hall sensor）：在手機、汽車中廣泛使用
- 量子 Hall 效應：精密測量基本常數

## 數值例題

### 【例題 1】電阻率計算

**題目**：N 型 Si，Nd = 10¹⁶ cm⁻³，μn = 1200 cm²/V·s。求電阻率。

**解答**：

```
N型，n >> p，可忽略電洞的貢獻

σ = qnμn = (1.6×10⁻¹⁹)(10¹⁶)(1200)
  = 1.92 (Ω·cm)⁻¹

ρ = 1/σ = 1/1.92 = 0.521 Ω·cm
```

**答案：ρ ≈ 0.52 Ω·cm**

---

### 【例題 2】漂移電流密度

**題目**：Si 樣本中 n = 10¹⁶ cm⁻³，p = 10⁴ cm⁻³，μn = 1200 cm²/V·s，μp = 400 cm²/V·s。若施加電場 E = 100 V/cm，求漂移電流密度。

**解答**：

```
Jn,drift = qnμnE = (1.6×10⁻¹⁹)(10¹⁶)(1200)(100)
         = 1.92 A/cm²

Jp,drift = qpμpE = (1.6×10⁻¹⁹)(10⁴)(400)(100)
         = 6.4×10⁻¹⁴ A/cm²（可忽略）

Jtotal ≈ Jn,drift = 1.92 A/cm²
```

**重點**：少數載子的漂移電流通常可以忽略（差了 12 個數量級）。

---

### 【例題 3】Einstein 關係與擴散電流

**題目**：Si 在 300K，μn = 1350 cm²/V·s。電子濃度從 x=0 的 10¹⁸ cm⁻³ 線性下降到 x=1μm 的 10¹⁶ cm⁻³。求擴散電流密度。

**解答**：

```
步驟 1：由 Einstein 關係求 Dn
Dn = μn × VT = 1350 × 0.026 = 35.1 cm²/s

步驟 2：求濃度梯度
dn/dx = (10¹⁶ - 10¹⁸) / (10⁻⁴ cm)
      = -9.9×10¹⁷ / 10⁻⁴
      = -9.9×10²¹ cm⁻⁴

步驟 3：計算擴散電流
Jn,diff = qDn(dn/dx)
        = (1.6×10⁻¹⁹)(35.1)(-9.9×10²¹)
        = -5.56×10⁴ A/cm² ← 方向為 -x（從高濃度到低濃度）

|Jn,diff| = 55.6 kA/cm²
```

**注意**：這個電流密度非常大！實際上這麼大的濃度梯度不會長期維持（會很快被擴散平衡掉）。

---

### 【例題 4】Hall 效應測量

**題目**：一塊半導體樣本，厚度 t = 0.1 cm，通過電流 I = 1 mA，施加磁場 B = 0.5 T。測得 Hall 電壓 VH = +5 mV。求載子類型、濃度和 Hall 遷移率（已知 ρ = 5 Ω·cm）。

**解答**：

```
步驟 1：載子類型
VH > 0 → Hall 係數 RH > 0 → P 型半導體

步驟 2：Hall 係數
RH = VH × t / (I × B)
   = (5×10⁻³)(0.1) / (10⁻³ × 0.5)
   = 5×10⁻⁴ / 5×10⁻⁴
   = 1.0 cm³/C

步驟 3：載子濃度
p = 1/(q × RH) = 1/(1.6×10⁻¹⁹ × 1.0)
  = 6.25×10¹⁸ cm⁻³

步驟 4：Hall 遷移率
μH = RH / ρ = 1.0 / 5 = 0.2 cm²/(V·s) × 10⁴ ...

等等，讓我重新檢查單位。
RH = 1.0 cm³/C，ρ = 5 Ω·cm
μH = |RH| / ρ = 1.0 / 5 = 0.2 cm²/(V·s) × ...

注意：RH 的單位是 cm³/C = cm³/(A·s)
σ = 1/ρ = 0.2 (Ω·cm)⁻¹

μH = |RH| × σ = 1.0 × 0.2 = 200 cm²/(V·s)
```

**答案**：P 型，p = 6.25×10¹⁸ cm⁻³，μH = 200 cm²/(V·s)

---

### 【例題 5】Matthiessen 規則

**題目**：某 Si 樣本在 300K，晶格散射遷移率 μL = 1500 cm²/V·s，雜質散射遷移率 μI = 600 cm²/V·s。求總遷移率。

**解答**：

```
1/μ = 1/μL + 1/μI = 1/1500 + 1/600

1/μ = 0.000667 + 0.001667 = 0.002333

μ = 1/0.002333 = 428.6 cm²/(V·s)
```

**觀察**：總遷移率（429）比兩者中較小的（600）還要低。遷移率由最強的散射機制主導，但不是簡單取最小值。

## ✅ 自我檢測

### 基礎題

**Q1**：漂移電流和擴散電流分別由什麼驅動？

<details>
<summary>點擊查看答案</summary>

- **漂移電流**：由**電場（Electric Field）**驅動。J_drift = σE
- **擴散電流**：由**濃度梯度（Concentration Gradient）**驅動。Jn_diff = qDn(dn/dx)

兩者可以同時存在，也可以互相抵銷。

</details>

**Q2**：Einstein 關係是什麼？室溫下 VT 是多少？

<details>
<summary>點擊查看答案</summary>

$$\frac{D}{\mu} = \frac{kT}{q} = V_T$$

室溫（300K）下：VT ≈ 0.026 V = **26 mV**

這個關係連結了擴散（D）和漂移（μ），本質上都源自載子的熱運動。

</details>

**Q3**：為什麼溫度升高時，晶格散射使遷移率下降，但雜質散射使遷移率上升？

<details>
<summary>點擊查看答案</summary>

- **晶格散射**：溫度升高 → 原子振動更劇烈（聲子增多） → 散射更頻繁 → μL ∝ T⁻³/² 下降
- **雜質散射**：溫度升高 → 載子熱速度更快 → 通過雜質附近的時間更短 → 被偏折的角度更小 → μI ∝ T³/²/Ni 上升

兩者的溫度依賴性剛好相反，在某個溫度有最大遷移率。

</details>

### 進階題

**Q4**：為什麼 GaAs 的電子遷移率比 Si 高很多，但 GaAs 沒有取代 Si 做邏輯晶片？

<details>
<summary>點擊查看答案</summary>

GaAs 的 μn = 8500 cm²/V·s（Si 的 6 倍多），原因是 GaAs 的電子有效質量很小（mn* = 0.067m₀ vs Si 的 1.08m₀）。

但 GaAs 沒有取代 Si 做邏輯晶片的原因：
1. GaAs 沒有天然的優質氧化物（Si 有 SiO₂），做 MOSFET 困難
2. GaAs 的電洞遷移率不高，CMOS 需要 NMOS+PMOS 配對
3. GaAs 晶圓貴，且機械性質較差（容易碎）
4. Si 已有數十年的成熟製程和生態系

GaAs 主要用於 RF 功率放大器和光電元件。

</details>

**Q5**：一塊 N 型 Si，Nd = 5×10¹⁵ cm⁻³，μn = 1300 cm²/V·s，截面積 A = 10⁻⁴ cm²，長度 L = 0.5 cm。求電阻值。

<details>
<summary>點擊查看答案</summary>

```
ρ = 1/(qNdμn) = 1/(1.6×10⁻¹⁹ × 5×10¹⁵ × 1300)
  = 1/(1.04) = 0.962 Ω·cm

R = ρL/A = 0.962 × 0.5 / 10⁻⁴
  = 4808 Ω ≈ 4.8 kΩ
```

</details>

**Q6**：Hall 效應可以測量本質半導體的載子濃度嗎？可能遇到什麼困難？

<details>
<summary>點擊查看答案</summary>

理論上可以，但實際困難：

在本質半導體中，n = p = ni，電子和電洞同時存在且濃度相等。由於電子和電洞受磁場偏向**同一側**（因為電荷符號和漂移方向都相反，效果疊加），但它們的 Hall 電壓正負號**相反**，會部分抵銷。

實際上，Hall 電壓為：
RH = (pμp² - nμn²) / [q(pμp + nμn)²]

對本質半導體（n=p=ni）：
RH = (μp² - μn²) / [q·ni·(μp + μn)²]

由於 μn > μp，RH < 0（看起來像 N 型），而且 |RH| 被縮小了。所以 Hall 效應對本質半導體的測量精度較差。

</details>

## 本章重點整理

```
1. 漂移電流：J_drift = q(nμn + pμp)E = σE
2. 遷移率：低摻雜用晶格散射，高摻雜用雜質散射
3. Matthiessen規則：1/μ = 1/μL + 1/μI
4. 高電場→速度飽和 vsat ≈ 10⁷ cm/s
5. 擴散電流：Jn = qDn(dn/dx), Jp = -qDp(dp/dx)
6. Einstein關係：D/μ = kT/q = VT ≈ 26mV
7. 總電流 = 漂移 + 擴散
8. Hall效應：RH正→P型，RH負→N型
9. 載子濃度：n(或p) = 1/(q|RH|)
10. Si: μn=1350, μp=480 cm²/(V·s) — 常用數值
```
