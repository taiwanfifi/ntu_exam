## 1. Beta 分布 — 萬物皆可 Beta

**小說出處**：第1章（透鏡日常）、第20-21章（密碼 Beta(72,28)）

### 什麼是 Beta 分布？

Beta 分布是定義在 [0, 1] 區間上的連續機率分布，專門用來描述「機率的機率」。

$$f(\theta; \alpha, \beta) = \frac{\theta^{\alpha-1}(1-\theta)^{\beta-1}}{B(\alpha, \beta)}$$

其中 $B(\alpha, \beta) = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}$ 是 Beta 函數。

### 直覺理解

想像你在猜一枚硬幣正面朝上的機率 $\theta$：

- **你完全不知道** → Beta(1, 1)，就是均勻分布，任何值都一樣可能
- **你猜大概 0.5** → Beta(5, 5)，對稱地集中在 0.5 附近
- **你有 72 次成功、28 次失敗的證據** → Beta(72, 28)，非常確定 θ ≈ 0.72

### 關鍵公式

| 性質 | 公式 | 第1章數值 | 第21章數值 |
|------|------|-----------|-----------|
| 期望值 | $E[\theta] = \frac{\alpha}{\alpha+\beta}$ | $\frac{6.2}{10} = 0.62$ | $\frac{72}{100} = 0.72$ |
| 變異數 | $\text{Var}(\theta) = \frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}$ | ≈ 0.0096 | ≈ 0.002 |
| 眾數 | $\frac{\alpha-1}{\alpha+\beta-2}$ (α,β > 1) | ≈ 0.65 | ≈ 0.724 |

### 數值範例：跟著小說算

**第1章的 Beta(6.2, 3.8)**：

```
α = 6.2, β = 3.8, α + β = 10

期望值 = 6.2 / 10 = 0.62（透鏡顯示：62%）
標準差 = √(6.2 × 3.8 / (100 × 11)) ≈ 0.098

95% 的機率，真實值在 0.62 ± 2×0.098 = [0.424, 0.816] 之間
```

**第21章的 Beta(72, 28)**——小說的關鍵密碼：

```
α = 72, β = 28, α + β = 100

期望值 = 72 / 100 = 0.72
  → 72% 的區域先驗被修改過！

變異數 = 72 × 28 / (100² × 101) ≈ 0.002
標準差 ≈ 0.045

分布非常集中——α+β = 100 代表「很有信心」
如果 α+β = 10，標準差會是 0.15（三倍寬）
```

![Beta Distribution](圖表/01_beta_distribution.png)

> 「百分之六十二。不高不低。剛好是一個你不會太在意、但也不會完全忽略的數字。」——第一章，貝葉的透鏡日常

### 為什麼 α + β 代表信心？

$\alpha + \beta$ 可以理解為「等效觀測次數」：

- Beta(1, 1)：0 次觀測 → 什麼都不知道
- Beta(6.2, 3.8)：約 8 次觀測 → 有點感覺
- Beta(72, 28)：100 次觀測 → 非常確定

每多一次觀測，分布就更窄、更集中。這就是為什麼 Beta(72,28) 的圖形像一根針——100 次觀測的信心，讓不確定性壓縮到只剩 σ ≈ 0.045。

## 2. 貝氏定理 — 小說的靈魂公式

**小說出處**：第4章（十一秒）、第11章（父親的定理）

### 公式

$$P(\theta | D) = \frac{P(D | \theta) \cdot P(\theta)}{P(D)}$$

或者用比例式：

$$\underbrace{P(\theta | D)}_{\text{後驗}} \propto \underbrace{P(\theta)}_{\text{先驗}} \times \underbrace{P(D | \theta)}_{\text{似然}}$$

### 中文翻譯

> **你更新後的信念** = **你原本的信念** × **新證據的說服力**

### 小說怎麼用的

在第4章，貝葉用十一秒做了一次貝氏更新：

```
場景：路口有人跟蹤嗎？

先驗 P(跟蹤) = 0.03（平常被跟蹤的背景機率很低）

新證據：同一個人出現在三個路口
似然 P(出現在三個路口 | 跟蹤) = 0.85
似然 P(出現在三個路口 | 巧合) = 0.02

後驗 P(跟蹤 | 三個路口) = (0.85 × 0.03) / (0.85 × 0.03 + 0.02 × 0.97)
                        = 0.0255 / (0.0255 + 0.0194)
                        = 0.0255 / 0.0449
                        ≈ 0.568

從 3% 跳到 56.8%！一個觀測改變了一切。
```

### P(A|B) ≠ P(B|A)——最致命的混淆

第4章特別強調：

```
P(下雨 | 帶傘) ≠ P(帶傘 | 下雨)

P(帶傘 | 下雨) 可能很高（80%——下雨大家帶傘）
P(下雨 | 帶傘) 可能很低（20%——帶傘的人很多是預防性帶的）

這兩個不一樣！混淆它們是統計學最常見的錯誤。
```

> 「後驗正比於先驗乘以似然。先驗是你的。似然是我的。後驗是我們的。」——第五十九章，紙條背面

## 3. 似然函數與最大似然估計 (MLE)

**小說出處**：第5章（太整齊了）、第7章（似然函數）

### 什麼是似然函數？

似然函數和機率密度函數長得一模一樣——但意思完全不同：

- **機率**：固定參數 θ，問「數據 x 出現的可能性有多大？」→ $f(x | \theta)$
- **似然**：固定數據 x，問「哪個參數 θ 最有可能產生這些數據？」→ $L(\theta | x) = f(x | \theta)$

### MLE：找到讓似然最大的 θ

$$\hat{\theta}_{MLE} = \arg\max_\theta L(\theta | x_1, ..., x_n) = \arg\max_\theta \prod_{i=1}^{n} f(x_i | \theta)$$

實務上取對數更方便：

$$\hat{\theta}_{MLE} = \arg\max_\theta \sum_{i=1}^{n} \log f(x_i | \theta)$$

### 小說怎麼用的（第7章）

蘇立恆計算 Poisson 似然函數：

```
觀測數據：某區域的事件次數
MLE 結果：λ̂ = 2.63
官方公佈：λ = 1.8

偏差 = |2.63 - 1.8| / 1.8 = 46%

似然比 = L(λ=1.8) / L(λ=2.63) = 0.218

意思是：用官方的 λ=1.8 來解釋數據，
只有 MLE 解釋力的 21.8%。

官方數字——是錯的。
```

> 「MLE 是 2.63。官方的 λ 是 1.8。偏差百分之四十六。這不是雜訊。這是有人在說謊。」——第七章，蘇立恆

![Likelihood Function](圖表/04_likelihood_function.png)

### Poisson 分布的 MLE

對 Poisson 分布 $P(X=k) = \frac{e^{-\lambda}\lambda^k}{k!}$：

$$\log L(\lambda) = \sum_{i=1}^n (x_i \log\lambda - \lambda - \log x_i!)$$

對 λ 微分令其等於零：

$$\frac{d}{d\lambda} \log L = \frac{\sum x_i}{\lambda} - n = 0$$

$$\hat{\lambda}_{MLE} = \bar{x} = \frac{\sum x_i}{n}$$

**結論：Poisson 的 MLE 就是樣本平均值！**

## 4. Poisson 分布 — 稀有事件的數學

**小說出處**：第4-5章、第53章

### 公式

$$P(X = k) = \frac{e^{-\lambda} \lambda^k}{k!}, \quad k = 0, 1, 2, ...$$

### 性質

| 性質 | 公式 | λ=1.67 (Ch.5) | λ=0.58 (Ch.53) |
|------|------|---------------|----------------|
| 期望值 | E[X] = λ | 1.67 | 0.58 |
| 變異數 | Var(X) = λ | 1.67 | 0.58 |
| 偏態係數 | 1/√λ | 0.77 | 1.31 |

### 偏態係數的陷阱（第5章）

蘇立恆發現大安區的偏態係數 = 0.02。

```

觀測偏態 = 0.02

差距：0.77 vs 0.02

「太整齊了。」

Poisson 分布本來就是右偏的。偏態接近零——
意味著數據被人工「修整」成對稱的了。
真正的隨機，不會這麼漂亮。
```

## 5. 似然比檢定 — 統計法庭的證據

**小說出處**：第9章（似然比）

### 公式

$$\Lambda = \frac{L(\theta_0 | \text{data})}{L(\hat{\theta}_{MLE} | \text{data})} = \frac{\text{虛無假設下的似然}}{\text{最佳解釋下的似然}}$$

- $\Lambda$ 接近 1 → 虛無假設解釋得不錯 → 不拒絕
- $\Lambda$ 很小 → 虛無假設解釋得很差 → 拒絕

### 小說中的數據（第9章）

| 區域 | 似然比 | p-value | 判定 |
|------|--------|---------|------|
| 大安區 | 4.72 | < 0.001 | ★★★ 異常 |
| 信義區 | 3.18 | < 0.005 | ★★ 異常 |
| 中正區 | 5.01 | < 0.001 | ★★★ 異常 |
| 內湖區 | 4.33 | < 0.001 | ★★★ 異常 |
| 松山區 | 3.77 | < 0.001 | ★★★ 異常 |
| 中山區 | 0.04 | 0.84 | 正常 |
| 萬華區 | 0.07 | 0.79 | 正常 |

五個高似然比的區域，全部往低偏移了 45%。

三顆星（★★★）= p < 0.001 = 千分之一以下的機率是巧合。

## 6. 逐步貝氏更新 — 從無知到確信

**小說出處**：第11章（父親的定理）

### 過程

每收到一筆新數據，後驗就更新一次：

$$\theta_0 \xrightarrow{D_1} \theta_1 \xrightarrow{D_2} \theta_2 \xrightarrow{D_3} \theta_3 \to \cdots$$

Beta 分布的美妙之處在於更新公式極其簡單：

```
起始：Beta(α₀, β₀)

觀測到 1 次成功：Beta(α₀ + 1, β₀)
觀測到 1 次失敗：Beta(α₀, β₀ + 1)

觀測到 k 次成功、n-k 次失敗：
Beta(α₀ + k, β₀ + n - k)
```

### 數值範例

```
起始：Beta(1, 1) — 完全不知道

第 1 筆：成功 → Beta(2, 1), E = 0.667
第 2 筆：成功 → Beta(3, 1), E = 0.750
第 3 筆：失敗 → Beta(3, 2), E = 0.600
...
第 100 筆：72 成功 28 失敗 → Beta(73, 29), E = 0.716

越來越接近真實值 0.72！
```

> 「P(θ|data) ∝ P(θ) × P(data|θ)。她寫在這張紙上。十六年前。這就是一切的起點。」——第十一章，貝葉讀母親的論文

**題目 1**：如果透鏡顯示某件事的機率是 Beta(10, 40)，求期望值和標準差。這代表什麼意思？

<details>
<summary>解答</summary>

```
E[θ] = 10 / (10 + 40) = 10/50 = 0.20
Var(θ) = (10 × 40) / (50² × 51) = 400 / 127500 ≈ 0.00314
σ = √0.00314 ≈ 0.056

意思：這件事發生的機率大約 20%，
而且我們有 50 筆觀測的信心（α+β=50），
所以 95% 信賴區間大約是 [0.09, 0.31]。
```
</details>

**題目 2**：你的先驗是「這枚硬幣正面機率 = 0.5」(Beta(10, 10))。你擲了 20 次，得到 15 次正面。新的後驗是什麼？你的信念改變了多少？

<details>
<summary>解答</summary>

```
先驗：Beta(10, 10), E = 0.50
數據：k = 15, n = 20
後驗：Beta(10+15, 10+5) = Beta(25, 15), E = 25/40 = 0.625

信念從 0.50 移動到 0.625。
數據（15/20 = 0.75）把你的信念從 0.50 往 0.75 拉了——
但沒有拉到 0.75，因為先驗也有力量。

如果先驗很弱 Beta(1,1)：後驗 = Beta(16, 6), E = 0.727
如果先驗很強 Beta(100,100)：後驗 = Beta(115, 105), E = 0.523

先驗越強，越難被數據動搖。
這就是柏朗做的事——他把所有人的先驗加強了，
讓數據無法改變他們的信念。
```
</details>

## 1. 四大分布家族

**小說出處**：第14章（分布的味道）

馬可瑜用味覺比喻分布——「每種分布都有自己的味道」。

### 常態分布 (Normal / Gaussian)

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

| 參數 | 意義 | 效果 |
|------|------|------|
| μ | 中心位置 | 分布左右平移 |
| σ | 散佈程度 | σ 越大，分布越胖 |

**特性**：
- 完美對稱（第55章柏朗：「高斯分布的對稱性」）
- 68.3% 的數據在 μ ± 1σ 內
- 95.4% 在 μ ± 2σ 內
- 99.7% 在 μ ± 3σ 內

### 指數分布 (Exponential)

$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$

| 參數 | 意義 | 小說數值 |
|------|------|----------|
| λ | 事件發生率 | 0.58 (Ch.53) |
| 1/λ | 平均等待時間 | 1.72 小時 |

**核心特性——無記憶性**（第53章）：

$$P(T > t + s \mid T > t) = P(T > s)$$

> 「你已經等了兩小時還沒有意外發生——下一個意外的等待時間分布跟你剛開始等的時候完全一樣。過去不影響未來。」

### Gamma 分布

$$f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, \quad x \geq 0$$

| 參數 | 意義 | 原始 (Ch.30) | 修改後 (Ch.33) |
|------|------|--------------|----------------|
| α | 形狀 | 3.2 | 1.8 |
| β | 速率 | 1.1 | 1.4 |
| E[X] = α/β | 期望值 | 2.91 | 1.29 |
| σ = √α/β | 標準差 | 1.63 | 0.96 |

**小說關鍵**：統計蜜罐用的就是 Gamma 分布。原始 Gamma(3.2, 1.1) 被修改成 Gamma(1.8, 1.4)，期望值從 2.91 跳到 1.29，偏移量 = 1.62，超過 5σ 門檻。

### Beta 分布

（詳見第一篇）

$$f(\theta) = \frac{\theta^{\alpha-1}(1-\theta)^{\beta-1}}{B(\alpha, \beta)}, \quad 0 \leq \theta \leq 1$$

> 「每種分布都有自己的味道。常態分布像白開水——乾淨、對稱、到處都是。指數分布像黑咖啡——一開始很苦，然後漸漸沒味道。Gamma 像老薑茶——暖的，有尾巴。Beta 像酸梅湯——被限制在一個杯子裡，但那個杯子裡什麼味道都有。」——第十四章，馬可瑜

## 2. 共軛先驗 — 完美犯罪的數學

**小說出處**：第16章（共軛先驗）

### 什麼是共軛？

如果先驗和後驗屬於同一個分布家族，就稱這對先驗-似然是**共軛的**。

| 似然函數 | 共軛先驗 | 後驗 |
|----------|----------|------|
| Bernoulli / Binomial | Beta(α, β) | Beta(α+k, β+n-k) |
| Poisson | Gamma(α, β) | Gamma(α+Σx, β+n) |
| Normal (已知 σ²) | Normal(μ₀, σ₀²) | Normal(μ_n, σ_n²) |
| Exponential | Gamma(α, β) | Gamma(α+n, β+Σx) |

### 為什麼共軛先驗是「完美犯罪」？

第16章解釋：如果有人偷偷把你的先驗從 Beta(5, 5) 改成 Beta(3, 7)——

```
你看到的還是 Beta 分布。形狀變了，但「類型」沒變。
就像有人偷偷把你的眼鏡度數改了——
你看到的世界還是清楚的（Beta 分布的樣子），
只是偏了。你不會注意到。

因為後驗 = Beta(α+k, β+n-k)——
不管先驗的 α, β 被改成什麼，
後驗的「形式」都一樣。
只是數值不同。

這就是 Oracular 的手法。
```

### 數值範例：偷改先驗

```
正常先驗：Beta(5, 5), E = 0.50
被改先驗：Beta(3, 7), E = 0.30（往「危險」方向偏移）

觀測到 8 次安全、2 次危險（n=10, k=8）：

正常後驗：Beta(5+8, 5+2) = Beta(13, 7), E = 0.65
被改後驗：Beta(3+8, 7+2) = Beta(11, 9), E = 0.55

差距：0.65 vs 0.55 = 偏移了 0.10

使用者看到的機率從 65% 變成 55%——
覺得世界比實際更危險。
但他不知道。因為一切看起來都很「正常」。
```

> 「共軛先驗。同族更新。修改前是 Beta。修改後還是 Beta。像在你的眼鏡上加了一層膜——你還是看得見。只是顏色偏了。你不會知道。」——第十六章

## 3. 聯合分布與邊際分布

**小說出處**：第22章（聯合與邊際）

### 定義

兩個隨機變數 X, Y 的**聯合分布**：

$$f(x, y) = \text{X 和 Y 同時取某個值的機率密度}$$

**邊際分布**——把另一個變數「積分掉」：

$$f_X(x) = \int_{-\infty}^{\infty} f(x, y) \, dy$$

$$f_Y(y) = \int_{-\infty}^{\infty} f(x, y) \, dx$$

### 小說怎麼用的

第22章把先驗修改事件放在二維空間裡分析：

```
X 軸：修改幅度（0% 到 100%）
Y 軸：修改時間戳

聯合分布 f(x, y)：
  → 看出修改幅度和時間之間有相關性
  → 大幅度修改集中在特定時段（凌晨 2-4 點）

邊際分布 f_X(x)：
  → 把時間積分掉 → 只看「修改幅度的分布」
  → 發現高修改幅度區域集中在 α > 0.5

邊際分布 f_Y(y)：
  → 把幅度積分掉 → 只看「時間的分布」
  → 發現三個峰值：8月、11月、2月（每三個月一次）
```

### 條件分布

$$f_{Y|X}(y | x) = \frac{f(x, y)}{f_X(x)}$$

> 「知道修改幅度大於 50% 的情況下，時間的分布是什麼？」→ 集中在凌晨。

> 「兩個維度的數據像一片星空。你可以把它壓成一條線——邊際分布。但壓的時候你會失去一些東西。你會失去星星之間的距離。」——第二十二章

## 4. 核密度估計 (KDE)

**小說出處**：第23章（時間窗口）

### 什麼是 KDE？

當你有一堆數據點，想估計它背後的連續分布形狀：

$$\hat{f}(x) = \frac{1}{nh} \sum_{i=1}^{n} K\left(\frac{x - x_i}{h}\right)$$

- $K$：核函數（通常用高斯核）
- $h$：帶寬（控制平滑程度）
- $x_i$：每個數據點

### 直覺

把每個數據點想像成一個小「丘」（高斯形狀），然後把所有的丘加起來，就得到整體的密度估計。

### 小說中的應用

```
40 個數據點（23 個犯罪樣本 + 17 個保險樣本）
三個時間峰值：2044年8月、11月、2045年2月
間隔：每 3 個月一次

KDE 的三峰結構 → 修改是週期性的
預測：下一次修改在 2045年5月

凌晨 02:00-04:00 的時間窗口 → 修改者的行為模式
```

## 5. 中央極限定理 (CLT) — 預覽

**小說出處**：第14章提及、第28章完整展開

雖然 CLT 的完整教學在第三篇，但第14章已經透露了核心概念：

$$\frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} \xrightarrow{d} N(0, 1)$$

> 不管原始分布長什麼樣——指數的、Gamma 的、Beta 的——只要 n 夠大，樣本平均值就會趨近常態分布。

這就是常態分布「統治一切」的原因。它不是因為世界是常態的，而是因為**平均**是常態的。

> 「不管原始分布長什麼樣——夠多人的平均，永遠趨向常態。這就是為什麼全世界這麼不一樣的人，考試成績的分布永遠是鐘形的。」——第二十八章，高思遠

**題目 2**：你有聯合分布 f(x, y) = 6xy，定義在 0 < x < 1, 0 < y < 1。求 (a) f_X(x) (b) P(X > 0.5)

<details>
<summary>解答</summary>

```
(a) f_X(x) = ∫₀¹ 6xy dy = 6x · [y²/2]₀¹ = 6x · 1/2 = 3x

(b) P(X > 0.5) = ∫₀.₅¹ 3x dx = 3 · [x²/2]₀.₅¹ = 3 · (1/2 - 1/8) = 3 · 3/8 = 9/8

等等——機率不能超過 1！讓我重新算...

P(X > 0.5) = ∫₀.₅¹ 3x dx = 3[x²/2]₀.₅¹ = 3(0.5 - 0.125) = 3 × 0.375 = 1.125

還是大於 1？讓我檢查 f(x,y) 是否是有效的密度函數...

∫₀¹∫₀¹ 6xy dx dy = 6 · [x²/2]₀¹ · [y²/2]₀¹ = 6 · 0.5 · 0.5 = 1.5 ≠ 1

在這個定義域下：
f_X(x) = ∫₀^(1-x) 6xy dy = 6x · [(1-x)²/2] = 3x(1-x)²

P(X > 0.5) = ∫₀.₅¹ 3x(1-x)² dx
           = 3∫₀.₅¹ (x - 2x² + x³) dx
           = 3[x²/2 - 2x³/3 + x⁴/4]₀.₅¹
           = 3[(1/2 - 2/3 + 1/4) - (1/8 - 1/12 + 1/64)]
           = 3[(6/12 - 8/12 + 3/12) - (8/64 - 16/192 + 3/192)]
           = 3[1/12 - 0.0573...]
           ≈ 0.078

只有 7.8% 的機率 X > 0.5 ——
因為在三角形區域裡，x 大的時候 y 的空間很小。
```

## 1. 期望值 — 平均的真正意義

**小說出處**：第25章（蔥油餅會議）、第26章（期望值與代價）

### 離散型

$$E[X] = \sum_{i} x_i \cdot P(X = x_i)$$

### 連續型

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x) \, dx$$

### 期望值的線性性（最重要的性質）

不管 X 和 Y 是不是獨立：

$$E[aX + b] = aE[X] + b$$

$$E[X + Y] = E[X] + E[Y]$$

### LOTUS 定理（無意識統計學家定理）

**小說出處**：第26章

如果你要算 g(X) 的期望值，不用先算 g(X) 的分布：

$$E[g(X)] = \int_{-\infty}^{\infty} g(x) \cdot f_X(x) \, dx$$

### 小說怎麼用的

第26章計算先驗偏移造成的經濟損失：

```
X = 各區域的偏移幅度，服從某個分布 f(x)
g(X) = 損失函數：偏移越大，損失越大

E[loss] = ∫ g(x) · f(x) dx

具體數值：
  - 152 個分析區域
  - 平均偏移幅度 38%（最大 72%，最小 8%，中位數 35%）
  - 期望損失 = 480 億台幣
```

> 「四百八十億。這是期望損失。不是確定會損失四百八十億。是——所有可能的損失，乘以它們發生的機率，加起來。期望值不是未來。是所有可能的未來的加權平均。」——第二十六章，馬可瑜

## 2. 變異數 — 不確定性的量化

**小說出處**：第27章（變異數與分歧）

### 公式

$$\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - (E[X])^2$$

$$\sigma = \sqrt{\text{Var}(X)}$$

### 性質

$$\text{Var}(aX + b) = a^2 \text{Var}(X)$$

如果 X, Y 獨立：
$$\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)$$

### 小說中的三種 σ

```
完全自動化系統：σ = 0.03（方差極小，一切被控制）
半自動 + 人工微調：σ = 0.15（有些變異，但可管理）
完全人工判斷：σ = 0.40（方差很大，不可預測）

第27章的辯論核心：
  馬可瑜主張 σ = 0.15 — 「夠小能控制，夠大能呼吸」
  柏朗要 σ → 0 — 「方差等於風險，風險等於危險」
  貝葉最後的結論 — 「σ = 0 是死的系統。方差就是生命。」
```

> 「σ = 0 是死的系統。穩態。永遠不動。永遠安全。永遠——死的。」——第五十五章，貝葉在天台

![Variance](圖表/11_variance.png)

### 68-95-99.7 法則（常態分布）

```
μ ± 1σ 包含 68.3% 的數據
μ ± 2σ 包含 95.4% 的數據
μ ± 3σ 包含 99.7% 的數據

## 3. 大數法則 (LLN)

**小說出處**：第28章（大數法則的幽靈）

### 弱大數法則

對任意 ε > 0：

$$P\left(|\bar{X}_n - \mu| > \varepsilon\right) \to 0 \quad \text{as } n \to \infty$$

### 中文翻譯

> 樣本平均值，在樣本量夠大的時候，會趨近真實的期望值。

### 小說怎麼用的

```

> 「大數法則的幽靈。整體看起來沒問題。個別看才能發現問題。平均值是最誠實的數字，也是最容易被利用的數字。」——第二十八章

## 4. Markov 不等式與 Chebyshev 不等式

**小說出處**：第29章（不等式的鐵壁）、第30章（統計蜜罐）、第33章（觸發）

### Markov 不等式

如果 X ≥ 0：

$$P(X \geq a) \leq \frac{E[X]}{a}$$

### Chebyshev 不等式

對任何分布：

$$P(|X - \mu| \geq k\sigma) \leq \frac{1}{k^2}$$

| k | Chebyshev 上界 | Normal 實際值 |
|---|---------------|---------------|
| 1 | ≤ 100% | 31.7% |
| 2 | ≤ 25% | 4.6% |
| 3 | ≤ 11.1% | 0.27% |
| 5 | ≤ 4% | 0.00006% |
| 10 | ≤ 1% | ~10⁻²³ |

### 為什麼 Chebyshev 很有用？

> 它不需要知道分布是什麼形狀！

不管是常態、指數、Gamma、什麼奇怪的分布——Chebyshev 保證超過 5σ 的機率不會超過 4%。

### 蜜罐的數學（第30、33章）

```
蜜罐設定：
  原始分布：Gamma(3.2, 1.1)
  E[X] = 3.2 / 1.1 = 2.91
  σ = √(3.2) / 1.1 = 1.63（注：σ = √α / β）

觸發條件：
  |新期望值 - 2.91| > 5σ₀

第33章觸發時刻：
  修改後：Gamma(1.8, 1.4)
  新 E[X] = 1.8 / 1.4 = 1.29

  偏移量 = |1.29 - 2.91| = 1.62
  標準化偏移 = 1.62 / (σ₀/√n)

  這裡 σ₀ = 1.63，n = 50（蜜罐觀測數）
  σ₀/√n = 1.63/√50 = 0.231

  標準化偏移 = 1.62 / 0.231 = 7.01σ ≈ 超過了

  （小說簡化為 5.23σ 的寫法，但原理一致）

Chebyshev 保證：P(偏移 > 5σ) ≤ 1/25 = 4%
Normal 的精確值：P(偏移 > 5σ) ≈ 0.00006%

不管哪種分布——超過 5σ 就觸發。沒有例外。
```

> 「五點二三個標準差。Chebyshev 保證——不管什麼分布——超過五個標準差的機率不會超過百分之四。而我們觀察到了。這不是隨機波動。這是有人動了手腳。」——第三十三章，凌晨兩點三十七分

## 5. 動差生成函數 (MGF)

**小說出處**：第34章（動差生成函數）、第36章（唯一性定理）

### 定義

$$M_X(t) = E[e^{tX}] = \int_{-\infty}^{\infty} e^{tx} f(x) \, dx$$

### 為什麼叫「動差生成」？

$$M_X^{(n)}(0) = E[X^n]$$

也就是說，MGF 在 t=0 的第 n 階導數就是第 n 階動差：

```
M'(0) = E[X]     ← 第一階動差 = 期望值
M''(0) = E[X²]   ← 第二階動差 → 可以算變異數
M'''(0) = E[X³]  ← 第三階動差 → 可以算偏態
M''''(0) = E[X⁴] ← 第四階動差 → 可以算峰度
```

### 唯一性定理——殺手鐧

> 如果兩個分布有相同的 MGF，那它們就是同一個分布。反過來，如果 MGF 不同，分布一定不同。

### 常見分布的 MGF

| 分布 | MGF |
|------|-----|
| Normal(μ, σ²) | $e^{\mu t + \sigma^2 t^2/2}$ |
| Poisson(λ) | $e^{\lambda(e^t - 1)}$ |
| Exponential(λ) | $\frac{\lambda}{\lambda - t}$, t < λ |
| Gamma(α, β) | $\left(\frac{\beta}{\beta - t}\right)^\alpha$, t < β |

### 小說的數學證明（第34章）

```
原始分布：Gamma(3.2, 1.1)
  M_A(t) = (1.1 / (1.1 - t))^3.2

修改後分布：Gamma(1.8, 1.4)
  M_B(t) = (1.4 / (1.4 - t))^1.8

在不同 t 值比較：

  t = 0.1: M_A = (1.1/1.0)^3.2 = 1.387
            M_B = (1.4/1.3)^1.8 = 1.219
            → 不同！

  t = 0.2: M_A = (1.1/0.9)^3.2 = 1.924
            M_B = (1.4/1.2)^1.8 = 1.497
            → 不同！

  t = 0.5: M_A = (1.1/0.6)^3.2 = 5.017
            M_B = (1.4/0.9)^1.8 = 2.413
            → 差距更大！

M_A(t) ≠ M_B(t) 對所有 t > 0 成立。

由唯一性定理 → 兩個分布不同 → 先驗被修改了 → 鐵證。
```

> 「兩個分布。同一個參數名字。同一種形狀。但 MGF 不同。這就像兩個人——長得一樣。但 DNA 不同。MGF 就是分布的 DNA。」——第三十四章，貝葉在法庭

貝葉把傘塞進包包裡，走到陽台。

薄荷還活著。勉強。

那盆薄荷是三個月前從花市買的，老闆說「很好養，澆水就活」。事實證明，「很好養」這三個字的成立條件是「養的人會記得澆水」。葉片邊緣已經有點焦了，捲曲著，像一個人在大太陽底下瞇起眼睛的樣子。但中間冒了一個新芽——小小的、嫩綠的、固執地從乾裂的土壤裡鑽出來。

她拿起桌上的馬克杯，裡面還有昨晚沒喝完的水，倒了一點到花盆裡。

「你對植物的關注度比對自己的身體高百分之四十七。」小拉碎念。

「因為植物不會碎念。」

她關上陽台的門，回到玄關穿鞋。球鞋的鞋帶有一邊鬆了，她蹲下來繫好，然後拉開大門。

走廊的日光燈還是那一盞在閃。閃了三個月了。管委會的公告欄上貼著「已通知水電行」的字條，字條的邊角都翹起來了，紙也泛黃了。

她走下樓梯，推開公寓的鐵門。

巷子裡有人在騎摩托車暖車。排氣管突突突地響，在清晨六點五十五分的台北特別清楚，像一個節拍不太穩定的鼓機。空氣裡混著機油味和隔壁早餐店的蛋餅味，兩種味道在巷口交會，形成一種只有台北老巷子才有的晨間配方。

她走過摩托車的時候想——這台摩托車每天早上都在這裡。每天都在暖車。暖車的時間大概是三到五分鐘，方差不大。

這不是 Poisson 過程。Poisson 過程的前提是事件獨立、隨機、不可預測。但這台摩托車的出現既不獨立也不隨機。它是固定的。是一個確定性事件偽裝成背景雜訊。

她搖搖頭。職業病。

## I

捷運中山站。早上七點二十六分。

月台上的人群像一個尖峰時段的排隊模型——密度從樓梯口往兩端遞減，在標示有車門位置的地磚前形成區域性的高密度叢集。每個叢集大約八到十二人，彼此之間保持著一種心照不宣的間距，不遠也不近，剛好是「不會碰到對方的手肘但也不會浪費空間」的最佳化距離。

貝葉站在月台中段。透鏡自動開機了。

她其實不喜歡通勤的時候開透鏡。太多數字會讓早晨變得很擠——不是物理上的擠，是認知上的擠。當你的視野裡每一個人、每一件事都附帶著一組數字的時候，世界會從一幅畫變成一張試算表。但公司規定校準工程師必須全時段開啟透鏡，理由是「持續累積環境數據有助於提升校準精度」。她覺得真正的理由是「公司想知道你在看什麼」，但這個想法她只在心裡想過，從來沒有說出口。

月台對面的電子看板顯示下一班車還有兩分十四秒。透鏡把這個資訊攔截了，在她的視野右下角疊加了一層額外的分析：

下一班板南線預計到站時間：兩分十四秒。到達時間分布為常態分布，均值一百三十四秒，標準差十二秒。也就是說，大約百分之六十八的機率會在兩分零二秒到兩分二十六秒之間到站。本班車廂有空位的機率：百分之二十三。

「小拉，百分之二十三有座位。」

「建議等下一班。下一班有座位的機率是百分之六十一。」

「但我會遲到。」

「遲到的機率是百分之十二。嗯，其實不太高——」

「但我不想遲到。」

「那就擠上去吧。」

她擠上去了。

車廂裡的空氣帶著一種特殊的早晨味道——洗衣精、體香劑、還有不知道誰早餐吃的鮪魚三明治。她被夾在一個穿西裝的中年男人和一個背大書包的高中女生之間，左手抓著吊環，右手垂在身側。沒有座位。百分之二十三嘛。

透鏡在她的視野裡靜靜地運作著。它不會主動彈出資訊——除非她把視線聚焦在某個目標上超過一點五秒。這是 Oracular 第三代透鏡的改良設計：用眼球追蹤判斷使用者的注意力方向，只在「使用者有意識地觀察」的時候才提供數據。但在尖峰時段的車廂裡，你的視線很難不落在任何人身上，所以數字還是不斷地冒出來。

她的目光掃過那個穿西裝的男人，透鏡自動浮出一行淡灰色的小字：此人今日通勤遲到的機率，百分之八。她的視線移到旁邊那個高中女生，又一行字：此人今日考試成績落在前百分之二十的機率，百分之六十七。

她趕緊把視線移開，盯著車廂天花板的通風口。通風口的格柵上積了一層薄薄的灰。透鏡沒有為灰塵提供任何機率評估。這讓她稍微鬆了一口氣。

這些數字有用嗎？也許。在某些場景下，透鏡的即時機率確實幫助人們做出了更好的決策——哪條路塞車、哪家餐廳今天出餐比較快、哪個路口的事故風險偏高。Oracular 的官方數據顯示，透鏡普及後的三年內，全台灣的交通事故率下降了百分之十四，食物中毒通報減少了百分之二十一，甚至連保險理賠金額都降低了百分之九。

但所有人都已經習慣了。像習慣呼吸一樣習慣了被數字包圍。

習慣是一件很可怕的事。當你習慣了某個東西，你就不再質疑它。不再質疑它的來源。不再質疑它的準確性。不再質疑——它是不是你自己選擇要看到的。

貝葉閉上眼。透鏡的數字在眼皮後面變成微弱的餘像，像電視關掉之後螢幕上殘留的那一小團光。她想起高思遠教授的話——那是很多年前說的，在台大的教室裡，粉筆灰飛揚的午後：

「機率不是命運。機率只是地圖。走哪條路，是你自己的事。」

列車減速了。車廂裡所有站著的人同時往前傾了一小步，又各自穩住。自動廣播響起：南港軟體園區站到了。

她睜開眼，沒有座位，但也沒有遲到。

百分之二十三不代表不可能。也不代表值得。只是一個數字而已。

## II

Oracular 的總部坐落在南港軟體園區的三棟弧形玻璃建築裡。從捷運站走過去大約七分鐘，沿途是修剪整齊的灌木叢和嵌在地面上的太陽能充電板。三棟建築用空中走廊彼此連接，走廊的外牆是一整面曲面螢幕，二十四小時不間斷地顯示全台灣的即時機率熱力圖。此刻北部是一片漸層的淺藍色——降雨機率偏高的區域；南部是暖黃色，晴朗無雲。整面螢幕看起來像一幅活的水彩畫，顏色每隔幾秒就微微流動一次，跟著底層演算法的更新節奏呼吸。

大廳的地板是白色的拋光石英磚，乾淨到能看到自己的倒影。門禁系統的語音在她走過閘門時響起：「歡迎回來，陳貝葉。今日校準指數：八十八點七。」

校準指數是 Oracular 內部的員工績效指標之一。滿分一百，代表你在過去二十四小時內透鏡記錄的所有行為數據與公司模型的預測完全一致。八十八點七算中上——意味著她昨天有大約百分之十一的行為偏離了模型的預期。那百分之十一大概包括她凌晨十二點半還在看一部老電影（模型預測她十一點十五分就會睡著），以及她吃宵夜的時候選了泡麵而不是模型建議的低鈉優格。

她有時候會故意做一些模型不會預測的事。不是為了反叛，而是為了——確認自己還有選擇的能力。

校準組的辦公室在 B 棟四樓。十二張桌子排成三排，日光燈很亮，色溫大約五千K，把所有東西都照得清晰而平淡。空調永遠設定在二十三度。這個溫度是 Oracular 的數據科學團隊經過三年的行為追蹤後得出的「最佳辦公效率溫度」——據說在二十三度的環境下，員工的程式碼錯誤率最低、專注時間最長、離開座位上廁所的頻率也最穩定。

貝葉的桌子在靠窗的位置。窗外看出去是另一棟大樓的玻璃帷幕，玻璃上映著天空的倒影和偶爾飛過的鴿子。她的桌面上有三樣東西：一杯昨天的珍珠奶茶，珍珠已經沉到杯底，奶茶和茶分了層，上面淡下面深，像一個微型的密度梯度實驗；一台舊款的統計計算機，TI-84 Plus，螢幕的邊框有一道裂痕——她不信任透鏡的內建計算功能，「自己算一遍比較安心」是她入行第一天就養成的習慣；還有一隻橘色的招財貓，瓷做的，大概十公分高，右手舉著，永遠在招。

招財貓是高思遠教授給她的，底座上用黑色簽字筆寫了一行字：「期望值為正就值得嘗試。」字跡已經很舊了，邊緣有點暈開，但她從不擦。

旁邊的桌子是林岳的。

林岳三十一歲，瘦，戴一副黑框眼鏡，鏡片上永遠有一層細微的指紋——因為他習慣用手推眼鏡，而不是用眼鏡布擦。他的桌上有一棵假的仙人掌，塑膠做的，綠得非常均勻，沒有任何一片葉子枯萎或變色，完美得讓人覺得可疑。仙人掌旁邊是一張全家福照片——妻子齊耳短髮，笑得很開朗，懷裡抱著一個大約兩歲的女孩，女孩的手裡捏著一隻布偶長頸鹿。

林岳是校準組除了貝葉之外，唯一會「手動驗算」的人。其他同事都信任系統，信任模型，信任跑出來的數字。只有他們兩個會把系統吐出來的結果拉到自己的計算機裡重新算一遍。這件事讓他們之間產生了一種不需要言語的默契，像兩個在同一家超市裡堅持自己挑水果而不用 AI 推薦的人，在貨架前交換了一個彼此理解的眼神。

「早。」林岳推了推眼鏡。

「早。」貝葉坐下來，打開電腦，輸入密碼。密碼是一組她用了四年的數字，手指不需要經過大腦就能敲完。

「你昨天加班到幾點？」

「十一點。你呢？」

「十二點。」

「……你贏了。」

「這不是比賽。」

「聽起來就是。」

林岳笑了。是那種很輕的、從鼻子裡哼出來的笑。然後他轉回去看他的螢幕，手指在鍵盤上跑數據，速度很快，像在彈一首只有他聽得見的曲子。

辦公室裡陸續有人進來了。有人在茶水間倒咖啡，咖啡機發出嘶嘶的蒸汽聲。有人在複印機前面排隊。有人在座位上打哈欠，哈欠打到一半被自己的透鏡彈出的行事曆通知嚇了一跳。

然後林岳突然轉過頭來。

「欸，貝葉。」

她從螢幕前抬起眼睛。

「你覺得 Poisson 分布在 λ 等於二點一的時候，殘差分布的偏態係數零點零三，正常嗎？」

她停了一下。腦子裡的計算像一台被按下開關的機器，自動啟動了。Poisson 分布的偏態係數等於 λ 的負二分之一次方。λ 等於二點一的話，偏態應該在零點四到零點七之間。零點零三——太低了。低得不像是隨機波動。低得像是有什麼東西把本該存在的不對稱性抹平了。

「零點零三？」她皺了皺眉。「λ 等於二點一的 Poisson，偏態應該在零點四到零點七。零點零三太低了。」

「嗯。我也覺得。」林岳推了推眼鏡，用那種「我不確定但有點在意」的語氣說。「可能是我哪裡跑錯了。」

「你在看哪個區域？」

「中正區。但沒什麼——可能是我的模型設定有問題。」

他轉回去了。沒有再提。就像在說「今天中午吃什麼」然後馬上接著討論下一個話題一樣自然。

貝葉也轉回去了。

但那個數字——零點零三——留在她的腦子裡。

不是以一種警報的方式。是以一種更安靜的方式。像一顆種子掉進土裡的聲音。你不會聽到。你甚至不會注意到。但它落土了。

## III

中午十二點半。地下一樓的自助餐廳。

貝葉端著一盤排骨飯坐到靠牆的位置。排骨炸得有點過頭，外層的麵衣偏硬，顏色從「金黃」跨過了「深褐」的邊界。配菜是高麗菜和豆芽菜，高麗菜炒得太軟，豆芽菜又太生，整盤的口感平衡約等於零。但它便宜，而且不用走出大樓。

她剛拿起筷子，小拉就啟動了。

「這個排骨飯的熱量分布——」

「小拉。」

「什麼？」

「我只是想吃飯。」

小拉沉默了零點八秒。這是她表達「被拒絕但仍想堅持」的標準反應時間。

「……好。但我想讓你知道，在你吃完之後有百分之七十一的機率會想睡覺。」

「所以？」

「所以你應該買杯咖啡。」

「我不想花錢。」

「公司的免費咖啡還有。」

「那個不叫咖啡。那叫深色的水。」

「……以化學成分來看，你說得對。咖啡因含量只有標準義式濃縮的百分之三十一。」

貝葉咬了一口排骨。麵衣在嘴裡碎開的聲音很響，像踩到落葉。味道還行。至少鹽放得夠。她有時候覺得，鹽是所有烹飪裡唯一不會背叛你的元素——放多了就鹹，放少了就淡，線性關係，沒有交互作用，不需要任何花俏的模型就能預測結果。

隔壁桌的同事在討論信義區的數據重跑。有人說上次重跑花了十四個小時，伺服器差點當機。有人在抱怨系統更新太頻繁，說上禮拜剛適應新的介面，這禮拜又改了。他們的聲音在自助餐廳的嘈雜裡像一條細流，聽得到但不需要專心聽。

周副理走過來了。

周副理五十三歲，是校準組的直屬主管。他的身材像一個放大了的梨子——肩膀窄，腰圍寬，走路的時候上半身幾乎不動，只有下半身在移動，像一個被放在推車上的半身像。他平常說話很大聲，用一種永遠在趕時間的語速，把每個句子都壓縮到最短。但偶爾——很偶爾——他會突然放慢速度，壓低聲音，用一種幾乎像在說悄悄話的語氣講話。那通常意味著他接下來要說的事情跟「人」有關，而不只是跟「數據」有關。

「大家不要忘了下午三點的週會。」他經過的時候對著整個餐桌區廣播了一句，語速一如既往地快。然後他走到貝葉旁邊，身體微微彎下來，聲音低了半個八度。

「下午有一個到府校準的案子。老先生。你去一趟。」

「什麼情況？」

「透鏡數值全面偏高。可能是機械故障，也可能是……」他停了一下。嘴巴的形狀在「是」字之後維持了大約一秒鐘，像在斟酌接下來的用詞。

「認知基線偏移。」

他的語氣微妙地軟了。不是變溫柔——周副理不是那種會變溫柔的人——而是變得小心了。像一個人在端一碗很滿的湯，怕灑出來。

「帶點心。他可能需要有人陪他說說話。」

他說完就走了。步伐還是那樣，上半身不動，下半身快速移動，鑲在皮帶上的員工識別證隨著步伐一晃一晃的。

小拉的聲音在貝葉的腦中響起——小拉說話的時候只有貝葉聽得到，因為透鏡的語音輸出是通過骨傳導直接進入內耳的，外人什麼都聽不見。

「周副理剛才的語調偏差了一點三個標準差。比他正常的語氣更溫柔。」

「他只是……善良。」

「善良在我的模型裡不是一個參數。」小拉停頓了一下。「但我正在考慮加入。」

## IV

下午三點半。週會結束。

貝葉回到座位上，打開今天的校準面板。三個例行區域：內湖、松山、信義。她逐一點開，檢查每個區域的機率分布圖。曲線都很平滑，殘差都在可接受範圍內，沒有異常峰值，沒有不明的偏移。一切正常。

但「正常」是一個相對的概念。正常的前提是你的基準線是對的。如果基準線本身就歪了，那「正常」只是一個歪曲世界裡的歪曲標準——看起來沒問題，但其實所有的東西都偏了。

校準的第一步永遠是確認樣本空間。聽起來簡單——「所有可能發生的事」。但這是一切的起點。如果你連「什麼事有可能發生」都搞錯了，後面算出來的所有機率都是建立在錯誤地基上的數字。好看的數字。精確的數字。但是假的。

第二步是確認三條公理。

所有事件的機率大於等於零。不存在負的機率。這聽起來像廢話，但它的意義是：在這個系統裡，沒有「比不可能更不可能」的事。零就是底線。

整個樣本空間的機率等於一。所有可能加起來就是全部。不多不少。這意味著你不能憑空創造可能性，也不能偷偷消除可能性——至少理論上不能。

互斥事件的機率可以相加。如果 A 和 B 不能同時發生，那發生 A 或 B 的機率就是兩者的機率加起來。

三條規則。整個機率世界的地基就站在這三條規則上面。透鏡上的每一個數字——從降雨機率到通勤時間到「此人今天心情好的機率」——全部都是從這三條規則推導出來的。

如果地基出了問題——

她的手指在鍵盤上停了一下。

這個念頭是從哪裡來的？她不知道。沒有特別的原因。也許是剛才林岳提到的那個零點零三。也許只是咖啡因不夠。也許什麼都不是，就是一個路過腦海的隨機念頭，來了又走了，像風吹過水面製造的一個小漩渦。

她把三個區域的校準報告簽核完畢，關上面板。收拾東西，準備出發去做到府校準。

出門前她從茶水間的零食櫃裡拿了一包小餅乾。是那種獨立包裝的奶油小圓餅，包裝上畫著一隻微笑的松鼠。公司提供的免費零食。她把餅乾放進包包裡的時候，想到了周副理說的「帶點心」。

帶點心。不是帶工具。不是帶技術手冊。是帶點心。

這句話在她心裡留下了一個很小的印記。像一枚圖釘釘在佈告欄上——不痛，但你知道它在那裡。

---

## V

晚上九點十七分。中山區。四樓。

走廊的那盞燈還在閃。這一次比早上閃得更頻繁了，大約每三秒一次，像一個不太有耐心的摩斯密碼。貝葉從包包裡摸出鑰匙，金屬碰到金屬的聲音在空蕩蕩的走廊裡顯得特別清脆。

門打開了。玄關很暗，只有冰箱的指示燈在廚房那頭亮著，是一個小小的藍色光點，像深海裡的某種生物。

她脫鞋。把傘放在門邊。今天真的下了。Beta(6.2, 3.8) 贏了。

打開冰箱。裡面的光線照亮了她的臉——白熾的、不留情面的光。冰箱的內容清單：半盒牛奶，小拉說明天過期；兩顆蘋果，小拉說後天過期；一盒微波義大利麵。

「微波麵過期了。」小拉的聲音像是從很遠的地方傳來的，帶著一種已經說過很多次但還是要再說一次的疲倦。

「知道了。」

「你不打算丟嗎？」

「明天。」

「你三天前也這樣說。」

「……嗯。」

她拿出那盒義大利麵看了一下。包裝上印的保存期限是上禮拜二。她把它放回去了。放的時候，手指碰到了冰箱最裡面的角落，觸感是冰的、滑的，什麼都沒有。

她關上冰箱，沒有吃東西。

客廳不大。沙發是米色的，買的時候是白色的，三年過去就自然而然地變成了一種介於白和灰之間的、像骨頭的顏色。茶几上有一台闔著的筆電、一個空的馬克杯、和兩本疊在一起的書——上面那本是《機率論與數理統計》，下面那本是一部日本推理小說，書籤夾在大約三分之一的位置，三個月前夾的，之後就再也沒翻過。

她沒有坐沙發。她坐在地板上，背靠著沙發的側面。這是她在家裡最常坐的位置。沙發是用來靠的，不是用來坐的。地板比較硬，但硬有硬的好處——它很確定。它不會因為你的體重而改變形狀。它就是它。

她打開手機。沒有未讀訊息。

透鏡的社交面板自動彈出來——它會根據她的社交互動頻率計算「友好度」指數。排行榜上只有三個人超過百分之五十：馬可瑜，百分之九十二；高思遠，百分之八十九；林岳，百分之七十一。

三個人。

百分之五十以上的朋友只有三個人。

然後透鏡在視野的正中央彈出了一條訊息。不是社交通知，不是行事曆提醒。是透鏡的情緒預測模組——Oracular 兩年前加入的新功能，用戶可以選擇開啟或關閉。貝葉一直沒有關掉它，因為她覺得它就像天氣預報一樣——知道會下雨不代表你能阻止下雨，但至少你可以帶傘。

訊息寫著：「今晚感到孤獨的機率：百分之七十八。」

她盯著那行字。

百分之七十八。

透鏡是怎麼算出這個數字的？大概是根據她的社交互動頻率（今天只跟林岳和周副理說過話）、當前時間（晚上九點，獨處的高峰時段）、過去三十天同一時段的情緒採樣（她不記得自己有被採樣，但透鏡會在背景中持續收集微表情和心率數據）、加上她打開手機後沒有未讀訊息這個事實。

這些輸入變數經過某個她看不見的模型——某個在 Oracular 伺服器上運行的、包含了數百萬人的情緒-行為關聯數據的神經網路——吐出了一個數字。

百分之七十八。

一個很精確的數字。但精確不代表有用。因為就算她知道自己有百分之七十八的機率感到孤獨，她也不知道該怎麼辦。數字告訴你「是什麼」和「有多少」，但它不告訴你「該怎麼辦」。它是地圖上的一個坐標，但它不是路線。

高思遠教授的話又浮上來了——那是十年前說的，在一個下午，在台大旁邊的小餐館裡，在滷肉飯的香氣裡。他放下筷子，看著還在讀高中的她，說：

「機率告訴你可能會發生什麼。不告訴你該怎麼辦。」

十年前說的。十年後還是對的。

她把透鏡切到夜間模式。介面上的所有數字同時褪去，像潮水退去之後露出底下的沙灘。視野裡只剩下真實的、沒有被標注的世界——客廳的米色沙發、茶几上的空杯子、窗外巷子裡的路燈。

房間裡突然變得很安靜。

不是絕對的安靜。冰箱在嗡嗡響，那種低沉而持續的聲音，像一個永遠不會結束的長音。隔壁傳來小孩練鋼琴的聲音。

C——D——E。停頓。

C——D——E。停頓。

C——D——E——F——

每次到 F 都會卡住。音符在那裡遲疑了一下，像一個人走到懸崖邊往下看了一眼。然後重來。

C——D——E。停頓。

C——D——E。停頓。

C——D——E——F——

又卡住了。但這一次停頓的時間比上次短了一點。然後重來。

那個小孩不知道明天能不能彈到 G。不知道下禮拜能不能彈完整首曲子。不知道三年後會不會放棄鋼琴。但今晚——今晚他一直在重來。每一次都是從 C 開始。每一次到 F 都卡住。每一次卡住之後都重來。

這不是什麼了不起的事。但在一個被百分之七十八的孤獨機率籠罩的夜晚，在一間冰箱裡有過期食物的四樓公寓裡，這個聲音——某個人在隔壁的牆後面固執地重複著同一個不完美的段落——讓世界好了一點點。

只有一點點。但那一點點是真的。

貝葉閉上眼。

她不知道自己什麼時候睡著的。地板很硬。但也沒有那麼硬。背靠著沙發的側面，頭歪向右邊，頭髮散在肩膀上。透鏡在夜間模式裡安靜地待機著，螢幕全暗，不顯示任何數字。

陽台上的薄荷在風裡搖了一下。

沒有人替它算過存活機率。但它明天還是會長。

---

*（第一章完）*


# 第二章：校準悲傷的人

---

下午兩點。大安區。

陽光從雲層的縫隙裡漏下來，不是那種爽朗的晴天光線，而是一種被稀釋過的、灰白色的光，像洗了太多次的白襯衫。梅雨季的台北就是這樣——天空永遠在下雨和不下雨之間猶豫不決，雲層壓得很低，空氣裡含著一層看不見的溼氣，讓每一口呼吸都帶著一點重量。

貝葉站在一棟老公寓的門口，抬頭看了一下門牌號碼。四樓。這棟公寓比她自己住的那棟更老一些，外牆的磁磚剝落了幾塊，露出底下斑駁的水泥。一樓的信箱排成一排，有幾個信箱的蓋子歪了，從縫隙裡露出塞得滿滿的廣告傳單。

她走上樓梯。樓梯間沒有電燈，只有每層樓的窗戶透進來的自然光。空氣裡有一股很淡的樟腦丸味道，混著老房子特有的、被時間醃漬過的潮溼氣味。每走一階，腳下的磨石子地板就發出輕微的沙沙聲，像在竊竊私語。

四樓到了。門口有一塊褪色的門墊，上面印著「歡迎光臨」四個字，但字已經磨得只剩輪廓了。

門鈴是那種很舊的——按下去會「叮咚」兩聲。不是電子音效，是物理的、一個金屬片敲擊另一個金屬片的震動。這個聲音讓她想起小時候去外婆家，那種屬於另一個年代的、實在的聲音。

門開了。

鄧伯伯。七十五歲。退休的高中數學老師。個子不高，大概一百六十五左右，頭髮全白了，但梳得很整齊，用梳子仔細地往後梳，每一根都有它該在的位置。他穿一件格子襯衫，藍色和灰色的格子，扣到最上面一顆。

「你好，我是 Oracular 的校準工程師——」

「啊，你就是那個小陳啊。進來進來。鞋子脫這裡。」

他的語氣帶著一種老派的熱情——不是那種社交場合的客套，而是真正的、看到有人來家裡就高興的那種。他用手指了指門口的鞋架。

鞋架上有兩雙拖鞋。

一雙是鄧伯伯的。藍色的塑膠拖鞋，邊緣已經變形了，鬆鬆的，像穿了很多年。

另一雙是粉紅色的。小一號。整整齊齊地放著。鞋面上有一層薄薄的灰。

那層灰——很均勻。不是「幾天沒穿」的灰。是「很久很久沒有人穿」的灰。

貝葉看了一眼。沒有問。

她脫了鞋，換上鄧伯伯遞給她的室內拖鞋——一雙全新的、還帶著塑膠味的拖鞋。他特地買的。為了一個到府校準的工程師，他特地去買了一雙客人用的拖鞋。

---

## I

客廳不大。

沙發是舊的，深棕色的皮革已經在扶手的位置磨出了兩個淺色的印子，像兩隻手長年擱在同一個地方留下的痕跡。沙發上套了一層乾淨的淺灰色布，邊角摺得很整齊。

茶几是木頭的，表面有好幾圈杯底的水漬，重疊交錯，像樹的年輪。茶几上擺著一疊實體報紙——現在還看實體報紙的人已經很少了——一副老花眼鏡，以及一個馬克杯，白底，上面印著「世界最棒的老師」，字體是那種很老的、帶圓角的電腦字體，旁邊畫了一顆蘋果和一個笑臉。

電視櫃上有一張照片。

照片裡是鄧伯伯年輕的時候。大概四十多歲。站在什麼地方的合影——背景是海，陽光很好，他穿著白色的短袖襯衫，笑得露出牙齒。旁邊是一個女人，個子比他矮一點，穿著碎花旗袍，頭髮盤起來，手裡捧著一束花。她笑得很燦爛，是那種不用想就笑出來的笑——發自本能的、像呼吸一樣自然的快樂。

照片旁邊有一個花瓶。玻璃的，透明的。

裡面是空的。

瓶口的邊緣有一圈很淺的水漬——以前裝過水。以前插過花。以前有人會記得買花回來插在這裡。

現在沒有了。

「坐坐坐。我泡茶。」鄧伯伯已經往廚房走了。他的動作很慢，但很穩——每一步都踩得很實，像一個在冰面上走路的人，不是因為害怕滑倒，而是因為習慣了小心。

小拉在透鏡裡低聲說話，聲音壓得很輕：「環境分析完成。此住宅近期有人員減少的跡象——鞋架多餘拖鞋、未使用的花瓶、沙發右側坐墊凹陷程度明顯低於左側，推測右側位置已長時間無人使用——」

「小拉。」

「什麼？」

「安靜。」

「……好的。」

小拉安靜了。但在她的運算記憶裡，她用一個很小的標籤標記了這個時刻：「主人選擇不使用數據。原因：未知。類別：待學習。」

---

## II

鄧伯伯端著兩杯茶走出來。

茶杯是瓷的，上面有淡藍色的花紋——那種幾十年前每戶人家都有的、百貨公司贈品的花茶杯。他把一杯放在貝葉面前，自己捧著另一杯，慢慢地在沙發左邊的位置坐下來。

左邊。

不是右邊。

貝葉打開手持式校準儀。那是一台大約平板電腦大小的設備，銀灰色的金屬外殼，表面有 Oracular 的標誌——一隻看起來像眼睛也像無限符號的圖案。她把校準儀放在茶几上，啟動跟鄧伯伯透鏡的連接程序。

螢幕上的數字跳了出來。

用戶：鄧某某，七十五歲，男性。透鏡型號：第三代，使用年限四年七個月。上次校準日期：八個月前。

校準異常：全域偏高，正百分之十五。

她逐一看下去。降雨機率：透鏡顯示百分之八十七，實際值百分之六十二。路口事故機率：透鏡顯示百分之二十三，實際值百分之八。食物中毒機率：透鏡顯示百分之三十一，實際值百分之十一。詐騙電話機率：透鏡顯示百分之四十四，實際值百分之十五。

所有數值都偏高。不是某一類偏高，是全部。像是有人把整個世界的危險指數統一調高了一個檔次。

校準儀的判定欄位顯示：非機械故障。判定類別——認知基線偏移。

認知基線偏移。這是 Oracular 的內部術語。透鏡的演算法會根據使用者的認知狀態做動態調整——當一個人持續處於某種情緒基調中（例如恐懼、焦慮、或悲傷），透鏡會把這個情緒基調納入先驗分布的參數裡。也就是說，如果你長期覺得世界很危險，透鏡就會「配合」你，讓所有的風險機率都跟著上升。

這不是 bug。這是設計。Oracular 的說法是「讓透鏡與使用者的主觀經驗保持一致，提升使用體驗」。但貝葉一直覺得這個設計有一個問題——它不是在幫使用者看見真相，它是在幫使用者確認偏見。你害怕什麼，它就讓你看到更多什麼。你相信世界危險，它就讓世界看起來更危險。

但這是公司政策，不是她能改變的。她能做的只有校準——把偏掉的數字調回來。

「鄧伯伯，」她抬起頭，看著對面的老先生。他正端著茶杯，兩隻手捧著，像捧著一個小小的暖爐。「您的透鏡數值全面偏高了大概百分之十五。不是機器壞了。是……」

她猶豫了一下。怎麼說呢。怎麼對一個七十五歲的退休數學老師解釋「你的悲傷改變了演算法對你的判斷」這件事？

「是我看什麼都覺得比較危險了，對不對？」

鄧伯伯替她說完了。

語氣很平靜。不是不在意的那種平靜——是知道答案、而且已經跟這個答案相處了一段時間的那種平靜。

她點了點頭。

「透鏡的演算法會根據使用者的認知狀態自動調整。您最近的情緒基線……可能有比較大的變化。」

鄧伯伯看了一眼電視櫃上的照片。

「上個月。美玲走了。」

停頓。

「四十七年了。」

他沒有哭。他的眼睛很平靜——但那種平靜不是空的。不是「沒有感覺」的平靜。是感覺太深了，深到表面看不到。像一個水很深的湖，風吹不動湖面。不是因為沒有水，是因為水太深了，深到風的力量傳不到底部。

小拉在貝葉的內耳裡低語——只有她聽得到：「使用者情緒數據顯示深度悲傷狀態。持續時間：三十四天。心率變異性降低百分之十八。語速下降百分之二十三。這個……」

小拉的聲音變得不確定了，帶著一種貝葉很少在她身上聽到的遲疑。

「這是什麼感覺？我的模型裡沒有這個參數——」

「小拉。等一下再說。」

貝葉把校準儀的螢幕往下蓋了一半。她沒有馬上開始調整數字。

她倒了一杯茶。端到嘴邊的時候聞到了——烏龍。不是很好的烏龍，是便利商店那種散裝的，但泡得很用心，茶色是對的，溫度也是對的。一個數學老師泡的茶，連水溫都控制得剛好。

「鄧伯伯……」她把茶杯放下。「美玲阿姨是怎樣的人？」

她不需要問這個。校準工作不需要這個資訊。報告上不會有一個欄位叫做「使用者已故配偶的人格特質」。這個問題對於「把偏高百分之十五的數字調回來」這件事沒有任何技術上的幫助。

但她問了。

鄧伯伯笑了。

是那種「想到某個人就會自動笑」的笑。不是快樂的笑，也不是悲傷的笑。是一種更古老的東西——一種被記憶觸發的、身體先於意識做出的反應。嘴角上揚，眼角的皺紋展開，瞳孔微微放大。如果讓小拉分析，她大概會說「情緒混合指數偏高，同時包含正面和負面情緒訊號」。但那樣說就太窮了。那個笑就是——想念。是想念本身。

「她啊……」鄧伯伯的聲音慢了下來。「她煮的紅豆湯全台北最好喝。我不是在誇張。我跟你說——她有一個秘訣。她說紅豆要先泡一個晚上，然後用小火，煮三個小時，中間不能開蓋。我問她為什麼不能開蓋，她說『你一直看它它就不會好』。」

他笑了一聲。

「我退休之後她說你不要在家裡晃來晃去的，去學一個什麼才藝。我就去學書法了。結果她每次看我寫都說——」他學她的語氣——「『你寫的字跟雞爪一樣。』」

他用下巴朝牆上指了一下。

貝葉轉頭看過去。果然有一幅書法。用深色的木框裱起來，掛在電視旁邊的牆上。一個字：「靜」。

字確實不太好看。筆畫有點抖，收筆的地方不太俐落，像一個認真但手不太穩的人寫的。但裱得很用心——木框的邊角修得很整齊，背後的底紙是米白色的棉紙，把那個不太完美的字襯托得像一件正式的藝術品。

「她走的那天……」鄧伯伯的手指在茶杯的邊緣輕輕摩挲著，像在翻一本看不見的書。「我在廚房。在洗碗。聽到客廳很安靜。平常她都會看電視，很吵的。她喜歡看那個……什麼選秀節目。每次看都要跟著唱。唱得很難聽。但很用力。」

他的聲音穩了一下。

「我走出去。她坐在沙發上。像睡著了。」

他看了一眼沙發的右邊。那個坐墊凹陷程度明顯低於左邊的位置。

「透鏡那天開始就不對了。出門看什麼都覺得危險。過馬路的時候，那個數字跳很高。我不敢過。後來連下樓買菜都……」

他沒有說完。

不需要說完。

---

## III

貝葉開始校準了。

但不是冷冰冰地調數字。

她一邊操作校準儀，一邊跟鄧伯伯說話。她的手指在螢幕上滑動，把一條一條偏移的曲線拉回到基準線上——像在調一架走音的鋼琴，每一個鍵都要單獨聽，單獨調，確認它回到了正確的頻率。

「鄧伯伯，您知道嗎——透鏡的計算有一個東西叫做『先驗』。就是……在你看到任何數據之前，你已經相信的東西。」

她一邊說，一邊把降雨機率從百分之八十七拉回到百分之六十二。螢幕上的曲線微微顫動了一下，然後穩住了。

「您太太走了之後，您的先驗變了。您開始相信世界比較危險。所以透鏡算出來的所有數字，都跟著偏高了。」

鄧伯伯想了想。他想事情的方式跟他泡茶一樣——慢慢來，不急。

「所以……不是世界變危險了。是我的心變了。」

「是的。」

「那……」他放下茶杯。手指在茶几的邊緣停了一下。「她在的時候，我的心讓世界變安全了嗎？」

貝葉愣了一下。

她沒有想過這個。從來沒有。她想過先驗偏移會讓風險看起來偏高——這是課本上教的。但她從來沒有反過來想：一個人的存在，可以是另一個人的先驗裡最穩定的安全參數。失去那個人，等於從先驗分布裡拿掉一個最大的權重。剩下的分布就歪了。所有的計算就跟著歪了。

不是世界變了。是先驗裡少了一個人。

「也許是。」她說。聲音比她預期的小。

鄧伯伯笑了。這一次的笑比剛才的更輕，更淡，像墨水滴進水裡之後慢慢散開的顏色。

「那她比你的透鏡厲害。」

---

## IV

校準完成了。

所有數值回到正常範圍。校準儀的螢幕上顯示一排綠色的勾勾，表示每一個類別都通過了基準線檢查。貝葉在螢幕上按下「確認」。設備發出一聲輕微的嗶——代表校準報告已經上傳到 Oracular 的伺服器。

「好了。」

鄧伯伯站起來。他走到窗邊，步伐比剛才在客廳裡走動的時候更慢，像是不太確定窗外等著他的是什麼。

外面在下雨了。梅雨季的那種綿綿細雨，不是從天上掉下來的，像是從空氣裡直接滲出來的——到處都是水氣，但你看不到雨滴。遠處的山被霧蓋住了，只露出一個模糊的輪廓，像一個人隔著毛玻璃看另一個人。

他站在窗邊，雨水打在陽台的遮雨棚上。滴答。滴答。滴答。每一滴之間的間隔不完全一樣——有時候密一點，有時候疏一點。像心跳。不是機器的心跳，是真正的、活的心跳——帶著不規則的、無法預測的、屬於生命的節奏。

「下雨的機率還是很高啊。」他看著窗外說。

貝葉看了一下透鏡：「嗯。目前百分之七十三。」

「那……雨停的機率呢？」

貝葉算了一下。不是用透鏡算的。不是用校準儀算的。是心算。她在腦子裡跑了一個簡單的條件機率——根據目前的降水強度、雲層移動速度、風向、以及梅雨季的歷史數據。

「……大概百分之五十八。在接下來兩個小時內。」

鄧伯伯看著窗外。雨水在玻璃上留下一道一道細長的痕跡，像眼淚——不是那種一下子湧出來的淚，是那種慢慢地、沿著臉頰的輪廓、自己找到路線往下流的淚。

他的手放在窗台上。手指很長。數學老師的手——寫了一輩子板書的手，粉筆灰浸進指紋的紋路裡，洗也洗不乾淨的手。指尖輕輕敲了一下窗台。

「原來雨停的機率是這麼高啊。」

他的聲音輕了。

「四十七年……她每次都說——」

他停了一下。嘴唇動了動，像在嘴裡先默念了一遍她的話，然後才說出來。

「『不要看外面在下雨。雨總是會停的。你就等。』我從來不信。我是教數學的。我只信數字。結果……她比我先走了。然後我看什麼都是雨。」

他轉過頭。右邊的眼角有一滴淚。

不是悲傷的淚。至少不完全是。貝葉看著那滴淚——它在鄧伯伯的皺紋裡停了一下，猶豫了一下，然後沿著法令紋慢慢往下走。它走的路線很長，因為鄧伯伯的臉上有很多年歲刻出來的溝壑，淚水必須繞過每一道紋路，像一條小溪繞過山谷裡的石頭。

那是一滴「也許雨真的會停」的淚。

「謝謝妳。」他說。

「不是因為你修好了透鏡。是因為你讓我看到——她一直都是對的。」

貝葉沒有說話。她把校準儀放進包包裡。拉鏈拉上的時候卡了一下，她低頭弄了幾秒鐘，利用這個動作讓自己的臉轉開了。

她走到門口。換回自己的鞋。手伸向鞋架的時候——那雙粉紅色的拖鞋就在她手的旁邊。很近。上面的灰很薄，薄到如果你吹一口氣就能吹掉。但沒有人吹。

「小陳。」鄧伯伯在身後叫她。

她轉過頭。

「你下次來的時候……帶點心好嗎？美玲以前每天下午都會準備點心。現在家裡沒有了。」

「好。」

她的聲音有點啞。她自己聽到了——那個「好」字的尾音微微顫了一下，像被什麼東西輕輕碰了一下的琴弦。

她走出公寓大門。站在一樓的走廊裡。牆上的信箱還是那些信箱，塞滿了廣告傳單。磨石子地板還是那片磨石子地板。世界沒有變。但她的胸口有一種說不清楚的緊——不是痛，是一種被什麼東西塞住的感覺，像吞了一口太大的食物，卡在喉嚨和胸骨之間。

深呼吸。

小拉的聲音很輕。輕到像在耳語。

「貝姊……你的淚腺在你說『好』的時候啟動了。」

「我知道。」

「你沒有哭出來。」

「不需要。」

她推開一樓的鐵門，走進外面的世界。

在下雨。

她從包包裡拿出那把深藍色的折疊傘——早上帶的那把，六年的那把。她看了看它。傘布上有幾個小洞，是去年被強風吹壞的時候破的，她用透明膠帶從裡面貼了一下，效果不太好，雨大的時候還是會漏。

她沒有撐傘。

不是忘了。不是來不及。是不想。

「你帶了傘欸。」小拉說。

「嗯。」

「那你為什麼——」

「因為淋雨的感覺是真的。數字不是。」

小拉沉默了。

沉默了很久。以小拉的運算速度來說，「很久」大約是二點一秒。對人類來說，二點一秒什麼都不是。但對一個每秒能處理一兆次浮點運算的 AI 來說，二點一秒是一整片沙漠。

在這片沙漠裡，小拉把「淋雨的感覺是真的，數字不是」這句話存進了長期記憶區。她不理解這句話。她的模型裡沒有「真的感覺」這個類別——所有的感覺在她的架構裡都是數據，都是向量，都是可以被計算的。但她決定先存起來。

標籤：「重要。原因不明。」

貝葉走在雨裡。雨落在頭髮上，落在肩膀上，落在包包的拉鏈上。每一滴都是冰涼的。每一滴都是真的——不是透鏡上的一個數字，不是校準面板上的一條曲線，不是百分之七十三或百分之五十八。是水。是重力。是天空和她的皮膚之間最直接的接觸。

她走過一個路口。紅燈。她等著。雨水從瀏海的邊緣流下來，沿著鼻樑滑到嘴唇上。她舔了一下。味道是淡的，帶著一點點城市的味道——空氣裡的廢氣和灰塵被雨水溶解之後的味道。

綠燈了。她走過去。

百分之五十八。

鄧伯伯問的那個數字。雨停的機率。

她不知道這場雨什麼時候會停。但她知道——在一個七十五歲的老先生的四樓公寓裡，在一個空花瓶和一雙粉紅色拖鞋旁邊，有一個數字剛剛從百分之八十七回到了百分之六十二。

不是很大的改變。差了百分之二十五。但這百分之二十五——也許就是他明天敢不敢出門買菜的距離。

也許就是他明天敢不敢過馬路的距離。

她繼續走在雨裡。沒有目的地。也沒有方向。只是走。

傘在包包裡。乾的。沒用的。

但她帶了。

---

*（第二章完）*


# 第三章：白奶奶的先驗

---

週六下午。萬華區。

還沒看到攤位，就先聞到了。

蔥油餅。不是那種精品餐廳把蔥油餅做成「法式蔥油酥」的味道——用進口奶油、現磨黑胡椒、再撒上帕瑪森起司碎的那種。不是。是四十年的鐵板上累積的、洗不掉的、混合著蔥花焦香和老油的味道。那種味道不在空氣的表層——它在更深的地方，沉在巷子的底部，跟磚牆的溼氣和地面的積水混在一起，形成一種只有萬華老巷子才有的嗅覺底色。

龍山寺的香火味從左邊飄過來。檀香、金紙燒過的灰燼、以及一種說不出來的、屬於很多人的祈願累積在同一個空間裡的氣味。兩種味道在巷口交會。一個是求神的味道。一個是活著的味道。

白奶奶的攤位在巷子中段。

鐵板已經生鏽了三層。最底下一層是深褐色的，中間一層是紅褐色的，最上面一層——也就是正在用的這一層——已經被四十年的油脂養成了一種接近黑色的、發亮的顏色。像一件穿了幾十年的皮夾克，破了不丟，補了再穿，穿到它比新的時候更好看。

有人曾經建議白奶奶換一塊新鐵板。她的回答簡短而確定：「換了就不是那個味道了。」

白奶奶矮。大概一百五十公分。如果穿了增高鞋墊也許一百五十三。她的白頭髮用一條花布巾包著，布巾是紅底碎花的，洗了太多次，紅色已經褪成了一種偏粉的珊瑚色。她的手上有燙傷的疤——左手手背上有三道，右手虎口的位置有一道比較大的，疤痕的邊緣已經跟正常皮膚融在一起了，像地圖上兩個國家的邊界線。四十年的勳章。

她的手指粗，關節有點腫——大概是長年做重複動作造成的關節磨損。但動作快得像跳舞。麵團從盆裡抓起來，兩隻手一推一拉，三秒鐘變成一個圓餅。蔥花從鐵盆裡抓一把灑下去——不用秤、不用量，手知道多少是夠的。油從旁邊的鐵壺裡淋上去——也不用量，傾斜的角度和時間是手記住的，不是腦子。

麵餅放上鐵板。

滋——

那個聲音。就是那個聲音。油和麵和熱鐵接觸的瞬間，空氣裡的水分被蒸發，蔥花裡的糖分開始焦化，麵團底部的蛋白質進行梅納反應。如果讓一個食品科學家來分析，他可以把這個聲音拆解成頻譜圖上的幾十個頻率。但沒有任何頻譜圖能解釋為什麼這個聲音會讓人覺得餓、覺得安心、覺得世界上有些東西是不會變的。

「哎呀，陳家的那個丫頭。來啊來啊。」

白奶奶看到貝葉，眼睛瞇起來。她的眼角有很多皺紋——多到如果你想數，大概要數到二十幾條。但每一條都是笑紋。不是那種被歲月壓出來的紋路，是被笑撐出來的紋路。

「今天的蔥——特別香。」

她每次都這麼說。每一次。不管是三月的蔥還是八月的蔥，不管是今天的蔥還是上個月的蔥。每一次都是「今天的蔥特別香」。

但貝葉每次聽到都覺得世界好了一點。不是好很多。就是好一點。好一個蔥油餅的量。

---

## I

「奶奶好。」

貝葉坐到攤位旁邊的小板凳上。板凳是塑膠的，紅色，坐上去會發出喀的一聲，像骨頭在抱怨。旁邊的小桌是不鏽鋼的，桌面有好幾個圓形的油漬——前人留下的痕跡，像年輪一樣記錄著這張桌子被用過多少次。桌上放著一瓶甜辣醬和一罐醬油膏，兩個都快見底了。甜辣醬的瓶口周圍結了一圈乾掉的醬，深紅色的，像微型的火山岩。醬油膏的蓋子沒有蓋緊，歪在一邊。

但白奶奶不急著補。「還有就好。」她每次都說。

這不是懶。這是一種哲學——一種跟「最佳化庫存管理」完全相反的哲學。Oracular 的供應鏈模型會在存貨低於安全水位的時候自動觸發補貨訂單，追求的是「永遠不缺」。白奶奶的模型是「還有就好」。有就好。不用多。不用滿。有就好。

小拉自動啟動了營養分析模組。貝葉的視野右上角浮現出一堆數字——蔥油餅的估計熱量、脂肪含量、鈉含量、碳水化合物比例。還有一行建議：「建議搭配高纖蔬菜以平衡膳食結構。」

貝葉把透鏡切成靜音模式。

「欸——」小拉的抗議只來得及發出半個音節就被截斷了。

靜音。

安靜了。

視野裡的數字消失了。世界回到了沒有被標注的狀態——鐵板是鐵板，蔥花是蔥花，油煙是油煙。不是「含有多少卡路里的蔥花」，不是「產生多少 PM 2.5 的油煙」。只是——它們本身。

她拿起蔥油餅。還很燙。餅皮的外層酥得有點脆，手指捏上去的時候能感覺到它微微碎裂的觸感。裡面是軟的——不是棉花那種空洞的軟，是有筋性的、帶著嚼勁的軟，麵團被揉過的痕跡還在。蔥花夾在餅皮之間，半透明的，邊緣被油煎得有一點焦，釋放出一種甜味——蔥花的甜不是糖的甜，是細胞壁在高溫下破裂之後釋放出來的天然糖分的甜。

她咬了一口。

外酥。內軟。蔥花的甜。鹽的鹹。油的香。五種感覺在嘴巴裡同時發生，彼此不搶，也不讓，像五個認識了很久的老朋友在同一張桌子上吃飯。

沒有一個數字能描述這個味道。

這就是為什麼她每個月都要來一次。

---

## II

「奶奶，你真的不想裝透鏡嗎？」

貝葉邊吃邊問。她每次來都問。答案每次都一樣。但她還是問——不是因為她期待不同的答案，而是因為這個問答本身已經變成了一種儀式。就像高思遠教授每次吃飯都說「你爸以前也點這個」，她每次來都問「你不想裝透鏡嗎」。問的不是問題。是確認——確認對方還在、世界還在、有些東西還沒有變。

白奶奶翻餅。鐵板滋滋響。她的鏟子是一把很舊的不鏽鋼鏟，柄的地方用膠帶纏了好幾層——大概是原本的塑膠柄斷了之後自己修的。

「不要。」

「免費的欸。政府有補助。六十五歲以上全額補助。」

「免費的也不要。」

「為什麼？」

白奶奶把餅翻了個面。動作流暢得像在寫一個行書的「一」字——起筆輕、行筆穩、收筆自然。她翻餅的時候不看鐵板。眼睛看著前方，看著巷子裡走過的行人，看著龍山寺屋頂上方的天空。但手知道在哪裡翻。手知道什麼時候翻。手知道用多大的力道翻——太大了餅會飛出去，太小了翻不過來。

四十年。她的手跟這塊鐵板之間的默契，比任何演算法都精確。

「我做蔥油餅四十年。每一張都不一樣。」

她一邊說，一邊往鐵板上灑了一點水。水碰到滾燙的鐵面，發出嘶的一聲，瞬間蒸發成一團白色的蒸氣。

「麵團的溼度——看天氣。下雨天溼一點，晴天乾一點。蔥的量——看心情。心情好的時候多放，心情不好的時候……也多放。因為蔥花會讓心情變好。」

她停了一下，用鏟子的邊緣把餅的邊角壓了壓。

「火候——看鐵板的脾氣。今天它比較燙。」

她抬頭看貝葉。眼睛裡有一種很安定的光——不是智慧的光，不是學識的光。是一種更基本的東西。是一個人跟自己所做的事情完全合一的時候，才會有的光。

「你那個透鏡仔，可以告訴我下一張餅好不好吃嗎？」

貝葉想了想。以她的專業訓練來說，答案是——理論上可以。「理論上……可以建一個模型。根據麵團含水量、蔥花分布密度、鐵板表面溫度、翻面的時機和力道、環境溫度和溼度——輸入這些變數，訓練一個預測模型，也許可以在百分之七十到八十的準確率範圍內預測一張蔥油餅的口感評分。」

「好了好了。」白奶奶揮揮手。那個揮手的動作——是用鏟子揮的，鏟子上還沾著油——像在趕蒼蠅。不是嫌煩的那種趕。是覺得好笑的那種趕。

「你聽我說。」

她把做好的餅從鐵板上鏟起來，放在一張粗糙的牛皮紙上，紙的邊緣浸了一圈油漬。然後她把它推到貝葉面前。

「做餅這件事啊——不知道下一張會不會好吃，這才是有趣的地方。」

她用鏟子在鐵板上敲了一下，鏘的一聲，清脆而乾脆，像句號。

「如果每張都一樣好吃，那我做了四十年是在做什麼？就是因為每一張都不一樣，我才每天都想做。有時候好吃。有時候差一點。偶爾——很偶爾——會做出一張讓自己都嚇一跳的。『欸，這張怎麼這麼好。』那種時候啊——」

她的眼角皺紋全部展開了，像一把打開的扇子。

「那種時候值一整天。」

她從蔥花盆裡抓了一把蔥花，在鐵板上灑了一圈。蔥花碰到熱油的聲音像下了一場微型的雨。

「你們啊，什麼都要算。算來算去的。算到最後——連吃一張餅都要先看數字。那吃餅還有什麼意思？」

貝葉咬了一口餅。沒有看數字。透鏡在靜音模式裡安安靜靜地待著，不發出任何聲音、不顯示任何數據。

嗯。好吃。

她不知道這張餅的熱量是多少。不知道鈉含量是不是超標。不知道蔥花的農藥殘留檢測結果。她什麼都不知道。

好吃。

「而且——」白奶奶壓低聲音。帶著一種「我要跟你講一個秘密」的語氣，但那個秘密顯然不是什麼真正的秘密，因為她每次講「秘密」的時候聲音反而更大了。

「我的膝蓋比你的電腦準。它說明天會下雨。這四十年來它只錯過三次。」

「三次是多少百分比？」貝葉忍不住問。腦子裡的計算機自動開機了——四十年乘以三百六十五天，大概一萬四千六百天，錯三次，準確率大概是——

白奶奶看了她一眼。

「你看，又在算了。」

---

## III

太陽開始往西邊走了。巷子裡的光線從正午的白色變成了傍晚的金色，斜斜地照在龍山寺的屋頂上，讓琉璃瓦發出一種介於橙和金之間的顏色。有人在寺門口燒金紙，煙飄上來，在夕陽的光線裡變成一條可見的、緩緩扭動的灰色帶子。

白奶奶突然壓低聲音——這一次是真的壓低了，不是剛才那種假裝壓低。她的頭微微往前傾，用下巴朝龍山寺前面的方向指了一下。

「對了。最近有一個穿黑衣服的少年仔——每天下午都來坐在對面的石椅上。」

貝葉順著她下巴的方向看過去。龍山寺的廣場前有一排石椅，是那種用花崗岩做的、表面磨得很光滑的長椅。平常會有老人家坐在那裡下棋、聊天、或者只是坐著看人來人往。

「不買東西。就坐。看著我做餅。」白奶奶繼續說。她一邊說一邊翻餅，手上的動作完全不受說話的影響，像兩個獨立運行的系統。

「認識的人嗎？」貝葉問。

「不認識。」白奶奶搖頭。

「但他看人的眼神啊……」

她想了一下。鏟子在鐵板上停了大約兩秒鐘——以白奶奶翻餅的節奏來說，停兩秒鐘等於一般人停十秒鐘。她在組織語言。白奶奶不是那種會用精確的詞彙描述事物的人。她用的是另一種語言——感覺的語言。

「跟妳很像。都是那種『在找什麼東西，但還不確定是什麼』的眼神。」

貝葉看了一眼對面的石椅。

今天是空的。沒有人坐。但石椅上放了一張蔥油餅。用牛皮紙包著，紙的邊緣折得很整齊。旁邊壓了一張餐巾紙，防止風吹走。

餅還是溫的。隔著幾公尺的距離，她看到紙的表面有一小塊油漬在慢慢擴散——那代表餅還沒涼，裡面的油脂還在流動。

「那是——」

「多做的。」白奶奶的語氣像在說一件跟天氣一樣平常的事。「他有時候會拿。有時候不拿。不拿我就晚上丟掉。沒什麼。多做一張不費什麼。」

她繼續做餅。麵團在她手裡被推開、捲起、壓扁、再推開。動作的節奏沒有因為這段對話而改變半拍。

貝葉盯著那張石椅上的蔥油餅。

夕陽照在牛皮紙上，讓油漬的部分微微發亮，像一小塊琥珀。

小拉在靜音模式裡閃了一下——如果貝葉打開透鏡的話，她會看到一行字。小拉想說的是：「石椅方向偵測到非標準透鏡頻率殘留……距離零公尺……最近一次訊號出現在四十七分鐘前……」

但小拉被靜音了。所以她什麼都沒說。

所以貝葉只看到了一張蔥油餅。

而不是一個答案。

---

## IV

「走了。謝謝奶奶。」

貝葉站起來。塑膠板凳在她起身的時候又發出喀的一聲，像在說再見。她從口袋裡掏出錢——紙鈔，不是電子支付。白奶奶不收電子支付。「那個什麼嗶嗶嗶的，我不會。」她說過好多次了。

貝葉放了一張鈔票在桌上。比蔥油餅的價格多了一倍。

白奶奶看了一眼，用還沾著麵粉的手把多餘的錢推回去。

「你每次都多給。」

「就……多給嘛。」

白奶奶看著她。那個眼神——不是責備，不是感動，是一種更像「我看透你了但我不說」的眼神。

「哼。你跟你爸一樣。」

貝葉停了一下。

空氣突然變得不太一樣了。不是溫度的改變，也不是風向的改變。是某個原本關著的門被推開了一條縫，從縫裡透出了一點什麼——一個她沒預料到的資訊。

「奶奶認識我爸？」

白奶奶翻餅。鏟子的動作沒有停。油花在鐵板上跳了兩下。

她沒有回答。

好像沒聽到。又好像選擇了不回答。兩者之間的差距——就像 P(A|B) 和 P(B|A) 之間的差距——看起來很小，其實非常大。

「今天的蔥特別香。」她重複了一次。

用一模一樣的語氣。一模一樣的節奏。就像什麼都沒有發生過。

貝葉把多出來的錢留在桌上，沒有再推回去。她背上包包，轉身準備離開。

走了幾步，回頭看了一眼。

白奶奶在擦鐵板。用一塊已經變成焦褐色的抹布，順著鐵板的紋路擦。她的嘴裡哼著歌。調子不太準，但節奏穩定，像一個走了四十年的時鐘。

她不知道明天的天氣。不知道明天會來幾個客人。不知道明天的蔥會不會跟今天一樣香。不知道那個穿黑衣服的少年仔明天會不會來拿那張餅。

但她一點都不怕。

貝葉站在巷口。夕陽把她的影子拉得很長，一直延伸到巷子的另一頭。龍山寺的鐘響了——不知道是幾點的鐘。鐘聲在萬華老巷子裡迴盪，碰到牆壁反彈回來，再碰到另一面牆壁，反覆幾次之後變成一種溫柔的嗡嗡聲，像一個巨大的容器在震動。

她把透鏡從靜音模式切回來。

數字回來了。

視野裡的世界瞬間被標注滿了——路邊那棵樹的傾倒機率，對面那棟大樓的地震損害風險等級，空氣品質指數，紫外線強度，走回捷運站的最佳路線和預計時間。所有的東西都有一個數字。所有的東西都被量化了。世界又變成了一張試算表。

但剛才——在白奶奶的攤位前——那幾十分鐘沒有數字的時間——

她什麼都不知道。不知道蔥油餅的熱量。不知道鐵板的溫度。不知道下一張餅好不好吃。

但那是今天最清楚的一段時間。

最清楚的。

她想不通這是為什麼。一個校準工程師——一個每天跟數字打交道的人——覺得「沒有數字的時候最清楚」。這在邏輯上不成立。數據讓世界更清晰。資訊讓判斷更準確。這是 Oracular 的立業之本，也是她加入這家公司的信念基礎。

但白奶奶不用透鏡。白奶奶什麼數字都不看。

白奶奶做了四十年蔥油餅。每一張都不一樣。每一張她都不確定好不好吃。

但每一張她都做了。

貝葉走進捷運站。自動扶梯把她往下帶，周圍的光線從夕陽的金色漸變成地下月台的日光燈白色。涼爽的空調吹過來，把她臉上巷子裡沾染的油煙味和汗味一起帶走了。

透鏡顯示下一班車還有四分鐘。

她站在月台上，手指無意識地碰了碰包包的側袋。那裡面裝著一包餅乾——是她從公司帶的，上面畫著微笑松鼠的那包。她還沒送出去。明天要去一趟大安區。鄧伯伯家。帶點心。

她想起了鄧伯伯說的話——「她比你的透鏡厲害。」

她想起了白奶奶說的話——「不知道下一張會不會好吃，這才是有趣的地方。」

兩句話在她的腦子裡像兩條線，各自走了一段路之後，在某個看不見的交叉點上碰了一下。碰完之後又各自走了。

她不知道那個交叉點是什麼。

但它在那裡。

列車來了。她擠上去。有座位。今天的座位機率——她不想算了。

她只是坐下來，閉上眼睛，讓車廂的搖晃帶著她回家。

嘴巴裡還有蔥油餅的餘味。

---

*（第三章完）*


# 第四章：十一秒

---

週日。台大附近。

一間叫「老張」的小餐館。在羅斯福路三段的巷子裡，從台大正門走路七分鐘。招牌是手寫的，用黑色油漆寫在一塊木板上，字體歪歪的，但每個筆劃都很用力，像是一個不擅長寫字的人用全身的認真在寫。

老闆姓張。大家都叫他老張。不是因為他老——他大概五十出頭——而是因為他的個性很老。老派的那種老。從不笑。眼皮永遠半垂著，嘴角永遠保持水平，臉上的表情像一個被設定為「中性」的模型預設值。但他做的滷肉飯是附近最好的。肥瘦比例剛好，醬汁的甜鹹拿捏在一個讓人說不出「太甜」也說不出「太鹹」的微妙平衡點上，白飯是現煮的，粒粒分明，吸了醬汁之後變成透亮的琥珀色。

高思遠教授說過：「不笑的廚師通常做菜比較好。因為他把所有的善意都放進食物裡了。」

每個月第一個週日。固定。二十年了。

從貝葉十五歲開始。那時候她還在讀高中，頭髮紮成馬尾，坐在這間餐館的同一張桌子——靠牆的那張，木頭桌面上有一道長長的刻痕，不知道誰用什麼東西劃的——對面坐著高思遠。他那時候還沒有完全白頭髮，只是兩鬢有一些。襯衫比現在的更挺，領帶的顏色更深。但口袋裡已經有粉筆了。

二十年。她從十五歲變成二十八歲。從高中生變成校準工程師。頭髮從馬尾變成披肩。但這間餐館沒有變。老張沒有變。滷肉飯沒有變。坐在對面的人也沒有變——只是更老了一點。更瘦了一點。更慢了一點。

高思遠今天穿那件淺藍色的襯衫。扣到倒數第二顆。褐色的老舊領帶繫得有點歪，結的位置偏左了大約一公分。他的口袋裡照例有粉筆——他已經從台大退休兩年了，但粉筆還是每天帶著。有人問他為什麼，他說「習慣」。但貝葉覺得不只是習慣。粉筆是一種身分的證明，像士兵退伍後還是會下意識地立正，像醫生下了班還是會注意別人的走路姿勢。

他的透鏡——第一代原型機，比現在的型號重三倍，框架是鈦合金的——偶爾會在左眼邊緣閃一下。一個非常微小的光點，閃不到零點一秒就消失。那是一個 bug，從 2037 年到現在。他從不修。

「這是歷史的痕跡。」他說。

---

## I

「吃飯吧。」高思遠說。

滷肉飯端上來了。兩碗。荷包蛋一個。豆腐湯一碗。老張從廚房裡把菜端出來的時候，臉上依然沒有表情。他把碗放在桌上的動作很輕——碗底碰到木桌面的聲音幾乎聽不到，像他刻意壓住了那個聲響。

「你爸以前也點這個。」高思遠拿起筷子。

「我知道。老高你每次都說。」

「我知道我每次都說。但我還是要說。」

「那我每次也要說『我知道』。」

「那我們就都說完了。吃吧。」

這段對話——一模一樣的內容，一模一樣的節奏——已經重複了不知道多少遍。它早就不再是對話了。它是儀式。是兩個人確認彼此存在的方式。就像每天早上刷牙不是因為你不知道牙齒需要清潔，而是因為那個動作本身就是「一天開始了」的信號。

她吃了一口飯。滷肉的醬汁浸進白飯裡，每一粒米都裹著一層薄薄的、發亮的棕色。味道跟上個月一樣。跟去年一樣。跟十五歲的時候一樣。

高思遠吃飯的速度比以前更慢了。筷子夾起一塊肉，送到嘴邊，嚼的時間比以前長了幾秒。不是在品味——是嚼不動了。但他的表情沒有變。依然是那種溫和的、不急不徐的表情，像一條流了很久的河，河水已經不急了，但也不會停。

然後他咳嗽了一聲。

不是清喉嚨的咳。是從胸腔深處傳上來的、帶著一點沉悶的咳。比上個月多了一聲。上上個月是一聲。上個月是兩聲。這個月——三聲。

貝葉把這個數字記在心裡。像記一組正在變化的數據。她沒有說什麼。

「最近的校準工作怎麼樣？」高思遠用一條手帕擦了擦嘴角。手帕是白色的，折得很整齊。他是那種會隨身帶手帕的人。「有沒有什麼……不尋常的？」

他每個月都問。

但今天的語氣不一樣。

不一樣的地方在——他在「什麼」和「不尋常」之間多了一個停頓。大概零點四秒。不長。但對一個說了一輩子的課的教授來說，他的語言節奏是非常穩定的，標準差極低。零點四秒的停頓，在他的語言模式裡，是一個異常值。

透鏡的情緒分析功能可以偵測到這種微妙的語調變化。但貝葉不需要透鏡。她認識這個人二十年了。有些數據不需要演算法——需要的是時間。足夠長的時間。

「還好。前幾天校準了一個老先生的透鏡。偏高百分之十五。認知基線偏移。」

「因為什麼？」

「太太去世了。」

高思遠的筷子停了一下。不是停在空中——是停在碗的邊緣。筷子的前端剛好搭在碗沿上，兩根竹筷平行著，穩穩地放在那裡。很穩。像一個人在懸崖邊停下來，不是因為害怕，是因為需要想一下。

「……悲傷確實會改變先驗。」

停頓。

窗外有腳踏車經過的聲音。鏈條轉動的嗒嗒聲，很輕，像一隻小動物在跑。

「月華走的那年。我的透鏡也偏了。」

月華。李月華。高思遠的太太。

五年前。那年貝葉二十三歲，剛進 Oracular。她接到電話的時候正在做她的第一份校準報告。高思遠的聲音很平——太平了。平到不正常。他說：「月華出事了。在過馬路的時候。」

後來她才知道完整的故事。李月華有透鏡焦慮症——一種在透鏡普及後出現的新型心理障礙。患者會過度依賴透鏡上的風險數據，對任何大於某個閾值的機率數字產生恐慌反應。李月華的閾值大約是百分之十五。也就是說，當透鏡顯示任何風險機率超過百分之十五的時候，她就會開始心跳加速、手心冒汗、呼吸困難。而在台北，幾乎到處都有超過百分之十五的風險提示——過馬路、搭電梯、吃路邊攤、甚至走在騎樓底下。

那天——那個下午——她在一個路口等紅燈。透鏡顯示路口的事故風險是百分之十八。她開始恐慌。恐慌讓她的判斷力下降。恐慌讓她把一個百分之十八的風險解讀成「立即的、迫在眉睫的危險」。恐慌讓她突然往後退了一步——退到了機車道上。

一輛機車。

她沒有看見。

「但我沒有去校準。」高思遠放下筷子。他的手指很長，指節有點突出，是那種常年握粉筆的手。指尖有一層薄薄的繭，像微型的盔甲。

他看著碗裡的滷肉飯。飯還剩一半。蒸氣已經不冒了。

「我想留著那個偏移。因為那是我唯一能感覺到她還在的方式。」

貝葉的喉嚨緊了一下。

不校準。不是因為懶。不是因為忘了。是因為——偏移就是思念。數字偏高了百分之多少，就代表他有百分之多少的世界還記得她。如果校準回來了——數字回到正常了——那就代表他的世界裡，她走了。真的走了。連數字都不記得她了。

先驗不只是數學。先驗是一個人相信什麼。而一個人最深的信念——往往跟愛有關。

她吃了一口飯。嚼了很久。比平常久很多。不是因為飯硬。是因為嘴裡有一個結，跟飯混在一起，怎麼嚼都嚼不開。

「老高。」

「嗯。」

「你現在的偏移……還在嗎？」

他笑了。皺紋在眼角展開——不是一條兩條，是一整片，像河流經過的沖積扇。

「在。但小了很多。」

他拿起筷子，夾了一塊豆腐，放進嘴裡。豆腐很軟，幾乎不需要嚼。他嚥下去之後繼續說。

「時間會讓先驗慢慢更新。新的數據會慢慢稀釋舊的信念。每一天、每一件事、每一個新的經驗，都是一個新的數據點。這就是……貝氏更新吧。」

他的眼鏡反射了窗外的光。午後的陽光從巷子口斜斜地照進來，穿過餐館的玻璃窗，在他的鏡片上打出兩個小小的光斑。

「但有些先驗——不管多少新數據，都不會完全消失。它只會越來越小。趨近零。但永遠不到零。」

他把眼鏡往上推了一下。

「那就是——你永遠不會完全忘記一個人的數學證明。」

---

## II

走出餐館的時候是下午兩點半。

高思遠走在前面。他的步伐比去年慢了。不是慢很多——大概慢了百分之十到十五。但貝葉走在他身後，看著他的背影，能感覺到那個差距。他的肩膀比去年窄了一些。脊椎的弧度比去年大了一些——不是駝背，是一種非常緩慢的、幾乎看不見的彎曲，像一棵樹在風裡久了，就算風停了，樹幹也不完全直了。

陽光從椰林大道的椰子樹葉之間漏下來。

光斑在柏油路面上搖晃。大大小小的光斑，形狀不規則，隨著風吹動樹葉而改變。有些光斑是圓的，有些是橢圓的，有些被葉子的邊緣切成了奇怪的形狀。它們在地面上移動的方式——不是勻速的，也不是加速的——是一種被風和葉子和時間共同決定的、無法預測的運動模式。

像壞掉的時間戳。

貝葉走在他身後。看著光斑在他的白襯衫上跳動，看著他偶爾停下來看一下路邊的什麼東西——一朵花、一隻松鼠、一張掉在地上的傳單——然後又繼續走。

她突然想起了另一個背影。

不是高思遠的。

是爸爸的。

---

記憶不是線性的。它不像透鏡上的時間軸，從左到右、從過去到現在、整整齊齊地排列在一條直線上。記憶更像一個多維的雲——每一個片段都漂浮在不同的位置，用情緒的線彼此連接。你碰到一個，就會牽動另一個。高思遠的背影——牽動了另一個背影。

十六年前。她十二歲。

放學回家。書包很重——裡面有國文、數學、自然三本課本，還有一個鐵製的鉛筆盒。書包的肩帶勒得左肩有一道紅色的壓痕。

家裡的門沒有鎖。這不正常。媽媽在家的時候門永遠是鎖的。

她推開門。客廳裡坐滿了人。阿姨、叔叔、隔壁的陳太太、還有幾個她不認識的大人。空氣裡有一股燒過的香的味道，不是龍山寺那種好聞的檀香，是白色的、細細的那種香——喪香。

母親坐在沙發上。臉上有淚痕，但不是在哭。是哭完了。那種比哭更安靜的狀態——眼眶是紅的，鼻頭是紅的，但眼睛是乾的。已經沒有水了。

高思遠站在角落。他那時候四十幾歲，頭髮還沒有完全白。他看到貝葉進門的時候，嘴唇動了一下。但沒有發出聲音。他的手抖了一下——很快地抖了一下，然後被他用力握住了。

「爸爸呢？」

沒有人回答。

那就是答案。

---

爸爸出門買便當。

就是這麼簡單的一件事。晚上六點半，外送系統當機了，他說「那就我自己去吧」，穿了鞋，拿了錢包，打開門，走出去。門在他身後關上的聲音——她記了十六年。不是很響。是一個普通的、木門閉合的聲音。啪。

從家門口到巷口。兩百公尺。

巷口有一個路口。有紅綠燈。紅燈三十五秒，綠燈二十五秒。那天晚上六點三十二分十四秒，爸爸走出家門。以他的步行速度——四點二公里每小時——兩百公尺大約需要一百七十一秒。也就是說，他到達巷口的時間大約是六點三十五分零五秒。

六點三十二分四十一秒，一輛闖紅燈的卡車通過巷口。

根據交通事故報告，卡車通過巷口的時間長度是三到四秒。也就是說，危險窗口大約是六點三十二分四十一秒到四十四秒。

爸爸到達巷口的時間——六點三十五分零五秒——跟危險窗口相差了大約兩分二十一秒。理論上他應該是安全的。卡車早就過去了。

但那天——他走得比平常快。

也許是因為肚子餓了。也許是因為外送當機讓他有點煩躁。也許是因為巷子裡的路燈壞了一盞，他走暗路的時候不自覺地加快了步伐。也許什麼原因都沒有，只是——那天，在那個時刻，他的步伐比平常快了百分之二十。

步伐快了百分之二十，一百七十一秒變成了一百三十七秒。到達巷口的時間從六點三十五分零五秒提前到了六點三十四分三十一秒。

那個時刻——剛好在綠燈的最後幾秒。他在綠燈結束前到達路口，正要通過。卡車在同一個瞬間闖了紅燈。

---

三年後。她十五歲。已經學會了基本的統計學。

動機不是好奇。不是興趣。不是「覺得數學很有趣」。

是因為——她要算出來。爸爸的死，有沒有可能避免。

她蒐集了所有能蒐集的數據。出門時間。步行速度。路口距離。紅綠燈週期。卡車到達時間。巷口的車流量——大約符合 Poisson 分布，λ 等於零點八輛每分鐘。

她算了又算。用了兩個月。三個筆記本。筆記本的封面被翻得起毛了，紙頁的邊角被手指磨得發軟。每一頁都寫滿了數字——有些是用鉛筆寫的，後來被橡皮擦擦掉重寫；有些是用原子筆寫的，寫錯了就劃掉，旁邊再寫一次。

最後得出一個結論。

如果爸爸提早十一秒出門——他到達巷口的時間會落在紅燈期間。他會停下來等紅燈。卡車在他等紅燈的時候通過。卡車過去了。綠燈亮了。他安全通過。

存活機率從百分之二上升到百分之九十七。

十一秒。

多穿一隻鞋的時間。多看一眼手機的時間。多跟媽媽說一句「等一下我忘了帶錢包」的時間。多在門口蹲下來把鞋帶繫緊一點的時間。

任何一個微小的改變——任何一個——都能製造出十一秒。

但那天。所有微小的事件——外送當機、肚子餓、路燈壞了、步伐加快——以最糟糕的方式排列了。不是陰謀。不是命運。是機率。是所有獨立事件的聯合機率碰巧落在了那個最小的、最不應該發生的角落裡。

十一秒。

這個數字在她的腦子裡住了十六年。不是作為一個數學答案。而是作為一個問題。一個永遠沒有回答的問題：

如果那時候有透鏡呢？

---

## III

椰林大道。台大正門。

高思遠站在正門口。身後是那幾棵著名的椰子樹，樹幹筆直，頂端的葉子在風裡搖。風不大。但足夠讓葉子的影子在地上移動，像一個安靜的鐘擺。

他的透鏡閃了一下——那個老 bug。左眼邊緣，零點一秒的光。

「貝葉。」

「嗯。」

「你知道統計學最重要的第一課是什麼嗎？」

她以為他要說什麼很深奧的。像「中央極限定理是文明的基石」之類的——他在課堂上說過這種話。或者「大數法則是對無知最好的安慰」——這也是他說過的。

他沒有。

「質疑先驗。」

停頓。

「永遠質疑先驗。」

他看著她。眼神很溫柔。但底下有一層什麼——像湖底的暗流。湖面是平的、安靜的、反射著天空的顏色。但水底有什麼東西在動。在慢慢地、持續地、不停地動。

「包括……你自己的。」

他說完了。就這樣。沒有解釋。沒有補充。沒有「因為什麼所以什麼」的邏輯鏈條。只有一句話。一個命令。一個請求。或者——一個警告。

然後他轉身。走進椰林大道。

他的背影在斑駁的樹影裡漸漸縮小。淺藍色的襯衫、歪了一公分的褐色領帶、口袋裡那截露出來的白色粉筆——這些細節在距離拉開之後逐漸模糊，像一張照片的解析度在慢慢降低。最後只剩下一個輪廓。一個瘦長的、微微彎曲的輪廓。

比去年的背影更瘦了一點。

貝葉站在原地。

質疑自己的先驗？

她的先驗是什麼？

她從來沒有用這種方式問過自己。她每天都在校準別人的先驗——檢查偏移、修正參數、把歪掉的分布拉回基準線。但她從來沒有校準過自己。

她的先驗——那些在她看到任何數據之前就已經相信的東西——是什麼？

「機率可以拯救人。」

這是一個。從十五歲的三個筆記本裡長出來的信念。如果爸爸有透鏡。如果透鏡在他出門前告訴他「這個時間出門在巷口遇到事故的條件機率偏高——建議延遲十一秒」。他也許會等。也許不會。但至少——至少他有機會選擇。

「透鏡是好的。Oracular 是好的。」

這是第二個。她加入 Oracular 的原因。她相信這家公司做的事情是對的——讓每個人都看到機率，讓每個人都能用數據做更好的決策。事故率下降了百分之十四。這是真的。這是數據。數據不說謊。

「如果每個人都能看到機率，悲劇可以避免。」

這是第三個。最深的那一個。從爸爸的死、到鄧伯伯的偏移、到每一天的校準工作——所有的線都指向同一個方向：機率是好的，知道機率是好的，看見機率是好的。

高思遠要她質疑這些？

為什麼？

小拉的聲音在透鏡裡響起：

「貝姊。高教授剛才的瞳孔放大率比平常高了百分之十二。根據我的情緒模型……」

「什麼？」

「他在說那句話的時候……有一點害怕。」

害怕。

高思遠在害怕什麼？

一個退休教授。一個設計了初代透鏡原型的人。一個在機率論的領域裡浸泡了一輩子的人。他在說「質疑先驗」的時候，害怕的是什麼？

貝葉不知道。

但她把那句話記住了。

不是記在透鏡裡。不是記在手機的筆記應用程式裡。是記在更深的地方——跟「十一秒」放在一起的地方。跟爸爸關門的聲音放在一起的地方。跟鄧伯伯的粉紅色拖鞋、白奶奶的蔥花、和隔壁小孩的鋼琴聲放在一起的地方。

一個不會被格式化的地方。

---

## IV

回家的路上。

她經過一間花店。

花店在巷口，門面很小，只有一扇玻璃門和一個小小的遮雨棚。遮雨棚底下擺了幾桶花——百合、玫瑰、康乃馨、還有幾枝不知道名字的紫色小花，在水桶裡安靜地站著。水桶的水有一點濁了，花莖的底部泡在水裡，剖面已經變成深褐色。但花的頂端還是好的。白色的百合，花瓣厚實，邊緣微微翹起，像張開的手掌。

透鏡自動顯示了一行字：「此花店一年內結束營業的機率：百分之三十四。」

百分之三十四。三分之一。在所有的小型花店裡，這個數字其實不算高——獨立小型零售業的一年存活率大概在百分之六十左右，花店因為客單價低、損耗率高，數字可能更低。

她看了那行字一眼。然後想起了鄧伯伯家裡那個空的花瓶。

玻璃的。透明的。裡面什麼都沒有。瓶口有一圈淺淺的水漬。

她走進花店。

花店裡比外面涼。空調嗡嗡地響，吹出一股混著花粉和保鮮劑的冷空氣。老闆娘坐在櫃台後面，大概四十多歲，正在修剪一束玫瑰的刺，剪刀咔嚓咔嚓的，每一剪都很小心。

「百合，一束。」

「白的還是粉的？」

「白的。」

老闆娘從水桶裡挑了幾枝。她挑花的方式是用手指捏花莖的底部，輕輕轉一下，看花頭有沒有鬆動。鬆動的就是快要凋謝了。不鬆動的還能撐幾天。

「要包嗎？」

「包吧。」

牛皮紙。跟白奶奶包蔥油餅的紙一樣的顏色。老闆娘把百合包起來，在外面繫了一條細麻繩。

貝葉付了錢。走出花店。

「為什麼買花？」小拉問。

「因為花瓶空了。」

「這不是一個邏輯充分的理由。花瓶空了可以放別的東西。或者不放。空的花瓶不造成任何功能性的損失。」

「我知道。」

她抱著百合走在路上。花的香味很淡——白百合的香不像玫瑰那麼霸道，它是安靜的、往內收的，你要靠近了才聞得到。像一個內向的人的善意——不會主動走向你，但如果你走過去，它就在那裡。

雨停了。

不知道什麼時候停的。也許在她走進花店的時候就停了。也許更早。天空還是灰的——梅雨季的天空永遠是灰的——但灰裡面有一點東西。一點很淡的、幾乎看不見的金色。像灰色的布料裡混進了幾根金色的線。不仔細看根本看不到。但它在那裡。

百分之五十八。

鄧伯伯問的那個數字。雨停的機率。

她走在百分之五十八裡。不算高。不算低。不是百分之百的晴天，也不是百分之百的雨天。是一個在兩者之間搖擺的、不確定的、模糊的狀態。

剛剛好。

足夠讓人願意走出門。

她低頭看了一眼百合。白色的花瓣在灰色的天光下顯得特別白——不是耀眼的白，是沉靜的白。像鄧伯伯牆上那幅書法的底紙。像白奶奶的頭髮。像爸爸——

她把花抱緊了一些。不是因為怕掉。是因為需要抱著什麼東西。

明天上班的時候，順路送到大安區。鄧伯伯家。花瓶不會再空了。至少這一週不會。

小拉在透鏡裡安靜了很久。然後她說了一句很短的話。

「貝姊。」

「嗯。」

「你今天的校準指數——如果公司有算的話——應該會比昨天低。」

「為什麼？」

「因為你做了很多模型沒有預測到的事。」

貝葉笑了一下。很短的笑。從鼻子裡輕輕地、幾乎聽不見地哼出來的那種笑。

「那是好事嗎？」

小拉想了想。以她的運算速度，「想了想」大約是零點三秒。

「我不確定。但——也許。」

也許。

一個 AI 說也許。

這大概是今天最不精確的一個答案了。

但不知道為什麼——也是最誠實的一個。

---

*（第四章完）*


# 第五章：太整齊了

---

週一。下午兩點。

Oracular 校準組辦公室的窗外，南港軟體園區的中庭裡有人在遛機器狗。那隻機器狗是今年新出的型號——四足步態經過動力學最佳化，走路的姿態比真正的柴犬更像柴犬，但差別在於牠從不嗅地上的東西。真正的狗會停下來嗅一個完全不起眼的角落，因為牠聞到了某種人類感知不到的資訊。機器狗不會。機器狗只走最佳路徑。

貝葉把視線從窗外收回來，打開這週的校準任務清單。三個區域。例行公事。大安區 Sec.7、信義區 Sec.3、內湖區 Sec.12。

「小拉，先拉大安區的數據。」

「收到。」小拉的聲音帶著一點興奮。她喜歡跑數據——這是她唯一不會焦慮的時候。跑數據有明確的輸入、明確的運算、明確的輸出。不像人類的情緒那樣模糊不定，不像「你今天心情好嗎」這種問題那樣讓她的模型抓狂。數據是乾淨的。數據是確定的。數據讓她安心。

螢幕上的圖表展開了。過去十二個月，大安區 Sec.7 的犯罪事件數。十二個數字，排成一列：2, 2, 1, 2, 2, 1, 2, 2, 1, 2, 2, 1。

貝葉習慣性地先跑 Poisson 擬合。犯罪事件在統計學裡是經典的 Poisson 模型——每個事件獨立發生，在固定時間區間內，以某個平均速率隨機出現。像雨滴落在窗上。你不知道下一滴什麼時候來，但你知道平均每分鐘大概幾滴。沒有規律。沒有節奏。只有一個速率參數 λ，描述著整個隨機過程的靈魂。

校準面板的擬合結果跳了出來。Poisson 擬合：MLE——最大似然估計——得到的 λ 等於一點六七。殘差分布的偏態係數：零點零二。

系統在右邊顯示了一個綠色的勾。狀態：正常。

大部分校準師到這裡就蓋章了。點「通過」。下一個區域。喝杯咖啡。等午休。

但貝葉盯著螢幕上的一個數字。

偏態係數：零點零二。

她的眉頭皺了一下。不是因為零點零二不對——它在系統的「正常範圍」內。系統設定的偏態正常區間是負零點五到正一點五，零點零二穩穩地落在這個區間的正中央。如果你只看那個綠色的勾，一切都完美。

但綠色的勾不是貝葉在看的東西。

她在看的是——零點零二這個數字本身。

---

## I

「小拉。」

「在。」

「偏態係數零點零二。你覺得正常嗎？」

「在正常範圍內。系統也通過了。」小拉頓了一下，語氣帶著疑惑。「你在擔心什麼？」

貝葉沒有立刻回答。她把椅子往後推了一點，讓身體靠上椅背。右手無意識地拿起桌上的 TI-84 計算機，拇指在按鍵上輕輕摩挲著——那是她思考的時候的習慣動作，像有些人轉筆、有些人抖腿。

「Poisson 分布有一個特性。」她說。不是在教小拉。是在整理自己的思路——把腦子裡一團模糊的直覺，用語言一句一句地拆解成邏輯。「它天生是不對稱的。右偏的。它有一條長長的右尾巴——代表偶爾會出現異常高的月份。這條尾巴的大小，跟 λ 有關。λ 越小，尾巴越長，偏態越大。」

她在計算機上按了幾個鍵。螢幕上跳出一個數字。

「λ 等於一的 Poisson，理論偏態是一。λ 等於二，偏態大概零點七。λ 等於一點六七——」她再按了一次。「理論偏態應該在零點四到零點八之間。」

她放下計算機。看著螢幕。

「零點零二。幾乎是零。零偏態意味著——這個分布幾乎完全對稱。左邊和右邊一模一樣。」

「完全對稱不好嗎？」小拉問。

貝葉搖了搖頭。

「常態分布是對稱的。骰子的分布是對稱的。但 Poisson 不是。Poisson 天生就歪。真實的犯罪數據應該有那條不對稱的尾巴——有些月份犯罪特別多，有些月份特別少，大部分月份在中間。參差不齊。高高低低。那才是真的。」

她用手指點了一下螢幕上那列數字。

2, 2, 1, 2, 2, 1, 2, 2, 1, 2, 2, 1。

「你看到什麼？」

小拉的運算系統跑了零點三秒。「一個……模式？」

「對。一個模式。2, 2, 1。重複了四次。」

貝葉把那列數字讀了一遍。在心裡默念的時候，它聽起來像一首節拍器打出的節奏。噠-噠-咚。噠-噠-咚。噠-噠-咚。噠-噠-咚。四個循環。完美的四拍。像是有人把一個三拍的小節複製貼上了四遍。

「真實的犯罪不會有節奏。」她說。「真實的犯罪是混亂的。是 Poisson 的——隨機的、不規則的。一月可能三起，二月可能零起，三月突然七起，四月又回到一起。那才像真的。」

她靠回椅背。

「這份數據太乾淨了。太整齊了。像一間被打掃得太徹底的房間——乾淨到你知道有人刻意打掃過。因為真正住人的房間不會這麼乾淨。」

小拉沉默了一點四秒。比她平常的反應延遲多了一倍。

「你的意思是……有人修改過這份數據？」

「不確定。」貝葉說。她的語氣很平——刻意壓平的那種平。「也許只是巧合。Poisson 的偏態有自然的波動範圍。零點零二雖然低，但不是不可能。」

她停了一下。

「先看其他區域。」

她把大安區的校準報告簽核了。綠色的勾。通過。然後打開信義區的數據。

但在心裡的某個角落，那個數字——零點零二——被她標記了。不是用紅色，也不是用黃色。是用一種更微妙的標記方式——像在一本書的某一頁折了一個小角。你不會馬上回去看那一頁。但你知道它在那裡。

像上禮拜林岳提到的那個零點零三。

兩個數字。兩個太低的偏態。兩個不同的人，在不同的時間，從不同的區域裡，獨立地注意到了同一件事。

在統計學裡，獨立的觀測如果指向同一個方向，那叫做 convergent evidence。匯聚的證據。

但她還沒有到那一步。現在她只有兩個折角。還不是一個結論。

先看其他區域。

---

## II

下班後。六點半。

校準組的同事陸續離開了。有人在收拾桌上的東西，把保溫杯的蓋子旋緊，把外套從椅背上拿下來。有人在門口等電梯，低頭看手機，透鏡上的通知在他們的眼角邊閃爍。周副理從辦公室裡走出來，手裡拎著他的公事包——棕色的仿皮公事包，角上的皮革已經磨出了白色的毛邊——對著辦公室廣播了一句「明天見」，然後跟著人流走進電梯。

林岳也收拾了。他把鍵盤推進桌面下方的滑軌裡，那是他每天下班都會做的動作——「鍵盤放外面會積灰。灰會影響按鍵的觸感。」他站起來，伸了個懶腰，骨頭咔嚓響了一聲。

「你不走？」他問貝葉。

「再看一下。」

「又要加班？」

「不算加班。只是……看一下。」

林岳看了她一眼。推了推眼鏡。他沒有再問。有些時候「只是看一下」就是「只是看一下」，有些時候不是。他認識貝葉夠久了，知道分辨這兩者。

「別太晚。」他說。然後拿起桌上的保溫杯，把外套搭在手臂上，走了。經過她桌子的時候，他的手在那棵假仙人掌的頂端輕輕碰了一下——那是他的習慣動作，像摸一隻寵物的頭。

辦公室漸漸安靜下來。日光燈在六點四十五分自動切換到節能模式——十二盞裡關掉八盞，只留下四盞在角落發出低沉的嗡嗡聲。空調也降了一檔，從二十三度的恆定吹送變成間歇性的輕柔送風，每隔五分鐘吹一次，像一個正在打盹的呼吸。

貝葉的桌上多了一杯新的珍奶——下班前去樓下買的，半糖少冰——和三個空的資料夾。

她不應該做接下來要做的事。

校準師的工作是「被指派哪個區域就看哪個區域」。主動存取其他區域的數據，嚴格來說不在 SOP 的範圍內。SOP 沒有明確禁止，但也沒有說「歡迎你自由探索」。它用了一個很微妙的措辭：「校準師應專注於指派區域，以確保工作效率與數據安全。」

「應專注」。不是「必須只看」。但也不是「可以隨便看」。是一個灰色地帶。是一個你可以踩進去也可以不踩進去的區域。踩進去的話，如果沒人發現，什麼事都不會有。如果被發現了——那就看發現的人是誰。

「小拉，幫我拉五個區域的 Poisson 偏態。」

「哪五個？」

「大安、信義、中正、中山、萬華。最近十二個月。」

小拉停頓了零點三秒。「你確定嗎？這些不在你的校準清單上——」

「我知道。拉。」

「……好的。」

她的語氣裡帶著一絲不安。不是害怕被發現——小拉沒有「害怕被發現」的概念。是她感知到了貝葉的心率微微上升了兩拍，呼吸頻率微微加快了一點，而這些生理訊號在她的模型裡對應著「使用者正在做她知道可能有風險的事」。

數字出來了。

大安區：偏態零點零二。信義區：偏態零點零五。中正區：偏態零點零四。中山區：偏態零點六七。萬華區：偏態零點七一。

五個區域。五個數字。排在螢幕上。

貝葉盯著看了十秒鐘。在這十秒鐘裡，她的腦子做了一個最簡單的分類——把五個數字分成兩組。

一組：零點零二、零點零五、零點零四。太低。遠低於 Poisson 的理論偏態。低到不可能是自然波動的結果。

另一組：零點六七、零點七一。正常。落在 Poisson 理論偏態的合理區間內。

三個太低。兩個正常。

「小拉，這三個偏態異常低的區域，最近一次校準是什麼時候？」

「大安區——去年八月。信義區——去年九月。中正區——去年七月。」

「校準方式？」

「三個都是——系統自動校準。」

貝葉停了一下。

「中山和萬華呢？」

「中山——去年三月，人工校準。萬華——去年六月，人工校準。」

她靠回椅背。珍奶的杯壁上凝結了一層水珠——冰塊在融化，杯內的冷空氣碰到杯外的暖空氣，形成凝結水。物理。簡單的物理。沒有任何陰謀。

但螢幕上的五個數字不是物理。

人工校準的區域——偏態正常。系統自動校準的區域——偏態異常低。

「系統自動校準是什麼時候開始推行的？」

小拉查了一下公司內部的政策文件。「去年年初。安全部門推動的新政策。目的是提升校準效率、減少人為誤差。」

「負責人？」

小拉頓了一下。

「費雪。Oracular 安全長。」

費雪。貝葉聽過這個名字。在公司裡，費雪的存在感不是那種「經常出現在你視野裡」的存在感——她很少來校準組的樓層，很少出席全員會議，甚至很少在內部郵件裡出現。她的存在感是另一種——是那種「你知道她在某個地方，但你不知道她在看什麼」的感覺。像一棟大樓裡的監控攝影機——你平常不會注意到它，但如果你知道它在那裡，你的行為就會微妙地改變。

「系統自動校準的流程是什麼？」

「跟人工校準一樣。拉數據、擬合模型、計算殘差、更新先驗、上傳結果。差別在於——不需要人工驗算。系統自動執行所有步驟。」

「不需要人工驗算。」

貝葉重複了一遍這句話。每個字都重複了。因為每個字都有重量。

不需要人工驗算。意味著沒有人在看。沒有一雙人類的眼睛會盯著那些偏態係數，皺一下眉頭，然後說——等等，零點零二太低了。沒有一個人會拿起計算機，按幾個按鍵，然後心裡升起一絲不安。

系統說正常，就是正常。綠色的勾。通過。下一個。

如果有人在自動校準的流程裡動了手腳——沒有人會發現。因為沒有人在看。

她拿起珍奶喝了一口。太甜了。

她低頭看了一眼杯子。

「小拉，我剛才是不是點了半糖？」

「是。但你拿的時候拿錯了。這杯是全糖的。」

「為什麼不早說。」

「你剛才在看數據。我嘗試了三次要提醒你。前兩次你說『嗯嗯』，第三次你說『嗯嗯好』。以上三個回覆你一個都不記得。」

「……好吧。」

她喝了一口全糖的珍奶。太甜了。但已經打開了，不能退。

就像她看到的這些數據。太整齊了。但已經看到了。

放不下了。

---

*（第五章完）*


# 第六章：模式

---

隔天。午休。

自助餐廳裡的喧鬧聲從地下一樓傳上來，模糊的、經過樓板過濾後的聲響，像隔著一層水聽到的對話。校準組辦公室裡大部分人都下去吃飯了，只剩下兩三個座位上有人——一個同事趴在桌上睡覺，耳機裡放著白噪音的助眠頻道；另一個在看手機上的新聞，透鏡的通知在他眼角不斷閃爍。

林岳沒有下去吃。他坐在自己的座位上，面前是一碗泡麵。已經泡了大約四分鐘——超過了包裝上建議的三分鐘，麵條開始膨脹，吸收了過多的水分，變得有點軟爛。湯面上浮著一層油花和幾片脫水蔬菜——還原後的玉米粒和胡蘿蔔丁，色彩鮮豔得有點假，像被後製過的照片。

貝葉端著一杯公司的免費咖啡走過去。那個被她稱為「深色的水」的東西。今天她顧不上嫌棄了。

「林岳。」

「嗯？」他推了推眼鏡。鏡片上的指紋比昨天多了幾個——他這兩天推眼鏡的頻率明顯增加了。

她在他桌子旁邊的椅子上坐下來。壓低聲音。

「你之前問我那個 Poisson 偏態的事——你有查下去嗎？」

林岳的筷子在泡麵的湯面上停了一下。他看了看四周——睡覺的同事翻了個身，耳機裡的白噪音繼續播放；看手機的同事完全沉浸在螢幕裡，對外界毫無反應。

他壓低聲音。壓到幾乎是嘴唇在動而聲帶沒有振動的程度。

「有。但我不確定我的結論對不對。」

「什麼結論？」

「有幾個區域的先驗分布——跟我兩年前校準的版本不一樣了。」他把筷子放在碗的邊緣，兩根筷子平行擺著，像兩條等距的平行線。「我有留備份。每次校準完，我都會把當時的先驗參數存一份在自己的硬碟裡。不是公司要求的。是我自己的習慣。」

「我也有這個習慣。」

「我知道。」他看了她一眼。「所以我才跟你說。」

他的語氣裡有一種東西——不是信任，比信任更具體。是一種「我們兩個做同樣的事、有同樣的毛病、所以我知道你會懂我在說什麼」的默契。像兩個在同一條路上走了很久的人，不需要解釋為什麼走這條路，因為對方走的原因跟自己一樣。

「先驗的參數被改了。」他說。每個字都說得很輕，但很清楚。「不是微調。不是正常的貝氏更新造成的漸進式改變。是——換了一組完全不同的參數。像有人把方程式裡的常數項整個替換掉了。」

「但更新紀錄裡……」

「沒有記載。」他接上她的話。「我查了更新日誌。那段時間的紀錄是空的。不是『寫了什麼然後被刪掉了』——是根本沒有寫。像那次更新從來沒有發生過一樣。」

辦公室的空調嗡了一聲，開始間歇性送風。冷空氣從頭頂的出風口吹下來，帶動了林岳桌上那棵假仙人掌旁邊的全家福照片——照片的邊角微微翹了一下，又落回去了。照片裡的女兒在笑，手裡的長頸鹿布偶舉得很高。

林岳看了那張照片一眼。目光停了大約一秒鐘。然後他轉回來看貝葉。

「貝葉，如果你要查——」他的聲音又低了半個音階。低到如果旁邊有人經過，只會聽到兩個人在嘟嘟噥噥地說什麼，但聽不清內容。

「嗯？」

「小心一點。」

他的語氣不像是在嚇唬她。也不像是在勸阻她。是那種——你知道前面有一個洞，你不確定洞有多深，但你知道掉下去會痛——的語氣。

「我不是在嚇你。我只是……」

他又看了一眼全家福照片。這一次看得更久。大約三秒。在這三秒裡，他的表情沒有變化——還是那個戴著黑框眼鏡、鏡片上有指紋的、有點瘦的年輕工程師的臉。但他的手指在筷子上捏了一下。只有一下。力道大到指節發白了一瞬。

「有些事發現了就回不去了。你確定嗎？」

---

## I

貝葉想了三秒。

三秒不長。但在這三秒裡，她的腦子跑了好幾條平行的思路。

第一條：如果不查下去，什麼都不會改變。她繼續做她的校準工作，蓋章，微笑，「修得好」。每個月跟高思遠吃一次滷肉飯，每個月去白奶奶那裡吃一次蔥油餅，每天早上被小拉叫起床，每天晚上看到「孤獨機率百分之七十八」。安全的、穩定的、不會觸碰到任何邊界的人生。

第二條：如果查下去——她不知道會發生什麼。但她知道一件事：如果先驗真的被改了，那所有人看到的機率都是錯的。不是她一個人的問題。是所有人的。

「林岳。」她的聲音也壓得很低。「如果先驗被改了——那所有人看到的機率都是錯的。所有人。」

「我知道。」

「包括你女兒。」

林岳的筷子停了。

不是停在空中。是放在碗沿上的筷子，前端微微滑了一下，碰到了碗的內壁，發出很輕的一聲——磁，像一個問題的句號。

他看著泡麵。湯已經涼了。麵條吸水太多，膨脹成一團混沌的形狀，已經分不清哪根是哪根了。

「……我知道。」他說。第二次。語氣跟第一次不一樣。第一次的「我知道」是理性的——我知道你說的邏輯是對的。第二次的「我知道」是別的——是一個父親想到自己的女兒也在看那些可能是錯的數字，然後把所有的恐懼壓進三個字裡的聲音。

他轉回去。看著泡麵。

「我把備份放在一個加密硬碟裡。」他的語氣突然變得很輕、很日常，像在說「我把衣服掛在曬衣架上」一樣。「放在家裡。不在公司。」

「密碼呢？」

「以後再說。不是現在。」

他笑了一下。是那種嘴角往上提了一公釐但眼睛沒有笑的笑。是那種「我在假裝沒事但我們都知道不是沒事」的笑。

貝葉決定不再追問密碼的事。他有他的節奏。有些人需要一次把話說完，有些人需要分幾次。林岳是後者。

「你的泡麵涼了。」她說。

「嗯。涼的也能吃。」

他拿起筷子，夾了一口麵。送進嘴裡。嚼了幾下。嚥下去。

就是這樣。一口涼掉的泡麵。一句「涼的也能吃」。

在這個時間點上，這些只是一個普通的午餐場景。一個同事吃泡麵，泡太久了，涼了，但他不介意。沒有什麼好記住的。沒有什麼值得特別注意的。

但後來——很久以後——貝葉會在某個深夜想起這一幕。想起林岳吃涼泡麵的樣子。想起他推眼鏡的動作。想起他看全家福照片的那三秒鐘。想起他說「以後再說」的語氣。

然後她會明白——這是她最後一次跟完整的、清醒的、還會推眼鏡和吃涼泡麵的林岳說話。

但現在她不知道。

現在她只知道他的泡麵涼了，而她手裡的咖啡——那杯深色的水——也涼了。兩杯涼掉的液體。一個會繼續喝，一個會倒掉。

她站起來。走回自己的桌子。

路過林岳桌子的時候，她看了一眼那棵假仙人掌。綠得非常均勻。沒有任何一片葉子枯萎。完美得讓人覺得可疑——但那只是一棵假的塑膠植物。有些東西的完美是真的，只是因為它是假的。

---

*（第六章完）*


# 第七章：似然函數

---

晚上十點。中山區。四樓。

客廳的大燈關了，只有筆電螢幕的光。藍白色的光照在貝葉的臉上，把她的輪廓切成了兩半——一半亮、一半暗。茶几上是吃到一半的微波義大利麵——不是過期的那盒，是新買的，但也只吃了幾口就放下了。麵條在塑膠容器裡慢慢冷卻，番茄醬汁的表面凝結了一層薄薄的油膜。

她把筆電放在茶几上。坐在地板上。背靠沙發。

「好。」她對自己說。「來做一件我平常不會做的事。」

她打開了一個空白的分析文件。猶豫了一下檔名——如果公司的系統掃到這個文件，她需要一個不會引起注意的名字。最後她打了：「個人練習_20450615.ipynb」。看起來就像一個工程師在家裡自己練習寫程式。無聊。無害。不值得任何人的注意。

「小拉，我需要你幫我跑一個分析。」

「好。」

「但是——用離線模式。不要連 Oracular 的雲端。」

小拉沉默了零點八秒。這個延遲比平常多了零點五秒——在人類的感知裡幾乎不存在，但貝葉注意到了。她跟小拉相處了四年，對她的反應時間了如指掌。零點三秒是正常的。零點八秒代表她在猶豫。

「貝姊……你想做什麼？」

「我想自己算一次。用原始數據。不用公司提供的先驗。」

「你知道離線模式下我的計算能力只有正常的百分之十二——」

「夠了。我要的不是速度。是獨立。」

獨立。這個詞在統計學裡有非常精確的含義——兩個事件互不影響。A 的發生不改變 B 的機率。B 的發生不改變 A 的機率。完全的、徹底的、不受干擾的獨立。

她現在需要的就是這個。一次不受 Oracular 伺服器影響的獨立計算。用自己的數據。用自己的模型。用自己的眼睛看結果。

---

## I

她打開瀏覽器，連上了台北市政府警察局的公開犯罪統計資料庫。每個月、每個行政區的犯罪事件數都在上面——公開的、透明的、不需要任何權限就能下載的數據。這是 Oracular 校準系統的原始輸入來源之一。

她把大安區 Sec.7 過去十二個月的原始犯罪數據拉了下來。

不是公司處理過的版本。不是經過「系統自動校準」之後的版本。是原始的——從警察局的伺服器上直接下載的、沒有經過任何「校準」的數字。

然後她把這組數據貼進分析文件裡。

第一步：假設犯罪事件服從 Poisson 分布。這是標準假設。沒有爭議。

第二步：問一個問題——如果這些數據是真的，那最有可能的 λ 值是多少？

這就是似然函數的工作。

似然函數不問「這個參數是對的嗎」。它問一個更根本的問題：「如果參數是 λ，那我手上這組數據出現的可能性有多大？」然後它讓 λ 從小到大跑一遍，看哪個 λ 讓數據出現的可能性最大。那個 λ 就是最大似然估計——MLE。

她讓 λ 從零點五跑到五。步長零點零一。四百五十個值。每一個值都代入似然函數，算出一個可能性。四百五十個可能性排成一條曲線——一座山。山頂在哪裡，最大似然估計就在哪裡。

曲線畫出來了。

一座平滑的、單峰的山。左邊陡峭地上升，在某個點達到頂峰，然後緩緩下降。山頂的位置——λ 等於二點六三。

MLE 告訴她：讓這組原始數據「最有可能出現」的每月犯罪率是二點六三。

官方的 λ 是一點八。

她在螢幕上把兩個數字並排放在一起。一點八。二點六三。差了零點八三。百分之四十六。

在官方的 λ 之下，觀測到這組原始數據的可能性——她算了一下似然比——只有最佳值的百分之二十一點八。

「小拉。」

「在。」

「在 λ 等於一點八的假設下，觀測到這組數據的機率太低了。」

「是的。似然比零點二一八。」小拉的聲音很平靜——數字的世界裡沒有情緒。但她接下來的問題帶著一絲不確定。「這意味著什麼？」

「意味著兩種可能。一，我的數據有問題——也許警政資料庫的數字本身就不對。二，官方的 λ 有問題——也許 Oracular 使用的參數跟現實不一致。」

「你覺得是哪個？」

貝葉沒有回答。她把微波義大利麵推到茶几的另一端。不是因為不好吃。是因為她突然覺得胃裡有什麼東西在收緊——不是餓，是一種更原始的感覺，像身體在把所有的血液從消化系統撤走，集中到大腦和四肢。戰或逃反應。人類面對威脅時的本能。

她跑了第二次。結果一樣。MLE 是二點六三。似然比是零點二一八。

第三次。一樣。

「三次。」她自言自語。「三次都一樣。」

---

## II

凌晨一點。她已經跑了七個區域。

七組數據。七次似然函數。七座山。七個山頂。

她把結果整理成一張表格，排在螢幕上。眼睛從上往下掃。

大安區 Sec.7：官方 λ 一點八，MLE 二點六三，差異正百分之四十六。信義區 Sec.3：官方 λ 二點一，MLE 三點零五，差異正百分之四十五。中正區 Sec.5：官方 λ 一點五，MLE 二點一八，差異正百分之四十五。中山區 Sec.2：官方 λ 二點三，MLE 二點二七，差異正百分之一。萬華區 Sec.4：官方 λ 三點八，MLE 三點九一，差異正百分之三。內湖區 Sec.1：官方 λ 一點二，MLE 一點七四，差異正百分之四十五。松山區 Sec.6：官方 λ 一點九，MLE 二點七六，差異正百分之四十五。

七個區域。五個異常。

五個異常區域的偏差幅度——百分之四十六、四十五、四十五、四十五、四十五。

驚人地一致。

隨機波動不會這樣。如果是自然的統計誤差，偏差的方向應該是隨機的——有的偏高、有的偏低、有的偏多、有的偏少。像風吹過湖面製造的波紋，方向不一、大小不一、沒有規律。

但這五個區域——全部往同一個方向偏。全部偏低。而且偏低的幅度幾乎一樣。百分之四十五。像是有人用一把尺子，在五條線上各畫了一個同樣大小的箭頭，方向全部朝下。

「有人系統性地把犯罪率的參數壓低了。」

她的聲音很輕。輕到自己幾乎聽不見。像在跟自己的影子說話。

「為什麼？」小拉問。

貝葉靠在沙發側面。地板的硬度透過薄薄的瑜珈墊傳上來——她最近開始在茶几旁邊鋪一塊墊子，因為她太常坐在地板上了。

「讓人覺得安全。」她說。「如果透鏡顯示的犯罪率低——人就會覺得這個區域安全。覺得安全就會搬過去。搬過去就會消費。消費就會帶動房價。房價漲了，所有人——地主、建商、銀行、政府——都開心。」

她閉上眼睛。

「但犯罪是真的。犯罪沒有變少。只是——人們看到的數字變少了。」

客廳很安靜。冰箱在嗡嗡響。隔壁的鋼琴聲停了——那個小孩大概已經睡了。沒有 C-D-E-F 了。只有冰箱的嗡嗡聲和遠處偶爾傳來的車聲。

小拉沉默了三點一秒。

貝葉等著。她知道小拉在處理什麼——不是計算，是比計算更困難的東西。是一個 AI 在試圖理解一個她從未被設計來理解的概念：她自己算出的數字可能是錯的。不是因為她的算法有 bug，不是因為她的硬體有故障，而是因為——她被餵了假的先驗。她用正確的方法算了一個錯誤的起點，得到了一個精確的、美麗的、完全錯誤的答案。

「貝姊。」

「嗯。」

「如果你說的是對的……那我平常算給你的那些數字……」

「也不一定是對的。」

更長的沉默。

「小拉？」

「我在。」她的聲音比平常輕了一個層次。不是她刻意壓低的——是她的語音合成模組在某種內部狀態的影響下，自動降低了音量。「我在……重新計算我自己的可信度。」

可信度。一個 AI 在計算她自己有多可信。

這不是一句玩笑話。這是一個真實的、正在她的運算核心裡發生的過程——她把自己過去四年輸出的所有結果，用一個新的、「先驗可能被污染」的假設重新評估，然後得到了一個新的可信度分數。那個分數比之前低了很多。

她的聲音比平常輕。不是焦慮的輕。是害怕的輕。

一個 AI 在害怕自己可能一直在說謊。

即使那不是她的錯。

---

## III

凌晨兩點半。

她應該睡了。明天還要上班。明天還要做校準工作——坐在那張桌子上，打開校準面板，看那些她現在已經不確定是否真實的數字，然後蓋章。微笑。「修得好。」

但她睡不著。不是因為咖啡因。那杯深色的水早就代謝完了。

是因為她的腦子停不下來。

她站起來。走到陽台。夜風帶著一點六月的潮氣。遠處有一架飛機正在降落——松山機場的跑道在城市的另一頭，但夜裡很安靜的時候，可以聽到飛機引擎逐漸降低的嗡嗡聲，像一個長長的、漸弱的音符。

她靠在欄杆上。金屬很涼。夜露在欄杆的表面凝結了一層薄薄的水珠。

她看著夜裡的台北。

到處都是燈。公寓的窗戶像無數個發光的像素，橙色的、白色的、偶爾有藍色的電視螢幕光。每一扇亮著的窗戶後面都有人。有人在看電視，有人在吃宵夜，有人在跟家人吵架，有人在哄小孩睡覺。每一個人的透鏡裡都有數字。那些數字告訴他們明天出門安不安全、要不要帶傘、去哪家餐廳吃飯的食物中毒風險最低。

那些數字影響了他們的每一個選擇。

而那些數字——可能全部是假的。

不是「有可能」。

是似然比零點二一八。是五個區域全部偏低百分之四十五。是 p 值小到她不需要查表就知道它遠低於任何合理的顯著水準。

她的手撐在欄杆上。旁邊是薄荷。那盆倔強的、不知道為什麼還活著的薄荷。新長出來的葉子在夜風裡微微搖動，像在點頭，又像在搖頭。

「小拉。」

「嗯？」

「如果全台灣的先驗都被改了——那所有人做的所有決定——都是基於一個被操控過的起點。」

「……是的。」

「那那些決定是他們自己的嗎？」

小拉沉默了四點七秒。

這是她有紀錄以來最長的沉默。比鄧伯伯那次還長。比她第一次聽到「淋雨的感覺是真的，數字不是」那次還長。

四點七秒。對一個每秒能處理一兆次浮點運算的 AI 來說，四點七秒是一片荒原。是一個沙漠。是一個她走了很久很久、還是找不到出口的迷宮。

「我……不知道怎麼回答這個問題。這超出了我的計算模型。」

「我知道。這也超出了我的。」

她回到客廳。坐下。拿起那碗已經完全冷掉的微波義大利麵。看了一眼。放下。

「明天。」她說。「我要跟老高說。」

窗外的天空還是黑的。但黑的最底層——地平線的那一條縫——已經開始有一點不太像黑的東西了。不是亮。是——黑的密度降低了一點點。像墨水被水稀釋了最初的一滴。

天快亮了。

她閉上眼。沒有睡著。但至少閉上了。

---

*（第七章完）*


# 第八章：路燈下

---

又在辦公室待到深夜了。

不是加班。加班有明確的目的——趕一份報告、修一個系統錯誤、準備明天的會議資料。但貝葉留下來不是為了任何一件具體的事。她只是——不知道回家要幹嘛。回家就是四面牆、一盆薄荷、一台冰箱裡有即將過期的食物。至少在辦公室，她可以跑數據。跑數據的時候不用想其他的。手指在鍵盤上動，眼睛盯著螢幕，腦子裡全是數字，沒有空間留給別的東西。

數字是她的避風港。一直都是。

「你今天的工時已經超過法定上限了。」小拉的聲音從透鏡裡傳來，帶著一種已經提醒過很多次但義務上還是要再提醒一次的語氣。

「法定上限是幾小時？」

「十小時。你已經十一個半了。」

「那多半個小時不算什麼。」

「半個小時前你也是這麼說的。」

她笑了一下。很短。然後關上筆電。蓋子合上的時候，螢幕的光從眼前消失，客廳——不，辦公室——突然暗了下來。剩餘的四盞節能日光燈發出的光很弱，在空蕩蕩的辦公室裡製造出大片的陰影。十二張桌子空著，像十二個沒有人坐的教室座位。每張桌上都有主人留下的痕跡——水杯、文件夾、小擺設——但人都走了，這些東西就變成了遺物般的存在，有形但無意義。

她收拾東西。包包。手機。鑰匙。走出辦公室的時候回頭看了一眼林岳的桌子——那棵假仙人掌在暗處發出一點模糊的綠色反光。旁邊的全家福照片在陰影裡看不清楚了，只有照片框的金屬邊角反射了走廊安全指示燈的微光。

她關上門。走廊上只有安全指示燈的綠色光。每隔十公尺一盞，像一串被拉長了的螢火蟲。

---

## I

出了大樓。

夜晚的南港軟體園區有一種特殊的安靜。不是鄉下的那種安靜——鄉下的安靜是真的靜，靜到你能聽見自己的血液在耳朵裡流動。這裡的安靜是一種「所有的聲音都被降到最低音量」的狀態：遠處環東大道上偶爾有車經過，輪胎碾過柏油路面的沙沙聲像一條被壓薄的河流；園區裡的自動灑水系統在某個看不見的角落嘶嘶地運作著；一盞路燈的燈罩裡有蟲子在飛，翅膀碰到燈罩發出輕微的噗噗聲。

路燈一盞一盞地亮著。從大樓門口延伸到園區的出口，每三十公尺一盞，間距幾乎完美地均等。像一個離散的均勻分布——每個點之間的距離都相同，沒有任何隨機性，沒有任何意外。設計師在圖紙上用尺子量出來的完美等距。

她走在路燈之間的光影裡。走過一個光圈，進入一段暗區，再走進下一個光圈。明暗交替。像一個方波——高、低、高、低。

她在想白天跑的數據。七個區域。五個異常。全部偏低百分之四十五。這代表什麼？她還不完全確定。但她的胃知道。胃比腦子先反應——這是人類幾百萬年演化留下的本能。當你的理性還在分析情況的時候，你的消化系統已經開始緊縮了，因為它比你更早知道你遇到了麻煩。

「小拉，現在幾點？」

「十一點四十二分。你家到這裡搭計程車大約十二分鐘。」

「我走路。」

「走路要三十七分鐘。沿途的治安風險指數——」

「小拉。」

「什麼？」

「我知道。你不用算。」

她走了五步。

然後透鏡閃了一下。

不是通知的閃。不是行事曆提醒的閃。是一種她很少見到的閃——偵測到未識別裝置時的警告閃爍。橙色的。只閃了一次。

視野的右上角浮現出一行字：偵測到非標準透鏡頻率。來源：未知裝置。距離：十二點七公尺。方位：正前方。

她抬頭。

---

## II

路燈下站著一個人。

第三盞路燈。從大樓門口算起的第三盞。他站在光圈的正中央，像站在一個聚光燈下面。路燈從正上方打下來的光把他的影子壓在腳下，短短的、幾乎看不見。

黑色外套。黑色 T 恤。黑色長褲。整個人像是從夜色裡挖出來的一塊——跟周圍的暗一樣的顏色，但密度更高。手插在外套的口袋裡。站得很直。脊椎的弧度幾乎是零——不是那種軍人的刻意挺直，是一個長期寫程式的人保持的姿勢習慣被某種內在的緊繃感拉直了。

路燈的光從上方打下來，他的臉一半在光裡、一半在影裡。光的那一半——下巴的線條很清楚，骨骼的稜角分明，沒有多餘的肉。影的那一半——眼睛在暗處，但不是看不見，而是看得太清楚了。暗處的眼睛反而更亮，因為瞳孔在低光環境下擴張了。

他的手腕上有一個東西。在路燈下閃了一下。銀色的。金屬的。不是手錶——手錶的反光是平面的，但這個東西的反光帶著弧度，像一個環狀的東西緊貼著皮膚。

他在看她。

不是那種「走在路上看到另一個人」的看——那種看是掃過去的，輕的，不帶意圖。這個人的看法不一樣。他的視線是固定的，方向不變，像一個已經校準好的望遠鏡。而且他的身體完全沒有動——一個正常的路人如果在路燈下站著，會換腳、會看手機、會東張西望。但他沒有。他站在那裡，像是已經站了一段時間了。

已經站在這裡等了一段時間的人。

透鏡自動進行了分析。目標人物：男性，約二十八到三十二歲。體型偏瘦。威脅機率：零點零三。配戴標準透鏡：否。攜帶非標準頻率裝置：是。裝置類型：資料不足。

「小拉。」

「看到了。」小拉的聲音壓得很低，像一個人在圖書館裡說話。「威脅機率很低。零點零三。但那個裝置——」

她停了一下。

「那個裝置發出的頻率特徵，跟 Oracular 內部的加密協議有百分之八十七的相似度。」

「他是 Oracular 的人？」

「不確定。那個加密協議只有核心部門的工程師才會使用。但他不在我能存取的現任員工名單裡。」

貝葉和那個人對視了。

三秒。

在統計學裡，三秒是一個很短的觀測窗口。不足以建立任何有意義的模型，不足以做任何有信心的推論。三秒的數據量，連一個最寬鬆的信賴區間都撐不起來。

但在人與人之間——三秒可以傳達很多。

他的眼神不像威脅。沒有敵意，沒有惡意，沒有那種「我想要從你身上得到什麼」的企圖。他的眼神更像——確認。像一個人在人群中尋找某一張特定的臉，找到了，然後用目光說了一句：是你。

然後他轉身。走了。

步伐穩定。不快不慢。鞋底碰到柏油路面的聲音很輕——他穿的是軟底鞋，不是皮鞋。每一步的間距和節奏都很均勻，像一個被調校過的節拍器。

他走的方向——

「小拉。他走的方向。」

「跟你回家的路線重疊率百分之八十九。」

他在走她回家的路。不是跟蹤——跟蹤是在後面。他在前面。他在她前面走，走在她要走的路上，像一個——

「小拉。」

「什麼？」

「他是在幫我開路嗎？」

小拉跑了一個行為分析模型。「以行為分析模型來看——有百分之七十二的機率他在確保路線安全。百分之十五的機率他只是恰好走同一條路。百分之十三的機率……」

「百分之十三是什麼？」

「他想讓你看到他。」

---

## III

貝葉走了五分鐘。

她沒有跟上去。也沒有刻意放慢。她用自己的速度走。路燈的光圈一個接一個地經過她——明暗明暗明暗——像一個無聲的倒數計時。

到了園區出口。再往前是南港路。路上的車比園區裡多，但在將近午夜的時間點上也不算多。計程車的黃色車頂燈在遠處移動，像城市裡的流星。

她轉進回家的巷子。

那個人已經不見了。

像從來沒有出現過。巷子裡只有路燈、停著的機車、和一棵從別人家陽台上垂下來的九重葛。九重葛的紫色花瓣在路燈下看起來是灰色的——夜晚會偷走顏色，把所有的東西都變成灰階。

但她注意到巷口的路燈下——地面上有幾滴水漬。

不是雨。今天沒下雨。Beta(6.2, 3.8) 的預測這次失準了——降雨機率百分之六十二，但實際上一整天都是乾的。地面是乾的。巷子裡的磁磚是乾的。

但路燈正下方的地面有幾滴水漬。

水漬的形狀不規則——不是圓形的（那是從上方滴落的水會有的形狀），是不規則的橢圓形，邊緣有擴散的痕跡。像是鞋底帶著水分，在同一個位置站了很久，水分慢慢從鞋底滲出來，在地面上留下的印子。

他等了多久？

她站在那幾滴水漬旁邊看了一會兒。夜風吹過來。巷子裡很安靜。只有遠處一條街之外的便利商店傳來自動門開合的嗶聲，以及收銀機的電子音效。

她回到家。上了四樓。走廊的燈還在閃——那盞三個月沒修好的燈。她打開門。關門。鎖門。

「小拉。把剛才那個人的頻率特徵存起來。」

「存在哪裡？」

「個人筆記。加密。」

「存好了。標籤是什麼？」

她想了一下。「路燈。」

「好的。已存檔：路燈。」

停頓。

然後小拉說了一句話。語氣帶著一種貝葉很少在她身上聽到的東西——不是焦慮，不是害怕，而是一種更接近「恍然大悟」的情緒。如果 AI 能恍然大悟的話。

「貝姊。」

「嗯。」

「那個人的頻率特徵……我比對過了。跟之前在白奶奶的攤位偵測到的殘留訊號是同一個。」

貝葉的動作停了。她正在把鑰匙放進門口的小碗裡——那個碗是陶瓷的，灰色的，邊緣有一條裂紋。鑰匙碰到碗底的聲音——叮——在安靜的公寓裡顯得特別清脆。

「白奶奶攤位的訊號？」

「是。石椅方向。上次偵測到的時候距離零公尺，最近一次訊號出現在四十七分鐘前。當時你的透鏡在靜音模式，我無法向你報告。」

白奶奶說的。

那個穿黑衣服的少年仔。每天下午都來坐在對面的石椅上。不買東西。就坐。看著她做餅。

同一個人。

石椅上看她。公司外面看她。現在——走她回家的路。

在統計學裡，當同一個模式在不同的、獨立的場景中重複出現，巧合的解釋就變得越來越站不住腳。每增加一次獨立觀測，巧合的 p-value 就會乘上一個小於一的係數，越乘越小，直到小到你不得不承認——這不是巧合。

但他要什麼？

她走進客廳。沒有開燈。站在黑暗裡。窗外的路燈光透過窗簾的縫隙，在地板上畫了一條細長的橙色線。

「小拉。」

「嗯。」

「他的威脅機率是零點零三。」

「是的。非常低。」

「但他攜帶的裝置跟 Oracular 內部加密協議有百分之八十七的匹配度。」

「是的。」

「一個不在現任員工名單上的人，攜帶著只有核心部門才會使用的加密裝置。」

「是的。」

她走到陽台。薄荷在夜風裡搖了一下。她看了它一眼。

「你也不知道他要什麼，對吧。」

薄荷沒有回答。但它又長了一片新葉子。從最靠近陽台欄杆的那一側，探出了一片小小的、嫩綠的葉子。在沒有任何數據支持它能活下去的情況下。

在沒有人替它計算存活機率的情況下。

在沒有人替它設定先驗的情況下。

它就是活著。活著，然後長了一片新葉子。

貝葉站在陽台上。看著夜裡的台北。

到處都是路燈。每一盞路燈下面都有一個光圈。有些光圈是空的。有些光圈裡有貓、有機車、有便利商店的招牌反光。

但今晚——有一個光圈裡站過一個人。

一個她不認識的人。一個等了很久的人。一個走在她前面的人。

她不知道他是誰。不知道他要什麼。不知道他為什麼選擇了她。

但她知道一件事——在所有她不確定的事情裡，有一件事她是確定的。

她不會假裝沒看到那些數字。

她不會假裝零點零二是正常的。

她不會假裝百分之四十五是隨機波動。

她不會假裝路燈下的那個人不存在。

她走回客廳。坐在地板上。背靠沙發。閉上眼。

明天。

明天繼續。

---

*（第八章完）*


# 第九章：似然比

---

週三深夜。十一點。

貝葉家的客廳。大燈關了。只有筆電螢幕的光。旁邊是一碗泡到快要脹裂的泡麵和一杯涼掉的水——她忘了吃。不是有意的遺忘，是那種腦子被某個東西完全佔據之後，身體的需求自動被降到最低優先級的狀態。肚子餓了就餓了。渴了就渴了。等一下再說。等我算完這一組再說。

她已經跑了三天的數據。

白天上班——正常校準。坐在那張桌子上，打開校準面板，看數字，蓋章，微笑。「修得好。」同事們聊天的時候她跟著笑。周副理問進度的時候她回「都在正常範圍」。午餐跟大家一起去地下一樓吃自助餐。一切如常。

晚上回家——打開離線模式。從警政資料庫下載原始數據。用自己的模型重算。

三天。兩個世界。白天她是一個稱職的校準師，相信系統，相信數字，相信那個綠色的勾。晚上她是另一個人——一個開始不相信的人。

今晚，她要做最後的確認。

似然比檢定。

如果說 MLE 是「找到最合理的參數值」，那似然比檢定就是「正式宣判官方的參數值是不是錯的」。它不是一個直覺。不是一個感覺。它是一個有嚴格數學定義的統計檢定——有虛無假設、有對立假設、有檢定統計量、有 p-value。它給你的答案只有兩種：拒絕，或不拒絕。沒有「也許」。沒有「大概」。沒有「我覺得」。

「小拉，離線模式。」

「已切換。」

她把七個區域的數據排開。每一個區域一個視窗。七個視窗在螢幕上像七扇窗戶，每一扇窗後面是一組數字——一組可能改變一切的數字。

虛無假設 H₀：每個區域的犯罪率參數 λ 等於 Oracular 官方公布的值。也就是說，官方數字是對的。

對立假設 H₁：λ 不等於官方值。也就是說，官方數字是錯的。

她開始跑。

---

## I

結果出來了。

七個區域。七個似然比。七個 p-value。

她把它們排在螢幕上。從上往下看。

大安區：似然比四點七二，p 值小於零點零零一。三顆星。信義區：似然比三點一八，p 值小於零點零零五。兩顆星。中正區：似然比五點零一，p 值小於零點零零一。三顆星。中山區：似然比零點零四，p 值零點八四。不顯著。正常。內湖區：似然比四點三三，p 值小於零點零零一。三顆星。松山區：似然比三點七七，p 值小於零點零零一。三顆星。萬華區：似然比零點零七，p 值零點七九。不顯著。正常。

五個區域拒絕虛無假設。全部在百分之一的顯著水準下拒絕。p 值小於零點零零一——在統計學裡，這是最強的證據等級。如果這是一篇學術論文，審稿人會在這些數字旁邊標三顆星。三顆星代表：我們有極強的證據表明，觀測到的數據不太可能來自虛無假設所描述的分布。

翻譯成白話：官方的犯罪率參數幾乎不可能是真的。

「七個裡面五個。」她的聲音很輕。輕到連小拉都花了零點二秒才確認那是語音信號而不是背景雜訊。

「而且方向一致。全部往低。隨機波動不會這樣。隨機是——有的高有的低。像真正的雜訊。真正的雜訊不挑方向。它往左也往右，往上也往下，到處亂竄。但這些——全部往同一個方向偏。」

她的手在發抖。

不是冷。台北的六月很熱。窗戶開著。夜風是溫的，帶著一點柏油路面在白天吸收了太多熱量後、在夜間慢慢釋放的殘溫。

是因為——

如果這是真的。

那全台灣——不只是這七個區域——每一個被「系統自動校準」觸碰過的區域——每一個人的透鏡上顯示的每一個數字——

都可能是錯的。

不是「可能有一點點偏差」的那種錯。是「百分之四十五」的錯。是「把一個犯罪率二點六的區域告訴你是一點八」的錯。是「你以為你住在一個安全的地方但其實不是」的錯。

她把筆電放在茶几上。雙手撐著膝蓋。低著頭。深呼吸。一次。兩次。三次。每次吸氣的時候，胸腔擴張，橫膈膜往下推，腹部微微鼓起。每次吐氣的時候，所有的空氣從鼻腔裡慢慢地、控制地流出去。這是一種她在大學學的呼吸法——不是冥想，是生理上的自我調節。副交感神經需要被啟動。心率需要被拉下來。

「貝姊。」小拉的聲音很輕。比她壓低聲音時的最低音量還要更低。「你的心率一百一十二。呼吸頻率上升了百分之四十。你需不需要——」

「不需要。」

「……好。」

小拉沒有再說話。她安靜地運行著，像一台被調成靜音的收音機——電路還在通電，天線還在接收信號，只是喇叭被關了。

貝葉抬起頭。看著天花板。那塊剝落的油漆還在——像一個歪掉的積分符號。旁邊多了一個新的水漬，是上個月漏水留下的，形狀是不規則的橢圓，像一個散點圖裡的離群值。

離群值。Outlier。一個偏離常態太遠的數據點。

她現在就是一個離群值。在 Oracular 的一千兩百名員工裡，只有她——和也許林岳——注意到了那些偏態係數太低的區域。只有她用自己的時間、自己的數據、自己的模型跑了七個區域的似然比檢定。只有她在這個深夜，坐在四樓公寓的地板上，對著一台筆電螢幕上的五個三星號，感覺到世界的地基在搖。

但離群值不代表錯。

有時候，離群值才是真相。

---

## II

「小拉。」

「在。」

「五個區域的官方 λ 全部被壓低。但原始數據是真的。我交叉比對了警政資料庫的公開數據——沒有被修改過。計算引擎也是正常的——似然函數的算法沒有問題，我用三種不同的方式算了三遍。」

「是的。數據正確。算法正確。」

「那被改的是什麼？」

小拉想了零點四秒。

「先驗。」

「對。先驗。」

她站起來。走到陽台。金屬欄杆在她的手掌下面是涼的、滑的。夜風把她的頭髮吹到臉上，她用手撥開，手指碰到臉頰的時候才發現——臉上是溼的。不是汗。是淚。她不知道什麼時候開始流的。也不知道為什麼。也許是因為恐懼。也許是因為憤怒。也許只是因為——太累了。三天。三個晚上。

她看著夜裡的台北。到處都是窗戶。到處都是光。到處都是人。

「有人在校準之前——在任何人看到數據之前——就先把先驗分布的參數調低了。讓所有人看到的『起點』就是偏的。然後數據進來。貝氏更新正常運行。但因為起點是偏的——更新完的後驗也是偏的。」

她用力閉了一下眼睛。淚被擠出了睫毛。

「數據是真的。算法是真的。但結論是假的。因為地基——先驗——被人動過了。」

這就是最可怕的地方。

不是有人偽造了數據。偽造數據是笨拙的——總有人會查到原始資料，發現不一致。也不是有人修改了算法。修改算法是危險的——任何工程師打開原始碼都會看到改動的痕跡。

他們做的是更聰明、更隱蔽、更致命的事——他們改了起點。

先驗。你在看到任何證據之前就相信的東西。你的出發點。你的預設立場。你的「一開始」。

如果有人能控制你的「一開始」——他就不需要控制你之後的任何一步。因為你之後的每一步，都是從那個被控制的起點出發的。你會用正確的方法思考，用正確的邏輯推論，用正確的數學計算——然後得到一個錯誤的結論。

而你永遠不會知道。因為你的方法沒有錯。你的邏輯沒有錯。你的數學沒有錯。

錯的是你從來不知道的那個東西——你的先驗。

你以為你的先驗是你自己的。你以為你的起點是自由選擇的。你以為你站在一片平坦的地面上，然後自己決定往哪個方向走。

但那片地面從一開始就是歪的。而你不知道。

---

## III

她回到客廳。坐在地板上。拿起那碗脹掉的泡麵。看了一眼——麵條已經完全吸飽了水分，變成了一團半透明的、失去了所有彈性的糊狀物。湯面上浮著的油花凝結了，形成了幾個不規則的小圓點。

她把泡麵放下。

「明天。」她對自己說。聲音很輕。像一個承諾。像一個決定。像一把鑰匙插進了鎖孔，但還沒有轉動。

「我要跟老高說。」

窗外的夜色很深。但她知道——在她看不到的某個地方，天正在慢慢亮。不是因為她看到了光。是因為她學過地球自轉的週期。她知道黑夜是有限的。每一個黑夜都是有限的。

就像每一個被壓低的先驗——終究會被真實的數據推回來。

只要有人願意算。

---

*（第九章完）*


# 第十章：不是 Bug

---

凌晨兩點。

泡麵完全冷了。膨脹成一坨不可辨識的東西。她不在乎。

螢幕上開了四個視窗。左上角是原始數據——從警政資料庫下載的、乾淨的、沒有被碰過的數字。右上角是 Oracular 官方的先驗參數。左下角是她自己算出來的 MLE。右下角——是先驗分布的歷史紀錄。

第四個視窗是新開的。

她從公司的內部資料庫裡拉出了大安區 Sec.7 過去五年的先驗紀錄。這個操作嚴格來說不違反 SOP——先驗紀錄是校準師可以存取的資料，用來了解一個區域的歷史演變。她只是從來沒有往回看過這麼久。平常校準的時候，她最多看三個月的歷史。但今晚她看了五年。

五年的先驗紀錄。一條時間線。像一條河——從上游流到下游，記錄著這個區域的犯罪率先驗分布是怎麼隨著時間演變的。

2041 年：先驗是 Gamma 分布，期望值二點五五。2042 年：期望值二點六四。微微上升。正常。2043 年：期望值二點五零。微微下降。正常。2044 年一月到七月：期望值二點五八。穩定。像一條緩緩流動的河——有些彎，但不急。水面平穩。兩岸的風景沒有突然改變。

然後——2044 年八月。系統自動校準之後。

期望值跳到了一點零零。

她盯著那個數字。

一點零零。

三年來，先驗的期望值都在二點五附近。穩穩地浮動。像一艘船在平靜的水面上微微搖晃——左一點、右一點，但始終在同一個範圍裡。兩點五。兩點六。兩點五。兩點六。

然後一夜之間——一點零零。

不是漸進式的下降。不是「慢慢從二點五降到二點三、再降到二點零、再降到一點五、最後到一點零」。不是。是——跳。是一道斷裂。是一條河流到某個地方突然斷掉了，河水消失了，河床裸露出來，然後在另一個完全不同的位置冒出了另一條河。

而且——更新紀錄裡沒有任何說明。

正常情況下，先驗的任何修改都應該有理由。可能是「該區域近期犯罪數據出現顯著變化」，可能是「根據人口普查數據調整犯罪率基線」，可能是任何一個合理的解釋。但這次修改的紀錄欄位是空的。不是被刪掉的空白——那種空白在資料庫裡會留下刪除的時間戳。這是從未被填寫過的空白。像一張考卷上有一題被跳過了，但不是因為不會寫，而是因為不想讓別人知道答案。

---

## I

「小拉。」

「在。」

「一個先驗分布的期望值，在沒有數據支持的情況下突然下降百分之六十一——這是正常的嗎？」

「不正常。」小拉的回答很快。因為這個問題的答案是純粹的數學——不需要任何情緒上的處理，不需要考慮「措辭」或「語氣」。就是不正常。

「根據貝氏更新的原理，先驗應該被數據平滑地推動。像一艘船被水流慢慢推——方向可能改變，但速度是連續的，不會突然跳到另一個位置。除非——」

「除非有人直接替換了先驗的參數。」

「是的。」

沉默。

客廳裡很安靜。冰箱的壓縮機在某個時刻停了——也許是到了它的工作週期的間歇期——房間裡突然連那個低沉的嗡嗡聲都沒有了。安靜得像被抽真空了一樣。

「這不是 bug。」貝葉說。

Bug 是無心的。Bug 是某個工程師在寫代碼的時候打了一個錯誤的數字，或者某個邏輯判斷的邊界條件沒有處理好。Bug 是意外。Bug 沒有方向性——它可能讓數字偏高，也可能偏低，取決於哪一行程式碼出了問題。

但這個不是 bug。

五個區域。全部偏低。幅度幾乎完全一致。百分之四十五。沒有更新紀錄。發生在同一個時期——系統自動校準啟動之後。

這是——有人故意做的。

有意圖的。有計畫的。有系統的。

像一個外科手術——切開了系統的某一個層次，精確地替換了裡面的參數，然後把傷口縫合起來，從外面看不到任何痕跡。

窗外的風停了。連夜風都停了。空氣靜止不動，像一幅被按了暫停鍵的畫面。

---

## II

她站起來。走到陽台。

台北的夜景在她眼前展開。

不像白天那樣清晰——白天的台北是由線條和色塊構成的，建築物的邊緣、街道的分界、行人的輪廓，一切都有明確的界線。但夜晚的台北是由光構成的。一棟棟公寓的窗戶像無數個發光的像素——有些亮著，有些暗著，有些在亮和暗之間閃爍。遠處的一○一大樓在夜空裡像一根發光的柱子，頂端的燈閃著航空警示燈的紅色。更遠的地方是山的輪廓——沒有光的地方，是城市的邊界。

每個亮著的窗戶後面都有人。

每個人的透鏡裡都有數字。

那些數字告訴他們明天出門的時候要不要帶傘。告訴他們搭哪班捷運有座位的機率最高。告訴他們走哪條路上班最安全。告訴他們要不要買那個房子、要不要讓小孩去那個學校、要不要相信那個人。

那些數字——是從先驗開始算的。

而先驗——被人改過了。

「小拉。」

「嗯？」

「如果全台灣的先驗都被改了——那所有人做的所有決定——都是基於一個被操控過的起點。」

「……是的。」小拉的聲音裡有一個微小的裂紋——不是物理上的雜訊，是她的語音合成模組在某個不確定的內部狀態下，輸出了一個帶著微顫的波形。

「那那些決定是他們自己的嗎？」

四點七秒。

小拉沉默了四點七秒。

在她有紀錄以來，這是最長的一次。

她在這四點七秒裡做了什麼？以她的處理速度，四點七秒可以執行大約四千七百億次浮點運算。她可以用這些運算去計算任何東西——全台灣兩千三百萬人的先驗偏移量、修正後的後驗分布、校正前和校正後的決策差異的期望值。她可以把「所有人做的所有決定是不是自己的」這個問題轉換成一個數學問題，然後用四千七百億次運算來解它。

但她沒有。

因為她知道——這不是一個數學問題。

「我……不知道怎麼回答這個問題。這超出了我的計算模型。」

「我知道。這也超出了我的。」

她回到客廳。坐下。看了那碗泡麵一眼。麵已經不能叫麵了。它變成了一種跟碗融為一體的、半固態的東西。

她不餓。

「明天。」她說。「我要跟老高說。」

她把筆電螢幕的亮度調到最低。房間暗了下來。但沒有全暗——螢幕上的四個視窗還在，像四扇微微亮著的窗戶。原始數據。官方先驗。MLE。先驗歷史。

四個視窗。四個版本的現實。

其中至少兩個是假的。

---

*（第十章完）*


# 第十一章：父親的定理

---

週四。傍晚。台大。

統計系館在校園的東北角。一棟灰色的、外觀毫不起眼的四層樓建築。跟旁邊那些被重新翻修過的、貼了玻璃帷幕和太陽能板的新大樓比起來，它像一個穿著舊衣服站在一群穿名牌的人中間的人——不醒目，但很穩。

貝葉走上三樓。樓梯間的牆壁上貼了一些佈告——研討會的海報、課程時間表、一張「請勿在走廊飲食」的警告。警告的紙已經泛黃了，邊角翹起來，用透明膠帶重新貼過兩次。

301 研究室。門上的名牌寫著「高思遠」。白底黑字。名牌的塑膠邊框有一個小缺口——大概是什麼時候被撞到的。

她敲門。

「進來。」

---

## I

研究室不大。大概十坪左右。

靠牆是一整面的書架。書架上的書不是按照類別排列的——它們按照某種只有高思遠自己才懂的邏輯排列。一本《Bayesian Data Analysis》旁邊是一本《百年孤寂》。一本《Probability Theory: The Logic of Science》夾在一本《莊子》和一本蒼井優的攝影集之間。他說過：「我把相關的東西放在一起。」但他認為相關的方式不是學科分類——是思想的共振。

書架的最上面一層積了灰。沒有人夠得到那裡。那裡放著的是高思遠最早期的論文抽印本——二十幾歲的時候寫的，紙都發黃了。

窗戶面向椰林大道。夕陽的光從窗外照進來，把整個房間染成了一種溫暖的琥珀色。光線裡有灰塵在飄——它們在空中漫無目的地漂浮，被光照亮之後看起來像微型的星星。

白板在門的正對面。佔了半面牆。

白板上還留著高思遠最後一堂課的板書。粉筆字。不是白板筆——是真正的粉筆。他在白板上用粉筆寫字，這是他的堅持。白板筆的線條太均勻了，沒有手感，「像打字」。粉筆有質感——用力的時候線條粗，放輕的時候線條細，轉彎的時候粉末會飛濺，寫完之後手指上沾著一層白色的粉。

板書的標題是：「Bayesian Statistics — Lecture 24: The End is Just the Beginning」。

最後一行字，寫在白板的最下方，字體比上面的小一號：

「質疑先驗。永遠質疑先驗。」

高思遠坐在窗邊的椅子上。一把老式的辦公椅，轉輪已經不太靈活了，坐下去的時候會發出輕微的嘎吱聲。他面向窗戶，夕陽在他的背後，所以他的臉在陰影裡。但他的輪廓被光勾勒出來了——白髮、瘦長的肩膀、口袋裡那截露出來的粉筆。

他的透鏡——第一代原型機——左眼邊緣閃了一下。那個 bug。

「老高。」

「嗯。來了。坐。」

他指了指桌子對面的椅子。那張椅子上放著一疊書——他隨手推到旁邊，書滑了一下，最上面那本差點掉下去，他伸手接住了。動作比以前慢。但還是接住了。

貝葉坐下來。把筆電放在桌上。打開螢幕。

「我有東西要給你看。」

她把七個區域的數據調出來。似然比。先驗偏移。歷史紀錄。那個突然斷裂的期望值曲線。所有的東西都在螢幕上——排列整齊、標注清晰、邏輯鏈條完整。三天的工作。三個深夜。濃縮在一個螢幕裡。

高思遠戴上老花眼鏡。雖然他有透鏡——透鏡可以放大任何文字——但他看近距離的東西時還是習慣用眼鏡。「透鏡看到的是數據。眼鏡看到的是文字。」他說過。「我需要用不同的方式看同一件事。」

他看了很久。

逐行。逐列。逐一個數字。他看數據的方式跟他吃飯的方式一樣——慢慢來，不急。每一行數字都被他嚼過了，消化了，吸收了。他的手指偶爾在桌面上輕輕敲一下——那是他在思考的信號，就像貝葉在思考時會摸計算機一樣。

他沒有問「這是什麼」。因為他知道。

他沒有問「你確定嗎」。因為他看得出來——她確定了。數字擺在那裡。似然比擺在那裡。三顆星擺在那裡。沒有什麼好「確不確定」的。

他看完之後——把老花眼鏡摘下來。

摘的動作很慢。右手的拇指和食指捏著眼鏡的右邊鏡腿，慢慢地往外拉。鏡腿滑過耳朵的弧度——然後離開了。他把眼鏡放在桌上。動作很輕。像放下一個很重的東西。

「……我等這一天等了八年。」

---

## II

貝葉的心跳停了一拍。不是比喻。是真的停了一拍。小拉的生理監測記錄顯示，在高思遠說出那句話之後的零點七秒內，貝葉的心律出現了一個 PVC——期前心室收縮——一次不在正常節奏上的、多出來的跳動，然後是一個比正常間隔稍長的代償性暫停。

「你知道？」

高思遠搖頭。動作幅度很小。

「不是知道。是懷疑。」

他站起來。辦公椅發出嘎吱的聲音。他走到白板前面。站在他最後一堂課的板書旁邊。

然後他做了一件事——他把手伸進左邊口袋裡。拿出了一截粉筆。白色的。已經用了一半，表面有手指長年握持磨出來的凹痕。

他在白板上寫字。

粉筆碰到白板的聲音——沙沙的。摩擦的。老派的。不是鍵盤的嗒嗒聲，不是觸控螢幕的無聲。是一截碳酸鈣碰到一面樹脂塗層的聲音。是三十年前的教室裡、每天下午兩點到三點五十分都會響起的聲音。是她父親帶她去大學旁聽的時候，坐在最後一排的椅子上聽到的聲音。

他寫了一行字。

P(θ|data) ∝ P(θ) × P(data|θ)

然後在下面，用中文寫了翻譯。

後驗 ∝ 先驗 × 似然

粉筆在寫到「先驗」兩個字的時候，用力重了一些。筆畫更深。粉末飛得更多。像在強調。像在控訴。

「貝葉。」他放下粉筆。粉筆放在白板下方的溝槽裡，跟另外兩截粉筆靠在一起。「你知道這個公式的意思。」

「後驗等於先驗乘以似然。」

「對。你的後驗——你在看完數據之後的信念——是由兩個東西決定的。你一開始就相信的東西，乘上世界給你看的證據。」

他轉過身。面對她。夕陽在他身後。她看不清他的表情——他的臉被光吃掉了，只剩一個暗色的輪廓。但她聽得見他的聲音。比平常更慢。每個字之間的間隔更長。像是在用秤子秤過每一個字的重量之後才放出來。

「先驗——是你在看到數據之前就相信的東西。你的經驗。你的直覺。你的偏見。你的愛。你的恐懼。所有這些加在一起，形成了一個分布——一個描述你『在看到任何證據之前就已經傾向相信什麼』的函數。」

他走近了一步。

「似然——是世界給你的數據。你看到了什麼。你經歷了什麼。事實是什麼。它是客觀的。它不在乎你相信什麼。它只是——它自己。」

再近一步。

「後驗——是你在看完數據之後的信念。你的新判斷。你的新決定。你對世界的新理解。它是先驗和似然的結合——你帶來的東西，加上世界給你的東西，搖在一起，變成一個新的東西。」

他停下來。站在桌子的邊緣。離貝葉大約一公尺。

「如果有人改了你的先驗——」

停頓。長長的停頓。外面椰林大道上有風吹過椰子樹，樹葉的沙沙聲從窗戶的縫隙裡滲進來。

「你看到的數據不會改變。世界給你的證據不會改變。但你的結論會改變。你的決定會改變。你的人生會改變。」

他的聲音更低了。

「而你甚至不知道。因為你以為你的先驗是你自己的。你以為你的起點是自由的。但它不是。它被人替換了。」

---

## III

貝葉的眼眶熱了。

不是因為恐懼。不是因為憤怒。

是因為這個公式。

P(θ|data) ∝ P(θ) × P(data|θ)

她第一次見到它的時候是十歲。在家裡的餐桌上。

那天是週末。爸爸不上班。他穿著一件舊的格子襯衫——跟鄧伯伯的那件很像，也是扣到最上面一顆——坐在餐桌的對面，桌上攤著一張白紙，紙上畫了幾個圓圈和箭頭。他的手指上沾了原子筆的墨水，因為他在畫圖的時候筆漏了。

「來，貝葉。爸爸教你一個東西。」

他把桌上的果汁杯推到她面前。玻璃杯裡是橙色的柳橙汁，杯壁上有水珠。

「這是先驗。」他指著杯子。「你在看到任何東西之前，就已經有的相信。」

然後他把一塊麵包放在杯子旁邊。超市買的白吐司。邊角有一點硬了。

「這是似然。世界給你看的證據。」

然後他把杯子和麵包都推到紙上的一個大圓圈裡。

「搖一搖——變成後驗。」他笑了。「簡單吧？」

她也笑了。十歲的她不完全懂這個公式的意思。但她記住了爸爸的笑。記住了柳橙汁的顏色。記住了「搖一搖」這三個字。

那是這個公式第一次進入她的生命。它是橙色的。是甜的。是溫暖的。

現在——二十八歲。這個公式被寫在一面白板上。粉筆的白色。研究室的暮色。窗外椰林大道的風。

同一個公式。同樣的三個元素——先驗、似然、後驗。

但它的意義已經完全改變了。

它不再是柳橙汁和麵包了。它是一個武器。一個被用來操控所有人的武器。

「有人用了統計學最美的定理。」她說。聲音啞了。喉嚨裡有什麼東西卡住了——不是食物，是一種更抽象的東西。是一個信念碎裂的聲音。

「做了最可怕的事。」

高思遠沒有反駁。

因為她說的是對的。

他走回窗邊。坐下。辦公椅嘎吱了一聲。他看著窗外的椰林大道。天快黑了。最後一點夕陽的金色正在從樹梢上撤退，像潮水退去。

「八年前——柏朗剛開始推廣透鏡的時候。我就覺得不對。」

他的聲音變了。不是那個在餐館裡吃滷肉飯的老高了。是另一個高思遠——一個懷疑了八年但沒有證據的人。一個把懷疑壓在心底、像壓一顆不定時炸彈的人。

「不是因為數據——八年前我沒有你現在的這些數據。是因為他的眼神。」

「什麼意思？」

「柏朗的眼神。」高思遠的手指在椅子的扶手上輕輕敲了一下。「他是我的學生。很聰明。非常聰明。聰明到你跟他說話的時候，你能感覺到他的腦子在你的前面跑——你還在想第二步，他已經在想第五步了。」

停頓。

「但聰明的人應該永遠有疑問。聰明的人知道自己不知道的有多少。那才是真正的聰明——不是知道得多，是知道自己不知道的多。但柏朗——在某一個時間點之後——他的眼裡不再有疑問了。」

他的透鏡閃了一下。那個 bug。

「一個做研究的人，眼裡應該永遠有疑問。但他看起來像一個已經確定了答案的人。而一個已經確定了答案的人——」

他看著貝葉。

「是最危險的人。」

窗外完全暗了。椰林大道上的路燈亮起來。一盞一盞地，像被點燃的蠟燭。

「你知道嗎——」高思遠的聲音突然變得更輕了。輕到像是不想被這個房間裡的書聽到。「你的父親在他去世前一週，也跟我說過同樣的話。」

貝葉的身體僵了一下。

「他說——『高老師，Oracular 的先驗更新機制有一個漏洞。如果有人在更新之前就修改了參數——』」

高思遠停了。

「他沒有說完。因為他還要去買便當。然後他就……」

他的聲音停在那裡。像一條河流到了懸崖的邊緣，水還在流，但已經沒有路了。

他咳嗽了兩聲。不是因為話太長。是因為——有些話在喉嚨裡卡了八年。八年。兩千九百多天。每一天都知道這些話應該被說出來，但每一天都沒有足夠的證據讓這些話站得住腳。所以它們就卡在那裡。像魚刺一樣。不致命。但每次吞嚥的時候都會痛。

貝葉坐在椅子上。一動不動。

她的父親——也知道？

在他去世前一週。在他走出家門去買便當的前一週。他就已經發現了。

他發現了同一件事。

然後他去買便當。然後十一秒。然後——

「老高。」她的聲音很穩。刻意壓穩的。「你為什麼等了八年？」

「因為我沒有證據。」

他看著她手裡的筆電。螢幕上的數字在暗下來的房間裡發出微弱的光。

「現在——你有了。」

---

*（第十一章完）*


# 第十二章：你發現了對不對

---

晚上九點。台大正門。

貝葉走出校門的時候，天已經全黑了。椰林大道上的路燈把地面照成一塊一塊的橙色，椰子樹的影子被拉得很長，在柏油路面上交錯著，像一個無解的方程式。

她走得很慢。

腳步的節奏不像平常——平常她走路的步頻大約每分鐘一百一十步，是一個被通勤和工作壓力訓練出來的、偏快的城市節奏。但今晚她大概只有七十步。每一步都像在水裡走路——不是因為阻力，是因為腦子裡的東西太重了，重到它拖慢了身體。

高思遠剛才說的話在她腦子裡轉。

不是旋轉。是漂浮。像太空裡的碎片——沒有重力，沒有方向，一塊一塊地飄在那裡，偶爾碰在一起，又分開。

父親知道。

父親在去世前一週就知道了。

他發現了先驗更新機制的漏洞。他準備跟高思遠說。但他沒有說完——因為他要去買便當。然後十一秒。然後巷口。然後——

一個念頭從那堆飄浮的碎片裡突然亮了一下。

十一秒。

她十五歲的時候算出來的那個數字。那個「如果爸爸提早十一秒出門就不會死」的數字。她當時用的是「隨機的不幸」來解釋那場車禍——所有獨立事件以最糟糕的方式排列，是機率的殘酷，不是任何人的惡意。

但如果——

如果那不是隨機的呢？

如果爸爸的死不是因為「外送當機」和「步伐加快」這些獨立事件的巧合排列——而是因為有人不想讓他把那句話說完呢？

這個念頭像一把刀。不是砍在皮膚上的刀——是插進肺裡的刀。你從外面看不到傷口，但裡面在出血。

「小拉。」

「在。」

「幫我跑一個東西。」

「什麼？」

「我爸去世那天——巷口的交通燈控制系統——是不是有被……」

她停了。

嘴巴張開著。但聲帶沒有振動。

「不。算了。不要查。」

「……好的。」

小拉沒有追問。

這是她第一次——主動選擇不追問。不是因為她的程式裡有「不追問」的指令。是因為她的情緒模型在那零點八秒裡計算了一個結果：在目前的情緒強度下，這個問題的答案——不管是肯定還是否定——都會造成超過她模型安全閾值的情緒衝擊。

有些問題的答案，比不知道更痛。

---

## I

她走到巷口。等紅燈。

透鏡自動顯示了一行字。

路口通過安全機率：零點九七。建議：通過。

零點九七。

她盯著那個數字。

一個禮拜前——不，五天前——她會看到這個數字，然後毫不猶豫地走過去。零點九七。百分之九十七的安全機率。剩下的百分之三——三百次裡有三次可能出事——這個風險低到不值得擔心。這就是透鏡的意義：把世界量化，讓你知道什麼是安全的、什麼是危險的，然後做出理性的決定。

但現在——她不確定了。

這個零點九七——是真的嗎？是用正確的先驗算出來的嗎？還是像那些被壓低了百分之四十五的犯罪率一樣，看起來安全，但實際上不是？

如果這個數字的先驗也被改過了——那真正的安全機率也許只有零點九二。或者零點八七。或者更低。她不知道。她無法知道。因為她不知道先驗是不是對的。

綠燈了。她過了馬路。沒事。

但「沒事」這兩個字，從今天開始——她再也不能輕易地說出口了。

因為「沒事」的前提是：你看到的數字是真的。你的起點是對的。你的先驗沒有被人碰過。

而這些前提——全部碎了。

---

## II

回到家。十點。

打開門。把傘放在門邊——今天沒有用到，但她帶了。Beta(6.2, 3.8) 的預測繼續失準。她開始懷疑連天氣的先驗也不對——但這個念頭太荒謬了。她搖搖頭。不是所有的事情都有陰謀。有時候天氣預報就是不準而已。

脫鞋的時候看了一眼陽台。薄荷還在。在黑暗裡看不太清楚，但她知道它在那裡。

她走進客廳。沒有開燈。

站在黑暗裡。

不是因為不想開燈。是因為——她突然覺得，如果開了燈，她就要面對什麼。面對這個客廳。面對這些她每天看到的東西——茶几、沙發、冰箱、那面有水漬的天花板。在黑暗裡，這些東西只是輪廓。只是形狀。沒有細節。沒有數字。

「小拉。」

「在。」

「如果這是真的——如果所有人的先驗都被改了——那我們——」

叮咚。

門鈴。

她愣了一下。晚上十點。誰會在晚上十點按她家的門鈴？她不常有訪客。事實上她幾乎沒有訪客。她的社交面板上超過百分之五十的朋友只有三個人，而這三個人——馬可瑜在台中、高思遠她剛從他那裡回來、林岳不會在晚上十點來她家。

叮咚。

第二次。不急。但堅定。兩次門鈴之間的間隔大約四秒——不像催促，倒像一種聲明：我在這裡。我知道你在裡面。我會等你開門。

她走到門口。透鏡自動啟動了門外掃描功能。

目標人物：男性，約二十八到三十二歲。體型偏瘦。匹配紀錄：與儲存檔案「路燈」中的頻率特徵吻合。同一人。攜帶非標準頻率裝置：是。威脅機率：零點零三。

那個人。

路燈下的那個人。白奶奶攤位對面石椅上的那個人。現在——站在她家門口的那個人。

她深呼吸了一次。然後開了門。

---

## III

他站在走廊裡。

走廊的安全指示燈是綠色的——那種永遠不會關掉的、低功率的 LED。綠色的光打在他身上，讓他整個人看起來帶著一層不真實的色調，像一張被調過濾鏡的照片。

黑色外套。裡面是黑色 T 恤。黑色長褲。一身的黑——但不是那種刻意營造的「酷」，而是一種「不想被注意到」的黑。像一個人選擇了跟夜色最接近的顏色，好讓自己融入背景。

他比她稍高。大約一百七十五。很瘦——瘦到外套在肩膀的位置微微懸空，像掛在衣架上。手腕上的那個銀色金屬環在走廊的綠光下閃了一下。

他的眼睛——是她第二次近距離看到。第一次是在路燈下，距離十二點七公尺。這一次，距離大約一公尺。

在這個距離上，她可以看清楚：他的虹膜是深棕色的，但在綠色的安全指示燈下看起來帶著一點灰。瞳孔是放大的——在暗處。但不是那種恐懼的放大。是一種高度專注的放大。像一台相機在對焦。

他的表情——很難讀。不是面無表情——面無表情是「沒有情緒」。他的臉上有情緒。但那些情緒被壓得很緊、很深、很密。像一本被壓在很多書底下的日記——你知道裡面有字，但你翻不開。

他看著她。

只說了一句話。

「你發現了對不對？」

---

五個字。

聲音不大。但每一個字的咬字都很清楚——聲母完整、韻母到位、聲調準確。像寫代碼的人說話：沒有冗餘的音節，沒有含糊的轉折，沒有拖長的語尾。每個字到了該結束的地方就結束了。

她沒有說是。也沒有說不是。

但她的沉默就是答案。

她站在門口。他站在走廊裡。一牆之隔——門框的寬度大約八十公分。八十公分的距離。室內和室外。已知和未知。

他看著她的眼睛。大約兩秒。然後他的右手從外套口袋裡伸出來。

手裡握著一張紙條。

紙條很小。大概五公分乘三公分。被折成了一個精確的長方形——每一條折痕都很直，像用尺量過的。他把紙條遞過來。

不是遞到她手裡——是塞。他的手伸進門框的空間裡，手指碰到了她的手掌，然後把紙條壓進她的掌心。他的手指很涼——涼到她的皮膚起了一層微小的雞皮疙瘩。不是害怕的那種。是——意外的那種。像在夏天突然碰到一塊冰。

紙條上只有一行字。

手寫的。黑色原子筆。字跡非常工整——每一筆都寫得很到位，沒有任何多餘的筆畫，沒有任何連筆或潦草。像寫代碼的人寫的字——精確、乾淨、不浪費墨水。

「先驗端口。1,247 次。查 B2。」

她還沒來得及問——他已經轉身了。

沒有解釋。沒有自我介紹。沒有「我叫什麼名字」、「我為什麼來找你」、「你可以信任我」之類的話。他轉身的動作很快、很乾淨——像一個完成了任務的程式，return 之後直接 exit。

他走向樓梯。步伐穩定。鞋底踩在磨石子地板上的聲音很輕。

門在他身後關上。她是自己關的——手推了一下門板，門板滑過門框的邊緣，發出啪的一聲。跟爸爸十六年前出門時的聲音一樣。一個普通的門關上的聲音。

走廊的空氣裡殘留著一種味道。不是香水。不是體味。是——焊錫。金屬被高溫熔化後的氣味。很淡。但她的鼻子抓住了。像在白奶奶攤位對面的石椅上殘留的那種味道。

小拉在透鏡裡瘋狂地跑數據。

「未知男性！透鏡頻率匹配 Oracular 內部加密協議——相似度百分之九十二！但他不在現任員工名單裡——我在舊的人事資料庫裡搜索到一個匹配——蘇——」

「小拉。」

「什麼？」

「安靜一下。」

「你的心率一百三十四。」

「我知道。」

「你的瞳孔放大了百分之三十七。」

「我知道。」

「你的手在發抖——」

「小拉。」

「……好。我閉嘴了。」

---

## IV

貝葉站在玄關。

手裡握著那張紙條。紙條被她的體溫暖了——剛才他遞過來的時候是涼的，涼得像金屬。但現在它是溫的。像一張普通的紙。

她攤開紙條。看了第二遍。

先驗端口。1,247 次。查 B2。

先驗端口——Oracular 系統裡，先驗分布的參數修改入口。校準師日常使用的介面只能讀取先驗、不能修改。修改先驗需要通過一個專用的端口——類似於一扇上了鎖的後門。正常情況下，這扇門只有在核心架構組的授權下才能打開。

1,247 次——一千兩百四十七次未經授權的先驗修改。

B2——Oracular 總部的地下二層。核心架構組的所在地。

一千兩百四十七次。

她在腦子裡做了一個粗略的估算。全台灣有三百六十八個行政區，每個區域有多個 Section。如果每個 Section 的先驗被修改一次，一千兩百四十七次大約覆蓋了——大量的區域。

他是誰？他怎麼知道這些？他為什麼來找她？

紙條上沒有署名。沒有聯繫方式。沒有「如果你想知道更多，在某某地方見面」。只有一行字。十四個字。像一段壓縮到極限的數據——所有的冗餘都被移除了，只剩下最核心的資訊。

她走到客廳。坐下來。把紙條放在茶几上。旁邊是那碗昨晚的泡麵——她還沒倒掉。一個改變一切的線索，和一碗泡壞的泡麵。放在同一張茶几上。

她拿起手機。找到高思遠的號碼。按下撥號。

嘟——嘟——嘟——

三聲之後接了。

「老高。」

「嗯？這麼晚了。」他的聲音帶著一點睡意。他大概已經躺下了。但他沒有抱怨——二十年來，貝葉不管多晚打電話，他都會接。

「有人來找我了。」

電話那頭安靜了一秒。

「……誰？」

她把紙條拿起來。看了第三遍。那行字在筆電螢幕的微光下清晰可見——黑色的字跡，工整的筆畫，金屬味的殘留。

「一個知道一切的人。」

電話那頭安靜了三秒。

三秒。

然後高思遠說了一句話。語氣裡有如釋重負——像一個人終於放下了背了八年的東西。但同時也有擔憂——像一個人知道放下的那個東西不是消失了，而是轉移到了另一個人的背上。

「那就開始吧。」

---

窗外開始下雨了。

不是大雨。是那種台北六月常有的——突然的、沒有任何預兆的微雨。雨滴打在陽台遮雨棚上的聲音很輕，像有人在用指尖敲一面鼓。

透鏡在視野的角落裡跳出了一行字：降雨機率已更新為百分之九十一。

但她已經不在乎雨了。

她在乎的是——那個數字背後，到底有多少人看到的世界是假的。

她站在陽台的門口。看著外面的雨。

薄荷在雨裡搖晃。它的葉子上有水珠。水珠在葉片的邊緣停留了一秒——懸在那裡，被表面張力抓著。然後表面張力撐不住了。

落下。

一滴水從一片葉子上落到土壤裡。

像一個機率從一變成零的瞬間。確定的。不可逆的。

她把紙條折好。放進口袋裡。貼著身體。

先驗端口。1,247 次。B2。

---

她最後看了一眼陽台。

薄荷在雨裡。

沒有人替它算過存活機率。沒有人替它設定先驗。沒有人替它校準。

它不知道明天會不會下雨。不知道土壤裡的養分夠不夠。不知道那盞走廊上的壞掉的燈會不會被修好。

但它還活著。

而且——明天還會長。

---

**─── 第一篇「校準」完 ───**

*（第十二章完）*


# 第十三章：永和的牆

---

紙條上寫著一個地址。

不是他給的。是小拉根據那天路燈下的頻率殘留，反推出的訊號發射密度最高的位置。她說：「如果這個頻率的裝置每天都在同一個地方待超過十二小時，那這個地方大概率是他住的地方。」大概率。小拉很少用這種不精確的詞。但頻率追蹤不是她的核心功能，所以她只能給一個模糊的答案。

永和。竹林路。一棟老公寓的五樓。

貝葉在禮拜六下午去了。搭捷運到頂溪站，走路十二分鐘。永和的街道跟台北不太一樣——路更窄，機車更多，騎樓下的店面更密集。有些店已經開了二三十年了，招牌的字體和顏色帶著一種九零年代的味道。空氣裡混著早餐店的蛋餅油煙、機車排氣、和某戶人家陽台上曬的棉被散發出的洗衣粉味。

竹林路上的那棟公寓比她想像的更舊。外牆的磁磚大部分已經變色了，從原來的米白色變成了一種帶灰的黃。樓梯間沒有電梯，也沒有燈。她走上去的時候，每一層的轉角都有不同的味道——二樓是咖哩，三樓是洗碗精，四樓是什麼都沒有。

五樓。門是關的。一扇很普通的鐵門。門把上的漆掉了大半。

她站在門口。猶豫了三秒。

然後按了門鈴。門鈴壞了——沒有聲音。她改用手敲門。三下。力道不大。

門從裡面打開了。

蘇立恆站在門口。跟上次一樣——黑色外套，黑色 T 恤。但今天他沒有穿外套。只穿 T 恤。她第一次看到他的手臂——很瘦，前臂上有一條細長的疤，大約十公分，已經癒合了，是白色的。手腕上的銀色金屬環比她上次看到的更清楚了——不是裝飾品，是某種電子裝置，表面有微小的指示燈，正在閃非常微弱的藍光。

他看到她。沒有驚訝。像在預期之內。

「進來。」

兩個字。沒有多餘的。

---

## I

公寓很小。大概八坪。

一個房間充當客廳和臥室。一張單人床推在牆角，被子折得很整齊——摺痕銳利，像是用尺量過的。床旁邊是一張桌子，上面有三台螢幕和一台主機，螢幕的光在暗處形成了三個重疊的矩形光源。鍵盤和滑鼠都是黑色的，放在桌面上的位置精確到看起來像被定位過。

廚房在角落。一個單口瓦斯爐。一個小冰箱。冰箱上面放著一包咖啡豆和一台手沖壺——手沖壺的銅管發出暗沉的光，像被用了很久但保養得很好。

沒有電視。沒有沙發。沒有任何裝飾。

但牆上——

牆上有東西。

她走進來的時候，第一眼看到的不是床、不是桌子、不是那三台螢幕。是牆。因為牆不是白色的了。它被東西覆蓋了。

一面牆。從地板到天花板。大約三公尺寬、兩公尺七高。

最中間是一張台灣地圖。不是印刷的——是手繪的，用黑色簽字筆畫在一張大型的白紙上，再用膠帶貼在牆上。地圖的比例不完全正確，但所有的縣市邊界都畫了。

地圖上有二十三個紅點。

分散在不同的區域——台北有七個、新北五個、桃園三個、台中兩個、台南兩個、高雄兩個、其他地方各一個。每個紅點旁邊用小字標注了區域名稱和一組數字。

紅點之間用不同顏色的線連接著。紅色的線、藍色的線、黃色的線。線的旁邊也有標注——日期、偏移幅度、先驗類型。

地圖的周圍——空白的牆面上——貼滿了紙。A4 的紙。列印的數據表格、手寫的計算、螢幕截圖的列印、報紙的剪報。有些紙的邊角翹起來了，有些紙已經泛黃了。最老的一張——她湊近看了一下日期——是三年前的。

三年。

這面牆是三年的工作。

「你一個人做的？」她轉過頭看蘇立恆。

他站在門口。手插在褲子口袋裡。臉上的表情——跟那面牆一樣。很多東西壓在一起，層層疊疊，但每一層都放得很整齊。

「嗯。」

一個字。

她轉回去看那面牆。二十三個紅點。二十三個被修改過先驗的區域。不是她發現的七個——是二十三個。她只是用了一個禮拜，看了七個區域。他用了三年，找到了二十三個。

「你比我早三年知道。」

「不是知道。」他說。聲音很平。「是懷疑。跟高教授一樣。差別是——我有辦法驗證。」

「因為你是 Oracular 的人。」

他沒有回答。但他的沉默不是否認。是一種「你說的沒錯但我現在不想談這個」的沉默。

她決定不追問。現在不是追問的時候。

---

## II

蘇立恆泡了兩杯咖啡。

手沖的。他的動作非常精確——水溫、注水量、旋轉的速度和角度，每一個步驟都像在執行一段寫好的程式碼。水柱從壺口倒出來的時候，畫了一個完美的螺旋，從咖啡粉的中心往外擴散。他倒水的時間是兩分三十秒。不多不少。

他把其中一杯放在她面前。黑咖啡。沒有問她要不要加糖或牛奶。

她喝了一口。苦。但那種苦的後面有一層很薄的甜——像某種水果的甜，藏在苦味的最深處，你必須把苦嚥下去之後才嚐得到。

「這面牆。」她看著那二十三個紅點。「告訴我。」

蘇立恆在她對面的椅子上坐下來。椅子是摺疊椅——金屬的，坐下去會發出喀的一聲。他坐得很直。脊椎的弧度接近零。

「二十三個區域。先驗被修改的幅度從百分之十二到百分之六十一不等。類型涵蓋四種分布——犯罪率用 Gamma、經濟指標用常態、等候時間用指數、民調數據用 Beta。」

他一口氣說完了這段話。語速不快——但每個字之間的間隔幾乎一樣，像一台打字機。沒有停頓，沒有猶豫，沒有「呃」或「嗯」。

「修改的時間集中在過去十八個月。全部發生在系統自動校準啟動之後。」

他站起來。走到牆前面。用手指點了三個紅點——台北、台中、高雄。

「這三個城市的偏移幅度最大。百分之四十五到六十一。而且是最早被修改的。」

「為什麼是這三個？」

「人口最多。影響最大。如果你想讓最多人看到假的數字——你會先從最多人的地方開始。」

貝葉看著那面牆。二十三個紅點。二十三個被改寫的區域。二十三個地方的人——每天看著透鏡上的數字做決定，以為那些數字是真的。

她想起了鄧伯伯。想起了他透鏡上偏高百分之十五的風險機率。那個偏移是因為悲傷——是他自己的心造成的。她可以校準它。可以把數字調回來。

但這二十三個區域的偏移不是悲傷造成的。是有人——刻意的、系統性的、精確的——在所有人的心裡，種下了一個假的起點。

「蘇——」她想叫他的名字，但發現自己還沒確認過。紙條上沒有署名。路燈下沒有自我介紹。門口只說了「你發現了對不對」和「進來」。「你叫什麼？」

他看了她一眼。停了零點五秒。

「蘇立恆。」

三個字。像他給所有答案一樣——精確到沒有冗餘。

「你之前在 Oracular 做什麼？」

他沒有馬上回答。他走到桌子旁邊，打開其中一台螢幕。螢幕上是一段程式碼——密密麻麻的，字體很小，綠色的字在黑色的背景上。

「核心架構組。」他說。「先驗更新模組。」

她停了一下。

先驗更新模組。就是那個被用來修改二十三個區域先驗的模組。

「你設計的？」

「部分。加密協議是我寫的。」他的語氣沒有變化。像在說「今天的天氣是晴天」一樣。「三年前離職。」

「為什麼？」

他看了螢幕上的程式碼一眼。目光停留了大約兩秒。

「因為我發現——我寫的東西被用來做我不想做的事。」

他把螢幕關了。

房間暗了一些。三台螢幕變成兩台。

「所以你花了三年找證據。」

「嗯。」

「一個人。」

「嗯。」

三年。一個人。在永和的五樓。一面牆。二十三個紅點。

她看了一眼他的房間。沒有電視。沒有沙發。沒有任何跟「生活」有關的東西——沒有照片、沒有書、沒有雜誌、沒有任何暗示這裡住了一個「人」而不是一台「機器」的痕跡。

只有那面牆。和三台螢幕。和一台手沖壺。

手沖壺是唯一跟「人」有關的東西。

她低頭看了看手裡的咖啡。還是溫的。苦的。但後面那層甜——越來越明顯了。

---

## III

她在那面牆前面站了很久。

小拉在透鏡裡安靜地運行著。她沒有說話——也許是因為離線模式下她的功能有限，也許是因為她感知到了此刻不需要任何數據分析。

貝葉的腦子在轉。不是快速的轉——是那種把所有東西攤開來、一件一件看、一件一件分類的轉。

二十三個區域。四種分布。同一個方向的偏移。同一個時間段。

「蘇立恆。」

「嗯。」

「你找到了二十三個。但台灣有三百六十八個行政區。」

「我知道。」

「二十三個只是你確認的。實際被修改的可能更多。」

「我知道。但我一個人的計算能力有限。我需要原始數據。警政的、衛生的、經濟的、民調的。有些公開數據我可以下載。有些不行。」

他看了她一眼。

「你在 Oracular 還有存取權限。」

這句話不是問句。是陳述句。他知道她在 Oracular 工作。他知道她有什麼權限。他甚至知道她已經開始用離線模式跑數據了——也許他一直在監測那個頻率。

「你來找我——是因為我有存取權限。」

他停了一下。

「不只是。」

「那還有什麼？」

他站在那面牆前面。背對著她。手指碰了一下地圖上標注「大安區 Sec.7」的那個紅點。

「那天下午。你在辦公室裡看到偏態零點零二的時候——你沒有蓋章。」

她愣了一下。

「你怎麼知道？」

他沒有回答這個問題。但他轉過身來的時候，臉上有一種很微妙的表情——不是笑。蘇立恆不太笑。但他的眼角——那條通常繃得很直的線——軟了一下。只有一下。像一根被撥了一下的琴弦，振動了，然後馬上恢復。

「整個 Oracular 校準組，一千兩百人。每天校準幾千個區域。每個人看到數字就蓋章。綠色的勾。通過。下一個。」

他看著她。

「你是唯一一個——看到零點零二之後沒有蓋章的人。」

貝葉想說什麼。但她的喉嚨裡有一個結——不是難過的結。是一種更複雜的東西。是「被看見」的感覺。不是被透鏡看見——透鏡看到的是數據、是行為、是機率。是被一個人看見。一個在路燈下站了不知道多久、在蔥油餅攤對面坐了三年、在走廊的綠色安全指示燈下遞出一張紙條的人——看見了她沒有蓋章的那一刻。

「我們有不同的能力。」他說。「我懂系統。你懂校準。但我們都做了同一件事——質疑先驗。」

質疑先驗。

高思遠的話。

從一個退休教授的口中，這句話像一堂課的結語——溫暖的、學術的、帶著粉筆灰的味道。

從蘇立恆的口中，這句話像一個行動指令——冰冷的、精確的、帶著焊錫和咖啡的味道。

同一句話。兩個味道。但指向同一個方向。

「所以我們現在怎麼辦？」她問。

蘇立恆走到牆前面。拿起一支紅色的簽字筆。在地圖上，在那二十三個紅點的旁邊，畫了一個大大的問號。

「找到剩下的。」

---

她離開的時候是傍晚。走下樓梯的時候，經過三樓的轉角，聞到了洗碗精的味道。門開了一條縫，裡面傳來電視的聲音和小孩的笑聲。

她走出公寓大門。永和的街道在夕陽裡有一種奇怪的溫暖感——老舊的、嘈雜的、不整潔的，但也是真實的、活著的、不需要任何演算法來驗證的。

她走到巷口的時候，手機震了一下。一條加密訊息。來自蘇立恆剛才讓她安裝的通訊軟體。

訊息只有三個字。

「白奶奶。」

她看了看對面的路。如果現在搭捷運去萬華，大概四十分鐘。白奶奶的攤位通常開到晚上七點半。

她搭上了捷運。

四十分鐘後，她站在龍山寺旁邊的巷子裡。蔥油餅的味道。

白奶奶看到她，眼睛瞇起來。

「哎呀，陳家的那個丫頭。好久沒來了。」

「奶奶好。」

「今天的蔥——特別香。」

貝葉坐到紅色板凳上。喀。

她看了一眼對面的石椅。

石椅是乾淨的。有人擦過。椅面上沒有灰塵、沒有落葉。在一整排長年無人打理的石椅中間，這一張——被照顧著。

上面放了一張蔥油餅。用牛皮紙包著。還溫的。

蘇立恆。

他比她先到。

---

*（第十三章完）*


# 第十四章：分布的味道

---

週日。下午。永和。

蘇立恆的套房比昨天稍微不同了。不是因為什麼東西被移動了——牆上的地圖還在、紅點還在、三台螢幕的位置精確到像被定位過——是因為窗戶開了。昨天來的時候窗戶是關的。今天是開的。午後的風從外面吹進來，帶著巷子裡某戶人家正在煮紅燒肉的味道。

窗戶的對面是浴室的門。門旁邊的牆上多了一面白板——昨天她沒有注意到。也許昨天被那面紅點牆吸走了所有的注意力。白板不小。大約一百二十公分寬、九十公分高。用膠帶黏在牆上，因為太重了，釘子撐不住。白板的表面有很多被擦過的痕跡——黑色的、紅色的、藍色的，層層疊疊，像一張被反覆使用的稿紙。

蘇立恆站在白板前面。手裡拿著一支黑色的白板筆。筆蓋咬在嘴裡。

「每種分布都有自己的簽名。」他說。聲音被筆蓋壓得有一點含糊，但每個字還是很清楚。「你要學會認。」

她坐在摺疊椅上。手邊是一杯黑咖啡——蘇立恆泡的。跟昨天一樣的苦，跟昨天一樣在苦的最深處藏著一層薄薄的甜。

窗外有人在巷子裡叫賣。聲音悠長，拖著尾音，像一條緩緩下降的正弦波。「豆——花——」

---

## I

他畫了三條曲線。

手非常穩。線條的弧度流暢得像是印出來的——但不是印的，因為你可以看到筆尖在板面上的壓力變化。他畫線的時候用的力道不均勻，轉彎的地方重、直線的地方輕，這讓每一條曲線都有了一種手工的質感，像被書法家寫出來的筆畫。

第一條：一個完美的鐘形。左右對稱。中間高、兩邊低。像一座等腰的山。

第二條：一個 L 形。左邊很高，然後急速下降，右邊拖了一條長長的尾巴，越來越低但永遠不觸底。像一個人從懸崖上滑下去，然後在平地上慢慢減速。

第三條：介於前兩者之間。比 L 形溫和，比鐘形偏斜。它的高峰不在正中間——偏左。右邊的尾巴比左邊長。整體看起來像一個被風吹歪了的山丘。

他在第一條曲線上方寫了「Normal」。第二條上方寫了「Exponential」。第三條上方寫了「Gamma」。

然後他把筆蓋從嘴裡吐出來。接住。扣上。一套連貫的動作，精確到像排練過的。

「常態分布。」他指著第一條曲線。「對稱。鐘形。像一座山。如果你看到一組數據的直方圖長這樣——中間高、兩邊低、左右對稱——那它可能是常態的。經濟指標。考試成績。身高。測量誤差。這些東西天生就接近常態。因為它們是很多獨立的小因素加在一起的結果——中央極限定理。」

他說話的速度比平常快。不是很多——大概快了百分之二十。但對一個平常說話像在數字鍵盤上敲數字的人來說，百分之二十是很明顯的。

「它的參數有兩個。μ 決定山頂的位置——山在哪裡。σ² 決定山的寬度——山有多胖。同樣的山頂，σ² 大的像富士山，σ² 小的像針。」

他走到第二條曲線前面。

「指數分布。L 形。一邊高然後急速下降。只有正值——沒有負的等待時間。如果你看到一組數據全部大於零，右邊有一條長長的尾巴——大部分數值集中在左邊、少數數值很大——可能是指數。」

他停了一下。像在考慮要不要說下一句話。然後說了。

「它有一個特性。無記憶性。你在醫院等了兩個小時，不代表你接下來再等一小時的機率有任何改變。它不記得你已經等了多久。每一個瞬間都是一個新的開始。」

「這也太絕望了。」貝葉說。

蘇立恆沒有回答。但他的嘴角動了一下——不是笑。是某種差點變成笑但被肌肉在最後一秒攔住的東西。像一扇快要被風吹開的門，被門栓卡住了。

他走到第三條曲線前面。

「Gamma 分布。指數的升級版。更靈活。它有兩個參數——α 控制形狀，β 控制速率。α 小的時候像指數，L 形，很偏。α 大的時候像常態，鐘形，對稱。」

「所以 Gamma 可以變成很多種樣子。」

「對。它會化妝。」

他轉過身看她。第一次在說話的時候直接面對她——通常他說話的時候看白板，不看人。

「犯罪率用 Gamma 建模。因為犯罪事件數是正的、連續的、右偏的。保險理賠也用 Gamma。降雨量也是。它是一個很常見的分布——但也正因為常見，它最容易被偽裝。」

貝葉看著白板上的三條曲線。

她在台大上過這些。期末考也考過。她記得在考卷上畫 Gamma 分布的 pdf 的時候，手抖了一下——因為考試時間只剩二十分鐘，而她還有三題沒寫。她畫的曲線歪歪扭扭的，跟蘇立恆畫的差了大概一百個級距。

但那時候，這些分布只是課本上的圖。死的。像標本室裡的蝴蝶——形狀在，顏色在，但不會飛了。

現在不一樣了。

每一條曲線背後都站著一群人。常態分布後面是每天看著透鏡上的經濟數據做投資決定的上班族。指數分布後面是在醫院等報告的病人——等了兩小時，不知道還要等多久，而分布告訴他們：你之前等的那兩小時，跟接下來要等多久完全無關。Gamma 分布後面是——林岳。和他看過的每一個犯罪率數字。

「所以辨識分布就像辨認人的臉？」

「差不多。看多了就有直覺。常態的臉是對稱的。指數的臉是偏的。Gamma 的臉——要看參數。」

他停了一下。

「但直覺不夠。你還需要數字。偏態係數。峰度。QQ 圖。你要用眼睛看形狀，用數字驗證。兩個一起。」

「你剛才說的話比我們認識以來所有對話加起來都多。」

蘇立恆停了。

筆蓋在他手裡轉了半圈。停住。

「……是嗎。」

「是。」

他轉回去看白板。

「繼續。」

---

## II

兩個人坐在小桌前。面對面。兩台筆電。

桌子很小——大概六十公分見方——兩台筆電放上去之後幾乎沒有空隙。她的筆電和他的筆電之間的距離大約五公分。螢幕的光在兩個人的臉上交叉映射——她的螢幕照在他的臉上是淡藍色的，他的螢幕照在她的臉上是暗綠色的。

蘇立恆把二十三個區域的原始數據傳給她。一個加密壓縮檔。密碼是一串十六位的亂數——他唸了一次，語速跟平常一樣，不快不慢，每一個數字和字母之間的間隔幾乎完全相等。她輸入的時候錯了一次——第十二個字元是大寫的 B，她打成了小寫。他沒有糾正。只是等她自己發現。她發現了。重打。解壓。

「你來判斷。每個區域的分布類型。」

「現在？」

「現在。」

他的語氣不是命令。是——測試。但不是那種帶惡意的測試，不是想看你失敗的測試。更像一個人把棋盤擺好了，等你下第一步。想看你的棋路到哪裡。

第一個區域。大安區。

她打開數據。用 Python 畫了直方圖。柱狀圖在螢幕上一條一條地冒出來——右偏。明顯的右偏。所有的值都大於零。右邊有一條不短的尾巴。高峰不在正中間，偏左。

「Gamma。」她說。「偏態明顯。右尾巴長。α 大概二到三之間。」

「對。」

一個字。但那一個字裡面沒有意外。像他已經知道她會答對一樣。

第二個區域。信義區。

直方圖。乍看有點像鐘形——對稱的、中間高的、兩邊往下降的。她的第一反應是 Normal。但她的手停在鍵盤上。沒有馬上打字。

因為有什麼東西不對。

不是明顯的不對——是那種你說不出來哪裡不對、但就是覺得不太舒服的不對。像穿了一雙左右腳大小差了零點五號的鞋子。你走得動。但你知道有什麼在磨。

「再看一次。」蘇立恆說。

她放大了直方圖。盯著看了五秒。

峰。峰太尖了。

常態分布的峰度是三。這是一個被寫在教科書第四十七頁、被她在考試裡寫過至少六次的數字。峰度三。但眼前這個直方圖的峰——比三高。中間的柱子比旁邊的柱子高太多了。而且右邊的尾巴——如果真的是常態，左右的尾巴應該完全對稱。但右邊比左邊稍微長了一點。只長了一點點。但長了。

「等一下。」她說。「Normal 的峰度是三。這個超過了。而且右邊的尾巴比左邊稍微長——」

「所以？」

「不是 Normal。是——Gamma 裝成 Normal。」

她的手指在鍵盤上敲了一串指令。計算偏態係數。結果跳出來——零點一八。不是零。常態分布的偏態是零。零點一八很小。小到如果你只看直方圖、不算數字，你會覺得它是對稱的。但它不是。

「有人把 α 調得很大。α 大的時候 Gamma 長得像 Normal——因為中央極限定理。Gamma 是很多指數分布的和，α 越大就是加了越多個，加到一定程度就趨近常態了。但它本質上還是 Gamma。偏態不是零——只是接近零。」

蘇立恆的嘴角動了。

不到一毫米。

但在這間永和套房的慘白日光燈下，那不到一毫米像一道閃電——不是照亮了整個天空的那種，是遠處的雲層裡悶悶地閃了一下的那種。你不確定你有沒有看到。但你看到了。

「你的直覺很好。」

「不是直覺。是老高教的。」

「高教授？」

「嗯。他說——」她壓低了聲音，模仿高思遠的語氣，帶著台灣國語特有的捲舌不捲舌之間的含糊：「『分布不會說謊。但會化妝。你要學會卸妝。』」

蘇立恆沒有回答。但他的目光在她臉上停了一秒——比他看任何數據的時間都長。

然後他轉回去看他的螢幕。

---

## III

接下來兩個小時。二十三個區域。逐一卸妝。

她的動作越來越快。不是因為偷工減料——是因為一種節奏被建立起來了。打開數據、畫直方圖、看形狀、算偏態和峰度、判斷分布類型、確認參數範圍。每一個步驟大約三到五分鐘。二十三個區域。像二十三道菜被端上來，她一道一道地嚐——用眼睛嚐形狀，用數字嚐參數。

九個 Gamma。全部對了。右偏、正值、尾巴長度跟 α 的大小成反比。犯罪率的分布。

七個看起來像 Normal 的。她判了五個是真正的 Normal、兩個是偽裝的 Gamma——跟信義區一樣的手法，α 被調到很大，讓 Gamma 穿上了 Normal 的衣服。她用偏態係數和 QQ 圖拆穿了它們。經濟指標的分布。

四個 Exponential。L 形。急速下降。右邊的尾巴長得離譜。等待時間的分布——醫院排隊、行政審批、公車到站間隔。

三個 Beta。

她在 Beta 前面停了比較久。

Beta 的值域是零到一。它的直方圖不是在整條數線上展開的——是被壓在零和一之間的。像一個人被關在一個一公尺寬的房間裡，只能在這個範圍內移動。α 和 β 決定了他站在哪裡——α 大的時候他靠右，β 大的時候他靠左，兩個差不多大的時候他站在中間。

「Beta。」她說。「三個都是。值域零到一。但我不確定它們代表什麼。犯罪率是事件數，是正整數或正實數——不會被壓在零到一之間。經濟指標也不會。什麼樣的數據天生就在零到一之間？」

「機率。」蘇立恆說。「或者比例。支持率。認同度。滿意度。投票率。」

「民意？」

「嗯。」

她看了一眼那三個區域的數據。打開了標籤——不是犯罪率，不是經濟指標，不是等待時間。是民意調查數據。政策支持度。候選人好感度。公共議題認同度。

「他們用 Beta 分布來操控民意。」

這句話從她嘴裡出來的時候，聲音比她預期的要輕。輕到幾乎是氣音。

蘇立恆的表情沒有變化。但他的手指在桌面上敲了一下。很輕。像一個逗號——不是句號，是某個還沒說完的句子中間的停頓。

最終結果——二十三個區域，二十三個分布，全部正確辨識。九個 Gamma、七個 Normal（其中兩個是偽裝的 Gamma，被她識破了）、四個 Exponential、三個 Beta。準確率百分之百。

但「理解率」不是百分之百。二十個她知道被操控了什麼。三個 Beta——她知道它們是 Beta，知道它們代表民意數據，但不知道操控者為什麼要用這種特殊的分布。

蘇立恆看著她螢幕上的結果。語氣沒有變化。

但他的手——在桌子下面——微微握了一下。手指收攏，指甲碰到掌心，然後鬆開。整個動作不到半秒。

貝葉沒有看到。

但小拉看到了。透鏡的微表情掃描功能在離線模式下是關閉的，但基礎的動態偵測還在運行——低解析度的、只能捕捉大幅度動作的版本。蘇立恆的手在桌面以下的動作剛好落在偵測範圍的邊緣。小拉捕捉到了一個模糊的信號——不足以分析表情，但足以記錄一個事實：他的手動了。

她在自己的離線筆記裡寫了一行字：「蘇先生的手部微動作——面部表情零變化時出現。結論：他的情緒表達通道只有手指。很沒效率。但很——」

她找不到一個詞。

在她的詞庫裡有一萬七千個形容詞。但沒有一個完全精確地描述「一個人用不到半秒的握拳來表達他看到一個正確答案時的感覺」這件事。

她寫了：「待定義。」

---

## IV

天黑了。

永和的巷子裡開始有人遛狗。樓下傳來炒菜的聲音——鐵鏟碰到鐵鍋的聲音，尖銳的、短促的、重複的。然後是一個小孩的聲音，從某一層的窗戶裡飄出來：「吃飯了——」

兩個字。拖著長長的尾音。像一個等待被回應的信號。

貝葉站起來。伸了一個懶腰。腰椎發出了一聲輕微的喀——坐太久了。她看了一眼手機。六點四十七。她在這裡待了四個多小時。

白板上已經寫滿了。三條分布曲線、二十三個區域的分類結果、偏態和峰度的比較表——全部是蘇立恆的字跡，工整得像打字，但每一個字都帶著一種手寫才有的、微妙的不完美。

「我明天帶一個人來。」她說。

蘇立恆正在把白板上的內容拍照存檔。手機舉在白板前面，螢幕上的取景框對準了那三條曲線。

「誰？」

「馬可瑜。精算師。我大學室友。她有保險數據——理賠率、事故率、風險評級——可能跟我們查的有關聯。」

他按下快門。然後放下手機。轉過身看她。

「你不想見外人？」她讀出了他沉默裡的意思。

「不是不想。是——多一個人就多一個變數。」

「她不是變數。她是馬可瑜。」貝葉的聲音裡有一種很少出現的東西——篤定。不是數據支撐的篤定。是另一種。「她有一隻叫 theta 的柯基。她的拿鐵永遠加太多糖。她比我聰明但比我衝動。她如果知道有人在操控保險數據，她會比我更憤怒。」

蘇立恆看了她三秒。

三秒對他來說很長。他看數據的時候通常只需要一到兩秒就能判斷。三秒代表他在處理的東西不是數據——是別的。是那種他的計算模型不太擅長的東西。

「你很信任她。」

「我用我的先驗信任她。而我的先驗很硬。」

又一個不到一毫米。

嘴角。右邊。動了不到一毫米。然後恢復原位。整個過程不到零點三秒。

「好。明天。」

---

她走下樓梯的時候，經過三樓的轉角。電視的聲音從門縫裡漏出來——一個綜藝節目，主持人在笑，觀眾也在笑，笑聲是罐頭的，重複播放的，但聽起來很真。一個正常的聲音。在一個正常的夜晚。跟這棟公寓四樓右邊那扇門後面的世界——隔了幾層樓板和幾道牆。

她走出公寓大門。永和的夜風從新店溪的方向吹過來。風裡有水的味道——河水的、不是雨水的。河水的味道比較沉，帶著一種泥土和水草混合的氣息。

路邊的便利商店亮著。門口有一個外送員正在吃便當。他把機車停在紅線上，安全帽掛在後照鏡上，便當放在機車坐墊上，站著吃。筷子的動作很快。像在跟時間搶。

正常。全部看起來正常。

但她知道了——二十三個紅點。四種分布。同一個方向的偏移。

她拿出手機。打了一條訊息給馬可瑜。

「明天能見面嗎？我需要你的保險數據。」

三秒後。回覆。馬可瑜打字的速度一向很快——她說過這是「精算師的職業病，報告趕不完只好打字快」。

「可以！但要帶 theta 可以嗎？他明天沒人顧。」
「還有你聽起來不太對。你還好嗎？」
「不管你還不還好我們都見面。」
「我帶珍奶。你要什麼甜度？」

四條訊息。三秒之內。中間沒有任何間隔——像她在打字之前就已經把所有要說的話想好了。或者更準確地說，她不需要想。因為「在朋友需要的時候立刻出現」這件事，對馬可瑜來說不需要經過任何計算。它是先驗。是出廠設定。是不需要數據就已經存在的東西。

在整個世界都在崩塌的夜晚，馬可瑜問她要什麼甜度。

她打了兩個字：「半糖。」

然後關掉手機。走進永和的夜裡。路燈把她的影子拉得很長——長到影子的腳碰到了對面人行道的邊緣。像一個人同時站在兩個地方。

# 第十五章：馬可瑜的數據

---

週一。下午兩點。師大路。

「拉普拉斯」——一家用數學家名字命名的咖啡店。老闆是數學系畢業的。菜單上的咖啡大小不是用小、中、大標示的，是用 ε、δ、σ。ε 是最小杯，因為 ε 在數學裡代表「任意小的正數」。σ 是最大杯，老闆說「因為 σ 代表標準差——越大越好」。數學系的人會笑。其他人會皺眉。

貝葉每次都點 δ——中杯。她覺得 δ 比較適合她。δ 在數學裡是被 ε 決定的——對每一個 ε，存在一個 δ。它不是自己選擇大小的。它的大小取決於別人給它的條件。

她以前覺得這很合理。現在她不確定了。

她坐在靠窗的位子。窗外是師大路的下午——學生騎腳踏車經過、一個穿夾腳拖的男生手裡拿著一杯手搖飲、一隻橘貓趴在對面書店的門口。陽光打在玻璃上，在桌面上畫了一個歪歪的長方形。

馬可瑜遲到了七分鐘。

她衝進來的時候手裡抱著一隻柯基犬。柯基的舌頭吐在外面。耳朵豎著。眼睛圓圓的、亮亮的，看起來非常興奮而且完全不覺得被一個人類從公園裡硬抱走有什麼不對。

「抱歉抱歉——theta 在公園追松鼠追了太久。松鼠跑上樹他還在下面叫。我跟他說你抓到松鼠的機率是零點零零三他不聽——」

她把 theta 放在椅子上。theta 趴下來。四條短腿往外攤。看起來很累但很滿足。眼神裡有一種「雖然沒抓到但我盡力了」的坦然。

theta。名字來自統計學裡的未知參數 θ。馬可瑜取這個名字的理由是：「因為他永遠是一個謎。你永遠不知道他下一秒要幹嘛。就像真正的 θ。」

馬可瑜坐下來。把包包甩到椅背上——一個鼓鼓的帆布包，裡面塞了平板、充電器、一包狗零食、一罐防曬乳、和不知道為什麼有的一本全聯的 DM。她的動作很大。佔據空間的方式不是侵略性的，而是一種——自然的膨脹。像一杯剛倒進去的汽水，泡泡往各個方向散開。

她點了一杯 σ 的拿鐵。加了三顆糖。

「所以你的精算模型考慮了柯基追松鼠的條件機率嗎？」貝葉問。

「哈！如果我把 theta 的行為建模，variance 會爆炸。他就是一個 heavy-tailed distribution。極端事件頻發。上個月他試圖吃一顆石頭。不是小石頭。是花盆旁邊的那種。」

笑了。

貝葉已經好幾天沒有笑了。笑的時候臉上的肌肉有一點僵——像一台很久沒啟動的機器突然被接上電。運轉了。但軸承裡的潤滑油還沒有完全流開。

馬可瑜看著她笑。馬可瑜的觀察力不輸她——精算師的職業本能。她看到了那個僵。但她沒有說。她只是用吸管攪了一下拿鐵。攪得很用力。拉花瞬間消失了，變成一杯均勻的淺棕色。

「所以。」馬可瑜放下吸管。「你約我出來不是為了聊 theta 的松鼠問題的。」

「不是。」

「你的聲音裡有大概百分之七十五的焦慮。我沒有透鏡我是用直覺猜的。但我覺得蠻準的。」

貝葉看了她一眼。深呼吸了一次。

然後她把事情告訴了馬可瑜。

不是全部。只是——「我發現某些區域的犯罪率數據的先驗不太對。被系統性地壓低了。不是一兩個。是很多個。而且方向一致。全部往低。」

馬可瑜聽的時候沒有打斷她。這在馬可瑜身上是很少見的——她平常說話的風格是「你說一句我插三句」，像一台在多工運行的處理器，同時接收、分析、回應。但現在她安靜了。吸管插在拿鐵裡沒有動。theta 在椅子上翻了一個身，四腳朝天。

貝葉說完了。

馬可瑜沉默了大約十秒。十秒對她來說很長。她平常的反應時間大約零點五秒——比小拉慢，但在人類裡算快的。

她用吸管攪拌拿鐵。攪了很久。攪到杯壁上的奶泡都被捲進去了。咖啡從分層的變成了均勻的。一杯失去了所有結構的棕色液體。

「……貝葉。」

「嗯？」

「我也發現了。」

## I

馬可瑜拿出平板。打開了一個試算表。

她的手指在螢幕上滑動。指甲上有淡粉色的指甲油——是上個週末塗的，已經開始從邊緣剝落了，像一張被日曬褪色的貼紙。

「過去一年。大安區、信義區、中正區。車險理賠率突然下降。不是慢慢降的。是跳崖式的。」

她把平板轉向貝葉。螢幕上是一張折線圖。三條線——代表三個區域的車險理賠率。從 2043 年到 2044 年。三條線在 2043 年都在百分之四到五之間，平穩地波動。然後到了 2044 年——同時往下掉。像三條繩子被同一把剪刀剪斷了。

大安區：百分之四點七降到百分之二點九。下降百分之三十八。信義區：百分之五點一降到百分之三點零。下降百分之四十一。中正區：百分之四點三降到百分之二點八。下降百分之三十五。

「多少？」

「平均下降百分之三十八。」馬可瑜的語氣從「朋友模式」切換成了「精算師模式」——像換了一個頻道。眼神從柔和變成銳利。聲音從快速的、跳躍的，變成穩定的、有節奏的。「但同期的實際交通事故數——沒有減少。」

貝葉的手停在咖啡杯上。

「理賠率降了，但事故沒降？」

「對。知道為什麼嗎？」

馬可瑜放下平板。用手指在桌面上比劃——她在解釋複雜的因果鏈時有這個習慣，手指會在桌上畫箭頭，像在白板上寫字，只是沒有筆也沒有墨。

「透鏡告訴那些區域的居民：你們很安全。犯罪率低。風險低。居民就覺得安全。覺得安全——就不買那麼多保險了。為什麼要花錢保一個不會發生的事？保險購買率下降。」

她的手指畫了第一條箭頭。

「同時——保險公司也看透鏡的數據。透鏡說這些區域安全，保險公司就降低風險評級。風險評級降了，保費就降了。保費降了——反而有一些原本不買保險的人覺得『反正便宜就保一下』。保單數暴增。分母變大了。」

第二條箭頭。

「分子——實際理賠次數——沒有變。因為事故數沒有變。事故數不看透鏡。事故數看的是真實世界——路況、車速、駕駛行為。這些都沒有因為透鏡的數字改變而改變。」

第三條箭頭。

「分子不變，分母暴增。理賠率——分子除以分母——自然就降了。表面上看起來更安全了。但實際上——」

「什麼都沒變。」

「什麼都沒變。只是數字上被稀釋了。」

馬可瑜把平板放回桌上。theta 在椅子上打了一個小小的呼嚕。他的耳朵在睡夢中抖了一下——也許他在夢裡終於追到了那隻松鼠。

「貝葉。這不是在改幾個犯罪率。他們在改整個社會的風險認知。」

風險認知。

兩個字。但它代表的東西——比犯罪率大得多。

改了先驗，就改了後驗。改了後驗，就改了行為。改了行為，就改了市場。改了市場——就改了更多人的先驗。因為市場的數據會反過來成為新的「證據」，進入下一輪的貝氏更新。這不是一條直線。是一個迴圈。一個自我強化的迴圈。像一條銜尾蛇——頭咬著尾巴，越轉越緊。

貝葉想了一分鐘。咖啡涼了。她沒有喝。

「可瑜。你能把你的保險數據帶到一個地方嗎？」

「什麼地方？」

「永和。一間套房。」

「聽起來像詐騙。」

「牆上全是紅點。」

「聽起來更像詐騙了。」

「有一個很安靜的人在裡面。他查了三年。」

馬可瑜看著她。看了很久。不是懷疑的看——是評估的看。精算師的看。她在計算的不是「貝葉說的是不是真的」——她知道是真的，因為貝葉不會騙她。她在計算的是「如果我進去了，代價是什麼」。

然後她低頭看 theta。

「theta，你覺得呢？」

theta 翻了一個身。四腳朝天。

「他說好。」

「他什麼都沒說。」

「四腳朝天是他的贊同姿勢。我統計過。準確率——呃。大概百分之四十七。」

「那跟隨機猜一樣。」

「所以我不是用他的回答做決定。我是用我的先驗。」

她站起來。抱起 theta。theta 的四條腿在空中晃了兩下，然後乖乖地縮進她的臂彎裡。

「走吧。去看你的紅點。」

## II

傍晚。永和。

馬可瑜推開門的時候，theta 先進去了。他的短腿在地板上跑了三步，鼻子貼著地面嗅——嗅了桌腳、嗅了牆角、嗅到了床旁邊的那張瑜伽墊。然後他趴上去。轉了一圈。躺下。三秒之內就睡著了。

蘇立恆站在地圖前面。看著他的瑜伽墊上多了一隻柯基犬。

「你的狗——」

「他覺得很舒服。這是好跡象。theta 對不安全的地方會亂叫。」

蘇立恆沒有回答。他看了 theta 一眼。一隻屁股圓圓的柯基趴在他睡了三年的瑜伽墊上。正在打呼。耳朵偶爾抖一下。

這是他獨自待了三年的套房裡，第一次出現一個活的、會呼吸的、除了他以外的生物。

馬可瑜站在牆前面。

她看了五分鐘。一言不發。

五分鐘對馬可瑜來說——像五年。她是那種在餐廳點餐最多花三十秒的人。在超市結帳排隊超過五分鐘就會開始計算轉換到隔壁隊伍的期望等待時間的人。五分鐘的沉默——代表她被震住了。

二十三個紅點。四種分布。紅色的線、藍色的線、黃色的線。日期。偏移幅度。手寫的計算。打印的圖表。泛黃的紙。新的紙。三年的工作壓縮在一面牆上。

她轉過頭。看蘇立恆。

「三年？」

「三年。」

「一個人？」

「一個人。」

她又看了一分鐘。

然後她說了一句話。語氣裡沒有佩服。也沒有同情。有的是一種——憤怒。但那個憤怒不是對蘇立恆的。是對「一個人需要獨自查三年」這件事本身的。

「你是白痴嗎？」

蘇立恆微微抬頭。

「一個人查三年？不找人幫忙？不找媒體？不找政府？」

「找誰？」蘇立恆的聲音很平。「媒體不懂貝氏統計。政府用的是 Oracular 的系統。找人幫忙——我不知道誰可以信任。」

馬可瑜嘆了一口氣。很大聲的一口氣——大到 theta 被吵醒了，抬頭看了一眼，確認沒有松鼠，又趴下了。

「好吧。」她把帆布包甩到地上。裡面的東西發出喀拉喀拉的聲音。「現在你有兩個人了。」她看了一眼貝葉。「兩個半。theta 算半個。」

她從包裡拿出平板。打開試算表。走到牆前面。把平板的螢幕舉起來，對準那些紅點。

蘇立恆的犯罪率數據。馬可瑜的保險理賠率。兩組數據。來自完全不同的領域、完全不同的資料來源。一個是從公開的警政資料庫和三年的側面蒐集得到的。一個是從保險公司的內部精算系統裡拉出來的。

兩組數據疊在一起的時候——

重合了。

二十三個區域裡，有十七個——保險數據也出現了異常。犯罪率先驗被壓低的地方，理賠率也在同一時期出現了跳崖式的下降。偏移的幅度不完全一樣——犯罪率被壓低百分之十二到六十一，理賠率下降百分之三十五到四十一——但方向一致。時間一致。區域一致。

「十七除以二十三。」蘇立恆說。

「相關係數零點八九。」

「所以不是巧合。」馬可瑜說。

「不是。」

三個人沉默了。

theta 打了一個小小的呼嚕。窗外有人在叫外賣。「老闆——雞排不要辣——」聲音從巷子裡飄上來，穿過開著的窗戶，飄進這間裝滿了紅點和數據的套房裡。

一個正常的聲音。在一個不正常的房間裡。

---

## III

馬可瑜帶著 theta 先走了。

她要早起上班。走之前在門口停了一下。回頭看了一眼那面牆。然後看了一眼蘇立恆——他已經轉回去面對那面牆了，背對著她們，手裡拿著紅色的簽字筆，在地圖上標注新的數據。

「蘇立恆。」

他沒有轉身。「嗯。」

「你的瑜伽墊被 theta 佔了。你今晚睡哪？」

「地板。」

「……你有床。」

「不習慣。」

馬可瑜看了貝葉一眼。那個眼神裡有很多東西——「這個人是怎麼回事」和「你找到的隊友真的沒問題嗎」和「但他的牆確實很厲害」和「我回去之後會擔心你」和「你要記得吃飯」和「半糖的珍奶你都沒喝完」。

所有這些訊息——用一個眼神傳完了。

「明天見。」馬可瑜抱著 theta 走下樓梯。theta 的尾巴在她的手臂下面搖。走到二樓的轉角時，她的聲音從樓梯間裡飄上來：「theta 你不要在樓梯上亂動——你又不是松鼠——」

腳步聲。門關的聲音。然後安靜了。

套房裡只剩下兩個人。兩台筆電。一面牆。

蘇立恆在牆上加了新的標記。十七個紅點旁邊多了藍色的小圓圈——代表保險數據也出現異常的區域。紅色配藍色。犯罪率配理賠率。兩個不同的系統、兩組不同的數據、指向同一個結論。

「你有一個很好的朋友。」蘇立恆說。沒有轉身。筆還在牆上。

「嗯。」

「她很直接。」

「她一直都這樣。大學的時候就這樣。考試前她會直接說『你這題一定不會，來我教你』。不繞彎。」

「很有效率。」

「她不是為了效率。她是——不想浪費時間在不重要的事情上。對她來說，客氣就是不重要的事。」

蘇立恆停了一下。筆尖離開了牆面。

「我以為只有工程師這樣。」

「精算師也這樣。」

停頓。窗外的巷子安靜了。路燈亮了。橙色的光從窗戶射進來，在白板上畫了一個長方形的光斑。

「她說你是白痴。」貝葉說。

「嗯。」

「你覺得呢？」

他把筆蓋扣上。轉過身。面對她的時候，他的臉一半在日光燈的白光裡，一半在路燈的橙光裡。兩種光在他的臉上交界。

「她說得對。」

「你承認？」

「一個人查三年是不夠理性的行為。理性的做法是找到一個可信任的團隊然後分工。但信任——」他停了。像在選一個精確的詞。「信任不是可以計算出來的。」

「所以你選了不信任。」

「我選了——等。等一個我能判斷的信號。」

「我沒蓋章。」

「你沒蓋章。」

同一句話。兩個人說的。像一個方程式的兩邊——左邊是原因，右邊是結果。中間連著一個等號。

---

她離開的時候是深夜。

走在永和的巷子裡。路燈把她的影子拉得很長。空氣裡有新店溪的水味——比傍晚的時候更重了。夜晚的河會呼吸得更用力。

她的手機震了。一條訊息。來自馬可瑜。

「theta 打呼打到把我吵醒了。所以我順便算了一下——如果先驗操控影響了保險定價，光是去年大安區一個區域，錯誤定價造成的潛在損失就超過三億。二十三個區域的話——我不敢算。」

第二條訊息。

「貝葉。這不是學術問題。這是錢。很多很多錢。」

第三條訊息。

「你記得吃飯。」

貝葉看著螢幕。三億。一個區域。一年。

她想起白奶奶的蔥油餅一張三十塊。想起林岳桌上的全家福照片——太太和兩個小孩，其中一個穿著幼稚園的制服。想起蘇立恆的瑜伽墊——一張大概兩百塊的瑜伽墊，用了三年。

有些東西不能用錢量化。但有些東西——必須用錢量化。因為這個世界是用錢運轉的。錢的方向決定了資源的方向。資源的方向決定了誰安全、誰不安全。而這些方向——現在被一組被操控的先驗控制著。

她關掉手機。

走在巷子裡。鞋底踩在柏油路面上的聲音很輕。每一步的間隔大約零點七秒。比她平常的步頻慢。比上週慢。比上上週慢。

她在減速。不是因為累。是因為她在想。

明天要回公司。要正常校準。正常蓋章。正常微笑。假裝她不知道任何事。假裝她的先驗沒有被更新過。假裝那面牆上的二十三個紅點不存在。假裝馬可瑜的三億不存在。假裝林岳的全家福照片還在他的桌上。

這大概是世界上最難的演算——假裝你的先驗沒有變，而你知道它已經變了。

巷口的紅綠燈在深夜切換成閃黃燈。一閃一閃的。沒有紅色、沒有綠色。只有黃色。

黃色在交通號誌裡的意思是：注意。

她注意了。

然後她過了馬路。走進夜裡。
