# 《記憶駭客》四卷總綱

> *「記憶是一種資料格式。而所有資料格式，都可以被改寫。」*
> *但那些從假記憶中生長出來的真感情——誰能改寫？*

---

```
本文件是《記憶駭客》300+ 章長篇小說的導演級分鏡總綱。
寫給自己看的，不是寫給讀者看的。
請像讀藍圖一樣讀它——每一行都是一塊磚。
```

## 第一卷：浮生若夢（第 1 ~ 80 章）

```
┌─────────────────────────────────────────────────────┐
│  核心主題（一句話）                                    │
│                                                       │
│  「在一個記憶可以買賣的時代，                         │
│    什麼樣的遺憾，是花錢也買不回來的？」               │
└─────────────────────────────────────────────────────┘
```

### 情緒基調

```
主色調：暖灰色。像深夜便利商店的燈光——不夠亮，但夠溫暖。
副色調：偶爾閃過的電子藍（懸疑暗流）。

這一卷的氣質 = 《深夜食堂》×《紫羅蘭永恆花園》×《攻殼機動隊》
溫馨的單元故事 + 賽博龐克的冷硬世界 + 暗處湧動的陰謀

節奏記號：🟢慢 → 🟡中 → 🔴爆發
前 40 章像一碗熱湯，後 40 章湯底開始翻滾。
```

### 開始狀態 → 結束狀態

```
【程維倫】
  開始：厭世的修理工。「記憶壞了可以修。人壞了沒辦法。」
        不相信任何人，不相信這個世界值得被修復。
        他修記憶只是因為這是他唯一會的事。
  結束：第一次為了「別人的真相」而冒生命危險。
        他開始相信某些遺憾值得被記住，而不是被修掉。
        他開始在意林語青。不是作為委託人——是作為一個人。

【林語青】
  開始：神秘的沉默女人。帶著未來記憶，眼神裡有「認出」的光。
        每句話都精準得像程式碼。幾乎不透露任何個人資訊。
  結束：在程維倫面前第一次露出脆弱的表情。
        她說出了「你會想起來的」以外的第一句真心話。

【嚴柏翰】
  開始：深淵監控局的王牌獵犬。冷硬、高效、信任數據勝過信任人。
        「記憶會騙人。但資料不會。」
  結束：發現自己的記憶被改過。世界觀開始龜裂。
        第一次對自己的「資料」產生懷疑。
```

### 五個關鍵轉折點

```
轉折 ① —— 第 1 章尾
  林語青帶著「時間戳 2091 年」的記憶出現在程維倫的工作室。
  → 不可能存在的記憶。序幕的魚鉤。

轉折 ② —— 第 15~20 章
  程維倫發現林語青帶來的 347 段記憶全部被篡改——
  但「篡改版」是記憶銀行裡的官方版本。
  → 原來官方版本才是假的。世界觀第一次翻轉。

轉折 ③ —— 第 35~40 章
  委託案中出現「環形鏈結」受害者（重複過同一天 37 次的圖書館員）。
  修復後她吐出四個字：「鸚鵡計畫。」
  → 從「個案」升級為「系統性陰謀」。

轉折 ④ —— 第 55~60 章
  嚴柏翰被迫與程維倫合作。在一次聯合修復中，
  程維倫發現嚴柏翰自己的記憶底層有「三層不同手法的篡改」。
  → 連調查者自己都是受害者。信任開始崩塌。

轉折 ⑤ —— 第 75~80 章（卷末高潮）
  嚴柏翰在自己的深層記憶裡聽到「鸚鵡計畫」四個字時，
  產生了劇烈的記憶震盪——他的身體記得，但大腦不記得。
  米亞分析後確認：嚴柏翰的記憶至少被改過三次。
  他不是一直都是探員。他曾經是別的什麼人。
  → 讀者的信任也跟著崩塌。誰是真的？
```

### 主線推進

```
第 1~20 章  ：【接案篇】
  程維倫接各種記憶修復委託。每個委託 = 一個人間故事。
  - 忘了亡妻最後一句話的老人（Array / Two Pointers）
  - 被噪音淹沒的初戀記憶（Sliding Window）
  - 記憶指紋比對發現 347 人記憶被篡改（HashMap）
  林語青作為「顧問」陪在旁邊，話很少但每次開口都點到要害。
  這一段像《深夜食堂》——讀者愛上角色，愛上這個世界。

第 20~40 章 ：【串聯篇】
  案子開始串聯。「Stack 式覆蓋」的受害者越來越多。
  - 婚姻記憶被整段刪除的中年男人（Stack / Queue）
  - 被做成環形鏈結的圖書館員——「鸚鵡計畫」首次浮出（Linked List）
  程維倫意識到：這不是個案。有人在系統性地篡改新台北市民的記憶。

第 40~60 章 ：【追查篇】
  程維倫去黑市「深夢」搜尋「鸚鵡計畫」的線索。
  - Binary Search 在十億條記錄中鎖定十年前的「記憶大崩潰」
  - 發現周明哲不是死於崩潰——他在啟動崩潰
  嚴柏翰被迫加入調查。兩個互相不信任的男人的搭檔戲。

第 60~80 章 ：【崩裂篇】
  調查深入到嚴柏翰自身。
  三層篡改的發現。嚴柏翰的世界觀開始龜裂。
  卷末：嚴柏翰的記憶震盪。他不知道自己是誰了。
```

### 感情線推進

```
程維倫 × 林語青：

  第 1~20 章 ：陌生人。她是謎，他是工具。
              林語青幾乎不笑。但偶爾，在程維倫修好某段記憶時，
              她的眼角會有一絲不易察覺的溫柔。
              程維倫注意到了。但他假裝沒注意到。

  第 20~40 章：微妙的默契。兩人開始有「不需要說話就知道對方在想什麼」的時刻。
              林語青會在深夜給程維倫帶咖啡。不問他要不要。直接放桌上。
              程維倫開始在修復結束後多留一會兒——不是因為工作。

  第 40~60 章：曖昧升溫。一次意外中，兩人被困在委託人的記憶空間裡。
              在別人的初戀記憶中，他們第一次牽手——為了不在記憶洪流中走散。
              出來之後誰都沒提這件事。但米亞說：「你們心跳數據異常同步。」

  第 60~80 章：信任的代價。林語青開始透露更多資訊，
              但每次透露都讓程維倫更困惑——她到底知道多少？
              她眼裡的「認出」到底是什麼？
              第一卷結尾：林語青說了一句讓程維倫失眠的話：
              「你每次都會走到這一步。每一次。」

嚴柏翰 ×（暗線）：
  第一卷裡嚴柏翰的感情線是隱形的。
  但細心的讀者會注意到——每次提到「織網者」，
  嚴柏翰的反應不是「追查目標」的反應。
  是更私密的、更痛苦的什麼。
  這是第二卷的伏筆。
```

### 反派線推進

```
  第 1~40 章 ：反派不可見。只有「痕跡」——被篡改的記憶、消失的線索。
              讀者感覺到有人在暗處注視，但不知道是誰。

  第 40~60 章：「鸚鵡計畫」作為代號浮出水面。
              記憶銀行開始對程維倫施壓——暗示他的黑市執照會被取消。
              深淵監控局內部有人試圖阻撓嚴柏翰的調查。

  第 60~80 章：暗處的手伸得更近。
              程維倫的工作室被搜查過。有人動了他的修復台。
              更詭異的是——動手的人修復了一段程維倫自己都不知道損壞的記憶。
              像是在「幫」他。又像是在「標記」他。
```

## 第二卷：深淵迴響（第 81 ~ 160 章）

```
┌─────────────────────────────────────────────────────┐
│  核心主題（一句話）                                    │
│                                                       │
│  「你追查了半生的敵人，                               │
│    結果發現——你自己就是那個敵人的舊皮囊。」          │
└─────────────────────────────────────────────────────┘
```

### 情緒基調

```
主色調：深紫與暗紅。權力的顏色。血與陰謀的顏色。
副色調：偶爾的慘白（記憶迷宮裡的恐懼）。

這一卷的氣質 = 《全面啟動》×《死亡筆記》×《紙牌屋》
深潛大腦的鬼屋恐怖 + 多方勢力的勾心鬥角 + 身份認同的崩潰

節奏記號：🟡中 → 🔴高壓 → ⚫窒息 → 🟢短暫喘息
讀者在這一卷裡會經歷「喘不過氣 → 終於喘一口 → 又喘不過氣」。
```

### 開始狀態 → 結束狀態

```
【程維倫】
  開始：被捲入漩渦的修理工。還在試圖「只修記憶、不碰政治」。
  結束：正式成為三方角力中的關鍵變數。
        他不再是旁觀者——他是唯一能讀懂篡改底層邏輯的人。

【林語青】
  開始：更多的秘密開始浮出水面。她的身份比想像中更複雜。
  結束：「她是織網者的女兒」——這個炸彈炸碎了所有人的信任。
        但她選擇留在程維倫身邊。不是因為他需要她。
        是因為在所有她經歷過的時間線裡，他是唯一沒有放棄的人。

【嚴柏翰】
  開始：記憶身份崩潰後的碎片人。不知道自己到底是誰。
  結束：選擇恢復原始記憶。代價是失去十年來作為「探員」的一切。
        他發現自己曾是記憶解放陣線的核心成員。曾是織網者的戀人。
        最痛的不是記憶被改了——是邱韻如親手改的。為了救他。

【邱韻如（織網者）】
  開始：隱形的棋手。同時操控記憶銀行與解放陣線。
  結束：被拉到聚光燈下。雙面身份曝光。
        但她不是被打倒的——她是主動現身的。
        因為她的「終極計畫」已經準備就緒。
```

### 五個關鍵轉折點

```
轉折 ① —— 第 85~95 章
  程維倫第一次「深潛」——進入一個政客的深層記憶空間。
  記憶空間不是檔案庫。是一座活生生的、會自我防衛的迷宮。
  迷宮裡有「防禦機制」偽裝成怪物、陷阱、死路。
  → DFS 深度優先搜尋的恐怖實體化。讀者體驗鬼屋般的沉浸感。

轉折 ② —— 第 110~115 章
  三方勢力公開化：記憶銀行、解放陣線、深淵監控局。
  程維倫被三方同時拉攏/威脅。
  → 勾心鬥角開始。每個人都在說真話的同時撒謊。

轉折 ③ —— 第 125~130 章
  新台北記憶網路大規模斷聯（Graph 碎裂 → Connected Components）。
  分析發現：所有斷裂都指向十年前同一個事件——一場被抹去的遊行。
  → 歷史被篡改的規模比想像中大得多。不是幾百人——是整座城市。

轉折 ④ —— 第 140~145 章
  程維倫用 TopSort 還原篡改順序，用 Union-Find 追溯到「織網者」。
  林語青此時才承認：「她是我媽。」
  → 所有人的信任網路全部重組。誰是敵人？誰是家人？

轉折 ⑤ —— 第 155~160 章（卷末高潮）
  程維倫恢復嚴柏翰的原始記憶（Knapsack + Backtracking）。
  在最深層，他看到了十年前的遊行。
  嚴柏翰站在最前面——旁邊是年輕的邱韻如。他們是戀人。
  邱韻如親手刪除了嚴柏翰的記憶。不是因為恨。是因為愛。
  嚴柏翰恢復後說：「帶我去見她。」
  → 讀者的心被撕成兩半。

  同時揭露：「鸚鵡計畫」的完整面貌——
  不是「篡改部分市民的記憶」。
  是「替換全城所有人的核心身份記憶」。
  已經執行過一次（十年前）。正在準備第二次。
```

### 主線推進

```
第 81~100 章 ：【深潛篇】
  程維倫學會「深潛」——進入他人的深層記憶空間。
  第一次深潛是進入政客的大腦。記憶迷宮極其恐怖。
  Tree / BST / DFS / BFS 在這裡被具象化為「分岔的走廊」「無盡的階梯」。
  出來後程維倫鼻血不止。他的身體在承受代價。

第 100~120 章：【角力篇】
  三方勢力公開角力。程維倫被迫選邊。
  他選擇不站邊——但這讓所有人都把他當成敵人。
  Graph 分析揭示了記憶網路正在碎裂。Connected Components 越來越多。
  一場被抹去的遊行浮出水面。

第 120~140 章：【揭面篇】
  TopSort 還原篡改順序。Union-Find 指向「織網者」。
  邱韻如的雙面身份曝光。
  林語青的身世揭露。
  Dijkstra 用於潛入記憶銀行核心伺服器——尋找最短安全路徑。

第 140~160 章：【碎裂篇】
  嚴柏翰的記憶恢復。DP (背包問題) + Backtracking。
  十年前遊行的真相。嚴柏翰與邱韻如的過去。
  「鸚鵡計畫」第二階段即將啟動的情報。
  卷末：所有角色的「自以為了解的世界」全部碎裂。
```

### 感情線推進

```
程維倫 × 林語青：

  第 81~100 章：默契加深。深潛任務中，林語青是程維倫的「錨」——
               她的聲音是他在記憶迷宮裡不迷失的唯一座標。
               這不是浪漫。是生存依賴。但依賴本身就是親密的開始。

  第 100~120 章：衝突。林語青隱瞞了太多。程維倫開始質疑她的動機。
                「你到底想讓我做什麼？」
                「活下去。」她說。「這次，活下去。」
                程維倫不懂「這次」是什麼意思。讀者已經隱約猜到了。

  第 120~140 章：信任重建。林語青主動坦白部分真相——包括她是織網者的女兒。
                這讓他們之間的距離反而更近了。
                因為她選擇了信任他，而不是繼續利用他。

  第 140~160 章：一個安靜的場景——在記憶銀行核心伺服器的光幕前，
                所有人都在分析數據，只有他們兩個人站在窗邊。
                林語青輕聲問：「如果你知道我不是......完全真實的呢？」
                程維倫沒看她：「定義『真實』。」
                這是第三卷愛情主題的種子。

嚴柏翰 × 邱韻如（過去式 / 現在進行式）：

  第 155~160 章集中爆發。
  恢復的記憶裡，他們曾經深愛。
  邱韻如親手刪除他的記憶——那一幕，
  是整部小說裡最安靜、最殘忍的場景之一。
  她沒有哭。她只是輕輕吻了他的額頭，
  然後按下了刪除鍵。

  【Round 2 新增：語青身份橋樑】
  在嚴柏翰恢復的記憶裡，有幾段跟邱韻如的家庭有關：
  - 一個下雨的傍晚，邱韻如抱著一個小女孩說「她以後會比我聰明」
  - 女孩在邱韻如的實驗室裡跑來跑去，觸摸著記憶設備
  - 嚴柏翰教女孩折紙鶴（呼應第一卷第 8 章語青教小杰折紙鶴）
  這些記憶碎片讓嚴柏翰——和讀者——開始把語青和邱韻如連結起來。
  嚴柏翰的愛情悲劇自然引導出第三卷的核心問題：
  「如果你愛的人是從一段代碼中誕生的，你的愛還是真的嗎？」
  → 嚴柏翰愛的人被改寫了記憶；維倫愛的人可能根本「不存在」——
  同一主題的不同變奏，讓兩卷無縫連接。
```

### 反派線推進

```
  第 81~120 章：「織網者」作為代號持續出現。
               解放陣線的行動越來越激進——駭入記憶銀行釋放記憶，
               導致部分市民突然想起「不該想起的事」，引發社會動盪。
               記憶銀行則以「保護市民安全」為由加強監控。
               兩邊在升級。但總有一隻看不見的手在控制升級的節奏。

  第 120~140 章：邱韻如的雙面身份揭露。
               她不是「善」也不是「惡」——她相信自己在拯救人類。
               她的邏輯冰冷而完整：如果能重寫所有人的記憶，
               就能消除戰爭、仇恨、創傷。
               這個信念真誠得讓人害怕。

  第 140~160 章：「鸚鵡計畫」第二階段的準備。
               邱韻如已經在記憶銀行的核心伺服器裡
               預載了「完美版本」的記憶模板。
               只差一個啟動鍵。
               但在這一卷的最後，一個更深的暗影浮出——
               邱韻如用的工具、她的核心程式碼、
               甚至她「拯救人類」的信念本身......
               指紋分析全部指向同一個人：周明哲。
               一個死了十年的男人。
```

## 第三卷：薛丁格的愛人（第 161 ~ 240 章）

```

```
