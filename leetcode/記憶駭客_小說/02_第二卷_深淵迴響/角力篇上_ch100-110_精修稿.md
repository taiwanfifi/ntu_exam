```
ch104 深潛    ★   震撼（記憶在但邊被刪了 = 孤島）

ch106 幽靈    ★   恐懼（同一個操縱者）
```

### ch103 消失的母親——BFS 的母愛載體

**委託場景**：

```
姊弟走進工作室。
姊姊牽著弟弟的手。弟弟不說話。

「我們的媽媽——三天前——」
姊姊停了一下。深吸一口氣。

「她看著我們，問我們是誰。」

不是失憶。
她的其他記憶完好。
她記得自己的名字、工作、住址。
她記得自己有兩個孩子。

她就是不認識站在面前的這兩個人。
```

**演算法：BFS（母親記憶的廣度搜索）**

```
A 線：在母親的記憶網絡中廣度搜尋「親子」關聯節點
     從「孩子」這個標籤開始 → 第一圈：出生記憶 ✓（存在）
     → 第二圈：養育記憶 ✓（存在）
     → 第三圈：日常互動 ✓（存在）

     所有記憶都在。
     但維倫找不到任何一條「通往它們的路」。
     像一座座島嶼，中間的橋全被炸了。

B 線：母親記得「我有孩子」這個事實。
     但她的大腦找不到那些記憶。
     就像你知道鑰匙在家裡，但所有房間的門都鎖了。

     愛的記憶還在。但通往愛的路斷了。

同步點：維倫用 BFS 一圈一圈向外搜索時——
       每找到一個「存在但無法到達」的記憶，
       就像在水面上看到波紋但摸不到水底的石頭。
       A 線的「搜索到了但無法連接」= B 線的「愛在但觸不到」
```

**米亞比喻**：「她的記憶像一座群島。每座島都在——孩子的笑聲、第一次走路、發燒時抱著她們的夜晚。但島跟島之間的橋全部消失了。她站在主島上，看得到對面有島，但游不過去。」

**寫作指令**：
- 姊姊的反應是「理性崩潰」——她帶了一疊醫院檢查報告、腦掃描、問診紀錄
- 弟弟「不說話」= 他每天去看媽媽。媽媽每天問他「你是誰」。他每天回答。每天心碎一次
- ≤2,800 字。這是 Vol 2 最催淚的委託案開場

### ch104 節點與邊——Graph 最核心的概念

**深潛場景**：

```
維倫深潛進母親的記憶空間。

第一個發現：
所有關於孩子的記憶——節點——完好無損。
出生。第一次走路。第一天上學。發燒的夜晚。

第二個發現：
所有連結邊——全部被切斷。
不是損壞。不是衰退。
是精準的切割。

「節點完好。邊被刪除。」
米亞的聲音帶著一種維倫從未聽過的困惑。
「我從來沒見過這種損壞模式。
 正常的記憶損壞是節點退化——邊會跟著弱化。
 但這裡——邊被單獨切掉了。
 像是有人懂得這棵樹的結構，然後只切橋不砍島。」
```

**演算法：Graph（節點 vs 邊的分離）**

```
A 線：記憶是節點。關係是邊。
     節點在，代表記憶存在。
     邊在，代表你「找得到」那段記憶。

     切掉邊 = 記憶還在，但你永遠找不到它。
     這比刪除記憶更殘忍——
     因為某種意義上，你知道它在那裡。
     你只是碰不到。

B 線：親情不是一個記憶——是無數條連結。
     你記得孩子，不是因為一段特定的畫面。
     是因為每天早上的擁抱、每頓晚餐的交談、
     每次生病時的擔心——
     這些「邊」織成了一張網。
     網沒了，節點就成了孤島。

同步點：維倫站在母親記憶空間的正中間——
       四周是完好的記憶節點，像星星一樣散布。
       但它們之間沒有任何連線。
       一座座孤島。
       「這就是 Graph。」他想。
       「連接比存在更重要。」
```

**寫作指令**：
- 「節點完好但邊被刪除」= Vol 2 最核心的 Graph 概念。這個意象必須讓讀者印象深刻
- 用「星星」或「島嶼」做視覺——讀者一看就懂
- 米亞的「困惑」= 她是 AI，她見過很多損壞模式。這是新的 = 讀者也跟著困惑
- ≤2,500 字

### ch105 重新連線——修復的不完美

**核心場景**：

```
維倫開始一條一條重建被切斷的連結。

第一條：「出生」← →「母親」。連上了。
第二條：「第一次走路」← →「母親」。連上了。
第三條到第十八條。一條一條。慢慢地。

然後他碰到了三條特殊的邊。

這三條——無論如何都接不回去。
不是技術問題。是它們已經被「替換」了。
原本指向姊弟的連結——
現在指向一個陌生人。

一個不存在的人。
一個「幽靈帳號」。

米亞查了帳號編碼：
「跟第一卷陳美玲案的 #50742 格式完全一致。」
```

**演算法：Graph（邊的重建與異常邊檢測）**

```
A 線：重建邊的過程 = 在斷裂的圖上一條一條畫線
     大多數邊可以重建（因為兩端節點都在）
     但被「替換」的邊 = 它們已經指向了別的節點
     無法覆蓋回來——除非先刪掉異常邊

B 線：修復不是完美的。
     有些關係斷了就是斷了。
     你可以重建大部分。
     但有三條裂縫——永遠在那裡。
     殘缺比完全失敗更痛。

同步點：維倫成功重建了 18 條邊中的 15 條。
       母親認出了孩子。哭著抱住他們。
       但有三段記憶——她永遠無法自然想起。
       那三段是最好的三段：
       姊姊第一次叫「媽媽」。
       弟弟學騎腳踏車。
       一家四口的最後一次旅行。
```

**寫作指令**：
- 「殘缺比完全失敗更痛」= 本案的核心情感。母親會記得孩子。但最好的三段永遠消失了
- 被替換的三條邊指向幽靈帳號 #50742 = 連接 Vol 1 和 Vol 2 的偵探線索
- 修復成功時母親的擁抱 ≤100 字。不要過度渲染。讓擁抱本身說話
- ≤2,500 字

## 參、ch107-108.5 壓力與呼吸——三連節奏

### ch107 四十七個家庭——城市危機升級

```
嚴柏翰帶來情報：

「不只是你那個案子。
 過去兩個月。
 新台北至少四十七個家庭。
 同樣的症狀。
 父母忘記孩子。
 從來沒有反過來。」

四十七個家庭。
四十七對父母不認識自己的孩子。
四十七組孩子每天問「媽媽你記得我嗎」。

維倫忽然覺得——
他面前的這個案子，
不是一個家庭的悲劇。
是一座城市的癌症。
```

**寫作指令**：
- 「從來沒有反過來」= 一個不自然的模式。自然損壞是隨機的。這是單向的 = 人為的
- 數字「四十七」要由嚴柏翰說出。他是探員。他有數據
- 維倫的反應不是「震驚」——是「安靜的恐懼」。比震驚更重
- ≤2,000 字

```
嚴柏翰的調查被局長叫停。
理由：「可能引發社會恐慌」。

「我越來越不確定——
```

### ch108.5 最難的案子——★★ 橘子貓呼吸章

**詳細場景設計**：

```
前台：老太太 + 橘子貓

「牠會說話。」
「……貓不會說話。」
「你聽。」

播放錄音。
是打鼾聲。
規律的、溫暖的打鼾聲。

「牠明明在說我愛你。」

米亞分析波形：
「頻率跟『我愛你』的語譯確實有 12% 的相似度。」

嚴柏翰站在旁邊看完全程。
嘴角動了兩毫米。
語青偷偷拍了下來。

維倫沒收費。
但他幫老太太把橘子打鼾的錄音調了一下相位——
至少聽起來更像「我愛你」了。
```

```
暗台：邱韻如暗線 #1

老太太離開時在門口撞到一個穿記憶銀行技術員制服的女人。
女人側身讓路。微笑。「不好意思。」
```

- 暗線只有三行。語青皺眉但不說。讀者在 ch132 回翻時才會注意到

- ❌ 暗線不要超過三行。不要讓任何角色表示在意
- ≤2,200 字（含暗線）

## 肆、ch109-110 恐懼升級——Graph/BFS 的城市版

### ch109 語青的地圖

```
語青攤開一張地圖。

不是普通的地圖。
是她秘密繪製了數月的——
新台北「記憶汙染分布圖」。

四十七個斷聯家庭的位置在地圖上形成了——
一個正在擴散的同心圓。

米亞：「每個斷聯區域是一個獨立的連通分量。
 它們之間沒有邊——完全隔離。
 但每個分量內部的家庭彼此相鄰。」

語青：「不是隨機的。是有規律的。
 從中心向外。一圈一圈。」

維倫：「中心在哪裡？」

語青用手指點了一下地圖。
那個位置所有人都認得。
```

**演算法：Graph/Connected Components**

```
A 線：47 個斷聯家庭形成數個獨立連通分量
     每個分量內部的家庭在物理上相鄰（同一社區）
     分量之間完全隔離（不同社區）
     但所有分量在地圖上形成同心圓 = 有中心源

B 線：災難不是一次性的。
     它是一圈一圈擴散的。
     像投進水裡的石頭——漣漪從中心向外。
     你在第一圈的時候以為是巧合。
     到第四十七圈的時候才知道——水面下有怪物。

同步點：米亞計算連通分量時 → 維倫看到了漣漪的模式
       A 線的「獨立分量形成同心圓」= B 線的「有計劃的擴散」
```

### ch110 圓心

```
擴散圓的中心。
精確地落在——
記憶銀行的中央伺服器位址。

有人在從網絡核心向外釋放記憶斷聯病毒。
以 BFS 的模式。
逐層感染。

維倫：「是銀行自己在做這件事？」
語青：「或者——有人在銀行內部做這件事。」
嚴柏翰：「有人在利用銀行的基礎設施。」
米亞：「三種可能。目前數據無法區分。」
```

**演算法：BFS（病毒傳播模式）**

```
A 線：病毒從圓心（銀行伺服器）以 BFS 模式向外傳播
     第一層：伺服器直接連接的家庭
     第二層：第一層鄰居
     第三層……
     逐層感染。不跳層。不隨機。

B 線：恐懼也是這樣傳播的。
     從一個點開始。然後一圈一圈。
     你在第一圈的時候不害怕。
     你在第十圈的時候不害怕。
     你在看到圓心的時候——才真正害怕。

同步點：維倫知道圓心在銀行的瞬間——
       「有人在從城市的心臟釋放毒素。」
       A 線的「BFS 傳播」= B 線的「從核心腐爛」
```

**寫作指令**：
- ch109 和 ch110 是連續的恐懼升級——不要在中間放呼吸
- 「中心在銀行」的揭露用語青的手指 = 動作比語言更有力
- ch110 的三人三種反應（維倫問、語青推理、嚴柏翰嚴謹、米亞客觀）= 四人各有位置
- ≤2,000 字（兩章合計 ≤4,000 字）
