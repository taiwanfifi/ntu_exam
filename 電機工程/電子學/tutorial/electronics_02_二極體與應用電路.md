# 電子學（二）：二極體與應用電路

## 🔰 本章基礎觀念（零基礎必讀）

### 二極體是什麼？——最簡單的半導體元件

上一章我們學了 PN 接面，而**二極體（Diode）**就是一個被封裝起來的 PN 接面。它有一個非常重要的特性：**電流只能往一個方向流**。

想像一個**單向閥門**：
- 水從左往右可以通過（**順偏 Forward Bias**）
- 水從右往左被擋住（**逆偏 Reverse Bias**）

二極體的兩個端子：
- **陽極（Anode, A）**：接 P 型那邊
- **陰極（Cathode, K）**：接 N 型那邊

> **記憶口訣**：電流從陽極（A）流向陰極（K），「A到K」= 順向導通。

## 一、二極體的 I-V 特性

### 1.1 Shockley 方程式

$$I = I_S \left( e^{V/(nV_T)} - 1 \right)$$

其中：
- I：二極體電流
- IS：逆向飽和電流（Reverse Saturation Current），矽二極體約 10⁻¹⁴ ~ 10⁻¹⁵ A
- V：二極體兩端電壓
- n：理想因子（Ideality Factor），n = 1 ~ 2
- VT：熱電壓 = kT/q ≈ 26 mV（室溫）

### 1.2 各區域分析

| 區域 | 條件 | 行為 |
|------|------|------|
| 順偏導通 | V > 0 且夠大 | I ≈ IS × e^(V/nVT)，電流指數增長 |
| 逆偏 | V < 0 | I ≈ -IS ≈ 0（極小漏電流） |
| 崩潰 | V < -VBR | 大量逆向電流（齊納崩潰或雪崩崩潰） |

### 數值例題 2.1：Shockley 方程計算

> **題目**：矽二極體，IS = 10⁻¹⁴ A，n = 1，室溫下 V = 0.7V，求電流 I。
>
> **解答**：
> I = IS(e^(V/VT) - 1)
> = 10⁻¹⁴ × (e^(0.7/0.026) - 1)
> = 10⁻¹⁴ × (e^26.92 - 1)
> = 10⁻¹⁴ × 4.84 × 10¹¹
> ≈ 4.84 × 10⁻³ A ≈ 4.84 mA
>
> **結果**：I ≈ 4.84 mA

### 數值例題 2.2：求二極體電壓

> **題目**：同一個二極體（IS = 10⁻¹⁴ A, n = 1），若電流 I = 1 mA，求二極體電壓。
>
> **解答**：
> 因為 I >> IS，可以忽略 -1：
> V = nVT × ln(I/IS)
> = 0.026 × ln(10⁻³/10⁻¹⁴)
> = 0.026 × ln(10¹¹)
> = 0.026 × 25.33
> ≈ 0.659 V
>
> **結果**：V ≈ 0.659 V（接近常說的 0.7V）

## 二、三種二極體模型

在電路分析中，我們根據需要的精確度選擇不同的二極體模型：

### 2.1 理想二極體模型（Ideal Diode Model）

- 順偏：短路（V = 0, I > 0）
- 逆偏：斷路（I = 0, V < 0）

**適用時機**：快速估算電路行為，定性分析

### 2.2 恆壓降模型（Constant Voltage Drop Model）

- 順偏：V = VD（矽約 0.7V，鍺約 0.3V）
- 逆偏：斷路

**適用時機**：大多數工程計算，考試最常用

### 2.3 指數模型（Exponential Model）

使用完整的 Shockley 方程式。

**適用時機**：精確分析、需要考慮溫度效應

### 模型比較

| 模型 | 順偏 | 逆偏 | 精確度 | 複雜度 |
|------|------|------|--------|--------|
| 理想 | 短路 | 斷路 | 低 | 最簡單 |
| 恆壓降 | V = 0.7V | 斷路 | 中 | 簡單 |
| 指數 | Shockley方程 | I = -IS | 高 | 需迭代求解 |

### 數值例題 2.3：三種模型比較

> **題目**：電路中，VCC = 5V 串聯 R = 1kΩ 和一個矽二極體（IS = 10⁻¹⁴A），用三種模型分別求二極體電流。
>
> **解答**：
>
> **(a) 理想模型**：
> VD = 0，I = VCC/R = 5/1000 = 5 mA
>
> **(b) 恆壓降模型**：
> VD = 0.7V，I = (VCC - VD)/R = (5 - 0.7)/1000 = 4.3 mA
>
> **(c) 指數模型**（需迭代）：
> VCC = ID × R + VD
> 5 = ID × 1000 + 0.026 × ln(ID/10⁻¹⁴)
>
> 以恆壓降結果為初始猜測：
> ID = 4.3 mA → VD = 0.026 × ln(4.3×10⁻³/10⁻¹⁴) = 0.026 × 26.69 = 0.694V
> → ID = (5 - 0.694)/1000 = 4.306 mA（已收斂）
>
> **比較**：理想模型 5mA，恆壓降模型 4.3mA，指數模型 4.306mA

## 三、小信號模型

### 3.1 為什麼需要小信號模型？

當二極體工作在某個直流操作點（Q點），加上一個小的交流信號時，我們可以用**線性近似**來分析。

### 3.2 小信號電阻 rd

$$r_d = \frac{nV_T}{I_D}$$

其中 ID 是直流工作點電流。

**物理意義**：在直流操作點附近，二極體表現得像一個小電阻。

### 數值例題 2.4：小信號電阻

> **題目**：二極體工作在 ID = 1 mA，n = 1，求小信號電阻 rd。
>
> **解答**：
> rd = nVT/ID = 1 × 0.026 / 0.001 = 26 Ω
>
> **結果**：rd = 26 Ω
>
> **規律**：ID = 1 mA → rd = 26 Ω；ID = 0.1 mA → rd = 260 Ω；ID = 10 mA → rd = 2.6 Ω

## 四、半波整流器（Half-Wave Rectifier）

### 4.1 基本電路

輸入交流信號 vi = Vpsin(ωt)，串聯一個二極體和負載電阻 RL。

### 4.2 工作原理

- **正半週**（vi > 0.7V）：二極體導通，vo ≈ vi - 0.7V
- **負半週**（vi < 0.7V）：二極體截止，vo = 0

### 4.3 重要參數

- **峰值輸出電壓**：Vo,peak = Vp - 0.7（恆壓降模型）
- **直流平均值**：Vdc = Vo,peak / π
- **PIV（Peak Inverse Voltage）**：二極體承受的最大逆向電壓 = Vp

### 數值例題 2.5：半波整流器

> **題目**：vi = 10sin(ωt) V，二極體用恆壓降模型（0.7V），RL = 1kΩ，
> 求峰值輸出電壓和直流平均值。
>
> **解答**：
> 峰值輸出 = 10 - 0.7 = 9.3V
> 直流平均值 = 9.3/π ≈ 2.96V
> PIV = 10V
>
> **結果**：峰值 9.3V，直流平均 2.96V

---

## 五、全波整流器（Full-Wave Rectifier）

### 5.1 中心抽頭式（Center-Tapped）

使用中心抽頭變壓器 + 2個二極體。

- 正負半週都有輸出
- **Vdc = 2Vo,peak / π**
- PIV = 2Vp - 0.7

### 5.2 橋式整流器（Bridge Rectifier）

使用4個二極體排列成橋式結構。

- 正半週：D1、D3導通
- 負半週：D2、D4導通
- 輸出峰值 = Vp - 2 × 0.7 = Vp - 1.4（兩個二極體壓降）
- PIV = Vp - 0.7

### 5.3 加濾波電容

在 RL 兩端並聯一個電容 C，可以減少輸出的漣波（ripple）。

**漣波電壓**：

$$V_r \approx \frac{V_p}{fRC}$$

其中 f 是漣波頻率（半波為 f，全波為 2f）。

### 數值例題 2.6：橋式整流器含濾波

> **題目**：輸入 vi = 15sin(377t) V（60Hz），橋式整流器 + C = 100μF，RL = 1kΩ。
> 求輸出直流電壓和漣波電壓（用恆壓降模型）。
>
> **解答**：
> 峰值輸出 = 15 - 1.4 = 13.6V
> 漣波頻率 = 2 × 60 = 120 Hz（全波）
> Vr ≈ Vp / (f × R × C) = 13.6 / (120 × 1000 × 100 × 10⁻⁶) = 13.6/12 ≈ 1.13V
> 直流輸出 ≈ 峰值 - Vr/2 = 13.6 - 0.567 ≈ 13.03V
>
> **結果**：直流輸出 ≈ 13.0V，漣波 ≈ 1.13V peak-to-peak

## 六、穩壓電路（齊納二極體）

### 6.1 齊納二極體（Zener Diode）

特殊的二極體，設計成在特定的**逆向崩潰電壓（VZ）**下工作。

**兩種崩潰機制**：
- **齊納崩潰（Zener Breakdown）**：VZ < 5V，重摻雜造成
- **雪崩崩潰（Avalanche Breakdown）**：VZ > 7V，碰撞游離造成

### 6.2 齊納穩壓器電路

基本結構：VS 串聯 RS，並聯齊納二極體和 RL。

**設計條件**：
1. 齊納要導通：IS(min) > IZ(knee)
2. 電流限制：不超過齊納的額定功率

**穩壓範圍**：

$$I_Z = \frac{V_S - V_Z}{R_S} - \frac{V_Z}{R_L}$$

### 數值例題 2.7：齊納穩壓器設計

> **題目**：VS = 15V，VZ = 5.1V，RS = 470Ω，RL = 1kΩ。
> 求齊納電流和負載電壓。
>
> **解答**：
> 假設齊納導通，VL = VZ = 5.1V
> IL = VL/RL = 5.1/1000 = 5.1 mA
> IS = (VS - VZ)/RS = (15 - 5.1)/470 = 9.9/470 = 21.06 mA
> IZ = IS - IL = 21.06 - 5.1 = 15.96 mA
>
> IZ > 0，所以齊納確實導通。
> **結果**：VL = 5.1V，IZ = 15.96 mA

## 七、限幅器（Clipper）

### 7.1 基本概念

限幅器用來「剪掉」信號超過某個電壓的部分。

### 7.2 常見類型

**正向限幅器**（Positive Clipper）：
- 二極體正端接輸出，負端接 VREF
- 當 vi > VREF + 0.7V 時，輸出被限制在 VREF + 0.7V

**負向限幅器**（Negative Clipper）：
- 把二極體方向反過來
- 限制信號的負向部分

**雙向限幅器**：
- 兩個反向並聯的二極體
- 限制正負兩邊

### 數值例題 2.8：限幅器分析

> **題目**：vi = 10sin(ωt) V，二極體陽極接到輸出節點，陰極接到 3V 電源。
> 串聯電阻 R = 1kΩ。用恆壓降模型分析輸出。
>
> **解答**：
> 二極體導通條件：vi > 3 + 0.7 = 3.7V
> - 當 vi < 3.7V：二極體截止，vo = vi
> - 當 vi > 3.7V：二極體導通，vo = 3.7V
>
> **結果**：輸出在 3.7V 處被限幅，負半週正常通過

## 八、箝位器（Clamper）

### 8.1 基本概念

箝位器（也叫直流恢復器 DC Restorer）將信號的基準電位平移。

### 8.2 工作原理

基本電路：串聯電容 C + 並聯二極體 D。

1. 第一個負半週：二極體導通，電容充電到 Vp - 0.7
2. 之後：電容上的電壓加到輸入信號上，整個波形往上平移

### 數值例題 2.9：正向箝位器

> **題目**：vi = 5sin(ωt) V，電容夠大（不會放電），二極體陽極接地、陰極接輸出。
> 求穩態輸出波形。
>
> **解答**：
> 第一個負半週，vi = -5V 時二極體導通
> 電容充電到 VC = 5 - 0.7 = 4.3V（左正右負）
>
> 之後二極體截止，vo = vi + VC
> vo = 5sin(ωt) + 4.3
>
> 波形範圍：
> - 最大值：5 + 4.3 = 9.3V
> - 最小值：-5 + 4.3 = -0.7V（= -VD）
>
> **結果**：整個正弦波往上平移，最小值箝位在 -0.7V

## 題型鑑別

| 看到什麼條件 | 用什麼方法 |
|-------------|-----------|
| 「求二極體電流/電壓」、簡單電路 | 先假設導通/截止，用恆壓降模型驗算 |
| 「求精確電壓」或需要迭代 | 用指數模型 + 迭代法 |
| 「小信號」、「交流等效」 | 求rd = nVT/ID，用小信號等效電路 |
| 「整流器」、有交流輸入 | 畫出正負半週的二極體狀態 |
| 「穩壓」、齊納二極體 | 先假設齊納導通，檢查 IZ > 0 |
| 「限幅器」 | 找到二極體導通門檻，分段分析 |
| 「箝位器」、有電容 | 先分析電容充電，再疊加 |

1. Shockley 方程中，IS 代表什麼？它的典型值大約是多少？
2. 恆壓降模型假設導通電壓是多少？為什麼是這個值？
3. 半波整流和全波整流的漣波頻率有什麼不同？
4. 齊納二極體工作在順偏還是逆偏？
5. 限幅器和箝位器的功能有什麼不同？

<details>
<summary>點擊查看答案</summary>

1. **IS 是逆向飽和電流**，代表逆偏時流過的極小電流。矽二極體典型值約 10⁻¹⁴ ~ 10⁻¹⁵ A，與接面面積和溫度有關。

2. **矽二極體約 0.7V。** 這是因為在典型電流（約 1mA）下，由 V = VT × ln(I/IS) 計算得來。鍺二極體約 0.3V。

3. **半波整流的漣波頻率等於輸入頻率 f，全波整流的漣波頻率是 2f。** 因為全波整流每半週都有輸出，所以漣波頻率加倍，漣波更小。

4. **逆偏。** 齊納二極體是設計在逆向崩潰區工作的，崩潰電壓 VZ 就是穩壓值。

5. **限幅器「剪掉」信號超過門檻的部分（改變波形形狀），箝位器「平移」整個信號的直流電位（不改變波形形狀）。**
