# 貳、Vol 1 → Vol 2 的弧線銜接

## 一、Vol 1 的探員形象：為反轉埋種子

Vol 1 的嚴柏翰必須建立得夠「真」，才能讓 Vol 2 的反轉夠痛。

| Vol 1 特徵 | 設計意圖 | Vol 2 如何反轉 |
|-----------|---------|--------------|
| 黑風衣、疤痕、權力感 | 建立「獵犬」的視覺印記 | ch140 遊行錄像：穿白大褂的研究生走在最前面 |
| 正直但不反叛 | 探員人格的核心：忠誠 | ch108「我越來越不確定自己效忠的機構」= 植入忠誠開始裂開 |
| 與維倫的信任 | 全書最純粹的男性友誼 | ch149「你是這十年裡唯一真的東西」= 友誼是唯一跨越真假的存在 |
| 左眼疤痕的「官方故事」 | 「追捕時被砍」 | ch157 真相：遊行中被鎮暴盾牌劃傷 |
| 職業性冷靜 | 讀者以為這是天性 | ch152 記憶恢復：24歲的他會笑、會哭、會把公式寫錯 |

**銜接核心**：Vol 1 結束時讀者對嚴柏翰的認知是「一個正直的探員，知道了一些可怕的真相，選擇沉默」。Vol 2 要做的是讓這個認知一層一層剝落——他不是「選擇沉默」，他是「被沉默」的。

## 二、暗線回收：茉莉花的四次進化

茉莉花是嚴柏翰弧線中最重要的單一符號。它的每次出現都比上一次更沉重。

```
                        心率 150。茉莉花的味道             （神秘、悲傷）

                        嚴柏翰沉默                         茉莉花從「個人記憶」
```

- ch35/73：味道（感官層）
- ch118：別人的記憶（社會層）
- ch149：無意識的選擇（意志層）
- ch152：具體的人（真相層）

讀者在 ch152 會經歷一個瞬間的「收束」——前三次的茉莉花全部在這一刻匯聚成一個人的臉。

## 三、Vol 1 ch80 → Vol 2 ch98 的缺席設計

嚴柏翰在 Vol 1 結尾（ch73-80）沉默、碎裂、缺席。Vol 2 前 17 章（ch81-97）他不出場。

**缺席的敘事功能**：
1. 讓讀者的焦慮累積——「他去哪了？他還好嗎？」
2. 深潛篇的恐怖節奏不能被嚴柏翰的情感線分散
3. ch98 回歸時的「聲音先到人後到」才有衝擊力

**缺席期間的暗示**（不超過 3 處）：
- ch83 阿嬤問「你身邊是不是少了一個人」→ 暗示嚴柏翰曾來過

# 參、ch147-149 背包問題精修

> **段落級寫法指令**
> **品質目標**：★★★★★ —— Vol 2 最佳教學場景 + 最佳情感場景的合體
> **核心問題**：你的大腦只能裝這麼多。你選擇帶走什麼？

---

## ch147 容量極限的設定呈現

### 技術設定

```
嚴柏翰的大腦神經容量：
  當前佔用 = 十年探員記憶（100%）
  需要加載 = 被覆蓋的原始記憶（34%）
  總需求 = 134%
  超出極限 = 34%

結論：不可能全部保留。必須釋放至少 34% 的空間。
```

### 呈現方式

**不要用數字開場**。用身體開場。

```
寫法指令：

第一段：維倫在修復台旁邊看嚴柏翰的腦神經掃描。
        畫面上的顏色——紅色 = 飽和，藍色 = 空閒。
        「全紅。」維倫說。「沒有一塊是空的。」

第二段：米亞翻譯成數字。
        「如果把你的大腦想像成一個背包——」
        嚴柏翰：「已經裝滿了。」
        米亞：「不只裝滿。你要放進去的東西，比你的背包大 34%。」

第三段：沉默。
        嚴柏翰看著掃描畫面。
        他看了很久。
        然後問了一個讓維倫心碎的問題：
        「我可以選擇丟掉什麼嗎？」

        這不是背包問題的技術起點。
        這是一個人對自己十年人生說「可以放棄」的起點。
```

### DP 教學融入（A線）

```
米亞的解釋（嵌入對話，不單獨成段）：

「0/1 背包。每段記憶是一個物品。
 有重量——佔多少神經容量。
 有價值——對你的人格有多重要。
 你只能選『全部保留』或『全部放棄』。
 不能只留一半。」

維倫補充（看著嚴柏翰）：
「這就是為什麼這個問題叫背包問題。
 背包只有這麼大。
 你必須決定——帶走什麼，留下什麼。」

嚴柏翰的回應：
（沉默。看著掃描圖。）
「開始吧。」
```

**教學原則**：ch147 只建立問題。不給答案。不給選擇清單。讀者在這一章結束時應該感受到的是「天啊，他要怎麼選」的焦慮。

## ch148 逐段秤量

### 記憶清單的呈現順序

順序經過精心設計：從「容易放棄的」到「越來越難放棄的」。

| 順序 | 記憶段落 | 重量 | 價值 | 嚴柏翰的決定 | 身體反應 |
|------|---------|------|------|-------------|---------|
| 1 | 榮譽獎章（表揚、升職、勳章） | 中 | 外界認可 | 放棄 | 無。「這些從來不是為了我自己」|
| 2 | 十年破案紀錄（所有案件細節） | 大 | 職業成就 | 放棄 | 微微搖頭。「這些……不是我的故事」|
| 3 | 審訊技巧/辦案方法論 | 小 | 可轉移工具 | ★保留 | 「工具可以帶走。」語氣平靜 |
| 4 | 獨處的夜晚（失眠、窗外的雨、無法命名的空洞） | 小但無法釋放 | 不明 | 放棄（被動） | 米亞：「這些記憶已經跟你的神經基底層融合了。放棄它們不會釋放空間——但也不會佔用額外容量。」嚴柏翰：「所以它們留著。不是因為我選擇。」|
| 5 | 同事情誼（一起出任務的人、慶功宴、默契） | 中 | 人際連結 | 放棄 | 手指微微收緊。「他們認識的嚴柏翰……本來就不存在。」|
| 6 | 跟維倫的友誼 | 中 | 唯一真的人際連結 | ★★保留 | 「你是這十年裡唯一真的東西。」|
| 7 | 茉莉花的味道 | 幾乎為零 | 無法歸類 | ★★★保留 | 手不肯放開（見下文詳述）|

### 維倫在秤量中的角色

**維倫不是旁觀者。他是秤。**

```
寫法指令：

維倫坐在修復台旁邊。
他的工作是把每段記憶從嚴柏翰的神經網絡中提取出來，
讓嚴柏翰「看到」它，評估它，然後決定去留。

每段記憶被提取時，修復台上方會浮現一個半透明的畫面——
像一段無聲電影。

維倫提取。
嚴柏翰觀看。
米亞計算剩餘容量。

三個人的分工：
  維倫 = 手（操作）
  嚴柏翰 = 眼（判斷）
  米亞 = 腦（計算）
```

### 嚴柏翰的身體反應遞進

```
秤量 #1（榮譽）：無反應。乾淨利落。他不在乎這些。
秤量 #2（破案）：搖頭。有一點惋惜，但不多。
秤量 #3（方法論）：第一次猶豫——但只猶豫了兩秒。「留。」
秤量 #4（獨處）：身體僵硬。這些夜晚他不想要，但它們黏在他身上。
秤量 #5（同事）：手指收緊。這是第一次需要放棄「人」。
秤量 #6（維倫）：手停在半空中。三秒。然後握拳。「留。」
秤量 #7（茉莉花）：——（見 ch149）
```

### DP 教學融入（A線）

```
嵌入在秤量過程中，米亞的即時計算：

秤量 #1 後：
  米亞：「釋放 12%。剩餘需釋放 22%。」

秤量 #2 後：
  米亞：「釋放 18%。累計釋放 30%。剩餘需釋放 4%。」

秤量 #3（保留方法論）：
  米亞：「方法論佔 3%。保留。剩餘需釋放 7%。」

  維倫：「為什麼保留這個？」
  嚴柏翰：「工具可以帶走。身份會變，但能力不會。」

  ——這句話是全書對「方法論 vs 知識」的核心比喻。
  讀者在這裡學到的不只是 DP，
  而是「什麼是可遷移的、什麼是綁定在特定身份上的」。

秤量 #5 後：
  米亞：「累計釋放 38%。已超過 34% 門檻。
         理論上可以開始恢復了。」

  嚴柏翰：「等一下。」

  他還沒有決定最後兩樣。
  在 DP 的邏輯裡，他已經「解決了問題」——
  容量夠了，可以停了。

  但他不停。因為問題從來不是「夠不夠」。
  問題是「什麼值得帶走」。

  這是 A 線（DP 最優解）和 B 線（人生選擇）的分離點：
  演算法說「已經最優了」。
  人說「我還沒決定完」。
```

## ch149 三樣東西的揭露

### 揭露順序的設計邏輯

```
第一樣揭露：方法論（ch148 已決定）
  → 理性。讀者點頭。「合理。」

第二樣揭露：友誼
  → 情感。讀者心軟。「當然要留。」

第三樣揭露：茉莉花
  → 直覺。讀者心碎。「他自己都不知道為什麼。」

三樣的遞進：理性 → 情感 → 直覺
讀者跟嚴柏翰一起從「腦」滑向「心」再滑向「身體」。
```

### 「友誼」的揭露寫法

```
寫法指令（段落級）：

維倫提取出最後一段探員記憶——
跟維倫第一次見面。

畫面：一個下雨的夜晚。嚴柏翰站在維倫工作室的門口。
他不是來修復記憶的。他是來調查維倫的。

但那天晚上維倫給了他一杯水。

不是什麼大事。就是一杯水。
一個被調查的人，給調查他的人倒了一杯水。

嚴柏翰看完這段記憶。
沉默了很久。

「留。」

維倫：「這段記憶很大。佔 8%。」

嚴柏翰看著維倫。

「你是這十年裡唯一真的東西。」

——然後不要接維倫的反應。
讓沉默留著。
讓這句話自己在空氣裡待夠。
```

### 「茉莉花」的揭露寫法（★★★★★ 全章最重要的段落）

```
寫法指令（逐句級）：

所有的記憶都秤量完了。
米亞宣布：容量已足夠。可以開始恢復。

但嚴柏翰沒有動。

他的手——右手——還停在修復台的操作面板上。
手指輕輕覆蓋著一段極小的記憶碎片。
碎片小到幾乎沒有重量。
米亞甚至沒有把它列入清單——
因為它太小了。不到 0.1%。可以忽略。

維倫：「那是什麼？」

米亞掃描。
停頓了 0.4 秒。

「一段不完整的感覺。沒有畫面。沒有聲音。
 只有……一個味道。」

嚴柏翰的手沒有離開。

維倫：「要放棄嗎？省一點空間。」

嚴柏翰看著自己的手。

他的手不肯放開。

不是他「選擇」不放開。
是手本身——那隻在 ch35 握拳、在 ch107 發抖、
在 ch123 再次發抖的手——
拒絕了鬆開的指令。

「留。」

他的聲音很輕。
像是在問自己為什麼。

但他沒有問出來。
因為有些選擇不需要理由。
身體知道的事，腦子不需要同意。

——三樣東西。
友誼。方法論。茉莉花。

一個真的人。
一套能帶走的工具。
一段沒有名字的愛。
```

### 不要做的事（ch149 禁忌清單）

1. 不要讓嚴柏翰猶豫太久——他是果斷的人。每個決定不超過十秒
2. 不要讓維倫問「茉莉花是什麼」——他不會在這個時候追問

## 三、Backtracking 在恢復中的教學呈現

### 教學策略

Backtracking 在 ch115-117（退役探員迷宮）已經建立概念。ch150-155 是同一個技術在情感場景中的重新應用。

```
Vol 2 Backtracking 的兩次使用對比：

ch115-117（迷宮案）：
  場景 = 恐怖
  目標 = 找到出口
  回溯原因 = 碰壁（牆擋住了）
  讀者感受 = 緊張

ch150-155（記憶恢復）：
  場景 = 心碎
  目標 = 找到被覆蓋的真相
  回溯原因 = 碰到覆寫層（假記憶擋住了真記憶）
  讀者感受 = 痛苦

同一個演算法，兩種完全不同的情感色彩。
讀者學到：工具是中性的。用它做什麼，取決於你在找什麼。
```

### 嵌入方式

```
不要在 ch150-155 中單獨設立「教學段落」。
Backtracking 的概念應該隱含在維倫的操作描寫中：

「維倫在第一層覆寫記憶的下方碰到了第二層——
 監控局加固的忠誠人格。
 穿不過去。
 退回。標記。
 嘗試另一條路徑。
 從情感記憶的邊緣繞過去——
 那裡的防禦最薄。因為情感無法被完全覆寫。」

讀者在閱讀時不會停下來想「哦這是 Backtracking」。
但回頭看會發現：維倫的每一步操作都是「嘗試→碰壁→回退→換路」。
這就是 Backtracking。

教學在潛意識裡完成。
```

## ch157：額頭上的吻

這一章是全書最重要的場景之一。每個動作、每個停頓都有設計意圖。

```
【場景開始】

記憶的最後一幀：

   → 「最後一幀」= 電影語言。這不是散文，是影像。
     讀者的閱讀速度應該像看慢鏡頭

邱韻如坐在嚴柏翰的床邊。
```

### 嚴柏翰醒來

```
【場景跳轉：現實。修復台。】

嚴柏翰在修復台上睜開眼睛。

   → 從記憶跳回現實。不需要過渡。直接切

眼淚從臉頰兩側滑落到耳朵旁邊。

   → 躺著時流淚的物理軌跡。淚水不是往下流——是往兩邊流，滑進耳朵
     這個細節 = 精確。像維倫觀察到的。冷靜的描寫包裹著最熱的情感

他說了一句話：

「帶我去見她。」

   → 四個字。獨立成段。
     前後各留一行空白。
     讓這句話在頁面上站著。像一個人站在廢墟中間

   → 不要接任何人的反應。不要寫維倫聽到後的表情。
     不要寫米亞的數據分析。不要寫語青的眼淚。
     讓這句話成為 ch157 的最後一行。

   → 然後斷章。
```

### 「帶我去見她」的重量

```
這句話為什麼是 Vol 2 最重要的一句台詞：

1. 它不是復仇宣言。
   「帶我去見她」不是「帶我去殺她」。
   他想見她。不是要報復。
   是因為他記起了愛。

2. 它證明了愛和恨可以共存。
   他知道她刪了他的人生。
   他也知道她是為了保護他。
   他不原諒。但他要見她。
   理解 ≠ 原諒。要見 ≠ 要復合。
   這些不等式才是真正的情感複雜度。

3. 它是 Vol 3 的起點。
   ch203：這句話兌現——嚴柏翰與邱韻如重逢。
   但讀者在 Vol 2 結尾不知道這一點。
   「帶我去見她」在 Vol 2 是一個懸崖。
   讀者被推下去。Vol 3 才有地面。

4. 它呼應了身體記憶。
   他的手在 ch149 選了茉莉花。
   現在他的嘴巴替手完成了句子：
   「帶我去見（茉莉花的那個人）。」
   身體選擇在前，語言確認在後。
```

```
  1. 用自己開發的技術覆寫他的記憶
  2. 刪除所有關於她、關於遊行、關於陣線的一切
  3. 植入「忠誠探員」人格
  4. 把他送進深淵監控局——他曾經反抗的機構
  5. 設置授權警報防止任何人追查到他的過去

她為什麼這樣做：
  表層：她相信「記憶改寫可以消除痛苦」（被植入的信念）
  中層：把他從「被通緝的反抗者」變成「安全的探員」= 保護
  底層：她不知道自己的信念是周明哲植入的（Vol 3 ch217 揭露）

  他在監控局追查記憶篡改案
  她在記憶銀行執行記憶篡改
  他追的就是她做的
  但他不知道。她知道

  她設置的授權警報（ch123）= 她一直在監控會不會有人追查到他
  不是為了保護自己——是為了保護他
  如果有人追查到十一月二日→追查到遊行→追查到他的真實身份→他就不安全了
```

## 三、ch157 後：恨和愛不是對立的

```
嚴柏翰知道的一切：
```

```
□ 2. 嚴柏翰的台詞超過三句話就檢查：他是不是說太多了？
```

# 附錄：嚴柏翰的情感崩潰模型

```
每恢復一段記憶 = 一次微型五階段循環：

1. 否認（3 秒）：「這是偽造的」
   → 職業本能。十年的探員訓練讓他的第一反應永遠是質疑

2. 驗證（10 分鐘）：用探員的方法確認真相
   → 他不會盲目接受。他會找證據。這是方法論——他保留的三樣之一

3. 無力（取代憤怒）：無法恨邱韻如——她是為了保護他
   → 不是「不想恨」。是「恨不出來」。因為每一段記憶裡她都在保護他

4. 沉默（取代抑鬱）：不是不說話，是找不到語言
   → 他的詞彙表裡沒有能描述這種痛苦的詞。所以他閉嘴

5. 接受（站起來）：帶著傷痛繼續走
   → 不是放下。不是釋懷。是「我現在知道了。我站著。接下來呢。」

最終的「接納」不是放下——是「帶著傷痛繼續活」。
嚴柏翰的弧線終點不是「他治癒了」。
是「他帶著所有的傷，選擇繼續走」。
這比治癒更真實。也更有力量。
```
