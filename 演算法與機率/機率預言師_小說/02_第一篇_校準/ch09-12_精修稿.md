# 第一篇「校準」終章（ch9-12）—— 初稿級精修

> **精修目的**：篇末高潮。從懷疑到確認。蘇立恆登場。世界觀裂開。
> **原則**：節奏快。情緒密度高。貝氏定理的完整呈現必須是「心碎的」而不是「教學的」。

## 第 9 章：似然比

### 夜晚——七個區域的鐵證

```
【環境】
週三深夜。十一點。貝葉家。
客廳的燈關了。只有筆電螢幕的光。
旁邊是一碗泡到脹的泡麵（她忘了吃）和一杯涼掉的水。
陽台的窗開著。夜風帶著一點潮氣。
遠處有一台救護車的聲音。

她已經跑了三天的數據。
白天上班——正常校準。蓋章。微笑。「修得好。」
晚上回家——打開離線模式。用自己下載的原始數據。重算。

三天。七個區域。
每一個都跑了三遍以上。
結果每次都一樣。

今晚，她要做最後的確認。
似然比檢定。Likelihood Ratio Test。

「小拉，離線模式。」
「已切換。……貝姊，你知道離線模式下我很多功能都不能用——」
「我知道。但你能算。」
「是的。我能算。」

她把七個區域的數據排開。
虛無假設 H₀：λ = 官方值（Oracular 宣稱的參數）
對立假設 H₁：λ ≠ 官方值

  ┌─ 似然比檢定結果 ──────────────────────────┐
  │                                             │
  │ 區域 A（大安）：LR = 4.72, p < 0.001 *** │
  │ 區域 B（信義）：LR = 3.18, p < 0.005 **  │
  │ 區域 C（中正）：LR = 5.01, p < 0.001 *** │
  │ 區域 D（中山）：LR = 0.04, p = 0.84  n.s.│ ← 正常
  │ 區域 E（內湖）：LR = 4.33, p < 0.001 *** │
  │ 區域 F（松山）：LR = 3.77, p < 0.001 *** │
  │ 區域 G（萬華）：LR = 0.07, p = 0.79  n.s.│ ← 正常
  │                                             │
  │ 異常區域：5/7                               │
  │ 全部在 α=0.01 水準下拒絕 H₀               │
  └─────────────────────────────────────────────┘

五個區域。
全部拒絕虛無假設。
p < 0.001。
在統計學裡，這是最強的證據等級。

「七個裡面五個。」她的聲音很輕。像在自言自語。

「而且方向一致。全部往低。
 隨機波動不會這樣。
 隨機是——有的高有的低。像真正的雜訊。
 但這些全部往同一個方向偏。」

她的手在發抖。
不是冷。
台北的六月很熱。窗戶開著。夜風是溫的。

是因為——

如果這是真的。
那全台灣——不只是這七個區域——
每一個被「系統自動校準」過的區域——
每一個人透鏡上看到的數字——

都是錯的。

她把筆電放在茶几上。手撐著膝蓋。低著頭。
呼吸很深。一次。兩次。三次。

「貝姊。」小拉的聲音很輕。比平常輕。
「你的心率 112。呼吸頻率上升 40%。
 你需不需要——」

「不需要。」

「……好。」

貝葉抬起頭。看著天花板。
天花板上有一個水漬。是上個月漏水留下的。
看起來像一個不規則的橢圓。像一個散點圖裡的離群值。

「小拉。」
「在。」
「五個區域的官方 λ 全部被壓低。
 但原始數據是真的。我查過警政資料庫——沒有被改過。
 計算引擎也是正常的——似然函數的算法沒有問題。
 那被改的是什麼？」

小拉想了 0.4 秒。
「先驗。」

「對。先驗。」

「有人在校準之前——在任何人看到數據之前——
 就先把先驗分布的參數調低了。
 讓人看到的『起點』就是偏的。
 然後數據進來。貝氏更新。
 但因為起點是偏的——更新完的後驗也是偏的。」

她用力閉了一下眼睛。

「數據是真的。算法是真的。
 但結論是假的。
 因為地基——先驗——被人動過了。」
```

## 第 10 章：不是 Bug

### 凌晨——先驗的幽靈

```
【環境】
凌晨兩點。她還沒睡。
泡麵完全冷了。膨脹成一坨。不能吃了。
她不在乎。

螢幕上開了四個視窗。
一個是原始數據。一個是官方先驗。一個是她算的 MLE。
第四個——是先驗分布的歷史紀錄。

她從公司資料庫裡拉出了大安區過去五年的先驗紀錄。
正常情況下，先驗會隨著新數據慢慢更新。
像一條河——緩緩流動。偶爾轉彎。但不會突然斷裂。

但在去年——

  ┌─ 先驗分布歷史 ─────────────────────────────┐
  │ 大安區 Sec.7 犯罪率先驗：                   │
  │                                              │
  │ 2041：Gamma(α=2.8, β=1.1)  → E[λ]=2.55    │
  │ 2042：Gamma(α=2.9, β=1.1)  → E[λ]=2.64    │
  │ 2043：Gamma(α=3.0, β=1.2)  → E[λ]=2.50    │
  │ 2044.01-07：Gamma(α=3.1, β=1.2) → E[λ]=2.58 │
  │                                              │
  │ 2044.08（系統自動校準後）：                   │
  │   Gamma(α=1.5, β=1.5) → E[λ]=1.00          │
  │                                              │
  │ ⚠ 先驗期望值從 2.58 跳到 1.00               │
  │ ⚠ 下降幅度：61%                             │
  │ ⚠ 無對應的數據變化支持此修改                │
  └──────────────────────────────────────────────┘

三年來，先驗的期望值都在 2.5 附近。穩定。正常。
然後在去年八月——「系統自動校準」之後——
突然跳到 1.0。

不是漸進式的更新。是突然的替換。
像有人把一首歌的第三段換成了另一首歌。

而且——更新紀錄裡沒有任何說明。
沒有「因為XX原因，先驗從 Gamma(3.1, 1.2) 改為 Gamma(1.5, 1.5)」。
就是——換了。靜悄悄地。

「小拉。」
「在。」
「一個先驗分布的期望值在沒有數據支持的情況下突然下降 61%——
 這是正常的嗎？」

「不正常。根據貝氏更新的原理，
 先驗應該被數據「平滑地」推動——
 像一艘船被水流慢慢推。
 不會突然跳。除非——」

「除非有人直接替換了先驗的參數。」

「是的。」

沉默。

「這不是 bug。」貝葉說。

「不是。」

「這是——有人故意做的。」

窗外的風停了。連夜風都在屏息。
遠處的救護車聲音也消失了。
整個世界安靜得像一個被設定為零的隨機變數。

貝葉站起來。走到陽台。
看著夜裡的台北。
到處都是燈。一棟棟公寓的窗戶像像素。
每個窗戶後面都有一個人。每個人的透鏡裡都有數字。
那些數字決定了他們明天要不要出門、
要不要過那個路口、
要不要買那個房子、
要不要相信那個人。

而那些數字——可能全部是假的。

不是「有可能」。
是 p < 0.001。

她的手撐在陽台的欄杆上。金屬很涼。
薄荷在她旁邊。新長出來的葉子在夜風裡微微搖動。

「小拉。」
「嗯？」
「如果全台灣的先驗都被改了——
 那所有人做的所有決定——
 都是基於一個被操控過的起點。」

「……是的。」

「那那些決定是他們自己的嗎？」

小拉沉默了 4.7 秒。
這是她有記錄以來最長的沉默。

「我……不知道怎麼回答這個問題。
 這超出了我的計算模型。」

「我知道。這也超出了我的。」

她回到客廳。坐下。
拿起那碗脹掉的泡麵。看了一眼。放下。

「明天。」她說。
「我要跟老高說。」
```

## 第 11 章：父親的定理

### 貝氏定理——心碎的版本

```
【環境】
週四。下班後。貝葉沒有回家。
她去了台大。統計系館。三樓。301 研究室。
高思遠教授退休之後，研究室還保留著。
學校說「等有新教授來再收回」。但一直沒有新教授。
所以它就一直在那裡。門上的名牌還是「高思遠」。

她敲門。
「進來。」高思遠的聲音。
研究室裡有書、有灰塵、有一面白板。
白板上還留著高思遠最後一堂課的板書：
「Bayesian Statistics — Lecture 24: The End is Just the Beginning」
最後一行字是：「質疑先驗。永遠質疑先驗。」

高思遠坐在窗邊的椅子上。看著窗外的椰林大道。
夕陽把他的輪廓鍍了一層金色。
他的透鏡——第一代原型機——左眼邊緣閃了一下。

「老高。」
「嗯。來了。」
「我有東西要給你看。」

她把筆電放在桌上。打開數據。
七個區域。似然比。先驗偏移。

高思遠戴上老花眼鏡（雖然有透鏡，他還是習慣用眼鏡看近的東西）。
看了很久。

他沒有問「這是什麼」。
他沒有問「你確定嗎」。

他看完之後——
把老花眼鏡摘下來。放在桌上。
摘得很慢。像在做一個儀式。

「……我等這一天等了八年。」

★★★ 全書第一個三星級場景。

貝葉的心跳停了一拍。

「你知道？」

高思遠搖頭。
「不是知道。是懷疑。」

他站起來。走到白板前面。
拿起一支粉筆——不是白板筆。是真的粉筆。
他從口袋裡拿出來的。

在白板上——在他最後一堂課的板書旁邊——
寫下了全書最重要的一行字。

  P(θ|data) ∝ P(θ) × P(data|θ)

  後驗 ∝ 先驗 × 似然

粉筆在白板上的聲音。沙沙的。老派的。溫暖的。
像三十年前的教室。
像她父親還在的時候。

「貝葉。」高思遠放下粉筆。「你知道這個公式的意思。」

「後驗等於先驗乘以似然。」

「對。但不只是數學。」

他轉過身。面對她。
夕陽在他身後。她看不清他的表情。但能聽到他的聲音。
比平常更慢。每個字都像被秤過重量。

「先驗——是你在看到數據之前就相信的東西。
 你的經驗。你的直覺。你的偏見。你的愛。你的恐懼。
 所有這些，都是先驗。

 似然——是世界給你的數據。
 你看到了什麼。你經歷了什麼。事實是什麼。

 後驗——是你在看完數據之後的信念。
 你的新判斷。你的新決定。你對世界的新理解。」

他走近一步。

「如果有人改了你的先驗——
 你看到的數據不會改變。
 但你的結論會改變。
 你的決定會改變。
 你的人生會改變。」

停頓。

「而你甚至不知道——
 因為你以為你的先驗是你自己的。
 你以為你的起點是自由的。
 但它不是。
 它被人替換了。」

貝葉的眼眶熱了。
不是因為恐懼。
是因為——

這個公式。
她第一次見到它的時候是十歲。
在家裡的餐桌上。
爸爸用果汁杯和麵包解釋：
「這是先驗。這是似然。搖一搖——變成後驗。簡單吧？」

她笑了。那時候。

現在。
這個公式被用來操控所有人。

「有人用了統計學最美的定理。」她說。
聲音啞了。
「做了最可怕的事。」

高思遠沒有反駁。
因為她說的是對的。

他走回窗邊。坐下。
看著窗外的椰林大道。天快黑了。

「八年前——柏朗剛開始推廣透鏡的時候。
 我就覺得不對。不是因為數據——
 是因為他的眼神。」

「什麼意思？」

「他的眼神不再有疑問了。
 一個做研究的人，眼裡應該永遠有疑問。
 但柏朗——
 他看起來像一個已經確定了答案的人。
 那讓我害怕。」

他看著貝葉。

「你知道嗎——
 你的父親在他去世前一週，也跟我說過同樣的話。
 他說：『高老師，Oracular 的先驗更新機制有一個漏洞——
 如果有人在更新之前就修改了參數……』
 他沒有說完。因為他還要去買便當。
 然後他就……」

高思遠的聲音停了。
他咳嗽了兩聲。不是因為話太長。
是因為——有些話在喉嚨裡卡了八年。

貝葉坐在椅子上。一動不動。

她父親——也知道？

「老高。」
「嗯。」
「你為什麼等了八年？」

「因為我沒有證據。」
他看著她手裡的筆電。
「現在——你有了。」
```

## 第 12 章：你發現了對不對

### 回家路上——世界的裂縫

```
【場景一】
晚上九點。貝葉走出台大校門。

天已經全黑了。
椰林大道上的路燈一盞盞亮著。
她走在路上。腳步很慢。

剛才高思遠說的話在她腦子裡轉。
父親知道。父親在去世前一週就知道了。
他沒來得及說完。

11 秒。
不只是一場車禍的隨機悲劇。
也許——

不。
她不能這樣想。不能。
那條路太深了。

「小拉。」
「在。」
「幫我跑一個東西。」
「什麼？」
「我爸去世那天——巷口的交通燈控制系統——
 是不是有被……」

她停了。

「不。算了。不要查。」

「……好的。」

小拉沒有追問。
這是她第一次——選擇不追問。
因為她的情緒模型告訴她：
有些問題的答案，比不知道更痛。

貝葉走到巷口。等紅燈。
透鏡自動顯示：

  ┌────────────────────────────────────────┐
  │ 路口通過安全機率：0.97                 │
  │ 建議：通過                             │
  └────────────────────────────────────────┘

0.97。
她盯著那個數字。

這個數字是真的嗎？
她以前從不懷疑。
現在——她不確定了。

綠燈了。她過了馬路。沒事。
但「沒事」這兩個字，從今天開始——
她再也不能輕易地說出口了。
```

### 門鈴——蘇立恆

```
【場景二——全書最重要的場景之一】

回到家。十點。
打開門。把傘放在門邊。
脫鞋的時候看了一眼陽台——薄荷還在。

她走進客廳。沒有開燈。
站在黑暗裡。
不是因為不想開。是因為——
她突然覺得，如果開了燈，就要面對什麼。

「小拉。」
「在。」
「如果這是真的——如果所有人的先驗都被改了——
 那我們——」

叮咚。

門鈴。

她愣了一下。晚上十點。誰會——

叮咚。

第二次。不急。但堅定。

她走到門口。透鏡自動掃描。

  ┌────────────────────────────────────────┐
  │ 門外人物：男性，約 28-32 歲            │
  │ 匹配紀錄：與 [路燈下未知人物] 同一人  │
  │ 攜帶非標準頻率裝置：是                 │
  │ 威脅機率：0.03                         │
  └────────────────────────────────────────┘

那個人。
路燈下的。蔥油餅攤對面的。
現在——站在她家門口。

她開了門。

蘇立恆。
黑衣。手環。瘦長的身影在走廊的安全指示燈下像一道陰影。
他的眼睛在暗處很亮。不是攻擊的亮。
是急迫的亮。

「你——」

他看著她。
只說了一句話。

「你發現了對不對？」

★★★

聲音不大。但每個字都清楚。
像一把手術刀——精確地切開了她最後的猶豫。

她沒有說是。也沒有說不是。
但她的沉默就是答案。

蘇立恆從口袋裡拿出一張紙條。
摺成很小的一塊。
塞進她的手裡。
他的手指碰到了她的手掌。很涼。
像金屬。

紙條上只有一行字：

  「先驗端口。1,247 次。查 B2。」

他轉身就走了。
沒有解釋。沒有自我介紹。
像他從來不是一個需要被介紹的人。
他是一段數據。他把數據留下了。人走了。

門在他身後關上。
走廊的空氣裡有一點——很淡的——
皮革的味道？不。是金屬的味道。焊錫。
像白奶奶攤位對面石椅上殘留的。

小拉在透鏡裡瘋狂跑數據：
「未知男性！透鏡頻率匹配 Oracular 內部加密協議——
 相似度 92%！但他不在現任員工名單裡——
 我在舊的人事資料庫裡找到一個匹配——
 蘇——」

「小拉。」
「什麼？」
「安靜一下。」

「你的心率 134。」
「我知道。」
「你的瞳孔放大了 37%。」
「我知道。」
「你的手在發抖——」
「小拉。」
「……好。我閉嘴了。」

貝葉站在門口。
手裡握著那張紙條。
紙條被她的體溫暖了。
但上面的字還是冰的。

先驗端口。1,247 次。B2。

1,247 次未經授權的先驗修改。
B2——Oracular 的地下二層。核心架構組。

他是誰？他怎麼知道？
他為什麼來找她？

她把紙條攤開。看了第二遍。第三遍。
字跡很工整。像寫代碼的人寫的字——
沒有冗餘。沒有裝飾。每一筆都到位就停。

她走到客廳。坐下。
把紙條放在茶几上。旁邊是那碗脹掉的泡麵。
一個改變世界的線索。和一碗泡壞的泡麵。
```
