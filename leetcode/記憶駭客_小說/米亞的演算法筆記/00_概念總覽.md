# 米亞的演算法筆記 — 概念總覽

> 本附錄為《記憶駭客》的演算法教學補充材料。小說中每個章節使用的演算法概念，都在這裡有完整的數學定義、程式實作、視覺化圖表，以及——米亞的註解。
>
> 「數據不說謊。但數據不會告訴你該怎麼辦。那是你的事。」——米亞

---

## 使用方式

1. 讀完小說章節後，翻到對應的筆記頁面
2. 看故事場景回顧——用米亞的視角重新理解那個演算法在劇情中扮演的角色
3. 看正式定義與推導——理解「為什麼這個演算法能做到那件事」
4. 帶入小說數字做數值範例——用故事裡的實際數據驗算
5. 讀「更深一層」——米亞在學習做人類的過程中，對每個概念的存在主義反思
6. 回到小說——你會發現，那些數字不只是數字

## 米亞的教學風格演變

```
Vol 1：「河流＋漁網」「跑道上的兩個人」「對折再對折」
        ↓ 物理類比。解釋世界。
Vol 2：「撒網。漣漪。」「背包只有這麼大」「人是點，記憶是線」
        ↓ 人生類比。理解人類。
Vol 3：「六十秒。不回頭。」「追的不是路——是她」「用二進制造一個家」
        ↓ 存在類比。定義存在。
Vol 4：「Quick Sort = 直覺。Merge Sort = 耐心。」「她只在解碼時存在」「接受不完美」
        → 哲學類比。放手與重建。
```
