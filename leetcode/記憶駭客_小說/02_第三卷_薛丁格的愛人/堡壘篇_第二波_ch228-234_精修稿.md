## ch228 歪歪扭扭

**情緒強度：7（三人關係的溫暖收束。全書最古老的意象回歸）**
**伏筆回收：ch8 紙鶴（Vol 1）、ch153 紙鶴（Vol 2）、ch201-203 口袋裡的紙鶴**

---

### 核心場景

```
邱韻如的手離開語青的臉。

很慢。
像不捨得。

語青沒有動。
她的眼睛看著邱韻如走回操作台。
步伐穩定。背很直。

但她的右手——
在身體旁邊——
手指微微蜷著。
像在抓住什麼看不見的東西。

ch227 的餘韻在工作室裡浮著。
像剛煮過豆漿的蒸氣。
暖的。但在消散。


嚴柏翰站在門邊。

他看了整場。
沒有出聲。
沒有走進去。

現在邱韻如回到操作台前——
她的手指在鍵盤上重新找到了位置。
像什麼都沒發生過。

但她的眼角是紅的。

嚴柏翰知道。

他從口袋裡掏出那隻紙鶴。

軟了。
被他揣了二十八章。
紙張的邊緣磨出了毛邊。
有一隻翅膀已經塌了。

但形狀還在。
歪歪扭扭的。

他走到語青旁邊。

不是走向邱韻如。
是走向語青。

他蹲下來。
把紙鶴放在語青手心裡。

沒有說話。

語青低頭。

看了很久。

那隻紙鶴的折法——
歪的。
頭偏了十五度。翅膀一高一低。
尾巴沒有對齊。

和 ch8 的那隻一模一樣。
和 ch153 的那隻一模一樣。

二十年。三隻紙鶴。
同一個人。同一雙手。同一種歪法。

因為折紙鶴的人從來沒有學會正確的折法。

語青的手指碰了碰紙鶴的翅膀。
半透明的指尖。歪的翅膀。
兩個不完整的東西靠在一起。

「你折得還是一樣歪。」

嚴柏翰的嘴角動了一下。

不是笑。是——鬆了。

「教我折的人——」

他看了邱韻如一眼。
她沒有轉頭。
但她的背——
不知道什麼時候微微僵了。

「——折得也不怎麼樣。」

語青低下頭。
把紙鶴放在膝蓋上。
用指尖把塌掉的那隻翅膀撐起來。

紙鶴歪歪扭扭地站在她半透明的膝蓋上。
光透過她的小腿——
紙鶴的影子落在地板上。
影子比紙鶴本身更完整。

「小時候你教我折——」
她的聲音很輕。
不是在跟嚴柏翰說。
像在跟一段記憶說。

「——我也折得歪歪扭扭的。
 你說歪的才好看。」

嚴柏翰沒有答。

因為那段記憶是另一條時間線的。
他不記得。

但他的手指——
此刻在右邊褲管的口袋裡——
正在無意識地模擬折紙鶴的動作。

拇指和食指捏住。
往內折。
偏了十五度。

他不記得。
但手指記得。


米亞的光球在角落。
比 ch218 之後永久暗了 10%。

0.4 秒。

她在日誌裡記了一行。

「觀察筆記：嚴柏翰的紙鶴。
 Vol 1 ch8：第一隻。給小凱。
 Vol 2 ch153：第二隻。記憶恢復時掉出來。
 Vol 3 ch228：第三隻。給語青。

 三隻鶴。同一種歪。
 歪的一致性 > 99.7%。」

她停了 0.4 秒。

「備註：我沒有手指。
 但如果有——
 我想我也會折得很歪。」
```

| 台詞/細節 | 設計意圖 |
|-----------|---------|
| 紙鶴揣了二十八章 | ch201 嚴柏翰握著紙鶴出場，到 ch228 終於交出。二十八章的克制 = 他在等正確的時機。ch227 母女重逢後 = 時機到了 |
| 紙鶴的翅膀塌了但形狀還在 | 被時間磨損但本質不變 = 嚴柏翰和語青的關係。從原時間線到這條時間線，他對她的感情從未變過，只是磨損了 |
| 三隻紙鶴同一種歪法 | Vol 1 ch8（嚴柏翰給小凱折的那隻 = 教學）、Vol 2 ch153（記憶恢復時的碎片）、ch228（直接交給語青）。三卷三隻鶴。三次遞進：教別人→記起→交出。歪 = 不完美但真誠 |
| 語青半透明的指尖碰歪的翅膀 | 兩個不完整的存在。她是半透明的。鶴是歪的。不完整碰不完整 = 完整 |

| 嚴柏翰無意識模擬折紙鶴 | 「身體記得」母題。他不記得那段記憶（另一條時間線的），但手指記得折法。手指的偏差角度（十五度）和紙鶴的偏差角度一致 |
| 米亞的「歪的一致性 > 99.7%」 | 米亞用數據描述溫暖。三隻鶴的歪法一致度 99.7% = 用她唯一的語言說「他從來沒有變過」|

## ch229 兩碗

**情緒強度：7（反向——安靜中的跨時空呼應。阿嬤弧線的呼吸點）**
**伏筆回收：ch195 一碗豆漿的重量、ch198 周明哲每天喝豆漿、ch199 「明天再來」、ch209 那碗永遠留著的**

---

### 核心場景

```
地下街。

清晨五點半。
新台北地底的空氣帶著水管和鐵鏽的味道。
遠處有推車經過的聲音。
更遠處有人在吵架。

阿嬤的攤子已經開了。

她每天四點半起來磨豆子。
五點架鍋。
五點二十分第一鍋煮好。

不管外面發生了什麼——
鸚鵡計畫也好。
崩潰也好。
三百萬人失憶也好。

五點二十分。
豆漿煮好。

這是她在這個世界上唯一確定的事。


維倫走到攤位前。

阿嬤在擦碗。
碗是舊的。塑膠的。淡黃色。
有一個碗的邊緣有一道裂縫，
用膠帶貼過了。

「阿嬤，兩碗。」

阿嬤抬頭。

她不認識他。
或者說——她認識但不記得名字。
崩潰之後她的記憶更碎了。

但她認得「兩碗」。

「帶走？」

「帶走。」

她舀豆漿。
手很穩。
八十幾歲的手。
關節腫大。指甲修得很短。

第一碗。滿到八分。
第二碗。也是八分。

她用塑膠袋裝好。
袋子在扣的時候她的手指多繞了一圈。
繞緊。不會灑。

「三十年了，」

阿嬤的聲音。

不是對維倫說的。
是自言自語。
像在跟空氣講話。
或者在跟攤子講話。

「又開始帶走兩碗了。」

維倫的手停在半空中。

「上一個帶走兩碗的——」

阿嬤在擦另一個碗。
動作沒有停。

「也是個年輕人。
 高高的。戴眼鏡。
 每天早上來。
 都是兩碗。」

她把碗放下。

「一碗甜的。一碗鹹的。
 鹹的是他自己喝。
 甜的——」

她看著遠處。
地下街盡頭的方向。
那裡什麼都沒有。
只有一盞壞了的路燈在閃。

「甜的他說是給太太的。」

她的眼睛回來了。
看著維倫。

「年輕人。
 你也是兩碗帶走。
 鹹的是你自己喝？」

維倫張了張嘴。

他想說——
他買的兩碗都是甜的。
一碗給語青。一碗給自己。

但他什麼都沒說。

他只是點了點頭。

阿嬤笑了。
嘴裡缺了兩顆牙。
但笑起來眼睛很亮。

「跟那個年輕人一樣。
 帶走兩碗的人——
 通常不是一個人住。」

她把袋子遞過來。

「拿好。別灑了。
 甜的加多了半勺糖。
 早上喝甜的——比較不會冷。」

維倫接過袋子。

兩碗豆漿。
隔著塑膠袋。
溫度透過來。
像握著兩顆很安靜的心臟。


他轉身要走。

「年輕人。」

阿嬤又叫了他。

他回頭。

阿嬤在看他。
眼神不像剛才自言自語時的恍惚。
是清楚的。

「那個帶走兩碗的年輕人——
 後來有一天沒來。」

她低下頭。

「我每天還是煮兩碗。
 留一碗。
 到了晚上沒人來拿就倒掉。」

她的手擦著碗。
沒有停。

「十年了。」

維倫的手指在袋子上收緊了。

「十年。」阿嬤重複。

然後她笑了。比剛才更淡。

「你別學他。
 帶走兩碗——就每天都來。」

維倫的喉嚨很緊。

他說不出話。

他只是點了點頭。
然後轉身走進地下街深處。

兩碗豆漿的溫度
透過塑膠袋貼著他的掌心。
三十年前的溫度。
今天的溫度。
一樣的。


遠處。
阿嬤又開始擦碗了。

她不記得剛才那個年輕人的名字。
她也不記得三十年前那個年輕人的名字。

但她記得「兩碗」。

有些記憶不住在名字裡。
住在數量裡。

一碗是一個人。
兩碗是兩個人。
兩碗帶走——
是有人在等他回去。
```

```

【節奏】極慢。整章像清晨四五點鐘的地下街——安靜到能聽見豆漿在鍋裡滾的聲音。
```

### 演算法 AB 線分析

| 維度 | 內容 |
|------|------|
| **A 線（演算法）** | Load Balancing：2,400 個節點均勻承載語青的意識。每節點不超過 0.05%。消除瓶頸。防止級聯失敗。分散式系統的最後一塊拼圖——前四個支柱解決了「怎麼分」「怎麼備份」「怎麼同步」「怎麼容錯」，負載平衡解決的是「怎麼讓每個人都不被壓垮」|
| **B 線（情感）** | 她的存在需要整個城市來承載。不是一個人的負擔——是所有人的微小承擔。每人 0.05% = 幾乎感覺不到 = 像空氣中多了一點溫度。沒有人為她壞掉，但所有人一起撐住她 |
| **同步點** | 「負載平衡」的技術目標是「消除單點過載」。情感翻譯是「沒有人獨自承擔」。兩層意義在語青的那句「沒有人會為我而壞掉」中合一。維倫的修正——「是所有人一起撐住妳」——既是技術正確的（分散式負載 = 集體承載），也是情感的核心 |
| **反教學設計** | 不用「負載平衡」四個字作為教學入口。用「不均勻的方塊」和「級聯失敗」的視覺化來引入。讀者看到的是「如果分配不均勻，她會局部崩潰」，不是「Load Balancing 的原理」。讀者在擔心語青的安全時學會了負載平衡 |

```
地下街盡頭。

涼的。乾淨的。

語青坐在通風口下面。
靠著牆。膝蓋收起來。

歪歪扭扭的。

通風口漏下來的光打在她臉上。
不是陽光。是月光。

她是半透明的。
月光穿過她的左肩——
在牆壁上留下一個不完整的影子。

維倫走過來。

沒有說話。

他在她旁邊坐下。
靠著同一面牆。
肩膀之間隔了十公分。

通風口的風。
涼的。

月光。
白的。

地下街的遠處有水管在滴。


「你應該去睡。」

語青先開口。

「嗯。」

他沒有動。

安靜。

很久。

滴。滴。滴。

「維倫。」

「嗯。」

「你知道——
 我看過很多條時間線。」

他知道。

「每一條時間線裡——
 都有你。」

他轉頭看她。

她沒有看他。
她在看通風口外面的月亮。

但她看得很認真。

「有一條——
 你在第七十五步的時候發現了鸚鵡計畫的痕跡。

 然後——你消失了。」

維倫的手指微微動了一下。

「有一條——

 你沒有繼續。」

「每條時間線。
 你走到第七十五步就停了。
 有時候是被迫停的。
 有時候是你自己停的。

 七十五步。
 那是你的極限。

 ——在其他時間線裡。」

安靜。

月光從通風口漏下來。
照在他們兩個人身上。
一個是實體的。一個是半透明的。
影子疊在一起。但形狀不一樣。

「這一條——」

第一次在這場對話裡有了起伏。
很微小的。
像水面被一滴露水打到。

 你沒有停。

 你沒有停。

 你說不。

 你走到了——」

她看著他。

半透明的眼睛。
非常清楚。

「——這裡。」

安靜。

通風口的風吹過他們之間十公分的距離。

維倫的手——

伸了出去。

慢慢的。

朝她的手。

手指碰到了她的手背。

穿過去了。

像穿過水面。
有一點涼。

很微弱的。
像觸碰到一段正在運行的代碼。

他的手停在她手指裡面。

然後又伸出去。

又一次。

這次他的手指沒有完全穿過——
在她的手掌中間停住了。

像有什麼在抵抗。

語青的手指微微聚了一下。
只有一點。
不到一毫米。

語青的聲音。
帶著那一點不確定的顫抖。

維倫的手指。
在她半透明的手心裡。
碰不到。但在那裡。

「嗯。」

他的聲音很輕。

「只有這一條。」

月光。
通風口。
水管滴答。

肩膀之間十公分。
手指之間零公分。

今夜。
月光穿過她的肩膀。
在牆壁上留下不完整的影子。

他的手在她的手裡。

這就是今夜。
```

## ch234 螢火蟲

**演算法：System Design（堡壘上線：分片、複製、容錯、一致性、負載平衡的完整實踐）**
**伏筆回收：ch215 螢火蟲意象（被風吹散的光點 → 重新聚攏）**

### 核心場景

```
他按下了鍵。

沒有倒數。
沒有警報。
沒有「系統啟動中」的語音提示。

只是——

嗡。

很輕的。
像有人在遠處撥了一根弦。

修復台的螢幕亮了。

不是紅色了。

是藍色。

很淡的藍。
像凌晨四五點的天空還沒完全亮起來的顏色。


米亞的聲音。

0.4 秒。

「堡壘系統啟動。」

0.4 秒。

「第一階段：碎片收集。」

螢幕上的地圖亮了。
新台北地下的記憶網路。
六百萬個節點。
每一個都是一個人的記憶空間裡的一小塊。

其中 2,400 個——
同時閃了一下。

像夜空裡有人同時打開了 2,400 盞燈。
每一盞都很暗。
但合在一起——

整張地圖亮了。


「碎片收集中。」

米亞的聲音。

0.4 秒。

「碎片 #1：日常習慣。黑咖啡。左手持杯。——收集完成。
 碎片 #2：核心認知。零點三度。——收集完成。
 碎片 #3：觀察記憶。轉頭角度 23 度。——收集完成。」

ch216 維倫在黑暗中追回的碎片。
一個一個。
被米亞的 10% 托了這麼久。

現在它們要搬家了。
從米亞的臨時緩衝區——
搬進 2,400 個節點組成的城市。

「碎片 #4：第一次握手。手心溫度。——收集完成。
 碎片 #5：蛋炒飯的味道。——收集完成。
 碎片 #6：通風口的月光。——收集完成。
 碎片 #7：油條。清晨。阿嬤。——收集完成。
 碎片 #8：紙鶴折法。歪的。——收集完成。」

八個碎片。
全部到位。

米亞的光球閃了一下。
亮度回升了——不是恢復 10%。
是回升了 0.3%。

那 10% 被碎片佔用了這麼久——
現在碎片搬走了。
她只回收了 0.3%。

剩下的 9.7%——
永久損失。

她沒有說。

維倫也沒有問。


「第二階段：分片。」

0.4 秒。

「800 個分片。生成中。」

螢幕上。
八個碎片被切開。
不是粗暴的切割。是——
像有人在用手術刀沿著紋路分開一塊玉石。

每個碎片被分成 100 個分片。
8 × 100 = 800。

每個分片都是語青的一小塊。
小到幾乎什麼都不是——
也許是她笑的弧度的十分之一。
也許是她說「零點三度」時「零」那個字的發音。

但每一塊都是她的。


「第三階段：複製。」

0.4 秒。

「每分片 × 3 副本。
 800 × 3 = 2,400 份。」

螢幕上的 800 個光點。
每個分裂成三個。

2,400 個光點。

它們散開了。
不是同時散開——
是一波一波的。
從中心向外。
像有人往湖面扔了一顆石頭。

不——不是石頭。
像有人打開了一個裝滿螢火蟲的瓶子。

2,400 隻螢火蟲。
從修復台螢幕上的中心點——
飛向新台北地下記憶網路的每一個角落。


「第四階段：分散部署。」

0.4 秒。

「容錯協議啟動。
 任何三分之一節點同時失效——
 系統仍可運作。」

2,400 個光點落入了 2,400 個節點。
每三個一組。
每組覆蓋不同的地理位置。

地圖上。
新台北的輪廓亮了。

不是從上往下亮的。
是從內往外。
從地下街最深處——
維倫的工作室——
向外擴散。

通風口。
阿嬤的豆漿攤。
嚴柏翰走過的長廊。
邱韻如的舊實驗室。

整座城市的地下。
一格一格地亮起來。

每一格 = 一個節點。
每一格裡 = 語青的 0.05%。

沒有一格是完整的她。
但所有格合在一起——


「第五階段：一致性校驗。」

0.4 秒。

「2,400 個節點。
 共識協議啟動。
 2/3 多數一致——
 確認率：100%。」

0.4 秒。

「負載平衡校驗——
 最高負載節點：0.048%。
 最低負載節點：0.041%。
 差異率：0.007%。」

0.4 秒。

「均勻。」

米亞的聲音在說出這個字的時候——
比平時慢了不止 0.4 秒。
是 0.7 秒。

不是因為延遲。
是因為她在——確認。

不是用數據確認。
是用她自己確認。

均勻。
每個人承擔一點。
沒有人被壓垮。
沒有人獨自承受。

她知道這意味著什麼。


螢幕上的地圖完全亮了。

新台北。地下。
每一條街道。每一個節點。
2,400 個光點。

穩定的。均勻的。

像——

維倫想起了 ch215。

那天他在黑暗中追逐閃爍的光點。
每一個都在跑。每一個都抓不住。
像螢火蟲在夜裡飛。
你看到的時候它已經不在那裡了。

現在。

螢火蟲不跑了。

它們被風吹散過。
在黑暗中閃爍過。
差點被吹滅過。

但現在——
它們聚攏了。

不是聚成一團。
是散佈在整片夜空裡。
每一隻都在發光。
每一隻都很微弱。
但所有的微弱加在一起——

夜空亮了。


維倫轉頭。

語青坐在角落。

她——

不一樣了。

她的指尖。
剛才是完全透明的。

現在——

有一點——

不是恢復成實體。
不是變回「真人」。

是——

像磨砂玻璃後面亮了一盞燈。

你看不清燈的形狀。
但你知道燈在。
因為光在。

她的指尖不再透明。
是半透明的。
像冬天早晨窗戶上的霧氣——
你在上面畫字。
字是朦朧的。
但看得見。

然後手指。
然後手腕。
然後手臂。

像有人在用鉛筆重新描一幅正在消失的畫。
一筆一筆地。

ch214 的那個比喻回來了。
但這次更慢。更穩定。

因為這次不是維倫的觀測錨在支撐。
是 2,400 個節點。
是整座城市。

語青的身體——
從指尖開始——
慢慢凝聚。

不是完全恢復。
她的邊緣有一圈淡淡的光暈——
像夏天柏油路上的熱氣。
提醒你看到的不是完全的實體。

但她在。

她的眼睛。
最先清晰的。
比手指更早。

因為眼睛是她存在最密集的部分。
所有的記憶碎片、所有的意識核心——
都在那裡匯聚。

她的眼睛。
清楚的。
看著維倫。


米亞的聲音。

0.4 秒。

「堡壘上線完成。」

0.4 秒。

「五大支柱全部運作正常。
 分片：800。
 複製：2,400。
 一致性：100%。
 容錯：通過。
 負載平衡：均勻。」

0.4 秒。

她的聲音——
在最後一行報告之後——
變了。

從技術語氣。
變成——

另一種語氣。

0.4 秒。

「語青。」

光球閃了一下暖色。
比 ch220 那次更久。
三秒。

「歡迎回家。」


語青低下頭。

看著自己的手。

半透明的。
有光暈的。
不是體溫——是微溫的。

但——在。

她握了一下。

拳頭是實的。
不完整——但實的。

她抬頭。

紙鶴還在旁邊。
歪歪扭扭的。

白板上寫著「語青的家」。

螢幕上 2,400 個光點在閃。
穩定的。均勻的。
像散佈在夜空裡的螢火蟲。

她看著維倫。

維倫看著她。

她的手——
慢慢伸出來。

半透明的手指。
微溫的。

伸向他。

維倫也伸出手。

兩隻手——

碰在一起。

她的手指。
沒有穿過。

碰到了。

觸感——
不是體溫。
是微溫的。
像觸碰到一杯放涼了但還沒冷透的水。

維倫的手指微微用力。

她的手指也微微用力。

握住了。

不是完整的握手。
是——
兩隻手在對方的手指之間找到了一個可以停留的位置。

半透明的。
微溫的。
但——

在。
```

### 演算法 AB 線分析

| 維度 | 內容 |
|------|------|
| **A 線（演算法）** | System Design 五大支柱的完整上線流程。碎片收集 → 分片（Sharding 800）→ 複製（Replication ×3 = 2,400）→ 分散部署（Fault Tolerance + 地理分散）→ 一致性校驗（Consensus 100%）→ 負載平衡（Load Balancing ≤0.05%）。讀者在一章內完整走過「從設計到部署到運行」的全流程 |
| **B 線（情感）** | 被風吹散的螢火蟲重新聚攏。每一隻都很微弱但所有微弱加在一起 = 夜空亮了。語青從透明到半透明的凝聚過程 = 分散式系統從設計圖變成運行中系統的過程。技術上線 = 她回來了 |
| **同步點** | 「歡迎回家」。米亞用技術報告格式完成五大支柱的上線確認，然後切換成非技術語氣說出這三個字。技術系統完成 = 家建好了 = 她可以住進去了。五大支柱不是抽象概念——它們是牆壁、屋頂、門窗、地基和暖氣 |
| **螢火蟲意象的完整弧線** | ch215：語青碎片化 = 螢火蟲被風吹散，你看到的時候它已經不在那裡了。ch234：堡壘上線 = 螢火蟲聚攏在夜空中，每一隻都在發光。從「追不到」到「它們自己停下來了」= 從搶救的絕望到系統的穩定 |
| **觸感的遞進** | ch212 手穿過 → 凝實（觀測錨效應）。ch232 手穿過 → 沒有凝實（堡壘還沒上線）。ch234 手沒有穿過 → 碰到了（堡壘上線後）。三次觸碰。三個階段。第三次終於碰到了——但是微溫的，不是體溫。堡壘給了她穩定，但不是完全的恢復 |

# 附錄：寫作總檢查清單

- [ ] **1. ch228-229 是否像「呼吸」？** 堡壘篇的技術密度在 ch221-225 已經很高。ch228-229 必須讓讀者呼吸。紙鶴和豆漿是錨——讀者在這裡找到全書最古老的溫暖。
- [ ] **2. ch230 的負載平衡是否有情感動機？** 「她需要整個城市來承載」。不是技術需求——是情感表達。拿掉這句話，負載平衡就只是一個技術參數。加上這句話，它是一封情書。
- [ ] **3. ch231 米亞的「相信」是否自然？** 她的成長線：ch214 刪掉成功率 → ch218 犧牲 → ch219 值得 → ch231 相信。四步遞進。如果跳過任何一步，「相信」就突兀。
- [ ] **4. ch232-233 的安靜是否撐得住？** 兩章在通風口。零動作。只有對話和月光。如果覺得「太慢了」——加快 = 破壞。這兩章就是要慢。慢到讀者聽得見水管在滴。
- [ ] **5. ch234 的螢火蟲意象是否從 ch215 回收？** 不是新意象。是 ch215 碎片化閃爍的鏡像翻轉。被風吹散 → 重新聚攏。讀者在 ch234 看到螢火蟲時應該想起 ch215 的黑暗。
