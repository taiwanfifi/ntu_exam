# 語青「偏移率遞增」追蹤表

> **設計核心**：語青從「先知」降格為「帶過期地圖的旅人」。偏移率是她人性化的刻度——越不準，越像人。
> **底層機制**：蝴蝶效應累積法則。語青的記憶來自 2091 年的時間線。維倫的每一個選擇（尤其是因愛而生的選擇）都讓兩條時間線進一步偏離。
> **最終意義**：當她不再能預知未來，只能握著你的手一起走——那才是真正的連結。

---

## 壹、偏移率公式與階段劃分

### 概念公式

```
偏移率 ≈ (時間差)² × (關鍵人物行動差異數)
```

愛是演算法無法預測的變數。維倫愛上語青在周明哲的所有計算中不應該發生——但每一條時間線裡都發生了。這個「不可預測的常數」是偏移率持續上升的根源。

### 四階段劃分

```
第一階段：先知期        ch1-20     偏移率 < 5%
第二階段：裂縫期        ch21-50    偏移率 5%-30%
第三階段：過期地圖期    ch51-70    偏移率 30%-60%
第四階段：失準期        ch71-80    偏移率 > 60%
─── 第一卷結束 ──────────────────────
第五階段：正式追蹤期    ch91-158   偏移率 31%-67%（米亞建檔）
第六階段：崩潰期        ch170-193  偏移率 78%-89%（檔案關閉）
第七階段：後追蹤期      ch204+     偏移率無意義——語青的存在本身開始不穩定
```

**第二階段寫法**：
- ch25-26 的「麵店」是第一個裂縫。用一句話帶過。不要讓維倫追問
- ch51 是轉折——語青自己說出「看來我帶的地圖開始過期了」
- 從 ch51 起，語青不再是引路人。兩人並肩

| 章 | 語青的預測 | 現實 | 偏移率 | 關鍵台詞 |
|----|-----------|------|--------|---------|

| 56 | 周明哲的意識分布範圍 | 比預期更廣 | ~40% | 米亞：「她的預測偏移率已經超過 34%。」|

| 65 | 周明哲會拒絕交出鑰匙 | 他主動交出（附帶警告）| ~53% | 維倫開始不問語青的「預測」|

**第三階段寫法**：
- 大事件方向仍然準確（周明哲在記憶網路裡、邱韻如還活著）
- **但細節越來越不可靠**——時間、數量、人物反應都偏差
- 語青開始說「不確定」、「也許」、「在另一條時間線裡是這樣，但——」

### 第四階段：失準期（ch71-80）

| 章 | 語青的預測 | 現實 | 偏移率 | 意義 |
|----|-----------|------|--------|------|
| 71-75 | 語青幾乎無法預測任何細節 | 事件走向與預期大幅偏離 | >60% | 她不再嘗試預測 |
| 75 | ★★★ 真正炸彈揭露 | 語青的真正目的被揭示 | — | 偏移率不再重要——身份危機取代 |
| 78-80 | 語青不再提供任何預測 | 維倫自己做判斷 | >70% | 兩人關係從「依賴」完成轉為「對等」|

**第四階段寫法**：
- 不再計算具體百分比——偏移率的概念已經完成了它的使命
- 語青不再是「先知」—— 她是「跟你一樣不知道未來的人」
- 這反而讓她更真實、更動人

## 參之二、Vol 2 偏移率場景級精修指令（Round 24 新增）

### 記錄點 0：ch91 鼻血（非正式記錄）

```
語青預測：鼻血持續 3 分鐘
現實：7 分鐘
偏移率：未計算（米亞此時尚未建檔）
```

**場景寫法**：
- 語青說「三分鐘」的語氣是自信的——這是她「先知殘留」的最後一次確定語氣
- 三分鐘過後沒停。語青開始在心裡算——讀者看到她「在算」= 裂縫的開始
- 七分鐘停了之後，語青說「不用算了」= 她不想面對。但米亞「還是算了」
- **關鍵台詞**：不要寫出偏移率數字。只有「她沒說話」和「米亞在報告底部記了一行」

### 記錄點 1：ch95 鋼琴師修復（★ 正式建檔）

```
語青預測：1 次深潛可修復
現實：需要 3 次深潛
偏移率：31%（米亞正式建立追蹤檔案）
```

**場景寫法**：
- 31% 這個數字要由米亞說出來。不是旁白
- 維倫問「要告訴她嗎？」→ 他開始保護語青 = 關係從「被引導」到「會擔心」
- 語青的「肩膀微微縮了一下。然後挺直了」= 用身體語言而非台詞傳遞「她知道」
- **台詞長度變化**：ch93 語青介紹鋼琴師案時，台詞長度 vs ch85 縮短約 20%。已經開始「說少一點」

### 記錄點 2：ch119 退役探員迷宮

```
語青預測：迷宮 5 層
現實：7 層
偏移率：42%
```

**場景寫法**：
- ★★ 維倫第一次正面質問語青：「妳以前不是這樣的。為什麼妳的預測越來越不準？」
- 語青沒有回答。沉默。這個沉默 ≥3 秒
- 此處不要讓語青解釋——解釋會降低懸念。她的沉默比任何解釋都有力
- **「在另一條時間線裡」使用頻率**：ch95 附近語青還會說這句話（約 1 次/3 章）。到 ch119 已經降到 0——她不再提「另一條時間線」了。因為提了也沒用

### 記錄點 3：ch136 嚴柏翰冷靜時間

```
語青預測：嚴柏翰 3 天內冷靜
現實：5 天仍拒絕跟語青說話
偏移率：53%
```

**場景寫法**：
- 偏移率突破 50% = 語青的預測「不如猜」
- 維倫「開始認真思考語青為什麼越來越不準」= 從「擔心她不準」到「懷疑她的存在方式」
- 語青的台詞長度此時已經是先知期的 40%——她說得越來越少
- **身體語言遞進**：ch95 肩膀微縮 → ch119 沉默 → ch136 她開始避開維倫的目光
- ch136 在 ch135 通風口場景之後。ch135 她剛說了 Vol 2 最美的話（「我記得所有在你身邊看過的東西」）。ch136 緊接偏移率 53% = 最美之後最痛

### 記錄點 4：ch158 嚴柏翰恢復後情緒（★ Vol 2 高潮）

```
語青預測：嚴柏翰恢復記憶後會暴怒
現實：他安靜地坐著。沉默。眼淚從臉頰滑到耳朵旁邊
偏移率：67%
```

**場景寫法**：
- 語青預測「暴怒」但嚴柏翰「安靜」= 完全相反。不是偏差——是方向性錯誤
- 語青「第一次在維倫面前哭了」。不是大聲的。是「眼淚無聲地流下來，像壞掉的水龍頭」
- 67% = 接近三分之二的預測是錯的。她帶的「地圖」已經是另一座城市的了
- **米亞更新檔案**用一句話帶過。不要讓米亞評論。數字本身就是評論
- **核心設計**：ch158 的偏移率記錄是在 ch156-157 的毀滅性情感場景之後。讀者還在心碎中——偏移率只是傷口上的鹽

## 伍、偏移率對維倫×語青關係的影響

### 關係轉變七階段

| 偏移率 | 維倫對語青的看法 | 關係定位 |
|--------|----------------|---------|
| 0-5% | 她知道一切。她是先知 | 依賴者 → 引路人 |
| 5-15% | 她偶爾不對……但大方向可靠 | 信任但開始留意 |
| 15-34% | 她的地圖過期了。但她仍然有用 | 合作者 |
| **34%** | **★ 轉折**：「我們得一起摸索」| **並肩 = 對等** |
| 42-53% | 她不是不準——她可能在崩潰 | 擔心 > 依賴 |
| 67-78% | 她快要失去自己了 | 保護者 → 被保護者逆轉 |
| **89%** | **她什麼都不知道了。但她還在** | **愛 ≠ 有用。愛 = 在** |

### 關鍵寫法規則

1. **偏移率上升 = 語青的台詞變短**。先知期她能說出長段分析；失準期她只說「我不確定」
2. **偏移率上升 = 維倫的主動性增加**。從「語青說怎麼做就怎麼做」→「維倫自己判斷」
3. **偏移率上升 = 兩人身體語言變近**。先知期保持距離；失準期並排；崩潰期握手
4. **米亞建檔（ch95）= 讀者才意識到「原來從很早開始就不對了」**

## 柒、寫作檢查清單

每次語青出場，確認以下項目：

- [ ] 她在這一章的偏移率大約是多少？（不需要精確——感覺對就好）
- [ ] 她的台詞長度是否符合偏移率？（越高越短）
- [ ] 她是否說了「在另一條時間線裡」？（先知期常說，失準期減少，崩潰期不再說）
- [ ] 這一章有沒有「身體記得」的時刻？（如果偏移率 > 30%，建議至少有一處）
- [ ] 維倫是主動還是被動？（偏移率 > 34% 後維倫應該是主動的）
