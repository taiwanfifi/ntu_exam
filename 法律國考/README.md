# 法律國考完全攻略 — 從零到上榜

> 專為非法律背景的轉職者設計，不需要任何法律基礎，只靠這套教材就能建立法律思維、通過國考。

---

## 這套教材是什麼？

這不是一份書單，也不是考古題的簡單彙整。這是一套**完整的法律思維訓練系統**。

如果你是工程師、業務、設計師、護理師——任何非法律背景的人——想轉職進入法律領域，這套教材就是為你設計的。

我們的目標：**讓你從「完全不懂法律」到「能像法律人一樣思考，並通過律師或司法官考試」。**

## 學習方法論

### 法律思維不是背法條，是學會一種「看世界的方式」

一般人看到車禍，想的是「誰撞誰」。
法律人看到車禍，想的是「誰對誰有什麼請求權？根據什麼法律基礎？對方可以主張什麼抗辯？」

這個思維轉換，就是我們要幫你完成的事。

### 我們的教學方法：像練數學一樣練法律

1. **先教算法（法律原理）**：用白話解釋每個法律概念，搭配生活案例
2. **再給基本題（標準考古題）**：看懂原理後，用真實考題練手感
3. **然後變化題（進階變體）**：改變條件，看你是否真的理解
4. **最後陷阱題（常見錯誤）**：為什麼不能這樣想？為什麼改一個條件結論就不同？

每一個法律爭點，都會有 3-5 個不同角度的案例，讓你徹底掌握。
