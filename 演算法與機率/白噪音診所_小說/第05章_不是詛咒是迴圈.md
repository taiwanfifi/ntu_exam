# 第五章：不是詛咒，是迴圈

---

下午兩點十七分，陳柏宇第三次走進白噪音心理諮商所。

前兩次他都在門口站了很久才進來。若嵐知道，因為門口的監視器有紀錄——第一次站了九分鐘，第二次站了四分鐘。今天，他只猶豫了四十秒。

這是一個好的趨勢。

柏宇今天穿了一件洗得發白的台大資工系T恤，背著一個塞得鼓鼓囊囊的書包，走進來的時候差點被門檻絆到。他穩住身體，苦笑了一下，像是在說「你看吧」。

「白醫師，我把你上次要我記的東西帶來了。」

他從書包裡掏出一本B5大小的筆記本，封面上用黑色麥克筆寫著三個大字：

**倒楣日記**

若嵐接過來。筆記本比她預期的重。

她翻開第一頁。柏宇的字跡工整得近乎偏執——每一行都壓在橫線上，沒有一個字超出格子。這種書寫習慣本身就在說話：控制感。他需要至少在某個地方維持秩序。

> **Day 1**
> 7:42 出門。跑步趕785公車，差三秒關門。等下一班，遲到12分鐘。
> 12:15 午餐。便利商店飲料架拿冰紅茶，開蓋噴出來，白襯衫毀了。
> 18:30 下課走回宿舍，經過新生南路，突然下雨。沒帶傘。

> **Day 2**
> 7:42 出門。又沒搭上785。
> 10:00 小考。四題選擇猜了兩題，全錯。
> 15:20 圖書館還書，發現逾期罰款87元。

> **Day 3**
> 7:42 出門......

若嵐沒有立刻說話。她一頁一頁翻下去，眼睛掃過每一條記錄。柏宇坐在沙發上，雙手交握，指節因為用力而微微發白。

沙發旁的空氣清淨機發出低沉的嗡嗡聲。午後的陽光從百葉窗的縫隙裡篩進來，在木質地板上畫出一排排平行的金色線條。

若嵐翻完三十頁。整整三十天，一天不漏。

「你很認真。」她說。

「我媽說我是被髒東西跟到了。」柏宇的聲音悶悶的，「她上禮拜帶我去行天宮收驚。師父說我最近犯太歲。」

「收驚有用嗎？」

「那天回家的路上，我踩到水溝蓋滑倒了。」

若嵐差一點笑出來。她忍住了——不是因為不該笑，而是因為她已經很久沒有這種想笑的衝動了。自從林哲宇死後，她以為自己把笑的機能也一起關掉了。

「柏宇，我可以把你的數據輸入電腦跑一下分析嗎？」

「呃......可以。」他猶豫了一下，「但你會發現什麼？」

「我不知道。這就是分析的意義——在看到結果之前，不預設答案。」

她打開筆電，啟動RStudio。柏宇好奇地湊過來看螢幕——畢竟是資工系的學生，看到程式碼會本能地靠近。

若嵐的手指在鍵盤上跳動，把三十天的記錄逐條輸入。她建了一個簡單的表格：日期、事件類型、發生時間、嚴重程度。

「你記錄得很詳細，」她一邊打字一邊說，「每天幾點幾分發生什麼事，全都有。這對分析來說非常好。」

「我是工程師嘛。」柏宇小聲說，「debug的習慣。」

若嵐停下手指，看了他一眼。這孩子剛剛說了一句比他自己意識到的更深刻的話。

*Debug的習慣。*

她把這句話記在心裡，繼續輸入數據。

---

十五分鐘後，數據全部進了系統。若嵐把筆電螢幕轉向柏宇。

「先看原始數據。三十天裡，你總共記錄了47件『倒楣事件』。我把它們分成五類——」

螢幕上出現一個彩色的表格：

| 事件類型 | 發生次數 | 單次機率估計 |
|---------|---------|------------|
| 沒搭上公車 | 22次 | 0.15 |
| 飲料打翻/食物出問題 | 8次 | 0.05 |
| 被淋雨 | 5次 | 0.03 |
| 考試/作業出錯 | 7次 | 0.10 |
| 其他（跌倒、罰款等） | 5次 | 0.08 |

「單次機率是怎麼估的？」柏宇問。他的眼睛已經亮起來了——這是他熟悉的語言。

「根據一般大學生的日常數據。」若嵐說，「台北市785公車在尖峰時段的誤點率大約15%。便利商店飲料出問題的機率，根據消保會的統計大約5%。被淋雨取決於你有沒有帶傘，但台北年均降雨天數換算下來，隨機一天遇到無預警降雨的機率大約3%。」

「所以每一件事單獨看，都不算特別倒楣？」

「對。每一件都是日常。」若嵐在鍵盤上敲了幾行指令，螢幕上開始跑計算。「但現在讓我們問一個關鍵問題——」

她在螢幕上打出一行公式：

```
P(至少一件倒楣事) = 1 - P(五件都沒發生)
                   = 1 - (1-0.15) x (1-0.05) x (1-0.03) x (1-0.10) x (1-0.08)
                   = 1 - 0.85 x 0.95 x 0.97 x 0.90 x 0.92
                   = 1 - 0.6363
                   ≈ 0.364
```

「一天之內至少發生一件倒楣事的機率，大約是36.4%。」若嵐說，「也就是說，平均每三天就有一天會遇到至少一件不順心的事。這很正常。」

柏宇皺眉。「但我是每天。連續三十天。」

「對。所以讓我們算連續三十天每天至少一件的機率——如果這些事件是獨立的話。」

她打出第二行：

```
P(連續30天，每天至少一件) = 0.364^30
                          ≈ 0.0000000024
```

螢幕上跳出那個數字。柏宇盯著看了五秒鐘，嘴巴微微張開。

「十億分之二點四。」他喃喃地說。

「你不是被詛咒。」若嵐說，語氣平靜得像在念天氣預報，「但0.364的30次方確實太小了。」

柏宇的表情很複雜——介於被驗證的欣慰和更深的恐懼之間。「所以......我真的是超級倒楣？宇宙級的倒楣？」

「不。」若嵐搖搖頭。她把螢幕再轉回來，打了幾行新的指令。一張圖表浮現——三十天的事件時間序列，每個點代表一次倒楣事件。

「你看這張圖。如果事件是隨機獨立的，散布應該像這樣——」她點開一個對照圖，上面的點隨機分散，有些日子密集，有些日子空白，「但你的數據長這樣——」

柏宇的圖表上，幾乎每一天都有事件，而且分布驚人地規律。早上七點四十二分左右一個cluster，中午十二點到一點一個cluster，傍晚六點到七點又一個cluster。

「看到了嗎？」若嵐用手指在螢幕上畫了三個圈，「不是機率有問題。是『連續三十天每天都中』這件事有問題。它在告訴我們一件事——」

她停頓了一拍。

「你的事件不是獨立的。」

柏宇眨了眨眼。「不是獨立的？什麼意思？」

「統計學上，如果兩件事是獨立的，一件事的發生不會影響另一件事的機率。但你的數據顯示出強烈的自相關——今天發生的倒楣事，和昨天發生的倒楣事之間存在關聯。」

她跑了一個自相關分析，螢幕上出現一排藍色的柱狀圖。lag-1的自相關係數高達0.82。

「0.82。」若嵐指著數字說，「完美獨立事件的自相關應該趨近於零。你的是0.82。這代表你昨天遇到的倒楣事，有超過八成的機會預測你今天也會遇到類似的事。」

「為什麼？」

若嵐把筆電闔上。她不需要螢幕了。她需要他聽。

「柏宇，你每天幾點出門？」

「七點四十二。」他不假思索地回答。

「為什麼是七點四十二？」

「因為......鬧鐘設七點二十，起床洗臉刷牙十五分鐘，穿衣服三分鐘，穿鞋子兩分鐘，下樓兩分鐘。七點四十二分到公車站。」

「每天都是七點四十二？」

「對。」

「一天都沒變過？」

「沒有。我......」他停住了。

若嵐看著他。窗外有一隻鴿子飛過，影子在百葉窗上一閃而逝。

「我設了鬧鐘。」柏宇說，聲音變小了。

「七點四十二分到公車站，」若嵐慢慢地說，「785公車的班距是十到十五分鐘。如果你每天同一個時間到站，而那個時間點剛好卡在兩班車的空檔——」

「我每天都會錯過同一班車。」柏宇的眼睛忽然睜大了。

「對。你不是運氣差。你是每天都在重複同一個條件。同一個時間出門，搭同一班你必然趕不上的公車。」

若嵐站起來，走到白板前面，拿起一支藍色白板筆。她畫了一條時間線，然後在上面標出三個節點。

「同一時間出門——同一班公車——同樣錯過。這是第一條因果鏈。」她畫了一條紅色的線連接三個節點。

「然後呢？你中午去哪裡買午餐？」

「學校對面那家全家。」

「每天？」

「......每天。」

「同一家全家——同一個冰櫃——」她在白板上畫了第二條線，「如果那個冰櫃的排水管有問題，導致某些飲料結冰後膨脹，開蓋就會噴，那你每天去拿，每次都可能中獎。」

柏宇的嘴巴張得更大了。

「傍晚走回宿舍呢？」若嵐繼續問。

「走新生南路。因為最近。」

「新生南路靠台大那一段，有好幾百公尺沒有騎樓。」若嵐在白板上畫了第三條線。「台北午後雷陣雨最常發生在五點半到七點之間。你每天六點半走那段路——」

「每次都淋到。」柏宇的聲音裡有一種奇怪的顫抖——不是恐懼，是某種正在碎裂又正在重組的東西。

若嵐放下白板筆，轉過身面對他。

白板上畫著三條紅色的因果鏈，它們沒有交叉，但都從同一個起點出發——一個被框起來的字：

**Routine**

「陳柏宇，」若嵐說，聲音放低了，不是冷淡，而是一種小心翼翼的溫柔，像是在拆除一個纏繞了很久的結，「你不是被詛咒。你是被自己的routine困住了。」

他沒有說話。

「你的『隨機』事件其實一點都不隨機。它們有一個共同的隱藏條件——你的行為模式。同一個時間，同一條路線，同一家便利商店。這些條件沒有變過，所以結果也不會變。」

「在統計學裡，這叫做違反獨立性假設。你以為每天是一次新的抽獎，但其實你每天都在用同一張彩券。當然每天都中同樣的獎——只不過是倒楣的獎。」

她停頓了一下，讓這句話沉下去。

柏宇低著頭，盯著自己的球鞋。他的鞋帶鬆了一邊，但他沒有去綁。

然後他笑了。

不是苦笑，不是自嘲。是一種忽然想通了什麼的、帶著一點傻氣的笑。那種笑讓若嵐想起很久以前的自己——在大學統計課上第一次理解中央極限定理時，那種「原來如此」的驚喜。

「所以......」他抬起頭，眼眶微紅但笑容燦爛，「我只要改變出門時間就好了？」

「改變出門時間，換一條路，試試不同的便利商店。」若嵐說，「打破routine，讓每一天的條件真正獨立。你的p值會降下來的。」

「我的p值會降下來。」他重複了一遍，像在品嚐一句咒語。然後他忽然笑出聲，「靠，這是我聽過最理工的祝福。」

這次若嵐真的笑了。

很小的笑，只有嘴角輕輕上揚。但在那個瞬間，診間裡的空氣好像暖了一度。

---

柏宇站起來收拾書包的時候，把那本倒楣日記留在了茶几上。

「這個......你不帶走嗎？」若嵐問。

「不用了。」他搖搖頭，背上書包，「我要開始寫新的日記了。叫『不倒楣日記』。」

他走到門口，手放在把手上，忽然停住了。

他回過頭。

逆光中，若嵐看不太清楚他的表情，但她聽見他的聲音很輕、很認真：

「白醫師，你是我遇過最怪的心理師。」

停頓。

「但......謝謝。」

門關上了。空氣清淨機的嗡嗡聲重新填滿了房間。

若嵐坐在原位沒有動。她低頭看著柏宇留下的筆記本，翻到最後一頁。在Day 30的記錄下方，柏宇用很小的字寫了一行附註：

> *我不知道這些記錄有沒有用。但至少寫下來的時候，我覺得有人會看。*

若嵐把筆記本闔上，按住封面，閉了一下眼睛。

有人會看。

她深吸一口氣，重新打開筆電。

---

螢幕上還留著柏宇的數據。自相關係數0.82，lag-1的柱狀圖高高豎起，像一根指控的手指。

若嵐沒有關掉這個頁面。她另外開了一個視窗，調出了另一組數據。

林哲宇的。

這組數據她已經看了無數遍。十四件事，發生在林哲宇死前的三天裡。她用同樣的方法跑過自相關分析——結果是0.11。幾乎為零。

她把兩組數據並列在螢幕上。

左邊是柏宇：自相關0.82。事件高度聚集在固定時段。原因清晰——行為模式的重複。

右邊是林哲宇：自相關0.11。事件分散在不同時間、不同地點、不同領域。工作上被主管無故責難。感情上被女友突然提分手。住處的熱水器壞了，房東三天不修。銀行帳戶被盜刷了一筆不大不小的錢——不到報警的門檻，但足以讓人焦躁。手機莫名其妙收到大量垃圾簡訊。社群帳號的追蹤者一夜之間少了三分之一。

柏宇的倒楣可以解釋。

因為自相關。因為重複的行為條件。改變routine，問題就會消失。

但林哲宇的呢？

若嵐把手指放在螢幕上，慢慢滑過那十四個數據點。

沒有自相關。沒有共同的時間模式。沒有重複的行為條件。這些事件涉及完全不同的生活領域——工作、感情、居住、財務、社交。它們之間沒有任何可見的因果鏈。

如果事件是真正獨立的，那十四件同時發生的機率小到荒謬。

如果事件不是獨立的，那隱藏的共同條件是什麼？

不是行為模式。林哲宇的生活沒有柏宇那種固定的routine——他是接案的軟體工程師，作息不規律，每天走不同的路線，去不同的地方吃飯。他的生活裡沒有可以製造自相關的結構。

所以那個共同條件，不在林哲宇身上。

在別處。

若嵐的手指停在螢幕上，指尖微微發涼。

*除非，有人從外部製造了那些條件。*

她閉上眼睛。不，她告訴自己。這個想法太跳躍了。你在把柏宇的案例過度延伸。柏宇的問題有一個簡單而溫暖的答案——改變行為就好。不是所有的異常都需要一個陰謀論。

但0.11的自相關和十四件事的聯合機率，像兩塊不吻合的拼圖，卡在她的腦子裡。

她正要關掉筆電，手機響了。

螢幕上顯示：**蘇瑾行**。

若嵐猶豫了兩秒。自從上次在咖啡廳不歡而散之後，她沒有主動聯繫過這個記者。他的調查方式讓她不舒服——太依賴直覺，太少數據支撐。但她不得不承認，他挖到的東西有時候比她的R語言跑得更快。

她接起來。

「白醫師。」蘇瑾行的聲音有一種壓抑的興奮，像一個獵人發現了腳印。「我找到林哲宇的女友了。」

「陳品蓁？」若嵐坐直了身體。

「對。她現在在高雄，跟她姊姊住。我飛下去見了她。」

「她怎麼說？」

「她說分手不是她的決定。」蘇瑾行停頓了一下，像是在斟酌措辭，「她說，她收到了一封匿名信。信裡有林哲宇出軌的證據——照片、對話截圖、開房紀錄。」

若嵐的呼吸慢了半拍。「那些證據是真的嗎？」

「她當時相信是真的。但現在——」電話那頭傳來翻動紙張的聲音，「她說她把那些照片拿給做攝影的朋友看了。朋友說有幾張的光影角度不太對，可能經過後製。」

「可能？」

「她原話是：『我現在很懷疑那些照片是假的。但我已經跟他分手了。然後他就死了。』」

蘇瑾行的聲音裡那股獵人的興奮消退了，取而代之的是一種沉重。

「白醫師，如果那些照片是偽造的，那這封匿名信——」

「就是十四件事裡的一件。」若嵐替他說完。

沉默。

若嵐的右手不自覺地握緊了左手腕上那只不走的錶。金屬錶帶冰冰涼涼的，秒針永遠停在十一點零七分——若晴的死亡時間。

「有人在系統性地拆毀林哲宇的生活。」她聽見自己的聲音說。那聲音很穩，但她的手在抖。

「我也是這麼想的。」蘇瑾行說，「不是巧合。不是運氣差。是有人刻意製造了那些『獨立事件』。」

「但為什麼？他只是一個普通的軟體工程師。」

「這就是我接下來要查的。」蘇瑾行頓了一下，「白醫師，妳還好嗎？」

若嵐張了張嘴，想說「我很好」。但這四個字在喉嚨裡卡住了——因為她忽然想起來，若晴死前也總是說這四個字。

「我在想一件事。」她說。

「什麼？」

「......沒什麼。晚點再說。」

她掛了電話。

---

診間裡很安靜。空氣清淨機的白噪音持續運轉，像一條永不停歇的河。

若嵐坐在沙發上，把左手腕舉到眼前。那只不走的錶在日光燈下反射出暗淡的光。錶面有一道很細的裂痕，是若晴摔倒時磕到的。若嵐用拇指摩挲那道裂痕，觸感像一條癒合不完全的疤。

五年前。

若晴死前的那個月，也是「突然很多事不順」。

若嵐曾經把它歸類為壓力。博士班的壓力、感情的壓力、跟指導教授的衝突。每一件事單獨看都說得通，合在一起也不過就是「那陣子比較倒楣」。

她身為心理師，身為雙胞胎姊姊，身為一個比任何人都擅長讀取數據的人——她什麼也沒看出來。

因為她當時只看了表面的敘事，沒有看底層的結構。

現在她看到了。

柏宇的倒楣是行為自相關。林哲宇的倒楣不是。那若晴呢？

若晴當時的「不順」——跟指導教授吵架、男朋友突然冷淡、論文被退回重寫、合租的室友搬走讓她付不起房租——這些事件之間，有自相關嗎？還是......

*不。*

若嵐用力搖了搖頭，像要把腦子裡的某個東西甩出去。

還不到想這個的時候。

她站起來，走到洗手台前，用冷水洗了把臉。水珠順著下巴滴落，在白色瓷磚上碎成幾瓣。她抬頭看鏡子——鏡子裡的自己臉色發白，眼眶下面有淡淡的青色。

若晴也有同一張臉。

她把水龍頭擰緊，回到辦公桌前，拿起筆。

在日誌本上，她寫下今天的諮商手記。手指握著筆，一筆一畫，每個字都壓在線上。就像柏宇一樣——在失控的世界裡，至少在紙上維持秩序。

## 白若嵐的諮商手記 #05

**日期**：（Day 38）

**患者**：陳柏宇，男，22歲，台大資工系

**主訴**：「我覺得自己被詛咒了。連續三十天，每天至少一件倒楣事。」

**數據摘要**：

三十天共記錄47件負面事件，分為五類。以獨立事件模型估計：

- P(每天至少一件) = 1 - (0.85 x 0.95 x 0.97 x 0.90 x 0.92) = 0.364
- P(連續30天每天至少一件) = 0.364^30 ≈ 2.4 x 10^-9

此機率值極端偏低，顯示「獨立性假設」被嚴重違反。

**自相關分析結果**：lag-1自相關係數 = 0.82（遠高於獨立事件的期望值0）

**診斷**：

事件並非獨立。患者每日行為模式高度固定——同一時間出門（07:42）、同一公車路線（785）、同一便利商店、同一步行路線。這些固定條件構成了「隱藏的共同變數（hidden common cause）」，使看似隨機的事件產生了強烈的自相關。

用白話說：他不是運氣差。他是每天用同一個方式去撞同一面牆。

**處方**：

改變行為模式的「初始條件」。調整出門時間（建議錯開尖峰）、更換通勤路線、嘗試不同的午餐地點。當底層條件改變，事件之間的自相關會下降，表觀機率也會回歸正常水準。

**諮商師備注**：

柏宇的案例教科書式地展示了一個常被忽略的統計陷阱——我們以為生活中的事件是獨立抽樣的骰子，但其實它們共享同一隻擲骰子的手。那隻手就是我們自己的習慣。

大多數人的「倒楣」不是命運的刻意為難，而是行為模式的無意識重複。改變命運的第一步，不是去廟裡拜拜，是改變你出門的時間。

但——

如果一個人的倒楣事件之間確實沒有自相關呢？如果那些事件涉及完全不同的領域，不存在任何共同的行為條件呢？

那就只有兩種可能：

一、那是極端罕見的真正隨機不幸。

二、有人從外部，人為地製造了那些看似獨立的條件。

後者的恐怖之處在於——受害者永遠不會意識到。因為每一件事單獨看都「說得通」，每一個打擊都像是日常的一部分。直到所有的巧合加在一起，壓垮他。

而到那時候，沒有人會覺得他是被謀殺的。

每個人都只會說：「他運氣真差。」

*——今天，柏宇在門口對我說謝謝的時候，我笑了。很小的笑。但那是林哲宇死後我第一次笑。*

*我不知道這代表什麼。但我希望這個數據點，是一個新趨勢的開始。*
