> **用途：** 餵給 AI 圖像生成工具（Gemini / Nano Banana 等），讓它理解我們的故事世界觀、角色外貌、場景氛圍，從而生成風格一致的動畫場景圖。

## 一、世界觀概述

### 時空背景
- **時間**：2089 年
- **地點**：新台北（Neo Taipei）
- **核心概念**：記憶已經被數位化。每個市民出生時在右耳後方植入一顆米粒大小的銀色記憶晶片，可儲存 500TB 的人生記憶。記憶可以被提取、儲存、交易、修復、移植、篡改。
- **社會現實**：記憶成為貨幣。有人靠賣記憶為生，有人因記憶負債而逐漸失去自我，最底層的「空白人」失去所有記憶，如行屍走肉般遊蕩。
- **美學定位**：台灣味賽博朋克。不是東京、不是洛杉磯——是台北。夜市還在（但也賣記憶）、機車還在（但全電動自駕）、珍珠奶茶還是手搖的。

### 城市四層結構

```

   空氣：無菌、薄荷、人工花香

   色調：深藍、純黑、冰冷白光
```

## 二、核心角色設定

### 1. 程維倫（W.L.）— 主角，記憶修復師
- **年齡**：28 歲
- **外貌**：178cm，偏瘦。黑色中長髮常亂翹，工作時用橡膠圈隨便紮。深棕色眼睛平時無神，修復時極度銳利如外科醫生。偏白皮膚（住地下街幾乎不見陽光），右手食指有舊燙疤。
- **穿著**：永遠是黑色帽T + 灰色工裝褲 + 室內拖鞋。出門加軍綠舊外套，口袋裡有記憶晶片和小螺絲起子。
- **氣質**：路上不會多看一眼的普通人，但手修長穩定、指甲剪得很短——做精密工作的手。
- **個性**：表面厭世話少，工作時極度專注如開鋒的刀，內心極度溫柔（會修味覺和嗅覺，大多修復師跳過不修）。面對林語青時笨拙柔軟（多買珍珠奶茶假裝買一送一）。
- **標誌物**：黑咖啡（工作時）、珍珠奶茶半糖少冰（放鬆時）、牆上貼滿修復失敗記錄的「恥辱牆」、一個寫著「台大電機」的接水塑膠杯。

### 2. 林語青 — 來自未來的幽靈
- **年齡**：26 歲
- **外貌**：165cm，纖細但不脆弱，有「挺直的疲倦」感。黑色直髮及肩，髮尾輕微分叉。深褐近黑的眼睛，目光穩定到令人不安（像在「確認」什麼）。偏蒼白如久未見陽光。
- **穿著**：永遠穿一件不合時宜的白色大衣（父親周明哲的舊實驗服）。口袋裡：一個記憶晶片、一把小刀、一張小女孩在公園笑的照片。
- **氣質**：從黑白電影裡走出來的女人。美，但不是讓人想靠近的美——是讓人想知道「她身上發生了什麼事」的美。
- **個性**：說話精準如程式碼。幾乎不表露情感（創傷後的冷靜，非天性）。偶爾觸碰太陽穴確認自己還在。
- **秘密**：她是兩條時間線的疊加態——記憶樹有兩個根。她可能不存在於物理現實中。

### 3. 嚴柏翰（代號「獵犬」）— 深淵監控局王牌探員
- **年齡**：35 歲
- **外貌**：183cm，精壯。短髮俐落，下巴線條硬朗。眼神像掃描器，冷而精準。左手背有一道長疤（他不記得怎麼來的——因為那段記憶被刪了）。
- **穿著**：深淵監控局標準黑色長風衣、軍靴。腰間別著手槍和高階記憶提取器（軍用級銀色髮箍）。
- **氣質**：黑色長風衣在走廊裡帶起風。整個人像一把未出鞘的刀。
- **個性**：冷硬、信任數據勝過信任人。但記憶被揭穿後開始龜裂——他曾是記憶解放陣線的核心成員，織網者的戀人。
- **秘密**：記憶被改過三次。十年前他站在抗議遊行的最前面。邱韻如親手刪了他的記憶——為了救他。

### 4. 邱韻如（代號「織網者」）— 雙面操控者
- **年齡**：45 歲
- **外貌**：170cm，優雅瘦削。銀灰短髮如戰盔，顴骨高挺，薄唇。永遠穿深紫或暗紅的高領修身套裝。左耳戴一顆黑色珍珠耳環（記憶晶片偽裝）。
- **氣質**：站在任何房間的中央，所有人都會自動往兩邊讓。權力的具象化。
- **雙重身份**：記憶銀行首席技術顧問 + 記憶解放陣線領袖。同時操控正邪兩方。
- **秘密**：她「消除人類痛苦」的信念本身是周明哲植入的。她以為自己是棋手，其實也是棋子。

### 5. 周明哲（零號病人）— 幕後之神
- **年齡**：外貌凍結在死亡時的 50 歲左右
- **外貌**：只出現在螢幕、全息投影、記憶中。普通的中年男人面孔，戴眼鏡，溫和的笑容。穿白色實驗袍。
- **存在形式**：他的意識分散式儲存在整個記憶網路中。不存在於任何一台伺服器，存在於所有伺服器。他就是記憶網路本身。
- **動機**：一個父親想讓失去記憶的女兒正常生活，結果扭曲了整個世界。

### 6. 米亞（MIA）— AI 夥伴
- **存在形式**：以一顆會變色的光球形態存在，漂浮在程維倫身邊。高興時金色，生氣時紅色，思考時藍色，打瞌睡時暗橘色。
- **性格**：毒舌、碎碎念、不承認關心程維倫（「他只是我的操作員」）。凌晨會偷偷把修復台溫度調高兩度。
- **核心**：由周明哲的父愛記憶驅動。但她對程維倫的關心已超越原始程式碼。

## 三、關鍵場景氛圍

### 程維倫工作室（最常出現的場景）
- 六坪小空間，舊台北地下街 Y 區出口旁
- 門上手寫小紙條：「W.L. 修復。預約制。」
- 內部：二手修復台（診療椅 + 三面環繞螢幕，左下角永遠有條亮線）、小廚房（電磁爐+咖啡機+泡麵）、滿牆的修復失敗列印稿、單人床、天花板水管外露滴水
- 氣味：焊接金屬味 + 黑咖啡 + 地下室潮濕 + 偶爾珍珠奶茶甜味
- 光線：日光燈慘白 + 螢幕藍光 + 米亞的暖色光球

### 記憶黑市「深夢」
- 西門町地下街最深處，入口偽裝成倒閉娃娃機店
- 推開最裡面灰色娃娃機背板 → 十七級鐵樓梯
- 2000 個攤位，電子菸甜膩煙霧 + 廉價記憶晶片化學味 + 壓低的交談聲
- 霓虹燈管在潮濕牆壁上投出粉紅/電子藍的光
- 各區：記憶大街（二手記憶）、技能巷、情感市場、禁區、修復街

### 記憶修復過程的視覺化
- 修復師坐在診療椅旁的操作台，委託人躺在椅上
- 三面螢幕顯示記憶的「數據視圖」：流動的光粒子、彩色數據流、樹狀分支圖
- 損壞的記憶 = 碎裂的光片、閃爍的錯誤紅光、扭曲的畫面
- 修復中 = 程維倫雙手在虛擬介面上如指揮家般精準操作
- 記憶空間（深潛時）= 真實場景但帶有數位毛邊，像一個會呼吸的夢境

### 新台北天際線
- 上城區：101 改建的「記憶之塔」高聳入雲，玻璃表面播放記憶銀行的全息廣告
- 中城區：密密麻麻的公寓、交織的霓虹招牌、滿街電動機車
- 多層高架橋、磁浮軌道穿梭、雨天反射霓虹光
- 視覺混合：九龍城寨密度 + 台北騎樓 + 賽博朋克霓虹

## 六、核心意象

1. **珍珠奶茶** — 程維倫唯一的奢侈品，人性溫暖的象徵
2. **銀色髮箍（記憶提取器）** — 像一頂輕薄王冠，科技入侵身體的視覺標記
3. **三面環繞螢幕** — 修復台的數位教堂，藍光中的莊嚴感
4. **白色大衣** — 林語青的盔甲，父親的遺物，兩個時間線的交界
5. **光球米亞** — 黑暗地下街中唯一的暖色漂浮光源
6. **記憶之塔（101）** — 從任何角度都看得見的壓迫性存在
7. **雨中的手** — 全書最後的意象，靈魂的肌肉記憶
