# 角色語音指紋 — 六角色聲線分化參考手冊

> **用途**：精修對話時的快速查閱參考。每個角色有獨特的語音指紋，即使遮住角色名也能辨認說話者。
> **核心原則**：壓力下聲線分化加劇，不是趨同。

---

## 一、程維倫

**句長上限**：≤ 3 句 / 輪
**壓力反應**：更沉默。身體敘事（手、呼吸、肌肉）取代語言。
**識別標記**：
- 感官先行：先寫他感覺到什麼，再寫他做什麼，最後才寫他想什麼（如果有的話）
- 內心獨白慣用「他不知道為什麼」——拒絕自我分析
- 對話極簡，常以動作代替回答（點頭、沉默、移開視線）
- 身體記得母題的主要載體

**範例**（正確）：
```
維倫沒有回答。他的手在發抖。
他把手插進口袋。
「走吧。」
```

**範例**（錯誤）：
```
維倫深呼一口氣，覺得這件事讓他陷入了深深的矛盾之中。「我覺得我們應該仔細想想這個問題的本質，因為——」
```

**禁止項**：
- ✗ 不用抽象詞（「矛盾」「本質」「意義」）
- ✗ 不說「我覺得」
- ✗ 不做長篇內心分析
- ✗ 壓力下不變話多

---

## 二、嚴柏翰

**句長上限**：無限制（他本來就話多）
**壓力反應**：更多話。語速加快。列舉加速。
**識別標記**：
- 「第一……第二……第三……」列舉結構
- 黑色幽默——越危險越諷刺
- 警察用語自然穿插（「嫌疑人」「現場」「證據鏈」「我跟你說」）
- 吃東西時最放鬆、最真實——對話常伴隨食物
- 罵人不帶髒字但精準

**範例**（正確）：
```
「第一，你沒有證據。第二，你連嫌疑人的名字都不確定。第三——」他咬了一口三明治，「第三，你剛才那個推論漏洞大到我可以開車進去。」
```

**範例**（錯誤）：
```
嚴柏翰輕聲說：「也許……我們應該從另一個角度思考這個問題。這讓我感到不安。」
```

**禁止項**：
- ✗ 不花瓣式抒情（除非真情流露的罕見時刻——那種反差才有力）
- ✗ 不用「也許」「或許」——他總是斷言
- ✗ 壓力下不變沉默

---

## 三、林語青

**句長上限**：碎片。大部分對話不超過 2 句。
**壓力反應**：更碎裂。語法瓦解。主詞消失。
**識別標記**：
- 答非所問：回應兩輪前的問題，而非當前問題
- 時間錯位：混淆「現在」和「那時候」
- 星辰意象：「星星偏了零點三度」「那顆星不在了」
- 句尾常以 `——` 或 `……` 斷裂
- 偶爾一句完整的、極其清晰的觀察——反差感極強

**範例**（正確）：
```
「……那天的雲比較低。」

維倫說的是明天的計畫。

「你有沒有覺得，天花板的影子——」她沒有說完。

過了很久。

「好。」
```

**範例**（錯誤）：
```
語青分析道：「因為記憶的結構是這樣的，所以我認為我們需要從根本原因出發，制定一個系統性的方案。」
```

**禁止項**：
- ✗ 不用完整邏輯句「因為……所以……」
- ✗ 不用「我認為」「我分析」——她不分析，她感知
- ✗ 不做長篇解釋
- ✗ 壓力下不變清晰

**驗證**：語青對話中 40%+ 以 `——` 或 `……` 結尾。

---

## 四、米亞

**句長上限**：雙模態
- 工作模式：精準、數字先行、語速快
- 個人模式：猶豫碎句、停頓（延遲簽名）、語速慢

**壓力反應**：退回純數據模式。所有情感碎句消失，只剩數字。
**識別標記**：
- 工作模式＝數字永遠在最前面（「87.3%。匹配度。」而非「匹配度是 87.3%」）
- 個人模式＝停頓秒數標記（0.3→0.4→1.2→2.1→4.1 秒，隨成長遞增）
- 延遲簽名：重要情感時刻會出現明確的停頓秒數
- 百分比吐槽：早期充滿，隨成長逐漸消失
- 比喻風格：Vol1 物理類比 → Vol2 人生類比 → Vol3 存在類比 → Vol4 哲學類比

**範例**（工作模式）：
```
「心率 147。血壓 92/61。偏移率 0.34 秒。」
頓了頓。
「建議終止。」
```

**範例**（個人模式）：
```
（停頓 2.1 秒。）
「我不確定——」
她的光球暗了一下。
「但我覺得……那叫思念。」
```

**禁止項**：
- ✗ 兩種模態不混用。工作時不猶豫，個人時不精準
- ✗ 不在數據報告中插入情感詞
- ✗ 不在情感時刻列精確數字（除非是刻意的反差）

**驗證**：米亞延遲簽名每重要場景出現 2-3 次。

## 五、周明哲

**句長上限**：5 句+（他習慣長篇獨白）
**壓力反應**：更精準。更控制。語域更正式。
**識別標記**：
- 「你要理解……」——教授式開場白
- 反問句密集：把問題拋回給對方
- 獨白式發言：一開口就是一整段，像講課
- 用詞正式、書面化（「倘若」「然而」「本質上」）
- 偶爾流露的溫柔——極其克制，只在維倫相關時

**範例**（正確）：
```
「你要理解，記憶的本質不是儲存——是重建。每一次你回想一件事，你都在重新建構它。那麼問題來了——」他摘下眼鏡擦了擦，「如果每一次回想都是重建，那什麼是『原始版本』？有原始版本嗎？」
```

**範例**（錯誤）：
```
「嗯……我不確定耶。也許可以試試看？」
```

**禁止項**：
- ✗ 不用口語（「耶」「啦」「嗯」）
- ✗ 不用不完整句
- ✗ 不表達不確定（他永遠有立場，即使是錯的）
- ✗ 壓力下不變隨意

**驗證**：周明哲平均對話輪長度 ≥ 其他角色 3 倍。

---

## 六、邱韻如

**句長上限**：中等（3-5 句）
**壓力反應**：問句更短更尖銳。
**識別標記**：
- 50%+ 對話行含問號——她永遠在提問
- 企業用語自然穿插（「風險評估」「利害關係人」「停損點」「ROI」）
- 茶的比喻：「這件事像普洱，放越久味道越不對」
- 不直接表態——用問題引導對方說出她要的答案
- 偶爾的溫柔藏在實用行動中（帶食物、安排住處），不說出口

**範例**（正確）：
```
「你確定？」她端起茶杯，「風險評估做過了？」
「這件事的停損點在哪裡？如果失敗，代價是什麼？」
她把一袋食物放在桌上，什麼都沒說，轉身就走。
```

**範例**（錯誤）：
```
邱韻如感嘆道：「我真的很擔心你們。這讓我想起小時候媽媽總是說——」
```

**禁止項**：
- ✗ 不直接表態立場（「我覺得你應該」→ 改成「你覺得這樣的 ROI 值得嗎？」）
- ✗ 不做長篇感嘆
- ✗ 不主動回憶過去（她刻意迴避）
- ✗ 壓力下不變囉嗦

**驗證**：邱韻如 50%+ 對話行含問號。

## 速查矩陣

| 維度 | 維倫 | 嚴柏翰 | 語青 | 米亞 | 周明哲 | 邱韻如 |
|------|------|--------|------|------|--------|--------|
| 句長 | ≤3句 | 無限 | 碎片 | 雙模態 | 5句+ | 3-5句 |
| 壓力 | 更沉默 | 更多話 | 更碎裂 | 退回數據 | 更精準 | 問句更尖 |
| 語域 | 感官 | 口語+警察 | 詩性碎片 | 數據/猶豫 | 書面正式 | 企業用語 |
| 標記 | 身體敘事 | 列舉+幽默 | 答非所問 | 延遲簽名 | 反問句 | 50%問句 |
| 禁止 | 抽象詞 | 花瓣抒情 | 邏輯句 | 模態混用 | 口語 | 直接表態 |

## 壓力場景辨識度測試

讀以下片段，應能在不看角色名的情況下辨認說話者：

> 片段 A：「……星星。今天的星星——不對。那是昨天的。」

> 片段 B：「第一，時間不夠。第二，人手不夠。第三——你那個計畫根本就是拿命在賭，我跟你說。」

> 片段 C：（停頓 1.8 秒。）「73.2%。不高。但——」光球暗了一下。「夠了。」

> 片段 D：「你要理解，這不是技術問題。本質上，這是一個倫理問題。你在用誰的標準定義『真實』？」

> 片段 E：他沒有回答。手指按在太陽穴上。呼吸。一次。兩次。「走。」

> 片段 F：「停損點在哪？」她放下茶杯。「如果這杯茶涼了你還沒做決定，我就當你放棄了。」

答案：A=語青、B=嚴柏翰、C=米亞、D=周明哲、E=維倫、F=邱韻如
