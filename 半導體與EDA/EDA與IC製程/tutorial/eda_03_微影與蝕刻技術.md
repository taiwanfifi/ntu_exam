## 🔰 本章基礎觀念（零基礎必讀）

### 微影是什麼？一句話

微影（Lithography）就是把設計好的電路圖案**「印」到晶圓上**，就像用印章蓋章一樣。

### 為什麼微影是最關鍵的製程？

- IC 製造的每一層都需要微影（一顆先進 IC 可能需要 **80～100 道微影**）
- 微影決定了**最小線寬**（Feature Size）→ 決定了製程節點（如 3nm, 5nm）
- 微影設備是所有半導體設備中**最昂貴的**（一台 EUV 曝光機超過 **1 億美元**）
- 微影製程佔整個 IC 製造成本的 **30～40%**

### 直覺比喻

| 微影步驟 | 生活比喻 |
|---------|---------|
| 光阻塗佈 | 在紙上塗一層感光材料 |
| 曝光 | 用投影機把圖案照上去 |
| 顯影 | 沖洗照片（被光照到的部分溶解或保留） |
| 蝕刻 | 用酸刻掉露出來的部分 |
| 去光阻 | 把剩餘的感光材料洗掉 |

## 一、光學微影流程

### 1.1 完整流程

```
                    光學微影流程
                    ===========

  ① 光阻塗佈          ② 軟烤              ③ 對準 + 曝光
  (Spin Coating)     (Soft Bake)          (Alignment + Exposure)
  ┌──────────┐      ┌──────────┐         ┌──────────┐
  │▓▓▓▓▓▓▓▓▓▓│ 光阻  │▓▓▓▓▓▓▓▓▓▓│        │▓▓░░▓▓░░▓▓│ ← 光照到
  │──────────│ 基板  │──────────│        │──────────│    的地方
  │ Si       │      │ Si       │        │ Si       │
  └──────────┘      └──────────┘        └──────────┘

  ④ 曝光後烤           ⑤ 顯影              ⑥ 硬烤
  (Post-Exposure     (Development)        (Hard Bake)
   Bake, PEB)
  ┌──────────┐      ┌──────────┐         ┌──────────┐
  │▓▓░░▓▓░░▓▓│      │▓▓  ▓▓  ▓▓│ 正光阻   │▓▓  ▓▓  ▓▓│
  │──────────│      │──────────│ 被照到   │──────────│
  │ Si       │      │ Si       │ 的溶掉   │ Si       │
  └──────────┘      └──────────┘         └──────────┘

  ⑦ 蝕刻               ⑧ 去光阻
  (Etching)            (Stripping)
  ┌──────────┐        ┌──────────┐
  │▓▓  ▓▓  ▓▓│        │    完成    │
  │──  ──  ──│ 露出   │──  ──  ──│ 圖案轉移
  │ Si Si  Si│ 的被蝕刻│ Si Si  Si│ 完成！
  └──────────┘        └──────────┘
```

### 1.2 正光阻 vs 負光阻

| 特性 | 正光阻（Positive） | 負光阻（Negative） |
|------|-------------------|-------------------|
| 曝光效果 | 被照到的部分**溶解** | 被照到的部分**硬化保留** |
| 顯影後 | 圖案與光罩**相同** | 圖案與光罩**相反** |
| 解析度 | **較好** ★ 先進製程使用 | 較差 |
| 應用 | IC 微影主流 | PCB、MEMS |
| 化學原理 | 曝光後變成可溶的羧酸 | 曝光後交聯(cross-link)硬化 |

> **記憶口訣**：正光阻 = 「照到就走」，負光阻 = 「照到就留」

### 1.3 光阻塗佈細節

- **旋轉塗佈（Spin Coating）**：滴光阻液 → 高速旋轉（3000～6000 RPM）→ 離心力展開成均勻薄膜
- 光阻厚度 $\propto 1/\sqrt{\omega}$（轉速越快越薄）
- 典型厚度：100 nm ～ 1 μm

## 二、解析度極限 ★核心考點

### 2.1 瑞利準則（Rayleigh Criterion）

光學微影的最小可解析特徵尺寸受到**繞射極限**限制：

$$\boxed{R_{min} = k_1 \cdot \frac{\lambda}{NA}}$$

- $R_{min}$：最小可解析線寬（Resolution）
- $k_1$：製程因子（理論極限 0.25，實際 0.3～0.8）
- $\lambda$：曝光光源波長
- $NA$：數值孔徑（Numerical Aperture）= $n \cdot \sin\theta$

### 2.2 焦深（Depth of Focus, DOF）

$$\boxed{DOF = k_2 \cdot \frac{\lambda}{NA^2}}$$

- $k_2$：另一個製程因子（~0.5）
- DOF 代表焦點允許的偏移量
- DOF 太小 → 製程困難（晶圓表面必須非常平坦）

### 2.3 解析度 vs DOF 的矛盾

| 想要更好解析度 | 需要 | 但 DOF 的變化 |
|--------------|------|--------------|
| 減小 $\lambda$ | 換光源 | DOF 也減小 ★ |
| 增大 $NA$ | 改鏡頭 | DOF 減小更快（$NA^2$） ★★ |
| 減小 $k_1$ | RET 技術 | DOF 可能也受影響 |

**核心矛盾**：追求高解析度會犧牲焦深 → 製程容許度（Process Window）變小。

### 2.4 光源波長演進

| 光源 | 波長 (nm) | 製程節點 | 時代 |
|------|----------|---------|------|
| g-line (Hg) | 436 | > 0.5 μm | 1980s |
| i-line (Hg) | 365 | 0.35～0.5 μm | 1990s |
| KrF 準分子雷射 | 248 | 180～250 nm | Late 1990s |
| ArF 準分子雷射 | 193 | 65～130 nm | 2000s |
| ArF 浸潤式 | 193（水中 NA↑） | 22～65 nm | 2007～今 |
| EUV | 13.5 | ≤ 7 nm | 2019～今 ★ |

### 2.5 浸潤式微影（Immersion Lithography）

在鏡頭和晶圓間填入**超純水**（折射率 $n = 1.44$），提升有效 NA：

$$NA_{immersion} = n_{water} \cdot \sin\theta \approx 1.44 \times 0.93 = 1.35$$

比乾式（空氣 $n = 1$）的 NA ≈ 0.93 提升了 **45%**！

---

### 數值例題 1：解析度計算

**題目**：ArF 浸潤式微影，$\lambda = 193$ nm, $NA = 1.35$, $k_1 = 0.30$。求最小可解析線寬。

**解答**：

$$R_{min} = k_1 \cdot \frac{\lambda}{NA} = 0.30 \times \frac{193}{1.35} = 0.30 \times 143 \approx \boxed{42.9 \text{ nm}}$$

這就是為什麼 193nm 浸潤式微影可以做到 ~40nm 的圖案。但要做到更小（如 10nm 以下），就需要**多重圖案化**或 **EUV**。

---

### 數值例題 2：DOF 計算

**題目**：承例題 1，$k_2 = 0.5$，求焦深。

**解答**：

$$DOF = k_2 \cdot \frac{\lambda}{NA^2} = 0.5 \times \frac{193}{1.35^2} = 0.5 \times \frac{193}{1.8225} = 0.5 \times 105.9 \approx \boxed{53 \text{ nm}}$$

焦深只有 53 nm！這意味著晶圓表面的高度變化必須控制在 ±53 nm 以內，非常嚴格。

---

### 數值例題 3：EUV vs ArF 比較

**題目**：EUV 微影 $\lambda = 13.5$ nm, $NA = 0.33$, $k_1 = 0.35$。ArF 浸潤式 $\lambda = 193$ nm, $NA = 1.35$, $k_1 = 0.30$。比較兩者的解析度和焦深。

**解答**：

**EUV**：
$$R_{EUV} = 0.35 \times \frac{13.5}{0.33} = 0.35 \times 40.9 \approx 14.3 \text{ nm}$$
$$DOF_{EUV} = 0.5 \times \frac{13.5}{0.33^2} = 0.5 \times \frac{13.5}{0.109} = 0.5 \times 123.9 \approx 62 \text{ nm}$$

**ArF 浸潤式**：
$$R_{ArF} = 0.30 \times \frac{193}{1.35} \approx 42.9 \text{ nm}$$
$$DOF_{ArF} = 0.5 \times \frac{193}{1.35^2} \approx 53 \text{ nm}$$

| | EUV | ArF Immersion |
|--|-----|---------------|
| 解析度 | **14.3 nm** ★ 好很多 | 42.9 nm |
| 焦深 | **62 nm** ★ 也較好 | 53 nm |

EUV 在解析度和焦深上都更好！波長的大幅縮短（193→13.5 nm）帶來巨大優勢。

## 三、光學增強技術（RET, Resolution Enhancement Techniques）

### 3.1 為什麼需要 RET？

當想要的圖案尺寸接近甚至小於光源波長時，繞射效應會嚴重扭曲圖案。RET 就是用各種技巧來彌補。

### 3.2 OPC（Optical Proximity Correction，光學鄰近校正）

**問題**：密集圖案和孤立圖案的曝光表現不同（鄰近效應）

**解法**：預先在光罩上加入修正圖形

```
    設計想要的圖案          光罩上的修正圖案
    ┌──────────┐          ┌──────────┐
    │          │          │  ┌┐  ┌┐  │  ← 加了 serif
    │ ████████ │    →     │ █████████│     （輔助特徵）
    │          │          │  └┘  └┘  │
    └──────────┘          └──────────┘
```

- **Serif**：在轉角加小方塊，防止圓角化
- **Hammer Head**：在線段端點加延伸，防止縮短
- **Sub-Resolution Assist Features (SRAF)**：加入比解析度還小的輔助圖案，改善主圖案品質

### 3.3 PSM（Phase Shift Mask，相移光罩）

**問題**：相鄰開口的光波互相干涉，影響對比度

**解法**：讓相鄰開口的光相位差 180°，利用破壞性干涉增強對比度

```
  傳統光罩：  │透│不透│透│不透│透│
  光波：      ∪  │    ∪  │    ∪   ← 相鄰開口同相位，中間對比低

  相移光罩：  │ 0°│不透│180°│不透│ 0°│
  光波：      ∪  │    ∩    │    ∪   ← 相鄰開口反相，中間「更暗」→ 對比度↑
```

類型：
- **交替式 PSM（Alternating PSM）**：最強的解析度提升，但設計複雜
- **衰減式 PSM（Attenuated PSM）**：半透過（~6% 透射率 + 180° 相移），較實用

### 3.4 OAI（Off-Axis Illumination，離軸照明）

**問題**：傳統正向照明（On-axis）繞射光資訊損失多

**解法**：改變照明方式，讓更多繞射光進入鏡頭

```
  正向照明          離軸照明
    │                ╲  ╱
    │                 ╲╱
    ▼                  ▼
  ┌───┐            ┌───┐
  │光罩│            │光罩│
  └───┘            └───┘
```

常見模式：
- **環狀（Annular）**：適用於各方向圖案
- **雙極（Dipole）**：適用於單一方向的密集線段
- **四極（Quadrupole）**：適用於水平 + 垂直線段

### 3.5 三種 RET 的比較

| 技術 | 解析度提升 | 複雜度 | 成本 |
|------|----------|--------|------|
| OPC | 中等 | 高（計算量大） | 光罩成本增加 |
| PSM | 最佳 | 很高 | 光罩成本大增 |
| OAI | 中等 | 低 | 改照明系統即可 |

實際上，先進製程**三者同時使用**（稱為 computational lithography）。

## 四、EUV 微影 ★先進製程關鍵

### 4.1 為什麼需要 EUV？

| 問題 | 說明 |
|------|------|
| 193nm 的極限 | 即使用了所有 RET + 多重圖案化，$k_1$ 也不能低於 0.25 |
| 多重圖案化太貴 | 做一層需要 2～4 次曝光 → 成本和時間倍增 |
| 解法 | 直接用更短波長：**13.5 nm**（極紫外光） |

### 4.2 EUV 的根本不同

| 特性 | ArF（193nm） | EUV（13.5nm） |
|------|-------------|---------------|
| 光學系統 | **折射式**（透鏡） | **反射式**（多層膜鏡面） |
| 環境 | 空氣或水 | **真空**（EUV 被任何物質吸收） |
| 光罩 | 透射式 | **反射式**（多層膜） |
| 光源 | 準分子雷射 | 錫(Sn)液滴 + CO₂ 雷射 |
| 鏡面 | 玻璃透鏡 | Mo/Si 多層膜反射鏡 |
| 功率挑戰 | 低 | **極高**（光源效率很低） |
| 設備價格 | ~$5,000 萬 | **> $1.5 億**（High-NA > $3 億） |

### 4.3 EUV 光源：LPP（Laser-Produced Plasma）

```
  CO₂ 雷射 ──→ 打擊錫(Sn)液滴 ──→ 產生電漿 ──→ 發出 13.5nm EUV 光
                                                      │
                    收集鏡 ←────────────────────────────┘
                      │
                    反射鏡系統（6～11 面）
                      │
                    反射式光罩
                      │
                    晶圓
```

### 4.4 EUV 多層膜反射鏡

每面鏡子由約 **40～50 對** Mo/Si 薄膜堆疊而成：
- Mo 層厚度：~2.8 nm
- Si 層厚度：~4.1 nm
- 一對週期：~6.9 nm（= 13.5nm / 2，布拉格反射）
- 單面反射率：~67%
- 6 面鏡子：$0.67^6 \approx 9\%$（非常低，這就是功率問題的來源）

### 4.5 High-NA EUV

下一代 EUV 將 NA 從 0.33 提升到 **0.55**：

$$R_{High-NA} = 0.35 \times \frac{13.5}{0.55} \approx 8.6 \text{ nm}$$

可以支撐 **2nm 以下**製程。但 DOF 會變小，需要更好的聚焦控制。

## 五、蝕刻（Etching）

### 5.1 蝕刻的目的

蝕刻是用化學或物理方法**移除**材料，將光阻上的圖案**轉移**到下方的薄膜上。

### 5.2 濕式蝕刻（Wet Etching）

- 用**化學溶液**溶解材料
- 例：HF 蝕刻 SiO₂，KOH 蝕刻 Si
- **等向性（Isotropic）**：各方向蝕刻速率相同 → 會有底切（undercut）

```
        光阻
    ┌──┤    ├──┐
    │  └────┘  │
    │ ╱        ╲│  ← 底切（undercut）
    │╱ 蝕刻區域 ╲│    等向性 → 側面也被蝕刻
    ├────────────┤
    │   基板      │
```

- **優點**：設備簡單、成本低、選擇比好
- **缺點**：等向性 → **無法做細線條**
- **用途**：清洗、犧牲層去除等

### 5.3 乾式蝕刻 / RIE（Reactive Ion Etching）★IC 製造主流

- 用**電漿（Plasma）**中的活性離子和自由基蝕刻
- **非等向性（Anisotropic）**：垂直方向蝕刻快，側面很慢

```
        光阻
    ┌──┤    ├──┐      離子 ↓↓↓↓↓ 轟擊
    │  └────┘  │
    │  │    │  │  ← 幾乎無底切
    │  │蝕刻│  │    非等向性 → 側壁垂直
    │  │區域│  │
    ├──┴────┴──┤
    │   基板     │
```

#### RIE 的原理

1. **化學反應**：活性自由基（如 F*, Cl*）與材料反應，產生揮發性產物
2. **物理轟擊**：離子（如 Ar⁺）垂直加速打到表面
3. **側壁保護**：反應副產物或添加氣體在側壁形成保護層（Passivation）

```
    反應氣體（如 CF₄, Cl₂, SF₆）
         │
         ▼
    ┌──────────────┐
    │   電漿腔體    │  RF 電源產生電漿
    │              │
    │  F*, Cl*,    │  ← 自由基（化學蝕刻）
    │  CF₃⁺, Ar⁺  │  ← 離子（物理轟擊）
    │              │
    │   ┌──────┐   │
    │   │ 晶圓  │   │  ← 放在下電極上
    └───┴──────┴───┘
```

### 5.4 蝕刻的關鍵參數

#### 選擇比（Selectivity）

$$\boxed{S = \frac{\text{被蝕刻材料的蝕刻速率}}{\text{不想被蝕刻材料的蝕刻速率}}}$$

- 選擇比越高越好（表示只蝕刻想要蝕刻的材料）
- 例：SiO₂ 對 Si 的選擇比 > 40:1（表示蝕刻 SiO₂ 比蝕刻 Si 快 40 倍）

#### 蝕刻均勻性（Uniformity）

$$\text{Uniformity} = \frac{R_{max} - R_{min}}{R_{max} + R_{min}} \times 100\%$$

- 希望晶圓上所有位置蝕刻速率一致
- 好的均勻性：< 3%

#### 蝕刻輪廓（Profile）

| 輪廓 | 說明 | 好壞 |
|------|------|------|
| 垂直（Vertical） | 側壁完全垂直 | ★ 最理想 |
| 漸縮（Tapered） | 側壁有斜度 | 有時刻意做（有利金屬填充） |
| 底切（Undercut） | 蝕刻超過光阻範圍 | ✗ 通常不好 |
| 微溝（Micro-trench） | 底部邊緣過度蝕刻 | ✗ 不好 |

---

### 數值例題 4：蝕刻選擇比應用

**題目**：需要蝕刻 200 nm 厚的 SiO₂ 層（下方是 Si 基板）。光阻厚度 300 nm。已知：
- SiO₂ 蝕刻速率：100 nm/min
- Si 蝕刻速率：5 nm/min
- 光阻蝕刻速率：50 nm/min

加上 20% 的過蝕刻（Over-etch）以確保完全蝕刻乾淨。求：
(a) 選擇比（SiO₂ 對 Si 和 SiO₂ 對光阻）
(b) 蝕刻時間
(c) 蝕刻後光阻剩餘厚度
(d) Si 基板被蝕刻掉多少？

**解答**：

**(a)** 選擇比：
- SiO₂ : Si = 100 : 5 = **20 : 1**
- SiO₂ : 光阻 = 100 : 50 = **2 : 1**

**(b)** 蝕刻時間（含 20% 過蝕刻）：
$$t = \frac{200 \text{ nm}}{100 \text{ nm/min}} \times 1.2 = 2 \times 1.2 = \boxed{2.4 \text{ min}}$$

**(c)** 光阻消耗量：
$$\text{光阻消耗} = 50 \times 2.4 = 120 \text{ nm}$$
$$\text{剩餘} = 300 - 120 = \boxed{180 \text{ nm}}$$

**(d)** 過蝕刻階段 Si 被蝕刻量：
過蝕刻時間 = $2.4 - 2.0 = 0.4$ min（只有過蝕刻的時間 Si 才被暴露）
$$\text{Si 消耗} = 5 \times 0.4 = \boxed{2 \text{ nm}}$$

## 六、CMP（Chemical Mechanical Polishing/Planarization，化學機械研磨）

### 6.1 為什麼需要 CMP？

經過多層沉積和蝕刻後，晶圓表面會凹凸不平。但微影需要**平坦的表面**（DOF 很小）。CMP 就是磨平表面的技術。

### 6.2 CMP 原理

```
    施壓
     ↓
  ┌──────┐
  │ 晶圓  │  （面朝下）
  └──┬───┘
     │     ← 研磨漿料（Slurry）：
     │        含奈米級研磨顆粒 + 化學溶液
  ═══╪═══════════════════
     │  研磨墊（Pad）  ← 旋轉
  ═══╧═══════════════════
```

- **化學**：溶液與材料反應，軟化表面
- **機械**：研磨顆粒物理磨除
- 兩者結合 → 高效且高品質的平坦化

### 6.3 CMP 的應用

| 應用 | 說明 |
|------|------|
| STI CMP | 淺溝槽隔離後磨平 SiO₂ |
| ILD CMP | 層間介電質平坦化 |
| 銅 CMP | 大馬士革製程中移除多餘的銅 ★ |
| Metal Gate CMP | 先進製程的金屬閘極平坦化 |

### 6.4 Preston 方程式

$$\boxed{RR = K_p \cdot P \cdot V}$$

- $RR$：移除速率（Removal Rate, nm/min）
- $K_p$：Preston 常數（取決於材料和漿料）
- $P$：施壓壓力
- $V$：相對速度

## ✅ 自我檢測

### 基礎題

<details>
<summary>Q1：正光阻和負光阻被曝光後的差異是什麼？先進 IC 製程通常用哪種？</summary>

**答案**：
- **正光阻**：曝光部分變得可溶，顯影後被沖掉 → 圖案與光罩相同
- **負光阻**：曝光部分交聯硬化，顯影後保留 → 圖案與光罩相反
- 先進 IC 製程用**正光阻**，因為解析度較好
</details>

<details>
<summary>Q2：提升微影解析度有哪三個方向？各有什麼代價？</summary>

**答案**：
1. **減小波長 λ**：從 193nm → 13.5nm (EUV)。代價：設備極其昂貴、技術困難
2. **增大數值孔徑 NA**：如浸潤式微影（水 n=1.44）。代價：焦深變小（$DOF \propto 1/NA^2$）
3. **減小 k₁**：用 OPC、PSM、OAI 等 RET 技術。代價：光罩複雜度和成本增加、計算量大
</details>

<details>
<summary>Q3：EUV 微影為什麼必須在真空中進行？光學系統為什麼是反射式？</summary>

**答案**：
- **真空**：13.5nm 的 EUV 光會被幾乎所有材料（包括空氣）吸收。即使幾公分的空氣路徑也會把光完全吸收
- **反射式**：同樣原因，EUV 光無法穿透任何透鏡材料，只能用反射鏡。反射鏡用 Mo/Si 多層膜實現布拉格反射，但單面反射率只有 ~67%
</details>

<details>
<summary>Q4：乾式蝕刻（RIE）如何實現非等向性（垂直側壁）？</summary>

**答案**：
三個機制的結合：
1. **離子垂直轟擊**：電場加速的離子垂直打到底部，增強底部的蝕刻速率
2. **側壁保護層**：反應副產物或添加的聚合物氣體在側壁形成保護膜（passivation），阻止側向蝕刻
3. **化學反應**：活性自由基（F*, Cl*）提供化學蝕刻，但因為底部被離子轟擊破壞了保護層，所以底部蝕刻快；側壁保護層完好，所以側向蝕刻慢

結果：底部蝕刻快 + 側壁幾乎不蝕刻 = **非等向性**
</details>

### 計算題

<details>
<summary>Q5：某製程使用 ArF 浸潤式微影（λ=193nm, NA=1.35）。若 k₁ 由 0.35 降低到 0.28（物理極限附近），解析度改善多少？</summary>

**答案**：

$R_1 = 0.35 \times 193/1.35 = 50.0$ nm

$R_2 = 0.28 \times 193/1.35 = 40.0$ nm

改善：$(50.0 - 40.0)/50.0 = 20\%$

解析度從 50nm 改善到 40nm，改善了 **20%**。但 k₁ = 0.28 已經極接近理論極限 0.25，需要同時使用 OPC + PSM + OAI 才可能達到。
</details>

<details>
<summary>Q6：若 CMP 的 Preston 常數 K_p = 3×10⁻¹³ cm²/dyne，壓力 P = 4 psi = 2.76×10⁵ dyne/cm²，速度 V = 50 cm/s。求移除速率（nm/min）。</summary>

**答案**：

$$RR = K_p \cdot P \cdot V = 3 \times 10^{-13} \times 2.76 \times 10^5 \times 50$$
$$= 3 \times 10^{-13} \times 1.38 \times 10^7 = 4.14 \times 10^{-6} \text{ cm/s}$$

換算：$4.14 \times 10^{-6}$ cm/s $= 41.4$ nm/s $= 41.4 \times 60 = \boxed{2484 \text{ nm/min} \approx 2.5 \text{ μm/min}}$
</details>

<details>
<summary>Q7：為什麼 TSMC 在 7nm 以下製程大量導入 EUV？用 ArF 多重圖案化不行嗎？</summary>

**答案**：
ArF 多重圖案化理論上可以做到很小的線寬，但有嚴重問題：
1. **成本**：每層需要 2～4 次曝光（LELE 或 SADP），成本倍增
2. **對準精度（Overlay）**：多次曝光之間的對準誤差會累積
3. **良率**：步驟越多，缺陷機率越高
4. **設計複雜度**：需要額外的分割（decomposition）步驟
5. **週期時間（Cycle Time）**：製造時間大幅增加

EUV 只需單次曝光就能達到相同解析度，雖然設備貴，但長期看總體成本更低、良率更高。這就是為什麼 TSMC 從 N7+（7nm 改良版）開始部分導入 EUV，N5 開始大量使用。
</details>
