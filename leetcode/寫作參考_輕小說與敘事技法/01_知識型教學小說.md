# 知識型教學小說 — 最直接的參考

> 這些作品成功地將 STEM 教育嵌入敘事中，是演算法教學小說最直接的範本。

---

## 1. 《數學少女》(Math Girls) ⭐ 首推

| 項目 | 內容 |
|------|------|
| **作者** | 結城浩 (Hiroshi Yuki) |
| **類型** | 數學教學小說 / 輕小說 |
| **規模** | 主系列 6 卷 + 「秘密筆記」衍生系列 15 卷 |
| **平台** | SoftBank Creative (日)；人民郵電出版社 (中譯) |
| **豆瓣** | https://book.douban.com/subject/26677354/ |
| **維基** | https://zh.wikipedia.org/zh-hans/數學少女 |

### 故事概要
男高中生透過與兩位女同學——天才型的米爾嘉和好奇型的蒂蒂拉——的對話探索數學。每卷聚焦一個重大數學概念（費波那契、費馬最後定理、哥德爾不完備定理、隨機算法、伽羅瓦理論、龐加萊猜想）。

### 寫作技法分析
- **對話×公式×敘事無縫交織**：文本中自然穿插數學推導，讀者在跟著角色對話的過程中不知不覺完成了證明
- **雙軌讀者設計**：數學強的讀者可以逐步跟著推導；對數學沒興趣的讀者也能享受人際互動
- **角色個性化教學風格**：
  - 米爾嘉 = 直覺跳躍型（像 quick sort，快但有時跳太遠）
  - 蒂蒂拉 = 步步確認型（像 bubble sort，慢但穩）
  - 「我」= 在兩者之間學習的讀者替身
- **內容跨度**：從國中算術到大學線性代數/數論，漸進式擴展

### 對我們的啟示
> **這是結構上最接近「演算法偵探筆記」的作品。**
> 關鍵差異：數學少女用「學校日常」驅動，我們用「偵探案件」驅動。
> 但核心手法相同：讓概念在角色互動中自然浮現。

## 2. 《數學超展開教室》

| 項目 | 內容 |
|------|------|
| **作者** | 賴以威（台大數學系教授） |
| **類型** | 校園數學教學小說 |
| **規模** | ~25 章 |
| **平台** | 台灣出版 |
| **巴哈心得** | https://home.gamer.com.tw/creationDetail.php?sn=4557841 |

### 故事概要
雲芳老師用學生的真實生活問題來示範數學思維。一邊是校園故事，一邊揭露老師的秘密身份——她其實是位知名作家。

### 寫作技法分析
- **由數學系教授親自操刀**，確保教學內容的專業度
- **生活化切入**：不是「來，我們學數學」，而是「你的問題，數學可以解決」
- **雙重懸念**：數學謎題 + 老師的秘密身份

### 對我們的啟示
> 證明了中文語境下「教學小說」這個形式是行得通的。
> 重點：用生活問題驅動，不是用考試驅動。

## 3. 《天才基本法》

| 項目 | 內容 |
|------|------|
| **作者** | 長洱 |
| **類型** | 數學+穿越+成長 |
| **規模** | 229 章（含 3 番外） |
| **平台** | 晉江文學城 (JJWXC) |
| **百度** | https://baike.baidu.com/item/天才基本法/23621736 |
| **晉江** | https://www.jjwxc.net/onebook.php?novelid=3193259 |

### 故事概要
22 歲的林朝夕面臨父親的阿茲海默症與自己的職業迷茫，觸發穿越回 12 歲。重新經歷數學競賽生涯，在三次時間循環中找到天賦與努力的真正意義。

### 寫作技法分析
- **數學做為敘事骨幹**：數學競賽不是背景裝飾，而是推動劇情的核心引擎
- **情感深度**：天賦 vs 努力、父女關係、自我認同——讓數學有了溫度
- **229 章的規模**證明這類題材可以長篇持續
- 已改編為熱門電視劇，證明商業可行性

### 對我們的啟示
> 229 章的數學主題小說成功運行，說明演算法小說完全可以做長篇連載。
> 關鍵：要有情感錨點，不能只有知識。

## 4. The Phoenix Project（鳳凰計畫）

| 項目 | 內容 |
|------|------|
| **作者** | Gene Kim, Kevin Behr, George Spafford |
| **類型** | IT 管理 / DevOps 教學小說 |
| **規模** | 35 章 |
| **平台** | IT Revolution Press；中譯本可得 |
| **Goodreads** | https://www.goodreads.com/book/show/17255186-the-phoenix-project |

### 故事概要
IT 經理 Bill 必須在 90 天內拯救一個嚴重超支的專案，否則整個部門將被外包。在導師 Erik 的指引下，他發現了 DevOps 的「三道」原則。

### 寫作技法分析
- **deadline 驅動的驚悚結構**：90 天期限製造持續緊迫感
- **導師-學徒模式**：Erik（導師）不直接給答案，而是用蘇格拉底式提問引導 Bill（學徒）自己發現
- **真實工作場景**：讀者邊讀邊想「這根本就是我們公司」
- **被企業廣泛用作 DevOps 教材**

### 對我們的啟示
> **「期限驚悚」+ 「導師提問」模式**非常適合演算法教學。
> 我們的「學姊」角色就是 Erik。案件的緊迫性就是 Bill 的 90 天期限。

## 5. 《算法少女》

| 項目 | 內容 |
|------|------|
| **作者** | 遠藤寬子 (Endo Hiroko) |
| **類型** | 歷史數學冒險 / 兒童文學 |
| **規模** | 15+ 章 |
| **年代** | 1974 年出版 |
| **維基** | https://zh.wikipedia.org/zh-hans/算法少女 |
| **豆瓣** | https://book.douban.com/subject/3617757/ |

### 故事概要
1775 年江戶時代，13 歲的千葉秋（醫生之女）發現一座寺廟的算額有誤，捲入了數學流派之間的競爭。

### 寫作技法分析
- **歷史冒險包裝數學教育**：不是現代教室，而是古代冒險
- **衝突來源自然**：數學學派之爭，比「上課考試」有趣一萬倍
- 獲三鷹兒童出版文化獎

### 對我們的啟示
> 書名就叫「算法少女」！證明「算法」可以是引人入勝的故事題材。
> 衝突不一定要來自考試/面試，可以來自更有趣的場景。

## 6. 《博士熱愛的算式》(The Housekeeper and the Professor)

| 項目 | 內容 |
|------|------|
| **作者** | 小川洋子 (Ogawa Yoko) |
| **類型** | 文學小說 / 數學小說 |
| **規模** | ~15 章 |
| **平台** | 新潮社(日)；獲本屋大獎第一名 |
| **博客來** | https://www.books.com.tw/products/0010903429 |

### 故事概要
一位家政婦被派去照顧一位只有 80 分鐘記憶的老年數學家。透過日常互動，親和數(220/284)、完美數、質數等概念成為人際連結的隱喻。

### 寫作技法分析
- **數學概念承載情感重量**：220 和 284 是「親和數」——兩個數字，各自的因數之和等於對方。這成了博士和家政婦關係的象徵
- **80 分鐘記憶限制**：天然的重複教學機制（每天重新認識，每天重新解釋）
- **溫暖、可親、人性化**的數學處理方式

### 對我們的啟示
> 數學概念可以有溫度。「親和數」不只是數論定義，它是一種關係。
> 「Hash collision」不只是碰撞——它是兩個本不相關的人被命運安排到同一頁。

## 7. JavaScript 百煉成仙

| 項目 | 內容 |
|------|------|
| **作者** | 楊逸飛（7 年 Java 開發者） |
| **類型** | 教學型 / 修仙 / 程式設計 |
| **規模** | 75 章 |
| **出版社** | 清華大學出版社 |
| **豆瓣** | https://book.douban.com/subject/35524665/ |
| **知乎討論** | https://www.zhihu.com/question/498434233 |

### 故事概要
主角「葉小凡」在修仙之旅中學習 JavaScript。函數（7 關）、jQuery、DOM 操作、Vue 基礎、ES6 語法等都被包裝成修煉內容。

### 寫作技法分析
- **修仙體系 = 知識體系**：每一層修為對應一個技術概念
- **耗時 3 年寫作**，是「程式教學小說」最直接的中文先行者
- **目標讀者**：JS 初學者，想要有趣的入門方式

### 對我們的啟示
> 中文程式教學小說已有先例。但「修仙」隱喻偏離現實。
> 我們的「偵探」隱喻更接近真實工程場景——debug、分析、推理。
> 這是優勢。

## 8. Dr. STONE 新石紀

| 項目 | 內容 |
|------|------|
| **作者** | 稻垣理一郎（原作）/ Boichi（作畫） |
| **類型** | 科學 / 冒險 / 教育 |
| **規模** | 26 卷，232 章（2017-2022，已完結） |
| **平台** | 週刊少年Jump |
| **維基** | https://zh.wikipedia.org/zh-tw/Dr.STONE_新石紀 |

### 故事概要
全人類被石化 3700 年後，天才科學家千空復活，從零開始用真實科學（化學、物理、工程）重建文明。

### 寫作技法分析
- **「從零建構」框架**：每個科學過程都是一個具體的冒險挑戰
- **教學不說教**：科學過程自然嵌入生存需求中
- **漸進式複雜度**：從肥皂 → 火藥 → 電燈 → 手機 → 火箭

### 對我們的啟示
> 「從零建構」的框架非常適合演算法教學：
> 從 O(n) 線性搜尋 → Binary Search → Hash Map → Graph → DP
> 每一步都因為「前一步不夠用」而自然引出。
