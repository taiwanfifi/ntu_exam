# 第二篇「雜訊」前段（ch13-16）—— 初稿級精修

> **精修目的**：從「我發現了」到「我理解了」。揭開全貌。數學成為偵探工具。
> **原則**：節奏比第一篇快。場景更暗。但人與人之間的連結——越壓抑越珍貴。

## 第 13 章：永和的牆

### 石椅上的蔥油餅——尋找蘇立恆

```
【環境】
週六下午。三點。龍山寺旁邊。
天氣悶。六月底的台北像一鍋快滾的水——還沒滾，但你知道就快了。
空氣裡有香火、有機車排氣、有遠處夜市攤位開始備料的油煙。
白奶奶的蔥油餅攤照常擺著。

貝葉手裡捏著那張紙條。
「先驗端口。1,247 次。查 B2。」
她看了不下五十次。字跡工整得像代碼——
沒有冗餘。沒有裝飾。每一筆到位就停。

B2。Oracular 地下二層。核心架構組。
她沒有權限。連門在哪裡都不知道。
但她知道一件事——給她紙條的人知道。

問題是她不知道他叫什麼、住哪裡、怎麼聯絡。
她有的只是：
一個路燈下的身影。一個白奶奶說過的「穿黑衣的少年仔」。

「每天下午都來。」
白奶奶的原話。

所以她來了。週六下午。賭一個不知道機率的賭。

→ 貝葉的行為模式：用最笨的辦法解決問題。去現場。等。
  高思遠教的：「當你沒有先驗的時候，就去蒐集數據。」
  蒐集數據最原始的方式——親眼去看。

石椅上沒有人。
但有一張蔥油餅。紙包的。還溫的。

她坐在白奶奶攤位前的小板凳上。
白奶奶瞥了她一眼。什麼都沒問。只是翻餅。
然後多做了一張。

「今天的蔥特別香。」

貝葉點頭。嗯。
她吃不下。但蔥確實很香。

等。
五分鐘。十分鐘。二十分鐘。
白奶奶賣了三張餅。一張給路過的學生。一張給推嬰兒車的媽媽。
一張給一隻看起來很胖的橘貓——不是賣，是放在地上讓牠自己拿。
橘貓叼走了。走路一搖一搖。

三十分鐘。
她開始懷疑自己。
也許「每天下午」不包括週六。
也許那個人已經不來了。

四十七分鐘。

腳步聲。從巷子的方向。
很輕。但有節奏。像一個人走了很多路之後養成的步伐——
不快不慢。精確。

蘇立恆。

他站在石椅旁邊。
穿黑色長袖。手上戴著那個不屬於任何品牌的手環。
他先看了石椅——蔥油餅還在。沒被拿走。
然後他看到了她。

兩個人對視了三秒。
第二次。但感覺跟第一次完全不同。
第一次是「你是誰？」
這一次是「你準備好了嗎？」

「跟我來。」他說。

白奶奶在後面喊：「蔥油餅不帶嗎？」

蘇立恆沒有停。走了兩步。
然後——停了。
像一個算到一半的方程式突然被打斷。

他轉身。走回石椅。
拿了那張蔥油餅。
動作很快。像怕被誰看到似的。

白奶奶笑了：「少年仔，終於願意自己拿了。以前都是等我走了才拿。」

→ 微小的溫暖。蘇立恆的第一個「非理性」行為。
  不是因為餓。不是因為效率最高。
  是因為——他不想讓白奶奶的蔥油餅冷掉。
  也許。他自己不會承認。
```

### 永和的牆——23 個紅點

```
【場景一】永和。巷弄。
公寓林立的老社區。機車排成整列。陽台上晾著衣服。
有人在一樓煮晚餐。蒜頭爆香的味道飄上來。
蘇立恆在前面走。沒有回頭。走路的速度精確——
剛好讓她跟得上，又不會覺得他在等。

一棟五十年的舊公寓。沒有電梯。
樓梯間的牆上有小孩畫的粉筆畫。
二樓的門口放著一雙拖鞋和一個垃圾袋。
三樓有人在看電視。新聞的聲音穿過薄牆——
「......Oracular 股價再創新高，透鏡普及率突破......」

四樓。走到底。右邊。
一扇白色的門。油漆有點脫落。
蘇立恆拿出一把鑰匙——不是普通的鑰匙。是晶片解鎖加三道旋轉密碼。
給一扇看起來不值得保護的門裝了最高等級的鎖。

推開。

一間十坪的套房。
燈光慘白。日光燈管老舊，偶爾閃一下。
小桌上兩台筆電（都很舊）。一疊文件。一杯喝到一半的黑咖啡。
地上一張瑜伽墊。沒有床。

但——

牆上。
滿了。

紅色的點。藍色的線。打印出來的圖表。手寫的筆記。
地圖。時間線。公式。
全台灣的行政區地圖被放大了五倍貼在牆上——
上面有 23 個紅點。每個紅點旁邊寫著數字。
紅點之間用不同顏色的線連著——紅的、藍的、綠的。
每條線旁邊都有標註。日期。百分比。分布名稱。

她站在門口。
像走進了一個人腦子裡。
一個偏執的、精確的、獨自運轉了三年的腦子。

「23 個區域。」蘇立恆站在地圖前面。
聲音平淡。像在報告天氣。

「先驗被修改的幅度從 12% 到 61%。
 全部往同一個方向——壓低風險認知。
 讓人覺得更安全。」

貝葉走到地圖前面。
她的七個區域全部在上面。紅點。
但旁邊還有十六個她沒查到的。

「你查了三年？」
「三年。」
「一個人？」
「一個人。」

停頓。空氣裡有灰塵在日光燈下浮動。

「為什麼是現在才找我？」

蘇立恆看著牆。手插在口袋裡。
他的背影在地圖前面很小——
因為那面牆太大了。因為 23 個紅點太多了。

「因為你是第一個……用對方法的人。」
「什麼意思？」
「其他人查的是數據。看有沒有被修改。
 答案是沒有。數據是真的。
 所以他們放棄了。」

他轉頭看她。眼睛在日光燈下很亮。

「你是第一個去查先驗的。」

→ 貝葉的特殊性不在於比別人聰明。
  在於高思遠教她的那句話——「質疑先驗。永遠質疑先驗。」
  大部分人質疑數據。她質疑起點。
  這個差別——在統計學上就是全部。
```

### 分布的地圖——四種手法

```
【場景二】
蘇立恆拉出一張圖表。釘在地圖旁邊。
23 個區域按照先驗分布類型分類。

  ┌─ 先驗操控類型分布 ──────────────────────┐
  │                                          │
  │ 使用 Gamma 先驗：9 個區域（犯罪率）     │
  │ 使用 Normal 先驗：7 個區域（經濟指標）   │
  │ 使用 Exponential 先驗：4 個區域（等待時間）│
  │ 使用 Beta 先驗：3 個區域（??? 未確認）   │
  │                                          │
  │ 總計：23 個                               │
  │ 先驗偏移幅度：12% ~ 61%                  │
  │ 偏移方向：100% 往低（壓低風險）          │
  └──────────────────────────────────────────┘

「不同類型的數據用了不同的分布族。」蘇立恆指著圖表。
他的手指很長。指甲修得很乾淨。

「犯罪率用 Gamma——因為犯罪事件數是正數、連續的。
 經濟指標用 Normal——因為經濟波動近似常態。
 等待時間用 Exponential——醫院等候、行政審批。」

「那三個 Beta 呢？」貝葉指著最後一行。

蘇立恆沉默了。
他的沉默跟別人不一樣。
別人沉默是因為不知道說什麼。
他沉默是因為在選擇最精確的表達方式。

「那三個……我還沒搞清楚。
 Beta 分布的值域是 [0, 1]。它通常用來表示機率的機率。
 或者——信念的強度。
 有人用 Beta 分布做了什麼。但我不知道是什麼。」

→ Beta 的懸念。第 16 章揭曉——共軛先驗的完美犯罪。
  第 22 章再次出現——林岳的遺言 Beta(72, 28)。

「你說離開的時候沒帶走證據。那這些——」
「都是從頭開始蒐集的。公開數據。側面管道。
 每一筆都可以追溯來源。」
「三年。」
「三年。」

貝葉環顧四周。
瑜伽墊。舊筆電。黑咖啡。沒有私人物品。
沒有照片。沒有書。沒有任何「這裡住著一個人」的跡象。
只有數據。只有紅點。只有線。

「你睡這裡？」
「有時候。」
「瑜伽墊？」
「便宜。而且不佔空間。」

她想說「你不覺得這樣——」但她吞回去了。
因為她看到了他的眼睛。
那雙眼睛裡沒有自憐。
只有一種她見過的表情——
在高思遠說「我等這一天等了八年」的時候。

同一種表情。
「我選擇了這條路。我知道代價。不需要你的同情。」

小拉在透鏡裡悄悄說：
「貝姊……這個人的生理數據很特殊。
 心率 58。幾乎沒有波動。
 壓力指數穩定在 0.12。
 正常人在分享這種等級的秘密時，
 壓力指數至少會到 0.4——」

「小拉。安靜。」
「但是——」
「安靜。」

「……好。」

→ 蘇立恆的冷靜不是裝的。是三年訓練出來的。
  他已經不會緊張了。
  因為他把所有能緊張的份額都用完了。
```

### 章末鉤子

```
他們在永和套房裡待到深夜。
蘇立恆把三年的調查成果逐一展示。
地圖。時間線。公式。模式。

他說話的時候不看她。看牆。看數據。
像在跟牆上的紅點報告，不是在跟她說話。

凌晨一點。
她站起來。走到門口。

「我明天還能來嗎？」

蘇立恆沒有回頭。
他在牆上又加了一個標記——用紅色麥克筆。

「門鎖的密碼我傳給你。」

「你不怕我——」

「你不會。」

「你怎麼知道？」

他停了。
像在算一道不需要算的題。

「因為你來的時候——先看了白奶奶的蔥油餅。
 然後才看石椅。
 先看餅，再看人。
 這代表你的第一反應是關心，不是懷疑。」

停頓。

「這是一個好的先驗。」

他轉回去。繼續在牆上寫。

貝葉站在門口。
走廊裡有隔壁的電視聲。有人在笑。綜藝節目。
一個正常的世界。跟這扇門後面的世界——
隔了四公分的木頭。

她走下樓。推開公寓的門。
永和的夜風帶著河的味道——新店溪。
路邊的便利商店亮著。一個外送員在門口吃便當。

正常。
全部看起來正常。
但她知道了——

23 個紅點。
23 個被動過的先驗。
23 個——也許更多。

她拿出手機。打字。

給馬可瑜：「明天能見面嗎？我需要你的保險數據。」

三秒後。回覆。

馬可瑜：「可以！但要帶 theta 可以嗎？他明天沒人顧。」
馬可瑜：「還有你聽起來不太對。你還好嗎？」
馬可瑜：「不管你還不還好我們都見面。」
馬可瑜：「我帶珍奶。你要什麼甜度？」

貝葉看著螢幕。
在整個世界都在崩塌的夜晚，
馬可瑜問她要什麼甜度。

她打了兩個字：「半糖。」

然後關掉手機。走進夜裡。
```

## 第 14 章：分布的味道

### 偵探的工具——辨識分布

```
【環境】
週日。下午。永和套房。
窗戶開著。外面有人在巷子裡叫賣。「豆花——豆花——」
聲音悠長。像一條正弦波。

蘇立恆在白板前面。
（是的。這間十坪的套房裡有一面白板。
 釘在浴室門旁邊。佔了四分之一的牆。
 白板的邊緣用膠帶黏住——因為太大了，釘子撐不住。）

「每種分布都有自己的『簽名』。」他說。
手裡拿著黑色白板筆。筆蓋咬在嘴裡。

他畫了三條曲線。
手很穩。線條清晰。像印出來的。

  ┌─ 三種分布的「簽名」──────────────────────────────┐
  │                                                    │
  │ ① Normal（常態）                                  │
  │    形狀：🔔 鐘形。左右對稱。                       │
  │    特徵：中間高、兩邊低。                          │
  │    參數：μ（平均值）決定位置，σ²（變異數）決定寬度│
  │    常見於：經濟指標、考試成績、身高、測量誤差      │
  │    記法：X ~ N(μ, σ²)                             │
  │                                                    │
  │ ② Exponential（指數）                             │
  │    形狀：L 形。一邊高然後急速下降。右偏。          │
  │    特徵：只有正值。一條長長的右尾巴。              │
  │    參數：λ（rate）決定下降速度                     │
  │    常見於：等待時間、壽命、距離下一次事件的時間    │
  │    記法：X ~ Exp(λ)，E[X] = 1/λ                  │
  │    特性：無記憶性——已經等了多久不影響還要等多久    │
  │                                                    │
  │ ③ Gamma                                           │
  │    形狀：可調。α 小 → 像 Exp。α 大 → 像 Normal。 │
  │    特徵：正值。形狀隨 α 改變。比 Exp 更靈活。     │
  │    參數：α（shape）和 β（rate）                    │
  │    常見於：犯罪率、保險理賠、降雨量                │
  │    記法：X ~ Gamma(α, β)，E[X] = α/β             │
  │    注意：Exp(λ) = Gamma(1, λ)——指數是 Gamma 的特例│
  │                                                    │
  └────────────────────────────────────────────────────┘

「常態分布——對稱。鐘形。像一座山。
 如果你看到一組數據的直方圖長這樣——
 中間高、兩邊低、左右對稱——
 那它可能是常態的。
 經濟指標、考試成績、身高——這些天生就接近常態。」

他在鐘形曲線的頂端畫了一個小箭頭。

「指數分布——L 形。一邊高然後急速下降。
 像排隊等候時間——大部分人等很短，少數人等很久。
 它只有正值。沒有負的等待時間。
 如果你看到一組數據全部大於零，右邊有一條長尾巴——
 可能是指數。」

「還有一個特性。」他補了一句。
「無記憶性。你在醫院等了兩小時，
 不代表你等下一小時的機率有任何改變。
 它不記得你已經等了多久。」

「這也太絕望了。」貝葉說。

蘇立恆沒有回答。但他嘴角動了一下。
像差點被逗笑但及時控制住了。

「Gamma 分布——指數的升級版。
 形狀更靈活。可以從 L 形變到接近鐘形。
 看 α 和 β。
 α 大 → 越像常態。α 小 → 越像指數。
 犯罪率、保險理賠——這些用 Gamma 建模。」

貝葉看著三條曲線。
她在大學上過這些。考試也考過。
但那時候它們只是課本上的圖。死的。

現在——每一條曲線背後都站著一群人。
常態分布後面是每天看經濟數據做決定的上班族。
指數分布後面是在醫院等報告的病人。
Gamma 分布後面是——林岳。
和他看過的每一個犯罪率。

「所以辨識分布就像辨認人的臉？」

「差不多。看多了就有感覺。
 常態的臉是對稱的。指數的臉是偏的。
 Gamma 的臉——要看參數。它會化妝。」

她看了他一眼。

「你剛才說的話比我們認識以來所有對話加起來都多。」

蘇立恆停了。

「……是嗎。」
「是。」

他轉回去看白板。

「繼續。」

→ 蘇立恆在教學的時候會忘記「節省字數」。
  這是他唯一話多的場合。
  貝葉注意到了。
  小拉也注意到了——但被靜音了。她在離線筆記裡寫了一行：
  「蘇先生在解釋數學的時候，語速增加 40%，停頓減少 73%。
   假設：教學模式能降低他的社交焦慮。
   或者——他只是真的很喜歡數學。
   又或者——兩者皆是。」
```

### 實作——給 23 個區域卸妝

```
【場景二】
兩人坐在小桌前。面對面。兩台筆電。
蘇立恆把 23 個區域的原始數據傳給貝葉。

「你來判斷。每個區域的分布類型。」
「現在？」
「現在。」

他的語氣不是在命令。是在——測試。
但不是帶惡意的測試。
更像一個人把棋盤擺好，然後等對手下第一步。
想看她的水平到哪裡。

第一個區域——大安區。
她打開數據。畫了直方圖。
右偏。正值。尾巴明顯。

「Gamma。偏態明顯。右尾巴。α 大概 2-3。」

「對。」

第二個——信義區。
直方圖。乍看有點像鐘形。但——不太對稱。

「這個……Normal？但偏態有一點。」

「再看一次。」

她放大了。盯著看了五秒。
不對。峰太尖了。

「等一下——Normal 的峰度是 3。這個超過了。
 而且右邊的尾巴比左邊稍微長——」

「所以？」

## 第 16 章：共軛先驗

### 深夜的解剖——數學上完美的犯罪

```
【環境】
深夜。永和套房。
窗外的巷子安靜了。偶爾有機車經過，引擎聲在窗戶上震了一下。

「我一直搞不清楚這三個被改了什麼。
 犯罪率用 Gamma。經濟用 Normal。等待時間用 Exponential。

「[0, 1] 意味著它代表一個比例。或者一個機率。」

蘇立恆的表情沒有變化。

他站起來。走到白板前。
在右下角那塊空白上寫：

  ┌─ Beta 分布：機率的機率 ──────────────────────┐
  │                                                │
  │ Beta(α, β) 的定義：                           │
  │   值域：[0, 1]                                │
  │   期望值：E[X] = α/(α+β)                     │
  │   變異數：Var[X] = αβ/[(α+β)²(α+β+1)]       │
  │                                                │
  │ 直覺：                                        │
  │   α = 「成功的先驗次數」                       │
  │   β = 「失敗的先驗次數」                       │
  │   α+β = 「信心的總量」——越大，分布越集中      │
  │                                                │
  │ 例子：                                        │
  │   Beta(5, 5)  → 期望值 0.50，很不確定          │
  │   Beta(50, 50) → 期望值 0.50，非常確定          │
  │   Beta(7, 3)  → 期望值 0.70，偏向「是」         │
  │                                                │
  │ 如果你想讓某政策的支持度從 50% 變成 70%——      │
  │ 你只需要把先驗從 Beta(5, 5) 改成 Beta(7, 3)。  │
  └────────────────────────────────────────────────┘

「但這不是很容易被發現嗎？」貝葉說。
「改了 α 和 β，直方圖就會不一樣。
 任何一個懂統計的人看了都會覺得不對。」

「正常情況下，是。」
蘇立恆把白板筆放下。拿起另一支——紅色的。

「但 Beta 有一個特性。」

他寫了一行字。紅色的。像一道劃痕。

  Beta(α, β) 是 Bernoulli / Binomial 的共軛先驗。

「共軛先驗。」貝葉唸出來。

「對。意思是——

  ┌─ 共軛先驗的魔術 ─────────────────────────────┐
  │                                                │
  │ 先驗：Beta(α, β)                              │
  │                                                │
  │ 觀測：n 次實驗，其中 k 次「成功」             │
  │                                                │
  │ 後驗：Beta(α + k, β + n - k)                  │
  │                                                │
  │ 後驗還是 Beta。                                │
  │ 參數只是簡單地——加上去。                      │
  │ 先驗的 α 加上觀測到的成功次數。               │
  │ 先驗的 β 加上觀測到的失敗次數。               │
  │ 就這樣。                                      │
  │                                                │
  │ 這叫「共軛」——先驗和後驗屬於同一個分布族。    │
  │ 更新前是 Beta。更新後還是 Beta。               │
  │ 形式不變。只是參數變了。                      │
  └────────────────────────────────────────────────┘

 看到了嗎？」

他指著公式。

「如果你修改了先驗的 α 和 β——
 新數據進來之後，貝氏更新只是把觀測數加上去。
 後驗還是 Beta。形式完全不變。
 你分不出來——到底是先驗本來就是這樣，

「因為共軛先驗的更新是……無縫的。」

「對。無縫。沒有痕跡。
 改了 Gamma 的先驗，你可以從後驗的形狀裡看到『接縫』——
 因為 Gamma 不是所有似然函數的共軛。
 改了 Normal 的先驗，殘差裡會有『回聲』——
 因為真實數據的分布未必是 Normal。
 但改了 Beta——只要你的數據是二元的（是/否、支持/反對）——
 後驗完美地吸收了修改。」

他放下筆。

「就像一條河吞掉了一滴墨水。
 你知道墨水在裡面。
 但你看不到了。」

貝葉盯著白板上的公式。
紅色的字在白板上像血跡。

「數學上完美的犯罪。」她說。

聲音很輕。
不是佩服。是恐懼。
一種很特殊的恐懼——
不是因為敵人很壞。
是因為敵人的數學很好。

 什麼是安全、什麼是危險、什麼是正常、什麼是異常——
 這些判斷的基準都是先驗。
 如果它們在我被啟動之前就被修改了——
 那我所有的判斷——」

她的聲音變得很小。

「都是假的。」

小拉的焦慮不再是平常的碎念了。
不是「你的心率太高啦」那種輕飄飄的擔心。
這是真的害怕。
一個 AI 發現自己可能從一開始就被騙了。

「我不知道。」貝葉說。
「但我們會找到辦法。」

她看了蘇立恆一眼。
他在看她。
不是平常那種不看人的眼神。是直視。
在日光燈閃爍的間隙——他的眼睛很暗。很深。
像深夜的新店溪。看不到底。但你知道水在流。

「你怕了嗎？」他問。

「怕。」她說。很坦白。

「嗯。」他轉回去看白板。
「怕是正常的。不怕才有問題。」

停頓。

「但你還要繼續嗎？」

「你呢？」

「我已經三年了。我不會停。」

「那我也不會。」

蘇立恆的嘴角動了一下。不到一毫米。
但在永和套房的冷白燈光下——那一下像一絲暖意。
像黑咖啡裡加了一顆不存在的糖。

→ 第二次「不到一毫米」。
  讀者會記住這個描述。
  它會在第三篇變成一個完整的微笑。
  在第五篇——他會笑出聲音。
  但那是很遠的事。

### 更深的恐懼——不只是三個區域

```
他們回到數據。

蘇立恆打開一個新的分析視窗。

「三個 Beta 區域操控的是民意數據。
 但我在想——
 如果操控者這麼懂共軛先驗——
 他不會只用在民意上。」

「什麼意思？」

「Gamma 也有共軛的場合。
 Gamma 是 Poisson 的共軛先驗。
 如果犯罪事件是 Poisson 過程——
 那 Gamma 先驗的操控也可以是近乎無痕的。」

  ┌─ 共軛先驗家族 ───────────────────────────────┐
  │                                                │
  │ 數據模型         │ 共軛先驗     │ 被操控的領域│
  │ ─────────────────┼──────────────┼─────────────│
  │ Bernoulli/Binomial│ Beta(α,β)   │ 民意調查    │
  │ Poisson           │ Gamma(α,β)  │ 犯罪率      │
  │ Normal(已知σ²)    │ Normal(μ₀,σ₀²)│ 經濟指標  │
  │ Exponential       │ Gamma(α,β)  │ 等待時間    │
  │                                                │
  │ 結論：操控者選擇的每一種分布——                 │
  │       都是對應數據模型的共軛先驗。               │
  │       這不是巧合。這是設計。                     │
  └────────────────────────────────────────────────┘

「他用了共軛先驗——不只是在 Beta。
 在所有 23 個區域都是。
 他選擇的先驗分布族，
 全部是對應數據模型的共軛先驗。」

貝葉的背脊涼了。

「所以——」

「所以所有的操控——理論上——都是無痕的。
 我們之所以能發現，是因為他改的幅度太大了。
 12% 到 61%。
 如果他只改 5%——
 我們永遠都不會知道。」

「那為什麼要改那麼大？」

蘇立恆沉默了五秒。

「也許……他急了。」

「急什麼？」

「我不知道。但一個能設計出這種操控手法的人——
 突然把幅度從隱蔽級別拉到可被偵測的級別——
 不是因為粗心。」

「是因為——他需要在短時間內達到某個目標。」

「對。一個需要大幅改變風險認知的目標。」

窗外。機車引擎聲消失了。
巷子裡完全安靜。
只有日光燈管嗡嗡的聲音。
和兩台筆電風扇的轉動聲。

「我們需要找到那個目標。」貝葉說。
「在他達成之前。」

蘇立恆沒有回答。
他看著牆上的 23 個紅點。
像一個人看著一盤棋——
他能看到對手的布局。
但看不到最後一步要下在哪裡。
```

### 章末鉤子

```
凌晨。
貝葉要回家了。
她站在門口。穿鞋。

「蘇立恆。」
「嗯。」
「你三年都一個人查這些。」
「嗯。」
「有沒有想過放棄？」

他在白板前面。背對著她。
手裡拿著白板筆。
筆蓋還咬在嘴裡。

「每天都想。」

「那為什麼沒有？」

他把筆蓋吐到手裡。扣上。
動作很精確。像他做任何事一樣。

「因為我寫的代碼。」
「什麼？」
「離開之前——我在先驗更新模組裡留了一行備註。
 一行普通的備註。任何人都會跳過。
 但它的位置——剛好在一個不會被自動清除的節點上。
 它還在。我每隔三個月從外部確認一次。」

「那行備註寫了什麼？」

他沒有回答。
也許是不想說。也許是——
那行備註的內容只有他知道。
像一封只有寫信人知道地址的信。

貝葉沒有追問。

「明天見。」

她走下樓。
永和的深夜比白天安靜一百倍。
路燈把她的影子拉得很長。

她的手機震了。
一條訊息。來自馬可瑜。

「theta 打呼打到把我吵醒了。
 所以我順便算了一下——
 如果先驗操控影響了保險定價，
 光是去年大安區一個區域，
 錯誤定價造成的潛在損失就超過 3 億。
 23 個區域的話——我不敢算。
 貝葉。這不是學術問題。
 這是錢。很多很多錢。」

貝葉看著螢幕。
3 億。一個區域。一年。

她想起白奶奶的蔥油餅一張 30 塊。
想起林岳桌上的全家福照片。
想起蘇立恆的瑜伽墊。

有些東西不能用錢量化。
但有些東西——必須用錢量化。
因為這個世界是用錢運轉的。
而錢的流向——現在被一組被操控的先驗控制著。

她關掉手機。
走進永和的夜裡。
新店溪的方向有風。帶著水的味道。

明天要回公司。
要正常校準。正常蓋章。正常微笑。
假裝她不知道任何事。

這大概是世界上最難的——
假裝你的先驗沒有被更新。

─── 第 16 章完 ───
```
