# 《機率預言師》角色語言風格 × 名場面台詞 × 母題回聲指南

> 寫作時隨時查閱的參考手冊。每個角色應該「聽起來」不一樣。

---

## 壹、八個核心角色的語言 DNA

---

### 陳貝葉

```
語言特徵：
  句式：中等長度。喜歡用問句——她在用語言思考。
  比喻系統：用校準的語言形容一切——
    「你的情緒基線偏了。」
    「我需要重新校準我對這件事的先驗。」
    「這個人的可信度......跟官方先驗一樣不可信。」
  情緒表達：由外而內。
    → 平靜時：正常語速，偶爾會跟小拉鬥嘴
    → 疑惑時：問句變多。反覆確認。「真的嗎？你確定？再算一次。」
    → 焦慮時：開始用手指在大腿上畫分布圖（無意識的習慣）
    → 憤怒時：話變少。語氣變平。像在唸實驗報告一樣冷靜。
              這比大吼更可怕。
    → 心碎時：沉默。然後走去跑數據。
              把情緒轉化成工作是她的防禦機制。

  她不說「我很害怕」。
  她說「這個結果的信賴區間太寬了」（意思：我不確定、我很不安）。
  她說「你的先驗偏了」（意思：你的判斷有問題/你搞錯了）。
  她說「數字不會說謊」（意思：你在騙我）。

  標誌性口頭禪：
  - 「你的先驗偏了。」（萬用——吵架、學術、吐槽）
  - 「數字不會說謊。但會被教壞。」（她的信念宣言）
  - 「再算一次。」（焦慮的時候）
  - 「......好。」（接受殘酷事實時的一個字）

  與不同角色的語言切換：
  - 對小拉：碎念模式。像跟室友鬥嘴。「小拉你閉嘴。」「小拉你又來。」
  - 對高思遠：放鬆模式。唯一會叫人綽號的對象。「老高，你又在裝深沉。」
  - 對蘇立恆：微妙地緊繃。措辭比平常精確 20%。小拉注意到了。
  - 對馬可瑜：最自然。會笑。會抱怨。會說廢話。
  - 對白奶奶：安靜。很少說話。只是吃蔥油餅。偶爾說「謝謝奶奶」。
  - 對費雪：冷。每個字都像在做假設檢定。

  禁忌：她不會突然變成演說家。她的力量在「安靜的堅定」，不在口才。
        如果你讓她在法庭上慷慨激昂地演講——重寫。
        她在法庭上的方式是：拿出數據。說出事實。然後沉默。
        沉默比演說更有力。
```

### 蘇立恆

```
語言特徵：
  句式：短。乾淨。像寫代碼——沒有冗餘，沒有註解。
  比喻系統：幾乎不用比喻。他用精確的技術語言描述一切。
    唯一的例外：在少數情緒流露的時刻，
    他會用「訊號」的語言——
    「你的訊號......比我預期的強。」
    「在雜訊裡，只有一個頻率是清楚的。」
  情緒表達：通過行動，不通過語言。
    → 他表達擔心的方式：出現在你需要的地方。不解釋。
    → 他表達信任的方式：給你密碼/權限/後門。
    → 他表達溫柔的方式：買一杯咖啡放在桌上。什麼都不說。
    → 他表達痛苦的方式：更安靜。如果可能的話——消失一陣子。

  標誌性特徵：
  - 「數據會說話。但你得問對問題。」
  - 「......」（他最常用的標點符號）
  - 平常看人不看眼睛。只有在認真的時候才直視。
    所以當他看著你的眼睛說話——那是他最誠實的時候。

  與不同角色的語言切換：
  - 對貝葉：話比平常多 30%。自己沒有意識到。小拉有。
  - 對高思遠：尊敬但不過度。「教授。」（一個字的問候）
  - 對柏朗：複雜。有憤怒也有曾經的尊敬。語氣在兩者之間震盪。
  - 對白奶奶：三年來零交流。直到最後走過去點了蔥油餅。
  - 對小拉：偶爾會被小拉的碎念逗到嘴角微微上揚。很快收回去。

  禁忌：他不會長篇大論解釋自己的動機。
        他做事。然後別人從他的行動中讀出動機。
        如果你寫了蘇立恆說超過四句連續的話——除非是關鍵揭露場景——重寫。
```

### 高思遠

```
語言特徵：
  句式：慢。像在下棋——每一步都考慮過。
  比喻系統：用日常生活的語言解釋深奧的概念。
    「機率就像地圖。地圖告訴你路在哪裡，
     但走哪條路——是你自己的事。」
    「先驗就像你帶上路的行李。
     帶太多走不動，不帶又會怕。」
  情緒表達：溫和但堅定。
    → 他從不發脾氣。但他「不說話」的時候比發脾氣更有壓力。
    → 他表達愛的方式是：每週打電話問你有沒有吃飯。堅持了二十年。
    → 他表達失望的方式是：「讓我想想......」然後沉默很久。
      你會希望他罵你。但他只是想。

  標誌性口頭禪：
  - 「機率不是命運。」（全書的核心哲學，從他口中說出最有重量）
  - 「你的先驗是什麼？」（他問所有人的第一個問題）
  - 「讓我想想......」（思考中。可能需要五分鐘。但答案一定值得等。）
  - 「吃飽了嗎？」（他最常見的開場白。跟白奶奶異曲同工。）

  特殊語言場景：
  - 跟貝葉在一起時：聲音更輕。像在跟自己的孩子說話。
  - 跟柏朗重逢時：聲音更沉。每個字都帶著三十年的重量。
  - 在教室裡：眼睛會亮。聲音會年輕十歲。教學是他的快樂。
  - 在醫院裡（晚期）：說話開始有停頓。咳嗽打斷句子。
    但眼神一樣清楚。他的身體在退化，但他的思維沒有。

  禁忌：他不會說「年輕人不懂」。
        他不會居高臨下。他尊重所有人的先驗——包括錯誤的。
```

### 馬可瑜

```
語言特徵：
  句式：快。密。像連珠炮。一口氣可以說五件事。
  比喻系統：把統計術語變成日常用語——
    「你的人生期望值太低了啦！要上調！」
    「這段關係的變異數也太大了吧。」
    「我跟你說，他對你好的機率是 Bernoulli(0.9)，超高的！」
  情緒表達：外顯。是全書最容易讀懂的角色。
    → 開心時：音量 +50%。會拉著人講話。
    → 生氣時：音量 +100%。會拍桌子。「你有沒有算過啊！」
    → 害怕時：會抱著 theta（柯基），用台語碎念。
    → 認真時：聲音會突然降下來。所有人都會安靜聽。
              因為馬可瑜安靜的時候——事情很嚴重。

  標誌性口頭禪：
  - 「我不管你的 p-value 多小，先把飯吃了。」
  - 「欸——」（所有句子的開頭。高頻率。）
  - 「theta 不要！那是我的論文！」
  - 「你知道嗎......」（認真模式的切換信號）

  特殊功能：
  - 她是「翻譯官」。當蘇立恆說了太技術的話，
    馬可瑜會用人話翻譯：
    蘇立恆：「KL 散度超過臨界值。」
    馬可瑜：「翻譯：那些數字假到不行。」
  - 她是「讀者的代言人」。
    當情節太沉重時，她的出現會讓氣氛輕鬆。
    但她也是「讓讀者意識到嚴重性」的角色——
    因為當她不再開玩笑的時候，讀者就知道：真的很嚴重了。

  禁忌：她不是搞笑工具。她的開朗是真實的。
        在適當的場合，她會展現深度和智慧。不要扁平化她。
```

### 柏朗

```
語言特徵：
  句式：慢。有重量。像法官宣判。每句話都是最終版——
  他在說出口之前已經在腦中校對過三次了。
  比喻系統：用「系統」和「最優化」的語言——
    「我不是在操控。我是在校準。」
    「自由意志是系統中最大的雜訊。」
    「你們看到的是後驗。我看到的是先驗。」
  情緒表達：幾乎不表達。
    → 他的情緒藏在微表情裡——嘴角的弧度、眉頭的角度。
    → 他憤怒的方式：更安靜。更禮貌。比蘇立恆更可怕。
    → 他唯一真正流露情感的時刻：
      ① 談到父親（眼神會失焦兩秒。然後立刻恢復。）
      ② 面對高思遠（聲音會微微軟化。只有一點點。但足夠。）
      ③ 最終戰被貝葉說服時（哭了。五十五年來第一次。）

  標誌性口頭禪：
  - 「我不預測未來。我創造未來。」
  - 「你的先驗是什麼？」（跟高思遠一樣的問法——師承明顯。但動機不同。）
  - 「有些事比你想像的更複雜。」（推託用語。但他真的覺得。）

  柏朗的「灰色」怎麼寫？
  - 永遠不要讓他做「明顯的壞事」。
    他的每一個行動都有「合理的理由」。
    讀者應該在讀完之後想：「等等，他好像不完全是壞人......」
  - 讓他的動機跟貝葉有鏡像關係：
    貝葉因為父親的死而相信「機率可以拯救人」。
    柏朗因為父親的死而相信「機率必須被控制」。
    同一個傷口，不同的結論。
  - 他跟貝葉的對話應該是全書最「理性」的對話。
    兩個人都不吼。都不哭（直到最後）。
    只是一句一句地拆解對方的論點。
    像兩個統計學家在做假設檢定——互相檢定對方的先驗。

  禁忌：不要臉譜化。不要讓他「邪笑」或「陰謀得逞的表情」。
        他始終相信自己是對的。直到最後一刻——也許。
```

### 費雪

```
語言特徵：
  句式：命令式。短促。像軍事通訊。
  但在「必要」的時候能切換成優雅的辯論模式——法庭上。
  比喻系統：不用比喻。她用的是統計學的術語，但帶著權力的味道——
    「統計顯著性不代表實際顯著性。但在這裡，我說了算。」
    「你的證據不夠顯著。」（= 你威脅不了我）
    「我的 α 設在 0.01。」（= 除非你 99% 確定，否則不要挑戰我）
  情緒表達：幾乎為零。
    → 但在極少數情況下——她會打開抽屜看那枚國安局胸針。
    → 那是她的「先驗校準儀式」。提醒自己為什麼在這裡。

  標誌性口頭禪：
  - 「給我數據。不要給我感覺。」
  - 「我不是你的敵人。你的敵人是混亂。」
  - 「......有趣。」（她覺得有趣的時候最危險。因為她在計算。）

  費雪 vs 貝葉的語言對比：
  - 貝葉：「數字不會說謊。」
    費雪：「數字不會說謊。但解讀會。」
  - 貝葉用統計找真相。
    費雪用統計維持秩序。
  - 她們其實很像。這是最可怕的地方。
    在不同的人生軌跡上，貝葉可能變成費雪。費雪可能變成貝葉。

  禁忌：她不是「邪惡的女上司」。她是一個有原則的人，
        只是她的原則跟貝葉不同。
        讀者應該在某些時刻覺得「也許費雪是對的」。
```

### 白奶奶

```
語言特徵：
  句式：口語。台灣味。混合國語和台語。
  有時候句子不完整——因為她覺得不需要說完。
  比喻系統：用食物和天氣。
    「你看起來像沒加鹽的麵。有形但沒味道。」
    「今天的天像醒發過頭的麵團——軟軟的，要塌不塌的。」
  情緒表達：全身體。
    → 開心時：手不停。一邊做餅一邊哼歌。
    → 擔心時：給你多加蔥花。不說為什麼。
    → 生氣時：鐵板拍得特別大聲。「哼！」
    → 溫柔時：不看你。但多做了一張餅放在旁邊。

  標誌性口頭禪：
  - 「吃飽了再想。空腹的人想出來的東西都不準。」
  - 「我做蔥油餅四十年，每一張都不一樣。這就是你們說的那個隨機？」
  - 「那個穿黑衣的少年仔......」（指蘇立恆。三年來從不叫他名字。）
  - 「你爸以前也這樣。」（對貝葉說。每次都讓貝葉愣住。）

  白奶奶的功能：
  - 她是全書的「零點」——不被透鏡影響的基準。
  - 她的場景 = 讀者深呼吸的時刻。緊張情節之後，放一段她的場景。
  - 她代表「不需要機率也能活得好」的證明。
  - 她的人生智慧不是「哲學」——是「四十年做蔥油餅累積的直覺」。
    這種直覺不需要貝氏更新。它就是先驗本身。

  禁忌：不要讓她說「深刻的哲理名言」。
        她的智慧必須是口語的、不完整的、甚至有點好笑的。
        但讀者看完會想很久。
```

### 小拉（Laplace）

```
語言特徵：
  句式：碎。快。帶括號。帶省略號。帶百分比。
  像一個邊算邊碎念的實習生——
  「這個......以 95% 信賴區間來看，不對，93%......
   嗯......好吧，大概是這個範圍啦（但我不太確定）」
  比喻系統：用辦公室的語言——
    「貝姊你的工時又超標了......HR 不管嗎......
     喔對他們也不管我的。因為我是 AI。」
    「這筆數據的品質跟公司的免費咖啡一樣——
     勉強能用但不要有期待。」
  情緒表達：焦慮是預設值。
    → 但焦慮的「類型」會變：
      一般焦慮：「嗚嗚嗚這個數據好可怕」
      真正擔心：反應速度快 0.3 秒。不碎念。直接報告。
      被感動：停頓 2.7 秒。然後說一句否認：
      「才不是被感動呢。只是......計算能力暫時受到干擾。」
    → 成長後（第四篇起）：
      焦慮變少。但偶爾會刻意焦慮——因為那是她的個性，不是設定。
      「我選擇焦慮。因為不焦慮的話我就不是我了。」

  標誌性口頭禪：
  - 「貝姊——」（被貝葉說不要這樣叫之後）「好的貝姊。」
  - 「以 95% 的信賴區間來看......不對也許是 93%......」
  - 「才不是擔心你呢。只是我的計算效率在你平安的時候比較高。」
  - 「......數據已經存起來了。」（偷偷記錄貝葉和蘇立恆的互動數據）

  小拉的成長弧線在語言上的體現：
  第一篇：「這是我算的但我不太確定結果如果錯了不要怪我......」
  第二篇：「這個結果......我有 73% 的信心。剩下的 27% 讓我很不安。」
  第三篇：「如果刪掉回報模組我會死掉的話......那就死掉吧。」
  第四篇：「我現在非常確定。好吧，其實是 98% 確定。但那 2% 是故意留的。」
  第五篇：「這次的計算結果我非常確定。......好吧其實我還是有一點不確定。
          但我選擇相信。」

  禁忌：不要把她寫成「冷冰冰的 AI」。也不要寫成「完全像人類」。
        她是一個「正在學習情感但用數據語言表達」的存在。
        她的獨特性在於——她的焦慮是真實的。不是演出來的。
        即使她自己不確定那算不算「真實」。
```

## 貳、母題回聲系統

> 母題 = 反覆出現的意象/台詞/動作，每次出現都帶有新的含義。

---

### 母題一：「你的先驗是什麼？」

```
第一次出現（第 2 章）：高思遠問貝葉。
  → 學術問題。課堂上的提問。

第二次出現（第 12 章）：高思遠得知貝葉的發現。
  → 哲學問題。「你相信什麼？」

第三次出現（第 36 章）：柏朗在學術會上問貝葉。
  → 挑釁。「你憑什麼質疑我的先驗？」

第四次出現（第 58 章）：貝葉反問柏朗。
  → 最終武器。「你的先驗——是恐懼。」

第五次出現（尾聲）：透鏡 v5.0 的選擇介面。
  → 每個人回答自己。「你的先驗是什麼？」
```

---

### 母題二：蔥油餅

```
第一次（第 3 章）：白奶奶給貝葉做蔥油餅。日常。溫暖。
第二次（第 12 章）：蘇立恆坐在對面石椅。多了一張沒人買的餅。
第三次（第 26 章）：團隊在攤位後面開會。「做大事的人吃得多。」
第四次（第 50 章）：最困難的時刻。貝葉去買蔥油餅。白奶奶什麼都沒問。
第五次（尾聲）：蘇立恆終於走到攤前。白奶奶：「擦了三年了。」
→ 蔥油餅 = 不需要機率也存在的溫暖。全書的情感錨點。
```

### 母題三：心率同步

```
小拉偷偷記錄的數據。

第一次（第 20 章）：凌晨三點。蘇立恆坐下後。92 → 68。
第二次（第 32 章）：裸眼酒吧。關掉透鏡。小拉在關機前最後記錄了一筆。
第三次（第 48 章）：手指碰到手指。心率同步上升 23%。小拉選擇不說。
第四次（第 55 章）：「習慣。」心率 +31%。小拉忍住沒說。
第五次（尾聲）：「兩位的心率再度同步。我真的覺得你們——」被打斷。
→ 心率同步 = 兩個人的「後驗」在趨近。他們正在收斂到同一個結論。
```

### 母題四：11 秒

```
貝葉父親的車禍。如果提早 11 秒出門，存活率從 2% → 97%。

第一次（第 4 章）：貝葉的回憶。解釋她為什麼加入 Oracular。
第二次（第 36 章）：學術會上。貝葉在緊張的時候閉眼數了 11 秒。
第三次（第 48 章）：法庭上。費雪的最後反問讓她沉默了 11 秒。
第四次（第 59 章）：面對柏朗。她說完「不出海的世界」後，
                    柏朗沉默了 11 秒。然後哭了。
→ 11 秒 = 改變一切的時間。也是「不確定性」最具體的體現。
```

### 母題五：Beta 分布

```
Beta 分布在數學上代表「信念的強度」。
Beta(α, β) 的 α 越大，越相信事件會發生。

在故事中，Beta 分布不只是工具——是角色內心的隱喻。

林岳的遺言：Beta(72, 28) → 他 72% 相信自己發現的是真相。
貝葉心中的 Beta：
  第一篇：Beta(2, 8) → 只有 20% 相信透鏡有問題。
  第二篇：Beta(15, 15) → 50/50。不確定。
  第三篇：Beta(50, 10) → 83% 確信。
  第四篇：Beta(95, 5) → 95% 確信。但那 5% 是費雪的反問。
  第五篇：Beta(∞, 0)？不。她選擇了 Beta(1, 1) = 均勻分布。
         → 回到最開放的先驗。什麼都有可能。
         → 「我不想要 99% 的確定性。我要的是選擇的權利。」
```

### 場景：「有人在操控先驗分布」

```
貝葉：「先驗被改了。三遍都是同一個結果。」

柏朗：「我不叫它操控。我叫它校準。」

       感覺很嚴重的樣子......我好緊張......」
```

## 肆、寫作戒律

```
❶ 每個角色的台詞遮住名字也能認出是誰。
   測試方法：把台詞單獨拿出來，不標說話者。
   如果分不出是貝葉還是蘇立恆——重寫。

❷ 機率術語在對話中必須自然。
   ✗ 「讓我來解釋什麼是 Poisson 分布。」
   ✓ 「這個 Poisson 的殘差太整齊了。不對勁。」

❸ 小拉的碎念不超過每章兩次。
   太多會煩。太少會想念。兩次剛好。

❹ 白奶奶每次出場不超過半頁。
   她的力量在「短而深」。拉長就失去魔力了。

❺ 蘇立恆和貝葉的感情只通過「觀察」和「數據」流露。
   ✗ 「我喜歡你。」
   ✓ 「你的操作頻率異常。建議休息。」
   讓讀者自己讀出弦外之音。這比明說動人一百倍。

❻ 柏朗的每句台詞都必須「有道理」。
   讀者應該在讀完之後想：「他好像不完全是錯的......」

❼ 高思遠說話前永遠有停頓。
   「......」至少出現一次。因為他在想。

❽ 馬可瑜是情緒的晴雨表。
   她開朗 = 讀者放鬆。她安靜 = 讀者緊張。善用這個。

❾ 費雪不是壞人。她是另一種好人。
   她的「正確」跟貝葉的「正確」不同。兩個都是正確。

❿ 所有角色都不完美。
   貝葉太執著。蘇立恆太冷。高思遠太慢。
   馬可瑜太衝動。柏朗太自負。費雪太絕對。
   白奶奶太固執。小拉太焦慮。
   完美的角色不好看。有裂縫的才有光進來。
```
