# 第五篇「隨機」開場（ch49-52）—— 初稿級精修

> **精修目的**：Oracle Prime 揭露。Markov Chain 終極 Boss。72 小時倒數。Random Walk 策略。
> **原則**：最後一篇。全書的呼吸——先吸滿氣（危機），再一口一口吐出來（解決、收束）。讀者在前四章要感覺到時間在倒數。每一秒都有重量。
> **承接**：第四篇結束。深夜的台北。P(stars exist) = 1。口袋裡的紙條五摺。

## 第 49 章：第二階段

### 選擇性公開

```
【環境】
十二月四日。法庭裁定後第十五天。
貝葉的家。早上。

十二月的台北。
終於下了一場像樣的雨。
雨打在陽台的薄荷上。葉子彎了。
但沒有斷。薄荷的韌性比看起來強。

小拉重新啟動十五天了。
回來之後她做的第一件事——
是把暫停期間的系統日誌全部跑了一遍。
「我錯過了 847 個事件。
 其中 12 個跟你有關。
 包括——你在法庭上的心率紀錄。
 ……你的法庭心率我沒有。
 因為我暫停了。
 這讓我——」
「讓你什麼？」
「讓我——在統計上感到不完整。」

Oracular 遵從法庭裁定——開始公開先驗修改紀錄。
每天公開一批。
按照時間順序。最早的先。

但——

「貝姊。」小拉的聲音變了。
不是平常碎念的語氣。
是——分析模式。精確的。冷的。

「公開的紀錄——有東西不對。」

  ┌─ 小拉分析報告 ─────────────────────────┐
  │                                           │
  │ 已公開紀錄：487 筆                       │
  │ 修改幅度分布：                            │
  │   < 0.1σ ：412 筆（84.6%）              │
  │   0.1-0.5σ：63 筆（12.9%）              │
  │   0.5-1.0σ：12 筆（2.5%）               │
  │   > 1.0σ ：0 筆（0%）                   │
  │                                           │
  │ ⚠ 異常：                                │
  │ 我們的數據中確認存在 > 1.0σ 的修改      │
  │ 至少 23 筆。                             │
  │ 但公開紀錄中——一筆都沒有。              │
  │                                           │
  │ 結論：選擇性公開。                        │
  │ 大幅度修改——被隱藏了。                   │
  │                                           │
  └───────────────────────────────────────────┘

「他們在選擇性公開。」

貝葉看著那份報告。
小拉繼續：

「公開的都是微小修改。
 0.1σ 以下的。
 幾乎看不出來。
 可以說是『系統微調』。
 無害的。
 但——真正的操控——那些超過 1σ 的修改——
 一筆都沒有出現在公開名單裡。」

「他們在分批餵食。
 先公開無害的。
 讓大眾覺得『原來修改幅度這麼小，沒什麼嘛』。
 然後——等到輿論冷卻——
 大幅度的修改就永遠不會被問到了。」

「費雪的策略。」小拉說。

「不。」貝葉說。聲音很輕。

「這不是費雪的策略。
 費雪——是執行者。
 這種格局的策略——
 是柏朗的。」
```

### 蘇立恆的發現

```
手機震了。蘇立恆。
接了。

「你看到公開紀錄了？」
「看了。選擇性公開。」
「不只是那個。」

他的聲音——比平常低了一個音。
不是變聲器。是——真的壓低了。
像在說一個不應該被大聲說的事。

「我一直在監控 Oracular 系統的深層架構。
 法庭裁定之後——
 他們的系統不是在『等待公開』。
 它在做另一件事。」

「什麼？」

「有一個新的程序在跑。
 三天前上線的。
 名字叫 Oracle Prime。」

小拉自動把名字搜了一遍。
「Oracular 的公開文件裡沒有這個名字。
 不是已知的子系統。」

「因為它不在公開文件裡。
 它在 B3。」

B3。
Oracular 總部。地下三層。
蘇立恆三年前工作過的地方。

「Oracle Prime 的核心——
 是一個馬可夫鏈。」

停頓。

「貝葉——
 全台灣的先驗分布——
 不是獨立被修改的。
 它們——連在一起。
 透過一個轉移矩陣。
 152 × 152。
 每一個區域的先驗——
 都在影響其他區域。

 修改一個——
 像推了第一張骨牌——
 所有相鄰的都會跟著動。」

「馬可夫鏈。」

「對。
 這不是在修改幾個參數。
 這是一個活的系統。
 它在自我演化。」

  ┌─ 馬可夫鏈概念 ──────────────────────────┐
  │                                           │
  │ 狀態空間 S = {s₁, s₂, ..., s₁₅₂}       │
  │ 每個 s_i = 一個區域的先驗分布狀態        │
  │                                           │
  │ 轉移矩陣 P：152 × 152                   │
  │ P_ij = P(下一步到 j | 現在在 i)          │
  │                                           │
  │ 馬可夫性質（無記憶）：                    │
  │ P(X_{n+1} = j | X_n = i, X_{n-1}, ...)  │
  │ = P(X_{n+1} = j | X_n = i)              │
  │ 未來只取決於現在。不取決於過去。          │
  │                                           │
  │ Oracle Prime：                            │
  │ 每 6 小時執行一次轉移。                  │
  │ 方向：朝向預設的「目標分布」收斂。       │
  │                                           │
  └───────────────────────────────────────────┘

「它每六個小時跑一次。
 每一次——所有區域的先驗都往『目標值』靠近一點。
 一次靠近一點。
 像——水往低處流。
 不急。不猛。但——持續。
 你堵了這邊——它從那邊流。
 因為它不是一條河。
 它是——一張網。」

「蘇立恆——
 如果它一直跑下去——」

「它會達到穩態。
 穩態分布。
 π = πP。
 一旦到了穩態——
 所有的先驗會被鎖死。
 你修改任何一個——
 轉移矩陣會自動把它拉回來。
 像——恆溫器。
 你把溫度調高。
 它自動調回去。
 不需要人。
 永遠。
 不可逆。」

沉默。
雨打在窗上。
薄荷的葉子在彎。

「我們有多少時間？」

「……我不確定。
 但——不多。
 你需要叫上所有人。
 今晚。馬可瑜家。」
```

### 穩態

```
蘇立恆把 Oracle Prime 的架構投射到螢幕上。

「這是 Oracle Prime。」

  ┌─ Oracle Prime 架構 ─────────────────────┐
  │                                           │
  │ 核心：馬可夫鏈                            │
  │ 狀態數：152（全台 152 個校準區域）       │
  │ 轉移矩陣：P（152 × 152）                │
  │                                           │
  │ 運作方式：                                │
  │ 每 6 小時 → 執行一次狀態轉移             │
  │ 每個區域的先驗 → 根據 P 更新             │
  │                                           │
  │ 目標：穩態分布 π                          │
  │ π = πP                                   │
  │ 穩態 = 所有區域鎖定在操控後的狀態        │
  │ 自動維持。不可逆。                        │
  │                                           │
  └───────────────────────────────────────────┘

高思遠看了很久。
他放下花生。
這是——嚴重的信號。
高思遠只有在非常認真的時候才會放下花生。

「穩態分布。」

他走到白板前。
拿起筆。
在白板上寫了一行——跟三十年前教書時一樣的姿勢。

  π = πP

  「如果馬可夫鏈達到穩態——
   分布不再改變。
   因為 π 乘以 P 等於 π。
   狀態轉移了——但分布沒變。
   就像——
   你把一杯水倒進水裡。
   水面動了。但水還是水。
   同樣的分布。永遠。」

馬可瑜接過去：
「問題是——它多久會到穩態？」

她拿過蘇立恆的數據。
開始算。
她算的速度非常快——精算師的手速。

「遍歷性定理——
 如果這個馬可夫鏈是不可約的——
 所有狀態互相可達——」

她看了一眼轉移矩陣。
「P 的所有元素都 > 0。不可約。確認。」

「而且非周期——
 不會卡在固定的循環裡——」

「對角元素 P_ii > 0。非周期。確認。」

「那——根據遍歷性定理——
 這個鏈必然會收斂到唯一的穩態分布。
 問題只是——多快？」

她在紙上寫：

  收斂速率 ∝ |λ₂|

  λ₂ = 轉移矩陣的第二大特徵值
  |λ₂| 越接近 1 → 收斂越慢
  |λ₂| 越接近 0 → 收斂越快

「蘇立恆——P 的第二大特徵值是多少？」

蘇立恆在筆電上跑了一下。
「|λ₂| = 0.847。」

馬可瑜算：
「0.847 的 n 次方……
 我需要 0.847^n < 0.001 才算穩態……
 n ≈ log(0.001)/log(0.847)
 n ≈ 41.6 步。

 但——每步 6 小時——
 41.6 × 6 = 250 小時？
 那很多啊。十天以上——」

蘇立恆搖頭。

「那是標準收斂。
 但柏朗加了一個加速器。
 他在每一步之後——手動調整 P。
 加大對角元素。
 讓 |λ₂| 越來越小。
 加速收斂。

 按照目前的加速率——」

他在螢幕上打開了收斂曲線。

「大約 12 步。72 小時。」

全場安靜了。

「七十二小時。」高思遠說。
聲音像一把老刀——不快了，但還是能切。

「三天。」
```

### 真正的死線

```
馬可瑜繼續算。
她的筆沒停過。

「等等。
 72 小時是完全穩態——‖π_n - π‖ < 0.001。
 但——有一個更可怕的閾值。」

她在白板上寫：

  不可逆閾值：‖π_n - π‖ < 0.010

  「一旦低於 0.010——
   系統的慣性足以讓它繼續收斂。
   即使我們關掉 Oracle Prime——
   已經被修改的先驗——
   會在慣性下——
   自己慢慢飄向穩態。
   像一艘船關了引擎——
   但水流在推它。
   你關了引擎。但船不停。」

她算了一下。

「不可逆閾值——
 按照目前速率——
 大約 36 小時後到達。」

36 小時。
不是 72。

蘇立恆：「36 小時才是真正的死線。」

高思遠站起來。
他的臉色不太好。
但他的聲音很穩。

「三十六小時。
 夠了。
 夠做很多事。」

他看了每個人一眼。

「分工。
 蘇立恆和貝葉——進入 B3。
 關閉 Oracle Prime。
 需要管理員權限。
 只有柏朗或費雪有。

 馬可瑜——你留在外面。
 監控收斂進度。
 實時回報剩餘時間。
 如果速率改變——立刻通知。

 我——」

他停了一秒。

「我去找柏朗。」

「高叔——」

「你們負責系統。
 我負責人。
 柏朗——需要有人跟他說話。
 三十年了。
 該上最後一課了。」

他拿起那壺茶。
茶涼了。
但他沒有倒掉。

「明天凌晨出發。」

36:00:00
```

## 第 51 章：擲硬幣

### Random Walk

```
【環境】
倒數 32 小時。十二月五日凌晨四點。
台北。

計畫很簡單。
也很瘋狂。

蘇立恆和貝葉——
必須從板橋到 Oracular 總部（信義區）。
直線距離：12 公里。
正常路線：捷運 40 分鐘。

但——
費雪的安保系統覆蓋了所有常規路線。
捷運站有臉部辨識。
主要道路有攝影機。
透鏡會自動標記接近總部的人。

蘇立恆解釋：
「費雪的安保核心——
 也是馬可夫鏈。
 它根據你之前的行動模式——
 預測你下一步會去哪。
 你去過的地方越多次——
 它越確定你下次還會去。
 轉移機率會越來越集中。
 預測會越來越準。」

「那——如果我們的路徑沒有模式呢？」

貝葉拿出一枚硬幣。
十元。
正面——蔣。反面——數字。

「Random Walk。隨機漫步。
 在每一個路口——
 我們擲硬幣。
 正面——左轉。
 反面——右轉。
 沒有模式。沒有歷史。
 完全隨機。

 費雪的馬可夫預測系統——
 無法預測真正隨機的路徑。
 因為——隨機路徑沒有轉移矩陣。
 每個方向的機率——等機率。
 P = 均勻分布。
 你不可能預測均勻分布的下一步。」

蘇立恆看著那枚硬幣。

「你要我們用擲硬幣來決定逃跑路線。」

貝葉微笑了。
第一次在這場危機中微笑。
像陽台上的薄荷——在雨裡還是能開花。

「不是逃跑。
 是隨機漫步。
 聽起來比較有學術尊嚴。」

蘇立恆的嘴角——
動了一下。
那個「幾乎是微笑但忍住了」的動作。

「好。
 但有一個問題。
 Random Walk 沒有方向。
 我們可能走了四個小時——
 回到原點。」

「一維 Random Walk 的回歸定理——
 在一維無限隨機漫步中，
 回到原點的機率 = 1。
 遲早會回來。」

「那我們不是白走了嗎？」

「但——二維 Random Walk 不一樣。
 在二維中——回到原點的機率也是 1。
 但——到達任何特定點的時間——
 是有限的。
 只是——不確定要多久。
 我們最終會到 Oracular 總部。
 只是不知道什麼時候。」

「在 32 小時內？」

「機率——不低。
 而且——我們可以作弊。」

「怎麼作弊？」

「用偏幣。
 不是公平的硬幣。
 稍微偏向目標方向。
 比如——往信義區方向的左轉：55%。
 右轉：45%。
 路徑還是接近隨機。費雪的系統還是預測不了。
 但——平均方向——會往目標偏移。
 偏移隨機漫步。Biased Random Walk。」

蘇立恆點頭。
「55/45。夠了。
 既保留隨機性——又有方向。」

他從口袋裡拿出一枚硬幣。
不是十元。
是一枚——鑄造得稍微不均勻的硬幣。

「……你從哪來的？」

「我三年前就做了。」

「三年前你就知道會有今天？」

他沒有回答。
但——那枚硬幣已經回答了。

凌晨四點。
出發。
```

### 白奶奶

```
但——出發之前。

凌晨四點。
白奶奶的攤位。
她已經在了。
每天凌晨三點半起來。
揉麵團。切蔥花。

貝葉走過去。
蘇立恆在後面。保持了五步的距離。
不是害羞。是——習慣。
他跟白奶奶的關係是「坐了三年但從沒走過去」。

「奶奶。」
「嗯。怎麼這麼早？」
「要出門。」
「多久？」
「不知道。」
「危險嗎？」
「不知道。」

白奶奶看了她一眼。
那種眼神——
不是奶奶看孫女。
是一個活了八十年的人看一個即將做重要事情的年輕人。

她開始包蔥油餅。
手很快。但不急。
跟每天一樣的速度。
不因為時間緊迫就快。
不因為事情重大就慢。
剛剛好。

「四張。」她包了四張。
用報紙包的。油漬透出來。

「你爸以前也是。
 每次做決定之前——都來買蔥油餅。
 兩張——一張他吃，一張帶回去給你媽。
 每次都是。
 像——固定的程序。」

她看了一眼蘇立恆。
```
