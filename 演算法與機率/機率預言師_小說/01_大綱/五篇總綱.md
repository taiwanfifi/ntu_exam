> *「後驗 ∝ 先驗 × 似然。改了先驗，就改了世界。」*
> *但有一個人決定不再讓別人替她選先驗。*

## 第一篇：校準（第 1 ~ 12 章）

```
┌─────────────────────────────────────────────────────┐
│  核心主題（一句話）                                    │
│                                                       │
│  「所有的信任，都建立在一個假設上：                   │
│    你看到的數字是真的。」                             │
└─────────────────────────────────────────────────────┘
```

### 情緒基調

```
主色調：淡藍灰色。像台北梅雨季的天空——不晴不雨，有點悶。
副色調：偶爾的金色（貝葉發現異常時眼中的光）。

這一篇的氣質 = 《PSYCHO-PASS》第一集 × 《深夜食堂》× 《模仿遊戲》
日常的科幻世界 + 溫暖的人物速寫 + 暗處隱隱浮現的不對勁

節奏記號：🟢慢 → 🟢慢 → 🟡中 → 🔴爆發
前 6 章像一個下午的散步。後 6 章突然起霧——你才發現自己走進了迷宮。
```

### 開始狀態 → 結束狀態

```
【陳貝葉】
  開始：盡職的校準工程師。相信透鏡。相信 Oracular。
        每天的生活 = 上班校準 → 買珍奶 → 回家跟小拉鬥嘴 → 睡覺。
        她有一點點不快樂，但不知道為什麼。
        也許是因為她太習慣用機率看世界了，連自己的寂寞都被量化了：
        「今晚感到孤獨的機率：78%。」
  結束：世界觀的第一道裂縫。她發現了異常。
        不是抽象的懷疑——是親眼看到數字不對。
        而且她算了三遍。每次都不對。
        她知道，從這一刻起，她回不去了。

【蘇立恆】
  開始：不存在。只是一個坐在蔥油餅攤對面的陌生人。
        穿黑衣。不說話。手腕上戴著奇怪的手環。
  結束：在第 12 章尾聲，他主動出現在貝葉面前。
        說了第一句話：「你發現了對不對？」
        然後走了。
        留下貝葉和一個在她腦中瘋狂跑計算的小拉。

【高思遠】
  開始：慈祥的退休教授。每週跟貝葉吃一次飯。
        總是點一樣的東西（滷肉飯加荷包蛋），說一樣的話：
        「最近的校準工作怎麼樣？有沒有什麼......不尋常的？」
  結束：貝葉告訴他異常的發現。他沉默了很久。
        然後說：「......我等這一天等了八年。」
        他的表情不是驚訝。是如釋重負。
```

### 五個關鍵轉折點

```
轉折 ① —— 第 1-2 章
  透鏡世界的日常建立。
  貝葉為一位老先生做透鏡校準——
  老先生的透鏡所有機率都偏高 15%。
  原因：他的妻子剛去世。悲傷改變了他的認知基線。
  貝葉手動校準後，老先生看著窗外說：
  「原來雨停的機率是這麼高啊。......謝謝你。」
  → 這個場景建立「校準」的溫暖意義。
  → 也埋下伏筆：如果連悲傷都能偏移先驗，那人為操控呢？

轉折 ② —— 第 3-4 章
  貝葉校準老人的過程中，發現一個技術性的奇怪點：
  老人透鏡裡的「主觀先驗修正」跟「系統預設先驗」之間有落差。
  按照設計，系統預設先驗應該是中性的。
  但老人的系統預設先驗......已經被設定成偏樂觀的了。
  為什麼？
  小拉說：「也許是版本更新？你要查嗎？」
  貝葉猶豫了一秒。「......先記下來。」
  → 第一顆種子。

轉折 ③ —— 第 5-7 章
  貝葉接到一個大區域的校準任務：大安區。
  她跑 Poisson 模型檢查犯罪率數據。
  數字太漂亮了。殘差分布的偏態係數接近零。
  真實的犯罪數據不可能這麼對稱。
  她對小拉說：「小拉，你看這個分布。」
  小拉：「看起來......很正常？」
  貝葉：「不。看起來太正常了。」
  → 核心異常被發現。

轉折 ④ —— 第 8-10 章
  貝葉開始暗中調查。她用似然函數（MLE）重新估計參數。
  官方的 λ = 1.8。她算出來的 λ = 2.63。
  差了 46%。
  她跑了 likelihood ratio test。p < 0.001。
  這不是隨機波動。是人為修改。
  但改的不是數據——數據本身是對的。
  改的是先驗分布的參數。
  「有人在校準之前，就先把基準線調低了。」
  → 先驗被竄改的真相浮出水面。

轉折 ⑤ —— 第 11-12 章（篇末高潮）
  貝葉用貝氏定理做了一個完整的推導：
  先驗 → 似然 → 後驗。三個環節。
  先驗被改了。似然（觀測數據）是真的。
  所以後驗——所有人看到的「最終機率」——是扭曲的。
  「每個人看到的世界......都是假的？」
  她跑回家。關上門。對著空氣說了一句：
  「小拉，如果這是真的，那我們——」
  門鈴響了。
  門外站著蘇立恆。
  「你發現了對不對？」
  → 第一篇結束。第二篇的大門被踢開。
```

### 主線推進

```
第 1-4 章  ：【日常篇】
  世界觀建立。透鏡的日常。貝葉的工作與生活。
  - 老先生的校準（機率公理、樣本空間、條件機率）
  - 白奶奶的蔥油餅（不用透鏡的生活方式）
  - 與馬可瑜的日常閒聊（精算師的視角）
  - 高思遠教授的週末聚餐
  這一段像散文。讀者愛上角色，愛上這個世界。

第 5-8 章  ：【懷疑篇】
  貝葉發現大安區犯罪數據異常。
  - Poisson 分布的殘差太整齊（第 5-6 章）
  - 開始暗中比對其他區域數據（第 7 章）
  - 學會用似然函數驗證數據真偽（第 8 章）
  節奏開始收緊。像一滴墨水落入水中——慢慢擴散。

第 9-12 章 ：【覺醒篇】
  從懷疑到確認。
  - MLE 估計與官方數據嚴重不符（第 9 章）
  - 深夜獨自重算：先驗被竄改了（第 10-11 章）
  - 蘇立恆的出現（第 12 章）
  節奏加速。讀者跟著貝葉一起心跳加速。
```

### 感情線推進

```
貝葉 × 蘇立恆：

  第 1-4 章 ：零接觸。

  第 5-8 章 ：一次偶然的交集。

              ┌─────────────────────────────────────┐
              │ 偵測到附近有非標準透鏡頻率          │
              │ 來源：未知裝置                      │
              │ 距離：8.3m                          │
              └─────────────────────────────────────┘
              她四處看了看。沒有找到來源。

  第 9-12 章：第一次對視。
              貝葉在深夜加班趕分析。走出 Oracular 大樓。
              蘇立恆站在路燈下。

              貝葉的透鏡自動分析：「目標人物威脅機率：0.03」
              蘇立恆轉身走了。
              → 但他走的方向是她回家的方向。他在確保她安全到家。

貝葉 × 白奶奶：
  全篇最溫暖的線。
  貝葉每次去買蔥油餅，白奶奶都不問她過得好不好。
```

### 機率概念整合

```
  第 1-2 章  ：機率公理、樣本空間
              → 透鏡的基本運作原理。世界觀的數學根基。
  第 3-4 章  ：條件機率、主觀 vs 客觀機率

  第 8-10 章 ：似然函數、MLE

  第 11-12 章：貝氏定理、先驗/後驗/似然的關係
              → 全書核心公式的第一次完整呈現。先驗被竄改的發現。
```

## 第二篇：雜訊（第 13 ~ 24 章）

```
┌─────────────────────────────────────────────────────┐
│  核心主題（一句話）                                    │
│                                                       │
│  「你以為雜訊只是隨機的干擾。                         │
│    但如果雜訊本身被精心設計過呢？」                   │
└─────────────────────────────────────────────────────┘
```

### 情緒基調

```
主色調：深藍灰。像暴風雨前的海面——平靜但壓力巨大。
副色調：電子綠（分析數據時的螢幕光）和暗紅（危險逼近）。

這一篇的氣質 = 《全面啟動》的層層深入 ×《死亡筆記》的智力對決 × 《聯合公園 Numb》的壓抑
深入數據的迷宮 + 信任開始瓦解 + 有人要阻止貝葉

節奏記號：🟡中 → 🔴高壓 → ⚫窒息 → 🟢短暫喘息
讀者在這一篇會開始感到不安。不只是為貝葉——也為自己。
「如果我看到的機率也是假的呢？」
```

### 開始狀態 → 結束狀態

```
【陳貝葉】
  開始：震驚但決心調查。蘇立恆的出現讓她知道——她不是第一個發現的人。
        但也意味著：這件事比她想像的大得多。
  結束：同事林岳因「校準失敗」被強制住院。
        貝葉趕到醫院時，林岳的眼神是空的。
        他的透鏡被重置了。他不記得自己發現了什麼。
        但他在被重置前偷偷留了一段加密訊息：
        一組 Beta 分布的參數——Beta(72, 28)。
        貝葉看著那組參數，手在發抖。
        「小拉......Beta(72, 28) 是什麼意思？」
        「以這個參數來看，它代表......某件事有 72% 的信心度。」
        「什麼事？」
        「我不知道。但不管是什麼——有人非常確定它是真的。」

【蘇立恆】
  開始：謹慎地向貝葉透露部分資訊。但還保留了很多。
        「我需要知道你能走多遠。」
  結束：開始真正信任貝葉。
        不是因為她聰明——是因為她在看到林岳的空洞眼神後，
        沒有退縮。她更堅定了。
        蘇立恆在自己的筆記本上劃掉了一行字。
        那行字是：「如果她退縮就放棄這條線。」
        他從未讓貝葉看到這行字。

【費雪】
  開始：幕後存在。貝葉還不知道她的名字。
        但「有人」在監控貝葉的分析行為。
  結束：費雪正式出場。她走進校準組的辦公室。
        對貝葉微笑。
        「陳工程師，最近的校準數據我看了。非常精確。」
        停頓。
        「精確得有點......不尋常。」
        她走了。
        貝葉的手心出汗了。
```

### 五個關鍵轉折點

```
轉折 ① —— 第 13-14 章
  蘇立恆帶貝葉去一個秘密地點——他三年來的調查基地。
  一間在永和巷弄裡的出租套房，牆上貼滿了數據圖表。
  他給她看了「先驗偏移地圖」：全台灣，不只是大安區。
  至少有 23 個區域的先驗被不同程度地修改過。
  「這不是 bug。這是系統。」
  → 從個案升級為系統性陰謀。

轉折 ② —— 第 16-18 章
  貝葉和蘇立恆發現最精密的操控使用了 Beta 分布。
  因為 Beta 是 Bernoulli/Binomial 的共軛先驗——
  修改 Beta 的參數，後驗會自動「吸收」修改，毫無痕跡。
  「他們用共軛先驗的性質來隱藏操控。數學上完美的犯罪。」
  貝葉盯著公式，感到前所未有的恐懼。
  不是因為壞人很強。是因為壞人的數學很好。
  → 共軛先驗的美被用來做壞事。讓讀者感受到數學的雙面性。

轉折 ③ —— 第 19-20 章
  同事林岳——校準組裡唯一跟貝葉一樣會「手動驗算」的人——
  突然被通知「校準失敗」。
  公司說他的「校準認知基線偏移過大，需要重新校準」。
  這是公司術語。翻譯成白話：強制記憶重置。
  貝葉趕去醫院。太遲了。
  林岳坐在病床上，笑著對她說：「你是誰？」
  → 危險離主角只有一步之遙。

轉折 ④ —— 第 22-23 章
  貝葉拿到林岳留下的加密訊息：Beta(72, 28)。
  她和蘇立恆花了兩天破解。
  那不只是一組參數——那是林岳用邊際分布和條件分布，
  從聯合分布中提取出來的「信號」。
  他把發現壓縮成一組共軛先驗的參數。
  「他知道自己會被抓。所以把證據藏在數學裡。」
  → 數學成為救命的語言。

轉折 ⑤ —— 第 24 章（篇末）
  費雪正式出場。走進校準組辦公室。
  那段「精確得有點不尋常」的對話。
  → 獵人和獵物正式進入同一個房間。
  → 第二篇結束。戰場從「暗處」移到「明面」。
```

### 主線推進

```
第 13-16 章 ：【深入篇】
  蘇立恆帶貝葉深入調查。學會辨識不同的分布。
  - 常態 vs 指數 vs Gamma 的辨識（第 13-14 章）
  - 某些區域用 Gamma 做先驗，某些用 Normal（第 15 章）
  - 發現 Beta 分布的操控——共軛先驗的完美犯罪（第 16 章）

第 17-20 章 ：【風險篇】
  調查開始有了代價。
  - 馬可瑜發現保險數據也有異常（第 17 章）
  - 貝葉的分析行為被系統標記（第 18 章）
  - 林岳的「校準失敗」——第一個犧牲者（第 19-20 章）
  節奏驟緊。溫暖消退。危險靠近。

第 21-24 章 ：【解碼篇】
  從林岳留下的數學密碼中尋找真相。
  - 聯合分布/邊際分布的數學推導（第 21-22 章）
  - 條件分布和變數轉換的實戰應用（第 23 章）
  - 費雪正式出場。暗轉明。（第 24 章）
```

### 感情線推進

```
貝葉 × 蘇立恆：

  第 13-16 章：合作初期。蘇立恆教貝葉讀他的架構代碼。
               兩個人面對面坐在永和套房的小桌子前。
               中間隔著兩台筆電和一疊資料。
               蘇立恆解釋代碼的時候，聲音比平常低。
               貝葉後來意識到——他只有在跟她說話時聲音才會低。
               跟別人說話都是正常音量。

  第 17-20 章：信任開始建立。
               林岳出事的那天晚上，貝葉沒有回家。
               她坐在醫院門口的長椅上，一直到天亮。
               蘇立恆在凌晨三點出現。沒說話。
               在她旁邊坐下來。遞過一杯溫熱的便利商店咖啡。
               兩個人坐到太陽出來。
               貝葉的透鏡記錄了那一夜的心率數據。
               小拉偷偷看了一眼：「平均心率從 92 降到 68......
               是在他坐下之後降的。」
               小拉把這筆數據標記為「私人」然後鎖起來。

  第 21-24 章：第一次衝突。
               貝葉問蘇立恆：「你在 Oracular 的時候就知道了，為什麼不公開？」
               蘇立恆：「因為我算過。公開的後果比隱瞞更糟。」
               貝葉：「那你現在為什麼要幫我？」
               蘇立恆沉默了五秒。
               「因為......你算了不一樣的東西。」
               「什麼？」
               「我只算了後果。你算了......值不值得。」
               → 先驗（貝葉的信念）和似然（蘇立恆的證據）開始融合。
```

### 機率概念整合

```
  第 13-15 章：常態/指數/Gamma 分布辨識
              → 不同區域的先驗用了不同的分布族。學會辨認它們。
  第 16-18 章：Beta 分布、共軛先驗
              → 操控的核心手法。Beta 是共軛先驗，修改後無痕。
  第 19-21 章：聯合分布、邊際分布
```

### 五個關鍵轉折點

```

轉折 ③ —— 第 43-45 章

  她的信賴區間越來越寬——因為樣本量在縮減。
  「他們在刪證據。我需要更多數據。但數據正在消失。」

轉折 ④ —— 第 46-47 章

  貝葉用馬可瑜提供的保險數據，對 Oracular 的先驗分布做卡方檢定。
  H₀：觀測數據符合 Oracular 宣稱的先驗分布。
  H₁：觀測數據不符合。
  卡方統計量 = 47.3。自由度 = 8。
  p < 0.00001。
  「以任何常見的顯著水準——0.05、0.01、甚至 0.001——
   我們都應該拒絕虛無假設。
   這些數據不是從 Oracular 宣稱的先驗產生的。」

  「但您的獨立性假設成立嗎？如果數據不獨立——」

轉折 ⑤ —— 第 48 章（篇末）
```

### 五個關鍵轉折點

```
轉折 ① —— 第 49-51 章
  蘇立恆分析 Oracle Prime 系統，發現它的核心是一個巨大的 Markov Chain。
  全台灣的先驗分布不是獨立被修改的——它們透過轉移矩陣互相影響。
  修改一個區域的先驗，會像連鎖反應一樣影響所有相鄰區域。
  「這不是在修改幾個參數。這是一個活的系統。它在自我演化。」
  → Markov Chain 成為全書的終極 Boss。

轉折 ② —— 第 52-54 章
  團隊必須預測 Markov Chain 的下一步。
  如果系統達到穩態分布（stationary distribution），操控將不可逆。
  因為一旦穩態，所有的先驗都會「鎖定」在操控後的狀態。
  「它在收斂。我們的時間不多了。」
  馬可瑜用遍歷性定理（ergodic theorem）算出：
  「按照目前的轉移速率，系統在 72 小時後達到穩態。」
  72 小時。三天。
  → 終極倒數開始。

轉折 ③ —— 第 55-57 章
  蘇立恆決定物理入侵 Oracular 的 B3 實驗室。
  必須穿過城市才能到達。問題：費雪已經封鎖了所有常規路線。
  貝葉用 Random Walk 理論計算了一條「最不可預測的路徑」——
  在每個路口做隨機選擇。
  「如果我們的路徑是真正隨機的，她就無法預測。」
  蘇立恆：「你要我們用擲硬幣來決定逃跑路線？」
  貝葉：「不是逃跑。是隨機漫步。聽起來比較有學術尊嚴。」
  → 最緊張的追逐戲。Random Walk 變成逃亡策略。

轉折 ④ —— 第 58-59 章
  貝葉面對柏朗。最後的對話。
  不是打鬥——是兩種世界觀的碰撞。
  柏朗：「我給了人類確定性。確定性消除了恐懼、減少了死亡。
         你要把它拿走？你準備好為後果負責嗎？」
  貝葉：「確定性不是你給的。是你偽造的。
         而且......確定性不是人類需要的。」
  柏朗：「那人類需要什麼？」
  貝葉：「選擇。即使選擇可能是錯的。」
  → 全書哲學主題的最終辯論。

轉折 ⑤ —— 第 60+ 章（尾聲）
  Oracle Prime 被關閉。先驗控制權被釋放。
  全台灣的透鏡同時彈出一個新的介面：
  ┌─ 機率透鏡 v5.0 ─────────────────────────────┐
  │                                               │
  │ ⚠ 系統更新                                   │
  │                                               │
  │ 您現在可以選擇自己的先驗模式：               │
  │                                               │
  │ ○ 官方先驗（Oracular 建議）                  │
  │ ○ 個人先驗（基於您的歷史數據）               │
  │ ○ 無先驗模式（僅顯示似然函數）               │
  │ ○ 關閉透鏡                                   │
  │                                               │
  │ 您的選擇是？                                  │
  └───────────────────────────────────────────────┘
  沒有正確答案。
  但這一次——是你自己的選擇。
```
