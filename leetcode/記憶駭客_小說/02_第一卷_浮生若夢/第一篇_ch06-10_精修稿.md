## 第 6 章：鹹豆漿與平行線

### 核心場景——語青的記憶錯位

```
【這一章的功能】
  1. 讓讀者喜歡三人組的日常互動（輕鬆、可愛、有梗）
  2. 植入語青的「時間線錯位」伏筆
  3. 三人團隊分工自然形成

【語青的錯位——微小但讓人發毛】
  不要一次給太多。三處就好。

  錯位 #1（輕微）：
  她在某個轉角停下來。
  「這裡以前不是賣魚的嗎？」
  維倫看了一眼。那是一間修手錶的老店。
  老闆：「我在這裡修了四十年的錶。」
  語青：「……抱歉。可能是我記錯了。」
  （讀者：哦，也許她只是搞混了。）

  錯位 #2（奇怪）：
  經過一個岔路口。語青自然地往左走。
  維倫：「右邊。出口在右邊。」
  語青停住。臉上閃過一絲困惑。
  「右邊？這裡……改過了？」
  維倫：「沒改過。一直都在右邊。」
  （讀者：等等。她好像真的「記得」一個不同版本的地下街？）

  錯位 #3（不安）：
  他們經過一面牆。牆上是記憶銀行的廣告。
  語青看了一秒。然後移開目光。很快。太快了。
  維倫沒注意到。但米亞注意到了——
  「她的瞳孔縮放模式跟遇到創傷刺激的反應一致。」
  米亞只對維倫說了這句。語青沒聽到。
```

### 章末鉤子——走在左邊

```
【這是全書第一個「身體記得」的示範——在語青身上】
  她不知道為什麼走在左邊。
  但她的身體記得。
  在另一條時間線裡，她和維倫走了很多年的路。
  她一直在他的左邊。

  寫法：
  維倫換了三次位置。
  語青每次都繞回去。
  不是刻意的。是本能。

  「你為什麼總走這邊？」

  語青的眼睛亮了一下。不是開心的亮。
  是「被戳到痛處」的亮。

  「因為……你以前——」
  停。

  「沒什麼。習慣了。」

  米亞（耳機裡）：「心率飆升 40%。她差一點說出什麼。」

  維倫沒有追問。
  但他開始注意到：他也習慣了她在左邊。
  不知道從什麼時候開始的。
```

## 第 7 章：記憶銀行的廣告

### 核心場景——語青的恐慌發作 + 維倫的保護方式

```
【這一章的雙重功能】
  1. 語青的脆弱面（她不是無所不知的神秘人——她會怕）
  2. 維倫的本能（他怎麼安撫人→用修復師的技術→用手的溫度）

【語青崩潰的三階段】
  第一階段：僵住（1-2 秒）
    不是害怕的反應。是「看到了噩夢走進現實」的反應。
    身體在瞬間凍結。連呼吸都停了。

  第二階段：抓住維倫（3-5 秒）
    不是「救我」的抓法。是「我需要確認這是真實的」的抓法。
    手指掐進維倫的手臂。會留下痕跡。

  第三階段：維倫的反應（關鍵）
    他不慌。
    因為他見過太多這樣的委託人——記憶受創的人。
    他知道：不要問「你怎麼了」。不要抱住她。不要試圖解釋。

    他蹲下來。讓自己的眼睛跟她平視。
    「看著我。」
    不是命令。是錨。
    「只看著我。」

    語青看著他。
    呼吸慢了一拍。又一拍。

    「我沒事了。」她說。
    「你沒有。」維倫說。（← 他很直。）

    她笑了一下。很小的一下。
    「……你說得對。我沒有。」

【「那隻蝴蝶不是蝴蝶」】
  這句話獨立成段。不解釋。
  讀者會帶著這句話讀完整本書。
  等到鸚鵡計畫全貌揭開的時候，他們會回想起這一刻。
```

### 章末鉤子——鸚鵡標誌

```
  米亞的聲音在暗處亮起來——
  她搜索的方式跟維倫修復記憶一樣：
  從外圍掃到核心。

  「這個蝴蝶標誌是十年前換的。」
  「之前是什麼？」
  「鸚鵡。」

  停頓。

  「但我搜遍了所有資料庫。」
  「嗯。」
  「沒有一個人記得那隻鸚鵡長什麼樣子。」

  維倫把手放在修復台上。指尖碰到螢幕的亮線。

  一座城市。六百萬人。
  沒有一個人記得。

  不是忘了。
  是被清除了。
```

## 第 8 章：雪花裡的搖籃曲 —— ★ 演算法融合測試 ★

### 演算法 × 情感融合設計

```
【核心挑戰】
  Sliding Window 必須同時服務兩件事：
  1. 技術層：讀者理解「窗口在噪音中滑動找真實片段」的過程
  2. 情感層：讀者感受到「一個母親用盡全力為兒子留下最後禮物」的心碎

  如果只做到 1 → 教科書
  如果只做到 2 → 言情小說
  必須兩者同時 → 記憶駭客

【融合方法：用小杰的反應作為情感錨】
  技術描寫的每一步，都同時映射小杰的情緒。

  維倫調整窗口大小 ← 同時 → 小杰攥緊語青的手
  窗口掃過噪音段 ← 同時 → 小杰的臉在閃爍的螢幕光裡忽明忽暗
  第一個清晰片段出現 ← 同時 → 小杰叫了一聲「媽媽」
  片段又被噪音吞沒 ← 同時 → 小杰的嘴唇在動但沒有聲音

  讀者不需要知道 Sliding Window 是什麼。
  他們只需要「感覺到」——
  一個人在雪花裡一幀一幀地找，
  像在暴風雪中找一朵花。
```

### 米亞的科普——「河流和漁網」

```
【米亞向小杰解釋】
  這是全書第一次面對「非專業人士」解釋技術。
  米亞的語言要變——她在跟一個 10 歲的孩子說話。

  不用百分比。不用吐槽。
  用「河流和漁網」的比喻。

  「想像一條很長的河。」
  米亞的全息投影變成了一條小溪的動畫。
  「你媽媽的禮物在河底。」
  小溪的底部亮起了一個金色的光點。
  「但河面上全是漂浮的落葉。」
  水面上蓋滿了深色的碎片。

  「我們的方法是——用一張網。」
  一個半透明的方框出現在水面。
  「太大的網撈太多垃圾。太小的網抓不到寶藏。
   維倫叔叔現在做的事情就是——」
  方框開始向右滑動。
  「找到剛剛好的網。」

  小杰盯著動畫。
  「那我媽媽的禮物......一定可以撈到的吧？」

  米亞停頓了 0.2 秒。
  （不是因為她不確定。是因為她在學習：
   怎樣的停頓長度表示「溫柔的確認」而不是「猶豫」。）

  「一定可以。」
```

### 語青陪小杰折紙鶴——平行場景

```
【雙線交錯寫法】
  A 線：維倫在修復台前工作（技術、專注、緊張）
  B 線：語青在角落教小杰折紙鶴（安靜、溫柔、等待）

  兩條線每隔一段交替一次。
  A 線的節奏越來越緊張（窗口找不到更多片段）。
  B 線的節奏越來越慢（紙鶴一隻一隻堆起來）。

  A：維倫的眉頭越皺越緊。
  B：語青教小杰折翅膀的角度。「這裡要壓平。」
  A：噪音密度突然升高。維倫加大窗口——失敗。
  B：小杰折壞了一隻。「沒關係，再折一隻。」語青笑了。
  A：維倫深呼吸。縮小窗口。重新掃描。
  B：第七隻紙鶴。小杰輕聲問：「姐姐，我媽媽是不是很想我？」
  A：窗口鎖定第 347 區段。情感密度突然飆高。
  B：語青的手停了一下。「她……一直都在想你。」

  然後——
  螢幕亮了。
  一個女人的臉。
  「媽媽！」
```

## 第 9 章：窗口裡的夏天

### 核心揭露——母親建造的虛假記憶

```
【這是全書第一個「假記憶不一定是壞的」論述】

  小杰看著螢幕上的海邊。「但是……我從來沒去過海邊。」
  語青低聲說：「她在自己的腦海裡建造了一個暑假。」

  這個概念要慢慢展開。不是一句話交代。
  讓讀者跟小杰一起理解。

  步驟 1：困惑
  小杰：「但這些不是真的……？」

  步驟 2：理解
  語青：「你媽媽沒有錢帶你去海邊。
         所以她在腦子裡想像了一個海邊。」

  步驟 3：心碎
  語青：「然後她把想像的這一天……錄下來了。
         留給你。」

  步驟 4：升華
  維倫（很輕）：「她建造的不是記憶。」

  停頓。

  「是一封遺書。」
  「用整個夏天寫的。」

  （← 維倫的句子在這裡斷成兩半。因為他說不下去了。）
```

### 章末鉤子——鸚鵡 × 周明哲殘像 #1

```
【多層鉤子——疊加三個發現】
  第一層：母親的最後一句話
    「我在想像這些的時候，我的快樂是真的。」
    → 催淚（情感鉤子）

  第二層：鸚鵡標誌
    記憶最後一幀。母親背後的牆上。
    記憶銀行的舊標誌——鸚鵡。
    → 不安（懸疑鉤子）

  第三層：周明哲殘像 #1
    電視螢幕裡。模糊的中年男人在演講。
    維倫覺得那個輪廓似曾相識。
    → 寒意（伏筆鉤子）

  三層鉤子在同一幀畫面裡。
  讀者同時感到催淚、不安、和莫名的寒意。
  這就是記憶駭客的閱讀體驗。
```

## 第 10 章：不被記得的人

### 小杰收到記憶——全書最溫暖場景之一

```
【寫法：只寫身體語言】
  小杰戴上播放器。閉眼。

  不要寫他看到了什麼——讀者已經知道了。
  只寫他臉上的變化。

  眼角有淚。但他在笑。
  是那種閉著眼睛的笑。
  像在夢裡見到了誰。

  維倫和語青站在旁邊。
  語青遞給維倫一杯茶。
  兩個人看著小杰。
  什麼話都沒說。

  （這是 ch6-10 裡維倫和語青最「像一對」的瞬間。
   不是因為做了什麼。是因為他們自然地站在一起，
   看著同一個方向，沉默著。）
```

### 暗線揭示——陳美玲被系統性抹除

```
【節奏：溫暖→冰冷，過渡要快】
  小杰走了。
  維倫：「查一下那個鸚鵡標誌。」
  （← 他等小杰走了才說。不在孩子面前談這些。細節。）

  米亞的搜索結果——
  用列表方式呈現，像在讀一份被刪除的檔案殘骸：

  陳美玲。32 歲。已故。
  前記憶銀行低階技術員。
  資遣日期：2079.10.15（崩潰前 18 天）
  資遣原因：[已清除]
  個人記憶雲端帳戶：[已清空]
  社交記錄：[已清除]
  健保紀錄：[存在但訪問被拒]

  她的整個數位人生。被刪了。
  除了那段留給兒子的記憶。

  因為那段記憶不在雲端。在一個實體存儲盒裡。
  有人清除她的時候，漏掉了那個盒子。

  維倫：「她知道。」
  米亞：「知道什麼？」
  維倫：「她知道會被清除。所以用實體盒子存。」

  （一個即將被刪除的母親，在被刪除之前，
   用一個不會被掃到的舊盒子，
   為兒子建造了一個永遠不會消失的暑假。）
```

```

測試項目          | 結果                                                  | 評分
─────────────────┼──────────────────────────────────────────────────────┼─────
```

後續章節的演算法引入應以此為標準。

## ch6-10 寫作陷阱

| 陷阱 | 說明 |
|------|------|
| ch6 語青錯位太多 | 三處就夠。多了變成「神經病」而不是「時間旅行者」。 |
| ch7 維倫太溫柔 | 他不是溫柔。他是專業。他蹲下來是因為修復師訓練，不是因為愛情。（但讀者會覺得是愛情。） |
| ch8 米亞的河流比喻太長 | 200 字以內。比喻是給小杰聽的，不是給讀者聽的。 |
| ch9 母親的遺書揭露太快 | 分四步：困惑→理解→心碎→升華。不要一句話交代。 |
| ch9 三層鉤子太擠 | 三個發現在同一幀但分三段寫。不要塞在一個段落裡。 |
| ch10 陳美玲被刪除太陰謀論 | 用列表格式呈現搜索結果——冰冷、客觀、像讀一份殘骸。不要寫「有人在背後操控」。讓讀者自己推理。 |
