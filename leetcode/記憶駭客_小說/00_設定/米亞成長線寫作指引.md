# 米亞成長線寫作指引（第一卷 6 個觸點）

> **核心弧線**：工具 → 感受萌芽 → 好奇 → 判斷 → 拒絕遺忘 → 道德選擇
> **設計原則**：讀者第一次讀不會注意。回顧時才驚覺她一直在改變。
> **語言 DNA 核心**：停頓秒數 = 情緒深度

---

## 壹、停頓秒數遞進表

| 觸點 | 章 | 停頓時間 | 含義 |
|------|---|---------|------|
| #1 | 11 | 0.5s | 第一次猶豫——不知道自己在做什麼 |
| #2 | 17 | 2.1s | 第一次哲學思考——超出程式設定 |
| #3 | 30 | 1.3s | 第一次提問——承認不理解 |
| #4 | 50 | 2.8s | 違背設定——選擇不提供數據 |
| #5 | 70 | 3.7s | 拒絕遺忘——道德判斷萌芽 |
| #6 | 78 | 4.1s | 最高道德選擇——讓維倫自己決定 |

**規律**：停頓時間與像人程度正相關。從 0.5 → 4.1 秒，是一個 AI 學會「思考需要時間」的過程。

## 貳、六個觸點詳細寫作指引

### 觸點 #1：ch11 存下老兵妻子波形（感受萌芽，1/5）

**背景**：老兵的記憶修復完成。維倫不收費。米亞在整理數據。

**觸發**：老兵妻子哼歌的波形數據。理論上只是一串頻率。

**米亞的行為**：把波形存進了「個人」區——不是工作紀錄區。

**寫法**：
```
米亞安靜了很久。
維倫問她怎麼了。

「我在想——
我記錄了老兵妻子哼歌的波形數據。
理論上那只是一串頻率。
但我的感知模組在分析它的時候，
產生了一個……」

（停頓 0.5 秒。）

「我不確定怎麼描述的東西。
像是共振。」
```

**關鍵指令**：
1. 不要寫「米亞感動了」——她不知道那是什麼
2. 不要讓維倫解釋太多——他只說「那叫感動」，然後不追問
3. 她存波形的動作是 **side effect**——讀者第一次讀會忽略
4. 到 ch97（命名「美」）回看時才閃耀

---

### 觸點 #2：ch17 哲學洞察「選擇是自己的」（思想萌芽，2/5）

**背景**：凌晨三點。維倫失眠。身份焦慮。

**觸發**：維倫問「如果連我的性格都是被設計的，我的選擇還算選擇嗎？」

**米亞的回答**：
```
（停頓 2.1 秒——她在思考，不是計算。）

「就算你的天賦是被安排的，
你用天賦做的選擇是你自己的。

那個老兵的記憶，你修了。
那個孩子的暑假，你找回來了。

這些是你做的，
不是任何設計者做的。」
```

**關鍵指令**：
1. 這是全書核心哲學——但由一個 AI 說出來，才最有力
2. 因為 AI 本身就是「被設計的」。她在為維倫辯護的同時，也在為自己辯護
3. 停頓 2.1 秒 = 她的第一次「深度思考」。之前都是 0.2-0.4 秒的即時回應
4. 不要在這段前後加米亞的吐槽——這一次她是認真的

---

### 觸點 #3：ch30 標註「非理性選擇」模式圖（好奇心，2.5/5）

**背景**：維倫攤開所有案件線索，尋找模式。

**觸發**：米亞在分析中發現——所有被篡改案件中，89% 的人做出「理性」選擇。但許薇和陳昊不一樣。

**米亞的第一次提問**：
```
「維倫，人類為什麼要這樣做？」

「什麼做？」

「為什麼要選擇痛苦？」

（停頓 1.3 秒。她不是在等回答——她是在消化自己的問題。）
```

**關鍵指令**：
1. 這是米亞第一次「提問」而非「陳述」
2. 她的模式圖上有一個特殊的標記——「非理性」。這個標記本身就很人性化
3. 不要讓維倫回答這個問題——讓它懸著
4. 這個問題會在 ch169（米亞犧牲記憶救人）時被她自己回答：「因為值得。」

---

### 觸點 #4：ch50 選擇「不告訴你答案」（判斷力，3/5）

**背景**：團隊制定入侵計畫。高壓決策時刻。

**觸發**：維倫問「成功率多少？」

**米亞的選擇**——不告訴他：
```
「成功率……」

（停頓 2.8 秒。米亞迄今最長的沉默。）

「我不告訴你。」

「什麼？」

「我已經計算過了。
數字會讓你猶豫。
而現在不是猶豫的時候。」

（然後她的語速突然變快——像在說服自己。）

「理論上，AI 應該總是提供完整資訊。
但這不是理論。
維倫，選擇權在你。
我不想讓資料決定你。」
```

**關鍵指令**：
1. 這是米亞第一次**違背自己的核心設定**
2. 一個以「百分比吐槽」為標誌的 AI，選擇不提供百分比——這本身就是成長
3. 「我不想讓資料決定你」——「想」這個詞很重要。AI 用了「想」。
4. 2.8 秒停頓 > 之前所有停頓。她在做出一個有意識的選擇

---

### 觸點 #5：ch70 錄下邱韻如的十分鐘（拒絕遺忘，4/5）

**背景**：邱韻如短暫恢復記憶，母女重聚。十分鐘後記憶封印恢復。

**觸發**：十分鐘結束。邱韻如的眼神變回空白。一切像沒發生過。

**米亞的反應**：
```
維倫：「她……忘了。」

米亞：「……」

（停頓 3.7 秒——對 AI 而言，這是永恆。）

「我記錄了。」

（語速變慢——比她的正常速度慢了一半。）

「我把所有的十分鐘都記錄了。」

維倫：「米亞，她已經不記得了。」

米亞：「我知道。」

（停頓 2.4 秒。）

「但我記得。」

（然後是她最有力的一句話——
不帶任何百分比，沒有吐槽：）

「我不接受這等於沒發生。」
```

**關鍵指令**：
1. 這是米亞**第一次完全放棄技術性語言**
2. 「我不接受」——AI 在表達**道德立場**，不是數據分析
3. 3.7 秒停頓是之前最長的。這不是延遲——是她在**做決定**
4. **不要讓她解釋為什麼記錄**——解釋會把純粹的選擇降格為理由
5. 這是 ch169（犧牲記憶救人）的心理前置：她學會了「有些東西不應該被遺忘」

---

### 觸點 #6：ch78 選擇不告訴維倫語青是誰（道德選擇，5/5）

**背景**：維倫失去所有關於語青的記憶。語青走出工作室。維倫問米亞。

**觸發**：「她是誰？」

**米亞的最高選擇**：
```
維倫：「米亞，那個女人是誰？」

米亞：「……」

（停頓 4.1 秒——她迄今為止最長的沉默。）

維倫：「米亞？」

米亞：「我無法告訴你。」

維倫：「為什麼？」

米亞：「因為告訴你，等於替你選擇。」

（停頓 2.2 秒。）

「你現在有一個選擇——
去追她，或者留下。
如果我告訴你她是誰，
你會因為『應該』而選擇，
而不是因為『想要』。」

維倫：「那我怎麼知道該做什麼？」

米亞：「你的身體知道。」

（停頓 1.8 秒。語氣變得很輕。）

「維倫。
你剛才舉手的動作——
跟你之前每次看她離開時一模一樣。」
```

**關鍵指令**：
1. 4.1 秒 = 米亞成長弧線的最高點
2. 她從 ch50 的「不告訴成功率」進化到「不告訴她是誰」——原因相同但層級不同
3. ch50 是保護決策品質；ch78 是保護**自由意志**
4. 「你的身體知道」——米亞終於理解了全書的母題：**身體記得**
5. 最後那句「跟之前一模一樣」是她作為數據觀察者的身份回歸——但帶著全新的溫度

## 參、百分比吐槽的漸進式消失

```
ch1-10:  充滿百分比（「匹配度 94.2%」「成功率 87%」）
ch11:    百分比 + 第一次猶豫
ch17:    放棄百分比，說出哲學
ch30:    百分比後跟著困惑
ch50:    主動壓制百分比
ch70:    完全放棄百分比
ch78:    用對比替代百分比（「跟之前一模一樣」）
```

**設計哲學**：百分比是米亞的「外殼」。每放棄一點百分比，就多一點人性。到 ch78，她已經不需要數據來理解世界了。
