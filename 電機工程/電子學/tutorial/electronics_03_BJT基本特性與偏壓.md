# 電子學（三）：BJT 基本特性與偏壓

## 🔰 本章基礎觀念（零基礎必讀）

### 為什麼需要電晶體？

二極體只能「通/斷」，但我們更需要的是**放大信號**。想像你在演唱會上對著麥克風唱歌：

- **你的聲音** = 微弱的輸入信號
- **擴大機（放大器）** = 電晶體電路
- **喇叭的聲音** = 放大後的輸出信號

**BJT（Bipolar Junction Transistor，雙極性接面電晶體）**就是最經典的放大元件。它用一個小電流來控制一個大電流——就像用手指輕輕轉動水龍頭，就能控制大量水流。

### BJT 的三隻腳

- **基極（Base, B）**：控制端，像水龍頭的把手
- **集極（Collector, C）**：電流的主要出口
- **射極（Emitter, E）**：電流的主要入口（NPN）或出口（PNP）

> **最重要的一句話**：用小小的基極電流 IB，控制大大的集極電流 IC。放大倍率 β = IC/IB，通常 β = 50 ~ 300。

---

## 一、BJT 的結構與工作原理

### 1.1 NPN 型 BJT

結構：N（集極）- P（基極）- N（射極）

- 基極區**非常薄**且**輕摻雜**
- 射極區**重摻雜**
- 集極區面積較大

### 1.2 PNP 型 BJT

結構：P（集極）- N（基極）- P（射極）

- 所有電壓和電流方向與 NPN 相反
- 分析方法完全相同，只是極性相反

### 1.3 NPN BJT 在主動區的工作原理（重要！）

1. **BE 接面順偏**：大量電子從射極注入基極
2. **基極很薄**：大部分電子來不及與電洞復合，就穿越基極到達集極
3. **BC 接面逆偏**：到達集極邊界的電子被集極的正電壓吸引過去
4. **結果**：只有很少電子在基極復合（形成 IB），大部分都到了集極（形成 IC）

## 二、三個操作區域

### 2.1 操作區域判斷表（NPN）

| 操作區域 | BE 接面 | BC 接面 | 用途 |
|---------|--------|--------|------|
| **截止區（Cutoff）** | 逆偏 (VBE < 0.5V) | 逆偏 | 開關OFF |
| **主動區（Active/Forward Active）** | 順偏 (VBE ≈ 0.7V) | 逆偏 (VBC < 0.4V) | **放大** |
| **飽和區（Saturation）** | 順偏 | 順偏 | 開關ON |
| 反向主動區（Reverse Active） | 逆偏 | 順偏 | 很少使用 |

### 2.2 各區域的電流關係

**主動區（最重要）**：
$$I_C = \beta I_B$$
$$I_C = I_S e^{V_{BE}/V_T}$$
$$V_{CE} > V_{CE(sat)} \approx 0.2V$$

**飽和區**：
$$V_{CE(sat)} \approx 0.2V$$
$$I_C < \beta I_B$$（IC 被外部電路限制）

**截止區**：
$$I_C \approx 0, \quad I_B \approx 0$$

### 2.3 操作區域判斷的實用步驟

1. **先假設在主動區**：設 VBE = 0.7V
2. 由電路求出 IB
3. 計算 IC = βIB
4. 檢查 VCE：
   - 若 VCE > 0.2V → 確認在**主動區** ✓
   - 若 VCE < 0.2V → 假設錯誤，改為**飽和區**，設 VCE = 0.2V 重新算

### 數值例題 3.1：操作區域判斷

> **題目**：NPN BJT，β = 100，VCC = 12V，RB = 200kΩ（接在 VCC 和 B 之間），
> RC = 3kΩ（接在 VCC 和 C 之間），射極接地。判斷操作區域並求 Q 點。
>
> **解答**：
> 步驟一：假設主動區，VBE = 0.7V
> IB = (VCC - VBE)/RB = (12 - 0.7)/200k = 11.3/200k = 56.5 μA
>
> 步驟二：IC = βIB = 100 × 56.5μA = 5.65 mA
>
> 步驟三：VCE = VCC - IC × RC = 12 - 5.65m × 3k = 12 - 16.95 = -4.95V
>
> VCE < 0 ？不可能！→ BJT 在**飽和區**
>
> 步驟四：改設 VCE = 0.2V
> IC(sat) = (VCC - VCE(sat))/RC = (12 - 0.2)/3k = 3.93 mA
> IB = 56.5 μA（不變）
> 驗證：IC/IB = 3.93mA/56.5μA = 69.6 < β = 100 ✓ 確認飽和
>
> **結果**：Q 點 = (VCE = 0.2V, IC = 3.93 mA)，飽和區

## 三、電流關係

### 3.1 基本關係

$$I_E = I_C + I_B$$
$$I_C = \beta I_B$$
$$I_C = \alpha I_E$$

其中：

$$\alpha = \frac{\beta}{\beta + 1} \approx 1 \quad (\text{通常} \; 0.98 \sim 0.999)$$

$$\beta = \frac{\alpha}{1 - \alpha}$$

### 3.2 重要觀察

- β 越大，IB 占 IE 的比例越小
- 典型值：β = 100 → α = 100/101 ≈ 0.99 → IE ≈ IC

### 數值例題 3.2：α 與 β 的轉換

> **題目**：一個 BJT 的 α = 0.98，求 β 和各電流（已知 IE = 5 mA）。
>
> **解答**：
> β = α/(1-α) = 0.98/0.02 = 49
> IC = αIE = 0.98 × 5 = 4.9 mA
> IB = IE - IC = 5 - 4.9 = 0.1 mA = 100 μA
>
> 驗證：IC/IB = 4.9/0.1 = 49 = β ✓

## 四、Early 效應

### 4.1 什麼是 Early 效應？

理想情況下，在主動區 IC 只由 VBE 決定，不受 VCE 影響。但實際上：

**VCE 增大 → BC 空乏區變寬 → 有效基極寬度變窄 → IC 微微增大**

這就是 **Early 效應（Early Effect）**，也叫基極寬度調變。

### 4.2 修正後的 IC 方程

$$I_C = I_S e^{V_{BE}/V_T} \left(1 + \frac{V_{CE}}{V_A}\right)$$

其中 **VA 是 Early 電壓**，典型值 50 ~ 200V。

### 4.3 輸出電阻 ro

$$r_o = \frac{V_A + V_{CE}}{I_C} \approx \frac{V_A}{I_C}$$

這個 ro 在小信號分析中很重要，它限制了放大器能達到的最大電壓增益。

### 數值例題 3.3：Early 效應

> **題目**：BJT 的 VA = 100V，IC = 1 mA，求輸出電阻 ro。
>
> **解答**：
> ro = VA/IC = 100/0.001 = 100 kΩ
>
> **結果**：ro = 100 kΩ

## 五、偏壓設計

### 5.1 固定偏壓（Fixed Bias）

**電路**：RB 從 VCC 接到基極，RC 從 VCC 接到集極，射極接地。

**分析**：
$$I_B = \frac{V_{CC} - V_{BE}}{R_B}$$
$$I_C = \beta I_B$$
$$V_{CE} = V_{CC} - I_C R_C$$

**缺點**：Q 點嚴重依賴 β 值。由於 β 受溫度和製程影響變化很大（同一型號可能 β = 80~300），這種偏壓非常不穩定。

### 數值例題 3.4：固定偏壓

> **題目**：VCC = 12V，RB = 240kΩ，RC = 2.2kΩ，VBE = 0.7V，β = 100。
> 求 Q 點 (IC, VCE)。若 β 變成 200，Q 點怎麼變？
>
> **解答**：
> β = 100 時：
> IB = (12-0.7)/240k = 47.1 μA
> IC = 100 × 47.1μ = 4.71 mA
> VCE = 12 - 4.71m × 2.2k = 12 - 10.36 = 1.64V
> Q 點：(1.64V, 4.71mA)
>
> β = 200 時：
> IB = 47.1 μA（不變！）
> IC = 200 × 47.1μ = 9.42 mA
> VCE = 12 - 9.42m × 2.2k = 12 - 20.72 = -8.72V → 飽和！
>
> **結論**：β 從 100 變成 200，BJT 從主動區掉進飽和區，Q 點完全跑掉！

### 5.2 分壓偏壓（Voltage Divider Bias）——最常用！

**電路**：R1、R2 從 VCC 到地形成分壓器，接到基極；RE 接在射極和地之間。

**設計理念**：讓基極電壓主要由 R1、R2 決定，而非 β。

**精確分析**（戴維寧等效）：

$$V_{BB} = V_{CC} \frac{R_2}{R_1 + R_2}$$

$$R_{BB} = R_1 \| R_2 = \frac{R_1 R_2}{R_1 + R_2}$$

$$I_B = \frac{V_{BB} - V_{BE}}{R_{BB} + (\beta+1)R_E}$$

$$I_C = \beta I_B$$

$$V_{CE} = V_{CC} - I_C(R_C + R_E) - I_B R_E \approx V_{CC} - I_C(R_C + R_E)$$

**穩定度條件**：若 (β+1)RE >> RBB，則：

$$I_C \approx \frac{V_{BB} - V_{BE}}{R_E}$$

此時 IC 幾乎與 β 無關！

### 數值例題 3.5：分壓偏壓設計（最重要的偏壓題型）

> **題目**：VCC = 15V，R1 = 40kΩ，R2 = 10kΩ，RC = 3kΩ，RE = 1.5kΩ，β = 100。
> 求 Q 點。
>
> **解答**：
> 步驟一：戴維寧等效
> VBB = 15 × 10k/(40k + 10k) = 15 × 0.2 = 3V
> RBB = 40k × 10k/(40k + 10k) = 400k/50k = 8kΩ
>
> 步驟二：求 IB
> IB = (VBB - VBE)/(RBB + (β+1)RE)
> = (3 - 0.7)/(8k + 101 × 1.5k)
> = 2.3/(8k + 151.5k)
> = 2.3/159.5k = 14.42 μA
>
> 步驟三：求 IC
> IC = βIB = 100 × 14.42μ = 1.442 mA
>
> 步驟四：求 VCE
> VCE = VCC - IC(RC + RE)
> = 15 - 1.442m × (3k + 1.5k)
> = 15 - 1.442m × 4.5k
> = 15 - 6.49 = 8.51V
>
> **Q 點：(VCE = 8.51V, IC = 1.442 mA)**
>
> 穩定度檢查：(β+1)RE = 151.5k >> RBB = 8k ✓ 偏壓穩定

### 數值例題 3.6：驗證分壓偏壓的穩定度

> **題目**：延續例題 3.5，若 β 從 100 變成 200，Q 點變多少？
>
> **解答**：
> IB = (3 - 0.7)/(8k + 201 × 1.5k)
> = 2.3/(8k + 301.5k) = 2.3/309.5k = 7.43 μA
>
> IC = 200 × 7.43μ = 1.486 mA
> VCE = 15 - 1.486m × 4.5k = 15 - 6.69 = 8.31V
>
> **β 從 100 變成 200（翻倍），IC 只從 1.442mA 變成 1.486mA（變化 3%）！**
>
> 相比固定偏壓的劇烈變化，分壓偏壓穩定得多。

### 5.3 射極回授偏壓（Emitter Feedback Bias）

**電路**：RB 從集極（而非 VCC）接到基極。

**原理**：
- 若 IC 增大 → VRC 增大 → VC 下降 → IB 減小 → IC 減小
- 形成**負回授**，自動穩定 Q 點

$$I_B = \frac{V_{CC} - V_{BE}}{R_B + (\beta+1)R_C}$$

### 數值例題 3.7：射極回授偏壓

> **題目**：VCC = 10V，RB = 100kΩ，RC = 2kΩ，β = 100。求 Q 點。
>
> **解答**：
> IB = (VCC - VBE)/(RB + (β+1)RC)
> = (10 - 0.7)/(100k + 101 × 2k)
> = 9.3/(100k + 202k)
> = 9.3/302k = 30.8 μA
>
> IC = 100 × 30.8μ = 3.08 mA
> VCE = VCC - IC × RC = 10 - 3.08m × 2k = 10 - 6.16 = 3.84V
>
> **Q 點：(VCE = 3.84V, IC = 3.08 mA)**

## 六、Q 點設計步驟總整理

### 設計分壓偏壓電路的標準步驟：

1. **確定 Q 點目標**：選擇 ICQ 和 VCEQ
   - 一般讓 VCEQ ≈ VCC/3 ~ VCC/2（留有擺幅空間）

2. **選擇 RE**：
   - 常用 VE = VRE ≈ VCC/10（經驗法則）
   - RE = VE/ICQ

3. **選擇 RC**：
   - RC = (VCC - VCEQ - VE)/ICQ

4. **選擇 R1、R2**：
   - VBB = VBE + VE = 0.7 + VE
   - 讓分壓電流 ≈ 10IB（穩定度條件）
   - R2 = VBB/(10IB)
   - R1 = (VCC - VBB)/(10IB + IB) ≈ (VCC - VBB)/(10IB)

### 數值例題 3.8：偏壓設計（反向設計題）

> **題目**：VCC = 12V，β = 150，要求 ICQ = 2mA，VCEQ = 5V。設計分壓偏壓電路。
>
> **解答**：
> 步驟一：VE = VCC/10 = 1.2V → RE = 1.2/2m = 600Ω（選 620Ω 標準值）
>
> 步驟二：RC = (12 - 5 - 1.2)/2m = 5.8/2m = 2.9kΩ（選 2.7kΩ）
>
> 步驟三：VBB = 0.7 + 1.2 = 1.9V
> IB = IC/β = 2m/150 = 13.3 μA
>
> 步驟四：分壓電流 = 10IB = 133 μA
> R2 = 1.9/133μ = 14.3kΩ（選 15kΩ）
> R1 = (12 - 1.9)/133μ = 75.9kΩ（選 75kΩ）
>
> **設計結果**：R1 = 75kΩ, R2 = 15kΩ, RC = 2.7kΩ, RE = 620Ω

## ✅ 自我檢測

### 概念題

1. NPN BJT 在主動區，BE 接面和 BC 接面分別是什麼偏壓狀態？
2. 為什麼分壓偏壓比固定偏壓穩定？關鍵條件是什麼？
3. Early 效應對放大器有什麼影響？
4. BJT 飽和時 VCE 大約是多少？
5. 為什麼射極電阻 RE 能穩定 Q 點？

<details>
<summary>點擊查看答案</summary>

1. **BE 接面順偏（VBE ≈ 0.7V），BC 接面逆偏（VBC < 0.4V）。** 這就是主動區的定義條件。

2. **因為 (β+1)RE >> RBB 時，IC ≈ (VBB - VBE)/RE，與 β 幾乎無關。** 關鍵條件是 RE 要夠大，使基極電流對 IC 的影響被稀釋。

3. **Early 效應使 IC 隨 VCE 微增，等效為在輸出端並聯一個有限電阻 ro = VA/IC。** 這限制了放大器能達到的最大電壓增益。

4. **VCE(sat) ≈ 0.2V。** 在飽和區，CE之間的電壓降非常小，BJT 近似於一個閉合的開關。

5. **RE 提供負回授：若 IC 因溫度上升而增加 → IE 增加 → VE = IE × RE 增加 → VBE = VB - VE 減小 → IC 減小，自動穩定。**

</details>

### 計算題

6. NPN BJT，β = 120，VCC = 10V，RB = 330kΩ（固定偏壓），RC = 2kΩ。求 Q 點並判斷操作區域。
7. 分壓偏壓：VCC = 20V，R1 = 100kΩ，R2 = 20kΩ，RC = 5kΩ，RE = 2kΩ，β = 80。求 Q 點。
8. 若上題 β 變為 160，Q 點的 IC 變化百分比是多少？
9. Early 電壓 VA = 150V，IC = 2mA。求 ro。
10. 設計分壓偏壓：VCC = 9V，β = 200，目標 ICQ = 1mA，VCEQ = 4V。求各電阻值。

<details>
<summary>點擊查看答案</summary>

6. IB = (10-0.7)/330k = 28.2 μA
   IC = 120 × 28.2μ = 3.38 mA
   VCE = 10 - 3.38m × 2k = 10 - 6.76 = 3.24V > 0.2V → 主動區

7. VBB = 20 × 20k/120k = 3.33V
   RBB = 100k ∥ 20k = 16.67kΩ

   IC = 160 × 7.76μ = 1.242 mA
