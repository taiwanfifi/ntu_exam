# 揭面篇（ch120-140）情感密度圖

> **覆蓋範圍**：Vol 2 第三篇「真相的聚光燈」
> **核心挑戰**：21 章連續揭露 + 情感核爆，必須避免「震撼疲勞」
> **解決方案**：(1) ch125-126 教學段提供半呼吸；(2) ch135 通風口的天空作為反向高峰——用溫柔打斷暴力；(3) 在「偵探追查」與「情感爆發」之間交替切換，讓讀者的腦和心輪流運作
> **與角力篇的本質差異**：角力篇是「腦」的篇章（政治博弈），揭面篇是「心」的篇章（真相 = 情感核彈）。銜接點 ch120 同時是角力篇的終點和揭面篇的起點

揭面篇的情緒強度是角力篇的 1.25 倍。更關鍵的是：角力篇有 ch108.5（3 分喜劇）作為情緒出口，揭面篇完全沒有。這是刻意設計——揭面篇的主題是「真相」，真相沒有喜劇。

### 二、風險診斷

```
風險 1：ch130-134 連續五章情緒強度 ≥ 8
  → 「震撼疲勞」。讀者在第三個震撼後可能開始麻木
  → 嚴重程度：★★★★（高）

風險 2：ch125-126 的教學段可能「太冷」
  → 從 ch124（8 分）直接降到 ch125（6 分）= 落差感
  → 嚴重程度：★★（中低）

風險 3：ch135→ch136 的溫柔→痛轉折太快
  → 讀者還沉浸在 ch135 的美好中就被 ch136 打碎
  → 嚴重程度：★★★（中高）

風險 4：揭面篇沒有任何喜劇呼吸
  → 21 章純高壓
  → 嚴重程度：★★★（中高）
```

### 三、解決方案

| 風險 | 對策 | 具體設計 |
|------|------|---------|
| 風險 1 | ch131 用「冷偵探」語調對沖連續震撼 | 讓米亞主導 ch131 的語氣。Union-Find 教學保持邏輯冷靜。ch132-133 分別攻擊讀者的「認知」和「情感」——不重複 |
| 風險 2 | ch125-126 埋入微型懸念 | ch125 結尾發現「W」的第一個線索 → 教學＋懸念並行。ch125 結尾加感官殘像（遊行前有人吃了一碗牛肉麵，麵的熱氣穿越十年）|
| 風險 3 | ch136 開場要有過渡 | 先寫維倫看著語青的背影。讓 ch135 的餘韻持續 3-5 行再揭開偏移率 |
| 風險 4 | 微型日常植入 | ch121 結尾：米亞遞咖啡（「你 18 小時沒喝水了」）。ch137 中段：米亞的概率幽默（「活著回來的概率 34%」「上次你算的是多少？」「12%。你活著回來了。我的模型可能有 bug。」）|

### 四、呼吸章分布

| 呼吸章 | 章節 | 類型 | 前後張力 | 是否足夠？ |
|--------|------|------|---------|----------|
| ch125-126 教學段 | 半呼吸 | 偵探＋教學（非喜劇或日常）| ch124（8）→ 教學（6）→ ch127（8）| 半夠。腦休息但心沒有 |
| ch135 通風口 | 反向高峰 | 溫柔（非呼吸——是另一種頂峰）| ch134（9）→ 溫柔（9）→ ch136（8）| 不算呼吸。強度仍高，只是方向不同 |

**結論**：揭面篇只有 0.5 個呼吸點。全卷呼吸密度最低的篇章。

**是否需要增加呼吸？**

不需要大幅增加。理由：
1. 角力篇結尾（ch118-119）已有情感呼吸的餘韻，讀者進入揭面篇有情緒儲備
2. 揭面篇的主題是「揭露」。揭露不等待。鬆懈 = 失去讀者的急迫感
3. ch135 的「反向高峰」雖不是呼吸，但提供了情緒方向的切換。讀者哭出來 = 釋放，比喜劇呼吸更有效
4. 21 章的高壓區段是可以承受的——前提是寫法精準

**建議的微型呼吸植入**：
```
ch121 結尾微型日常（咖啡）   → 衝擊後緩衝
ch125 結尾感官殘像（牛肉麵） → TopSort 前休息
ch135 通風口天空              → ★ 核心呼吸段
ch137 中段米亞幽默            → Dijkstra 前輕鬆

修正後呼吸密度 ≈ 3.5/21 ≈ 17%。接近理想值 15%。
```
