### 蔥油餅紙上的期望值

```
【場景一】團隊分工與期望值

高思遠把蔥油餅的包裝紙翻過來。
油漬的背面。白色的。
他從口袋裡拿出一支筆。
（不是粉筆。在蔥油餅紙上用粉筆會滑掉。）

「第一步。」他說。
「我們需要知道操控的全貌。
 不只是 23 個區域。不只是 109 個區域。
 是——盡可能多的區域。」

「林岳的 Beta(72, 28) 告訴我們大約 72% 被操控。」貝葉說。
「全台 368 個分區。72% ≈ 264 個。
 但這只是估計。我們需要確認。」

「保險數據我有。」馬可瑜舉手。
theta 也抬頭了——好像在舉手。
「但只覆蓋有投保的區域。大概 60%。」

蘇立恆：「公開的政府數據庫能補剩下的。
 但品質不好。缺失值多。需要清理。」

「所以——」高思遠在蔥油餅紙上寫了一行字：

  E[X] = Σ xᵢ · P(xᵢ)

「你們清理完數據之後。第一個要算的東西。
 先驗操控的期望損害。」

他在公式旁邊畫了一個小箭頭。

「E[X]——期望值。
 不是平均值——雖然很多時候它們數值一樣。
 但概念不同。
 平均值是你『看到的』。
 期望值是你『預期的』。
 平均值是已經發生的事實。
 期望值是——如果這個過程重複無限多次，你會得到的平均結果。」

  ┌─ 期望值 ─────────────────────────────────────┐
  │                                                │
  │ E[X] = Σ xᵢ · P(xᵢ)  （離散）              │
  │ E[X] = ∫ x · f(x) dx  （連續）              │
  │                                                │
  │ 直覺：                                        │
  │   如果你玩一個遊戲無限多次——                  │
  │   E[X] 是你平均每次會得到的結果。             │
  │                                                │
  │ 性質：                                        │
  │   E[aX + b] = aE[X] + b  （線性）            │
  │   E[X + Y] = E[X] + E[Y]  （不需要獨立）     │
  │                                                │
  │ 為什麼重要：                                  │
  │   因為我們需要知道——                          │
  │   平均來說，每個區域的先驗被偏移了多少。      │
  │   這決定了損害的「規模」。                    │
  └────────────────────────────────────────────────┘

「用蔥油餅紙推導期望值。」貝葉看著那張油漬的紙。
「老高，你真的很老派。」

「蔥油餅紙跟白板一樣好用。
 而且比較香。」

馬可瑜拍了一張照片。
「我要把這個裱起來。
 史上第一篇在蔥油餅包裝紙上完成的統計推導。」

theta 趁大家不注意的時候，偷舔了蔥油餅紙的角。
油漬的那面。
被馬可瑜抓住了。
「theta！那是歷史文件！」

→ 期望值不是在教室裡學的。
  是在蔥油餅攤後面、用油漬的紙、
  被一個七十五歲老奶奶的廣播音樂伴奏著學的。
  這就是這本小說的教學方式。
```

### LOTUS——把偏移翻譯成金額

```
「我們知道了偏移幅度。」貝葉說。
「但——法庭不會對『偏移 38%』有感覺。
 他們需要的是——錢。損失。具體的數字。」

馬可瑜點頭。
「我需要把先驗偏移轉換成經濟損失。
 但問題是——偏移幅度的分布我有了。
 經濟損失的分布我沒有。
 我要怎麼算損失的期望值？」

高思遠拿起粉筆。
他在白板上——在三十年前最後一堂課的板書痕跡上——
寫了一行新的公式。

  ┌─ LOTUS（無意識統計學家定理）──────────────────┐
  │                                                 │
  │ E[g(X)] = Σ g(xᵢ) · P(xᵢ)    （離散）       │
  │ E[g(X)] = ∫ g(x) · f(x) dx    （連續）       │
  │                                                 │
  │ 你知道 X 的分布。                              │
  │ 你想算 g(X) 的期望值——                        │
  │ 其中 g 是某個函數（比如「偏移→損失」）。      │
  │                                                 │
  │ 你不需要先求 g(X) 的分布。                     │
  │ 直接用 X 的分布來算。                          │
  │                                                 │
  │ 為什麼叫「無意識」？                           │
  │ 因為很多人「無意識地」就這樣用了——             │
  │ 不知道這其實需要一個定理來保證它是對的。       │
  │                                                 │
  │ 直覺：                                         │
  │ 你知道每個員工遲到幾分鐘（X 的分布）。         │
  │ 你想算遲到造成的薪資損失（g(X) = 時薪×分鐘）。│
  │ 你不需要先算「損失的分布」。                   │
  │ 直接套函數就好。                               │
  └─────────────────────────────────────────────────┘

「LOTUS。」高思遠放下粉筆。
「名字聽起來像一朵蓮花。
 但它的功能是——幫你省掉一大堆計算。」

馬可瑜的眼睛亮了。
她拿出平板。開始打字。

「如果我定義 g(x) = 先驗偏移幅度 → 保險錯誤定價金額——
 這個函數我可以用精算模型建——
 然後直接用偏移幅度的分布來算期望損失——」

她打了二十分鐘的字。
theta 在沙發上翻了三次身。
蘇立恆喝完了他的黑咖啡。
高思遠閉目養神——但他的耳朵在聽。

結果。

馬可瑜抬起頭。臉色不太好。

「全台灣。一年。
 先驗操控造成的保險錯誤定價：
 約 480 億台幣。」

沒有人說話。

「如果加上間接損失——
 民眾因為錯誤風險認知做出的決策偏差——
 買錯房、走錯路、投錯票——」

她把平板放下。

「無法估計。」

高思遠睜開眼睛。

「480 億。這個數字——法庭聽得懂。」

→ LOTUS 的出現不是因為課本排到了。
  是因為馬可瑜需要算錢。
  讀者學到 LOTUS 是因為——
  有一個精算師需要把數學翻譯成法官能理解的語言。
```

## 第 27 章：變異數與分歧

### 不確定性——不只是數字的

```
【環境】
同一天。傍晚。研究室。
夕陽從窗戶照進來。
白板上的公式在夕陽下像鍍了金。
高思遠的沙發上——theta 和高思遠各佔一半。
theta 的頭靠在高思遠的膝蓋上。
高思遠的手放在 theta 的背上。
兩個老靈魂。

「我們有了期望損害。」蘇立恆說。
他站在白板前。手裡拿著紅色白板筆。

「但期望值只是平均。
 問題是——不確定性有多大？」

  ┌─ 變異數與標準差 ─────────────────────────────┐
  │                                                │
  │ Var(X) = E[(X - μ)²] = E[X²] - (E[X])²      │
  │ σ = √Var(X)  （標準差）                       │
  │                                                │
  │ 直覺：                                        │
  │   期望值 = 射擊的瞄準點（你平均打在哪裡）     │
  │   變異數 = 子彈的散布範圍（打得多不準）       │
  │                                                │
  │ σ 小 → 數據集中在平均值附近 → 穩定           │
  │ σ 大 → 數據分散 → 不穩定                     │
  │                                                │
  │ 在我們的情境中：                              │
  │   σ = 0.15 → 不小也不大                       │
  │   → 操控有統一策略，但執行有彈性              │
  │   → 不是一個人手動改的（太均勻）              │
  │   → 也不是完全自動化（有變異）                │
  │   → 可能是：半自動系統 + 人工微調             │
  └────────────────────────────────────────────────┘

「σ = 0.15。」蘇立恆在白板上圈了這個數字。
「這告訴我們——操控不是隨機的。
 但也不是完全機械化的。
 有一個人在背後做微調。」

「費雪？」馬可瑜問。

「可能。或者是費雪指揮的人。
 但——微調的人很懂統計。
 因為他把每個區域的偏移量控制在一個範圍裡——
 不會太大（會被發現）、不會太小（達不到效果）。」

「一個精確的 σ。」貝葉說。
「像一個精心校準過的分散範圍。」

蘇立恆點頭。
然後——氣氛變了。
```

「保險業——三天內崩盤。所有保單需要重新定價。

## 第 28 章：大數法則的幽靈

### 為什麼沒有人發現？——LLN 的庇護

```
【環境】
三天後。同一間研究室。
白板被擦了又寫了不知道多少遍。
蘇立恆帶了新的白板筆——六支。不同顏色。
「黑色是定義。藍色是推論。紅色是關鍵。綠色是例子。
 紫色是——我不知道答案的地方。橘色是危險。」
貝葉看著六支筆：「你連寫字都要校準。」
「效率。」

今天的議題：
為什麼操控存在了至少三年——
全台灣那麼多統計學家、數據科學家、精算師——
沒有人發現？

蘇立恆站在白板前。拿起黑色筆。

「答案分兩層。
 第一層——大數法則。
 第二層——中央極限定理。
 柏朗用了兩道防線。」

他寫了第一個公式。黑色。

  ┌─ 大數法則（LLN）─────────────────────────────┐
  │                                                │
  │ 弱大數法則：                                  │
  │                                                │
  │ 對任何 ε > 0，                                │
  │ P(|X̄ₙ - μ| > ε) → 0，當 n → ∞             │
  │                                                │
  │ X̄ₙ = (X₁ + X₂ + ... + Xₙ)/n  （樣本平均）  │
  │ μ = E[X]  （真實期望值）                      │
  │                                                │
  │ 直覺：                                        │
  │ 如果你投硬幣投夠多次——                        │
  │ 正面的比例一定會接近 50%。                    │
  │ 樣本夠大 → 樣本平均趨近真實平均。             │
  │ 雜訊被平均掉了。                              │
  │                                                │
  │ 這是統計學最基礎的保證：                      │
  │ 「多看幾次就會看清楚。」                      │
  └────────────────────────────────────────────────┘

「大數法則告訴我們——
 如果你蒐集夠多的數據，平均值會趨近真實值。
 所以統計局看全台灣的犯罪率平均值——
 完全正常。因為——」

他換了紅色筆。在旁邊寫：

  柏朗讓高偏移和低偏移的區域互相抵消。

  大安區 ↓61%  +  中山區 ↑0%  +  萬華區 ↑0%
  → 平均偏移 ≈ 0。看起來沒事。

「他不是讓所有區域都往同一個方向偏。
 他讓某些區域大幅偏低——
 但保留了一些區域完全不動——
 甚至有些區域微微偏高。
 整體平均——跟歷史值一樣。」

「LLN 保護了他。」貝葉說。

「對。因為任何人看大趨勢——
 全台灣的平均犯罪率、平均等候時間、平均保險理賠——
 全部正常。
 大數法則保證了平均值的穩定。
 他只要確保——偏移的期望值是零。
 個別區域偏多少都無所謂——只要加起來抵消。」

「所以——」馬可瑜聽懂了。
「所以只有逐一看個別區域的人才會發現。」

「只有校準師。」蘇立恆說。
「因為你們是唯一逐一處理區域數據的人。
 統計局看 X̄ₙ。你們看 Xᵢ。
 這就是為什麼林岳發現了。為什麼貝葉發現了。
 而所有其他人——看到的是一個完美的平均值。」

高思遠在旁邊。他的眼睛閉著。但他在聽。

「LLN 是統計學最偉大的定理之一。」
他沒有睜開眼睛。
「也是最危險的。
 因為它告訴人們：相信平均值。
 但——如果你只看平均值——
 你永遠不會看到平均值底下的黑暗。」

→ LLN 在課本上 = 「大量數據的平均值趨近真值」。
  在這裡 = 「操控者把謊言藏在平均值的掩護下」。
  一條保護善良統計的定理——被用來保護邪惡的操控。
  數學沒有善惡。人有。
```

### CLT——第二道防線

```
「LLN 是第一道。」蘇立恆說。
「但——如果有人不只看平均值呢？
 如果有人看分布呢？
 有人可能會畫直方圖。做檢定。
 那就會發現個別區域的偏移——」

「所以柏朗有第二道防線。」

他換了藍色筆。

  ┌─ 中央極限定理（CLT）─────────────────────────┐
  │                                                 │
  │ 如果 X₁, X₂, ..., Xₙ 是 iid 隨機變數，      │
  │ E[Xᵢ] = μ, Var(Xᵢ) = σ²，                    │
  │                                                 │
  │ 那麼當 n 夠大時：                              │
  │                                                 │
  │ √n(X̄ₙ - μ)/σ → N(0, 1)                       │
  │                                                 │
  │ 也就是說：                                     │
  │ X̄ₙ 近似 N(μ, σ²/n)                            │
  │                                                 │
  │ 直覺：                                         │
  │ 不管原始數據長什麼樣——                         │
  │ 只要樣本夠大——                                 │
  │ 樣本平均值的分布就像常態分布。                 │
  │ 鐘形。對稱。95% 落在 ±2σ/√n 以內。            │
  │                                                 │
  │ 所以——                                         │
  │ 如果操控的偏移量 < 2σ/√n——                     │
  │ 任何統計檢定都會判定：「正常波動」。           │
  │ z-test？通過。t-test？通過。                    │
  │ 因為偏移在 CLT 的正常範圍裡。                  │
  └─────────────────────────────────────────────────┘

「他精確計算了——
 每個區域的偏移量不超過 2σ/√n。
 這意味著——
 不管你用什麼標準統計檢定——
 你的 p-value 都不會低於 0.05。
 你的結論永遠是『無顯著差異』。」

蘇立恆在白板上寫了一行字。紅色。很大。

  「他把謊言穿上了常態分布的外衣。」

馬可瑜的臉色變了。
「所以——用標準方法永遠抓不到他？」

「用『標準』方法——不行。」
蘇立恆看了貝葉一眼。

「但我們不打算用標準方法。」

貝葉接上。

「CLT 有一個前提。σ²/n。
 n 越大——允許的波動範圍越小。
 他必須隨著樣本量的增加而縮小操控幅度——
 否則就會暴露。」

「所以——如果我們能在某個特定區域
 大量增加觀測數據——」

「CLT 的容許範圍會縮到——
 連他的操控都藏不住。」

「但我們不能增加觀測。
 我們沒有辦法讓某個區域突然多出很多犯罪事件——」

蘇立恆的嘴角——
動了一毫米。
在這個嚴肅的場合。一毫米。

「我們不增加觀測。
 我們——放一組假的觀測進去。
 一個陷阱。
 一個蜜罐。」

高思遠睜開了眼睛。

「蜜罐。」他重複。
然後——微微點頭。

「好棋。」

→ CLT = 敵人的鎧甲。
  它讓所有標準檢定都失效。
  但鎧甲也有縫隙——n 的大小。
  增大 n → 縫隙變大 → 鎧甲失效。
  而蜜罐——就是那把穿過縫隙的箭。
```
