# 第三篇「信號」中段（ch29-32）—— 初稿級精修

> **精修目的**：設計武器。等待觸發。在壓力與寧靜之間深化角色。
> **原則**：Markov/Chebyshev 不等式 = 鍛造武器的過程。讀者要感受到「我們終於有了反擊的手段」的興奮。第 32 章是暴風雨前的寧靜——情感章。

## 第 29 章：不等式的鐵壁

### Markov 與 Chebyshev——萬用的保證

```
【環境】
一週後。台大 301 研究室。
白板上五個紫色的問題還在。
蘇立恆在旁邊加了新的內容——
回答那五個問題需要的工具。

今天。高思遠站在白板前。
他今天穿了一件少見的格子襯衫。
馬可瑜偷偷跟貝葉說：「老高穿格子的時候通常心情很好。」
「為什麼？」
「因為格子 = 他覺得今天要教的東西很優雅。」

高思遠拿起粉筆。
（是的。他隨身帶粉筆。即使在白板的世界。）

「在設計蜜罐之前——
 你們需要一個『觸發器』。
 一個不管數據長什麼樣都能用的警報系統。
 不依賴任何分布假設。」

  ┌─ Markov 不等式 ─────────────────────────────┐
  │                                               │
  │ 如果 X ≥ 0（非負隨機變數），對任何 a > 0：  │
  │                                               │
  │ P(X ≥ a) ≤ E[X] / a                         │
  │                                               │
  │ 白話：                                       │
  │ 一個非負的東西超過某個門檻的機率——           │
  │ 不會超過「平均值除以門檻」。                 │
  │                                               │
  │ 例子：                                       │
  │ 如果全班平均考 60 分——                       │
  │ 考超過 120 分的人比例 ≤ 60/120 = 50%。       │
  │ （廢話。但數學保證的廢話。）                 │
  │                                               │
  │ 更有用的例子：                               │
  │ 如果平均偏移量是 0.38——                      │
  │ 偏移超過 0.76 的區域比例 ≤ 0.38/0.76 = 50%。│
  │ 很粗糙。但——                                │
  │ 對任何分布都成立。不需要知道分布長什麼樣。  │
  └───────────────────────────────────────────────┘

「Markov 很粗糙。」高思遠說。
「它像一堵很厚的牆——擋住了所有東西。
 但因為太厚了，所以不精確。
 你知道超過門檻的機率不會太大——
 但到底有多大，它說不清楚。」

「所以我們需要更好的工具。」

「Chebyshev。」

  ┌─ Chebyshev 不等式 ──────────────────────────┐
  │                                               │
  │ P(|X - μ| ≥ kσ) ≤ 1/k²                     │
  │                                               │
  │ 白話：                                       │
  │ 偏離平均值超過 k 個標準差的機率——            │
  │ 不超過 1/k²。                                │
  │                                               │
  │ k=2: P(偏離 > 2σ) ≤ 1/4 = 25%              │
  │ k=3: P(偏離 > 3σ) ≤ 1/9 ≈ 11%              │
  │ k=4: P(偏離 > 4σ) ≤ 1/16 ≈ 6.25%           │
  │ k=5: P(偏離 > 5σ) ≤ 1/25 = 4%              │
  │ k=10: P(偏離 > 10σ) ≤ 1/100 = 1%            │
  │                                               │
  │ 關鍵優勢：                                   │
  │ 不需要知道 X 的分布。                        │
  │ 只需要 μ（平均值）和 σ（標準差）。           │
  │ 對任何分布都成立。                            │
  │ Normal、Gamma、Beta、Poisson——全部。          │
  │                                               │
  │ 這是一個「萬用上界」。                       │
  │ 粗糙但絕對。                                 │
  │ 像一把不鋒利但保證砍得到的刀。               │
  └───────────────────────────────────────────────┘

「Chebyshev 比 Markov 好。
 因為它用了兩個資訊——平均值和標準差。
 而 Markov 只用了平均值。」

「所以 Chebyshev 更精準。」

「相對來說。
 但最重要的是——
 它對任何分布都成立。
 不管操控者用了什麼詭計——
 不管他怎麼偽裝分布——
 只要偏移超過 Chebyshev 的上界——
 就一定是異常。」

高思遠放下粉筆。

「一定。
 不是『很可能』。
 不是『在某些假設下』。
 是——數學證明的一定。」

他看了看四個人。

「你們要用這個——做觸發器。」

→ Markov 和 Chebyshev 不是考試題目。
  它們是武器。
  像偵探小說裡的槍——
  你在第一幕看到它掛在牆上，
  第三幕它一定會開火。
  現在——武器已經展示了。
  接下來——裝填。
```

## 第 30 章：統計蜜罐

### 蜜罐的設計——用數學設下陷阱

```
【環境】
第二天。研究室。
白板上五個紫色問題開始被一一回答。

蘇立恆站在白板前。
今天他的狀態不一樣——
平常他的姿勢是靠在牆上、手插口袋。
今天他站得很直。像一個準備上場的棋手。

「五個問題。逐一回答。」

他拿起黑筆。

問題一：蜜罐放在哪裡？

「我檢查了全台灣的區域列表。
 有 14 個區域到目前為止沒有被操控。
 這些是柏朗保留的——
 可能是因為數據太少不值得改，
 也可能是作為『對照組』讓整體統計看起來正常。」

「我們選其中一個——
 數據量適中、位於都會區、有保險數據可以交叉驗證的。
 新北市三重區 Sec.3。」

問題二：什麼時候放？

「下一個時間窗口是五月。
 我們在四月底部署。
 給操控者兩週的時間——
 讓數據融入系統。看起來自然。」

問題三：放什麼數據？

「一組犯罪率觀測數據。
 來源：我們自己生成。
 但——必須看起來完全真實。
 必須符合三重區的歷史模式。
 我會用歷史數據的分布參數來生成——
 但加入了我們知道、操控者不知道的特徵。」

  ┌─ 蜜罐數據的設計 ──────────────────────────────┐
  │                                                 │
  │ 真實先驗（三重區歷史）：Gamma(α=3.2, β=1.1)  │
  │ E[λ] = 3.2/1.1 = 2.91                         │
  │ σ₀ = √(α/β²) = √(3.2/1.21) ≈ 1.63           │
  │                                                 │
  │ 蜜罐數據：                                     │
  │ 用 Gamma(3.2, 1.1) 生成 50 個觀測值。          │
  │ 看起來完全正常。                               │
  │ 統計檢定全部通過。                             │
  │                                                 │
  │ 但我們知道真實的 μ₀ = 2.91 和 σ₀ = 1.63。    │
  │ 操控者不知道。                                 │
  │                                                 │
  │ 如果他修改先驗——                               │
  │ 比如把 Gamma(3.2, 1.1) 改成 Gamma(1.8, 1.4)—— │
  │ 後驗的期望值會從 2.91 降到某個更低的值。       │
  │ 偏離 = |新期望值 - 2.91|。                     │
  │ 如果偏離 > 5σ₀ ≈ 8.15——                       │
  │ Chebyshev 說：P(正常波動) ≤ 4%。              │
  │                                                 │
  │ 觸發。                                         │
  └─────────────────────────────────────────────────┘

問題四：觸發條件？

「Chebyshev 5σ。
 偏離超過 5 個標準差——觸發。
 4% 以下的機率是正常波動造成的。
 這在法庭上——足以構成初步證據。」

問題五：被發現了怎麼辦？

蘇立恆停了。
他拿起紫色筆。沒有在白板上寫。
把筆蓋轉了一圈。

「如果被發現——
 操控者會知道有人在反擊。
 他會加強監控。
 他會修改更多區域來覆蓋痕跡。
 他會——找到我們。」

「就像找到林岳那樣。」貝葉的聲音很輕。

「所以——蜜罐只能用一次。
 一次機會。
 如果他改了——我們就有證據。
 如果他沒改——我們就暴露了。」

高思遠在沙發上。theta 在他旁邊。

「一次機會。」他重複。
「就像考試只考一次。
 你必須在那一次——做對。」

→ 蜜罐的緊張感。
  不是因為技術複雜。
  是因為——只有一次機會。
  一個數學陷阱。
  如果對手踩進去——一切改變。
  如果對手沒踩——一切結束。
```

## 第 31 章：回聲

### 等待期間——貝葉與高思遠

```
【環境】
四月最後一週。蜜罐已部署。
等待。

台大研究室。下午。
只有貝葉和高思遠。
其他人各自忙——蘇立恆在監控後門，馬可瑜在準備法庭用的經濟模型。

高思遠在泡茶。
他有一個很舊的紫砂壺。三十年了。
壺的表面已經被茶水養得發亮。像古銅。

「老高。」
「嗯。」
「你剛才說——白奶奶認識我爸。你也認識白奶奶。」

高思遠把茶倒進兩個小杯子。
茶的顏色是琥珀色的。像液態的夕陽。

「認識。」
「多久了？」
「很久。」

他把一杯茶推給貝葉。

「你爸在 Oracular 工作的時候——
 每天下班都會去白奶奶那裡買蔥油餅。
 雷打不動。颱風天也去。
 白奶奶颱風天不擺攤——
 你爸就站在龍山寺的走廊下面等。
 等到白奶奶來收前一天留下的東西。
 然後跟她聊天。」

「聊什麼？」

「聊統計。」

「跟白奶奶聊統計？」

「你爸有一個理論——
 他說白奶奶是全台灣最好的貝氏統計學家。
 因為她每天煎餅——每張餅都是一次實驗。
 三十年。一天一百張。超過一百萬次實驗。
 她的先驗是用一百萬張餅校準的。
 比任何模型都精確。」

貝葉端著茶杯。手微微發抖。

「他——他真的這樣說？」

「原話是——」
高思遠模仿了一個語氣。比他自己的輕快。帶著一點台灣國語的尾音。
那個語氣——讓貝葉的心跳停了一拍。
因為她認識那個語氣。
那是她小時候聽過的——爸爸的語氣。

「『老高你知道嗎，白奶奶的蔥油餅
  就是一個完美的 Beta 分布。
  每張好吃的是 success，不好吃的是 failure。
  她煎了三十年——Alpha 和 Beta 都大到爆炸——
  她的後驗比我們任何模型都穩。
  我們搞什麼統計？不如去煎蔥油餅。』」

貝葉笑了。
眼淚也掉了。
同時。

「他真的——」

「他真的是那種人。」高思遠說。
聲音很柔。
「用蔥油餅解釋貝氏定理的人。
 你覺得你的教學天賦是從哪裡來的？」

她低頭。茶杯裡的琥珀色液體模糊了。

「老高。我爸……是怎麼死的？」

高思遠沉默了。
很久。
茶涼了。

「你知道的。車禍。十一秒。」

「我知道官方說法。
 但——你相信嗎？」

高思遠看著窗外。椰林大道。
三十年前他和柏朗和貝葉的父親走過的那條路。

「我——」

他停了。

「等這一切結束之後。
 我會告訴你所有我知道的。
 但現在——你需要專注。
 你不能帶著這個問題去打仗。
 它太重了。」

他看著她。眼神很認真。

「你爸如果在——
 他會說：『先驗可以等。似然不能等。
 現在手上有似然。先跑模型。
 先驗的問題……之後再更新。』」

貝葉深呼吸。一次。兩次。

「好。之後再說。」

她喝了茶。
涼了。但味道很好。
像——某種很久以前的味道。
她想不起來在哪裡嚐過。
也許是——
爸爸泡的茶。

→ 父親的線索越來越密。
  白奶奶認識他。高思遠是他的同學。
  他在 Oracular 工作過。他發現了同樣的事。
  然後——11 秒。
  這條線在第五篇完全揭開。
  現在——它是一根越來越粗的伏筆。
```

```
兩個人在永和套房裡。
螢幕的光。咖啡的味道。

叮叮。叮叮。

看起來一模一樣。

她的杯子上有口紅印。淡粉色的。
他的杯子上什麼都沒有。
乾淨。像他這個人。

  作為一個提醒：有人在他之前就發現了。有人為此付出了代價。
  他不能放棄。
  但——他不想讓貝葉知道。
  因為那會讓她問更多關於父親的問題。
  而那些問題的答案——他還沒有準備好面對。
```

## 第 32 章：暴風雨前

### 馬可瑜與 theta——深夜電話

```
【環境】
四月最後一天。
明天開始是五月。
時間窗口——隨時可能打開。

馬可瑜家。深夜。
theta 趴在她腳邊。沒有睡。
他今天不安。一直抖耳朵。
馬可瑜說：「theta 不安的時候跟我做精算一樣——
 算不出來就焦慮。」

她打了一通電話給貝葉。

「睡了嗎？」
「沒有。」
「我也沒有。theta 也沒有。所以我們三個都醒著。」

停頓。

「貝葉。我有一個數字要告訴你。」
「什麼？」
「如果先驗操控的事被公開——
 不是被控制地公開。是爆炸式的——
 保險業會在三天內崩盤。」

「三天？」

「三天。因為所有保單的定價基礎都要重算。
 重算需要六到八週。
 但恐慌不會等六到八週。
 恐慌在第一天就開始了。
 到第三天——退保潮、股市暴跌、保險公司流動性危機——
 一條完整的崩潰鏈。」

「所以我們不能直接公開。」

「不能。絕對不能。
 必須有一個受控的方式。
 法庭。學術論文。官方調查。
 讓消息像水一樣慢慢流出來——
 不是像洪水一樣衝出來。」

theta 在她腳邊嗚了一聲。
馬可瑜摸了摸他的頭。

「你知道嗎。」她的聲音突然變了。
不是平常的馬可瑜——開朗的、愛開玩笑的。
是一個——疲憊的、認真的精算師。

「我做精算這麼多年——
 每天都在跟風險打交道。
 車禍的機率、疾病的機率、死亡的機率。
 我以為我已經免疫了。
 數字就是數字。概率就是概率。

 但——當我算出 480 億的時候。
 我突然覺得——
 那不是數字。
 是人。
 每一億裡面——有多少人因為錯誤的風險認知做了錯誤的決定？
 有多少人因為透鏡說『安全』而走了那條路、買了那棟房、信了那個人？

 我以前覺得統計是冷的。
 現在——我覺得它是熱的。
 燙的。」

電話那頭。貝葉安靜了很久。

「可瑜。」
「嗯。」
「謝謝你。」
「謝什麼？」
「謝謝你——把數字變成溫度。」

馬可瑜笑了。
是她平常的笑。
但底下有一層東西——不是眼淚。
是一種——決心。

「好了。不煽情了。
 明天開始我們要嚴陣以待。
 theta——你也是。」

theta 抬起頭。看了她一眼。
然後趴回去。嘆了一口氣。
那口氣裡有一種——超越物種的理解。

「他說他準備好了。」
「他明明就是嘆氣。」
「47% 準確率。但今天我選擇相信它。」

掛了電話。
深夜。
台北。
五月——即將到來。
```

### 最後的寧靜——白奶奶的蔥油餅

```
【場景三——最後一個安靜的場景】

五月一日。傍晚。
貝葉去了龍山寺。

白奶奶在翻餅。
今天的廣播在播〈雨夜花〉。台語的。

她坐在小板凳上。
白奶奶遞了一張餅。
「今天的蔥特別香。」

每次都說。但貝葉每次聽到都覺得比較好了。

「奶奶。」
「嗯？」
「如果——有一天——
 你發現你一直相信的東西是假的。
 你會怎麼辦？」

白奶奶想了想。
很認真地想。

「假的啊……」
她翻了一張餅。完美的金黃。

「假的就假的嘛。
 我信了三十年。那三十年是真的啊。
 我用它做了很多決定——那些決定是我做的。
 不管它是真是假——
 做決定的是我。不是它。」

她看了貝葉一眼。

「你是不是在煩惱那些數字的事？」

「什麼數字——」

「我不懂數字。
 但我知道——你最近看東西的眼神不一樣了。
 以前你看東西是——」
她做了一個動作。用鏟子指了指自己的眼睛。
「直直的。像看一條路。

 現在你看東西是——」
她做了另一個動作。鏟子在空中畫了一個圓。
「轉的。像在找什麼東西。但不知道在哪裡。」

貝葉看著白奶奶。
七十五歲。不戴透鏡。不用手機。
賣了三十年的蔥油餅。

但她看人的方式——比任何透鏡都準。

「奶奶。你怎麼這麼厲害？」

「什麼厲害。
 煎了三十年的餅。
 看了三十年的人。
 人的臉跟餅一樣——
 火候不對就會焦。表情不對就會苦。
 我不需要那個——」
她又指了指太陽穴。透鏡的位置。
「我用我自己的眼睛看。就夠了。」

她翻了最後一張餅。收進紙袋。
遞給貝葉。

「多的。不要錢。
 你明天有大事吧？」

「你怎麼——」

「做大事的人前一天都會來買蔥油餅。
 你爸也是。」

又是爸爸。
這條線——像一根越來越亮的光。

貝葉接過餅。很溫。
蔥花的味道。
最簡單的味道。
也是——最真實的。

「謝謝奶奶。」

「去吧。做你的大事。
 餅涼了就不好吃了。
 人也是——別等太久。」

她不知道白奶奶最後那句話是什麼意思。
是在說蔥油餅。
還是在說——別的什麼。

─── 第 32 章完 ───
─── 等待結束。五月。時間窗口打開。 ───
```
