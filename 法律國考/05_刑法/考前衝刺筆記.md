## 壹、三階層犯罪審查 SOP ★★★

> **口訣：「該違責」**（構成要件該當性→違法性→罪責）

### 一、審查架構總覽

```
【三階層檢查流程】

第一階：構成要件該當性
  ├─ 客觀構成要件：行為主體、行為客體、行為、結果、因果關係、客觀歸責
  └─ 主觀構成要件：故意（§13）或過失（§14）、特殊主觀要件（意圖等）
         │
         ▼ 該當 → 進入第二階

第二階：違法性（有無阻卻違法事由？）
  ├─ 正當防衛 §23
  ├─ 緊急避難 §24
  ├─ 依法令行為 §21
  ├─ 業務正當行為 §22
  └─ 超法規阻卻違法事由（得被害人承諾等）
         │
         ▼ 無阻卻事由 → 進入第三階

第三階：罪責（有責性）
  ├─ 責任能力：§18（年齡）、§19（精神狀態）
  ├─ 不法意識：§16（禁止錯誤）
  └─ 期待可能性
```

### 二、各階段檢查細項

| 階層 | 檢查項目 | 條文 | 說明 |
|---|---|---|---|
| **構成要件** | 行為主體 | 各條 | 自然人為原則，法人另論 |
| | 行為 | — | 作為或不作為 |
| | 結果 | — | 結果犯須有法益侵害結果 |
| | 因果關係 | — | 條件說→客觀歸責 |
| | 故意/過失 | §13/§14 | 主觀歸責 |
| **違法性** | 阻卻違法事由 | §21-§24 | 有→不罰；無→違法 |
| | 容許構成要件錯誤 | 學說 | 誤想防衛等，效果有爭議 |
| **罪責** | 責任能力 | §18/§19 | 14歲以下不罰、精神障礙 |
| | 不法意識 | §16 | 禁止錯誤：不知法律→得減輕 |
| | 期待可能性 | 學說 | 無期待可能性→阻卻罪責 |

### 三、責任能力速查

| 年齡/狀態 | 條文 | 效果 |
|---|---|---|
| 未滿14歲 | §18 I | 不罰 |
| 14歲以上未滿18歲 | §18 II | 得減輕其刑 |
| 滿80歲 | §18 III | 得減輕其刑 |
| 精神障礙（不能辨識） | §19 I | 不罰 |
| 精神障礙（辨識顯著降低） | §19 II | 得減輕其刑 |
| 原因自由行為 | §19 III | 不適用前二項（如酒駕故意醉酒） |

## 貳、故意過失速查 ★★★

### 一、四格比較表

| | **有預見（認知）** | **無預見** |
|---|---|---|
| **有意欲（意志）** | **直接故意** §13 I：明知並有意使其發生 | （不存在此組合） |
| **無意欲但容任** | **間接故意** §13 II：預見其發生而不違背本意 | — |
| **無意欲** | **有認識過失** §14 II：預見其發生，確信其不發生 | **無認識過失** §14 I：應注意能注意而不注意 |

### 二、故意與過失辨別口訣

```
直接故意：「明知＋有意」→ 我就是要他死
間接故意：「預見＋不違背本意」→ 死了也無所謂
有認識過失：「預見＋確信不發生」→ 應該不會死吧
無認識過失：「應注意＋能注意＋未注意」→ 根本沒想到會死
```

### 三、重要區分：間接故意 vs 有認識過失

| 比較 | 間接故意 §13 II | 有認識過失 §14 II |
|---|---|---|
| 認知面 | 預見可能發生 | 預見可能發生 |
| 意志面 | 不違背本意（容任） | 確信不發生（排斥） |
| 白話 | 「死了也沒差」 | 「應該不會出事」 |
| 法律效果 | 以故意犯論 | 以過失犯論（刑度較低） |

## 參、不作為犯要件 ★★

### 一、不純正不作為犯檢查流程

```
① 有保證人地位？
② 有作為可能性？
③ 不作為與作為等價？（等價性）
④ 主觀故意/過失？
→ 全部肯定 → 成立不純正不作為犯
```

### 二、保證人地位類型

| 類型 | 說明 | 例子 |
|---|---|---|
| **法律明文** | 法律規定的作為義務 | 父母對子女的保護義務 |
| **契約/職務** | 基於契約負擔的義務 | 保母照顧嬰兒、救生員 |
| **危險前行為（自招危險）** | 因自己行為製造危險源 | 開車撞人後的救助義務 |
| **危險源監控** | 對危險源有支配管領力 | 飼養猛犬的飼主 |
| **特殊信賴關係** | 緊密生活關係 | 登山團領隊 |

### 三、等價性判斷

| 要素 | 內容 |
|---|---|
| 核心 | 不作為在法律評價上須等同於以作為方式實現構成要件 |
| 判斷重點 | 保證人地位之強度、對結果之支配程度 |
| 例外 | 純正不作為犯不需等價性判斷（如§149、§294遺棄罪） |

## 肆、因果關係與客觀歸責 ★★★

### 一、因果關係理論

| 理論 | 內容 | 公式 |
|---|---|---|
| **條件理論** | 若無該行為，結果即不發生 | 非A則非B |
| **相當因果關係** | 依一般經驗法則，該行為足以導致該結果 | 社會通念上相當 |
| **客觀歸責** | 現代通說，三層次判斷 | 見下表 |

### 二、客觀歸責三層次

```
【客觀歸責檢查流程】

Step 1：行為人是否「製造法所不容許的風險」？
  ├─ 否 → 不歸責（如：合法駕駛但他人自撞）
  └─ 是 → 進入 Step 2

Step 2：該風險是否「在具體結果中實現」？
  ├─ 否 → 不歸責（如：被害人送醫途中車禍死亡）
  └─ 是 → 進入 Step 3

Step 3：結果是否在「構成要件效力範圍內」？
  ├─ 否 → 不歸責（如：自我負責領域、他人負責領域）
  └─ 是 → 客觀歸責肯定
```

### 三、客觀歸責重要案例類型

| 類型 | 說明 | 結論 |
|---|---|---|
| 合法替代行為 | 即使合法行為結果仍會發生 | 風險未實現→不歸責 |
| 風險升高理論 | 行為顯著升高風險 | 有爭議，多數肯定歸責 |
| 被害人自我負責 | 被害人自陷風險 | 不在構成要件效力範圍→不歸責 |
| 介入因素 | 第三人行為介入 | 視是否異常判斷 |

## 伍、阻卻違法事由速查 ★★★

### 一、五大阻卻違法事由比較表

| 比較 | 正當防衛 §23 | 緊急避難 §24 | 依法令 §21 | 業務正當 §22 | 超法規（被害人承諾） |
|---|---|---|---|---|---|
| **情狀** | 現在不法侵害 | 自己或他人生命等之危難 | 依法律或命令 | 業務上正當行為 | 被害人有效承諾 |
| **對象** | 只限對侵害者 | 可對第三人 | — | — | 處分個人法益 |
| **比例原則** | 不要求（但不得過當） | 嚴格要求（手段不得逾越危難） | — | — | — |
| **補充性** | 不要求 | 要求（無其他避難方法） | — | — | — |
| **過當效果** | 得減免 §23但書 | 得減免 §24但書 | — | — | — |

### 二、正當防衛要件（§23）

| 要件 | 內容 |
|---|---|
| 現在 | 侵害已開始尚未結束（非過去、非未來） |
| 不法 | 侵害行為具違法性（不含合法行為） |
| 侵害 | 對法益有侵害或危險 |
| 防衛意思 | 主觀上認識到防衛狀態 |
| 必要性 | 選擇有效排除侵害之最溫和手段 |
| 不得過當 | 防衛行為未超過必要程度 |

> **重點**：正當防衛不要求「退避原則」，但有社會倫理限制（對幼童、精神病患宜退避）

### 三、緊急避難要件（§24）

| 要件 | 內容 |
|---|---|
| 緊急危難 | 現在之危難（自然災害、動物攻擊等亦可） |
| 避難行為 | 犧牲第三人法益以保全自己或他人法益 |
| 補充性 | 別無其他避難方法 |
| 利益衡量 | 保全利益須大於犧牲利益 |
| 避難意思 | 主觀上須有避難意思 |

> **比較口訣**：「正當防衛打壞人，緊急避難傷好人」

## 陸、錯誤論速查 ★★★

### 一、錯誤類型總覽

```
錯誤
├── 構成要件錯誤（事實錯誤）§12
│   ├── 客體錯誤
│   ├── 打擊錯誤
│   └── 因果歷程錯誤
└── 禁止錯誤（法律錯誤）§16
```

### 二、構成要件錯誤三類型比較

| 類型 | 定義 | 例子 | 法律效果 |
|---|---|---|---|
| **客體錯誤** | 誤認攻擊客體（A當成B） | 甲想殺乙，誤認丙為乙而殺丙 | 等價：故意既遂（概括故意）；不等價：阻卻故意 |
| **打擊錯誤** | 攻擊目標正確但打偏 | 甲開槍射乙但打中旁邊丙 | 通說：對乙未遂＋對丙過失，想像競合 |
| **因果歷程錯誤** | 因果經過與預想不同 | 甲推人下橋想淹死，被害人撞石頭死 | 偏離未逾一般經驗→不影響故意既遂 |

### 三、打擊錯誤的學說爭議

| 學說 | 結論 | 理由 |
|---|---|---|
| **具體符合說**（通說） | 對目標未遂＋對被害人過失→想像競合 | 故意須指向具體客體 |
| **法定符合說** | 對被害人故意既遂 | 同一構成要件內故意可轉用 |

### 四、禁止錯誤（§16）

| 項目 | 內容 |
|---|---|
| 定義 | 行為人不知其行為違法 |
| 可避免 | 得減輕其刑（§16但書） |
| 不可避免 | 免除其刑（§16本文） |
| 判斷標準 | 行為人是否盡查詢義務（諮詢律師、主管機關） |

### 五、容許構成要件錯誤（誤想防衛）

| 學說 | 處理方式 |
|---|---|
| 嚴格罪責說 | 禁止錯誤處理→§16 |
| 限制罪責說（通說） | 類推適用§12→阻卻故意，僅論過失 |
| 法律效果限縮說 | 排除故意刑罰→依過失犯處罰 |

## 柒、未遂犯速查 ★★★

### 一、未遂犯審查流程

```
行為人已否「著手」？
├─ 否 → 預備犯（原則不罰，例外如§271 III預備殺人罪）
└─ 是 → 未遂犯
       ├─ 結果發生？ → 既遂犯
       └─ 結果未發生？
            ├─ 不能未遂？（§26）
            ├─ 中止未遂？（§27）
            └─ 普通未遂？（§25）
```

### 二、著手時點判斷

| 學說 | 標準 | 說明 |
|---|---|---|
| 形式客觀說 | 開始實行構成要件行為 | 殺人＝開始砍殺 |
| 實質客觀說（通說） | 對法益產生直接危險 | 殺人＝舉刀瞄準 |
| 主觀說 | 行為人主觀認為已開始 | 過於前置，少採 |

### 三、三種未遂比較

| 比較 | 普通未遂 §25 | 不能未遂 §26 | 中止未遂 §27 |
|---|---|---|---|
| **定義** | 已著手未既遂 | 行為不能發生結果且無危險 | 己意中止或防止結果發生 |
| **法律效果** | 得減輕其刑 | 不罰 | 必減或免除其刑 |
| **例子** | 開槍未中 | 以砂糖當毒藥 | 開槍後主動送醫救活 |
| **要件** | 著手＋未既遂 | 著手＋客觀不能＋無危險 | 著手＋己意＋中止/防果 |

### 四、中止未遂要件詳解（§27）

| 要件 | 說明 |
|---|---|
| 己意性 | 出於自願而非外力阻止（Frank公式：「欲達目的而不能」→普通未遂；「能達目的而不欲」→中止未遂） |
| 中止行為 | 未發生結果前即中止行為，或積極防止結果發生 |
| 結果未發生 | 結果已發生則為既遂，不成立中止未遂 |
| 因果關係 | 中止行為與結果未發生間須有因果關係 |

> **Frank 公式口訣**：「能達而不欲＝中止；欲達而不能＝障礙（普通）未遂」

## 捌、共犯論速查 ★★★

### 一、正犯與共犯體系

```
【廣義共犯】
├── 正犯
│   ├── 直接正犯：親自實行
│   ├── 間接正犯 §28反面：利用他人為工具
│   └── 共同正犯 §28：共同實行犯罪
└── 狹義共犯
    ├── 教唆犯 §29：教唆他人犯罪
    └── 幫助犯 §30：幫助他人犯罪
```

### 二、三者比較表

| 比較 | 共同正犯 §28 | 教唆犯 §29 | 幫助犯 §30 |
|---|---|---|---|
| **主觀** | 犯意聯絡（共同行為決意） | 教唆故意＋使他人產生犯意 | 幫助故意＋使正犯易於實行 |
| **客觀** | 行為分擔（功能支配） | 教唆行為（唆使決意） | 幫助行為（物質/精神） |
| **處罰** | 皆為正犯 | 依所教唆之罪處罰 §29 II | 得減輕其刑 §30 II |
| **從屬性** | — | 限制從屬（正犯須該當＋違法） | 限制從屬 |

### 三、共同正犯重要爭點

| 爭點 | 說明 |
|---|---|
| 相互歸責 | 「一部行為全部責任」，各共同正犯就全部結果負責 |
| 犯意聯絡 | 明示或默示均可，不以事前謀議為必要 |
| 行為分擔 | 須有功能上的重要支配角色 |
| 共謀共同正犯 | 未參與實行但主導謀議者，仍為共同正犯（實務肯定） |
| 逾越 | 共同正犯一人逾越共同計畫→他人就逾越部分不負責 |

### 四、教唆犯重點

| 項目 | 內容 |
|---|---|
| 雙重故意 | ①教唆故意（使他人萌生犯意）②被教唆之罪的故意 |
| 被教唆人已有犯意 | 非教唆→可能為幫助犯 |
| 教唆未遂 | §29 III：被教唆人未至犯罪→教唆犯依未遂犯處罰 |
| 連鎖教唆 | 教唆教唆犯→仍以教唆犯論 |

### 五、幫助犯重點

| 項目 | 內容 |
|---|---|
| 幫助方式 | 物質幫助（提供工具）＋精神幫助（鼓勵、建議） |
| 幫助時點 | 事前幫助、事中幫助（事後幫助原則非幫助犯） |
| 因果關係 | 通說要求幫助行為對正犯行為有促進效果 |
| 處罰 | 得按正犯之刑減輕之 §30 II |

## 玖、競合論速查 ★★

### 一、競合類型總覽

```
一行為 → 想像競合 §55（從一重處斷）
數行為 → 實質競合 §50（數罪併罰）
法條間 → 法條競合（擇一適用）
附隨行為 → 不罰前後行為
```

### 二、四種競合比較

| 比較 | 想像競合 §55 | 實質競合 §50 | 法條競合 | 不罰前後行為 |
|---|---|---|---|---|
| **行為數** | 一行為 | 數行為 | 一行為 | 數行為 |
| **觸犯罪名** | 數罪名 | 數罪名 | 數罪名（外觀） | 主行為＋附隨行為 |
| **處斷** | 從一重處斷 | 數罪併罰 | 擇一適用 | 僅論主行為 |
| **例子** | 一槍打死二人 | 今天偷、明天搶 | 殺人vs傷害（特別關係） | 偽造文書後行使 |

### 三、法條競合四類型

| 類型 | 說明 | 例子 |
|---|---|---|
| **特別關係** | 特別法優於普通法 | 殺直系血親尊親屬 §272 優於 §271 |
| **補充關係** | 基本規定補充適用 | 未遂犯補充既遂犯 |
| **吸收關係** | 高度行為吸收低度行為 | 既遂吸收未遂 |
| **擇一關係** | 互斥適用 | 竊盜 §320 vs 侵占 §335 |

### 四、數罪併罰（§50-§51）

| 項目 | 內容 |
|---|---|
| 要件 | 裁判確定前犯數罪 |
| 執行 | 宣告多數有期徒刑→定應執行刑 §51 |
| 上限 | 不得逾各刑合計，最長不超過30年 |

## 拾、10大必考罪名 ★★★

### 一、§271 殺人罪

| 項目 | 內容 |
|---|---|
| 客觀 | 殺人行為＋死亡結果＋因果關係 |
| 主觀 | 殺人故意（直接或間接） |

### 二、§277 傷害罪

| 項目 | 內容 |
|---|---|
| 客觀 | 傷害行為＋身體或健康受損 |
| 主觀 | 傷害故意 |
| 加重結果犯 | §277 II 傷害致死/致重傷 |
| 重要爭點 | ①傷害故意vs殺人故意之區分 ②重傷 §10 IV 定義 ③加重結果犯之主觀要件（客觀能預見） |
| 告訴乃論 | §287：普通傷害為告訴乃論 |

### 三、§320 竊盜罪

| 項目 | 內容 |
|---|---|
| 客觀 | 竊取他人動產（破壞持有＋建立持有） |
| 主觀 | 竊盜故意＋不法所有意圖 |
| 重要爭點 | ①竊盜vs侵占（是否已持有）②竊佔不動產§320 II ③使用竊盜（欠缺不法所有意圖→不成立） |
| 加重 | §321 加重竊盜（夜間侵入、結夥三人等） |

### 四、§339 詐欺罪

| 項目 | 內容 |
|---|---|
| 構成要件流程 | 施詐術→使人陷於錯誤→為財產處分→財產損害 |
| 主觀 | 詐欺故意＋不法所有意圖 |
| 重要爭點 | ①詐欺vs竊盜（有無處分意思）②詐欺vs背信（主體不同）③三角詐欺（處分人≠被害人） |
| 加重 | §339-4 加重詐欺（三人以上、網路等） |

### 五、§304 強制罪

| 項目 | 內容 |
|---|---|
| 客觀 | 以強暴脅迫使人行無義務之事或妨害人行使權利 |
| 主觀 | 強制故意 |
| 重要爭點 | ①強制罪之補充性（其他罪之強暴脅迫已評價→不再論強制）②強暴概念（直接/間接強暴） |

### 六、§354 毀損罪

| 項目 | 內容 |
|---|---|
| 客觀 | 毀棄損壞他人之物或致令不堪用 |
| 主觀 | 毀損故意 |
| 告訴乃論 | §357：毀損罪為告訴乃論 |
| 重要爭點 | 「致令不堪用」之解釋（減損物之效用） |

### 七、§169 誣告罪

| 項目 | 內容 |
|---|---|
| 客觀 | 意圖使他人受刑事處分，向該管公務員誣告 |
| 主觀 | 誣告故意＋使人受刑事處分之意圖 |
| 重要爭點 | ①須「明知無此事實」而虛構 ②自首減免§172 ③反坐可能性 |

### 八、§210 偽造文書罪

| 項目 | 內容 |
|---|---|
| 類型 | 偽造（冒名製作）vs 變造（竄改內容） |
| 客體 | 私文書 §210 / 公文書 §211 / 特種文書 §212 |
| 行使 | §216 行使偽造變造文書罪 |
| 重要爭點 | ①文書之定義（有體性、持續性、名義人可辨識）②有形偽造vs無形偽造 |

### 九、§302 剝奪行動自由罪

|---|---|

### 十、§346 恐嚇取財罪

## 拾壹、易混淆罪名對照表 ★★★

### 一、竊盜 vs 侵占

| 比較 | 竊盜 §320 | 侵占 §335 |
|---|---|---|
| **持有狀態** | 行為人原本無持有 | 行為人原本已合法持有 |
| **行為** | 破壞他人持有＋建立自己持有 | 易持有為所有（變更持有意思） |
| **關鍵區分** | 「拿別人的」 | 「把別人寄放的據為己有」 |
| **例子** | 從商店偷東西 | 借用朋友手機後不還並變賣 |

### 二、詐欺 vs 背信

| 比較 | 詐欺 §339 | 背信 §342 |
|---|---|---|
| **行為人角色** | 外部人（無信任關係） | 為他人處理事務者（有信任關係） |
| **行為** | 施詐術→陷於錯誤→處分財產 | 違背任務→損害本人利益 |
| **被害人** | 因錯誤而處分 | 不以陷於錯誤為必要 |
| **關鍵區分** | 「騙外人的錢」 | 「背叛雇主的信任」 |

### 三、強盜 vs 恐嚇取財

| 比較 | 強盜 §328 | 恐嚇取財 §346 |
|---|---|---|
| **手段強度** | 至使不能抗拒 | 使人心生畏怖但仍有意思自由 |
| **被害人狀態** | 完全喪失意思自由 | 心生畏懼但仍有選擇餘地 |
| **關鍵區分** | 「硬搶」 | 「威脅交出」 |
| **法定刑** | 5年以上 §328 | 6月以上5年以下 §346 |

### 四、殺人 vs 傷害致死

| 比較 | 殺人 §271 | 傷害致死 §277 II |
|---|---|---|
| **主觀** | 殺人故意（直接或間接） | 傷害故意（無殺人故意）＋對死亡結果客觀能預見 |
| **判斷標準** | 下手力道、攻擊部位、兇器種類、行為人與被害人關係 |  |
| **法定刑** | 死刑、無期、10年以上 | 無期或7年以上 |
| **關鍵區分** | 「要他死」 | 「只想打他但打死了」 |

### 五、搶奪 vs 強盜

| 比較 | 搶奪 §325 | 強盜 §328 |
|---|---|---|
| **手段** | 趁人不備公然奪取 | 強暴脅迫至使不能抗拒 |
| **被害人狀態** | 來不及反應 | 被壓制不能抗拒 |
| **例子** | 飛車搶皮包 | 持刀威脅交出錢包 |

## 拾貳、刑法答題 SOP ★★★

### 一、三階層套用模板

```
一、甲之行為是否成立○○罪？
（一）構成要件該當性
  1. 客觀構成要件
     (1) 行為：甲為○○行為（涵攝事實）
     (2) 結果：發生○○結果
     (3) 因果關係與客觀歸責：（逐層檢驗）
  2. 主觀構成要件
     (1) 故意/過失：甲具有○○故意/過失
     (2) 特殊主觀要件（如有）：不法所有意圖等
  → 構成要件該當

（二）違法性
  有無阻卻違法事由？
  → 無 → 具違法性

（三）罪責
  1. 責任能力：甲為成年人，具完全責任能力
  2. 不法意識：無禁止錯誤問題
  3. 期待可能性：可期待為合法行為
  → 具罪責

（四）結論：甲成立○○罪
```

### 二、共犯題型答題順序

```
Step 1：先論正犯（實行者）的犯罪是否成立
Step 2：再論共犯（教唆/幫助）是否成立
Step 3：檢查共犯從屬性（限制從屬：正犯須該當＋違法）
Step 4：各自的競合處理
```

### 三、考前口訣總整理

| 主題 | 口訣 | 對應內容 |
|---|---|---|
| 三階層 | 該違責 | 該當→違法→罪責 |
| 客觀歸責 | 製實範 | 製造風險→實現風險→構成要件效力範圍 |
| 正當防衛vs緊急避難 | 防打壞人避傷好人 | 防衛只對侵害者，避難可對第三人 |
| 故意層次 | 明知有意＝直故；預見容任＝間故 | 直接vs間接故意 |
| 過失層次 | 預見確信＝有認識；該注意沒注意＝無認識 | 有認識vs無認識過失 |
| 中止未遂 | 能達不欲＝中止 | Frank公式 |
| 共犯類型 | 共教幫 | 共同正犯、教唆犯、幫助犯 |
| 競合 | 一行為想像，數行為實質 | 想像競合vs實質競合 |
| 打擊錯誤 | 對目標未遂＋對被害人過失 | 具體符合說結論 |
| 詐欺流程 | 詐錯處損 | 施詐→錯誤→處分→損害 |

## 附錄：考前最後提醒

### 作答注意事項

```
1. 一定要按三階層寫，不要跳階
2. 每個構成要件都要「涵攝」（把題目事實套入法律要件）
3. 有爭點要「正反並呈」再採立場
4. 共犯問題先寫正犯再寫共犯
5. 競合問題不要忘記寫
6. 分則罪名要寫條號
```

### 常見扣分原因

| 扣分原因 | 正確做法 |
|---|---|
| 跳過三階層直接寫結論 | 完整走過該當→違法→罪責 |
| 漏寫主觀構成要件 | 每次都要寫故意/過失的論述 |
| 忘記寫因果關係 | 結果犯一定要檢驗因果與客觀歸責 |
| 錯誤論只寫結論 | 要明確說是哪種錯誤、用哪個學說 |
| 漏寫競合 | 結尾一定要處理罪數與競合 |
