> **預備知識**：第二章（載子濃度）、第三章（漂移與擴散）、第四章（產生復合與連續性方程）

## 🔰 本章基礎觀念（零基礎必讀）

### PN 接面為什麼重要？

PN 接面是幾乎所有半導體元件的基礎構件：

| 元件 | PN 接面的角色 |
|------|-------------|
| 二極體 | 就是一個 PN 接面 |
| 太陽能電池 | PN 接面分離光產生的載子 |
| LED | 在 PN 接面注入載子，輻射復合發光 |
| MOSFET | 源極/汲極和基板之間形成 PN 接面 |
| BJT | 兩個 PN 接面（BE 和 BC） |

### 一分鐘抓住核心

```
P 型 ──│── N 型
(多電洞) │ (多電子)
        │
電洞→→→→│←←←←電子    ← 擴散
        │
   ⊖⊖⊖ │ ⊕⊕⊕        ← 留下離子（空乏區）
        │
←←←←←←←│→→→→→→→     ← 內建電場阻止擴散

結果：形成空乏區 + 內建電位 Vbi
正偏（P接正）→ 空乏區變窄 → 電流大（指數增長）
反偏（P接負）→ 空乏區變寬 → 電流極小（飽和電流 Is）
```

## 一、PN 接面的形成

### 1.1 空乏區的形成過程

當 P 型和 N 型半導體接觸時：

1. **擴散開始**：N 區的電子向 P 區擴散（多→少），P 區的電洞向 N 區擴散
2. **留下離子**：擴散走的載子留下不能移動的離子
   - N 區靠近接面處留下帶正電的施體離子 Nd⁺
   - P 區靠近接面處留下帶負電的受體離子 Na⁻
3. **內建電場形成**：這些離子形成從 N 到 P 的電場
4. **達到平衡**：電場產生的漂移電流剛好抵銷擴散電流

這個沒有自由載子、只有固定離子的區域就是**空乏區（Depletion Region）**，又稱空間電荷區。

```
    P 型          空乏區           N 型
 ─────────┤ ⊖⊖⊖⊖ │ ⊕⊕⊕⊕ ├─────────
 多電洞    │ Na⁻  │ Nd⁺  │    多電子
           │←─xp──│──xn─→│
           │←──── W ────→│
                  ↑
               冶金接面
```

### 1.2 內建電位（Built-in Potential, Vbi）

在平衡下，整個系統只有一個費米能階。由此可推導：

$$V_{bi} = \frac{kT}{q}\ln\left(\frac{N_a N_d}{n_i^2}\right)$$

**Si 在 300K 的典型值**：

| Na (cm⁻³) | Nd (cm⁻³) | Vbi (V) |
|-----------|-----------|---------|
| 10¹⁶      | 10¹⁶      | 0.72    |
| 10¹⁸      | 10¹⁶      | 0.83    |
| 10¹⁸      | 10¹⁸      | 0.95    |

**注意**：Vbi 不能用電壓表直接測量（因為接觸電位差會抵銷）。

### 1.3 空乏區寬度

使用突變接面近似（Abrupt Junction）和電中性條件 NaXp = NdXn：

**總空乏區寬度**：

$$W = \sqrt{\frac{2\varepsilon_s}{q}\left(\frac{1}{N_a} + \frac{1}{N_d}\right)(V_{bi} - V)}$$

其中：
- εs = εr·ε₀：半導體的介電常數（Si：εr = 11.7，εs = 1.04×10⁻¹² F/cm）
- V：外加電壓（正偏 V > 0，反偏 V < 0）

**N 側和 P 側的空乏區寬度**：

$$x_n = W \cdot \frac{N_a}{N_a + N_d}, \quad x_p = W \cdot \frac{N_d}{N_a + N_d}$$

**關鍵觀察**：空乏區主要延伸到**摻雜濃度較低**的那一側。

### 1.4 單邊突變接面（One-Sided Junction）

如果 Na >> Nd（P⁺N 接面）：

$$W \approx x_n \approx \sqrt{\frac{2\varepsilon_s(V_{bi} - V)}{qN_d}}$$

空乏區幾乎全部在輕摻雜的 N 側。

### 1.5 最大電場

空乏區中心（冶金接面處）的電場最大：

$$E_{max} = \frac{qN_d x_n}{\varepsilon_s} = \frac{qN_a x_p}{\varepsilon_s} = \frac{2(V_{bi} - V)}{W}$$

## 二、能帶圖分析

### 2.1 平衡下的能帶圖

```
能量 E ↑
      │    P 型        空乏區         N 型
      │
  Ec  │    ─────╲                ╱─────── Ec
      │          ╲   qVbi      ╱
  Ei  │    ─────  ╲           ╱   ─────── Ei
      │            ╲         ╱
  Ef  │    ════════╗╲═══════╱╔════════════ Ef（平的！）
      │            ╚═╲═════╱═╝
  Ev  │    ─────    ╲     ╱      ─────── Ev
      │              ╲   ╱
      │               ╲ ╱
      └──────────────────────────────→ x
```

**重點**：
- 平衡下費米能階 Ef 是水平的（整個系統一條線）
- 能帶在空乏區彎曲，彎曲量 = qVbi
- P 側能帶較高，N 側較低

### 2.2 正偏下的能帶圖

外加電壓 V > 0（P 接正極），降低了能帶彎曲：

- 位障從 qVbi 降低為 q(Vbi - V)
- 空乏區變窄
- 少數載子注入增加 → 大量電流

### 2.3 反偏下的能帶圖

外加電壓 V < 0（P 接負極），增加了能帶彎曲：

- 位障從 qVbi 增加為 q(Vbi + |V|)
- 空乏區變寬
- 少數載子注入減少 → 極小的反向飽和電流

## 三、理想二極體方程

### 3.1 邊界條件

在空乏區邊界，少數載子濃度由偏壓決定：

$$p_n(x_n) = p_{n0} \exp\left(\frac{V}{V_T}\right) = \frac{n_i^2}{N_d}\exp\left(\frac{V}{V_T}\right)$$

$$n_p(-x_p) = n_{p0} \exp\left(\frac{V}{V_T}\right) = \frac{n_i^2}{N_a}\exp\left(\frac{V}{V_T}\right)$$

### 3.2 少數載子分佈

利用少數載子擴散方程（第四章），在空乏區外：

$$\Delta p_n(x) = \Delta p_n(x_n) \cdot \exp\left(-\frac{x - x_n}{L_p}\right) \quad (x > x_n)$$

$$\Delta n_p(x) = \Delta n_p(-x_p) \cdot \exp\left(\frac{x + x_p}{L_n}\right) \quad (x < -x_p)$$

### 3.3 電流推導

在空乏區邊界計算擴散電流：

$$J_p(x_n) = -qD_p\frac{dp_n}{dx}\bigg|_{x_n} = \frac{qD_p p_{n0}}{L_p}\left[\exp\left(\frac{V}{V_T}\right) - 1\right]$$

$$J_n(-x_p) = qD_n\frac{dn_p}{dx}\bigg|_{-x_p} = \frac{qD_n n_{p0}}{L_n}\left[\exp\left(\frac{V}{V_T}\right) - 1\right]$$

### 3.4 理想二極體方程（Shockley Equation）

$$I = I_s\left[\exp\left(\frac{V}{nV_T}\right) - 1\right]$$

其中**反向飽和電流**：

$$I_s = qA\left(\frac{D_p p_{n0}}{L_p} + \frac{D_n n_{p0}}{L_n}\right) = qA\cdot n_i^2\left(\frac{D_p}{L_p N_d} + \frac{D_n}{L_n N_a}\right)$$

**理想因子 n（Ideality Factor）**：
- n = 1：理想二極體（擴散電流主導）
- n = 2：空乏區復合電流主導
- 實際：1 < n < 2

### 3.5 Is 的溫度依賴性

$$I_s \propto n_i^2 \propto \exp\left(-\frac{E_g}{kT}\right)$$

Is 對溫度非常敏感。每升高約 10°C，Si 二極體的 Is 大約翻倍。

### 3.6 I-V 特性圖

```
I (mA) ↑
       │         ╱  正偏：指數增長
  10   │        ╱
       │       ╱
   5   │      ╱
       │     ╱
   0 ──│────╱─────────── V (V)
  -V   │  0.5  0.7
       │
  -Is  │─────────── 反偏：飽和電流（非常小）
       │
```

**實用記憶**：Si 二極體在正偏 ~0.7V 時「導通」，GaAs ~1.2V。

## 四、PN 接面電容

### 4.1 空乏電容（Junction Capacitance / Depletion Capacitance）

空乏區就像一個平行板電容器：

$$C_j = \frac{\varepsilon_s A}{W} = A\sqrt{\frac{q\varepsilon_s}{2(V_{bi} - V)} \cdot \frac{N_a N_d}{N_a + N_d}}$$

零偏電容：

$$C_{j0} = A\sqrt{\frac{q\varepsilon_s}{2V_{bi}} \cdot \frac{N_a N_d}{N_a + N_d}}$$

反偏時（V < 0）：

$$C_j = \frac{C_{j0}}{(1 - V/V_{bi})^m}$$

其中 m = 1/2（突變接面）或 m = 1/3（線性漸變接面）。

**應用**：
- 變容二極體（Varactor）：利用反偏電壓調整電容 → 用於射頻調諧
- C-V 測量：從 1/C² vs V 的斜率可以求出摻雜濃度

### 4.2 擴散電容（Diffusion Capacitance）

正偏時，空乏區外儲存了大量少數載子，形成擴散電容：

$$C_d = \frac{\tau_p I}{V_T}$$

其中 τp 是少數載子生命期，I 是正偏電流。

- 反偏時 Cd ≈ 0（沒有少數載子儲存）
- 正偏時 Cd >> Cj（因為 I 很大）

### 4.3 總電容

$$C_{total} = C_j + C_d$$

- **反偏**：C ≈ Cj（空乏電容主導）
- **正偏**：C ≈ Cd（擴散電容主導）

## 五、崩潰（Breakdown）

### 5.1 雪崩崩潰（Avalanche Breakdown）

在高反偏下，空乏區中的電場很強。自由載子被加速到足夠能量時，可以撞擊晶格原子，產生新的電子-電洞對（**碰撞游離，Impact Ionization**）。新載子再被加速、再碰撞 → 連鎖反應 → 電流急劇增加。

```
     電子 →  ─→─ 碰撞 ─→ 產生新電子+電洞
                          │
                    新電子 ─→─ 碰撞 ─→ 更多...
                    新電洞 ←─← 碰撞 ←─← 更多...
                    （雪崩效應）
```

**碰撞游離係數**：α（電子）和 β（電洞），單位 cm⁻¹

**雪崩崩潰條件**：

$$\int_0^W \alpha \, dx = 1 \quad \text{（簡化條件，α = β 時）}$$

**崩潰電壓與摻雜的關係**：

$$V_{BR} \propto \frac{E_{crit}^2}{N_{light}} \propto \frac{1}{N_{light}}$$

輕摻雜側摻雜越低，崩潰電壓越高。

| 材料 | 臨界電場 Ecrit (V/cm) | 說明 |
|------|---------------------|------|
| Si   | ~3 × 10⁵           | - |
| GaAs | ~4 × 10⁵           | 稍高 |
| SiC  | ~3 × 10⁶           | 10 倍！功率元件優勢 |
| GaN  | ~3.3 × 10⁶         | 同樣很高 |

### 5.2 齊納崩潰（Zener Breakdown）

在**高摻雜**的 PN 接面中，空乏區很窄。當反偏電壓增加到某個值時，電子可以直接**量子穿隧（Quantum Tunneling）**從 P 側價帶穿過窄的空乏區到達 N 側導帶。

```
     P 側 VB          N 側 CB
     ─────╲          ╱─────
           ╲        ╱
            ╲──→→──╱   ← 電子穿隧
             ╲    ╱       （空乏區很窄）
              ╲  ╱
               ╲╱
```

**齊納 vs 雪崩的區分**：

| 特性 | 齊納崩潰 | 雪崩崩潰 |
|------|---------|---------|
| 摻雜濃度 | 高（Na, Nd > ~5×10¹⁷） | 低到中 |
| 崩潰電壓 | < ~5V | > ~7V |
| 空乏區寬度 | 窄 | 寬 |
| 機制 | 量子穿隧 | 碰撞游離 |
| 溫度係數 | 負（VBR ↓ as T ↑） | 正（VBR ↑ as T ↑） |

5V~7V 之間的二極體通常兩種機制都有，溫度係數接近零 → 適合做電壓參考。

## 六、異質接面（Heterojunction）

### 6.1 基本概念

由兩種不同半導體材料組成的接面，例如 AlGaAs/GaAs 或 Si/SiGe。

$$\Delta E_c = \chi_1 - \chi_2 \quad \text{（導帶偏移，Conduction Band Offset）}$$
$$\Delta E_v = (E_{g2} - E_{g1}) - \Delta E_c \quad \text{（價帶偏移）}$$

其中 χ 是電子親和力（Electron Affinity）。

### 6.2 異質接面的優勢

1. **能帶工程**：可以設計能帶形狀，控制載子注入
2. **高效 LED/雷射**：用寬能隙材料包夾窄能隙材料（雙異質接面），將載子侷限在活性層
3. **HBT（異質接面雙極電晶體）**：用寬能隙的射極提高注入效率
4. **HEMT（高電子遷移率電晶體）**：在異質接面形成 2DEG（二維電子氣）

### 6.3 Type I vs Type II 異質接面

- **Type I（Straddling）**：窄能隙完全在寬能隙內 → LED、雷射常用
- **Type II（Staggered）**：兩個能隙錯開 → 特殊光偵測器
- **Type III（Broken）**：能帶完全不重疊 → 穿隧元件

## 數值例題

### 【例題 1】內建電位計算

**題目**：Si PN 接面，Na = 10¹⁸ cm⁻³，Nd = 10¹⁶ cm⁻³，T = 300K。求 Vbi。

**解答**：

```
Vbi = (kT/q) × ln(NaNd/ni²)
    = 0.026 × ln(10¹⁸ × 10¹⁶ / (1.5×10¹⁰)²)
    = 0.026 × ln(10³⁴ / 2.25×10²⁰)
    = 0.026 × ln(4.44×10¹³)
    = 0.026 × 31.42
    = 0.817 V

答案：Vbi ≈ 0.82 V
```

---

### 【例題 2】空乏區寬度

**題目**：承上題，求零偏和反偏 V = -5V 時的空乏區寬度。

**解答**：

```
εs = 11.7 × 8.854×10⁻¹⁴ = 1.036×10⁻¹² F/cm

因為 Na >> Nd（P⁺N 接面），W ≈ xn

零偏（V = 0）：
W = √(2εs × Vbi / (q × Nd))
  = √(2 × 1.036×10⁻¹² × 0.817 / (1.6×10⁻¹⁹ × 10¹⁶))
  = √(1.692×10⁻¹² / 1.6×10⁻³)
  = √(1.058×10⁻⁹)
  = 3.25 × 10⁻⁵ cm = 0.325 μm

反偏 V = -5V：
W = √(2εs × (Vbi + 5) / (q × Nd))
  = √(2 × 1.036×10⁻¹² × 5.817 / (1.6×10⁻¹⁹ × 10¹⁶))
  = √(1.205×10⁻¹¹ / 1.6×10⁻³)
  = √(7.531×10⁻⁹)
  = 8.68 × 10⁻⁵ cm = 0.868 μm

比值：W(反偏)/W(零偏) = 0.868/0.325 = 2.67
```

反偏使空乏區寬度增加了 2.67 倍。

---

### 【例題 3】理想二極體電流

**題目**：Si 二極體，Is = 10⁻¹⁴ A，n = 1，T = 300K。求 V = 0.6V 和 V = 0.7V 時的電流。

**解答**：

```
I = Is × [exp(V/nVT) - 1]

V = 0.6V：
I = 10⁻¹⁴ × [exp(0.6/0.026) - 1]
  = 10⁻¹⁴ × [exp(23.08) - 1]
  = 10⁻¹⁴ × 1.058×10¹⁰
  = 1.058 × 10⁻⁴ A = 0.106 mA

V = 0.7V：
I = 10⁻¹⁴ × [exp(0.7/0.026) - 1]
  = 10⁻¹⁴ × [exp(26.92) - 1]
  = 10⁻¹⁴ × 4.85×10¹¹
  = 4.85 × 10⁻³ A = 4.85 mA
```

**觀察**：電壓只增加 0.1V（從 0.6V 到 0.7V），電流增加了 **46 倍**！這就是指數特性的威力。每增加 60mV，電流增加約 10 倍（理想因子 n=1 時）。

---

### 【例題 4】反向飽和電流 Is

**題目**：Si PN 接面，A = 10⁻⁴ cm²，Na = 10¹⁸，Nd = 10¹⁶，Dp = 10 cm²/s，Dn = 25 cm²/s，τp = 1 μs，τn = 5 μs，T = 300K。求 Is。

**解答**：

```
Lp = √(Dp·τp) = √(10 × 10⁻⁶) = 3.16×10⁻³ cm = 31.6 μm
Ln = √(Dn·τn) = √(25 × 5×10⁻⁶) = 1.118×10⁻² cm = 112 μm

pn0 = ni²/Nd = (1.5×10¹⁰)²/10¹⁶ = 2.25×10⁴ cm⁻³
np0 = ni²/Na = (1.5×10¹⁰)²/10¹⁸ = 2.25×10² cm⁻³

Is = qA × (Dp·pn0/Lp + Dn·np0/Ln)
   = 1.6×10⁻¹⁹ × 10⁻⁴ × (10×2.25×10⁴/3.16×10⁻³ + 25×2.25×10²/1.118×10⁻²)
   = 1.6×10⁻²³ × (7.12×10⁷ + 5.03×10⁵)
   = 1.6×10⁻²³ × 7.17×10⁷
   = 1.15×10⁻¹⁵ A ≈ 1.15 fA
```

**觀察**：Is 主要由 N 側（輕摻雜側）的電洞擴散電流貢獻（7.12×10⁷ >> 5.03×10⁵）。這是因為輕摻雜側的少數載子濃度較高。

### 【例題 5】空乏電容

**題目**：承例題 1/2，求零偏空乏電容（A = 10⁻⁴ cm²），以及 V = -5V 時的電容。

**解答**：

```
零偏：
Cj0 = εs × A / W(0) = 1.036×10⁻¹² × 10⁻⁴ / 3.25×10⁻⁵
    = 3.19×10⁻¹² F = 3.19 pF

V = -5V：
Cj = εs × A / W(-5) = 1.036×10⁻¹² × 10⁻⁴ / 8.68×10⁻⁵
   = 1.19×10⁻¹² F = 1.19 pF

或用公式：
Cj = Cj0 / √(1 - V/Vbi) = 3.19 / √(1 + 5/0.817)
   = 3.19 / √(7.12) = 3.19 / 2.67 = 1.19 pF ✓
```

### 【例題 6】崩潰電壓估算

**題目**：Si 單邊 P⁺N 接面，Nd = 10¹⁵ cm⁻³，臨界電場 Ecrit = 3×10⁵ V/cm。估算崩潰電壓。

**解答**：

```
在崩潰時，Emax = Ecrit

Emax = qNd·xn/εs，且 VBR ≈ Emax·W/2 = Emax·xn/2（P⁺N接面）

從 Emax = qNd·xn/εs 得：
xn = Ecrit·εs/(qNd) = 3×10⁵ × 1.036×10⁻¹² / (1.6×10⁻¹⁹ × 10¹⁵)
   = 3.108×10⁻⁷ / 1.6×10⁻⁴
   = 1.94×10⁻³ cm = 19.4 μm

VBR ≈ Ecrit × xn / 2 = 3×10⁵ × 1.94×10⁻³ / 2
    = 291 V

答案：VBR ≈ 291 V
```

如果 Nd = 10¹⁷ cm⁻³：
VBR ≈ 291 × (10¹⁵/10¹⁷) = 2.91 V → 摻雜越高，崩潰電壓越低。

## ✅ 自我檢測

### 基礎題

**Q1**：空乏區在哪一側比較寬？為什麼？

<details>
<summary>點擊查看答案</summary>

空乏區在**輕摻雜側**比較寬。

原因：電荷中性要求 Na·xp = Nd·xn。如果 Na >> Nd，則 xn >> xp。

物理理解：輕摻雜側的離子濃度低，需要更大的空間才能提供足夠的電荷來平衡對面的電荷。

</details>

**Q2**：正偏時電流為什麼那麼大？

<details>
<summary>點擊查看答案</summary>

正偏降低了 PN 接面的位障（從 qVbi 降到 q(Vbi-V)），使得大量多數載子可以跨過接面：

- P 側的電洞大量注入 N 側
- N 側的電子大量注入 P 側

這些注入的少數載子在對面擴散並復合，形成持續的擴散電流。由於位障的降低是指數效果，電流與電壓是指數關係：I ∝ exp(V/VT)。

</details>

**Q3**：反向飽和電流 Is 主要由哪一側決定？

<details>
<summary>點擊查看答案</summary>

Is 主要由**輕摻雜側**決定。

因為 Is = qAni²(Dp/LpNd + Dn/LnNa)，較小的 Nd（或 Na）對應較大的項。

例如 P⁺N 接面（Na >> Nd）：Is ≈ qAni²Dp/(LpNd)

輕摻雜側的少數載子濃度高（pn0 = ni²/Nd 較大），所以注入的少數載子多，Is 大。

</details>

### 進階題

**Q4**：如何從 C-V 測量中提取摻雜濃度？

<details>
<summary>點擊查看答案</summary>

對於單邊突變接面（如 P⁺N）：

1/Cj² = 2(Vbi - V) / (qεsNdA²)

畫 **1/Cj² vs V** 的圖，是一條直線：
- **斜率** = -2/(qεsNdA²) → 可算出 Nd
- **x 截距** = Vbi → 可算出內建電位

這是半導體材料分析中最常用的技術之一。

</details>

**Q5**：為什麼 SiC 功率元件的崩潰電壓比 Si 高？

<details>
<summary>點擊查看答案</summary>

SiC 的臨界電場 Ecrit ≈ 3×10⁶ V/cm，是 Si（3×10⁵ V/cm）的 **10 倍**。

崩潰電壓 VBR ∝ Ecrit² / N

所以在相同摻雜下，SiC 的 VBR 是 Si 的 ~100 倍（10²）。

或者反過來說，同樣的 VBR，SiC 可以用高 10 倍的摻雜 → 導通電阻降低 10 倍。

這就是 SiC 在電動車功率元件中的巨大優勢。

</details>

**Q6**：擴散電容 Cd 在什麼情況下重要？它為什麼限制二極體的高頻應用？

<details>
<summary>點擊查看答案</summary>

Cd 在**正偏**時重要。Cd = τI/VT，與正偏電流 I 成正比。

高頻應用中的問題：
- 正偏到反偏切換時，儲存在空乏區外的少數載子必須先被移除（復合或漂移走）
- 這需要時間 ~τ（少數載子壽命）
- 在這段「反向恢復時間」內，二極體仍然導電
- 等效地，大的 Cd 限制了開關速度

解決方案：
- 使用 Schottky 二極體（沒有少數載子注入，Cd ≈ 0）
- 加入復合中心降低 τ（如金摻雜 Si）

</details>

## 本章重點整理

```
1. PN接面形成空乏區：擴散→離子→電場→平衡
2. Vbi = (kT/q)ln(NaNd/ni²)，Si典型值 ~0.7-0.9V
3. W = √(2ε(Vbi-V)(1/Na+1/Nd)/q)，反偏W增大
4. 空乏區主要在輕摻雜側
5. I = Is[exp(V/nVT)-1]，n=1理想，n=2有復合
6. Is = qAni²(Dp/LpNd + Dn/LnNa) ∝ ni² ∝ exp(-Eg/kT)
7. 空乏電容 Cj = εA/W（反偏用），反偏越深電容越小
8. 擴散電容 Cd = τI/VT（正偏用），限制高頻應用
9. 雪崩崩潰：碰撞游離，低摻雜，VBR > 7V
10. 齊納崩潰：量子穿隧，高摻雜，VBR < 5V
11. 異質接面：ΔEc = χ₁-χ₂，用於LED、雷射、HEMT
```
